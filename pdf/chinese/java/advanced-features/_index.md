## 什么是 pdf 可选内容组？
pdf 可选内容组（OCG）是 PDF 图层背后的技术术语。它们允许您将页面对象分组，以便查看器能够独立切换每个组的可见性。这使得构建多语言 PDF、水印或交互式表单成为可能，在不同条件下显示不同内容。

## 为什么使用 pdf 可选内容组？
- **动态呈现：** 根据用户偏好或设备能力显示或隐藏内容。  
- **减小文件体积：** 一个 PDF 可以包含多个内容版本，而无需复制整个文档。  
- **提升可访问性：** 将 OCG 与标签结合，并运行 **pdf 可访问性验证**，以改善屏幕阅读器的导航。  
- **更好的 SEO：** 带标签的分层 PDF 更易被搜索引擎索引，尤其是在配合适当的元数据时。

## 常见问题

**Q: 我可以在 PDF 创建后添加或删除图层吗？**  
A: 可以。Aspose.PDF 允许您随时修改可选内容组，实现动态更新而无需重新生成整个文档。

**Q: PDF 图层会影响文件大小吗？**  
A: 图层共享相同的页面资源，通常只会带来相对于为每个版本单独存储 PDF 的轻微开销。

**Q: 所有查看器都支持 PDF 图层吗？**  
A: 大多数现代 PDF 查看器（Adobe Acrobat、Foxit、PDF.js）都支持可选内容组，但老旧或轻量级查看器可能会忽略它们。

**Q: 使用图层需要设置特殊权限吗？**  
A: 不需要特殊权限；图层是 PDF 规范的一部分，像其他页面内容一样处理。

**Q: 我可以将图层与 PDF/UA 标记结合，以提升可访问性吗？**  
A: 完全可以。您可以为每个图层的内容添加标签，使屏幕阅读器能够更有效地导航分层文档。

**Q: pdf 可访问性验证在分层 PDF 中如何工作？**  
A: Aspose.PDF 对整个文档进行 pdf 可访问性验证检查，包括每个可选内容组，以确保符合 PDF/UA 标准。

**Q: 渲染大量图层会影响性能吗？**  
A: 渲染大量 OCG 可能会增加处理时间，但 Aspose.PDF 会优化资源共享，在大多数使用场景下保持可接受的性能。

## 安装说明
要开始使用 Aspose.PDF for .NET，请遵循以下安装说明：

1. 下载 Aspose.PDF： 
   - 访问 [Aspose 网站](https://www.aspose.com/downloads/pdf/net) 下载最新版本的 Aspose.PDF for .NET。

2. 通过 NuGet 包管理器安装：
   - 在 Visual Studio 中打开您的项目。
   - 转到工具>NuGet 包管理器>管理解决方案的 NuGet 包。
   - 搜索“Aspose.PDF”并安装该包。

3. 添加参考文献：
   - 确保您的项目引用了 Aspose.PDF DLL `packages` 文件夹。
   
4. 在您的项目中包含 Aspose.PDF 命名空间：  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

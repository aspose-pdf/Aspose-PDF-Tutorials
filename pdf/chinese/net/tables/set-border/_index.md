## 装配说明书
为了开始使用Aspose.PDF在.NET中,请按照以下安装说明:

1. 接下来我们将把它转换成一个新的版本. 
   * 访问[作为网站](https://www.aspose.com/downloads/pdf/net)为了下载最新版本的Aspose.PDF

2. 通过NuGet包管理器安装:
   * 在视觉工作室打开你的项目.
   * 进入工具> 云端包管理器> 管理云端包解决方案.
   * 搜索Aspose.PDF并安装该包.

3. 添加参考:
   * 确保您的项目引用了Aspose.PDFDLL`包装`文件.
   
4. 在您的项目中包含Aspose.PDF命名空间:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

# 使用 Aspose.PDF for .NET 在 PDF 中添加下划线文本

Aspose.PDF for .NET 是一个功能强大的 PDF 处理库，允许开发人员无需 Adobe Acrobat 即可创建、修改和操作 PDF 文档。本示例演示了如何使用 TextBuilder 和 TextFragment 类创建新的 PDF 文档并添加带有下划线格式的文本。该代码创建了一个简单的 PDF，其中包含一条 Arial 字体格式的测试消息，并带有下划线样式，并将其定位在页面中的特定坐标处。

## 安装说明

1：安装 Aspose.PDF for .NET

您可以使用以下方法之一安装 Aspose.PDF for .NET：

使用 NuGet 包管理器：

```csharp
Install-Package Aspose.PDF
```

使用 .NET CLI：

```bash
dotnet add package Aspose.PDF
```

在项目文件中使用 PackageReference：

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2：添加必需的 Using 语句

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

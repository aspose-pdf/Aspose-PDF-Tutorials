# 添加下划线文本到PDF中使用Aspose.PDF

Aspose.PDF为.NET提供了一个强大的PDF处理库,允许开发人员在不需要Adobe Acrobat的情况下创建,修改和操纵PDF文档.本示例展示了如何创建一个新的PDF文档并使用TextBuilder和TextFragment类添加下划线格式化的文本.代码创建了一个简单的PDF,以Arial字体格式化的测试消息,并将其定位在页面内的特定坐标.

## 装配说明书

1,安装Aspose.PDF用于.NET

您可以使用以下方法之一安装Aspose.PDF

使用Nuget包管理器:

```csharp
Install-Package Aspose.PDF
```

使用.NET CLI:

```bash
dotnet add package Aspose.PDF
```

在您的项目文件中使用PackageReference:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: 添加需要的使用语句

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

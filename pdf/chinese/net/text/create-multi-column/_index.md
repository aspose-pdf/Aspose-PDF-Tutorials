## 装配说明书  

为了在项目中使用Aspose.PDF为.NET,请按照以下步骤:  

1. 通过NuGet安装 (推)  
在 NuGet 包管理器控制台中运行下面命令:  
```powershell
Install-Package Aspose.PDF
```

2. 通过.NET CLI 安装  
对于.NET Core或.NET 5+使用:  
```bash
dotnet add package Aspose.PDF
```

3. 在安装后,导入项目中所需的命名空间:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

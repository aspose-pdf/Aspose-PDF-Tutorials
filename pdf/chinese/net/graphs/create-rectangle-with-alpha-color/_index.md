## 装配说明书  
为了在项目中使用Aspose.PDF为.NET,请按照以下步骤:  

1. 通过NuGet安装  
在 NuGet 包管理器控制台中运行下面命令:  
```powershell
Install-Package Aspose.PDF
```

2. 在C#中添加名字空间  
在C#文件的开始,包含所需的命名空间:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

详细的文件请访问:  
[对于.NET 文档,使用Aspose.PDF](https://docs.aspose.com/pdf/net/)  
[对于.NET API 参考,使用Aspose.PDF](https://reference.aspose.com/pdf/net/)

## 装备说明:  

为了使用 .NET 的 Aspose.PDF,请按照以下步骤进行:  

1. 通过 NuGet 包管理器安装  
* 打开视觉工作室,创建一个新的 .NET项目.  
* 打开包管理器控制台 (工具 → NuGet包管理器 →包管理器控制台) 然后运行:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. 通过 .NET CLI 安装  
另一个选择是使用 .NET CLI 安装包:  
```bash
dotnet add package Aspose.PDF
```

3. 在C#代码中添加名字空间  
为了开始使用Aspose.PDF,在C#文件中添加以下名字空间:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

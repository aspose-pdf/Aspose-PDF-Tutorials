## 装配说明:  

为了在项目中使用Aspose.PDF为.NET,请按照以下步骤:  

1. 通过NuGet包管理器安装  
* 打开视觉工作室,创建一个新的.NET项目.  
* 打开包管理器控制台 (工具 → NuGet包管理器 →包管理器控制台) 并运行:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. 通过.NET CLI 安装  
另一个选择是使用.NET CLI 安装该包:  
```bash
dotnet add package Aspose.PDF
```

3. 在您的C#代码中添加命名空间  
要使用Aspose.PDF功能,请包含以下名称空间:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Aspose.PDF for .NET 使开发人员能够轻松创建、操作和编辑 PDF 文档。在本例中，我们将演示如何使用 C# 以编程方式在 PDF 页面上绘制两条对角线。您将学习如何初始化 PDF 文档、设置页面属性、添加图形元素以及保存最终输出。本指南将提供分步指导，让您轻松地将此功能集成到您的 .NET 应用程序中。

## 安装说明：  

要在您的项目中使用 Aspose.PDF for .NET，请按照以下步骤操作：  

1. 通过 NuGet 包管理器安装  
- 打开 Visual Studio 并创建一个新的 .NET 项目。  
- 打开包管理器控制台（工具 → NuGet 包管理器 → 包管理器控制台）并运行：  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. 通过 .NET CLI 安装  
或者，使用 .NET CLI 安装包：  
```bash
dotnet add package Aspose.PDF
```

3. 在 C# 代码中添加命名空间  
要使用 Aspose.PDF 功能，请包括以下命名空间：  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

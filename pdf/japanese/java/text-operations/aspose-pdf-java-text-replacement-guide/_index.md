ステップ1：PDF文書の読み込み\n\nまず、ソースPDFファイルを読み込みます。これは、 `Document` クラス:\n\n```java\nimport com.aspose.pdf.Document;\n\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nDocument doc = new Document(dataDir + \"/input.pdf\");\n```\n\n#### ステップ2: TextFragment Absorberを作成する\n\n `TextFragmentAbsorber` 正規表現パターンに一致するテキストを検索します。これにより、ドキュメント内で柔軟な検索が可能になります。\n\n```java\nimport com.aspose.pdf.TextFragmentAbsorber;\n\nTextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(\"[Cname,companyname,Textbox,50]\");\ndoc.getPages().accept(textFragmentAbsorber);\n```\n\n#### ステップ3: テキストの反復処理と置換\n\n見つかったテキスト断片をループ処理して、目的の文字列に置き換えます。ここでフォントのプロパティもカスタマイズします:\n\n```java\nimport com.aspose.pdf.FontRepository;\nimport com.aspose.pdf.Color;\n\nfor (TextFragment textFragment : (Iterable<TextFragment>) textFragmentAbsorber.getTextFragments()) {\n    textFragment.getTextState().setFont(FontRepository.findFont(\"Arial\"));\n    textFragment.getTextState().setFontSize(12);\n    textFragment.getTextState().setForegroundColor(Color.getNavy());\n    textFragment.setText(\"This is a Larger String to Testing of this issue\");\n}\n```\n\n#### ステップ 4: 変更した PDF を保存する\n\n変更を加えたら、ドキュメントを保存します:\n\n```java\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\ndoc.save(outputDir + \"/29860_out_large_NoHyphenation_1020.pdf\");\n```\n\n### 機能 2: テキスト置換調整 (オプション)\n\n最初の例ではコメントアウトされていますが、この機能はハイフネーション オプションを使用してテキスト置換を調整する方法を示しています:\n\n```java\nimport com.aspose.pdf.TextReplaceOptions;\n\n// Load source PDF file\nDocument doc = new Document(dataDir + \"/input.pdf\");\n\nTextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(\"[Cname,companyname,Textbox,50]\");\ndoc.getPages().accept(textFragmentAbsorber);\n\ntextFragmentAbsorber.getTextReplaceOptions().setReplaceAdjustmentAction(TextReplaceOptions.ReplaceAdjustment.WholeWordsHyphenation);\n```\n\nこれは、テキストがドキュメントのレイアウト制約内にきちんと収まるようにするのに役立ちます。\n\n##

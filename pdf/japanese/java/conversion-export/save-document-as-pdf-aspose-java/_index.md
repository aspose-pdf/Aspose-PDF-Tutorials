### 文書をPDFとして保存する
**概要：** この機能を使用すると、現在の状態を保存できます。 `Document` オブジェクトをディスク上の PDF ファイルに変換し、永続的に保存して共有できるようにします。

#### 手順:
1. **ドキュメントを準備する:** 既存の `Document` すべての変更が適用されたインスタンス。
```java
// 「document」はページとコンテンツを持つ既に変更されたDocumentオブジェクトであると仮定します。
```
2. **出力ディレクトリを指定します:** PDF ファイルを保存するパスを設定します。
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // 希望するパスに置き換えます
```
3. **ドキュメントを保存します:** 使用 `save` ドキュメントを指定された場所に書き込むメソッド。
```java
// 文書をPDFファイルとして保存する
document.save(outputDir + "/HelloWorld_out.pdf");
```
**説明：**
その `outputDir` 変数はターゲットディレクトリのパスを保持し、 `save` このメソッドは、ドキュメントの現在の状態を「HelloWorld_out.pdf」という名前の PDF に書き込みます。

#### トラブルシューティングのヒント:
- **ファイルパスの問題:** 出力ディレクトリが存在し、書き込み可能であることを確認してください。ファイルI/O操作に関連する例外を処理します。
- **ライブラリバージョンの互換性:** 予期しない動作が発生した場合は、Aspose.PDF ライブラリのバージョンを再確認してください。

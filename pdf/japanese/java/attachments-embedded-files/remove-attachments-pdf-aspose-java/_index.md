PDF文書からすべての添付ファイルを削除する\nこの機能では、PDF文書に埋め込まれたすべての添付ファイルを削除する方法を説明します。手順を一つずつ説明します。\n\n#### ステップ1：PDF文書を読み込む\nまず、 `Document` Aspose.PDF によって提供されるクラス:\n\n```java\n// Open the specified PDF

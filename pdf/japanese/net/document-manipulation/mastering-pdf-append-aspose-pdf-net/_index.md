## 実装ガイド
PDF ファイルを追加するプロセスを明確な手順で説明します。

### PdfFileEditorオブジェクトを作成する
その `PdfFileEditor` クラスは追加操作の中心であり、複数の PDF を簡単に結合する方法を提供します。

#### 概要
この機能では、2つ以上のPDFファイルを結合するために、 `PdfFileEditor` クラスとそのメソッドを使用します。

#### 実装手順
**ステップ1: ファイルパスを設定する**
すべてのファイル操作が正しく実行されるように、入力 PDF ファイルと出力 PDF ファイルのパスを定義します。
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Pages();
```

**ステップ2: PdfFileEditorインスタンスを作成する**
初期化する `PdfFileEditor` 追加操作のオブジェクト。
```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**ステップ3: PDFファイルを追加する**
使用 `Append` どのページを含めるかを指定して、PDF を 1 つの出力ファイルに結合する方法:
```csharp
pdfEditor.Append(dataDir + "input.pdf", dataDir + "input2.pdf", 1, 1, dataDir + "AppendFiles_out.pdf");
```

**パラメータの説明:**
- **データディレクトリ + "input.pdf"**: 最初の PDF ファイルへのパス。
- **データディレクトリ + "input2.pdf"**: 2 番目の PDF ファイルへのパス。
- **1、1**追加するページを指定するページインデックス。ここでは、各入力ファイルの最初のページのみを追加します。
- **データディレクトリ + "AppendFiles_out.pdf"**: 結合された出力ファイルのパス。

### トラブルシューティングのヒント
- すべてのファイル パスが正しく、アクセス可能であることを確認します。
- 入力 PDF が破損していないか、編集を妨げる権限でロックされていないことを確認します。

## 実装ガイド

### ラジオボタンフィールドの作成と追加

#### 概要
このセクションでは、C#を使用してPDF文書にラジオボタンフィールドを作成する方法を説明します。 `RadioButtonField`オプションを追加してフォームに添付します。

**ステップ1: ドキュメントオブジェクトのインスタンス化**
まずは新規作成 `Document` オブジェクト。これは PDF コンテンツのコンテナとして機能します。
```csharp
// 新しいPDF文書を作成する
Document pdfDocument = new Document();
```

**ステップ2: ドキュメントにページを追加する**
フィールドを配置する前に、ページを追加する必要があります。
```csharp
// 空白ページを追加する
pdfDocument.Pages.Add();
```

**ステップ3: RadioButtonFieldオブジェクトのインスタンス化**
その `RadioButtonField` オブジェクトはラジオボタングループを表します。作成時にページ番号を指定してください。
```csharp
// ページ1に新しいラジオボタンフィールドを作成します
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

**ステップ4: ラジオボタンにオプションを追加する**
ラジオボタンフィールド内に複数のオプションを定義し、それらの位置を次のように指定します。 `Rectangle` オブジェクト。
```csharp
// 特定の位置で最初のオプションを追加する
radio.AddOption("Test", new Rectangle(0, 0, 20, 20));

// 別の場所に2番目のオプションを追加する
radio.AddOption("Test1", new Rectangle(20, 20, 40, 40));
```

**ステップ5: ラジオボタンフィールドをフォームに追加する**
ラジオ ボタン フィールドをドキュメントのフォーム コレクションに追加します。
```csharp
// PDFフォームにラジオボタンフィールドを追加する
pdfDocument.Form.Add(radio);
```

**ステップ6: ドキュメントを保存する**
設定が完了したら、すべての要素をそのままにしてドキュメントを保存します。
```csharp
// ファイルパスを定義してドキュメントを保存する
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();
dataDir += "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```

## 設置説明書  

この手順を実行してください.  

1. NuGet でインストール (推奨)  
NuGet パッケージマネージャー コンソールで次のコマンドを実行します:  
```powershell
Install-Package Aspose.PDF
```

2. .NET CLI でインストール  
.NET Core または .NET 5+ では,以下を使用します.  
```bash
dotnet add package Aspose.PDF
```

3. プロジェクトに必要な名前空間をインポートします.  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

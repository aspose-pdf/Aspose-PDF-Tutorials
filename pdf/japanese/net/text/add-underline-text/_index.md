# Aspose.PDF for .NET で PDF に下線付きテキストを追加する

Aspose.PDF for .NETは、Adobe Acrobatを必要とせずにPDFドキュメントを作成、変更、操作できる強力なPDF処理ライブラリです。この例では、TextBuilderクラスとTextFragmentクラスを使用して、新しいPDFドキュメントを作成し、下線付きのテキストを追加する方法を示します。このコードは、Arialフォントで下線付きのテキストを書式設定し、ページ内の特定の座標に配置するシンプルなPDFを作成します。

## インストール手順

1: Aspose.PDF for .NET をインストールする

次のいずれかの方法で Aspose.PDF for .NET をインストールできます。

NuGet パッケージ マネージャーの使用:

```csharp
Install-Package Aspose.PDF
```

.NET CLI の使用:

```bash
dotnet add package Aspose.PDF
```

プロジェクト ファイルで PackageReference を使用する:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: 必要なusingステートメントを追加する

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

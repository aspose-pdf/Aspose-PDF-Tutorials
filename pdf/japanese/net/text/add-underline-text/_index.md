# Aspose.PDFで PDF に強調されたテキストを追加する.

Aspose.PDFは,Adobe Acrobat を必要としなく,PDF 文書を作成,修正,操作する開発者に可能にする強力な PDF 処理ライブラリです.この例では,新しい PDF 文書を作成し,TextBuilder と TextFragment クラスを使用して下書きフォーマットでテキストを追加する方法を示しています.コードは,ページ内の特定の座標に位置付け,下書きスタイリングで Arial フォントでフォーマットされたテストメッセージのシンプルな PDF を作成します.

## 設置説明書

インストールする

インストールするには,次の方法のいずれかを使用します.

NuGet パッケージマネージャを使用する:

```csharp
Install-Package Aspose.PDF
```

.NET CLI を使用する:

```bash
dotnet add package Aspose.PDF
```

パッケージ参照をプロジェクトファイルで使用する:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: 必要な使用式を追加する

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

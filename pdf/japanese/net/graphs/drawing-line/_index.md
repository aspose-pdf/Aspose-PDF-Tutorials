# .NET のAspose.PDF を使って,PDF で Diagonal Line を 描く

## 設置指示:  

プロジェクトで .NET のAspose.PDFを使用するには,次の手順を実行してください.  

1. NuGet パッケージ管理者でインストールする  
* Visual Studioを開いて新しい .NET プロジェクトを作成します  
* パッケージ管理者コンソール (ツール → NuGet パッケージ管理者 → パッケージ管理者コンソール) を開いて実行します:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. .NET CLI を介してインストールする  
代替的に,パッケージをインストールするには .NET CLI を使用してください.  
```bash
dotnet add package Aspose.PDF
```

3. C#コードに名前空間を追加する  
Aspose.PDF機能を使用するには,次の名前空間を入力してください.  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

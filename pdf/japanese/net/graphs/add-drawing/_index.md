## 設置指示:  

.NET のAspose.PDFを使用するには,次の手順を実行してください.  

1. NuGet パッケージ管理者でインストールする  
* Visual Studioを開いて新しい .NET プロジェクトを作成します  
* パッケージ管理者コンソール (ツール → NuGet パッケージ管理者 → パッケージ管理者コンソール) を開いて実行します:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. .NET CLI を介してインストールする  
代替的に, .NET CLI を使用してパッケージをインストールします:  
```bash
dotnet add package Aspose.PDF
```

3. C#コードに名前空間を追加する  
C#ファイルに次の名前空間の追加をします.  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

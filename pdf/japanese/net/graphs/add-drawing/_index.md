## 設置説明書:  

.NET でAspose.PDFを使用するには,次の手順を実行します:  

1. NuGet パッケージマネージャーでインストール  
* Visual Studioを開いて新しい .NET プロジェクトを作成します.  
* パッケージマネージャコンソールを開いて (Tools → NuGet パッケージマネージャ → パッケージマネージャ コンソール) を実行します.  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. .NET CLI でインストール  
代替として, .NET CLI を使ってパッケージをインストールします.  
```bash
dotnet add package Aspose.PDF
```

3. C#コードに名前空間に追加  
C#ファイルに次の名前空間を追加します.  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

### PDFからフォントを抽出する
#### 概要
ここでの主な目的は、特定の PDF ファイルで使用されているすべてのフォントを抽出することです。これは、ドキュメントの一貫性を監査したり、特定のワークフロー用にファイルを準備したりするために重要です。

#### ステップバイステップの実装
**1. ドキュメントを読み込む**
まず、PDF文書を `Document` クラス。

```csharp
using Aspose.Pdf;

// ドキュメントオブジェクトを初期化する
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
Document doc = new Document(dataDir + "input.pdf");
```

**2. フォントの抽出**
活用する `FontUtilities.GetAllFonts()` ドキュメントで使用されているフォントの配列を取得するメソッド。

```csharp
Aspose.Pdf.Text.Font[] fonts = doc.FontUtilities.GetAllFonts();

foreach (Aspose.Pdf.Text.Font font in fonts)
{
    Console.WriteLine(font.FontName);
}
```

**説明：**
- `FontUtilities.GetAllFonts()`: このメソッドは PDF をスキャンし、使用されているすべての一意のフォントの配列を返します。
- ループは各フォントを反復処理し、その名前をコンソールに出力します。

#### 主要な設定オプション
この例では基本的な抽出に重点を置いていますが、Aspose.PDF では、フォント サブセットの指定や埋め込みフォントの最適化など、より高度な使用ケース向けにさまざまな構成オプションを提供しています。

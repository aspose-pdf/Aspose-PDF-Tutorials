# 添付ファイルを使ったプログラミング

Aspose.PDF for .NET の添付ファイルを使ったプログラミングチュートリアルでは、PDF ドキュメント内の添付ファイルの操作方法をステップバイステップで解説します。添付ファイルの追加、抽出、削除、そしてプロパティの管理方法を学びましょう。

これらのチュートリアルでは、PDF ドキュメント内の添付ファイルを操作するためのさまざまな手順を説明し、PDF ファイルの機能とインタラクティブ性を向上させることができます。

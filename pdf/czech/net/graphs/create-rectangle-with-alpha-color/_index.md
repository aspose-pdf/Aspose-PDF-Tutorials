# Vytvořte PDF s obdélníky a alfa barvami v Aspose.PDF

Aspose.PDF pro .NET je výkonná knihovna pro vytváření, manipulaci a převod PDF dokumentů v jazyce C#. V tomto příkladu se naučíte, jak vytvořit PDF soubor a kreslit obdélníky s alfa barvami. Používáme... `Aspose.Pdf.Drawing.Graph` třída pro přidávání tvarů a aplikaci efektů průhlednosti pomocí barev ARGB. Tento tutoriál poskytuje podrobný návod s ukázkovým kódem, který vám zajistí snadnou implementaci ve vašich aplikacích .NET.

## Pokyny k instalaci  
Chcete-li ve svém projektu použít Aspose.PDF pro .NET, postupujte takto:  

1. Instalace přes NuGet  
Spusťte následující příkaz v konzoli Správce balíčků NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Přidání jmenného prostoru v C#  
Na začátek souboru C# uveďte požadované jmenné prostory:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Podrobnou dokumentaci naleznete na adrese:  
[Dokumentace k souboru Aspose.PDF pro .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF pro referenční příručku k .NET API](https://reference.aspose.com/pdf/net/)

# Kreslení diagonálních čar v PDF pomocí Aspose.PDF pro .NET

Aspose.PDF pro .NET umožňuje vývojářům snadno vytvářet, manipulovat a upravovat dokumenty PDF. V tomto příkladu si ukážeme, jak programově nakreslit dvě diagonální čáry přes stránku PDF pomocí jazyka C#. Naučíte se, jak inicializovat dokument PDF, nastavit vlastnosti stránky, přidat grafické prvky a uložit konečný výstup. Tato příručka poskytuje podrobný postup, který usnadňuje integraci této funkce do vašich aplikací .NET.

## Pokyny k instalaci:  

Chcete-li ve svém projektu použít Aspose.PDF pro .NET, postupujte takto:  

1. Instalace pomocí Správce balíčků NuGet  
- Otevřete Visual Studio a vytvořte nový .NET projekt.  
- Otevřete konzoli Správce balíčků (Nástroje → Správce balíčků NuGet → Konzola Správce balíčků) a spusťte:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalace přes .NET CLI  
Nebo můžete k instalaci balíčku použít rozhraní .NET CLI:  
```bash
dotnet add package Aspose.PDF
```

3. Přidání jmenného prostoru do kódu C#  
Chcete-li používat funkce Aspose.PDF, zahrňte následující jmenné prostory:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

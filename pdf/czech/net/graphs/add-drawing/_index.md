# Vytvořte PDF s průhlednými grafy pomocí Aspose.PDF  

Aspose.PDF pro .NET usnadňuje programatické vytváření a manipulaci s PDF dokumenty. V tomto příkladu ukážeme, jak vytvořit PDF dokument a přidat graf obsahující obdélník s transparentní barvou vyplnění. Naučíte se nastavit barvy, přidat stránky a kreslit tvary s přesnou kontrolou transparentnosti. Tento tutoriál poskytuje krok za krokem průvodce, včetně snippetů kódu a vysvětlení, které vám pomohou integrovat Aspose.PDF do vašich aplikací .NET.

## Příkazy na instalaci:  

Chcete-li použít Aspose.PDF pro .NET, postupujte podle těchto kroků:  

1. Instalace prostřednictvím programu NuGet Package Manager  
* Otevři Visual Studio a vytvoři nový projekt .NET.  
* Otevři konzoli pro správu balíček (tools → NuGet package manager → package manager console) a spěchejte:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalace přes .NET CLI  
Alternativně instalovat balíček pomocí CLI .NET:  
```bash
dotnet add package Aspose.PDF
```

3. Přidejte prostor pro názvy do svého kódu C#  
Chcete-li začít používat Aspose.PDF, přidejte do souboru C# následující názevové prostory:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

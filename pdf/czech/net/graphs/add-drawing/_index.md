# Vytvořte PDF s průhlednými grafy pomocí Aspose.PDF  

Aspose.PDF pro .NET usnadňuje vytváření a manipulaci s PDF dokumenty programaticky. V tomto příkladu ukážeme, jak vytvořit PDF dokument a přidat graf obsahující obdélník s transparentní barvou plnění. Naučíte se nastavit barvy, přidat stránky a kreslit tvary s přesným ovládáním transparentnosti. Toto tutoriál poskytuje postupnou příručku, včetně úryvků kódu a vysvětlení, které vám pomohou integrovat Aspose.PDF do vašich aplikací .NET.

## Návod k instalaci:  

Chcete-li použít Aspose.PDF pro .NET, postupujte následujícím způsobem:  

1. Instalace pomocí NuGet Package Manager  
* Otevřete Visual Studio a vytvořte nový .NET projekt.  
* Otevřete konzoli správce balíčků (Tools → NuGet Package Manager → Package Manager Console) a spustíte:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalace přes .NET CLI  
Alternativně, nainstalujte balíček pomocí .NET CLI:  
```bash
dotnet add package Aspose.PDF
```

3. Přidejte název do vašeho kódu C#  
Chcete-li začít používat Aspose.PDF, přidejte následující prostor jmen do souboru C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

# Vytvořte PDF s průhlednými grafy pomocí Aspose.PDF  

Aspose.PDF pro .NET usnadňuje programově generování a manipulaci s PDF dokumenty. V tomto příkladu si ukážeme, jak vytvořit PDF dokument a přidat graf obsahující obdélník s průhlednou barvou výplně. Naučíte se, jak nastavit barvy, přidat stránky a kreslit tvary s přesnou kontrolou nad průhledností. Tento tutoriál poskytuje podrobný návod, včetně úryvků kódu a vysvětlení, které vám pomohou integrovat Aspose.PDF do vašich .NET aplikací.

## Pokyny k instalaci:  

Chcete-li použít Aspose.PDF pro .NET, postupujte takto:  

1. Instalace pomocí Správce balíčků NuGet  
- Otevřete Visual Studio a vytvořte nový .NET projekt.  
- Otevřete konzoli Správce balíčků (Nástroje → Správce balíčků NuGet → Konzola Správce balíčků) a spusťte:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalace přes .NET CLI  
Nebo nainstalujte balíček pomocí rozhraní .NET CLI:  
```bash
dotnet add package Aspose.PDF
```

3. Přidání jmenného prostoru do kódu C#  
Chcete-li začít používat Aspose.PDF, přidejte do souboru C# následující jmenný prostor:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

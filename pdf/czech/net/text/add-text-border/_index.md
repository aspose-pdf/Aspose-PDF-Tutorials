# Vytváření stylizovaného textu s hranicemi v dokumentech PDF

Aspose.PDF pro .NET je výkonné API pro zpracování PDF, které umožňuje vývojářům vytvářet, upravovat, manipulovat a konverzovat dokumenty PDF bez nutnosti Adobe Acrobat. Tento příklad ukazuje, jak vytvořit PDF dokument s textovým fragmentem, který má specifický styl a hranici, což je obzvláště užitečné pro zvýraznění důležitého obsahu v vašich dokumentech.

## Návod k instalaci

1. Instalace pomocí NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Manuální instalace:
   * Stáhněte si nejnovější verzi z[Aspose.PDF pro vydání .NET](https://releases.aspose.com/pdf/net/)
   * Přidejte odkaz na Aspose.PDF.dll do projektu

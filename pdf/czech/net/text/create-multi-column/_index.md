V tomto tutoriálu vám ukážeme, jak vytvořit více sloupcový PDF dokument pomocí Aspose.PDF pro .NET. Budeme se zabývat nastavením okrajů, přidáním formátovaného textu s HTML, kreslením vodorovných čár a organizováním obsahu do více sloupců. ## Návod k instalaci  

Chcete-li použít Aspose.PDF pro .NET ve vašem projektu, postupujte podle těchto kroků:  

1. Instalace pomocí NuGet (doporučeno)  
Spusťte následující příkaz v konzoli Manažera balíčků NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Instalace přes .NET CLI  
Pro .NET Core nebo .NET 5+ použijte:  
```bash
dotnet add package Aspose.PDF
```

3. Po instalaci, importovat potřebné název v projektu:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

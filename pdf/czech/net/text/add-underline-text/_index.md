# Přidání podtrženého textu do PDF pomocí Aspose.PDF pro .NET

Aspose.PDF pro .NET je výkonná knihovna pro zpracování PDF, která umožňuje vývojářům vytvářet, upravovat a manipulovat s dokumenty PDF bez nutnosti použití aplikace Adobe Acrobat. Tento příklad ukazuje, jak vytvořit nový dokument PDF a přidat text s podtržením pomocí tříd TextBuilder a TextFragment. Kód vytvoří jednoduchý PDF soubor s testovací zprávou formátovanou písmem Arial s podtržením a umístí ji na určité souřadnice na stránce.

## Pokyny k instalaci

1: Instalace Aspose.PDF pro .NET

Aspose.PDF pro .NET můžete nainstalovat jednou z následujících metod:

Použití Správce balíčků NuGet:

```csharp
Install-Package Aspose.PDF
```

Použití .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Použití PackageReference v souboru projektu:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Přidání povinných příkazů Using

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

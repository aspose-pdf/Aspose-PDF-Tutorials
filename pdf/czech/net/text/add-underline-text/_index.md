# Přidání podtrženého textu do PDF s Aspose.PDF pro .NET

Aspose.PDF pro .NET je výkonná knihovna pro zpracování PDF, která umožňuje vývojářům vytvářet, upravovat a manipulovat s PDF dokumenty bez nutnosti Adobe Acrobat. Tento příklad ukazuje, jak vytvořit nový PDF dokument a přidat text s podtrženým formátováním pomocí tříd TextBuilder a TextFragment. Kód vytváří jednoduchý PDF s testovací zprávou formátovanou v písmu Arial s podtrženým stylováním, umístěním na konkrétní souřadnice uvnitř stránky.

## Návod k instalaci

1: Nainstalujte Aspose.PDF pro .NET

Můžete nainstalovat Aspose.PDF pro .NET pomocí jedné z následujících metod:

Použití NuGet Package Manageru:

```csharp
Install-Package Aspose.PDF
```

Používání .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Použití PackageReference ve vašem souboru projektu:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Přidat požadovaná použití výroků

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## Instalace  

Chcete-li použít Aspose.PDF pro .NET, postupujte takto:  

1. Stáhněte a nainstalujte Aspose.PDF pro .NET  
   - Stáhnout z [Aspose.PDF pro .NET ke stažení](https://releases.aspose.com/pdf/net/)  
   - Instalace přes NuGet:  
     ```sh
     Install-Package Aspose.PDF
     ```

   - Dočasná licence: [Dočasná licence Aspose.PDF](https://purchase.aspose.com/temporary-license/)

## Pokyny k instalaci
Chcete-li začít s Aspose.PDF pro .NET, postupujte podle těchto pokynů k instalaci:

1. Stáhnout Aspose.PDF: 
   - Navštivte [Webové stránky Aspose](https://www.aspose.com/downloads/pdf/net) stáhnout nejnovější verzi souboru Aspose.PDF pro .NET.

2. Instalace pomocí Správce balíčků NuGet:
   - Otevřete svůj projekt ve Visual Studiu.
   - Přejděte do nabídky Nástroje > Správce balíčků NuGet > Spravovat balíčky NuGet pro řešení.
   - Vyhledejte „Aspose.PDF“ a nainstalujte balíček.

3. Přidat reference:
   - Ujistěte se, že váš projekt odkazuje na knihovnu DLL Aspose.PDF v `packages` složku.
   
4. Zahrňte do projektu jmenný prostor Aspose.PDF:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

Tento příklad ukazuje, jak vytvořit PDF dokument pomocí Aspose.PDF pro .NET, který obsahuje tabulku s dvojitými hranicemi na horní a dolní straně buňky.

## Návod k instalaci
Chcete-li začít s Aspose.PDF pro .NET, postupujte podle těchto návodů pro instalaci:

1. Stáhněte Aspose.PDF: 
   * Navštivte[Webová stránka Aspose](https://www.aspose.com/downloads/pdf/net)ke stažení nejnovější verze Aspose.PDF pro .NET.

2. Instalace pomocí NuGet Package Manager:
   * Otevřete svůj projekt ve Visual Studio.
   * Přejděte na Nástroje > Správce balíčků NuGet > Správa balíčků NuGet pro řešení.
   * Vyhledat Aspose.PDF a nainstalovat balíček.

3. Přidat odkazy:
   * Ujistěte se, že váš projekt odkazuje na DLL Aspose.PDF v `balení`složku.
   
4. Zahrnout Aspose.PDF Namespace ve vašem projektu:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

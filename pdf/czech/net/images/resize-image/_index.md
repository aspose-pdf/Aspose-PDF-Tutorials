## Příkazy na instalaci
1. Stáhněte si Aspose.PDF pro .NET: Navštivte [Aspose.PDF pro stránku s stažením .NET](https://docs.aspose.com/pdf/net/)\- Aby se dostala nejnovější verze.
2. Instalace prostřednictvím NuGet: Otevři svůj projekt v Visual Studio a použijte následující příkaz v konzole pro správu balíček:
   ```
   Install-Package Aspose.PDF
   ```

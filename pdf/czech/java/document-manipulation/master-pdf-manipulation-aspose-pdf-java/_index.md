Document(dataDir + \"/input.pdf\");\n```\n**Vysvětlení:** \n- Ten `Document` Třída se používá k otevření existujícího PDF souboru.\n- Nahradit `\"YOUR_DOCUMENT_DIRECTORY\"` s vaší skutečnou cestou k pdfDocument.getForm().get(\"textbox1\");\n```\n**Vysvětlení:** \n- Použití `getForm()` pro přístup k polím formuláře.\n- Nahradit `"textbox1"` s názvem vašeho pole.\n\n### Nastavení popisku pro pole tool tip\");\n```\n**Vysvětlení:** \n- `setAlternateName()` přidá do pole popisek, díky čemuž je pro uživatele intuitivnější.\n\n### Uložení upraveného dokumentu PDF\n**Přehled:**\nPo úpravě uložte

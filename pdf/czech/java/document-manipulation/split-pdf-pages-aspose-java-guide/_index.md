pdfPage <= pdfDocument1.getPages().size(); pdfPage++) {\n    // Create a new document for each individual page\n    Document newDocument = new Document();\n    \n    // Add the current page to the new document\n    newDocument.getPages().add(pdfDocument1.getPages().get_Item(pdfPage));\n    \n    // Save the new document as a separate PDF file named according to its page number\n    newDocument.save(\"YOUR_OUTPUT_DIRECTORY/page_\" + pdfPage + \".pdf\");\n}\n```\n\n**Vysvětlení:**\n- **`dataDir`:** Cesta k adresáři, kde je uložen váš vstupní PDF soubor.\n- **`newDocument`:** Čerstvý případ `Document` pro každé rozdělení stránky.\n- **`get_Item(pdfPage)`:** Načte konkrétní stránku z původního dokumentu.\n\n### Tipy pro řešení problémů\n- Ujistěte se, že cesty k adresářům existují a mají odpovídající oprávnění.\n- Ověřte kompatibilitu verzí souboru Aspose.PDF, abyste předešli problémům za běhu.\n\n## Praktické aplikace\n\nRozdělení PDF souborů na jednotlivé stránky může být neocenitelné v různých scénářích:\n\n1. **Sdílení dokumentů**Snadno sdílejte konkrétní části bez nutnosti odesílat celé dokumenty.\n2. **Zpracování dat**: Usnadněte automatizované zpracování zpracováním jedné stránky najednou.\n3. **Správa souborů**Uspořádejte velké soubory PDF pro snazší navigaci a archivaci.\n\n## Aspekty výkonu\n\nPři práci s rozsáhlými soubory PDF je klíčová optimalizace výkonu:\n\n- Používejte vhodné techniky správy paměti pro efektivní zpracování velkých dokumentů.\n- Sledujte využití zdrojů, abyste předešli úzkým hrdlům ve vaší aplikaci.\n- Využijte vestavěné metody Aspose.PDF ke zvýšení rychlosti zpracování.\n\n## Závěr\n\nNyní jste se naučili, jak rozdělit dokument PDF na jednotlivé stránky pomocí Aspose.PDF pro Javu.

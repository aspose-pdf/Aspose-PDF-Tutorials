Importujte potřebné třídy\n```java\nimport com.aspose.pdf.facades.Form;\n```\n\n#### Krok 2: Definování adresářů souborů\nNastavte adresáře pro vstupní a výstupní soubory.\n```java\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n```\n\n#### Krok 3: Vytvořte metodu pro sloučení polí\nDefinujte `documentForm` metoda, která otevírá, splošťuje a ukládá váš PDF dokument.\n\n```java\nvoid documentForm() {\n    // Initialize Form object to interact with the PDF\n    Form pdfForm = new Form();\n    \n    // Bind the source PDF file from 'dataDir/input.pdf'\n    pdfForm.bindPdf(dataDir + \"/input.pdf\");\n    \n    // Flatten all fields in the PDF document\n    pdfForm.flattenAllFields();\n    \n    // Save the modified document to 'outputDir/output.pdf'\n    pdfForm.save(outputDir + \"/output.pdf\");\n}\n```\n\n#### Vysvětlení:\n- **`Form` Třída**Používá se pro manipulaci s poli formuláře v PDF.\n- **`bindPdf` Metoda**: Otevře a připraví soubor PDF ke zpracování.\n- **`flattenAllFields` Metoda**Převede všechna interaktivní pole na statický text.\n- **`save` Metoda**: Zapíše upravený obsah do nového souboru.\n\n### Tipy pro řešení problémů\n- Ujistěte se, že jsou v `dataDir` a `outputDir`.\n- Zpracování výjimek, které mohou nastat během operací se soubory, pomocí bloků try-catch pro robustní zpracování chyb.\n\n##

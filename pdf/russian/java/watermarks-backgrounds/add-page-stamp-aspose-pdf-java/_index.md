## Руководство по внедрению

### Добавление штампа страницы в PDF-документ

Эта функция позволяет накладывать штамп из одного PDF-файла на другой. Давайте разберемся:

#### Обзор
Вы возьмете страницу из `stamp.pdf` и поместите его в качестве фона на первой странице `input.pdf`.

#### Пошаговая реализация

**1. Загрузите ваши документы**
Начните с загрузки основного документа и источника штампа.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/input.pdf");
Document pdfStampSource = new Document(dataDir + "/stamp.pdf");
```

**2. Создайте объект «Штамп»**
Используйте первую страницу `stamp.pdf` для создания вашего объекта-штампа.
```java
import com.aspose.pdf.PdfPageStamp;

PdfPageStamp pageStamp = new PdfPageStamp(pdfStampSource.getPages().get_Item(1));
pageStamp.setBackground(true); // Печать на заднем плане
// Установите положение и вращение для точного размещения
pageStamp.setXIndent(100);
pageStamp.setYIndent(100);
pageStamp.setRotate(com.aspose.pdf.Rotation.on180);
```

**3. Добавьте штамп к вашему документу**
Теперь примените этот штамп к вашему основному документу.
```java
pdfDocument.getPages().get_Item(1).addStamp(pageStamp);
// Сохраните обновленный документ
String outputDir = "YOUR_OUTPUT_DIRECTORY";
pdfDocument.save(outputDir + "/output_pdfpagestamp.pdf");
```

### Поворот штампа страницы в документе PDF
Чтобы повернуть штамп, отрегулируйте его ориентацию перед добавлением в целевой PDF-файл.

#### Обзор
Этот процесс включает в себя установку вращения на вашем `PdfPageStamp` объект, а затем применить его к документу.

**1. Загрузка документов**
Как и прежде, загрузите оба документа.
```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
Document pdfStampSource = new Document(dataDir + "/stamp.pdf");
```

**2. Создание и настройка штампа**
Создайте штамп и задайте его вращение.
```java
PdfPageStamp pageStamp = new PdfPageStamp(pdfStampSource.getPages().get_Item(1));
pageStamp.setRotate(com.aspose.pdf.Rotation.on180);
```

**3. Подать заявку и сохранить**
Добавьте повернутый штамп в свой PDF-файл, затем сохраните его.
```java
pdfDocument.getPages().get_Item(1).addStamp(pageStamp);
pdfDocument.save(outputDir + "/output_rotated_stamp.pdf");
```

### Извлечение и составление списка встроенных изображений с определенной страницы
#### Обзор
После загрузки документа вам может потребоваться доступ к встроенным изображениям. Вот как можно получить имена изображений с первой страницы.
```java
import com.aspose.pdf.Document;

public class ImageRetriever {
    public static void listImages(String filePath) {
        Document pdfDocument = new Document(filePath);
        
        // Доступ к ресурсам первой страницы
        var pageResources = pdfDocument.getPages().get_Item(1).getResources();
        
        if (pageResources.getImages() != null) {
            for (int i = 1; i <= pageResources.getImages().size(); i++) {
                String imageName = pageResources.getImages().getNames()[i - 1];
                System.out.println("Image Name: " + imageName);
            }
        } else {
            System.out.println("No images found on the first page.");
        }
    }
}
```
#### Параметры и возвращаемые значения
- **filePath**: Путь к вашему PDF-файлу. Эта строка должна быть допустимым путем к каталогу.
- **Объект документа**Обрабатывает загруженный PDF-документ, предоставляя доступ к его содержимому.
- **Ресурсы и изображения**: Методы, позволяющие взаимодействовать с ресурсами, такими как изображения на определенных страницах.

## Руководство по внедрению

Давайте разберем процесс на две основные функции: установку и получение коэффициента масштабирования в PDF-документе с помощью Java.

### Установка коэффициента масштабирования

#### Обзор
Эта функция позволяет вам указать, насколько большим или маленьким будет содержимое при открытии PDF-файла, что с самого начала улучшает пользовательский опыт.

#### Пошаговая реализация:
**1. Импортируйте необходимые пакеты**
Начнем с импорта необходимых классов:
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.XYZExplicitDestination;
import com.aspose.pdf.GoToAction;
```

**2. Определите пути к файлам и уровень масштабирования**
Настройте каталоги документов и желаемый коэффициент масштабирования.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";
double zoom = 0.5; // Пример: уровень масштабирования 50%
```

**3. Загрузите PDF-документ.**
Создать `Document` объект для загрузки существующего PDF-файла, который вы хотите изменить.
```java
Document doc = new Document(dataDir + "/HelloWorld.pdf");
```

**4. Создайте XYZExplicitDestination**
Настройте целевой объект с желаемым уровнем масштабирования, шириной и высотой на основе размеров медиа-окна первой страницы.
```java
XYZExplicitDestination destination = new XYZExplicitDestination(
    doc.getPages().get_Item(1),
    doc.getPages().get_Item(1).getMediaBox().getWidth(),
    doc.getPages().get_Item(1).getMediaBox().getHeight(), 
    zoom);
```

**5. Назначьте Zoom как действие открытия**
Создать `GoToAction` используя место назначения и установите его в качестве действия открытия документа.
```java
GoToAction actionzoom = new GoToAction(destination);
doc.setOpenAction(actionzoom);
```

**6. Сохраните изменения.**
Наконец, сохраните измененный PDF-файл, чтобы применить настройки масштабирования.
```java
String outputPath = outputDir + "/Zoomed_actionzoom.pdf";
doc.save(outputPath);
```

### Получение коэффициента масштабирования

#### Обзор
Доступ к текущему уровню масштабирования, установленному в PDF-файле, имеет решающее значение для понимания того, как будет представлен ваш документ при открытии.

#### Пошаговая реализация:
**1. Загрузите измененный документ**
Начните с загрузки ранее сохраненного документа.
```java
Document doc1 = new Document(outputDir + "/Zoomed_actionzoom.pdf");
```

**2. Доступ к открытому действию и получение уровня масштабирования**
Извлечь `GoToAction` из открытого действия, затем получите уровень масштабирования из `XYZExplicitDestination`.
```java
GoToAction action = (GoToAction) doc1.getOpenAction();
XYZExplicitDestination destination = (XYZExplicitDestination) action.getDestination();
double retrievedZoom = destination.getZoom();
```
Переменная `retrievedZoom` теперь содержит текущий коэффициент масштабирования вашего документа.

## Практические применения

Настройка уровней масштабирования PDF-файлов может быть полезна в нескольких реальных сценариях:
1. **Автоматизированные отчеты**: Установите начальный вид для выделения ключевых разделов в отчетах.
2. **Шаблоны документов**: Предварительно заданный масштаб в шаблонах для единообразного представления.
3. **Инструменты для совместной работы**: Убедитесь, что во время рецензирования члены команды видят документы оптимального размера.

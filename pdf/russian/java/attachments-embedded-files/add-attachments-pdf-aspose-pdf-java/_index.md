## Что такое «создание вложенных вложений PDF»?
Создание вложенных вложений PDF означает вставку дополнительных файлов — таких как изображения, электронные таблицы или текстовые документы — непосредственно в контейнер PDF. Прикреплённые файлы перемещаются вместе с PDF, позволяя получателям открывать их без необходимости отдельной загрузки.

## Настройка Aspose.PDF for Java

Настройка Aspose.PDF проста как с Maven, так и с Gradle. Ниже приведена **aspose pdf maven setup**, которая вам понадобится.

### Maven
Добавьте следующую зависимость в ваш `pom.xml`:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

### Gradle
Добавьте это в ваш файл `build.gradle`:
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## Руководство по реализации

Разделим реализацию на две основные функции: прикрепление файлов и загрузка PDF‑документов.

### Функция 1: Прикрепить файл к PDF‑документу

#### Обзор
Эта функция позволяет встраивать дополнительные файлы непосредственно в PDF‑документ, что полезно для предоставления дополнительной информации без отдельных вложений.

#### Шаги реализации
**Step 1: Set Up Your Environment**  
Укажите пути к исходному PDF и файлу, который вы хотите прикрепить:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Update this path
String targetPdfPath = dataDir + "/input.pdf";
String attachmentFilePath = dataDir + "/test.txt";
String targetOutputPath = "YOUR_OUTPUT_DIRECTORY/output.pdf";
```

**Step 2: Open the PDF Document**  
Создайте объект `Document` для загрузки вашего существующего PDF:
```java
Document document = new Document(targetPdfPath);
```

**Step 3: Create File Specification**  
Используйте `FileSpecification` для описания файла, который вы хотите прикрепить, включая его описание:
```java
FileSpecification fileSpecification = new FileSpecification(attachmentFilePath, "Sample text file");
```

**Step 4: Add Attachment to PDF**  
Прикрепите файл к документу, добавив его в коллекцию встроенных файлов:
```java
document.getEmbeddedFiles().add(fileSpecification);
```

**Step 5: Save the Modified Document**  
Наконец, сохраните документ с новым вложением:
```java
document.save(targetOutputPath);
```

### Функция 2: Загрузка PDF‑документа

#### Обзор
Загрузка и доступ к PDF‑документу необходимы для любой манипуляции. Этот шаг демонстрирует, как получить доступ к различным элементам загруженного PDF.

#### Шаги реализации
**Step 1: Open the PDF**  
Загрузите существующий PDF, используя его путь:
```java
Document document = new Document(targetPdfPath);
```

**Step 2: Access Document Elements**  
Теперь вы можете получить доступ к различным частям документа, таким как текст и вложения. Например, чтобы получить количество страниц:
```java
int numberOfPages = document.getPages().size();
```

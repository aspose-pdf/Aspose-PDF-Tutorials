### Извлечь файлы из портфолио PDF

#### Обзор

В этом разделе рассказывается, как извлекать отдельные файлы, встроенные в портфолио PDF. Это особенно полезно при работе с документами, содержащими несколько типов носителей или файлов данных.

#### Пошаговая реализация

**1. Загрузите исходный PDF-файл портфолио**

Начните с загрузки PDF-документа в Aspose.PDF `Document` объект:

```java
import com.aspose.pdf.Document;

// Загрузить исходный PDF-файл портфолио
document = new Document("Portfolio_output.pdf");
```

**2. Извлечь коллекцию встроенных файлов**

Доступ к коллекции встроенных файлов с помощью `getEmbeddedFiles()` метод:

```java
import com.aspose.pdf.EmbeddedFileCollection;

// Получить коллекцию встроенных файлов
embeddedFiles = document.getEmbeddedFiles();
```

**3. Повторите и извлеките каждый файл**

Просмотрите каждый файл в портфолио, чтобы извлечь их по отдельности:

```java
import java.io.File;
import java.io.FileOutputStream;
import java.io.IOException;
import java.io.InputStream;

// Просмотрите отдельные файлы портфолио
for (int counter = 1; counter <= embeddedFiles.size(); counter++) {
    com.aspose.pdf.FileSpecification fileSpecification = embeddedFiles.get_Item(counter);
    try (InputStream input = fileSpecification.getContents()) {
        File file = new File(fileSpecification.getName());
        
        // Создать путь к файлу из PDF
        if (!file.getParentFile().exists()) {
            file.getParentFile().mkdirs();
        }
        
        // Извлеките и сохраните файл.
        try (FileOutputStream output = new FileOutputStream(file)) {
            byte[] buffer = new byte[4096];
            int n;
            while ((n = input.read(buffer)) != -1) {
                output.write(buffer, 0, n);
            }
        }
    } catch (IOException e) {
        System.err.println("An error occurred: " + e.getMessage());
    }
}
```

В этом фрагменте:
- `input` извлекает поток содержимого каждого встроенного файла.
- Мы создаем каталоги по мере необходимости, используя `mkdirs()`.
- Файлы извлекаются и сохраняются с помощью `FileOutputStream`.

#### Советы по устранению неполадок

- **Файл не найден:** Убедитесь, что путь к PDF-документу указан правильно.
- **Проблемы с разрешениями:** Проверяйте права доступа к каталогам при создании папок или записи файлов.

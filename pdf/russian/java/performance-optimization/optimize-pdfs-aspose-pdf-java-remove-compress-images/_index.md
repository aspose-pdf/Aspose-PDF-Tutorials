## Руководство по внедрению

### Удаление ненужных объектов

**Обзор:** Эта функция позволяет навести порядок в PDF-файле, удаляя неиспользуемые объекты и подстановочные шрифты, что позволяет уменьшить размер файла.

#### Шаг 1: Загрузите PDF-документ
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.optimization.OptimizationOptions;

String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Заменить фактическим путем к документу
Document pdfDocument = new Document(dataDir + "/YourPDFFile.pdf");
```

#### Шаг 2: Настройте параметры оптимизации
```java
OptimizationOptions optimizeOptions = new OptimizationOptions();
optimizeOptions.setRemoveUnusedObjects(true);
optimizeOptions.setSubsetFonts(true);
```
- **`setRemoveUnusedObjects(boolean)`:** Удаляет все объекты, не используемые в документе.
- **`setSubsetFonts(boolean)`:** Сокращает данные шрифта, включая только необходимые символы.

#### Шаг 3: Оптимизация и сохранение
```java
pdfDocument.optimizeResources(optimizeOptions);
pdfDocument.save("YOUR_OUTPUT_DIRECTORY/removeUnnecessaryObjects_out.pdf");
```

### Сжатие PDF с изображениями

**Обзор:** Эта функция уменьшает размер изображений в PDF-файле, ускоряя его загрузку без существенного ухудшения качества.

#### Шаг 1: Загрузка и подготовка
```java
Document doc = new Document(dataDir + "/YourPDFFile.pdf");

OptimizationOptions opt = new OptimizationOptions();
opt.setRemoveUnusedObjects(false); // Сохраните объекты для этой операции
```

#### Шаг 2: Настройте параметры сжатия изображения
```java
opt.getImageCompressionOptions().setCompressImages(true);
opt.getImageCompressionOptions().setImageQuality(10); // Меньшее значение означает большую степень сжатия.
```
- **`setCompressImages(boolean)`:** Включает сжатие изображений.
- **`setImageQuality(int)`:** Устанавливает уровень качества изображений (1-100).

#### Шаг 3: Примените оптимизацию и сохраните
```java
doc.optimizeResources(opt);
doc.save("YOUR_OUTPUT_DIRECTORY/compressingPDFWithImages_out.pdf");
```

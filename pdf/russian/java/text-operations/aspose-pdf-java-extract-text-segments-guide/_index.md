## Руководство по внедрению

В этом разделе вы узнаете, как реализовать извлечение текстового сегмента с помощью Aspose.PDF для Java.

### Поиск и извлечение текстовых сегментов (H2)

Давайте разберем реализацию функции на логические этапы:

#### Шаг 1: Загрузите PDF-документ

Начните с загрузки вашего PDF-документа. Это делается путем создания экземпляра `Document` класс, указав ему путь к вашему файлу.

```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String inputFilePath = dataDir + "/input.pdf";

// Загрузите PDF-документ
Document pdfDocument = new Document(inputFilePath);
```

#### Шаг 2: Создайте TextFragmentAbsorber

The `TextFragmentAbsorber` класс используется для поиска определенного текста в вашем документе. Вам нужно будет создать экземпляр этого класса, указав текст, который вы ищете.

```java
import com.aspose.pdf.TextFragmentAbsorber;

// Создайте объект TextFragmentAbsorber с ключевым словом «sample»
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("sample");
```

#### Шаг 3: Примените поглотитель ко всем страницам

Используйте `accept` метод на всех страницах вашего документа. Этот шаг находит вхождения указанного текста во всем документе.

```java
// Нанесите поглотитель на все страницы.
pdfDocument.getPages().accept(textFragmentAbsorber);
```

#### Шаг 4: Извлечение и итерация фрагментов текста

После применения поглотителя извлеките коллекцию найденных фрагментов текста. Выполните итерацию по каждому фрагменту и его сегментам для извлечения подробной информации.

```java
import com.aspose.pdf.TextFragmentCollection;
import com.aspose.pdf.TextFragment;
import com.aspose.pdf.TextSegment;

// Получить коллекцию найденных фрагментов текста
textFragmentCollection = textFragmentAbsorber.getTextFragments();

// Пройдитесь по каждому TextFragment и его сегментам для извлечения деталей
for (TextFragment textFragment : (Iterable<TextFragment>) textFragmentCollection) {
    for (TextSegment textSegment : (Iterable<TextSegment>) textFragment.getSegments()) {
        // Извлечение различных свойств каждого текстового сегмента
        String text = textSegment.getText();
        Object position = textSegment.getPosition();
        
        double xIndent = position != null ? ((com.aspose.pdf.Point)position).getXIndent() : 0;
        double yIndent = position != null ? ((com.aspose.pdf.Point)position).getYIndent() : 0;
        
        String fontName = textSegment.getTextState().getFont().getFontName();
        boolean isAccessible = textSegment.getTextState().getFont().isAccessible();
        boolean isEmbedded = textSegment.getTextState().getFont().isEmbedded();
        boolean isSubset = textSegment.getTextState().getFont().isSubset();
        
        double fontSize = textSegment.getTextState().getFontSize();
        String foregroundColor = textSegment.getTextState().getForegroundColor() != null ? textSegment.getTextState().getForegroundColor().toString() : "null";
        
        // Пример: Вывод извлеченных данных
        System.out.println("Text: " + text);
        System.out.println("XIndent: " + xIndent);
    }
}
```

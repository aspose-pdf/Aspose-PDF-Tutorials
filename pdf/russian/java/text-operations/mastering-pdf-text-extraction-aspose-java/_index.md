## Настройка Aspose.PDF для Java
Чтобы начать работу с Aspose.PDF для Java, добавьте библиотеку в свой проект. Вот как это можно сделать с помощью Maven или Gradle:

**Знаток**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

**Градл**
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

### Этапы получения лицензии
1. **Бесплатная пробная версия:** Начните с бесплатной пробной версии, чтобы протестировать возможности Aspose.PDF.
2. **Временная лицензия:** Получите временную лицензию для расширенного тестирования.
3. **Покупка:** Для полного доступа рассмотрите возможность приобретения подписки.

### Базовая инициализация и настройка
После добавления зависимости инициализируйте свой проект, создав экземпляр `Document` сорт:
```java
import com.aspose.pdf.Document;

public class AsposePDFSetup {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        Document pdfDocument = new Document(dataDir + "/sample.pdf");
        System.out.println("Aspose.PDF for Java setup complete!");
    }
}
```
Настроив среду, давайте рассмотрим, как извлекать текст на основе колонок и настроек шрифта.

## Извлечение текста на основе столбцов
### Обзор
Эта функция позволяет настраивать извлечение текста путем изменения размера шрифта, что полезно для документов, в которых текст организован в столбцы.

### Пошаговая реализация
**1. Загрузите PDF-документ.**
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.TextFragmentAbsorber;
import com.aspose.pdf.TextFragmentCollection;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/net_New-age NED's.pdf");
```
**2. Настройте размер шрифта**
Здесь мы уменьшаем размер шрифта, чтобы продемонстрировать эффект масштабирования:
```java
TextFragmentAbsorber tfa = new TextFragmentAbsorber();
pdfDocument.getPages().accept(tfa);
TextFragmentCollection tfc = tfa.getTextFragments();

for (TextFragment tf : (Iterable<TextFragment>) tfc) {
    tf.getTextState().setFontSize(tf.getTextState().getFontSize() * 0.7f);
}
```
**3. Сохраните и снова откройте документ.**
Временно сохраните измененный документ и снова откройте его для извлечения текста:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
pdfDocument.save(outputDir + "/TempOutput.pdf");
pdfDocument = new Document(outputDir + "/TempOutput.pdf");
```
**4. Извлечь текст**
Использовать `TextAbsorber` для извлечения скорректированного текста:
```java
import com.aspose.pdf.TextAbsorber;

TextAbsorber textAbsorber = new TextAbsorber();
pdfDocument.getPages().accept(textAbsorber);
String extractedText = textAbsorber.getText();

java.io.FileWriter writer = new java.io.FileWriter(outputDir + "/Extracted_text.txt");
writer.write(extractedText);
writer.close();
```
## Извлечение текста с использованием метода установки масштабного коэффициента
### Обзор
Эта функция фокусируется на извлечении текста с использованием масштабного коэффициента для эффективной обработки столбчатых данных.

### Пошаговая реализация
**1. Загрузите PDF-документ.**
```java
Document pdfDocument = new Document(dataDir + "/inputFile.pdf");
```
**2. Настройте TextAbsorber с коэффициентом масштабирования**
Установите параметры извлечения и коэффициент масштабирования:
```java
import com.aspose.pdf.TextExtractionOptions;

TextAbsorber textAbsorber = new TextAbsorber();
textAbsorber.setExtractionOptions(new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Pure));
textAbsorber.getExtractionOptions().setScaleFactor(0.5);
```
**3. Извлечение и извлечение текста**
Примите поглотитель для всех страниц и получите извлеченный текст:
```java
pdfDocument.getPages().accept(textAbsorber);
String extractedText = textAbsorber.getText();
```

## Настройка Aspose.PDF для Java

Чтобы начать использовать Aspose.PDF, добавьте его в зависимости вашего проекта. Вот как:

**Мейвен:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

**Градл:**
```gradle

```

### Установка шрифта по умолчанию в PDF-документе

**Обзор:** Эта функция позволяет вам установить шрифт по умолчанию для всего текста в существующем документе PDF. Это особенно полезно, когда исходные шрифты отсутствуют или требуется стандартизация во всех документах.

#### Шаг 1: Загрузите ваш PDF-документ
Начните с загрузки вашего PDF-документа с помощью Aspose.PDF:
```java
import com.aspose.pdf.*;

Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

#### Шаг 2: Настройте параметры сохранения
Далее инициализируем `PdfSaveOptions` и задайте желаемое имя шрифта по умолчанию:
```java
String newName = "Arial";
PdfSaveOptions ops = new PdfSaveOptions();
ops.setDefaultFontName(newName);
```
Здесь, `"Arial"` указан как новый шрифт по умолчанию. Вы можете выбрать любой доступный шрифт.

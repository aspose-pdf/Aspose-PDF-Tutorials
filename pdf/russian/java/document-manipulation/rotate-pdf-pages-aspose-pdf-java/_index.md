## Руководство по внедрению
Мы разобьем нашу реализацию на две основные функции: изменение ориентации страницы на альбомную и соответствующая подгонка контента.

### Функция 1: изменение ориентации страницы на альбомную
**Обзор**: эта функция преобразует каждую страницу PDF-файла из книжной в альбомную, регулируя размеры для плавного перехода.

#### Пошаговая реализация
##### Загрузить исходный документ
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/Input.pdf");
```
*Почему*: Загрузка документа необходима, поскольку она позволяет нам манипулировать его страницами.

##### Изменить ориентацию страницы
Пройдитесь по каждой странице и отрегулируйте размеры:
```java
int pageCount = doc.getPages().size();
for (int i = 1; i <= pageCount; i++) {
    Rectangle r = doc.getPages().get_Item(i).getMediaBox();

    double newHeight = r.getWidth();
    double newWidth = r.getHeight();
    double newLLX = r.getLLX();
    double newLLY = r.getLLY() + (r.getHeight() - newHeight);

    doc.getPages().get_Item(i).setMediaBox(new Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight));
    doc.getPages().get_Item(i).setCropBox(new Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight));

    // Поверните страницу на 90 градусов для альбомной ориентации.
    doc.getPages().get_Item(i).setRotate(Rotation.on90);
}
```
*Почему*: Этот блок кода изменяет размеры каждой страницы и поворачивает ее на 90 градусов, чтобы получить альбомный вид.

##### Сохранить измененный документ
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.save(outputDir + "/Output.pdf");
```
*Почему*: Сохранение документа гарантирует постоянное хранение изменений, что позволяет вам просматривать или делиться измененным PDF-файлом.

### Функция 2: Подгонка содержимого страницы под новую ориентацию

#### Регулировка размеров
```java
Rectangle r = doc.getPages().get_Item(0).getMediaBox();
double newHeight = r.getHeight();
double newWidth = r.getWidth() * (r.getHeight() / r.getWidth());

```
*Почему*: Обеспечение надлежащего размещения контента на новых страницах повышает читабельность и качество представления.

## Руководство по внедрению
В этом разделе мы разобьем реализацию на управляемые части, сосредоточившись на добавлении функциональности истечения срока действия с помощью действий JavaScript, используя Aspose.PDF Java.
### Добавление функции истечения срока действия
Основная функция нашего руководства — встроить действие JavaScript в ваш PDF, которое проверяет, истек ли срок действия документа при открытии. Вот как это можно сделать:
#### Шаг 1: Загрузите ваш PDF-документ
Начните с загрузки существующего PDF-файла из вашего каталога с помощью Aspose.PDF.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Установить путь к каталогу входного PDF-файла
Document doc = new Document(dataDir + "/input.pdf");
```
Этот код инициализирует `Document` объект, который представляет ваш PDF. Вам нужно будет заменить `"YOUR_DOCUMENT_DIRECTORY"` с фактическим путем хранения ваших PDF-файлов.
#### Шаг 2: Определите JavaScript для проверки срока действия
Далее определите JavaScript, который проверяет, превышает ли текущая дата дату истечения срока действия, и выводит предупреждение, если это так.
```java
JavascriptAction javaScript = new JavascriptAction(
    "var year=2014; var month=4; today = new Date(); today = new Date(today.getFullYear(), today.getMonth());" +
    "expiry = new Date(year, month);" +
    "if (today.getTime() > expiry.getTime()) app.alert('The file is expired. You need a new one.');"
);
```
Этот скрипт устанавливает дату истечения срока действия (`2014-04`) и сравнивает ее с текущей датой каждый раз при открытии PDF-файла. Если дата открытия документа превышает эту дату, пользователь получает уведомление.
#### Шаг 3: Установите JavaScript в качестве действия открытия
Интегрируйте действие JavaScript в свой PDF-файл, чтобы оно запускалось при открытии:
```java
doc.setOpenAction(javaScript);
```
Эта строка кода гарантирует, что каждый раз при открытии PDF-файла автоматически запускается JavaScript, проверяющий срок действия.
#### Шаг 4: Сохраните измененный PDF-файл.
Наконец, сохраните документ с новым добавленным действием JavaScript в указанном выходном каталоге.
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // Установить путь к выходному каталогу PDF
doc.save(outputDir + "/JavaScript-Added.pdf");
```
Заменять `"YOUR_OUTPUT_DIRECTORY"` с желаемым путем к выходной папке. Это сохранит ваш измененный PDF, готовый к распространению с функцией истечения срока действия.

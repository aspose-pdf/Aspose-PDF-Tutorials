### Руководство по реализации
Мы разобьём решение на четыре чётких шага: загрузка документа, установка режима страницы, раскрытие закладок и сохранение обновлённого PDF.

#### Загрузка PDF‑документа
```java
// Load an existing PDF document from your specified directory
document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
**Пояснение:** Это создаёт объект `Document`, представляющий PDF, который вы хотите изменить. Замените путь‑заполнитель на фактическое расположение вашего файла.

#### Установка режима страницы
```java
doc.setPageMode(PageMode.UseOutlines);
```
**Пояснение:** `setPageMode()` указывает просмотрщику PDF, как открывать файл. `UseOutlines` заставляет просмотрщик автоматически отображать панель закладок.

#### Раскрытие закладок
```java
// Iterate over each bookmark and set its state to open
for (int counter = 1; counter <= doc.getOutlines().size(); counter++) {
    doc.getOutlines().get_Item(counter).setOpen(true);
}
```
**Пояснение:** Цикл проходит по каждому элементу outline (закладке) и устанавливает его свойство `Open` в `true`, гарантируя, что все закладки будут раскрыты при открытии PDF.

#### Сохранение изменённого PDF
```java
doc.save("YOUR_OUTPUT_DIRECTORY/Bookmarks_Expanded.pdf");
```
**Пояснение:** Метод `save()` записывает изменения в новый файл. При необходимости скорректируйте путь вывода.

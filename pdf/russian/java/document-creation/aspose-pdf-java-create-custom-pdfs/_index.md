## Руководство по внедрению

### Функция 1: Создание и настройка PDF-документа

#### Обзор
Создание нового документа PDF подразумевает инициализацию экземпляра `Document` класс, добавление страниц и настройка таких свойств, как размер и поля.

#### Пошаговая реализация
**Инициализация и настройка документа**
Начните с создания нового PDF-документа:
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.Page;

String outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создайте новый экземпляр класса Document.
Document doc = new Document();
```

**Добавить страницу в документ**
Вы можете добавлять страницы, используя `getPages().add()` метод:
```java
Page page = doc.getPages().add(); // Добавляет новую страницу в документ
```

**Настройте размер страницы и поля**
При необходимости настройте размер страницы и поля:
```java
// Установить размер страницы
page.setPageSize(375, 300);

// Установите поля для страницы
page.getPageInfo().getMargin().setLeft(0);
page.getPageInfo().getMargin().setTop(0);
```

**Сохранить документ**
Наконец, сохраните документ в указанном каталоге:
```java
doc.save(outputDir + "/Created_PDF.pdf");
```

### Функция 2: Добавление прямоугольника на страницу PDF-файла

#### Обзор
Добавление графических элементов, таких как прямоугольники, подразумевает использование `Graph` и `Rectangle` классы из Aspose.PDF.

#### Пошаговая реализация
**Инициализация графика и установка положения**
Создайте графический объект для вашего прямоугольника:
```java
import com.aspose.pdf.drawing.Graph;
import com.aspose.pdf.drawing.Rectangle;

Page page = new Page(); // Предположим, что страница является частью документа.
Graph graph = new Graph(60, 40); // Создать график с указанными размерами

// Исправить положение относительно страницы
graph.setChangePosition(false);
```

**Установить положение и размеры**
Поместите прямоугольник на страницу PDF-файла:
```java
graph.setLeft(50); // Установить левую позицию
graph.setTop(40);  // Установить верхнюю позицию

Rectangle rect = new Rectangle(0, 0, 60, 40); // Определить размеры
rect.getGraphInfo().setFillColor(Color.getRed()); // Цвет заливки
rect.getGraphInfo().setColor(Color.getRed()); // Цвет границы

graph.getShapes().add(rect); // Добавить прямоугольник в коллекцию фигур графика
```

**Управление Z-порядком и добавление на страницу**
Обеспечьте правильный порядок рендеринга, установив Z-индекс, затем добавьте график на страницу:
```java
graph.setZIndex(2); // Установить порядок рисования

page.getParagraphs().add(graph);
```

## Руководство по внедрению

Это руководство разделено на две основные части: преобразование изображения в PDF с использованием прямых потоков файлов и добавление изображения из BufferedImage в PDF.

### Конвертируйте изображение в PDF с помощью Direct File Stream

Эта функция позволяет преобразовать изображение непосредственно в PDF-документ, используя мощные возможности Aspose.PDF с минимальными накладными расходами.

#### Шаг 1: Создание экземпляра объекта документа
```java
doc = new Document();
```
Создать `Document` объект, который будет служить контейнером для вашего PDF-файла.

#### Шаг 2: Добавьте страницу в документ
```java
page = doc.getPages().add();
```
Добавьте новую страницу в документ. Здесь будет размещено ваше изображение.

#### Шаг 3: Загрузите файл изображения
```java
FileInputStream fs = new FileInputStream("YOUR_DOCUMENT_DIRECTORY/source.tif");
```
Загрузите исходный файл изображения в `FileInputStream`. Замените «YOUR_DOCUMENT_DIRECTORY/source.tif» на фактический путь к вашему изображению.

#### Шаг 4: Установите поля страницы и область обрезки
```java
page.getPageInfo().getMargin().setBottom(0);
page.getPageInfo().getMargin().setTop(0);
page.getPageInfo().getMargin().setLeft(0);
page.getPageInfo().getMargin().setRight(0);
page.setCropBox(new Rectangle(0, 0, 400, 400));
```
Отрегулируйте поля и область обрезки, чтобы изображение хорошо вписывалось в страницу.

#### Шаг 5: Создание и добавление объекта изображения
```java
Image image1 = new Image();
page.getParagraphs().add(image1);
image1.setImageStream(fs);
```
Создайте `Image` объект, добавьте его в коллекцию абзацев страницы и установите его поток в соответствии с потоком ввода вашего файла.

#### Шаг 6: Сохраните PDF-документ.
```java
doc.save("YOUR_OUTPUT_DIRECTORY/Image2PDF_DOM.pdf");
```
Сохраните документ в нужном месте. Замените "YOUR_OUTPUT_DIRECTORY/Image2PDF_DOM.pdf" на ваш выходной путь.

### Добавить изображение из BufferedImage в PDF

Этот метод подразумевает преобразование изображения в `BufferedImage`, который затем добавляется в PDF-документ.

#### Шаг 1: Создание документа и добавление страницы
```java
doc = new Document();
page = doc.getPages().add();
```
Начните с создания документа и добавления страницы, аналогично предыдущей функции.

#### Шаг 2: Создание BufferedImage из файла изображения
```java
Image image1 = new Image();
java.awt.image.BufferedImage bufferedImage = ImageIO.read(new File("YOUR_DOCUMENT_DIRECTORY/source.gif"));
ByteArrayOutputStream baos = new ByteArrayOutputStream();

// Записать BufferedImage как GIF
ImageIO.write(bufferedImage, "gif", baos);
baos.flush();
ByteArrayInputStream bais = new ByteArrayInputStream(baos.toByteArray());
```
Конвертируйте ваш файл изображения в `BufferedImage` а затем запишите его в `ByteArrayOutputStream`. Этот поток будет использоваться для добавления изображения в PDF.

#### Шаг 3: Добавьте изображение на страницу и настройте поток
```java
page.getParagraphs().add(image1);
image1.setImageStream(bais);
```
Добавьте объект изображения в коллекцию абзацев страницы и задайте его поток с помощью `ByteArrayInputStream`.

#### Шаг 4: Сохраните PDF-документ.
```java
doc.save("YOUR_OUTPUT_DIRECTORY/BufferedImage.pdf");
```
Сохраните документ. Обязательно укажите правильный выходной каталог.

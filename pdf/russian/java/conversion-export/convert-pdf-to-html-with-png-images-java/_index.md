# Как конвертировать PDF в HTML с помощью Aspose.PDF для Java без сохранения изображений в формате SVG

### Руководство по внедрению

Настроив среду, выполните следующие действия, чтобы преобразовать PDF в HTML и встроить изображения в формате PNG.

#### Конвертируйте PDF в HTML со встроенными изображениями PNG

**1. Открытие исходного PDF-документа**

Загрузите PDF-документ в память с помощью Aspose.PDF:
```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document pdfDocument = new Document(dataDir);
```
Убедитесь, что путь правильно указывает на местонахождение вашего PDF-файла.

**2. Настройка параметров сохранения HTML**

Настройте, как преобразование должно обрабатывать изображения и макет:
```java
import com.aspose.pdf.HtmlSaveOptions;

HtmlSaveOptions saveOptions = new HtmlSaveOptions();
// Поддерживайте визуальную последовательность с помощью фиксированной компоновки
saveOptions.setFixedLayout(true);
// Встраивайте изображения в фон страницы в формате PNG вместо SVG
saveOptions.setRasterImagesSavingMode(HtmlSaveOptions.RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground);
```
The `setFixedLayout(true)` гарантирует, что HTML отражает визуальную структуру вашего PDF. Ключевой метод здесь, `setRasterImagesSavingMode`, указывает Aspose.PDF встраивать изображения в формате PNG, избегая формата SVG, который может отображаться неравномерно в разных браузерах.

**3. Сохранение преобразованного HTML-файла**

Сохраните документ в формате HTML с указанными вами параметрами:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/converted_file.html";
pdfDocument.save(outputDir, saveOptions);
```
Это сохранит преобразованный файл в нужном вам месте, сохранив встроенные изображения PNG.

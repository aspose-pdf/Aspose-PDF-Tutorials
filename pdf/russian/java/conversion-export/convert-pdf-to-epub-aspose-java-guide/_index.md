## Как конвертировать pdf в epub с помощью Aspose.PDF for Java
В этом разделе мы пройдем все необходимые шаги для преобразования PDF‑файла в документ EPUB.

### Шаг 1: Загрузка PDF‑документа
Сначала загрузите исходный PDF в объект `Document`. Этот объект станет основой для конвертации:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf"; // Path to the source PDF document

// Load the existing PDF document
Document pdfDocument = new Document(dataDir);
```

### Шаг 2: Настройка EpubSaveOptions
Создайте экземпляр `EpubSaveOptions`, чтобы управлять процессом генерации EPUB. Режим **Flow** переупорядочивает текст для лучшей читаемости на разных размерах экрана:

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/convertPDFToEPUBFormat.epub"; // Path for saving the converted EPUB file

// Instantiate EpubSaveOptions object to specify conversion options
EpubSaveOptions options = new EpubSaveOptions();

// Set content recognition mode to 'Flow' which adjusts text flow for better readability on e-readers
options.setContentRecognitionMode(EpubSaveOptions.RecognitionMode.Flow);
```

**Почему стоит использовать режим Flow?**  
Режим Flow перестраивает абзацы и строки так, чтобы полученный EPUB плавно адаптировался к различным размерам устройств, обеспечивая более комфортное чтение.

### Шаг 3: Сохранение в EPUB
Наконец, вызовите метод `save` с настроенными параметрами:

```java
// Save the document in EPUB format using specified save options
doc.save(outputDir, options);
```

#### Советы по устранению неполадок
- Убедитесь, что `dataDir` указывает на существующий PDF; иначе возникнет `FileNotFoundException`.  
- Проверьте, что приложение имеет права записи в `outputDir`.  
- Если EPUB выглядит смещённым, двойной проверкой убедитесь, что включён режим `Flow`.

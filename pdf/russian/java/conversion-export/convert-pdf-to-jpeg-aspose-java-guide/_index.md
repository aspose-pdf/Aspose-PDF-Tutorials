## Функция 1: Конвертация всех страниц PDF в изображения JPEG

### Что означает «convert all pdf pages»?
Эта функция проходит по каждой странице PDF‑документа и сохраняет её как отдельный JPEG‑файл. Идеально подходит для **batch convert pdf jpeg** сценариев, таких как создание архивов изображений или подготовка миниатюр для веб‑галереи.

### Пошаговая реализация

**Шаг 1: Загрузка PDF‑документа**  
Создаём объект `Document`, представляющий исходный PDF.

```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

**Шаг 2: Перебор каждой страницы**  
Простой цикл `for` проходит по всем индексам страниц.

```java
for (int pageCount = 1; pageCount <= pdfDocument.getPages().size(); pageCount++) {
    // Processing code here
}
```

**Шаг 3: Настройка вывода и параметров конвертации**  
Конфигурируем `JpegDevice` с разрешением 300 DPI и максимальным качеством (100).

```java
Resolution resolution = new Resolution(300);
JpegDevice jpegDevice = new JpegDevice(resolution, 100);
```

**Шаг 4: Конвертация каждой страницы в JPEG**  
Внутри цикла записываем каждую страницу в JPEG‑файл, используя блок try‑with‑resources для корректного закрытия потока.

```java
try (OutputStream imageStream = new FileOutputStream(outputDir + "/Converted_Image" + pageCount + ".jpg")) {
    jpegDevice.process(pdfDocument.getPages().get_Item(pageCount), imageStream);
}
```

#### Распространённые подводные камни и советы
- **Корректность путей:** Тщательно проверяйте значения `dataDir` и `outputDir`; относительные пути легко ошибиться.  
- **Разрешения файлов:** Убедитесь, что процесс Java имеет права записи в целевую папку.  
- **Использование памяти:** Для очень больших PDF рассмотрите обработку страниц небольшими партиями или увеличьте размер кучи JVM.

## Функция 2: Конвертация одной страницы PDF в изображение JPEG

### Когда использовать «convert single pdf page»?
Иногда нужен только снимок конкретной страницы — например, обложки, графика или страницы с подписью. Эта функция показывает, как целенаправленно конвертировать одну страницу.

### Пошаговая реализация

**Шаг 1: Загрузка PDF‑документа** (как и ранее)

```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

**Шаг 2: Настройка выходного потока и параметров конвертации**  
Повторно используем те же настройки `Resolution` и `JpegDevice`.

```java
Resolution resolution = new Resolution(300);
JpegDevice jpegDevice = new JpegDevice(resolution, 100);
```

**Шаг 3: Конвертация указанной страницы в JPEG**  
Здесь конвертируем только первую страницу (измените индекс, чтобы выбрать другую страницу).

```java
try (OutputStream imageStream = new FileOutputStream(outputDir + "/Converted_Image.jpg")) {
    jpegDevice.process(pdfDocument.getPages().get_Item(1), imageStream);
}
```

#### Быстрый чек‑лист
- Убедитесь, что номер страницы существует (`pdfDocument.getPages().size()`).  
- Обрабатывайте `IndexOutOfBoundsException`, если индекс выходит за пределы.  
- Используйте тот же шаблон try‑with‑resources, чтобы избежать утечек ресурсов.

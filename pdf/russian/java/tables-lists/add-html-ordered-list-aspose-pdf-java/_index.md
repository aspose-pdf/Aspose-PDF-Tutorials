## Руководство по внедрению
Теперь давайте разберем шаги, необходимые для добавления упорядоченного списка HTML в ваш PDF-документ с помощью Aspose.PDF для Java.
### Создание нового PDF-документа
Во-первых, создайте новый экземпляр `Document` объект как основа нашего PDF-файла:
```java
// Создать экземпляр объекта Document для создания нового PDF-документа
doc = new Document();
```
### Добавление HTML-фрагмента
Используйте `HtmlFragment` класс для интерпретации и отображения HTML в PDF-файле.
#### Шаг 1: Создание HtmlFragment
Создайте `HtmlFragment` с желаемым HTML-контентом:
```java
// Создайте HtmlFragment с HTML-содержимым для упорядоченного списка.
t = new HtmlFragment("<body style='line-height: 100px;'><ul><li>First</li><li>Second</li><li>Third</li><li>Fourth</li><li>Fifth</li></ul>Text after the list.<br/>Next line<br/>Last line</body>");
```
#### Шаг 2: Добавьте новую страницу в документ
Добавьте новую страницу в ваш документ:
```java
// Добавить новую страницу в PDF-документ
page = doc.getPages().add();
```
#### Шаг 3: Вставьте HtmlFragment на страницу
Добавьте свой `HtmlFragment` на вновь созданную страницу:
```java
// Вставьте HtmlFragment на только что добавленную страницу.
page.getParagraphs().add(t);
```
### Сохранение PDF-документа
Сохраните документ в указанном каталоге:
```java
// Сохраните созданный PDF-документ
doc.save(outputDir + "/AddHTMLOrderedListIntoDocuments_out.pdf");
```

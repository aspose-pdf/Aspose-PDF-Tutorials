# Учебники по соответствию PDF/A для Aspose.PDF Java

Научитесь создавать документы PDF архивного качества с помощью наших всеобъемлющих учебных пособий Aspose.PDF для соответствия PDF/A в Java. Эти подробные руководства показывают, как создавать документы, соответствующие стандартам PDF/A-1b, PDF/A-2b и PDF/A-3b, проверять существующие PDF-файлы на соответствие стандартам, преобразовывать стандартные PDF-файлы в архивный формат и устранять распространенные проблемы соответствия. Каждое учебное пособие содержит рабочие примеры кода Java для сценариев архивных документов, помогая вам создавать приложения, которые создают документы для долгосрочного хранения, соответствующие международным стандартам.

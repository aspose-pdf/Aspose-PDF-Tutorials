# Как аннотировать PDF-файлы в Java с помощью зачеркнутых аннотаций с помощью Aspose.PDF

## Настройка Aspose.PDF для Java
Интеграция Aspose.PDF в ваш проект Java проста. Вот как вы можете добавить его с помощью Maven или Gradle:

**Знаток**

## Руководство по внедрению
Мы разобьем нашу реализацию на разделы в зависимости от функциональности: инициализация документа, поиск фрагментов текста, создание аннотаций и сохранение документа.

### Инициализация и загрузка документа
#### Обзор
Сначала нам нужно загрузить PDF-файл с помощью Aspose.PDF для Java. Этот шаг включает в себя инициализацию `Document` class с путем к вашему PDF-файлу.

**Шаг 1**Импортировать необходимые классы.
```java
import com.aspose.pdf.Document;
```

**Шаг 2**: Определите каталог документов и инициализируйте объект Document.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document document = new Document(dataDir + "/input.pdf");
```
Этот код загружает PDF-файл из `dataDir` в `Document` например, позволяя выполнять дальнейшие операции.

### Поглощение фрагмента текста
#### Обзор
Поиск текста в вашем PDF-файле необходим перед созданием аннотаций. Мы будем использовать `TextFragmentAbsorber`.

**Шаг 1**: Импортируйте класс.
```java
import com.aspose.pdf.TextFragmentAbsorber;
```

**Шаг 2**: Создайте экземпляр TextFragmentAbsorber с поисковым запросом.
```java
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("Sample");
```

**Шаг 3**: Примите поглотитель на каждой странице, чтобы найти все совпадающие фрагменты текста.
```java
for (int i = 1; i <= document.getPages().size(); i++) {
    document.getPages().get_Item(i).accept(textFragmentAbsorber);
}
```
Этот цикл просматривает каждую страницу, собирая совпадения по запросу «Sample».

### Извлечь коллекцию фрагментов текста
#### Обзор
После того, как фрагменты текста усвоены, мы извлекаем их в коллекцию для обработки аннотаций.

**Шаг 1**: Импортируйте требуемый класс.
```java
import com.aspose.pdf.TextFragmentCollection;
```

**Шаг 2**: Достаньте коллекцию из поглотителя.
```java
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.getTextFragments();
```
Это дает нам доступ ко всем найденным экземплярам «Sample».

### Создать зачеркнутую аннотацию
#### Обзор
Мы добавим зачеркнутую аннотацию к каждому выявленному фрагменту текста, визуально выделяя или отмечая его.

**Шаг 1**Импортировать необходимые классы.
```java
import com.aspose.pdf.StrikeOutAnnotation;
import com.aspose.pdf.Rectangle;
import com.aspose.pdf.Border;
import com.aspose.pdf.Color;
import com.aspose.pdf.Page;
```

**Шаг 2**: Просмотрите фрагменты текста и создайте аннотации.
```java
for (int j = 1; j <= textFragmentCollection.size(); j++) {
    com.aspose.pdf.TextFragment textFragment = textFragmentCollection.get_Item(j);

    // Определите область прямоугольника для аннотации на основе местоположения фрагмента текста.
    Rectangle rect = new Rectangle((float) textFragment.getPosition().getXIndent(),
                                   (float) textFragment.getPosition().getYIndent(),
                                   (float) textFragment.getPosition().getXIndent() + (float) textFragment.getRectangle().getWidth(),
                                   (float) textFragment.getPosition().getYIndent() + (float) textFragment.getRectangle().getHeight());

    // Создайте и настройте зачеркнутую аннотацию.
    StrikeOutAnnotation strikeOut = new StrikeOutAnnotation(textFragment.getPage(), rect);
    strikeOut.setOpacity(0.80f);  // Установите непрозрачность на 80%
    strikeOut.setBorder(new Border(strikeOut));  // Инициализируйте границу для аннотации
    strikeOut.setColor(Color.getRed());  // Установить красный цвет аннотации

    // Добавьте аннотацию на страницу.
    textFragment.getPage().getAnnotations().add(strikeOut);
}
```
Перед добавлением каждой аннотации настраиваются параметры прозрачности, границы и цвета.

### Сохранить документ с аннотациями
#### Обзор
После создания аннотаций мы сохраняем аннотированный документ.

**Шаг 1**: Импортируйте необходимые классы, если это еще не сделано.
```java
import com.aspose.pdf.Document;
```

**Шаг 2**: Определите выходной каталог и сохраните документ.
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
document.save(outputDir + "/StrikeOut.pdf");
```
На этом этапе вы завершаете работу над аннотированным PDF-файлом, сохраняя все изменения.

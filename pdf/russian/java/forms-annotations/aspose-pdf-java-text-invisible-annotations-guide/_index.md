## Руководство по внедрению

### Добавление текстовых аннотаций

#### Обзор
Текстовые аннотации позволяют добавлять описательные комментарии или метаданные непосредственно на страницы PDF. Они полезны для обзоров, заметок или выделения разделов в документах.

**Пошаговая реализация:**

##### Шаг 1: Инициализация документа
Загрузите исходный PDF-документ:
```java
document = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
Эта строка загружает существующий PDF-файл в ваше Java-приложение.

##### Шаг 2: Создание и настройка текстовой аннотации
Создать `TextAnnotation` объект для первой страницы, устанавливающий такие свойства, как заголовок, тема, содержимое и граница:
```java
TextAnnotation textAnnotation = new TextAnnotation(document.getPages().get_Item(1), new Rectangle(200, 400, 400, 600));
textAnnotation.setTitle("Sample Annotation Title");
textAnnotation.setSubject("Sample Subject");
textAnnotation.setState(com.aspose.pdf.AnnotationState.Accepted);
textAnnotation.setContents("Sample contents for the annotation");
textAnnotation.setOpen(true);
textAnnotation.setIcon(TextIcon.Key);

Border border = new Border(textAnnotation);
border.setWidth(5);
border.setDash(new Dash(1, 1));
textAnnotation.setBorder(border);
```

##### Шаг 3: Добавьте аннотацию к документу
Добавьте эту аннотацию и сохраните документ:
```java
document.getPages().get_Item(1).getAnnotations().add(textAnnotation);
document.save("YOUR_OUTPUT_DIRECTORY/addAnnotationToPDF.pdf");
```

### Добавление невидимых аннотаций

#### Обзор
Невидимые аннотации полезны для встраивания метаданных или данных в PDF-файл, которые не должны быть видны конечным пользователям, например для отслеживания или внутренних заметок.

**Пошаговая реализация:**

##### Шаг 1: Инициализация нового документа
Создайте и добавьте пустую страницу в свой документ:
```java
document = new Document();
document.getPages().add();
```

##### Шаг 2: Создание и настройка невидимой аннотации
Использовать `FreeTextAnnotation` со специальными флагами для контроля видимости:
```java
FreeTextAnnotation annotation = new FreeTextAnnotation(document.getPages().get_Item(1), new Rectangle(50, 600, 250, 650), new DefaultAppearance("Helvetica", 16, Color.RED));
annotation.setContents("ABCDEFG");
annotation.getCharacteristics().setBorder(Color.RED);

// Установите флаги, чтобы сделать аннотацию невидимой
annotation.setFlags(AnnotationFlags.Print | AnnotationFlags.NoView);
```

##### Шаг 3: Добавьте и сохраните документ
Добавьте эту аннотацию и сохраните документ:
```java
document.getPages().get_Item(1).getAnnotations().add(annotation);
document.save("YOUR_OUTPUT_DIRECTORY/Invisible_Annotation.pdf");
```

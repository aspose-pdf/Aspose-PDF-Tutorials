## Руководство по внедрению
Теперь, когда настройка завершена, давайте добавим действие закрытия документа в ваш PDF-файл.

### Добавить действие закрытия документа
Эта функция выполняет JavaScript, когда пользователь закрывает PDF-документ. Выполните следующие действия:

#### Шаг 1: Подготовьте среду
Убедитесь, что у вас есть существующий PDF-файл с именем `CreateDocumentLink.pdf` в указанном вами каталоге.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
```

#### Шаг 2: Откройте PDF-документ.
Использовать `PdfContentEditor` чтобы открыть и изменить PDF-файл:
```csharp
PdfContentEditor contentEditor = new PdfContentEditor();
contentEditor.BindPdf(dataDir + "CreateDocumentLink.pdf");
```
На этом этапе вы привязываете существующий документ к редактированию.

#### Шаг 3: Определите действие закрытия
Укажите действие, используя `AddDocumentAdditionalAction`. При закрытии срабатывает предупреждение JavaScript:
```csharp
contentEditor.AddDocumentAdditionalAction(PdfContentEditor.DocumentClose,
    "app.alert('Thank you for using Aspose products!');");
```
The `PdfContentEditor.DocumentClose` параметр идентифицирует событие, а строка JavaScript определяет действие.

#### Шаг 4: Сохраните обновленный PDF-файл.
После настройки документа с помощью дополнительных действий сохраните его, чтобы применить изменения:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/";
contentEditor.Save(outputDir + "CreateDocAdditionalAction_out.pdf");
```
На этом этапе измененный PDF-файл сохраняется в нужном месте.

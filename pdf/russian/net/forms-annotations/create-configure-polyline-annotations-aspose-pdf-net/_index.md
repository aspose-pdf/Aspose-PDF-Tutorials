### Создание аннотации полилинии

#### Обзор
Аннотация полилинии используется для рисования нескольких соединенных сегментов линий в документе PDF. Вы можете определить ее путь с помощью вершин и настроить ее с помощью различных свойств, таких как цвет и стиль линии.

#### Пошаговая реализация

##### Шаг 1: Инициализация документа
Начните с загрузки существующего PDF-документа в свой проект:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

##### Шаг 2: Определите вершины и площадь прямоугольника
Далее настройте вершины для вашей полилинии. Эти точки определяют путь аннотации.
```csharp
Point[] vertices = new Point[] {
    new Point(100, 600),
    new Point(500, 600),
    new Point(500, 500),
    new Point(400, 300),
    new Point(100, 500),
    new Point(100, 600)
};

Rectangle rect = new Rectangle(100, 500, 500, 600);
```

##### Шаг 3: Создание и настройка аннотации
Создать `PolylineAnnotation` Объект с определенными вершинами и прямоугольником. Настройте его внешний вид, задав такие свойства, как цвет и окончания линий.
```csharp
PolylineAnnotation area = new PolylineAnnotation(doc.Pages[1], rect, vertices);

// Установить красный цвет аннотации
area.Color = Color.Red;

// Настройте окончания строк
area.StartingStyle = LineEnding.OpenArrow;
area.EndingStyle = LineEnding.OpenArrow;
```

##### Шаг 4: Настройте параметры измерения
Вы также можете задать единицы измерения для полилинии, что полезно в технической документации.
```csharp
area.Measure = new Measure(area);
area.Measure.DistanceFormat = new Measure.NumberFormatList(area.Measure);
area.Measure.DistanceFormat.Add(new Measure.NumberFormat(area.Measure));
area.Measure.DistanceFormat[1].UnitLabel = "mm";
```

##### Шаг 5: Добавьте аннотацию к документу
Наконец, добавьте настроенную аннотацию в документ и сохраните ее.
```csharp
doc.Pages[1].Annotations.Add(area);

string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputDir + "/UseMeasureWithPolylineAnnotation_out.pdf");
```

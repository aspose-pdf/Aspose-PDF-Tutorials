# Как добавить всплывающие подсказки в текст PDF с помощью Aspose.PDF для .NET

## Руководство по внедрению
### Функция 1: Создание и сохранение образца PDF-документа
Создание простого PDF-документа — это первый шаг на нашем пути.

#### Обзор
В этом разделе показано, как создать простой PDF с текстом с помощью Aspose.PDF для .NET. Вы научитесь добавлять страницы и текстовые фрагменты в свой документ.

**Шаги по реализации:**
##### 1. Определите путь к вашему каталогу
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputFile = dataDir + "/Tooltip_out.pdf";
```
Здесь, `dataDir` это каталог, в котором вы хотите сохранить файлы PDF, и `outputFile` указывает имя PDF-файла.

##### 2. Создать новый документ
```csharp
Document doc = new Document();
```
Эта строка инициализирует новый экземпляр PDF-документа.

##### 3. Добавить страницы и фрагменты текста
```csharp
doc.Pages.Add().Paragraphs.Add(new TextFragment("Move the mouse cursor here to display a tooltip"));
doc.Pages[1].Paragraphs.Add(new TextFragment("Move the mouse cursor here to display a very long tooltip"));
```
На каждую страницу мы добавляем фрагменты текста, которые в дальнейшем будут использоваться для подсказок.

##### 4. Сохраните документ
```csharp
doc.Save(outputFile);
```
Эта команда сохраняет ваш документ в указанном месте.

### Функция 2: Добавление подсказки с ButtonField к фрагментам текста PDF
Улучшите свой PDF-файл, добавив интерактивные подсказки — вот что становится интереснее!

#### Обзор
В этом разделе мы добавляем подсказки к текстовым фрагментам в существующем PDF-файле с помощью Aspose.PDF. Это включает в себя создание невидимых кнопок, которые вызывают подсказки при наведении на них.

**Шаги по реализации:**
##### 1. Откройте существующий документ.
```csharp
Document document = new Document("YOUR_DOCUMENT_DIRECTORY/Tooltip_out.pdf");
```
Загрузите ранее созданный документ, чтобы добавить всплывающие подсказки.

##### 2. Найдите фрагменты текста с определенными фразами
```csharp
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Move the mouse cursor here to display a tooltip");
document.Pages.Accept(absorber);
TextFragmentCollection textFragments = absorber.TextFragments;
```
Здесь, `TextFragmentAbsorber` ищет определенные фразы в вашем PDF-файле.

##### 3. Создание полей кнопок и настройка подсказок
```csharp
foreach (TextFragment fragment in textFragments)
{
    ButtonField field = new ButtonField(fragment.Page, fragment.Rectangle);
    field.AlternateName = "Tooltip for text.";
    document.Form.Add(field);
}
```
Этот цикл создает `ButtonField` для каждого найденного фрагмента текста и устанавливает его подсказку.

##### 4. Повторите для более длинной подсказки
```csharp
absorber = new TextFragmentAbsorber("Move the mouse cursor here to display a very long tooltip");
document.Pages.Accept(absorber);
textFragments = absorber.TextFragments;

foreach (TextFragment fragment in textFragments)
{
    ButtonField field = new ButtonField(fragment.Page, fragment.Rectangle);
    field.AlternateName = "Lorem ipsum dolor sit amet..."; // Добавьте сюда длинный текст подсказки
    document.Form.Add(field);
}
```
Повторите процесс для других фраз, требующих более длинных подсказок.

##### 5. Сохраните обновленный документ.
```csharp
document.Save(outputFile);
```
Наконец, сохраните обновленный PDF-файл с включенными подсказками.

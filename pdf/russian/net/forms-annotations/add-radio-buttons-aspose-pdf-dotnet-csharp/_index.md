## Руководство по внедрению

### Создание и добавление полей радиокнопок

#### Обзор
В этом разделе вы узнаете, как создать поля радиокнопок в вашем документе PDF с помощью C#. Вы узнаете, как создать экземпляр `RadioButtonField`, добавьте параметры и прикрепите его к форме.

**Шаг 1: Создание экземпляра объекта документа**
Начните с создания нового `Document` объект. Он служит контейнером для содержимого вашего PDF-файла.
```csharp
// Создать новый PDF-документ
Document pdfDocument = new Document();
```

**Шаг 2: Добавьте страницу в свой документ**
Страницу необходимо добавить до размещения на ней каких-либо полей.
```csharp
// Добавить пустую страницу
pdfDocument.Pages.Add();
```

**Шаг 3: Создание экземпляра объекта RadioButtonField**
The `RadioButtonField` Объект представляет группу радиокнопок. Укажите номер страницы при ее создании.
```csharp
// Создайте новое поле радиокнопки на странице 1
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

**Шаг 4: Добавьте параметры к вашим радиокнопкам**
Определите несколько параметров в поле переключателя, указав их позиции с помощью `Rectangle` объекты.
```csharp
// Добавить первый вариант с определенной позицией
radio.AddOption("Test", new Rectangle(0, 0, 20, 20));

// Добавить второй вариант в другом месте
radio.AddOption("Test1", new Rectangle(20, 20, 40, 40));
```

**Шаг 5: Прикрепите поле радиокнопки к форме**
Добавьте поле переключателя в коллекцию форм документа.
```csharp
// Добавьте поле переключателя в форму PDF
pdfDocument.Form.Add(radio);
```

**Шаг 6: Сохраните документ**
После настройки сохраните документ со всеми его элементами.
```csharp
// Определите путь к файлу и сохраните документ
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();
dataDir += "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```

### Советы по устранению неполадок
- **Отсутствует библиотека**: Убедитесь, что Aspose.PDF установлен правильно. Проверьте ссылки на проект.
- **Неправильный индекс страницы**: Убедитесь, что индекс страницы соответствует существующей странице в вашем документе.

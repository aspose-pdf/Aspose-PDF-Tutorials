## Руководство по внедрению

### Добавление текстовой аннотации в PDF-документ

Давайте разобьем процесс добавления текстовой аннотации на удобные для выполнения шаги.

#### Загрузите ваш PDF-документ

Начните с загрузки существующего документа PDF. Укажите каталог, в котором будут храниться ваши входные и выходные документы.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/AddAnnotation.pdf");
```

#### Создать объект текстовой аннотации

Далее создайте `TextAnnotation` объект на определенной странице с требуемыми свойствами, такими как заголовок, тема и местоположение.

```csharp
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

#### Настроить свойства границы

Задайте свойства границ, чтобы улучшить визуальную привлекательность вашей аннотации.

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1); // Установить стиль штриховки для границы.
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

#### Добавить аннотацию к документу

Наконец, добавьте настроенную вами аннотацию в коллекцию аннотаций определенной страницы и сохраните измененный документ.

```csharp
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/AddAnnotation_out.pdf");
```

### Советы по устранению неполадок

- Убедитесь, что пути к каталогам ввода/вывода указаны правильно.
- Перед загрузкой убедитесь, что ваш PDF-документ существует по указанному пути.

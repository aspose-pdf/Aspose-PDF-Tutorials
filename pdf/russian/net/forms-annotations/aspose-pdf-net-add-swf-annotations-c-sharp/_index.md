## Руководство по внедрению

### Открыть и загрузить PDF-документ

#### Обзор

Эта функция демонстрирует, как загрузить существующий PDF-документ с помощью **Aspose.PDF для .NET**.

##### Шаг 1: Определите каталог

```csharp
using System;
using Aspose.Pdf;

public class FeatureOpenPDF
{
    public static void Run()
    {
        // Укажите каталог вашего документа
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        // Открыть существующий PDF-файл
        Document doc = new Document(dataDir + "AddSwfFileAsAnnotation.pdf");
        
        Console.WriteLine("The document is now loaded and ready for manipulation.");
    }
}
```

- **Параметры:** Путь к каталогу ваших документов.
- **Цель:** Загружает PDF-документ в память, подготавливая его к дальнейшим операциям.

### Добавить SWF-файл как аннотацию

#### Обзор

Добавьте мультимедийный контент, вставив аннотацию SWF-файла на указанную страницу вашего PDF-файла.

##### Шаг 2: Ссылка и аннотация первой страницы

```csharp
using System;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;

public class FeatureAddSwfAnnotation
{
    public static void Run()
    {
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        // Загрузить документ
        Document doc = new Document(dataDir + "AddSwfFileAsAnnotation.pdf");
        
        // Получить ссылку на первую страницу
        Page page = doc.Pages[1];
        
        // Создание и добавление аннотации SWF
        ScreenAnnotation annotation = new ScreenAnnotation(page, 
            new Aspose.Pdf.Rectangle(0, 400, 600, 700), dataDir + "input.swf");
        
        page.Annotations.Add(annotation);
        
        Console.WriteLine("An SWF file annotation is added to the first page.");
    }
}
```

- **Параметры:** Путь к файлу SWF и местоположение документа.
- **Цель:** Помещает интерактивный мультимедийный файл SWF в ваш PDF-файл.

### Сохранить обновленный PDF-документ

#### Обзор

После внесения изменений обязательно сохраните обновленный PDF-файл обратно на диск.

##### Шаг 3: Сохраните изменения

```csharp
using System;
using Aspose.Pdf;

public class FeatureSavePDF
{
    public static void Run()
    {
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        // Загрузите и измените документ по мере необходимости.
        Document doc = new Document(dataDir + "AddSwfFileAsAnnotation.pdf");
        
        // Укажите выходной путь для сохранения
        string outputPath = dataDir + "AddSwfFileAsAnnotation_out.pdf";
        
        // Сохранить изменения в новом файле
        doc.Save(outputPath);
        
        Console.WriteLine("The modified PDF document is saved with annotations.");
    }
}
```

- **Цель:** Сохраняет добавленные аннотации и любые другие изменения.

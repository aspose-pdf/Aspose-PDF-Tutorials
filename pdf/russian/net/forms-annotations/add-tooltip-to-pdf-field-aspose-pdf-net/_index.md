### Руководство по внедрению

#### Добавление подсказки к полю PDF

**Обзор:**
Эта функция позволяет добавлять подсказки к текстовым полям в документе PDF, предоставляя пользователям дополнительный контекст или инструкции.

**Пошаговые инструкции**

**Шаг 1: Загрузите документ**
Загрузите исходную PDF-форму:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Замените фактическим путем к вашим документам.
Document doc = new Document(dataDir + "/AddTooltipToField.pdf");
```
*Объяснение*: Этот шаг инициализирует `Document` объект, загрузив существующий файл PDF. Убедитесь, что путь указан правильно.

**Шаг 2: Доступ к текстовому полю и его изменение**
Получите доступ к определенному текстовому полю и установите его `AlternateName`, который будет действовать как подсказка:
```csharp
doc.Form["textbox1"].AlternateName = "Text box tool tip";
```
*Объяснение*: `Form` Коллекция позволяет вам получать доступ к полям по их именам. Установка `AlternateName` свойство добавляет всплывающую подсказку, которая отображается при наведении.

**Шаг 3: Сохраните обновленный документ.**
Сохраните изменения:
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY"; // Заменить на желаемый выходной путь
outputPath += "/AddTooltipToField_out.pdf";
doc.Save(outputPath);
```
*Объяснение*: Этот шаг записывает измененный документ в новый файл. Убедитесь, что выходной каталог указан правильно.

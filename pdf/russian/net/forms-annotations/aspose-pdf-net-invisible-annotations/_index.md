## Руководство по внедрению

### Добавление невидимых аннотаций

Теперь давайте сосредоточимся на основной задаче — добавлении невидимой аннотации в ваш PDF-документ.

#### Шаг 1: Загрузите ваш PDF-документ

Начните с загрузки файла PDF, с которым вы хотите работать. Это включает в себя указание пути и создание нового `Document` объект.

```csharp
// Путь к каталогу документов.
string dataDir = RunExamples.GetDataDir_AsposePdf_Annotations();

// Открыть документ
Document doc = new Document(dataDir + "input.pdf");
```

#### Шаг 2: Создайте аннотацию

Создать экземпляр `FreeTextAnnotation`. Этот тип позволяет добавлять свободный текст в виде аннотации.

```csharp
FreeTextAnnotation annotation = new FreeTextAnnotation(doc.Pages[1], 
    new Aspose.Pdf.Rectangle(50, 600, 250, 650), 
    new DefaultAppearance("Helvetica", 16, System.Drawing.Color.Red));
```

- **Объясняемые параметры:**
  - `Rectangle`: Определяет местоположение и размер аннотации.
  - `DefaultAppearance`: Устанавливает шрифт, размер и цвет текста.

#### Шаг 3: Настройка свойств аннотации

Задайте такие свойства, как содержимое, цвет границы и флаги, чтобы сделать аннотацию невидимой.

```csharp
// Установите содержание и характеристики
annotation.Contents = "ABCDEFG";
annotation.Characteristics.Border = System.Drawing.Color.Red;
annotation.Flags = AnnotationFlags.Print | AnnotationFlags.NoView;
```

- **Флаги аннотаций:**
  - `Print`: Позволяет распечатать аннотацию.
  - `NoView`: Делает аннотацию невидимой на экране.

#### Шаг 4: Добавьте и сохраните аннотацию.

Добавьте настроенную вами аннотацию к документу и сохраните ее в новом файле.

```csharp
// Добавить аннотацию на первую страницу
doc.Pages[1].Annotations.Add(annotation);

dataDir = dataDir + "InvisibleAnnotation_out.pdf";

// Сохраните выходной файл.
doc.Save(dataDir);
```

## Раздел часто задаваемых вопросов

1. **Что такое невидимая аннотация?**
   - Невидимая аннотация позволяет встраивать в PDF-файл информацию, которая не видна при просмотре документа, но может использоваться программно или во время определенных операций, таких как печать.

2. **Можно ли изменить размер шрифта невидимой аннотации?**
   - Да, отрегулируйте его через `DefaultAppearance` параметр при создании объекта аннотации.

3. **Как сделать так, чтобы мои аннотации были только напечатаны, а не просмотрены на экране?**
   - Используйте `AnnotationFlags.NoView | AnnotationFlags.Print` комбинация в настройках флагов аннотаций.

4. **Можно ли использовать Aspose.PDF бесплатно в коммерческих проектах?**
   - Доступна бесплатная пробная версия, но для полного коммерческого использования без ограничений требуется покупка лицензии.

5. **Что делать, если мои аннотации сохраняются неправильно?**
   - Убедитесь, что вы используете последнюю версию Aspose.PDF, и проверьте правильность указания путей к документам перед сохранением.

#### Функция 1: загрузка и итерация PDF-документа

#### Функция 2: Извлечение и отображение аннотаций ссылок

**Обзор**
Извлечение аннотаций ссылок из PDF позволяет легко осуществлять навигацию или извлечение данных на основе гиперссылок. Эта функция продемонстрирует, как находить и отображать эти ссылки с помощью Aspose.PDF для .NET.

**Этапы внедрения**

**Шаг 1**: Загрузите документ, как и прежде, убедившись, что путь указан правильно.
```csharp
Document document = new Document(dataDir);
```

**Шаг 2**: Определите метод извлечения аннотаций ссылок с каждой страницы.
```csharp
public static void ExtractLinkAnnotations(Page page)
{
    foreach (Annotation annot in page.Annotations) {
        if (annot is LinkAnnotation linkAnnot && linkAnnot.Action is GoToURIAction uriAction) {
            Console.WriteLine("URI: " + uriAction.URI);
            
            TextAbsorber absorber = new TextAbsorber();
            absorber.TextSearchOptions.LimitToPageBounds = true;
            absorber.TextSearchOptions.Rectangle = linkAnnot.Rect;

            page.Accept(absorber);
            string extractedText = absorber.Text;
            Console.WriteLine("Extracted Text: " + extractedText);
        }
    }
}
```
**Объяснение**: Этот метод проверяет каждую аннотацию, чтобы определить, является ли она `LinkAnnotation` с действием URI. Если это так, он извлекает и печатает URI ссылки и связанный текст.

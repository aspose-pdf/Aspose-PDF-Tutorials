## Руководство по внедрению
### Добавление поля TextBox
В этом разделе объясняется, как добавить текстовое поле в PDF-документ.

#### Обзор
Поля TextBox позволяют пользователям вводить данные напрямую, что упрощает отправку форм и сбор данных.

##### Шаг 1: Откройте ваш PDF-документ
Загрузите существующий PDF-файл с помощью Aspose.PDF:
```csharp
// Путь к каталогу документов.
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();

// Открыть документ
Document pdfDocument = new Document(dataDir + "TextField.pdf");
```

##### Шаг 2: Создание и настройка TextBoxField
Создайте текстовое поле, указав его положение:
```csharp
// Создать поле
TextBoxField textBoxField = new TextBoxField(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(100, 200, 300, 300));
textBoxField.PartialName = "textbox1";
textBoxField.Value = "Text Box";
```
**Объяснение:**
- `PartialName`: Уникальный идентификатор поля.
- `Value`: Текст по умолчанию, отображаемый в поле.

##### Шаг 3: Настройте внешний вид
Настройте текстовое поле с помощью границ и цветов:
```csharp
// Настроить границу
Border border = new Border(textBoxField)
{
    Width = 5,
    Dash = new Dash(1, 1) // Пунктирная линия
};
textBoxField.Border = border;
textBoxField.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
```
**Основные параметры конфигурации:**
- `Width`: Толщина границы.
- `Dash`: Образец для пунктирных линий.
- `Color`: Цвет заливки текстового поля.

##### Шаг 4: Добавьте поле в документ
Добавьте настроенное вами текстовое поле:
```csharp
// Добавить поле в документ
pdfDocument.Form.Add(textBoxField, 1);
```

##### Шаг 5: Сохраните измененный PDF-файл
Сохраните изменения в новом файле:
```csharp
dataDir = dataDir + "TextBox_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nTextbox field added successfully.\nFile saved at " + dataDir);
```

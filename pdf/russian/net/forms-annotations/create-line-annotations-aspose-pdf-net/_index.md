## Руководство по внедрению

Давайте разобьем реализацию на выполнимые этапы.

### Функция 1: Конфигурация информации о линии

#### Обзор
В этом разделе мы настроим свойства линии, такие как координаты вершины, видимость, цвет и ширину, используя `LineInfo` сорт.

**Шаг 1**: Определить свойства линии
```csharp
// Настройте координаты вершины для линии
LineInfo lineInfo = new LineInfo();
lineInfo.VerticeCoordinate = new float[] { 55, 55, 70, 70, 70, 90, 150, 60 };
lineInfo.Visibility = true;
lineInfo.LineColor = System.Drawing.Color.Red;
lineInfo.LineWidth = 2;

// Рассчитайте количество точек по координатам вершин
int length = lineInfo.VerticeCoordinate.Length / 2;
Aspose.Pdf.Point[] gesture = new Aspose.Pdf.Point[length];
for (int i = 0; i < length; i++) {
    // Создайте точку для каждой пары координат
    gesture[i] = new Aspose.Pdf.Point(lineInfo.VerticeCoordinate[2 * i], lineInfo.VerticeCoordinate[2 * i + 1]);
}
```

**Объяснение**: 
- The `VerticeCoordinate` Массив определяет путь линии с использованием координат x и y.
- Мы делаем линию видимой, красного цвета и шириной в 2 единицы.

### Функция 2: Создание и настройка рукописных аннотаций

#### Обзор
Далее мы создаем рукописную аннотацию, которая использует нашу настроенную линию в качестве графического элемента.

**Шаг 2**: Создание и настройка рукописных аннотаций
```csharp
using Aspose.Pdf.Annotations;

IList<Point[]> inkList = new List<Point[]>();
inkList.Add(gesture);

Document doc = new Document();
doc.Pages.Add();

InkAnnotation a1 = new InkAnnotation(doc.Pages[1], new Aspose.Pdf.Rectangle(100, 100, 300, 300), inkList);
a1.Subject = "Test";
a1.Title = "Title";
a1.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);

doc.Pages[1].Annotations.Add(a1);
```

**Объяснение**: 
- Мы добавляем конфигурацию линии в новую рукописную аннотацию.
- Задайте такие свойства, как тема, заголовок и цвет.

### Функция 3: Конфигурация границы аннотации

#### Обзор
Настройте свои аннотации с помощью специальных стилей границ для повышения визуальной привлекательности.

**Шаг 3**: Настройка границ аннотаций
```csharp
using Aspose.Pdf.Facades;

Border border = new Border(a1);
border.Width = 3;
border.Effect = BorderEffect.Cloudy;
border.Dash = new Dash(1, 1); // 1 единица включена, 1 единица выключена, шаблон тире
border.Style = BorderStyle.Solid;
```

**Объяснение**: 
- Мы применяем сплошную границу с «облачным» эффектом и специфические штриховые узоры.

Наконец, сохраните документ:

```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/lnkAnnotationLineWidth_out.pdf";
doc.Save(dataDir);
```

## Руководство по внедрению: Добавление свободной текстовой аннотации

### Обзор функций

Добавление свободной текстовой аннотации позволяет вам вставлять пользовательский текст в определенные места в вашем PDF-файле. Эта функция необходима для выделения разделов, добавления комментариев или маркировки важной информации.

### Пошаговая реализация

#### Шаг 1: Определите пути и инициализируйте PdfContentEditor

Начните с настройки путей для входных и выходных документов и инициализации `PdfContentEditor` объект:

```csharp
using System.IO;
using Aspose.Pdf.Facades;

public static void AddFreeTextAnnotation()
{
    string dataDir = @"YOUR_DOCUMENT_DIRECTORY";  
    string outputPath = @"YOUR_OUTPUT_DIRECTORY";

    using (PdfContentEditor contentEditor = new PdfContentEditor())
    {
        // Открыть существующий PDF-документ
        contentEditor.BindPdf(dataDir + "AddFreeTextAnnotation.pdf");
```

#### Шаг 2: Определите прямоугольник аннотации

Определите область, где вы хотите разместить аннотацию. Здесь мы используем прямоугольник:

```csharp
        System.Drawing.Rectangle rect = new System.Drawing.Rectangle(50, 50, 100, 100);
```

#### Шаг 3: Создайте и добавьте бесплатную текстовую аннотацию

Теперь создайте свободную текстовую аннотацию внутри указанного прямоугольника и добавьте ее в свой PDF-файл:

```csharp
        // Создайте бесплатную текстовую аннотацию
        contentEditor.CreateFreeText(rect, "Sample content", 1);

        // Сохраните обновленный документ
        contentEditor.Save(outputPath + "AddFreeTextAnnotation_out.pdf");
    }
}
```

### Объяснение параметров и методов

- `rect`: Определяет, где на странице будет отображаться ваша аннотация.
- `CreateFreeText()`: Добавляет свободную текстовую аннотацию с указанными размерами и текстом.
- `Save()`: Сохраняет изменения в новом PDF-файле.

## Руководство по внедрению

В этом разделе вы узнаете, как добавлять текстовые поля в существующие формы PDF с помощью Aspose.PDF для .NET. Каждая функция разбита на понятные шаги.

### Определение полей формы

#### Обзор:
Сначала нам необходимо определить и извлечь поля формы из существующего PDF-документа.

**Шаг 1: Загрузите ваш PDF-документ**
```csharp
string dataDir = "path/to/your/pdf/";
Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form(dataDir + "FilledForm.pdf");
```

**Шаг 2: Извлечение всех имен полей**
```csharp
String[] allfields = form.FieldNames;
```

#### Объяснение:
Этот код загружает PDF-файл и извлекает имена всех полей, предоставляя основу для дальнейших изменений.

### Добавление новых текстовых полей

#### Обзор:
Теперь, когда мы определили существующие поля формы, давайте добавим в эти места новые текстовые поля.

**Шаг 1: Подготовка координат**
```csharp
System.Drawing.Rectangle[] box = new System.Drawing.Rectangle[allfields.Length];
for (int i = 0; i < allfields.Length; i++)
{
    FormFieldFacade facade = form.GetFieldFacade(allfields[i]);
    box[i] = facade.Box;
}
```

**Шаг 2: Создание и добавление новых текстовых полей**
```csharp
Document doc = new Document(dataDir + "FilledForm - 2.pdf");
FormEditor editor = new FormEditor(doc);

for (int i = 0; i < allfields.Length; i++)
{
    editor.AddField(FieldType.Text, "TextField" + i, allfields[i], 1,
                    box[i].Left, box[i].Top, box[i].Left + 50, box[i].Top + 10);
}
editor.Save(dataDir + "IdentifyFormFields_out.pdf");
```

#### Объяснение:
Этот фрагмент добавляет новые текстовые поля прямо над каждым существующим полем формы, используя полученные ранее координаты.

### Советы по устранению неполадок
- **Несоответствие координат**: Убедитесь, что значения координат прямоугольника рассчитаны правильно.
- **Проблемы с путями к файлам**Дважды проверьте пути к файлам и убедитесь, что они существуют, прежде чем запускать код.

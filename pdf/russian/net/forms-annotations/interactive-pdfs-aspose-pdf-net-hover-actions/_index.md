## Руководство по внедрению

### Функция 1: Создание документа с текстом

#### Обзор
Эта функция демонстрирует, как создать новый PDF-документ, содержащий начальный текстовый блок, который закладывает основу для дальнейшего улучшения интерактивности.

#### Пошаговая реализация

**1. Создайте и сохраните новый документ**
```csharp
using Aspose.Pdf;

// Создать образец документа с текстом
Document doc = new Document();
doc.Pages.Add().Paragraphs.Add(new TextFragment("Move the mouse cursor here to display floating text"));
doc.Save(@"YOUR_DOCUMENT_DIRECTORY\TextBlock_HideShow_MouseOverOut_out.pdf");
```

- **Объяснение:** Начнем с создания `Document` объект и добавление страницы. A `TextFragment` Затем на эту страницу добавляется инструкция по взаимодействию с пользователем.

### Функция 2: Загрузка документа и создание скрытого текстового поля

#### Обзор
Эта функция включает загрузку ранее созданного документа, поиск определенного текста и встраивание скрытого текстового поля, которое становится видимым при наведении мыши.

#### Пошаговая реализация

**1. Загрузить существующий документ**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
using Aspose.Pdf.Forms;
using System.Drawing;

// Откройте ранее сохраненный документ с текстом
Document document = new Document(@"YOUR_DOCUMENT_DIRECTORY\TextBlock_HideShow_MouseOverOut_out.pdf");
```

**2. Найти текст и добавить скрытое поле**
```csharp
// Создайте объект TextAbsorber для поиска всех фраз, соответствующих указанному тексту.
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Move the mouse cursor here to display floating text");
document.Pages.Accept(absorber);
TextFragmentCollection textFragments = absorber.TextFragments;
TextFragment fragment = textFragments[1];

// Создать скрытое текстовое поле для плавающего текста в указанном прямоугольнике страницы
TextBoxField floatingField = new TextBoxField(fragment.Page, new Rectangle(100, 700, 220, 740));
floatingField.Value = "This is the \"floating text field\".";
floatingField.ReadOnly = true;
floatingField.Flags |= AnnotationFlags.Hidden;
floatingField.PartialName = "FloatingField_1";

// Настройте внешний вид плавающего поля
floatingField.DefaultAppearance = new DefaultAppearance("Helv", 10, Color.Blue);
floatingField.Characteristics.Background = Color.LightBlue;
floatingField.Characteristics.Border = Color.DarkBlue;
floatingField.Border = new Border(floatingField);
floatingField.Border.Width = 1;
floatingField.Multiline = true;

// Добавить текстовое поле в форму документа
document.Form.Add(floatingField);
```

- **Объяснение:** Мы находим определенный текст, используя `TextFragmentAbsorber` и создайте скрытый `TextBoxField`. Это поле настраивается с помощью пользовательских стилей, что гарантирует его невидимость до тех пор, пока оно не будет активировано пользователем.

### Функция 3: Добавление невидимой кнопки с действиями

#### Обзор
Эта функция демонстрирует добавление невидимой кнопки, которая переключает видимость текстового поля при наведении мыши.

#### Пошаговая реализация

**1. Создание и настройка невидимой кнопки**
```csharp
using Aspose.Pdf.Forms;
using System.Drawing;

// Создать невидимую кнопку на месте текстового фрагмента
ButtonField buttonField = new ButtonField(fragment.Page, fragment.Rectangle);

// Добавить действия для отображения/скрытия плавающего поля при входе/выходе мыши из области кнопки
buttonField.Actions.OnEnter = new HideAction(floatingField, false); // Показывать поле при нажатии мыши
buttonField.Actions.OnExit = new HideAction(floatingField); // Скрыть поле при выходе мыши

// Добавить поле кнопки в форму документа
document.Form.Add(buttonField);
```

- **Объяснение:** The `ButtonField` позиционируется в месте расположения фрагмента текста. Мы определяем действия, используя `HideAction` для управления видимостью плавающего текста, повышая интерактивность.

### Сохранить изменения
```csharp
// Сохранить изменения в документе
document.Save(@"YOUR_DOCUMENT_DIRECTORY\TextBlock_HideShow_MouseOverOut_out.pdf");
```

- **Объяснение:** После внедрения всех функций сохраните измененный PDF-файл, чтобы отразить эти изменения.

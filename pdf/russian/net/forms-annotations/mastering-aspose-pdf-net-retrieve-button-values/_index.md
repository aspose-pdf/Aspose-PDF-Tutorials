## Руководство по внедрению
Мы рассмотрим две основные функции: получение значений параметров кнопки и получение текущего выбранного значения поля кнопки.

### Функция 1: Получить значения параметров кнопки
#### Обзор
Эта функция позволяет извлекать все доступные параметры для определенного поля кнопки из формы PDF, предоставляя ценную информацию о выборе пользователя или конфигурациях формы.

#### Пошаговая реализация
**Шаг 1:** Создайте и инициализируйте объект Form.
```csharp
Aspose.Pdf.Facades.Form pdfForm = new Aspose.Pdf.Facades.Form();
```

**Шаг 2:** Свяжите свой PDF-документ с объектом «Форма».
```csharp
pdfForm.BindPdf("YOUR_DOCUMENT_DIRECTORY/FormField.pdf");
```
*Объяснение:* Сшивание PDF-файла гарантирует, что все его элементы доступны для манипуляций.

**Шаг 3:** Извлечение и отображение значений параметров кнопок.
```csharp
var optionValues = pdfForm.GetButtonOptionValues("Gender");

IDictionaryEnumerator optionValueEnumerator = optionValues.GetEnumerator();
while (optionValueEnumerator.MoveNext()) {
    Console.WriteLine("Key : {0} , Value : {1}", optionValueEnumerator.Key, optionValueEnumerator.Value);
}
```
*Объяснение:* The `GetButtonOptionValues` Метод извлекает перечисление всех параметров кнопки для указанного поля.

**Совет по устранению неполадок:** Во избежание ошибок убедитесь, что имя поля («Пол») точно совпадает с именем поля вашей PDF-формы.

### Функция 2: Получить текущее значение параметра кнопки
#### Обзор
Понимание того, какой параметр выбран в данный момент в PDF-форме, может иметь решающее значение, особенно при обработке пользовательских данных или конфигураций.

#### Пошаговая реализация
**Шаг 1:** Как и прежде, инициализируйте объект Form и привяжите свой документ.
```csharp
Aspose.Pdf.Facades.Form pdfForm = new Aspose.Pdf.Facades.Form();
pdfForm.BindPdf("YOUR_DOCUMENT_DIRECTORY/FormField.pdf");
```

**Шаг 2:** Получить и вывести текущее выбранное значение.
```csharp
string optionValue = pdfForm.GetButtonOptionCurrentValue("Gender");
Console.WriteLine("Current Value : {0}", optionValue);
```
*Объяснение:* `GetButtonOptionCurrentValue` извлекает текущую активную опцию, предоставляя прямое представление о состояниях формы.

**Совет по устранению неполадок:** Если значение не возвращается, убедитесь, что поле PDF содержит допустимые данные и соответствует указанному вами имени поля.

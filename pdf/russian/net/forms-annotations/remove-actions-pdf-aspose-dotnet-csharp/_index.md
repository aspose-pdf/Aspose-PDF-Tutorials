### Удаление действий по открытию документа

#### Обзор:
Удаление действий открытия документа может быть критически важным, когда вы хотите предотвратить определенное поведение или соответствовать стандартам безопасности. Давайте посмотрим, как этого можно добиться.

#### Пошаговая реализация:

**1. Подготовьте свое окружение**
Сначала убедитесь, что ваш проект настроен и Aspose.PDF установлен, как указано выше.

```csharp
using System.IO;
using Aspose.Pdf.Facades;
```

**2. Откройте PDF-документ.**
Создать экземпляр `PdfContentEditor` для работы с PDF-файлом:

```csharp
// Укажите путь к каталогу ваших документов
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_LinksActions();

// Инициализировать PdfContentEditor
PdfContentEditor contentEditor = new PdfContentEditor();
contentEditor.BindPdf(dataDir + "RemoveOpenAction.pdf");
```

**3. Удалить действие «Открытый документ»**
Используйте `RemoveDocumentOpenAction` Метод удаления открытого действия из документа:

```csharp
// Удалить открытое действие
contentEditor.RemoveDocumentOpenAction();
```

**4. Сохраните обновленный PDF-файл.**
Наконец, сохраните изменения в новом файле:

```csharp
// Сохраните обновленный PDF-файл
contentEditor.Save(dataDir + "RemoveOpenAction_out.pdf");
```

#### Объяснение:
- **Параметры**: `BindPdf` метод принимает путь к входному файлу.
- **Возвращаемые значения**: `RemoveDocumentOpenAction` не возвращает никакого значения, а изменяет документ на месте.

### Советы по устранению неполадок
- Убедитесь, что пути к PDF-файлам указаны правильно и доступны.
- Убедитесь, что Aspose.PDF правильно указан в вашем проекте, чтобы избежать ошибок во время выполнения.

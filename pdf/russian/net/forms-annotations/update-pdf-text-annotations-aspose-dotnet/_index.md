### Обновление функции бесплатных текстовых аннотаций

#### Обзор

Измените размер шрифта и цвет свободных текстовых аннотаций в ваших PDF-файлах. Эта функция идеально подходит для выделения заметок или обеспечения единообразия в вашем документе.

**Пошаговое руководство:**

##### Шаг 1: Откройте ваш PDF-документ

Сначала загрузите PDF-файл:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc1 = new Document(dataDir + "/input.pdf");
```
Этот код открывает документ из указанного вами каталога.

##### Шаг 2: Доступ к аннотации

Далее, откройте и примените первую аннотацию на первой странице к `FreeTextAnnotation`:
```csharp
var freeTextAnnotation = (FreeTextAnnotation)doc1.Pages[1].Annotations[0];
```
Рассмотрите возможность добавления проверок для реальных приложений.

##### Шаг 3: Обновите размер и цвет шрифта

Теперь измените размер и цвет шрифта:
```csharp
freeTextAnnotation.TextStyle.FontSize = 18;
freeTextAnnotation.TextStyle.Color = System.Drawing.Color.Green;
```
Эти изменения сделают текст аннотации крупнее и зеленее.

**Совет по устранению неполадок:** Убедитесь, что путь к файлу указан правильно, а на первой странице есть хотя бы одна аннотация, чтобы избежать исключений.

### Функция обработки документов

#### Обзор

Откройте документ PDF и получите доступ к его элементам с помощью Aspose.PDF. Эта функция обеспечивает понимание структуры документа, например, количества аннотаций.

**Пошаговое руководство:**

##### Шаг 1: Откройте ваш PDF-документ

Загрузите PDF-файл:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc1 = new Document(dataDir + "/input.pdf");
```

##### Шаг 2: Доступ к первой странице и аннотациям

Доступ к первой странице и отображение информации аннотации:
```csharp
Page firstPage = doc1.Pages[1];
Console.WriteLine($"Number of Annotations on First Page: {firstPage.Annotations.Count}");
```
В этом фрагменте показано количество аннотаций, что помогает понять структуру документа.

**Совет по устранению неполадок:** Проверьте, есть ли в PDF-файле страницы или аннотации, чтобы избежать ошибок доступа.

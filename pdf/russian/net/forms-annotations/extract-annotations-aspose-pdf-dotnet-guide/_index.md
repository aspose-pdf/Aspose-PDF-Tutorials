## Руководство по внедрению

### Извлечение аннотаций из PDF-документа

Давайте погрузимся в извлечение аннотаций. В этом уроке мы сосредоточимся на свободном тексте и линейных аннотациях.

#### Шаг 1: Настройка вашего проекта
Создайте новый проект C# в предпочитаемой вами среде IDE, добавив Aspose.PDF в качестве ссылки или пакета.

#### Шаг 2: Создание метода извлечения аннотаций
Вот как можно извлечь определенные типы аннотаций:
```csharp
using System;
using System.Collections;
using Aspose.Pdf.Facades;

namespace PdfAnnotationExtractor
{
    public class ExtractAnnotations
    {
        public static void Run()
        {
            try
            {
                string dataDir = "path/to/your/documents/directory/";
                PdfAnnotationEditor annotationEditor = new PdfAnnotationEditor();
                
                // Связать PDF-документ
                annotationEditor.BindPdf(dataDir + "ExtractAnnotations.pdf");
                
                // Определите типы аннотаций для извлечения
                string[] annotType = { AnnotationType.FreeText.ToString(), AnnotationType.Line.ToString() };
                
                // Извлечь указанные аннотации со страниц 1–2
                ArrayList annotList = (ArrayList)annotationEditor.ExtractAnnotations(1, 2, annotType);
                
                foreach (Annotation annotation in annotList)
                {
                    Console.WriteLine(annotation.Contents); // Вывести содержимое каждой аннотации
                }
            }
            catch (Exception ex)
            {
                Console.WriteLine("An error occurred: " + ex.Message);
            }
        }
    }
}
```
**Объяснение:**
- **`PdfAnnotationEditor`**: Этот класс облегчает извлечение и обработку аннотаций.
- **`BindPdf` Метод**: Открывает указанный PDF-файл для обработки.
- **`ExtractAnnotations` Метод**: Извлекает аннотации на основе типа и диапазона страниц, возвращая их в `ArrayList`.

#### Шаг 3: Запуск вашего кода
Скомпилируйте и запустите свое приложение. Убедитесь, что ваш PDF-файл находится по указанному пути.

## Инструкция по установке
Чтобы начать работу с Aspose.PDF для .NET, следуйте этим инструкциям по установке:

1. Загрузить Aspose.PDF: 
   - Посетите [Сайт Aspose](https://www.aspose.com/downloads/pdf/net) чтобы загрузить последнюю версию Aspose.PDF для .NET.

2. Установка через менеджер пакетов NuGet:
   - Откройте свой проект в Visual Studio.
   - Перейдите в Инструменты > Диспетчер пакетов NuGet > Управление пакетами NuGet для решения.
   - Найдите «Aspose.PDF» и установите пакет.

3. Добавить ссылки:
   - Убедитесь, что ваш проект ссылается на Aspose.PDF DLL в `packages` папка.
   
4. Включите пространство имен Aspose.PDF в свой проект:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

# PDF-портфолио и руководства по пакетам для Aspose.PDF .NET

Освойте создание портфолио PDF с помощью наших подробных учебных пособий Aspose.PDF для разработчиков .NET. Эти исчерпывающие руководства показывают, как создавать портфолио PDF, содержащие несколько файлов, настраивать макеты портфолио и навигацию, извлекать и обрабатывать встроенные документы и программно реализовывать расширенные функции портфолио. Каждое учебное пособие содержит рабочие примеры кода C# для создания сложных пакетов документов, объединяющих несколько файлов в один контейнер PDF с возможностью навигации, идеально подходящий для распространения связанных документов в качестве единого целого.

### Поиск регулярных выражений в PDF

#### Обзор

Цель здесь — поиск определенных текстовых шаблонов, определенных регулярным выражением, на всех страницах документа PDF. Это может быть полезно для извлечения дат, телефонных номеров или других форматированных данных.

#### Пошаговая реализация

1. **Загрузить документ**

   Начните с загрузки вашего PDF-файла с помощью Aspose.PDF `Document` сорт.

   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Document pdfDocument = new Document(dataDir + "/SearchRegularExpressionAll.pdf");
   ```

2. **Создать объект TextAbsorber**

   Используйте `TextFragmentAbsorber` чтобы найти все текстовые фрагменты, соответствующие вашему шаблону регулярного выражения.

   ```csharp
   TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("\\d{4}-\\d{4}"); // Пример: Соответствует годам типа «1999-2000».
   ```

3. **Настройте параметры текстового поиска**

   Настройте параметры поиска, чтобы указать использование регулярных выражений.

   ```csharp
   TextSearchOptions textSearchOptions = new TextSearchOptions(true);
   textFragmentAbsorber.TextSearchOptions = textSearchOptions;
   ```

4. **Принять Поглотитель для Всех Страниц**

   Нанесите поглотитель на все страницы документа.

   ```csharp
   pdfDocument.Pages.Accept(textFragmentAbsorber);
   ```

5. **Извлечение и отображение фрагментов текста**

   Просмотрите собранные текстовые фрагменты, отобразив их свойства.

   ```csharp
   foreach (TextFragment textFragment in textFragmentAbsorber.TextFragments)
   {
       Console.WriteLine("Text : {0} ", textFragment.Text);
       // При необходимости можно регистрировать дополнительные свойства.
   }
   ```

#### Советы по устранению неполадок

- **Ошибки синтаксиса регулярных выражений:** Убедитесь, что шаблон регулярного выражения правильный и соответствует целевому формату данных.
- **Проблемы с путем к файлу:** Проверьте правильность пути к файлу PDF-документа.

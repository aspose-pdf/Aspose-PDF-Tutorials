# Как извлечь выделенный текст из PDF-файлов с помощью Aspose.PDF для .NET

Это руководство проведет вас через использование Aspose.PDF для .NET для извлечения и отображения выделенного текста с определенной страницы в документе PDF.

К концу этого подробного руководства вы узнаете, как:

## Руководство по внедрению

Давайте разобьем реализацию на управляемые шаги. Мы рассмотрим две основные функции: извлечение выделенного текста и настройку среды для манипуляций с PDF.

### Функция 1: Извлечение и отображение выделенного текста из PDF-файлов

#### Обзор
Эта функция позволяет извлекать и отображать текст, выделенный в документе PDF, специально предназначенный для аннотаций на определенной странице.

#### Этапы внедрения

**Шаг 1: Загрузите документ**

Начните с загрузки файла PDF с помощью Aspose.PDF. Убедитесь, что путь к файлу указан правильно:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/ExtractHighlightedText.pdf");
```

Этот фрагмент инициализирует `Document` объект, представляющий ваш PDF-файл.

**Шаг 2: Перебор аннотаций**

Просмотрите аннотации на первой странице и обратите внимание на основные моменты:

```csharp
foreach (Annotation annotation in doc.Pages[1].Annotations)
{
    if (annotation is TextMarkupAnnotation highlightedAnnotation)
    {
        // Обработать выделенный текст
    }
}
```

Здесь мы перебираем каждый `Annotation` объект. Если аннотация является `TextMarkupAnnotation`, это указывает на выделение.

**Шаг 3: Извлечение и отображение выделенного текста**

Извлеките отмеченные фрагменты текста и выведите их:

```csharp
TextFragmentCollection collection = highlightedAnnotation.GetMarkedTextFragments();
foreach (TextFragment tf in collection)
{
    Console.WriteLine(tf.Text);
}
```

Этот код получает доступ к каждому `TextFragment` внутри выделения, отображающего его содержимое.

#### Советы по устранению неполадок

- Убедитесь, что ваш PDF-файл не зашифрован и не защищен паролем.
- Убедитесь, что на указанной странице присутствуют выделенные фрагменты.

### Функция 2: Настройка среды Aspose.PDF

#### Обзор
Настройте и управляйте основными свойствами вашего PDF-документа с помощью Aspose.PDF.

**Шаг 1: Загрузите документ**

```csharp
Document doc = new Document(dataDir + "/Sample.pdf");
```

**Шаг 2: Измените свойства страницы**

При необходимости получите доступ к размерам страницы и измените их:


В этом примере изменяются ширина и высота первой страницы, а затем документ сохраняется.

## Руководство по внедрению

### Поиск точного совпадения слов

**Обзор:** Эта функция позволяет находить точные совпадения определенных слов в документе с помощью регулярных выражений.

#### Этапы реализации:

**1. Создайте TextFragmentAbsorber:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"\bWord\b", new TextSearchOptions(true));
```
- **Объяснение:** The `\b` обозначает границы слов, гарантируя соответствие только точному «Слову».

**2. Принять поглотитель:**

```csharp
pdfDocument.Pages.Accept(textFragmentAbsorber);
TextFragmentCollection textFragments = textFragmentAbsorber.TextFragments;
```
- **Цель:** Этот метод обрабатывает каждую страницу документа и извлекает совпадающие фрагменты.

### Поиск строк без учета регистра

**Обзор:** Выполняйте поиск, не беспокоясь о чувствительности к регистру, что полезно для пользовательского контента или непоследовательного форматирования.

#### Этапы реализации:

**1. Определите TextFragmentAbsorber:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("(?i)Line", new TextSearchOptions(true));
```
- **Объяснение:** The `(?i)` флаг делает поиск нечувствительным к регистру.

### Анализ всех строк внутри PDF-документа

**Обзор:** Извлеките все текстовое содержимое, чтобы не пропустить ни одной данные в документе.

#### Этапы реализации:

**1. Инициализируйте поглотитель для всех слов:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"[\S]+");
```
- **Объяснение:** `[\S]+` соответствует любой последовательности символов, не являющихся пробелами, эффективно захватывая все строки.

### Поиск совпадений в строке поиска и извлечение следующего контента

**Обзор:** Полезно для извлечения контента, который соответствует определенному шаблону или ключевому слову в вашем документе.

#### Этапы реализации:

**1. Создайте поглотитель регулярных выражений:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"(?i)the ((.)*)");
```
- **Объяснение:** Это позволяет захватить «the», за которым следуют любые символы до символа переноса строки.

### Поиск текста после совпадения с регулярным выражением

**Обзор:** Извлечение контента после сопоставления с регулярным выражением, идеально подходит для идентификации и обработки последующей информации.

#### Этапы реализации:

**1. Дайте определение поглотителю:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"(?<=word).*");
```
- **Объяснение:** The `(?<=word)` является положительным ретроспективным утверждением, охватывающим все, что находится после «слова».

### Поиск гиперссылок/URL-адресов в PDF-документе

**Обзор:** Определите и извлеките URL-адреса, полезные для каталогизации или проверки ссылок в документах.

#### Этапы реализации:

**1. Настройте TextFragmentAbsorber:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"(http|ftp|https):\/\/(?:[\w\-_]+(?:(?:\.[\w\-_]+)+))(?:[\w\-.,@?^=%&amp;:/~\\#+]*[\w\-@?^=%&amp;/~\\#+])?");
```
- **Объяснение:** Этот шаблон соответствует URL-адресам, начинающимся с http, https или ftp.

## Руководство по внедрению
В этом разделе реализация подразделяется на две основные функции: внедрение всех шрифтов и выделение только полностью внедренных шрифтов.

### Функция 1: Встраивание всех шрифтов как подмножества
#### Обзор
Эта функция гарантирует, что каждый шрифт, используемый в вашем PDF-файле, будет встроен в качестве подмножества, обеспечивая единообразие в различных средах просмотра.

#### Этапы внедрения
**Загрузите ваш документ**
Начните с загрузки существующего PDF-документа из файловой системы:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

**Подмножество всех шрифтов**
Используйте `SubsetAllFonts` Стратегия внедрения всех шрифтов как подмножеств:

```csharp
doc.FontUtilities.SubsetFonts(FontSubsetStrategy.SubsetAllFonts);
```

Это гарантирует включение даже невстроенных шрифтов, что повышает совместимость.

**Сохранить документ**
Наконец, сохраните измененный документ в новом файле:

```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputPath + "/Output_All_Fonts_Subset.pdf");
```

#### Советы по устранению неполадок
- Убедитесь, что все файлы шрифтов доступны на случай возникновения ошибок при внедрении.
- Проверьте правильность путей к каталогам.

### Функция 2: Встраивание только встроенных шрифтов как подмножества
#### Обзор
Эта функция фокусируется на выделении только тех шрифтов, которые уже встроены, оставляя невстроенные шрифты нетронутыми.

#### Этапы внедрения
**Загрузите ваш документ**
Загрузите PDF-файл, как это было сделано ранее:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

**Только встроенные шрифты подмножества**
Применить `SubsetEmbeddedFontsOnly` стратегия:

```csharp
doc.FontUtilities.SubsetFonts(FontSubsetStrategy.SubsetEmbeddedFontsOnly);
```

Этот метод не влияет на невстроенные шрифты.

**Сохранить документ**
Сохраните изменения с помощью:

```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputPath + "/Output_Embedded_Fonts_Subset.pdf");
```

#### Советы по устранению неполадок
- Перед применением этой стратегии проверьте статус встроенного шрифта.
- Подтвердите пути к файлам и разрешения.

## Раздел часто задаваемых вопросов
1. **Что такое подмножество шрифтов?**
   Подмножество шрифтов подразумевает внедрение только тех частей шрифта, которые необходимы в документе для уменьшения размера.
2. **Можно ли использовать подмножества шрифтов в невстроенных PDF-файлах?**
   Да, с помощью `SubsetAllFonts` Стратегия будет включать невстроенные шрифты в качестве подмножеств.
3. **Как Aspose.PDF обрабатывает различные типы шрифтов?**
   Среди прочего он поддерживает шрифты TrueType, OpenType и Type1.
4. **Что делать, если шрифт встраивается неправильно?**
   Проверьте доступность шрифта и убедитесь, что у вас есть необходимые разрешения.
5. **Есть ли поддержка пользовательских каталогов шрифтов?**

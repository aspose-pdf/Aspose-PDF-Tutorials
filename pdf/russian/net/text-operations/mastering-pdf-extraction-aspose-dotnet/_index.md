## Руководство по внедрению

В этом разделе вы узнаете, как извлекать текст и изображения из PDF-файлов с помощью различных функций Aspose.PDF для .NET.

### Извлечение текста из PDF-файла

#### Обзор

Извлечение текста имеет решающее значение для анализа данных, отчетности или интеграции контента в другие приложения. Вот как эффективно извлекать текст:

1. **Полное извлечение документа**
2. **Извлечение отдельных страниц**

#### Полное извлечение документа

Чтобы извлечь текст из всего документа, используйте `PdfExtractor` сорт.

```csharp
using Aspose.Pdf.Facades;

namespace PdfTextExtraction
{
    class Program
    {
        static void Main()
        {
            string dataDir = "YOUR_DOCUMENT_DIRECTORY";
            PdfExtractor extractor = new PdfExtractor();
            extractor.Password = ""; // Установите, защищен ли документ паролем
            extractor.BindPdf(dataDir + "/inFile.pdf");
            extractor.ExtractText();

            // Сохранить извлеченный текст в файл
            extractor.GetText(dataDir + "/extracted_text.txt");
        }
    }
}
```

**Объяснение:**
- `BindPdf`: Прикрепляет PDF-файл для извлечения.
- `ExtractText`: Запускает процесс извлечения текста.
- `GetText`: Сохраняет извлеченное содержимое в указанный файл.

#### Извлечение отдельных страниц

Для извлечения текста с определенных страниц выполните итерацию по каждой странице:

```csharp
using Aspose.Pdf.Facades;

namespace PdfPageTextExtraction
{
    class Program
    {
        static void Main()
        {
            string dataDir = "YOUR_DOCUMENT_DIRECTORY";
            PdfExtractor extractor = new PdfExtractor();
            extractor.Password = "";
            extractor.BindPdf(dataDir + "/inFile.pdf");

            while (extractor.HasNextPageText())
            {
                string fileName = $"{dataDir}/{DateTime.Now.Ticks}_out.txt";
                extractor.GetNextPageText(fileName);
            }
        }
    }
}
```

**Объяснение:**
- `HasNextPageText`: Проверяет, есть ли еще страницы для извлечения.
- `GetNextPageText`: Извлекает текст со следующей страницы и сохраняет его.

### Извлечение изображений из PDF-файла

#### Обзор

Извлечение изображений может быть необходимо для повторного использования или анализа контента. Вот как этого можно добиться с помощью Aspose.PDF:

```csharp
using System;
using Aspose.Pdf.Facades;
using System.Drawing.Imaging;

namespace PdfImageExtraction
{
    class Program
    {
        static void Main()
        {
            string dataDir = "YOUR_DOCUMENT_DIRECTORY";
            PdfExtractor extractor = new PdfExtractor();
            extractor.Password = "";
            extractor.BindPdf(dataDir + "/inFile.pdf");
            extractor.ExtractImage();

            while (extractor.HasNextImage())
            {
                string fileName = $"{dataDir}/{DateTime.Now.Ticks}_out.jpg";
                extractor.GetNextImage(fileName, ImageFormat.Jpeg);
            }
        }
    }
}
```

**Объяснение:**
- `ExtractImage`: Извлекает все изображения из PDF-файла.
- `HasNextImage` и `GetNextImage`: Итерирует каждое изображение для извлечения.

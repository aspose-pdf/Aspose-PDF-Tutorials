### Добавление FloatingBox для нумерации страниц

А `FloatingBox` позволяет размещать контент в определенных позициях на страницах PDF. Вот как можно использовать его для добавления номеров страниц:

#### Шаг 1: Создание документа и добавление страниц
Сначала создайте новый документ и добавьте страницу, на которой будет размещено плавающее поле.

```csharp
// Создать новый PDF-документ
Document document = new Document();

// Добавить страницу в документ PDF
Page page = document.Pages.Add();
```

#### Шаг 2: Инициализация FloatingBox

Создать экземпляр `FloatingBox` с желаемыми размерами и разместите его соответствующим образом на странице.

```csharp
// Инициализируйте FloatingBox с шириной и высотой
FloatingBox box1 = new FloatingBox(140, 80);

// Установите левое и верхнее положение для точного размещения
box1.Left = 2;
box1.Top = 10;

// Добавить текст нумерации страниц в абзацы FloatingBox
box1.Paragraphs.Add(new Text.TextFragment("Page: ($p/ $P )"));

// Добавить плавающее поле на текущую страницу
page.Paragraphs.Add(box1);
```

**Объяснение:**  
- `FloatingBox(140, 80)`: Определяет размер поля.
- `$p` и `$P`: Заполнители для текущих и общих страниц.

#### Шаг 3: Сохраните документ.

Наконец, сохраните документ в указанном месте.

```csharp
// Определить выходной путь
string outputPath = "YOUR_OUTPUT_DIRECTORY/PageNumberinHeaderFooterUsingFloatingBox_out.pdf";

// Сохранить документ
document.Save(outputPath);
```

## Руководство по внедрению

### Внедрение стандартных шрифтов Type 1

Встраивание шрифтов имеет решающее значение для сохранения целостности и внешнего вида документа на разных платформах. Эта функция гарантирует единообразный вид текста, независимо от того, где просматривается PDF-файл.

#### Пошаговый процесс

**Загрузите существующий документ**

Начните с загрузки PDF-файла, который вы хотите изменить:
```csharp
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

**Установить свойство Embed Standard Fonts**

Убедитесь, что в ваш документ встроены стандартные шрифты типа 1:
```csharp
pdfDocument.EmbedStandardFonts = true;
```

**Перебор страниц и шрифтов**

Далее пройдитесь по каждой странице и ее ресурсам шрифтов, чтобы внедрить необходимые шрифты:
```csharp
foreach (Page page in pdfDocument.Pages)
{
    if (page.Resources.Fonts != null)
    {
        foreach (Text.Font pageFont in page.Resources.Fonts)
        {
            // Встроить шрифт, если он еще не встроен
            if (!pageFont.IsEmbedded)
            {
                pageFont.IsEmbedded = true;
            }
        }
    }
}
```

Это гарантирует, что все шрифты, используемые в документе, будут внедрены, сохранив их внешний вид.

**Сохраните обновленный документ**

Наконец, сохраните обновленный PDF-файл:
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/EmbeddedFonts-updated_out.pdf");
```

### Советы по устранению неполадок

- **Отсутствующие шрифты**: Убедитесь, что файлы шрифтов доступны и на них правильно указаны ссылки.

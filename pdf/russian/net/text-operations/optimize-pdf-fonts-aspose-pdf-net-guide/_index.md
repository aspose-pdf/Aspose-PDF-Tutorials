# Подробное руководство: Оптимизация шрифтов PDF с помощью Aspose.PDF .NET

**Что вы узнаете:**
- Удаление неиспользуемых шрифтов из PDF-документов
- Замена удаленных шрифтов на Arial Bold с помощью Aspose.PDF для .NET
- Практическое применение оптимизированных PDF-файлов в различных проектах
Это руководство упростит обработку документов и повысит производительность. Давайте подготовимся, прежде чем начать.

## Руководство по внедрению
Давайте рассмотрим две основные функции: удаление неиспользуемых шрифтов и замену их на Arial Bold.
### Функция 1: Удаление неиспользуемых шрифтов
#### Обзор
Основное внимание уделяется очистке PDF-файла путем удаления всех неиспользуемых шрифтов, уменьшения размера файла и сокращения времени загрузки.
#### Шаги по реализации
**Шаг 1:** Загрузите исходный PDF-документ
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/ReplaceTextPage.pdf";
document doc = new Document(dataDir);
```
Укажите путь к исходному PDF-документу. Убедитесь, что `dataDir` указывает правильно.
**Шаг 2:** Создайте TextFragmentAbsorber с опциями удаления шрифтов
```csharp
TextFragmentAbsorber absorber = new TextFragmentAbsorber(new TextEditOptions(TextEditOptions.FontReplace.RemoveUnusedFonts));
```
The `TextFragmentAbsorber` определяет и отмечает неиспользуемые шрифты для удаления.
**Шаг 3:** Принять Поглотитель для Всех Страниц
```csharp
doc.Pages.Accept(absorber);
```
На этом этапе обрабатывается каждая страница документа, что обеспечивает комплексную оптимизацию шрифтов.
**Шаг 4:** Заменить шрифты на Arial Bold
```csharp
foreach (TextFragment textFragment in absorber.TextFragments)
{
    textFragment.TextState.Font = FontRepository.FindFont("Arial, Bold");
}
```
После выявления неиспользуемых шрифтов замените их на Arial Bold, чтобы сохранить единообразие и профессиональный вид.
**Шаг 5:** Сохраните обновленный документ
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/RemoveUnusedFonts_out.pdf";
doc.Save(outputDir);
```
Сохраните оптимизированный документ в нужном месте, используя `outputDir`.
### Функция 2: Загрузка и сохранение PDF-документа
#### Обзор
Демонстрирует, как загрузить PDF-файл из одного места и сохранить его в другом, что упрощает его изменение и распространение.
#### Шаги по реализации
**Шаг 1:** Определить входные и выходные пути
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/Input.pdf";
string outputDir = "YOUR_OUTPUT_DIRECTORY/Output.pdf";
```
Настройте входные и выходные каталоги соответствующим образом.
**Шаг 2:** Загрузить документ
```csharp
document doc = new Document(dataDir);
```
Загрузите PDF-файл, используя его путь.
**Шаг 3:** Сохранить документ
```csharp
doc.Save(outputDir);
```
Сохраните документ в новом месте или с внесенными изменениями.

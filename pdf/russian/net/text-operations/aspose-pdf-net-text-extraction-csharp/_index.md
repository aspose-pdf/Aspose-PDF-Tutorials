## Руководство по внедрению

### Шаг 1: Откройте ваш PDF-документ

Сначала загрузите PDF-документ в свое приложение:

```csharp
// Загрузите PDF-документ
Document pdfDocument = new Document("input.pdf");
```

Это инициализирует `Document` объект, представляющий ваш PDF-файл.

### Шаг 2: Настройте параметры извлечения текста

Настройте параметры извлечения текста с помощью `TextExtractionOptions` класс. Это позволяет указать, как следует извлекать текст:

```csharp
// Установить режим извлечения текста (чистый или необработанный)
TextExtractionOptions textExtOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Pure);
```

### Шаг 3: Извлечение текста с каждой страницы

Пройдитесь по каждой странице документа, извлеките текст и добавьте его в построитель строк:

```csharp
StringBuilder builder = new StringBuilder();
string extractedText;

foreach (Page pdfPage in pdfDocument.Pages)
{
    using (MemoryStream textStream = new MemoryStream())
    {
        // Создать TextDevice для извлечения
        TextDevice textDevice = new TextDevice();
        textDevice.ExtractionOptions = textExtOptions;
        
        // Извлечь текст со страницы и сохранить в потоке памяти
        textDevice.Process(pdfPage, textStream);
        
        extractedText = Encoding.Unicode.GetString(textStream.ToArray());
        builder.Append(extractedText);
    }
}
```

### Шаг 4: Сохраните извлеченный текст

Наконец, запишите извлеченный текст в выходной файл:

```csharp
// Определить путь для выходного текстового файла
string outputPath = "input_Text_Extracted_out.txt";

// Сохранить извлеченный текст в файл
File.WriteAllText(outputPath, builder.ToString());
Console.WriteLine("\nText extracted successfully using text device from page of PDF Document.\nFile saved at " + outputPath);
```

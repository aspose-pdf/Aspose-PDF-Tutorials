### Функция: Извлечение текста из аннотации штампа

Эта функция позволяет извлекать текст, встроенный в аннотацию штампа на странице PDF. Выполните следующие шаги для ее реализации:

#### Шаг 1: Загрузите PDF-документ

Сначала укажите каталог вашего документа и загрузите PDF-файл.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/test.pdf";
Document doc = new Document(dataDir);
```

#### Шаг 2: Доступ к аннотации штампа

Получите доступ к нужной вам аннотации штампа. В этом примере мы работаем с аннотацией на первой странице.
```csharp
StampAnnotation annot = (StampAnnotation)doc.Pages[1].Annotations[3];
```

#### Шаг 3: Извлечение текста с помощью TextAbsorber

Инициализировать `TextAbsorber` извлечь текст из внешнего вида марки.
```csharp
TextAbsorber ta = new TextAbsorber();
ta.Visit(annot.Appearance.Form);
string extractedText = ta.Text;
```

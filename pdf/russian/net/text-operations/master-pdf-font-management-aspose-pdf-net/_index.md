## Руководство по внедрению

Чтобы установить шрифт по умолчанию в существующем PDF-документе, выполните следующие действия.

### Загрузка и изменение документа

#### Обзор
Загрузите целевой PDF-файл, затем примените шрифт по умолчанию с помощью Aspose.PDF `PdfSaveOptions`.

#### Пошаговый процесс

##### 1. Загрузите ваш PDF-документ
Начните с открытия PDF-файла, в котором необходимо установить шрифт по умолчанию.
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
string documentName = dataDir + "input.pdf";
```

##### 2. Настройте параметры замены шрифтов
Создать экземпляр `PdfSaveOptions` и укажите желаемое имя шрифта по умолчанию.
```csharp
using (Document document = new Document(documentName))
{
    PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
    pdfSaveOptions.DefaultFontName = "Arial";
```

##### 3. Сохраните обновленный PDF-файл.
Наконец, сохраните измененный документ в новом файле, применив шрифт по умолчанию.
```csharp
document.Save(dataDir + "output_out.pdf", pdfSaveOptions);
}
```
**Объяснение**: 
- `PdfSaveOptions.DefaultFontName` позволяет определить, какой шрифт следует использовать, если другие недоступны.
- Во избежание ошибок убедитесь, что выбранный вами шрифт установлен на компьютере, где выполняется этот процесс.

### Советы по устранению неполадок
- **Ошибка отсутствия шрифта**: Убедитесь, что указанный шрифт по умолчанию доступен в вашей среде.
- **Проблемы с доступом к файлам**: Проверьте правильность и доступность пути к входному PDF-файлу.

## Руководство по внедрению

### Поиск текста в PDF-документе

**Обзор**: эта функция позволяет искать определенный текст в документе PDF и извлекать эти сегменты для дальнейшей обработки.

#### Шаг 1: Определите путь к вашему PDF-документу
Начните с настройки пути к файлу. Заменить `YOUR_DOCUMENT_DIRECTORY` с фактическим каталогом, содержащим ваш PDF-файл.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/SearchAndGetTextFromAll.pdf");
```

#### Шаг 2: Создайте экземпляр TextFragmentAbsorber

The `TextFragmentAbsorber` класс используется для поиска текста в документе. Вы можете указать фразу поиска при создании экземпляра.

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("text");
```

#### Шаг 3: Принять поглотитель для обработки

Чтобы обработать все страницы документа, примите `TextFragmentAbsorber`:

```csharp
pdfDocument.Pages.Accept(textFragmentAbsorber);
```

#### Шаг 4: Извлечение извлеченных фрагментов текста

Соберите фрагменты текста, соответствующие вашей поисковой фразе.

```csharp
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.TextFragments;
```

#### Шаг 5: Пройдитесь по каждому фрагменту текста

Доступ и использование свойств каждого фрагмента:

```csharp
foreach (TextFragment textFragment in textFragmentCollection)
{
    string text = textFragment.Text;
    double xIndent = textFragment.Position.XIndent;
    double yIndent = textFragment.Position.YIndent;
    string fontName = textFragment.TextState.Font.FontName;
    bool isAccessible = textFragment.TextState.Font.IsAccessible;
    bool isEmbedded = textFragment.TextState.Font.IsEmbedded;
    bool isSubset = textFragment.TextState.Font.IsSubset;
    double fontSize = textFragment.TextState.FontSize;
}
```

### Соображения производительности

При работе с большими PDF-файлами для достижения оптимальной производительности примите во внимание следующие советы:
- **Управление памятью**: Утилизировать `Document` объекты, когда они больше не нужны, чтобы освободить ресурсы.
- **Пакетная обработка**: Обрабатывайте несколько PDF-файлов пакетами, а не все сразу, чтобы эффективно управлять использованием ресурсов.
- **Оптимизация поисковых запросов**: Сузьте условия поиска, чтобы сократить время обработки.

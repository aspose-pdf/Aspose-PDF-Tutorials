## Руководство по внедрению

Теперь, когда у вас все настроено, давайте сосредоточимся на извлечении текста из определенных страниц в документе PDF. Мы разобьем эту задачу на управляемые разделы.

### Извлечение текста с определенных страниц

#### Обзор
Эта функция позволяет точно извлекать текст из определенных диапазонов страниц в файле PDF, что полезно при работе с большими документами или при фокусировке только на соответствующих разделах.

#### Этапы реализации:

**Шаг 1: Настройте пути к файлам**
Определите входные и выходные каталоги:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**Шаг 2: Инициализация PdfExtractor**
Создать экземпляр `PdfExtractor` для обработки операций с PDF-файлами.
```csharp
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "/ExtractText-PageRange.pdf");
```

**Шаг 3: Определите диапазон страниц для извлечения**
Укажите, с каких страниц вы хотите извлечь текст:
```csharp
pdfExtractor.StartPage = 1;
pdfExtractor.EndPage = 1; // При необходимости отрегулируйте для конкретного диапазона страниц.
```
The `StartPage` и `EndPage` свойства определяют границы извлечения.

**Шаг 4: Извлечение текста**
Вызовите метод для извлечения текста с указанных страниц:
```csharp
pdfExtractor.ExtractText();
```

**Шаг 5: Сохранение извлеченного текста в потоке памяти**
Создать `MemoryStream` для временного хранения извлеченного текста.
```csharp
using System.IO;
using System.Text;

MemoryStream tempMemoryStream = new MemoryStream();
pdfExtractor.GetText(tempMemoryStream);
```

**Шаг 6: Прочитайте и сохраните извлеченный текст.**
Используйте `StreamReader` для чтения из потока памяти, а затем сохранения его в выходной файл:
```csharp
string text = "";
using (StreamReader sr = new StreamReader(tempMemoryStream, Encoding.Unicode))
{
    sr.BaseStream.Seek(0, SeekOrigin.Begin);
    text = sr.ReadToEnd();
}
File.WriteAllText(outputDir + "/output_out.txt", text);
```
Этот процесс захватывает извлеченный текст и сохраняет его для дальнейшего использования.

### Советы по устранению неполадок
- **Ошибки диапазона страниц**: Гарантировать `StartPage` меньше или равно `EndPage`.
- **Проблемы с MemoryStream**: Всегда сбрасывайте позицию потока перед чтением.
- **Действительность пути к файлу**: Убедитесь, что пути к каталогам указаны правильно и доступны.

## Руководство по внедрению
### Инициализировать документ и добавить страницу
**Обзор:**
Создайте новый экземпляр PDF-документа и добавьте в него страницу.

**Шаги:**
1. **Инициализировать документ:**
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Document pdfDocument = new Document();
   ```
2. **Добавить новую страницу:**
   ```csharp
   Page pdfPage = (Page)pdfDocument.Pages.Add();
   ```

### Создать текстовый абзац и задать позицию
**Обзор:**
Создайте текстовый абзац для хранения фрагментов текста и задайте его начальную позицию на странице.

**Шаги:**
1. **Создать текстовый абзац:**
   ```csharp
   TextParagraph paragraph = new TextParagraph();
   ```
2. **Установить позицию абзаца:**
   ```csharp
   paragraph.Position = new Position(200, 600);
   ```

### Создание и настройка повернутого текстового фрагмента
**Обзор:**
Создайте фрагмент текста с поворотом, чтобы продемонстрировать гибкость Aspose.PDF.

**Шаги:**
1. **Создать RotatedTextFragment:**
   ```csharp
   TextFragment rotatedText = new TextFragment("rotated text");
   ```
2. **Настройте шрифт и поворот:**
   ```csharp
   rotatedText.TextState.FontSize = 12;
   rotatedText.TextState.Font = FontRepository.FindFont("TimesNewRoman");
   rotatedText.TextState.Rotation = 45; // Повернуть на 45 градусов
   ```

### Создать и настроить основной текстовый фрагмент
**Обзор:**
Добавьте стандартный фрагмент текста для сравнения.

**Шаги:**
1. **Создать MainTextFragment:**
   ```csharp
   TextFragment mainText = new TextFragment("main text");
   ```
2. **Установить настройки шрифта:**
   ```csharp
   mainText.TextState.FontSize = 12;
   mainText.TextState.Font = FontRepository.FindFont("TimesNewRoman");
   ```

### Создать и настроить еще один повернутый текстовый фрагмент
**Обзор:**
Добавьте еще один повернутый фрагмент текста с отрицательным поворотом, чтобы продемонстрировать универсальность.

**Шаги:**
1. **Создайте еще один повернутый фрагмент текста:**
   ```csharp
   TextFragment anotherRotatedText = new TextFragment("another rotated text");
   ```
2. **Установить шрифт и отрицательное вращение:**
   ```csharp
   anotherRotatedText.TextState.FontSize = 12;
   anotherRotatedText.TextState.Font = FontRepository.FindFont("TimesNewRoman");
   anotherRotatedText.TextState.Rotation = -45; // Повернуть на -45 градусов
   ```

### Добавить фрагменты текста в абзац и на страницу
**Обзор:**
Объедините фрагменты текста в абзац, затем добавьте этот абзац на страницу PDF, используя `TextBuilder`.

**Шаги:**
1. **Добавить фрагменты к абзацу:**
   ```csharp
   paragraph.AppendLine(rotatedText);
   paragraph.AppendLine(mainText);
   paragraph.AppendLine(anotherRotatedText);
   ```
2. **Добавить абзац на страницу с помощью TextBuilder:**
   ```csharp
   TextBuilder textBuilder = new TextBuilder(pdfPage);
   textBuilder.AppendParagraph(paragraph);
   ```

### Сохранить документ
**Обзор:**
Сохраните созданный PDF-документ.

**Шаги:**
1. **Сохраните документ:**
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   pdfDocument.Save(outputDir + "/TextFragmentTests_Rotated2_out.pdf");
   ```

1. **Можно ли повернуть текст на любой угол?**

2. **Как динамически изменить размер шрифта?**
   - Отрегулируйте `FontSize` атрибут в пределах `TextState` объект для каждого фрагмента текста.
3. **Что делать, если повернутый текст перекрывает другие элементы?**
   - Поэкспериментируйте с различными начальными позициями или отрегулируйте макет страницы, чтобы избежать наложения.

## Руководство по внедрению

Мы разобьем процесс выравнивания текста в плавающих блоках на удобные для выполнения этапы.

### Создание и выравнивание плавающих блоков

#### Обзор

Плавающие блоки позволяют вам контролировать размещение текста независимо от потока страницы, идеально подходит для боковых панелей или подписей. Мы создадим три плавающих блока, выровненных в разных вертикальных положениях (внизу, по центру, сверху) на странице PDF.

#### Пошаговая реализация

**1. Создайте документ и добавьте страницу**

```csharp
// Инициализировать новый экземпляр документа
doc = new Aspose.Pdf.Document();
doc.Pages.Add(); // Добавляет новую страницу в документ
```

**2. Определите плавающий блок с выравниванием по нижнему краю**

```csharp
// Создайте плавающее поле для выравнивания по нижнему краю
Aspose.Pdf.FloatingBox floatBox = new Aspose.Pdf.FloatingBox(100, 100);
floatBox.VerticalAlignment = VerticalAlignment.Bottom;
floatBox.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right;

// Добавьте текст в плавающее поле
doc.Pages[1].Paragraphs.Add(floatBox.Paragraphs.Add(new TextFragment("FloatingBox_bottom")));

// Установить границу для видимости
floatBox.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**3. Определите плавающий блок с выравниванием по центру**

```csharp
// Создайте плавающую область для выравнивания по центру
doc.Pages[1].Paragraphs.Add(floatBox = new Aspose.Pdf.FloatingBox(100, 100) {
    VerticalAlignment = VerticalAlignment.Center,
    HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right
});

floatBox.Paragraphs.Add(new TextFragment("FloatingBox_center"));
floatBox.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**4. Определите плавающий блок с выравниванием по верхнему краю**

```csharp
// Создайте плавающее поле для выравнивания по верхнему краю
doc.Pages[1].Paragraphs.Add(floatBox2 = new Aspose.Pdf.FloatingBox(100, 100) {
    VerticalAlignment = VerticalAlignment.Top,
    HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right
});

floatBox2.Paragraphs.Add(new TextFragment("FloatingBox_top"));
floatBox2.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**5. Сохраните документ.**

```csharp
// Определите выходной каталог и сохраните документ
string dataDir = RunExamples.GetDataDir_AsposePdf_Text();
doc.Save(dataDir + "FloatingBox_alignment_review_out.pdf");
```

### Объяснение основных параметров

- **Размеры FloatingBox (100, 100):** Определяет ширину и высоту.
- **Вертикальное выравнивание:** Управляет вертикальным позиционированием на странице.
- **Горизонтальное выравнивание:** Регулирует горизонтальное выравнивание относительно других элементов.
- **Информация о границе:** Настраивает внешний вид границ для лучшей видимости во время разработки.

1. **Можно ли изменить цвет границы плавающего поля?**

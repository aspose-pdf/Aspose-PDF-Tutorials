## Руководство по внедрению
В этом разделе вы узнаете, как извлечь шрифты из документа PDF с помощью Aspose.PDF для .NET. Каждая функция разбита на управляемые шаги:

### Извлечение шрифтов из PDF-файлов
#### Обзор
Основная цель здесь — извлечь все шрифты, используемые в данном PDF-файле, что имеет решающее значение для проверки согласованности документа или подготовки файлов для определенных рабочих процессов.

#### Пошаговая реализация
**1. Загрузите документ**
Начните с загрузки вашего PDF-документа в экземпляр `Document` сорт.

```csharp
using Aspose.Pdf;

// Инициализировать объект документа
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
Document doc = new Document(dataDir + "input.pdf");
```

**2. Извлечение шрифтов**
Используйте `FontUtilities.GetAllFonts()` метод для извлечения массива шрифтов, используемых в документе.

```csharp
Aspose.Pdf.Text.Font[] fonts = doc.FontUtilities.GetAllFonts();

foreach (Aspose.Pdf.Text.Font font in fonts)
{
    Console.WriteLine(font.FontName);
}
```

**Объяснение:**
- `FontUtilities.GetAllFonts()`: Этот метод сканирует PDF-файл и возвращает массив всех используемых уникальных шрифтов.
- Цикл повторяется по каждому шрифту, выводя его имя на консоль.

#### Основные параметры конфигурации
Хотя этот пример фокусируется на базовом извлечении, Aspose.PDF предлагает различные варианты конфигурации для более сложных вариантов использования, таких как указание подмножеств шрифтов или оптимизация для встроенных шрифтов.

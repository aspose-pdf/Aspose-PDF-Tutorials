## Руководство по внедрению

### Импорт закладок из XML в PDF-документ

Эта функция позволяет вам заполнять закладки в документе PDF с помощью существующего файла XML. Выполните следующие действия:

#### Шаг 1: Настройте пути к каталогам и файлам

определите пути для входного PDF-файла, источника закладок XML и выходного PDF-файла:
```csharp
double YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
string pdfFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ImportFromXML.pdf");
string xmlFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "bookmarks.xml");
string outputPdfPath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ImportFromXML_out.pdf");
```

#### Шаг 2: Инициализация PdfBookmarkEditor

Создать экземпляр `PdfBookmarkEditor` для управления закладками PDF:
```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```

#### Шаг 3: Привязка и импорт закладок

Откройте целевой PDF-файл, затем импортируйте закладки из XML:
```csharp
bookmarkEditor.BindPdf(pdfFilePath);
bookmarkEditor.ImportBookmarksWithXML(xmlFilePath);
```
- **СвязатьPdf:** Открывает PDF-файл для редактирования.
- **ИмпортироватьЗакладкиСXML:** Импортирует закладки, определенные в XML-файле.

#### Шаг 4: Сохраните обновленный PDF-файл.

Наконец, сохраните изменения, чтобы получить выходной PDF-файл:
```csharp
bookmarkEditor.Save(outputPdfPath);
```

### Экспорт закладок из PDF-документа в XML-файл

И наоборот, вы можете извлечь существующие закладки в формат XML:

#### Шаг 1: Настройте пути к каталогам и файлам

Укажите пути для исходного PDF-файла и целевого XML-файла:
```csharp
double YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
string pdfFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ExportToXML.pdf");
string xmlOutputPath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "bookmarks_exported.xml");
```

#### Шаг 2: Инициализация PdfBookmarkEditor

Снова инициализируйте `PdfBookmarkEditor`:
```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```

#### Шаг 3: Привязка и экспорт закладок

Откройте PDF-файл и экспортируйте его закладки в XML-файл:
```csharp
bookmarkEditor.BindPdf(pdfFilePath);
bookmarkEditor.ExportBookmarksToXML(xmlOutputPath);
```
- **ЭкспортироватьЗакладкиВXML:** Конвертирует и сохраняет закладки в формат XML.

## Руководство по внедрению

Мы рассмотрим две основные функции: поиск текста и добавление гиперссылок.

### Функция 1: Поиск текста в PDF-файле

#### Обзор
Эта функция позволяет выполнять поиск определенных текстовых шаблонов с использованием регулярных выражений, что идеально подходит для структурированных данных, таких как даты или идентификаторы в документе PDF.

#### Пошаговая реализация
**Шаг 1: Определите путь к документу**
Укажите путь к вашему PDF-файлу:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/SearchRegularExpressionPage.pdf";
```
**Шаг 2: Создание TextFragmentAbsorber**
Создайте объект-поглотитель для поиска всех вхождений входной поисковой фразы с использованием регулярных выражений.
```csharp
TextFragmentAbsorber absorber = new TextFragmentAbsorber("\\d{4}-\\d{4}");
// Включить поиск по регулярным выражениям
absorber.TextSearchOptions = new TextSearchOptions(true);
```
**Шаг 3: Откройте и свяжите PDF-документ**
Откройте документ с помощью `PdfContentEditor` и привяжите его к исходному файлу.
```csharp
PdfContentEditor editor = new PdfContentEditor();
editor.BindPdf(dataDir);
editor.Document.Pages[1].Accept(absorber);
```
### Функция 2: Добавить гиперссылку к найденному тексту

#### Обзор
После обнаружения текста повысьте интерактивность, добавив гиперссылки непосредственно на эти фрагменты.

#### Пошаговая реализация
**Шаг 4: Цикл по фрагментам текста**
Пройдитесь по каждому найденному фрагменту текста и примените гиперссылку:
```csharp
using System;
using Aspose.Pdf.Annotations;
using System.Drawing;

int[] dashArray = { };
String[] LEArray = { };
Color blue = Color.Blue;

foreach (TextFragment textFragment in absorber.TextFragments)
{
    // Установить цвет фрагмента текста
    textFragment.TextState.ForegroundColor = Aspose.Pdf.Color.Blue;

    // Определите прямоугольник вокруг фрагмента текста.
    Rectangle rect = new Rectangle((int)textFragment.Rectangle.LLX,
        (int)Math.Round(textFragment.Rectangle.LLY),
        (int)Math.Round(textFragment.Rectangle.Width + 2),
        (int)Math.Round(textFragment.Rectangle.Height + 1));

    // Создать гиперссылку в прямоугольной области
    editor.CreateWebLink(rect, "http://www.aspose.com", 1, синий);

    // Добавьте подчеркивающую линию под фрагментом текста для визуального акцента.
    editor.CreateLine(rect, "",
        (float)textFragment.Rectangle.LLX + 1,
        (float)textFragment.Rectangle.LLY - 1,
        (float)textFragment.Rectangle.URX,
        (float)textFragment.Rectangle.LLY - 1,
        1, 1, blue, "S", dashArray, LEArray);
}
```
**Шаг 5: Сохраните отредактированный PDF-файл.**
Наконец, сохраните отредактированный документ:
```csharp
dataDir = dataDir.Replace("SearchRegularExpressionPage.pdf", "SearchTextAndAddHyperlink_out.pdf");
editor.Save(dataDir);
editor.Close();
```
### Советы по устранению неполадок
- **Отсутствующие фрагменты текста**Убедитесь, что ваше регулярное выражение точно соответствует желаемому текстовому шаблону.
- **Ссылка не видна**: Проверьте, что размеры прямоугольника правильно охватывают фрагмент текста.

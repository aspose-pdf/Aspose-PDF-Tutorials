### Создание и настройка локальных гиперссылок
Локальные гиперссылки позволяют пользователям легко перемещаться в пределах одного документа.

#### Шаг 1: Добавьте свою первую локальную гиперссылку
Создайте текст, который будет действовать как гиперссылка:
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Укажите ваш каталог
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Выходной каталог

// Создать новый экземпляр документа
document doc = new Document();
Page page = doc.Pages.Add();

// Шаг 1: Добавьте первую локальную гиперссылку
textFragment text1 = new TextFragment("link to page 7");
hyperlink link1 = new LocalHyperlink();
link1.TargetPageNumber = 7; // Установить номер целевой страницы для навигации
text1.Hyperlink = link1;
doc.Pages[page.Number].Paragraphs.Add(text1);
```

#### Шаг 2: Добавьте еще одну локальную гиперссылку
Добавьте еще одну гиперссылку на новую страницу документа:
```csharp
// Шаг 2: Добавьте вторую локальную гиперссылку
textFragment text2 = new TextFragment("link to page 1");
text2.IsInNewPage = true; // Убедитесь, что эта ссылка отображается на новой странице.
hyperlink link2 = new LocalHyperlink();
link2.TargetPageNumber = 1;
text2.Hyperlink = link2;
doc.Pages.Add().Paragraphs.Add(text2);
```

### Сохранение документа
Сохраните документ, включив в него гиперссылки:
```csharp
string outputFilePath = Path.Combine(outputDir, "CreateLocalHyperlink_out.pdf");
doc.Save(outputFilePath);
```

# Как извлечь гиперссылки из PDF-файлов с помощью Aspose.PDF для .NET

**Что вы узнаете:**
- Настройка Aspose.PDF для .NET в вашем проекте
- Загрузка и перемещение по PDF-документу
- Извлечение гиперссылок с определенных страниц
- Обработка действий URI внутри гиперссылок

Давайте начнем с необходимых предварительных условий.

## Руководство по внедрению
Мы разобьем это руководство на две основные функции: загрузка PDF-документа и извлечение гиперссылок.

### Загрузить и просмотреть PDF-документ
#### Обзор
Первый шаг — загрузить PDF-документ с помощью Aspose.PDF и просмотреть его страницы.

#### Пошаговая реализация
**Загрузка документа**
Начните с указания пути к входному PDF-документу:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document document = new Document(dataDir);
```

**Перемещение страниц**
Далее пройдитесь по каждой странице документа:
```csharp
foreach (Aspose.Pdf.Page page in document.Pages)
{
    // Этот блок позволит вам выполнять операции на каждой странице.
}
```
Эта настройка подготавливает ваше приложение к доступу и обработке отдельных страниц.

### Извлечение гиперссылок-аннотаций из страницы PDF
#### Обзор
Теперь давайте сосредоточимся на извлечении аннотаций гиперссылок с помощью Aspose.PDF. Мы определим гиперссылки на страницах документа и извлечем их действия URI.

#### Пошаговая реализация
**Итерация по страницам**
Продолжайте итерацию по каждой странице, как и прежде:
```csharp
foreach (Aspose.Pdf.Page page in document.Pages)
{
    AnnotationSelector selector = new AnnotationSelector(new Aspose.Pdf.Annotations.LinkAnnotation(page, Aspose.Pdf.Rectangle.Trivial));
    page.Accept(selector);
    IList<Annotation> list = selector.Selected;
```

**Извлечение гиперссылок**
В этом цикле извлеките и обработайте каждую аннотацию гиперссылки:
```csharp
foreach (LinkAnnotation link in list)
{
    if (link.Action is Aspose.Pdf.Annotations.GoToURIAction uriAction)
    {
        string destination = uriAction.URI; // Получите URI гиперссылки.
    }
}
```

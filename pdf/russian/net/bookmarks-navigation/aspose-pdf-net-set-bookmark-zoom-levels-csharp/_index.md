## Руководство по внедрению
Теперь давайте шаг за шагом реализуем функцию масштабирования закладок в вашем .NET-приложении.

### Шаг 1: Загрузите PDF-документ
Сначала загрузите PDF-документ, в котором вы хотите установить закладки:
```csharp
// Путь к каталогу документов.
string dataDir = RunExamples.GetDataDir_AsposePdf_Bookmarks();

// Открыть существующий документ
Document doc = new Document(dataDir + "input.pdf");
```

### Шаг 2: Создание и настройка закладки
Далее создайте закладку с пользовательским уровнем масштабирования, используя `XYZExplicitDestination`:
```csharp
// Инициализируйте элемент закладки с пользовательским масштабированием
OutlineItemCollection item = new OutlineItemCollection(doc.Outlines);

// Установить масштаб 0 (просмотр всей страницы) в определенных координатах
XYZExplicitDestination dest = new XYZExplicitDestination(2, 100, 100, 0);
item.Action = new GoToAction(dest);

// Добавить закладку в коллекцию структуры документа
doc.Outlines.Add(item);
```

### Шаг 3: Сохраните обновленный документ.
Наконец, сохраните изменения в файле PDF:
```csharp
dataDir += "InheritZoom_out.pdf";

// Сохраните измененный PDF-файл с обновленными закладками.
doc.Save(dataDir);

Console.WriteLine("\nBookmarks updated successfully.\nFile saved at " + dataDir);
```

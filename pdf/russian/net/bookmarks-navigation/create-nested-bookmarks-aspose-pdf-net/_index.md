### Создание PDF-файла с вложенными закладками

#### Обзор
В этом разделе вы узнаете, как создать PDF-документ с иерархическими закладками с помощью Aspose.PDF для .NET.

#### Шаг 1: Подготовьте свой проект
Создайте новое консольное приложение C# и убедитесь, что `Aspose.Pdf` библиотека указана в вашем проекте.

#### Шаг 2: Определите закладки
Закладки в Aspose.PDF представлены `Bookmark` класс. Мы создадим вложенные закладки, используя дочерние элементы.

```csharp
using System.IO;
using Aspose.Pdf.Facades;

public static void CreateNestedBookmarks()
{
    string dataDir = RunExamples.GetDataDir_AsposePdfFacades_TechnicalArticles();
    PdfContentEditor editor = new PdfContentEditor();

    // Связать PDF-документ
    editor.BindPdf(dataDir + "inFile.pdf");

    // Определить дочерние закладки
    Bookmark bm11 = new Bookmark { Action = "GoTo", PageNumber = 3, Title = "Section - 1.1.1" };
    Bookmark bm1 = new Bookmark { Action = "GoTo", PageNumber = 2, Title = "Section - 1.1" };

    // Создать коллекцию детских закладок
    Aspose.Pdf.Facades.Bookmarks bms1 = new Aspose.Pdf.Facades.Bookmarks();
    bms1.Add(bm11);
    bm1.ChildItems = bms1;

    // Определить закладку родительской главы
    Bookmark bm = new Bookmark { Action = "GoTo", PageNumber = 1, Title = "Chapter - 1" };
    Aspose.Pdf.Facades.Bookmarks bms = new Aspose.Pdf.Facades.Bookmarks();
    bms.Add(bm1);

    // Добавить дополнительные дочерние закладки
    Bookmark bm2 = new Bookmark { Action = "GoTo", PageNumber = 4, Title = "Section - 1.2" };
    bms.Add(bm2);
    
    // Назначить всю коллекцию родительской закладке
    bm.ChildItems = bms;

    // Определите другую главу с помощью дочерней закладки
    Bookmark bm_parent2 = new Bookmark { Action = "GoTo", PageNumber = 5, Title = "Chapter - 2" };
    Bookmark bm22 = new Bookmark { Action = "GoTo", PageNumber = 6, Title = "Section - 2.1" };

    Aspose.Pdf.Facades.Bookmarks bms_parent2 = new Aspose.Pdf.Facades.Bookmarks();
    bms_parent2.Add(bm22);
    bm_parent2.ChildItems = bms_parent2;

    // Сохраните документ с вложенными закладками
    editor.Save(dataDir + "Nested_BookMarks_out.pdf");
}
```

#### Объяснение параметров и методов
- **Действие**: Указывает тип действия, связанного с закладкой. Здесь он установлен на `"GoTo"` для навигации.
- **Номер страницы**: Указывает номер целевой страницы, на которую ведет закладка.
- **Заголовок**: Отображаемое имя закладки.

### Советы по устранению неполадок

1. **Неправильные номера страниц**: Убедитесь, что ваш `PageNumber` значения верны и соответствуют существующим страницам в PDF-файле.
2. **Проблемы со сбережением**: Проверьте пути к файлам и разрешения при сохранении выходного PDF-файла.

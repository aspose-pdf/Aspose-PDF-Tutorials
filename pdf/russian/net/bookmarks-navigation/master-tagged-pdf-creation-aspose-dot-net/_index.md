### Функция 1: Создание и настройка тегированного содержимого PDF-файла
#### Обзор
Создание нового документа PDF со структурированным, тегированным содержимым обеспечивает доступность. В этом разделе рассматривается инициализация документа и настройка его основных свойств, таких как заголовок и язык.

**Пошаговая реализация**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.LogicalStructure;
using Aspose.Pdf.Tagged;

// Укажите каталог для сохранения документов.
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outFile = dataDir + "/AddStructureElementIntoElement_Output.pdf";

// Инициализируйте новый экземпляр документа.
Document document = new Document();

// Получайте доступ к свойствам помеченного контента и устанавливайте необходимые метаданные.
ITaggedContent taggedContent = document.TaggedContent;
taggedContent.SetTitle("Text Elements Example");
taggedContent.SetLanguage("en-US");
```
**Объяснение:**
- **`ITaggedContent`:** Этот интерфейс используется для управления логической структурой вашего PDF-файла, делая его доступным и удобным для навигации.
- **Настройка заголовка и языка:** Эти свойства помогают определить метаданные доступности документа, помогая программам чтения с экрана правильно обрабатывать контент.

### Функция 2: Добавление элементов Paragraph и Span
#### Обзор
Добавление структурированных элементов, таких как абзацы и интервалы, имеет решающее значение для логической организации текста. В этом разделе вы узнаете, как добавить эти элементы в ваш PDF-файл с тегами.

**Пошаговая реализация**
```csharp
using Aspose.Pdf.LogicalStructure;
using Aspose.Pdf.Tagged;

// Получить корневой элемент логической структуры документа.
StructureElement rootElement = taggedContent.RootElement;

// Создайте и добавьте абзац с элементами span.
ParagraphElement p1 = taggedContent.CreateParagraphElement();
rootElement.AppendChild(p1);

SpanElement span11 = taggedContent.CreateSpanElement();
span11.SetText("Span_11");
SpanElement span12 = taggedContent.CreateSpanElement();
span12.SetText(" and Span_12.");

p1.SetText("Paragraph with ");
p1.AppendChild(span11);
p1.AppendChild(span12);
```
**Объяснение:**
- **`StructureElement`:** Выступает в качестве контейнера для других элементов логической структуры.
- **`ParagraphElement & SpanElement`:** Они используются для определения текстовых сегментов, улучшая поток документов и их читабельность.

### Функция 4: Проверка соответствия PDF/UA
#### Обзор
Обеспечение соответствия стандартам доступности, таким как PDF/UA, имеет решающее значение. В этом разделе объясняется, как проверить документ на соответствие этим стандартам.

**Выполнение**
```csharp
using Aspose.Pdf;

string logFile = "YOUR_DOCUMENT_DIRECTORY/46144_log.xml";

// Повторно откройте сохраненный документ для проверки.
Document document = new Document(outFile);

// Проверка соответствия стандарту PDF/UA-1 и вывод результатов.
bool isPdfUaCompliance = document.Validate(logFile, PdfFormat.PDF_UA_1);
```

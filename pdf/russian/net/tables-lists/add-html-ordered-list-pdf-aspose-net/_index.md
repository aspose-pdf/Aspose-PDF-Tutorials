### Добавление упорядоченного списка HTML в PDF-документ

**Обзор:**  
Эта функция позволяет вам вставлять структурированный HTML-контент, такой как списки, непосредственно в недавно созданный PDF-файл. Это идеально подходит, когда вам нужно создавать документы с динамическим веб-контентом.

#### Шаг 1: Создание объекта документа

Начните с создания нового `Document` объект:

```csharp
// Создать экземпляр класса Document
Document doc = new Document();
```

Этот объект будет служить контейнером для вашего PDF-файла.

#### Шаг 2: Определите HTML-контент

Создайте строку, содержащую ваш HTML-контент. Этот пример включает упорядоченный список:

```csharp
// Определите HTML-контент с помощью упорядоченного списка
string htmlContent = "<body style='line-height: 100px;'>\n" +
                     "<ul>\n" + 
                     "<li>First</li>\n" + 
                     "<li>Second</li>\n" + 
                     "<li>Third</li>\n" + 
                     "<li>Fourth</li>\n" + 
                     "<li>Fifth</li>\n" + 
                     "</ul>\n" +
                     "Text after the list.<br/>Next line<br/>Last line\n" + 
                     "</body>";
```

Эта строка будет преобразована в формат PDF.

#### Шаг 3: Создание экземпляра объекта HtmlFragment

Преобразуйте ваш HTML-контент в `HtmlFragment` объект:

```csharp
// Создайте HtmlFragment с HTML-содержимым
HtmlFragment htmlFragment = new HtmlFragment(htmlContent);
```

The `HtmlFragment` класс анализирует и форматирует HTML-строку для включения в PDF-файл.

#### Шаг 4: Добавьте страницу в документ

Добавьте в документ новую страницу, на которой будет отображаться список:

```csharp
// Добавить пустую страницу в документ
Page page = doc.Pages.Add();
```

Это гарантирует, что наш контент найдет место в структуре PDF-файла.

#### Шаг 5: Добавьте HtmlFragment на страницу

Включить `HtmlFragment` на вновь созданной странице:

```csharp
// Добавить HtmlFragment к коллекции абзацев страницы
page.Paragraphs.Add(htmlFragment);
```

Коллекция абзацев содержит все элементы страницы, включая текст и изображения.

#### Шаг 6: Сохраните документ.

Наконец, сохраните документ по указанному пути вместе с HTML-содержимым:

```csharp
// Определить путь к выходному файлу
string outFile = "YOUR_OUTPUT_DIRECTORY" + "/AddHTMLOrderedListIntoDocuments_out.pdf";

// Сохраните PDF-документ
doc.Save(outFile);
```

На этом этапе создается окончательный PDF-файл с вашим упорядоченным списком.

## Раздел часто задаваемых вопросов

1. **Как устранить неполадки, если HTML-контент отображается неправильно?**  
   Убедитесь, что ваш синтаксис HTML правильный и поддерживается Aspose.PDF. Проверьте наличие неподдерживаемых тегов или атрибутов.

2. **Можно ли добавить несколько фрагментов HTML на одну страницу PDF-файла?**  
   Да, вы можете добавить несколько `HtmlFragment` возражает против коллекции абзацев той же страницы.

3. **Можно ли стилизовать мои списки с помощью CSS при использовании HtmlFragment?**  
   Хотя прямая поддержка CSS ограничена, встроенные стили в вашем HTML-контенте будут применены в PDF-файле.

4. **Как мне осуществлять лицензирование для крупномасштабных приложений?**  
   Рассмотрите возможность приобретения коммерческой лицензии, если вам требуется более широкое использование, выходящее за рамки пробного периода.

5. **Какие проблемы чаще всего возникают при конвертации HTML в PDF?**  
   К распространенным проблемам относятся неподдерживаемые теги, неправильный стиль и узкие места производительности при использовании сложных макетов.

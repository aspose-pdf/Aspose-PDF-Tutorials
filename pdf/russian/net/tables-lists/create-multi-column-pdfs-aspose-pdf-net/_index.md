## Руководство по внедрению
Теперь, когда мы все настроили, давайте перейдем к созданию нашего многоколоночного PDF-документа. Мы разобьем этот процесс на управляемые шаги.

### 1. Инициализируйте документ
Начните с создания нового `Document` объект:
```csharp
Document doc = new Document();
```
Этот объект представляет весь ваш PDF-файл.

### 2. Настройте поля страницы
Настройте поля страницы, чтобы обеспечить правильное расположение контента на каждой странице:
```csharp
doc.PageInfo.Margin.Left = 40;
doc.PageInfo.Margin.Right = 40;
Page page = doc.Pages.Add();
```
Эти значения определяют расстояние между содержимым документа и его краями.

### 3. Добавьте горизонтальную линию
Чтобы визуально разделить разделы, добавьте горизонтальную линию по всей ширине страницы:
```csharp
Graph graph1 = new Graph(500.0, 2.0);
page.Paragraphs.Add(graph1);

float[] posArr = { 1, 2, 500, 2 };
Line l1 = new Line(posArr);
graph1.Shapes.Add(l1);
```
Это создает линию с использованием `Graph` и `Line` классы.

### 4. Вставьте заголовок с HTML-форматированием
Для стилизованного текста, например заголовков, используйте фрагменты HTML:
```csharp
string headingHtml = "<font face=\"Times New Roman\" size=4><strong> How to Steer Clear of Money Scams</strong></font>";
HtmlFragment headingText = new HtmlFragment(headingHtml);
page.Paragraphs.Add(headingText);
```
Это позволяет использовать HTML-теги для стилизации текста в вашем PDF-файле.

### 5. Создайте FloatingBox для многоколоночной компоновки
The `FloatingBox` класс используется для организации содержимого в столбцы:
```csharp
FloatingBox box = new FloatingBox();
box.ColumnInfo.ColumnCount = 2; // Две колонки
box.ColumnInfo.ColumnSpacing = "5";
box.ColumnInfo.ColumnWidths = "105 105";

TextFragment authorText = new TextFragment("By A Googler (The Official Google Blog)");
authorText.TextState.FontSize = 8;
authorText.TextState.FontStyle = FontStyles.Italic;
box.Paragraphs.Add(authorText);
```
Этот фрагмент создает макет из двух колонок с заданными интервалами и шириной.

### 6. Добавьте больше контента
Продолжайте добавлять контент, например горизонтальные линии или текстовые блоки:
```csharp
Graph graph2 = new Graph(50.0, 10.0);
float[] posArr2 = { 1, 10, 100, 10 };
Line l2 = new Line(posArr2);
graph2.Shapes.Add(l2);
box.Paragraphs.Add(graph2);

TextFragment sampleText = new TextFragment("Your content here...");
box.Paragraphs.Add(sampleText);
```
Это иллюстрирует добавление различных типов элементов в столбцы.

### 7. Завершите и сохраните документ.
Наконец, добавьте `FloatingBox` на страницу и сохраните документ:
```csharp
page.Paragraphs.Add(box);

string outputDir = "YOUR_OUTPUT_DIRECTORY/CreateMultiColumnPdf_out.pdf";
doc.Save(outputDir);
```
Это позволит сохранить ваш многоколоночный PDF-файл в указанном каталоге.

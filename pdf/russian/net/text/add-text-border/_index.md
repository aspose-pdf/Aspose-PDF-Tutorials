# Создание стилизованного текста с границами в PDF-документах

Aspose.PDF для .NET - это мощный API обработки PDF, который позволяет разработчикам создавать, редактировать, манипулировать и конвертировать PDF- документы без необходимости Adobe Acrobat. Этот пример показывает, как создать PDF-документ с текстовым фрагментом, который имеет определенный стиль и границу, что особенно полезно для выделения важного контента в ваших документах.

## Инструкция по установке

1. Установка через NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Ручная установка:
   * Скачать последнюю версию с [Aspose.PDF для выпусков .NET](https://releases.aspose.com/pdf/net/)
   * Добавить ссылку на Aspose.PDF.dll в вашем проекте

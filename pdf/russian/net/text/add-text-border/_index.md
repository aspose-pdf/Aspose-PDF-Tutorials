# Создание стилизованного текста с границами в PDF-документах

Aspose.PDF для .NET — это мощный API обработки PDF, который позволяет разработчикам создавать, редактировать, изменять и конвертировать PDF-документы без необходимости использования Adobe Acrobat. В этом примере показано, как создать PDF-документ с текстовым фрагментом, имеющим определенный стиль и границу, что особенно полезно для выделения важного контента в ваших документах.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.PDF
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Ручная установка:
   - Загрузите последнюю версию с сайта [Aspose.PDF для релизов .NET](https://releases.aspose.com/pdf/net/)
   - Добавьте ссылку на Aspose.PDF.dll в свой проект

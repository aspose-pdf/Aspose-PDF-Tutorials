# Добавление подчеркнутого текста в PDF с помощью Aspose.PDF для .NET

Aspose.PDF для .NET — это мощная библиотека обработки PDF, которая позволяет разработчикам создавать, изменять и манипулировать PDF-документами без необходимости использования Adobe Acrobat. В этом примере показано, как создать новый PDF-документ и добавить текст с подчеркиванием с помощью классов TextBuilder и TextFragment. Код создает простой PDF-файл с тестовым сообщением, отформатированным шрифтом Arial с подчеркиванием, размещая его в определенных координатах на странице.

## Инструкция по установке

1: Установите Aspose.PDF для .NET

Установить Aspose.PDF для .NET можно одним из следующих способов:

Использование менеджера пакетов NuGet:

```csharp
Install-Package Aspose.PDF
```

Использование .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Использование PackageReference в файле проекта:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Добавьте необходимые операторы using

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

# Добавление подчеркнутого текста в PDF с помощью Aspose.PDF для .NET

Aspose.PDF для .NET - это мощная библиотека обработки PDF, которая позволяет разработчикам создавать, изменять и манипулировать PDF-документами без необходимости Adobe Acrobat. Этот пример демонстрирует, как создать новый PDF- документ и добавлять текст с форматированием подчеркивания с использованием классов TextBuilder и TextFragment. Код создает простой PDF с тестовым сообщением, форматированным в шрифте Arial со стилированием подчеркивания, размещая его на конкретных координатах на странице.

## Инструкция по установке

1: Установка Aspose.PDF для .NET

Вы можете установить Aspose.PDF для .NET, используя один из следующих методов:

Использование NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Использование .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Использование PackageReference в файле проекта:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Добавить требуемые использования заявлений

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

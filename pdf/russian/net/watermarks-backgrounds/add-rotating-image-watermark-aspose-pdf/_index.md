## Руководство по внедрению

### Добавление водяного знака изображения с поворотом
В этом разделе вы узнаете, как добавить вращающийся водяной знак с изображением с помощью Aspose.PDF для .NET. Мы разобьем его на понятные шаги:

#### 1. Настройте пути к файлам и инициализируйте объекты
Начните с определения путей к входному PDF-файлу, выходному PDF-файлу и изображению водяного знака.

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Facades;
using System.IO;

// Определить пути к файлам
string imageFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
string inFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "inFile.pdf");
string outFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "RotatingStamp_out.pdf");

// Создайте объект PdfFileInfo для получения размеров страницы.
PdfFileInfo fileInfo = new PdfFileInfo(inFile);
```

#### 2. Создание и настройка объекта «Штамп»
Далее инициализируем `Stamp` объект и привяжите свое изображение.

```csharp
// Инициализируйте объект Stamp
Aspose.Pdf.Facades.Stamp aStamp = new Aspose.Pdf.Facades.Stamp();

// Прикрепить изображение к штампу
aStamp.BindImage(imageFile);

// Установить свойства водяного знака
aStamp.IsBackground = false; // Водяной знак как элемент переднего плана
aStamp.Pages = new int[] { 1 }; // Укажите страницы для нанесения водяного знака
aStamp.Rotation = 90; // Поворот на 90 градусов

// Расположите штамп на странице.
aStamp.SetOrigin(fileInfo.GetPageWidth(1) / 2, fileInfo.GetPageHeight(1) / 2);

// Определить размер водяного знака
aStamp.SetImageSize(100, 100);
```

#### 3. Нанесите и сохраните водяной знак.
Наконец, используйте `PdfFileStamp` чтобы применить штамп и сохранить выходной файл.

```csharp
Document doc = new Document(inFile);
PdfFileStamp stamper = new PdfFileStamp(doc);

// Добавить водяной знак в PDF
stamper.AddStamp(aStamp);

// Сохранить изменения в новом файле
stamper.Save(outFile);

// Очистите ресурсы
stamper.Close();
```

### Советы по устранению неполадок
- **Изображение не видно**: Убедитесь, что путь к изображению указан правильно и формат поддерживается.
- **Проблемы ротации**: Убедитесь, что `aStamp.Rotation` настроен правильно; по умолчанию вращение происходит вокруг центра.

## Раздел часто задаваемых вопросов
**В1: Могу ли я добавить водяные знаки на все страницы?**
- Да, установить `aStamp.Pages` в массив, содержащий все номера страниц, на которые вы хотите нанести водяной знак.

**В2: Как повернуть водяной знак на другой угол?**
- Изменять `aStamp.Rotation` до желаемого градуса (например, 45 для диагонального эффекта).

**В3: Могу ли я использовать этот метод в приложении .NET Core?**
- Безусловно, Aspose.PDF поддерживает обе среды: .NET Framework и .NET Core.

**В4: Какие форматы файлов поддерживаются в качестве водяных знаков?**
- JPEG, PNG, BMP, GIF, TIFF и др. Убедитесь, что формат изображения совместим с вашим вариантом использования.

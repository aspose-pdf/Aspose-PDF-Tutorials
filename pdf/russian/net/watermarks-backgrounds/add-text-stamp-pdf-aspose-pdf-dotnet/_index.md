### Добавление текстового штампа в PDF

#### Обзор

Добавление текстового штампа подразумевает создание экземпляра `PdfFileStamp` и настройка `Stamp` объект с желаемыми свойствами, такими как шрифт, размер, цвет и положение. Затем мы применим этот штамп к файлу PDF.

#### Пошаговая реализация
1. **Создать объект PdfFileStamp**
   Начните с создания экземпляра `PdfFileStamp`, который позволяет манипулировать PDF-файлами:
   ```csharp
   PdfFileStamp fileStamp = new PdfFileStamp();
   ```
2. **Связать PDF-документ**
   Загрузите существующий PDF-документ, используя `BindPdf`. Замените фактическим путем к вашему документу:
   ```csharp
   fileStamp.BindPdf("YOUR_DOCUMENT_DIRECTORY\AddTextStampAll.pdf");
   ```
3. **Настроить свойства штампа**
   Создать `Stamp` объект и задайте его свойства, такие как текст, стиль шрифта, размер, цвет и угол поворота:
   ```csharp
   Aspose.Pdf.Facades.Stamp stamp = new Aspose.Pdf.Facades.Stamp();
   
   // Задайте текст, стиль шрифта, размер и цвет для штампа.
   stamp.BindLogo(new FormattedText("Hello World!", Color.Blue, Color.Gray, FontStyle.Helvetica, EncodingType.Winansi, true, 14));
   
   // Определить положение на странице (координаты x, y)
   stamp.SetOrigin(200, 200);
   
   // Установить вращение для штампа
   stamp.Rotation = 90.0F;
   
   // Укажите, является ли штамп фоновым элементом.
   stamp.IsBackground = true;
   ```
4. **Добавить штамп в PDF**
   Использовать `AddStamp` Метод нанесения настроенного вами штампа на документ:
   ```csharp
   fileStamp.AddStamp(stamp);
   ```
5. **Сохранить и закрыть**
   Сохраните измененный PDF-файл под новым именем, затем закройте. `PdfFileStamp` для высвобождения ресурсов:
   ```csharp
   fileStamp.Save("YOUR_OUTPUT_DIRECTORY\AddTextStampAll_out.pdf");
   fileStamp.Close();
   ```

#### Советы по устранению неполадок
- **Отсутствующие зависимости**: Убедитесь, что все зависимости Aspose.PDF установлены правильно.
- **Ошибки пути**: Еще раз проверьте пути к файлам на наличие опечаток или неправильных имен каталогов.

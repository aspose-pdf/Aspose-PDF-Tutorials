## Руководство по внедрению

Теперь, когда все готово, давайте реализуем цифровую подпись.

### Функция: цифровая подпись PDF-файла с индивидуальным внешним видом

Эта функция позволяет настраивать то, как ваша подпись будет выглядеть в PDF-файлах. Выполните следующие действия:

#### Шаг 1: Инициализация объекта PdfFileSignature

Создать экземпляр `PdfFileSignature` скрепить и подписать документ.

```csharp
using (PdfFileSignature pdfSign = new PdfFileSignature()) {
    // Свяжите PDF-файл, который вы хотите подписать
    pdfSign.BindPdf("input.pdf");
}
```

#### Шаг 2: Определите место подписи

Создайте прямоугольник, который определит место расположения вашей подписи.

```csharp
Rectangle rect = new Rectangle(310, 45, 200, 50);
```

#### Шаг 3: Инициализация объекта PKCS7

Используйте ваш PFX-файл и пароль для инициализации `PKCS7` объект. Это представляет собой цифровой сертификат для подписи.

```csharp
PKCS7 pkcs = new PKCS7("SampleCertificate.pfx", "idsrv3test");
```

#### Шаг 4: Настройте внешний вид подписи

Настройте внешний вид своей подписи, используя `SignatureCustomAppearance`.

```csharp
SignatureCustomAppearance signatureCustomAppearance = new SignatureCustomAppearance();
signatureCustomAppearance.FontSize = 6;
signatureCustomAppearance.FontFamilyName = "Times New Roman";
signatureCustomAppearance.DigitalSignedLabel = "Signed by me";
pkcs.CustomAppearance = signatureCustomAppearance;
```

#### Шаг 5: Подпишите PDF-файл

Примените свою цифровую подпись к документу с помощью `Sign` метод.

```csharp
pdfSign.Sign(1, true, rect, pkcs);
pdfSign.Save("output.pdf");
```

### Советы по устранению неполадок
- Убедитесь, что ваш PFX-файл и пароль верны.
- Убедитесь, что у вас есть разрешения на чтение/запись соответствующих PDF-файлов.

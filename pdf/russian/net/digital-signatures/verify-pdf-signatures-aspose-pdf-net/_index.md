## Руководство по внедрению

### Проверка цифровых подписей
Основная функция этого руководства — проверка цифровых подписей в файлах PDF. Мы разобьем процесс на управляемые шаги.

#### Шаг 1: Сшиваем PDF-документ
Сначала вам необходимо связать ваш PDF-документ с помощью `PdfFileSignature`.
```csharp
// Путь к каталогу ваших документов
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_SecuritySignatures();
PdfFileSignature pdfSign = new PdfFileSignature();

// Свяжите PDF-файл
pdfSign.BindPdf(dataDir + "DigitallySign.pdf");
```
**Почему?**: Переплет PDF-файла имеет решающее значение, поскольку он позволяет `PdfFileSignature` объект для доступа и управления полями подписи в документе.

#### Шаг 2: Проверьте подпись
Далее проверьте, подписан ли документ. Этот метод проверяет наличие определенного поля подписи в вашем PDF-файле:
```csharp
// Проверьте, подписан ли документ.
if (pdfSign.VerifySigned("Signature1"))
{
    Console.WriteLine("PDF Signed");
}
```
**Почему?**: С использованием `VerifySigned` помогает определить действительность подписи в указанном поле, гарантируя целостность документа.

### Проверка подписей
Чтобы проверить наличие подписей в PDF-файле:
```csharp
// Проверьте наличие подписей.
pdfSign.ContainsSignature();
```
**Объяснение:** Этот метод возвращает логическое значение, указывающее, есть ли в документе какие-либо подписи. Это полезно для документов, которые могут содержать несколько полей подписи.

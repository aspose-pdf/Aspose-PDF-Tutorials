## Руководство по внедрению
### Проверка цифровой подписи
Проверка цифровой подписи подтверждает, что документ не был изменен с момента подписания, что гарантирует его подлинность и целостность.

#### Шаг 1: Настройте свою среду
Убедитесь, что ваша среда готова и установлены все необходимые библиотеки:
```csharp
using Aspose.Pdf.Facades;
```

#### Шаг 2: Создайте метод проверки подписи
Используйте этот фрагмент кода для создания метода проверки цифровых подписей в PDF-файлах:
```csharp
class VerifyValidSignature
{
    public static void Run()
    {
        try
        {
            // Укажите путь к каталогу для ваших документов
            string dataDir = "path_to_your_documents_directory";
            
            // Создать экземпляр PdfFileSignature
            PdfFileSignature pdfSign = new PdfFileSignature();
            
            // Связать PDF-документ
            pdfSign.BindPdf(dataDir + "DigitallySign.pdf");
            
            // Проверьте подпись
            if (pdfSign.VerifySignature("Signature1"))
            {
                Console.WriteLine("Signature Verified");
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine(ex.Message);
        }
    }       
}
```

#### Объяснение параметров кода
- `dataDir`: Каталог, в котором хранятся ваши PDF-файлы.
- `PdfFileSignature pdfSign = new PdfFileSignature()`: Инициализирует новый экземпляр для обработки подписей PDF.
- `pdfSign.BindPdf(dataDir + "DigitallySign.pdf")`: Привязывает PDF-файл к обработке.
- `pdfSign.VerifySignature("Signature1")`: Проверяет, действительна ли указанная подпись.

## Руководство по внедрению

### Обзор функций: извлечение информации о подписи

Эта функция фокусируется на извлечении информации сертификата из полей подписи PDF-файла для проверки подлинности и целостности документа.

#### Шаг 1: Определите путь к документу и загрузите PDF-файл

Укажите путь к каталогу вашего PDF-документа и загрузите его:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string input = dataDir + "ExtractSignatureInfo.pdf";

using (Document pdfDocument = new Document(input))
{
    // Продолжить обработку...
}
```
**Объяснение**: `dataDir` Переменная хранит путь к вашим документам. `input` строка формирует полный путь к файлу для загрузки.

#### Шаг 2: Перебор полей формы

Просмотрите все поля в форме PDF, чтобы найти поля подписи:

```csharp
foreach (Field field in pdfDocument.Form)
{
    SignatureField sf = field as SignatureField;
    if (sf != null)
    {
        // Поле подписи процесса
    }
}
```
**Объяснение**: Мы используем `foreach` цикл для итерации по каждому `Field` объект. `as` ключевое слово проверяет, является ли поле `SignatureField`.

#### Шаг 3: Извлечение потока сертификатов

Извлеките и считайте поток сертификата из поля подписи:

```csharp
Stream cerStream = sf.ExtractCertificate();
if (cerStream != null)
{
    byte[] bytes = new byte[cerStream.Length];
    using (cerStream)
    {
        cerStream.Read(bytes, 0, bytes.Length);
    }

    // Сохранить сертификат в файл
    string outputCerPath = dataDir + "input.cer";
    using (FileStream fs = new FileStream(outputCerPath, FileMode.CreateNew))
    {
        fs.Write(bytes, 0, bytes.Length);
    }
}
```
**Объяснение**: `ExtractCertificate()` Метод извлекает поток сертификатов. Мы считываем этот поток в массив байтов и сохраняем его как `.cer` файл.

### Советы по устранению неполадок

- **Файл не найден**: Убедитесь, что путь к документу указан правильно.
- **Нулевой поток**: Убедитесь, что PDF-файл содержит поля подписи с сертификатами.

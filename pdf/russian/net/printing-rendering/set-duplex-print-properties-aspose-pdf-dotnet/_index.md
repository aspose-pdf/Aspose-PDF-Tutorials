### Настройка предустановленных свойств для диалогового окна печати (H2)
Эта функция демонстрирует настройку дуплексных свойств в документе PDF. Давайте рассмотрим, как настроить это с помощью Aspose.PDF.

#### Обзор
Следующий код устанавливает свойство двусторонней печати, чтобы страницы печатались вдоль длинного края, что идеально подходит для профессиональных презентаций или отчетов.

#### Пошаговая реализация
**1. Создание и настройка документа**
```csharp
using Aspose.Pdf;

// Инициализировать новый PDF-документ
Document doc = new Document();

// Добавить страницу в документ
doc.Pages.Add();

// Установить дуплексное свойство
doc.Duplex = PrintDuplex.DuplexFlipLongEdge;
```
- **Объяснение:** Мы создаем новый `Document` экземпляр и добавить страницу. Настройка `doc.Duplex` к `PrintDuplex.DuplexFlipLongEdge` обеспечивает печать страниц вдоль длинной стороны.

**2. Сохраните документ.**
```csharp
// Определить путь к выходному файлу
string outputFilePath = "YOUR_OUTPUT_DIRECTORY/SetPresetPropertiesForPrintDialog_out.pdf";

// Сохранить документ с указанными настройками
doc.Save(outputFilePath, SaveFormat.Pdf);
```
- **Объяснение:** The `Save` метод записывает ваш настроенный PDF на диск. Не забудьте заменить `"YOUR_OUTPUT_DIRECTORY"` на фактический путь, по которому вы хотите сохранить файл.

### Настройка свойств диалогового окна «Печать» с помощью редактора содержимого PDF (H2)
Для существующих документов изменение настроек просмотра может иметь решающее значение для единообразного поведения печати на разных платформах.

#### Обзор
В этом разделе изменяются параметры двусторонней печати существующего PDF-файла с помощью PdfContentEditor из Aspose.PDF.

#### Пошаговая реализация
**1. Настройка и привязка документа**
```csharp
using Aspose.Pdf.Facades;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
string documentPath = "YOUR_OUTPUT_DIRECTORY/SetPrintDlgPropertiesUsingPdfContentEditor_out.pdf";

// Создать экземпляр PdfContentEditor
using (PdfContentEditor editor = new PdfContentEditor())
{
    // Связать PDF-файл для редактирования
    editor.BindPdf(inputFile);
```
- **Объяснение:** `PdfContentEditor` позволяет вносить изменения в существующие PDF-файлы. Документ привязан к редактированию путем указания его пути.

**2. Измените настройки просмотра**
```csharp
    // Получить и проверить текущие настройки
    ViewerPreferences prefs = editor.GetViewerPreference();
    
    if ((prefs & ViewerPreference.DuplexFlipShortEdge) > 0)
    {
        // Текущие предпочтения включают переворачивание короткой стороной
    }

    // Установить новые настройки просмотра для перелистывания по короткому краю
    editor.ChangeViewerPreference(ViewerPreference.DuplexFlipShortEdge);
```
- **Объяснение:** При этом проверяются текущие настройки и обновляются для переворота по короткой стороне бумаги, что повышает универсальность макетов печати.

**3. Сохранить изменения**
```csharp
    // Сохраните измененный документ.
    editor.Save(documentPath);
}
```
- **Объяснение:** The `Save` метод сохраняет изменения обратно в место хранения.

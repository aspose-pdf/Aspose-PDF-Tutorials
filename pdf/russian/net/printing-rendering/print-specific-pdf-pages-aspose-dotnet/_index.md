## Руководство по внедрению

Давайте для ясности разберем реализацию на отдельные этапы:

### Шаг 1: Определите настройки задания печати

Настройте структуру для эффективного управления заданиями печати, указав диапазоны страниц, пути к выходным файлам и настройки дуплекса. Вот как вы определяете `PrintingJobSettings` структура:

```csharp
type PrintingJobSettings = {
    ToPage: int,
    FromPage: int,
    OutputFile: string,
    Mode: Duplex
}
```

### Шаг 2: Настройте средство просмотра PDF-файлов

Далее настройте `PdfViewer` объект для обработки фактического отображения страниц:

```csharp
let viewer = new PdfViewer()
viewer.BindPdf(inPdf)
viewer.AutoResize <- true
viewer.AutoRotate <- true
viewer.PrintPageDialog <- false
```

**Объяснение:**
- **СвязатьPdf:** Прикрепляет ваш PDF-файл к средству просмотра.
- **Автоматическое изменение размера/автоматический поворот:** Обеспечивает автоматическое изменение размера и поворот страниц для оптимальной печати.
- **PrintPageDialog:** Подавляет диалоговые окна печати во время выполнения.

### Шаг 3: Настройте параметры принтера

Определите настройки принтера, которые определяют, как и где будет напечатан ваш документ:

```csharp
let ps = new PrinterSettings()
ps.PrinterName <- "Microsoft XPS Document Writer"
ps.PrintFileName <- Path.GetFullPath(printingJobs[0].OutputFile)
ps.PrintToFile <- true
ps.FromPage <- printingJobs[0].FromPage
ps.ToPage <- printingJobs[0].ToPage
ps.Duplex <- printingJobs[0].Mode
ps.PrintRange <- PrintRange.SomePages

let pgs = new PageSettings()
pgs.PaperSize <- new Aspose.Pdf.Printing.PaperSize("A4", 827, 1169)
ps.DefaultPageSettings.PaperSize <- pgs.PaperSize
pgs.Margins <- new Aspose.Pdf.Devices.Margins(0, 0, 0, 0)
```

**Объяснение:**
- **ИмяПринтера/ИмяФайлаПечати:** Указывает принтер и выходной файл.
- **Со страницы/На страницу/Дуплекс:** Управляет тем, какие страницы печатать и следует ли выполнять их двустороннюю печать.

### Шаг 4: Управление последовательной печатью

Для обработки нескольких заданий печати присоедините обработчик событий:

```csharp
type EndPrintHandler(sender: obj, args: PrintEventArgs) =
    if ++printingJobIndex < printingJobs.Count then
        ps.PrintFileName <- Path.GetFullPath(printingJobs[printingJobIndex].OutputFile)
        ps.FromPage <- printingJobs[printingJobIndex].FromPage
        ps.ToPage <- printingJobs[printingJobIndex].ToPage
        ps.Duplex <- printingJobs[printingJobIndex].Mode
        viewer.PrintDocumentWithSettings(pgs, ps)
viewer.EndPrint.AddHandler(EndPrintHandler)
```

### Шаг 5: Выполнение печати

Наконец, запустите процесс печати:

```csharp
viewer.PrintDocumentWithSettings(pgs, ps)
```

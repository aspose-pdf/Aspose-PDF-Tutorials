# Создать и редактировать PDF с помощью изображений с помощью Aspose.PDF

Aspose.PDF для .NET - мощная библиотека, которая позволяет разработчикам программируя создавать, редактировать и манипулировать PDF-документами. В этом примере вы узнаете, как создавать PDF-документ, добавлять изображение, регулировать его размер и изменять свойства страниц с использованием C#. Код демонстрирует инициализацию PDF-документа, вставку изображения, настройку размеров страницы и сохранение окончательного файла.

## Инструкция по установке:  

Чтобы использовать Aspose.PDF для .NET в вашем проекте, следуйте этим шагам:  

1. Установка через управляющий пакетами NuGet:  
Откройте Visual Studio и запустите следующую команду в консоли Package Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Добавить пространство имен в код C#:  
```csharp
using Aspose.Pdf;
```

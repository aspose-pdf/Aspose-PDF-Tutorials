## Руководство по внедрению

### Обзор вставки страницы
Вставка страниц между определенными разделами в PDF может улучшить организацию документа или добавить необходимое содержимое без ручного редактирования. Этот раздел проведет вас через использование `PdfFileEditor` для эффективного выполнения этой задачи.

#### Пошаговая реализация
**1. Создайте экземпляр PdfFileEditor**
Начните с инициализации `PdfFileEditor` объект, который предоставляет методы для работы с PDF-файлами:

```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**2. Подготовка входных и выходных потоков**
Настройте потоки для чтения исходных PDF-файлов и записи выходных данных:

```csharp
FileStream inputStream = new FileStream("source.pdf", FileMode.Open);
FileStream portStream = new FileStream("pagesToInsert.pdf", FileMode.Open);
FileStream outputStream = new FileStream("output.pdf", FileMode.Create);
```

**3. Вставка страниц с использованием потоков**
Теперь вставьте страницы из одного PDF-файла в другой в указанном месте:

```csharp
// Вставьте страницы 1–4 второго потока после страницы 1 первого потока.
pdfEditor.Insert(inputStream, 1, portStream, 1, 4, outputStream);
```

**Объяснение:**
- `inputStream`: Исходный PDF-файл, куда вставлены страницы.
- `1`: Номер страницы в исходном PDF-файле, где начинается новый контент.
- `portStream`: Поток с дополнительными страницами для вставки.
- `1, 4`: Диапазон страниц от `portStream` быть включенным.

#### Советы по устранению неполадок
- **Ошибки «Файл не найден»**: Убедитесь, что пути к файлам верны и доступны.
- **Проблемы с памятью**: Используйте потоки разумно, чтобы избежать чрезмерного использования памяти.

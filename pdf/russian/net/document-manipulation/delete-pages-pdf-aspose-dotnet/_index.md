## Руководство по внедрению

### Удаление страниц из PDF-документа
Эта функция позволяет эффективно удалять определенные страницы из документа PDF. Выполните следующие шаги для реализации этой функции:

#### 1. Создайте объект PdfFileEditor
```csharp
using Aspose.Pdf.Facades;

// Создайте экземпляр объекта PdfFileEditor
class PdfPageDeleter {
    static void Main() {
        using (PdfFileEditor pdfEditor = new PdfFileEditor()) {
            // Массив номеров страниц для удаления (индекс начинается с 1)
            int[] pagesToDelete = new int[] { 1, 2 };

            // Пути для входных и выходных файлов
            string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
            string outputFile = "YOUR_OUTPUT_DIRECTORY/DeletePagesUsingFilePath_out.pdf";

            // Выполнить удаление страницы
            pdfEditor.Delete(inputFile, pagesToDelete, outputFile);
        }
    }
}
```
Этот код инициализирует экземпляр `PdfFileEditor`, который предоставляет методы редактирования PDF-файлов.

#### 2. Укажите страницы для удаления
Определите страницы, которые вы хотите удалить, с помощью целочисленного массива:
```csharp
// Массив номеров страниц для удаления (индекс начинается с 1)
int[] pagesToDelete = new int[] { 1, 2 };
```
Здесь мы указываем, что хотим удалить первую и вторую страницы.

#### 3. Удалить страницы из PDF
Используйте `Delete` метод удаления указанных страниц:
```csharp
// Пути для входных и выходных файлов
class PdfPageDeleter {
    static void Main() {
        using (PdfFileEditor pdfEditor = new PdfFileEditor()) {
            string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
            string outputFile = "YOUR_OUTPUT_DIRECTORY/DeletePagesUsingFilePath_out.pdf";

            // Выполнить удаление страницы
            pdfEditor.Delete(inputFile, pagesToDelete, outputFile);
        }
    }
}
```
Этот код удаляет указанные страницы из `input.pdf` и сохраняет результат в `output.pdf`.

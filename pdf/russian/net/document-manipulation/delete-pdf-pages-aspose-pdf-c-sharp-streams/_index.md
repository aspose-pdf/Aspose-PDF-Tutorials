## Руководство по внедрению
Следуйте этому руководству, чтобы удалить определенные страницы из PDF-файла с помощью потоков:

### Инициализация объекта PdfFileEditor
The `PdfFileEditor` класс является центральным для изменения PDF-файлов. Начните с создания экземпляра:
```csharp
// Создать объект PdfFileEditor
PfPdFileEditor pdfEditor = new PfPdFileEditor();
```
Этот объект обрабатывает все задачи по манипулированию страницами, включая удаление.

### Создание потоков
Инициализировать `FileStream` объекты для чтения и записи данных PDF:
- **Входной поток:** Указывает на исходный PDF-файл.
- **Выходной поток:** Указывает, где будет сохранен измененный PDF-файл.
```csharp
// Создание входных и выходных потоков
using (FileStream inputStream = new FileStream("input.pdf", FileMode.Open))
using (FileStream outputStream = new FileStream("output.pdf", FileMode.Create))
{
    // Операции идут здесь
}
```

### Удаление страниц
Укажите, какие страницы следует удалить, используя целочисленный массив. Ниже показано, как удалить страницы 1 и 3:
```csharp
// Массив страниц для удаления
int[] pagesToDelete = new int[] { 1, 3 };

// Удалить указанные страницы
cpdfEditor.Delete(inputStream, pagesToDelete, outputStream);
```
- **Параметры:**
  - `inputStream`: Исходный поток PDF.
  - `pagesToDelete`: Массив, содержащий номера страниц для удаления.
  - `outputStream`Место назначения для измененного PDF-файла.

### Закрытие потоков
Всегда закрывайте потоки после операций, чтобы освободить ресурсы:
```csharp
// Потоки автоматически закрываются с помощью операторов «using» выше.
```

### Советы по устранению неполадок
- Убедитесь, что пути к файлам верны и доступны.
- Подтвердите, что номера страниц в `pagesToDelete` являются действительными (т.е. находятся в пределах диапазона PDF).

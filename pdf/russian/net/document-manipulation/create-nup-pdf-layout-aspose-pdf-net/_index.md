## Руководство по внедрению

### Понимание создания макета N-up

Макет N-up позволяет объединить несколько страниц в одну страницу с несколькими столбцами и строками. Это особенно полезно для уменьшения размера файла и расходов на печать.

#### Шаг 1: Инициализация PdfFileEditor

Начните с создания экземпляра `PdfFileEditor`:

```csharp
using Aspose.Pdf.Facades;

class MakeNUpUsingPageSizeAndPaths
{
    public static void Run()
    {
        // Определите пути исходного и выходного документа
        string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
        string outputPath = @"YOUR_OUTPUT_DIRECTORY";

        using (PdfFileEditor pdfEditor = new PdfFileEditor())
```

#### Шаг 2: Настройка параметров N-up

Укажите количество строк и столбцов для вашего макета, а также желаемый размер страницы:

```csharp
int nUpRows = 2;   // Количество строк в новом макете
int nUpColumns = 3; // Количество столбцов в новом макете
PageSize pageSize = PageSize.A5; // Размер страницы для каждой выходной страницы
```

#### Шаг 3: Выполнение операции MakeNUp

Используйте `MakeNUp` метод перестановки страниц:

```csharp
pdfEditor.MakeNUp(
    dataDir + "MultiplePages.pdf",         // Путь к входному файлу
    outputPath + "MakeNUpUsingPageSizeAndPaths_out.pdf", // Путь к выходному файлу
    nUpRows,                              // Количество строк в новом макете
    nUpColumns,                           // Количество столбцов в новом макете
    pageSize                             // Размер каждой страницы в выходном документе
);
}
```

#### Объяснение

- **Параметры**: `MakeNUp` Для этого метода требуются пути к входным и выходным файлам, количество строк и столбцов, а также размер страницы.
- **Возвращаемые значения**: Возвращает логическое значение, указывающее на успешность или неудачу операции.

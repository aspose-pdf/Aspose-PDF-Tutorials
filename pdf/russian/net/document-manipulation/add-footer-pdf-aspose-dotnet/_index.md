## Руководство по внедрению

### Добавление нижнего колонтитула на каждую страницу

В этом разделе мы покажем, как добавить текст нижнего колонтитула на каждую страницу PDF-документа с помощью `PdfFileStamp` класс из Aspose.PDF для .NET.

#### Шаг 1: Создайте объект PdfFileStamp
Начните с создания экземпляра `PdfFileStamp`. Этот объект позволяет вам размещать дополнительную информацию на страницах PDF-файла.

```csharp
PdfFileStamp fileStamp = new PdfFileStamp();
```

#### Шаг 2: Сшиваем PDF-документ

Свяжите исходный документ с `fileStamp` объект. Заменить `"YOUR_DOCUMENT_DIRECTORY"` с путем, по которому хранится ваш PDF-файл.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
fileStamp.BindPdf(dataDir + "/AddFooter.pdf");
```

#### Шаг 3: Создание форматированного текста для нижнего колонтитула

Далее создайте `FormattedText` объект, который определяет внешний вид текста вашего нижнего колонтитула. Этот шаг позволяет настраивать стиль шрифта, размер и цвет.

```csharp
FormattedText formattedText = new FormattedText(
    "Aspose - Your File Format Experts!", 
    System.Drawing.Color.Blue, 
    System.Drawing.Color.Gray, 
    Aspose.Pdf.Facades.FontStyle.Courier, 
    EncodingType.Winansi, 
    false, 14
);
```

- **Объясняемые параметры:**
  - `text`Текст нижнего колонтитула.
  - `foregroundColor`: Цвет текста.
  - `backgroundColor`: Цвет фона (необязательно).
  - `fontStyle`: Стиль шрифта, используемый для текста.
  - `encodingType`: Тип кодировки вашего текста.
  - `isHtmlTagSupported`: Логическое значение для указания поддержки HTML в тексте.
  - `fontSize`: Размер шрифта нижнего колонтитула.

#### Шаг 4: Добавьте нижний колонтитул на каждую страницу

Используйте `AddFooter` метод применения форматированного текста нижнего колонтитула с указанным отступом (в данном случае 10 единиц).

```csharp
fileStamp.AddFooter(formattedText, 10);
```

#### Шаг 5: Сохраните и закройте PDF-документ.

Наконец, сохраните обновленный документ и освободите ресурсы, закрыв `PdfFileStamp` объект.

```csharp
fileStamp.Save(dataDir + "/AddFooter_out.pdf");
fileStamp.Close();
```

## Руководство по внедрению

### Добавление текстового штампа нижнего колонтитула

#### Обзор

Эта функция позволяет автоматически размещать пользовательский текст внизу каждой страницы вашего документа PDF. Она идеально подходит для добавления уведомлений об авторских правах или элементов брендинга последовательно во всех ваших файлах.

#### Пошаговая реализация

1. **Откройте ваш PDF-документ**
   
   Загрузите PDF-файл из указанного вами каталога.
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY/TextinFooter.pdf";
   Document pdfDocument = new Document(dataDir);
   ```

2. **Создать объект TextStamp**
   
   Инициализируйте `TextStamp` с желаемым текстом нижнего колонтитула.
   
   ```csharp
   TextStamp textStamp = new TextStamp("Footer Text");
   ```

3. **Настроить свойства штампа**
   
   Установите поля и выравнивание, чтобы разместить его как нижний колонтитул на каждой странице.
   
   ```csharp
   textStamp.BottomMargin = 10;
   textStamp.HorizontalAlignment = HorizontalAlignment.Center;
   textStamp.VerticalAlignment = VerticalAlignment.Bottom;
   ```

4. **Добавить текстовый штамп на все страницы**
   
   Просмотрите страницы документа и примените штамп.
   
   ```csharp
   foreach (Page page in pdfDocument.Pages)
   {
       page.AddStamp(textStamp);
   }
   ```

5. **Сохраните обновленный документ**
   
   Укажите выходной путь и сохраните измененный PDF-файл.
   
   ```csharp
   string outputPath = "YOUR_OUTPUT_DIRECTORY/TextinFooter_out.pdf";
   pdfDocument.Save(outputPath);
   ```

#### Советы по устранению неполадок

- Убедитесь, что пути к файлам указаны правильно, чтобы избежать `FileNotFoundException`.
- Проверьте свойства выравнивания, если нижний колонтитул расположен не на своем месте.

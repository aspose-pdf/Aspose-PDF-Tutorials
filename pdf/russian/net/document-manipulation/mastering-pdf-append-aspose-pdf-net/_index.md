## Руководство по внедрению
Мы подробно рассмотрим процесс добавления PDF-файлов.

### Создать объект PdfFileEditor
The `PdfFileEditor` Класс играет центральную роль в операциях добавления, предлагая методы для легкого объединения нескольких PDF-файлов.

#### Обзор
Эта функция позволяет объединять два или более PDF-файлов, создавая экземпляр `PdfFileEditor` класс и использование его методов.

#### Этапы внедрения
**Шаг 1: Настройте пути к файлам**
Определите пути для входных и выходных файлов PDF, чтобы гарантировать правильное направление всех файловых операций:
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Pages();
```

**Шаг 2: Создание экземпляра PdfFileEditor**
Инициализировать `PdfFileEditor` объект для операции добавления.
```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**Шаг 3: Добавьте PDF-файлы**
Используйте `Append` метод объединения PDF-файлов в один выходной файл с указанием страниц, которые следует включить:
```csharp
pdfEditor.Append(dataDir + "input.pdf", dataDir + "input2.pdf", 1, 1, dataDir + "AppendFiles_out.pdf");
```

**Пояснение параметров:**
- **dataDir + "input.pdf"**: Путь к первому PDF-файлу.
- **dataDir + "input2.pdf"**: Путь ко второму PDF-файлу.
- **1, 1**Индексы страниц, указывающие, какие страницы следует добавить. Здесь добавляется только первая страница каждого входного файла.
- **dataDir + "AppendFiles_out.pdf"**: Путь к объединенному выходному файлу.

### Советы по устранению неполадок
- Убедитесь, что все пути к файлам верны и доступны.
- Убедитесь, что входные PDF-файлы не повреждены и не заблокированы разрешениями, препятствующими редактированию.

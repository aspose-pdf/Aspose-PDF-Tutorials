## Руководство по внедрению

### Извлечение страниц с использованием потоков

Эта функция позволяет извлекать определенные страницы из PDF-файла с помощью потоков, что идеально подходит для больших файлов или решений для удаленного хранения.

#### Шаг 1: Создание объекта PdfFileEditor
```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```
The `PdfFileEditor` Класс предоставляет методы для работы с PDF-документами, включая извлечение страниц.

#### Шаг 2: Настройка входных и выходных потоков
Откройте файловый поток для чтения и еще один для записи:
```csharp
FileStream inputStream = new FileStream("input.pdf", FileMode.Open);
FileStream outputStream = new FileStream("output.pdf", FileMode.Create);
```
На этом этапе определяется источник вашего PDF-документа и место сохранения извлеченных страниц.

#### Шаг 3: Укажите страницы для извлечения
Определите, какие страницы вы хотите извлечь:
```csharp
int[] pagesToExtract = { 1, 2 };
```
Измените этот массив, чтобы указать необходимые вам номера страниц.

#### Шаг 4: Выполнение извлечения
Используйте `Extract` метод извлечения указанных страниц:
```csharp
pdfEditor.Extract(inputStream, pagesToExtract, outputStream);
```
Этот метод считывает данные из входного потока и записывает в выходной поток только выбранные страницы.

### Советы по устранению неполадок

- Убедитесь, что пути к файлам указаны правильно.
- Убедитесь, что у вас есть разрешения на чтение/запись соответствующих файлов.
- Проверьте, существуют ли указанные номера страниц в исходном документе.

## Руководство по внедрению

В этом разделе мы рассмотрим загрузку и итерацию страниц PDF-файла, а также настройку размеров страниц.

### Загрузка и итерация страниц PDF
Эта функция позволяет загружать PDF-документ и просматривать его страницы для доступа или внесения изменений.

#### Шаг 1: Загрузите документ
```csharp
using System.IO;
using Aspose.Pdf;

// Загрузите ваш PDF-файл
document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

#### Шаг 2: Просмотрите страницы
Вы можете перебирать каждую страницу, используя `foreach` петля:
```csharp
foreach (Page page in doc.Pages)
{
    // Доступ к прямоугольнику MediaBox текущей страницы
    Rectangle r = page.MediaBox;
}
```
The `MediaBox` свойство предоставляет вам размеры и положение, имеющие решающее значение для настройки ориентации.

### Настройка размеров страницы
Изменение ориентации подразумевает изменение ширины и высоты страницы. Вот как:

#### Шаг 1: Рассчитайте новые размеры
Чтобы переключить страницу с книжной на альбомную или наоборот, рассчитайте новые размеры следующим образом:
```csharp
foreach (Page page in doc.Pages)
{
    Rectangle r = page.MediaBox;

    // Поменять местами высоту и ширину для изменения ориентации
    double newWidth = r.Height;
    double newHeight = r.Width;

    // Применить новые размеры к MediaBox
    page.MediaBox = new Rectangle(0, 0, newWidth, newHeight);
}
```
**Объяснение:**
- `r.Height` становится новой шириной, и `r.Width` становится новой высотой для альбомной ориентации.

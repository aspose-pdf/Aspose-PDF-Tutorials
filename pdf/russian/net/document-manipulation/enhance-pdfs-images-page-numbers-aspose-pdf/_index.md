## Руководство по внедрению

### Добавление изображения в заголовок PDF-файла

#### Обзор
Добавление изображения в заголовок PDF может сделать документы более интересными и профессиональными. В этом разделе вы узнаете, как добиться этого с помощью Aspose.PDF для .NET.

**Шаг 1: Инициализация объекта документа**
Создать новый `Document` объект, представляющий весь PDF-файл:
```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

**Шаг 2: Создание страницы и раздела заголовка**
Добавьте страницу в свой документ и создайте для нее раздел заголовка:
```csharp
// Добавить страницу
Aspose.Pdf.Page page = doc.Pages.Add();

// Инициализируйте раздел заголовка
Aspose.Pdf.HeaderFooter header = new Aspose.Pdf.HeaderFooter();
page.Header = header;
```

**Шаг 3: Вставьте изображение в заголовок**
Создайте объект изображения, укажите путь к его файлу и добавьте его в заголовок:
```csharp
Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
image1.File = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
header.Paragraphs.Add(image1);
```

**Шаг 4: Сохраните документ**
Наконец, сохраните документ, включив в него изображение заголовка:
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "HeaderWithImage_out.pdf");
doc.Save(outputFilePath);
```

### Добавление номеров страниц в нижний колонтитул PDF-файла

#### Обзор
Включение номеров страниц в нижний колонтитул PDF имеет решающее значение для документов, охватывающих несколько страниц. В этом руководстве объясняется, как это сделать с помощью Aspose.PDF для .NET.

**Шаг 1: Инициализация документа и добавление страницы**
Начните с создания нового `Document` объект:
```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
// Добавить страницу
Aspose.Pdf.Page page = doc.Pages.Add();
```

**Шаг 2: Создание раздела нижнего колонтитула**
Создайте раздел нижнего колонтитула для вашего документа:
```csharp
Aspose.Pdf.HeaderFooter footer = new Aspose.Pdf.HeaderFooter();
page.Footer = footer;
```

**Шаг 3: Добавьте номер страницы в нижний колонтитул**
Вставьте фрагмент текста, который динамически отображает номер страницы:
```csharp
Aspose.Pdf.Text.TextFragment txt = new Aspose.Pdf.Text.TextFragment("Page: ($p of $P)");
footer.Paragraphs.Add(txt);
```

**Шаг 4: Сохраните документ с нижним колонтитулом**
Сохраните документ, включив в него нижний колонтитул с номерами страниц:
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "FooterWithPageNumber_out.pdf");
doc.Save(outputFilePath);
```

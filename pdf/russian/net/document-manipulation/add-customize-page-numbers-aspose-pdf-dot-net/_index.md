### Добавить номер страницы в PDF-документ (H2)

Добавьте номера страниц внизу каждой страницы в формате «Страница X из Y».

#### Обзор
Мы будем использовать `PdfFileStamp` для проставления номеров страниц на документе. 

**Шаг 1: Инициализация PdfFileStamp**
```csharp
using Aspose.Pdf.Facades;

// Создайте новый объект PdfFileStamp
fileStamp = new PdfFileStamp();
fileStamp.BindPdf("YOUR_DOCUMENT_DIRECTORY\AddPageNumber.pdf");
```

**Шаг 2: Получите общее количество страниц**
Получите общее количество страниц для правильного форматирования номеров страниц.
```csharp
int totalPages = new PdfFileInfo("YOUR_DOCUMENT_DIRECTORY\AddPageNumber.pdf").NumberOfPages;
```

**Шаг 3: Отформатируйте и добавьте номера страниц**
Настройте внешний вид номеров страниц, задав их цвет, стиль шрифта и положение.
```csharp
using System.Drawing;
using System.Text;

FormattedText formattedText = new FormattedText(
    "Page # Of " + totalPages,
    Color.Blue,
    Color.Gray,
    FontStyle.Courier,
    EncodingType.Winansi,
    false, 14
);

fileStamp.StartingNumber = 1;
fileStamp.AddPageNumber(formattedText, 0); // Положение внизу по центру

// Сохраните и закройте объект PdfFileStamp.
testDocument.Save("YOUR_OUTPUT_DIRECTORY\AddPageNumber_out.pdf");
testDocument.Close();
```

### Настройте стиль нумерации страниц в PDF-документе (H2)

Измените стиль нумерации страниц, например, используя римские цифры.

#### Обзор
Вы можете легко настроить стиль нумерации с помощью `PdfFileStamp`.

**Шаг 1: Инициализация PdfFileStamp**
При необходимости выполните повторную инициализацию или продолжите предыдущее использование.
```csharp
// Предположим, что fileStamp уже инициализирован и привязан к PDF-документу.
```

**Шаг 2: Установка стиля нумерации**
Для этого примера выберите римские цифры в верхнем регистре.
```csharp
fileStamp.NumberingStyle = NumberingStyle.NumeralsRomanUppercase;
```

**Шаг 3: Добавьте номера страниц в пользовательском формате**
Примените пользовательский стиль нумерации к вашему документу.
```csharp
// Добавление номеров страниц в указанном формате внизу по центру
testDocument.AddPageNumber("#");

// Сохраните и закройте объект PdfFileStamp.
testDocument.Save("YOUR_OUTPUT_DIRECTORY\CustomNumberStyle_out.pdf");
testDocument.Close();
```

## Руководство по внедрению
### Разделить PDF на отдельные страницы
Эта функция позволяет разделить каждую страницу документа на отдельный файл, что может быть очень полезно для организации и совместного использования отдельных разделов.

#### Шаг 1: Откройте исходный документ.
Начните с загрузки вашего PDF-файла с помощью `Document` сорт:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/SplitToPages.pdf");
```
Этот код открывает PDF-файл из указанного каталога, подготавливая его к обработке.

#### Шаг 2: Перебор страниц
Пройдитесь по каждой странице, чтобы создать отдельные документы:
```csharp
int pageCount = 1;
foreach (Page pdfPage in pdfDocument.Pages)
{
    // Создайте новый документ для этой страницы.
    Document newDocument = new Document();
    newDocument.Pages.Add(pdfPage);

    // Сохраните одностраничный документ как PDF-файл.
    newDocument.Save($"YOUR_OUTPUT_DIRECTORY/page_{pageCount}_out.pdf");
    pageCount++;
}
```
**Объяснение**: Этот фрагмент проходит по каждой странице вашего PDF-файла, создавая новый `Document` объект для каждого из них и сохранение его как отдельного файла.

### Советы по устранению неполадок

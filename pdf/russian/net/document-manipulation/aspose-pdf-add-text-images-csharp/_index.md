## Руководство по внедрению
Давайте рассмотрим, как реализовать каждую функцию с помощью Aspose.PDF для .NET.

### Создание TextFragment
#### Обзор
Создание и добавление фрагмента текста в PDF-файл — это просто. Этот раздел проведет вас через создание «Hello World» в качестве образца контента.
**Шаг 1: Создание экземпляра документа**
```csharp
Document doc = new Document();
```
*Почему?*: Инициализирует новый PDF-документ для обработки.

**Шаг 2: Добавьте страницу**
```csharp
Page page = doc.Pages.Add();
```
*Почему?*: Добавляет пустую страницу в документ, на которой будет размещен фрагмент текста.

**Шаг 3: Создание и добавление TextFragment**
```csharp
TextFragment text = new TextFragment("Hello World.. ");
page.Paragraphs.Add(text);
```
*Почему?*: Этот фрагмент создает текстовый объект и добавляет его в коллекцию абзацев страницы.

**Шаг 4: Сохраните документ.**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(Path.Combine(outputDir, "TextFragment_out.pdf"));
```
*Почему?*: Сохраняет изменения в виде PDF-файла в указанном каталоге.

### Добавление встроенного изображения
#### Обзор
Добавление изображений в текст повышает визуальную привлекательность вашего PDF-файла. Вот как встроить изображение сразу после ранее добавленного фрагмента текста.
**Шаг 1: Создание экземпляра изображения**
```csharp
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
image.IsInLineParagraph = true;
```
*Почему?*: Настраивает изображение на обработку как встроенного, позволяя ему плавно перетекать в текстовое содержимое.

**Шаг 2: Задайте свойства изображения**
```csharp
string imagePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
image.File = imagePath;
image.FixHeight = 30;
image.FixWidth = 100;
```
*Почему?*: указывает путь к изображению и его размеры, гарантируя, что оно хорошо впишется в макет документа.

**Шаг 3: Добавьте изображение на страницу**
```csharp
page.Paragraphs.Add(image);
```
*Почему?*: Добавляет настроенное изображение как элемент абзаца в коллекцию страницы.

**Шаг 4: Сохраните документ.**
```csharp
doc.Save(Path.Combine(outputDir, "InlineImage_out.pdf"));
```
*Почему?*: Сохраняет обновленный документ со встроенным изображением.

### Добавление фрагмента встроенного текста после изображения
#### Обзор
Продолжая с того места, на котором мы остановились, давайте добавим еще один фрагмент текста после нашего изображения, чтобы создать плавный поток контента.
**Шаг 1: Повторная инициализация TextFragment**
```csharp
TextFragment text = new TextFragment(" Hello Again..");
text.IsInLineParagraph = true;
```
*Почему?*Подготавливает новый текстовый блок для добавления в строку, сразу после изображения.

**Шаг 2: Добавьте новый текстовый фрагмент на страницу**
```csharp
page.Paragraphs.Add(text);
```
*Почему?*: размещает фрагмент текста в соответствии с другими элементами контента на странице.

**Шаг 3: Сохраните документ.**
```csharp
doc.Save(Path.Combine(outputDir, "InlineTextAfterImage_out.pdf"));
```
*Почему?*: Завершает и сохраняет документ, содержащий как изображение, так и дополнительные текстовые фрагменты.

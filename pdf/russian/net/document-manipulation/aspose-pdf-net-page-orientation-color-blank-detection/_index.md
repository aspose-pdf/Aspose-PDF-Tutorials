### Изменить ориентацию страницы

#### Обзор

Изменение ориентации страницы с портретной на альбомную или наоборот осуществляется легко с помощью Aspose.PDF. Эта функция может быть решающей при подготовке документов к печати или презентации.

#### Шаги по реализации

**Шаг 1: Загрузите документ**
Начните с загрузки PDF-документа в `Aspose.Pdf.Document` объект.

```csharp
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

**Шаг 2: Перебор страниц и изменение ориентации**
Пройдитесь по каждой странице, отрегулируйте размеры и поверните их:

```csharp
foreach (Page page in doc.Pages)
{
    Aspose.Pdf.Rectangle r = page.MediaBox;
    double newHeight = r.Width; // Новая высота — это ширина страницы.
    double newWidth = r.Height; // Новая ширина — это высота страницы.

    double newLLY = r.LLY + (r.Height - newHeight);

    page.MediaBox = new Aspose.Pdf.Rectangle(r.LLX, newLLY, r.LLX + newWidth, newLLY + newHeight);
    page.CropBox = page.MediaBox;
    page.Rotate = Rotation.on90; // Поверните страницу на 90 градусов.
}
```

**Шаг 3: Сохраните измененный документ.**
Наконец, сохраните документ с обновленными ориентациями:

```csharp
doc.Save("YOUR_OUTPUT_DIRECTORY/ChangeOrientation_out.pdf");
```

#### Советы по устранению неполадок
- **Неправильные размеры:** Убедитесь, что вы правильно меняете ширину и высоту для точного изменения ориентации.
- **Проблемы с поворотом страниц:** Подтвердите, что `page.Rotate` установлен на 90 градусов (`Rotation.on90`).

### Проверьте наличие белого цвета на странице

#### Обзор
Чтобы гарантировать, что страницы будут чисто белыми (полезно при проверке качества), Aspose.PDF позволяет проверять цветовые операции.

#### Шаги по реализации

**Шаг 1: Определите метод**
Создайте метод, который проверяет, содержит ли страница только белый цвет:

```csharp
bool HasOnlyWhiteColor(Page page)
{
    foreach (Operator op in page.Contents)
    {
        if (op is SetColorOperator opSC)
        {
            Color color = opSC.getColor();
            if (color.R != 255 || color.G != 255 || color.B != 255)
                return false;
        }
    }
    return true;
}
```

**Шаг 2: Примените метод**
Используйте этот метод для проверки цветового содержания каждой страницы:

```csharp
foreach (Page page in doc.Pages)
{
    bool isWhite = HasOnlyWhiteColor(page);
    Console.WriteLine($"Page {page.Number} is {(isWhite ? "white" : "not white")}");
}
```

### Проверьте наличие пустой страницы

#### Обзор
Обнаружение пустых страниц помогает оптимизировать обработку документов за счет выявления ненужного содержимого.

#### Шаги по реализации

**Шаг 1: Определите метод**
Реализуйте метод проверки того, является ли страница пустой:

```csharp
bool IsBlankPage(Page page)
{
    return page.Contents.Count == 0 && page.Annotations.Count == 0;
}
```

**Шаг 2: Примените метод**
Просмотрите страницы и определите, какие из них пустые:

```csharp
foreach (Page page in doc.Pages)
{
    bool isBlank = IsBlankPage(page);
    Console.WriteLine($"Page {page.Number} is {(isBlank ? "blank" : "not blank")}");
}
```

## Руководство по внедрению
### Функция 1: Создание и настройка документа с содержимым LaTeX в ячейках
#### Обзор
Эта функция демонстрирует создание нового PDF-документа, добавление страницы, настройку полей ячеек и вставку текста LaTeX в виде фрагмента HTML в ячейку таблицы.
**Этапы реализации:**
##### Шаг 1: Создание нового объекта документа
```csharp
// Инициализируйте объект Document
Document document = new Document();
```
*Почему этот шаг?* Создание `Document` экземпляр необходим для начала создания структуры вашего PDF-файла.
##### Шаг 2: Добавьте страницу в свой документ
```csharp
// Добавить страницу в коллекцию страниц документа
Page page = document.Pages.Add();
```
*Цель:* Добавление страницы позволяет размещать на ней контент, например таблицы и текстовые фрагменты.
##### Шаг 3: Настройте поля ячеек
```csharp
// Определить текст LaTeX для вставки в ячейку
string latexText = "@\documentclass{article}\n@\begin{document}\nLaTeX and the document class will normally take care of page layout issues for you.\n@\end{document}";

// Создать новую таблицу, строку и ячейку с указанными полями
Aspose.Pdf.Table pdfTable = new Aspose.Pdf.Table();
pdfTable.Rows.Add().Cells.Add(latexText);
MarginInfo marginInfo = new MarginInfo { Left = 20, Right = 20, Top = 20, Bottom = 20 };
pdfTable.DefaultCellPadding = marginInfo;
```
*Конфигурация ключа:* Настройка полей гарантирует, что содержимое ячейки будет аккуратно выровнено и визуально сбалансировано.
##### Шаг 4: Добавьте текст LaTeX как HTML-фрагмент в ячейку
```csharp
// Создайте HtmlFragment, содержащий текст LaTeX.
HtmlFragment htmlFragment = new HtmlFragment(latexText);
pdfTable.Rows[0].Cells[0].Paragraphs.Add(htmlFragment);

// Добавьте таблицу в коллекцию абзацев вашей страницы.
page.Paragraphs.Add(pdfTable);

// Сохранить документ
document.Save("YOUR_DOCUMENT_DIRECTORY/LatextScriptInPdf_out.pdf");
```
*Почему?* Преобразование текста LaTeX в `HtmlFragment` позволяет сохранить его форматирование при встраивании в ячейку PDF-файла.
### Функция 2: Создание и заполнение таблицы содержимым LaTeX
#### Обзор
Эта функция расширяет предыдущую, демонстрируя, как создать таблицу, добавить строки и ячейки и заполнить их содержимым LaTeX для более сложных структур документов.
**Этапы реализации:**
##### Шаг 1: Создайте и настройте свой стол
```csharp
// Инициализировать новый объект таблицы
Table table = new Table();
```
*Цель:* Таблицы имеют основополагающее значение для организации данных в структурированных форматах в ваших PDF-документах.
##### Шаг 2: Добавьте строку в таблицу
```csharp
// Добавить строку в эту таблицу
Row row = table.Rows.Add();
```
*Объяснение:* Добавление строк позволяет размещать несколько ячеек, каждая из которых может содержать различные типы контента или стили.
##### Шаг 3: Вставьте текст в формате LaTeX в ячейки
```csharp
// Определить текст LaTeX для вставки
string latexText2 = "@\documentclass{article}\n@\begin{document}\nLaTeX and the document class will normally take care of page layout issues for you.\n@\end{document}";

// Добавить ячейку с указанными полями в эту строку
Cell cell = row.Cells.Add();
MarginInfo marginInfo2 = new MarginInfo { Left = 20, Right = 20, Top = 20, Bottom = 20 };
cell.Margin = marginInfo2;

// Вставьте LaTeX как HTMLFragment в ячейку
HtmlFragment htmlFragment2 = new HtmlFragment(latexText2);
cell.Paragraphs.Add(htmlFragment2);

// Добавьте таблицу в коллекцию абзацев вашей страницы.
page.Paragraphs.Add(table);

// Сохраните обновленный документ
document.Save("YOUR_DOCUMENT_DIRECTORY/LatextScriptInPdf_withTable_out.pdf");
```
*Почему?* С использованием `HtmlFragment` для LaTeX гарантирует сохранение сложного форматирования в выходном PDF-файле.

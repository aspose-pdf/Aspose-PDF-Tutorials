### Руководство по внедрению
В этом разделе мы рассмотрим, как поворачивать страницы PDF-файла и получать их размеры с помощью Aspose.PDF в C#.

#### Добавление пустой страницы и получение размеров
**Обзор:**
Начнем с открытия существующего PDF-документа, добавления пустой страницы при необходимости, а затем получения текущих размеров страницы.

1. **Открыть документ**

   ```csharp
   string dataDir = RunExamples.GetDataDir_AsposePdf_Pages();
   Document pdfDocument = new Document(dataDir + "UpdateDimensions.pdf");
   ```

2. **Добавить пустую страницу (при необходимости)**

   ```csharp
   // Добавляет пустую страницу, если в документе ее нет.
   Page page = pdfDocument.Pages.Count > 0 ? pdfDocument.Pages[1] : pdfDocument.Pages.Add();
   ```

3. **Получить размеры**

   ```csharp
   Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
   ```

#### Поворот страницы
**Обзор:**
Поверните страницу PDF-файла на 90 градусов и проверьте, как изменяются размеры при повороте.

1. **Повернуть страницу**

   ```csharp
   // Повернуть страницу на 90 градусов
   page.Rotate = Rotation.on90;
   ```

2. **Получить новые измерения**

   ```csharp
   Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
   ```

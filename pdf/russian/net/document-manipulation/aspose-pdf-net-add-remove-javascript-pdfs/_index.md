## Руководство по внедрению
Изучите две основные функции: добавление JavaScript в PDF-файлы и его удаление.

### Добавление функций JavaScript в PDF-документ
Добавление JavaScript может превратить ваши статические PDF-файлы в динамические инструменты. Вот как можно реализовать эту функцию:

#### Обзор
В этом разделе демонстрируется внедрение функций JavaScript в ваш PDF-документ с использованием Aspose.PDF для .NET.

#### Пошаговая реализация
**1. Создайте и настройте PDF-документ**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Инициализируйте новый документ.
Document doc = new Document();
doc.Pages.Add();  // Добавьте страницу в документ.
```

**2. Добавьте функции JavaScript**
Здесь мы добавим две простые функции с именами `func1` и `func2`.
```csharp
// Встраивайте функции JavaScript в коллекцию скриптов PDF-файла.
doc.JavaScript["func1"] = "function func1() { hello(); }";
doc.JavaScript["func2"] = "function func2() { hello(); }";

// Сохраните документ со встроенными скриптами.
doc.Save(dataDir + "/AddJavascript.pdf");
```
*Объяснение*: Мы используем `doc.JavaScript` для добавления функций. Каждая функция связана с уникальным ключом, что обеспечивает легкий доступ и манипуляции.

**Совет по устранению неполадок**: Убедитесь, что ваш код JavaScript синтаксически верен и не конфликтует с существующими скриптами в PDF-файле.

### Удаление функции JavaScript из PDF-документа
Иногда вам может понадобиться удалить определенные функции JavaScript. Вот как это сделать:

#### Обзор
В этом разделе вы узнаете, как удалить функции JavaScript из PDF-документа с помощью Aspose.PDF для .NET.

#### Пошаговая реализация
**1. Загрузите существующий PDF-файл**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Откройте документ, содержащий скрипты.
Document doc1 = new Document(dataDir + "/AddJavascript.pdf");
```

**2. Отображение функций JavaScript**
Перед удалением полезно составить список существующих функций.
```csharp
IList keys = (System.Collections.IList)doc1.JavaScript.Keys;

foreach (string key in keys)
{
    // Выведите имя каждой функции и ее код.
    Console.WriteLine($"{key}: {doc1.JavaScript[key]}");
}
```

**3. Удалить определенную функцию**
Здесь мы удаляем `func1` из документа.
```csharp
// Удалить «func1» по его ключу.
doc1.JavaScript.Remove("func1");

// При необходимости сохраните измененный PDF-файл.
doc1.Save(dataDir + "/ModifiedJavascript.pdf");
```
*Объяснение*Используйте `Remove` метод с ключом функции, чтобы исключить ее из коллекции скриптов.

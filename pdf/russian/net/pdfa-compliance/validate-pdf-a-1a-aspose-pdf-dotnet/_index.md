## Проверить стандарт PDF/A-1a
Проверка PDF-документа на соответствие стандарту PDF/A-1a обеспечивает соответствие архивным требованиям, сохраняя доступность в будущем.

### Шаг 1: Загрузите документ
Загрузите ваш PDF-файл с помощью Aspose.PDF:
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
Document pdfDocument = new Document(dataDir + "ValidatePDFAStandard.pdf");
```

### Шаг 2: Выполните проверку
Выполнить проверку на соответствие стандарту PDF/A-1a:
```csharp
string validationResultPath = dataDir + "validation-result-A1A.xml";
pdfDocument.Validate(validationResultPath, PdfFormat.PDF_A_1A);
```
**Объясняемые параметры:**
- `validationResultPath`: Путь для хранения результатов проверки.
- `PdfFormat.PDF_A_1A`: Определяет стандарт для проверки.

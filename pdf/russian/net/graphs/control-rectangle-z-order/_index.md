## Инструкция по установке
1. Убедитесь, что вы настроили среду разработки .NET (например, Visual Studio).
2. Скачать библиотеку Aspose.PDF с [Сайт Aspose](https://www.aspose.com/purchase)\- Я не знаю .
3. Установите пакет NuGet Aspose.PDF выполнением `Установка пакета Aspose.PDF`в консоли управления пакетами.
4. Включите необходимые пространства имен в свой проект:
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

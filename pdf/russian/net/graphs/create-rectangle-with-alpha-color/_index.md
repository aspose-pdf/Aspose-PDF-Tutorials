## Инструкции по установке  
Чтобы использовать Aspose.PDF для .NET в вашем проекте, следуйте этим шагам:  

1. Установка через NuGet  
Используйте следующую команду в консоли NuGet Package Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Добавить пространство имен в C#  
Включите необходимые пространства имен в начале файла C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

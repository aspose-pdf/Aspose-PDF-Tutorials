# Нарисуйте диагональные линии в PDF с помощью Aspose.PDF для .NET

Aspose.PDF для .NET позволяет разработчикам легко создавать, манипулировать и редактировать PDF-документы. В этом примере мы покажем, как программировать две диагональные линии на странице PDF с использованием C#. Вы узнаете, как инициировать документ PDF, установить свойства страницы, добавить графические элементы и сохранить конечный результат. Это руководство предоставляет пошаговый подход, облегчая интеграцию этой функциональности в ваши приложения .NET.

## Инструкция по установке:  

Чтобы использовать Aspose.PDF для .NET в вашем проекте, выполните следующие шаги:  

1. Установка с помощью NuGet Package Manager  
* Откройте Visual Studio и создайте новый .NET проект.  
* Откройте консоль менеджера пакетов (Tools → NuGet Package Manager → Package Manager Console) и запустите:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Установка через .NET CLI  
В качестве альтернативы, используйте .NET CLI для установки пакета:  
```bash
dotnet add package Aspose.PDF
```

3. Добавьте пространство имен в код C#  
Для использования функциональности Aspose.PDF включите следующие пространства имен:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Aspose.PDF для .NET позволяет разработчикам создавать, манипулировать и редактировать PDF-документы с легкостью. В этом примере мы покажем, как программируем два диагональных строка по странице PDF с использованием C#. Вы узнаете, как инициировать PDF-документ, установить свойства страницы, добавить графические элементы и сохранить окончательный выход.

## Инструкция по установке:  

Чтобы использовать Aspose.PDF для .NET в вашем проекте, следуйте этим шагам:  

1. Установка через управляющий пакетами NuGet  
* Откройте Visual Studio и создавайте новый проект .NET.  
* Откройте консоль управления пакетами (инструменты → NuGet Package Manager → Консоль управления пакетами) и запустите:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Установка через .NET CLI  
В качестве альтернативы, используйте .NET CLI для установки пакета:  
```bash
dotnet add package Aspose.PDF
```

3. Добавьте пространство имен в код C#  
Для использования функции Aspose.PDF включите следующие пространства имен:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

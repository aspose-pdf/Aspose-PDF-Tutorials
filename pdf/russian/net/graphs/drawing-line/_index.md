# Рисование диагональных линий в PDF с помощью Aspose.PDF для .NET

Aspose.PDF для .NET позволяет разработчикам с легкостью создавать, изменять и редактировать PDF-документы. В этом примере мы демонстрируем, как программно нарисовать две диагональные линии на странице PDF с помощью C#. Вы узнаете, как инициализировать PDF-документ, задать свойства страницы, добавить графические элементы и сохранить конечный результат. Это руководство предоставляет пошаговый подход, что упрощает интеграцию этой функциональности в ваши приложения .NET.

## Инструкция по установке:  

Чтобы использовать Aspose.PDF для .NET в своем проекте, выполните следующие действия:  

1. Установить через менеджер пакетов NuGet  
- Откройте Visual Studio и создайте новый проект .NET.  
- Откройте консоль диспетчера пакетов (Инструменты → Диспетчер пакетов NuGet → Консоль диспетчера пакетов) и выполните:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Установка через .NET CLI  
Либо используйте .NET CLI для установки пакета:  
```bash
dotnet add package Aspose.PDF
```

3. Добавьте пространство имен в свой код C#  
Чтобы использовать функциональность Aspose.PDF, включите следующие пространства имен:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

### Добавление заголовка изображения на каждую страницу
#### Обзор
В этом разделе вы узнаете, как использовать `PdfFileStamp` для добавления изображения в качестве заголовка на каждой странице вашего документа PDF. Это особенно полезно для брендинга или единообразного представления в документах.
#### Пошаговая реализация
**1. Создание и привязка объекта PdfFileStamp**
Начните с создания экземпляра `PdfFileStamp`, который позволяет работать с PDF-файлами:
```csharp
// Инициализировать объект PdfFileStamp
PdfFileStamp fileStamp = new PdfFileStamp();
```
**2. Откройте PDF-документ.**
Откройте целевой PDF-документ с помощью `BindPdf` Метод. Заменить `"YOUR_DOCUMENT_DIRECTORY\AddImage-Header.pdf"` с путем к вашему фактическому PDF-файлу.
```csharp
// Связать PDF-документ
fileStamp.BindPdf("YOUR_DOCUMENT_DIRECTORY\\AddImage-Header.pdf");
```
**3. Добавить изображение заголовка**
Используйте `AddHeader` метод вставки изображения в качестве заголовка на каждой странице документа. Убедитесь, что вы указали правильный путь к файлу изображения и при необходимости измените его положение.
```csharp
// Откройте FileStream для изображения заголовка
using (FileStream headerStream = new FileStream("YOUR_DOCUMENT_DIRECTORY\\AddImageHeader.jpg", FileMode.Open))
{
    // Добавить заголовок с указанной позицией
    fileStamp.AddHeader(headerStream, 10);
}
```
**4. Сохраните обновленный PDF-файл.**
Сохраните обновленный документ в новом месте с помощью `Save` метод.
```csharp
// Сохранить выходной PDF-файл
fileStamp.Save("YOUR_OUTPUT_DIRECTORY\\AddImage-Header_out.pdf");
```
**5. Освобождение ресурсов**
Наконец, убедитесь, что вы закрыли `PdfFileStamp` возражает против освобождения любых ресурсов, которыми он владеет:
```csharp
// Закрыть объект PdfFileStamp
fileStamp.Close();
```
### Советы по устранению неполадок
- **Изображение не отображается:** Убедитесь, что путь к изображению указан правильно и доступен для вашего приложения.
- **Проблемы с памятью:** Всегда удаляйте объекты FileStream правильно, используя `using` заявления или ручной вызов `Dispose()`.

## Руководство по внедрению
Теперь, когда вы настроили среду, давайте извлечем изображения из определенных страниц PDF-документа.

### Обзор функций
Наша цель — выбрать и сохранить изображения с указанных страниц в файле PDF. Эта функция может быть особенно полезна для проектов, требующих выборочного извлечения контента.

#### Шаг 1: Определите пути к каталогам
Начните с указания того, где находятся ваши входные документы и куда вы хотите сохранить извлеченные изображения:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
Заменять `YOUR_DOCUMENT_DIRECTORY` с фактическим путем к вашим PDF-файлам.

#### Шаг 2: Инициализация PdfExtractor
Создайте экземпляр `PdfExtractor` класс, который будет управлять процессом извлечения изображения:

```csharp
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "ExtractImages-Page.pdf");
```
The `BindPdf` Метод связывает ваш PDF-файл с объектом-экстрактором.

#### Шаг 3: Установите диапазон страниц
Укажите номер(а) страниц, с которых вы хотите извлечь изображения:

```csharp
pdfExtractor.StartPage = 2;
pdfExtractor.EndPage = 2;
```
Этот пример нацелен только на вторую страницу. Отрегулируйте эти значения по мере необходимости для вашего варианта использования.

#### Шаг 4: Извлечение и сохранение изображений
Используйте цикл для перебора каждого извлеченного изображения и сохранения его на диск:

```csharp
while (pdfExtractor.HasNextImage())
{
    using (MemoryStream memoryStream = new MemoryStream())
    {
        pdfExtractor.GetNextImage(memoryStream);
        
        string outputPath = @"YOUR_OUTPUT_DIRECTORY" + DateTime.Now.Ticks.ToString() + "_out.jpg";
        using (FileStream fileStream = new FileStream(outputPath, FileMode.Create))
        {
            memoryStream.WriteTo(fileStream);
        }
    }
}
```
Каждое изображение сохраняется с уникальным именем файла, основанным на текущей временной метке.

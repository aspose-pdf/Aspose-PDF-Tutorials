## Руководство по внедрению

### Добавить изображение в существующий PDF-файл

Эта функция позволяет добавлять изображения в определенные места существующих PDF-документов.

#### Обзор

Узнайте, как размещать и вставлять изображения в PDF-файл, что идеально подходит для добавления логотипов или пользовательской графики в ваши документы.

#### Шаги по реализации

##### Шаг 1: Загрузите PDF-документ

Начните с открытия существующего PDF-файла с помощью Aspose.PDF `Document` сорт.

```csharp
using System.IO;
using Aspose.Pdf;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/AddImage.pdf");
```

##### Шаг 2: Задайте координаты изображения

Определите, где на странице PDF-файла должно располагаться изображение, задав координаты.

```csharp
int lowerLeftX = 100;
int lowerLeftY = 100;
int upperRightX = 200;
int upperRightY = 200;

Page page = pdfDocument.Pages[1];
```

##### Шаг 3: Загрузите изображение в поток

Загрузите файл изображения в поток, чтобы Aspose.PDF мог получить к нему доступ и управлять им.

```csharp
FileStream imageStream = new FileStream(dataDir + "/aspose-logo.jpg", FileMode.Open);
page.Resources.Images.Add(imageStream);
```

##### Шаг 4: Разместите и нарисуйте изображение

Используйте такие операторы, как `GSave`, `ConcatenateMatrix`, и `Do` разместить и визуализировать изображение.

```csharp
page.Contents.Add(new GSave());

Rectangle rectangle = new Rectangle(lowerLeftX, lowerLeftY, upperRightX, upperRightY);
Matrix matrix = new Matrix(new double[] { rectangle.URX - rectangle.LLX, 0, 0, rectangle.URY - rectangle.LLY, rectangle.LLX, rectangle.LLY });

page.Contents.Add(new ConcatenateMatrix(matrix));
XImage ximage = page.Resources.Images[page.Resources.Images.Count];
page.Contents.Add(new Do(ximage.Name));

page.Contents.Add(new GRestore());
```

##### Шаг 5: Сохраните обновленный документ.

Наконец, сохраните документ с только что добавленным изображением.

```csharp
string outputDir = dataDir + "/AddImage_out.pdf";
pdfDocument.Save(outputDir);
```

### Добавить изображение DICOM в новый PDF-файл

Эта функция демонстрирует создание нового PDF-файла из файла DICOM, обычно используемого в медицинской визуализации.

#### Обзор

Узнайте, как преобразовывать и включать изображения DICOM в новые PDF-файлы, расширяя возможности обработки документов.

#### Шаги по реализации

##### Шаг 1: Создайте новый документ

Инициализировать новый `Document` объект, который будет служить контейнером PDF-файла.

```csharp
using Aspose.Pdf;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

using (Document pdfDocument = new Document())
{
    pdfDocument.Pages.Add();
```

##### Шаг 2: Настройте изображение DICOM

Настройте `Aspose.Pdf.Image` объект для обработки файла DICOM.

```csharp
    Aspose.Pdf.Image image = new Aspose.Pdf.Image();
    image.FileType = ImageFileType.Dicom;
    image.ImageStream = new FileStream(dataDir + "/0002.dcm", FileMode.Open, FileAccess.Read);
```

##### Шаг 3: Добавьте изображение в PDF-файл

Вставьте изображение на страницу вашего документа.

```csharp
    pdfDocument.Pages[1].Paragraphs.Add(image);

    string dicomOutputDir = dataDir + "/PdfWithDicomImage_out.pdf";
    pdfDocument.Save(dicomOutputDir);
}
```

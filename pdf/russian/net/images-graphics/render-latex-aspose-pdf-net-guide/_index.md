## Руководство по внедрению
В этом разделе мы наглядно расскажем вам о каждом этапе преобразования LaTeX в PDF-файлы.

### Создание нового документа и страницы (H2)
#### Обзор:
Начните с создания экземпляра `Document` класс и добавьте к нему новую страницу.

**Шаг 1: Инициализация документа**
```csharp
var doc = new Document();
```
*Объяснение:* Эта строка создает новый документ PDF. Впоследствии вы добавите в этот документ содержимое, включая ваш скрипт LaTeX.

#### Шаг 2: Добавить страницу
```csharp
var page = doc.Pages.Add();
```
*Цель:* Добавляет новую страницу в документ, где вы можете визуализировать свой скрипт LaTeX. Каждая страница действует как контейнер для своего содержимого в файлах PDF.

### Добавление скрипта LaTeX (H3)
Теперь давайте добавим наш скрипт LaTeX с помощью Aspose.PDF `TeXFragment`.

#### Шаг 1: Определите скрипт LaTeX
```csharp
string latexScript = """
    \usepackage{amsmath,amsthm}
    \begin{document}
    \begin{proof} The proof is as follows:
    \begin{align}
    (x+y)^3&=(x+y)(x+y)^2
    (x+y)(x^2+2xy+y^2)\\
    &=x^3+3x^2y+3xy^2+x^3.\qedhere
    \end{align}
    \end{proof}
    \end{document}"
```
*Объяснение:* Этот скрипт содержит код LaTeX для визуализации математического доказательства, включая пакеты и выравнивание для правильного форматирования.

#### Шаг 2: Создание TeXFragment
```csharp
var latex = new TeXFragment(latexScript);
```
*Цель:* Преобразует вашу строку LaTeX в `TeXFragment`, который Aspose.PDF может интерпретировать для отображения в документе PDF.

#### Шаг 3: Добавьте фрагмент на страницу
```csharp
page.Paragraphs.Add(latex);
```
*Объяснение:* Добавляет `TeXFragment` к набору абзацев страницы, встраивая ваш скрипт LaTeX в содержимое PDF-файла.

### Сохранение документа (H2)
Наконец, сохраните документ в указанном месте:

#### Шаг 1: Определите путь сохранения
```csharp
var dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
*Цель:* Задает каталог, в котором будет сохранен выходной PDF-файл. Убедитесь, что этот путь существует или доступен для записи вашим приложением.

#### Шаг 2: Сохраните документ
```csharp
doc.Save(dataDir + "Script_out.pdf");
```
*Объяснение:* Эта команда записывает отрисованный документ на диск, создавая PDF-файл с содержимым LaTeX в указанном месте.

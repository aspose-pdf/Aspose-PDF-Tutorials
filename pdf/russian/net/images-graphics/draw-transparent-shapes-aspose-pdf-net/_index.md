### Рисование фигур на странице с помощью Aspose.PDF .NET

#### Обзор

Добавление пользовательских фигур, таких как прямоугольники, в ваши PDF-файлы может сделать их более интересными и информативными. С Aspose.PDF вы можете легко добавлять эти элементы.

#### Пошаговая реализация

**1. Создание экземпляра объекта документа**

Начните с создания нового `Document` объект, который будет служить контейнером для ваших страниц и контента.

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;

// Создать экземпляр объекта Document
Document document = new Document();
```

**2. Добавить страницу в PDF-документ**

Добавьте новую страницу, на которой вы будете рисовать фигуры:

```csharp
Page page = document.Pages.Add();
```

**3. Создание и настройка графического объекта**

The `Graph` объект используется для определения области рисования на странице:

```csharp
// Создать объект Graph с определенными размерами
Graph graph = new Graph(300.0, 400.0);
graph.Border = (new BorderInfo(BorderSide.All, Color.Black));
page.Paragraphs.Add(graph);
```

**4. Нарисуйте прямоугольник**

Определите размер и положение прямоугольника:

```csharp
Rectangle rectangle = new Rectangle(0, 0, 100, 50);
GraphInfo graphInfo = rectangle.GraphInfo;
graphInfo.Color = Color.Red; // Цвет контура
graphInfo.FillColor = Color.FromArgb(10, 100, 0, 0); // Прозрачная заливка

// Добавить прямоугольник в коллекцию фигур графического объекта
graph.Shapes.Add(rectangle);
```

**5. Сохраните PDF-файл.**

Наконец, сохраните документ со всеми нарисованными элементами:

```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY" + "/AddDrawing_out.pdf";
document.Save(outputFilePath);
```

### Установка прозрачного цвета заливки для фигур

#### Обзор

Использование прозрачности в цветах заливки может добавить глубину и четкость вашим фигурам. Aspose.PDF позволяет устанавливать значения RGBA для точного управления.

#### Пошаговая реализация

**1. Определите компоненты RGBA**

Установите значение альфа для определения прозрачности:

```csharp
int alpha = 10; // Уровень прозрачности: от 0 (полностью прозрачный) до 255 (непрозрачный)
Color alphaColor = Color.FromArgb(alpha, 100, 0, 0);
```

**2. Применить прозрачный цвет заливки**

Назначьте этот цвет вашему `GraphInfo` объект для формы:

```csharp
rectangle.GraphInfo.FillColor = alphaColor;
graph.Shapes.Add(rectangle);
document.Save(outputFilePath);
```

## Руководство по внедрению: извлечение изображений из PDF-файла

Когда все настроено, давайте извлечем изображения из файла PDF. Эта функция полезна для автоматизации задач обработки документов.

### Обзор

Мы будем использовать `PdfExtractor` класс для извлечения и сохранения изображений, встроенных в PDF-файл. Следуйте этому пошаговому процессу:

### Пошаговая реализация

#### 1. Настройка каталогов

Определите входные и выходные каталоги:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Убедитесь, что этот каталог существует
```

Такая организация помогает эффективно управлять файлами во время извлечения.

#### 2. Инициализируйте PdfExtractor

Создать экземпляр `PdfExtractor` и привяжите PDF-файл:

```csharp
// Создать экземпляр PdfExtractor
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "/ExtractImages.pdf");
```

**Зачем нужна привязка?** Привязка связывает ваш документ с экстрактором для последующих операций.

#### 3. Выполнить извлечение изображения

Вызовите `ExtractImage` метод начала извлечения:

```csharp
// Извлечение изображений из PDF-файла
pdfExtractor.ExtractImage();
```

Он сканирует страницы и идентифицирует встроенные объекты изображений.

#### 4. Сохраните извлеченные изображения

Пройдитесь по каждому извлеченному изображению, сохраняя их под уникальными именами файлов:

```csharp
while (pdfExtractor.HasNextImage())
{
    string outputFileName = $"{outputDir}/{DateTime.Now.Ticks}_out.jpg";
    pdfExtractor.GetNextImage(outputFileName);
}
```

**Зачем использовать DateTime?** С использованием `DateTime.Now.Ticks` гарантирует, что каждый файл изображения имеет уникальное имя, предотвращая перезапись.

### Функция 1: Загрузка документа и извлечение информации о странице

**Обзор:** Узнайте, как загрузить PDF-документ и получить размеры его страницы с помощью Aspose.PDF.

#### Шаг 1: Определите входной путь
Укажите каталог, в котором находится ваш входной PDF-файл. 

```csharp

```

#### Шаг 2: Загрузите документ

Использовать `Document` класс для загрузки PDF:

```csharp
document doc = new Document(dataDir);
```

#### Шаг 3: Информация о странице доступа

Получить размеры первой страницы:

```csharp
Page page = doc.Pages[1];
float width = page.PageInfo.Width;
float height = page.PageInfo.Height;

Console.WriteLine($"Page Width: {width}, Page Height: {height}");
```

**Объяснение:** Этот код получает доступ к `PageInfo` свойство для извлечения размеров страницы, что может быть полезно для корректировки макета или проверки.

#### Шаг 2: Сохраните растровое изображение

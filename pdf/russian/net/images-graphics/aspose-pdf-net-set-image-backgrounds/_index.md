## Руководство по внедрению (H2)

### Установить изображение как фон (H2)

В этом разделе вы узнаете, как настроить фоновое изображение в вашем PDF-файле.

#### Шаг 1: Создайте новый документ

Начните с создания `Document` объект:

```csharp
// Инициализировать документ
document doc = new Document();
```

Этот фрагмент инициализирует новый экземпляр `Document` класс, представляющий ваш PDF-файл.

#### Шаг 2: Добавьте страницу в документ

Добавьте страницу, на которой вы установите фоновое изображение:

```csharp
Page page = doc.Pages.Add();
```

Здесь мы добавляем пустую страницу с помощью `Pages.Add()` метод.

#### Шаг 3: Создание и настройка BackgroundArtifact

The `BackgroundArtifact` класс используется для установки фонов в Aspose.PDF. Вот как его настроить:

```csharp
BackgroundArtifact background = new BackgroundArtifact();
background.BackgroundImage = File.OpenRead(@"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg");
page.Artifacts.Add(background);
```

- **Почему?** The `BackgroundArtifact` класс предназначен для обработки фонов на страницах PDF. Добавление этого артефакта гарантирует, что изображение будет отображаться как фон.
- Заменять `"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg"` с желаемым путем к фоновому изображению.

#### Шаг 4: Сохраните документ

Наконец, сохраните ваш документ:

```csharp
string dataDir = @"YOUR_OUTPUT_DIRECTORY\ImageAsBackground_out.pdf";
doc.Save(dataDir);
```

Этот код сохраняет ваш PDF-файл с указанным выходным каталогом и именем файла. Заменить `"YOUR_OUTPUT_DIRECTORY"` с желаемым местом сохранения.

## Руководство по внедрению

В этом разделе мы рассмотрим изменение паролей PDF-файлов с помощью `PdfFileSecurity` класс из Aspose.PDF.

### Изменение паролей PDF-файлов

**Обзор:** Эта функция позволяет изменять пароли владельца и пользователя PDF-документа, повышая его безопасность.

#### Шаг 1: Создайте экземпляр PdfFileSecurity
```csharp
using System;
using Aspose.Pdf.Facades;

// Инициализируйте объект безопасности файла
PdfFileSecurity fileSecurity = new PdfFileSecurity();
```
**Объяснение:** Начнем с создания экземпляра `PdfFileSecurity`, который обрабатывает все операции, связанные с безопасностью PDF-файлов.

#### Шаг 2: Загрузите существующий PDF-документ
```csharp
// Загрузите существующий PDF-документ
fileSecurity.BindPdf("YOUR_DOCUMENT_DIRECTORY/ChangePassword.pdf");
```
**Параметры:** Метод принимает в качестве входных данных путь к файлу, указывающий на PDF, который вы хотите защитить. Убедитесь, что путь правильный, чтобы избежать ошибок.

#### Шаг 3: Измените пароли пользователя и владельца.
```csharp
// Измените пароли пользователя и владельца.
fileSecurity.ChangePassword("owner", "newuserpassword", "newownerpassword");
```
**Параметры:**
- `oldUserPassword`: Текущий пароль (если есть) для открытия документа.
- `newUserPassword`: Новый пароль для открытия PDF-файла.
- `newOwnerPassword`: Пароль нового владельца, контролирующий такие разрешения, как печать и изменение.

#### Шаг 4: Сохраните обновленный документ.
```csharp
// Сохраните обновленный PDF-файл с новым паролем.
fileSecurity.Save("YOUR_OUTPUT_DIRECTORY/ChangeFilePassword_out.pdf");
```
**Объяснение:** Сохраните изменения в желаемом выходном каталоге. Это гарантирует, что у вас будет обновленная копия документа с новыми установленными паролями.

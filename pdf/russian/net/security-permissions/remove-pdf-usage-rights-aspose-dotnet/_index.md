## Руководство по внедрению

Теперь, когда вы настроили свою среду, давайте перейдем к удалению прав использования из PDF-документа.

### Функция: удаление прав на использование документа

Эта функция позволяет вам удалить ограничения использования, встроенные в файл PDF. Выполните следующие действия:

#### Шаг 1: Определите входные и выходные пути
Укажите пути для входного PDF-файла и выходного файла.
```csharp
string inputPath = @"YOUR_DOCUMENT_DIRECTORY\DigitallySign.pdf";
string outputPath = @"YOUR_OUTPUT_DIRECTORY\RemoveRights_out.pdf";
```

#### Шаг 2: Инициализация объекта PdfFileSignature
Создать `PdfFileSignature` объект для взаимодействия с вашим PDF-документом.
```csharp
using (PdfFileSignature pdfSign = new PdfFileSignature())
{
    // Свяжите входной PDF-документ.
    pdfSign.BindPdf(inputPath);
}
```
Этот объект позволяет вам манипулировать и сохранять изменения в PDF-файле.

#### Шаг 3: Проверьте права использования
Прежде чем пытаться удалить документ, проверьте, содержит ли он какие-либо права на использование.
```csharp
if (pdfSign.ContainsUsageRights())
{
    // При наличии прав продолжите удаление.
}
```

#### Шаг 4: Удаление прав использования
Удалите все встроенные права использования из PDF-файла.
```csharp
pdfSign.RemoveUsageRights();
```
Этот шаг гарантирует, что ваш документ больше не будет ограничен предыдущими разрешениями пользователя.

#### Шаг 5: Сохраните измененный документ.
Сохраните изменения в новом выходном файле.
```csharp
pdfSign.Document.Save(outputPath);
```

### Советы по устранению неполадок
- Убедитесь, что пути указаны правильно и доступны.
- Обрабатывайте исключения с помощью блоков try-catch для перехвата любых ошибок во время выполнения.

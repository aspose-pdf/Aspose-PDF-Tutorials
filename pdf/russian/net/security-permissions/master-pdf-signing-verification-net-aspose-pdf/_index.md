## Руководство по внедрению

### Механизм подписания PDF-файлов

#### Обзор

Подписание PDF-документа гарантирует его подлинность с помощью цифровых сертификатов. Эта функция защищает документы цифровой подписью, которую можно проверить позже.

#### Действия по подписанию PDF-документа

##### 1. Подготовьте свое окружение
Убедитесь, что у вас готовы входные PDF-файлы и цифровой сертификат.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/inFile.pdf");
PdfFileSignature pdfSign = new PdfFileSignature(doc);
```

##### 2. Определите внешний вид подписи

Настройте внешний вид подписи с помощью файла изображения.

```csharp
Rectangle rect = new Rectangle(100, 100, 200, 100);
pdfSign.SignatureAppearance = dataDir + "/aspose-logo.jpg";
```

##### 3. Создание и применение подписи PKCS1

Используйте сертификат для создания подписи PKCS1 и примените ее к документу.

```csharp
PKCS1 signature = new PKCS1(dataDir + "/inFile2.pdf", "password");
pdfSign.Sign(1, "Signature Reason", "Alice", "Odessa", true, rect, signature);
```

#### Советы по устранению неполадок

- Убедитесь, что файл сертификата доступен и имеет правильные разрешения.
- Убедитесь, что пути к изображениям указаны правильно.

### Добавить поля подписи

#### Обзор

Добавление полей подписи позволяет пользователям подписывать документы цифровой подписью в определенных местах.

#### Действия по добавлению полей подписи

##### 1. Загрузите ваш PDF-документ
Инициализируйте объект FormEditor с вашим документом.

```csharp
Document doc = new Document(dataDir + "/inFile.pdf");
FormEditor editor = new FormEditor(doc);
```

##### 2. Определите и добавьте поля подписи
Укажите местоположение каждого поля подписи на нужной странице.

```csharp
teditor.AddField(FieldType.Signature, "Signature from Alice", 1, 10, 10, 110, 110);
editor.AddField(FieldType.Signature, "Signature from John", 1, 120, 150, 220, 250);
editor.AddField(FieldType.Signature, "Signature from Smith", 1, 300, 200, 400, 300);
```

##### 3. Сохраните документ.
Сохранить изменения в новом файле.

```csharp
teditor.Save(dataDir + "/AddSignatureFields_1_out.pdf");
```

#### Советы по устранению неполадок

- Убедитесь, что координаты полей находятся в пределах границ страницы.
- Убедитесь, что ваш документ не защищен паролем и не заблокирован от редактирования.

### Проверить подписи

#### Обзор
Проверка гарантирует, что цифровые подписи в PDF-файле действительны и не были подделаны.

#### Действия по проверке подписей

##### 1. Загрузите документ для проверки
Инициализируйте PdfFileSignature с целевым файлом.

```csharp
PdfFileSignature pdfVerify = new PdfFileSignature();
pdfVerify.BindPdf(dataDir + "/inFile.pdf");
```

##### 2. Проверка и подтверждение подписей
Определите, существуют ли какие-либо подписи, затем проверьте их по имени или индексу.

```csharp
bool isSigned = pdfVerify.ContainsSignature();
bool isSignatureVerified = pdfVerify.VerifySignature("Signature1");
bool isSignatureVerified2 = pdfVerify.VerifySignature("Signature from Alice");
```

#### Советы по устранению неполадок

- Убедитесь, что документ не был изменен после подписания.
- Для проверки используйте правильные названия подписей или индексы.

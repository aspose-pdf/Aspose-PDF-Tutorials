### Понимание проверки пароля

Цель состоит в том, чтобы определить, может ли заданный набор паролей разблокировать зашифрованный PDF-файл. Вот как это реализовать:

#### Шаг 1: Загрузите PDF-документ

Загрузите исходный PDF-файл и проверьте, защищен ли он паролем, используя `PdfFileInfo`.
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Facades;

// Путь к каталогу документов.
string dataDir = "path/to/your/files/";

// Загрузить исходный PDF-файл
PdfFileInfo info = new PdfFileInfo();
info.BindPdf(dataDir + "IsPasswordProtected.pdf");

// Определите, зашифрован ли исходный PDF-файл
Console.WriteLine("File is password protected: " + info.IsEncrypted);
```

#### Шаг 2: Перебор паролей

Попробуйте каждый пароль из предопределенного списка и отловите любые исключения для неверных паролей.
```csharp
String[] passwords = new String[5] { "test", "test1", "test2", "test3", "sample" };

for (int passwordcount = 0; passwordcount < passwords.Length; passwordcount++)
{
    try
    {
        Document doc = new Document(dataDir + "IsPasswordProtected.pdf", passwords[passwordcount]);
        if (doc.Pages.Count > 0)
            Console.WriteLine("Password is correct. Number of pages: " + doc.Pages.Count);
    }
    catch (InvalidPasswordException)
    {
        Console.WriteLine("Password = " + passwords[passwordcount] + " is not correct");
    }
}
```

### Основные параметры конфигурации

- **PdfFileInfo**: Предоставляет информацию о PDF-файле, включая статус шифрования.
- **Конструктор документов**: Попытка открыть документ с указанным паролем.

### Советы по устранению неполадок

- Убедитесь, что путь к вашим PDF-файлам указан правильно. `dataDir`.
- Правильно обрабатывайте исключения, чтобы избежать сбоев при вводе неверных паролей.

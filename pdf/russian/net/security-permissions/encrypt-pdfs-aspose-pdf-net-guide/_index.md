## Руководство по внедрению

### Шифрование PDF-файлов с помощью паролей пользователя и владельца

Эта функция позволяет вам защитить ваши PDF-файлы, установив пароли пользователя и владельца. Вот как это работает:

#### Шаг 1: Сшиваем PDF-документ
Сначала загрузите документ, который вы хотите зашифровать:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        fileSecurity.BindPdf("YOUR_DOCUMENT_DIRECTORY/Encrypt.pdf");
    }
}
```
**Почему?** Этот шаг имеет решающее значение, поскольку он загружает целевой PDF-файл в память для обработки.

#### Шаг 2: Примените шифрование
Теперь примените шифрование с указанными разрешениями и 256-битное шифрование AES:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        // Зашифруйте файл с помощью паролей пользователя и владельца, разрешив только печать
        fileSecurity.EncryptFile("user", "owner", DocumentPrivilege.Print, KeySize.x256, Algorithm.AES);
    }
}
```
**Объясняемые параметры:**
- `"user"`: Пароль, необходимый для открытия PDF-файла.
- `"owner"`: Пароль для изменения разрешений или снятия шифрования.
- `DocumentPrivilege.Print`: Ограничивает доступ только к печати.
- `KeySize.x256`: Определяет 256-битное шифрование AES.
- `Algorithm.AES`: Использует расширенный стандарт шифрования.

#### Шаг 3: Сохраните зашифрованный файл.
Наконец, сохраните зашифрованный документ:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        fileSecurity.Save("YOUR_OUTPUT_DIRECTORY/Encrypt_out.pdf");
    }
}
```
**Почему?** На этом этапе все изменения записываются в новый файл, сохраняя исходный PDF-файл.

### Советы по устранению неполадок
- **Файл не найден**: Убедитесь, что ваши пути правильны и доступны.
- **Проблемы с разрешением**: Убедитесь, что ваше приложение имеет достаточные разрешения для чтения/записи файлов в указанных каталогах.
- **Неправильные пароли**: Еще раз проверьте пароли пользователя и владельца на наличие опечаток или ошибок формата.

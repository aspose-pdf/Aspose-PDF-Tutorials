### Руководство по внедрению

#### Создание PDF-буклета с использованием потоков

**Обзор**: эта функция демонстрирует, как создать брошюру из PDF-файла, используя указанные левую и правую страницы, используя возможности потоков файлов для эффективной обработки ввода-вывода.

```csharp
using System.IO;
using Aspose.Pdf.Facades;

// Создать объект PdfFileEditor
class Example {
    public void CreateBooklet() {
        PdfFileEditor pdfEditor = new PdfFileEditor();
        
        // Определите путь для входных и выходных PDF-файлов
        string inputFilePath = "YOUR_DOCUMENT_DIRECTORY/MultiplePages.pdf";
        string outputFilePath = "YOUR_OUTPUT_DIRECTORY/MakeBookletUsingLeftRightPagesAndStreams_out.pdf";
        
        // Использование файловых потоков для эффективной обработки больших файлов
        using (FileStream inputStream = new FileStream(inputFilePath, FileMode.Open)) {
            using (FileStream outputStream = new FileStream(outputFilePath, FileMode.Create)) {
                // Определите левую и правую страницы для макета буклета.
                int[] leftPages = { 1, 5 };
                int[] rightPages = { 2, 3 };
                
                // Создайте буклет из PDF-файла, используя указанные левую и правую страницы.
                pdfEditor.MakeBooklet(inputStream, outputStream, leftPages, rightPages);
            }
        }
    }
}
```

**Объяснение**: 
- **Редактор PDF-файлов**: Центральный класс для работы с PDF-файлами.
- **ФайлПоток**: Используется для эффективного чтения и записи больших файлов без их полной загрузки в память.
- **Метод создания буклета**: Принимает входные и выходные потоки, а также массивы страниц, определяющие макет буклета.

## Руководство по внедрению

### Открыть и прочитать PDF-файл
Для извлечения метаданных используйте `PdfFileInfo` сорт:
```csharp
using Aspose.Pdf.Facades;
using System;

namespace PdfInfoExtractor
{
    public class GetFileInfo
    {
        public static void Run()
        {
            string dataDir = "path_to_your_directory/"; // Обновите путь к каталогу

            // Открыть документ с помощью PdfFileInfo
            PdfFileInfo fileInfo = new PdfFileInfo(dataDir + "GetFileInfo.pdf");

            // Извлечение и печать метаданных PDF
            Console.WriteLine("Subject: {0}", fileInfo.Subject);
            Console.WriteLine("Title: {0}", fileInfo.Title);
            Console.WriteLine("Keywords: {0}", fileInfo.Keywords);
            Console.WriteLine("Creator: {0}", fileInfo.Creator);
            Console.WriteLine("Creation Date: {0}", fileInfo.CreationDate);
            Console.WriteLine("Modification Date: {0}", fileInfo.ModDate);

            // Проверьте, является ли файл корректным PDF-файлом и зашифрован ли он.
            Console.WriteLine("Is Valid PDF: {0}", fileInfo.IsPdfFile);
            Console.WriteLine("Is Encrypted: {0}", fileInfo.IsEncrypted);
        }
    }
}
```

#### Ключевые моменты:
- **Параметры**: `PdfFileInfo` Конструктор принимает путь к файлу в качестве параметра.
- **Методы и свойства**: Методы, подобные `Subject`, `Title`и т. д., обеспечивают доступ к метаданным PDF. 
- **Проверка файла**: Проверьте, действителен ли документ, используя `IsPdfFile` и зашифровано ли оно с помощью `IsEncrypted`.

### Советы по устранению неполадок
- Убедитесь, что путь к файлу правильный и доступный.
- При возникновении проблем с шифрованием файлов проверьте разрешения или используйте соответствующие методы дешифрования.

## Руководство по внедрению

### Настройка метаданных XMP

Эта функция позволяет вам эффективно добавлять и управлять метаданными. Вот как реализовать это шаг за шагом:

#### Создание объекта PdfXmpMetadata

Начните с создания экземпляра `PdfXmpMetadata` для обработки метаданных вашего PDF-файла.

```csharp
// Инициализируйте объект PdfXmpMetadata
PdfXmpMetadata xmpMetaData = new PdfXmpMetadata();
```

#### Потоки чтения и письма

Откройте поток для чтения входного PDF-файла и еще один для записи измененного документа с метаданными XMP.

```csharp
// Определите пути к файлам (замените YOUR_DOCUMENT_DIRECTORY соответствующим образом)
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Входной поток PDF-файлов
FileStream input = new FileStream(dataDir + "/FilledForm.pdf", FileMode.Open);

// Выходной поток PDF-файлов для сохранения изменений
FileStream output = new FileStream(dataDir + "/xmp_out.pdf", FileMode.Create);
```

#### Привязка и изменение метаданных

Привяжите PDF к `PdfXmpMetadata` объект, затем добавьте или измените свойства метаданных по мере необходимости.

```csharp
// Свяжите входной PDF-документ
xmpMetaData.BindPdf(input);

// Добавить различные свойства метаданных XMP
xmpMetaData.Add(DefaultMetadataProperties.BaseURL, "xmlns:pdf=http://Ns.adobe.com/pdf/1.3/");
xmpMetaData.Add(DefaultMetadataProperties.CreateDate, System.DateTime.Now.ToString());
xmpMetaData.Add(DefaultMetadataProperties.MetadataDate, System.DateTime.Now.ToString());
xmpMetaData.Add(DefaultMetadataProperties.CreatorTool, "Creator Tool Name");
xmpMetaData.Add(DefaultMetadataProperties.ModifyDate, System.DateTime.Now.ToString());
xmpMetaData.Add(DefaultMetadataProperties.Nickname, "Test");

// Сохраните изменения в выходном PDF-файле.
xmpMetaData.Save(output);
```

#### Закрытие потоков

Всегда закрывайте свои потоки, чтобы освободить ресурсы.

```csharp
input.Close();
output.Close();
```

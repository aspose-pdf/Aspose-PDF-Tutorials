# Как получить свойства страницы PDF с помощью Aspose.PDF для .NET (пошаговое руководство)

## Введение

Работа с PDF-документами в среде .NET часто требует извлечения определенных свойств страницы, таких как поворот, количество страниц и размеры полей. Эти данные необходимы для таких задач, как автоматическое создание отчетов или подготовка файлов к печати. Это руководство покажет вам, как эффективно извлекать эти свойства с помощью Aspose.PDF для .NET.

**Что вы узнаете:**
- Как повернуть страницу PDF-файла.
- Получить общее количество страниц в PDF-файле.
- Извлекайте поля различных размеров (обрезка, художественное оформление, обрезка под обрез, кадрирование, медиа) из страниц PDF.
- Эффективно внедряйте Aspose.PDF для .NET.

Давайте начнем с настройки вашей среды!

## Руководство по внедрению

В этом разделе мы рассмотрим, как использовать различные функции Aspose.PDF для .NET для извлечения определенных свойств из страниц PDF.

### Получить ротацию страниц

**Обзор**: Определите ориентацию страницы PDF-файла, используя значения поворота (0, 90, 180 или 270 градусов).

#### Пошаговая реализация

1. **Инициализировать PdfPageEditor**:
   ```csharp
   PdfPageEditor pageEditor = new PdfPageEditor();
   ```

2. **Связать PDF-документ**:
   ```csharp
   pageEditor.BindPdf(dataDir + "/input.pdf");
   ```

3. **Получить поворот страницы**:
   ```csharp
   int rotation = pageEditor.GetPageRotation(1);
   Console.WriteLine($"Rotation of Page 1: {rotation} degrees");
   ```
   - `GetPageRotation(int pageNumber)`Выбирает поворот в градусах для указанной страницы.

### Получить количество страниц

**Обзор**: Получите общее количество страниц в вашем PDF-документе.

#### Пошаговая реализация

1. **Связать PDF-документ**:
   ```csharp
   pageEditor.BindPdf(dataDir + "/input.pdf");
   ```

2. **Получить общее количество страниц**:
   ```csharp
   int pageCount = pageEditor.GetPages();
   Console.WriteLine($"Total Pages: {pageCount}");
   ```
   - `GetPages()`: Возвращает общее количество страниц в документе.

### Получить размеры страниц

#### Размер коробки обрезки

**Обзор**: Извлекает размеры обрезки для определенной страницы PDF.

1. **Получить размер коробки обрезки**:
   ```csharp
   SizeF trimBoxSize = pageEditor.GetPageBoxSize(1, "trim");
   Console.WriteLine($"Trim Box Size: {trimBoxSize}");
   ```

#### Размер художественной коробки

1. **Получить размер коробки для произведений искусства**:
   ```csharp
   SizeF artBoxSize = pageEditor.GetPageBoxSize(1, "art");
   Console.WriteLine($"Art Box Size: {artBoxSize}");
   ```

#### Размер выпускного короба

1. **Получить размер коробки для кровотечения**:
   ```csharp
   SizeF bleedBoxSize = pageEditor.GetPageBoxSize(1, "bleed");
   Console.WriteLine($"Bleed Box Size: {bleedBoxSize}");
   ```

#### Размер поля обрезки

1. **Получить размер поля обрезки**:
   ```csharp
   SizeF cropBoxSize = pageEditor.GetPageBoxSize(1, "crop");
   Console.WriteLine($"Crop Box Size: {cropBoxSize}");
   ```

#### Размер медиа-бокса

1. **Получить размер медиа-бокса**:
   ```csharp
   SizeF mediaBoxSize = pageEditor.GetPageBoxSize(1, "media");
   Console.WriteLine($"Media Box Size: {mediaBoxSize}");
   ```
   - `GetPageBoxSize(int pageNumber, string boxType)`: Извлекает размеры указанного типа блока для заданной страницы.

## Руководство по внедрению

### Извлечение и отображение пользовательских свойств PDF-файла

#### Обзор
Доступ к пользовательским свойствам, встроенным в PDF-файл, полезен для извлечения метаданных, необходимых для индексации или категоризации.

##### Шаг 1: Импорт необходимых библиотек
```csharp
using Aspose.Pdf.Facades;
```

##### Шаг 2: Инициализация PdfFileInfo
Укажите каталог, в котором находится ваш PDF-файл:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/inFile1.pdf";
PdfFileInfo fInfo = new PdfFileInfo(dataDir);
```

##### Шаг 3: Извлечение пользовательских свойств
Доступ к пользовательским свойствам и их отображение с помощью хэш-таблицы:
```csharp
var hTable = new System.Collections.Hashtable(fInfo.Header);
foreach (DictionaryEntry entry in hTable)
{
    Console.WriteLine($"Key: {entry.Key}, Value: {entry.Value}"); // Отображение пользовательского свойства
}
```

### Установка и извлечение пользовательских метасвойств в PDF

#### Обзор
Установка и извлечение метасвойств имеет решающее значение для динамического обновления метаданных документа.

##### Шаг 1: Инициализация PdfFileInfo
Используйте ту же инициализацию, что и раньше:
```csharp
PdfFileInfo fInfo = new PdfFileInfo(dataDir);
```

##### Шаг 2: Установка пользовательского метасвойства
Добавьте или обновите пользовательское свойство в вашем PDF-файле:
```csharp
fInfo.SetMetaInfo("CustomAttribute", "test value");
```

##### Шаг 3: Извлечение пользовательского метасвойства
Извлеките новое заданное свойство, чтобы проверить его существование:
```csharp
string value = fInfo.GetMetaInfo("CustomAttribute");
Console.WriteLine($"Retrieved Value: {value}"); // Выход: тестовое значение
```

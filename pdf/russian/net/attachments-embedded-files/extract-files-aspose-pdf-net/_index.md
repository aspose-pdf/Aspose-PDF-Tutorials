## Руководство по внедрению
Теперь, когда у нас все настроено, давайте извлечем файлы из PDF-портфолио с помощью Aspose.PDF.

### Загрузить исходный PDF-портфолио (H2)
Сначала загрузите PDF-документ, содержащий встроенные файлы:
```csharp
// Определите путь к каталогу документов.
string dataDir = RunExamples.GetDataDir_AsposePdf_TechnicalArticles();

// Загрузить исходный PDF-файл портфолио
Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document(dataDir + "PDFPortfolio.pdf");
```

### Доступ к встроенным файлам (H2)
Далее, получите доступ и выполните итерацию по встроенным файлам:
```csharp
// Получить коллекцию встроенных файлов
EmbeddedFileCollection embeddedFiles = pdfDocument.EmbeddedFiles;

// Итерация по отдельным файлам портфолио
foreach (FileSpecification fileSpecification in embeddedFiles)
{
    // Извлечь содержимое и сохранить в файл или поток
    byte[] fileContent = new byte[fileSpecification.Contents.Length];
    fileSpecification.Contents.Read(fileContent, 0, fileContent.Length);
    
    string filename = Path.GetFileName(fileSpecification.Name);

    // Сохраните извлеченный файл в каком-либо месте.
    using (FileStream fileStream = new FileStream(dataDir + "_out" + filename, FileMode.Create))
    {
        fileStream.Write(fileContent, 0, fileContent.Length);
    }
}
```
#### Объяснение
- **Встроенная коллекция файлов:** Предоставляет доступ ко всем встроенным файлам в PDF-файле.
- **Спецификация файла:** Представляет каждый файл в коллекции. Его `Contents` свойство позволяет читать и извлекать данные.

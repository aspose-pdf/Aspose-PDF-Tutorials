# Как задать размеры страницы PDF и преобразовать в HTML с помощью Aspose.PDF .NET

## Введение

Вам когда-нибудь нужно было преобразовать PDF-документ в формат HTML, при этом гарантируя, что размеры страницы будут идеально подобраны? Это руководство призвано решить именно эту проблему, демонстрируя, как использовать **Aspose.PDF для .NET** для установки пользовательских размеров для страниц PDF и их бесшовного преобразования в HTML. Aspose.PDF предоставляет надежные функции, позволяющие разработчикам эффективно манипулировать документами PDF в среде .NET.

Из этого руководства вы узнаете, как:
- Установите новые размеры для страниц PDF-файла
- Конвертируйте измененный PDF-файл в формат HTML
- Настройте параметры преобразования, такие как границы страниц.

Давайте перейдем непосредственно к настройке Aspose.PDF и реализации этих функций!

## Руководство по внедрению

Давайте разберем реализацию по ключевым функциям:

### Функция 1: Установка размеров выходного файла

#### Обзор
Эта функция позволяет вам настраивать размеры страниц PDF перед их конвертацией в HTML. Она гарантирует, что содержимое идеально впишется в новые размеры страниц, вычисляя соответствующий уровень масштабирования.

#### Пошаговая реализация

**Инициализировать и привязать PDF-документ**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Facades;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Укажите путь к каталогу документов
PdfPageEditor pdfEditor = new PdfPageEditor();
pdfEditor.BindPdf(dataDir + "/input.pdf"); // Свяжите входной PDF-файл
```

**Установить новые размеры страницы**
```csharp
// Выберите желаемый размер страницы
float newPageWidth = 400f;
float newPageHeight = 400f;

// Установите новые размеры и выравнивание
pdfEditor.PageSize = new PageSize(newPageWidth, newPageHeight);
pdfEditor.VerticalAlignmentType = VerticalAlignment.Center;
pdfEditor.HorizontalAlignment = HorizontalAlignment.Center;
```

**Рассчитать коэффициент масштабирования**
```csharp
// Рассчитать масштаб, чтобы пропорционально разместить содержимое в новом размере страницы
float zoom = Math.Min((float)newPageWidth / (float)pdfEditor.Document.Pages[1].Rect.Width,
                      (float)newPageHeight / (float)pdfEditor.Document.Pages[1].Rect.Height);
pdfEditor.Zoom = zoom; // Применить расчетный зум
```

**Сохранить для трансляции и конвертации**
```csharp
// Сохраните обновленный PDF-файл с новыми размерами в потоке памяти.
MemoryStream output = new MemoryStream();
pdfEditor.Save(output);

// Перезагрузите масштабированный документ для преобразования в HTML.
Document exportDoc = new Document(output);
HtmlSaveOptions htmlOptions = new HtmlSaveOptions();

// Настройте границы страницы в полученном HTML-файле
SaveOptions.BorderPartStyle borderStyle = new SaveOptions.BorderPartStyle()
{
    LineType = SaveOptions.HtmlBorderLineType.Dotted,
    Color = System.Drawing.Color.Gray
};
htmlOptions.PageBorderIfAny = new SaveOptions.BorderInfo(borderStyle);

// Конвертируйте и сохраните PDF в формат HTML
exportDoc.Save("YOUR_OUTPUT_DIRECTORY/SetOutputFileDimensions_out.html", htmlOptions);
output.Close(); // Закрыть поток памяти
```

### Функция 2: Конфигурация преобразования

#### Обзор
Эта функция фокусируется на настройке процесса преобразования из PDF в HTML, включая настройку границ страниц для лучшей видимости.

**Настроить и преобразовать**
```csharp
// Инициализируйте HtmlSaveOptions для конфигурации
HtmlSaveOptions htmlOptions = new HtmlSaveOptions();

// Настройте видимые границы страницы в результирующем HTML-файле
SaveOptions.BorderPartStyle borderStyle = new SaveOptions.BorderPartStyle()
{
    LineType = SaveOptions.HtmlBorderLineType.Dotted,
    Color = System.Drawing.Color.Gray
};
htmlOptions.PageBorderIfAny = new SaveOptions.BorderInfo(borderStyle);

// Предположим, что создан объект Document (например, из PDF).
Document exportDoc = new Document(dataDir + "/input.pdf");

// Конвертируйте и сохраните документ как HTML с настроенными параметрами
exportDoc.Save("YOUR_OUTPUT_DIRECTORY/ConvertedWithBorders_out.html", htmlOptions);
```

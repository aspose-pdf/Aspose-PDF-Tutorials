## Руководство по внедрению
Теперь, когда вы настроили свою среду, давайте приступим к конвертации Markdown в PDF с помощью Aspose.PDF для .NET. Мы разобьем это на простые шаги.

### Шаг 1: Загрузите документ Markdown
Начните с загрузки вашего файла Markdown. Для этого необходимо указать путь и использовать `MdLoadOptions`.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MdLoadOptions loadOptions = new MdLoadOptions();
Document doc = new Document(dataDir + "/sample.md", loadOptions);
```

**Объяснение:**
- `dataDir` ваш каталог, содержащий файлы Markdown.
- `MdLoadOptions` инициализирует параметры загрузки специально для Markdown.

### Шаг 2: Сохраните документ как PDF-файл.
После загрузки сохраните документ в формате PDF:

```csharp
doc.Save("YOUR_OUTPUT_DIRECTORY/MarkdownToPDF.pdf");
```

**Объяснение:**
- На этом этапе содержимое Markdown преобразуется и сохраняется в виде файла PDF в указанном выходном каталоге.

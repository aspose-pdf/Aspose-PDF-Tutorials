## Руководство по внедрению

### Функция: конвертация EPUB в PDF

Конвертация файла EPUB в PDF может быть выполнена всего за несколько шагов. Мы разберем каждый шаг, объяснив его цель и реализацию.

#### Шаг 1: Укажите параметры загрузки для EPUB
Чтобы начать процесс конвертации, определите параметры загрузки, адаптированные для файлов EPUB, используя `EpubLoadOptions`.

```csharp
// Создайте экземпляр EpubLoadOptions для указания параметров загрузки.
EpubLoadOptions epubload = new EpubLoadOptions();
```

#### Шаг 2: Загрузите файл EPUB
Используя Aspose.PDF, создайте объект Document, который загружает ваш файл EPUB. `Document` класс обрабатывает процесс преобразования.

```csharp
// Загрузите файл EPUB с указанными параметрами в Aspose.Pdf.Document.
Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document(dataDir + "/EPUBToPDF.epub", epubload);
```

#### Шаг 3: Сохранить как PDF
Наконец, сохраните загруженный документ в формате PDF в указанном вами выходном каталоге.

```csharp
// Конвертируйте и сохраните файл EPUB как PDF.
pdfDocument.Save(outputDir + "/EPUBToPDF_out.pdf");
```

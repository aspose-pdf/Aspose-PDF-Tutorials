## Руководство по внедрению

### Преобразование PDF в HTML с отчетом о ходе работ

В этом разделе вы узнаете, как преобразовать PDF-документ в формат HTML, а также получите информацию о ходе выполнения в режиме реального времени.

#### Обзор

Используя настраиваемые обработчики событий, вы можете эффективно преобразовывать содержимое PDF в HTML и контролировать процесс преобразования на каждом существенном этапе.

#### Пошаговая реализация

**1. Инициализация объекта документа**

Загрузите исходный PDF-документ:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

Этот код инициализирует `Document` объект, представляющий PDF-файл, который вы хотите преобразовать.

**2. Настройте параметры сохранения HTML**

Создать экземпляр `HtmlSaveOptions` и настройте его:

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions();
saveOptions.SplitIntoPages = false; // Сохраняет содержимое в одном HTML-файле
```

Параметр `SplitInкPages` to `false` обеспечивает преобразование всего PDF-файла в один HTML-файл.

**3. Установите пользовательский обработчик прогресса**

Назначьте пользовательский обработчик прогресса для отслеживания статуса конверсии:

```csharp
saveOptions.CustomProgressHandler = new HtmlSaveOptions.ConversionProgessEventHandler(ShowProgressOnConsole);
```

Это соединяет `CustomProgressHandler` с нашим методом `ShowProgressOnConsole`, регистрация событий прогресса.

**4. Сохранить документ как HTML**

Выполните преобразование и сохраните результат:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputDir + "/ProgressDetails_out_.html", saveOptions);
```

#### Обработчик событий прогресса

Реализуйте обработчик событий прогресса для регистрации подробных обновлений статуса:

```csharp
class ProgressEventHandler
{
    public static void ShowProgressOnConsole(HtmlSaveOptions.ProgressEventHandlerInfo eventInfo)
    {
        switch (eventInfo.EventType)
        {
            case ProgressEventType.TotalProgress:
                Console.WriteLine($"{DateTime.Now.TimeOfDay} - Conversion progress: {eventInfo.Value}%");
                break;

            case ProgressEventType.SourcePageAnalysed:
                Console.WriteLine($"{DateTime.Now.TimeOfDay} - Source page {eventInfo.Value} of {eventInfo.MaxValue} analyzed.");
                break;

            case ProgressEventType.ResultPageCreated:
                Console.WriteLine($"{DateTime.Now.TimeOfDay} - Result page's {eventInfo.Value} of {eventInfo.MaxValue} layout created.");
                break;

            case ProgressEventType.ResultPageSaved:
                Console.WriteLine($"{DateTime.Now.TimeOfDay} - Result page {eventInfo.Value} of {eventInfo.MaxValue} exported.");
                break;
        }
    }
}
```

Этот метод обрабатывает различные события во время преобразования, предоставляя информацию о каждом этапе.

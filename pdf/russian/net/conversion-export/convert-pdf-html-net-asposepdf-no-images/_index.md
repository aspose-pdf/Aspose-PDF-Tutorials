## Руководство по внедрению

Теперь давайте разобьем процесс внедрения на управляемые этапы.

### Настройка параметров сохранения HTML

**Обзор**
Мы настроим параметры для преобразования PDF-файлов в HTML, сохраняя изображения, встроенные в фон страницы. Такой подход гарантирует, что ваши преобразованные документы сохранят свой исходный макет без отдельного экспорта изображений.

#### Шаг 1: Инициализация документа и сохранение параметров
Настройте экземпляр документа и укажите параметры сохранения для преобразования:

```csharp
// Путь к входному PDF-файлу.
string dataDir = "YourDirectoryPath";

Document pdfDocument = new Document(dataDir + "/input.pdf");
HtmlSaveOptions saveOptions = new HtmlSaveOptions();
```

#### Шаг 2: Установите макет и режим сохранения изображения

Настройте `HtmlSaveOptions` для сохранения фиксированного макета и встраивания изображений:

```csharp
saveOptions.FixedLayout = true;
// Изображения сохраняются в виде встроенных PNG-файлов на фоне страницы.
saveOptions.RasterImagesSavingMode = HtmlSaveOptions.RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground;
```

#### Шаг 3: Сохраните документ
Наконец, сохраните ваш PDF-документ в формате HTML, используя следующие настройки:

```csharp
pdfDocument.Save(dataDir + "/AvoidSavingImages_out.html", saveOptions);
```

**Совет по устранению неполадок**: Убедитесь, что путь к входному файлу правильный и доступный, чтобы избежать исключений во время загрузки.

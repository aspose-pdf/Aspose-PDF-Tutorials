## Руководство по внедрению

### Замена шрифтов в PDF-файлах

Замена шрифта имеет решающее значение для сохранения внешнего вида документа, когда исходный шрифт недоступен. Давайте реализуем эту функцию с помощью Aspose.PDF .NET.

#### Обзор
В этом разделе основное внимание уделяется замене отсутствующих шрифтов указанной альтернативой для сохранения форматирования текста и визуальной согласованности.

#### Пошаговая реализация

**1. Настройка замены шрифтов**

Сначала определите правило подстановки для вашего документа:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

// Путь к каталогу для хранения документов
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; 

// Создайте объект SimpleFontSubstitution для замены «AgencyFB» на «Arial»
var substituteFont = new SimpleFontSubstitution("AgencyFB", "Arial");

try
{
    // Попытайтесь найти оригинальный шрифт в репозитории.
    FontRepository.FindFont("AgencyFB");
} catch (Exception)
{
    // Если не найдено, добавьте правило замены
    FontRepository.Substitutions.Add(substituteFont);
}
```

**Объяснение:**
- `SimpleFontSubstitution` сопоставляет недоступный шрифт («AgencyFB») с резервным вариантом («Arial»).
- Код проверяет, существует ли "AgencyFB". Если нет, блок исключений добавляет правило подстановки.

### Преобразование и сохранение PDF-файлов

Преобразование PDF-файлов в стандартизированные форматы, такие как PDF/A, обеспечивает совместимость и соответствие архивным стандартам.

#### Обзор
В этом разделе вы узнаете, как преобразовать и сохранить PDF-документ в другом формате с помощью Aspose.PDF .NET.

**2. Конвертация и сохранение PDF-документов**

Вот как преобразовать стандартный PDF-файл в формат PDF/A-1b, который широко используется для долгосрочного архивирования:

```csharp
using System.IO;
using Aspose.Pdf;

// Определите каталоги и пути к файлам
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; 
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "newfile_out.pdf");

// Загрузите входной PDF-документ
var pdfDocument = new Document(Path.Combine(dataDir, "input.pdf"));

// Конвертировать в формат PDF/A-1b и сохранить результат
pdfDocument.Convert(Path.Combine(dataDir, "log.xml"), PdfFormat.PDF_A_1B, ConvertErrorAction.Delete);
pdfDocument.Save(outputFile);
```

**Объяснение:**
- `Convert` метод используется для преобразования документа в указанный формат (PDF/A-1b).
- Параметры включают путь к файлу журнала и действия по обработке ошибок (`ConvertErrorAction.Delete`).

### Советы по устранению неполадок

- Убедитесь, что указанные файлы шрифтов существуют в каталоге вашей системы или приложения.
- Проверьте пути к файлам для каталогов ввода/вывода, чтобы предотвратить ошибки во время выполнения.

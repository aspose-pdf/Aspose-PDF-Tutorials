## Руководство по внедрению

### Функция 1: Преобразование HTML в PDF с помощью пользовательских ресурсов

Эта функция позволяет вам настраивать способ загрузки внешних ресурсов во время конвертации. Это особенно полезно при работе с определенными изображениями или активами, требующими специальной обработки.

#### Обзор
Целью здесь является загрузка изображения в виде массива байтов и использование его в результирующем PDF-файле, переопределяя механизм загрузки ресурсов Aspose.PDF по умолчанию.

**Шаг 1: Инициализация HtmlLoadOptions**

Мы начинаем с создания `HtmlLoadOptions` и определение пользовательского загрузчика для внешних ресурсов.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// Стратегия загрузки пользовательских ресурсов
HtmlLoadOptions options = new HtmlLoadOptions();
options.CustomLoaderOfExternalResources = new LoadOptions.ResourceLoadingStrategy(SamePictureLoader);
```

**Шаг 2: Создание PDF-документа из HTML**

Установив параметры, мы загружаем наш HTML-файл и конвертируем его в PDF-документ.

```csharp
Document pdfDocument = new Document(dataDir + "/HTMLToPDF.html", options);
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/HTMLToPDF_out.pdf");
```

**Шаг 3: Определите пользовательский загрузчик ресурсов**

The `SamePictureLoader` Функция определяет, как загружаются внешние ресурсы. Здесь мы загружаем определенный файл изображения как байты.

```csharp
private static LoadOptions.ResourceLoadingResult SamePictureLoader(string resourceURI)
{
    byte[] resultBytes = File.ReadAllBytes(dataDir + "/aspose-logo.jpg");
    return new LoadOptions.ResourceLoadingResult(resultBytes);
}
```

### Функция 2: преобразование HTML-контента в одностраничный PDF-файл

Преобразование многостраничного HTML-контента в одностраничный может иметь важное значение для создания кратких отчетов или сводок.

#### Обзор
Мы включим `IsRenderToSinglePage` свойство в наших опциях, чтобы гарантировать, что весь документ уместится на одной странице.

**Шаг 1: Настройка HtmlLoadOptions**

Включите одностраничный рендеринг, настроив `HtmlLoadOptions`.

```csharp
HtmlLoadOptions options = new HtmlLoadOptions();
options.IsRenderToSinglePage = true;
```

**Шаг 2: Загрузите и сохраните HTML как одностраничный PDF-файл**

Теперь загружаем документ с этими параметрами и сохраняем его.

```csharp
Document doc = new Document(dataDir + "/HTMLToPDF.html", options);
doc.Save("YOUR_OUTPUT_DIRECTORY/RenderContentToSamePage.pdf");
```

### Функция 3: Отображение HTML с данными SVG в PDF

Обработка данных SVG в HTML может быть сложной задачей, но Aspose.PDF упрощает ее.

#### Обзор
Эта функция предназначена для преобразования HTML-файлов, содержащих встроенные изображения SVG, в PDF-документ.

**Шаг 1: Укажите пути ввода и вывода**

Определите пути для входного HTML-файла и выходного PDF-файла.

```csharp
string inFile = dataDir + "/HTMLSVG.html";
string outFile = "YOUR_OUTPUT_DIRECTORY/RenderHTMLwithSVGData.pdf";
```

**Шаг 2: Инициализация параметров с путем к каталогу**

Настраивать `HtmlLoadOptions` с путем к каталогу HTML-файла, чтобы гарантировать правильное разрешение SVG.

```csharp
HtmlLoadOptions options = new HtmlLoadOptions(Path.GetDirectoryName(inFile));
Document pdfDocument = new Document(inFile, options);
pdfDocument.Save(outFile);
```

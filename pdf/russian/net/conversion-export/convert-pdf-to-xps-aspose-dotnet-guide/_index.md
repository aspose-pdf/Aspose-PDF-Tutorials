## Руководство по внедрению
В этом разделе описывается преобразование PDF-документа в XPS-файл с помощью Aspose.PDF для .NET.

### Конвертировать PDF в XPS
Преобразуйте существующие файлы PDF в формат XPS без особых усилий:

#### Шаг 1: Загрузите PDF-документ
Загрузите PDF-файл, требующий преобразования:
```csharp
using System;
using Aspose.Pdf;

// Укажите входной каталог для вашего PDF-файла.
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";

document = new Document(inputDirectory + "/input.pdf");
```

#### Шаг 2: Настройте параметры сохранения XPS
Настройте параметры, необходимые для сохранения в формате XPS:
```csharp
// Параметры сохранения экземпляра XPS
var saveOptions = new Aspose.Pdf.XpsSaveOptions();
```

#### Шаг 3: Сохранить как документ XPS
Конвертируйте и сохраните ваш PDF-файл в файл XPS:
```csharp
// Определите выходной каталог для преобразованного XPS-файла.
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

document.Save(outputDirectory + "/PDFToXPS_out.xps", saveOptions);
```
**Объяснение**: `XpsSaveOptions` класс позволяет выполнять настройку, специфичную для преобразования XPS.

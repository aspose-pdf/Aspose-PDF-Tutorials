### Функция: Преобразование PDF в XML (H2)
Эта функция фокусируется на преобразовании PDF-документа в формат XML с использованием Aspose.PDF Facades.

#### Обзор
Преобразование PDF в XML позволяет улучшить обработку данных и интеграцию с другими системами. В этом разделе вы узнаете, как настроить потоки файлов и выполнить преобразование.

##### Шаг 1: Создание потоков файлов (H3)
Вам понадобятся два файловых потока: один для чтения входного PDF-файла и другой для записи выходного XML-файла.
```csharp
using System.IO;

FileStream inputStream = new FileStream("YOUR_DOCUMENT_DIRECTORY\inFile.pdf", FileMode.Open);
FileStream xmlOutputStream = new FileStream("YOUR_OUTPUT_DIRECTORY\ConvertPdfToXML_out.xml", FileMode.Create);
```
**Объяснение:**
- `inputStream`: Читает ваш PDF-файл.
- `xmlOutputStream`: Записывает преобразованное XML-содержимое.

##### Шаг 2: Преобразование PDF в XML (H3)
Используйте `FormDataConverter` класс для преобразования:
```csharp
using Aspose.Pdf.Facades;

// Выполнить преобразование из PDF (FDF) в XML.
FormDataConverter.ConvertFdfToXml(inputStream, xmlOutputStream);
```
**Объяснение:**
- Метод эффективно преобразует ваш PDF-документ в формат XML и сохраняет его в указанном выходном потоке.

The `FormDataConverter` класс имеет решающее значение в преобразовании PDF-документов в XML. Это пошаговое руководство покажет вам, как использовать его эффективно.

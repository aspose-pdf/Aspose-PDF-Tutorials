## Руководство по внедрению

### Функция 1: Преобразование PDF в HTML с внедрением ресурсов

Эта функция позволяет преобразовать PDF-документ в формат HTML, встраивая все ресурсы в выходной HTML-файл.

#### Обзор:
Преобразование гарантирует, что все шрифты, изображения и стили будут инкапсулированы в HTML, обеспечивая автономный и визуально единообразный интерфейс в различных браузерах.

##### Шаг 1: Загрузите исходный PDF-файл

```csharp
using System;
using Aspose.Pdf;

string documentPath = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document doc = new Document(documentPath);
```
Здесь мы загружаем исходный PDF-файл. Убедитесь, что путь к файлу указан правильно, чтобы избежать ошибок загрузки.

##### Шаг 2: Настройте параметры сохранения HTML

```csharp
using Aspose.Pdf.Saving;

HtmlSaveOptions options = new HtmlSaveOptions();
options.PartsEmbeddingMode = PartsEmbeddingModes.EmbedAllIntoHtml;
```
Мы создаем экземпляр `HtmlSaveOptions` и настроить его для встраивания всех ресурсов в HTML-файл.

##### Шаг 3: Оптимизация для Internet Explorer

```csharp
options.LettersPositioningMethod = LettersPositioningMethods.UseEmUnitsAndCompensationOfRoundingErrorsInCss;
```
Эта настройка обеспечивает корректное отображение текста в IE за счет использования единиц измерения em и компенсации ошибок округления CSS.

##### Шаг 4: Сохранение растровых изображений как встроенных PNG-файлов

```csharp
options.RasterImagesSavingMode = RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground;
```
Отображает изображения как встроенные части фона страницы, обеспечивая визуальную точность на всех платформах.

##### Шаг 5: Настройте режим сохранения шрифтов

```csharp
options.FontSavingMode = FontSavingModes.SaveInAllFormats;
```
Эта опция сохраняет шрифты во всех форматах для максимальной совместимости и сохранения качества.

##### Шаг 6: Сохраните преобразованный HTML-файл.

```csharp
string outputHtmlPath = "YOUR_OUTPUT_DIRECTORY/SingleHTML_out.html";
doc.Save(outputHtmlPath, options);
```
Наконец, сохраните преобразованный HTML-файл в указанном вами каталоге. Убедитесь, что ваш выходной путь указан правильно, чтобы избежать ошибок.

**Что вы узнаете:**
- Эффективное преобразование файлов PDF в формат HTML.
- Настройте URL-адреса изображений во время конвертации с помощью Aspose.PDF для .NET.

Прежде чем начать, давайте настроим вашу среду и рассмотрим предварительные условия!

## Руководство по внедрению

### Преобразование PDF в HTML с использованием пользовательских URL-адресов изображений

Преобразование документа PDF в HTML с одновременным контролем URL-адресов изображений требует понимания возможностей Aspose.PDF и настройки соответствующих параметров. Мы разберем это пошагово.

#### Шаг 1: Загрузите документ
Сначала загрузите ваш PDF-документ с помощью `Document` сорт:

```csharp
// Загрузить существующий PDF-документ
document pdfDocument = new Document("input.pdf");
```

#### Шаг 2: Настройте параметры сохранения
Настраивать `HtmlSaveOptions` чтобы указать, как изображения обрабатываются во время конвертации. Ключевым моментом здесь является настройка `RasterImagesSavingMode` и определение индивидуальной стратегии экономии ресурсов:

```csharp
// Настройте параметры сохранения HTML
HtmlSaveOptions options = new HtmlSaveOptions();

// Определить режим сохранения изображения
options.RasterImagesSavingMode = HtmlSaveOptions.RasterImagesSavingModes.AsExternalPngFilesReferencedViaSvg;

// Установить индивидуальную стратегию экономии ресурсов
customResourceSavingStrategy: new HtmlSaveOptions.ResourceSavingStrategy(Custom_processor_of_embedded_images);
```

#### Шаг 3: Внедрите индивидуальную стратегию экономии ресурсов
Здесь вы можете настроить способ сохранения и ссылки на изображения:

```csharp
private static string Custom_processor_of_embedded_images(SaveOptions.ResourceSavingInfo resourceSavingInfo)
{
    if (!(resourceSavingInfo is HtmlSaveOptions.HtmlImageSavingInfo))
    {
        resourceSavingInfo.CustomProcessingCancelled = true;
        return "";
    }

    // Определить выходной каталог для изображений
    string dataDir = "your/data/directory/path";
    string outDir = Path.Combine(dataDir, @"output\files");
    string outPath = Path.Combine(outDir, Path.GetFileName(resourceSavingInfo.SupposedFileName));

    if (!Directory.Exists(outDir))
    {
        Directory.CreateDirectory(outDir);
    }

    // Сохранить изображение
    using (BinaryReader reader = new BinaryReader(resourceSavingInfo.ContentStream))
    {
        System.IO.File.WriteAllBytes(outPath, reader.ReadBytes((int)resourceSavingInfo.ContentStream.Length));
    }

    // Возвращает пользовательский URL для ссылки на изображение в HTML/SVG
    HtmlSaveOptions.HtmlImageSavingInfo imgInfo = (HtmlSaveOptions.HtmlImageSavingInfo)resourceSavingInfo;
    
    if (imgInfo.ParentType == HtmlSaveOptions.ImageParentTypes.SvgImage)
    {
        return "http://localhost:255/" + resourceSavingInfo.ПредполагаемоеИмяФайла;
    }
    else
    {
        return "http://localhost:GetImage/imageID=" + resourceSavingInfo.SupposedFileName;
    }
}
```

Эта функция определяет, как сохраняются и ссылаются на изображения, позволяя вам указывать пользовательские URL-адреса.

#### Шаг 4: Выполнение преобразования
Наконец, сохраните документ в формате HTML, используя настроенные вами параметры:

```csharp
// Сохранить PDF как HTML
document.Save("output.html", options);
```

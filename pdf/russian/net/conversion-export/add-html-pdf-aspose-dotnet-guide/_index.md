## Руководство по внедрению

Подготовив среду, давайте рассмотрим, как реализовать добавление HTML с использованием DOM и точного размещения на страницах PDF.

### Функция 1: добавление HTML с использованием DOM
Эта функция позволяет включать в PDF-документы расширенный HTML-контент.

#### Обзор
Добавьте динамический HTML для создания визуально привлекательных PDF-файлов с форматированным текстом, изображениями или стилизованными таблицами.

#### Этапы внедрения

**Шаг 1: Инициализация документа**
```csharp
// Создать новый объект документа
Document doc = new Document();
```

**Шаг 2: Добавьте страницу**
```csharp
// Добавить страницу в коллекцию страниц документа
Page page = doc.Pages.Add();
```

**Шаг 3: Создайте HTML-фрагмент**
```csharp
// Инициализируйте HtmlFragment желаемым HTML-содержимым.
HtmlFragment title = new HtmlFragment("<fontsize=10><b><i>Table</i></b></fontsize>");
```

**Шаг 4: Настройка полей**
```csharp
// Установите поля для управления позиционированием
title.Margin.Bottom = 10;
title.Margin.Top = 200;
```

**Шаг 5: Добавьте HTML-фрагмент на страницу**
```csharp
// Вставьте фрагмент HTML в коллекцию абзацев страницы.
page.Paragraphs.Add(title);
```

**Шаг 6: Сохраните документ.**
```csharp
// Определите выходной путь и сохраните документ
string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/AddHTMLUsingDOM_out.pdf";
doc.Save(dataDir);
```

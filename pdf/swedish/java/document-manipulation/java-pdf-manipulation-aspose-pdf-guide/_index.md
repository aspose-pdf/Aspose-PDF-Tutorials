Ersätt `YOUR_DOCUMENT_DIRECTORY` med din faktiska mappsökväg.\n\n#### Steg 2: Ladda PDF-filen\n\nSkapa en `Document` objekt och ladda din PDF-fil:\n\n```java\ndocument = new Document(dataDir + \"/Field_Added_output.pdf\");\n```\n\n### Åtkomst till ett specifikt fält i ett PDF-formulär\n\n**Översikt:** Få åtkomst till specifika fält i ett formulär för att extrahera eller manipulera data.\n\n#### Steg 1: Hämta fältet efter namn\n\nAnvänd `getForm()` metod för att komma åt formuläret och hämta fältet:\n\n```java\nimport com.aspose.pdf.TextBoxField;\n\nTextBoxField textBoxField = (TextBoxField) pdfDocument.getForm().get(\"textbox1\");\n```\n\nHär, `"textbox1"` är namnet på det fält du vill komma åt.\n\n### Hämta fältinformation från ett PDF-dokument\n\n**Översikt:** Extrahera detaljerad information såsom delvisa namn och värden från fält.\n\n#### Steg 2: Hämta fältinformation\n\nHämta det delvisa namnet och värdet för det angivna fältet:\n\n```java\nString fieldName = textBoxField.getPartialName();\nString fieldValue = textBoxField.getValue();\n\nSystem.out.println(\"Field Name: \" + fieldName);\nSystem.out.println(\"Field Value: \" + fieldValue);\n```\n\n##

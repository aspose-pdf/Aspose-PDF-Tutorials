# Skapa en PDF med öppen grafik med hjälp av Aspose.PDF  

Aspose.PDF för .NET gör det enkelt att generera och manipulera PDF-dokument programmatiskt. I detta exempel visar vi hur man skapar ett PDF-dokument och lägger till en graf som innehåller en rektangel med en transparent fyllfärg. Du kommer att lära dig att ställa in färger, lägga till sidor och rita former med exakt kontroll över transparens. Denna handledning ger en steg för steg guide, inklusive kodsnitter och förklaringar för att hjälpa dig integrera Aspose.PDF i dina .NET-applikationer.

## Installationsinstruktioner:  

För att använda Aspose.PDF för .NET, följ dessa steg:  

1. Installa via NuGet Package Manager  
* Öppna Visual Studio och skapa ett nytt .NET-projekt.  
* Öppna paketet Manager Console (verktyg → NuGet Paket Manager → Paket Manager Console) och kör:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Installa via .NET CLI  
Alternativt installerar du paketet med .NET-CLI:  
```bash
dotnet add package Aspose.PDF
```

3. Lägg till Namespace i C#-koden  
För att börja använda Aspose.PDF lägg till följande namnrum i C#-filen:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

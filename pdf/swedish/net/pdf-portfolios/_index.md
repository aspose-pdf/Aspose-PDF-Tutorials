# PDF-portfölj och pakethandledningar för Aspose.PDF .NET

Bemästra skapandet av PDF-portföljer med våra detaljerade Aspose.PDF-handledningar för .NET-utvecklare. Dessa omfattande guider visar hur man skapar PDF-portföljer som innehåller flera filer, anpassar portföljlayouter och navigering, extraherar och manipulerar inbäddade dokument och implementerar avancerade portföljfunktioner programmatiskt. Varje handledning ger fungerande C#-kodexempel för att bygga sofistikerade dokumentpaket som kombinerar flera filer till en enda, navigerbar PDF-behållare, perfekt för att distribuera relaterade dokument som en sammanhängande enhet.

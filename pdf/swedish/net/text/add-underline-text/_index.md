# Lägga till understruken text i PDF med Aspose.PDF för .NET

Aspose.PDF för .NET är ett kraftfullt PDF-behandlingsbibliotek som låter utvecklare skapa, modifiera och manipulera PDF-dokument utan att behöva Adobe Acrobat. Det här exemplet visar hur man skapar ett nytt PDF-dokument och lägger till text med understrykningsformatering med hjälp av klasserna TextBuilder och TextFragment. Koden skapar en enkel PDF med ett testmeddelande formaterat i Arial-teckensnittet med understrykningsformatering, och placerar det vid specifika koordinater på sidan.

## Installationsanvisningar

1: Installera Aspose.PDF för .NET

Du kan installera Aspose.PDF för .NET med någon av följande metoder:

Använda NuGet-pakethanteraren:

```csharp
Install-Package Aspose.PDF
```

Använda .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Använda PackageReference i din projektfil:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Lägg till obligatoriska användsatser

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

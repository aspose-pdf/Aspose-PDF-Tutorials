# Lägg till understruken text i PDF med Aspose.PDF för .NET

Aspose.PDF för .NET är ett kraftfullt PDF-behandlingsbibliotek som gör det möjligt för utvecklare att skapa, ändra och manipulera PDF-dokument utan att behöva Adobe Acrobat. Detta exempel visar hur man skapar ett nytt PDF-dokument och lägger till text med understräckt formatering med hjälp av klasserna TextBuilder och TextFragment. Koden skapar en enkel PDF med ett testmeddelande formaterat i Arial-typsnitt med understräckt stil, och placerar det på specifika koordinater inom sidan.

## Installationsinstruktioner

1: Installera Aspose.PDF för .NET

Du kan installera Aspose.PDF för .NET med hjälp av en av följande metoder:

Använd NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Använda .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Använda PackageReference i projektfilen:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Lägg till nödvändiga användningsord

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

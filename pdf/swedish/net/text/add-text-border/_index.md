# Skapa stiliserad text med gränser i PDF-dokument

Aspose.PDF för .NET är ett kraftfullt PDF-bearbetnings API som gör det möjligt för utvecklare att skapa, redigera, manipulera och konvertera PDF-dokument utan att behöva Adobe Acrobat. Detta exempel visar hur man skapar ett PDF- dokument med ett textfragment som har en specifik stil och en kant, vilket är särskilt användbart för att markera viktigt innehåll i dina dokument.

## Installationsinstruktioner

1. Installera via NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Manuell installation:
   * Ladda ner den senaste versionen från[Aspose.PDF för .NET-utgåvor](https://releases.aspose.com/pdf/net/)
   * Lägg till referens till Aspose.PDF.dll i ditt projekt

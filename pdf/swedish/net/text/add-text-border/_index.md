# Skapa stiliserad text med ramar i PDF-dokument

Aspose.PDF för .NET är ett kraftfullt PDF-bearbetnings-API som låter utvecklare skapa, redigera, manipulera och konvertera PDF-dokument utan att behöva Adobe Acrobat. Det här exemplet visar hur man skapar ett PDF-dokument med ett textfragment som har en specifik stil och en kantlinje, vilket är särskilt användbart för att markera viktigt innehåll i dina dokument.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.PDF
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Manuell installation:
   - Ladda ner den senaste versionen från [Aspose.PDF för .NET-versioner](https://releases.aspose.com/pdf/net/)
   - Lägg till referens till Aspose.PDF.dll i ditt projekt

# Programmering med bilagor

Handledningarna i Aspose.PDF för .NET om programmering med bilagor ger steg-för-steg-instruktioner för att arbeta med bilagor i PDF-dokument. Lär dig hur du lägger till, extraherar och tar bort bilagor, samt hanterar deras egenskaper.

Dessa handledningar guidar dig genom de olika stegen för att manipulera bilagor i dina PDF-dokument, vilket gör att du kan förbättra funktionaliteten och interaktiviteten hos dina PDF-filer.

| [Hämta alla bilagor i PDF-fil](./get-all-the-attachments/) | Lär dig hur du extraherar alla bilagor från en PDF-fil med Aspose.PDF för .NET i den här steg-för-steg-handledningen. |

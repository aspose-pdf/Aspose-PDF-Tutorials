# Desenhar linhas diagonais em PDF usando Aspose.PDF para .NET

Aspose.PDF para .NET permite que desenvolvedores criem, manipulem e editem documentos PDF com facilidade. Neste exemplo, demonstramos como desenhar programaticamente duas linhas diagonais em uma página PDF usando C#. Você aprenderá a inicializar um documento PDF, definir propriedades de página, adicionar elementos gráficos e salvar o resultado final. Este guia fornece uma abordagem passo a passo, facilitando a integração dessa funcionalidade em seus aplicativos .NET.

## Instruções de instalação:  

Para usar o Aspose.PDF para .NET em seu projeto, siga estas etapas:  

1. Instalar via Gerenciador de Pacotes NuGet  
- Abra o Visual Studio e crie um novo projeto .NET.  
- Abra o Console do Gerenciador de Pacotes (Ferramentas → Gerenciador de Pacotes NuGet → Console do Gerenciador de Pacotes) e execute:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalar via .NET CLI  
Como alternativa, use o .NET CLI para instalar o pacote:  
```bash
dotnet add package Aspose.PDF
```

3. Adicione namespace ao seu código C#  
Para usar a funcionalidade Aspose.PDF, inclua os seguintes namespaces:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

# Desenhe linhas diagonais em PDF usando Aspose.PDF para .NET

Aspose.PDF para .NET permite que os desenvolvedores criem, manipulem e editem documentos PDF com facilidade. Neste exemplo, demonstramos como desenhar programaticamente duas linhas diagonais em uma página PDF usando C#. Você aprenderá como inicializar um documento PDF, definir propriedades de página, adicionar elementos gráficos e salvar a saída final. Este guia fornece uma abordagem passo a passo, facilitando a integração desta funcionalidade em seus aplicativos .NET.

## Instruções de instalação:  

Para usar Aspose.PDF para .NET no seu projeto, siga estes passos:  

1. Instalar através do Gerente de Pacotes NuGet  
* Abra o Visual Studio e crie um novo projeto .NET.  
* Abra o Console do Gerente de Pacotes (Tools → NuGet Package Manager → Console do Gerente de Pacotes) e execute:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalação através do .NET CLI  
Alternativamente, use o CLI .NET para instalar o pacote:  
```bash
dotnet add package Aspose.PDF
```

3. Adicionar espaço de nomes no seu código C#  
Para utilizar a funcionalidade Aspose.PDF, incluam os seguintes espaços de nomes:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

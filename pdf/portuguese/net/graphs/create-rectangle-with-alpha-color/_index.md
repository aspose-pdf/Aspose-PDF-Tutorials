# Crie PDF com retângulos e cores alfa no Aspose.PDF

Aspose.PDF para .NET é uma biblioteca poderosa para criar, manipular e converter documentos PDF em C#. Neste exemplo, você aprenderá a criar um arquivo PDF e desenhar retângulos com cores alfa. Usamos o `Aspose.Pdf.Drawing.Graph` Classe para adicionar formas e aplicar efeitos de transparência usando cores ARGB. Este tutorial fornece um guia passo a passo com código de exemplo, garantindo que você possa implementá-lo facilmente em seus aplicativos .NET.

## Instruções de instalação  
Para usar o Aspose.PDF para .NET em seu projeto, siga estas etapas:  

1. Instalar via NuGet  
Execute o seguinte comando no Console do Gerenciador de Pacotes NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Adicionar namespace em C#  
Inclua os namespaces necessários no início do seu arquivo C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Para documentação detalhada, visite:  
[Documentação do Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/)  
[Referência da API Aspose.PDF para .NET](https://reference.aspose.com/pdf/net/)

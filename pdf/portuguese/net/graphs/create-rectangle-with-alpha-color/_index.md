## Instruções de instalação  
Para usar Aspose.PDF para .NET no seu projeto, siga estes passos:  

1. Instalação através de NuGet  
Execute o seguinte comando no Console do Gerente de Pacotes NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Adicionar espaço de nomes em C#  
Incluir os espaços de nomes necessários no início do seu arquivo C #:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Para obter documentação detalhada, visite:  
[Aspose.PDF para Documentação .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF para Referência de API .NET](https://reference.aspose.com/pdf/net/)

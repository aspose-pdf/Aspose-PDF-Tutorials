## Instruções de instalação  
Para usar Aspose.PDF para .NET no seu projeto, siga estes passos:  

1. Instalar através do NuGet  
Execute o seguinte comando no NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Adicionar espaço de nomes em C#  
Inclua os espaços de nomes necessários no início do seu arquivo C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Para obter documentação pormenorizada, visite:  
[Aspose.PDF para documentação .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF para .NET API Referência](https://reference.aspose.com/pdf/net/)

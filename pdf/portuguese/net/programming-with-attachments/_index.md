# Programação com anexos

Os tutoriais do Aspose.PDF para .NET sobre programação com anexos fornecem instruções passo a passo para trabalhar com anexos em documentos PDF. Aprenda a adicionar, extrair e excluir anexos, bem como gerenciar suas propriedades.

Estes tutoriais guiarão você pelas diferentes etapas para manipular anexos em seus documentos PDF, permitindo que você melhore a funcionalidade e a interatividade dos seus arquivos PDF.

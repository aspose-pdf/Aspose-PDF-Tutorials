Este exemplo demonstra como criar um documento PDF usando Aspose.PDF para .NET, com uma tabela com bordas duplas nos lados superior e inferior das células. Siga as instruções passo a passo para implementar esse recurso em seus aplicativos sem problemas.

## Instruções de instalação
Para começar a usar Aspose.PDF para .NET, siga estas instruções de instalação:

1. Baixe Aspose.PDF: 
   * Visita o[Site da Aspose](https://www.aspose.com/downloads/pdf/net)para baixar a versão mais recente de Aspose.PDF para .NET.

2. Instalar através do NuGet Package Manager:
   * Abra seu projeto no Visual Studio.
   * Ir para Ferramentas > NuGet Package Manager > Gerenciar NuGet Pacotes para Solução.
   * Procure por Aspose.PDF e instale o pacote.

3. Adicionar Referências:
   * Certifique-se de que o seu projeto faz referência ao DLL Aspose.PDF no`embalagens`\- A pasta.
   
4. Incluir Aspose.PDF namespace em seu projeto:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

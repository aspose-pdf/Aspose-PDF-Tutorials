## Instruções de instalação
Para começar a usar o Aspose.PDF para .NET, siga estas instruções de instalação:

1. Baixe Aspose.PDF: 
   - Visite o [Site Aspose](https://www.aspose.com/downloads/pdf/net) para baixar a versão mais recente do Aspose.PDF para .NET.

2. Instalar via Gerenciador de Pacotes NuGet:
   - Abra seu projeto no Visual Studio.
   - Acesse Ferramentas > Gerenciador de Pacotes NuGet > Gerenciar Pacotes NuGet para Solução.
   - Procure por "Aspose.PDF" e instale o pacote.

3. Adicionar referências:
   - Certifique-se de que seu projeto faça referência à DLL Aspose.PDF no `packages` pasta.
   
4. Incluir namespace Aspose.PDF em seu projeto:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

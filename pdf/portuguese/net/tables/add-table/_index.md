## Instalação  

Para usar o Aspose.PDF para .NET, siga estas etapas:  

1. Baixe e instale o Aspose.PDF para .NET  
   - Baixar de [Aspose.PDF para downloads .NET](https://releases.aspose.com/pdf/net/)  
   - Instalar via NuGet:  
     ```sh
     Install-Package Aspose.PDF
     ```  

2. Aplicar licença (opcional)  
   - Comprar: [Aspose.PDF Comprar](https://purchase.aspose.com/buy)  
   - Teste gratuito: [Teste grátis do Aspose.PDF](https://releases.aspose.com/)  
   - Licença temporária: [Licença temporária Aspose.PDF](https://purchase.aspose.com/temporary-license/)  

3. Consulte a documentação  
   - [Documentação do Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/)  
   - [Referência da API Aspose.PDF](https://reference.aspose.com/pdf/net/)

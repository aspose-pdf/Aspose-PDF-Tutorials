# Criando texto estilizado com bordas em documentos PDF

Aspose.PDF para .NET é uma poderosa API de processamento de PDF que permite aos desenvolvedores criar, editar, manipular e converter documentos PDF sem a necessidade do Adobe Acrobat. Este exemplo demonstra como criar um documento PDF com um fragmento de texto com estilo específico e uma borda, o que é particularmente útil para destacar conteúdo importante em seus documentos.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.PDF
   ```

2. Instalar via .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Instalação manual:
   - Baixe a versão mais recente em [Aspose.PDF para versões .NET](https://releases.aspose.com/pdf/net/)
   - Adicione referência a Aspose.PDF.dll em seu projeto

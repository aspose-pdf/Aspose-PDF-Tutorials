# Criar texto estilizado com bordas em documentos PDF

Aspose.PDF para .NET é uma API de processamento de PDF poderosa que permite aos desenvolvedores criar, editar, manipular e converter documentos PDF sem precisar do Adobe Acrobat. Este exemplo demonstra como criar um documento PDF com um fragmento de texto que tem estilo específico e uma borda, o que é particularmente útil para destacar conteúdo importante em seus documentos.

## Instruções de instalação

1. Instalar através do NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Instalar através do .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Instalação manual:
   * Baixe a versão mais recente do[Aspose.PDF para versões .NET](https://releases.aspose.com/pdf/net/)
   * Adicionar referência a Aspose.PDF.dll em seu projeto

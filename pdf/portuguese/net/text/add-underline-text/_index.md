# Adicionar texto sublinhado em PDF com Aspose.PDF para .NET

Aspose.PDF para .NET é uma poderosa biblioteca de processamento de PDF que permite aos desenvolvedores criar, modificar e manipular documentos PDF sem a necessidade do Adobe Acrobat. Este exemplo demonstra como criar um novo documento PDF e adicionar texto com formatação sublinhada usando as classes TextBuilder e TextFragment. O código cria um PDF simples com uma mensagem de teste formatada em fonte Arial com estilo sublinhado, posicionando-a em coordenadas específicas dentro da página.

## Instruções de instalação

1: Instale o Aspose.PDF para .NET

Você pode instalar o Aspose.PDF para .NET usando um dos seguintes métodos:

Usando o Gerenciador de Pacotes NuGet:

```csharp
Install-Package Aspose.PDF
```

Usando o .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Usando PackageReference no seu arquivo de projeto:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Adicionar instruções de uso necessárias

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

# Adicionar texto sublinhado em PDF com Aspose.PDF para .NET

Aspose.PDF para .NET é uma biblioteca de processamento de PDF poderosa que permite aos desenvolvedores criar, modificar e manipular documentos PDF sem a necessidade de Adobe Acrobat. Este exemplo demonstra como criar um novo documento PDF e adicionar texto com formatação sublinhada usando as classes TextBuilder e TextFragment. O código cria um PDF simples com uma mensagem de teste formatada em Arial, colocando-a em coordenadas específicas dentro da página.

## Instruções de instalação

1: Instalar Aspose.PDF para .NET

Você pode instalar Aspose.PDF para .NET usando um dos seguintes métodos:

Usando NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Usando .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Usando o PackageReference no arquivo do projeto:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Adicionar as instruções de uso necessárias

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

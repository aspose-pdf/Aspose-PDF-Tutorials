# Otimize PDF compactando imagens no Aspose.PDF
 
Otimizar arquivos PDF é essencial para reduzir o tamanho do arquivo e, ao mesmo tempo, manter a qualidade do documento. Com o Aspose.PDF para .NET, você pode compactar imagens em um PDF com eficiência usando opções de otimização integradas. Este guia demonstra um exemplo simples em C# em que um arquivo PDF é carregado, a compactação de imagem é ativada, as configurações de qualidade são ajustadas e o documento otimizado é salvo. Seguindo essas etapas, você garante que seus PDFs permaneçam leves e fáceis de compartilhar, sem comprometer a clareza.

## Instruções de instalação:  
Para usar o Aspose.PDF para .NET, siga estas etapas:  

1. Instalar via Gerenciador de Pacotes NuGet:  
   - Abra o Visual Studio e vá para Ferramentas > Gerenciador de Pacotes NuGet > Gerenciar Pacotes NuGet para Solução.  
   - Procure por Aspose.PDF e clique em Instalar.  

   OU  

   Execute o seguinte comando no Console do Gerenciador de Pacotes:  
   ```powershell
   Install-Package Aspose.PDF
   ```

2. Baixe do site da Aspose:  
   - Visita [Aspose.PDF para downloads .NET](https://releases.aspose.com/pdf/net/)  
   - Baixe e instale a versão mais recente.  

3. Aplicar uma licença (opcional):  
   - Obtenha uma licença de teste gratuita em [Licença Temporária Aspose](https://purchase.aspose.com/temporary-license/)  
   - Aplique a licença no seu código usando:  
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.PDF.lic");
   ```

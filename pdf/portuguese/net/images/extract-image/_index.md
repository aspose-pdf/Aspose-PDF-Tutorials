## Instruções de instalação:  

Para usar Aspose.PDF para .NET no seu projeto C#, siga estes passos:  

1. Instalar através do Gerente de Pacotes NuGet:  
* Abre o Visual Studio.  
* Vá para Ferramentas > Gerente de Pacotes NuGet > Gerencie Pacotes NuGet para Solução.  
* Procure Aspose.PDF e instale-o.  
* Ou, executar o seguinte comando no Console do Gestor de Pacotes:  

  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalação através do Download DLL:  
* Baixe a versão mais recente de [Aspose.PDF para .NET Download](https://releases.aspose.com/pdf/net/)\- Não .  
* Extrair o pacote baixado.  
* Adicione uma referência ao Aspose.PDF.dll no seu projeto .NET.  

3. Aplicar uma Licença (opcional):  
* Podes arranjar um .[licença temporária gratuita](https://purchase.aspose.com/temporary-license/)para a plena funcionalidade.

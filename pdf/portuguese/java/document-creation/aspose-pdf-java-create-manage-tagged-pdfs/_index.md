objeto Document**\n```java\nimport com.aspose.pdf.Document;\n\n// Define output directory for saving documents\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n\n// Create a new, empty PDF title and language for accessibility compliance\ntaggedContent.setTitle(\"Tagged Pdf Document\");\ntaggedContent.setLanguage(\"en-US\");\n```\n\n**Salvar o PDF marcado**\n```java\ndocument.save(dataDir + \"/TaggedPdfDocument.pdf\");\n```\n\n### Recurso 3: Adicionando elementos de estrutura de texto a um PDF marcado\n\n**Visão geral:** Estruture o conteúdo do seu documento para melhor legibilidade e acessibilidade.\n\n#### Passo a passo:\n\n**Criar e adicionar elemento de parágrafo**\n```java\nimport com.aspose.pdf.Document;\nimport com.aspose.pdf.tagged.ITaggedContent;\nimport com.aspose.pdf.tagged.logicalstructure.elements.ParagraphElement;\n\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n\nDocument document = new Document();\nITaggedContent taggedContent = document.getTaggedContent();\ntaggedContent.getRootElement().appendChild(taggedContent.createParagraphElement()\n    .setText(\"This is a sample paragraph in a tagged PDF.\"));\n```\n\n**Salvar o PDF aprimorado**\n```java\ndocument.save(outputDir + \"/TextStructureElementsAdded.pdf\");\n```\n\n## Aplicações práticas\n\nAqui estão alguns cenários

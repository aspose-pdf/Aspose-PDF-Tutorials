## Instrukcja montażu:  

Aby użyć Aspose.PDF dla .NET w projekcie C#, wykonaj następujące kroki:  

1. Zainstalowanie za pośrednictwem NuGet Package Manager:  
* Otwórz Studio wizualne.  
* Przejdź do Narzędzi > Menedżer pakietów NuGet > Zarządzaj pakietami NuGet dla rozwiązania.  
* Poszukaj Aspose.PDF i zainstaluj.  
* Lub uruchom następującą komendę w konsoli menedżera pakietów:  

  ```powershell
  Install-Package Aspose.PDF
  ```

2. Zainstalować za pomocą DLL Pobierz:  
* Pobierz najnowszą wersję z[Aspose.PDF dla .NET Download](https://releases.aspose.com/pdf/net/)\- Nie .  
* Wyciągnij pobrany pakiet.  
* Dodaj odniesienie do Aspose.PDF.dll w projekcie .NET.

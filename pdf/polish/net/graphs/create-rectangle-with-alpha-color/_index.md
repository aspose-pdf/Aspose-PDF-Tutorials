## Instrukcje instalacji  
Aby użyć Aspose.PDF dla .NET w swoim projekcie, wykonaj następujące kroki:  

1. Instalacja za pośrednictwem NuGet  
Wyślij następujące polecenie w konsoli Menedżera pakietów NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Dodaj przestrzeni nazw w C#  
Włączyć wymagane przestrzenie nazw na początku pliku C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Szczegółowe informacje można znaleźć w:  
[Aspose.PDF dla dokumentacji .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF dla odniesienia API .NET](https://reference.aspose.com/pdf/net/)

# Tworzenie PDF z przejrzystymi grafikami za pomocą Aspose.PDF

W tym przykładzie pokazujemy, jak tworzyć dokument PDF i dodać wykres zawierający prostokąt z przejrzystymi kolorami wypełnienia. ## Instrukcje instalacji:  

Aby użyć Aspose.PDF dla .NET, wykonaj następujące kroki:  

1. Instalacja za pomocą Menedżera pakietów NuGet  
* Otwórz Visual Studio i stworz nowy projekt .NET.  
* Otwórz konsolę zarządzającego pakietami (Narzędzia → NuGet Manager pakietu → Konsola zarządzającego pakietami) i uruchomi:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalacja za pośrednictwem .NET CLI  
W celu uzyskania informacji o potrzebach, należy zainstalować pakiet za pomocą .NET CLI:  
```bash
dotnet add package Aspose.PDF
```

3. Dodaj przestrzeni nazw w kodzie C#  
Aby zacząć używać Aspose.PDF, dodaj następujące przestrzenie nazw do pliku C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

## Instrukcje instalacji:  

Aby użyć Aspose.PDF dla platformy .NET, wykonaj następujące kroki:  

1. Zainstaluj za pomocą Menedżera pakietów NuGet  
- Otwórz program Visual Studio i utwórz nowy projekt .NET.  
- Otwórz konsolę Menedżera pakietów (Narzędzia → Menedżer pakietów NuGet → Konsola Menedżera pakietów) i uruchom:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Zainstaluj za pomocą .NET CLI  
Alternatywnie zainstaluj pakiet za pomocą interfejsu wiersza poleceń .NET:  
```bash
dotnet add package Aspose.PDF
```

3. Dodaj przestrzeń nazw w kodzie C#  
Aby rozpocząć korzystanie z pliku Aspose.PDF, dodaj następującą przestrzeń nazw do pliku C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

## Instrukcja montażu:  

Aby użyć Aspose.PDF dla .NET, wykonaj następujące kroki:  

1. Zainstalowanie za pośrednictwem NuGet Package Manager  
* Otwórz Visual Studio i utwórz nowy projekt .NET.  
* Otwórz konsolę menedżera pakietów (Tools → NuGet Package Manager → Package Manager Console) i uruchom:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Zainstalowanie za pomocą interfejsu .NET  
Alternatywnie, zainstaluj pakiet za pomocą interfejsu .NET CLI:  
```bash
dotnet add package Aspose.PDF
```

3. Dodaj przestrzenie nazw w kodzie C#  
Aby zacząć używać Aspose.PDF, dodaj następującą przestrzeń nazw do pliku C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

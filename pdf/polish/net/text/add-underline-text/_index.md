# Dodawanie podkreślonego tekstu w pliku PDF za pomocą Aspose.PDF dla platformy .NET

Aspose.PDF dla .NET to potężna biblioteka przetwarzania PDF, która umożliwia programistom tworzenie, modyfikowanie i manipulowanie dokumentami PDF bez konieczności korzystania z programu Adobe Acrobat. Ten przykład pokazuje, jak utworzyć nowy dokument PDF i dodać tekst z podkreśleniem za pomocą klas TextBuilder i TextFragment. Kod tworzy prosty plik PDF z wiadomością testową sformatowaną czcionką Arial ze stylem podkreślenia, umieszczając go w określonych współrzędnych na stronie.

## Instrukcje instalacji

1: Zainstaluj Aspose.PDF dla .NET

Możesz zainstalować Aspose.PDF dla platformy .NET, korzystając z jednej z następujących metod:

Korzystanie z Menedżera pakietów NuGet:

```csharp
Install-Package Aspose.PDF
```

Korzystanie z interfejsu wiersza poleceń .NET:

```bash
dotnet add package Aspose.PDF
```

Używanie PackageReference w pliku projektu:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Dodaj wymagane polecenia using

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

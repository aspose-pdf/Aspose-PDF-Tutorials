# Dodawanie podkreślonego tekstu w PDF z Aspose.PDF dla .NET

Aspose.PDF dla .NET to potężna biblioteka przetwarzania plików PDF, która pozwala programistom tworzyć, modyfikować i manipulować dokumentami PDF bez konieczności użycia Adobe Acrobat. Ten przykład pokazuje, jak tworzyć nowy dokument PDF i dodawać tekst z formatowaniem podkreślonym przy użyciu klas TextBuilder i TextFragment. Kod tworzy prosty plik PDF z komunikatem testowym sformatowanym w czcionce Arial ze stylizowaniem podkreślonym, umieszczając go na określonych współrzędnych w stronie.

## Instrukcje montażu

1: Zainstaluj Aspose.PDF dla .NET

Możesz zainstalować Aspose.PDF dla .NET przy użyciu jednej z następujących metod:

Korzystanie z NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Używanie interfejsu .NET:

```bash
dotnet add package Aspose.PDF
```

Użycie pakietu PackageReference w pliku projektu:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Dodaj wymagane wypowiedzi używające

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

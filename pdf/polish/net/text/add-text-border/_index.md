## Instrukcje montażu

1. Zainstalowanie za pośrednictwem NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Zainstalowanie za pomocą .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Instalacja ręczna:
   * Pobierz najnowszą wersję z[Aspose.PDF dla wersji .NET](https://releases.aspose.com/pdf/net/)
   * Dodaj odniesienie do Aspose.PDF.dll w projekcie

obiekt dokumentu**\N```java\nimport com.aspose.pdf.Document;\n\n// Define output directory for saving documents\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n\n// Create a new, empty PDF document\nDocument document = new Document();\n```\n\n**Zapisz dokument PDF**\n\nUstaw żądaną ścieżkę pliku i zapisz dokument.\n```java\n// Save the newly created PDF to the specified location\ndocument.save(outputDir + \"/CreatedPdfDocument.pdf\");\n```\n\n### Funkcja 2: Praca z oznaczoną zawartością PDF\n\n**Przegląd:** Zwiększ dostępność, dodając do plików PDF metadane, takie compliance\ntaggedContent.setTitle(\"Tagged Pdf Document\");\ntaggedContent.setLanguage(\"en-US\");\n```\n\n**Zapisz oznaczony plik PDF**\N```java\ndocument.save(dataDir + \"/TaggedPdfDocument.pdf\");\n```\n\n### Funkcja 3: Dodawanie elementów struktury tekstowej do oznaczonych plików PDF\n\n**Przegląd:** Ustrukturyzuj treść dokumentu tak, aby była bardziej czytelna i com.aspose.pdf.tagged.logicalstructure.elements.ParagraphElement;\n\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n\nDocument document = new Document();\nITaggedContent taggedContent = document.getTaggedContent();\ntaggedContent.getRootElement().appendChild(taggedContent.createParagraphElement()\n    .setText(\"This is a sample paragraph in a tagged PDF.\"));\n```\n\n**Zapisz ulepszony plik PDF**\N```java\ndocument.save(outputDir + \"/TextStructureElementsAdded.pdf\");\n```\n\n## Zastosowania praktyczne\n\nOto kilka praktycznych scenariuszy, w których Aspose.PDF dla Java

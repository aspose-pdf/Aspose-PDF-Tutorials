Te praktyczne przewodniki przeprowadzą Cię przez dostęp do standardowych metadanych dokumentów i ich modyfikację, tworzenie niestandardowych pól metadanych, implementację metadanych XMP i pracę ze słownikami informacji o dokumentach. Każdy samouczek zawiera szczegółowe przykłady kodu Java do manipulowania metadanymi zgodnie ze standardami PDF, pomagając Ci tworzyć aplikacje, które zachowują właściwą atrybucję i klasyfikację dokumentów.

Importuj wymagane biblioteki**\N```java\nimport com.aspose.pdf.Document;\nimport com.aspose.pdf.DocumentInfo;\n```\n\n**2. Załaduj dokument PDF**\nZdefiniuj katalog dokumentu i załaduj plik PDF.\n```java\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nDocument pdfDocument = new Document(dataDir + \"/Original.pdf\");\n```\n\n**3. Wyodrębnij informacje metadanych**\nUzyskaj dostęp i pobierz właściwości metadanych z dokumentu.\n\n```java\nDocumentInfo docInfo = pdfDocument.getInfo();\nString author = docInfo.getAuthor();\nString creationDate = docInfo.getCreationDate().toString();\nString keywords = docInfo.getKeywords();\nString modifyDate = docInfo.getModDate().toString();\nString subject = docInfo.getSubject();\nString title = docInfo.getTitle();\n```\n\n**Wyjaśnienie**:Ten fragment kodu otwiera dokument PDF i pobiera jego metadane za pomocą `DocumentInfo`która jest częścią biblioteki Aspose.PDF. Metody takie jak `getAuthor()` I `getCreationDate()` wyodrębnij określone fragmenty metadanych.\n\n### Funkcja 2: Zapisywanie metadanych pliku PDF\nTa funkcja obejmuje zapisywanie wyodrębnionych metadanych w pliku tekstowym w celu dalszego wykorzystania lub przechowywania danych.\n\n#### Omówienie\nPo wyodrębnieniu metadanych często przydatne jest zapisanie tych informacji w ustrukturyzowanym formacie, takim jak zwykły plik tekstowy.\n\n##### Implementacja krok po kroku\n\n**1. Zapisz metadane do pliku**\nSkonfiguruj katalog wyjściowy i przygotuj FileWriter.\n\n```java\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\ntry (FileWriter writer = new FileWriter(outputDir + \"/pdf_info.txt\")) {\n    // Writing each metadata field on a separate line for clarity\n    writer.write(\"Author:-\" + author + \"\\n\");\n    writer.write(\"Creation Date:-\" + creationDate + \"\\n\");\n    writer.write(\"Keywords:-\" + keywords + \"\\n\");\n    writer.write(\"Modify Date:-\" + modifyDate + \"\\n\");\n    writer.write(\"Subject:-\" + subject + \"\\n\");\n    writer.write(\"Title:-\" + title);\n} catch (IOException e) {\n    e.printStackTrace();\n}\n```\n\n**Wyjaśnienie**:Ten fragment kodu zapisuje każdy element metadanych w oddzielnym wierszu `pdf_info.txt`.

`documentForm` metoda, która otwiera, spłaszcza i zapisuje dokument PDF.\n\n```java\nvoid documentForm() {\n    // Initialize Form object to interact with the PDF\n    Form pdfForm = new Form();\n    \n    // Bind the source PDF file from 'dataDir/input.pdf'\n    pdfForm.bindPdf(dataDir + \"/input.pdf\");\n    \n    // Flatten all fields in the PDF document\n    pdfForm.flattenAllFields();\n    \n    // Save the modified document to 'outputDir/output.pdf'\n    pdfForm.save(outputDir + \"/output.pdf\");\n}\n```\n\n#### Wyjaśnienie:\n- **`Form` Klasa**: Służy do manipulowania polami formularzy w Zapisuje zmodyfikowaną zawartość do nowego pliku.\n\n### Porady dotyczące rozwiązywania problemów\n- Upewnij się, że podano prawidłowe ścieżki w `dataDir` I `outputDir`.\n- Obsługa wyjątków, które

### आप क्या सीखेंगे:
- Java के लिए Aspose.PDF सेट अप करना.
- किसी PDF दस्तावेज़ में विशिष्ट पाठ खोजना।

## कार्यान्वयन मार्गदर्शिका

यह अनुभाग आपको Java के लिए Aspose.PDF का उपयोग करके पाठ खंड निष्कर्षण को कार्यान्वित करने में मार्गदर्शन करेगा।

### पाठ खंडों को खोजना और निकालना (H2)

आइए सुविधा कार्यान्वयन को तार्किक चरणों में विभाजित करें:

#### चरण 1: पीडीएफ दस्तावेज़ लोड करें

अपने PDF दस्तावेज़ को लोड करके शुरू करें। यह एक उदाहरण बनाकर किया जाता है `Document` क्लास में जाकर, उसे अपनी फ़ाइल का पथ प्रदान करें।

```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String inputFilePath = dataDir + "/input.pdf";

// पीडीएफ दस्तावेज़ लोड करें
Document pdfDocument = new Document(inputFilePath);
```

#### चरण 2: एक टेक्स्टफ़्रेगमेंटअब्ज़ॉर्बर बनाएँ

The `TextFragmentAbsorber` क्लास का उपयोग आपके दस्तावेज़ में विशिष्ट टेक्स्ट खोजने के लिए किया जाता है। आपको इस क्लास का एक इंस्टेंस बनाना होगा, जिसमें आप जिस टेक्स्ट को खोज रहे हैं उसे निर्दिष्ट करना होगा।

```java
import com.aspose.pdf.TextFragmentAbsorber;

// 'sample' को खोज कीवर्ड के रूप में उपयोग करके TextFragmentAbsorber ऑब्जेक्ट बनाएँ
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("sample");
```

#### चरण 3: सभी पृष्ठों पर अवशोषक लागू करें

उपयोग `accept` आपके दस्तावेज़ के सभी पृष्ठों पर विधि। यह चरण पूरे दस्तावेज़ में निर्दिष्ट पाठ की घटनाओं को खोजता है।

```java
// अवशोषक को सभी पृष्ठों पर लागू करें
pdfDocument.getPages().accept(textFragmentAbsorber);
```

#### चरण 4: पाठ अंशों को पुनः प्राप्त करें और उन पर पुनरावृति करें

अवशोषक को लागू करने के बाद, पाए गए पाठ अंशों के संग्रह को पुनः प्राप्त करें। विस्तृत जानकारी निकालने के लिए प्रत्येक अंश और उसके खंडों पर पुनरावृत्ति करें।

```java
import com.aspose.pdf.TextFragmentCollection;
import com.aspose.pdf.TextFragment;
import com.aspose.pdf.TextSegment;

// पाए गए पाठ अंशों का संग्रह प्राप्त करें
textFragmentCollection = textFragmentAbsorber.getTextFragments();

// विवरण निकालने के लिए प्रत्येक TextFragment और उसके खंडों पर पुनरावृति करें
for (TextFragment textFragment : (Iterable<TextFragment>) textFragmentCollection) {
    for (TextSegment textSegment : (Iterable<TextSegment>) textFragment.getSegments()) {
        // प्रत्येक पाठ खंड के विभिन्न गुण निकालें
        String text = textSegment.getText();
        Object position = textSegment.getPosition();
        
        double xIndent = position != null ? ((com.aspose.pdf.Point)position).getXIndent() : 0;
        double yIndent = position != null ? ((com.aspose.pdf.Point)position).getYIndent() : 0;
        
        String fontName = textSegment.getTextState().getFont().getFontName();
        boolean isAccessible = textSegment.getTextState().getFont().isAccessible();
        boolean isEmbedded = textSegment.getTextState().getFont().isEmbedded();
        boolean isSubset = textSegment.getTextState().getFont().isSubset();
        
        double fontSize = textSegment.getTextState().getFontSize();
        String foregroundColor = textSegment.getTextState().getForegroundColor() != null ? textSegment.getTextState().getForegroundColor().toString() : "null";
        
        // उदाहरण: आउटपुट निकाले गए विवरण
        System.out.println("Text: " + text);
        System.out.println("XIndent: " + xIndent);
    }
}
```

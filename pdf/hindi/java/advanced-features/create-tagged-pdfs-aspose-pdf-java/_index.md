## प्रदर्शन संबंधी विचार
जब आप बड़े PDFs (सैकड़ों पृष्ठ या हाई‑रेज़ोल्यूशन इमेज) के साथ काम कर रहे हों, तो इन टिप्स को ध्यान में रखें:

- **Chunk Processing** – पूरे दस्तावेज़ को एक बार लोड करने के बजाय बैच में एलिमेंट्स बनाएं और टैग करें।  
- **Dispose Early** – काम समाप्त होने पर `document.dispose()` कॉल करके नेटिव रिसोर्सेज़ मुक्त करें।  
- **Memory Management** – आवश्यकता से अधिक बड़े ऑब्जेक्ट्स को रखे रखने से बचें; लूप के भीतर स्थानीय वेरिएबल्स का उपयोग करें।

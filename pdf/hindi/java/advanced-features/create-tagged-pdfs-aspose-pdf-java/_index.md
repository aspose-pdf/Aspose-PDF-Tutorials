## प्रदर्शन विचार
जब आप बड़े PDFs (सैकड़ों पृष्ठ या उच्च‑रिज़ॉल्यूशन इमेज) से निपट रहे हों, तो इन टिप्स को याद रखें:

- **Chunk Processing** – पूरे दस्तावेज़ को एक साथ लोड करने के बजाय बैच में तत्व बनाएं और टैग करें।  
- **Dispose Early** – काम समाप्त होने पर `document.dispose()` कॉल करके नेटिव रिसोर्सेज़ मुक्त करें।  
- **Memory Management** – बड़े ऑब्जेक्ट्स को आवश्यक से अधिक समय तक न रखें; लूप के अंदर स्थानीय वेरिएबल्स का उपयोग करें।

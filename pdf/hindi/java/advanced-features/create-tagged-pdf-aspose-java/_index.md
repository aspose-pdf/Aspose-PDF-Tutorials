### टैग किया हुआ PDF Java बनाने के चरण 1: दस्तावेज़ बनाएं और कॉन्फ़िगर करें

एक नया `Document` इंस्टेंस लोड करें, टैगिंग सक्षम करें, और शीर्षक व भाषा जैसे बुनियादी दस्तावेज़ गुण सेट करें। यह चरण लॉजिकल‑स्ट्रक्चर कंटेनर स्थापित करता है जिसमें सभी बाद के टैग शामिल होंगे।

#### अवलोकन
`Document` Aspose.PDF की मुख्य क्लास है जो मेमोरी में PDF दस्तावेज़ का प्रतिनिधित्व करती है।  
पहले, हम एक `Document` ऑब्जेक्ट बनाते हैं और उसकी **tagged content** इंटरफ़ेस प्राप्त करते हैं। यह इंटरफ़ेस हमें लॉजिकल स्ट्रक्चर को नियंत्रित करने की अनुमति देता है।  
`Document` क्लास Aspose.PDF का टॉप‑लेवल ऑब्जेक्ट है जो मेमोरी में एकल PDF फ़ाइल का प्रतिनिधित्व करता है। इंस्टैंसिएशन के बाद, सभी पढ़ने और लिखने के ऑपरेशन इस ऑब्जेक्ट के माध्यम से होते हैं।

```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY"; 
Document document = new Document();
```

```java
import com.aspose.pdf.tagged.ITaggedContent;

ITaggedContent taggedContent = document.getTaggedContent();
```

```java
String title = "Tagged Pdf Document";
String language = "en-US";

taggedContent.setTitle(title);
taggedContent.setLanguage(language);
```

### टैग किया हुआ PDF Java बनाने के चरण 2: कस्टम टैग के साथ पैराग्राफ तत्व जोड़ें

`SectElement` कंटेनर बनाएं और उसे `ParagraphElement` ऑब्जेक्ट्स से भरें, प्रत्येक को आसान पहचान और बाद में एक्सट्रैक्शन के लिए एक कस्टम टैग दिया जाता है।

#### अवलोकन
`SectElement` PDF के टैग ट्री में एक लॉजिकल सेक्शन का प्रतिनिधित्व करता है।  
`ParagraphElement` एक सेक्शन के भीतर टेक्स्ट ब्लॉक को परिभाषित करता है।  
हम एक **सेक्शन** (`SectElement`) बनाते हैं जो कंटेनर के रूप में कार्य करता है, फिर उसके अंदर कई **पैराग्राफ** (`ParagraphElement`) जोड़ते हैं। प्रत्येक पैराग्राफ को आसान पहचान के लिए एक कस्टम टैग मिलता है।

```java
import com.aspose.pdf.tagged.logicalstructure.elements.bls.ParagraphElement;
import com.aspose.pdf.tagged.logicalstructure.elements.grouping.SectElement;

SectElement sect = taggedContent.createSectElement();
taggedContent.getRootElement().appendChild(sect);

ParagraphElement p1 = taggedContent.createParagraphElement();
p1.setText("P1. ");
p1.setTag("P1");
// Repeat for other paragraphs...
```

```java
sect.appendChild(p1);
// Append other paragraphs similarly...
```

### टैग किया हुआ PDF Java बनाने के चरण 3: कस्टम टैग के साथ स्पैन तत्व जोड़ें

पैराग्राफ के भीतर `SpanElement` ऑब्जेक्ट्स डालें ताकि इनलाइन टेक्स्ट फ्रैगमेंट्स को टैग किया जा सके, जिससे आपको स्टाइलिंग और एक्सेसिबिलिटी मेटाडाटा पर सूक्ष्म नियंत्रण मिलता है।

#### अवलोकन
`SpanElement` पैराग्राफ के भीतर एक इनलाइन टेक्स्ट सेगमेंट को टैग करता है।  
`Spans` पैराग्राफ को छोटे टुकड़ों में विभाजित करते हैं, जिससे हम इनलाइन टेक्स्ट फ्रैगमेंट्स को टैग कर सकते हैं। यही वह जगह है जहाँ **tag spans pdf** वास्तव में चमकता है।

```java
import com.aspose.pdf.tagged.logicalstructure.elements.ils.SpanElement;

SpanElement span1 = taggedContent.createSpanElement();
span1.setText("Span 1.");
span1.setTag("SPAN");
// Repeat for other spans...
```

```java
p1.appendChild(span1);
// Append other spans similarly...
```

### टैग किया हुआ PDF Java बनाने के चरण 4: टैग किए हुए PDF दस्तावेज़ को सहेजें

एकल `save` कॉल के साथ पूरी‑टैग की हुई दस्तावेज़ को डिस्क पर सहेजें, यह सुनिश्चित करते हुए कि सभी लॉजिकल‑स्ट्रक्चर जानकारी आउटपुट फ़ाइल में लिखी गई है।

#### अवलोकन
लॉजिकल स्ट्रक्चर बनाने के बाद, हम बस दस्तावेज़ को इच्छित स्थान पर सहेजते हैं।  
`save` मेथड विज़ुअल कंटेंट और छिपे हुए टैग ट्री दोनों को लिखता है, जिससे एक ऐसा PDF बनता है जो PDF/UA के अनुरूप है और SEO क्रॉलिंग के लिए तैयार है।

```java
String outputPath = dataDir + "CustomTag.pdf";
document.save(outputPath);
```

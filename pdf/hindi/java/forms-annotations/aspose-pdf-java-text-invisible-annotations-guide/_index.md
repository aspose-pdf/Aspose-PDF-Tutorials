# Java के लिए Aspose.PDF का उपयोग करके PDF में टेक्स्ट और अदृश्य एनोटेशन जोड़ें

## परिचय

अपने PDF दस्तावेज़ों को उपयोगी एनोटेशन जोड़कर बेहतर बनाएँ, चाहे वह महत्वपूर्ण टेक्स्ट को हाइलाइट करना हो या मेटाडेटा एम्बेड करना हो। इस व्यापक ट्यूटोरियल में, आप सीखेंगे कि Aspose.PDF for Java का उपयोग करके दृश्यमान और अदृश्य दोनों एनोटेशन कैसे जोड़ें - एक मज़बूत लाइब्रेरी जो Java अनुप्रयोगों में PDF हेरफेर को सरल बनाती है।

**आप क्या सीखेंगे:**
- Java के लिए Aspose.PDF सेट अप करना.
- शीर्षक, विषय और बॉर्डर जैसी कस्टम सेटिंग्स के साथ टेक्स्ट एनोटेशन जोड़ना।
- अदृश्य एनोटेशन को क्रियान्वित करना जो दस्तावेज़ के भीतर मौजूद होते हैं लेकिन अदृश्य रहते हैं।
- इन सुविधाओं को अपने मौजूदा जावा प्रोजेक्ट्स में एकीकृत करना।

आइये समझते हैं कि शुरुआत करने के लिए आपको किन पूर्व-आवश्यकताओं की आवश्यकता है।

## Java के लिए Aspose.PDF सेट अप करना

Maven या Gradle का उपयोग करके अपने प्रोजेक्ट में लाइब्रेरी को निर्भरता के रूप में शामिल करें:

**मावेन:**
```xml

    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

**ग्रेडेल:**
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## कार्यान्वयन मार्गदर्शिका

### पाठ एनोटेशन जोड़ना

#### अवलोकन
टेक्स्ट एनोटेशन आपको सीधे पीडीएफ पेजों पर वर्णनात्मक टिप्पणियाँ या मेटाडेटा जोड़ने की अनुमति देते हैं। वे समीक्षा, नोट्स या दस्तावेज़ों के भीतर अनुभागों को हाइलाइट करने के लिए उपयोगी हैं।

**चरण-दर-चरण कार्यान्वयन:**

##### चरण 1: दस्तावेज़ को आरंभ करें
अपना स्रोत पीडीएफ दस्तावेज़ लोड करें:
```java
document = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
यह पंक्ति आपके जावा अनुप्रयोग में एक मौजूदा पीडीएफ फाइल लोड करती है।

##### चरण 2: टेक्स्ट एनोटेशन बनाएं और कॉन्फ़िगर करें
एक बनाने के `TextAnnotation` प्रथम पृष्ठ के लिए ऑब्जेक्ट, शीर्षक, विषय, सामग्री और बॉर्डर जैसे गुण सेट करना:
```java
TextAnnotation textAnnotation = new TextAnnotation(document.getPages().get_Item(1), new Rectangle(200, 400, 400, 600));
textAnnotation.setTitle("Sample Annotation Title");
textAnnotation.setSubject("Sample Subject");
textAnnotation.setState(com.aspose.pdf.AnnotationState.Accepted);
textAnnotation.setContents("Sample contents for the annotation");
textAnnotation.setOpen(true);
textAnnotation.setIcon(TextIcon.Key);

Border border = new Border(textAnnotation);
border.setWidth(5);
border.setDash(new Dash(1, 1));
textAnnotation.setBorder(border);
```

##### चरण 3: दस्तावेज़ में एनोटेशन जोड़ें
यह एनोटेशन जोड़ें और अपना दस्तावेज़ सहेजें:
```java
document.getPages().get_Item(1).getAnnotations().add(textAnnotation);
document.save("YOUR_OUTPUT_DIRECTORY/addAnnotationToPDF.pdf");
```

### अदृश्य एनोटेशन जोड़ना

#### अवलोकन
अदृश्य एनोटेशन पीडीएफ के भीतर मेटाडेटा या डेटा एम्बेड करने के लिए उपयोगी होते हैं, जो अंतिम उपयोगकर्ताओं के लिए अदृश्य रहना चाहिए, जैसे ट्रैकिंग या आंतरिक नोट्स के लिए।

**चरण-दर-चरण कार्यान्वयन:**

##### चरण 1: नया दस्तावेज़ आरंभ करें
अपने दस्तावेज़ में एक रिक्त पृष्ठ बनाएं और जोड़ें:
```java
document = new Document();
document.getPages().add();
```

##### चरण 2: अदृश्य एनोटेशन बनाएं और कॉन्फ़िगर करें
उपयोग `FreeTextAnnotation` दृश्यता नियंत्रण के लिए विशिष्ट झंडों के साथ:
```java
FreeTextAnnotation annotation = new FreeTextAnnotation(document.getPages().get_Item(1), new Rectangle(50, 600, 250, 650), new DefaultAppearance("Helvetica", 16, Color.RED));
annotation.setContents("ABCDEFG");
annotation.getCharacteristics().setBorder(Color.RED);

// एनोटेशन को अदृश्य बनाने के लिए फ़्लैग सेट करें
annotation.setFlags(AnnotationFlags.Print | AnnotationFlags.NoView);
```

##### चरण 3: दस्तावेज़ जोड़ें और सहेजें
यह एनोटेशन जोड़ें और अपना दस्तावेज़ सहेजें:
```java
document.getPages().get_Item(1).getAnnotations().add(annotation);
document.save("YOUR_OUTPUT_DIRECTORY/Invisible_Annotation.pdf");
```

## व्यावहारिक अनुप्रयोगों
- **दस्तावेज़ समीक्षा:** व्यावसायिक या शैक्षणिक सेटिंग में सहयोगात्मक समीक्षा के लिए पाठ एनोटेशन का उपयोग करें।
- **मेटाडेटा टैगिंग:** दृश्य सामग्री में परिवर्तन किए बिना दस्तावेज़ संपादन और संस्करण को ट्रैक करने के लिए अदृश्य एनोटेशन एम्बेड करें।
- **कानूनी दस्तावेजों:** आंतरिक उपयोग के लिए अनुबंधों या कानूनी कागजातों में गैर-हस्तक्षेपकारी नोट्स जोड़ें।
- **शिक्षण सामग्री:** पाठ्यपुस्तकों में मुख्य बिंदुओं को उजागर करें या स्पष्टीकरण जोड़ें।

- **स्मृति प्रबंधन:** उपयुक्त मेमोरी सेटिंग्स का उपयोग करें और जब आवश्यकता न हो तो संसाधन जारी कर दें।

## कार्यान्वयन मार्गदर्शिका
इस अनुभाग में, हम कार्यान्वयन को प्रबंधनीय भागों में विभाजित करेंगे, और Aspose.PDF Java का उपयोग करके जावास्क्रिप्ट क्रियाओं के साथ समाप्ति कार्यक्षमता जोड़ने पर ध्यान केंद्रित करेंगे।
### समाप्ति कार्यक्षमता जोड़ना
हमारे ट्यूटोरियल की मुख्य विशेषता आपके PDF में एक जावास्क्रिप्ट क्रिया एम्बेड करना है जो यह जांचता है कि दस्तावेज़ खोलने पर समाप्त हो गया है या नहीं। यहाँ बताया गया है कि आप इसे कैसे पूरा कर सकते हैं:
#### चरण 1: अपना PDF दस्तावेज़ लोड करें
Aspose.PDF का उपयोग करके अपनी निर्देशिका से एक मौजूदा PDF फ़ाइल लोड करके प्रारंभ करें।
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // इनपुट PDF निर्देशिका पथ सेट करें
Document doc = new Document(dataDir + "/input.pdf");
```
यह कोड एक आरंभीकरण करता है `Document` ऑब्जेक्ट, जो आपके PDF का प्रतिनिधित्व करता है। आपको प्रतिस्थापित करने की आवश्यकता होगी `"YOUR_DOCUMENT_DIRECTORY"` वास्तविक पथ के साथ जहां आपकी पीडीएफ संग्रहीत हैं।
#### चरण 2: समाप्ति जाँच के लिए जावास्क्रिप्ट को परिभाषित करें
इसके बाद, जावास्क्रिप्ट को परिभाषित करें जो यह जांच करेगा कि क्या वर्तमान तिथि समाप्ति तिथि से अधिक है और यदि ऐसा है तो अलर्ट प्रदर्शित करेगा।
```java
JavascriptAction javaScript = new JavascriptAction(
    "var year=2014; var month=4; today = new Date(); today = new Date(today.getFullYear(), today.getMonth());" +
    "expiry = new Date(year, month);" +
    "if (today.getTime() > expiry.getTime()) app.alert('The file is expired. You need a new one.');"
);
```
यह स्क्रिप्ट एक समाप्ति तिथि निर्धारित करती है (`2014-04`) और हर बार पीडीएफ खोले जाने पर इसकी तुलना वर्तमान तिथि से करता है। यदि दस्तावेज़ की खुली तिथि इससे अधिक है, तो उपयोगकर्ता को एक अलर्ट सूचित करता है।
#### चरण 3: जावास्क्रिप्ट को ओपन एक्शन के रूप में सेट करें
अपने पीडीएफ में जावास्क्रिप्ट क्रिया को एकीकृत करें ताकि इसे खोलने पर यह सक्रिय हो जाए:
```java
doc.setOpenAction(javaScript);
```
कोड की यह पंक्ति यह सुनिश्चित करती है कि जब भी आपका पीडीएफ खोला जाएगा, तो जावास्क्रिप्ट स्वचालित रूप से चलेगा और समाप्ति की जांच करेगा।
#### चरण 4: संशोधित पीडीएफ को सहेजें
अंत में, नए जोड़े गए जावास्क्रिप्ट एक्शन के साथ दस्तावेज़ को निर्दिष्ट आउटपुट निर्देशिका में सहेजें।
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // आउटपुट PDF निर्देशिका पथ सेट करें
doc.save(outputDir + "/JavaScript-Added.pdf");
```
प्रतिस्थापित करें `"YOUR_OUTPUT_DIRECTORY"` अपने इच्छित आउटपुट फ़ोल्डर के पथ के साथ। यह आपके संशोधित पीडीएफ को सहेजता है, जो समाप्ति सुविधा के साथ वितरण के लिए तैयार है।
### समस्या निवारण युक्तियों
आम समस्याओं में गलत फ़ाइल पथ या जावास्क्रिप्ट त्रुटियाँ शामिल हो सकती हैं। अपने निर्देशिका पथों की दोबारा जाँच करें और सुनिश्चित करें कि आपका जावास्क्रिप्ट सिंटैक्स त्रुटि-मुक्त है। यदि समस्याएँ बनी रहती हैं, तो सहायता के लिए Aspose.PDF के दस्तावेज़ या फ़ोरम देखें।

## कार्यान्वयन मार्गदर्शिका

### समाप्ति तिथि के साथ PDF बनाना

#### अवलोकन

हम एक सरल पीडीएफ बनाएंगे और पीडीएफ फाइल में जावास्क्रिप्ट का उपयोग करके समाप्ति तिथि निर्धारित करेंगे। यह उपयोगकर्ताओं को दस्तावेज़ की समाप्ति तिथि के बारे में सूचित करेगा।

#### चरण-दर-चरण कार्यान्वयन

**1. दस्तावेज़ सेट अप करना**

इसका एक उदाहरण बनाकर शुरू करें `Document` क्लास, जो आपके पीडीएफ का प्रतिनिधित्व करता है:

```csharp
// दस्तावेज़ ऑब्जेक्ट को इंस्टैंशिएट करें
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

**2. पीडीएफ में सामग्री जोड़ना**

प्रदर्शन प्रयोजनों के लिए अपने दस्तावेज़ में एक पृष्ठ और पाठ जोड़ें:

```csharp
// पीडीएफ फाइल के पेज टू पेज संग्रह को जोड़ें
doc.Pages.Add();

// पृष्ठ ऑब्जेक्ट के पैराग्राफ़ संग्रह में पाठ अंश जोड़ें
doc.Pages[1].Paragraphs.Add(new TextFragment("Hello World..."));
```

**3. जावास्क्रिप्ट के साथ समाप्ति तर्क को लागू करना**

समाप्ति तिथि लागू करने के लिए पीडीएफ में जावास्क्रिप्ट का उपयोग करें:

```csharp
// पीडीएफ समाप्ति तिथि निर्धारित करने के लिए जावास्क्रिप्ट ऑब्जेक्ट बनाएं
JavascriptAction javaScript = new JavascriptAction(
    "var year=2023;"  // वर्तमान वर्ष का अद्यतन
    + "var month=10;"  // इच्छित समाप्ति माह निर्धारित करें
    + "today = new Date(); today = new Date(today.getFullYear(), today.getMonth());"
    + "expiry = new Date(year, month);"
    + "if (today.getTime() > expiry.getTime())"
    + "app.alert('The file is expired. You need a new one.');");

// जावास्क्रिप्ट को पीडीएफ खोलने की क्रिया के रूप में सेट करें
doc.OpenAction = javaScript;
```

**4. दस्तावेज़ को सहेजना**

अंत में, अपने दस्तावेज़ को निर्धारित समाप्ति तर्क के साथ सहेजें:

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
dataDir += "SetExpiryDate_out.pdf";
// पीडीएफ दस्तावेज़ सहेजें
doc.Save(dataDir);
Console.WriteLine(\nPDF expiry date setup successfully.\nFile saved at " + dataDir);
```

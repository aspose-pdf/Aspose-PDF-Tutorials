# .NET के लिए Aspose.PDF का उपयोग करके PDF टेक्स्ट में टूलटिप्स कैसे जोड़ें

## परिचय
दस्तावेज़ को अव्यवस्थित किए बिना अतिरिक्त जानकारी या संदर्भ प्रदान करके उपयोगकर्ता की सहभागिता बढ़ाने के लिए गतिशील और इंटरैक्टिव PDF दस्तावेज़ बनाना महत्वपूर्ण है। .NET के लिए Aspose.PDF के साथ, आप टेक्स्ट अंशों पर टूलटिप्स जैसी सुविधाओं के साथ आसानी से समृद्ध PDF बना सकते हैं। यह ट्यूटोरियल आपको .NET वातावरण में Aspose.PDF का उपयोग करके टूलटिप्स जोड़ने के बारे में बताएगा।

**आप क्या सीखेंगे:**
- एक सरल पीडीएफ दस्तावेज़ कैसे बनाएं और सेव करें।
- किसी मौजूदा पीडीएफ फाइल के भीतर विशिष्ट पाठ में टूलटिप्स जोड़ना।
- .NET के लिए Aspose.PDF में ButtonField का उपयोग करके इंटरैक्टिव तत्वों को कार्यान्वित करना।

आइए इन सुविधाओं को लागू करने से पहले आवश्यक शर्तों पर गौर करें!

## कार्यान्वयन मार्गदर्शिका
### फ़ीचर 1: नमूना PDF दस्तावेज़ बनाएँ और सहेजें
एक सरल पीडीएफ दस्तावेज़ बनाना हमारी यात्रा का पहला कदम है।

#### अवलोकन
यह अनुभाग दर्शाता है कि .NET के लिए Aspose.PDF का उपयोग करके टेक्स्ट के साथ एक बुनियादी PDF कैसे बनाया जाए। आप अपने दस्तावेज़ में पृष्ठ और टेक्स्ट अंश जोड़ना सीखेंगे।

**कार्यान्वयन के चरण:**
##### 1. अपना निर्देशिका पथ परिभाषित करें
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputFile = dataDir + "/Tooltip_out.pdf";
```
यहाँ, `dataDir` वह निर्देशिका है जहाँ आप अपनी पीडीएफ फाइलें सहेजना चाहते हैं, और `outputFile` पीडीएफ फाइल का नाम निर्दिष्ट करता है.

##### 2. नया दस्तावेज़ बनाएँ
```csharp
Document doc = new Document();
```
यह पंक्ति एक नया PDF दस्तावेज़ उदाहरण आरंभ करती है।

##### 3. पृष्ठ और पाठ अंश जोड़ें
```csharp
doc.Pages.Add().Paragraphs.Add(new TextFragment("Move the mouse cursor here to display a tooltip"));
doc.Pages[1].Paragraphs.Add(new TextFragment("Move the mouse cursor here to display a very long tooltip"));
```
हम प्रत्येक पृष्ठ पर पाठ के अंश जोड़ते हैं। इन अंशों का उपयोग बाद में टूलटिप्स के लिए किया जाएगा।

##### 4. अपना दस्तावेज़ सहेजें
```csharp
doc.Save(outputFile);
```
यह आदेश आपके दस्तावेज़ को निर्दिष्ट स्थान पर सहेजता है।

### फ़ीचर 2: पीडीएफ टेक्स्ट फ़्रैगमेंट में बटन फ़ील्ड के साथ टूलटिप जोड़ें
इंटरैक्टिव टूलटिप्स जोड़कर अपने पीडीएफ को बेहतर बनाना रोमांचक हो जाता है!

#### अवलोकन
इस अनुभाग में, हम Aspose.PDF का उपयोग करके मौजूदा PDF में टेक्स्ट अंशों में टूलटिप्स जोड़ते हैं। इसमें अदृश्य बटन बनाना शामिल है जो माउस घुमाने पर टूलटिप्स को ट्रिगर करते हैं।

**कार्यान्वयन के चरण:**
##### 1. मौजूदा दस्तावेज़ खोलें
```csharp
Document document = new Document("YOUR_DOCUMENT_DIRECTORY/Tooltip_out.pdf");
```
टूलटिप्स जोड़ने के लिए अपना पहले से बनाया गया दस्तावेज़ लोड करें.

##### 2. विशिष्ट वाक्यांशों वाले पाठ अंश खोजें
```csharp
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Move the mouse cursor here to display a tooltip");
document.Pages.Accept(absorber);
TextFragmentCollection textFragments = absorber.TextFragments;
```
यहाँ, `TextFragmentAbsorber` आपके पीडीएफ में विशिष्ट वाक्यांशों की खोज करता है।

##### 3. बटन फ़ील्ड बनाएँ और टूलटिप्स सेट करें
```csharp
foreach (TextFragment fragment in textFragments)
{
    ButtonField field = new ButtonField(fragment.Page, fragment.Rectangle);
    field.AlternateName = "Tooltip for text.";
    document.Form.Add(field);
}
```
यह लूप एक बनाता है `ButtonField` प्रत्येक पाए गए पाठ खंड के लिए और उसका टूलटिप सेट करता है।

##### 4. लंबे समय तक दोहराएँ टूलटिप
```csharp
absorber = new TextFragmentAbsorber("Move the mouse cursor here to display a very long tooltip");
document.Pages.Accept(absorber);
textFragments = absorber.TextFragments;

foreach (TextFragment fragment in textFragments)
{
    ButtonField field = new ButtonField(fragment.Page, fragment.Rectangle);
    field.AlternateName = "Lorem ipsum dolor sit amet..."; // अपना लंबा टूलटिप टेक्स्ट यहां जोड़ें
    document.Form.Add(field);
}
```
लंबे टूलटिप्स की आवश्यकता वाले अन्य वाक्यांशों के लिए भी यही प्रक्रिया दोहराएं।

##### 5. अपना अपडेट किया गया दस्तावेज़ सहेजें
```csharp
document.Save(outputFile);
```
अंत में, अपडेट किए गए पीडीएफ को टूलटिप्स के साथ सेव कर लें।

## व्यावहारिक अनुप्रयोगों
इंटरैक्टिव पीडीएफ़ को लागू करने से विभिन्न परिदृश्यों में उपयोगकर्ता अनुभव को बेहतर बनाया जा सकता है:
1. **शैक्षिक सामग्री**: अतिरिक्त स्पष्टीकरण या परिभाषाएँ प्रदान करने के लिए टूलटिप्स का उपयोग करें।
2. **उपयोगकर्ता मैनुअल**त्वरित मार्गदर्शन के लिए टूलटिप्स के साथ निर्देशों के मुख्य भागों को हाइलाइट करें।
3. **रिपोर्ट और प्रस्तुतियाँ**: मुख्य विषय-वस्तु पर अधिक ज़ोर डाले बिना प्रासंगिक जानकारी जोड़ें।
4. **उत्पाद सूची**: डिज़ाइन को साफ़ रखते हुए, होवर पर विस्तृत उत्पाद विवरण दें।
5. **फार्म**: जब उपयोगकर्ता फ़ॉर्म फ़ील्ड के साथ इंटरैक्ट करते हैं तो सहायता पाठ या उदाहरण प्रदान करें.

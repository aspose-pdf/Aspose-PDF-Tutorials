## परिचय

क्या आप रेडियो बटन जैसे इंटरैक्टिव तत्व जोड़कर अपने PDF फ़ॉर्म को बेहतर बनाना चाहते हैं? गतिशील, उपयोगकर्ता-अनुकूल PDF बनाने से डेटा संग्रह दक्षता और सटीकता में उल्लेखनीय सुधार हो सकता है। इस गाइड में, हम आपको दिखाएंगे कि .NET लाइब्रेरी के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ों में रेडियो बटन कैसे लागू करें। यह शक्तिशाली उपकरण अपने मजबूत API के साथ जटिल कार्यों को सरल बनाता है, जिससे यह उन डेवलपर्स के लिए एक आदर्श विकल्प बन जाता है जो अपने अनुप्रयोगों में परिष्कृत फ़ॉर्म कार्यक्षमताओं को एकीकृत करना चाहते हैं।

**आप क्या सीखेंगे:**

- .NET के लिए Aspose.PDF को कैसे सेट अप और इंस्टॉल करें
- C# का उपयोग करके स्क्रैच से PDF दस्तावेज़ बनाना
- अपने PDF में रेडियो बटन फ़ील्ड जोड़ना
- रेडियो बटन फ़ील्ड के भीतर विकल्प कॉन्फ़िगर करना
- संशोधित PDF को सहेजना और निर्यात करना

आइए जानें कि आप आसानी से इंटरैक्टिव पीडीएफ फॉर्म कैसे बना सकते हैं।

## कार्यान्वयन मार्गदर्शिका

अब जब आपने सब कुछ सेट कर लिया है, तो आइए पीडीएफ में रेडियो बटन जोड़ने की सुविधा को क्रियान्वित करें।

### रेडियो बटन के साथ नया दस्तावेज़ बनाना

**अवलोकन:**
हम एक रिक्त पीडीएफ दस्तावेज़ बनाकर शुरुआत करेंगे और फिर एक पृष्ठ जोड़ेंगे जहां हम अपने रेडियो बटन फ़ील्ड रख सकते हैं।

#### चरण 1: नया दस्तावेज़ आरंभ करें

आवश्यक नामस्थानों के साथ अपना प्रोजेक्ट सेट अप करके आरंभ करें:

```csharp
using System;
using Aspose.Pdf;
```

इसका एक उदाहरण बनाएं `Document` क्लास, जो पीडीएफ फाइल का प्रतिनिधित्व करता है:

```csharp
// नया दस्तावेज़ बनाएँ
Document doc = new Document();
Page page = doc.Pages.Add(); // दस्तावेज़ में नया पृष्ठ जोड़ें
```

#### चरण 2: रेडियो बटन फ़ील्ड जोड़ना

अब हम अपने नवनिर्मित पीडीएफ पेज में रेडियो बटन फ़ील्ड जोड़ेंगे।

**रेडियो बटन फ़ील्ड बनाएँ और कॉन्फ़िगर करें:**

```csharp
// पहले पृष्ठ पर रेडियोबटनफील्ड बनाएं
RadioButtonField field = new RadioButtonField(page);
field.Rect = new Aspose.Pdf.Rectangle(40, 650, 100, 720); // स्थिति और आकार निर्धारित करें
field.PartialName = "NewField"; // संदर्भ के लिए नाम निर्दिष्ट करें
```

#### चरण 3: रेडियो बटन विकल्प जोड़ें

अपने रेडियो बटन फ़ील्ड में उपलब्ध विकल्प परिभाषित करें:

```csharp
// रेडियो बटन विकल्प बनाएं और गुण सेट करें
RadioButtonOptionField opt1 = new RadioButtonOptionField();
opt1.Rect = new Aspose.Pdf.Rectangle(40, 650, 60, 670);
opt1.OptionName = "Item1"; // विकल्प लेबल
opt1.Border = new Border(opt1) { Width = 1 };
opt1.Characteristics.Border = System.Drawing.Color.Black;

RadioButtonOptionField opt2 = new RadioButtonOptionField();
opt2.Rect = new Aspose.Pdf.Rectangle(60, 670, 80, 690);
opt2.OptionName = "Item2";
opt2.Border = new Border(opt2) { Width = 1 };
opt2.Characteristics.Border = System.Drawing.Color.Black;

RadioButtonOptionField opt3 = new RadioButtonOptionField();
opt3.Rect = new Aspose.Pdf.Rectangle(80, 690, 100, 710);
opt3.OptionName = "Item3";
opt3.Border = new Border(opt3) { Width = 1 };
opt3.Characteristics.Border = System.Drawing.Color.Black;

// रेडियो बटन फ़ील्ड में विकल्प जोड़ें
field.Add(opt1);
field.Add(opt2);
field.Add(opt3);

// फॉर्म में रेडियोबटनफील्ड जोड़ें
doc.Form.Add(field);
```

#### चरण 4: दस्तावेज़ सहेजें

अंत में, अपने दस्तावेज़ को नए जोड़े गए रेडियो बटनों के साथ सहेजें:

```csharp
string dataDir = "path/to/your/directory/";
dataDir += "CreateDoc_out.pdf"; // आउटपुट पथ परिभाषित करें

// दस्तावेज़ सहेजें
doc.Save(dataDir);

Console.WriteLine("\nNew doc with 3 items radio button created successfully.\nFile saved at " + dataDir);
```

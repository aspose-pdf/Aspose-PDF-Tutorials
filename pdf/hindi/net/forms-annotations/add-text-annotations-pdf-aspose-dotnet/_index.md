## कार्यान्वयन मार्गदर्शिका

### पीडीएफ दस्तावेज़ में टेक्स्ट एनोटेशन जोड़ना

आइए टेक्स्ट एनोटेशन जोड़ने की प्रक्रिया को प्रबंधनीय चरणों में विभाजित करें।

#### अपना पीडीएफ दस्तावेज़ लोड करें

किसी मौजूदा PDF दस्तावेज़ को लोड करके शुरू करें। वह निर्देशिका निर्दिष्ट करें जहाँ आपके इनपुट और आउटपुट दस्तावेज़ संग्रहीत किए जाएँगे।

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/AddAnnotation.pdf");
```

#### टेक्स्ट एनोटेशन ऑब्जेक्ट बनाएँ

इसके बाद, एक बनाएं `TextAnnotation` किसी विशिष्ट पृष्ठ पर शीर्षक, विषय और स्थान जैसे वांछित गुणों के साथ ऑब्जेक्ट।

```csharp
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

#### बॉर्डर गुण कॉन्फ़िगर करें

अपने एनोटेशन के दृश्य आकर्षण को बढ़ाने के लिए बॉर्डर गुण सेट करें।

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1); // बॉर्डर के लिए डैश शैली सेट करें.
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

#### दस्तावेज़ में एनोटेशन जोड़ें

अंत में, अपने कॉन्फ़िगर किए गए एनोटेशन को किसी विशिष्ट पृष्ठ के एनोटेशन संग्रह में जोड़ें और संशोधित दस्तावेज़ को सहेजें।

```csharp
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/AddAnnotation_out.pdf");
```

### समस्या निवारण युक्तियों

- सुनिश्चित करें कि इनपुट/आउटपुट निर्देशिकाओं के लिए पथ सही ढंग से निर्दिष्ट किए गए हैं।
- लोड करने से पहले सत्यापित करें कि आपका PDF दस्तावेज़ दिए गए पथ पर मौजूद है।
- यह सुनिश्चित करने के लिए कि एनोटेशन गुण आपकी आवश्यकताओं को पूरा करते हैं, उनकी दोबारा जांच करें।

# Aspose.PDF .NET के साथ PDF में मापित रेखा एनोटेशन कैसे जोड़ें

## परिचय

क्या आपको कभी भी PDF दस्तावेज़ों को सटीक माप के साथ एनोटेट करने की ज़रूरत पड़ी है? चाहे वह तकनीकी रेखाचित्रों, वास्तुशिल्प योजनाओं या किसी भी दस्तावेज़ के लिए हो जहाँ सटीकता महत्वपूर्ण है, मापी गई रेखा एनोटेशन जोड़ने से स्पष्टता और संचार में काफी वृद्धि हो सकती है। यह ट्यूटोरियल आपको शक्तिशाली Aspose.PDF .NET लाइब्रेरी का उपयोग करके आसानी से अपनी PDF फ़ाइलों में माप क्षमताओं के साथ एक लाइन एनोटेशन जोड़ने के लिए मार्गदर्शन करेगा।

**आप क्या सीखेंगे:**

- Aspose.PDF .NET के साथ काम करने के लिए अपना वातावरण कैसे सेट करें
- पीडीएफ दस्तावेज़ में माप के साथ लाइन एनोटेशन बनाने की प्रक्रिया
- लीडर लाइन, मापन इकाइयाँ और कैप्शन डिस्प्ले जैसी विभिन्न सेटिंग्स कॉन्फ़िगर करना

आइए इस सुविधा को लागू करने से पहले आवश्यक पूर्वापेक्षाओं पर गौर करें।

## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में, हम Aspose.PDF .NET का उपयोग करके एक PDF दस्तावेज़ में मापी गई लाइन एनोटेशन जोड़ने की प्रक्रिया को विभाजित करेंगे।

### माप के साथ लाइन एनोटेशन जोड़ना

#### अवलोकन

लाइन एनोटेशन जोड़ने से आप अपने पीडीएफ़ में विशिष्ट अनुभागों को चिह्नित कर सकते हैं और दूरी माप सकते हैं। यह सुविधा तकनीकी दस्तावेज़ीकरण के लिए विशेष रूप से उपयोगी है जहाँ परिशुद्धता मायने रखती है।

#### कार्यान्वयन चरण

1. **दस्तावेज़ लोड करें**

   उस मौजूदा पीडीएफ दस्तावेज़ को लोड करके आरंभ करें जिसमें आप एनोटेशन जोड़ना चाहते हैं।

   ```csharp
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY/input.pdf";
   Document doc = new Document(dataDir);
   ```

2. **एनोटेशन क्षेत्र परिभाषित करें**

   अपने पृष्ठ पर वह आयताकार क्षेत्र निर्दिष्ट करें जहां एनोटेशन दिखाई देगा.

   ```csharp
   Rectangle rect = new Rectangle(260, 630, 451, 662); // एनोटेशन के लिए निर्देशांक परिभाषित करें
   ```

3. **लाइन एनोटेशन बनाएं**

   एक बनाने के `LineAnnotation` परिभाषित आयत क्षेत्र के भीतर प्रारंभ और अंत बिंदु वाली वस्तु।

   ```csharp
   LineAnnotation line = new LineAnnotation(doc.Pages[1], rect, new Point(266, 657), new Point(446, 656));
   ```

4. **उपस्थिति कॉन्फ़िगर करें**

   एनोटेशन के लिए रंग, मुख्य पंक्तियां और शैलियाँ निर्धारित करें।

   ```csharp
   line.Color = Color.Red; // एनोटेशन का रंग लाल पर सेट करता है
   line.LeaderLine = -15; // आरंभ बिंदु से लीडर लाइन ऑफसेट
   line.LeaderLineExtension = 5; // लीडर लाइन की विस्तार लंबाई
   line.StartingStyle = LineEnding.OpenArrow;
   line.EndingStyle = LineEnding.OpenArrow;
   ```

5. **माप विवरण सेट करें**

   का उपयोग करो `Measure` माप विवरण निर्दिष्ट करने पर आपत्ति।

   ```csharp
   line.Measure = new Measure(line);
   line.Measure.DistanceFormat.Add(new Measure.NumberFormat(line.Measure));
   line.Measure.DistanceFormat[1].UnitLabel = "mm"; // माप के लिए इकाई लेबल सेट करें
   ```

6. **कैप्शन जोड़ें और सहेजें**

   माप प्रदर्शित करने के लिए एक कैप्शन जोड़ें और अपने दस्तावेज़ को सहेजें.

   ```csharp
   line.Contents = "155 mm";
   line.ShowCaption = true;
   line.CaptionPosition = CaptionPosition.Top;

   doc.Pages[1].Annotations.Add(line); // पृष्ठ 1 पर टिप्पणी जोड़ता है

   string outputDir = @"YOUR_OUTPUT_DIRECTORY/UseMeasureWithLineAnnotation_out.pdf";
   doc.Save(outputDir);
   ```

1. **तकनीकी दस्तावेज़ीकरण**सटीक माप दिखाकर इंजीनियरिंग चित्रों में स्पष्टता बढ़ाना।
2. **वास्तुकला योजनाएँ**निर्माण प्रयोजनों के लिए सटीक दूरियों के साथ ब्लूप्रिंट की व्याख्या करना।
3. **शिक्षण सामग्री**पाठ्यपुस्तकों में आरेखों और चित्रों में माप संबंधी टिप्पणियाँ जोड़ना।

## अक्सर पूछे जाने वाले प्रश्न अनुभाग

1. **मैं .NET के लिए Aspose.PDF कैसे स्थापित करूं?**
   - आप अपने प्रोजेक्ट में Aspose.PDF जोड़ने के लिए .NET CLI, पैकेज मैनेजर कंसोल या NuGet पैकेज मैनेजर UI का उपयोग कर सकते हैं।

2. **क्या मैं एनोटेशन में माप की इकाई बदल सकता हूँ?**
   - हाँ, संशोधित करें `UnitLabel` की संपत्ति `Measure.NumberFormat` इंच जैसी विभिन्न इकाइयाँ निर्धारित करने के लिए ऑब्जेक्ट (`"in"`), सेंटीमीटर (`"cm"`), वगैरह।

3. **यदि मेरा दस्तावेज़ सही ढंग से लोड नहीं होता तो क्या होगा?**
   - अपने फ़ाइल पथ को सत्यापित करें और सुनिश्चित करें कि Aspose.PDF आपके प्रोजेक्ट में ठीक से स्थापित है।

4. **मैं एनोटेशन की लाइन शैली को कैसे अनुकूलित करूँ?**
   - जैसे गुणों का उपयोग करें `StartingStyle` और `EndingStyle` यह समायोजित करने के लिए कि रेखाएँ अपने अंतिम बिंदु पर कैसे दिखाई देती हैं।

5. **क्या पीडीएफ को सहेजने से पहले परिवर्तनों का पूर्वावलोकन करने का कोई तरीका है?**
   - Aspose.PDF में अंतर्निहित पूर्वावलोकन सुविधा नहीं है, लेकिन आप परीक्षण प्रयोजनों के लिए मध्यवर्ती संस्करणों को सहेज सकते हैं।

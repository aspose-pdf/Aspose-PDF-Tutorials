## कार्यान्वयन गाइड (H2)
हम इस प्रक्रिया को प्रबंधनीय चरणों में विभाजित करेंगे।

### PDF दस्तावेज़ बनाना (H3)
**अवलोकन:** एक नया दस्तावेज़ बनाकर और उसमें एक पृष्ठ जोड़कर आरंभ करें।
```csharp
Document pdf = new Aspose.Pdf.Document();
Page sec1 = pdf.Pages.Add(); // नया पेज जोड़ें
```
- `Aspose.Pdf.Document`: आपके संपूर्ण PDF का प्रतिनिधित्व करता है.
- `Pages.Add()`: एक नया पृष्ठ जोड़ता है जहाँ आप सामग्री रख सकते हैं.

### पाठ खंड जोड़ना (H3)
**अवलोकन:** रंग और फ़ॉन्ट आकार जैसे स्वरूपण विकल्पों के साथ पाठ अंश सम्मिलित करें.
```csharp
TextFragment t1 = new Aspose.Pdf.Text.TextFragment("paragraph 3 segment");
sec1.Paragraphs.Add(t1);

t1.TextState.ForegroundColor = Color.Red; // पाठ का रंग लाल पर सेट करें
t1.TextState.FontSize = 12;                // फ़ॉन्ट आकार 12 पर सेट करें
```
- `TextFragment`: पाठ के एक ब्लॉक का प्रतिनिधित्व करता है.
- `TextState.ForegroundColor`: पाठ का रंग सेट करता है.
- `TextState.FontSize`: फ़ॉन्ट आकार को नियंत्रित करता है.

### छवियाँ सम्मिलित करना (H3)
**अवलोकन:** अपने पीडीएफ दस्तावेज़ में चित्र जोड़ें, जिससे इसकी विषय-वस्तु दृश्यात्मक रूप से समृद्ध हो जाए।
```csharp
Image image1 = new Aspose.Pdf.Image();
image1.File = "path\to\your\test_image.png"; // छवि पथ निर्दिष्ट करें
sec1.Paragraphs.Add(image1);
```
- `Aspose.Pdf.Image`: आपके दस्तावेज़ में एक छवि का प्रतिनिधित्व करता है.
- `image1.File`: छवि के लिए फ़ाइल पथ सेट करता है.

### फ़्लोटिंग बॉक्स जोड़ना (H3)
**अवलोकन:** किसी पृष्ठ पर सामग्री को प्रभावी ढंग से व्यवस्थित करने और स्थान देने के लिए फ्लोटिंग बॉक्स का उपयोग करें।
```csharp
FloatingBox box1 = new Aspose.Pdf.FloatingBox(117, 21);
box1.Left = -4; // बाएं किनारे से स्थिति निर्धारण
box1.Top = -4;  // शीर्ष किनारे से स्थिति निर्धारण
sec1.Paragraphs.Add(box1);

box1.Paragraphs.Add(image1); // फ़्लोटिंग बॉक्स में एक छवि जोड़ें
```
- `FloatingBox`: सामग्री को व्यवस्थित करने के लिए एक कंटेनर.
- स्थिति विशेषताएँ (`Left`, `Top`): पृष्ठ पर बॉक्स की स्थिति निर्धारित करें.

### PDF दस्तावेज़ को सहेजना (H3)
**अवलोकन:** अंत में, अपने दस्तावेज़ को फ़ाइल में सहेजें।
```csharp
pdf.Save("path\to\your\CreateMultiLayerPdf_out.pdf");
```
- `Document.Save()`: अंतिम आउटपुट को डिस्क पर लिखता है।

# .NET के लिए Aspose.PDF का उपयोग करके टेक्स्ट और छवियों के साथ व्यावसायिक PDF बनाएं

## परिचय

क्या आप अपने .NET एप्लीकेशन से सीधे प्रोफेशनल-क्वालिटी वाली PDF फ़ाइलें बनाकर अपने दस्तावेज़ निर्माण प्रक्रिया को सरल बनाना चाहते हैं? चाहे वह एक सुंदर स्वरूपित PDF में गतिशील टेक्स्ट, इमेज या दोनों जोड़ना हो, .NET के लिए Aspose.PDF में महारत हासिल करने से आपकी विकास क्षमताएँ काफ़ी हद तक बढ़ सकती हैं। यह गाइड आपको .NET के लिए Aspose.PDF का उपयोग करके टेक्स्ट और इमेज के साथ समृद्ध PDF दस्तावेज़ बनाने के लिए आवश्यक चरणों से परिचित कराएगी।

**आप क्या सीखेंगे:**
- अपने प्रोजेक्ट में .NET के लिए Aspose.PDF कैसे सेट करें।
- पीडीएफ दस्तावेज़ में पाठ जोड़ने की तकनीकें।
- अपने PDF में फ्लोटिंग बॉक्स और छवियों को शामिल करने के तरीके।
- अपने दस्तावेज़ों को अनुकूलित करने के लिए प्रमुख कॉन्फ़िगरेशन विकल्प.

इन कौशलों के साथ, आप अपने आवेदन की ज़रूरतों के अनुरूप पेशेवर-गुणवत्ता वाले PDF तैयार करने में सक्षम होंगे। शुरू करने से पहले आइए आवश्यक शर्तों पर नज़र डालें!

## कार्यान्वयन मार्गदर्शिका

यह अनुभाग .NET के लिए Aspose.PDF का उपयोग करके पाठ और छवियों के साथ PDF बनाने की प्रक्रिया को प्रबंधनीय चरणों में विभाजित करता है।

### पाठ के साथ दस्तावेज़ बनाना

**अवलोकन:**
टेक्स्ट जोड़कर एक सरल पीडीएफ दस्तावेज़ बनाना बहुत आसान है। यह सुविधा आपको प्रोग्रामेटिक रूप से दस्तावेज़ बनाने देती है, जिसे विशिष्ट स्टाइलिंग और फ़ॉर्मेटिंग शामिल करने के लिए अनुकूलित किया जा सकता है।

#### चरण-दर-चरण कार्यान्वयन:
1. **दस्तावेज़ आरंभ करें**
   इसका एक उदाहरण बनाकर शुरू करें `Document` कक्षा।
   ```csharp
   Document doc = new Document();
   ```
2. **पेज जोड़ें**
   दस्तावेज़ में वे पृष्ठ जोड़े जाते हैं जहां आप अपनी सामग्री रखेंगे.
   ```csharp
   Page page = doc.getPages().add();
   ```
3. **टेक्स्ट फ़्रैगमेंट बनाएँ और कॉन्फ़िगर करें**
   यहाँ, हम एक बनाते हैं `TextFragment` जिसमें हमारा वांछित टेक्स्ट, रंग और आकार जैसी विशिष्ट स्टाइलिंग सेटिंग्स शामिल होती हैं।
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;

   TextFragment t1 = new TextFragment("paragraph 3 segment");
   t1.getTextState().setForegroundColor(Color.RED);
   t1.setTextIsInLineParagraph(true);
   t1.getTextState().setFontSize(12);
   ```
4. **पृष्ठ पर पाठ जोड़ें**
   अंत में, पाठ अंश को पृष्ठ के पैराग्राफ संग्रह में जोड़ें।
   ```csharp
   page.getParagraphs().add(t1);
   doc.save("YOUR_DOCUMENT_DIRECTORY/CreateTextPDF_out.pdf");
   ```

### टेक्स्ट के साथ फ़्लोटिंगबॉक्स बनाना

**अवलोकन:**
फ्लोटिंग बॉक्स पीडीएफ के भीतर सामग्री को व्यवस्थित करने का एक उन्नत तरीका प्रदान करते हैं, जिससे गतिशील लेआउट की सुविधा मिलती है।

#### चरण-दर-चरण कार्यान्वयन:
1. **फ़्लोटिंग बॉक्स बनाएँ**
   परिभाषित करें `FloatingBox` विशिष्ट आयामों के साथ.
   ```csharp
   FloatingBox textFloatingBox = new FloatingBox(117, 21);
   ```
2. **स्थिति और Z-इंडेक्स कॉन्फ़िगर करें**
   पृष्ठ पर इसकी स्थिति को अनुकूलित करें.
   ```csharp
   textFloatingBox.setZIndex(1);
   textFloatingBox.setLeft(-4);
   textFloatingBox.setTop(-4);
   ```
3. **पेज पर फ़्लोटिंग बॉक्स जोड़ें**
   इसे दस्तावेज़ की संरचना में रखें.
   ```csharp
   page.getParagraphs().add(textFloatingBox);
   textFloatingBox.getParagraphs().add(t1); // पहले से निर्मित पाठ अंश जोड़ें
   doc.save("YOUR_DOCUMENT_DIRECTORY/TextFloatingBox_out.pdf");
   ```

### फ़्लोटिंगबॉक्स में छवि जोड़ना

**अवलोकन:**
फ्लोटिंग बॉक्स में छवियों को शामिल करने से लचीले डिजाइन और लेआउट विकल्प उपलब्ध होते हैं।

#### चरण-दर-चरण कार्यान्वयन:
1. **छवि आरंभ करें**
   फ़ाइल पथ के साथ अपनी छवि ऑब्जेक्ट तैयार करें.
   ```csharp
   using Aspose.Pdf;

   Image image = new Image();
   image.setFile("YOUR_DOCUMENT_DIRECTORY/test_image.png");
   ```
2. **छवि के लिए फ़्लोटिंग बॉक्स बनाएँ और कॉन्फ़िगर करें**
   पाठ के समान, एक फ्लोटिंग बॉक्स को परिभाषित करें लेकिन एक छवि के लिए अनुकूलित करें।
   ```csharp
   FloatingBox imageFloatingBox = new FloatingBox(117, 21);
   page.getParagraphs().add(imageFloatingBox);

   imageFloatingBox.setLeft(-4);
   imageFloatingBox.setTop(-4);
   imageFloatingBox.setZIndex(2);
   ```
3. **फ़्लोटिंग बॉक्स में छवि जोड़ें**
   अपनी छवि को फ्लोटिंग बॉक्स में रखें और उसे सेव करें।
   ```csharp
   imageFloatingBox.getParagraphs().add(image);
   doc.save("YOUR_DOCUMENT_DIRECTORY/ImageInFloatingBox_out.pdf");
   ```

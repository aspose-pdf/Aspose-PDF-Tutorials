## कार्यान्वयन मार्गदर्शिका

### अवलोकन

इस अनुभाग में, हम PDF दस्तावेज़ में डिजिटल हस्ताक्षरों से छवियाँ निकालने के बारे में जानेंगे। यह विशेष रूप से तब उपयोगी होता है जब आपको हस्ताक्षर विवरणों को अलग से सत्यापित या संग्रहीत करने की आवश्यकता होती है।

#### पीडीएफ दस्तावेज़ लोड करें

सबसे पहले, Aspose.PDF का उपयोग करके अपनी PDF फ़ाइल लोड करें `Document` कक्षा:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFilePath = Path.Combine(dataDir, "DigitallySign.pdf");

// नया दस्तावेज़ इंस्टेंस बनाएँ
Document doc = new Document(inputFilePath);
```

#### हस्ताक्षरों से छवियाँ निकालें

पीडीएफ हस्ताक्षरों में सन्निहित छवियों को निकालने का तरीका यहां दिया गया है:

```csharp
using (PdfFileSignature signature = new PdfFileSignature(doc))
{
    // जाँच करें कि दस्तावेज़ में कोई डिजिटल हस्ताक्षर है या नहीं
    if (signature.ContainsSignature())
    {
        foreach (string sigName in signature.GetSignNames())
        {
            string outputFilePath = Path.Combine(dataDir, "ExtractImages_out.jpg");

            using (Stream imageStream = signature.ExtractImage(sigName))
            {
                if (imageStream != null)
                {
                    // स्ट्रीम को इमेज ऑब्जेक्ट में बदलें
                    using (System.Drawing.Image image = Bitmap.FromStream(imageStream))
                    {
                        // निकाली गई छवि को JPEG फ़ाइल के रूप में सहेजें
                        image.Save(outputFilePath, System.Drawing.Imaging.ImageFormat.Jpeg);
                    }
                }
            }
        }
    }
}
```

**स्पष्टीकरण:**
- **`PdfFileSignature`:** यह वर्ग पीडीएफ हस्ताक्षरों पर परिचालन को सुगम बनाता है।
- **`ContainsSignature()`:** जाँचता है कि दस्तावेज़ में कोई डिजिटल हस्ताक्षर मौजूद है या नहीं।
- **`ExtractImage(sigName)`:** निर्दिष्ट हस्ताक्षर से छवि डेटा निकालता है.

### समस्या निवारण युक्तियों

- सुनिश्चित करें कि आपके फ़ाइल पथ सही हैं; अन्यथा, आपको समस्या का सामना करना पड़ेगा `FileNotFoundException`.
- यदि कोई छवि नहीं निकाली गई है, तो सत्यापित करें कि पीडीएफ में वास्तव में उसके हस्ताक्षरों में एम्बेडेड छवियां मौजूद हैं।

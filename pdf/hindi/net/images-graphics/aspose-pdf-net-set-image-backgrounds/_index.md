## कार्यान्वयन गाइड (H2)

### छवि को पृष्ठभूमि के रूप में सेट करें (H2)

यह अनुभाग आपको अपने PDF में छवि पृष्ठभूमि सेट करने में मार्गदर्शन करता है।

#### चरण 1: नया दस्तावेज़ बनाएँ

एक बनाकर शुरू करें `Document` वस्तु:

```csharp
// दस्तावेज़ आरंभ करें
document doc = new Document();
```

यह स्निपेट एक नया उदाहरण आरंभ करता है `Document` क्लास, जो आपकी पीडीएफ फाइल का प्रतिनिधित्व करता है।

#### चरण 2: दस्तावेज़ में एक पृष्ठ जोड़ें

वह पृष्ठ जोड़ें जहां आप पृष्ठभूमि छवि सेट करेंगे:

```csharp
Page page = doc.Pages.Add();
```

यहाँ, हम एक रिक्त पृष्ठ जोड़ते हैं `Pages.Add()` तरीका।

#### चरण 3: बैकग्राउंड आर्टिफैक्ट बनाएं और कॉन्फ़िगर करें

The `BackgroundArtifact` क्लास का उपयोग Aspose.PDF में बैकग्राउंड सेट करने के लिए किया जाता है। इसे कॉन्फ़िगर करने का तरीका यहां बताया गया है:

```csharp
BackgroundArtifact background = new BackgroundArtifact();
background.BackgroundImage = File.OpenRead(@"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg");
page.Artifacts.Add(background);
```

- **क्यों?** The `BackgroundArtifact` क्लास को पीडीएफ पेजों में बैकग्राउंड को संभालने के लिए डिज़ाइन किया गया है। इस आर्टिफैक्ट को जोड़ने से यह सुनिश्चित होता है कि छवि बैकग्राउंड के रूप में दिखाई दे।
- प्रतिस्थापित करें `"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg"` अपनी इच्छित पृष्ठभूमि छवि पथ के साथ।

#### चरण 4: अपना दस्तावेज़ सहेजें

अंत में, अपना दस्तावेज़ सहेजें:

```csharp
string dataDir = @"YOUR_OUTPUT_DIRECTORY\ImageAsBackground_out.pdf";
doc.Save(dataDir);
```

यह कोड आपकी पीडीएफ फाइल को निर्दिष्ट आउटपुट डायरेक्टरी और फ़ाइल नाम के साथ सहेजता है। `"YOUR_OUTPUT_DIRECTORY"` अपने इच्छित स्थान के साथ सहेजें.

## कार्यान्वयन मार्गदर्शिका

### पीडीएफ पृष्ठ का आकार बदलें

यह अनुभाग दर्शाता है कि .NET के लिए Aspose.PDF का उपयोग करके चयनित पृष्ठों के पृष्ठ आकार को कैसे बदला जाए।

#### अवलोकन

हम उपयोग करेंगे `PdfPageEditor` Aspose.Pdf.Facades से PDF दस्तावेज़ को उसके पृष्ठ आकार में परिवर्तन करके संशोधित करें।

**1. लाइब्रेरीज़ आयात करें**

सबसे पहले, सुनिश्चित करें कि आपके पास आवश्यक नामस्थान शामिल हैं:

```csharp
using System;
using Aspose.Pdf.Facades;
```

**2. PdfPageEditor प्रारंभ करें**

इसका एक उदाहरण बनाएं `PdfPageEditor` अपनी PDF फ़ाइल के साथ काम करने के लिए:

```csharp
PdfPageEditor pEdit = new PdfPageEditor();
```

**3. पीडीएफ फाइल को बाइंड करें**

उपयोग `BindPdf()` संपादक में पीडीएफ फाइल लोड करने की विधि:

```csharp
pEdit.BindPdf("YOUR_DOCUMENT_DIRECTORY/FilledForm.pdf");
```
"YOUR_DOCUMENT_DIRECTORY" को अपने वास्तविक पथ से प्रतिस्थापित करें।

**4. पृष्ठ निर्दिष्ट करें और आकार बदलें**

निर्धारित करें कि किन पृष्ठों को संशोधित करना है और उनका आकार निर्धारित करें `PageSize` एनम:

```csharp
// केवल प्रथम पृष्ठ (सूची 1) पर प्रक्रिया करें
pEdit.ProcessPages = new int[] { 1 };
pEdit.PageSize = PageSize.PageLetter;
```
यहां, हम पहले पृष्ठ का चयन करते हैं और इसका आकार "पत्र" में बदलते हैं।

**5. संशोधित पीडीएफ को सहेजें**

परिवर्तन करने के बाद, अपने PDF को किसी भिन्न नाम से सेव करें:

```csharp
pEdit.Save("YOUR_OUTPUT_DIRECTORY/ChangePageSizes_out.pdf");
```
"YOUR_OUTPUT_DIRECTORY" को उस स्थान से प्रतिस्थापित करें जहाँ आप संशोधित फ़ाइल को सहेजना चाहते हैं।

#### परिवर्तन सत्यापित करें

पुनः बाइंड करें और जांचें कि क्या पृष्ठ का आकार सही ढंग से अपडेट किया गया है:

```csharp
pEdit.BindPdf("YOUR_DOCUMENT_DIRECTORY/FilledForm.pdf");
PageSize size = pEdit.GetPageSize(1);
Console.WriteLine($"New Page Size: {size}");
```

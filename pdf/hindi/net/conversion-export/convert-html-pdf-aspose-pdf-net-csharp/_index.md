# Aspose.PDF का उपयोग करके C# में HTML को PDF में बदलें: एक संपूर्ण गाइड

## क्रेडेंशियल के साथ HTTP अनुरोध करना
### अवलोकन
इस अनुभाग में, हम दिखाएंगे कि C# का उपयोग करके प्रमाणित HTTP अनुरोध कैसे किया जाता है। इसमें क्रेडेंशियल सेट करना और सर्वर प्रतिक्रियाओं को संभालना शामिल है।
### चरण-दर-चरण कार्यान्वयन
**वेब अनुरोध बनाना**
```csharp
using System.Net;

// URL के लिए अनुरोध बनाएँ.
WebRequest request = WebRequest.Create("http://my.signchart.com/Report/PrintBook.asp?ProjectGuid=6FB9DBB0-");
```
**क्रेडेंशियल सेट करना**
यदि आपके सर्वर को प्रमाणीकरण की आवश्यकता है, तो क्रेडेंशियल इस प्रकार सेट करें:
```csharp
// डिफ़ॉल्ट नेटवर्क क्रेडेंशियल सेट करें
equest.Credentials = CredentialCache.DefaultCredentials;
```
**सर्वर प्रतिक्रिया को संभालना**
सर्वर से प्रतिक्रिया प्राप्त करें और पढ़ें:
```csharp
using (HttpWebResponse response = (HttpWebResponse)request.GetResponse())
{
    // डेटा स्ट्रीम प्राप्त करें
    using (Stream dataStream = response.GetResponseStream())
    {
        using (StreamReader reader = new StreamReader(dataStream))
        {
            string responseFromServer = reader.ReadToEnd();
            Console.WriteLine(responseFromServer);
        }
    }
}
```
**समस्या निवारण युक्तियों**
- सुनिश्चित करें कि आपका URL सही ढंग से स्वरूपित है.
- यदि प्रमाणीकरण आवश्यक है तो सत्यापित करें कि नेटवर्क क्रेडेंशियल सेट हैं।
## क्रेडेंशियल्स के साथ HTML को PDF में परिवर्तित करना
### अवलोकन
अब, आइए HTTP अनुरोध से प्राप्त HTML सामग्री को Aspose.PDF का उपयोग करके PDF दस्तावेज़ में परिवर्तित करें।
### चरण-दर-चरण कार्यान्वयन
**प्रतिक्रिया को मेमोरीस्ट्रीम में बदलें**
सबसे पहले, अपनी प्रतिक्रिया स्ट्रिंग को में बदलें `MemoryStream`:
```csharp
using System.IO;
using System.Text;

MemoryStream stream = new MemoryStream(Encoding.UTF8.GetBytes(responseFromServer));
```
**HTML लोड विकल्प सेट करना**
बनाएं `HtmlLoadOptions` बाह्य संसाधनों के लिए क्रेडेंशियल के साथ:
```csharp
// आधार URL के साथ HTML लोड विकल्प बनाएं और क्रेडेंशियल सेट करें
HtmlLoadOptions options = new HtmlLoadOptions("http://my.signchart.com/");
options.ExternalResourcesCredentials = CredentialCache.DefaultCredentials;
```
**पीडीएफ दस्तावेज़ लोड करना और सहेजना**
अंत में, HTML दस्तावेज़ को एक में लोड करें `Aspose.Pdf.Document` ऑब्जेक्ट चुनें और उसे PDF के रूप में सेव करें:
```csharp
// दस्तावेज़ को विकल्पों के साथ लोड करें
document pdfDocument = new Document(stream, options);

// आउटपुट सहेजें
doc.Save("YOUR_OUTPUT_DIRECTORY/ProvideCredentialsDuringHTMLToPDF_out.pdf");
```
**समस्या निवारण युक्तियों**
- सुनिश्चित करें कि आपकी HTML सामग्री में सभी URL पूर्ण हों.
- सत्यापित करें कि बाह्य संसाधनों के पास सही अनुमतियाँ हैं.

## कार्यान्वयन गाइड (H2)
### बड़ी CGM छवि को PDF (H3) में बदलें
**अवलोकन**
इस अनुभाग में, हम Aspose.PDF .NET का उपयोग करके एक बड़ी CGM छवि को PDF में बदलेंगे। यह सुविधा विशेष रूप से तब उपयोगी होती है जब उच्च-रिज़ॉल्यूशन वाले ग्राफ़िक्स से निपटना होता है जिन्हें साझा करने या अधिक सुलभ प्रारूप में प्रिंट करने की आवश्यकता होती है।

#### चरण-दर-चरण कार्यान्वयन
1. **पथ और विकल्प सेट करें**
   इनपुट और आउटपुट फ़ाइल पथ निर्दिष्ट करके शुरू करें। हम रूपांतरण प्रक्रिया के लिए पेज आकार विकल्प भी कॉन्फ़िगर करेंगे।
   ```csharp
   using System;
   using Aspose.Pdf.Facades;

   // अपनी दस्तावेज़ निर्देशिका और आउटपुट निर्देशिका सेट करें
   string inputFile = "YOUR_DOCUMENT_DIRECTORY/corvette.cgm";
   string outputFile = "YOUR_OUTPUT_DIRECTORY/LargeCGMImageToPDF_out.pdf";

   // विकल्प सेट करने के लिए CgmImportOptions का एक उदाहरण बनाएँ
   CgmImportOptions options = new CgmImportOptions();
   options.PageSize = new SizeF(1000, 1000); // आवश्यकतानुसार पृष्ठ का आकार निर्दिष्ट करें
   ```
2. **रूपांतरण करें**
   उपयोग `PdfProducer.Produce` CGM फ़ाइल को PDF प्रारूप में परिवर्तित करने के लिए.
   ```csharp
   PdfProducer.Produce(inputFile, ImportFormat.Cgm, outputFile);
   ```

#### स्पष्टीकरण
- **Cgmआयातविकल्प**: यह क्लास आपको पेज आकार जैसे विभिन्न आयात विकल्प निर्दिष्ट करने की अनुमति देता है। इस उदाहरण में, हमने इसे 1000x1000 इकाइयों पर सेट किया है।
- **PdfProducer.प्रोड्यूस**: इनपुट फ़ाइल पथ, प्रारूप और आउटपुट फ़ाइल पथ लेकर रूपांतरण प्रक्रिया को संभालता है।

# .NET के लिए Aspose.PDF का उपयोग करके PDF हस्ताक्षर कैसे बनाएं और सत्यापित करें

डिजिटल युग में, दस्तावेज़ की प्रामाणिकता सुनिश्चित करना महत्वपूर्ण है। चाहे आप सुरक्षित फ़ाइल हैंडलिंग या कानूनी दस्तावेज़ों के प्रबंधन के लिए काम करने वाले डेवलपर हों, सही टूल के बिना PDF हस्ताक्षर बनाना और सत्यापित करना कठिन हो सकता है। यह ट्यूटोरियल आपको .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ों को सहजता से कॉपी, साइन और सत्यापित करने के लिए मार्गदर्शन करेगा, जिससे आपके वर्कफ़्लो की सुरक्षा और दक्षता बढ़ेगी।

## आप क्या सीखेंगे

- **हस्ताक्षर फ़ील्ड बनाना:** अपने PDF में हस्ताक्षर फ़ील्ड जोड़ें.
- **दस्तावेजों पर हस्ताक्षर:** पीडीएफ फाइलों पर सुरक्षित हस्ताक्षर करने के लिए डिजिटल प्रमाणपत्रों का उपयोग करें।
- **हस्ताक्षर सत्यापित करना:** जाँचें कि आपके PDF दस्तावेज़ों पर हस्ताक्षर वैध हैं या नहीं।

इन सुविधाओं को लागू करने से पहले आइए हम आपके परिवेश को स्थापित करने की प्रक्रिया पर नजर डालें!

## कार्यान्वयन मार्गदर्शिका

आइये प्रत्येक फीचर को चरण दर चरण समझें।

### सुविधा 1: फ़ाइल कॉपी और हस्ताक्षर निर्माण

#### अवलोकन

यह सुविधा हस्ताक्षर करने के लिए एक नया दस्तावेज़ बनाने के लिए एक पीडीएफ फाइल की प्रतिलिपि बनाने में सक्षम बनाती है, फिर एक हस्ताक्षर फ़ील्ड जोड़ती है।

#### कदम

##### **चरण 1: पीडीएफ कॉपी करें**

अपने आधार दस्तावेज़ के रूप में काम करने के लिए किसी मौजूदा PDF की प्रतिलिपि बनाकर शुरुआत करें:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
File.Copy(dataDir + "/blank.pdf", dataDir + "/externalSignature1.pdf", true);
```

**क्यों:** इससे यह सुनिश्चित होता है कि आपको मूल फ़ाइल में कोई परिवर्तन किए बिना एक नया प्रारंभिक बिंदु मिलेगा।

##### **चरण 2: हस्ताक्षर फ़ील्ड बनाएँ और जोड़ें**

अपनी कॉपी की गई पीडीएफ को लोड करें और हस्ताक्षर फ़ील्ड जोड़ें:

```csharp
using (FileStream fs = new FileStream(dataDir + "/externalSignature1.pdf", FileMode.Open, FileAccess.ReadWrite))
{
    Document doc = new Document(fs);
    Rectangle rect = new Rectangle(100, 400, 200, 450); // हस्ताक्षर फ़ील्ड के लिए स्थिति और आकार निर्धारित करें.
    SignatureField field1 = new SignatureField(doc.Pages[1], rect);
    doc.Form.Add(field1, 1);
}
```

**क्यों:** हस्ताक्षर फ़ील्ड यह निर्धारित करता है कि उपयोगकर्ता अपने डिजिटल हस्ताक्षर कहां रखेंगे।

### फ़ीचर 2: PDF दस्तावेज़ पर हस्ताक्षर करना

#### अवलोकन

यह अनुभाग बताता है कि विंडोज प्रमाणपत्र स्टोर से प्रमाणपत्र का उपयोग करके पहले से निर्मित पीडीएफ पर हस्ताक्षर कैसे करें।

#### कदम

##### **चरण 3: प्रमाणपत्रों तक पहुंचें**

प्रमाणपत्र संग्रह खोलें और कोई प्रमाणपत्र चुनें:

```csharp
X509Store store = new X509Store(StoreLocation.CurrentUser);
store.Open(OpenFlags.ReadOnly);
X509Certificate2Collection sel = X509Certificate2UI.SelectFromCollection(store.Certificates, null, "Select a Certificate:", SelectionFlag.SingleSelection);
```

**क्यों:** प्रमाण पत्र दस्तावेजों पर हस्ताक्षर करने के लिए आवश्यक साख प्रदान करते हैं।

##### **चरण 4: पीडीएफ पर हस्ताक्षर करें**

एक बाह्य हस्ताक्षर बनाएं और उसे अपने दस्तावेज़ पर लागू करें:

```csharp
using (FileStream fs = new FileStream(dataDir + "/externalSignature1.pdf", FileMode.Open, FileAccess.ReadWrite))
{
    Document doc = new Document(fs);
    SignatureField field1 = (SignatureField)doc.Form["sig1"];
    
    Aspose.Pdf.Forms.ExternalSignature externalSignature = new Aspose.Pdf.Forms.ExternalSignature(sel[0])
    {
        Authority = "Me",
        Reason = "Reason",
        ContactInfo = "Contact"
    };

    field1.Sign(externalSignature);
    doc.Save();
}
```

**क्यों:** यह चरण आपके डिजिटल हस्ताक्षर को लागू करता है, तथा दस्तावेज़ को प्रमाणित करता है।

### फ़ीचर 3: हस्ताक्षरित PDF दस्तावेज़ का सत्यापन

#### अवलोकन

हस्ताक्षरित दस्तावेजों के हस्ताक्षरों का सत्यापन करके उनकी अखंडता सुनिश्चित करें।

#### कदम

##### **चरण 5: हस्ताक्षर सत्यापित करें**

पीडीएफ में सभी हस्ताक्षरों की वैधता की पुष्टि करने के लिए उनकी जांच करें:

```csharp
using (PdfFileSignature pdfSign = new PdfFileSignature(new Document(dataDir + "/externalSignature1.pdf")))
{
    IList<string> sigNames = pdfSign.GetSignNames();
    foreach (string sigName in sigNames)
    {
        bool isVerified = pdfSign.VerifySigned(sigName) && pdfSign.VerifySignature(sigName);
        if (!isVerified)
        {
            throw new ApplicationException("Not verified");
        }
    }
}
```

**क्यों:** सत्यापन यह सुनिश्चित करता है कि हस्ताक्षर करने के बाद दस्तावेज़ में कोई परिवर्तन नहीं किया गया है।

# .NET के लिए Aspose.PDF के साथ PDF हस्ताक्षर भाषा कैसे बदलें

## परिचय

क्या आप .NET का उपयोग करके अपने PDF दस्तावेज़ों में डिजिटल हस्ताक्षरों की भाषा को अनुकूलित करना चाहते हैं? चाहे आप अनुबंध, समझौते या हस्ताक्षर की आवश्यकता वाले किसी अन्य दस्तावेज़ को तैयार कर रहे हों, विभिन्न भाषाओं के लिए पाठ को स्थानीयकृत करना आवश्यक है। यह ट्यूटोरियल आपको .NET के लिए Aspose.PDF के साथ PDF में डिजिटल हस्ताक्षरों की भाषा सेटिंग बदलने के बारे में मार्गदर्शन करेगा, यह सुनिश्चित करते हुए कि आपके दस्तावेज़ अंतर्राष्ट्रीय मानकों को पूरा करते हैं और वैश्विक दर्शकों को पूरा करते हैं।

**आप क्या सीखेंगे:**
- .NET के लिए Aspose.PDF कैसे सेट करें।
- Aspose.PDF का उपयोग करके हस्ताक्षर पाठ भाषा बदलना `SignatureCustomAppearance` कक्षा।
- दिनांक प्रारूप, स्थान, कारण आदि जैसे हस्ताक्षर पैरामीटर कॉन्फ़िगर करना।
- अनुकूलित भाषा सेटिंग्स के साथ हस्ताक्षरित पीडीएफ दस्तावेज़ को सहेजना।

जैसा कि हम इस गाइड में आगे बढ़ रहे हैं, सुनिश्चित करें कि आप सुचारू रूप से आरंभ करने के लिए नीचे उल्लिखित पूर्वापेक्षाओं को पूरा करके तैयार हैं।

## कार्यान्वयन मार्गदर्शिका

आइए डिजिटल हस्ताक्षरों के लिए भाषा सेटिंग बदलने की प्रक्रिया को प्रबंधनीय चरणों में विभाजित करें।

### हस्ताक्षर का स्वरूप सेट करना

#### अवलोकन
हम आपके पीडीएफ पर हस्ताक्षर कैसे दिखाई देंगे, इसे कॉन्फ़िगर करेंगे, जिसमें विभिन्न भाषाओं के अनुरूप दिनांक, कारण और स्थान जैसे पाठ तत्वों को अनुकूलित करने पर ध्यान केंद्रित किया जाएगा।

**अपना दस्तावेज़ लोड करें**
सबसे पहले, इसका एक उदाहरण बनाएं `PdfFileSignature` और इसे अपने इनपुट पीडीएफ से बांधें:

```csharp
using (Aspose.Pdf.Facades.PdfFileSignature pdfSign = new Aspose.Pdf.Facades.PdfFileSignature())
{
    pdfSign.BindPdf("input.pdf");
}
```

**हस्ताक्षर आयत परिभाषित करें**
पृष्ठ पर वह क्षेत्र निर्धारित करें जहां हस्ताक्षर प्रदर्शित होंगे:

```csharp
System.Drawing.Rectangle rect = new System.Drawing.Rectangle(310, 45, 200, 50);
```

### हस्ताक्षर विवरण कॉन्फ़िगर करना

#### अवलोकन
अपने डिजिटल हस्ताक्षरों के लिए कारण और स्थान जैसे विभिन्न पैरामीटर सेट करें। इन्हें किसी भी भाषा में दर्शाने के लिए कस्टमाइज़ करें।

**PKCS#7 हस्ताक्षर ऑब्जेक्ट बनाएँ**
एक उदाहरण बनाना `PKCS7` आवश्यक प्रमाण पत्र विवरण के साथ आपत्ति:

```csharp
PKCS7 pkcs = new PKCS7("certificate.pfx", "12345");
pkcs.Reason = "Pruebas Firma";
pkcs.ContactInfo = "Contacto Pruebas";
pkcs.Location = "Población (Provincia)";
pkcs.Date = DateTime.Now;
```

**हस्ताक्षर स्वरूप को अनुकूलित करें**
उपयोग `SignatureCustomAppearance` पाठ गुण और भाषा सेटिंग समायोजित करने के लिए:

```csharp
SignatureCustomAppearance signatureCustomAppearance = new SignatureCustomAppearance();
signatureCustomAppearance.DateSignedAtLabel = "Fecha";
signatureCustomAppearance.DigitalSignedLabel = "Digitalmente firmado por";
signatureCustomAppearance.ReasonLabel = "Razón";
signatureCustomAppearance.LocationLabel = "Localización";
signatureCustomAppearance.FontFamilyName = "Arial";
signatureCustomAppearance.FontSize = 10d;
signatureCustomAppearance.Culture = CultureInfo.InvariantCulture;
signatureCustomAppearance.DateTimeFormat = "yyyy.MM.dd HH:mm:ss";

pkcs.CustomAppearance = signatureCustomAppearance;
```

### पीडीएफ पर हस्ताक्षर करना
**हस्ताक्षर लागू करें**
उपयोग `Sign` अपना कॉन्फ़िगर किया गया हस्ताक्षर लागू करने की विधि:

```csharp
pdfSign.Sign(1, true, rect, pkcs);
```

### आउटपुट फ़ाइल को सहेजना
**हस्ताक्षरित दस्तावेज़ सहेजें**
अंत में, अपने हस्ताक्षरित पीडीएफ को नई भाषा सेटिंग के साथ सहेजें:

```csharp
pdfSign.Save("output.pdf");
```

## व्यावहारिक अनुप्रयोगों
यहां कुछ वास्तविक परिदृश्य दिए गए हैं जहां इस सुविधा का उपयोग किया जा सकता है:
1. **अंतर्राष्ट्रीय व्यापार अनुबंध**: विभिन्न देशों के साझेदारों के लिए हस्ताक्षर पाठ का स्थानीयकरण करना।
2. **कानूनी दस्तावेजों**स्थानीय भाषा में हस्ताक्षर प्रदर्शित करके क्षेत्रीय कानूनी आवश्यकताओं के अनुपालन को सुनिश्चित करना।
3. **शैक्षिक सामग्री**अंतर्राष्ट्रीय छात्रों को उनकी मूल भाषा में प्रमाण पत्र या दस्तावेज उपलब्ध कराना।
4. **सरकारी फॉर्म**: ऐसे फॉर्मों को अनुकूलित करना जिनके लिए आधिकारिक हस्ताक्षर की आवश्यकता होती है, जैसे परमिट और पंजीकरण।
5. **बहुराष्ट्रीय निगम**विभिन्न क्षेत्रों के कर्मचारियों के लिए दस्तावेज़ वर्कफ़्लो को सुव्यवस्थित करना।

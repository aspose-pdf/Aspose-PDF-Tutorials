## कार्यान्वयन मार्गदर्शिका

### सुविधा अवलोकन: हस्ताक्षर जानकारी निकालें

यह सुविधा दस्तावेज़ की प्रामाणिकता और अखंडता को सत्यापित करने के लिए पीडीएफ हस्ताक्षर फ़ील्ड से प्रमाणपत्र जानकारी निकालने पर केंद्रित है।

#### चरण 1: दस्तावेज़ पथ निर्धारित करें और PDF लोड करें

अपने PDF दस्तावेज़ के लिए निर्देशिका पथ निर्दिष्ट करें और उसे लोड करें:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string input = dataDir + "ExtractSignatureInfo.pdf";

using (Document pdfDocument = new Document(input))
{
    // प्रसंस्करण जारी रखें...
}
```
**स्पष्टीकरण**: द `dataDir` चर आपके दस्तावेज़ों का पथ रखता है। `input` स्ट्रिंग लोडिंग के लिए पूर्ण फ़ाइल पथ का निर्माण करता है।

#### चरण 2: फ़ॉर्म फ़ील्ड पर पुनरावृत्ति करें

हस्ताक्षर फ़ील्ड ढूंढने के लिए पीडीएफ फॉर्म में प्रत्येक फ़ील्ड पर जाएँ:

```csharp
foreach (Field field in pdfDocument.Form)
{
    SignatureField sf = field as SignatureField;
    if (sf != null)
    {
        // प्रक्रिया हस्ताक्षर फ़ील्ड
    }
}
```
**स्पष्टीकरण**: हम एक का उपयोग करते हैं `foreach` प्रत्येक पर पुनरावृति करने के लिए लूप `Field` वस्तु. `as` कीवर्ड जाँचता है कि फ़ील्ड एक है या नहीं `SignatureField`.

#### चरण 3: प्रमाणपत्र स्ट्रीम निकालें

हस्ताक्षर फ़ील्ड से प्रमाणपत्र स्ट्रीम निकालें और पढ़ें:

```csharp
Stream cerStream = sf.ExtractCertificate();
if (cerStream != null)
{
    byte[] bytes = new byte[cerStream.Length];
    using (cerStream)
    {
        cerStream.Read(bytes, 0, bytes.Length);
    }

    // प्रमाणपत्र को फ़ाइल में सहेजें
    string outputCerPath = dataDir + "input.cer";
    using (FileStream fs = new FileStream(outputCerPath, FileMode.CreateNew))
    {
        fs.Write(bytes, 0, bytes.Length);
    }
}
```
**स्पष्टीकरण**: द `ExtractCertificate()` विधि प्रमाणपत्र स्ट्रीम को पुनः प्राप्त करती है। हम इस स्ट्रीम को बाइट सरणी में पढ़ते हैं और इसे एक के रूप में सहेजते हैं `.cer` फ़ाइल।

### समस्या निवारण युक्तियों

- **फ़ाइल प्राप्त नहीं हुई**: सुनिश्चित करें कि आपका दस्तावेज़ पथ सही है.
- **शून्य स्ट्रीम**सत्यापित करें कि PDF में प्रमाणपत्रों के साथ हस्ताक्षर फ़ील्ड शामिल हैं।

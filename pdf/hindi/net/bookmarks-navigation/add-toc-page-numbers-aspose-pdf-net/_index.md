### कार्यान्वयन मार्गदर्शिका

सब कुछ सेट हो जाने के बाद, आइए अपने PDF दस्तावेज़ में पृष्ठ संख्या के साथ एक कस्टम TOC जोड़ें।

#### पीडीएफ दस्तावेज़ लोड करें और संशोधित करें
सबसे पहले, अपने आवेदन में मौजूदा पीडीएफ दस्तावेज़ लोड करें। फिर हम अपने TOC के लिए इस दस्तावेज़ की शुरुआत में एक नया पेज डालेंगे।

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string inFile = "YOUR_DOCUMENT_DIRECTORY/42824.pdf";
string outFile = "YOUR_OUTPUT_DIRECTORY/42824_out.pdf";

// पीडीएफ दस्तावेज़ लोड करें
document doc = new Document(inFile);

// TOC के लिए शुरुआत में एक नया पृष्ठ डालें
Aspose.Pdf.Page tocPage = doc.Pages.Insert(1);
```

#### विषय-सूची जानकारी बनाएँ और कॉन्फ़िगर करें
इसके बाद, अपने TOC का शीर्षक, फ़ॉन्ट आकार, शैली और पृष्ठ संख्या उपसर्ग निर्धारित करके उसे कॉन्फ़िगर करें।

```csharp
// TOC जानकारी सेट करें
tocInfo tocInfo = new TocInfo();
textFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20; // TOC शीर्षक के लिए फ़ॉन्ट आकार
title.TextState.FontStyle = FontStyles.Bold; // ज़ोर देने के लिए बोल्ड शैली
tocInfo.Title = title;
tocInfo.PageNumbersPrefix = "P"; // पृष्ठ संख्या के आगे 'P' लगाएं
tocPage.TocInfo = tocInfo; // नए पृष्ठ पर TOC जानकारी निर्दिष्ट करें
```

#### विषय-सूची में प्रविष्टियाँ जोड़ें
अब, अपने दस्तावेज़ के प्रत्येक पृष्ठ के लिए प्रविष्टियों के साथ TOC भरें।

```csharp
for (int i = 1; i < doc.Pages.Count; i++) {
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
textSegment segment2 = new textSegment();

    // शीर्षक को TOC पृष्ठ से लिंक करें
    heading2.TocPage = tocPage;

    // शीर्षक में पाठ खंड जोड़ें
    heading2.Segments.Add(segment2); 

    // प्रत्येक प्रविष्टि के लिए गंतव्य पृष्ठ और स्थिति निर्धारित करें
    heading2.DestinationPage = doc.Pages[i + 1];
    heading2.Top = doc.Pages[i + 1].Rect.Height; 
    segment2.Text = "P" + i.ToString(); // उपसर्ग 'P' के साथ अनुकूलित पाठ

    // TOC पृष्ठ पर शीर्षक जोड़ें
    tocPage.Paragraphs.Add(heading2);
}
```

अंत में, अपने संशोधित दस्तावेज़ को नए जोड़े गए TOC के साथ सहेजें।


2. **उपयोगकर्ता मैनुअल**: विशिष्ट निर्देशों या विषयों पर त्वरित पहुंच की अनुमति देकर उपयोगकर्ता अनुभव में सुधार करें।
3. **रिपोर्ट और प्रस्ताव**प्रस्तुतियों या समीक्षाओं के दौरान आसान संदर्भ की सुविधा प्रदान करना।

## कार्यान्वयन मार्गदर्शिका

### पृष्ठ पर पाठ बनाना और जोड़ना

#### अवलोकन

इस ट्यूटोरियल का मुख्य उद्देश्य एक सरल PDF दस्तावेज़ बनाना और .NET के लिए Aspose.PDF का उपयोग करके उसमें "Hello World!" टेक्स्ट जोड़ना है।

#### चरण 1: दस्तावेज़ को आरंभ करें

इसका एक उदाहरण बनाकर शुरू करें `Document`.

```csharp
// एक नया दस्तावेज़ ऑब्जेक्ट आरंभ करें.
Document document = new Document();
```

#### चरण 2: पेज जोड़ें

इसके बाद, अपने दस्तावेज़ में एक पेज जोड़ें। यहीं पर आप अपना टेक्स्ट डालेंगे।

```csharp
// दस्तावेज़ में एक रिक्त पृष्ठ जोड़ें.
Page page = document.Pages.Add();
```

#### चरण 3: पृष्ठ पर पाठ जोड़ें

उपयोग `TextFragment` नए जोड़े गए पृष्ठ पर अपना पाठ बनाने और जोड़ने के लिए।

```csharp
// "हैलो वर्ल्ड!" के साथ एक पाठ खंड बनाएं।
page.Paragraphs.Add(new Aspose.Pdf.Text.TextFragment("Hello World!"));
```

#### चरण 4: अपना दस्तावेज़ सहेजें

अंत में, अपना दस्तावेज़ सेव करें। अपनी आउटपुट फ़ाइल के लिए उपयुक्त स्थान चुनें।

```csharp
// पीडीएफ को सहेजने के लिए डेटा निर्देशिका और पथ निर्धारित करें।
string dataDir = RunExamples.GetDataDir_AsposePdf_QuickStart();

// दस्तावेज़ को निर्दिष्ट पथ पर सहेजें.
document.Save(dataDir + "HelloWorld_out.pdf");

Console.WriteLine("\nFile saved at " + dataDir);
```

- **पथ संबंधी मुद्दे:** फ़ाइलों को सहेजने से पहले अपने फ़ाइल पथों की दोबारा जांच करें और सुनिश्चित करें कि निर्देशिकाएं मौजूद हैं।

### पृष्ठ क्रमांकन के लिए फ़्लोटिंगबॉक्स जोड़ना

ए `FloatingBox` आपको अपने PDF पृष्ठों में विशिष्ट स्थानों पर सामग्री रखने की अनुमति देता है। यहां बताया गया है कि आप पृष्ठ संख्या जोड़ने के लिए इसका उपयोग कैसे कर सकते हैं:

#### चरण 1: दस्तावेज़ को तत्काल बनाएँ और पृष्ठ जोड़ें
सबसे पहले, एक नया दस्तावेज़ बनाएं और एक पृष्ठ जोड़ें जहां फ्लोटिंग बॉक्स रखा जाएगा।

```csharp
// एक नया PDF दस्तावेज़ बनाएँ
Document document = new Document();

// पीडीएफ दस्तावेज़ में एक पृष्ठ जोड़ें
Page page = document.Pages.Add();
```

#### चरण 2: फ्लोटिंगबॉक्स आरंभ करें

इसका एक उदाहरण बनाएं `FloatingBox` वांछित आयामों के साथ और इसे अपने पृष्ठ पर उचित रूप से रखें।

```csharp
// चौड़ाई और ऊंचाई के साथ एक फ़्लोटिंगबॉक्स आरंभ करें
FloatingBox box1 = new FloatingBox(140, 80);

// सटीक प्लेसमेंट के लिए बाएं और शीर्ष स्थान सेट करें
box1.Left = 2;
box1.Top = 10;

// फ्लोटिंगबॉक्स पैराग्राफ़ में पृष्ठ क्रमांकन पाठ जोड़ें
box1.Paragraphs.Add(new Text.TextFragment("Page: ($p/ $P )"));

// वर्तमान पृष्ठ पर फ़्लोटिंग बॉक्स जोड़ें
page.Paragraphs.Add(box1);
```

**स्पष्टीकरण:**  
- `FloatingBox(140, 80)`: बॉक्स का आकार निर्धारित करता है.
- `$p` और `$P`: वर्तमान और कुल पृष्ठों के लिए प्लेसहोल्डर.

#### चरण 3: दस्तावेज़ सहेजें

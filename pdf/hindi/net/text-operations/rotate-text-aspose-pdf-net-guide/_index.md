## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में, हम .NET के लिए Aspose.PDF का उपयोग करके PDF में पाठ को घुमाने का तरीका बताएंगे।

### टेक्स्ट रोटेशन का अवलोकन

पाठ को घुमाना सौंदर्यपूर्ण लेआउट या तंग जगहों में सामग्री को फिट करने के लिए आवश्यक हो सकता है। हम इसका उपयोग करेंगे `TextFragment` और इसके घूर्णन गुण को सेट करें.

#### चरण-दर-चरण कार्यान्वयन

**1. दस्तावेज़ को आरंभ करें**
एक नया दस्तावेज़ ऑब्जेक्ट बनाकर प्रारंभ करें:
```csharp
Document pdfDocument = new Document();
```

**2. पेज जोड़ें**
अपने PDF दस्तावेज़ में वह पृष्ठ पुनः प्राप्त करें या जोड़ें जहां आप पाठ रखेंगे:
```csharp
Page pdfPage = (Page)pdfDocument.Pages.Add();
```

**3. टेक्स्ट फ़्रैगमेंट बनाएँ और कॉन्फ़िगर करें**
बनाएं `TextFragment` मुख्य और घुमाए गए पाठ के लिए उदाहरण.
- **मुख्य पाठ:**
  ```csharp
  TextFragment textFragment1 = new TextFragment("main text");
textFragment1.TextState.FontSize = 12;
textFragment1.TextState.Font = फ़ॉन्टRepository.FindFont('TimesNewRoman');
  ```

- **Rotated Text (315 degrees):**
  ```csharp
  TextFragment textFragment2 = new TextFragment("rotated text");
textFragment2.TextState.FontSize = 12;
textFragment2.TextState.Font = FontRepository.FindFont("TimesNewRoman");
textFragment2.TextState.Rotation = 315; // Rotate by 315 degrees
  ```

- **घुमाया गया पाठ (270 डिग्री):**
  ```csharp
  TextFragment textFragment3 = new TextFragment("rotated text");
textFragment3.TextState.FontSize = 12;
textFragment3.TextState.Font = फ़ॉन्टRepository.FindFont('TimesNewRoman');
textFragment3.TextState.Rotation = 270; // 270 डिग्री घुमाएँ
  ```

**4. Add Text Fragments to the Page**
Add these fragments to your page:
```csharp
pdfPage.Paragraphs.Add(textFragment1);
pdfPage.Paragraphs.Add(textFragment2);
pdfPage.Paragraphs.Add(textFragment3);
```

**5. दस्तावेज़ सहेजें**
अंत में, अपना दस्तावेज़ सहेजें:
```csharp
pdfDocument.Save("TextFragmentTests_Rotated3_out.pdf");
```

#### समस्या निवारण युक्तियों
- लेआउट संबंधी समस्याओं से बचने के लिए उन्हें जोड़ने से पहले सुनिश्चित करें कि सभी पाठ अंश सही ढंग से कॉन्फ़िगर किए गए हैं।
- यदि घूर्णन अपेक्षानुसार नहीं दिखाई देता है, तो डिग्री सेटिंग की दोबारा जांच करें (0 डिफ़ॉल्ट है, घूर्णन नहीं)।

पाठ को घुमाने से विभिन्न उद्देश्य पूरे हो सकते हैं:

2. **शीर्षलेख और पादलेख**: पृष्ठ लेआउट में परिवर्तन किए बिना सीमित स्थान में हेडर या फ़ुटनोट फ़िट करें।
3. **डिजाइन के तत्व**: ब्रोशर या प्रस्तुतियों में दृश्य रुचि के लिए पाठ को घुमाकर डिज़ाइन को बेहतर बनाएं।

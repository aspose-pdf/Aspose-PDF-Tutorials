# .NET के लिए Aspose.PDF के साथ PDF में टेक्स्ट रोटेशन में महारत हासिल करें

## परिचय

टेक्स्ट तत्वों को घुमाकर अपने PDF दस्तावेज़ों को बेहतर बनाएँ **.NET के लिए Aspose.PDF**चाहे आपको सौंदर्य में सुधार करने की आवश्यकता हो या किसी पृष्ठ पर अधिक जानकारी फिट करने की आवश्यकता हो, यह मार्गदर्शिका आपको प्रोग्रामेटिक रूप से पीडीएफ फाइलों को बनाने और उनमें हेरफेर करने में मदद करेगी।

**आप क्या सीखेंगे:**
- .NET के लिए Aspose.PDF के साथ PDF दस्तावेज़ आरंभ करना
- घुमाए गए और मानक पाठ अंशों का निर्माण और विन्यास
- इन पाठ तत्वों को PDF पृष्ठ में जोड़ना
- अंतिम रूप से तैयार दस्तावेज़ को सहेजना

Aspose.PDF को आरंभ करने के लिए, अपनी C# फ़ाइल में ये नामस्थान शामिल करें:
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## कार्यान्वयन मार्गदर्शिका
### दस्तावेज़ आरंभ करें और पृष्ठ जोड़ें
**अवलोकन:**
एक नया PDF दस्तावेज़ इंस्टैंस बनाएं और उसमें एक पृष्ठ जोड़ें.

**चरण:**
1. **दस्तावेज़ आरंभ करें:**
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Document pdfDocument = new Document();
   ```
2. **नया पृष्ठ जोड़ें:**
   ```csharp
   Page pdfPage = (Page)pdfDocument.Pages.Add();
   ```

### टेक्स्टपैराग्राफ बनाएं और स्थिति निर्धारित करें
**अवलोकन:**
पाठ अंशों को रखने के लिए एक पाठ पैराग्राफ बनाएं और पृष्ठ पर उसका प्रारंभिक स्थान निर्धारित करें।

**चरण:**
1. **टेक्स्टपैराग्राफ बनाएं:**
   ```csharp
   TextParagraph paragraph = new TextParagraph();
   ```
2. **पैराग्राफ़ स्थिति सेट करें:**
   ```csharp
   paragraph.Position = new Position(200, 600);
   ```

### रोटेटेड टेक्स्टफ़्रेगमेंट बनाएँ और कॉन्फ़िगर करें
**अवलोकन:**
Aspose.PDF की लचीलापन प्रदर्शित करने के लिए रोटेशन के साथ एक पाठ खंड उत्पन्न करें।

**चरण:**
1. **RotatedTextFragment बनाएं:**
   ```csharp
   TextFragment rotatedText = new TextFragment("rotated text");
   ```
2. **फ़ॉन्ट और रोटेशन कॉन्फ़िगर करें:**
   ```csharp
   rotatedText.TextState.FontSize = 12;
   rotatedText.TextState.Font = FontRepository.FindFont("TimesNewRoman");
   rotatedText.TextState.Rotation = 45; // 45 डिग्री घुमाएँ
   ```

### मुख्य टेक्स्टफ़्रेगमेंट बनाएँ और कॉन्फ़िगर करें
**अवलोकन:**
तुलना के लिए एक मानक पाठ अंश जोड़ें.

**चरण:**
1. **मुख्य टेक्स्ट फ़्रैगमेंट बनाएँ:**
   ```csharp
   TextFragment mainText = new TextFragment("main text");
   ```
2. **फ़ॉन्ट सेटिंग सेट करें:**
   ```csharp
   mainText.TextState.FontSize = 12;
   mainText.TextState.Font = FontRepository.FindFont("TimesNewRoman");
   ```

### एक और घुमाए गए टेक्स्टफ़्रेगमेंट को बनाएँ और कॉन्फ़िगर करें
**अवलोकन:**
बहुमुखी प्रतिभा को प्रदर्शित करने के लिए नकारात्मक घुमाव के साथ एक और घुमाए गए पाठ खंड को जोड़ें।

**चरण:**
1. **एक और घुमाए गए टेक्स्ट फ़्रैगमेंट बनाएँ:**
   ```csharp
   TextFragment anotherRotatedText = new TextFragment("another rotated text");
   ```
2. **फ़ॉन्ट और नकारात्मक रोटेशन सेट करें:**
   ```csharp
   anotherRotatedText.TextState.FontSize = 12;
   anotherRotatedText.TextState.Font = FontRepository.FindFont("TimesNewRoman");
   anotherRotatedText.TextState.Rotation = -45; // -45 डिग्री घुमाएँ
   ```

### पैराग्राफ में टेक्स्ट अंश जोड़ें और पेज में जोड़ें
**अवलोकन:**
पाठ अंशों को एक पैराग्राफ में संयोजित करें, फिर इस पैराग्राफ को अपने पीडीएफ पेज में जोड़ें `TextBuilder`.

**चरण:**
1. **पैराग्राफ में अंश जोड़ें:**
   ```csharp
   paragraph.AppendLine(rotatedText);
   paragraph.AppendLine(mainText);
   paragraph.AppendLine(anotherRotatedText);
   ```
2. **टेक्स्टबिल्डर का उपयोग करके पृष्ठ पर पैराग्राफ जोड़ें:**
   ```csharp
   TextBuilder textBuilder = new TextBuilder(pdfPage);
   textBuilder.AppendParagraph(paragraph);
   ```

### दस्तावेज़ सहेजें
**अवलोकन:**
निर्मित पीडीएफ दस्तावेज़ को सहेजें.

**चरण:**
1. **दस्तावेज़ सहेजें:**
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   pdfDocument.Save(outputDir + "/TextFragmentTests_Rotated2_out.pdf");
   ```

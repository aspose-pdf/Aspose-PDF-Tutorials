# .NET के लिए Aspose.PDF के साथ पीडीएफ में रेखांकित पाठ जोड़ना

Aspose.PDF .NET के लिए एक शक्तिशाली पीडीएफ प्रसंस्करण पुस्तकालय है जो डेवलपर्स को एडोब एक्रोबेट की आवश्यकता के बिना पीडीएफ दस्तावेज़ बनाने, संशोधित करने और हेरफेर करने की अनुमति देता है। यह उदाहरण दिखाता है कि कैसे एक नया पीडीएफ दस्तावेज़ बनाने और टेक्स्टबिल्डर और टेक्स्टफ्रेगमेंट वर्गों का उपयोग करके रेखांकित स्वरूपण के साथ पाठ जोड़ने के लिए। कोड पृष्ठ के भीतर विशिष्ट निर्देशांक पर स्थित, रेखांकित स्टाइल के साथ एरियल फ़ॉन्ट में स्वरूपित एक परीक्षण संदेश के साथ एक सरल पीडीएफ बनाता है।

## स्थापना के निर्देश

1: .NET के लिए Aspose.PDF स्थापित करें

आप निम्न विधियों में से किसी एक का उपयोग करके .NET के लिए Aspose.PDF स्थापित कर सकते हैंः

NuGet पैकेज प्रबंधक का उपयोग करनाः

```csharp
Install-Package Aspose.PDF
```

.NET CLI का उपयोग करनाः

```bash
dotnet add package Aspose.PDF
```

अपने प्रोजेक्ट फ़ाइल में PackageReference का उपयोग करनाः

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: आवश्यक उपयोग कथन जोड़ें

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

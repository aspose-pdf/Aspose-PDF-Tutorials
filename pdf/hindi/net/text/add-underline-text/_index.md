# .NET के लिए Aspose.PDF के साथ PDF में रेखांकित पाठ जोड़ना

.NET के लिए Aspose.PDF एक शक्तिशाली PDF प्रोसेसिंग लाइब्रेरी है जो डेवलपर्स को Adobe Acrobat की आवश्यकता के बिना PDF दस्तावेज़ बनाने, संशोधित करने और हेरफेर करने की अनुमति देती है। यह उदाहरण दर्शाता है कि TextBuilder और TextFragment क्लास का उपयोग करके एक नया PDF दस्तावेज़ कैसे बनाया जाए और अंडरलाइन फ़ॉर्मेटिंग के साथ टेक्स्ट कैसे जोड़ा जाए। कोड एक सरल PDF बनाता है जिसमें एक परीक्षण संदेश होता है जिसे Arial फ़ॉन्ट में अंडरलाइन स्टाइलिंग के साथ फ़ॉर्मेट किया जाता है, इसे पृष्ठ के भीतर विशिष्ट निर्देशांक पर स्थित किया जाता है।

## स्थापना निर्देश

1: .NET के लिए Aspose.PDF स्थापित करें

आप निम्न विधियों में से किसी एक का उपयोग करके .NET के लिए Aspose.PDF स्थापित कर सकते हैं:

NuGet पैकेज मैनेजर का उपयोग करना:

```csharp
Install-Package Aspose.PDF
```

.NET CLI का उपयोग करना:

```bash
dotnet add package Aspose.PDF
```

अपनी प्रोजेक्ट फ़ाइल में PackageReference का उपयोग करें:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: आवश्यक उपयोग कथन जोड़ें

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## Frequently Asked Questions

**Q:** *Can I use these tutorials in a commercial application?*  
**A:** Yes, you need a valid Aspose license for production use; a free trial is available for evaluation.

**Q:** *Do the .NET tutorials work with .NET Core?*  
**A:** Absolutely – all examples are compatible with .NET Core 3.1 and later, as well as .NET 5/6/7.

**Q:** *How do I extract images from an existing PDF?*  
**A:** Use the `PdfExtractor` class shown in the “Extract Images PDF” section to pull out raster assets.

**Q:** *What is the best way to add a table to a PDF?*  
**A:** Follow the “Add Table PDF” guide; it covers creating a `Table` object, populating rows, and applying borders.

**Q:** *Can I draw custom lines or shapes on a PDF page?*  
**A:** Yes – see the “Draw Line PDF” tutorial for using the `Line` and `Rectangle` primitives.

## Instruksi Pemasangan  
Untuk menggunakan Aspose.PDF untuk .NET dalam proyek Anda, ikuti langkah- langkah ini:  

1. Instal melalui NuGet  
Jalankan perintah berikut di NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Tambahkan ruang nama di C #  
Sertakan ruang nama yang diperlukan di awal file C # Anda:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Untuk dokumentasi rinci, kunjungi:  
[Aspose.PDF untuk Dokumentasi .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF untuk Referensi .NET API](https://reference.aspose.com/pdf/net/)

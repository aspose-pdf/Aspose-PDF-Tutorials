# Buat PDF dengan Persegi Panjang & Warna Alfa di Aspose.PDF

Aspose.PDF untuk .NET adalah pustaka yang hebat untuk membuat, memanipulasi, dan mengonversi dokumen PDF dalam C#. Dalam contoh ini, Anda akan mempelajari cara membuat file PDF dan menggambar persegi panjang dengan warna alfa. Kami menggunakan `Aspose.Pdf.Drawing.Graph` kelas untuk menambahkan bentuk dan menerapkan efek transparansi menggunakan warna ARGB. Tutorial ini menyediakan panduan langkah demi langkah dengan contoh kode, memastikan Anda dapat dengan mudah menerapkannya dalam aplikasi .NET Anda.

## Petunjuk Instalasi  
Untuk menggunakan Aspose.PDF untuk .NET di proyek Anda, ikuti langkah-langkah berikut:  

1. Instal melalui NuGet  
Jalankan perintah berikut di Konsol Pengelola Paket NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Menambahkan Namespace di C#  
Sertakan namespace yang diperlukan di awal file C# Anda:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Untuk dokumentasi terperinci, kunjungi:  
[Dokumentasi Aspose.PDF untuk .NET](https://docs.aspose.com/pdf/net/)  
[Referensi API Aspose.PDF untuk .NET](https://reference.aspose.com/pdf/net/)

# Buat PDF dengan persegi panjang & warna alfa di Aspose.PDF

Aspose.PDF untuk .NET adalah perpustakaan yang kuat untuk membuat, memanipulasi, dan mengkonversi dokumen PDF dalam C#. Dalam contoh ini, Anda akan belajar bagaimana membuat file PDF dan menggambar persegi panjang dengan warna alfa.`Aspose.Pdf.Drawing.Graph`kelas untuk menambahkan bentuk dan menerapkan efek transparansi menggunakan warna ARGB. Tutorial ini menyediakan panduan langkah demi langkah dengan kode sampel, memastikan Anda dapat dengan mudah menerapkannya dalam aplikasi .NET Anda.

## Instruksi Instalasi  
Untuk menggunakan Aspose.PDF untuk .NET dalam proyek Anda, ikuti langkah- langkah ini:  

1. Instal melalui NuGet  
Keluarkan perintah berikut di NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Tambahkan Nama Ruang dalam C#  
Masukkan ruang nama yang diperlukan di awal file C # Anda:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Untuk dokumentasi terperinci, kunjungi:  
[Aspose.PDF untuk Dokumen .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF untuk Referensi API .NET](https://reference.aspose.com/pdf/net/)

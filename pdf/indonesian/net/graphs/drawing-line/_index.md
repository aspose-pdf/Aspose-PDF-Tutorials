## Instruksi pemasangan:  

Untuk menggunakan Aspose.PDF untuk .NET dalam proyek Anda, ikuti langkah- langkah ini:  

1. Instal melalui NuGet Package Manager  
* Buka Visual Studio dan buat proyek .NET baru.  
* Buka Paket Manajer Konsol (Tools → NuGet Paket Manajer → Paket Manajer Konsol) dan jalankan:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instal melalui .NET CLI  
Atau, gunakan .NET CLI untuk menginstal paket:  
```bash
dotnet add package Aspose.PDF
```

3. Tambahkan ruang nama dalam kode C # Anda  
Untuk menggunakan fungsi Aspose.PDF, sertakan ruang nama berikut:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

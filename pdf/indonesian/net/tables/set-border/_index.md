## Petunjuk Instalasi
Untuk memulai Aspose.PDF untuk .NET, ikuti petunjuk instalasi berikut:

1. Unduh Aspose.PDF: 
   - Kunjungi [Situs web Aspose](https://www.aspose.com/downloads/pdf/net) untuk mengunduh versi terbaru Aspose.PDF untuk .NET.

2. Instal melalui Manajer Paket NuGet:
   - Buka proyek Anda di Visual Studio.
   - Buka Alat > Manajer Paket NuGet > Kelola Paket NuGet untuk Solusi.
   - Cari "Aspose.PDF" dan instal paketnya.

3. Tambahkan Referensi:
   - Pastikan proyek Anda merujuk ke Aspose.PDF DLL di `packages` map.
   
4. Sertakan Namespace Aspose.PDF dalam Proyek Anda:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

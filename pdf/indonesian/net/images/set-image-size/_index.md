# Buat dan Ubah PDF dengan Gambar Dengan Menggunakan Aspose.PDF

Aspose.PDF untuk .NET adalah perpustakaan yang kuat yang memungkinkan pengembang untuk membuat, mengedit, dan memanipulasi dokumen PDF secara programatik. Dalam contoh ini, Anda akan belajar bagaimana membuat dokumen PDF, menambahkan gambar, menyesuaikan ukurannya, dan memodifikasi properti halaman menggunakan C #. Kode menunjukkan menginisialisasi dokumen PDF, memasukkan gambar, mengatur dimensi halaman, dan menyimpan file akhir. ## Instruksi pemasangan:  

Untuk menggunakan Aspose.PDF untuk .NET dalam proyek Anda, ikuti langkah- langkah ini:  

1. Instal melalui NuGet Package Manager:  
Buka Visual Studio dan jalankan perintah berikut di Konsole Paket Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Tambahkan namespace dalam kode C# Anda:  
```csharp
using Aspose.Pdf;
```

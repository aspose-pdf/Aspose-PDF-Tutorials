# Menambahkan Teks Bergaris Bawah dalam PDF dengan Aspose.PDF untuk .NET

Aspose.PDF untuk .NET adalah pustaka pemrosesan PDF yang canggih yang memungkinkan pengembang untuk membuat, memodifikasi, dan memanipulasi dokumen PDF tanpa memerlukan Adobe Acrobat. Contoh ini menunjukkan cara membuat dokumen PDF baru dan menambahkan teks dengan format garis bawah menggunakan kelas TextBuilder dan TextFragment. Kode tersebut membuat PDF sederhana dengan pesan uji yang diformat dalam font Arial dengan gaya garis bawah, memposisikannya pada koordinat tertentu di dalam halaman.

## Petunjuk Instalasi

1: Instal Aspose.PDF untuk .NET

Anda dapat menginstal Aspose.PDF untuk .NET menggunakan salah satu metode berikut:

Menggunakan NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Menggunakan .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Menggunakan PackageReference dalam berkas proyek Anda:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Tambahkan Pernyataan Penggunaan yang Diperlukan

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

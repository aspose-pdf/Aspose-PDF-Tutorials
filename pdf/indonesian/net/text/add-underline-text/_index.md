# Menambahkan Teks yang Diterangkan di PDF dengan Aspose.PDF untuk .NET

Aspose.PDF untuk .NET adalah perpustakaan pengolah PDF yang kuat yang memungkinkan pengembang untuk membuat, memodifikasi, dan memanipulasi dokumen PDF tanpa memerlukan Adobe Acrobat. Contoh ini menunjukkan cara membuat dokumen PDF baru dan menambahkan teks dengan pemformatan digariskan menggunakan kelas TextBuilder dan TextFragment. Kode ini membuat PDF sederhana dengan pesan uji yang diformat dalam font Arial dengan gaya digariskan, memposisinya di koordinat tertentu di dalam halaman.

## Instruksi Pemasangan

1: Menginstal Aspose.PDF untuk .NET

Anda dapat menginstal Aspose.PDF untuk .NET menggunakan salah satu metode berikut:

Menggunakan NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Menggunakan .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Menggunakan PackageReference dalam file proyek Anda:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Tambahkan Pernyataan Penggunaan yang Dibutuhkan

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

sehingga PDF Anda tidak dapat diedit.\n\n#### Langkah 1: Impor Kelas yang Diperlukan\n```java\nimport com.aspose.pdf.facades.Form;\n```\n\n#### Langkah 2: Tentukan Direktori File\nSiapkan direktori untuk file input dan output.\n```java\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n```\n\n#### Langkah 3: Buat Metode untuk Meratakan Bidang\nTentukan `documentForm` metode yang membuka, meratakan, dan menyimpan dokumen PDF Anda.\n\n```java\nvoid documentForm() {\n    // Initialize Form object to interact with the PDF\n    Form pdfForm = new Form();\n    \n    // Bind the source PDF file from 'dataDir/input.pdf'\n    pdfForm.bindPdf(dataDir + \"/input.pdf\");\n    \n    // Flatten all fields in the PDF document\n    pdfForm.flattenAllFields();\n    \n    // Save the modified document to 'outputDir/output.pdf'\n    pdfForm.save(outputDir + \"/output.pdf\");\n}\n```\n\n#### Penjelasan:\n- **`Form` Kelas**: Digunakan untuk memanipulasi bidang formulir dalam PDF.\n- **`bindPdf` Metode**: Membuka dan menyiapkan file PDF untuk diproses.\n- **`flattenAllFields` Metode**Mengubah semua bidang interaktif

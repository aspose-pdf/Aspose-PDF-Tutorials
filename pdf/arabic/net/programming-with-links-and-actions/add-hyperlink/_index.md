# إضافة ارتباط تشعبي في ملف PDF

## مقدمة

إضافة روابط تشعبية إلى ملفات PDF تُحسّن بشكل كبير من تفاعلية المستند وسهولة تصفحه. سواءً كنت تُنشئ فاتورةً تُربط ببوابة دفع أو تقريرًا يُوجِّه القراء إلى موارد إلكترونية ذات صلة، تُضيف الروابط التشعبية طبقةً من الوظائف تجعل ملفات PDF أكثر سهولةً في الاستخدام. في هذا الدليل، سنستخدم Aspose.PDF لـ .NET لنوضح لك كيفية إضافة روابط تشعبية إلى ملفات PDF بسلاسة. لذا، استعد جيدًا؛ ستتعلم كل شيء خطوة بخطوة!

## المتطلبات الأساسية

قبل الخوض في التفاصيل الدقيقة لإضافة الروابط التشعبية، هناك بعض المتطلبات الأساسية التي يجب عليك التحقق منها في قائمتك:

1. تثبيت .NET Framework: تأكد من تثبيت .NET Framework متوافق على جهازك. يعمل Aspose.PDF مع إصدارات مختلفة، لذا تأكد من توافقه مع الإصدار الذي تستخدمه.
2. مكتبة Aspose.PDF لـ .NET: ستحتاج إلى مكتبة Aspose.PDF. يمكنك تنزيلها من [صفحة التحميل](https://releases.aspose.com/pdf/net/) إذا لم تكن قد فعلت ذلك بالفعل.
3. المعرفة الأساسية بلغة C#: إن الإلمام ببرمجة C# سيجعل هذا البرنامج التعليمي أكثر سلاسة وسهولة في الفهم.
4. بيئة التطوير: قم بإعداد IDE مثل Visual Studio لكتابة وتنفيذ التعليمات البرمجية الخاصة بك.

بمجرد توفر هذه المتطلبات الأساسية، فأنت جاهز للمتابعة!

## استيراد الحزم

للعمل مع Aspose.PDF، يجب استيراد مساحات الأسماء ذات الصلة إلى مشروع C#. افتح مشروعك، وفي أعلى ملف C#، أضف ما يلي باستخدام التوجيهات:

```csharp
using System;
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
```

وبعد أن تناولنا ذلك، دعنا ننتقل إلى عملية إضافة ارتباطات تشعبية إلى ملف PDF خطوة بخطوة.

## الخطوة 1: إعداد دليل المستندات الخاص بك

أول ما عليك فعله هو إنشاء مجلد عمل لحفظ ملفات PDF. إليك الطريقة:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يستبدل `YOUR DOCUMENT DIRECTORY` مع المسار الفعلي الذي تريد حفظ ملفات PDF فيه. سيساعدك هذا المسار على التنقل بين الملفات أثناء قراءتها وكتابتها.

## الخطوة 2: افتح مستند PDF الموجود

بعد ذلك، لنفتح ملف PDF الذي تريد إضافة الرابط التشعبي إليه. يمكنك فتح ملف PDF موجود باستخدام `Document` الفئة من مكتبة Aspose.PDF.

```csharp
Document document = new Document(dataDir + "AddHyperlink.pdf");
```

يقرأ هذا المقطع ملف PDF الخاص بك ويجهزه للتعديلات. تأكد من `"AddHyperlink.pdf"` موجود في الدليل المحدد أو قم بتعديل اسم الملف وفقًا لذلك.

## الخطوة 3: الوصول إلى صفحة PDF

الآن، علينا تحديد الصفحة التي سيظهر فيها الرابط التشعبي داخل المستند. على سبيل المثال، إذا أردنا إضافة الرابط إلى الصفحة الأولى:

```csharp
Page page = document.Pages[1];
```

تذكر أن فهرس الصفحة في Aspose يبدأ من 1، وليس 0. لذا، فإن الصفحة الأولى هي الصفحة 1.

## الخطوة 4: إنشاء كائن تعليق الارتباط

بعد ذلك، عليك تحديد منطقة المستطيل التي سيكون الرابط التشعبي قابلاً للنقر عليها. يمكنك تخصيص هذه المنطقة حسب احتياجاتك:

```csharp
LinkAnnotation link = new LinkAnnotation(page, new Aspose.Pdf.Rectangle(100, 100, 300, 300));
```

هنا، نقوم بإنشاء مستطيل يبدأ عند `(100, 100)` وتمتد إلى `(300, 300)`. قم بتعديل هذه الأرقام لتعديل حجم وموقع الرابط الخاص بك.

## الخطوة 5: تكوين حدود الرابط

بعد ضبط منطقة الرابط، نحتاج إلى منحها نمطًا بصريًا. يمكنك إنشاء حدود، مع أننا سنجعلها غير مرئية في هذه الحالة:

```csharp
Border border = new Border(link);
border.Width = 0;
link.Border = border;
```

يؤدي هذا إلى إنشاء حدود ارتباط غير مرئية، وتختلط بشكل أنيق مع تصميم ملف PDF الخاص بك.

## الخطوة 6: تحديد إجراء الارتباط التشعبي

ستحتاج إلى تحديد ما يحدث عند نقر المستخدم على هذا الرابط. في مثالنا، سنوجه المستخدمين إلى موقع Aspose الإلكتروني:

```csharp
link.Action = new GoToURIAction("http://www.aspose.com");
```

تأكد من استخدام `"http://"` في بداية عنوان الويب؛ وإلا، فقد لا يعمل بشكل صحيح.

## الخطوة 7: إضافة تعليق الرابط إلى الصفحة

في هذه المرحلة، دعنا نضع كل ما أنشأناه موضع التنفيذ عن طريق إضافة ارتباط تشعبي إلى مجموعة التعليقات التوضيحية للصفحة المحددة:

```csharp
page.Annotations.Add(link);
```

مع هذا السطر، أصبح الارتباط التشعبي الخاص بك جاهزًا وينتظر تفاعل المستخدم!

## الخطوة 8: إنشاء تعليق نصي مجاني

من المفيد إضافة سياق نصي إلى رابطك التشعبي. هذا يُساعد المستخدمين على فهم ما ينقرون عليه. لنُضيف تعليقًا توضيحيًا من FreeText:

```csharp
FreeTextAnnotation textAnnotation = new FreeTextAnnotation(document.Pages[1], new Aspose.Pdf.Rectangle(100, 100, 300, 300), new DefaultAppearance(FontRepository.FindFont("TimesNewRoman"), 10, Color.Blue));
textAnnotation.Contents = "Link to Aspose website";
textAnnotation.Border = border;
document.Pages[1].Annotations.Add(textAnnotation);
```

هنا، نُحدد نوع الخط وحجمه ولونه. يُمكنك تعديل هذه الخصائص وفقًا لاحتياجات تصميمك.

## الخطوة 9: حفظ المستند

بعد إضافة كل شيء من الارتباط التشعبي إلى التعليق النصي، حان الوقت لحفظ مستندك حتى تنعكس جميع التغييرات:

```csharp
dataDir = dataDir + "AddHyperlink_out.pdf";
document.Save(dataDir);
```

يؤدي هذا إلى حفظ ملف PDF المحدث كملف جديد باسم `"AddHyperlink_out.pdf"` في الدليل المحدد الخاص بك.

## خاتمة

إضافة روابط تشعبية إلى مستندات PDF باستخدام Aspose.PDF لـ .NET لا ترتقي باحترافية ملفات PDF فحسب، بل تُعزز أيضًا تفاعل المستخدم. الأمر سهل، ويمنح مستوى جديدًا كليًا من التفاعلية لا تُضاهيه المستندات الثابتة. باتباع الخطوات الموضحة في هذا الدليل، يمكنك إضافة روابط تشعبية بثقة إلى أي ملف PDF تُنشئه أو تُعدّله.

# تحويل النص إلى PDF

## مقدمة

في عالمنا الرقمي اليوم، لم تعد القدرة على تحويل الملفات النصية إلى صيغة PDF مجرد ترف، بل ضرورة. تُستخدم ملفات PDF على نطاق واسع لقدرتها على الحفاظ على التنسيق عبر مختلف الأجهزة والمنصات. سواء كنت مطورًا يسعى إلى تحسين تطبيقك أو متخصصًا في مجال الأعمال ويحتاج إلى إنشاء تقارير، فإن فهم كيفية العمل مع ملفات PDF أمر بالغ الأهمية. في هذا الدليل الشامل، سنشرح لك عملية استخدام Aspose.PDF لـ .NET لتحويل ملف نصي بسيط إلى مستند PDF مُحسّن. بفضل ميزاته القوية وواجهته سهلة الاستخدام، يُسهّل Aspose.PDF التعامل مع ملفات PDF. هيا بنا نبدأ!

## المتطلبات الأساسية
قبل الغوص في الكود، تأكد من أن لديك المتطلبات الأساسية التالية:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. هنا سنكتب وننفذ الكود.
2. Aspose.PDF لـ .NET: قم بتنزيل مكتبة Aspose.PDF وتثبيتها من [هنا](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.
4. ملف نصي نموذجي: في هذا البرنامج التعليمي، سنستخدم ملف نصي بسيط يسمى `log.txt`تأكد من أن هذا الملف جاهز في دليل المشروع الخاص بك.

## استيراد الحزم
لبدء استخدام Aspose.PDF، عليك استيراد الحزم اللازمة إلى مشروعك. إليك كيفية القيام بذلك:

1. افتح مشروع Visual Studio الخاص بك.
2. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول وحدد "إدارة حزم NuGet".
3. بحث عن `Aspose.PDF` وتثبيت الحزمة.

```csharp
using System.IO;
using System;
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

بمجرد تثبيت الحزمة، يمكنك البدء في كتابة الكود الخاص بك.

## الخطوة 1: إعداد مشروعك
قبل كتابة أي شيفرة برمجية، لنُعِدّ هيكل مشروعنا. أنشئ تطبيق وحدة تحكم C# جديدًا في Visual Studio. ستكون هذه البيئة التي سنُطبّق فيها منطق تحويل PDF.

## الخطوة 2: تحديد دليل المستندات
في هذه الخطوة، سنحدد مسار المجلد الذي يحتوي على ملفنا النصي. هذا مهم جدًا لأننا نحتاج إلى قراءة محتوى الملف النصي قبل تحويله إلى PDF.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يستبدل `YOUR DOCUMENT DIRECTORY` مع المسار الفعلي الذي تريده `log.txt` تم تخزين الملف.

## الخطوة 3: قراءة ملف النص المصدر
الآن بعد أن قمنا بإعداد دليل المستندات، لنقرأ محتوى الملف النصي. سنستخدم `StreamReader` لتحقيق ذلك.

```csharp
TextReader tr = new StreamReader(dataDir + "log.txt");
```

يفتح هذا السطر من التعليمات البرمجية `log.txt` ملف للقراءة. تأكد من وجود الملف في الدليل المحدد، وإلا ستواجه خطأً.

## الخطوة 4: إنشاء كائن مستند
بعد ذلك، نحتاج إلى إنشاء مستند PDF جديد. يتم ذلك عن طريق إنشاء مثيل لـ `Document` هدف.

```csharp
Document doc = new Document();
```

يقوم هذا السطر بإنشاء مستند PDF جديد سنقوم بملئه بالمحتوى.

## الخطوة 5: إضافة صفحة جديدة
كل مستند PDF يتكون من صفحات. في هذه الخطوة، سنضيف صفحة جديدة إلى مستندنا.

```csharp
Page page = doc.Pages.Add();
```

يضيف هذا السطر صفحة جديدة إلى `Pages` مجموعة من مستنداتنا. يمكنك اعتبارها إضافة لوحة فارغة لوضع نصنا عليها.

## الخطوة 6: إنشاء جزء نصي
الآن سوف نقوم بإنشاء `TextFragment` كائنٌ سيحمل النص الذي نقرؤه من الملف. هنا يحدث السحر!

```csharp
TextFragment text = new TextFragment(tr.ReadToEnd());
```

هنا نقرأ محتوى ملف النص بالكامل ونمرره إلى `TextFragment` سيمثل هذا الكائن النص الذي نريد إضافته إلى ملف PDF الخاص بنا.

## الخطوة 7: إضافة النص إلى الصفحة
مع شركائنا `TextFragment` جاهز، حان الوقت لإضافته إلى الصفحة التي أنشأناها سابقًا.

```csharp
page.Paragraphs.Add(text);
```

يضيف هذا الخط `TextFragment` الى `Paragraphs` مجموعة من الصفحات. الأمر أشبه بوضع نص على لوحتنا.

## الخطوة 8: حفظ مستند PDF
أخيرًا، علينا حفظ مستند PDF الذي أنشأناه حديثًا. هذه هي الخطوة الأخيرة في عملية التحويل.

```csharp
doc.Save(dataDir + "TexttoPDF_out.pdf");
```

يحفظ هذا السطر المستند في المسار المحدد. يمكنك تسمية ملف الإخراج بأي اسم تريده، ولكن في هذا الدرس، سنسميه `TexttoPDF_out.pdf`.

## الخطوة 9: التعامل مع الاستثناءات
من الجيد دائمًا التعامل مع الاستثناءات في الكود. هذا يضمن إمكانية اكتشاف الخطأ والاستجابة له بشكل مناسب في حال حدوث أي خطأ.

```csharp
catch (Exception ex)
{
    Console.WriteLine(ex.Message);
}
```

ستقوم كتلة الالتقاط هذه بطباعة أي رسائل خطأ في وحدة التحكم، مما يساعدك على تصحيح أي مشكلات تنشأ أثناء عملية إنشاء ملف PDF.

# استبدال الخطوط المفقودة

## مقدمة

هل سبق لك أن فتحت مستند PDF لتكتشف أن بعض الخطوط مفقودة؟ قد يكون الأمر محبطًا، أليس كذلك؟ قد يؤدي فقدان الخطوط إلى ظهور المستند بشكل مختلف تمامًا عما أراده مُنشئه. لحسن الحظ، مع Aspose.PDF لـ .NET، يمكنك بسهولة استبدال الخطوط المفقودة وضمان احتفاظ مستندات PDF بمظهرها المطلوب. في هذا البرنامج التعليمي، سنشرح لك العملية خطوة بخطوة، مما يجعلها بسيطة ومباشرة.

## المتطلبات الأساسية

قبل أن نبدأ، هناك بعض الأشياء التي ستحتاج إلى وضعها في مكانها:

1. Aspose.PDF لـ .NET: تأكد من تثبيت مكتبة Aspose.PDF. يمكنك تنزيلها من [هنا](https://releases.aspose.com/pdf/net/).
2. Visual Studio: بيئة تطوير يمكنك من خلالها كتابة واختبار الكود الخاص بك.
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.

## استيراد الحزم

للبدء، ستحتاج إلى استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

```csharp
using System;
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، عليك تحديد مسار مجلد المستندات. هذا هو المكان الذي يوجد فيه ملف PDF المُدخل، وهو المكان الذي سيتم فيه حفظ ملف الإخراج.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: تهيئة الخط الأصلي

بعد ذلك، حاول العثور على الخط الأصلي الذي قد يكون مفقودًا. في هذه الحالة، نبحث عن "AgencyFB".

```csharp
Aspose.Pdf.Text.Font originalFont = null;
try
{
    originalFont = FontRepository.FindFont("AgencyFB");
}
catch (Exception)
{
    // الخط مفقود على الجهاز الوجهة
    FontRepository.Substitutions.Add(new SimpleFontSubstitution("AgencyFB", "Arial"));
}
```

هنا، نحاول العثور على الخط. إذا لم نجده، نكتشف الاستثناء ونستبدله بخط أكثر شيوعًا، وهو "Arial". هذا يضمن أن يبدو مستندك جيدًا حتى لو لم يكن الخط الأصلي متوفرًا.

## الخطوة 3: تحميل مستند PDF

الآن، لنحمّل ملف PDF الذي تريد معالجته. ستحتاج إلى تحديد مسار ملف الإدخال.

```csharp
var fileNew = new FileInfo(dataDir + "newfile_out.pdf");
var pdf = new Document(dataDir + "input.pdf");
```

في هذه الخطوة نقوم بإنشاء جديد `FileInfo` كائن لملف الإخراج وتحميل مستند PDF المدخل في ملف جديد `Document` هدف.

## الخطوة 4: تحويل مستند PDF

قبل حفظ المستند، يُنصح بتحويله إلى صيغة PDF مُحددة. في هذه الحالة، سنحوّله إلى صيغة PDF/A-1B، وهي الصيغة القياسية للأرشفة الإلكترونية طويلة الأمد.

```csharp
pdf.Convert(dataDir + "log.xml", PdfFormat.PDF_A_1B, ConvertErrorAction.Delete);
```

يُحوّل هذا السطر ملف PDF ويُسجّل أي أخطاء في ملف XML مُحدّد. في حال وجود أي مشاكل أثناء التحويل، سيتم تسجيلها في "log.xml".

## الخطوة 5: حفظ مستند PDF المحدث

أخيرًا، حان الوقت لحفظ مستند PDF المحدث بالخطوط المستبدلة.

```csharp
pdf.Save(fileNew.FullName);
```

هذا السطر يحفظ ملف PDF المُعدَّل في مسار ملف الإخراج المُحدَّد. وهكذا، تكون قد استبدلت الخطوط المفقودة في مستند PDF بنجاح!

## خاتمة

استبدال الخطوط المفقودة في مستندات PDF ليس بالأمر الصعب. مع Aspose.PDF لـ .NET، يمكنك بسهولة إدارة عمليات استبدال الخطوط وضمان ظهور مستنداتك كما ينبغي. باتباع الخطوات الموضحة في هذا البرنامج التعليمي، يمكنك الحفاظ على سلامة ملفات PDF، حتى في حالة عدم توفر بعض الخطوط. لذا، في المرة القادمة التي تواجه فيها مشكلة فقدان الخطوط، ستعرف بالضبط ما يجب فعله!

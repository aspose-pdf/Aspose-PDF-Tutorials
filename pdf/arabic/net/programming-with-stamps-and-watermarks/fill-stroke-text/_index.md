# ملء النص في ملف PDF

## مقدمة

هل رغبتَ يومًا في تعديل ملف PDF لجعله مميزًا؟ ربما تحتاج إلى إضافة علامة مائية مميزة أو طابع عريض يجعل مستندًا مهمًا خاصًا بك. مع Aspose.PDF لـ .NET، يمكنك بسهولة ملء النص المُحدَّد في ملف PDF، مما يضيف لمسة فنية جذابة. في درس اليوم، سنشرح عملية القيام بذلك تمامًا - ملء النص المُحدَّد في ملف PDF باستخدام C#. في النهاية، ستكون قد أتقنتَ التعامل مع ملفات PDF باحترافية.

## المتطلبات الأساسية

قبل أن نتعمق في الترميز، هناك بعض الأشياء التي ستحتاج إلى وضعها في مكانها لجعل هذا البرنامج التعليمي سهلاً:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك لأننا سنكتب كود C#.
2. مكتبة Aspose.PDF: تأكد من تنزيل مكتبة Aspose.PDF لـ .NET. يمكنك الحصول عليها. [هنا](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم البرنامج التعليمي بسهولة أكبر.
4. ملف PDF نموذجي: ستحتاج إلى ملف PDF نموذجي (`input.pdf`) لأغراض الاختبار. يمكنك إنشاء ملف بسيط أو استخدام أي ملف PDF موجود لديك.

الآن بعد أن أصبح كل شيء في مكانه، دعنا ننتقل إلى التفاصيل الدقيقة لملء نص الخطوط في ملف PDF الخاص بك.

## استيراد الحزم

للبدء، سنحتاج إلى استيراد الحزم اللازمة. إليك لمحة موجزة عن الاستيرادات الأساسية لمشروعنا:

```csharp
using Aspose.Pdf.Text;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

ستسمح لنا هذه الحزم باستخدام الوظائف القوية لمكتبة Aspose.PDF.

لنُقسّم المهمة الرئيسية إلى خطوات واضحة. باتباع هذه الخطوات، يمكنك بسهولة ملء ملفات PDF بنصوص مُحدّدة. 

## الخطوة 1: إعداد البيئة الخاصة بك

أولاً، تأكد من إعداد كل شيء بشكل صحيح في مشروع Visual Studio. أنشئ مشروعًا جديدًا أو اختر مشروعًا موجودًا. إذا كنت بحاجة إلى مساعدة، فإليك الطريقة:

1. افتح Visual Studio.
2. إنشاء مشروع C# جديد (على سبيل المثال، تطبيق وحدة التحكم).
3. انقر بزر الماوس الأيمن على المشروع في مستكشف الحلول، ثم حدد "إدارة حزم NuGet".
4. بحث عن `Aspose.PDF` وتثبيته.

## الخطوة 2: تحديد دليل المستندات الخاص بك

تحتاج كل رحلة إلى نقطة بداية، وفي حالتنا، فهي دليل المستندات حيث توجد ملفات الإدخال والإخراج. 

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يستبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يوجد به ملف PDF المدخل الخاص بك. 

## الخطوة 3: إنشاء كائن TextState

تعتبر هذه الخطوة هي المكان الذي تبدأ فيه بتحديد خصائص النص الذي تريد إضافته. 

```csharp
TextState ts = new TextState();
```

ال `TextState` سيحمل الكائن خيارات التصميم لنصك المحدد.

## الخطوة 4: تعيين لون الخط

بعد ذلك، قد ترغب في تحديد لون خط النص الخاص بك. 

```csharp
ts.StrokingColor = Color.Gray;
```

في هذا الكود، نستخدم اللون الرمادي للحدود. لا تتردد في تغيير اللون ليناسب احتياجاتك!

## الخطوة 5: تكوين وضع العرض

لتتأكد من عرض النص بالشكل المقصود، اضبط وضع العرض:

```csharp
ts.RenderingMode = TextRenderingMode.StrokeText;
```

يخبر هذا مكتبة Aspose بأننا نعمل على نص السكتة الدماغية.

## الخطوة 6: تحميل مستند PDF المدخل

الآن حان الوقت لتحميل ملف PDF الذي تريد تعديله. 

```csharp
Facades.PdfFileStamp fileStamp = new Facades.PdfFileStamp(new Aspose.Pdf.Document(dataDir + "input.pdf"));
```

تأكد من إدخال ملف PDF الخاص بك (`input.pdf`) يقع في دليل المستندات المحدد في الخطوات السابقة.

## الخطوة 7: إنشاء كائن ختم

بعد ذلك، قم بإنشاء ختم سيحمل نصك. 

```csharp
Aspose.Pdf.Facades.Stamp stamp = new Aspose.Pdf.Facades.Stamp();
```

سيتم استخدام هذا الطابع لتراكب النص الخاص بك على ملف PDF.

## الخطوة 8: تحديد النص الذي سيتم ختمه

سيتعين عليك تحديد النص الذي تريد إضافته إلى ملف PDF:

```csharp
stamp.BindLogo(new Facades.FormattedText("PAID IN FULL", System.Drawing.Color.Gray, "Arial", Facades.EncodingType.Winansi, true, 78));
```

هنا، نص "مدفوع بالكامل" الذي نضيفه، مع سمات تنسيقه. خصّصه حسب احتياجاتك!

## الخطوة 9: ربط حالة النص

الآن، قم بربط `TextState` لقد قمت بتعريفها مسبقًا للطابع. 

```csharp
stamp.BindTextState(ts);
```

تطبق هذه الخطوة كافة الأنماط مثل اللون ووضع العرض على النص الخاص بك.

## الخطوة 10: تعيين موضع الختم

حدد المكان الذي ستظهر فيه طابعتك في ملف PDF:

```csharp
stamp.SetOrigin(100, 100);
```

الحجج `(100, 100)` حدد إحداثيات X وY (بالنقاط) لأصل النص. اضبط هذه القيم لوضع نصك بشكل مثالي!

## الخطوة 11: تكوين التعتيم والدوران

وهنا يمكنك اللعب بمظهر النص الخاص بك:

```csharp
stamp.Opacity = 5;
stamp.BlendingSpace = Facades.BlendingColorSpace.DeviceRGB;
stamp.Rotation = 45.0F;
```

في هذه الحالة، تُضفي قيمة التعتيم وزاوية الدوران 45 درجة لمسةً فريدةً على نصك. لا تتردد في تعديل هذه الإعدادات للحصول على تأثيرات مختلفة.

## الخطوة 12: إضافة الختم إلى ملف PDF

هذه هي الخطوة الحاسمة حيث نضيف أخيرًا طابعتنا التي تتضمن نص السكتة الدماغية إلى ملف PDF:

```csharp
fileStamp.AddStamp(stamp);
```

وهكذا، أصبح نصك جاهزًا للإدلاء ببيان!

## الخطوة 13: حفظ وإغلاق المستند

وأخيرًا، احفظ التغييرات وتأكد من تنظيف كل شيء بشكل صحيح. 

```csharp
fileStamp.Save(dataDir + "output_out.pdf");
fileStamp.Close();
```

سيتم حفظ ملف PDF الذي تم تعديله حديثًا والذي يحتوي على نص السكتة الدماغية باسم `output_out.pdf` في دليل المستندات الخاص بك. 

## خاتمة

وهكذا تكون قد انتهيت! باتباع هذه الخطوات البسيطة، يمكنك بسهولة ملء نص في ملف PDF باستخدام Aspose.PDF لـ .NET. سواءً لمستندات العمل أو المشاريع الشخصية، تتيح لك هذه التقنية إضافة لمسة فريدة إلى ملفات PDF، مما يجعلها مميزة بين أي مجموعة أوراق.

## دليل التنفيذ

### نظرة عامة حول تضمين SVG في خلايا جدول PDF
سيرشدك هذا القسم إلى كيفية تضمين صورة SVG في خلية جدول ضمن مستند PDF. هذه الميزة مفيدة لإضافة شعارات أو أيقونات أو أي رسومات متجهية أخرى.

#### الخطوة 1: إنشاء المستند ومثيل الصورة
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء كائن مستند
Document doc = new Document();

// إنشاء مثيل صورة لـ SVG
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.FileType = Aspose.Pdf.ImageFileType.Svg; // تعيين نوع الصورة كـ SVG
img.File = dataDir + "SVGToPDF.svg"; // حدد المسار إلى ملف SVG الخاص بك
img.FixWidth = 50; // تحديد العرض لنسخة الصورة
img.FixHeight = 50; // تحديد الارتفاع لنسخة الصورة
```

#### الخطوة 2: تكوين الجدول وإضافته
```csharp
// إنشاء جدول وتكوين عرض أعمدته
Aspose.Pdf.Table table = new Aspose.Pdf.Table();
table.ColumnWidths = "100 100";

// إضافة صف إلى الجدول
Aspose.Pdf.Row row = table.Rows.Add();

// أضف الخلية الأولى التي تحتوي على النص
Aspose.Pdf.Cell cell1 = row.Cells.Add();
cell1.Paragraphs.Add(new TextFragment("First cell")); // إضافة جزء نصي إلى مجموعة فقرات الخلية

// أضف خلية ثانية وأدرج صورة SVG فيها
Aspose.Pdf.Cell cell2 = row.Cells.Add();
cell2.Paragraphs.Add(img); // إضافة صورة SVG إلى مجموعة فقرات الخلية
```

#### الخطوة 3: إدراج جدول في المستند
```csharp
// إنشاء صفحة جديدة وإضافة جدول إلى مجموعة فقراتها
Page page = doc.Pages.Add();
page.Paragraphs.Add(table);

// تعيين دليل الإخراج لحفظ ملف PDF
string outputPath = outputDir + "AddSVGObject_out.pdf";
doc.Save(outputPath); // حفظ المستند مع إضافة كائن SVG
```

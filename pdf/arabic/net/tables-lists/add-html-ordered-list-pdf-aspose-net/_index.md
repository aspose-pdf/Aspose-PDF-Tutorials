## دليل التنفيذ

الآن بعد أن قمت بإعداد بيئتك، دعنا نتعمق في تنفيذ الوظيفة لإضافة قائمة مرتبة بتنسيق HTML إلى مستند PDF.

### إضافة قائمة HTML مرتبة إلى مستند PDF

**ملخص:**  
تتيح لك هذه الميزة إدراج محتوى HTML مُهيكل، مثل القوائم، مباشرةً في ملف PDF مُنشأ حديثًا. إنها مثالية لإنشاء مستندات بمحتوى ويب ديناميكي.

#### الخطوة 1: إنشاء كائن المستند

ابدأ بإنشاء حساب جديد `Document` هدف:

```csharp
// إنشاء مثيل لفئة المستند
Document doc = new Document();
```

سيعمل هذا الكائن كحاوية لملف PDF الخاص بك.

#### الخطوة 2: تعريف محتوى HTML

أنشئ سلسلة تحتوي على محتوى HTML. يتضمن هذا المثال قائمة مرتبة:

```csharp
// تعريف محتوى HTML بقائمة مرتبة
string htmlContent = "<body style='line-height: 100px;'>\n" +
                     "<ul>\n" + 
                     "<li>First</li>\n" + 
                     "<li>Second</li>\n" + 
                     "<li>Third</li>\n" + 
                     "<li>Fourth</li>\n" + 
                     "<li>Fifth</li>\n" + 
                     "</ul>\n" +
                     "Text after the list.<br/>Next line<br/>Last line\n" + 
                     "</body>";
```

سيتم تحويل هذا السلسلة إلى تنسيق PDF.

#### الخطوة 3: إنشاء كائن HtmlFragment

تحويل محتوى HTML الخاص بك إلى `HtmlFragment` هدف:

```csharp
// إنشاء HtmlFragment بمحتوى HTML
HtmlFragment htmlFragment = new HtmlFragment(htmlContent);
```

ال `HtmlFragment` تقوم الفئة بتحليل وتنسيق سلسلة HTML لتضمينها في ملف PDF.

#### الخطوة 4: إضافة صفحة إلى المستند

أضف صفحة جديدة إلى مستندك حيث سيتم عرض القائمة:

```csharp
// إضافة صفحة فارغة إلى المستند
Page page = doc.Pages.Add();
```

وهذا يضمن أن المحتوى الخاص بنا لديه مساحة ضمن بنية PDF.

#### الخطوة 5: إضافة HtmlFragment إلى الصفحة

تشمل `HtmlFragment` في الصفحة التي تم إنشاؤها حديثًا:

```csharp
// إضافة HtmlFragment إلى مجموعة فقرات الصفحة
page.Paragraphs.Add(htmlFragment);
```

تحتوي مجموعة الفقرات على جميع عناصر الصفحة، بما في ذلك النصوص والصور.

#### الخطوة 6: حفظ المستند

أخيرًا، احفظ مستندك في المسار المحدد مع تضمين محتوى HTML الخاص بك:

```csharp
// تحديد مسار ملف الإخراج
string outFile = "YOUR_OUTPUT_DIRECTORY" + "/AddHTMLOrderedListIntoDocuments_out.pdf";

// حفظ مستند PDF
doc.Save(outFile);
```

تؤدي هذه الخطوة إلى إنشاء ملف PDF النهائي بقائمتك المرتبة.

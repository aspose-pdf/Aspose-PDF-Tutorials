# إضافة كائن خطي في ملف PDF

## مقدمة

قد يكون إنشاء ملفات PDF برمجيًا مهمة شاقة، خاصةً إذا كنت جديدًا عليها. لكن لا تقلق! مع Aspose.PDF لـ .NET، أصبحت إضافة عناصر رسومية، كالخطوط، إلى ملفات PDF سهلة للغاية. في هذا البرنامج التعليمي، سنشرح لك العملية خطوة بخطوة، مع التأكد من فهمك لكل جزء من الكود. هيا، حضّر مشروبك المفضل، ولنبدأ!

## المتطلبات الأساسية

قبل أن نبدأ، هناك بعض الأشياء التي تحتاج إلى وضعها في مكانها:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. إنه أفضل بيئة تطوير متكاملة لتطوير .NET.
2. Aspose.PDF لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك العثور عليها [هنا](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.

## استيراد الحزم

للبدء، عليك استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

1. افتح مشروع Visual Studio الخاص بك.
2. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول وحدد "إدارة حزم NuGet".
3. بحث عن `Aspose.PDF` وتثبيته.

```csharp
using System.IO;
using System;
using Aspose.Pdf;
```

بمجرد تثبيت الحزمة، يمكنك البدء في الترميز!

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، عليك تحديد مكان حفظ ملف PDF. يتم ذلك بتحديد مسار مجلد المستندات. إليك الطريقة:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يستبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي تريد حفظ ملف PDF فيه. هذا مهم جدًا، لأنه إذا كان المسار غير صحيح، فلن يُحفظ ملفك.

## الخطوة 2: إنشاء مثيل مستند

بعد ذلك، تحتاج إلى إنشاء مثيل لـ `Document` هذه الفئة تُمثّل مستند PDF الخاص بك. إليك كيفية القيام بذلك:

```csharp
// إنشاء مثيل مستند
Document doc = new Document();
```

يقوم هذا السطر من التعليمات البرمجية بإنشاء مستند PDF جديد يمكنك البدء في إضافة المحتوى إليه.

## الخطوة 3: إضافة صفحة إلى المستند

الآن وقد حصلت على مستندك، حان وقت إضافة صفحة إليه. كل ملف PDF يحتاج إلى صفحة واحدة على الأقل، أليس كذلك؟ إليك كيفية إضافة صفحة:

```csharp
// إضافة صفحة إلى مجموعة صفحات ملف PDF
Page page = doc.Pages.Add();
```

يضيف هذا الكود صفحة جديدة إلى مستندك. يمكنك اعتباره لوحة فارغة للرسم أو الكتابة.

## الخطوة 4: إنشاء مثيل للرسم البياني

لرسم أشكال مثل الخطوط، تحتاج إلى إنشاء `Graph` مثال. هنا سيتم رسم خطك. إليك كيفية إنشاء رسم بياني:

```csharp
// إنشاء مثيل للرسم البياني
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
```

في هذا المثال، تم تعيين الرسم البياني على عرض 100 وارتفاع 400. يمكنك تعديل هذه القيم بناءً على احتياجاتك.

## الخطوة 5: إضافة الرسم البياني إلى الصفحة

الآن وقد أصبح لديك الرسم البياني، حان وقت إضافته إلى الصفحة التي أنشأتها سابقًا. يتم ذلك بإضافته إلى مجموعة فقرات الصفحة:

```csharp
// إضافة كائن الرسم البياني إلى مجموعة فقرات مثيل الصفحة
page.Paragraphs.Add(graph);
```

هذه الخطوة أشبه بوضع لوحتك على الصفحة. الآن يمكنك البدء بالرسم عليها!

## الخطوة 6: إنشاء كائن خطي

بعد تثبيت الرسم البياني، يمكنك الآن إنشاء كائن خطي. هنا يمكنك تحديد نقطتي البداية والنهاية للخط. إليك كيفية القيام بذلك:

```csharp
// إنشاء مثيل الخط
Aspose.Pdf.Drawing.Line line = new Aspose.Pdf.Drawing.Line(new float[] { 100, 100, 200, 100 });
```

في هذا المثال، يبدأ الخط عند الإحداثيات (100، 100) وينتهي عند (200، 100). يمكنك تغيير هذه القيم لوضع الخط في أي مكان تريده على الرسم البياني.

## الخطوة 7: تخصيص مظهر الخط

يمكنك تخصيص مظهر خطك من خلال ضبط خصائصه. على سبيل المثال، يمكنك تحديد نمط الخط المتقطع. إليك كيفية القيام بذلك:

```csharp
// تحديد لون التعبئة لكائن الرسم البياني
line.GraphInfo.DashArray = new int[] { 0, 1, 0 };
line.GraphInfo.DashPhase = 1;
```

في هذا الكود، نقوم بإنشاء خط متقطع. `DashArray` تحدد الخاصية نمط الشرطات والفجوات، بينما `DashPhase` يحدد نقطة البداية لنمط الشرطة.

## الخطوة 8: إضافة الخط إلى الرسم البياني

الآن وقد أصبح خطك جاهزًا ومُخصّصًا، حان وقت إضافته إلى الرسم البياني. إليك كيفية القيام بذلك:

```csharp
// إضافة كائن مستطيل إلى مجموعة أشكال كائن الرسم البياني
graph.Shapes.Add(line);
```

هذه الخطوة أشبه بوضع خطك على اللوحة التي أنشأتها سابقًا. أصبح الآن جزءًا من الرسم البياني!

## الخطوة 9: حفظ ملف PDF

أخيرًا، حان وقت حفظ ملف PDF. لقد بذلتَ جهدًا كبيرًا، والآن تريد رؤية النتيجة. إليك كيفية حفظ مستندك:

```csharp
dataDir = dataDir + "AddLineObject_out.pdf";
// حفظ ملف PDF
doc.Save(dataDir);
```

هذا الكود يحفظ ملف PDF الخاص بك باسم `AddLineObject_out.pdf` في الدليل الذي حددته سابقًا. 

## الخطوة 10: تأكيد العملية

لإعلام نفسك بأن كل شيء سار بسلاسة، يمكنك طباعة رسالة تأكيد إلى وحدة التحكم:

```csharp
Console.WriteLine("\nLine object added successfully to pdf.\nFile saved at " + dataDir);
```

ستظهر هذه الرسالة في وحدة التحكم، لتأكيد إضافة خطك بنجاح.

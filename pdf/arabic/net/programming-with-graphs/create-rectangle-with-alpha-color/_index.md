# إنشاء مستطيل باستخدام لون ألفا

## مقدمة

إنشاء ملفات PDF جذابة بصريًا غالبًا ما يتطلب أكثر من مجرد إضافة نص، بل يتعلق بالتصميم باستخدام الأشكال والألوان والأنماط. من الميزات الرائعة التي يمكنك استكشافها إنشاء أشكال بألوان ألفا، مما يتيح لك إنشاء مستطيلات شفافة في ملفات PDF. في هذا البرنامج التعليمي، سنتعمق في كيفية استخدام Aspose.PDF لـ .NET لإنشاء مستطيل بلون ألفا. تخيل أن ألوان ألفا تشبه نوافذ سيارتك الملونة؛ فهي تسمح بمرور بعض الضوء مع إبقاء العناصر الأخرى مرئية. يمكن أن يضيف هذا لمسة احترافية أو يبرز مناطق مهمة في مستنداتك.

## المتطلبات الأساسية

قبل أن ننتقل إلى الكود، تأكد من أن لديك بعض الأشياء في مكانها:

1. مكتبة Aspose.PDF لـ .NET: تأكد من تثبيت Aspose.PDF لـ .NET. يمكنك تنزيله من [تنزيلات Aspose.PDF](https://releases.aspose.com/pdf/net/).
2. بيئة تطوير .NET: يجب أن يكون لديك بيئة تطوير .NET جاهزة، مثل Visual Studio.
3. الفهم الأساسي لـ C#: ستساعدك المعرفة ببرمجة C# على متابعة أمثلة التعليمات البرمجية بسهولة أكبر.

## استيراد الحزم

لبدء استخدام Aspose.PDF لـ .NET، عليك استيراد مساحات الأسماء اللازمة إلى مشروع C#. إليك كيفية القيام بذلك:

```csharp
using System.IO;
using System;
using Aspose.Pdf;
```

توفر هذه المساحات الأسماء إمكانية الوصول إلى ميزات معالجة PDF ووظائف الرسم.

لنُقسّم عملية إنشاء مستطيل بلون ألفا إلى خطوات سهلة. سيوضح لك هذا المثال كيفية إضافة مستطيل إلى ملف PDF وتحديد لونه باستخدام الشفافية.

## الخطوة 1: تهيئة المستند

أولاً، عليك إنشاء مثيل جديد لـ `Document` هذا هو مستند PDF الخاص بك الذي ستضيف إليه كل المحتوى الخاص بك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// إنشاء مثيل مستند
Document doc = new Document();
```

## الخطوة 2: إضافة صفحة إلى المستند

الآن، أضف صفحةً إلى مستند PDF. هنا سيتم وضع الأشكال والمحتوى الآخر.

```csharp
// إضافة صفحة إلى مجموعة صفحات ملف PDF
Aspose.Pdf.Page page = doc.Pages.Add();
```

## الخطوة 3: إنشاء مثيل للرسم البياني

ال `Graph` تتيح لك هذه الفئة رسم أشكال على ملف PDF. هنا، ننشئ رسمًا بيانيًا بأبعاد محددة تناسب الصفحة.

```csharp
// إنشاء مثيل للرسم البياني
Aspose.Pdf.Drawing.Graph canvas = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
```

## الخطوة 4: تحديد المستطيل الأول وإضافته

أنشئ مستطيلاً بأبعاد محددة، واضبط لون تعبئته باستخدام قيمة ألفا. هذا يجعل اللون شفافًا جزئيًا.

```csharp
// إنشاء كائن مستطيل بأبعاد محددة
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 100);
// تعيين لون تعبئة الرسم البياني من بنية System.Drawing.Color من قيمة ARGB 32 بت
rect.GraphInfo.FillColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.FromArgb(128, System.Drawing.Color.FromArgb(12957183)));
// إضافة كائن مستطيل إلى مجموعة الأشكال الخاصة بمثيل الرسم البياني
canvas.Shapes.Add(rect);
```

## الخطوة 5: تحديد وإضافة مستطيل ثانٍ

وبالمثل، أنشئ مستطيلاً آخر بأبعاد وألوان مختلفة. يمكنك تجربة قيم ألفا وألوان مختلفة لرؤية تأثيرات متنوعة.

```csharp
// إنشاء كائن مستطيل ثانٍ
Aspose.Pdf.Drawing.Rectangle rect1 = new Aspose.Pdf.Drawing.Rectangle(200, 150, 200, 100);
rect1.GraphInfo.FillColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.FromArgb(128, System.Drawing.Color.FromArgb(16118015)));
canvas.Shapes.Add(rect1);
```

## الخطوة 6: إضافة الرسم البياني إلى الصفحة

بمجرد تحديد الأشكال الخاصة بك، أضف `Graph` إضافة كائن إلى مجموعة فقرات الصفحة. سيؤدي هذا إلى دمج رسمك في صفحة PDF.

```csharp
// إضافة مثيل الرسم البياني إلى مجموعة فقرات كائن الصفحة
page.Paragraphs.Add(canvas);
```

## الخطوة 7: حفظ المستند

أخيرًا، احفظ مستند PDF في المسار المحدد. سيؤدي هذا إلى إنشاء ملف PDF بالمستطيلات التي أنشأتها.

```csharp
dataDir = dataDir + "CreateRectangleWithAlphaColor_out.pdf";
// حفظ ملف PDF
doc.Save(dataDir);
Console.WriteLine("\nRectangle object created successfully with alpha color.\nFile saved at " + dataDir);
```

## دليل التنفيذ

### تحويل صفحات PDF إلى PNG

#### ملخص
يمكن أن يكون تحويل كل صفحة من مستند PDF إلى ملفات PNG عالية الدقة مفيدًا لتطبيقات مثل النشر على الويب وحلول التخزين الرقمي.

##### الخطوة 1: تحديد الدلائل
أولاً، قم بتعيين المسارات لملف PDF المصدر ودليل الإخراج:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

تأكد من وجود هذه الدلائل أو قم بتحديثها وفقًا لهيكل مشروعك.

##### الخطوة 2: تحميل المستند
قم بتحميل مستند PDF الخاص بك باستخدام Aspose.PDF `Document` فصل:

```csharp
document pdfDocument = new Document(dataDir + "/ConvertAllPagesToPNG.pdf");
```

تعمل هذه الخطوة على تهيئة عملية التحويل من خلال الوصول إلى ملف PDF المطلوب.

##### الخطوة 3: تحويل كل صفحة
قم بتكرار كل صفحة، وتحويلها إلى ملفات PNG:

```csharp
for (int pageCount = 1; pageCount <= pdfDocument.Pages.Count; pageCount++)
{
    string imagePath = Path.Combine(outputDir, "image" + pageCount + "_out.png");
    
    using (FileStream imageStream = new FileStream(imagePath, FileMode.Create))
    {
        Resolution resolution = new Resolution(300); // ضبط على DPI عالية للجودة
        PngDevice pngDevice = new PngDevice(resolution);
        
        pngDevice.Process(pdfDocument.Pages[pageCount], imageStream);
    }
}
```
- **دقة**:يضمن إعداد 300 نقطة في البوصة الحصول على صور واضحة ونقية.
- **جهاز PNG**:يتولى عملية التحويل من PDF إلى PNG بإعدادات الدقة المحددة.

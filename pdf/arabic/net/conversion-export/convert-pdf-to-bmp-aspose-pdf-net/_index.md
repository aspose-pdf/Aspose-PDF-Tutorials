## دليل التنفيذ

الآن، دعونا نقوم بتحويل صفحات PDF إلى صور BMP.

### تحويل صفحات PDF إلى صور BMP

تعمل هذه الميزة على تحويل كل صفحة PDF إلى ملفات صور BMP منفصلة، وهي مفيدة للتنسيقات الجاهزة للطباعة أو المحتوى الرقمي عالي الدقة.

#### الخطوة 1: إعداد الدلائل الخاصة بك

قم بتحديد الدلائل لملف PDF المصدر والصور الناتجة:

```csharp
private const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
private const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
```
استبدال العناصر النائبة بالمسارات الفعلية.

#### الخطوة 2: تحميل مستند PDF الخاص بك

قم بتحميل المستند باستخدام Aspose.PDF `Document` فصل:

```csharp
Document pdfDocument = new Document(DocumentDirectory + "/AddImage.pdf");
```

سيؤدي هذا إلى تحميل "AddImage.pdf" من الدليل المحدد.

#### الخطوة 3: تكوين إعدادات الإخراج

ضبط الدقة للصور الناتجة وإنشاء `BmpDevice`:

```csharp
Resolution resolution = new Resolution(300); // اضبط DPI المطلوب، على سبيل المثال، 300 DPI للصور عالية الجودة
BmpDevice bmpDevice = new BmpDevice(resolution);
```

#### الخطوة 4: تحويل كل صفحة إلى صورة BMP

قم بتكرار كل صفحة PDF وتحويلها إلى صورة BMP:

```csharp
for (int pageCount = 1; pageCount <= pdfDocument.Pages.Count; pageCount++)
{
    string outputPath = Path.Combine(OutputDirectory, "image" + pageCount + "_out.bmp");

    using (FileStream imageStream = new FileStream(outputPath, FileMode.Create))
    {
        bmpDevice.Process(pdfDocument.Pages[pageCount], imageStream);
    }
}
```

تعمل هذه الحلقة على معالجة كل صفحة وحفظها كملف BMP في دليل الإخراج المحدد.

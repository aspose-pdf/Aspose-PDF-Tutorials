## دليل التنفيذ

الآن، دعنا نقوم بتحويل ملف XPS إلى PDF باستخدام Aspose.PDF لـ .NET.

### تحويل XPS إلى PDF: خطوة بخطوة
**1. قم بإعداد مساراتك**
قم بتحديد المسارات لملفات الإدخال والإخراج الخاصة بك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // مسار دليل الإدخال
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // مسار دليل الإخراج
```

**2. قم بتحميل مستند XPS**
إنشاء `XpsLoadOptions` كائن لتحديد خيارات التحميل، ثم إنشاء مثيل له `Document` مع هذه الإعدادات:
```csharp
// استخدم XpsLoadOptions لتحميل مستندات XPS.
Aspose.Pdf.LoadOptions options = new Aspose.Pdf.XpsLoadOptions();

// قم بتحميل مستند XPS إلى كائن المستند.
Aspose.Pdf.Document document = new Aspose.Pdf.Document(dataDir + "/XPSToPDF.xps", options);
```

**3. احفظ بصيغة PDF**
وأخيرا، احفظ `Document` كملف PDF إلى دليل الإخراج المطلوب:
```csharp
// تحويل ملف XPS وحفظه بصيغة PDF.
document.Save(outputDir + "/XPSToPDF_out.pdf");
```

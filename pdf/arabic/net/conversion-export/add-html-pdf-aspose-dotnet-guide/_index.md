## دليل التنفيذ

بعد أن أصبحت بيئتنا جاهزة، دعنا نستكشف كيفية تنفيذ إضافة HTML باستخدام DOM والوضع الدقيق داخل صفحات PDF.

### الميزة 1: إضافة HTML باستخدام DOM
تتيح لك هذه الميزة دمج محتوى HTML الغني في مستندات PDF.

#### ملخص
أضف HTML ديناميكيًا لإنشاء ملفات PDF جذابة بصريًا مع نص منسق أو صور أو جداول مصممة.

#### خطوات التنفيذ

**الخطوة 1: تهيئة المستند**
```csharp
// إنشاء كائن مستند جديد
Document doc = new Document();
```

**الخطوة 2: إضافة صفحة**
```csharp
// إضافة صفحة إلى مجموعة صفحات المستند
Page page = doc.Pages.Add();
```

**الخطوة 3: إنشاء جزء HTML**
```csharp
// قم بتهيئة HtmlFragment بالمحتوى HTML المطلوب
HtmlFragment title = new HtmlFragment("<fontsize=10><b><i>Table</i></b></fontsize>");
```

**الخطوة 4: تكوين الهوامش**
```csharp
// تعيين الهوامش للتحكم في الموضع
title.Margin.Bottom = 10;
title.Margin.Top = 200;
```

**الخطوة 5: إضافة جزء HTML إلى الصفحة**
```csharp
// إدراج جزء HTML في مجموعة فقرات الصفحة
page.Paragraphs.Add(title);
```

**الخطوة 6: حفظ المستند**
```csharp
// تحديد مسار الإخراج وحفظ المستند
string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/AddHTMLUsingDOM_out.pdf";
doc.Save(dataDir);
```

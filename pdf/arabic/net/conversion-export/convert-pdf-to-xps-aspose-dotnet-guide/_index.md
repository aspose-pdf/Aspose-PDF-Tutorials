## دليل التنفيذ
يتناول هذا القسم كيفية تحويل مستند PDF إلى ملف XPS باستخدام Aspose.PDF لـ .NET.

### تحويل PDF إلى XPS
قم بتحويل ملفات PDF الموجودة لديك إلى تنسيق XPS بسهولة:

#### الخطوة 1: تحميل مستند PDF
قم بتحميل ملف PDF الذي يتطلب التحويل:
```csharp
using System;
using Aspose.Pdf;

// حدد دليل الإدخال لملف PDF الخاص بك
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";

document = new Document(inputDirectory + "/input.pdf");
```

#### الخطوة 2: تكوين خيارات حفظ XPS
إعداد الخيارات المطلوبة للحفظ بتنسيق XPS:
```csharp
// إنشاء خيارات حفظ XPS
var saveOptions = new Aspose.Pdf.XpsSaveOptions();
```

#### الخطوة 3: الحفظ كمستند XPS
تحويل ملف PDF وحفظه إلى ملف XPS:
```csharp
// قم بتحديد دليل الإخراج لملف XPS المُحوّل
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

document.Save(outputDirectory + "/PDFToXPS_out.xps", saveOptions);
```
**توضيح**: ال `XpsSaveOptions` تسمح الفئة بالتخصيص المخصص لتحويل XPS.

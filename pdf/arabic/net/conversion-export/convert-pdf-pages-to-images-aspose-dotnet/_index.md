### تحويل صفحة PDF إلى صورة

#### ملخص

تتيح لك هذه الميزة تحويل صفحة محددة من مستند PDF إلى صورة، مما يسمح بتخصيص عرض النص من خلال إعدادات الخط.

#### التنفيذ خطوة بخطوة

**1. قم بتحميل مستند PDF**

```csharp
// قم بتحميل ملف PDF الخاص بك باستخدام Aspose.PDF
using (Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf"))
{
    // متابعة خطوات التحويل داخل هذه الكتلة
}
```

*توضيح:* يقوم هذا الكود بتهيئة `Document` الكائن، وهو أمر ضروري للوصول إلى صفحات ملف PDF.

**2. تكوين إعدادات الإخراج**

```csharp
// إعداد تدفق الإخراج والدقة
using (FileStream imageStream = new FileStream("YOUR_OUTPUT_DIRECTORY/SetDefaultFontName.png", FileMode.Create))
{
    Resolution resolution = new Resolution(300);
    PngDevice pngDevice = new PngDevice(resolution);

    // خيارات العرض لإعدادات الخط
    RenderingOptions ro = new RenderingOptions();
    ro.DefaultFontName = "Arial";  // قم بتعيين الخط الافتراضي المطلوب

    // تطبيق خيارات العرض على الجهاز
    pngDevice.RenderingOptions = ro;
}
```

*توضيح:* هنا، نقوم بتكوين `FileStream` ووضع القرار. `RenderingOptions` تسمح لنا الفئة بتحديد خط افتراضي لعناصر النص أثناء التحويل.

**3. قم بإجراء التحويل**

```csharp
// تحويل الصفحة الأولى من ملف PDF إلى صورة
pngDevice.Process(pdfDocument.Pages[1], imageStream);
```

*توضيح:* وأخيرًا، نقوم بتحويل الصفحة المحددة وحفظها كصورة باستخدام `PngDevice`.

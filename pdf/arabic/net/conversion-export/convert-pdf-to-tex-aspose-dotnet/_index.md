## دليل التنفيذ: تحويل PDF إلى TeX

يوضح هذا الدليل كيفية تحويل مستند PDF إلى تنسيق TeX باستخدام Aspose.PDF لـ .NET:

### الخطوة 1: تحميل مستند PDF الخاص بك

إنشاء `Aspose.Pdf.Document` الكائن من ملف PDF الحالي لديك.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
document doc = new Document(dataDir + "/PDFToTeX.pdf");
```
**توضيح**:يؤدي هذا إلى تهيئة كائن المستند للتحويل إلى تنسيق TeX.

### الخطوة 2: تكوين TeXSaveOptions

إنشاء مثيل `TeXSaveOptions` لتحديد إعدادات التحويل.
```csharp
TeXSaveOptions saveOptions = new TeXSaveOptions();
saveOptions.OutDirectoryPath = "YOUR_OUTPUT_DIRECTORY";
```
**توضيح**: ال `OutDirectoryPath` يُحدد دليل الإخراج لحفظ الملف المُحوّل. هذه الخطوة أساسية لتحديد مكان حفظ الملف المُحوّل. `.tex` سيتم حفظ الملفات بعد التحويل.

### الخطوة 3: حفظ المستند بتنسيق TeX

احفظ ملف PDF كمستند TeX عن طريق تحديد مسار الإخراج والخيارات.
```csharp
doc.Save("YOUR_OUTPUT_DIRECTORY/PDFToTeX_out.tex", saveOptions);
```
**توضيح**:يؤدي هذا إلى حفظ ملف PDF الخاص بك بتنسيق TeX باستخدام `saveOptions`يتم حفظ الملف في الدليل المحدد باسم الملف المناسب.

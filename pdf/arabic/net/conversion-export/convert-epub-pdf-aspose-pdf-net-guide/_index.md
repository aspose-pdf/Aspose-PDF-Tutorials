## دليل التنفيذ

### الميزة: تحويل EPUB إلى PDF

يمكن تحويل ملف EPUB إلى PDF بخطوات قليلة. سنشرح كل خطوة بالتفصيل، ونشرح غرضها وطريقة تنفيذها.

#### الخطوة 1: تحديد خيارات التحميل لـ EPUB
لبدء عملية التحويل، قم بتحديد خيارات التحميل المخصصة لملفات EPUB باستخدام `EpubLoadOptions`.

```csharp
// قم بإنشاء مثيل لـ EpubLoadOptions لتحديد معلمات التحميل.
EpubLoadOptions epubload = new EpubLoadOptions();
```

#### الخطوة 2: تحميل ملف EPUB
باستخدام Aspose.PDF، أنشئ كائن مستند يُحمّل ملف EPUB الخاص بك. `Document` تتعامل الفئة مع عملية التحويل.

```csharp
// قم بتحميل ملف EPUB بالخيارات المحددة في Aspose.Pdf.Document.
Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document(dataDir + "/EPUBToPDF.epub", epubload);
```

#### الخطوة 3: الحفظ بتنسيق PDF
أخيرًا، احفظ المستند المحمّل بتنسيق PDF في دليل الإخراج المخصص لك.

```csharp

```

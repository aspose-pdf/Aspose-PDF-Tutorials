## تعليمات التثبيت:  

لاستخدام Aspose.PDF لـ .NET في مشروعك، اتبع هذه الخطوات:  

1. تثبيت عن طريق NuGet Package Manager  
* افتح ويزويل ستوديو وافعل مشروع .NET جديد  
* افتح وحدة التحكم في حزمة (أدوات → NuGet وحدة التحكم في حزمة → وحدة التحكم في حزمة) وافعل:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. تثبيت عبر .NET CLI  
بدلاً من ذلك، استخدم .NET CLI لتثبيت الحزمة:  
```bash
dotnet add package Aspose.PDF
```

3. إضافة مساحة أسماء في رمز C # الخاص بك  
لاستخدام وظيفة Aspose.PDF، تضم المساحات الاسمية التالية:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

## دليل التنفيذ

في هذا القسم، سنقوم بتوضيح عملية إضافة تعليق خطي مقاس إلى مستند PDF باستخدام Aspose.PDF .NET.

### إضافة تعليق توضيحي للخط مع القياس

#### ملخص

تتيح لك إضافة تعليق سطري تحديد أقسام محددة في ملف PDF وقياس المسافات. تُعد هذه الميزة مفيدة بشكل خاص للوثائق التقنية التي تتطلب الدقة.

#### خطوات التنفيذ

1. **تحميل المستند**

   ابدأ بتحميل مستند PDF الموجود الذي تريد إضافة التعليقات التوضيحية إليه.

   ```csharp
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY/input.pdf";
   Document doc = new Document(dataDir);
   ```

2. **تحديد منطقة التعليق التوضيحي**

   قم بتحديد منطقة المستطيل على صفحتك حيث سيظهر التعليق التوضيحي.

   ```csharp
   Rectangle rect = new Rectangle(260, 630, 451, 662); // تحديد إحداثيات الشرح التوضيحي
   ```

3. **إنشاء تعليق توضيحي للسطر**

   إنشاء `LineAnnotation` كائن به نقاط بداية ونهاية ضمن منطقة المستطيل المحددة.

   ```csharp
   LineAnnotation line = new LineAnnotation(doc.Pages[1], rect, new Point(266, 657), new Point(446, 656));
   ```

4. **تكوين المظهر**

   قم بتعيين اللون وخطوط القيادة والأنماط للتعليق التوضيحي.

   ```csharp
   line.Color = Color.Red; // تعيين لون التعليق التوضيحي إلى اللون الأحمر
   line.LeaderLine = -15; // إزاحة خط القائد من نقطة البداية
   line.LeaderLineExtension = 5; // طول امتداد خط القائد
   line.StartingStyle = LineEnding.OpenArrow;
   line.EndingStyle = LineEnding.OpenArrow;
   ```

5. **تعيين تفاصيل القياس**

   استخدم `Measure` كائن لتحديد تفاصيل القياس.

   ```csharp
   line.Measure = new Measure(line);
   line.Measure.DistanceFormat.Add(new Measure.NumberFormat(line.Measure));
   line.Measure.DistanceFormat[1].UnitLabel = "mm"; // تعيين تسمية الوحدة للقياسات
   ```

6. **أضف التسمية التوضيحية واحفظها**

   أضف تعليقًا لعرض القياس وحفظ مستندك.

   ```csharp
   line.Contents = "155 mm";
   line.ShowCaption = true;
   line.CaptionPosition = CaptionPosition.Top;

   doc.Pages[1].Annotations.Add(line); // يضيف التعليق التوضيحي إلى الصفحة 1

   string outputDir = @"YOUR_OUTPUT_DIRECTORY/UseMeasureWithLineAnnotation_out.pdf";
   doc.Save(outputDir);
   ```

## دليل التنفيذ

سيرشدك هذا القسم إلى كيفية إضافة حقول نصية إلى نماذج PDF الحالية باستخدام Aspose.PDF لـ .NET. كل ميزة مُقسّمة إلى خطوات واضحة.

### تحديد حقول النموذج

#### ملخص:
أولاً، نحتاج إلى تحديد حقول النموذج واستخراجها من مستند PDF موجود.

**الخطوة 1: تحميل مستند PDF الخاص بك**
```csharp
string dataDir = "path/to/your/pdf/";
Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form(dataDir + "FilledForm.pdf");
```

**الخطوة 2: استرداد جميع أسماء الحقول**
```csharp
String[] allfields = form.FieldNames;
```

#### توضيح:
يقوم هذا الكود بتحميل ملف PDF واسترجاع أسماء جميع الحقول، مما يوفر أساسًا لمزيد من التعديل.

### إضافة حقول نصية جديدة

#### ملخص:
الآن بعد أن حددنا حقول النماذج الموجودة، فلنقم بإضافة حقول نصية جديدة إلى هذه المواقع.

**الخطوة 1: إعداد الإحداثيات**
```csharp
System.Drawing.Rectangle[] box = new System.Drawing.Rectangle[allfields.Length];
for (int i = 0; i < allfields.Length; i++)
{
    FormFieldFacade facade = form.GetFieldFacade(allfields[i]);
    box[i] = facade.Box;
}
```

**الخطوة 2: إنشاء حقول نصية جديدة وإضافتها**
```csharp
Document doc = new Document(dataDir + "FilledForm - 2.pdf");
FormEditor editor = new FormEditor(doc);

for (int i = 0; i < allfields.Length; i++)
{
    editor.AddField(FieldType.Text, "TextField" + i, allfields[i], 1,
                    box[i].Left, box[i].Top, box[i].Left + 50, box[i].Top + 10);
}
editor.Save(dataDir + "IdentifyFormFields_out.pdf");
```

#### توضيح:
تضيف هذه القطعة حقول نصية جديدة فوق كل حقل نموذج موجود باستخدام الإحداثيات التي تم الحصول عليها سابقًا.

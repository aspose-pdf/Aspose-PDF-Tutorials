## دليل التنفيذ

دعونا نقسم التنفيذ إلى خطوات قابلة للإدارة.

### الميزة 1: تكوين معلومات الخط

#### ملخص
في هذا القسم، نقوم بتكوين خصائص الخط، مثل إحداثيات الرأس، والرؤية، واللون، والعرض باستخدام `LineInfo` فصل.

**الخطوة 1**:تحديد خصائص الخط
```csharp
// تكوين إحداثيات الرأس للخط
LineInfo lineInfo = new LineInfo();
lineInfo.VerticeCoordinate = new float[] { 55, 55, 70, 70, 70, 90, 150, 60 };
lineInfo.Visibility = true;
lineInfo.LineColor = System.Drawing.Color.Red;
lineInfo.LineWidth = 2;

// احسب عدد النقاط من إحداثيات الرأس
int length = lineInfo.VerticeCoordinate.Length / 2;
Aspose.Pdf.Point[] gesture = new Aspose.Pdf.Point[length];
for (int i = 0; i < length; i++) {
    // إنشاء نقطة لكل زوج من الإحداثيات
    gesture[i] = new Aspose.Pdf.Point(lineInfo.VerticeCoordinate[2 * i], lineInfo.VerticeCoordinate[2 * i + 1]);
}
```

**توضيح**: 
- ال `VerticeCoordinate` تعرف المصفوفة مسار الخط باستخدام إحداثيات x و y.
- قمنا بضبط الخط ليكون مرئيًا، ملونًا باللون الأحمر، وبعرض وحدتين.

### الميزة 2: إنشاء وتكوين التعليقات التوضيحية بالحبر

#### ملخص
بعد ذلك، نقوم بإنشاء تعليق حبر يستخدم الخط الذي قمنا بتكوينه كعنصر رسومي.

**الخطوة 2**:إنشاء وتكوين التعليقات التوضيحية بالحبر
```csharp
using Aspose.Pdf.Annotations;

IList<Point[]> inkList = new List<Point[]>();
inkList.Add(gesture);

Document doc = new Document();
doc.Pages.Add();

InkAnnotation a1 = new InkAnnotation(doc.Pages[1], new Aspose.Pdf.Rectangle(100, 100, 300, 300), inkList);
a1.Subject = "Test";
a1.Title = "Title";
a1.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);

doc.Pages[1].Annotations.Add(a1);
```

**توضيح**: 
- نضيف تكوين الخط إلى تعليق الحبر الجديد.
- تعيين خصائص مثل الموضوع والعنوان واللون.

### الميزة 3: تكوين حدود التعليقات التوضيحية

#### ملخص
قم بتخصيص تعليقاتك التوضيحية باستخدام أنماط حدود محددة لتحسين المظهر المرئي.

**الخطوة 3**:تكوين حدود التعليقات التوضيحية
```csharp
using Aspose.Pdf.Facades;

Border border = new Border(a1);
border.Width = 3;
border.Effect = BorderEffect.Cloudy;
border.Dash = new Dash(1, 1); // نمط لوحة القيادة وحدة واحدة قيد التشغيل ووحدة واحدة متوقفة
border.Style = BorderStyle.Solid;
```

**توضيح**: 
- نحن نطبق حدودًا صلبة بتأثير "غائم" وأنماط شرطات محددة.

وأخيرًا، احفظ المستند:

```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/lnkAnnotationLineWidth_out.pdf";
doc.Save(dataDir);
```

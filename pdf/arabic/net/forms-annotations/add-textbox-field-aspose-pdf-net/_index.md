## دليل التنفيذ
### إضافة حقل مربع نص
يوضح هذا القسم كيفية إضافة حقل مربع نص إلى مستند PDF الخاص بك.

#### ملخص
تتيح حقول مربع النص للمستخدمين إدخال البيانات بشكل مباشر، مما يسهل عمليات إرسال النماذج وجمع البيانات.

##### الخطوة 1: افتح مستند PDF الخاص بك
قم بتحميل ملف PDF موجود باستخدام Aspose.PDF:
```csharp
// المسار إلى دليل المستندات.
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();

// فتح المستند
Document pdfDocument = new Document(dataDir + "TextField.pdf");
```

##### الخطوة 2: إنشاء وتكوين TextBoxField
إنشاء حقل مربع نص يحدد موضعه:
```csharp
// إنشاء حقل
TextBoxField textBoxField = new TextBoxField(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(100, 200, 300, 300));
textBoxField.PartialName = "textbox1";
textBoxField.Value = "Text Box";
```
**توضيح:**
- `PartialName`:معرف فريد للحقل.
- `Value`:يتم عرض النص الافتراضي في المربع.

##### الخطوة 3: تخصيص المظهر
قم بتخصيص مربع النص الخاص بك باستخدام الحدود والألوان:
```csharp
// تكوين الحدود
Border border = new Border(textBoxField)
{
    Width = 5,
    Dash = new Dash(1, 1) // خط متقطع
};
textBoxField.Border = border;
textBoxField.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
```
**خيارات تكوين المفتاح:**
- `Width`:سمك الحدود.
- `Dash`:نمط للخطوط المتقطعة.
- `Color`:لون تعبئة مربع النص.

##### الخطوة 4: إضافة الحقل إلى المستند
أضف حقل مربع النص الذي قمت بتكوينه:
```csharp
// إضافة حقل إلى المستند
pdfDocument.Form.Add(textBoxField, 1);
```

##### الخطوة 5: احفظ ملف PDF المعدّل
حفظ التغييرات في ملف جديد:
```csharp
dataDir = dataDir + "TextBox_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nTextbox field added successfully.\nFile saved at " + dataDir);
```

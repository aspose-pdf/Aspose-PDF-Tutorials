## دليل التنفيذ
في هذا القسم، سنركز على ميزتين رئيسيتين: إنشاء FreeTextAnnotations مع خصائص التعليق التوضيحي واستيراد تعليقات XFDF.
### إنشاء FreeTextAnnotation مع خصائص التعليق التوضيحي
#### ملخص
تتضمن إضافة FreeTextAnnotation ضبط مظهرها وتحديد خصائص مثل لون النص وحجم الخط. تُحسّن ميزة التعليق التوضيحي هذه الميزة برسم خطوط تُشير إلى أجزاء مُحددة من محتوى PDF.
#### خطوات التنفيذ
**الخطوة 1: إعداد المظهر الافتراضي**
قم بتحديد مظهر التعليق التوضيحي باستخدام `DefaultAppearance`:
```csharp
// تكوين إعدادات المظهر الافتراضية للتعليق التوضيحي
da = new DefaultAppearance()
{
    TextColor = Color.Red,
    FontSize = 10
};
```
**الخطوة 2: إنشاء التعليقات التوضيحية وتكوينها**
تهيئة `FreeTextAnnotation`، مع تحديد موقعه وخصائصه ومحتواه:
```csharp
// إنشاء FreeTextAnnotation مع خصائص توضيحية محددة
fta = new FreeTextAnnotation(page, 
    new Rectangle(422.25, 645.75, 583.5, 702.75), da)
{
    Intent = FreeTextIntent.FreeTextCallout,
    EndingStyle = LineEnding.OpenArrow,
    Callout = new Point[]
    {
        new Point(428.25, 651.75),
        new Point(462.75, 681.375),
        new Point(474, 681.375)
    }
};

// أضف التعليق التوضيحي إلى الصفحة
page.Annotations.Add(fta);

// تعيين محتوى RichText للتعليق التوضيحي
fta.RichText = "<body xmlns=\"http://www.w3.org/1999/xhtml\" ... >هذه عينة</body>";
```
**الخطوة 3: حفظ المستند**
وأخيرًا، احفظ مستندك للحفاظ على التغييرات:
```csharp
// احفظ ملف PDF الموضح
doc.Save("YOUR_OUTPUT_DIRECTORY/SetCalloutProperty.pdf");
```
### استيراد التعليقات التوضيحية من ملف XFDF
#### ملخص
يتيح لك استيراد التعليقات التوضيحية إضافة تعليقات توضيحية مُعرّفة مسبقًا إلى ملف PDF جديد أو موجود. تُبسّط هذه العملية باستخدام XFDF، وهو تنسيق مُصمّم خصيصًا لإضافة التعليقات التوضيحية إلى المستندات.
#### خطوات التنفيذ
**الخطوة 1: تحميل مستند PDF الموجود**
افتح المستند المستهدف الذي تريد استيراد التعليقات التوضيحية فيه:
```csharp
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/AddAnnotation.pdf");
```
**الخطوة 2: إنشاء محتوى XFDF**
إنشاء منشئ سلسلة بمحتوى XFDF، وتحديد خصائص التعليق التوضيحي:
```csharp
StringBuilder Xfdf = new StringBuilder();
Xfdf.AppendLine("<?xml version=\"1.0\" encoding=\"UTF-8\"?><xfdf ... >");

// تعريف FreeTextAnnotation بتنسيق XFDF
CreateXfdf(ref Xfdf);
Xfdf.AppendLine("</xfdf>");
```
**الخطوة 3: استيراد التعليقات التوضيحية**
استخدم `ImportAnnotationsFromXfdf` طريقة إضافة التعليقات التوضيحية من بيانات XFDF:
```csharp
pdfDocument.ImportAnnotationsFromXfdf(new MemoryStream(Encoding.UTF8.GetBytes(Xfdf.ToString())));
```
**الخطوة 4: حفظ المستند المحدث**
قم بتخزين التغييرات عن طريق حفظ المستند مرة أخرى:
```csharp
// حفظ ملف PDF مع التعليقات التوضيحية المستوردة
doc.Save("YOUR_OUTPUT_DIRECTORY/SetCalloutPropertyXFDF.pdf");
```
#### طريقة المساعدة
ال `CreateXfdf` تقوم الطريقة بإنشاء عناصر XML اللازمة للتعليق التوضيحي:
```csharp
static void CreateXfdf(ref StringBuilder pXfdf)
{
    // إضافة تفاصيل FreeTextAnnotation بتنسيق XFDF
    pXfdf.Append("<freetext page=\"0\" rect=\"422.25,645.75,583.5,702.75\" ... >");
    
    // إضافة محتويات RichText وإعدادات المظهر الافتراضية
    pXfdf.Append("<contents-richtext><body style=\"font-size:10.0pt;...");
    pXfidf.AppendLine("</body></contents-richtext>");
    pXfidf.AppendLine("<defaultappearance>/Helv 12 Tf 1 0 0 rg</defaultappearance>");
    pXfdf.AppendLine("</freetext>");
}
```

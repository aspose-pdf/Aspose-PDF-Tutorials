## دليل التنفيذ

الآن بعد أن قمت بإعداد كل شيء، دعنا ننفذ ميزة إضافة أزرار الاختيار إلى ملف PDF.

### إنشاء مستند جديد باستخدام أزرار الاختيار

**ملخص:**
سنبدأ بإنشاء مستند PDF فارغ ثم نضيف صفحة يمكننا وضع حقول أزرار الاختيار الخاصة بنا فيها.

#### الخطوة 1: تهيئة مستند جديد

ابدأ بإعداد مشروعك بالمساحات الأساسية الضرورية:

```csharp
using System;
using Aspose.Pdf;
```

إنشاء مثيل لـ `Document` الفئة التي تمثل ملف PDF:

```csharp
// إنشاء مستند جديد
Document doc = new Document();
Page page = doc.Pages.Add(); // إضافة صفحة جديدة إلى المستند
```

#### الخطوة 2: إضافة حقول أزرار الاختيار

سنقوم الآن بإضافة حقول أزرار الاختيار إلى صفحة PDF التي تم إنشاؤها حديثًا.

**إنشاء وتكوين حقل زر الاختيار:**

```csharp
// إنشاء حقل زر راديو في الصفحة الأولى
RadioButtonField field = new RadioButtonField(page);
field.Rect = new Aspose.Pdf.Rectangle(40, 650, 100, 720); // تعيين الموضع والحجم
field.PartialName = "NewField"; // تعيين اسم للمرجع
```

#### الخطوة 3: إضافة خيارات أزرار الاختيار

قم بتحديد الخيارات المتاحة في حقل زر الاختيار الخاص بك:

```csharp
// إنشاء خيارات أزرار الراديو وتعيين الخصائص
RadioButtonOptionField opt1 = new RadioButtonOptionField();
opt1.Rect = new Aspose.Pdf.Rectangle(40, 650, 60, 670);
opt1.OptionName = "Item1"; // تسمية الخيار
opt1.Border = new Border(opt1) { Width = 1 };
opt1.Characteristics.Border = System.Drawing.Color.Black;

RadioButtonOptionField opt2 = new RadioButtonOptionField();
opt2.Rect = new Aspose.Pdf.Rectangle(60, 670, 80, 690);
opt2.OptionName = "Item2";
opt2.Border = new Border(opt2) { Width = 1 };
opt2.Characteristics.Border = System.Drawing.Color.Black;

RadioButtonOptionField opt3 = new RadioButtonOptionField();
opt3.Rect = new Aspose.Pdf.Rectangle(80, 690, 100, 710);
opt3.OptionName = "Item3";
opt3.Border = new Border(opt3) { Width = 1 };
opt3.Characteristics.Border = System.Drawing.Color.Black;

// إضافة خيارات إلى حقل زر الاختيار
field.Add(opt1);
field.Add(opt2);
field.Add(opt3);

// أضف RadioButtonField إلى النموذج
doc.Form.Add(field);
```

#### الخطوة 4: حفظ المستند

وأخيرًا، احفظ مستندك باستخدام أزرار الاختيار المضافة حديثًا:

```csharp
string dataDir = "path/to/your/directory/";
dataDir += "CreateDoc_out.pdf"; // تحديد مسار الإخراج

// حفظ المستند
doc.Save(dataDir);

Console.WriteLine("\nNew doc with 3 items radio button created successfully.\nFile saved at " + dataDir);
```

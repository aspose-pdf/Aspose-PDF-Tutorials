## دليل التنفيذ

### الميزة 1: إنشاء المستندات باستخدام النص

#### ملخص
توضح هذه الميزة كيفية إنشاء مستند PDF جديد يحتوي على كتلة نصية أولية تمهد الطريق لمزيد من تحسينات التفاعلية.

#### التنفيذ خطوة بخطوة

**1. إنشاء مستند جديد وحفظه**
```csharp
using Aspose.Pdf;

// إنشاء مستند نموذجي يحتوي على نص
Document doc = new Document();
doc.Pages.Add().Paragraphs.Add(new TextFragment("Move the mouse cursor here to display floating text"));
doc.Save(@"YOUR_DOCUMENT_DIRECTORY\TextBlock_HideShow_MouseOverOut_out.pdf");
```

- **توضيح:** نبدأ بإنشاء `Document` الكائن وإضافة صفحة. أ `TextFragment` يتم بعد ذلك إضافتها إلى هذه الصفحة، والتي تحتوي على تعليمات للتفاعل مع المستخدم.

### الميزة 2: تحميل مستند وإنشاء حقل نص مخفي

#### ملخص
تتضمن هذه الميزة تحميل المستند الذي تم إنشاؤه مسبقًا، وتحديد موقع نص معين، وتضمين حقل نص مخفي يصبح مرئيًا عند تحريك الماوس فوقه.

#### التنفيذ خطوة بخطوة

**1. تحميل المستند الموجود**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
using Aspose.Pdf.Forms;
using System.Drawing;

// افتح المستند المحفوظ مسبقًا بالنص
Document document = new Document(@"YOUR_DOCUMENT_DIRECTORY\TextBlock_HideShow_MouseOverOut_out.pdf");
```

**2. البحث عن النص وإضافة الحقل المخفي**
```csharp
// إنشاء كائن TextAbsorber للعثور على جميع العبارات المطابقة للنص المحدد
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Move the mouse cursor here to display floating text");
document.Pages.Accept(absorber);
TextFragmentCollection textFragments = absorber.TextFragments;
TextFragment fragment = textFragments[1];

// إنشاء حقل نص مخفي للنص العائم في المستطيل المحدد للصفحة
TextBoxField floatingField = new TextBoxField(fragment.Page, new Rectangle(100, 700, 220, 740));
floatingField.Value = "This is the \"floating text field\".";
floatingField.ReadOnly = true;
floatingField.Flags |= AnnotationFlags.Hidden;
floatingField.PartialName = "FloatingField_1";

// تخصيص مظهر الحقل العائم
floatingField.DefaultAppearance = new DefaultAppearance("Helv", 10, Color.Blue);
floatingField.Characteristics.Background = Color.LightBlue;
floatingField.Characteristics.Border = Color.DarkBlue;
floatingField.Border = new Border(floatingField);
floatingField.Border.Width = 1;
floatingField.Multiline = true;

// إضافة حقل نص إلى نموذج المستند
document.Form.Add(floatingField);
```

- **توضيح:** نحن نحدد نصًا محددًا باستخدام `TextFragmentAbsorber` وإنشاء مخفي `TextBoxField`تم تكوين هذا الحقل باستخدام أنماط مخصصة، مما يضمن بقائه غير مرئي حتى يتم تشغيله عن طريق تفاعل المستخدم.

### الميزة 3: إضافة زر غير مرئي مع الإجراءات

#### ملخص
تُظهر هذه الميزة إضافة زر غير مرئي يعمل على تبديل رؤية حقل النص عند تحريك الماوس فوقه.

#### التنفيذ خطوة بخطوة

**1. إنشاء وتكوين زر غير مرئي**
```csharp
using Aspose.Pdf.Forms;
using System.Drawing;

// إنشاء زر غير مرئي في موضع جزء النص
ButtonField buttonField = new ButtonField(fragment.Page, fragment.Rectangle);

// أضف إجراءات لإظهار/إخفاء الحقل العائم عند دخول/خروج الماوس من منطقة الزر
buttonField.Actions.OnEnter = new HideAction(floatingField, false); // إظهار الحقل عند إدخال الماوس
buttonField.Actions.OnExit = new HideAction(floatingField); // إخفاء الحقل عند خروج الماوس

// أضف حقل الزر إلى نموذج المستند
document.Form.Add(buttonField);
```

- **توضيح:** ال `ButtonField` يتم وضعه في موقع جزء النص. نُعرّف الإجراءات باستخدام `HideAction` للتحكم في رؤية النص العائم، مما يعزز التفاعلية.

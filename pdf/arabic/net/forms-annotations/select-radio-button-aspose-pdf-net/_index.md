## دليل التنفيذ

### الوصول إلى أزرار الراديو وتحديدها

#### ملخص
تعرف على كيفية الوصول إلى مجموعات أزرار الاختيار داخل مستندات PDF الخاصة بك وتحديد الخيارات برمجيًا.

#### تعليمات خطوة بخطوة

**الوصول إلى RadioButtonField:**
```csharp
// استرداد حقل زر الاختيار حسب اسمه
RadioButtonField radioButtonField = document.Form["radio"] as RadioButtonField;
```
*توضيح*: يستخدم `document.Form` للوصول إلى حقول النموذج. ستحتاج إلى اسم الحقل، والذي يمكنك العثور عليه باستخدام برامج تحرير PDF مثل Adobe Acrobat.

**حدد خيارًا محددًا:**
```csharp
// حدد الخيار الثالث (يبدأ الفهرس عند 0)
radioButtonField.Selected = 2;
```
*توضيح*:خيارات أزرار الاختيار مفهرسة صفريًا. هنا، تحديد الفهرس `2` يختار الخيار الثالث في المجموعة.

#### حفظ التغييرات

## دليل التنفيذ
سنستكشف ميزتين رئيسيتين: استخراج حقول النماذج وإضافة حقول جديدة.

### استخراج أسماء حقول النموذج ومواقعها
#### ملخص
استرداد الأسماء وإحداثيات المربع المحدد لجميع حقول النموذج في ملف PDF، وهو أمر بالغ الأهمية للتعامل مع النماذج الديناميكية وأتمتتها.

#### التنفيذ خطوة بخطوة
**1. استيراد المكتبات الضرورية**
```csharp
using Aspose.Pdf.Facades;
```

**2. تهيئة كائن النموذج**
إنشاء مثيل لـ `Aspose.Pdf.Facades.Form` للتفاعل مع ملف PDF الخاص بك.
```csharp
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form(dataDir + "/FilledForm.pdf");
```

**3. استخراج أسماء الحقول**
استرداد جميع أسماء الحقول باستخدام `FieldNames`.
```csharp
String[] allfields = form.FieldNames;
Rectangle[] box = new Rectangle[allfields.Length];
for (int i = 0; i < allfields.Length; i++) {
    FormFieldFacade facade = form.GetFieldFacade(allfields[i]);
    box[i] = facade.Box;
}
```
*توضيح:* تتكرر هذه الحلقة خلال كل حقل، وتستخرج إحداثيات المربع المحدد باستخدام `GetFieldFacade()`.

**4. حفظ التغييرات**
احفظ تعديلاتك في ملف PDF جديد.
```csharp
form.Save(dataDir + "/ExtractedFields_out.pdf");
```

### إضافة حقول نصية جديدة إلى نموذج PDF الحالي
#### ملخص
أضف حقول نصية جديدة أسفل الحقول الموجودة، مما يعزز تخصيص النماذج وتفاعل المستخدم.

#### التنفيذ خطوة بخطوة
**1. قم بتحميل المستند**
```csharp
Document document = new Document(dataDir + "/FilledForm - 2.pdf");
```

**2. تهيئة محرر النماذج**
يستخدم `FormEditor` لإضافة حقول جديدة.
```csharp
FormEditor editor = new FormEditor(document);
```

**3. أضف الحقول أسفل الحقول الموجودة**
قم بالمرور على الحقول الموجودة وأضف حقولًا جديدة أسفلها مباشرةً.
```csharp
for (int i = 0; i < allfields.Length; i++) {
    editor.AddField(Aspose.Pdf.Forms.FieldType.Text, "TextField" + i, allfields[i], 1,
        box[i].X, box[i].Y - 10, box[i].X + 50, box[i].Y);
}
```
*توضيح:* ال `AddField()` تقوم الطريقة بوضع الحقول الجديدة بالنسبة للحقول الموجودة.

**4. احفظ ملف PDF المعدّل**
```csharp
editor.Save(dataDir + "/AddedFields_out.pdf");
```

## دليل التنفيذ

### الميزة: تحميل مستند PDF

**ملخص**
ترتكز هذه الميزة على تحميل مستند PDF إلى Aspose.PDF لـ .NET، مما يمهد الطريق لمزيد من التلاعب مثل إزالة الارتباطات التشعبية.

#### تعليمات خطوة بخطوة

1. **تحديد مسار المستند**
   حدد المسار إلى ملف PDF الخاص بك:
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   ```

2. **تحميل مستند PDF**
   إنشاء جديد `Document` الكائن باستخدام Aspose.PDF لـ .NET:
   ```csharp
   Document doc = new Document(dataDir + "/SamplePdfFile.pdf");
   ```

### الميزة: إزالة الروابط التشعبية من مستند PDF المحمّل

**ملخص**
يوضح هذا القسم كيفية تكرار التعليقات التوضيحية في كل صفحة من مستندك، وتحديد تعليقات الارتباط التشعبي وإزالتها.

#### تعليمات خطوة بخطوة

3. **التكرار من خلال التعليقات التوضيحية**
   قم بالتكرار خلال كل تعليق توضيحي في كل صفحة:
   ```csharp
   using Aspose.Pdf.Annotations;
   foreach (var page in doc.Pages)
   {
       foreach (Annotation a in page.Annotations)
   ```

4. **تحديد تعليقات الرابط**
   التحقق مما إذا كان التعليق التوضيحي من النوع `Link` وألقيها وفقًا لذلك:
   ```csharp
   if (a.AnnotationType == AnnotationType.Link)
   {
       LinkAnnotation la = (LinkAnnotation)a;
   ```

5. **التعامل مع GoToURIAction**
   إذا كان إجراء الرابط هو `GoToURIAction`، قم بمسح عنوان URI الخاص به لإزالة وظيفة الارتباط التشعبي:
   ```csharp
   if (la.Action is GoToURIAction)
   {
       GoToURIAction gta = (GoToURIAction)la.Action;
       gta.URI = "";
   ```

6. **تحديث خصائص النص**
   يستخدم `TextFragmentAbsorber` لتحديث خصائص النص داخل مستطيل التعليق التوضيحي، وإزالة أي إشارات مرئية للارتباط التشعبي:
   ```csharp
   TextFragmentAbsorber tfa = new TextFragmentAbsorber();
   tfa.TextSearchOptions = new TextSearchOptions(a.Rect);
   page.Accept(tfa);

   foreach (TextFragment tf in tfa.TextFragments)
   {
       tf.TextState.Underline = false;
       tf.TextState.ForegroundColor = Aspose.Pdf.Color.Black;
   }
   ```

7. **حذف التعليق التوضيحي**
   إزالة تعليق الرابط من الصفحة:
   ```csharp
   page.Annotations.Delete(a);
   ```

### الميزة: حفظ المستند المعدل بتنسيق PDF

**ملخص**
ترتكز هذه الميزة على حفظ المستند المعدّل في ملف جديد، مع الحفاظ على كافة التغييرات التي تم إجراؤها أثناء المعالجة.

#### تعليمات خطوة بخطوة

8. **تحديد دليل الإخراج**
   حدد المكان الذي يجب حفظ ملف PDF الناتج فيه:
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   ```

9. **حفظ المستند**
   استخدم `Save` الطريقة لكتابة المستند المحدث إلى ملف جديد:
   ```csharp
   doc.Save(outputDir + "/RemoveHyperlinksFromPdf_out.pdf");
   ```

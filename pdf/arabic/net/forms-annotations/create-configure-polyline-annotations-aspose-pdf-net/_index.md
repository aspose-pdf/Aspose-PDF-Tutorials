## دليل التنفيذ

في هذا القسم، سنقوم بتقسيم عملية إنشاء وتكوين التعليقات التوضيحية متعددة الخطوط إلى خطوات قابلة للإدارة.

### إنشاء تعليق توضيحي متعدد الخطوط

#### ملخص
يُستخدم شرح الخطوط المتعددة لرسم عدة أجزاء خطية متصلة داخل مستند PDF. يمكنك تحديد مسارها باستخدام الرؤوس وتخصيصها بخصائص متنوعة، مثل اللون ونمط الخط.

#### التنفيذ خطوة بخطوة

##### الخطوة 1: تهيئة المستند
ابدأ بتحميل مستند PDF موجود إلى مشروعك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

##### الخطوة 2: تحديد الرؤوس ومساحة المستطيل
بعد ذلك، حدّد رؤوس خطك المتعدد. هذه النقاط تُحدّد مسار الشرح التوضيحي.
```csharp
Point[] vertices = new Point[] {
    new Point(100, 600),
    new Point(500, 600),
    new Point(500, 500),
    new Point(400, 300),
    new Point(100, 500),
    new Point(100, 600)
};

Rectangle rect = new Rectangle(100, 500, 500, 600);
```

##### الخطوة 3: إنشاء التعليقات التوضيحية وتكوينها
إنشاء `PolylineAnnotation` كائن ذو رؤوس ومستطيل محددين. خصّص مظهره بتعيين خصائص مثل اللون ونهايات الخطوط.
```csharp
PolylineAnnotation area = new PolylineAnnotation(doc.Pages[1], rect, vertices);

// تعيين لون التعليق التوضيحي إلى اللون الأحمر
area.Color = Color.Red;

// تكوين نهايات الأسطر
area.StartingStyle = LineEnding.OpenArrow;
area.EndingStyle = LineEnding.OpenArrow;
```

##### الخطوة 4: تكوين إعدادات القياس
يمكنك أيضًا تعيين وحدات القياس للخط المتعدد، وهو أمر مفيد في المستندات الفنية.
```csharp
area.Measure = new Measure(area);
area.Measure.DistanceFormat = new Measure.NumberFormatList(area.Measure);
area.Measure.DistanceFormat.Add(new Measure.NumberFormat(area.Measure));
area.Measure.DistanceFormat[1].UnitLabel = "mm";
```

##### الخطوة 5: إضافة التعليقات التوضيحية إلى المستند
وأخيرًا، أضف التعليق التوضيحي الذي تم تكوينه إلى مستندك واحفظه.
```csharp
doc.Pages[1].Annotations.Add(area);

string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputDir + "/UseMeasureWithPolylineAnnotation_out.pdf");
```

## دليل التنفيذ
دعنا نقسم التنفيذ إلى أقسام قابلة للإدارة، مع التركيز على إنشاء مستندات PDF وتكوينها باستخدام أزرار الاختيار.

### الميزة 1: إنشاء مستند PDF جديد
#### ملخص
الخطوة الأولى هي إنشاء مستند PDF أساسي. سيكون هذا أساسًا لإضافة عناصر تفاعلية، مثل أزرار الاختيار.
```csharp
using Aspose.Pdf;

// تعيين المسار لحفظ المستندات
double dataDir = "YOUR_DOCUMENT_DIRECTORY";

// إنشاء كائن مستند
Document pdfDocument = new Document();

// إضافة صفحة إلى ملف PDF
Page page = pdfDocument.Pages.Add();

// حفظ المستند
dataDir += "NewPDFDocument_out.pdf";
pdfDocument.Save(dataDir);
```
**توضيح:**
- **إنشاء مستند:** إنشاء مستند PDF فارغ.
- **إضافة صفحة:** يضيف صفحة فارغة، وهو أمر بالغ الأهمية حيث يجب وضع كل المحتوى على صفحة.
- **حفظ المستند:** يخزن ملف PDF الذي تم إنشاؤه حديثًا في الدليل المحدد.

### الميزة 2: إنشاء وتكوين RadioButtonField
#### ملخص
بعد ذلك، سنضيف حقل زر اختيار بخيارين. يتيح هذا العنصر التفاعلي للمستخدمين اختيار خيار واحد من بين خيارات متعددة.
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Forms;
using Aspose.Pdf.Annotations;

// تعيين المسار لحفظ المستندات
double dataDir = "YOUR_DOCUMENT_DIRECTORY";

// إنشاء كائن مستند
Document pdfDocument = new Document();

// إضافة صفحة إلى ملف PDF
Page page = pdfDocument.Pages.Add();

// إنشاء كائن RadioButtonField مع رقم الصفحة كحجة
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);

// إنشاء خيار زر الراديو الأول باستخدام المستطيل للموضع
RadioButtonOptionField opt1 = new RadioButtonOptionField(page, new Aspose.Pdf.Rectangle(0, 0, 20, 20));
RadioButtonOptionField opt2 = new RadioButtonOptionField(page, new Aspose.Pdf.Rectangle(100, 0, 120, 20));
opt1.OptionName = "Test1";
opt2.OptionName = "Test2";

// إضافة خيارات إلى حقل زر الاختيار
radio.Add(opt1);
radio.Add(opt2);

// تعيين أنماط لأزرار الراديو
opt1.Style = BoxStyle.Square;
opt2.Style = BoxStyle.Cross;

// تكوين نمط الحدود واللون لـ opt1
opt1.Border = new Border(opt1);
opt1.Border.Style = BorderStyle.Solid;
opt1.Border.Width = 1;
opt1.Characteristics.Border = System.Drawing.Color.Black;

// تكوين نمط الحدود واللون لـ opt2
opt2.Border = new Border(opt2);
opt2.Border.Style = BorderStyle.Solid;
opt2.Border.Width = 1;
opt2.Characteristics.Border = System.Drawing.Color.Black;

// إضافة زر اختياري إلى كائن النموذج الخاص بكائن المستند
doc.Form.Add(radio, 1);

// حفظ مستند PDF باستخدام أزرار الاختيار
dataDir += "RadioButtonField_out.pdf";
pdfDocument.Save(dataDir);
```
**توضيح:**
- **إنشاء حقل RadioButton:** ينشئ حقل زر اختيار جديد مرتبط بصفحة معينة.
- **خيارات الإنشاء:** قم بتحديد خيارين باستخدام `RadioButtonOptionField` وتحديد مواقعهم بالمستطيلات.
- **إضافة خيارات:** قم بإرفاق هذه الخيارات بحقل زر الاختيار.
- **تكوين النمط:** تخصيص المظهر، مثل نمط الحدود واللون.

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من إضافة كافة العناصر إلى الصفحة قبل الحفظ.
- التحقق من مسارات الدليل بحثًا عن أخطاء في إعدادات موقع الملف.

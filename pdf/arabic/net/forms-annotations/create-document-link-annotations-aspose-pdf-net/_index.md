## دليل التنفيذ

### إنشاء تعليق توضيحي لرابط المستند

تُظهر هذه الميزة كيفية إضافة تعليق ارتباط إلى ملف PDF موجود.

#### الخطوة 1: تحميل مستند PDF

أولاً، قم بتحميل مستند PDF الخاص بك باستخدام `Document` فصل:

```csharp
using Aspose.Pdf;

// تحديد مسارات لملفات الإدخال والإخراج
string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\CreateDocumentLink.pdf";
string outputFilePath = @"YOUR_OUTPUT_DIRECTORY\CreateDocumentLink_out.pdf";

// تحميل مستند PDF موجود
Document document = new Document(inputFilePath);
```

#### الخطوة 2: الوصول إلى الصفحة المطلوبة

بعد ذلك، قم بالوصول إلى الصفحة المحددة التي تريد إضافة التعليق التوضيحي إليها:

```csharp
// الوصول إلى الصفحة الأولى من المستند
Page page = document.Pages[1];
```

#### الخطوة 3: إنشاء وتكوين تعليق الرابط

إنشاء `LinkAnnotation` كائن، مع تحديد موقعه وحجمه على الصفحة. إليك كيفية تكوينه:

```csharp
using Aspose.Pdf.Annotations;

// تحديد مستطيل لمنطقة الرابط
Rectangle rect = new Rectangle(100, 100, 300, 300);

// إنشاء تعليق الرابط
LinkAnnotation link = new LinkAnnotation(page, rect);

// تعيين لون الرابط إلى اللون الأخضر
link.Color = Color.FromRgb(System.Drawing.Color.Green);

// تحديد إجراء بعيد للرابط (على سبيل المثال، الانتقال إلى ملف PDF آخر)
link.Action = new GoToRemoteAction(@"YOUR_DOCUMENT_DIRECTORY\RemoveOpenAction.pdf", 1);
```

#### الخطوة 4: إضافة التعليقات التوضيحية وحفظها

أضف التعليقات التوضيحية التي قمت بتكوينها إلى مجموعة التعليقات التوضيحية الخاصة بالصفحة، ثم احفظ المستند:

```csharp
// إضافة تعليق الرابط إلى الصفحة
page.Annotations.Add(link);

// احفظ ملف PDF المحدث
document.Save(outputFilePath);
```

### تكوين خصائص تعليق الرابط

يوضح هذا القسم كيفية تعيين خصائص مثل اللون والإجراءات لـ `LinkAnnotation`.

#### الخطوة 1: تحديد الصفحة والتعليق التوضيحي

على افتراض أن لديك `Page` هدف:

```csharp
Page page = new Page(); // استبدال بنسخة الصفحة الفعلية
LinkAnnotation link = new LinkAnnotation(page, new Rectangle(100, 100, 300, 300));
```

#### الخطوة 2: تعيين الخصائص

قم بتكوين خصائص التعليق التوضيحي مثل اللون والإجراء:

```csharp
// تعيين لون التعليق التوضيحي إلى اللون الأخضر
link.Color = Color.FromRgb(System.Drawing.Color.Green);

// تحديد إجراء (على سبيل المثال، الانتقال إلى صفحة معينة في مستند آخر)
link.Action = new GoToRemoteAction(@"YOUR_DOCUMENT_DIRECTORY\TargetDocument.pdf", 2);
```

#### الخطوة 3: إضافة التعليقات التوضيحية

أضف التعليق التوضيحي إلى الصفحة المطلوبة:

```csharp
// بافتراض أن "الصفحات" عبارة عن مجموعة من الصفحات
page.Annotations.Add(link);
```

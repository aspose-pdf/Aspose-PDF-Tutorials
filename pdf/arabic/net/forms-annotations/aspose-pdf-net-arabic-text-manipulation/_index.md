## دليل التنفيذ

### تحميل وتعديل نموذج PDF بالنص العربي

تتيح لك هذه الميزة تحميل مستند PDF، وتعديل حقول النص، وحفظه. إليك كيفية تحقيق ذلك في .NET باستخدام Aspose.PDF.

#### الخطوة 1: تهيئة تدفقات المستندات والملفات
ابدأ بتحديد المسار الذي يوجد فيه ملف PDF الخاص بك:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/FillFormField.pdf";
```

افتح مجرى ملف لقراءة وكتابة مستندك:

```csharp
using (FileStream fs = new FileStream(dataDir, FileMode.Open, FileAccess.ReadWrite))
{
    Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document(fs);
    
    // الوصول إلى حقل النص المسمى "textbox1"
    TextBoxField txtFld = pdfDocument.Form["textbox1"] as TextBoxField;
    
    // قم بتعيين النص العربي إلى الحقل المطلوب
    txtFld.Value = "يولد جميع الناس أحراراً متساوين في";
    
    string outputDir = "YOUR_OUTPUT_DIRECTORY" + "/ArabicTextFilling_out.pdf";
    pdfDocument.Save(outputDir);
}
```

**توضيح:**
- `FileStream` يفتح ملف PDF للتعديلات.
- ال `Aspose.Pdf.Document` يتم إنشاء الكائن للتعامل مع حقول النموذج.
- `TextBoxField` الوصول إلى مناطق نصية محددة وتعديلها داخل مستندك.

#### الخطوة 2: احفظ المستند المعدّل
بعد التعديل، احفظ التغييرات باستخدام:

```csharp
pdfDocument.Save(outputDir);
```

يضمن هذا تخزين ملف PDF المحدث الذي يحتوي على نص باللغة العربية في موقع محدد.

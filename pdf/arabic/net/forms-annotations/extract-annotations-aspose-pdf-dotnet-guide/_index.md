## دليل التنفيذ

### استخراج التعليقات التوضيحية من مستند PDF

لنبدأ باستخراج التعليقات التوضيحية. سنركز في هذا الدرس على النصوص الحرة والتعليقات التوضيحية على الأسطر.

#### الخطوة 1: إعداد مشروعك
قم بإنشاء مشروع C# جديد في بيئة التطوير المتكاملة المفضلة لديك، مع التأكد من إضافة Aspose.PDF كمرجع أو حزمة.

#### الخطوة 2: إنشاء طريقة استخراج التعليقات التوضيحية
إليك كيفية استخراج أنواع معينة من التعليقات التوضيحية:
```csharp
using System;
using System.Collections;
using Aspose.Pdf.Facades;

namespace PdfAnnotationExtractor
{
    public class ExtractAnnotations
    {
        public static void Run()
        {
            try
            {
                string dataDir = "path/to/your/documents/directory/";
                PdfAnnotationEditor annotationEditor = new PdfAnnotationEditor();
                
                // ربط مستند PDF
                annotationEditor.BindPdf(dataDir + "ExtractAnnotations.pdf");
                
                // تحديد أنواع التعليقات التوضيحية التي سيتم استخراجها
                string[] annotType = { AnnotationType.FreeText.ToString(), AnnotationType.Line.ToString() };
                
                // استخراج التعليقات التوضيحية المحددة من الصفحات 1 إلى 2
                ArrayList annotList = (ArrayList)annotationEditor.ExtractAnnotations(1, 2, annotType);
                
                foreach (Annotation annotation in annotList)
                {
                    Console.WriteLine(annotation.Contents); // إخراج محتويات كل تعليق
                }
            }
            catch (Exception ex)
            {
                Console.WriteLine("An error occurred: " + ex.Message);
            }
        }
    }
}
```
**توضيح:**
- **`PdfAnnotationEditor`**:تسهل هذه الفئة استخراج التعليقات التوضيحية ومعالجتها.
- **`BindPdf` طريقة**:يفتح ملف PDF المحدد للمعالجة.
- **`ExtractAnnotations` طريقة**:يستخرج التعليقات التوضيحية بناءً على النوع ونطاق الصفحة، ويعيدها في `ArrayList`.

#### الخطوة 3: تشغيل الكود الخاص بك
قم بتجميع تطبيقك وتشغيله. تأكد من وجود ملف PDF في المسار المحدد.

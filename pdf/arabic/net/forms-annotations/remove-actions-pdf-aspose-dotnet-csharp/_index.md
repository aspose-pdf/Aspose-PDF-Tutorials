### إزالة إجراءات فتح المستند

#### ملخص:
قد يكون حذف إجراءات فتح المستندات أمرًا بالغ الأهمية عند الرغبة في منع سلوكيات معينة أو الامتثال لمعايير الأمان. لنرَ كيف يُمكن تحقيق ذلك.

#### التنفيذ خطوة بخطوة:

**1. جهّز بيئتك**
أولاً، تأكد من إعداد مشروعك وتثبيت Aspose.PDF كما هو مذكور أعلاه.

```csharp
using System.IO;
using Aspose.Pdf.Facades;
```

**2. افتح مستند PDF**
إنشاء مثيل لـ `PdfContentEditor` للتلاعب بملف PDF:

```csharp
// حدد المسار إلى دليل المستند الخاص بك
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_LinksActions();

// تهيئة محرر محتوى Pdf
PdfContentEditor contentEditor = new PdfContentEditor();
contentEditor.BindPdf(dataDir + "RemoveOpenAction.pdf");
```

**3. إزالة إجراء فتح المستند**
استخدم `RemoveDocumentOpenAction` طريقة إزالة الإجراء المفتوح من المستند:

```csharp
// إزالة الإجراء المفتوح
contentEditor.RemoveDocumentOpenAction();
```

**4. احفظ ملف PDF المحدث**
وأخيرًا، احفظ التغييرات في ملف جديد:

```csharp
// احفظ ملف PDF المحدث
contentEditor.Save(dataDir + "RemoveOpenAction_out.pdf");
```

#### توضيح:
- **حدود**: ال `BindPdf` تأخذ الطريقة مسار ملف الإدخال.
- **قيم الإرجاع**: `RemoveDocumentOpenAction` لا يعيد أي قيمة ولكنه يعدل المستند في مكانه.

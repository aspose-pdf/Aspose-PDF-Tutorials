### حذف حقل النموذج

تتيح لك هذه الميزة إزالة الحقول غير المرغوب فيها من ملف PDF الخاص بك، مما يضمن الاحتفاظ بالبيانات الضرورية فقط.

#### ملخص
سوف نقوم بفتح مستند PDF موجود ونقوم برمجيًا بإزالة الحقل المسمى "textbox1".

#### التنفيذ خطوة بخطوة

**1. قم بإعداد بيئتك**

قم بإنشاء مشروع C# جديد في Visual Studio وتأكد من تثبيت Aspose.PDF لـ .NET كما هو موضح أعلاه.

**2. اكتب الكود لحذف الحقل**

إليك كيفية تنفيذ الحذف:

```csharp
using System;
using Aspose.Pdf;

namespace Aspose.Pdf.Examples.CSharp.AsposePDF.Forms
{
    public class DeleteFormField
    {
        public static void Run()
        {
            // حدد المسار إلى مستند PDF الخاص بك
            string dataDir = "Your_Directory_Path/";  // التحديث باستخدام الدليل الفعلي الخاص بك

            // افتح مستند PDF الموجود
            Document pdfDocument = new Document(dataDir + "DeleteFormField.pdf");

            // حذف حقل النموذج المسمى "textbox1"
            pdfDocument.Form.Delete("textbox1");

            // حفظ المستند المعدل في ملف جديد
            string outputFilePath = dataDir + "DeleteFormField_out.pdf";
            pdfDocument.Save(outputFilePath);

            Console.WriteLine(\nParticular field deleted successfully.\nFile saved at " + outputFilePath);
        }
    }
}
```

#### توضيح
- **فتح المستند**:نفتح ملف PDF موجود باستخدام `new Document("path")`.
- **حذف حقل**: ال `Delete` تقوم الطريقة بإزالة حقل النموذج المحدد حسب اسمه.
- **حفظ التغييرات**:بعد التعديل نقوم بحفظ المستند باسم الملف الجديد.

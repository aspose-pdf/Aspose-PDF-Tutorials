
ثم قم بتحميل مستند PDF الخاص بك:

```csharp

{

    string title = annotation.Title;
```

#### توضيح
- **حدود**: ال `Document` يأخذ الكائن مسار الملف لتحميل ملف PDF الخاص بك.
- **قيم الإرجاع**: ال `Annotations` تقوم الخاصية بإرجاع مجموعة من التعليقات التوضيحية على صفحة معينة.
- **غاية**:تتيح لك هذه الطريقة الوصول إلى بيانات التعليقات التوضيحية ومعالجتها برمجيًا داخل ملفات PDF الخاصة بك.

## دليل التنفيذ

### إنشاء حقول أزرار الاختيار وإضافتها

#### ملخص
يرشدك هذا القسم إلى كيفية إنشاء حقول أزرار الاختيار في مستند PDF باستخدام لغة C#. ستتعلم كيفية إنشاء مثيل لـ `RadioButtonField`، أضف الخيارات وأرفقها بالنموذج.

**الخطوة 1: إنشاء كائن مستند**
ابدأ بإنشاء حساب جديد `Document` هذا الكائن بمثابة حاوية لمحتوى PDF الخاص بك.
```csharp
// إنشاء مستند PDF جديد
Document pdfDocument = new Document();
```

**الخطوة 2: إضافة صفحة إلى مستندك**
يجب إضافة صفحة قبل وضع أي حقول عليها.
```csharp
// أضف صفحة فارغة
pdfDocument.Pages.Add();
```

**الخطوة 3: إنشاء كائن RadioButtonField**
ال `RadioButtonField` يُمثل الكائن مجموعة أزرار الاختيار. حدّد رقم الصفحة عند إنشائه.
```csharp
// إنشاء حقل زر اختيار جديد في الصفحة 1
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

**الخطوة 4: إضافة خيارات إلى أزرار الراديو الخاصة بك**
قم بتحديد خيارات متعددة داخل حقل زر الاختيار الخاص بك، مع تحديد مواقعها باستخدام `Rectangle` أشياء.
```csharp
// أضف الخيار الأول بموضع محدد
radio.AddOption("Test", new Rectangle(0, 0, 20, 20));

// أضف الخيار الثاني في موقع مختلف
radio.AddOption("Test1", new Rectangle(20, 20, 40, 40));
```

**الخطوة 5: إرفاق حقل زر الاختيار بالنموذج**
أضف حقل زر الاختيار الخاص بك إلى مجموعة النماذج الخاصة بالمستند.
```csharp
// أضف حقل زر الاختيار إلى نموذج PDF
pdfDocument.Form.Add(radio);
```

**الخطوة 6: احفظ مستندك**
بمجرد تكوينه، احفظ مستندك مع جميع عناصره سليمة.
```csharp
// تحديد مسار الملف وحفظ المستند
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();
dataDir += "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```

### نصائح استكشاف الأخطاء وإصلاحها
- **المكتبة المفقودة**تأكد من تثبيت Aspose.PDF بشكل صحيح. تحقق من مراجع مشروعك.
- **فهرس الصفحة غير صحيح**:تأكد من أن فهرس الصفحة يتطابق مع صفحة موجودة في مستندك.

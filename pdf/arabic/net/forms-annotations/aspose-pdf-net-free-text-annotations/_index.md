## دليل التنفيذ: إضافة تعليقات توضيحية مجانية للنص

### نظرة عامة على الميزة

تتيح لك إضافة تعليق نصي مجاني إدراج نص مخصص في مواقع محددة بملف PDF. هذه الميزة أساسية لتمييز الأقسام، وإضافة التعليقات، أو تحديد المعلومات المهمة.

### التنفيذ خطوة بخطوة

#### الخطوة 1: تحديد المسارات وتهيئة محرر محتوى Pdf

ابدأ بإعداد المسارات لمستندات الإدخال والإخراج الخاصة بك وقم بتهيئة `PdfContentEditor` هدف:

```csharp
using System.IO;
using Aspose.Pdf.Facades;

public static void AddFreeTextAnnotation()
{
    string dataDir = @"YOUR_DOCUMENT_DIRECTORY";  
    string outputPath = @"YOUR_OUTPUT_DIRECTORY";

    using (PdfContentEditor contentEditor = new PdfContentEditor())
    {
        // فتح مستند PDF موجود
        contentEditor.BindPdf(dataDir + "AddFreeTextAnnotation.pdf");
```

#### الخطوة 2: تعريف مستطيل الشرح التوضيحي

حدد المنطقة التي تريد وضع تعليقك فيها. هنا، نستخدم مستطيلًا:

```csharp
        System.Drawing.Rectangle rect = new System.Drawing.Rectangle(50, 50, 100, 100);
```

#### الخطوة 3: إنشاء وإضافة تعليقات توضيحية مجانية للنص

الآن، قم بإنشاء التعليق النصي الحر داخل المستطيل المحدد وأضفه إلى ملف PDF الخاص بك:

```csharp
        // إنشاء تعليق نصي مجاني
        contentEditor.CreateFreeText(rect, "Sample content", 1);

        // حفظ المستند المحدث
        contentEditor.Save(outputPath + "AddFreeTextAnnotation_out.pdf");
    }
}
```

### شرح المعلمات والطرق

- `rect`:يحدد المكان الذي ستظهر فيه تعليقاتك التوضيحية على الصفحة.
- `CreateFreeText()`:يضيف تعليقًا نصيًا مجانيًا بأبعاد ونص محددين.
- `Save()`:يحفظ التغييرات في ملف PDF جديد.

يُسهّل استخراج تعليقات الروابط من ملف PDF عملية التنقل أو استخراج البيانات بناءً على الروابط التشعبية. ستوضح هذه الميزة كيفية العثور على هذه الروابط وعرضها باستخدام Aspose.PDF لـ .NET.

**خطوات التنفيذ**

**الخطوة 1**:قم بتحميل المستند كما في السابق، مع التأكد من تعيين المسار بشكل صحيح.

**الخطوة 2**:قم بتحديد طريقة لاستخراج تعليقات الارتباط من كل صفحة.
```csharp
public static void ExtractLinkAnnotations(Page page)
{
    foreach (Annotation annot in page.Annotations) {
        if (annot is LinkAnnotation linkAnnot && linkAnnot.Action is GoToURIAction uriAction) {
            Console.WriteLine("URI: " + uriAction.URI);
            
            TextAbsorber absorber = new TextAbsorber();
            absorber.TextSearchOptions.LimitToPageBounds = true;
            absorber.TextSearchOptions.Rectangle = linkAnnot.Rect;

            page.Accept(absorber);
            string extractedText = absorber.Text;
            Console.WriteLine("Extracted Text: " + extractedText);
        }
    }
}
```
**توضيح**:تتحقق هذه الطريقة من كل تعليق لمعرفة ما إذا كان `LinkAnnotation` باستخدام إجراء URI. إذا كان الأمر كذلك، فسيتم استخراج وطباعة عنوان URI الخاص بالرابط والنص المرتبط به.

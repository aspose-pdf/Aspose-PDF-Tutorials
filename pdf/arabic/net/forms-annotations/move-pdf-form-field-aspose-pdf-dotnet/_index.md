## دليل التنفيذ

### نقل حقل نموذج PDF

في هذا القسم، سنركز على نقل حقول النماذج داخل مستند PDF. يُعد هذا مفيدًا بشكل خاص عند الحاجة إلى إعادة ترتيب العناصر لتحسين التخطيط أو سهولة الوصول.

#### نظرة عامة على الميزة
يتضمن نقل حقل نموذج تغيير إحداثياته داخل مستند PDF. يمكنك تعديل الموضع دون تغيير خصائص أخرى، مثل الحجم أو النمط.

#### خطوات التنفيذ
1. **افتح المستند**
   ابدأ بتحميل ملف PDF المستهدف إلى `Aspose.Pdf.Document` هدف:
   ```csharp
   string dataDir = "Path_to_your_PDF_directory";
   Document pdfDocument = new Document(dataDir + "MoveFormField.pdf");
   ```
2. **الوصول إلى حقل النموذج المستهدف**
   استرداد حقل النموذج الذي ترغب في نقله حسب اسمه:
   ```csharp
   TextBoxField textBoxField = pdfDocument.Form["textbox1"] as TextBoxField;
   ```
3. **تعديل موقع الحقل**
   ضبط الموقع باستخدام `Rect` الخاصية، التي تحدد مستطيلاً جديدًا لموضع الحقل:
   ```csharp
   // المعلمات: X السفلي الأيسر، Y السفلي الأيسر، X العلوي الأيمن، Y العلوي الأيمن
   textBoxField.Rect = new Aspose.Pdf.Rectangle(300, 400, 600, 500);
   ```
4. **حفظ المستند المعدل**
   احفظ التغييرات في ملف PDF جديد:
   ```csharp
   dataDir = dataDir + "MoveFormField_out.pdf";
   pdfDocument.Save(dataDir);
   ```

#### شرح المعلمات
- `Rectangle(300, 400, 600, 500)`:هذا يُحدد الموضع والحجم الجديدين. الرقمان الأولان (300، 400) هما إحداثيات أسفل اليسار، والرقمان الأخيران (600، 500) هما إحداثيات أعلى اليمين.

### نصائح استكشاف الأخطاء وإصلاحها
- **لم يتم العثور على الحقل**:تأكد من أن اسم الحقل يتطابق تمامًا مع ما هو موجود في ملف PDF.
- **قضايا الإحداثيات**:تحقق جيدًا من قيم المستطيل للتأكد من أنها تقع ضمن حدود المستند.

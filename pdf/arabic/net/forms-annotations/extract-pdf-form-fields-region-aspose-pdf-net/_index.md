## دليل التنفيذ

### استخراج الحقول من منطقة محددة

في هذا القسم، سنوضح كيفية استخراج حقول النماذج ضمن مساحة مستطيلة محددة في مستند PDF. تُعد هذه الميزة مفيدة بشكل خاص عند التعامل مع نماذج كبيرة الحجم حيث تحتاج فقط إلى نقاط بيانات محددة.

#### الخطوة 1: افتح مستند PDF

أولاً، قم بتحميل ملف PDF الخاص بك إلى كائن Aspose.Pdf.Document.

```csharp
string dataDir = "path-to-your-data-directory";
document doc = new Document(dataDir + "GetFieldsFromRegion.pdf");
```

#### الخطوة 2: تحديد المنطقة للاستخراج

أنشئ مستطيلاً يُحدد المنطقة التي تريد استخراج الحقول منها. تُحدَّد الإحداثيات بالنقاط، حيث (0, 0) هي الزاوية السفلية اليسرى.

```csharp
Rectangle rectangle = new Rectangle(35, 30, 500, 500);
```

#### الخطوة 3: الوصول إلى حقول النموذج واستخراجها

استخدم `GetFieldsInRect` طريقة لاسترجاع الحقول ضمن المنطقة المحددة. تُرجع هذه الطريقة مصفوفة من `Field` أشياء.

```csharp
Form form = doc.Form;
Field[] fields = form.GetFieldsInRect(rectangle);

// عرض أسماء الحقول والقيم
foreach (Field field in fields)
{
    Console.WriteLine("Field Name: " + field.FullName + "-" + "Field Value: " + field.Value);
}
```

#### شرح الكود

- **إنشاء المستطيل:** ال `Rectangle` يحدد الكائن الإحداثيات التي تحدد منطقة الاستخراج.
- **طريقة GetFieldsInRect:** يسترجع جميع حقول النموذج داخل المستطيل المحدد. هذا ضروري لاستخراج البيانات ذات الصلة فقط.

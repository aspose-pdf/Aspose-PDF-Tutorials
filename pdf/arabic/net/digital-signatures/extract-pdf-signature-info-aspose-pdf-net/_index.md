## دليل التنفيذ

### نظرة عامة على الميزة: استخراج معلومات التوقيع

ترتكز هذه الميزة على استخراج معلومات الشهادة من حقول توقيع PDF للتحقق من صحة المستند وسلامته.

#### الخطوة 1: تحديد مسار المستند وتحميل ملف PDF

حدد مسار الدليل لمستند PDF الخاص بك وقم بتحميله:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string input = dataDir + "ExtractSignatureInfo.pdf";

using (Document pdfDocument = new Document(input))
{
    // متابعة المعالجة...
}
```
**توضيح**: ال `dataDir` يحتوي المتغير على المسار إلى مستنداتك. `input` يقوم string بإنشاء مسار الملف الكامل للتحميل.

#### الخطوة 2: التكرار عبر حقول النموذج

قم بالمرور على كل حقل في نموذج PDF للعثور على حقول التوقيع:

```csharp
foreach (Field field in pdfDocument.Form)
{
    SignatureField sf = field as SignatureField;
    if (sf != null)
    {
        // حقل توقيع العملية
    }
}
```
**توضيح**:نحن نستخدم `foreach` حلقة للتكرار على كل `Field` الكائن. `as` التحقق من الكلمات الرئيسية لمعرفة ما إذا كان الحقل `SignatureField`.

#### الخطوة 3: استخراج تدفق الشهادة

استخراج وقراءة تدفق الشهادة من حقل التوقيع:

```csharp
Stream cerStream = sf.ExtractCertificate();
if (cerStream != null)
{
    byte[] bytes = new byte[cerStream.Length];
    using (cerStream)
    {
        cerStream.Read(bytes, 0, bytes.Length);
    }

    // حفظ الشهادة في ملف
    string outputCerPath = dataDir + "input.cer";
    using (FileStream fs = new FileStream(outputCerPath, FileMode.CreateNew))
    {
        fs.Write(bytes, 0, bytes.Length);
    }
}
```
**توضيح**: ال `ExtractCertificate()` تسترجع الطريقة تدفق الشهادة. نقرأ هذا التدفق في مصفوفة بايتات ونحفظه كملف `.cer` ملف.

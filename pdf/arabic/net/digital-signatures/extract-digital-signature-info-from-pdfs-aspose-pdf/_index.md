## دليل التنفيذ

الآن، دعونا نتناول عملية استخراج معلومات التوقيع الرقمي من مستند PDF.

### ملخص

المهمة هي استخراج وحفظ معلومات الشهادة المرتبطة بالتوقيعات الرقمية في ملف PDF باستخدام Aspose.PDF `PdfFileSignature` فصل.

### التنفيذ خطوة بخطوة

#### **1. استيراد مساحات الأسماء المطلوبة**

ابدأ باستيراد مساحات الأسماء الضرورية:

```csharp
using System.IO;
using Aspose.Pdf.Facades;
```

توفر هذه المساحات الأسماء إمكانية الوصول إلى معالجة الملفات ووظائف معالجة PDF التي نحتاجها.

#### **2. قم بتحديد فئتك الرئيسية وطريقتك**

إنشاء فصل دراسي `ExtractSignatureInfo` مع طريقة ثابتة `Run()` حيث ستنفذ منطقك:

```csharp
public class ExtractSignatureInfo
{
    public static void Run()
    {
        // سيتم التنفيذ هنا
    }
}
```

#### **3. قم بتحميل مستند PDF**

يستخدم `PdfFileSignature` لتحميل ملف PDF الذي تريد معالجته:

```csharp
string dataDir = "PathToYourDataDirectory";
string inputFilePath = Path.Combine(dataDir, "DigitallySign.pdf");

using (PdfFileSignature pdfFileSignature = new PdfFileSignature())
{
    pdfFileSignature.BindPdf(inputFilePath);
```

`BindPdf` يربط مستند PDF بـ `PdfFileSignature` هدف.

#### **4. استرداد أسماء التوقيع**

التحقق من التوقيعات داخل ملف PDF:

```csharp
IList<string> sigNames = pdfFileSignature.GetSignNames();
if (sigNames.Count > 0)
{
    // لدينا توقيع واحد على الأقل للمعالجة
}
```

تؤدي هذه الخطوة إلى استرجاع جميع أسماء التوقيعات الرقمية الموجودة في المستند.

#### **5. استخراج معلومات الشهادة**

استخراج وحفظ بيانات الشهادة لكل توقيع:

```csharp
string sigName = sigNames[0] as string;
if (!string.IsNullOrEmpty(sigName))
{
    using (Stream cerStream = pdfFileSignature.ExtractCertificate(sigName))
    {
        if (cerStream != null)
        {
            byte[] bytes = new byte[cerStream.Length];
            cerStream.Read(bytes, 0, (int)cerStream.Length);
            
            string outputFilePath = Path.Combine(dataDir, "ExtractedCertificate.pfx");
            using (FileStream fs = new FileStream(outputFilePath, FileMode.CreateNew))
            {
                fs.Write(bytes, 0, bytes.Length);
            }
        }
    }
}
```

ال `ExtractCertificate` تسترجع الطريقة الشهادة المرتبطة بالتوقيع وتحفظها على القرص.

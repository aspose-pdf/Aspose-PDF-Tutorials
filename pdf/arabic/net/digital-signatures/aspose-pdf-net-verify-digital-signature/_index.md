#### الخطوة 2: إنشاء طريقة للتحقق من التوقيع
استخدم مقتطف التعليمات البرمجية هذا لإنشاء طريقة للتحقق من التوقيعات الرقمية في ملفات PDF:
```csharp
class VerifyValidSignature
{
    public static void Run()
    {
        try
        {
            // تعيين مسار الدليل لمستنداتك
            string dataDir = "path_to_your_documents_directory";
            
            // إنشاء مثيل لـ PdfFileSignature
            PdfFileSignature pdfSign = new PdfFileSignature();
            
            // ربط مستند PDF
            pdfSign.BindPdf(dataDir + "DigitallySign.pdf");
            
            // التحقق من التوقيع
            if (pdfSign.VerifySignature("Signature1"))
            {
                Console.WriteLine("Signature Verified");
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine(ex.Message);
        }
    }       
}
```

#### شرح معلمات الكود
- `dataDir`:الدليل الذي يتم تخزين ملفات PDF الخاصة بك فيه.
- `PdfFileSignature pdfSign = new PdfFileSignature()`:يتم تهيئة مثيل جديد للتعامل مع توقيعات PDF.
- `pdfSign.BindPdf(dataDir + "DigitallySign.pdf")`:يربط ملف PDF المراد معالجته.
- `pdfSign.VerifySignature("Signature1")`:التحقق من صحة التوقيع المحدد.

## دليل التنفيذ

الآن بعد أن قمت بالإعداد، فلنبدأ في تنفيذ التوقيع الرقمي.

### الميزة: التوقيع الرقمي على ملف PDF بمظهر مخصص

تتيح لك هذه الميزة تخصيص كيفية ظهور توقيعك على ملفات PDF. اتبع الخطوات التالية:

#### الخطوة 1: تهيئة كائن PdfFileSignature

إنشاء مثيل لـ `PdfFileSignature` لربط الوثيقة وتوقيعها.

```csharp
using (PdfFileSignature pdfSign = new PdfFileSignature()) {
    // ربط ملف PDF الذي تريد التوقيع عليه
    pdfSign.BindPdf("input.pdf");
}
```

#### الخطوة 2: تحديد موقع التوقيع

قم بإنشاء مستطيل يحدد المكان الذي سيظهر فيه توقيعك.

```csharp
Rectangle rect = new Rectangle(310, 45, 200, 50);
```

#### الخطوة 3: تهيئة كائن PKCS7

استخدم ملف PFX وكلمة المرور لتهيئة `PKCS7` هذا الكائن يمثل الشهادة الرقمية للتوقيع.

```csharp
PKCS7 pkcs = new PKCS7("SampleCertificate.pfx", "idsrv3test");
```

#### الخطوة 4: تخصيص مظهر التوقيع

قم بتخصيص مظهر توقيعك باستخدام `SignatureCustomAppearance`.

```csharp
SignatureCustomAppearance signatureCustomAppearance = new SignatureCustomAppearance();
signatureCustomAppearance.FontSize = 6;
signatureCustomAppearance.FontFamilyName = "Times New Roman";
signatureCustomAppearance.DigitalSignedLabel = "Signed by me";
pkcs.CustomAppearance = signatureCustomAppearance;
```

#### الخطوة 5: التوقيع على ملف PDF

قم بتطبيق توقيعك الرقمي على المستند باستخدام `Sign` طريقة.

```csharp
pdfSign.Sign(1, true, rect, pkcs);
pdfSign.Save("output.pdf");
```

- تأكد من أن لديك الأذونات اللازمة لقراءة/كتابة ملفات PDF المعنية.

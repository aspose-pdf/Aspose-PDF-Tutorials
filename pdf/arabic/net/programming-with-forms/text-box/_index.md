# مربع النص

## مقدمة

في مجال التوثيق الرقمي، يُحسّن إنشاء نماذج PDF تفاعلية تجربة المستخدم وكفاءة جمع البيانات بشكل ملحوظ. يوفر Aspose.PDF لـ .NET طريقة فعّالة ومباشرة لدمج حقول النماذج المختلفة، مما يسمح للمطورين بإشراك المستخدمين بطريقة لا تُمكنهم من ذلك في المستندات الثابتة. من بين أنواع حقول النماذج المختلفة التي يُمكنك إضافتها إلى ملف PDF، تبرز مربعات النص لأنها تُسهّل إدخال المستخدم بطريقة واضحة ومنظمة. تخيّل إنشاء مستند PDF لا ينقل المعلومات فحسب، بل يدعو المستخدمين أيضًا للتفاعل معه! في هذا البرنامج التعليمي، سنتعمق في عملية إضافة مربع نص إلى ملف PDF باستخدام Aspose.PDF لـ .NET، مع شرح كل خطوة بالتفصيل والتأكد من استيعابك الكامل للمفهوم.

هل أنت مستعد لتحسين ملفات PDF الخاصة بك وجعلها تفاعلية حقًا؟ لنبدأ!

## المتطلبات الأساسية

قبل أن نبدأ في إنشاء مربع النص في مستند PDF، هناك بعض الأشياء التي ستحتاج إلى وضعها في مكانها:

1. المعرفة الأساسية بلغة C#: إن فهم بناء الجملة وهيكلية لغة C# سيساعدك على التنقل عبر الكود بسهولة أكبر.
2. تم تثبيت Aspose.PDF لـ .NET: تأكد من تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك الحصول عليها من [رابط التحميل](https://releases.aspose.com/pdf/net/).
3. بيئة التطوير: ستعمل بيئة التطوير المتكاملة مثل Visual Studio بشكل أفضل لتشغيل واختبار الكود الخاص بك.
4. .NET Framework: تم تصميم هذا البرنامج التعليمي لتطبيقات .NET، لذا فإن وجود إصدار متوافق مثبت يعد أمرًا بالغ الأهمية.

بعد استيفاء هذه المتطلبات الأساسية، أنت الآن جاهز للانطلاق في البرمجة. لنبدأ بالشرح!

## استيراد الحزم

قبل البدء بالبرمجة، عليك استيراد الحزم اللازمة من مكتبة Aspose.PDF. سيسمح لك هذا بالوصول إلى الفئات والأساليب اللازمة للتعامل مع ملفات PDF. 

إليك كيفية استيراد الحزم المطلوبة:

### افتح IDE الخاص بك

قم بتشغيل بيئة التطوير المفضلة لديك (يفضل Visual Studio). 

### إنشاء مشروع جديد

قم بإعداد مشروع C# جديد عن طريق تحديد "إنشاء مشروع جديد". اختر قالب تطبيق وحدة التحكم لتبسيط الأمور.

### تثبيت حزمة Aspose.PDF

استخدم مدير حزم NuGet لتثبيت Aspose.PDF لـ .NET. في وحدة تحكم مدير الحزم، شغّل الأمر:

```bash
Install-Package Aspose.PDF
```

تعمل هذه الخطوة على دمج مكتبة Aspose.PDF في مشروعك، مما يسمح لك بالعمل بسلاسة مع وظائف PDF.

### استيراد مساحة اسم Aspose.PDF

في أعلى ملف البرنامج الرئيسي الخاص بك (عادةً `Program.cs`), قم بتضمين السطر التالي للوصول إلى وظيفة Aspose.PDF:

```csharp
using System.IO;
using System;
using Aspose.Pdf.Forms;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
```

من خلال القيام بذلك، فإنك تمهد الطريق للسحر الذي على وشك الحدوث!

الآن بعد أن قمنا بإعداد كل شيء، حان الوقت للاستمتاع ببعض المرح في البرمجة.

دعونا نستعرض عملية إضافة مربع النص خطوة بخطوة!

## الخطوة 1: تحديد دليل المستندات الخاص بك

أولاً، علينا تحديد مكان حفظ ملف PDF. تأكد من استبدال `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي لملفاتك.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يقوم هذا السطر بإنشاء دليل العمل الخاص بنا ويخبر البرنامج بالمكان الذي يجب أن يبحث فيه عن ملف PDF الذي نريد التعامل معه.

## الخطوة 2: افتح مستند PDF 

بعد ذلك، افتح ملف PDF الذي ترغب بإضافة مربع النص إليه. إليك كيفية القيام بذلك:

```csharp
Document pdfDocument = new Document(dataDir + "TextField.pdf");
```

يقوم هذا السطر بتحميل ملف PDF في مثيل `Document` الصف. تأكد من ذلك `"TextField.pdf"` موجود في الدليل المحدد.

## الخطوة 3: إنشاء حقل مربع النص

الآن للجزء المثير - دعنا ننشئ مربع النص الخاص بنا:

```csharp
TextBoxField textBoxField = new TextBoxField(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(100, 200, 300, 300));
```

هذا الخط يفعل شيئين:
- يقوم بتهيئة ملف جديد `TextBoxField` الكائن الذي سيتم إضافته إلى الصفحة الثانية من ملف PDF الخاص بك (لاحظ أن الصفحات مفهرسة بدءًا من 1).
- ال `Rectangle` تعرف المعلمة على موضع وحجم مربع النص الخاص بك، المحدد على هيئة إحداثيات (x1، y1، x2، y2).

## الخطوة 4: تعيين خصائص حقل مربع النص 

يمكنك تخصيص مربع النص حسب احتياجاتك. إليك كيفية ضبط بعض الخصائص الأساسية:

```csharp
textBoxField.PartialName = "textbox1";
textBoxField.Value = "Text Box";
```

في هذا المثال:
- `PartialName` تعيين معرف فريد لمربع النص.
- `Value` يقوم بتعريف النص الافتراضي الذي يظهر داخل المربع.

## الخطوة 5: تخصيص الحدود

بعد ذلك، دعنا نعطي مربع النص الخاص بنا بعض الأناقة عن طريق تخصيص حدوده:

```csharp
Border border = new Border(textBoxField);
border.Width = 5; 
border.Dash = new Dash(1, 1);
textBoxField.Border = border;
textBoxField.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
```

هذه المقتطفة:
- إنشاء حدود وتحديد عرضها.
- إنشاء نمط متقطع للحدود.
- تعيين اللون الأخضر لمربع النص.

## الخطوة 6: إضافة مربع النص إلى المستند

الآن بعد أن قمنا بإعداد حقل مربع النص الخاص بنا، فلنقم بإضافته إلى مستند PDF الخاص بنا:

```csharp
pdfDocument.Form.Add(textBoxField, 1);
```

يخبر هذا السطر ملف PDF بأن يتضمن فعليًا مربع النص الذي تم إنشاؤه حديثًا في الصفحة الأولى.

## الخطوة 7: احفظ ملف PDF المعدّل

أخيرًا، حان وقت حفظ التغييرات. إليك الطريقة:

```csharp
dataDir = dataDir + "TextBox_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nTextbox field added successfully.\nFile saved at " + dataDir);
```

يحفظ هذا الكود ملف PDF المُعدَّل باسم جديد. تأكد من التحقق من مسار الإخراج لملف PDF الذي أنشأته حديثًا!

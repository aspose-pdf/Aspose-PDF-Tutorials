# إنشاء مستند

## مقدمة

إنشاء نماذج تفاعلية داخل ملفات PDF يُحسّن تجربة المستخدم وتفاعله بشكل ملحوظ. هل تساءلت يومًا عن كيفية تبسيط عملية جمع البيانات أو جمع الردود بفعالية من خلال مستنداتك؟ مع Aspose.PDF لـ .NET، أصبح إنشاء ملفات PDF مع حقول أزرار الاختيار أمرًا في غاية السهولة! في هذا البرنامج التعليمي، سنستكشف كيفية إنشاء مستند يتضمن حقل أزرار الاختيار باستخدام Aspose.PDF، ونشرح لك العملية خطوة بخطوة. سواء كنت مطورًا محترفًا أو مبتدئًا، صُمم هذا الدليل خصيصًا لتوفير تعليمات ورؤى واضحة. هيا بنا نغوص في عالم إنشاء ملفات PDF باستخدام .NET ونجعل مستنداتك تتألق!

## المتطلبات الأساسية

قبل أن ننتقل إلى البرمجة، هناك بعض الأساسيات التي تحتاجها للتأكد من أن كل شيء يسير بسلاسة:

1. بيئة تطوير .NET: يجب أن تكون على دراية ببيئة تطوير .NET، مثل Visual Studio، لكتابة التعليمات البرمجية الخاصة بك وتنفيذها.
2. Aspose.PDF لـ .NET: تأكد من تثبيت مكتبة Aspose.PDF. يمكنك الحصول عليها بسهولة من [صفحة التحميل](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: من الضروري أن يكون لديك فهم أساسي للغة C#، حيث أن هذه هي اللغة التي سنستخدمها في أمثلتنا.
4. دليل المستندات الخاص بك: قم بإنشاء دليل حيث سيتم تخزين مستنداتك لتجنب أي مشكلات متعلقة بالمسار.

بمجرد توفر هذه المتطلبات الأساسية لديك، ستكون جاهزًا لإنشاء مستند PDF تفاعلي!

## استيراد الحزم

للبدء، يجب عليك استيراد مكونات Aspose.PDF اللازمة إلى مشروعك. إليك كيفية القيام بذلك:

### تثبيت مكتبة Aspose.PDF

أولاً، عليك إضافة مكتبة Aspose.PDF إلى مشروعك. إذا كنت تستخدم NuGet، يمكنك تشغيل الأمر التالي في وحدة تحكم NuGet لإدارة الحزم:

```bash
Install-Package Aspose.PDF
```

سيؤدي هذا إلى إضافة المكتبة إلى مشروعك وجعل جميع الوظائف متاحة للاستخدام.

### استيراد مساحات الأسماء المطلوبة

بعد إضافة المكتبة، ستحتاج إلى استيراد مساحات الأسماء المطلوبة في ملف C#. إليك المقطع الذي يجب تضمينه في أعلى مستندك:

```csharp
using System;
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
using Aspose.Pdf.Forms;
```

ستتيح لك هذه المساحات الاسمية الوصول إلى الفئات والطرق اللازمة لإنشاء ملفات PDF ومعالجتها.

بعد أن أعددنا بيئتنا واستوردنا الحزم اللازمة، لنُنشئ مستند PDF يحتوي على حقول أزرار اختيار. سنُقسّم هذا إلى خطوات سهلة الفهم للتوضيح.

## الخطوة 1: تحديد دليل المستندات الخاص بك

الخطوة الأولى في رحلة البرمجة هي تحديد مسار مجلد المستندات. هنا سيتم حفظ ملف PDF النهائي بعد إنشائه.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";  // استبدل بمسارك
```
هذا الخط ينشئ متغير سلسلة `dataDir` حيث ستخزن ملف PDF الناتج. تأكد من استبدال `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي.

## الخطوة 2: إنشاء مستند جديد

بعد ذلك، سننشئ كائن المستند. هذا أشبه ببدء العمل على لوحة فارغة، في انتظار لمستنا الإبداعية!

```csharp
// إنشاء مستند جديد
Document doc = new Document();
```
هذا السطر يُشغّل مستند PDF جديد. يمكنك اعتباره إنشاء ملف جديد لإضافة محتوى إليه.

## الخطوة 3: إضافة صفحة إلى المستند

الآن وقد أصبح لدينا مستندنا، نحتاج إلى إضافة صفحة إليه. وكما أن كل عمل فني يحتاج إلى خلفية، فإن ملف PDF يحتاج إلى صفحة!

```csharp
Page page = doc.Pages.Add();
```
يُضيف هذا الأمر صفحة جديدة إلى المستند. بهذا، يُمكننا البدء بإضافة عناصر تفاعلية.

## الخطوة 4: إضافة حقل زر الاختيار

الآن، حان وقت إدخال حقل أزرار الاختيار. هنا يُمكن للمستخدمين اختيار خيارات الإجابة!

```csharp
// إضافة حقل زر الاختيار
RadioButtonField field = new RadioButtonField(page);
field.Rect = new Aspose.Pdf.Rectangle(40, 650, 100, 720);
field.PartialName = "NewField";
```
هنا، نقوم بإنشاء `RadioButtonField` الكائن الذي نخصصه لصفحتنا. `Rect` تعرف المعلمة على موضع وحجم حقل زر الاختيار على الصفحة.

## الخطوة 5: تحديد خيارات أزرار الراديو

الآن، علينا إنشاء بعض الخيارات لأزرار الاختيار. هذا يسمح للمستخدمين بالاختيار بين عناصر مختلفة.

وهنا كيف يمكننا تعريف الخيارات الثلاثة:

```csharp
// إضافة خيارات أزرار الراديو
RadioButtonOptionField opt1 = new RadioButtonOptionField();
opt1.Rect = new Aspose.Pdf.Rectangle(40, 650, 60, 670);
opt1.OptionName = "Item1";
opt1.Border = new Border(opt1);
opt1.Border.Width = 1;
opt1.Characteristics.Border = System.Drawing.Color.Black;

RadioButtonOptionField opt2 = new RadioButtonOptionField();
opt2.Rect = new Aspose.Pdf.Rectangle(60, 670, 80, 690);
opt2.OptionName = "Item2";
opt2.Border = new Border(opt2);
opt2.Border.Width = 1;
opt2.Characteristics.Border = System.Drawing.Color.Black;

RadioButtonOptionField opt3 = new RadioButtonOptionField();
opt3.Rect = new Aspose.Pdf.Rectangle(80, 690, 100, 710);
opt3.OptionName = "Item3";
opt3.Border = new Border(opt3);
opt3.Border.Width = 1;
opt3.Characteristics.Border = System.Drawing.Color.Black;
```
في كل من هذه الكتل، نقوم بإنشاء `RadioButtonOptionField` الأشياء وتحديد مواقعها باستخدام `.Rect`، تعيين الأسماء باستخدام `.OptionName`، وتعيين سمات الحدود الخاصة بهم.

## الخطوة 6: إضافة خيارات إلى الحقل

بعد تحديد خياراتنا، حان الوقت لإضافتها إلى حقل أزرار الاختيار الذي أنشأناه سابقًا. هذه الخطوة بالغة الأهمية لأنها تربط الخيارات بالحقل نفسه.

```csharp
field.Add(opt1);
field.Add(opt2);
field.Add(opt3);
```
تضمن مقتطفات التعليمات البرمجية هذه أن تكون خياراتنا جزءًا من حقل زر الاختيار، مما يجعلها تفاعلية للمستخدمين.

## الخطوة 7: حفظ المستند

أخيرًا، علينا حفظ ملف PDF المُصمّم بإتقان. بدون هذه الخطوة، سيذهب جهدنا هباءً!

```csharp
dataDir = dataDir + "CreateDoc_out.pdf";  // تعيين اسم ملف الإخراج
doc.Save(dataDir);  // حفظ المستند
Console.WriteLine("\nNew doc with 3 items radio button created successfully.\nFile saved at " + dataDir);
```
هنا، نحدد اسم ملف الإخراج ونحفظ المستند. تُطبع رسالة نجاح في وحدة التحكم للتأكيد. 

## الخطوة 8: معالجة الاستثناءات

من الجيد تضمين معالجة الاستثناءات لاكتشاف أي مشاكل قد تظهر أثناء التنفيذ. إليك طريقة بسيطة للقيام بذلك:

```csharp
catch (Exception ex)
{
    Console.WriteLine(ex.Message);
}
```
من خلال تغليف الكود الخاص بنا في كتلة try-catch، يمكننا التعامل بسلاسة مع أي أخطاء قد تحدث، وتوفير ردود الفعل بدلاً من التعطل.

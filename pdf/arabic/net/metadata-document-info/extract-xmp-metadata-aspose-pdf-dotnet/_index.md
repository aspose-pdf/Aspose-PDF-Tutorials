### استخراج بيانات التعريف XMP

**ملخص:**
يتضمن استخراج بيانات XMP قراءة خصائص محددة مُضمنة في ملف PDF. يمكن أن تتضمن هذه المعلومات تواريخ الإنشاء، وتفاصيل المؤلف، والحقول المخصصة.

#### التنفيذ خطوة بخطوة:

1. **تحميل مستند PDF**
   ابدأ بتحميل مستند PDF المستهدف إلى `Aspose.Pdf.Document` هدف.

   ```csharp
   // المسار إلى الدليل الذي يحتوي على المستندات
   string dataDir = "path/to/your/documents/";

   // تحميل ملف PDF
   Document pdfDocument = new Document(dataDir + "GetXMPMetadata.pdf");
   ```

2. **خصائص بيانات الوصول الوصفية**
   استخدم `Metadata` خاصية كائن المستند الخاص بك للوصول إلى بيانات XMP التعريفية.

   ```csharp
   // استرداد وعرض خصائص البيانات الوصفية المحددة
   Console.WriteLine(pdfDocument.Metadata["xmp:CreateDate"]);
   Console.WriteLine(pdfDocument.Metadata["xmp:Nickname"]);
   Console.WriteLine(pdfDocument.Metadata["xmp:CustomProperty"]);
   ```

**شرح المعلمات:**
- `dataDir`:مسار الدليل الذي يتم تخزين ملفات PDF الخاصة بك فيه.
- `Metadata`:كائن يشبه القاموس يحمل جميع إدخالات بيانات XMP.

أنشئ تطبيق وحدة تحكم C# جديدًا في Visual Studio. تأكد من أن مشروعك يدعم .NET Framework 4.6.1 أو أحدث (أو أي إصدار .NET متوافق).

#### إعداد Aspose.PDF

أولاً، قم بتضمين مساحات الأسماء الضرورية:

```csharp

```

#### تحميل واسترداد معلومات المستند

فيما يلي تفصيل خطوة بخطوة لاستخراج معلومات المستند:

**الخطوة 1: تحديد دليل البيانات الخاص بك**

حدد مكان تخزين ملفات PDF. استبدل `"GetFileInfo.pdf"` مع المسار إلى ملفك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YourFilePathHere/"; // قم بتعديل هذا السطر بشكل مناسب
```

**الخطوة 2: افتح المستند**

قم بتحميل مستندك باستخدام Aspose.PDF `Document` فصل:

```csharp
// تحميل مستند PDF موجود
Document pdfDocument = new Document(dataDir + "GetFileInfo.pdf");
```

**الخطوة 3: الوصول إلى معلومات المستند**

استرداد وعرض البيانات الوصفية من ملف PDF:

```csharp
// الحصول على معلومات الوثيقة
DocumentInfo docInfo = pdfDocument.Info;

// عرض خصائص المستند
Console.WriteLine("Author: {0}", docInfo.Author);
Console.WriteLine("Creation Date: {0}", docInfo.CreationDate);
Console.WriteLine("Keywords: {0}", docInfo.Keywords);
Console.WriteLine("Modify Date: {0}", docInfo.ModDate);
Console.WriteLine("Subject: {0}", docInfo.Subject);
Console.WriteLine("Title: {0}", docInfo.Title);
```

يفتح هذا المقتطف من الكود ملف PDF، ويستعيد بياناته الوصفية، ثم يطبعها في وحدة التحكم. إنه بسيط وفعّال للوصول إلى خصائص المستند الأساسية.

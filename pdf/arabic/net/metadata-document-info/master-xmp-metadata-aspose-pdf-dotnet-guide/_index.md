## دليل التنفيذ
يُقسّم هذا الدليل كل ميزة إلى خطوات سهلة. سنشرح بالتفصيل كيفية إضافة خصائص البيانات الوصفية وتعديلها وإزالتها باستخدام Aspose.PDF لـ .NET.

### إضافة خصائص البيانات الوصفية
#### ملخص
تتضمن إضافة البيانات الوصفية إنشاء ملف جديد `PdfXmpMetadata` الكائن وربطه بملف PDF الخاص بك.

**الخطوة 1: التهيئة والربط**
```csharp
// إنشاء كائن PdfXmpMetadata
PdfXmpMetadata xmpMetaData = new PdfXmpMetadata();

// ربط ملف pdf بالكائن
xmpMetaData.BindPdf("SetXMPMetadata.pdf");
```

**الخطوة 2: إضافة حقول البيانات الوصفية**
- **تاريخ الإنشاء**:تعيين تاريخ إنشاء المستند.
```csharp
xmpMetaData.Add(DefaultMetadataProperties.CreateDate, DateTime.Now.ToString());
```
- **أداة المنشئ**:حدد البرنامج المستخدم لإنشاء ملف PDF.
```csharp
xmpMetaData.Add(DefaultMetadataProperties.CreatorTool, "Your Application Name");
```

### تعديل خصائص البيانات الوصفية
#### ملخص
لتحديث خصائص البيانات الوصفية الموجودة، قم بالوصول إليها مباشرةً باستخدام مفاتيحها.

**تغيير الممتلكات الموجودة**
```csharp
// تعديل بيانات التعريف الخاصة بتاريخ الإنشاء
xmpMetaData[DefaultMetadataProperties.MetadataDate] = DateTime.Now.ToString();
```

### إزالة خصائص البيانات الوصفية
#### ملخص
إن إزالة البيانات الوصفية غير الضرورية أمر سهل باستخدام `Remove` طريقة.

**الخطوة 1: إزالة الخصائص غير المرغوب فيها**
```csharp
// قم بإزالة تاريخ التعديل إذا لم يكن هناك حاجة لذلك
xmpMetaData.Remove(DefaultMetadataProperties.ModifyDate);
```

### إضافة مساحات الأسماء والخصائص المخصصة
#### ملخص
بالنسبة للخصائص المخصصة، يتعين عليك تسجيل بادئة مساحة الاسم أولاً.

**الخطوة 1: تسجيل بادئة مساحة الاسم**
```csharp
// تسجيل عنوان URI لمساحة اسم مخصصة
xmpMetaData.RegisterNamespaceURI("customNamespace", "http://www.customNameSpaces.com/ns/");
```

**الخطوة 2: إضافة خاصية مخصصة**
```csharp
// إضافة خاصية محددة من قبل المستخدم باستخدام البادئة
xmpMetaData.Add("customNamespace:UserPropertyName", "Custom Value");
```

### الحفظ والإغلاق
بعد إجراء التغييرات، قم بحفظها مرة أخرى في ملف PDF:

```csharp
// حفظ بيانات التعريف xmp في ملف pdf
xmpMetaData.Save("SetXMPMetadata_out.pdf");

// إغلاق الكائن
xmpMetaData.Close();
```

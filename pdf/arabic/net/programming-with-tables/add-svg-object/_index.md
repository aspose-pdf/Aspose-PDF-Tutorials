# إضافة كائن SVG في ملف PDF

## مقدمة

هل تساءلت يومًا عن كيفية دمج الرسومات المتجهة القابلة للتطوير (SVG) في مستندات PDF؟ مع ازدياد استخدام التوثيق الرقمي، أصبح دمج الرسومات والنصوص بكفاءة أمرًا بالغ الأهمية. إذا كنت تعمل باستخدام .NET وترغب في تحسين ملفات PDF الخاصة بك باستخدام صور SVG، فأنت في المكان المناسب! في هذا البرنامج التعليمي، سنشرح لك خطوة بخطوة عملية إضافة كائنات SVG إلى ملفات PDF باستخدام Aspose.PDF لـ .NET. سنتعمق في كل خطوة، مع التأكد من فهمك الكامل لما يجب عليك فعله.

## المتطلبات الأساسية

قبل أن نتعمق في تفاصيل إضافة كائنات SVG إلى ملفات PDF، هناك بعض الأشياء التي تحتاج إلى إعدادها:

1. الفهم الأساسي لـ .NET: ستساعدك المعرفة بلغة البرمجة C# وبيئة .NET على المتابعة بسهولة.
2. مكتبة Aspose.PDF: عليك تنزيل وتثبيت مكتبة Aspose.PDF لـ .NET. يمكنك الحصول عليها عبر الرابط التالي: [تنزيل Aspose.PDF لـ .NET](https://releases.aspose.com/pdf/net/).
3. Visual Studio أو أي .NET IDE: قم بإعداد بيئة التطوير المتكاملة (IDE) المفضلة لديك حيث يمكنك كتابة التعليمات البرمجية الخاصة بك وتنفيذها.
4. ملف SVG نموذجي: ستحتاج إلى ملف SVG للعمل عليه. أنشئ ملفًا أو نزّل ملف SVG نموذجيًا لاستخدامه في هذا المثال.

## استيراد الحزم

الخطوة الأولى هي التأكد من استيراد الحزم اللازمة لمشروعك. إليك كيفية البدء:

### إنشاء مشروع جديد

افتح Visual Studio (أو IDE المفضل لديك) وقم بإنشاء مشروع تطبيق وحدة تحكم جديد.

### إضافة Aspose.PDF DLL

أضف ملف Aspose.PDF DLL إلى مراجع مشروعك. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول، ثم اختر "إضافة مرجع"، وانتقل إلى المكان الذي نزّلت منه مكتبة Aspose.PDF. 

### استيراد مساحات الأسماء المطلوبة

في أعلى ملف C# الخاص بك، قم باستيراد مساحات الأسماء المطلوبة:

```csharp
using System.IO;
using System;
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

ستتيح لك هذه المساحات الاسمية الوصول إلى فئات وطرق مختلفة للعمل مع ملفات PDF.

بعد أن انتهينا من إعداد كل شيء، لننتقل إلى البرمجة الفعلية. سنقسّم العملية إلى خطوات سهلة.

## الخطوة 1: إعداد كائن المستند

أول شيء عليك فعله هو إنشاء مثيل جديد لـ `Document` هذا هو المكان الذي سيتم فيه تخزين جميع محتويات PDF الخاصة بك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// إنشاء كائن مستند
Document doc = new Document();
```

يؤدي هذا السطر من التعليمات البرمجية إلى إنشاء مستند PDF جديد حيث يمكننا البدء في إضافة المحتوى الخاص بنا.

## الخطوة 2: إنشاء مثيل صورة

بعد ذلك، نحتاج إلى إنشاء نسخة صورة لملف SVG. هذا هو الكائن الذي سيحتوي على ملف SVG.

```csharp
// إنشاء مثيل للصورة
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
```

يقوم هذا السطر بتهيئة مثيل صورة جديد سنقوم لاحقًا بتكوينه لقراءة ملف SVG الخاص بنا.

## الخطوة 3: تعيين نوع الصورة والملف

الآن، حان الوقت لتحديد نوع الملف والملف الفعلي الذي نريد استخدامه:

```csharp
// تعيين نوع الصورة كـ SVG
img.FileType = Aspose.Pdf.ImageFileType.Svg;

// المسار لملف المصدر
img.File = dataDir + "SVGToPDF.svg"; // تأكد من الاستبدال بالمسار الفعلي الخاص بك
```

هنا، قمنا بتعيين نوع الصورة إلى SVG، وحددنا مسار ملف SVG. تأكد من صحة المسار!

## الخطوة 4: تحديد أبعاد الصورة

قد ترغب في تغيير حجم صورة SVG لتناسب ملف PDF بشكل مثالي. يمكنك القيام بذلك بتحديد عرضها وارتفاعها:

```csharp
// تعيين العرض لمثيل الصورة
img.FixWidth = 50;

// تعيين الارتفاع لمثيل الصورة
img.FixHeight = 50;
```

هذه الخطوة بالغة الأهمية إذا كنت تسعى لتصميم ملف PDF جذاب بصريًا. يمكنك تعديل هذه الأبعاد وفقًا لاحتياجاتك التصميمية الخاصة.

## الخطوة 5: إنشاء مثيل للجدول

الآن، لنُنشئ جدولًا سيُضيف صورة SVG وبعض النصوص. هذا رائع لتنظيم محتواك.

```csharp
// إنشاء مثيل للجدول
Aspose.Pdf.Table table = new Aspose.Pdf.Table();

// تعيين عرض خلايا الجدول
table.ColumnWidths = "100 100";
```

مع `ColumnWidths`يمكننا تحديد مساحة كل عمود في الجدول. يمكنك تعديل هذه القيم حسب متطلبات المحتوى لديك.

## الخطوة 6: إضافة الصفوف والخلايا إلى الجدول

الآن، سنضيف صفوفًا إلى الجدول ثم نضيف صورة SVG الخاصة بنا إلى خلية:

```csharp
// إنشاء كائن صف وإضافته إلى مثيل الجدول
Aspose.Pdf.Row row = table.Rows.Add();

// إنشاء كائن خلية وإضافته إلى مثيل الصف
Aspose.Pdf.Cell cell = row.Cells.Add();

// إضافة جزء نصي إلى مجموعة فقرات كائن الخلية
cell.Paragraphs.Add(new TextFragment("First cell"));

// إضافة خلية أخرى إلى كائن الصف
cell = row.Cells.Add();
```

يؤدي هذا إلى إنشاء صف في الجدول يحتوي على خليتين - تحتوي الأولى على تسمية نصية، وستحمل الثانية صورة SVG الخاصة بنا.

## الخطوة 7: إضافة صورة SVG إلى الجدول

الآن يمكننا إضافة صورة SVG الخاصة بنا إلى الخلية الثانية التي أنشأناها للتو:

```csharp
// إضافة صورة SVG إلى مجموعة فقرات مثيل الخلية المضافة مؤخرًا
cell.Paragraphs.Add(img);
```

وهكذا، تكون قد أدخلت صورة SVG الخاصة بك في ملف PDF!

## الخطوة 8: إنشاء صفحة PDF وإضافة الجدول

بعد ذلك، سنحتاج إلى إنشاء صفحة في مستند PDF الخاص بنا لحمل الجدول الذي أنشأناه للتو:

```csharp
// إنشاء كائن الصفحة وإضافته إلى مجموعة الصفحات الخاصة بمثيل المستند
Page page = doc.Pages.Add();

// إضافة جدول إلى مجموعة فقرات كائن الصفحة
page.Paragraphs.Add(table);
```

تضمن هذه الخطوة أن جدولنا، الذي يحتوي الآن على صورة SVG والنص، سيكون جزءًا من ملف PDF.

## الخطوة 9: حفظ ملف PDF

أخيرًا، حان الوقت لحفظ مستند PDF الذي قمت بإنشائه حديثًا:

```csharp
dataDir = dataDir + "AddSVGObject_out.pdf"; // توفير مسار الإخراج
// حفظ ملف PDF
doc.Save(dataDir);
```

هكذا تفعل ذلك! ببضعة أسطر من التعليمات البرمجية، تصبح صورة SVG جزءًا من ملف PDF.

## خاتمة

إضافة كائنات SVG إلى ملفات PDF باستخدام Aspose.PDF لـ .NET عملية سهلة وبسيطة بمجرد فهمك للعمليات. باتباع الخطوات الموضحة في هذا الدليل، يمكنك الجمع بكفاءة بين تنوع رسومات SVG والوظائف القوية لمستندات PDF. تذكر، مع كل مشروع، الممارسة تُتقن العمل. لا تتردد في تجربة تصاميم وتخطيطات مختلفة أثناء إضافة صور SVG.

### هل Aspose.PDF متوافق مع كافة إصدارات .NET؟

# إضافة تعليق توضيحي للرابط

## مقدمة

أهلاً بك في عالم معالجة ملفات PDF مع Aspose.PDF لـ .NET! إذا كنت ترغب في تحسين مستندات PDF الخاصة بك، سواءً للاستخدام المهني أو المشاريع الشخصية أو أي شيء آخر، فأنت في المكان المناسب. اليوم، سنتناول ميزةً عمليةً ومميزةً في Aspose.PDF: إضافة تعليق بالحبر إلى ملفات PDF. تُعد هذه الميزة مفيدةً للغاية عند إضافة ملاحظات أو توقيعات بخط اليد إلى مستنداتك، مما يجعلها أكثر تفاعليةً وجاذبية.

## المتطلبات الأساسية

قبل أن نتعمق في سحر البرمجة، دعنا نتأكد من أن لديك كل ما تحتاجه للبدء:

1. .NET Framework: تأكد من تثبيت .NET على جهازك. تعمل هذه المكتبة بسلاسة مع مختلف إصدارات .NET، بما في ذلك .NET Core.
2. مكتبة Aspose.PDF: ستحتاج إلى تنزيل مكتبة Aspose.PDF لـ .NET والرجوع إليها في مشروعك. إذا لم تقم بذلك بعد، يمكنك الحصول على أحدث إصدار من [رابط التحميل](https://releases.aspose.com/pdf/net/).
3. محرر الكود: يمكنك استخدام أي محرر كود من اختيارك، ولكن يوصى بشدة باستخدام Visual Studio لسهولة استخدامه مع تطبيقات .NET.
4. الفهم الأساسي للغة C#: ستساعدك المعرفة العملية بلغة C# على التنقل عبر أمثلة الترميز بسلاسة.
5. إعداد بيئة التطوير الخاصة بك: تأكد من إعداد بيئة التطوير المتكاملة لديك للتعامل مع مشروعات .NET ومن أنك قمت بالإشارة إلى مكتبة Aspose.PDF بشكل صحيح في مشروعك. 

بعد تلبية هذه المتطلبات الأساسية، ستكون جاهزًا لبدء إضافة التعليقات التوضيحية بالحبر إلى ملفات PDF الخاصة بك!

## استيراد الحزم

قبل البدء بالبرمجة، لنستورد الحزم اللازمة. في أعلى ملف C#، أضف عبارات الاستخدام التالية:

```csharp
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
using System;
using System.Collections;
using System.Collections.Generic;
```

سيمنحك هذا إمكانية الوصول إلى جميع الفئات والأساليب التي تحتاجها للعمل مع تعليقات PDF.

بعد أن هيأنا المكان، حان وقت العمل الجاد والدخول في التفاصيل! سنشرح كل خطوة بالتفصيل لضمان فهمك الدقيق لكيفية إنشاء وإضافة تعليق حبري إلى مستند PDF.

## الخطوة 1: تعيين المستند والدليل

أول شيء تريد القيام به هو إعداد مستندك والمسار الذي تريد حفظ ملف الإخراج فيه. 

```csharp
string dataDir = "YOUR DATA DIRECTORY";
Document doc = new Document();
```
نحن نحدد متغير `dataDir`، والذي يشير إلى الدليل الذي سيتم حفظ ملف PDF الناتج فيه. `Document` يتم بعد ذلك إنشاء الكائن، مما يؤدي إلى إنشاء مستند PDF جديد للتحرير.

## الخطوة 2: إضافة صفحة إلى مستندك

بعد ذلك، قد ترغب في إضافة صفحة إلى المستند الذي قمت بإنشائه حديثًا.

```csharp
Page pdfPage = doc.Pages.Add();
```
هنا نضيف صفحة جديدة إلى مستندنا. يحتاج كل ملف PDF إلى صفحة واحدة على الأقل، لذا تُعد هذه الخطوة أساسية.

## الخطوة 3: تحديد مستطيل الرسم

قبل أن تتمكن من رسم أي شيء، تحتاج إلى تحديد المكان على الصفحة الذي ستضع فيه تعليقات الحبر الخاصة بك.

```csharp
System.Drawing.Rectangle drect = new System.Drawing.Rectangle();
drect.Height = (int)pdfPage.Rect.Height;
drect.Width = (int)pdfPage.Rect.Width;
drect.X = 0;
drect.Y = 0;
Aspose.Pdf.Rectangle arect = Aspose.Pdf.Rectangle.FromRect(drect);
```
هنا، نقوم بإنشاء `Rectangle` كائن يُحدد المنطقة على الصفحة التي سنضيف إليها تعليقنا بالحبر. نضبط أبعاده لتناسب الصفحة بأكملها، بدءًا من (0,0).

## الخطوة 4: تحضير نقاط الحبر

الآن يأتي الجزء الممتع - تحديد النقاط التي تشكل تعليق الحبر الخاص بك. 

```csharp
IList<Point[]> inkList = new List<Point[]>();
Aspose.Pdf.Point[] arrpt = new Aspose.Pdf.Point[3];
inkList.Add(arrpt);
arrpt[0] = new Aspose.Pdf.Point(100, 800);
arrpt[1] = new Aspose.Pdf.Point(200, 800);
arrpt[2] = new Aspose.Pdf.Point(200, 700);
```
يُنشئ هذا الجزء من التعليمات البرمجية قائمةً بمصفوفات النقاط، حيث تُمثل كل مصفوفة مجموعةً من النقاط لخط الحبر. هنا، نُعرّف ثلاث نقاط تُشكّل مثلثًا؛ يُمكنك تعديل إحداثياتها لتناسب تصميمك.

## الخطوة 5: إنشاء تعليق الحبر

بعد تحديد النقاط التي تريدها، حان الوقت لإنشاء التعليق التوضيحي بالحبر الفعلي.

```csharp
InkAnnotation ia = new InkAnnotation(pdfPage, arect, inkList)
{
    Title = "XXX",
    Color = Aspose.Pdf.Color.LightBlue,
    CapStyle = CapStyle.Rounded
};
```
نحن ننشئ `InkAnnotation` الكائن، مع إدخال الصفحة والمستطيل ونقاط الحبر. بالإضافة إلى ذلك، نقوم بتعيين بعض الخصائص مثل `Title`، `Color`، و `CapStyle`قم بتخصيصها لتناسب احتياجاتك!

## الخطوة 6: تعيين الحدود والتعتيم

هل تريد أن يبرز تعليقك؟ دعنا نضفي عليه لمسةً من الأناقة.

```csharp
Border border = new Border(ia);
border.Width = 25;
ia.Opacity = 0.5;
```
هنا، نضيف حدودًا إلى التعليق التوضيحي بعرض محدد ونضبط درجة تعتيمه، مما يجعله شفافًا إلى حد ما.

## الخطوة 7: إضافة التعليق التوضيحي إلى الصفحة

الآن بعد أن أصبحت التعليقات التوضيحية جاهزة، فقد حان الوقت لإضافتها إلى صفحة PDF.

```csharp
pdfPage.Annotations.Add(ia);
```
يضيف هذا السطر تعليقات الحبر التي أنشأناها سابقًا إلى مجموعة تعليقات الصفحة. 

## الخطوة 8: حفظ المستند

وأخيرًا، دعونا نحفظ مستندنا المعدّل.

```csharp
dataDir = dataDir + "AddInkAnnotation_out.pdf";
doc.Save(dataDir);
Console.WriteLine("\nInk annotation added successfully.\nFile saved at " + dataDir);
```
نحن نقوم بتعديل `dataDir` لتضمين اسم ملف الإخراج وحفظ المستند. ستُطبع رسالة تأكيد على وحدة التحكم لإعلامك بأن كل شيء سار بسلاسة.

## خاتمة

ها قد انتهيت! لقد نجحت في إضافة تعليق بالحبر إلى مستند PDF باستخدام Aspose.PDF لـ .NET. هذه الميزة البسيطة والفعّالة تُحسّن مستنداتك وتجعلها تفاعلية. سواءً أكنت تُضيف توقيعات أو ملاحظات أو رسومات، تُوفر التعليقات بالحبر طريقة فريدة لإثراء المحتوى.

# تعيين تنسيق التعليقات التوضيحية للنص مجانًا

## مقدمة

في العصر الرقمي، أصبحت القدرة على التعامل مع مستندات PDF والتعليق عليها أمرًا بالغ الأهمية للمحترفين في مختلف المجالات. سواء كنت مدرسًا يُصحح واجباتك، أو محاميًا يُراجع العقود، أو مدير مشروع يُشارك ملاحظاتك، فإن امتلاك الأدوات المناسبة يُحدث فرقًا كبيرًا. ومن بين هذه الأدوات القوية Aspose.PDF for .NET، وهي مكتبة قوية تُمكّن المطورين من إنشاء ملفات PDF وتحريرها ومعالجتها بسهولة. في هذا البرنامج التعليمي، سنتعمق في تفاصيل ضبط تنسيق التعليقات النصية المجانية باستخدام Aspose.PDF for .NET. بنهاية هذا الدليل، ستكون مُجهزًا بالمعرفة اللازمة لتحسين مستندات PDF الخاصة بك باستخدام التعليقات المُخصصة، مما يجعل سير عملك أكثر سلاسة وكفاءة.

## المتطلبات الأساسية

قبل أن نتعمق في أساسيات البرمجة، لنتأكد من امتلاكك كل ما تحتاجه للبدء. إليك ما يجب أن يكون لديك:

1. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم الأمثلة ومقاطع التعليمات البرمجية المقدمة في هذا البرنامج التعليمي.
2. Aspose.PDF لـ .NET: يجب تثبيت مكتبة Aspose.PDF. يمكنك تنزيلها من [هنا](https://releases.aspose.com/pdf/net/).
3. Visual Studio: بيئة تطوير مثل Visual Studio ستجعل كتابة واختبار الكود الخاص بك أسهل.
4. مستند PDF: لهذا البرنامج التعليمي، ستحتاج إلى نموذج مستند PDF للعمل عليه. يمكنك إنشاء نموذج بسيط أو تنزيل نموذج من الإنترنت.

بمجرد توفر هذه المتطلبات الأساسية لديك، ستكون جاهزًا للانطلاق إلى عالم تعليقات PDF!

## استيراد الحزم

لبدء استخدام Aspose.PDF لـ .NET، عليك استيراد الحزم اللازمة إلى مشروعك. إليك كيفية القيام بذلك:

### الخطوة 1: إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع C# جديدًا. يمكنك اختيار تطبيق وحدة التحكم لتسهيل الأمر.

### الخطوة 2: إضافة مرجع Aspose.PDF

1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد "إدارة حزم NuGet".
3. ابحث عن "Aspose.PDF" وقم بتثبيت الإصدار الأحدث.

### الخطوة 3: استيراد مساحة الاسم

في أعلى ملف C# الخاص بك، قم باستيراد مساحة اسم Aspose.PDF:

```csharp
using System;
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا ننتقل إلى الجزء الرئيسي من البرنامج التعليمي الخاص بنا: ضبط تنسيق التعليقات التوضيحية للنص الحر.

## الخطوة 1: تحديد دليل المستندات

أولاً، عليك تحديد مسار مجلد المستندات. هذا هو المكان الذي سيُحفظ فيه ملف PDF. إليك الطريقة:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يستبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي لتخزين ملف PDF. هذه الخطوة بالغة الأهمية لأنها تُحدد لبرنامجك مكان ملف PDF الذي تريد العمل عليه.

## الخطوة 2: افتح مستند PDF

بعد ذلك، ستحتاج إلى فتح مستند PDF الذي ستُضيف إليه التعليقات التوضيحية. يتم ذلك باستخدام `Document` الفئة من مكتبة Aspose.PDF:

```csharp
// فتح المستند
Document pdfDocument = new Document(dataDir + "SetFreeTextAnnotationFormatting.pdf");
```

يقوم هذا السطر من التعليمات البرمجية بتهيئة ملف جديد `Document` الكائن ويُحمّل ملف PDF المُحدد. تأكد من أن اسم الملف يُطابق اسم الملف الموجود في مجلدك.

## الخطوة 3: إنشاء كائن DefaultAppearance

الآن، دعونا ننشئ `DefaultAppearance` الكائن. سيحدد هذا الكائن مظهر التعليق النصي الحر الخاص بك، مثل الخط والحجم واللون:

```csharp
// إنشاء كائن DefaultAppearance
DefaultAppearance default_appearance = new DefaultAppearance("Arial", 28, System.Drawing.Color.Red);
```

في هذا المثال، نستخدم خط Arial، وحجم الخط ٢٨، واللون الأحمر. يمكنك تخصيص هذه القيم لتناسب احتياجاتك!

## الخطوة 4: إنشاء التعليق النصي الحر

بعد ضبط المظهر، حان وقت إنشاء التعليق النصي الحر. هنا، يمكنك تحديد مكان ظهور التعليق في ملف PDF:

```csharp
// إنشاء تعليق توضيحي
FreeTextAnnotation freetext = new FreeTextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600), default_appearance);
```

في هذا السطر، نقوم بإنشاء سطر جديد `FreeTextAnnotation` في الصفحة الأولى من ملف PDF. يُحدد المستطيل موضع وحجم التعليق التوضيحي. يمكنك ضبط الإحداثيات (٢٠٠، ٤٠٠، ٤٠٠، ٦٠٠) لوضع التعليق التوضيحي في المكان المطلوب.

## الخطوة 5: تحديد محتويات التعليق التوضيحي

الآن بعد أن قمنا بإنشاء التعليق التوضيحي الخاص بنا، دعنا نضيف بعض النص إليه:

```csharp
// تحديد محتويات التعليق التوضيحي
freetext.Contents = "Free Text";
```

يمكنك استبدال `"Free Text"` مع أي رسالة تريد عرضها في التعليق التوضيحي. هذا هو النص الذي سيكون مرئيًا لأي شخص يشاهد ملف PDF.

## الخطوة 6: إضافة التعليق التوضيحي إلى الصفحة

بعد ذلك، نحتاج إلى إضافة التعليق التوضيحي إلى مجموعة التعليقات التوضيحية الخاصة بالصفحة:

```csharp
// إضافة تعليق توضيحي إلى مجموعة التعليقات التوضيحية للصفحة
pdfDocument.Pages[1].Annotations.Add(freetext);
```

يضمن هذا السطر من التعليمات البرمجية إضافة التعليق التوضيحي الذي أنشأته حديثًا إلى مستند PDF. بدون هذه الخطوة، لن يظهر التعليق التوضيحي في الناتج النهائي.

## الخطوة 7: حفظ المستند المحدث

أخيرًا، حان وقت حفظ التغييرات. ستحتاج إلى تحديد اسم ملف جديد للمستند المُحدّث:

```csharp
dataDir = dataDir + "SetFreeTextAnnotationFormatting_out.pdf";
// حفظ المستند المحدث
pdfDocument.Save(dataDir);
```

يحفظ هذا الكود ملف PDF المُعدَّل باسم جديد، مما يضمن بقاء مستندك الأصلي دون تغيير. يمكنك الآن فتح ملف PDF الجديد لمشاهدة شرحك النصي المجاني قيد الاستخدام!

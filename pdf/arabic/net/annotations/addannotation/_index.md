# إضافة تعليق توضيحي لملف PDF

## مقدمة

تُعد التعليقات التوضيحية وسيلة رائعة لإثراء مستندات PDF، مما يجعلها تفاعلية وغنية بالمعلومات. سواء كنت تترك ملاحظات لأحد زملائك أو تضيف معلومات إضافية للقراء، فإن التعليقات التوضيحية ضرورية. في هذا البرنامج التعليمي، سنتعمق في عملية إضافة تعليقات توضيحية إلى ملفات PDF باستخدام Aspose.PDF لـ .NET. سنشرح كل خطوة بالتفصيل، بحيث تصبح بنهاية هذا الدليل محترفًا في تضمين التعليقات التوضيحية في ملفات PDF. هيا بنا نبدأ!

## استيراد الحزم

قبل البدء في الترميز، تأكد من استيراد مساحات الأسماء الضرورية:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
```

توفر هذه المساحات الاسمية إمكانية الوصول إلى الفئات والطرق التي تحتاجها لمعالجة ملفات PDF والتعليق عليها.

## الخطوة 1: تحميل مستند PDF الخاص بك

أولاً وقبل كل شيء، عليك تحميل مستند PDF الذي تخطط لإضافة التعليق التوضيحي إليه.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DATA DIRECTORY";
// فتح المستند
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

إليك ما يحدث: أنت تحدد الدليل الذي يتم تخزين ملف PDF الخاص بك فيه، ثم تقوم بتحميله باستخدام `Document` الفئة التي يوفرها Aspose.PDF. هذه الخطوة بالغة الأهمية، لأنه بدون تحميل المستند، لا يمكنك إجراء أي تعديلات عليه.

## الخطوة 2: إنشاء تعليق توضيحي

### تحديد خصائص الشرح التوضيحي
الآن، لنُنشئ التعليق التوضيحي نفسه. سنستخدم `TextAnnotation`، وهو مثالي لإضافة تعليقات أو ملاحظات إلى ملف PDF الخاص بك.

```csharp
// إنشاء تعليق توضيحي
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

في هذه المقتطفة:
- الموقع والحجم: `Rectangle` تعرف الفئة المكان الذي ستظهر فيه تعليقاتك التوضيحية على الصفحة وأبعادها.
- العنوان والموضوع والمحتويات: تسمح لك هذه الخصائص بتحديد موضوع التعليق التوضيحي الخاص بك وما سيحتويه.
- الأيقونة: `TextIcon.Key` تعيين رمز للتعليق التوضيحي، مما يجعله أكثر جاذبية بصريًا.

## الخطوة 3: تخصيص مظهر التعليقات التوضيحية

بعد ذلك، دعنا نجعل هذا التعليق التوضيحي بارزًا عن طريق إضافة حدود وتعديل مظهره.

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

فيما يلي تفصيل لما يحدث:
- الحدود: نقوم بإنشاء `Border` الكائن وضبط عرضه إلى 5، مما يمنح تعليقنا مخططًا بارزًا.
- نمط الاندفاعة: `Dash` تتيح لك الخاصية إنشاء حدود متقطعة، مما يضيف القليل من الأناقة إلى التعليقات التوضيحية.

## الخطوة 4: إضافة التعليق التوضيحي إلى صفحة PDF

بعد إنشاء التعليق التوضيحي وتخصيصه، حان الوقت لإضافته إلى صفحة PDF الخاصة بك.

```csharp
// إضافة تعليق توضيحي إلى مجموعة التعليقات التوضيحية للصفحة
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

يضيف هذا الكود التعليق التوضيحي إلى الصفحة الأولى من ملف PDF الخاص بك. `Annotations` تحتوي المجموعة على كل التعليقات التوضيحية لصفحة معينة، وتضمن هذه الخطوة أن تكون التعليقات التوضيحية الجديدة جزءًا من هذه المجموعة.

## الخطوة 5: حفظ مستند PDF المحدث

وأخيرًا، دعنا نحفظ المستند حتى تتم إضافة تعليقاتك التوضيحية بشكل دائم.

```csharp
// حفظ ملف الإخراج
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nAnnotation added successfully.\nFile saved at " + dataDir);
```

عن طريق حفظ المستند باسم جديد (`AddAnnotation_out.pdf`)، يمكنك حفظ الملف الأصلي وإنشاء ملف جديد بإضافة التعليقات التوضيحية. تؤكد رسالة وحدة التحكم نجاح العملية، ويمكنك الآن العثور على ملف PDF المُعلّق عليه في الدليل المحدد.

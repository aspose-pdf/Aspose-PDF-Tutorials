# تضمين الخط في ملف PDF

## مقدمة

عند إنشاء ملفات PDF، يُعدّ تضمين الخطوط المستخدمة في مستندك من أهمّ الجوانب. فهذا لا يحافظ على مظهر المستند على مختلف الأجهزة فحسب، بل يمنع أيضًا مشاكل استبدال الخطوط. في هذا البرنامج التعليمي، سنشرح لك عملية تضمين الخطوط في ملف PDF باستخدام Aspose.PDF لـ .NET. 

## المتطلبات الأساسية

قبل أن نتعمق في الكود، هناك بعض المتطلبات الأساسية التي يجب أن تكون موجودة:

1. Aspose.PDF لـ .NET: تأكد من تثبيت مكتبة Aspose.PDF. يمكنك تنزيلها من [موقع إلكتروني](https://releases.aspose.com/pdf/net/).
2. Visual Studio: بيئة تطوير يمكنك من خلالها كتابة وتنفيذ كود .NET الخاص بك.
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.

## استيراد الحزم

للبدء، عليك استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

1. افتح مشروع Visual Studio الخاص بك.
2. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول وحدد "إدارة حزم NuGet".
3. بحث عن `Aspose.PDF` وتثبيت الإصدار الأحدث.

```csharp
using System.IO;
using System;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
using Aspose.Pdf.Text;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا نقوم بتقسيم عملية تضمين الخطوط في ملف PDF خطوة بخطوة.

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، عليك تحديد مسار مجلد المستندات. هنا سيتم حفظ ملف PDF المُدخل وملف الإخراج.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

تأكد من الاستبدال `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يتم تخزين ملفات PDF الخاصة بك فيه.

## الخطوة 2: تحميل ملف PDF الموجود

بعد ذلك، ستحتاج إلى تحميل ملف PDF الحالي الذي تريد تعديله. يتم ذلك باستخدام `Document` تم توفير الفئة بواسطة Aspose.PDF.

```csharp
// تحميل ملف PDF موجود
Document doc = new Document(dataDir + "input.pdf");
```

هنا نقوم بتحميل ملف PDF باسم `input.pdf`تأكد من وجود هذا الملف في الدليل المحدد.

## الخطوة 3: التكرار عبر جميع الصفحات

بعد تحميل مستندنا، علينا مراجعة جميع صفحات ملف PDF. يتيح لنا هذا التحقق من الخطوط التي يجب تضمينها في كل صفحة.

```csharp
// كرر كل الصفحات
foreach (Page page in doc.Pages)
{
    // التحقق مما إذا كانت الصفحة تحتوي على موارد
    if (page.Resources.Fonts != null)
    {
        foreach (Aspose.Pdf.Text.Font pageFont in page.Resources.Fonts)
        {
            // تحقق مما إذا كان الخط مضمنًا بالفعل
            if (!pageFont.IsEmbedded)
                pageFont.IsEmbedded = true;
        }
    }
}
```

في هذا الكود، نتحقق مما إذا كانت الصفحة تحتوي على أي خطوط. إذا كانت كذلك، نمرر كل خط على حدة ونتحقق مما إذا كان مُضمّنًا بالفعل. إذا لم يكن كذلك، نضبط `IsEmbedded` الممتلكات إلى `true`.

## الخطوة 4: التحقق من كائنات النموذج

بالإضافة إلى خطوط الصفحات العادية، قد تحتوي ملفات PDF على عناصر نموذجية تستخدم الخطوط أيضًا. يجب التأكد من تضمين هذه الخطوط أيضًا.

```csharp
// التحقق من كائنات النموذج
foreach (XForm form in page.Resources.Forms)
{
    if (form.Resources.Fonts != null)
    {
        foreach (Aspose.Pdf.Text.Font formFont in form.Resources.Fonts)
        {
            // تحقق مما إذا كان الخط مضمنًا
            if (!formFont.IsEmbedded)
                formFont.IsEmbedded = true;
        }
    }
}
```

يتحقق مقتطف التعليمات البرمجية هذا من أي كائنات نموذج على الصفحة وينفذ نفس فحص التضمين لخطوطها.

## الخطوة 5: حفظ مستند PDF المعدّل

بعد تضمين الخطوط، حان وقت حفظ مستند PDF المُعدَّل. يمكنك تحديد اسم جديد للملف الناتج.

```csharp
dataDir = dataDir + "EmbedFont_out.pdf";
// حفظ مستند PDF
doc.Save(dataDir);
```

في هذه الحالة، نقوم بحفظ ملف PDF المعدل كـ `EmbedFont_out.pdf` في نفس الدليل.

## الخطوة 6: تأكيد العملية

وأخيرًا، يُنصح دائمًا بتأكيد نجاح العملية. يمكنك القيام بذلك بطباعة رسالة إلى وحدة التحكم.

```csharp
Console.WriteLine("\nFont embedded successfully in a PDF file.\nFile saved at " + dataDir);
```

ستتيح لك هذه الرسالة معرفة أنه تم تضمين الخطوط وتم حفظ الملف بنجاح.

## خاتمة

يُعد تضمين الخطوط في ملفات PDF عملية سهلة مع Aspose.PDF لـ .NET. باتباع الخطوات الموضحة في هذا البرنامج التعليمي، يمكنك ضمان احتفاظ مستندات PDF بمظهرها المطلوب على مختلف المنصات. سواء كنت تُنشئ تقارير أو نماذج أو أي نوع آخر من المستندات، يُعد تضمين الخطوط خطوة أساسية في عملية إنشاء PDF.

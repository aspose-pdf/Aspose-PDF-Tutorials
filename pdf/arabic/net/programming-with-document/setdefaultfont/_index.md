# تعيين الخط الافتراضي في ملف PDF

## مقدمة

هل سبق لك أن فتحت مستند PDF لتكتشف أن الخطوط مفقودة أو لا تظهر بشكل صحيح؟ قد يكون الأمر محبطًا، أليس كذلك؟ لا تقلق! في هذا البرنامج التعليمي، سنتعمق في كيفية تعيين خط افتراضي في ملف PDF باستخدام Aspose.PDF لـ .NET. تتيح لك هذه المكتبة القوية التعامل مع مستندات PDF بسهولة، وتعيين خط افتراضي هو مجرد واحدة من العديد من الميزات التي تقدمها. هيا، هيا بنا نبدأ!

## المتطلبات الأساسية

قبل أن ننتقل إلى الكود، هناك بعض الأشياء التي ستحتاج إلى وضعها في مكانها:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. إنه أفضل بيئة تطوير متكاملة لتطوير .NET.
2. Aspose.PDF لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك العثور عليها [هنا](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: إن القليل من المعرفة ببرمجة C# سوف يساعدك كثيرًا في فهم الأمثلة التي سنغطيها.

## استيراد الحزم

للبدء، ستحتاج إلى استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

1. افتح مشروع Visual Studio الخاص بك.
2. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول وحدد "إدارة حزم NuGet".
3. بحث عن `Aspose.PDF` وتثبيت الإصدار الأحدث.

بمجرد تثبيت الحزمة، ستكون جاهزًا لبدء الترميز!

## الخطوة 1: إعداد مشروعك

### إنشاء مشروع جديد

أولاً وقبل كل شيء، دعنا ننشئ مشروع C# جديد في Visual Studio:

- افتح Visual Studio وحدد "إنشاء مشروع جديد".
- اختر "تطبيق وحدة التحكم (.NET Core)" ثم انقر فوق "التالي".
- قم بتسمية مشروعك (على سبيل المثال، `AsposePdfExample`) ثم انقر فوق "إنشاء".

### إضافة باستخدام التوجيهات

الآن، دعنا نضيف توجيهات الاستخدام الضرورية في الجزء العلوي من ملفك `Program.cs` ملف:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System.IO;
```

ستتيح لك هذه التوجيهات الوصول إلى فئات وطرق Aspose.PDF.

## الخطوة 2: تحميل مستند PDF

### تحديد مسار المستند

بعد ذلك، ستحتاج إلى تحديد مسار ملف PDF الذي ترغب بالعمل عليه. إليك كيفية القيام بذلك:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // استبدل بالدليل الفعلي الخاص بك
string documentName = Path.Combine(dataDir, "input.pdf");
```

تأكد من الاستبدال `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يوجد به ملف PDF الخاص بك.

### تحميل المستند

الآن، دعنا نحمل مستند PDF الموجود:

```csharp
using (FileStream fs = new FileStream(documentName, FileMode.Open))
{
    Document document = new Document(fs);
}
```

يفتح مقتطف التعليمات البرمجية هذا ملف PDF وينشئ `Document` كائن يمكنك التحكم فيه.

## الخطوة 3: تعيين الخط الافتراضي

### إنشاء خيارات حفظ ملف Pdf

الآن يأتي الجزء المثير! ستحتاج إلى إنشاء مثيل لـ `PdfSaveOptions` لتحديد الخط الافتراضي:

```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
```

### حدد اسم الخط الافتراضي

بعد ذلك، ستُعيّن اسم الخط الافتراضي. في هذا المثال، سنستخدم "Arial":

```csharp
pdfSaveOptions.DefaultFontName = "Arial";
```

يخبر هذا السطر Aspose.PDF باستخدام Arial كخط افتراضي لأي نص لا يحتوي على خط محدد.

## الخطوة 4: حفظ المستند

أخيرًا، حان الوقت لحفظ مستند PDF المعدّل باستخدام الخط الافتراضي الجديد:

```csharp
document.Save(Path.Combine(dataDir, "output_out.pdf"), pdfSaveOptions);
```

هذا السطر يحفظ المستند كـ `output_out.pdf` في الدليل المحدد.

## خاتمة

ها قد انتهيت! لقد نجحت في تعيين خط افتراضي في ملف PDF باستخدام Aspose.PDF لـ .NET. هذه الميزة البسيطة والفعّالة تضمن لك ظهور مستنداتك بالشكل الذي تريده تمامًا، حتى في حال عدم وجود خطوط. لذا، في المرة القادمة التي تواجه فيها مشكلة في الخطوط في ملف PDF، ستعرف بالضبط ما يجب فعله!

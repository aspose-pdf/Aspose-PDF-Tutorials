## دليل التنفيذ

### تحسين ملفات PDF باستخدام محتوى الصفحة القابل لإعادة الاستخدام

تتيح لك هذه الميزة تقليل حجم ملف PDF عن طريق إعادة استخدام المحتوى عبر الصفحات حيثما أمكن.

#### الخطوة 1: تهيئة مشروعك
أنشئ مشروع .NET جديدًا وتأكد من تثبيت Aspose.PDF كما هو موضح أعلاه. جهّز مجلدات ملفاتك:

```csharp
class Program
{
    static void Main(string[] args)
    {
        string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // الدليل الذي يحتوي على ملف PDF المصدر الخاص بك
        string outputDir = "YOUR_OUTPUT_DIRECTORY"; // دليل لحفظ ملف PDF المُحسَّن
        
        // تحميل مستند PDF موجود
        Document pdfDocument = new Document(dataDir + "/OptimizeDocument.pdf");
    }
}
```

#### الخطوة 2: تكوين خيارات التحسين
إنشاء مثيل جديد من `Pdf.Optimization.OptimizationOptions`، مما يتيح إعادة استخدام محتوى الصفحة.

```csharp
class Program
{
    static void Main(string[] args)
    {
        var optimizeOptions = new Pdf.Optimization.OptimizationOptions
        {
            AllowReusePageContent = true // قم بتمكين هذا للسماح بإعادة استخدام محتوى الصفحة عبر الصفحات
        };
    }
}
```

#### الخطوة 3: تحسين المستند وحفظه
قم بتطبيق خيارات التحسين على مستندك وحفظه في ملف جديد.

```csharp
class Program
{
    static void Main(string[] args)
    {
        // تحسين الموارد في مستند PDF باستخدام الخيارات المحددة
        pdfDocument.OptimizeResources(optimizeOptions);
        
        // احفظ المستند المُحسَّن في ملف جديد في دليل الإخراج
        pdfDocument.Save(outputDir + "/Optimized_OptimizeDocument_out.pdf");
    }
}
```

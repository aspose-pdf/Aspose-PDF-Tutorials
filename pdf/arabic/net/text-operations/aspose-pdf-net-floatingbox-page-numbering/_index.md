### إضافة FloatingBox لترقيم الصفحات

أ `FloatingBox` يتيح لك وضع المحتوى في مواضع محددة في صفحات PDF. إليك كيفية استخدامه لإضافة أرقام الصفحات:

#### الخطوة 1: إنشاء مستند وإضافة صفحات
أولاً، قم بإنشاء مستند جديد وأضف صفحة سيتم وضع المربع العائم فيها.

```csharp
// إنشاء مستند PDF جديد
Document document = new Document();

// إضافة صفحة إلى مستند pdf
Page page = document.Pages.Add();
```

#### الخطوة 2: تهيئة FloatingBox

إنشاء مثيل لـ `FloatingBox` بالأبعاد المطلوبة ووضعها بشكل مناسب على صفحتك.

```csharp
// تهيئة FloatingBox بالعرض والارتفاع
FloatingBox box1 = new FloatingBox(140, 80);

// اضبط الموضعين الأيسر والأعلى للحصول على وضع دقيق
box1.Left = 2;
box1.Top = 10;

// إضافة نص ترقيم الصفحات إلى فقرات FloatingBox
box1.Paragraphs.Add(new Text.TextFragment("Page: ($p/ $P )"));

// إضافة المربع العائم إلى الصفحة الحالية
page.Paragraphs.Add(box1);
```

**توضيح:**  
- `FloatingBox(140, 80)`:يحدد حجم المربع.
- `$p` و `$P`:عناصر نائبة للصفحات الحالية والإجمالية.

#### الخطوة 3: حفظ المستند

وأخيرًا، احفظ مستندك في الموقع المحدد.

```csharp
// تحديد مسار الإخراج
string outputPath = "YOUR_OUTPUT_DIRECTORY/PageNumberinHeaderFooterUsingFloatingBox_out.pdf";

// حفظ المستند
document.Save(outputPath);
```

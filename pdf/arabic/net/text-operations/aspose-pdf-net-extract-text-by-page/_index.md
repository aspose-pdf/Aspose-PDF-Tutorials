## دليل التنفيذ
بعد إعداد كل شيء، دعنا نستعرض خطوات التنفيذ.

### تهيئة PdfExtractor وربط PDF
#### ملخص
ابدأ بالتهيئة `PdfExtractor` مثال وربط ملف PDF المستهدف به.

```csharp
// المسار إلى دليل المستندات.
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Text();

// فتح ملف PDF المدخل
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "ExtractText-Page.pdf");
```
#### توضيح
- **`BindPdf(string)`**:يقوم بإعداد المستخرج عن طريق ربط ملف PDF، وإعداده لاستخراج النص.
- ال `dataDir` يشير المتغير إلى الدليل الذي يحتوي على ملف PDF.

### استخراج النص من كل صفحة
#### ملخص
قم بالمرور على كل صفحة من صفحات ملف PDF واستخراج محتواها في ملفات نصية منفصلة.

```csharp
// استخدم طريقة ExtractText بدون معلمات
currentPage = 1;
pdfExtractor.ExtractText();
while (pdfExtractor.HasNextPageText())
{
    using (MemoryStream tempMemoryStream = new MemoryStream())
    {
        pdfExtractor.GetNextPageText(tempMemoryStream);

        string text = Encoding.Unicode.GetString(tempMemoryStream.ToArray());

        File.WriteAllText(dataDir + "output" + currentPage + "_out.txt", text);
        currentPage++;
    }
}
```
#### توضيح
- **`ExtractText()`**:بدء عملية الاستخراج لجميع الصفحات.
- **`HasNextPageText()`**:التحقق مما إذا كانت هناك صفحات أخرى متاحة للمعالجة.
- **`GetNextPageText(MemoryStream)`**:يستخرج النص من الصفحة التالية إلى `MemoryStream`.
- **ترميز يونيكود**:يضمن قراءة دقيقة للنص المستخرج، وهو أمر مهم بشكل خاص بالنسبة للأحرف الخاصة.

### نصائح استكشاف الأخطاء وإصلاحها

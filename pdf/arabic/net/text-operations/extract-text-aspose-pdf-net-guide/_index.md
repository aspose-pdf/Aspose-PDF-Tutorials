## دليل التنفيذ

في هذا القسم، سنُقسّم تنفيذ الكود إلى أجزاء سهلة الإدارة. سنُركّز على استخراج النص من ملفات PDF باستخدام عبارات بحث مُحدّدة.

### استخراج أجزاء نصية باستخدام C#

#### ملخص

تتيح لك هذه الميزة البحث عن جميع تكرارات عبارة محددة واستخراجها من مستند PDF. يوضح المثال أدناه كيفية العثور على تكرارات كلمة "Figure" في ملف PDF.

#### خطوات التنفيذ

1. **قم بإعداد مشروعك**
   
   تأكد من تثبيت Aspose.PDF وإدراج المساحات الأساسية اللازمة:
   
   ```csharp
   using System;
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

2. **تحميل المستند**
   
   افتح ملف PDF الخاص بك باستخدام `Document` فصل.
   
   ```csharp
   string dataDir = RunExamples.GetDataDir_AsposePdf_Text();
   Document pdfDocument = new Document(dataDir + "SearchAndGetTextPage.pdf");
   ```

3. **إنشاء كائن امتصاص النص**
   
   يستخدم `TextFragmentAbsorber` لتحديد النص الذي تبحث عنه:
   
   ```csharp
   TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("Figure");
   pdfDocument.Pages.Accept(textFragmentAbsorber);
   ```

4. **استخراج وعرض أجزاء النص**
   
   قم بالتكرار خلال كل جزء، مع عرض التفاصيل ذات الصلة:
   
   ```csharp
   foreach (TextFragment textFragment in textFragmentAbsorber.TextFragments)
   {
       foreach (TextSegment textSegment in textFragment.Segments)
       {
           Console.WriteLine("Text: {0} ", textSegment.Text);
           Console.WriteLine("Position: {0} ", textSegment.Position);
           // تفاصيل إضافية مثل خصائص الخط
           Console.WriteLine("Font - Name: {0}", textSegment.TextState.Font.FontName);
       }
   }
   ```

- **ممتص شظايا النص**:البحث عن العبارات المحددة.
- **شظايا النص**:مجموعة من جميع الحالات التي تم العثور عليها.

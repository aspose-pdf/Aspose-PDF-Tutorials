## دليل التنفيذ

في هذا القسم، سنقوم بتقسيم عملية تضمين الخطوط المخصصة خطوة بخطوة.

### ملخص
يضمن تضمين الخط الحفاظ على المظهر والأسلوب المطلوبين لمستندك لدى مختلف المشاهدين. تُعد هذه الميزة بالغة الأهمية عند التعامل مع مستندات تحتوي على خطوط أو عناصر أسلوبية خاصة بعلامة تجارية.

#### الخطوة 1: إنشاء مستند PDF جديد

```csharp
// قم بإنشاء كائن Pdf عن طريق استدعاء المنشئ الفارغ الخاص به.
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

*توضيح:* نبدأ بإنشاء مثيل لـ `Document` الفئة، التي تعمل بمثابة الحاوية الخاصة بنا لإضافة النص والعناصر الأخرى.

#### الخطوة 2: إضافة صفحة إلى المستند

```csharp
// إنشاء قسم (صفحة) في كائن Pdf.
Aspose.Pdf.Page page = doc.Pages.Add();
```

*توضيح:* كل مستند PDF يتكون من صفحات. هنا، نضيف صفحة جديدة حيث سيتواجد النص.

#### الخطوة 3: تضمين الخطوط المخصصة
الآن يأتي الجزء الحاسم - تضمين الخط الذي اخترته:

```csharp
// إنشاء TextFragment بسلسلة فارغة.
Aspose.Pdf.Text.TextFragment fragment = new Aspose.Pdf.Text.TextFragment("");

// قم بإنشاء TextSegment باستخدام نص عينة وحدد الخط المخصص.
Aspose.Pdf.Text.TextSegment segment = new Aspose.Pdf.Text.TextSegment("This is a sample text using Custom font.");
Aspose.Pdf.Text.TextState ts = new Aspose.Pdf.Text.TextState();

// ابحث عن الخط من المستودع، واضبطه ليتم تضمينه
ts.Font = FontRepository.FindFont("Arial");
ts.Font.IsEmbedded = true;

segment.TextState = ts;
fragment.Segments.Add(segment);
page.Paragraphs.Add(fragment);
```

*توضيح:* نحن ننشئ `TextFragment` و أ `TextSegment`تم ضبط حالة النص لاستخدام خط "Arial"، والذي نُضبطه بعد ذلك ليتم تضمينه في ملف PDF. هذا يضمن ظهور خط Arial بشكل متسق عبر مختلف برامج العرض.

#### الخطوة 4: حفظ المستند
وأخيرًا، احفظ مستندك باستخدام الخطوط المضمنة:

```csharp
// قم بتحديد المسار لحفظ ملف الإخراج.
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
dataDir = dataDir + "EmbedFontWhileDocCreation_out.pdf";

// حفظ مستند PDF
doc.Save(dataDir);
```

*توضيح:* يتم حفظ المستند في الموقع المحدد، مع الحفاظ على جميع الخطوط المضمنة.

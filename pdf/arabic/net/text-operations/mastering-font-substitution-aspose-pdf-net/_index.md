### تنفيذ معالجات استبدال الخطوط

**ملخص**
تحدث عمليات استبدال الخطوط عندما يشير مستند PDF إلى خط غير متوفر. بمعالجة هذه الأحداث، يمكنك تسجيل هذه التغييرات وإدارتها بفعالية.

#### الخطوة 1: إعداد البيئة الخاصة بك
أنشئ مشروع C# جديدًا في بيئة التطوير المتكاملة المفضلة لديك. أضف حزمة Aspose.PDF كما هو موضح أعلاه.

#### الخطوة 2: إنشاء معالج حدث استبدال الخط

أضف معالج حدث لمراقبة استبدالات الخطوط:

```csharp
using System;
using Aspose.Pdf;

namespace AsposePdfExamples
{
    public class GetWarningsForFontSubstitution
    {
        public static void Run()
        {
            // المسار إلى دليل المستندات.
            string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();

            Document doc = new Document(dataDir + "input.pdf");

            // اشترك في حدث FontSubstitution
            doc.FontSubstitution += OnFontSubstitution;
        }

        static void OnFontSubstitution(Font oldFont, Font newFont)
        {
            Console.WriteLine($"Font '{oldFont.FontName}' was substituted with '{newFont.FontName}'.");
        }
    }
}
```

**توضيح:**
- ال `OnFontSubstitution` تسجل الطريقة أحداث استبدال الخطوط. هذا ضروري لتتبع وتصحيح الأخطاء الناتجة عن الخطوط المفقودة.
- يتلقى معالج الحدث معامِلين، `oldFont` و `newFont`، تمثل الخطوط الأصلية والبديلة على التوالي.

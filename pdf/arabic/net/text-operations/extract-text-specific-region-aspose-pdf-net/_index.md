## دليل التنفيذ

### استخراج النص من منطقة محددة
اتبع الخطوات التالية لاستخراج النص من مناطق محددة في صفحة PDF:

#### الخطوة 1: تحميل المستند الخاص بك
قم بتحميل مستند PDF إلى تطبيقك.
```csharp
Document pdfDocument = new Document("path/to/your/ExtractTextAll.pdf");
```

#### الخطوة 2: إنشاء وتكوين TextAbsorber
يستخدم `TextAbsorber` لتحديد النص الذي تريد استخراجه، وتقييد الاستخراج إلى منطقة صفحة محددة عن طريق تعيين إحداثيات المستطيل.
```csharp
// تهيئة كائن TextAbsorber
TextAbsorber absorber = new TextAbsorber();

// تقييد البحث النصي ضمن حدود الصفحة
absorber.TextSearchOptions.LimitToPageBounds = true;

// تحديد المنطقة المستطيلة التي سيتم استخراج النص منها (x، y، العرض، الارتفاع)
absorber.TextSearchOptions.Rectangle = new Aspose.Pdf.Rectangle(100, 200, 250, 350);
```

#### الخطوة 3: قبول الامتصاص للصفحة المستهدفة
تقبل `TextAbsorber` الكائن الموجود على الصفحة المطلوبة لبدء استخراج النص.
```csharp
// استخراج النص من الصفحة الأولى
document.Pages[1].Accept(absorber);
```

#### الخطوة 4: استرداد النص المستخرج وحفظه
استرجاع النص المستخرج وحفظه في ملف.
```csharp
// احصل على النص المستخرج
string extractedText = absorber.Text;

// كتابة النص المستخرج إلى ملف جديد
using (TextWriter tw = new StreamWriter("path/to/extracted-text.txt"))
{
    tw.WriteLine(extractedText);
}
```

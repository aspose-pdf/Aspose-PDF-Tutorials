## دليل التنفيذ
يقوم هذا القسم بتقسيم التنفيذ إلى ميزتين رئيسيتين: تضمين جميع الخطوط وتقسيم الخطوط المضمنة بالكامل فقط.

### الميزة 1: تضمين جميع الخطوط كمجموعة فرعية
#### ملخص
تضمن هذه الميزة تضمين كل خط مستخدم في ملف PDF الخاص بك كمجموعة فرعية، مما يوفر الاتساق عبر بيئات العرض المختلفة.

#### خطوات التنفيذ
**قم بتحميل مستندك**
ابدأ بتحميل مستند PDF موجود من نظام الملفات:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

**مجموعة فرعية من جميع الخطوط**
استخدم `SubsetAllFonts` استراتيجية لتضمين جميع الخطوط كمجموعات فرعية:

```csharp
doc.FontUtilities.SubsetFonts(FontSubsetStrategy.SubsetAllFonts);
```

ويضمن هذا تضمين حتى الخطوط غير المضمنة، مما يعزز التوافق.

**حفظ المستند**
وأخيرًا، احفظ المستند المعدّل في ملف جديد:

```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputPath + "/Output_All_Fonts_Subset.pdf");
```

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من إمكانية الوصول إلى جميع ملفات الخطوط في حالة حدوث أخطاء أثناء التضمين.
- التحقق من مسارات الدليل للتأكد من دقتها.

### الميزة 2: تضمين الخطوط المضمنة فقط كمجموعة فرعية
#### ملخص
ترتكز هذه الميزة على استبعاد الخطوط المضمنة فقط، مع ترك الخطوط غير المضمنة دون تأثير.

#### خطوات التنفيذ
**قم بتحميل مستندك**
قم بتحميل ملف PDF كما فعلت سابقًا:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

**مجموعة فرعية من الخطوط المضمنة فقط**
تطبيق `SubsetEmbeddedFontsOnly` الاستراتيجية:

```csharp
doc.FontUtilities.SubsetFonts(FontSubsetStrategy.SubsetEmbeddedFontsOnly);
```

لا تؤثر هذه الطريقة على الخطوط غير المضمنة.

**حفظ المستند**
احفظ التغييرات الخاصة بك مع:

```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputPath + "/Output_Embedded_Fonts_Subset.pdf");
```

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من حالة الخط المضمن قبل تطبيق هذه الاستراتيجية.
- تأكيد مسارات الملفات والأذونات.

2. **مشاركة المستندات**:شارك ملفات PDF مع ضمان إمكانية القراءة دون مشاكل استبدال الخط.
3. **تحسين حجم الملف**:تؤدي عملية تقسيم الملفات إلى أجزاء إلى تقليل حجم الملف، وهو أمر مفيد لمرفقات البريد الإلكتروني والمشاركة عبر الإنترنت.

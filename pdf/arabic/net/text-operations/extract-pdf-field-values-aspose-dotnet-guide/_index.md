#### 1. افتح المستند
ابدأ بتحميل ملف PDF الخاص بك إلى Aspose.PDF `Document` هدف:

```csharp

Document pdfDocument = new Document(dataDir + "GetValuesFromAllFields.pdf");
```
**لماذا:** تعمل هذه الخطوة على تهيئة المستند الذي ستستخرج منه قيم الحقول.

#### 2. استرداد قيم الحقول
قم بالتكرار خلال كل حقل نموذج وعرض اسمه وقيمته:

```csharp

{
    Console.WriteLine("Field Name: {0}", formField.PartialName);
    Console.WriteLine("Value: {0}", formField.Value);
}
```

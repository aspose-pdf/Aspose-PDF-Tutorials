تتيح لك هذه الميزة استخراج نص مُضمّن في تعليق ختم على صفحة PDF. اتبع الخطوات التالية لتطبيقها:

#### الخطوة 1: تحميل مستند PDF

أولاً، حدد دليل المستند الخاص بك وقم بتحميل ملف PDF.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/test.pdf";
Document doc = new Document(dataDir);
```

#### الخطوة 2: الوصول إلى شرح الختم

استخدم شرح الطابع الذي تحتاجه. في هذا المثال، نعمل على شرح في الصفحة الأولى.
```csharp
StampAnnotation annot = (StampAnnotation)doc.Pages[1].Annotations[3];
```

#### الخطوة 3: استخراج النص باستخدام TextAbsorber

تهيئة `TextAbsorber` لاستخراج النص من مظهر الختم.
```csharp
TextAbsorber ta = new TextAbsorber();
ta.Visit(annot.Appearance.Form);
string extractedText = ta.Text;
```

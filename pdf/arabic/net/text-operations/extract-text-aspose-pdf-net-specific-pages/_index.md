## دليل التنفيذ

بعد أن انتهيت من إعداد كل شيء، لنركز على استخراج النص من صفحات محددة في مستند PDF. سنقسم هذه المهمة إلى أقسام يسهل التعامل معها.

### استخراج النص من صفحات محددة

#### ملخص
تتيح هذه الميزة استخراج النص بدقة من نطاقات الصفحات المحددة داخل ملف PDF، وهي مفيدة للتعامل مع المستندات الكبيرة أو التركيز على الأقسام ذات الصلة فقط.

#### خطوات التنفيذ:

**الخطوة 1: إعداد مسارات الملفات**
قم بتحديد أدلة الإدخال والإخراج الخاصة بك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**الخطوة 2: تهيئة PdfExtractor**
إنشاء مثيل لـ `PdfExtractor` للتعامل مع عمليات ملفات PDF.
```csharp
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "/ExtractText-PageRange.pdf");
```

**الخطوة 3: تحديد نطاق الصفحات للاستخراج**
حدد الصفحات التي تريد استخراج النص منها:
```csharp
pdfExtractor.StartPage = 1;
pdfExtractor.EndPage = 1; // قم بالتعديل حسب الحاجة لنطاق الصفحة المحددة لديك.
```
ال `StartPage` و `EndPage` الخصائص تحدد حدود الاستخراج.

**الخطوة 4: استخراج النص**
استدعاء الطريقة لاستخراج النص من الصفحات المحددة:
```csharp
pdfExtractor.ExtractText();
```

**الخطوة 5: تخزين النص المستخرج في مجرى الذاكرة**
إنشاء `MemoryStream` للاحتفاظ بالنص المستخرج مؤقتًا.
```csharp
using System.IO;
using System.Text;

MemoryStream tempMemoryStream = new MemoryStream();
pdfExtractor.GetText(tempMemoryStream);
```

**الخطوة 6: قراءة النص المستخرج وحفظه**
استخدم `StreamReader` لقراءة من مجرى الذاكرة، ثم حفظه في ملف إخراج:
```csharp
string text = "";
using (StreamReader sr = new StreamReader(tempMemoryStream, Encoding.Unicode))
{
    sr.BaseStream.Seek(0, SeekOrigin.Begin);
    text = sr.ReadToEnd();
}
File.WriteAllText(outputDir + "/output_out.txt", text);
```
تقوم هذه العملية بالتقاط النص المستخرج وحفظه لاستخدامه لاحقًا.

### نصائح استكشاف الأخطاء وإصلاحها
- **أخطاء نطاق الصفحات**: يضمن `StartPage` أقل من أو يساوي `EndPage`.
- **مشاكل MemoryStream**:قم دائمًا بإعادة تعيين موضع التدفق قبل القراءة.
- **صحة مسار الملف**:تحقق من صحة مسارات الدليل وإمكانية الوصول إليها.

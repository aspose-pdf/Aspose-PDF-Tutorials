### استخراج الخطوط من ملفات PDF
#### ملخص
الهدف الأساسي هنا هو استخراج جميع الخطوط المستخدمة في ملف PDF معين، وهو أمر بالغ الأهمية لتدقيق اتساق المستندات أو إعداد الملفات لتدفقات عمل محددة.

#### التنفيذ خطوة بخطوة
**1. قم بتحميل المستند**
ابدأ بتحميل مستند PDF الخاص بك إلى مثيل `Document` فصل.

```csharp
using Aspose.Pdf;

// تهيئة كائن المستند
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
Document doc = new Document(dataDir + "input.pdf");
```

**2. استخراج الخطوط**
استخدم `FontUtilities.GetAllFonts()` طريقة لاسترجاع مجموعة من الخطوط المستخدمة في المستند.

```csharp
Aspose.Pdf.Text.Font[] fonts = doc.FontUtilities.GetAllFonts();

foreach (Aspose.Pdf.Text.Font font in fonts)
{
    Console.WriteLine(font.FontName);
}
```

**توضيح:**
- `FontUtilities.GetAllFonts()`:تقوم هذه الطريقة بمسح ملف PDF وإرجاع مجموعة من جميع الخطوط الفريدة المستخدمة.
- تتكرر الحلقة على كل خط، وتخرج اسمه إلى وحدة التحكم.

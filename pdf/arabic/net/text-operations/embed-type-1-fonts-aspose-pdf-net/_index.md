## دليل التنفيذ

### تضمين الخطوط القياسية من النوع 1

يُعدّ تضمين الخطوط أمرًا بالغ الأهمية للحفاظ على سلامة المستندات ومظهرها عبر مختلف المنصات. تضمن هذه الميزة ظهور نصك بشكل متسق، بغض النظر عن مكان عرض ملف PDF.

#### عملية خطوة بخطوة

**قم بتحميل مستندك الحالي**

ابدأ بتحميل ملف PDF الذي تريد تعديله:
```csharp
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

**تعيين خاصية تضمين الخطوط القياسية**

تأكد من تضمين الخطوط القياسية من النوع 1 في مستندك:
```csharp
pdfDocument.EmbedStandardFonts = true;
```

**التكرار عبر الصفحات والخطوط**

بعد ذلك، قم بالتكرار خلال كل صفحة وموارد الخطوط الخاصة بها لتضمين الخطوط الضرورية:
```csharp
foreach (Page page in pdfDocument.Pages)
{
    if (page.Resources.Fonts != null)
    {
        foreach (Text.Font pageFont in page.Resources.Fonts)
        {
            // تضمين الخط إذا لم يكن مضمنًا بالفعل
            if (!pageFont.IsEmbedded)
            {
                pageFont.IsEmbedded = true;
            }
        }
    }
}
```

يضمن هذا تضمين جميع الخطوط المستخدمة في مستندك، مع الحفاظ على مظهرها.

**حفظ المستند المحدث**

وأخيرًا، احفظ ملف PDF المحدث:
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/EmbeddedFonts-updated_out.pdf");
```

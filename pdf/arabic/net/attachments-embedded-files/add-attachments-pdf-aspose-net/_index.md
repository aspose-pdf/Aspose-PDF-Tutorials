## دليل التنفيذ

### إضافة المرفقات إلى مستند PDF
تتضمن إضافة المرفقات عدة خطوات. إليك كيفية القيام بذلك باستخدام Aspose.PDF:

#### الخطوة 1: جهّز بيئتك
تأكد من أن مشروعك يشير إلى مكتبة Aspose.PDF ويتضمن مساحات الأسماء الضرورية.

```csharp
using System.IO;
using Aspose.Pdf.Facades;
```

#### الخطوة 2: تهيئة محرر محتوى Pdf
ال `PdfContentEditor` تتيح لك هذه الفئة تعديل محتوى PDF. ستستخدمها لإضافة المرفقات.

```csharp
// إنشاء كائن PdfContentEditor
PdfContentEditor contentEditor = new PdfContentEditor();
```

#### الخطوة 3: ربط مستند PDF الخاص بك
افتح مستند PDF وقم بربطه حيث سيتم إضافة المرفق.

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Attachments(); // حدد مسار الدليل الخاص بك
contentEditor.BindPdf(dataDir + "AddAttachment-Stream.pdf");
```

#### الخطوة 4: إرفاق ملف
اقرأ الملف الذي تريد إرفاقه باستخدام تيار وأضفه كمرفق.

```csharp
// افتح الملف في FileStream
using (FileStream fileStream = new FileStream(dataDir + "test.txt", FileMode.Open))
{
    // أضف مرفق المستند مع الاسم والوصف
    contentEditor.AddDocumentAttachment(fileStream, "Attachment Name", "Attachment Description");
}
```

#### الخطوة 5: احفظ ملف PDF
وأخيرًا، احفظ ملف PDF المحدث للحفاظ على التغييرات.

```csharp
contentEditor.Save(dataDir + "AddAttachment-Stream_out.pdf");
```

## قسم الأسئلة الشائعة
1. **كيف يمكنني إضافة مرفقات متعددة إلى ملف PDF واحد؟**

2. **هل يمكنني إرفاق الملفات من عناوين URL مباشرة؟**

4. **هل هناك حد لحجم المرفقات في Aspose.PDF؟**
   - على الرغم من عدم وجود حد محدد، فإن القيود العملية مثل مساحة القرص والذاكرة يجب أن توجه أحجام المرفقات التي تستخدمها.

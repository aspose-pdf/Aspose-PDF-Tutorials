## دليل التنفيذ

### إضافة المرفقات إلى ملف PDF

اتبع الخطوات التالية لإرفاق الملفات باستخدام Aspose.PDF لـ .NET.

#### الخطوة 1: افتح مستند PDF الموجود
قم بتحميل ملف PDF الحالي الذي تريد إضافة المرفقات إليه:
```csharp
// المسار إلى دليل المستندات.
string dataDir = RunExamples.GetDataDir_AsposePdf_Attachments();

// تحميل مستند PDF موجود
Document pdfDocument = new Document(dataDir + "AddAttachment.pdf");
```
*توضيح*:استخدم `Document` فئة من Aspose.PDF لتحميل ملف PDF. تم إنشاء المسار باستخدام طريقة مساعدة لمسارات الدليل.

#### الخطوة 2: إعداد مواصفات الملف للمرفق
إنشاء مثيل لـ `FileSpecification`، الذي يحدد تفاصيل المرفق:
```csharp
// قم بتحديد الملف الذي تريد إرفاقه ووصفه
FileSpecification fileSpecification = new FileSpecification(dataDir + "test.txt", "Sample text file");
```
*توضيح*: ال `FileSpecification` يتطلب المنشئ مسارًا للملف ووصفًا اختياريًا لسهولة التعرف عليه.

#### الخطوة 3: إضافة مرفق إلى مستند PDF
أضف مواصفات الملف إلى مجموعة مرفقات المستند:
```csharp
// قم بإرفاق الملف بمجموعة الملفات المضمنة في ملف PDF
pdfDocument.EmbeddedFiles.Add(fileSpecification);
```
*توضيح*:تضيف هذه الطريقة الملف المحدد كمرفق مخزن في البنية الداخلية لملف PDF.

#### الخطوة 4: حفظ مستند PDF المحدث
احفظ المستند بالمرفقات المضافة حديثًا:
```csharp
// حدد مسار الإخراج واحفظ ملف PDF المحدث
dataDir = dataDir + "AddAttachment_out.pdf";
pdfDocument.Save(dataDir);
```
*توضيح*:تكتب هذه الخطوة جميع التغييرات مرة أخرى في ملف، وهو أمر بالغ الأهمية لاستمرار التحديثات الخاصة بك.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسارات الملفات صحيحة ويمكن الوصول إليها.
- تحقق من الأذونات إذا لم يتمكن التطبيق من قراءة/كتابة الملفات.

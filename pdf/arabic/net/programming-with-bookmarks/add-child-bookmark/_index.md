# إضافة إشارة مرجعية فرعية في ملف PDF

## مقدمة

في العصر الرقمي، تُعدّ إدارة المستندات بكفاءة أمرًا بالغ الأهمية، خاصةً فيما يتعلق بملفات PDF. هل سبق لك أن وجدت نفسك تتصفح ملف PDF طويلًا بلا نهاية، محاولًا العثور على قسم محدد؟ أمرٌ مُحبط، أليس كذلك؟ وهنا تبرز أهمية الإشارات المرجعية! فهي بمثابة جدول محتويات، مما يُتيح للقراء التنقل عبر المستند بسهولة. في هذا البرنامج التعليمي، سنستكشف كيفية إضافة إشارات مرجعية فرعية إلى ملف PDF باستخدام Aspose.PDF لـ .NET. بنهاية هذا الدليل، ستتمكن من تحسين مستندات PDF الخاصة بك، وجعلها أكثر سهولة في الاستخدام وتنظيمًا.

## المتطلبات الأساسية

قبل أن نتعمق في التفاصيل المتعلقة بإضافة الإشارات المرجعية، هناك بعض الأشياء التي تحتاج إلى وضعها في مكانها:

1. Aspose.PDF لـ .NET: تأكد من تثبيت مكتبة Aspose.PDF. يمكنك تنزيلها من [موقع](https://releases.aspose.com/pdf/net/).
2. Visual Studio: بيئة تطوير يمكنك من خلالها كتابة واختبار الكود الخاص بك.
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.

## استيراد الحزم

للبدء، عليك استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

### إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع C# جديدًا. اختر تطبيق وحدة التحكم لتسهيل الأمر.

### إضافة مرجع Aspose.PDF

1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد "إدارة حزم NuGet".
3. ابحث عن "Aspose.PDF" وقم بتثبيت الإصدار الأحدث.

### استيراد مساحات الأسماء المطلوبة

في الجزء العلوي من `Program.cs` الملف، استيراد المساحات الأساسية اللازمة:

```csharp
using System;
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```
الآن بعد أن قمت بإعداد كل شيء، دعنا نقوم بتقسيم عملية إضافة الإشارات المرجعية الفرعية خطوة بخطوة.

## الخطوة 1: إعداد دليل المستندات الخاص بك

قبل أن تتمكن من التعامل مع أي ملف PDF، عليك تحديد مكان تخزين مستنداتك. هذا ضروري ليتمكن الكود من تحديد موقع ملف PDF.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يستبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي لملف PDF الخاص بك. هذا يشبه إعطاء كودك خريطة للعثور على الكنز!

## الخطوة 2: افتح مستند PDF

الآن بعد أن قمنا بإعداد الدليل، حان الوقت لفتح مستند PDF الذي تريد العمل معه.

```csharp
// فتح المستند
Document pdfDocument = new Document(dataDir + "AddChildBookmark.pdf");
```

هنا، نقوم بإنشاء جديد `Document` كائن يُحمّل ملف PDF الخاص بك. تخيّل هذا كأنك تفتح كتابًا لتبدأ القراءة.

## الخطوة 3: إنشاء إشارة مرجعية للوالدين

بعد ذلك، سننشئ إشارة مرجعية رئيسية. ستكون هذه الإشارة المرجعية العنوان الرئيسي الذي سنضيف تحته الإشارات المرجعية الفرعية.

```csharp
// إنشاء كائن إشارة مرجعية رئيسي
OutlineItemCollection pdfOutline = new OutlineItemCollection(pdfDocument.Outlines);
pdfOutline.Title = "Parent Outline";
pdfOutline.Italic = true;
pdfOutline.Bold = true;
```

في هذا المقطع، نقوم بإنشاء جديد `OutlineItemCollection` للإشارة المرجعية الأصلية. حددنا عنوانها ونمطها (مائل وعريض) لإبرازها. كأنك تختار عنوانًا جذابًا لفصلك!

## الخطوة 4: إنشاء إشارة مرجعية فرعية

الآن، دعنا نضيف إشارة مرجعية فرعية أسفل الإشارة المرجعية الأصلية التي أنشأناها للتو.

```csharp
// إنشاء كائن إشارة مرجعية فرعية
OutlineItemCollection pdfChildOutline = new OutlineItemCollection(pdfDocument.Outlines);
pdfChildOutline.Title = "Child Outline";
pdfChildOutline.Italic = true;
pdfChildOutline.Bold = true;
```

على غرار العلامة المرجعية الأصلية، ننشئ علامة مرجعية فرعية بعنوانها ونمطها الخاصين. تُدمج هذه العلامة الفرعية تحت العلامة الأصلية، مما يُنشئ تسلسلًا هرميًا.

## الخطوة 5: إضافة إشارة مرجعية للطفل إلى الأصل

بعد إنشاء كلا الإشارتين المرجعيتين، حان الوقت لربطهما معًا.

```csharp
// إضافة إشارة مرجعية فرعية إلى مجموعة الإشارات المرجعية الأصلية
pdfOutline.Add(pdfChildOutline);
```

هذا السطر من التعليمات البرمجية يُضيف العلامة الفرعية إلى مجموعة العلامات الرئيسية. يشبه الأمر إضافة عنوان فرعي تحت عنوان فصل!

## الخطوة 6: إضافة الإشارة المرجعية الأصلية إلى المستند

الآن بعد أن قمنا بإعداد إشاراتنا المرجعية للوالد والطفل، نحتاج إلى إضافة إشارة المرجع للوالد إلى مجموعة الخطوط العريضة للمستند.

```csharp
// أضف إشارة مرجعية رئيسية إلى مجموعة الخطوط العريضة للمستند.
pdfDocument.Outlines.Add(pdfOutline);
```

تضمن هذه الخطوة أن تصبح الإشارة المرجعية الأصلية، مع الإشارة الفرعية التابعة لها، جزءًا من مستند PDF. وكأنك تنشر كتابك رسميًا بجميع فصوله!

## الخطوة 7: حفظ المستند

وأخيرًا، دعنا نحفظ التغييرات التي أجريناها على مستند PDF.

```csharp
dataDir = dataDir + "AddChildBookmark_out.pdf";
// حفظ الناتج
pdfDocument.Save(dataDir);
Console.WriteLine("\nChild bookmark added successfully.\nFile saved at " + dataDir);
```

هنا، نحدد اسم ملف الإخراج ونحفظ المستند. ستظهر لك رسالة تأكيد بعد اكتمال العملية. الأمر أشبه بإغلاق كتاب بعد كتابة تحفتك الفنية!

## خاتمة

تهانينا! لقد نجحت في إضافة إشارات مرجعية فرعية إلى ملف PDF باستخدام Aspose.PDF لـ .NET. هذه الميزة البسيطة والفعّالة تُحسّن بشكل كبير من سهولة استخدام مستنداتك، مما يُسهّل على القراء تصفحها. سواء كنت تُنشئ تقارير أو كتبًا إلكترونية أو أي مستندات PDF أخرى، فإن الإشارات المرجعية تُحدث فرقًا كبيرًا.

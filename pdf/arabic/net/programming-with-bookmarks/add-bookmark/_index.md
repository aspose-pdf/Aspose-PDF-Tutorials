# إضافة إشارة مرجعية في ملف PDF

## مقدمة

هل سبق لك أن وجدت نفسك تتصفح ملف PDF طويلًا، باحثًا عن القسم الذي تحتاجه؟ إذا كان الأمر كذلك، فأنت لست وحدك! قد يكون التنقل بين المستندات الطويلة أمرًا شاقًا للغاية. ولكن ماذا لو أخبرتك أن هناك طريقة لجعل ملفات PDF الخاصة بك أكثر سهولة في الاستخدام؟ أضف الإشارات المرجعية! في هذا البرنامج التعليمي، سنستكشف كيفية إضافة إشارات مرجعية إلى ملف PDF باستخدام Aspose.PDF لـ .NET. تتيح لك هذه المكتبة القوية التعامل مع مستندات PDF بسهولة، مما يجعل حياتك أسهل بكثير. هيا بنا!

## المتطلبات الأساسية

قبل أن نبدأ، هناك بعض الأشياء التي ستحتاج إلى وضعها في مكانها:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. إنه بيئة التطوير المتكاملة الأمثل لتطوير .NET.
2. Aspose.PDF لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك الحصول عليها من [رابط التحميل](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على المتابعة بسلاسة.

## استيراد الحزم

للبدء بإضافة الإشارات المرجعية، ستحتاج إلى استيراد الحزم اللازمة. إليك كيفية القيام بذلك:

### إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع C# جديدًا. اختر تطبيق وحدة التحكم لتسهيل الأمر.

### إضافة مرجع Aspose.PDF

بعد إعداد مشروعك، ستحتاج إلى إضافة مرجع إلى مكتبة Aspose.PDF. يمكنك القيام بذلك باتباع الخطوات التالية:

- النقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- تحديد "إدارة حزم NuGet".
- البحث عن "Aspose.PDF" وتثبيته.

### استيراد مساحات الأسماء المطلوبة

في الجزء العلوي من `Program.cs` الملف، استيراد المساحات الأساسية اللازمة:

```csharp
using System;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا ننتقل إلى الكود الفعلي لإضافة الإشارات المرجعية!

## الخطوة 1: تحديد دليل المستندات

أولاً، عليك تحديد مسار مجلد المستندات. هذا هو المكان الذي سيُحفظ فيه ملف PDF. إليك الطريقة:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

يستبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يتم تخزين ملف PDF الخاص بك فيه.

## الخطوة 2: افتح مستند PDF

بعد ذلك، افتح ملف PDF الذي تريد إضافة إشارات مرجعية إليه. استخدم الكود التالي:

```csharp
Document pdfDocument = new Document(dataDir + "AddBookmark.pdf");
```

يقوم هذا السطر من التعليمات البرمجية بتهيئة ملف جديد `Document` قم بربط الكائن بملف PDF الخاص بك.

## الخطوة 3: إنشاء كائن إشارة مرجعية

الآن حان وقت إنشاء كائن إشارة مرجعية. هنا يمكنك تحديد عنوان ومظهر الإشارة المرجعية. إليك كيفية القيام بذلك:

```csharp
OutlineItemCollection pdfOutline = new OutlineItemCollection(pdfDocument.Outlines);
pdfOutline.Title = "Test Outline";
pdfOutline.Italic = true;
pdfOutline.Bold = true;
```

في هذا المثال، سننشئ علامة مرجعية بعنوان "مخطط اختبار" ونجعلها غامقة ومائلة. يمكنك تخصيص العنوان كما يحلو لك!

## الخطوة 4: تعيين رقم صفحة الوجهة

لكل إشارة مرجعية وجهة. يمكنك تحديد رقم الصفحة التي سترتبط بها الإشارة المرجعية باستخدام الكود التالي:

```csharp
pdfOutline.Action = new GoToAction(pdfDocument.Pages[1]);
```

يُحدد هذا السطر كيفية انتقال الإشارة المرجعية إلى الصفحة الأولى من ملف PDF. يمكنك تغيير رقم الصفحة حسب الحاجة.

## الخطوة 5: إضافة الإشارة المرجعية إلى المستند

الآن بعد أن قمت بإنشاء الإشارة المرجعية الخاصة بك، حان الوقت لإضافتها إلى مجموعة مخططات المستند:

```csharp
pdfDocument.Outlines.Add(pdfOutline);
```

يضيف هذا السطر الإشارة المرجعية التي قمت بإنشائها حديثًا إلى مستند PDF.

## الخطوة 6: حفظ الناتج

أخيرًا، ستحتاج إلى حفظ مستند PDF المُعدَّل. إليك كيفية القيام بذلك:

```csharp
dataDir = dataDir + "AddBookmark_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nBookmark added successfully.\nFile saved at " + dataDir);
```

يحفظ هذا الكود ملف PDF مع الإشارة المرجعية المضافة باسم "AddBookmark_out.pdf" في الدليل المحدد.

## خاتمة

وها قد انتهيت! لقد نجحت في إضافة إشارة مرجعية إلى ملف PDF باستخدام Aspose.PDF لـ .NET. هذه الميزة البسيطة والفعّالة تُحسّن بشكل كبير من سهولة استخدام مستنداتك، مما يُسهّل على القراء تصفحها. لذا، في المرة القادمة التي تعمل فيها على ملفات PDF، تذكّر إضافة هذه الإشارات المرجعية!

# إعداد اللغة والعنوان

## مقدمة

يُعد إنشاء ملفات PDF المُعلَّمة نشاطًا بالغ الأهمية لضمان سهولة الوصول وتوفير تنسيق منظم للمستندات. مع توجهنا نحو بيئة رقمية أكثر شمولًا، يزداد فهم كيفية إنشاء مستندات مُعلَّمة أهميةً. سيرشدك هذا الدليل الشامل خلال عملية إعداد اللغة والعناوين في ملفات PDF المُعلَّمة باستخدام Aspose.PDF لـ .NET. سنُقسِّمها إلى خطوات سهلة الفهم، حتى لو كنت مبتدئًا، ستشعر بالاحترافية في النهاية. 

## المتطلبات الأساسية

قبل الخوض في عالم ملفات PDF المُعلَّمة، لنجمع كل ما تحتاجه. إليك ما يجب أن يكون جاهزًا:

- المعرفة الأساسية بـ .NET: على الرغم من أنك لست بحاجة إلى أن تكون مبرمجًا خارقًا، فإن الإلمام بمفاهيم .NET سيجعل هذه الرحلة أكثر سلاسة.
- تثبيت Aspose.PDF لـ .NET: تأكد من تثبيت المكتبة. يمكنك تنزيلها للتقييم أو شراء ترخيص. تحقق من [صفحة التحميل هنا](https://releases.aspose.com/pdf/net/).
- Visual Studio: هنا ستكتب وتختبر شفرتك البرمجية. إذا لم تكن لديك، يمكنك تنزيله من موقع Microsoft.
- إجادة لغة C#: هذا الدليل مكتوب بلغة C#. ستساعدك خبرة بسيطة في C# على فهم البرمجة بسهولة.

## استيراد الحزم

بعد إعداد المتطلبات الأساسية، حان وقت استيراد الحزم اللازمة. يمكنك القيام بذلك بإضافة الأمر التالي في أعلى ملف C#:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

تتيح لك هذه المساحات الوصول إلى المكونات اللازمة لإنشاء ملفات PDF ذات محتوى مُعلَّم ومعالجتها. قد تتساءل: "لماذا استيراد الحزم؟" الأمر أشبه بتجهيز صندوق أدوات قبل بناء أي شيء - فأنت بحاجة إلى الأدوات المناسبة.

## الخطوة 1: تهيئة المستند

الخطوة الأولى في رحلتنا هي إنشاء مستند جديد. يمكنك اعتبار هذا بمثابة وضع أساس منزل، حيث سيُبنى كل شيء عليه.

```csharp
Document document = new Document();
```

هنا، نقوم بإنشاء مستند PDF جديد. سيحتوي هذا المستند على جميع محتوياتك. 

## الخطوة 2: تحديد دليل المستندات

الخطوة التالية هي تحديد مكان تخزين مستنداتك. ستحتاج إلى مكان لحفظ ملف PDF الذي أنشأته حديثًا.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

تأكد من الاستبدال `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي تريد حفظ ملف PDF فيه. هذا أشبه بالبحث عن موقف لسيارتك الجديدة.

## الخطوة 3: الحصول على المحتوى المميز

الآن، لننتقل إلى المحتوى المُعلَّم في مستندنا. يُعدّ المحتوى المُعلَّم أساسًا لإنشاء ملفات PDF سهلة الوصول. 

```csharp
Tagged.ITaggedContent taggedContent = document.TaggedContent;
```

من خلال القيام بذلك، يمكنك تمكين إمكانية هيكلة ملف PDF الخاص بك، تمامًا مثل إنشاء مخطط لكتاب قبل كتابته فعليًا.

## الخطوة 4: تعيين العنوان واللغة

بعد أن أصبحت محتوياتك المميزة جاهزة، حان الوقت لتحديد عنوان المستند واللغة الأساسية. 

```csharp
taggedContent.SetTitle("Example Tagged Document");
taggedContent.SetLanguage("en-US");
```

اعتبر هذه الخطوة بمثابة منح مستندك هوية. يُمثل العنوان جوهر ما تدور حوله الوثيقة، بينما تنقل اللغة للقراء السياق اللغوي الأساسي.

## الخطوة 5: إنشاء عنصر الرأس

غالبًا ما يتضمن ملف PDF المُهيكل عناوين رئيسية لمساعدة القارئ على فهمه. لنُنشئ عنصر عنوان رئيسي.

```csharp
LogicalStructure.HeaderElement h1 = taggedContent.CreateHeaderElement(1);
h1.SetText("Phrase on different languages");
taggedContent.RootElement.AppendChild(h1);
```

هنا، أنشأنا رأسًا (H1) بنص. الأمر أشبه بوضع لافتة تُرشد القراء إلى ما سيقرؤونه لاحقًا. 

## الخطوة 6: إضافة فقرات بلغات متعددة

هنا يبدأ الجزء الممتع: إضافة محتوى بلغات مختلفة. سنضيف بعض الفقرات لتمثيل لغات مختلفة.

### إضافة فقرة باللغة الإنجليزية

لنبدأ باللغة الإنجليزية:

```csharp
LogicalStructure.ParagraphElement pEN = taggedContent.CreateParagraphElement();
pEN.SetText("Hello, World!");
pEN.Language = "en-US";
taggedContent.RootElement.AppendChild(pEN);
```

يُضيف هذا السطر تحيةً وديةً باللغة الإنجليزية. إنه أشبه بتحيةٍ لقرائك بلغتهم المفضلة.

### إضافة فقرة ألمانية

والآن دعونا نضيف فقرة باللغة الألمانية:

```csharp
LogicalStructure.ParagraphElement pDE = taggedContent.CreateParagraphElement();
pDE.SetText("Hallo Welt!");
pDE.Language = "de-DE";
taggedContent.RootElement.AppendChild(pDE);
```

من خلال هذا، يمكنك الوصول إلى جمهورك الناطق باللغة الألمانية، وتوسيع إمكانية الوصول إلى مستندك.

### إضافة فقرة فرنسية

وبالمثل بالنسبة للغة الفرنسية:

```csharp
LogicalStructure.ParagraphElement pFR = taggedContent.CreateParagraphElement();
pFR.SetText("Bonjour le monde!");
pFR.Language = "fr-FR";
taggedContent.RootElement.AppendChild(pFR);
```

مرة أخرى، نحتضن التنوع من خلال تضمين النص الفرنسي. 

### إضافة فقرة باللغة الإسبانية

وأخيرًا، دعونا نتحدث قليلاً باللغة الإسبانية:

```csharp
LogicalStructure.ParagraphElement pSP = taggedContent.CreateParagraphElement();
pSP.SetText("¡Hola Mundo!");
pSP.Language = "es-ES";
taggedContent.RootElement.AppendChild(pSP);
```

من خلال هذه الإضافة، يمكنك إظهار أن مستندك يتحدث لغات متعددة، مما يعزز الشمولية.

## الخطوة 7: حفظ مستند PDF المُعَلَّم

الآن بعد أن قمت ببناء مستندك باللغات المتعددة، فقد حان الوقت لحفظه. 

```csharp
document.Save(dataDir + "SetupLanguageAndTitle.pdf");
```

وهكذا، يُنجز إبداعك ويُحفظ! اعتبر هذا بمثابة إغلاق الظرف قبل إرسال رسالتك.

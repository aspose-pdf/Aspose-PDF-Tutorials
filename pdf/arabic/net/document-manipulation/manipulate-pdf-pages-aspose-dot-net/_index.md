## دليل التنفيذ

### فتح وربط مستند PDF

للتعامل مع ملف PDF، عليك أولاً فتحه. تتضمن هذه الخطوة ربط المستند باستخدام `PdfPageEditor` فصل.

#### الخطوة 1: إعداد مسار الملف
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY/input.pdf";
```

#### الخطوة 2: ربط ملف PDF
```csharp
pageEditor.BindPdf(dataDir);
```
يقوم هذا الكود بربط مستند PDF المحدد الخاص بك، مما يجعله جاهزًا للتعامل معه. `BindPdf` تقوم الطريقة بتحميل الملف إلى الذاكرة.

### نقل أصل الصفحة

يمكن أن يكون تعديل نقطة الأصل لصفحة ما أمرًا بالغ الأهمية لأغراض تصميم التخطيط والعرض.

#### الخطوة 1: إنشاء PdfPageEditor
على افتراض أنك قمت بالفعل بربط مستند كما هو موضح مسبقًا.

#### الخطوة 2: تعيين موضع الأصل الجديد
```csharp
pageEditor.MovePosition(100, 100);
```
يؤدي هذا إلى تحويل الأصل من (0، 0) إلى (100، 100)، مما يؤثر على كيفية وضع المحتوى في كل صفحة.

### تعيين تدوير الصفحات

يمكن أن تساعد الصفحات الدائرية في محاذاة اتجاه المستند أو لأغراض عرض محددة.

#### الخطوة 1: إعداد جدول التجزئة الدوراني
```csharp
Hashtable pageRotations = new Hashtable();
pageRotations.Add(1, 90); // تدوير الصفحة الأولى بمقدار 90 درجة
pageRotations.Add(2, 180); // تدوير الصفحة الثانية بمقدار 180 درجة
pageRotations.Add(3, 270); // تدوير الصفحة الثالثة بمقدار 270 درجة
```

#### الخطوة 2: تعيين التدويرات إلى محرر الصفحات
```csharp
pageEditor.PageRotations = pageRotations;
```
يقوم مقتطف التعليمات البرمجية هذا بتعيين زوايا الدوران لصفحات محددة باستخدام `Hashtable`.

### ضبط مستوى التكبير

يعد ضبط مستوى التكبير مفيدًا لإعداد المستندات التي تحتاج إلى تغيير حجمها.

#### الخطوة 1: تحديد عامل التكبير
```csharp
pageEditor.Zoom = 2.0f; // تعيين تكبير بنسبة 200٪
```

تعمل هذه الطريقة على تعديل عامل التكبير/التصغير للمستند بأكمله بشكل مباشر، مما يعزز إمكانية القراءة أو جودة العرض.

### حفظ ملف PDF المحدث

بمجرد اكتمال كافة التعديلات، يصبح حفظ ملف PDF المحدث أمرًا سهلاً.

#### الخطوة 1: تحديد مسار الإخراج
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY/SetPageProperties_out.pdf";
```

#### الخطوة 2: حفظ المستند
```csharp
pageEditor.Save(outputDir);
```
يكتب هذا الأمر جميع التغييرات في ملف جديد، مع الحفاظ على المستند الأصلي.

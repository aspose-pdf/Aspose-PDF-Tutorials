## دليل التنفيذ

بعد إعداد بيئتنا، دعنا ننفذ ميزة النص المخفي خطوة بخطوة.

### إضافة نص مخفي إلى ملف PDF

#### ملخص
سنبدأ بإنشاء مستند PDF وإضافة أجزاء نصية مرئية وغير مرئية. هذه الوظيفة أساسية للتحكم في رؤية المحتوى مع ضمان إمكانية البحث في جميع البيانات.

#### التنفيذ خطوة بخطوة
**إنشاء مستند جديد**
ابدأ بتهيئة ملف جديد `Document` هدف:
```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
Page page = doc.Pages.Add();
```

**إضافة أجزاء نصية**
أضف أجزاء نصية مرئية ومخفية إلى ملف PDF. هنا، نضبط `Invisible` ممتلكات `TextFragment` هدف:
```csharp
TextFragment frag1 = new TextFragment("This is common text.");
TextFragment frag2 = new TextFragment("This is invisible text.");

// تعيين خاصية النص - غير مرئية
frag2.TextState.Invisible = true;

page.Paragraphs.Add(frag1);
page.Paragraphs.Add(frag2);

doc.Save(dataDir + "Output.pdf");
```
**توضيح:**
- **جزء نصي**:يمثل كتلة من النص داخل المستند.
- **الممتلكات غير المرئية**:عند ضبطه على `true`، النص مخفي ولكن يظل قابلاً للبحث.

### البحث عن نص في ملف PDF

#### ملخص
الآن بعد أن قمت بإخفاء النص في مستندك، دعنا نرى كيفية البحث من خلاله بكفاءة.

**البحث عن النصوص المخفية والمرئية**
يستخدم `TextFragmentAbsorber` للبحث عن كافة أجزاء النص في صفحة محددة:
```csharp
doc = new Aspose.Pdf.Document(dataDir + "Output.pdf");
TextFragmentAbsorber absorber = new TextFragmentAbsorber();
asorber.Visit(doc.Pages[1]);

foreach (TextFragment fragment in absorber.TextFragments)
{
    Console.WriteLine("Text '{0}' on pos {1} invisibility: {2}", 
        fragment.Text, fragment.Position.ToString(), fragment.TextState.Invisible);
}
```
**توضيح:**
- **ممتص شظايا النص**:يبحث عن أجزاء نصية عبر المستند.
- يتم معالجة كل جزء لتحديد مدى رؤيته وموقعه.

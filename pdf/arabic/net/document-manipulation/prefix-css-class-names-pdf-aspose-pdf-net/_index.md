## دليل التنفيذ

الآن بعد أن قمنا بإعداد بيئتنا، دعنا ننفذ إضافة بادئة لأسماء فئات CSS أثناء تحويل PDF إلى HTML.

### تهيئة خيارات الحفظ وتكوينها

ابدأ بإنشاء مثيل لـ `HtmlSaveOptions` حيث يمكنك تحديد البادئات المخصصة لفئات CSS:

```csharp
using Aspose.Pdf;
using System.IO;

namespace Aspose.Pdf.Examples.CSharp.AsposePDF.DocumentConversion.PDFToHTMLFormat
{
    public class PrefixCSSClassNames
    {
        public static void Run()
        {
            try
            {
                string dataDir = RunExamples.GetDataDir_AsposePdf_DocumentConversion_PDFToHTMLFormat();
                
                Document pdfDocument = new Document(dataDir + "input.pdf");
                string outHtmlFile = dataDir + "PrefixCSSClassNames_out.html";
                
                HtmlSaveOptions saveOptions = new HtmlSaveOptions
                {
                    CssClassNamesPrefix = ".gDV__ .stl_"
                };
                
                pdfDocument.Save(outHtmlFile, saveOptions);
            }
            catch (Exception ex)
            {
                Console.WriteLine(ex.Message);
            }
        }
    }
}
```

#### شرح الخطوات الرئيسية
- **بادئة أسماء فئات Css**:تتيح لك هذه المعلمة تعيين بادئة مخصصة لأسماء فئات CSS. بتعيينها إلى `".gDV__ .stl_"`، ستبدأ جميع فئات CSS في HTML المُحوّل بهذه البادئة، مما يساعد على تجنب تعارضات التسمية.

## دليل التنفيذ
سنشرح عملية إضافة ملفات PDF بخطوات واضحة.

### إنشاء كائن محرر ملفات Pdf
ال `PdfFileEditor` تُعد الفئة أساسية لعمليات الإلحاق، حيث توفر طرقًا لدمج ملفات PDF المتعددة بسهولة.

#### ملخص
تتيح لك هذه الميزة دمج ملفين PDF أو أكثر عن طريق إنشاء مثيل من `PdfFileEditor` الصف واستخدام أساليبه.

#### خطوات التنفيذ
**الخطوة 1: إعداد مسارات الملفات**
قم بتحديد المسارات لملفات PDF المدخلة والمخرجة لضمان توجيه جميع عمليات الملف بشكل صحيح:
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Pages();
```

**الخطوة 2: إنشاء مثيل PdfFileEditor**
تهيئة `PdfFileEditor` كائن لعملية الإلحاق.
```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**الخطوة 3: إضافة ملفات PDF**
استخدم `Append` طريقة لدمج ملفات PDF في ملف إخراج واحد، مع تحديد الصفحات التي يجب تضمينها:
```csharp
pdfEditor.Append(dataDir + "input.pdf", dataDir + "input2.pdf", 1, 1, dataDir + "AppendFiles_out.pdf");
```

**شرح المعلمات:**
- **dataDir + "input.pdf"**:المسار إلى ملف PDF الأول.
- **dataDir + "input2.pdf"**:المسار إلى ملف PDF الثاني.
- **1, 1**فهرس الصفحات يُحدد الصفحات المراد إضافتها. هنا، تُضاف الصفحة الأولى فقط من كل ملف إدخال.
- **dataDir + "AppendFiles_out.pdf"**:المسار لملف الإخراج المدمج.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن جميع مسارات الملفات صحيحة ويمكن الوصول إليها.
- تأكد من أن ملفات PDF المدخلة ليست تالفة أو مقفلة بأذونات تمنع التحرير.

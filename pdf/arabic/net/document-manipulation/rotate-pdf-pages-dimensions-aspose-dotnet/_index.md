### دليل التنفيذ
في هذا القسم، سنتناول كيفية تدوير صفحات PDF واسترجاع أبعادها باستخدام Aspose.PDF في C#.

#### إضافة صفحة فارغة واسترجاع الأبعاد
**ملخص:**
لنبدأ بفتح مستند PDF موجود، وإضافة صفحة فارغة إذا لزم الأمر، ثم استرجاع الأبعاد الحالية للصفحة.

1. **فتح المستند**

   ```csharp
   string dataDir = RunExamples.GetDataDir_AsposePdf_Pages();
   Document pdfDocument = new Document(dataDir + "UpdateDimensions.pdf");
   ```

2. **إضافة صفحة فارغة (إذا لزم الأمر)**

   ```csharp
   // إضافة صفحة فارغة إذا لم يكن المستند يحتوي على واحدة
   Page page = pdfDocument.Pages.Count > 0 ? pdfDocument.Pages[1] : pdfDocument.Pages.Add();
   ```

3. **استرداد الأبعاد**

   ```csharp
   Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
   ```

#### تدوير الصفحة
**ملخص:**
قم بتدوير صفحة PDF بمقدار 90 درجة وتحقق من كيفية تغير الأبعاد بسبب الدوران.

1. **تدوير الصفحة**

   ```csharp
   // تدوير الصفحة بزاوية 90 درجة
   page.Rotate = Rotation.on90;
   ```

2. **استرداد أبعاد جديدة**

   ```csharp
   Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
   ```

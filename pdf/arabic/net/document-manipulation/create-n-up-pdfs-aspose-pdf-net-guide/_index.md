## دليل التنفيذ

### إنشاء ملف PDF N-Up

**ملخص:**
دمج ملفات PDF متعددة في مستند واحد باستخدام تخطيط N-Up يُتيح استغلالًا فعالًا للمساحة. يُرشدك هذا القسم خطوة بخطوة لإنشاء ملف PDF N-Up.

#### الخطوة 1: جهّز بيئتك
أولاً، قم بإعداد مشروعك وتضمين المساحات الأساسية اللازمة:

```csharp
using System.IO;
using Aspose.Pdf.Facades;
```

#### الخطوة 2: تهيئة محرر ملفات Pdf
إنشاء `PdfFileEditor` كائن لمعالجة عملية الدمج. توفر هذه الفئة طرقًا لدمج ملفات PDF.

```csharp
// إنشاء مثيل PdfFileEditor
class PdfMerger {
    private PdfFileEditor pdfEditor;

    public PdfMerger() {
        pdfEditor = new PdfFileEditor();
    }
}
```

#### الخطوة 3: إعداد تدفقات الملفات

#### الخطوة 4: تنفيذ طريقة MakeNUp
استدعاء `MakeNUp` طريقة لدمج ملفات PDF. تُرتّب هذه الطريقة الصفحات بتنسيق مُحدّد:


**توضيح:**
- `fileStreams` يحتوي على مسارات لملفات الإدخال.
- `outputStream` يحدد المكان الذي سيتم حفظ الملف المدمج فيه.
- ال `true` تشير المعلمة إلى أنك تريد الحفاظ على الإشارات المرجعية.

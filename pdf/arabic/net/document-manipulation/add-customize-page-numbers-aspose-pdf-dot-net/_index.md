## دليل التنفيذ

يغطي هذا القسم إضافة أرقام الصفحات وتخصيصها باستخدام Aspose.PDF لـ .NET.

### إضافة رقم الصفحة إلى مستند PDF (H2)

أضف أرقام الصفحات في أسفل كل صفحة بالتنسيق "الصفحة X من Y".

#### ملخص
سوف نستخدم `PdfFileStamp` لختم أرقام الصفحات على مستندك. 

**الخطوة 1: تهيئة PdfFileStamp**
```csharp
using Aspose.Pdf.Facades;

// إنشاء كائن PdfFileStamp جديد
fileStamp = new PdfFileStamp();
fileStamp.BindPdf("YOUR_DOCUMENT_DIRECTORY\AddPageNumber.pdf");
```

**الخطوة 2: الحصول على إجمالي عدد الصفحات**
استرداد العدد الإجمالي للصفحات لتنسيق أرقام الصفحات بشكل صحيح.
```csharp
int totalPages = new PdfFileInfo("YOUR_DOCUMENT_DIRECTORY\AddPageNumber.pdf").NumberOfPages;
```

**الخطوة 3: تنسيق وإضافة أرقام الصفحات**
قم بتخصيص مظهر أرقام صفحاتك عن طريق تعيين لونها ونمط الخط وموضعها.
```csharp
using System.Drawing;
using System.Text;

FormattedText formattedText = new FormattedText(
    "Page # Of " + totalPages,
    Color.Blue,
    Color.Gray,
    FontStyle.Courier,
    EncodingType.Winansi,
    false, 14
);

fileStamp.StartingNumber = 1;
fileStamp.AddPageNumber(formattedText, 0); // الموضع في أسفل المركز

// حفظ وإغلاق كائن PdfFileStamp
testDocument.Save("YOUR_OUTPUT_DIRECTORY\AddPageNumber_out.pdf");
testDocument.Close();
```

### تخصيص نمط ترقيم الصفحات في مستند PDF (H2)

قم بتغيير نمط ترقيم الصفحات، مثل استخدام الأرقام الرومانية.

#### ملخص
يمكنك تعديل نمط الترقيم بسهولة باستخدام `PdfFileStamp`.

**الخطوة 1: تهيئة PdfFileStamp**
قم بإعادة التهيئة إذا لزم الأمر أو استمر من الاستخدام السابق.
```csharp
// بافتراض أن fileStamp تم تهيئة ملف بالفعل وربطه بمستند PDF
```

**الخطوة 2: تعيين نمط الترقيم**
اختر الأرقام الرومانية بأحرف كبيرة في هذا المثال.
```csharp
fileStamp.NumberingStyle = NumberingStyle.NumeralsRomanUppercase;
```

**الخطوة 3: إضافة أرقام الصفحات بتنسيق مخصص**
قم بتطبيق نمط الترقيم المخصص على مستندك.
```csharp
// إضافة أرقام الصفحات بالتنسيق المحدد في أسفل المنتصف
testDocument.AddPageNumber("#");

// حفظ وإغلاق كائن PdfFileStamp
testDocument.Save("YOUR_OUTPUT_DIRECTORY\CustomNumberStyle_out.pdf");
testDocument.Close();
```

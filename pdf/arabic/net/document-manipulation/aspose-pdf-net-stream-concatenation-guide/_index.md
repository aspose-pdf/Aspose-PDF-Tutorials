## دليل التنفيذ

بعد اكتمال الإعداد، دعنا ننفذ عملية تجميع تدفق PDF باستخدام `Aspose.PDF for .NET`.

### إنشاء وتكوين كائن PdfFileEditor

يتضمن جوهر تنفيذنا استخدام `PdfFileEditor` فصل:
```csharp
// إنشاء مثيل لـ PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

#### إعداد تدفقات الإدخال

سنعمل مع التدفقات لقراءة ملفات PDF، مما يسمح بالتعامل المرن مع البيانات:
1. **تحديد المسارات وتهيئة التدفقات:**
    ```csharp
    // حدد الدليل لمستنداتك
    string dataDir = "Path to Your Documents";

    // إنشاء مصفوفة لتخزين تدفقات الإدخال
    FileStream[] inputStreams = new FileStream[2];
    
    // فتح التدفقات لكل ملف PDF تريد ربطه
    inputStreams[0] = new FileStream(dataDir + "input.pdf", FileMode.Open);
    inputStreams[1] = new FileStream(dataDir + "input2.pdf", FileMode.Open);
    ```

#### ربط التدفقات

ال `Concatenate` تجمع الطريقة بكفاءة تدفقات PDF في تدفق واحد:
```csharp
// إنشاء دفق إخراج للملف المتسلسل
using (FileStream outputStream = new FileStream(dataDir + "ConcatenateArrayOfPdfUsingStreams_out.pdf", FileMode.Create))
{
    // تنفيذ التسلسل
    pdfEditor.Concatenate(inputStreams, outputStream);
}
```

### شرح المعلمات والطرق

- **`inputStreams`:** مجموعة من `FileStream` الكائنات التي تحتوي على ملفات PDF المراد دمجها.
- **`outputStream`:** مجرى الوجهة للمستند المتسلسل.

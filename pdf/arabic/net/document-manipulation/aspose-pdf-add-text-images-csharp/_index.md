## دليل التنفيذ
دعنا نستكشف كيفية تنفيذ كل ميزة باستخدام Aspose.PDF لـ .NET.

### إنشاء جزء نصي
#### ملخص
إنشاء وإضافة جزء نصي إلى ملف PDF أمر سهل. يرشدك هذا القسم إلى كيفية إنشاء "مرحبًا بالعالم" كمثال.
**الخطوة 1: إنشاء مثيل مستند**
```csharp
Document doc = new Document();
```
*لماذا؟*:يتم تهيئة مستند PDF جديد للتعامل معه.

**الخطوة 2: إضافة صفحة**
```csharp
Page page = doc.Pages.Add();
```
*لماذا؟*:يضيف صفحة فارغة إلى مستندك حيث سيتم وضع جزء النص.

**الخطوة 3: إنشاء TextFragment وإضافته**
```csharp
TextFragment text = new TextFragment("Hello World.. ");
page.Paragraphs.Add(text);
```
*لماذا؟*:تعمل هذه القطعة على إنشاء كائن نصي وإضافته إلى مجموعة الفقرات الخاصة بالصفحة.

**الخطوة 4: حفظ المستند**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(Path.Combine(outputDir, "TextFragment_out.pdf"));
```
*لماذا؟*:يحفظ تغييراتك كملف PDF في الدليل المحدد.

### إضافة صورة مضمنة
#### ملخص
إضافة صور مضمنة مع النص تُحسّن من مظهر ملف PDF. إليك كيفية تضمين صورة مباشرةً بعد النص المُضاف سابقًا.
**الخطوة 1: إنشاء مثيل صورة**
```csharp
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
image.IsInLineParagraph = true;
```
*لماذا؟*:يقوم بتكوين الصورة ليتم التعامل معها باعتبارها مضمنة، مما يسمح لها بالتدفق داخل محتوى النص بسلاسة.

**الخطوة 2: تعيين خصائص الصورة**
```csharp
string imagePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
image.File = imagePath;
image.FixHeight = 30;
image.FixWidth = 100;
```
*لماذا؟*:يحدد مسار الصورة وأبعادها، مما يضمن ملاءمتها بشكل جيد لتخطيط المستند الخاص بك.

**الخطوة 3: إضافة صورة إلى الصفحة**
```csharp
page.Paragraphs.Add(image);
```
*لماذا؟*:يضيف الصورة المُهيأة كعنصر فقرة في مجموعة الصفحة.

**الخطوة 4: حفظ المستند**
```csharp
doc.Save(Path.Combine(outputDir, "InlineImage_out.pdf"));
```
*لماذا؟*:يحفظ المستند المحدث مع الصورة المضمنة المضمنة.

### إضافة جزء نصي مضمن بعد الصورة
#### ملخص
استمرارًا من حيث توقفنا، دعنا نضيف جزءًا آخر من النص بعد صورتنا لإنشاء تدفق سلس للمحتوى.
**الخطوة 1: إعادة تهيئة TextFragment**
```csharp
TextFragment text = new TextFragment(" Hello Again..");
text.IsInLineParagraph = true;
```
*لماذا؟*:يقوم بإعداد كتلة النص الجديدة للإضافة المضمنة، والتي تأتي مباشرة بعد الصورة.

**الخطوة 2: إضافة جزء نصي جديد إلى الصفحة**
```csharp
page.Paragraphs.Add(text);
```
*لماذا؟*:يضع جزء النص في خط واحد مع عناصر المحتوى الأخرى في الصفحة.

**الخطوة 3: حفظ المستند**
```csharp
doc.Save(Path.Combine(outputDir, "InlineTextAfterImage_out.pdf"));
```
*لماذا؟*:يقوم بإنهاء وحفظ مستندك الذي يحتوي على كل من الصور وأجزاء النص الإضافية.

## دليل التنفيذ
دعونا نقوم بتقسيم تنفيذ إضافة ختم النص إلى خطوات واضحة.

### إضافة طوابع نصية إلى ملفات PDF (H2)
تتيح لك هذه الميزة إدراج نص مخصص على أي صفحة في مستند PDF الخاص بك، وهي مفيدة لوضع علامة تجارية أو وضع علامة على المستندات باستخدام معرفات محددة مثل أرقام الإصدار أو التواريخ.

#### الخطوة 1: فتح مستند PDF موجود
ابدأ بتحميل مستند PDF حيث تريد إضافة طابع:

```csharp
// تحميل مستند PDF
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY\\AddTextStamp.pdf");
```

#### الخطوة 2: إنشاء وتكوين ختم النص
إنشاء `TextStamp` الكائن وتكوينه باستخدام محتوى النص المطلوب، والموضع، والتصميم، والخصائص الأخرى.

```csharp
// إنشاء طابع نصي
TextStamp textStamp = new TextStamp("Sample Stamp");

// اضبط الخلفية على "صحيح" إذا كنت تريد الختم خلف النص الموجود
textStamp.Background = true;

// وضع الختم
textStamp.XIndent = 100;
textStamp.YIndent = 100;

// قم بتدوير الختم بمقدار 90 درجة للحصول على مظهر ديناميكي
textStamp.Rotate = Rotation.on90;

// تكوين إعدادات الخط
textStamp.TextState.Font = FontRepository.FindFont("Arial");
textStamp.TextState.FontSize = 14.0F;
textStamp.TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;
textStamp.TextState.ForegroundColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Aqua);
```

#### الخطوة 3: إضافة الختم إلى الصفحة
بعد ذلك، قم بإضافة ختم النص إلى الصفحة الأولى من مستند PDF الخاص بك.

```csharp
// إضافة طابع إلى الصفحة الأولى
pdfDocument.Pages[1].AddStamp(textStamp);
```

#### الخطوة 4: حفظ المستند المعدّل
وأخيرًا، احفظ المستند المحدث مع إضافة الطابع الجديد:

```csharp
// تحديد مسار الإخراج وحفظ المستند
string outputDir = "YOUR_OUTPUT_DIRECTORY\\AddTextStamp_out.pdf";
pdfDocument.Save(outputDir);
```

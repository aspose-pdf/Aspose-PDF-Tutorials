## دليل التنفيذ

### حذف صفحات من مستند PDF
تتيح لك هذه الميزة إزالة صفحات محددة من مستند PDF بكفاءة. اتبع الخطوات التالية لتطبيق هذه الميزة:

#### 1. إنشاء كائن محرر ملفات Pdf
```csharp
using Aspose.Pdf.Facades;

// إنشاء كائن PdfFileEditor
class PdfPageDeleter {
    static void Main() {
        using (PdfFileEditor pdfEditor = new PdfFileEditor()) {
            // مجموعة من أرقام الصفحات المراد حذفها (فهرس يعتمد على 1)
            int[] pagesToDelete = new int[] { 1, 2 };

            // مسارات لملفات الإدخال والإخراج
            string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
            string outputFile = "YOUR_OUTPUT_DIRECTORY/DeletePagesUsingFilePath_out.pdf";

            // تنفيذ حذف الصفحة
            pdfEditor.Delete(inputFile, pagesToDelete, outputFile);
        }
    }
}
```
يقوم هذا الكود بتهيئة مثيل لـ `PdfFileEditor`، الذي يوفر طرقًا لتحرير ملفات PDF.

#### 2. حدد الصفحات التي تريد حذفها
قم بتحديد الصفحات التي تريد إزالتها باستخدام مصفوفة عددية صحيحة:
```csharp
// مجموعة من أرقام الصفحات المراد حذفها (فهرس يعتمد على 1)
int[] pagesToDelete = new int[] { 1, 2 };
```
هنا نحدد أننا نريد حذف الصفحتين الأولى والثانية.

#### 3. حذف الصفحات من ملف PDF
استخدم `Delete` الطريقة لإزالة الصفحات المحددة:
```csharp
// مسارات لملفات الإدخال والإخراج
class PdfPageDeleter {
    static void Main() {
        using (PdfFileEditor pdfEditor = new PdfFileEditor()) {
            string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
            string outputFile = "YOUR_OUTPUT_DIRECTORY/DeletePagesUsingFilePath_out.pdf";

            // تنفيذ حذف الصفحة
            pdfEditor.Delete(inputFile, pagesToDelete, outputFile);
        }
    }
}
```
يقوم هذا الكود بحذف الصفحات المحددة من `input.pdf` ويحفظ النتيجة في `output.pdf`.

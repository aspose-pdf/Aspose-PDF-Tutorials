## دليل التنفيذ
### نسخ حقول PDF باستخدام Aspose.PDF
#### ملخص
يوضح هذا القسم كيفية تكرار حقل داخل مستند PDF، وهو أمر مفيد للحفاظ على الاتساق عبر صفحات متعددة.

#### التنفيذ خطوة بخطوة
**1. تهيئة محرر النماذج**
إنشاء مثيل لـ `FormEditor` وربطه بملف PDF المستهدف:
```csharp
string dataDir = "your-data-directory-path";
FormEditor formEditor = new FormEditor();
formEditor.BindPdf(dataDir + "CopyInnerField.pdf");
```
**2. نسخ الحقل من موقع واحد**
يستخدم `CopyInnerField` لتكرار حقل، مع تحديد الحقول المصدر والوجهة بالإضافة إلى رقم الصفحة المستهدفة:
```csharp
// انسخ 'textfield' إلى 'newfieldname' في الصفحة 1
formEditor.CopyInnerField("textfield", "newfieldname", 1);
```
**3. احفظ مستند الإخراج**
احفظ التغييرات في ملف PDF جديد بعد نسخ الحقل:
```csharp
formEditor.Save(dataDir + "CopyInnerField_out.pdf");
```

## دليل التنفيذ
استكشف ميزتين أساسيتين: إضافة JavaScript إلى ملفات PDF وإزالتها.

### إضافة وظائف JavaScript إلى مستند PDF
إضافة JavaScript تُحوّل ملفات PDF الثابتة إلى أدوات ديناميكية. إليك كيفية تطبيق هذه الميزة:

#### ملخص
يوضح هذا القسم تضمين وظائف JavaScript داخل مستند PDF الخاص بك باستخدام Aspose.PDF لـ .NET.

#### التنفيذ خطوة بخطوة
**1. إنشاء مستند PDF وإعداده**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// تهيئة مستند جديد.
Document doc = new Document();
doc.Pages.Add();  // أضف صفحة إلى المستند.
```

**2. إضافة وظائف JavaScript**
هنا، سنضيف وظيفتين بسيطتين باسم `func1` و `func2`.
```csharp
// تضمين وظائف JavaScript في مجموعة النصوص البرمجية الخاصة بملف PDF.
doc.JavaScript["func1"] = "function func1() { hello(); }";
doc.JavaScript["func2"] = "function func2() { hello(); }";

// احفظ المستند باستخدام البرامج النصية المضمنة.
doc.Save(dataDir + "/AddJavascript.pdf");
```
*توضيح*:نحن نستخدم `doc.JavaScript` لإضافة وظائف. كل وظيفة مرتبطة بمفتاح فريد، مما يسهل الوصول إليها والتعامل معها.

**نصائح لاستكشاف الأخطاء وإصلاحها**:تأكد من أن كود JavaScript الخاص بك صحيح نحويًا ولا يتعارض مع البرامج النصية الموجودة في ملف PDF.

### إزالة دالة JavaScript من مستند PDF
في بعض الأحيان، قد تحتاج إلى إزالة وظائف JavaScript مُحددة. إليك الطريقة:

#### ملخص
يرشدك هذا القسم خلال عملية إزالة وظائف JavaScript من مستند PDF باستخدام Aspose.PDF لـ .NET.

#### التنفيذ خطوة بخطوة
**1. قم بتحميل ملف PDF الموجود**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// افتح المستند الذي يحتوي على البرامج النصية.
Document doc1 = new Document(dataDir + "/AddJavascript.pdf");
```

**2. عرض وظائف JavaScript**
قبل الإزالة، من المفيد إدراج الوظائف الموجودة.
```csharp
IList keys = (System.Collections.IList)doc1.JavaScript.Keys;

foreach (string key in keys)
{
    // إخراج اسم كل وظيفة والرمز الخاص بها.
    Console.WriteLine($"{key}: {doc1.JavaScript[key]}");
}
```

**3. إزالة الوظيفة المحددة**
هنا نقوم بإزالة `func1` من الوثيقة.
```csharp
// احذف 'func1' بواسطة مفتاحه.
doc1.JavaScript.Remove("func1");

// اختياريًا، يمكنك حفظ ملف PDF المعدّل إذا لزم الأمر.
doc1.Save(dataDir + "/ModifiedJavascript.pdf");
```
*توضيح*:استخدم `Remove` الطريقة مع مفتاح الوظيفة لإزالتها من مجموعة البرامج النصية.

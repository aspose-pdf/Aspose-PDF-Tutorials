## دليل التنفيذ
### تقسيم ملف PDF إلى صفحات منفصلة
تتيح لك هذه الميزة تقسيم كل صفحة من المستند إلى ملف خاص بها، وهو ما قد يكون مفيدًا جدًا لتنظيم الأقسام الفردية ومشاركتها.

#### الخطوة 1: افتح المستند الأصلي
ابدأ بتحميل ملف PDF الخاص بك باستخدام `Document` فصل:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/SplitToPages.pdf");
```
يفتح هذا الكود ملف PDF من دليل محدد، ويجهزه للمعالجة.

#### الخطوة 2: التكرار عبر الصفحات
قم بالتنقل عبر كل صفحة لإنشاء مستندات منفصلة:
```csharp
int pageCount = 1;
foreach (Page pdfPage in pdfDocument.Pages)
{
    // إنشاء مستند جديد لهذه الصفحة.
    Document newDocument = new Document();
    newDocument.Pages.Add(pdfPage);

    // احفظ المستند المكون من صفحة واحدة كملف PDF.
    newDocument.Save($"YOUR_OUTPUT_DIRECTORY/page_{pageCount}_out.pdf");
    pageCount++;
}
```
**توضيح**:تتكرر هذه القطعة على كل صفحة في ملف PDF الخاص بك، مما يؤدي إلى إنشاء صفحة جديدة `Document` كائن لكل واحد وحفظه كملف فردي.

### نصائح استكشاف الأخطاء وإصلاحها

## دليل التنفيذ

### الميزة 1: فتح مستند PDF وربطه

#### ملخص

فتح ملف PDF وربطه هو خطوتك الأولى في أي عملية تعديل. هذا يسمح لك بتحميل ملف PDF إلى الذاكرة لإجراء عمليات أخرى.

**خطوات:**

##### تهيئة محرر النماذج

```csharp
// تهيئة كائن FormEditor
FormEditor formEditor = new FormEditor();
```

يؤدي هذا إلى تهيئة مثيل لـ `FormEditor` الفئة التي ستتولى التعامل مع كافة العمليات على ملف PDF الخاص بك.

##### تعريف دليل المستندات

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
يستبدل `"YOUR_DOCUMENT_DIRECTORY"` مع مسار تخزين ملفات PDF. يضمن هذا الدليل إمكانية تبديل الأدلة بسهولة عند نشر تطبيقك.

##### فتح ملف PDF وربطه

```csharp
// افتح مستند PDF وقم بربطه
formEditor.BindPdf(dataDir + "DeleteFormField.pdf");
```

ال `BindPdf` تقوم هذه الطريقة بتحميل ملف PDF المحدد في الذاكرة، مُهيئًا إياه للتعديل. تأكد من صحة المسار لتجنب أخطاء التشغيل.

### الميزة 2: حذف حقل من مستند PDF

#### ملخص

قد يؤدي حذف الحقول (مثل مربعات النص أو مربعات الاختيار) إلى تبسيط مستنداتك عن طريق إزالة العناصر غير الضرورية.

**خطوات:**

##### فتح ملف PDF وربطه

أولاً، تأكد من أنك قمت بفتح ملف PDF وربطه كما هو موضح في القسم السابق.

```csharp
formEditor.BindPdf(dataDir + "DeleteFormField.pdf");
```

##### إزالة حقل

```csharp
// حذف الحقل المحدد من مستند PDF
formEditor.RemoveField("textfield");
```
ال `RemoveField` تأخذ الطريقة اسم الحقل الذي تريد حذفه. استبدل `"textfield"` مع اسم الحقل المستهدف الفعلي.

### الميزة 3: حفظ مستند PDF معدّل

#### ملخص

بعد إجراء التغييرات، من الضروري حفظ عملك. تضمن هذه الخطوة حفظ جميع التعديلات في ملف جديد.

**خطوات:**

##### تحديد دليل الإخراج

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```
على غرار دليل الإدخال، استبدل `"YOUR_OUTPUT_DIRECTORY"` حيث تريد تخزين الملفات المعدلة.

##### حفظ الملف المحدث

```csharp
// حفظ الملف المحدث في الدليل المحدد
formEditor.Save(outputDir + "DeleteFormField_out.pdf");
```

ال `Save` تكتب الطريقة تغييراتك في ملف PDF جديد في الموقع المحدد، مع الحفاظ على المستند الأصلي.

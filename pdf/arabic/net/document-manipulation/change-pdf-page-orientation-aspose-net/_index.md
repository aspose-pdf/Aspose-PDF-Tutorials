## دليل التنفيذ

في هذا القسم، سنتناول كيفية تحميل صفحات PDF والتنقل خلالها، وضبط أبعاد الصفحة.

### تحميل صفحات PDF والتكرار فيها
تتيح لك هذه الميزة تحميل مستند PDF والتنقل بين صفحاته للوصول إليه أو تعديله.

#### الخطوة 1: تحميل المستند
```csharp
using System.IO;
using Aspose.Pdf;

// قم بتحميل ملف PDF الخاص بك
document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

#### الخطوة 2: التكرار عبر الصفحات
يمكنك التكرار عبر كل صفحة باستخدام `foreach` حلقة:
```csharp
foreach (Page page in doc.Pages)
{
    // الوصول إلى مستطيل MediaBox للصفحة الحالية
    Rectangle r = page.MediaBox;
}
```
ال `MediaBox` تمنحك الخاصية الأبعاد والموضع، وهي ضرورية لضبط الاتجاه.

### ضبط أبعاد الصفحة
يتضمن تغيير الاتجاه تعديل عرض الصفحة وارتفاعها. إليك الطريقة:

#### الخطوة 1: حساب الأبعاد الجديدة
لتبديل الصفحة من الوضع الرأسي إلى الأفقي أو العكس، احسب الأبعاد الجديدة على النحو التالي:
```csharp
foreach (Page page in doc.Pages)
{
    Rectangle r = page.MediaBox;

    // تبديل الارتفاع والعرض لتغيير الاتجاه
    double newWidth = r.Height;
    double newHeight = r.Width;

    // تطبيق الأبعاد الجديدة على MediaBox
    page.MediaBox = new Rectangle(0, 0, newWidth, newHeight);
}
```
**توضيح:**
- `r.Height` يصبح العرض الجديد، و `r.Width` يصبح الارتفاع الجديد للتوجيه الأفقي.

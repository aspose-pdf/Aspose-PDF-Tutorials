## دليل التنفيذ

### الميزة: صفحات PDF متعددة

تعمل هذه الميزة على دمج صفحات متعددة من ملفات PDF مختلفة في عدد أقل من الصفحات باستخدام تنسيق N-Up، وهو مثالي لتكثيف المستندات دون فقدان المعلومات.

#### الخطوة 1: إنشاء كائن محرر ملفات Pdf
```csharp
using Aspose.Pdf.Facades;

// تهيئة كائن PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

#### الخطوة 2: تحديد مسارات الملفات المصدر والإخراج
حدد مسارات ملفات PDF المدخلة وملف الإخراج. استبدل العناصر النائبة بالدلائل الفعلية.
```csharp
string inputFile1 = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
string inputFile2 = "YOUR_DOCUMENT_DIRECTORY/input2.pdf";
string outputFile = "YOUR_OUTPUT_DIRECTORY/MakeNUpUsingPaths_out.pdf";
```

#### الخطوة 3: اصنع N-Up
استخدم `MakeNUp` طريقة لدمج المستندات في ملف إخراج واحد.
```csharp
// دمج "input.pdf" و"input2.pdf" باستخدام تكوين N-Up
pdfEditor.MakeNUp(inputFile1, inputFile2, outputFile);
```

**المعلمات موضحة:**
- `inputFile1`، `inputFile2`:المسارات إلى ملفات PDF التي تريد دمجها.
- `outputFile`:مسار الوجهة لملف PDF المدمج.

### الميزة: إعداد الدليل لأمثلة Aspose.PDF

دليل لإعداد مسارات الدليل، وضمان التخزين المنظم لملفات الإدخال والإخراج.

Console.WriteLine("Using document directory: " + dataDirectory);

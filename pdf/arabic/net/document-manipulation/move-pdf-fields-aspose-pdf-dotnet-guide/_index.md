### نظرة عامة: نقل حقل في مستند PDF

تتيح لك هذه الميزة إعادة وضع أي حقل نموذج، مما يعزز تخصيص التخطيط وتفاعل المستخدم.

#### الخطوة 1: إعداد الدلائل الخاصة بك

حدد المسارات الخاصة بمجلدات الإدخال والإخراج الخاصة بك:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

استبدل العناصر النائبة بالمسارات الفعلية على نظامك.

#### الخطوة 2: تهيئة مثيل محرر النماذج

إنشاء مثيل لـ `FormEditor` لتحرير نماذج PDF:

```csharp
using (FormEditor formEditor = new FormEditor())
{
    // سيتم إضافة الكود هنا في الخطوات اللاحقة.
}
```

#### الخطوة 3: افتح مستند PDF

قم بربط ملف PDF المستهدف الخاص بك بـ `FormEditor` مثال:

```csharp
formEditor.BindPdf(dataDir + "/input.pdf");
```

هنا، `"input.pdf"` هو اسم مستند المصدر الخاص بك.

#### الخطوة 4: نقل الحقل

استخدم `MoveField` طريقة نقل الحقل المسمى `textfield`. حدود:
- **اسم الحقل:** معرف لحقل النموذج الذي تريد نقله.
- **موضع البداية X، موضع البداية Y:** مواضع أفقية ورأسية جديدة (بالنقاط).
- **العرض والارتفاع:** أبعاد مساحة الحقل الجديد.

```csharp
formEditor.MoveField("textfield", 300, 300, 400, 400);
```

#### الخطوة 5: حفظ المستند المحدث

احفظ التغييرات في ملف PDF جديد:

```csharp
formEditor.Save(outputDir + "/MoveFormField_out.pdf");
```

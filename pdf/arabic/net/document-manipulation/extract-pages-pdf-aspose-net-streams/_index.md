## دليل التنفيذ

### استخراج الصفحات باستخدام التدفقات

تتيح لك هذه الميزة استخراج صفحات محددة من ملف PDF باستخدام التدفقات، وهو أمر مثالي للملفات الكبيرة أو حلول التخزين عن بعد.

#### الخطوة 1: إنشاء كائن محرر ملفات Pdf
```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```
ال `PdfFileEditor` توفر الفئة طرقًا للتعامل مع مستندات PDF، بما في ذلك استخراج الصفحات.

#### الخطوة 2: إعداد تدفقات الإدخال والإخراج
افتح مجرى ملف للقراءة وآخر للكتابة:
```csharp
FileStream inputStream = new FileStream("input.pdf", FileMode.Open);
FileStream outputStream = new FileStream("output.pdf", FileMode.Create);
```
تؤدي هذه الخطوة إلى تحديد مصدر مستند PDF الخاص بك ومكان حفظ الصفحات المستخرجة.

#### الخطوة 3: تحديد الصفحات التي سيتم استخراجها
حدد الصفحات التي تريد استخراجها:
```csharp
int[] pagesToExtract = { 1, 2 };
```
قم بتعديل هذه المجموعة لتحديد أي أرقام صفحات تحتاجها.

#### الخطوة 4: إجراء الاستخراج
استخدم `Extract` الطريقة لسحب الصفحات المحددة:
```csharp
pdfEditor.Extract(inputStream, pagesToExtract, outputStream);
```
تقوم هذه الطريقة بالقراءة من مجرى الإدخال وتكتب الصفحات المحددة فقط إلى مجرى الإخراج.

- تحقق مما إذا كانت أرقام الصفحات المحددة موجودة في المستند المصدر.

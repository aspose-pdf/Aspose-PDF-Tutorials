## دليل التنفيذ
في هذا القسم، سنقوم بتفصيل عملية إنشاء وإضافة الرؤوس إلى مستند PDF باستخدام Aspose.PDF.

### إنشاء مستند برأس
#### ملخص
سنبدأ بإعداد مستند PDF بسيط، ثم نضيف رأسًا مخصصًا يحتوي على نص وصورة. تُحسّن هذه الميزة مظهر مستندك وتناسق علامتك التجارية.

**الخطوة 1: إنشاء مثيل للمستند**
```csharp
// إنشاء مثيل جديد لفئة المستند
document = new Document();
```
سيؤدي هذا إلى تهيئة مستند PDF فارغ والذي سنقوم بتعديله عن طريق إضافة الصفحات والرؤوس.

#### الخطوة 2: إضافة صفحة
```csharp
// إضافة صفحة إلى المستند
page = document.Pages.Add();
```
من الضروري إضافة صفحة لأننا سنحتاج إلى مكان لوضع رأس الصفحة.

### إنشاء قسم الرأس
**الخطوة 3: إنشاء وتعيين الرأس**
```csharp
// إنشاء مثيل لفئة HeaderFooter وتعيينها للصفحة
header = new HeaderFooter();
page.Header = header;
```
تتضمن هذه الخطوة إنشاء `HeaderFooter` الكائن الذي سنستخدمه لإضافة عناصر مثل النص والصور.

#### الخطوة 4: إضافة نص إلى الرأس
```csharp
// إنشاء TextFragment بخصائص محددة
TextFragment textFragment1 = new TextFragment("Aspose.Pdf is a robust component by");
textFragment1.TextState.ForegroundColor = Color.Blue; // تعيين لون النص إلى اللون الأزرق
textFragment1.IsInLineParagraph = true;

header.Paragraphs.Add(textFragment1);
```
نضيف `TextFragment` الكائن الذي يحتوي على النص المطلوب ونقوم بضبط لون المقدمة إلى اللون الأزرق.

#### الخطوة 5: إضافة صورة إلى الرأس
```csharp
// إنشاء كائن صورة وتكوينه
Image image = new Image();
image.File = "YOUR_DOCUMENT_DIRECTORY/aspose-logo.jpg"; // المسار إلى ملف صورتك
image.FixWidth = 50;
image.FixHeight = 20;
image.IsInLineParagraph = true;

header.Paragraphs.Add(image);
```
تمت إضافة الصورة بأبعاد ثابتة، مما يضمن ملاءمتها بشكل جيد ضمن العنوان.

#### الخطوة 6: إضافة نص إضافي إلى العنوان
```csharp
// جزء نص آخر بلون مختلف
TextFragment textFragment2 = new TextFragment(" Pty Ltd.");
textFragment2.TextState.ForegroundColor = Color.Maroon; // تعيين لون النص إلى اللون العنابي
textFragment2.IsInLineParagraph = true;

header.Paragraphs.Add(textFragment2);
```
هذه الخطوة تضيف خطوة أخرى `TextFragment` كائن يوضح كيفية مزج العناصر والأنماط المختلفة.

### حفظ المستند
**الخطوة 7: احفظ ملف PDF**
```csharp
// حفظ المستند في المسار المحدد
document.Save("YOUR_OUTPUT_DIRECTORY/ImageAndPageNumberInHeaderFooter_UsingInlineParagraph_out.pdf");
```
وأخيرًا، قم بحفظ مستند PDF المعدّل في الدليل الذي اخترته.

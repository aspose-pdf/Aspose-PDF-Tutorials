# إضافة نص شفاف في ملف PDF

الآن بعد أن قمنا بتغطية المتطلبات الأساسية، دعنا ننتقل مباشرة إلى كيفية إضافة نص شفاف إلى مستند PDF.

## استيراد الحزم

قبل البدء بالترميز، يجب استيراد مساحات الأسماء اللازمة. تتيح لنا هذه المساحات الوصول إلى مكتبة Aspose.PDF، مما يُمكّننا من التعامل مع مستندات PDF.

```csharp
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System;
```

تُعد هذه الاستيرادات ضرورية للتعامل مع صفحات PDF وإضافة الرسومات ومعالجة النصوص في Aspose.PDF لـ .NET.

بعد أن انتهينا من إعداد كل شيء، لنبدأ بشرح عملية إضافة نص شفاف إلى ملف PDF باستخدام Aspose.PDF لـ .NET. ستشرح كل خطوة الكود، مما يضمن فهمك الكامل لوظيفة كل جزء.

## الخطوة 1: إعداد المستند

أول ما علينا فعله هو إنشاء مستند PDF جديد وصفحة لإضافة النص الشفاف. تخيل هذا كإنشاء لوحة فارغة لإضافة تصاميمك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// إنشاء مثيل مستند
Document doc = new Document();
// إنشاء مجموعة صفحات من ملف PDF
Aspose.Pdf.Page page = doc.Pages.Add();
```

هنا، نقوم بتهيئة `Document` كائن يُمثل ملف PDF الخاص بنا. نضيف إليه أيضًا صفحة فارغة. بسيط، أليس كذلك؟

## الخطوة 2: إنشاء رسم بياني وإضافة الأشكال

بعد ذلك، سنقوم بإنشاء `Graph` الكائن الذي سيعمل كحاوية للعناصر الرسومية التي نريد إضافتها إلى ملف PDF، مثل الأشكال أو المستطيلات.

```csharp
// إنشاء كائن الرسم البياني
Aspose.Pdf.Drawing.Graph canvas = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
// إنشاء مثيل مستطيل بأبعاد معينة
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 400, 400);
```

هنا، نقوم بتعريف `Graph` بأبعاد محددة، ثم أضف مستطيلاً. تخيّل هذا المستطيل مكانًا للنص.

## الخطوة 3: ضبط الألوان والشفافية

لإضفاء مظهر شفاف على المستطيل والنص، نحتاج إلى تعديل قناة ألفا للألوان. تتحكم قناة ألفا بشفافية الألوان في الصور الرقمية، فكلما انخفضت القيم، زادت شفافية الكائن.

```csharp
// إنشاء كائن ملون من قناة ألوان ألفا
rect.GraphInfo.FillColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.FromArgb(128, System.Drawing.Color.FromArgb(12957183)));
```

يقوم هذا المقطع بتعديل شفافية المستطيل. `FromArgb` تتيح لك هذه الطريقة التحكم في ألفا (الشفافية) مع قيم ألوان RGB.

## الخطوة 4: إضافة المستطيل إلى الرسم البياني

الآن بعد أن قمنا بإعداد المستطيل، فلنقم بإضافته إلى الرسم البياني حتى يصبح جزءًا من المستند.

```csharp
// إضافة مستطيل إلى مجموعة الأشكال الخاصة بكائن الرسم البياني
canvas.Shapes.Add(rect);
// إضافة كائن الرسم البياني إلى مجموعة فقرات كائن الصفحة
page.Paragraphs.Add(canvas);
```

هنا، يتم إضافة المستطيل إلى `Graph`، والتي تُضاف بعد ذلك إلى الصفحة. تخيل هذا كأنك تضع إطارًا شفافًا على صورة.

## الخطوة 5: إنشاء نص شفاف

الآن يأتي الجزء الممتع! لنُنشئ نصًا شفافًا ونُضيفه إلى المستند. هنا سيحصل ملف PDF على نص أنيق يشبه العلامة المائية.

```csharp
// إنشاء مثيل TextFragment بقيمة العينة
TextFragment text = new TextFragment("transparent text transparent text transparent text...");
```

نحن نستخدم `TextFragment` لتحديد النص الذي نريد عرضه. يمكنك استبدال النص البديل بأي نص آخر تريده.

## الخطوة 6: ضبط شفافية النص

ولجعل النص شفافًا، نستخدم قناة ألفا مرة أخرى.

```csharp
// إنشاء كائن ملون من قناة ألفا
Aspose.Pdf.Color color = Aspose.Pdf.Color.FromArgb(30, 0, 255, 0);
// تعيين معلومات اللون لنسخة النص
text.TextState.ForegroundColor = color;
```

هنا، `FromArgb` هذه الطريقة تُعطي النص لونًا أخضرًا شفافًا. يمكنك تخصيص اللون ليناسب تفضيلاتك.

## الخطوة 7: إضافة نص شفاف إلى ملف PDF

وأخيرًا، نضيف النص الشفاف إلى صفحة PDF الخاصة بنا.

```csharp
// إضافة نص إلى مجموعة فقرات من مثيلات الصفحة
page.Paragraphs.Add(text);
```

يضيف هذا الكود النص الشفاف إلى الصفحة `Paragraphs` المجموعة، مما يجعلها مرئية في ملف PDF.

## الخطوة 8: حفظ ملف PDF

الآن بعد أن أصبح كل شيء في مكانه، حان الوقت لحفظ مستند PDF.

```csharp
dataDir = dataDir + "AddTransparentText_out.pdf";
doc.Save(dataDir);
```

يحفظ هذا الكود المستند باسم ملف مخصص. تحقق من مجلد الإخراج لعرض ملف PDF مع النص الشفاف المُضاف حديثًا.

## دليل التنفيذ
### الميزة 1: ضغط صور PDF
ضغط الصور داخل ملف PDF يُقلل حجم الملف بشكل ملحوظ، مما يُسهّل مشاركته وتخزينه. إليك كيفية تحقيق ذلك باستخدام Aspose.PDF لـ .NET.

#### ملخص:
تتيح لك هذه الميزة ضغط الصور في ملفات PDF الخاصة بك مع الحفاظ على مستوى جودة مقبول.

#### التنفيذ خطوة بخطوة:
**1. افتح مستند PDF:**
ابدأ بتحميل مستند PDF الخاص بك إلى `Document` هدف.

```csharp
Document pdfDocument = new Document(dataDir);
```

**2. تهيئة خيارات التحسين:**
إعداد خيارات التحسين خصيصًا لضغط الصور.

```csharp
var optimizeOptions = new Pdf.Optimization.OptimizationOptions();
optimizeOptions.ImageCompressionOptions.CompressImages = true;
```

**3. ضبط جودة الصورة:**
ضبط جودة الصور المضغوطة لتحقيق التوازن بين تقليل الحجم والدقة البصرية.

```csharp
optimizeOptions.ImageCompressionOptions.ImageQuality = 50; // مقياس من 0 إلى 100
```

**4. تحسين الموارد:**
قم بتطبيق إعدادات التحسين على مستند PDF الخاص بك.

```csharp
pdfDocument.OptimizeResources(optimizeOptions);
```

**5. احفظ المستند المُحسَّن:**
وأخيرًا، احفظ ملف PDF المحسّن في مسار ملف جديد.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/Shrinkimage_out.pdf";
pdfDocument.Save(outputDir);
```

#### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من أن ملف PDF المدخل لديك غير تالف وقابل للوصول.
- قم بضبط إعدادات جودة الصورة استنادًا إلى الاحتياجات المحددة للحجم مقابل الوضوح.

### الميزة 2: إعداد دليل العنصر النائب
يؤدي استخدام العناصر النائبة لمسارات الملفات إلى تعزيز المرونة وإمكانية الصيانة في قاعدة التعليمات البرمجية الخاصة بك.

**مثال على الاستخدام:**

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY"; // الدليل الأساسي للمستندات
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // الدليل الأساسي لحفظ المخرجات

string templateInputFile = Path.Combine(documentDirectory, "example.pdf");
string templateOutputFile = Path.Combine(outputDirectory, "optimized_example.pdf");
```

## دليل التنفيذ
الآن بعد أن قمت بإعداد البيئة، دعنا نستخرج الصور من صفحات محددة في مستند PDF.

### نظرة عامة على الميزات
هدفنا هو استهداف وحفظ الصور من صفحات محددة داخل ملف PDF. تُعد هذه الميزة مفيدة بشكل خاص للمشاريع التي تتطلب استخراج محتوى محدد.

#### الخطوة 1: تحديد مسارات الدليل
ابدأ بتحديد مكان وجود مستندات الإدخال والمكان الذي تريد حفظ الصور المستخرجة فيه:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
يستبدل `YOUR_DOCUMENT_DIRECTORY` مع المسار الفعلي لملفات PDF الخاصة بك.

#### الخطوة 2: تهيئة PdfExtractor
إنشاء مثيل لـ `PdfExtractor` الفئة التي ستتعامل مع عملية استخراج الصورة:

```csharp
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "ExtractImages-Page.pdf");
```
ال `BindPdf` تربط الطريقة ملف PDF الخاص بك بكائن المستخرج.

#### الخطوة 3: تعيين نطاق الصفحات
حدد رقم الصفحة (الصفحات) التي تريد استخراج الصور منها:

```csharp
pdfExtractor.StartPage = 2;
pdfExtractor.EndPage = 2;
```
هذا المثال يستهدف الصفحة الثانية فقط. عدّل هذه القيم حسب حاجة استخدامك.

#### الخطوة 4: استخراج الصور وحفظها
استخدم حلقة لتكرار كل صورة مستخرجة وحفظها على القرص:

```csharp
while (pdfExtractor.HasNextImage())
{
    using (MemoryStream memoryStream = new MemoryStream())
    {
        pdfExtractor.GetNextImage(memoryStream);
        
        string outputPath = @"YOUR_OUTPUT_DIRECTORY" + DateTime.Now.Ticks.ToString() + "_out.jpg";
        using (FileStream fileStream = new FileStream(outputPath, FileMode.Create))
        {
            memoryStream.WriteTo(fileStream);
        }
    }
}
```
يتم حفظ كل صورة باسم ملف فريد استنادًا إلى الطابع الزمني الحالي.

## دليل التنفيذ

سيرشدك هذا القسم إلى كيفية حذف صور محددة أو كلها من مستند PDF.

### حذف صورة محددة

لإزالة صورة من ملف PDF الخاص بك:

#### الخطوة 1: تحميل مستند PDF

إنشاء مثيل لـ `Document` حمّل ملف PDF الخاص بك. حدد ملف PDF الذي تعمل عليه.

```csharp
// ExStart:تحميل مستند PDF
using System.IO;
using Aspose.Pdf;

string dataDir = "your_directory_path/";
Document pdfDocument = new Document(dataDir + "DeleteImages.pdf");
```

#### الخطوة 2: الوصول إلى الصورة وحذفها

ادخل إلى الصفحة التي تحتوي على صورتك المستهدفة. استخدم `Delete` طريقة إزالته.

```csharp
// ExStart:حذف صورة محددة
pdfDocument.Pages[1].Resources.Images.Delete(1);
dataDir = dataDir + "DeleteImages_out.pdf";
```

في هذا المثال، سنحذف الصورة الأولى من الصفحة الأولى. عدّل المعلمات لاستهداف صور أو صفحات مختلفة حسب الحاجة.

#### الخطوة 3: حفظ المستند المحدث

بعد إجراء التغييرات، احفظ المستند باستخدام `Save` طريقة.

```csharp
// ExStart:حفظ ملف PDF المحدث
pdfDocument.Save(dataDir);
```

### حذف جميع الصور من الصفحة

لإزالة كافة الصور من صفحة معينة:

```csharp
// قم بفحص كل صورة في موارد الصفحة ثم قم بحذفها.
foreach (var img in pdfDocument.Pages[1].Resources.Images)
{
    pdfDocument.Pages[1].Resources.Images.Delete(img.Index);
}
```

- حفظ ملفات PDF مع العناصر الرسومية

لنبدأ بالتأكد من أن لديك المتطلبات الأساسية جاهزة.

## دليل التنفيذ

### إنشاء مستطيلات شفافة باستخدام شفافية الألوان ألفا

الوظيفة الأساسية لهذا البرنامج التعليمي هي توضيح كيفية إنشاء مستطيلات داخل مستند PDF باستخدام الشفافية ألفا، وإثراء ملفات PDF الخاصة بك بعناصر شبه شفافة تعمل على تعزيز الجماليات البصرية.

#### الخطوة 1: إنشاء مستند
إنشاء مثيل لـ `Document` الفئة التي تمثل ملف PDF.

```csharp
// إنشاء مستند PDF جديد
Document doc = new Document();
```

#### الخطوة 2: إضافة صفحة
أضف صفحةً إلى المستند. هنا سيتم رسم المستطيلات.

```csharp
// إضافة صفحة إلى المستند
Aspose.Pdf.Page page = doc.Pages.Add();
```

#### الخطوة 3: إنشاء مثيل الرسم البياني
ال `Graph` يعمل الكائن كلوحة رسم داخل صفحة PDF، مما يسمح لك بإضافة أشكال مثل المستطيلات.

```csharp
// تحديد أبعاد الرسم البياني (اللوحة القماشية)
Aspose.Pdf.Drawing.Graph canvas = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
```

#### الخطوة 4: إنشاء المستطيلات وتخصيصها
أنشئ مستطيلاً واضبط لون تعبئته باستخدام شفافية ألفا. `Color.FromArgb` الطريقة من `System.Drawing.Color` يسمح بتحديد قيمة ARGB.

```csharp
// إنشاء مستطيل بأبعاد محددة
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 100);

// تعيين لون التعبئة باستخدام شفافية ألفا (128 في هذا المثال)
rect.GraphInfo.FillColor = Aspose.Pdf.Color.FromRgb(Color.FromArgb(128, Color.FromArgb(12957183)));

// إضافة مستطيل إلى الرسم البياني
canvas.Shapes.Add(rect);
```

#### الخطوة 5: كرر ذلك للمستطيلات الإضافية
يمكنك إضافة المزيد من المستطيلات بألوان ومواضع مختلفة حسب الحاجة.

```csharp
// إنشاء مستطيل ثاني بمواصفات مختلفة
Aspose.Pdf.Drawing.Rectangle rect1 = new Aspose.Pdf.Drawing.Rectangle(200, 150, 200, 100);
rect1.GraphInfo.FillColor = Aspose.Pdf.Color.FromRgb(Color.FromArgb(128, Color.FromArgb(16118015)));

// أضفه إلى الرسم البياني
canvas.Shapes.Add(rect1);
```

#### الخطوة 6: احفظ ملف PDF
وأخيرًا، احفظ مستندك في الدليل المحدد.

```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY/CreateRectangleWithAlphaColor_out.pdf";
doc.Save(dataDir);
```

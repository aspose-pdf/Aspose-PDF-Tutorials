## دليل التنفيذ
سنقوم بتقسيم عملية تحويل مجرى الصورة إلى مستند PDF.

### الميزة: تحويل تدفق الصور إلى PDF
تتضمن هذه الميزة قراءة صورة من مجرى ملف، وتحويلها إلى مصفوفة بايتات، ثم تضمينها داخل مستند PDF باستخدام Aspose.PDF لـ .NET.

#### الخطوة 1: إعداد مستندك

```csharp
using System.IO;
using Aspose.Pdf;

// إنشاء مثيل مستند جديد
Document pdf1 = new Document();
// إضافة صفحة إلى المستند
Page sec = pdf1.Pages.Add();
```

**توضيح**:هنا، نقوم بتهيئة ملف جديد `Document` الكائن وإضافة صفحة فارغة حيث سيتم وضع صورتنا.

#### الخطوة 2: قراءة الصورة من مجرى

```csharp
FileStream fs = File.OpenRead("YOUR_DOCUMENT_DIRECTORY/aspose.jpg");
byte[] data = new byte[fs.Length];
fs.Read(data, 0, data.Length);
fs.Close();
```

**توضيح**نفتح مسار ملف لقراءة صورة وتحويلها إلى مصفوفة بايتات. إغلاق المسار ضروري لتوفير الموارد.

#### الخطوة 3: إنشاء MemoryStream من مجموعة البايتات

```csharp
MemoryStream ms = new MemoryStream(data);
Image imageht = new Image();
imageht.ImageStream = ms;
```

**توضيح**:نحن ننشئ `MemoryStream` باستخدام مصفوفة البايتات لدينا. هذا يسمح لنا باستخدام التدفق كمصدر للصورة داخل Aspose.PDF.

#### الخطوة 4: إضافة صورة إلى ملف PDF

```csharp
sec.Paragraphs.Add(imageht);
pdf1.Save("YOUR_OUTPUT_DIRECTORY/ConvertMemoryStreamImageToPdf_out.pdf");
ms.Close();
```

**توضيح**:تتم إضافة الصورة إلى فقرات الصفحة، وأخيرًا، يتم حفظ المستند كملف PDF. إغلاق `MemoryStream` ضمان إدارة الموارد بشكل صحيح.

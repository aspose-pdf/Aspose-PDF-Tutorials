## دليل التنفيذ
في هذا القسم، سنرشدك خلال كل خطوة من خطوات تقديم LaTeX في ملفات PDF بوضوح.

### إنشاء مستند وصفحة جديدة (H2)
#### ملخص:
ابدأ بإنشاء مثيل لـ `Document` الصف وإضافة صفحة جديدة إليه.

**الخطوة 1: تهيئة المستند**
```csharp
var doc = new Document();
```
*توضيح:* يُنشئ هذا السطر مستند PDF جديدًا. ستضيف لاحقًا محتوىً، بما في ذلك نص LaTeX، إلى هذا المستند.

#### الخطوة 2: إضافة الصفحة
```csharp
var page = doc.Pages.Add();
```
*غاية:* يُضيف صفحة جديدة إلى المستند لعرض نص LaTeX. تعمل كل صفحة كحاوية لمحتواها في ملفات PDF.

### إضافة نص LaTeX (H3)
الآن، دعنا نضيف نص LaTeX الخاص بنا باستخدام Aspose.PDF `TeXFragment`.

#### الخطوة 1: تعريف نص LaTeX
```csharp
string latexScript = """
    \usepackage{amsmath,amsthm}
    \begin{document}
    \begin{proof} The proof is as follows:
    \begin{align}
    (x+y)^3&=(x+y)(x+y)^2
    (x+y)(x^2+2xy+y^2)\\
    &=x^3+3x^2y+3xy^2+x^3.\qedhere
    \end{align}
    \end{proof}
    \end{document}"
```
*توضيح:* يحتوي هذا البرنامج النصي على كود LaTeX لتقديم دليل رياضي، بما في ذلك الحزم والمحاذاة للتنسيق المناسب.

#### الخطوة 2: إنشاء TeXFragment
```csharp
var latex = new TeXFragment(latexScript);
```
*غاية:* يحول سلسلة LaTeX الخاصة بك إلى `TeXFragment`، والتي يمكن لـ Aspose.PDF تفسيرها لعرضها داخل مستند PDF.

#### الخطوة 3: إضافة جزء إلى الصفحة
```csharp
page.Paragraphs.Add(latex);
```
*توضيح:* يضيف `TeXFragment` إلى مجموعة فقرات الصفحة، ودمج نص LaTeX الخاص بك في محتوى PDF.

### حفظ المستند (H2)
وأخيرًا، احفظ مستندك في الموقع المحدد:

#### الخطوة 1: تحديد مسار الحفظ
```csharp
var dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
*غاية:* يُحدد الدليل الذي سيتم حفظ ملف PDF الناتج فيه. تأكد من وجود هذا المسار أو إمكانية كتابته بواسطة تطبيقك.

#### الخطوة 2: حفظ المستند
```csharp
doc.Save(dataDir + "Script_out.pdf");
```
*توضيح:* يكتب هذا الأمر المستند المُعالج على القرص، مما يؤدي إلى إنشاء ملف PDF يحتوي على محتوى LaTeX في الموقع المحدد.

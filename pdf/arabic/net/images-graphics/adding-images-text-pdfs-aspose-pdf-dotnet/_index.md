## دليل التنفيذ

الآن، لنشرح بالتفصيل كيفية استخدام Aspose.PDF لـ .NET لإضافة الصور والنصوص إلى ملفات PDF. سنتناول كل ميزة بالتفصيل.

### إضافة صورة إلى ملف PDF
#### ملخص
إضافة صورة إلى موقع محدد في ملف PDF يُحسّن مظهره أو يُضيف معلومات إضافية. سيرشدك هذا القسم خلال عملية تضمين صورة باستخدام Aspose.PDF لـ .NET.

#### التنفيذ خطوة بخطوة
1. **جهّز بيئتك**:تأكد من إعداد المسارات لملف الإدخال وملف الإخراج والصورة.
2. **تدفقات الملفات المفتوحة**: يستخدم `FileStream` لقراءة الصورة وإنشاء تدفق PDF الناتج.
   ```csharp
   using (FileStream inImgStream = new FileStream("image.jpg", FileMode.Open))
   {
       using (FileStream outputStream = new FileStream("output.pdf", FileMode.Create))
       ```
3. **تحميل المستند**:قم بتهيئة مستند PDF الخاص بك باستخدام `Document`.
   ```csharp
doc = مستند جديد("input.pdf");
```
4. **Create PdfFileMend Instance**: Use this to modify the document.
   ```csharp
   PdfFileMend mendor = new PdfFileMend(doc);
```
5. **أضف الصورة**:حدد رقم الصفحة وإحداثيات وضع الصورة.
   ```csharp
   mendor.AddImage(inImgStream, 1, 50, 50, 100, 100);
```
6. **Save Changes**: Close the `PdfFileMend` instance to save your modifications.
   ```csharp
   mendor.Close();
```
#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسارات الملفات صحيحة ويمكن الوصول إليها.
- تحقق من وجود إحداثيات صالحة تتناسب مع أبعاد الصفحة.

### إضافة نص إلى ملف PDF
#### ملخص
إدراج نص منسق في ملف PDF يُمكّنك من إبراز المعلومات أو إضافة تعليقات توضيحية. سيشرح هذا القسم كيفية إضافة نص بخصائص محددة باستخدام Aspose.PDF لـ .NET.

#### التنفيذ خطوة بخطوة
1. **إعداد مسارات الملفات**:قم بتحديد المسارات لملفات الإدخال والإخراج الخاصة بك.
2. **تدفق الإخراج المفتوح**:إنشاء مجرى إخراج لملف PDF.
   ```csharp
   using (FileStream outputStream = new FileStream("output.pdf", FileMode.Create))
```
3. **Load Document**: Initialize with `Document`.
   ```csharp
doc = new Document("input.pdf");
```
4. **إنشاء مثيل PdfFileMend**:لتعديل المستند.
   ```csharp
   PdfFileMend mendor = new PdfFileMend(doc);
```
5. **Define Formatted Text**: Specify text attributes like color, font style, and size.
   ```csharp
   FormattedText ft = new FormattedText(
       "Sample text",
       Color.FromArgb(0, 200, 0),
       FontStyle.TimesRoman,
       EncodingType.Winansi,
       false,
       12);
```
6. **إضافة نص**:تعيين رقم الصفحة والإحداثيات.
   ```csharp
   mendor.AddText(ft, 1, 50, 100, 100, 200);
```
7. **Save Changes**: Close `PdfFileMend` to apply changes.
   ```csharp
   mendor.Close();
```
#### نصائح استكشاف الأخطاء وإصلاحها
- التحقق من صحة ترميز النص وإعدادات الخط.
- تأكد من أن الإحداثيات تقع ضمن أبعاد الصفحة.

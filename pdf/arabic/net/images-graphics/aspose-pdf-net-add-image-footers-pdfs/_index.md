- خطوات إضافة صورة كتذييل في مستندات PDF الموجودة

## دليل التنفيذ

### نظرة عامة على إضافة تذييلات الصور

تتضمن إضافة تذييل الصورة إنشاء `PdfFileStamp` أداة لختم صفحات PDF بالصور. هذه الميزة مثالية لإضافة علامات تجارية أو علامات مائية إلى المستندات.

#### الخطوة 1: إنشاء كائن PdfFileStamp وربطه
```csharp
// تهيئة كائن PdfFileStamp
class Program
{
    static void Main(string[] args)
    {
        // إنشاء مثيل PdfFileStamp
        PdfFileStamp fileStamp = new PdfFileStamp();
```
ال `PdfFileStamp` توفر الفئة طرقًا لإضافة طوابع، بما في ذلك تذييلات الصور، إلى ملفات PDF. ابدأ بتهيئة هذا الكائن.

#### الخطوة 2: افتح المستند

قم بربط مستند PDF المستهدف الخاص بك بـ `PdfFileStamp` هدف:
```csharp
// تحديد المسار إلى دليل المستندات
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_StampsWatermarks();

// ربط مستند PDF
fileStamp.BindPdf(dataDir + "AddImage-Footer.pdf");
```
هنا، حدد مسارات الملفات لكل من الإدخال والإخراج. `BindPdf` تقوم الطريقة بإعداد ملف PDF الخاص بك للختم.

#### الخطوة 3: إضافة صورة التذييل

بعد ذلك، أضف صورة كتذييل لكل صفحة:
```csharp
// فتح تدفق ملف الصورة
using (FileStream imageStream = new FileStream(dataDir + "aspose-logo.jpg", FileMode.Open))
{
    // إضافة تذييل الصورة
    fileStamp.AddFooter(imageStream, 10);
}
```
ال `AddFooter` تتطلب هذه الطريقة تدفقًا للصور وقيمة هامش. يحدد الهامش المسافة بين حافة الصفحة وتذييلها.

#### الخطوة 4: حفظ ملف PDF المحدث

بمجرد إضافة تذييل الصورة، احفظ المستند المحدث:
```csharp
// احفظ ملف PDF المختوم
fileStamp.Save(dataDir + "AddImage-Footer_out.pdf");
```
تقوم هذه الخطوة بإتمام التغييرات عن طريق كتابتها في ملف جديد.

#### الخطوة 5: إغلاق كائن PdfFileStamp

وأخيرًا، أغلق `PdfFileStamp` هدف تحرير الموارد:
```csharp
// إصدار الموارد
fileStamp.Close();
}
```
**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن مسار صورتك صحيح ويمكن الوصول إليه.
- قم بضبط قيمة الهامش إذا لم يكن موضع التذييل كما هو متوقع.

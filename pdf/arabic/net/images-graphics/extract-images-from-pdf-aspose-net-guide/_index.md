## دليل التنفيذ: استخراج الصور من ملف PDF

بعد إعداد كل شيء، لنستخرج الصور من ملف PDF. هذه الميزة مفيدة لأتمتة مهام معالجة المستندات.

### ملخص

سوف نستخدم `PdfExtractor` فئة لاستخراج وحفظ الصور المُضمَّنة في ملف PDF. اتبع هذه العملية خطوة بخطوة:

### التنفيذ خطوة بخطوة

#### 1. إعداد الدلائل

قم بتحديد أدلة الإدخال والإخراج الخاصة بك:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // تأكد من وجود هذا الدليل
```

يساعد هذا التنظيم على إدارة الملفات بكفاءة أثناء الاستخراج.

#### 2. تهيئة PdfExtractor

إنشاء مثيل لـ `PdfExtractor` وربط ملف PDF:

```csharp
// إنشاء مثيل لـ PdfExtractor
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "/ExtractImages.pdf");
```

**لماذا الربط؟** يقوم الربط بربط مستندك بالمستخرج للعمليات اللاحقة.

#### 3. قم باستخراج الصورة

استدعاء `ExtractImage` طريقة البدء في الاستخراج:

```csharp
// استخراج الصور من ملف PDF
pdfExtractor.ExtractImage();
```

يقوم هذا بمسح الصفحات وتحديد كائنات الصور المضمنة.

#### 4. حفظ الصور المستخرجة

قم بالتكرار على كل صورة مستخرجة، وحفظها بأسماء ملفات فريدة:

```csharp
while (pdfExtractor.HasNextImage())
{
    string outputFileName = $"{outputDir}/{DateTime.Now.Ticks}_out.jpg";
    pdfExtractor.GetNextImage(outputFileName);
}
```

**لماذا استخدام DateTime؟** استخدام `DateTime.Now.Ticks` يضمن أن كل ملف صورة له اسم فريد، مما يمنع الكتابة فوقه.

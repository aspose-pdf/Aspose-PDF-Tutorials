## دليل التنفيذ
الآن بعد أن قمت بالإعداد، دعنا ننتقل إلى استخراج الصور من ملف PDF.

### استخراج الصور
يرشدك هذا القسم خلال الخطوات المطلوبة لاستخراج الصور من صفحة معينة من مستند PDF الخاص بك.

#### افتح مستند PDF الخاص بك
أولاً، قم بتحميل ملف PDF الخاص بك باستخدام Aspose.PDF:
```csharp
using System.IO;
using Aspose.Pdf;

namespace PdfImageExtraction
{
    public class ExtractImages
    {
        public static void Run()
        {
            // قم بتحديد المسار إلى الدليل الذي يحتوي على مستنداتك
            string dataDir = "path/to/your/documents/";

            // تحميل مستند PDF
            Document pdfDocument = new Document(dataDir + "ExtractImages.pdf");
```
#### الوصول إلى الصور وحفظها
بعد ذلك، انتقل عبر الصفحات واحفظ كل صورة تجدها:
```csharp
            // تكرار جميع الصور على صفحة معينة
            foreach (var image in pdfDocument.Pages[1].Resources.Images)
            {
                using (FileStream outputImage = new FileStream(dataDir + "output_" + image.Number + ".jpg", FileMode.Create))
                {
                    // احفظ الصورة المستخرجة بصيغة JPEG
                    image.Save(outputImage, System.Drawing.Imaging.ImageFormat.Jpeg);
                }
            }

            // احفظ أي تعديلات مرة أخرى على ملف PDF الخاص بك إذا لزم الأمر
            pdfDocument.Save(dataDir + "ExtractImages_out.pdf");
        }
    }
}
```
**توضيح:**
- `XImage` يمثل صورة داخل صفحة PDF.
- ال `Save` تكتب الطريقة الصورة على القرص بتنسيق JPEG.

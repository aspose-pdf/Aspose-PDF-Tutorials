## دليل التنفيذ
اتبع الخطوات التالية لإضافة نص مظلل بالتدرج اللوني:

### إضافة نص باستخدام ألوان التظليل المتدرجة
#### ملخص
خصّص أجزاء النص في ملف PDF باستخدام تدرجات لونية لتحسين المظهر. استخدم Aspose.PDF `GradientAxialShading` لانتقالات ألوان سلسة.

#### التنفيذ خطوة بخطوة
**1. قم بتحميل مستند PDF**
تحميل ملف PDF موجود في `Document` هدف.
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_Text();
using (Document pdfDocument = new Document(dataDir + "text_sample4.pdf"))
{
    // سيتم وضع الكود الخاص بالتلاعب بالمستند هنا.
}
```
**2. البحث عن نص محدد**
يستخدم `TextFragmentAbsorber` للعثور على أجزاء النص التي تريد تعديلها.
```csharp
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Lorem ipsum");
pdfDocument.Pages.Accept(absorber);
TextFragment textFragment = absorber.TextFragments[1];
```
**3. تطبيق التظليل المتدرج**
اضبط `ForegroundColor` باستخدام مساحة ألوان النمط.
```csharp
textFragment.TextState.ForegroundColor = new Aspose.Pdf.Color()
{
    PatternColorSpace = new Aspose.Pdf.Drawing.GradientAxialShading(Color.Red, Color.Blue)
};
```
**4. إضافة تسطير (اختياري)**
تحسين مظهر النص عن طريق التسطير.
```csharp
textFragment.TextState.Underline = true;
```
**5. احفظ المستند**
احفظ التغييرات في ملف PDF جديد.
```csharp
pdfDocument.Save(dataDir + "text_out.pdf");
```

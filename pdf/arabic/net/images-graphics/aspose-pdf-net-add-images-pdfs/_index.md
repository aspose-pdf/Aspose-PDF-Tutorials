## دليل التنفيذ
الآن، دعونا نوضح الخطوات اللازمة لإضافة صورة إلى صفحة PDF.

### إضافة صورة إلى موقع محدد على صفحة PDF
**ملخص**
تتيح لك هذه الميزة تراكب الصور على مستندات PDF الخاصة بك في أي إحداثيات محددة، مما يعزز جاذبيتها البصرية ووظائفها.

#### الخطوة 1: افتح مستند PDF الحالي
```csharp
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/PDFOperators.pdf");
```
- **توضيح**:يقوم هذا السطر بتحميل ملف PDF موجود في `pdfDocument` هدف.

#### الخطوة 2: تحديد إحداثيات الصورة
```csharp
int lowerLeftX = 100;
int lowerLeftY = 100;
int upperRightX = 200;
int upperRightY = 200;
```
- **توضيح**:تحدد هذه الإحداثيات المكان الذي سيتم وضع الصورة فيه على الصفحة، مع (lowerLeftX, lowerLeftY) كنقطة بداية و(upperRightX, upperRightY) كنقطة نهاية.

#### الخطوة 3: تحميل الصورة إلى مجرى الملف
```csharp
FileStream imageStream = new FileStream("YOUR_DOCUMENT_DIRECTORY/PDFOperators.jpg", FileMode.Open);
```
- **توضيح**يفتح ملف صورة لإضافته إلى صفحة PDF. تأكد من صحة مسار الملف واسمه.

#### الخطوة 4: إضافة صورة إلى موارد الصفحة
```csharp
Page page = pdfDocument.Pages[1];
page.Resources.Images.Add(imageStream);
```
- **توضيح**:استرجاع الصفحة الأولى من المستند وإضافة تدفق الصورة إلى مواردها.

#### الخطوة 5: تحديد حالة الرسومات باستخدام عامل GSave
```csharp
page.Contents.Add(new Aspose.Pdf.Operators.GSave());
```
- **توضيح**:يحفظ حالة الرسومات الحالية، مما يضمن عدم تأثير أي تعديلات على العناصر الأخرى في الصفحة.

#### الخطوة 6: إنشاء وتعيين مصفوفة وضع الصورة
```csharp
Aspose.Pdf.Rectangle rectangle = new Aspose.Pdf.Rectangle(lowerLeftX, lowerLeftY, upperRightX, upperRightY);
Matrix matrix = new Matrix(new double[] { rectangle.URX - rectangle.LLX, 0, 0, rectangle.URY - rectangle.LLY, rectangle.LLX, rectangle.LLY });
page.Contents.Add(new Aspose.Pdf.Operators.ConcatenateMatrix(matrix));
```
- **توضيح**:يحدد مصفوفة التحويل التي تضع الصورة وفقًا للإحداثيات المحددة.

#### الخطوة 7: ارسم صورة باستخدام عامل Do
```csharp
XImage ximage = page.Resources.Images[page.Resources.Images.Count];
page.Contents.Add(new Aspose.Pdf.Operators.Do(ximage.Name));
```
- **توضيح**:يستعيد الصورة المضافة حديثًا ويضعها على الصفحة باستخدام `Do` المشغل.

#### الخطوة 8: استعادة حالة الرسومات باستخدام عامل GRestore
```csharp
page.Contents.Add(new Aspose.Pdf.Operators.GRestore());
```
- **توضيح**:يستعيد حالة الرسومات إلى شكلها الأصلي بعد رسم الصورة.

#### الخطوة 9: حفظ مستند PDF المعدّل
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/PDFOperators_out.pdf";
pdfDocument.Save(outputDir);
```
- **توضيح**:يحفظ جميع التغييرات التي تم إجراؤها على ملف جديد في الدليل المحدد.

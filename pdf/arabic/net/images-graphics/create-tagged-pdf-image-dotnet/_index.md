## دليل التنفيذ
### إنشاء ملف PDF مُعلَّم بصورة
يوضح هذا القسم كيفية إنشاء ملف PDF مُعَلَّم وتضمين صورة فيه.

#### ملخص
يُوفر الوسم معلومات هيكلية للتقنيات المساعدة، مما يجعل مستنداتك أكثر سهولة في الوصول. بتضمين الصور في المحتوى المُوسوم، تضمن وصف العناصر المرئية بشكل صحيح للمستخدمين الذين يعتمدون على قارئات الشاشة.

#### التنفيذ خطوة بخطوة
##### 1. تهيئة مستند PDF
ابدأ بإنشاء حساب جديد `Document` هدف:
```csharp
// عنصر نائب لدليل المستندات الخاص بك
double dataDir = "YOUR_DOCUMENT_DIRECTORY";

// تهيئة مستند PDF جديد
Document document = new Document();
ITaggedContent taggedContent = document.TaggedContent;
```
##### 2. تعيين العنوان واللغة
يساعد تحديد العنوان واللغة في تحديد سياق المحتوى الخاص بك:
```csharp
// تعيين العنوان واللغة لسهولة الوصول
TaggedContent.SetTitle("CreatePDFwithTaggedImage");
taggedContent.SetLanguage("en-US");
```
##### 3. إنشاء عنصر الشكل
يمثل عنصر الشكل صورة في ملفات PDF المميزة، مما يعزز إمكانية الوصول من خلال توفير السياق والأوصاف:
```csharp
// إنشاء عنصر شكل لحمل الصورة
IllustrationElement figure1 = taggedContent.CreateFigureElement();
taggedContent.RootElement.AppendChild(figure1);

// تعيين نص بديل وعنوان للصورة
figure1.AlternativeText = "Aspose Logo";
figure1.Title = "Image 1";

// تعيين علامة لعنصر الشكل
figure1.SetTag("Fig");
```
##### 4. تضمين صورة
تأكد من أن صورتك ذات دقة عالية (يوصى بـ 300 نقطة في البوصة) وأضفها إلى الشكل:
```csharp
// إضافة صورة من المسار المحدد
figure1.SetImage(dataDir + @"aspose-logo.png");
```
##### 5. احفظ المستند
أخيرًا، احفظ ملف PDF الذي قمت بوضع علامة عليه مع الصور المضمنة:
```csharp
// عنصر نائب لدليل الإخراج الخاص بك
double outputDir = "YOUR_OUTPUT_DIRECTORY";

// حفظ المستند
document.Save(outputDir + "PDFwithTaggedImage.pdf");
```

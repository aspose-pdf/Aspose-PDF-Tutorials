## دليل التنفيذ (H2)
سنقوم بتقسيم العملية إلى خطوات قابلة للإدارة.

### إنشاء مستند PDF (H3)
**ملخص:** ابدأ بإنشاء مستند جديد وإضافة صفحة إليه.
```csharp
Document pdf = new Aspose.Pdf.Document();
Page sec1 = pdf.Pages.Add(); // إضافة صفحة جديدة
```
- `Aspose.Pdf.Document`:يمثل ملف PDF بأكمله.
- `Pages.Add()`:تضيف صفحة جديدة حيث يمكنك وضع المحتوى.

### إضافة أجزاء نصية (H3)
**ملخص:** إدراج أجزاء من النص مع خيارات التنسيق مثل اللون وحجم الخط.
```csharp
TextFragment t1 = new Aspose.Pdf.Text.TextFragment("paragraph 3 segment");
sec1.Paragraphs.Add(t1);

t1.TextState.ForegroundColor = Color.Red; // تعيين لون النص إلى اللون الأحمر
t1.TextState.FontSize = 12;                // ضبط حجم الخط إلى 12
```
- `TextFragment`:يمثل كتلة من النص.
- `TextState.ForegroundColor`:يحدد لون النص.
- `TextState.FontSize`:التحكم في حجم الخط.

### إدراج الصور (H3)
**ملخص:** أضف الصور إلى مستند PDF الخاص بك، مما يؤدي إلى إثراء محتواه بصريًا.
```csharp
Image image1 = new Aspose.Pdf.Image();
image1.File = "path\to\your\test_image.png"; // حدد مسار الصورة
sec1.Paragraphs.Add(image1);
```
- `Aspose.Pdf.Image`:يمثل صورة في مستندك.
- `image1.File`:يحدد مسار الملف للصورة.

### إضافة صناديق عائمة (H3)
**ملخص:** استخدم المربعات العائمة لتنظيم المحتوى وتحديد موضعه بشكل فعال داخل الصفحة.
```csharp
FloatingBox box1 = new Aspose.Pdf.FloatingBox(117, 21);
box1.Left = -4; // التموضع من الحافة اليسرى
box1.Top = -4;  // التموضع من الحافة العلوية
sec1.Paragraphs.Add(box1);

box1.Paragraphs.Add(image1); // أضف صورة إلى المربع العائم
```
- `FloatingBox`:حاوية لتنظيم المحتوى.
- سمات الموضع (`Left`، `Top`): قم بتعيين موضع المربع على الصفحة.

### حفظ مستند PDF (H3)
**ملخص:** وأخيرًا، احفظ مستندك في ملف.
```csharp
pdf.Save("path\to\your\CreateMultiLayerPdf_out.pdf");
```
- `Document.Save()`:يكتب الناتج النهائي على القرص.

## دليل التنفيذ
الآن بعد اكتمال عملية الإعداد، دعنا نضيف إجراء إغلاق المستند إلى ملف PDF الخاص بك.

### إضافة إجراء إغلاق المستند
تُنفّذ هذه الميزة جافا سكريبت عند إغلاق المستخدم لمستند PDF. اتبع الخطوات التالية:

#### الخطوة 1: جهّز بيئتك
تأكد من وجود ملف PDF موجود باسم `CreateDocumentLink.pdf` في الدليل المحدد الخاص بك.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
```

#### الخطوة 2: افتح مستند PDF
يستخدم `PdfContentEditor` لفتح ملف PDF والتلاعب به:
```csharp
PdfContentEditor contentEditor = new PdfContentEditor();
contentEditor.BindPdf(dataDir + "CreateDocumentLink.pdf");
```
تؤدي هذه الخطوة إلى ربط مستندك الحالي بالتحرير.

#### الخطوة 3: تحديد إجراء الإغلاق
حدد الإجراء باستخدام `AddDocumentAdditionalAction`. يتم تشغيل تنبيه JavaScript عند الإغلاق:
```csharp
contentEditor.AddDocumentAdditionalAction(PdfContentEditor.DocumentClose,
    "app.alert('Thank you for using Aspose products!');");
```
ال `PdfContentEditor.DocumentClose` يحدد المعامل الحدث، ويحدد سلسلة JavaScript الإجراء.

#### الخطوة 4: حفظ ملف PDF المحدث
بعد إعداد المستند الخاص بك بإجراءات إضافية، احفظه لتطبيق التغييرات:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/";
contentEditor.Save(outputDir + "CreateDocAdditionalAction_out.pdf");
```
تؤدي هذه الخطوة إلى حفظ ملف PDF المعدّل في الموقع المطلوب.

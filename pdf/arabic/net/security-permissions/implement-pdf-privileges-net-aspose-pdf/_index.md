## دليل التنفيذ

الآن بعد أن قمنا بإعداد بيئتنا، دعنا ننتقل إلى الخطوات لتقييد وتشفير مستندات PDF باستخدام Aspose.PDF لـ .NET.

### تقييد امتيازات PDF

#### ملخص
يركز هذا القسم على تعيين امتيازات المستندات، والسماح بإجراءات محددة فقط مثل قراءة الشاشة ومنع إجراءات أخرى.

#### خطوات التنفيذ

##### الخطوة 1: تحميل ملف PDF المصدر

قم بتحميل ملف PDF المصدر الخاص بك إلى `Aspose.Pdf.Document` الكائن. يسمح لك هذا بالتحكم في محتوياته وخصائصه.

```csharp
string dataDir = "path_to_your_directory";
using (Document document = new Document(dataDir + "input.pdf"))
{
    // المضي قدما في إعداد الامتيازات
}
```

##### الخطوة 2: إنشاء امتيازات المستند

إنشاء `DocumentPrivilege` لتحديد القيود التي تريد تطبيقها. على سبيل المثال، منع جميع الإجراءات باستثناء قراءة الشاشة:

```csharp
documentPrivilege = DocumentPrivilege.ForbidAll;
documentPrivilege.AllowScreenReaders = true; // تمكين قارئات الشاشة فقط
```

##### الخطوة 3: تشفير ملف PDF

شَفِّر الملف باستخدام كلمات مرور المستخدم والمالك. تضمن هذه الخطوة إمكانية عرض المستند للمستخدمين مع قيود محددة.

```csharp
document.Encrypt("user", "owner", documentPrivilege, CryptoAlgorithm.AESx128, false);
```
- **كلمة مرور المستخدم**:مطلوب لفتح المستند.
- **كلمة مرور المالك**:يمنح الوصول الكامل لتعديل الأذونات.
- **خوارزمية التشفير**:اختر خوارزمية التشفير المناسبة (على سبيل المثال، AES 128-bit).

##### الخطوة 4: حفظ المستند المحدث

وأخيرًا، احفظ التغييرات عن طريق كتابة المستند المعدّل في ملف جديد.

```csharp
document.Save(dataDir + "SetPrivileges_out.pdf");
```

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تحديد المسارات بشكل صحيح، وإلا فقد تفشل عمليات الملف.
- تأكد من منح الأذونات اللازمة لقراءة الملفات وكتابتها في بيئتك.

## دليل التنفيذ

### تشفير ملفات PDF باستخدام كلمات مرور المستخدم والمالك

تتيح لك هذه الميزة حماية ملفات PDF الخاصة بك عن طريق تعيين كلمة مرور لكلٍّ من المستخدم والمالك. إليك كيفية عملها:

#### الخطوة 1: ربط مستند PDF
أولاً، قم بتحميل المستند الذي تريد تشفيره:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        fileSecurity.BindPdf("YOUR_DOCUMENT_DIRECTORY/Encrypt.pdf");
    }
}
```
**لماذا؟** تعتبر هذه الخطوة بالغة الأهمية لأنها تقوم بتحميل ملف PDF المستهدف إلى الذاكرة للمعالجة.

#### الخطوة 2: تطبيق التشفير
الآن، قم بتطبيق التشفير باستخدام الأذونات المحددة وتشفير AES 256 بت:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        // تشفير الملف باستخدام كلمات مرور المستخدم والمالك، مما يسمح بالطباعة فقط
        fileSecurity.EncryptFile("user", "owner", DocumentPrivilege.Print, KeySize.x256, Algorithm.AES);
    }
}
```
**المعلمات موضحة:**
- `"user"`:كلمة المرور المطلوبة لفتح ملف PDF.
- `"owner"`:كلمة المرور لتغيير الأذونات أو إزالة التشفير.
- `DocumentPrivilege.Print`:يقيد الوصول للطباعة فقط.
- `KeySize.x256`:يحدد تشفير AES 256 بت.
- `Algorithm.AES`:يستخدم معيار التشفير المتقدم.

#### الخطوة 3: حفظ الملف المشفر
وأخيرًا، احفظ مستندك المشفر:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        fileSecurity.Save("YOUR_OUTPUT_DIRECTORY/Encrypt_out.pdf");
    }
}
```
**لماذا؟** تؤدي هذه الخطوة إلى كتابة جميع التغييرات في ملف جديد، مع الحفاظ على ملف PDF الأصلي.

### نصائح استكشاف الأخطاء وإصلاحها
- **لم يتم العثور على الملف**:تأكد من أن مساراتك صحيحة ويمكن الوصول إليها.
- **مشاكل الأذونات**:تأكد من أن تطبيقك لديه الأذونات الكافية لقراءة/كتابة الملفات في الدلائل المحددة.
- **كلمات مرور غير صحيحة**:تحقق جيدًا من كلمات مرور المستخدم والمالك بحثًا عن الأخطاء المطبعية أو أخطاء التنسيق.

## دليل التنفيذ

### التحقق من حماية كلمة المرور لملف PDF

تتيح لك هذه الميزة تحديد ما إذا كان ملف PDF مشفّرًا بكلمة مرور. إليك الخطوات اللازمة:

#### الخطوة 1: تحديد المسار إلى ملف PDF المستهدف

أولاً، حدد الدليل واسم ملف PDF الذي تريد التحقق منه:
```csharp
using Aspose.Pdf.Facades;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // استبدل بمسار المستند الفعلي الخاص بك
string filePath = Path.Combine(dataDir, "IsPasswordProtected.pdf");
```
**توضيح**:نحن نستخدم `Path.Combine` للتأكد من تنسيق مسار الملف بشكل صحيح عبر أنظمة التشغيل المختلفة.

#### الخطوة 2: تحميل مستند PDF

بعد ذلك، قم بتحميل ملف PDF المصدر باستخدام `PdfFileInfo` فصل:
```csharp
// تحميل معلومات مستند PDF
tPdfFileInfo fileInfo = new PdfFileInfo(filePath);
```
**توضيح**: ال `PdfFileInfo` توفر الفئة إمكانية الوصول إلى خصائص وطرق مختلفة متعلقة بملف PDF.

#### الخطوة 3: تحديد ما إذا كان ملف PDF مشفرًا

أخيرًا، تحقق مما إذا كان ملف PDF مشفرًا بكلمة مرور:
```csharp
// التحقق من التشفير
bool encrypted = fileInfo.IsEncrypted;
Console.WriteLine($"Is the PDF password protected? {encrypted}");
```
**توضيح**: ال `IsEncrypted` تعيد الخاصية قيمة منطقية تشير إلى ما إذا كانت المستند يتطلب كلمة مرور لفتحه.

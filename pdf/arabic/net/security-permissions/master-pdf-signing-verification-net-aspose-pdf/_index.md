## دليل التنفيذ

### آلية توقيع PDF

#### ملخص

يضمن توقيع مستند PDF صحته باستخدام شهادات رقمية. تُؤمّن هذه الميزة المستندات بتوقيع رقمي يُمكن التحقق منه لاحقًا.

#### خطوات التوقيع على مستند PDF

##### 1. جهّز بيئتك
تأكد من أن ملفات PDF المدخلة والشهادة الرقمية جاهزة.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/inFile.pdf");
PdfFileSignature pdfSign = new PdfFileSignature(doc);
```

##### 2. تحديد مظهر التوقيع

تعيين مظهر التوقيع باستخدام ملف صورة.

```csharp
Rectangle rect = new Rectangle(100, 100, 200, 100);
pdfSign.SignatureAppearance = dataDir + "/aspose-logo.jpg";
```

##### 3. إنشاء وتطبيق توقيع PKCS1

استخدم شهادة لإنشاء توقيع PKCS1 وتطبيقه على المستند.

```csharp
PKCS1 signature = new PKCS1(dataDir + "/inFile2.pdf", "password");
pdfSign.Sign(1, "Signature Reason", "Alice", "Odessa", true, rect, signature);
```

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من إمكانية الوصول إلى ملف الشهادة وأنه يحتوي على الأذونات الصحيحة.
- تأكد من تحديد مسارات الصورة بشكل صحيح.

### إضافة حقول التوقيع

#### ملخص

تتيح إضافة حقول التوقيع للمستخدمين التوقيع رقميًا على المستندات في مواقع محددة.

#### خطوات إضافة حقول التوقيع

##### 1. قم بتحميل مستند PDF الخاص بك
قم بتهيئة كائن FormEditor باستخدام مستندك.

```csharp
Document doc = new Document(dataDir + "/inFile.pdf");
FormEditor editor = new FormEditor(doc);
```

##### 2. تحديد حقول التوقيع وإضافتها
قم بتحديد موقع كل حقل توقيع على الصفحة المطلوبة.

```csharp
teditor.AddField(FieldType.Signature, "Signature from Alice", 1, 10, 10, 110, 110);
editor.AddField(FieldType.Signature, "Signature from John", 1, 120, 150, 220, 250);
editor.AddField(FieldType.Signature, "Signature from Smith", 1, 300, 200, 400, 300);
```

##### 3. احفظ المستند
استمرار التغييرات في الملف الجديد.

```csharp
teditor.Save(dataDir + "/AddSignatureFields_1_out.pdf");
```

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن إحداثيات الحقول تقع ضمن حدود الصفحة.
- تأكد من أن مستندك ليس محميًا بكلمة مرور أو مقفلاً ضد التحرير.

### التحقق من التوقيعات

#### ملخص
تضمن عملية التحقق أن التوقيعات الرقمية في ملف PDF صالحة ولم يتم العبث بها.

#### خطوات التحقق من التوقيعات

##### 1. تحميل المستند للتحقق
قم بتهيئة PdfFileSignature باستخدام الملف المستهدف.

```csharp
PdfFileSignature pdfVerify = new PdfFileSignature();
pdfVerify.BindPdf(dataDir + "/inFile.pdf");
```

##### 2. التحقق من صحة التوقيعات
حدد ما إذا كانت هناك أي توقيعات موجودة، ثم قم بالتحقق منها عن طريق الاسم أو الفهرس.

```csharp
bool isSigned = pdfVerify.ContainsSignature();
bool isSignatureVerified = pdfVerify.VerifySignature("Signature1");
bool isSignatureVerified2 = pdfVerify.VerifySignature("Signature from Alice");
```

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من عدم تغيير المستند بعد التوقيع.
- استخدم أسماء التوقيعات أو المؤشرات الصحيحة للتحقق.

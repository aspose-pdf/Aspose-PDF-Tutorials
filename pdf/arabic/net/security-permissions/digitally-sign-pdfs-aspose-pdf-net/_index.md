# كيفية التوقيع رقميًا على ملفات PDF باستخدام الطوابع الزمنية باستخدام Aspose.PDF .NET

في ظلّ العالم الرقميّ اليوم، يُعدّ ضمان صحة وسلامة المستندات أمرًا بالغ الأهمية. سواءً كنت تتعامل مع عقود قانونية أو تقارير رسمية، فإنّ التوقيع الرقميّ على ملفات PDF يُضيف طبقةً من الأمان تُؤكّد حقوق التأليف وتمنع التلاعب. كما تُعزّز إضافة الطوابع الزمنية إلى هذه التوقيعات هذه الضمانات بتوفير دليل على تاريخ توقيع المستند.

## دليل التنفيذ
سوف يرشدك هذا القسم خلال التوقيع الرقمي على ملف PDF باستخدام Aspose.PDF لـ .NET مع دمج إعدادات الطابع الزمني.

### إعداد مستندك

#### ملخص
سنبدأ بتحميل مستند PDF الذي ترغب في توقيعه وتهيئة الكائنات الأساسية مثل `PdfFileSignature`.

#### التنفيذ خطوة بخطوة

**1. قم بتحميل مستند PDF**
إنشاء مثيل لـ `Document` الفئة لتحميل ملف PDF الخاص بك:
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_SecuritySignatures();
using (Document document = new Document(dataDir + "DigitallySign.pdf"))
```

**2. تهيئة توقيع ملف Pdf**
استخدم `PdfFileSignature` فئة لإضافة التوقيعات الرقمية إلى ملف PDF.
```csharp
using (PdfFileSignature signature = new PdfFileSignature(document))
{
    // وسوف تتبع ذلك خطوات تنفيذية أخرى...
}
```

### إضافة توقيع رقمي

#### ملخص
تتم إضافة التوقيعات الرقمية باستخدام معيار PKCS#7، الذي يتضمن التوقيع المعتمد على الشهادات. ستحتاج إلى ملف PFX يحتوي على مفتاحك الخاص والشهادة المرتبطة به.

**3. إنشاء كائن PKCS#7**
تهيئة `PKCS7` الكائن مع مسار ملف PFX وكلمة المرور الخاصة بك:
```csharp
string pfxFile = "your_pfx_file.pfx";
PKCS7 pkcs = new PKCS7(pfxFile, "pfx_password");
```

**4. ضبط إعدادات الطابع الزمني**
قم بتكوين إعدادات الطابع الزمني لتشمل التوقيعات المختومة بالوقت.
```csharp
TimestampSettings timestampSettings = new TimestampSettings("https://your_timestamp_server"، "user:password"); // بيانات اعتماد اختيارية
pkcs.TimestampSettings = timestampSettings;
```

**5. تحديد تفاصيل التوقيع والمنطقة**
حدد المكان على صفحة PDF الذي تريد توقيعك فيه:
```csharp
System.Drawing.Rectangle rect = new System.Drawing.Rectangle(100, 100, 200, 100);
signature.Sign(1, "Signature Reason", "Contact", "Location", true, rect, pkcs);
```

### حفظ المستند الموقّع

**6. احفظ التغييرات**
بعد التوقيع، احفظ مستند PDF الخاص بك مع جميع التعديلات:
```csharp
signature.Save(dataDir + "DigitallySignWithTimeStamp_out.pdf");
```

# إنشاء وتعديل ملف PDF مع الصور باستخدام Aspose.PDF

## تعليمات التثبيت:  

لاستخدام Aspose.PDF لـ .NET في مشروعك، اتبع هذه الخطوات:  

1. تثبيت عن طريق NuGet Package Manager:  
افتح ويزويل ستوديو و اطلق الأمر التالي في وحدة التحكم في مدير الحزم:  
```powershell
Install-Package Aspose.PDF
```

2. إضافة مساحة أسماء في رمز C# الخاص بك:  
```csharp
using Aspose.Pdf;
```

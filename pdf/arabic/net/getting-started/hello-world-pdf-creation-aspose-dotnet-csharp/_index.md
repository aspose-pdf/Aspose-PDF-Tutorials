## دليل التنفيذ

### إنشاء نص وإضافته إلى صفحة

#### ملخص

جوهر هذا البرنامج التعليمي هو إنشاء مستند PDF بسيط وإضافة نص "Hello World!" إليه باستخدام Aspose.PDF لـ .NET.

#### الخطوة 1: تهيئة المستند

ابدأ بإنشاء مثيل لـ `Document`.

```csharp
// تهيئة كائن مستند جديد.
Document document = new Document();
```

#### الخطوة 2: إضافة صفحة

بعد ذلك، أضف صفحةً إلى مستندك. هنا ستضع نصك.

```csharp
// أضف صفحة فارغة إلى المستند.
Page page = document.Pages.Add();
```

#### الخطوة 3: إضافة نص إلى الصفحة

يستخدم `TextFragment` لإنشاء النص الخاص بك وإضافته إلى الصفحة المضافة حديثًا.

```csharp
// إنشاء جزء نصي يحتوي على "مرحبا بالعالم!"
page.Paragraphs.Add(new Aspose.Pdf.Text.TextFragment("Hello World!"));
```

#### الخطوة 4: احفظ مستندك

أخيرًا، احفظ مستندك. اختر موقعًا مناسبًا لملف الإخراج.

```csharp
// قم بتحديد دليل البيانات والمسار لحفظ ملف PDF.
string dataDir = RunExamples.GetDataDir_AsposePdf_QuickStart();

// احفظ المستند في المسار المحدد.
document.Save(dataDir + "HelloWorld_out.pdf");

Console.WriteLine("\nFile saved at " + dataDir);
```

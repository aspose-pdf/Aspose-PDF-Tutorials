## تعليمات التثبيت

1: الشروط المسبقة
* فيزيوال ستوديو 2017 أو أعلى
* .NET Framework 4.0 أو أعلى أو .NET Core 2.0+ / .NET 5.0+
* معرفة أساسية بالبرمجة C#

2: تثبيت Aspose.PDF لـ .NET
يمكنك تثبيت Aspose.PDF لـ .NET باستخدام إحدى الطرق التالية:

باستخدام وحدة التحكم في NuGet Package Manager:
```csharp
PM> Install-Package Aspose.PDF
```

3: باستخدام برنامج Visual Studio NuGet Package Manager:
	1. افتح مشروعك في Visual Studio
	2. انقر بزر الماوس الأيمن على المشروع في مستكشف الحلول واختر "إدارة حزم NuGet"
	3. ابحث عن Aspose.PDF وتثبيت الحزمة

# إنشاء PDF متعدد الأعمدة باستخدام Aspose.PDF لـ .NET

## تعليمات التثبيت  

لاستخدام Aspose.PDF لـ .NET في مشروعك، اتبع هذه الخطوات:  

1. تثبيت عن طريق NuGet (الموصى به)  
قم بتشغيل الأمر التالي في وحدة التحكم في حزم NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. تثبيت عبر .NET CLI  
لـ .NET Core أو .NET 5+ ، استخدم:  
```bash
dotnet add package Aspose.PDF
```

3. بعد التثبيت، استيراد المساحات الاسمية اللازمة في مشروعك:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

## تعليمات التثبيت

1. تثبيت عن طريق NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. تثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. التثبيت اليدوي:
   * قم بتنزيل أحدث إصدار من[Aspose.PDF للإصدارات .NET](https://releases.aspose.com/pdf/net/)
   * إضافة إشارة إلى "Aspose.PDF.dll" في مشروعك

## تعليمات التثبيت

1. تثبيت عن طريق NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. تثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. إضافة مرجع يدوياً:
   * قم بتنزيل المكتبة من[أسبوس إطلاق](https://releases.aspose.com/pdf/net/)
   * إضافة إشارة إلى "Aspose.PDF.dll" في مشروعك

4. إضافة بيان باستخدام:
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

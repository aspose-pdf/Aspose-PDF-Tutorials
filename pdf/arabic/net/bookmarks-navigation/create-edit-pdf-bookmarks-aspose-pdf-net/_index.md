## دليل التنفيذ

يرشدك هذا القسم خلال عملية إنشاء الإشارات المرجعية وتحريرها باستخدام Aspose.PDF لـ .NET.

### الميزة: إنشاء إشارات مرجعية وإضافة إشارات مرجعية فرعية

#### ملخص:
نظّم محتوى PDF بإضافة إشارات مرجعية منظمة، بما في ذلك الإشارات المرجعية الأصلية والفرعية. هذا يُسهّل التنقل داخل المستند.

**الخطوة 1:** إنشاء مثيل جديد من `Bookmarks` فصل
```csharp
Aspose.Pdf.Facades.Bookmarks bookmarks = new Aspose.Pdf.Facades.Bookmarks();
```
*لماذا؟* ال `Bookmarks` تساعد الفئة في إدارة مجموعات من كائنات الإشارة المرجعية.

**الخطوة 2:** إضافة إشارات مرجعية للأطفال
إنشاء وتكوين إشارات مرجعية فرعية مع أرقام الصفحات والعناوين.
```csharp
Bookmark childBookmark1 = new Bookmark { PageNumber = 1, Title = "First Child" };
Bookmark childBookmark2 = new Bookmark { PageNumber = 2, Title = "Second Child" };

bookmarks.Add(childBookmark1);
bookmarks.Add(childBookmark2);
```
*لماذا؟* ترتبط الإشارات المرجعية الفرعية بصفحة محددة وتوفر تنقلًا دقيقًا.

**الخطوة 3:** إنشاء إشارة مرجعية للوالدين
```csharp
Bookmark parentBookmark = new Bookmark { Action = "GoTo", PageNumber = 1, Title = "Parent" };
parentBookmark.ChildItems = bookmarks;
```
*لماذا؟* تقوم مجموعة الإشارات المرجعية الرئيسية بتجميع الإشارات المرجعية الفرعية تحت عنوان واحد.

### الميزة: تحرير الإشارات المرجعية في مستند PDF

#### ملخص:
تتيح لك هذه الميزة تعديل الإشارات المرجعية الموجودة داخل ملف PDF. 

**الخطوة 1:** تهيئة `PdfBookmarkEditor`
```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```
*لماذا؟* ال `PdfBookmarkEditor` توفر الفئة طرقًا لتحرير الإشارات المرجعية.

**الخطوة 2:** ربط مستند PDF المدخل
قم بربط مستند PDF المصدر الخاص بك بـ `bookmarkEditor`.
```csharp
bookmarkEditor.BindPdf("YOUR_DOCUMENT_DIRECTORY\\AddChildBookmark.pdf");
```

**الخطوة 3:** إضافة أو تعديل الإشارات المرجعية
أدخل الإشارة المرجعية الأصلية التي تم إنشاؤها مسبقًا.
```csharp
bookmarkEditor.CreateBookmarks(parentBookmark);
```

**الخطوة 4:** حفظ مستند PDF المحدث
```csharp
bookmarkEditor.Save("YOUR_OUTPUT_DIRECTORY\\AddChildBookmark_out.pdf");
```
*لماذا؟* تؤدي هذه الخطوة إلى حفظ كافة التعديلات في ملف جديد، مع الحفاظ على البيانات الأصلية.

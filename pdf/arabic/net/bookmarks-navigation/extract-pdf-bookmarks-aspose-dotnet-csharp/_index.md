### استخراج الإشارات المرجعية
#### ملخص
يتيح استخراج الإشارات المرجعية التنقل بكفاءة وإدارة أقسام مستند PDF. تُعد هذه الميزة مفيدة بشكل خاص للمستخدمين الذين يحتاجون إلى الوصول السريع إلى أجزاء محددة من المستندات الكبيرة.

#### خطوات التنفيذ
1. **إعداد مشروعك:**
   تأكد من أن مشروعك يتضمن مكتبة Aspose.PDF كما هو مذكور في قسم الإعداد.

2. **إنشاء فئة جديدة:**
   إنشاء فئة باسم `ExtractBookmarks`.

3. **اكتب رمز الاستخراج:**
```csharp
using System;
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Facades;

namespace Aspose.Pdf.Examples.CSharp.AsposePDFFacades.Bookmarks
{
    public class ExtractBookmarks
    {
        public static void Run()
        {
            // قم بتحديد المسار إلى ملف PDF الخاص بك.
            string dataDir = "path_to_your_directory/";
            
            // تهيئة كائن PdfBookmarkEditor
            PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
            
            // ربط مستند PDF
            bookmarkEditor.BindPdf(dataDir + "ExtractBookmarks.pdf");
            
            // استخراج الإشارات المرجعية من ملف PDF
            Aspose.Pdf.Facades.Bookmarks bookmarks = bookmarkEditor.ExtractBookmarks();

            foreach (Bookmark bookmark in bookmarks)
            {
                Console.WriteLine("Title: {0}", bookmark.Title);
                Console.WriteLine("Page Number: {0}", bookmark.PageNumber);
            }
        }
    }
}
```

**شرح المكونات الرئيسية:**
- **محرر إشارات مرجعية لملفات PDF:** تعتبر هذه الفئة محورية للتعامل مع الإشارات المرجعية داخل ملف PDF.
- **طريقة BindPdf:** يفتح ملف PDF المحدد، ويجهزه للمعالجة.
- **طريقة استخراج الإشارات المرجعية:** يسترجع جميع الإشارات المرجعية من ملف PDF. تحتوي كل إشارة مرجعية على تفاصيل مثل العنوان ورقم الصفحة.

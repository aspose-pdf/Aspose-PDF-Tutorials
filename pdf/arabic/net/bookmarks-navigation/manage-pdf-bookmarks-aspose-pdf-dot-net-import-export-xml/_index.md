## دليل التنفيذ

### استيراد الإشارات المرجعية من XML إلى مستند PDF

تتيح لك هذه الميزة ملء الإشارات المرجعية في مستند PDF باستخدام ملف XML موجود. اتبع الخطوات التالية:

#### الخطوة 1: إعداد مسارات الدليل والملفات

قم بتحديد المسارات لملف PDF المدخل، ومصدر الإشارة المرجعية XML، وملف PDF المخرج:
```csharp
double YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
string pdfFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ImportFromXML.pdf");
string xmlFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "bookmarks.xml");
string outputPdfPath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ImportFromXML_out.pdf");
```

#### الخطوة 2: تهيئة PdfBookmarkEditor

إنشاء مثيل لـ `PdfBookmarkEditor` للتلاعب بإشارات PDF المرجعية:
```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```

#### الخطوة 3: ربط الإشارات المرجعية واستيرادها

افتح ملف PDF المستهدف، ثم قم باستيراد الإشارات المرجعية من XML:
```csharp
bookmarkEditor.BindPdf(pdfFilePath);
bookmarkEditor.ImportBookmarksWithXML(xmlFilePath);
```
- **ربط PDF:** يفتح ملف PDF للتحرير.
- **استيراد الإشارات المرجعية مع XML:** استيراد الإشارات المرجعية المحددة في ملف XML.

#### الخطوة 4: حفظ ملف PDF المحدث

وأخيرًا، احفظ التغييرات لإنتاج ملف PDF الناتج:
```csharp
bookmarkEditor.Save(outputPdfPath);
```

### تصدير الإشارات المرجعية من مستند PDF إلى ملف XML

وعلى العكس من ذلك، يمكنك استخراج الإشارات المرجعية الموجودة إلى تنسيق XML:

#### الخطوة 1: إعداد مسارات الدليل والملفات

حدد المسارات لملف PDF المصدر وملف XML الوجهة:
```csharp
double YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
string pdfFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ExportToXML.pdf");
string xmlOutputPath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "bookmarks_exported.xml");
```

#### الخطوة 2: تهيئة PdfBookmarkEditor

مرة أخرى، قم بتهيئة `PdfBookmarkEditor`:
```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```

#### الخطوة 3: ربط الإشارات المرجعية وتصديرها

افتح ملف PDF الخاص بك وقم بتصدير إشاراته المرجعية إلى ملف XML:
```csharp
bookmarkEditor.BindPdf(pdfFilePath);
bookmarkEditor.ExportBookmarksToXML(xmlOutputPath);
```
- **تصدير الإشارات المرجعية إلى XML:** يقوم بتحويل الإشارات المرجعية وحفظها بتنسيق XML.

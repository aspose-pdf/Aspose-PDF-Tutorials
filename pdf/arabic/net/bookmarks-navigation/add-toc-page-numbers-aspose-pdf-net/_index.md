### دليل التنفيذ

بعد إعداد كل شيء، دعنا نضيف جدول محتويات مخصصًا مع أرقام الصفحات إلى مستند PDF الخاص بك.

#### تحميل وتعديل مستند PDF
أولاً، حمّل ملف PDF الحالي إلى تطبيقك. سنضيف بعد ذلك صفحة جديدة في بداية هذا الملف لجدول المحتويات.

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string inFile = "YOUR_DOCUMENT_DIRECTORY/42824.pdf";
string outFile = "YOUR_OUTPUT_DIRECTORY/42824_out.pdf";

// تحميل مستند PDF
document doc = new Document(inFile);

// إدراج صفحة جديدة في البداية لجدول المحتويات
Aspose.Pdf.Page tocPage = doc.Pages.Insert(1);
```

#### إنشاء معلومات جدول المحتويات وتكوينها
بعد ذلك، قم بتكوين جدول المحتويات الخاص بك عن طريق إعداد عنوانه وحجم الخط والنمط وأرقام الصفحات البادئة.

```csharp
// إعداد معلومات جدول المحتويات
tocInfo tocInfo = new TocInfo();
textFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20; // حجم الخط لعنوان جدول المحتويات
title.TextState.FontStyle = FontStyles.Bold; // أسلوب جريء للتأكيد
tocInfo.Title = title;
tocInfo.PageNumbersPrefix = "P"; // أرقام الصفحات تبدأ بـ 'P'
tocPage.TocInfo = tocInfo; // تعيين معلومات جدول المحتويات إلى الصفحة الجديدة
```

#### إضافة إدخالات إلى جدول المحتويات
الآن، قم بملء جدول المحتويات بالإدخالات لكل صفحة في المستند الخاص بك.

```csharp
for (int i = 1; i < doc.Pages.Count; i++) {
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
textSegment segment2 = new textSegment();

    // ربط العنوان بصفحة جدول المحتويات
    heading2.TocPage = tocPage;

    // إضافة جزء نصي إلى العنوان
    heading2.Segments.Add(segment2); 

    // تعيين صفحة الوجهة والموضع لكل إدخال
    heading2.DestinationPage = doc.Pages[i + 1];
    heading2.Top = doc.Pages[i + 1].Rect.Height; 
    segment2.Text = "P" + i.ToString(); // نص مخصص مع البادئة 'P'

    // أضف العنوان إلى صفحة جدول المحتويات
    tocPage.Paragraphs.Add(heading2);
}
```

#### حفظ مستند PDF المعدّل
وأخيرًا، احفظ مستندك المعدّل باستخدام جدول المحتويات المضاف حديثًا.

```csharp
doc.Save(outFile);
```

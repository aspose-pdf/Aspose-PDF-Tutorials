## دليل التنفيذ
الآن، دعنا ننفذ وظيفة تكبير الإشارة المرجعية في تطبيق .NET الخاص بك خطوة بخطوة.

### الخطوة 1: تحميل مستند PDF
أولاً، قم بتحميل مستند PDF الذي تريد تعيين الإشارات المرجعية فيه:
```csharp
// المسار إلى دليل المستندات.
string dataDir = RunExamples.GetDataDir_AsposePdf_Bookmarks();

// فتح المستند الموجود
Document doc = new Document(dataDir + "input.pdf");
```

### الخطوة 2: إنشاء وتكوين الإشارة المرجعية
بعد ذلك، قم بإنشاء إشارة مرجعية بمستوى تكبير مخصص باستخدام `XYZExplicitDestination`:
```csharp
// تهيئة عنصر الإشارة المرجعية باستخدام التكبير المخصص
OutlineItemCollection item = new OutlineItemCollection(doc.Outlines);

// ضبط التكبير إلى 0 (عرض الصفحة الكاملة) عند إحداثيات محددة
XYZExplicitDestination dest = new XYZExplicitDestination(2, 100, 100, 0);
item.Action = new GoToAction(dest);

// أضف الإشارة المرجعية إلى مجموعة مخططات المستند
doc.Outlines.Add(item);
```

### الخطوة 3: حفظ المستند المحدث
وأخيرًا، احفظ التغييرات مرة أخرى في ملف PDF:
```csharp
dataDir += "InheritZoom_out.pdf";

// احفظ ملف PDF المعدّل مع الإشارات المرجعية المحدثة
doc.Save(dataDir);

Console.WriteLine("\nBookmarks updated successfully.\nFile saved at " + dataDir);
```

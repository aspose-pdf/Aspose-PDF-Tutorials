## دليل التنفيذ: إضافة روابط محلية في ملفات PDF

### نظرة عامة على إنشاء الرابط المحلي

تتضمن إضافة رابط محلي تحديد منطقة داخل المستند كنقطة اتصال قابلة للنقر، مما يتيح للمستخدمين الانتقال إلى أقسام مختلفة دون الحاجة إلى التمرير اليدوي.

#### التنفيذ خطوة بخطوة

**1. افتح مستند PDF المستهدف**

```csharp
using Aspose.Pdf.Facades;
class CreateLocalLink {
    static void Main() {
        using (PdfContentEditor contentEditor = new PdfContentEditor()) {
            // تهيئة ملف PDF المستهدف وربطه
            contentEditor.BindPdf("YOUR_DOCUMENT_DIRECTORY\CreateApplicationLink.pdf");
        }
    }
}
```

**2. قم بتحديد منطقة مستطيلة للرابط**

حدد المكان داخل المستند الذي سيكون فيه الرابط نشطًا:

```csharp
Rectangle rectangle = new Rectangle(100, 100, 200, 200);
```

**3. إنشاء رابط محلي في المستطيل المحدد**

أضف رابطًا داخليًا باستخدام `CreateLocalLink` طريقة:

```csharp
contentEditor.CreateLocalLink(rectangle, 3, 1, Color.Red);
```

- **المستطيل**:المنطقة القابلة للنقر للرابط.
- **رقم الصفحة (3)**:صفحة الوجهة عند النقر على الرابط.
- **معرف الطبقة (1)**:إدارة طبقات التعليقات التوضيحية.
- **اللون (أحمر)**:إشارة مرئية لوجود الرابط.

**4. احفظ ملف PDF باستخدام الرابط المحلي الذي تم إنشاؤه حديثًا**

```csharp
contentEditor.Save("YOUR_OUTPUT_DIRECTORY\CreateLocalLink_out.pdf");
```

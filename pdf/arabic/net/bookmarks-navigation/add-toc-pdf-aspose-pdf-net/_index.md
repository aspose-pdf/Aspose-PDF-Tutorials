## دليل التنفيذ

### إضافة جدول المحتويات
تتضمن إضافة جدول محتويات تحميل ملف PDF موجود، وإنشاء صفحة جدول محتويات جديدة، وربطها بعناوين من صفحات أخرى. إليك الطريقة:

#### الخطوة 1: تحميل مستند PDF الخاص بك
```csharp
// تهيئة كائن المستند
document doc = new Document("YOUR_DOCUMENT_DIRECTORY\AddTOC.pdf");
```
قم بتحميل ملف PDF الحالي الخاص بك إلى Aspose.PDF `Document` كائن للتلاعب.

#### الخطوة 2: إنشاء صفحة جدول المحتويات
```csharp
// قم بإدراج صفحة جديدة في البداية لتكون بمثابة جدول المحتويات
Page tocPage = doc.Pages.Insert(1);
```
قم بإدراج صفحة جديدة في بداية ملف PDF الخاص بك لتضمين جدول المحتويات، مما يتيح لك سهولة الوصول إليه من أي مكان في المستند.

#### الخطوة 3: إعداد معلومات جدول المحتويات
```csharp
// تكوين عنوان جدول المحتويات ومظهره
tocInfo tocInfo = new TocInfo();
TextFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20;
title.TextState.FontStyle = FontStyles.Bold;
tocInfo.Title = title;
tocPage.TocInfo = tocInfo;
```
قم بتحديد مظهر ومحتوى جدول المحتويات باستخدام عنوان واضح وجريء.

#### الخطوة 4: تحديد العناوين وإضافتها
```csharp
// مجموعة من العناوين لإضافتها كعناصر جدول المحتويات
string[] titles = { "First page", "Second page", "Third page", "Fourth page" };

for (int i = 0; i < 2; i++)
{
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
    TextSegment segment2 = new TextSegment();
    heading2.TocPage = tocPage;
    heading2.Segments.Add(segment2);

    // ربط كل مدخل في جدول المحتويات بالصفحة المخصصة له
    heading2.DestinationPage = doc.Pages[i + 2];
    heading2.Top = doc.Pages[i + 2].Rect.Height;
    segment2.Text = titles[i];

    tocPage.Paragraphs.Add(heading2);
}
```
قم بالتنقل عبر العناوين المطلوبة، مما يؤدي إلى إنشاء `Heading` لكل عنصر. اضبط صفحة الوجهة وموقعها لضمان التنقل المباشر.

#### الخطوة 5: حفظ المستند
```csharp
// إخراج المستند المعدل مع جدول المحتويات
string outputFilePath = "YOUR_OUTPUT_DIRECTORY\TOC_out.pdf";
doc.Save(outputFilePath);
```
احفظ ملف PDF المحدث الخاص بك مع جدول المحتويات المدمج في البداية.

## دليل التنفيذ
يتناول هذا القسم بالتفصيل عملية إضافة إشارة مرجعية فرعية إلى مستند PDF موجود.

### ملخص
تتضمن إضافة إشارة مرجعية فرعية إنشاء هيكل هرمي داخل ملف PDF الخاص بك، مما يعزز التنقل من خلال تجميع الأقسام ذات الصلة أسفل الإشارات المرجعية الرئيسية.

### دليل خطوة بخطوة
#### **1. إعداد مسارات الإدخال والإخراج**
أولاً، قم بتحديد المسارات لملف PDF المدخل وموقع الإخراج:
```csharp
string inputDir = "YOUR_DOCUMENT_DIRECTORY" + "/AddChildBookmark.pdf";
string outputDir = "YOUR_OUTPUT_DIRECTORY" + "/AddChildBookmark_out.pdf";
```
#### **2. قم بتحميل مستند PDF الخاص بك**
افتح مستند PDF موجود باستخدام Aspose.PDF:
```csharp
Document pdfDocument = new Document(inputDir);
```
#### **3. إنشاء إشارة مرجعية للوالدين**
إنشاء وتصميم كائن إشارة مرجعية رئيسي ليبرز في قائمة الإشارات المرجعية:
```csharp
OutlineItemCollection parentOutline = new OutlineItemCollection(pdfDocument.Outlines);
parentOutline.Title = "Parent Outline";
parentOutline.Italic = true;
parentOutline.Bold = true;
```
*لماذا هذه الخطوة؟*:يساعد التصميم على التمييز بين الأقسام الأساسية، مما يجعل التنقل سهلاً.
#### **4. إنشاء إشارة مرجعية للأطفال**
إنشاء إشارة مرجعية للأطفال بتصميمها الخاص:
```csharp
OutlineItemCollection childOutline = new OutlineItemCollection(pdfDocument.Outlines);
childOutline.Title = "Child Outline";
childOutline.Italic = true;
childOutline.Bold = true;
```
*لماذا هذه الخطوة؟*:توفر الإشارات المرجعية الفرعية إمكانية الوصول المباشر إلى المحتوى المتداخل، مما يؤدي إلى تحسين تجربة المستخدم.
#### **5. أضف إشارة مرجعية للطفل**
قم بإرفاق إشارة مرجعية فرعية بإشارة مرجعية أصلية:
```csharp
parentOutline.Add(childOutline);
```
*لماذا هذه الخطوة؟*:إن إنشاء هذه العلاقة ينظم بنية مستندك منطقيًا.
#### **6. احفظ مستندك**
احفظ ملف PDF المحدث مع الإشارات المرجعية الجديدة المضافة:
```csharp
pdfDocument.Save(outputDir);
```

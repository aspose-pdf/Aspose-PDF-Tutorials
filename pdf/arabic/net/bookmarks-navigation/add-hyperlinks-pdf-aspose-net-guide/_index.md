# كيفية إضافة ارتباطات تشعبية إلى ملفات PDF باستخدام Aspose.PDF لـ .NET: دليل شامل

- كيفية تثبيت Aspose.PDF لـ .NET.

- تنفيذ تعليقات نصية مجانية تعمل كروابط تشعبية.

## دليل التنفيذ
دعنا نقسم التنفيذ إلى ميزتين رئيسيتين: إضافة تعليقات الارتباط التشعبي وتعليقات النص الحر مع الروابط التشعبية.

### إضافة تعليق ارتباط تشعبي إلى ملف PDF
#### ملخص
تعرف على كيفية إضافة تعليق توضيحي لارتباط تشعبي قابل للنقر باستخدام Aspose.PDF لـ .NET، وهو أمر مفيد لتوجيه المستخدمين إلى مواقع ويب خارجية أو أقسام مستندات داخلية.

#### خطوات التنفيذ
**1. افتح المستند**
ابدأ بتحميل مستند PDF الحالي لديك:
```csharp
Document document = new Document("YOUR_DOCUMENT_DIRECTORY\\AddHyperlink.pdf");
```

**2. إنشاء كائن تعليق الارتباط**
حدد الصفحة والمنطقة التي تريد ظهور الارتباط التشعبي فيها:
```csharp
Page page = document.Pages[1];
LinkAnnotation link = new LinkAnnotation(page, new Aspose.Pdf.Rectangle(100, 100, 300, 300));
```

**3. تكوين الحدود والإجراء**
اضبط عرض الحدود على الصفر للحصول على مظهر سلس وحدد إجراء الارتباط التشعبي:
```csharp
Border border = new Border(link) { Width = 0 };
link.Border = border;
link.Action = new GoToURIAction("http://www.aspose.com");
```

**4. إضافة تعليق توضيحي إلى الصفحة**
دمج تعليق الرابط في مجموعة التعليقات التوضيحية للمستند الخاص بك:
```csharp
page.Annotations.Add(link);
```

**5. حفظ المستند**
أخيرًا، احفظ ملف PDF المحدث مع إضافة الرابط التشعبي:
```csharp
document.Save("YOUR_OUTPUT_DIRECTORY\\AddHyperlink_out.pdf");
```

### إضافة تعليق نصي مجاني مع ارتباط تشعبي إلى ملف PDF
#### ملخص
بعد ذلك، استكشف إضافة تعليق نصي مجاني يعمل كارتباط تشعبي، مما يوفر للمستخدمين محتوى نصيًا وإمكانيات تفاعلية.

#### خطوات التنفيذ
**1. افتح المستند**
قم بتحميل مستندك بنفس الطريقة السابقة:
```csharp
Document document = new Document("YOUR_DOCUMENT_DIRECTORY\\AddHyperlink.pdf");
Page page = document.Pages[1];
```

**2. إنشاء تعليقات توضيحية مجانية للنص**
قم بتحديد مظهر التعليق النصي المجاني الخاص بك:
```csharp
FreeTextAnnotation textAnnotation = new FreeTextAnnotation(page, 
    new Aspose.Pdf.Rectangle(100, 100, 300, 300), 
    new DefaultAppearance(Aspose.Pdf.Text.FontRepository.FindFont("TimesNewRoman"), 10, System.Drawing.Color.Blue));
```

**3. تعيين محتوى التعليقات التوضيحية والحدود**
تعيين محتوى لتعليقك وتكوين حدوده:
```csharp
textAnnotation.Contents = "Link to Aspose website";
textAnnotation.Border = new Border(textAnnotation) { Width = 0 };
```

**4. إضافة تعليقات توضيحية مجانية إلى الصفحة**
أضف التعليق النصي المجاني إلى مجموعة التعليقات التوضيحية للصفحة:
```csharp
page.Annotations.Add(textAnnotation);
```

**5. حفظ المستند**
احفظ مستندك باستخدام رابط النص المجاني المضاف حديثًا:
```csharp
document.Save("YOUR_OUTPUT_DIRECTORY\\AddHyperlink_out.pdf");
```

## دليل التنفيذ
يرشدك هذا القسم خلال عملية إضافة ارتباطات تشعبية محلية إلى مستند PDF باستخدام Aspose.PDF لـ .NET.

### إنشاء وتكوين الارتباطات التشعبية المحلية
تتيح الارتباطات التشعبية المحلية للمستخدمين التنقل داخل المستند نفسه بسلاسة.

#### الخطوة 1: أضف أول رابط محلي لك
إنشاء نص يعمل كارتباط تشعبي:
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // حدد الدليل الخاص بك
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // دليل الإخراج

// إنشاء مثيل مستند جديد
document doc = new Document();
Page page = doc.Pages.Add();

// الخطوة 1: إضافة أول رابط تشعبي محلي
textFragment text1 = new TextFragment("link to page 7");
hyperlink link1 = new LocalHyperlink();
link1.TargetPageNumber = 7; // تعيين رقم الصفحة المستهدفة للتنقل
text1.Hyperlink = link1;
doc.Pages[page.Number].Paragraphs.Add(text1);
```

#### الخطوة 2: إضافة ارتباط تشعبي محلي آخر
أضف ارتباطًا تشعبيًا آخر على صفحة جديدة داخل المستند:
```csharp
// الخطوة 2: إضافة الارتباط التشعبي المحلي الثاني
textFragment text2 = new TextFragment("link to page 1");
text2.IsInNewPage = true; // تأكد من ظهور هذا الرابط على صفحة جديدة
hyperlink link2 = new LocalHyperlink();
link2.TargetPageNumber = 1;
text2.Hyperlink = link2;
doc.Pages.Add().Paragraphs.Add(text2);
```

### حفظ مستندك
احفظ مستندك مع الروابط التشعبية المضمنة:
```csharp
string outputFilePath = Path.Combine(outputDir, "CreateLocalHyperlink_out.pdf");
doc.Save(outputFilePath);
```

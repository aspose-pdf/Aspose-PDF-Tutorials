### دليل التنفيذ:
دعنا نقسم التنفيذ إلى ميزتين رئيسيتين: توسيع الإشارات المرجعية وتعيين وضع عرض الصفحة.

#### الميزة 1: توسيع جميع الإشارات المرجعية في ملفات PDF
تتيح لك هذه الميزة فتح جميع الإشارات المرجعية داخل مستند PDF، مما يعزز التنقل.

**الخطوة 1:** قم بتحميل مستند PDF الخاص بك
ابدأ بتحميل ملف PDF الخاص بك باستخدام Aspose.PDF `Document` تأكد من تحديد المسار الصحيح لملف PDF المدخل.
```csharp
using System;
using Aspose.Pdf;

// قم بتحميل مستند PDF من الدليل المحدد.
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input-bookmark.pdf");
```

**الخطوة 2:** تعيين وضع عرض الصفحة
تأكد من أن الإشارات المرجعية مرئية عن طريق ضبط وضع عرض الصفحة على `UseOutlines`.
```csharp
// تمكين رؤية الخطوط العريضة (الإشارات المرجعية) في عارض PDF.
doc.PageMode = PageMode.UseOutlines;
```

**الخطوة 3:** فتح جميع الإشارات المرجعية
كرر كل إشارة مرجعية وافتحها. عدّل `OutlineItemCollection` الأشياء على النحو التالي:
```csharp
// قم بالمرور على جميع الإشارات المرجعية وضبطها على الفتح.
foreach (OutlineItemCollection item in doc.Outlines)
{
    // قم بوضع علامة على الإشارة المرجعية الحالية على أنها مفتوحة.
    item.Open = true;
}
```

**الخطوة 4:** حفظ المستند المعدل
احفظ مستندك مع الإشارات المرجعية الموسعة في ملف جديد.
```csharp
// احفظ ملف PDF مع الإشارات المرجعية الموسعة في دليل الإخراج.
doc.Save("YOUR_OUTPUT_DIRECTORY/ExpandBookmarks_out.pdf");
```

#### الميزة 2: ضبط وضع عرض الصفحة
قم بضبط وضع عرض الصفحة في ملف PDF لعرض الإشارات المرجعية بشكل افتراضي.

**الخطوة 1:** قم بتحميل مستند PDF الخاص بك
كما في السابق، ابدأ بتحميل مستندك. هذه الخطوة مشابهة لما قمنا به في الميزة السابقة.
```csharp
using Aspose.Pdf;

// افتح مستند PDF من الدليل المحدد.
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input-bookmark.pdf");
```

**الخطوة 2:** تكوين وضع عرض الصفحة
ضبط وضع الصفحة إلى `UseOutlines` مرة أخرى، التأكد من عرض الإشارات المرجعية عند فتح الملف.
```csharp
// ضبط وضع عرض الصفحة لإظهار الخطوط العريضة (الإشارات المرجعية).
doc.PageMode = PageMode.UseOutlines;
```

**الخطوة 3:** احفظ التغييرات
احفظ ملف PDF الخاص بك بإعدادات الصفحة المحدثة.
```csharp
// احفظ المستند المعدل.
doc.Save("YOUR_OUTPUT_DIRECTORY/BookmarksPageView_out.pdf");
```

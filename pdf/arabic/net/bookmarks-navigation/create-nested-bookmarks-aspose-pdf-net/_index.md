## دليل التنفيذ

بعد أن أعددنا بيئتنا، لننتقل إلى إنشاء إشارات مرجعية متداخلة. سنُقسّم كل ميزة إلى خطوات سهلة.

### إنشاء ملف PDF مع إشارات مرجعية متداخلة

#### ملخص
سيرشدك هذا القسم خلال إنشاء مستند PDF مع إشارات مرجعية هرمية باستخدام Aspose.PDF لـ .NET.

#### الخطوة 1: إعداد مشروعك
قم بإنشاء تطبيق وحدة تحكم C# جديد وتأكد من `Aspose.Pdf` تمت الإشارة إلى المكتبة في مشروعك.

#### الخطوة 2: تحديد الإشارات المرجعية
يتم تمثيل العلامات المرجعية في Aspose.PDF بواسطة `Bookmark` سوف نقوم بإنشاء إشارات مرجعية متداخلة باستخدام العناصر الفرعية.

```csharp
using System.IO;
using Aspose.Pdf.Facades;

public static void CreateNestedBookmarks()
{
    string dataDir = RunExamples.GetDataDir_AsposePdfFacades_TechnicalArticles();
    PdfContentEditor editor = new PdfContentEditor();

    // ربط مستند PDF
    editor.BindPdf(dataDir + "inFile.pdf");

    // تحديد الإشارات المرجعية للأطفال
    Bookmark bm11 = new Bookmark { Action = "GoTo", PageNumber = 3, Title = "Section - 1.1.1" };
    Bookmark bm1 = new Bookmark { Action = "GoTo", PageNumber = 2, Title = "Section - 1.1" };

    // إنشاء مجموعة لإشارات مرجعية للأطفال
    Aspose.Pdf.Facades.Bookmarks bms1 = new Aspose.Pdf.Facades.Bookmarks();
    bms1.Add(bm11);
    bm1.ChildItems = bms1;

    // تحديد إشارة مرجعية للفصل الرئيسي
    Bookmark bm = new Bookmark { Action = "GoTo", PageNumber = 1, Title = "Chapter - 1" };
    Aspose.Pdf.Facades.Bookmarks bms = new Aspose.Pdf.Facades.Bookmarks();
    bms.Add(bm1);

    // إضافة إشارات مرجعية إضافية للأطفال
    Bookmark bm2 = new Bookmark { Action = "GoTo", PageNumber = 4, Title = "Section - 1.2" };
    bms.Add(bm2);
    
    // تعيين المجموعة الكاملة إلى الإشارة المرجعية الأصلية
    bm.ChildItems = bms;

    // تعريف فصل آخر بإشارة مرجعية فرعية
    Bookmark bm_parent2 = new Bookmark { Action = "GoTo", PageNumber = 5, Title = "Chapter - 2" };
    Bookmark bm22 = new Bookmark { Action = "GoTo", PageNumber = 6, Title = "Section - 2.1" };

    Aspose.Pdf.Facades.Bookmarks bms_parent2 = new Aspose.Pdf.Facades.Bookmarks();
    bms_parent2.Add(bm22);
    bm_parent2.ChildItems = bms_parent2;

    // حفظ المستند باستخدام الإشارات المرجعية المتداخلة
    editor.Save(dataDir + "Nested_BookMarks_out.pdf");
}
```

#### شرح المعلمات والطرق
- **فعل**: يُحدد نوع الإجراء المرتبط بالإشارة المرجعية. هنا، يتم ضبطه على `"GoTo"` للملاحة.
- **رقم الصفحة**: يشير إلى رقم صفحة الوجهة التي يرتبط بها الإشارة المرجعية.
- **عنوان**:اسم العرض للإشارة المرجعية.

### نصائح استكشاف الأخطاء وإصلاحها

1. **أرقام الصفحات غير صحيحة**:تأكد من `PageNumber` القيم صحيحة وتتطابق مع الصفحات الموجودة في ملف PDF.
2. **قضايا الادخار**:تحقق من مسارات الملفات والأذونات عند حفظ ملف PDF الناتج.

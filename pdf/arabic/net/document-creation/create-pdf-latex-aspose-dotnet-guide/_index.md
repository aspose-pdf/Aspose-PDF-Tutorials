## دليل التنفيذ
الآن، دعنا ننتقل إلى كل خطوة لإنشاء ملف PDF الخاص بنا باستخدام نصوص LaTeX.

### الخطوة 1: إنشاء كائن مستند جديد
ابدأ بالتهيئة `Document` هذا الكائن بمثابة حاوية لصفحاتك ومحتواك.

```csharp
// تهيئة مستند جديد
Document document = new Document();
```

### الخطوة 2: إضافة صفحة إلى مجموعة صفحات المستند
أضف صفحة إلى مستندك حيث ستضع المحتوى الخاص بك.

```csharp
// إضافة صفحة جديدة
Page page = document.Pages.Add();
```

### الخطوة 3: إنشاء جدول وإضافته إلى الصفحة
سيساعد هيكل الجدول في تنظيم المحتوى. هنا، سنضيفه مباشرةً إلى صفحتنا.

```csharp
// تهيئة كائن جدول جديد
Table table = new Table();
page.Paragraphs.Add(table);
```

### الخطوة 4: إضافة صف إلى الجدول
بعد ذلك، قم بإدراج صف في الجدول حيث يمكنك وضع الخلايا التي تحتوي على المحتوى.

```csharp
// إضافة صف إلى الجدول
Row row = table.Rows.Add();
```

### الخطوة 5: تعريف نص LaTeX للتعبير الرياضي
جهّز نص LaTeX الخاص بك. هنا يمكنك تحديد التعبير الرياضي الذي تريد عرضه.

```csharp
// تعبير LaTeX للجمع والجذر التربيعي والتكامل
string latexText1 = "$123456789+\\sqrt{1}+\\int_a^b f(x)dx$";
```

### الخطوة 6: إنشاء خلية وإضافة جزء LaTeX
أنشئ خلية في صفك لحفظ نص LaTeX. اضبط هوامشها لتحسين قابلية القراءة.

```csharp
// إضافة خلية جديدة بهامش محدد
Cell cell = row.Cells.Add();
cell.Margin = new MarginInfo { Left = 20, Right = 20, Top = 20, Bottom = 20 };

// إنشاء وإضافة جزء LaTeX
TeXFragment ltext1 = new TeXFragment(latexText1, true);
cell.Paragraphs.Add(ltext1);
```

### الخطوة 7: حفظ المستند
أخيرًا، احفظ مستندك في المجلد المحدد. تأكد من حصولك على أذونات الكتابة في هذا الموقع.

```csharp
// حدد دليل الإخراج واحفظ الملف
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // حدد مسار دليل الإخراج الخاص بك
document.Save(Path.Combine(outputDir, "LatextScriptInPdf_out.pdf"));
```

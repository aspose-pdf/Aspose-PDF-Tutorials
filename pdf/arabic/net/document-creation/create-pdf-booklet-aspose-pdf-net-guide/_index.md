## دليل التنفيذ

في هذا القسم، سنشرح كيفية إنشاء كتيب PDF باستخدام التدفقات.

### إنشاء الكتيب

#### ملخص
تتيح لك هذه الميزة أخذ ملف PDF موجود وإعادة تنظيم صفحاته إلى تنسيق كتيب بكفاءة.

#### التنفيذ خطوة بخطوة

**1. إعداد المسارات وإنشاء تدفقات الملفات**
```csharp
// تحديد مسارات لملفات الإدخال والإخراج
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Pages();
FileStream inputStream = new FileStream(dataDir + "input.pdf", FileMode.Open);
FileStream outputStream = new FileStream(dataDir + "MakeBookletUsingStreams_out.pdf", FileMode.Create);
```

**2. تهيئة محرر ملفات Pdf**
```csharp
// إنشاء مثيل لـ PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**3. إنشاء الكتيب**
```csharp
// استخدم طريقة MakeBooklet لإنشاء كتيب
pdfEditor.MakeBooklet(inputStream, outputStream);
```

#### خيارات تكوين المفاتيح
- **نطاق الصفحات**:تخصيص الصفحات التي سيتم تضمينها.
- **تنسيق الإخراج**:قم بتحديد تنسيق ملف الإخراج الخاص بك.

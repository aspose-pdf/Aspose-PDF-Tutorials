## دليل التنفيذ
### إعداد المستندات وتكوينها
الخطوة الأولى في إنشاء مستند PDF هي إعداد تخطيطه، بما في ذلك أبعاد الصفحة والهوامش.

#### تهيئة المستند
```csharp
using Aspose.Pdf;

public static void CreateDocumentWithPageSettings()
{
    // تهيئة مثيل مستند جديد
    Document pdfDoc = new Document();

    // تعيين عرض وارتفاع صفحات المستند بالنقاط (1 بوصة = 72 نقطة)
    pdfDoc.PageInfo.Width = 612.0;  // 8.5 بوصة
    pdfDoc.PageInfo.Height = 792.0; // 11 بوصة

    // تحديد هوامش موحدة لجميع الجوانب
    pdfDoc.PageInfo.Margin = new MarginInfo
    {
        Left = 72,
        Right = 72,
        Top = 72,
        Bottom = 72
    };

    // أضف صفحة إلى المستند، مع وراثة هذه الإعدادات
    Page pdfPage = pdfDoc.Pages.Add();
}
```
يُهيئ هذا المقتطف الشفري مستند PDF بأبعاد محددة وهوامش موحدة لجميع الصفحات. بتعيين هذه الخصائص، تضمن تنسيقًا متسقًا للمحتوى في جميع أنحاء المستند.

### إعداد الصندوق العائم والعنوان
بعد ذلك، سنستكشف إضافة مربعات عائمة - وهي أداة متعددة الاستخدامات لتنظيم النص - وتكوين العناوين داخلها.

#### إضافة صندوق عائم
```csharp
using Aspose.Pdf;

public static void AddFloatingBoxWithHeadings(Page pdfPage)
{
    // إنشاء مثيل FloatingBox جديد لاحتواء عناصر النص
    FloatingBox floatBox = new FloatingBox
    {
        Margin = pdfPage.PageInfo.Margin  // مطابقة هوامش الصفحة
    };

    // أضف المربع العائم إلى مجموعة فقرات الصفحة
    pdfPage.Paragraphs.Add(floatBox);

    // تكوين وإضافة عنوان بأسلوب الترقيم
    Heading heading1 = new Heading(1)
    {
        IsInList = true,
        StartNumber = 1,
        Text = "List 1",
        Style = NumberingStyle.NumeralsRomanLowercase,
        IsAutoSequence = true
    };

    floatBox.Paragraphs.Add(heading1);

    // أضف عنوانًا آخر برقم بداية ونمط مختلفين
    Heading heading2 = new Heading(1)
    {
        IsInList = true,
        StartNumber = 13,
        Text = "List 2",
        Style = NumberingStyle.NumeralsRomanLowercase,
        IsAutoSequence = true
    };

    floatBox.Paragraphs.Add(heading2);

    // أضف عنوانًا فرعيًا بأسلوب ترقيم الأحرف الصغيرة
    Heading heading3 = new Heading(2)
    {
        IsInList = true,
        StartNumber = 1,
        Text = "The value, as of the effective date of the plan, of property to be distributed under the plan on account of each allowed",
        Style = NumberingStyle.LettersLowercase,
        IsAutoSequence = true
    };

    floatBox.Paragraphs.Add(heading3);
}
```
توضح هذه الميزة كيفية إنشاء مربع عائم وإضافة عناوين متعددة بأنماط ترقيم مختلفة. تُفيد المربعات العائمة في تجميع المحتوى منطقيًا، `IsInList` تضمن الخاصية أن العناوين تتبع تسلسلًا منظمًا.

### حفظ المستند
أخيرًا، نحتاج إلى حفظ مستند PDF الذي قمنا بتكوينه في دليل محدد.

#### حفظ المستند
```csharp
using Aspose.Pdf;

public static void SaveDocument(Document pdfDoc)
{
    // قم بتحديد مسار دليل الإخراج (استبدله بالمسار الفعلي الخاص بك)
    string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/ApplyNumberStyle_out.pdf";

    // حفظ المستند في المسار المحدد
    pdfDoc.Save(dataDir);
}
```
تقوم هذه الوظيفة بحفظ ملف PDF في موقع محدد، مما يضمن تخزين مستندك بشكل آمن وإمكانية الوصول إليه عند الحاجة.

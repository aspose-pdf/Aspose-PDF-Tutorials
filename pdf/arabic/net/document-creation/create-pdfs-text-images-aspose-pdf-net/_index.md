## دليل التنفيذ

يقوم هذا القسم بتقسيم عملية إنشاء ملفات PDF تحتوي على نصوص وصور باستخدام Aspose.PDF لـ .NET إلى خطوات قابلة للإدارة.

### إنشاء مستند يحتوي على نص

**ملخص:**
إنشاء مستند PDF بسيط بإضافة نص أمر سهل. تتيح لك هذه الميزة إنشاء مستندات برمجيًا، ويمكن تخصيصها لتشمل أنماطًا وتنسيقات محددة.

#### التنفيذ خطوة بخطوة:
1. **تهيئة المستند**
   ابدأ بإنشاء مثيل لـ `Document` فصل.
   ```csharp
   Document doc = new Document();
   ```
2. **إضافة صفحة**
   سيتم إضافة الصفحات إلى المستند حيث ستضع المحتوى الخاص بك.
   ```csharp
   Page page = doc.getPages().add();
   ```
3. **إنشاء وتكوين جزء نصي**
   هنا، نقوم بإنشاء `TextFragment` الذي يحتوي على النص المطلوب، إلى جانب إعدادات التصميم المحددة مثل اللون والحجم.
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;

   TextFragment t1 = new TextFragment("paragraph 3 segment");
   t1.getTextState().setForegroundColor(Color.RED);
   t1.setTextIsInLineParagraph(true);
   t1.getTextState().setFontSize(12);
   ```
4. **أضف النص إلى الصفحة**
   وأخيرًا، أضف جزءًا من النص إلى مجموعة فقرات الصفحة.
   ```csharp
   page.getParagraphs().add(t1);
   doc.save("YOUR_DOCUMENT_DIRECTORY/CreateTextPDF_out.pdf");
   ```

### إنشاء صندوق عائم يحتوي على نص

**ملخص:**
توفر المربعات العائمة طريقة متقدمة لتنظيم المحتوى داخل ملف PDF، مما يسمح بتخطيطات ديناميكية.

#### التنفيذ خطوة بخطوة:
1. **إنشاء الصندوق العائم**
   تعريف أ `FloatingBox` بأبعاد محددة.
   ```csharp
   FloatingBox textFloatingBox = new FloatingBox(117, 21);
   ```
2. **تكوين الموضع ومؤشر Z**
   تخصيص موقعه على الصفحة.
   ```csharp
   textFloatingBox.setZIndex(1);
   textFloatingBox.setLeft(-4);
   textFloatingBox.setTop(-4);
   ```
3. **إضافة مربع عائم إلى الصفحة**
   ضعها داخل بنية المستند.
   ```csharp
   page.getParagraphs().add(textFloatingBox);
   textFloatingBox.getParagraphs().add(t1); // إضافة جزء نص تم إنشاؤه مسبقًا
   doc.save("YOUR_DOCUMENT_DIRECTORY/TextFloatingBox_out.pdf");
   ```

### إضافة صورة إلى FloatingBox

**ملخص:**
يتيح دمج الصور داخل المربعات العائمة خيارات تصميم وتخطيط مرنة.

#### التنفيذ خطوة بخطوة:
1. **تهيئة الصورة**
   قم بإعداد كائن صورتك باستخدام مسار الملف.
   ```csharp
   using Aspose.Pdf;

   Image image = new Image();
   image.setFile("YOUR_DOCUMENT_DIRECTORY/test_image.png");
   ```
2. **إنشاء وتكوين صندوق عائم للصورة**
   على غرار النص، قم بتعريف مربع عائم ولكنه مصمم خصيصًا للصورة.
   ```csharp
   FloatingBox imageFloatingBox = new FloatingBox(117, 21);
   page.getParagraphs().add(imageFloatingBox);

   imageFloatingBox.setLeft(-4);
   imageFloatingBox.setTop(-4);
   imageFloatingBox.setZIndex(2);
   ```
3. **أضف الصورة إلى الصندوق العائم**
   ضع صورتك داخل المربع العائم واحفظها.
   ```csharp
   imageFloatingBox.getParagraphs().add(image);
   doc.save("YOUR_DOCUMENT_DIRECTORY/ImageInFloatingBox_out.pdf");
   ```

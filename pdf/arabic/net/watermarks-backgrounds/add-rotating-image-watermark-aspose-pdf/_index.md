## دليل التنفيذ

### إضافة علامة مائية للصورة مع التدوير
يرشدك هذا القسم إلى كيفية إضافة علامة مائية لصورة دوارة باستخدام Aspose.PDF لـ .NET. سنشرحها في خطوات واضحة:

#### 1. إعداد مسارات الملفات وتهيئة الكائنات
ابدأ بتحديد المسارات إلى ملف PDF المدخل، وملف PDF المخرج، وصورة العلامة المائية.

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Facades;
using System.IO;

// تحديد مسارات الملفات
string imageFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
string inFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "inFile.pdf");
string outFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "RotatingStamp_out.pdf");

// إنشاء كائن PdfFileInfo للحصول على أبعاد الصفحة
PdfFileInfo fileInfo = new PdfFileInfo(inFile);
```

#### 2. إنشاء كائن الطوابع وتكوينه
بعد ذلك، قم بتهيئة `Stamp` كائن وربط صورتك.

```csharp
// تهيئة كائن الختم
Aspose.Pdf.Facades.Stamp aStamp = new Aspose.Pdf.Facades.Stamp();

// ربط الصورة بالطابع
aStamp.BindImage(imageFile);

// تعيين خصائص العلامة المائية
aStamp.IsBackground = false; // العلامة المائية كعنصر في المقدمة
aStamp.Pages = new int[] { 1 }; // حدد الصفحات لتطبيق العلامة المائية عليها
aStamp.Rotation = 90; // تدوير بزاوية 90 درجة

// ضع الختم على الصفحة
aStamp.SetOrigin(fileInfo.GetPageWidth(1) / 2, fileInfo.GetPageHeight(1) / 2);

// تحديد حجم العلامة المائية
aStamp.SetImageSize(100, 100);
```

#### 3. تطبيق وحفظ العلامة المائية
وأخيرا، استخدم `PdfFileStamp` لتطبيق الختم وحفظ ملف الإخراج.

```csharp
Document doc = new Document(inFile);
PdfFileStamp stamper = new PdfFileStamp(doc);

// أضف العلامة المائية إلى ملف PDF
stamper.AddStamp(aStamp);

// حفظ التغييرات في ملف جديد
stamper.Save(outFile);

// تنظيف الموارد
stamper.Close();
```

### نصائح استكشاف الأخطاء وإصلاحها
- **الصورة غير مرئية**:تأكد من أن مسار الصورة صحيح وأن التنسيق مدعوم.
- **قضايا الدوران**:تأكد من ذلك `aStamp.Rotation` تم ضبطه بشكل صحيح؛ يتم الدوران حول المركز بشكل افتراضي.

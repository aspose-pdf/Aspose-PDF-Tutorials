## دليل التنفيذ
سنستكشف طريقتين لتحرير المناطق داخل ملف PDF: استخدام تعليقات التحرير وواجهة PdfAnnotationEditor. لكل طريقة مزاياها، حسب احتياجاتك الخاصة.

### التحرير باستخدام تعليق التحرير
يتيح هذا النهج تخصيصًا تفصيليًا للمنطقة المحررة.

#### ملخص
يمكنك تحديد المكان الدقيق لتطبيق التحرير وتخصيص مظهره.

#### خطوات
##### 1. قم بتحميل المستند
```java
document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
هنا، نقوم بتحميل مستند PDF إلى الذاكرة.

##### 2. تحديد منطقة التحرير
```java
Rectangle rect = new Rectangle(200, 500, 300, 600);
```
يحدد هذا المستطيل إحداثيات المنطقة التي تريد تحريرها.

##### 3. إنشاء وتكوين التعليقات التوضيحية
```java
RedactionAnnotation annot = new RedactionAnnotation(doc.getPages().get_Item(1), rect);
annot.setFillColor(Color.getBlack());
annot.setBorderColor(Color.getYellow());
annot.setColor(Color.getBlue());
annot.setFontSize(20);
annot.setOverlayText("REDACTED");
annot.setTextAlignment(HorizontalAlignment.Center);
annot.setRepeat(true);
```
يؤدي هذا المقطع إلى إنشاء تعليق تحريري، وتعيين لونه ونص التراكب الخاص به.

##### 4. أضف التعليق التوضيحي إلى الصفحة
```java
doc.getPages().get_Item(1).getAnnotations().add(annot);
```
تمت إضافة التحرير إلى الصفحة الأولى من المستند.

##### 5. احفظ المستند
```java
doc.save("YOUR_OUTPUT_DIRECTORY/Redaction_out.pdf");
```
وأخيرًا، احفظ التغييرات في ملف PDF جديد.

### التحرير باستخدام نهج واجهات محرر التعليقات التوضيحية PdfAnnotationEditor
توفر هذه الطريقة طريقة سريعة لتطبيق التحرير دون تخصيص التعليقات التوضيحية.

#### ملخص
يستخدم `PdfAnnotationEditor` لمهام التحرير المباشرة.

#### خطوات
##### 1. تهيئة المحرر
```java
PdfAnnotationEditor editor = new PdfAnnotationEditor();
editor.bindPdf("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
قم بتحميل مستندك باستخدام المحرر.

##### 2. تطبيق التحرير
```java
color redactColor = Color.WHITE;
editor.redactArea(1, new Rectangle(100, 100, 20, 70), redactColor);
```
قم بتحديد وتطبيق التحرير على الصفحة 1 باستخدام الإحداثيات المحددة.

##### 3. احفظ المستند
```java
color outputDirectory = "YOUR_OUTPUT_DIRECTORY/Redaction_out.pdf";
editor.save(outputDirectory);
```
قم بتخزين مستندك المعدّل مع التحريرات المطبقة.

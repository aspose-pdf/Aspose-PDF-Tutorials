## دليل التنفيذ

### تحميل مستند PDF
#### ملخص

```java
        // قم بتهيئة كائن مستند Aspose.PDF باستخدام المسار إلى ملف PDF الخاص بك
        Document pdfDocument = new Document(filePath);
        
        System.out.println("PDF loaded successfully.");
    }
}
```
### استرداد الصور المضمنة وإدراجها من صفحة محددة
#### ملخص
بعد تحميل مستند، قد تحتاج إلى الوصول إلى الصور المُضمّنة. إليك كيفية استرجاع أسماء الصور من الصفحة الأولى.
```java
import com.aspose.pdf.Document;

public class ImageRetriever {
    public static void listImages(String filePath) {
        Document pdfDocument = new Document(filePath);
        
        // الوصول إلى موارد الصفحة الأولى
        var pageResources = pdfDocument.getPages().get_Item(1).getResources();
        
        if (pageResources.getImages() != null) {
            for (int i = 1; i <= pageResources.getImages().size(); i++) {
                String imageName = pageResources.getImages().getNames()[i - 1];
                System.out.println("Image Name: " + imageName);
            }
        } else {
            System.out.println("No images found on the first page.");
        }
    }
}
```
#### المعلمات وقيم الإرجاع
- **مسار الملف**: مسار ملف PDF الخاص بك. يجب أن يكون هذا المسار مسار دليل صالح.
- **كائن المستند**:يتعامل مع مستند PDF المحمّل، مما يوفر إمكانية الوصول إلى محتواه.
- **الموارد والصور**:طرق تسمح لك بالتفاعل مع الموارد مثل الصور على صفحات محددة.

# استخراج خصائص صفحة PDF باستخدام Aspose.PDF لـ Java

- استخراج خصائص مثل أبعاد ArtBox، وBleedBox، وCropBox، وMediaBox، وTrimBox، وRect.
- استرجاع تفاصيل الصفحة الإضافية مثل رقم الصفحة والدوران.

## إعداد Aspose.PDF لـ Java

Aspose.PDF مكتبة متعددة الاستخدامات تُمكّن المطورين من إنشاء ملفات PDF وتعديلها واستخراج المعلومات منها. إليك كيفية إضافتها إلى مشروعك بلغة جافا:

### مافن
قم بتضمين التبعية التالية في ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

### جرادل
أضف هذا السطر إلى `build.gradle` ملف:
```gradle

```

## دليل التنفيذ

### استخراج خصائص الصفحة

في هذا القسم، سنستكشف كيفية استخراج خصائص مختلفة لصفحة PDF باستخدام Aspose.PDF لجافا. سنشرح العملية خطوة بخطوة:

#### الخطوة 1: فتح مستند
ابدأ بتحميل مستند PDF الخاص بك إلى `Aspose.Pdf.Document` هدف.
```java
Document pdfDocument = new Document("input.pdf");
```
تؤدي هذه الخطوة إلى تهيئة المستند الذي تريد العمل عليه، مما يسمح لك بالوصول إلى محتوياته.

#### الخطوة 2: الوصول إلى مجموعة الصفحات
استرداد مجموعة الصفحات في المستند باستخدام:
```java
PageCollection pageCollection = pdfDocument.getPages();
```
ال `PageCollection` يحتوي الكائن على جميع الصفحات الموجودة داخل ملف PDF الخاص بك.

#### الخطوة 3: الحصول على صفحة محددة
اختر صفحة محددة من المجموعة. على سبيل المثال، للحصول على الصفحة الأولى:
```java
Page pdfPage = pageCollection.get_Item(1);
```

#### الخطوة 4: استخراج الخصائص
الآن، قم باستخراج خصائص الصندوق المختلفة والسمات الأخرى للصفحة المحددة.

- **صندوق الفن:** يحدد منطقة الطباعة المقصودة.
  ```java
  System.out.printf("\n ArtBox : Height = " + pdfPage.getArtBox().getHeight() +
                    ", Width = " + pdfPage.getArtBox().getWidth() +
                    ", LLX = " + pdfPage.getArtBox().getLLX() +
                    ", LLY = " + pdfPage.getArtBox().getLLY() +
                    ", URX = " + pdfPage.getArtBox().getURX() +
                    ", URY = " + pdfPage.getArtBox().getURY());
  ```

- **صندوق النزيف:** يحدد المنطقة التي سيتم تضمين النزيف فيها أثناء الطباعة.
  ```java
  System.out.printf("\n BleedBox : Height = " + pdfPage.getBleedBox().getHeight() +
                    ", Width = " + pdfPage.getBleedBox().getWidth() +
                    ", LLX = " + pdfPage.getBleedBox().getLLX() +
                    ", LLY = " + pdfPage.getBleedBox().getLLY() +
                    ", URX = " + pdfPage.getBleedBox().getURX() +
                    ", URY = " + pdfPage.getBleedBox().getURY());
  ```

- **CropBox، وMediaBox، وTrimBox، وRect:** تعمل هذه المربعات على تعريف مناطق مختلفة تتعلق بالقص، وحجم الوسائط، والتشذيب، والأبعاد الشاملة.

متابعة استخراج خصائص أخرى مثل رقم الصفحة والدوران:
```java
System.out.printf("\n Page Number :- " + pdfPage.getNumber());
System.out.printf("\n Rotate :-" + pdfPage.getRotate());
```

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسار ملف PDF الخاص بك صحيح.
- تأكد من إضافة مكتبة Aspose.PDF بشكل صحيح إلى تبعيات مشروعك.

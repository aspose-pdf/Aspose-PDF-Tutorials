## إعداد Aspose.PDF لـ Java

دمج Aspose.PDF في مشروعك سهل للغاية. إليك كيفية إضافته باستخدام Maven أو Gradle:

**مافن:**
```xml

    <groupId>com.aspose</groupId>

    <version>25.3</version>
</dependency>
```

**جرادل:**
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## دليل التنفيذ

دعونا نقسم العملية إلى ميزتين رئيسيتين: إعداد عامل التكبير والحصول عليه في مستند PDF باستخدام Java.

### ضبط عامل التكبير

#### ملخص
تتيح لك هذه الميزة تحديد مدى حجم المحتوى الذي يظهر عند فتح ملف PDF، مما يعزز تجربة المستخدم منذ البداية.

#### التنفيذ خطوة بخطوة:
**1. استيراد الحزم الضرورية**
ابدأ باستيراد الفئات الضرورية:
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.XYZExplicitDestination;
import com.aspose.pdf.GoToAction;
```

**2. تحديد مسارات الملفات ومستوى التكبير**
قم بإعداد أدلة المستندات الخاصة بك وعامل التكبير المطلوب.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";
double zoom = 0.5; // مثال: مستوى تكبير 50%
```

**3. قم بتحميل مستند PDF**
إنشاء `Document` كائن لتحميل ملف PDF الموجود الذي تريد تعديله.
```java
Document doc = new Document(dataDir + "/HelloWorld.pdf");
```

**4. إنشاء XYZExplicitDestination**
قم بإعداد الوجهة بمستوى التكبير والعرض والارتفاع المطلوب بناءً على أبعاد مربع الوسائط في الصفحة الأولى.
```java
XYZExplicitDestination destination = new XYZExplicitDestination(
    doc.getPages().get_Item(1),
    doc.getPages().get_Item(1).getMediaBox().getWidth(),
    doc.getPages().get_Item(1).getMediaBox().getHeight(), 
    zoom);
```

**5. تعيين التكبير كإجراء مفتوح**
إنشاء `GoToAction` باستخدام وجهتك وتعيينها كإجراء فتح للمستند.
```java
GoToAction actionzoom = new GoToAction(destination);
doc.setOpenAction(actionzoom);
```

**6. احفظ التغييرات**
أخيرًا، احفظ ملف PDF المعدّل لتطبيق إعداد التكبير/التصغير.
```java
String outputPath = outputDir + "/Zoomed_actionzoom.pdf";
doc.save(outputPath);
```

### استرجاع عامل التكبير

#### ملخص
يعد الوصول إلى مستوى التكبير الحالي الموجود في ملف PDF أمرًا بالغ الأهمية لفهم كيفية عرض مستندك عند فتحه.

#### التنفيذ خطوة بخطوة:
**1. قم بتحميل المستند المعدّل**
ابدأ بتحميل المستند المحفوظ مسبقًا.
```java
Document doc1 = new Document(outputDir + "/Zoomed_actionzoom.pdf");
```

**2. الوصول إلى إجراء الفتح واسترداد مستوى التكبير**
استخرج `GoToAction` من الإجراء المفتوح، ثم احصل على مستوى التكبير من `XYZExplicitDestination`.
```java
GoToAction action = (GoToAction) doc1.getOpenAction();
XYZExplicitDestination destination = (XYZExplicitDestination) action.getDestination();
double retrievedZoom = destination.getZoom();
```
المتغير `retrievedZoom` يحتفظ الآن بعامل التكبير الحالي للمستند الخاص بك.

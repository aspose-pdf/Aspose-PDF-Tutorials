## إعداد Aspose.PDF لـ Java

لتعديل تفضيلات عارض PDF، ستحتاج أولاً إلى دمج Aspose.PDF في مشروعك. إليك كيفية القيام بذلك باستخدام Maven أو Gradle:

**مافن:**

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

**جرادل:**

```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## دليل التنفيذ

دعنا نستعرض الخطوات لتغيير تفضيلات عارض PDF باستخدام Aspose.PDF لـ Java.

### الخطوة 1: تهيئة محرر محتوى Pdf

ابدأ بإنشاء `PdfContentEditor` توفر هذه الفئة طرقًا لتعديل محتوى ملف PDF الحالي وخصائصه.

```java
import com.aspose.pdf.facades.PdfContentEditor;

// إنشاء مثيل لـ PdfContentEditor
PdfContentEditor contentEditor = new PdfContentEditor();
```

### الخطوة 2: الارتباط بملف PDF الموجود

ربط الخاص بك `PdfContentEditor` اعترض على ملف PDF الموجود الذي تريد تعديله.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
contentEditor.bindPdf(dataDir + "/input.pdf");
```
*لماذا؟*:يعد الربط أمرًا ضروريًا لربط المحرر بمستند PDF المحدد، مما يتيح إجراء المزيد من التعديلات.

### الخطوة 3: تغيير تفضيلات المشاهد

الآن، لنُغيّر تفضيلات المُعاينات المُختلفة. يُغيّر كل استدعاء دالة جانبًا مُختلفًا من كيفية عرض ملف PDF في المُعاينات:

```java
import com.aspose.pdf.facades.ViewerPreference;

// مركز النافذة عند فتحها
contentEditor.changeViewerPreference(ViewerPreference.CENTER_WINDOW);

// إخفاء شريط القائمة
contentEditor.changeViewerPreference(ViewerPreference.HIDE_MENUBAR);    

// تعطيل أي إعدادات لوضع الصفحة
contentEditor.changeViewerPreference(ViewerPreference.PAGE_MODE_USE_NONE);
```
*لماذا؟*:تتيح لك هذه الطرق تخصيص سلوك المشاهد، وتحسين تجربة المستخدم استنادًا إلى احتياجات محددة.

### الخطوة 4: حفظ ملف PDF المعدل

بعد إجراء التغييرات، احفظ ملف PDF المعدّل في موقع جديد:

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
contentEditor.save(outputDir + "/ChangePreference_output.pdf");
```
*لماذا؟*:يضمن الحفظ تخزين كافة التعديلات بشكل دائم في دليل محدد.

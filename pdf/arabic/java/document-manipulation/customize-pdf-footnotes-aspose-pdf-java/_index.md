### الميزة 1: نمط خط مخصص للحاشية السفلية
#### ملخص
قم بتخصيص مظهر الحواشي السفلية عن طريق تغيير نمط خطوطها في مستندات PDF الخاصة بك.

#### خطوات:
**1. إنشاء GraphInfo وتكوينه**

```java
GraphInfo graph = new GraphInfo();
graph.setLineWidth(2); // ضبط عرض الخط إلى نقطتين.
graph.setColor(Color.getRed()); // تغير اللون إلى اللون الأحمر.
```

**توضيح**:تضبط هذه الإعدادات نمط سطر الحواشي السفلية. `GraphInfo` تتحكم الفئة في كيفية عرض الخطوط.

تطبق هذه الطريقة تكوين الخط المخصص الخاص بك على جميع الحواشي السفلية الموجودة في الصفحة.

- تأكد من قيم اللون ونمط الشرطة إذا لم يظهر النمط كما هو متوقع.

## إعداد Aspose.PDF لـ Java
لبدء العمل مع Aspose.PDF لـ Java، اتبع الخطوات التالية لإعداد بيئتك:

### تثبيت Maven
أضف التبعية التالية إلى ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

### تثبيت Gradle
قم بتضمين هذا في `build.gradle` ملف:
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## دليل التنفيذ
سنقوم بتقسيم تنفيذنا إلى ميزتين رئيسيتين: تغيير اتجاه الصفحة إلى الوضع الأفقي وملاءمة المحتوى وفقًا لذلك.

### الميزة 1: تغيير اتجاه الصفحة إلى الوضع الأفقي
**ملخص**:تعمل هذه الميزة على تحويل كل صفحة PDF من الوضع الرأسي إلى الوضع الأفقي، وضبط الأبعاد لتحقيق انتقال سلس.

#### التنفيذ خطوة بخطوة
##### تحميل المستند المصدر
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/Input.pdf");
```
*لماذا*:يعتبر تحميل المستند أمرًا ضروريًا لأنه يسمح لنا بالتعامل مع صفحاته.

##### تغيير اتجاه الصفحة
قم بالتكرار خلال كل صفحة وضبط الأبعاد:
```java
int pageCount = doc.getPages().size();
for (int i = 1; i <= pageCount; i++) {
    Rectangle r = doc.getPages().get_Item(i).getMediaBox();

    double newHeight = r.getWidth();
    double newWidth = r.getHeight();
    double newLLX = r.getLLX();
    double newLLY = r.getLLY() + (r.getHeight() - newHeight);

    doc.getPages().get_Item(i).setMediaBox(new Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight));
    doc.getPages().get_Item(i).setCropBox(new Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight));

    // قم بتدوير الصفحة إلى 90 درجة للحصول على اتجاه أفقي
    doc.getPages().get_Item(i).setRotate(Rotation.on90);
}
```
*لماذا*:يعمل هذا الجزء من التعليمات البرمجية على تغيير أبعاد كل صفحة وتدويرها بمقدار 90 درجة لتحقيق عرض أفقي.

##### حفظ المستند المعدل
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.save(outputDir + "/Output.pdf");
```
*لماذا*:يضمن حفظ المستند تخزين التغييرات التي أجريتها بشكل مستمر، مما يسمح لك بمراجعة ملف PDF المعدل أو مشاركته.

### الميزة 2: ملاءمة محتوى الصفحة للاتجاه الجديد
**ملخص**:ضبط أبعاد المحتوى بعد تغيير اتجاه الصفحة للحصول على ملاءمة أفضل.

#### ضبط الأبعاد
```java
Rectangle r = doc.getPages().get_Item(0).getMediaBox();
double newHeight = r.getHeight();
double newWidth = r.getWidth() * (r.getHeight() / r.getWidth());

// يمكن تنفيذ المزيد من التعديلات هنا إذا لزم الأمر.
```
*لماذا*:إن التأكد من أن المحتوى يتناسب جيدًا مع الصفحات الموجهة حديثًا يعزز قابلية القراءة وجودة العرض.

## دليل التنفيذ
### الميزة 1: ربط ملفات PDF باستخدام مسارات الملفات
#### ملخص
يُعدّ دمج ملفات PDF أمرًا بالغ الأهمية عند الحاجة إلى مستند واحد من مصادر متعددة. تُبسّط هذه الميزة عملية دمج المستندات في مستند واحد باستخدام المسارات المُتاحة في الكود.

#### خطوات:
##### الخطوة 1: إنشاء كائن محرر ملفات Pdf
```java
import com.aspose.pdf.facades.PdfFileEditor;

PdfFileEditor pdfEditor = new PdfFileEditor();
```
يعد هذا الكائن بمثابة بوابتك لتحرير ملفات PDF باستخدام Aspose.PDF.

##### الخطوة 2: ربط الملفات
فيما يلي كيفية تحديد مسارات الإدخال والإخراج:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

pdfEditor.concatenate(
    dataDir + "/input1.pdf",
    dataDir + "/input2.pdf",
    outputDir + "/output.pdf"
);
```
ال `concatenate` تأخذ هذه الطريقة مسارات ملفات PDF المدخلة ومسار إخراج المستند المدمج. تأكد من إعداد المجلدات بشكل صحيح لتجنب أي أخطاء تتعلق بعدم العثور على الملف.

### الميزة 2: إعداد خيار نسخ الخطوط العريضة
#### ملخص
عند ربط ملفات PDF، غالبًا ما تكون إدارة الخطوط العريضة (الإشارات المرجعية) ضرورية. تتيح لك هذه الميزة تحديد ما إذا كان ينبغي تضمين هذه الخطوط العريضة في المستند النهائي.

#### خطوات:
##### الخطوة 1: تكوين PdfFileEditor للتحكم في المخطط التفصيلي
```java
PdfFileEditor pfe = new PdfFileEditor();
pfe.setCopyOutlines(false); // لن يتم نسخ الإشارات المرجعية.
```
جلسة `CopyOutlines` يضمن تعيين "إلى false" عدم تضمين الإشارات المرجعية من ملفات PDF الفردية في الملف المترابط.

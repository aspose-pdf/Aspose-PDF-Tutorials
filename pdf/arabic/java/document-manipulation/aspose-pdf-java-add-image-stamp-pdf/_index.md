## دليل التنفيذ

الآن بعد اكتمال عملية الإعداد، دعنا نركز على تنفيذ كل ميزة خطوة بخطوة.

### فتح مستند PDF

**ملخص:**
فتح ملف PDF موجود سهل للغاية مع Aspose.PDF. ستُحمّله في `Document` كائن لإجراء عمليات أخرى.

```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
// افتح المستند من الدليل المحدد.
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

**توضيح:**
- **فئة المستند:** يمثل ملف PDF، مما يسمح بالتلاعب بمحتواه.
- **دليل البيانات:** المسار الذي سيتم تخزين ملفات الإدخال الخاصة بك فيه.

### إنشاء وتكوين ختم الصورة

**ملخص:**
تتضمن عملية إنشاء ختم صورة مخصص تحديد خصائص مثل الحجم والموضع والدوران والتعتيم لتناسب احتياجاتك.

```java
import com.aspose.pdf.ImageStamp;
import com.aspose.pdf.Rotation;

// إنشاء الختم باستخدام ملف صورة.
ImageStamp imageStamp = new ImageStamp(dataDir + "/sample.jpg");

// تعيين الختم كعنصر خلفية.
imageStamp.setBackground(true);
// قم بتعيين المسافة البادئة الأفقية والرأسية لموضع الختم.
imageStamp.setXIndent(100); 
imageStamp.setYIndent(100);
// تحديد ارتفاع وعرض الطوابع.
imageStamp.setHeight(300);
imageStamp.setWidth(300);
// قم بتدوير الختم 270 درجة.
imageStamp.setRotate(Rotation.on270);
// اضبط مستوى التعتيم لجعل الختم شفافًا إلى حد ما.
imageStamp.setOpacity(0.5);
```

**توضيح:**
- **فئة ImageStamp:** يسمح بإضافة الصور كطوابع مع خيارات تخصيص واسعة النطاق.
- **setBackground(صحيح):** تمت إضافة الختم خلف المحتوى الموجود.
- **الدوران والتعتيم:** تخصيص زاوية الدوران ومستوى الشفافية.

### إضافة ختم صورة إلى صفحة محددة في ملف PDF

**ملخص:**
بعد إعداد ختم الصورة، يمكنك الآن إضافتها إلى أي صفحة في مستندك. إليك كيفية وضعها في الصفحة الأولى:

```java
// أضف الختم إلى الصفحة الأولى من المستند.
pdfDocument.getPages().get_Item(1).addStamp(imageStamp);
```

**توضيح:**
- **الحصول على الصفحات ():** الوصول إلى كافة الصفحات في ملف PDF.
- **الحصول على العنصر(1):** استرجاع صفحة محددة حسب الفهرس (لاحظ أن الفهرسة تعتمد على 1).

### حفظ مستند PDF المعدّل

وأخيرًا، احفظ التغييرات لضمان استمرارها.

```java
import com.aspose.pdf.Document;

String outputDir = "YOUR_OUTPUT_DIRECTORY";
// احفظ المستند مع التغييرات في ملف جديد.
pdfDocument.save(outputDir + "/PageNumberStamp_output.pdf");
```

**توضيح:**
- **يحفظ():** يكتب التعديلات مرة أخرى في ملف PDF في المسار المحدد.

## دليل التنفيذ

### إضافة وظيفة انتهاء الصلاحية
الميزة الأساسية في برنامجنا التعليمي هي تضمين إجراء JavaScript في ملف PDF للتحقق من انتهاء صلاحية المستند عند فتحه. إليك كيفية القيام بذلك:
#### الخطوة 1: تحميل مستند PDF الخاص بك
ابدأ بتحميل ملف PDF موجود من الدليل الخاص بك باستخدام Aspose.PDF.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // تعيين مسار دليل PDF المدخل
Document doc = new Document(dataDir + "/input.pdf");
```
يقوم هذا الكود بتهيئة `Document` الكائن الذي يمثل ملف PDF الخاص بك. ستحتاج إلى استبدال `"YOUR_DOCUMENT_DIRECTORY"` مع المسار الفعلي الذي يتم تخزين ملفات PDF الخاصة بك فيه.
#### الخطوة 2: تعريف JavaScript للتحقق من انتهاء الصلاحية
بعد ذلك، قم بتعريف JavaScript الذي يتحقق مما إذا كان التاريخ الحالي يتجاوز تاريخ انتهاء الصلاحية ويعرض تنبيهًا إذا حدث ذلك.
```java
JavascriptAction javaScript = new JavascriptAction(
    "var year=2014; var month=4; today = new Date(); today = new Date(today.getFullYear(), today.getMonth());" +
    "expiry = new Date(year, month);" +
    "if (today.getTime() > expiry.getTime()) app.alert('The file is expired. You need a new one.');"
);
```
يحدد هذا البرنامج النصي تاريخ انتهاء الصلاحية (`2014-04`) ويقارنه بالتاريخ الحالي في كل مرة يُفتح فيها ملف PDF. إذا تجاوز تاريخ فتح المستند هذا التاريخ، يُرسَل تنبيه للمستخدم.
#### الخطوة 3: تعيين JavaScript كإجراء مفتوح
دمج إجراء JavaScript في ملف PDF الخاص بك بحيث يتم تشغيله عند الفتح:
```java
doc.setOpenAction(javaScript);
```
يضمن هذا السطر من التعليمات البرمجية أنه في كل مرة يتم فيها فتح ملف PDF الخاص بك، يتم تشغيل JavaScript تلقائيًا، والتحقق من انتهاء الصلاحية.
#### الخطوة 4: حفظ ملف PDF المعدل
أخيرًا، احفظ المستند باستخدام إجراء JavaScript المضاف حديثًا إلى دليل الإخراج المحدد.
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // تعيين مسار دليل PDF الناتج
doc.save(outputDir + "/JavaScript-Added.pdf");
```
يستبدل `"YOUR_OUTPUT_DIRECTORY"` مع مسار مجلد الإخراج المطلوب. هذا يحفظ ملف PDF المعدّل، جاهزًا للتوزيع مع تفعيل ميزة انتهاء الصلاحية.

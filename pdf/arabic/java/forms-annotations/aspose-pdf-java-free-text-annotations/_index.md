## دليل التنفيذ
سنقوم بتقسيم التنفيذ إلى عدة ميزات رئيسية: فتح ملف PDF، وإنشاء مستطيلات التعليقات التوضيحية، وإضافة التعليقات التوضيحية، وحفظ الملف.

### فتح وربط مستند PDF
**ملخص**:تتضمن هذه الخطوة تحضير ملف PDF الخاص بك للتحرير عن طريق تحميله في مثيل `PdfContentEditor`.

#### التنفيذ خطوة بخطوة
**الخطوة 1**:إنشاء مثيل
```java
import com.aspose.pdf.facades.PdfContentEditor;

class OpenAndBindPDF {
    public void openAndBind() {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        PdfContentEditor contentEditor = new PdfContentEditor();
        
        // ربط ملف PDF المدخل باستخدام مسار إلى مستندك
        String inputFile = dataDir + "input.pdf";
        contentEditor.bindPdf(inputFile);
    }
}
```
**توضيح**:هذا الكود يقوم بالتهيئة `PdfContentEditor` ويربطه بملف PDF موجود. `bindPdf()` تعتبر هذه الطريقة حاسمة لأنها تمهد الطريق لأي تعديلات.

### إنشاء مستطيل لوضع التعليقات التوضيحية
**ملخص**:قم بتحديد المنطقة على ملف PDF الخاص بك حيث تريد وضع التعليقات التوضيحية باستخدام Java `Rectangle` فصل.

#### التنفيذ خطوة بخطوة
**الخطوة 2**:تحديد أبعاد المستطيل
```java
import java.awt.Rectangle;

class CreateRectangle {
    public Rectangle create() {
        int x = 50; // إحداثيات X للزاوية العلوية اليسرى
        int y = 50; // إحداثي Y للزاوية العلوية اليسرى
        int width = 100; // عرض المستطيل
        int height = 100; // ارتفاع المستطيل
        
        return new Rectangle(x, y, width, height);
    }
}
```
**توضيح**: ال `Rectangle` تم إنشاء الكائن بأبعاد محددة. هذه المعلمات (س، ص، العرض، الارتفاع) تُحدد مكان وحجم التعليق التوضيحي.

### إضافة تعليقات نصية مجانية إلى ملف PDF
**ملخص**:بمجرد تحديد المستطيل المراد وضعه، سنستخدمه لإضافة تعليق نصي حر إلى المستند.

#### التنفيذ خطوة بخطوة
**الخطوة 3**:إنشاء وإضافة التعليقات التوضيحية
```java
import com.aspose.pdf.facades.PdfContentEditor;
import java.awt.Rectangle;

class AddFreeTextAnnotation {
    public void addAnnotation(PdfContentEditor contentEditor, Rectangle rect) {
        int pageNumber = 1; // حدد رقم الصفحة للتعليق التوضيحي
        String annotationText = "Sample content"; 
        
        contentEditor.createFreeText(rect, annotationText, pageNumber);
    }
}
```
**توضيح**: ال `createFreeText()` تُستخدم هذه الطريقة لإضافة ملاحظة نصية في المكان المحدد. هنا، نضعها في الصفحة الأولى من ملف PDF.

### حفظ مستند PDF المحدث
**ملخص**:بعد إضافة التعليقات التوضيحية، احفظ المستند المعدل للحفاظ على التغييرات.

#### التنفيذ خطوة بخطوة
**الخطوة 4**:حفظ التعديلات
```java
import com.aspose.pdf.facades.PdfContentEditor;

String outputDir = "YOUR_OUTPUT_DIRECTORY";
class SavePDF {
    public void save(PdfContentEditor contentEditor) {
        String outputFile = outputDir + "output.pdf";
        
        // استمرار التغييرات في مستند PDF
        contentEditor.save(outputFile);
    }
}
```
**توضيح**: ال `save()` تكتب الطريقة جميع التغييرات التي تم إجراؤها أثناء الجلسة مرة أخرى إلى ملف، مما يضمن الاحتفاظ بالتعليقات التوضيحية.

# إنشاء ختم نصي محاذي للمركز في ملفات PDF باستخدام Aspose.PDF لـ Java

هل أنت مستعد لتبسيط عملية تخصيص مستنداتك؟ هيا بنا!

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

### المكتبات والتبعيات المطلوبة

ستحتاج إلى تضمين Aspose.PDF لجافا كاعتمادية. اختر Maven أو Gradle بناءً على إعدادات مشروعك.

**مافن:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

**جرادل:**
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

- تأكد من تثبيت Java Development Kit (JDK) وتكوينه.

## دليل التنفيذ

الآن، دعونا ننتقل إلى عملية التنفيذ خطوة بخطوة.

### إنشاء ختم نص مركزي في PDF

**1. افتح مستند PDF موجود**

أولاً، نقوم بتحميل مستند PDF الموجود باستخدام `Document` فصل.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

**2. تكوين كائن FormattedText**

نحن ننشئ `FormattedText` الهدف هو تحديد محتوى النص وبنيته.
```java
FormattedText text = new FormattedText("This");
text.addNewLineText("is sample");
text.addNewLineText("Center Aligned");
text.addNewLineText("TextStamp");
text.addNewLineText("Object");
```
- **لماذا هذه الخطوة؟** نحن نستخدم `FormattedText` لإنشاء كتلة نصية غنية التنسيق يمكن تخصيصها بسمات مختلفة.

**3. إنشاء وتكوين كائن TextStamp**

بعد ذلك، نقوم بإنشاء مثيل `TextStamp` الكائن باستخدام ما تم تكوينه `FormattedText`.
```java
TextStamp stamp = new TextStamp(text);
```

**4. تعيين خصائص المحاذاة**

وهنا حيث يحدث السحر: ضبط المحاذاة.
```java
stamp.setHorizontalAlignment(HorizontalAlignment.Center);
stamp.setVerticalAlignment(VerticalAlignment.Center);
stamp.setTextAlignment(HorizontalAlignment.Center);
```
- **حدود:** تعمل هذه الطرق على ضبط كيفية وضع النص داخل الختم وعلى الصفحة.

**5. تحديد الهوامش**

نقوم بتحديد هامش علوي لضبط موضع ختم النص الخاص بنا.
```java
stamp.setTopMargin(20);
```

**6. إضافة ختم إلى صفحة PDF**

وأخيرًا، نضيف الختم الذي تم تكوينه إلى الصفحة المطلوبة في المستند.
```java
pdfDocument.getPages().get_Item(1).addStamp(stamp);
```
- **ملحوظة:** يمكنك استهداف أي صفحة عن طريق تغيير `get_Item(1)`.

**7. احفظ المستند المعدّل**

احفظ التغييرات في ملف جديد.
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
pdfDocument.save(outputDir + "/TextStamp_output.pdf");
```

### نصائح استكشاف الأخطاء وإصلاحها

- **مشكلة شائعة:** النص غير متمركز بشكل صحيح؟ تحقق جيدًا من إعدادات المحاذاة.

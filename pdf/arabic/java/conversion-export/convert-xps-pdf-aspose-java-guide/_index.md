**التهيئة والإعداد الأساسي**
بمجرد التثبيت، يمكنك تهيئة Aspose.PDF لـJava على النحو التالي:
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.LoadOptions;
import com.aspose.pdf.XpsLoadOptions;

public class ConvertXPS2PDF {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY/AsposePdf/Conversion/xpstopdf/";
        String outputDir = "YOUR_OUTPUT_DIRECTORY/";

        // إنشاء كائن LoadOption باستخدام خيار تحميل XPS.
        LoadOptions options = new XpsLoadOptions();

        // إنشاء كائن مستند باستخدام دليل البيانات المحدد وLoadOptions.
        Document document = new Document(dataDir + "test.xps", options);

        // احفظ مستند PDF الناتج في دليل الإخراج المحدد.
        document.save(outputDir + "resultant.pdf");
    }
}
```

### الميزة: تحويل ملف XPS إلى PDF

#### الخطوة 1: تحميل مستند XPS
ابدأ بإنشاء `LoadOptions` كائن مصمم خصيصًا لملفات XPS. هذا يضمن تطبيق جميع التكوينات الخاصة بـ XPS.
```java
// إنشاء كائن LoadOption باستخدام خيار تحميل XPS.
LoadOptions options = new XpsLoadOptions();
```
**توضيح**: ال `XpsLoadOptions()` يقوم المنشئ بتهيئة الإعدادات المخصصة للتعامل مع مستندات XPS، مما يضمن التحليل المناسب ودقة التحويل.

#### الخطوة 2: إنشاء كائن مستند
بعد ذلك، قم بإنشاء مثيل لـ `Document` باستخدام مسار ملف XPS وخيارات التحميل. يمثل هذا الكائن المستند الذي تُحوّله.
```java
// إنشاء كائن مستند باستخدام دليل البيانات المحدد وLoadOptions.
Document document = new Document(dataDir + "test.xps", options);
```
**توضيح**: هنا، `new Document(String, LoadOptions)` يقوم بتهيئة ملف XPS الخاص بك كمستند Aspose.PDF قابل للتلاعب، وجاهز للتحويل.

#### الخطوة 3: احفظ ملف PDF
أخيرًا، احفظ المستند الذي تم إنشاؤه حديثًا بتنسيق الإخراج والموقع المطلوبين:
```java
// احفظ مستند PDF الناتج في دليل الإخراج المحدد.
document.save(outputDir + "resultant.pdf");
```
**توضيح**: ال `save(String)` تنهي الطريقة عملية التحويل عن طريق كتابة الملف بتنسيق PDF إلى الدليل الذي حددته.

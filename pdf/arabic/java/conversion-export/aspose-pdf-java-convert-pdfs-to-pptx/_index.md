## كيفية تحويل pdf إلى pptx باستخدام Aspose.PDF for Java
لتحويل PDF إلى PPTX، أنشئ كائن `Document` مع الملف المصدر، أنشئ كائن `PptxSaveOptions` لتحديد حجم الشريحة، جودة الصورة، وتعامل الخطوط، ثم استدعِ `doc.save` بالاسم المطلوب للملف والخيارات. هذه العملية الموجزة من ثلاث خطوات تنتج عرض PowerPoint قابل للتحرير مع الحفاظ على التخطيط الأصلي.

### الإجابة المباشرة
حمّل ملف PDF باستخدام `new Document("source.pdf")`، أنشئ كائن `PptxSaveOptions` لتعيين أي تفضيلات لحجم الشريحة أو جودة الصورة، ثم استدعِ `doc.save("output.pptx", saveOptions)`. هذا النمط من ثلاث أسطر يحول PDF إلى PPTX مع الحفاظ على التخطيط والرسومات تلقائيًا.

### تحميل مستند PDF
**التعريف:** تمثل فئة `Document` ملف PDF واحد في الذاكرة وتوفر طرقًا للقراءة، التحرير، والحفظ.

#### تنفيذ خطوة بخطوة:
1. **Initialize the Document Class**  
```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Replace with your actual directory path

public static void documentLoadingExample(String dataDir) {
    Document doc = new Document(dataDir + "/input.pdf");
    System.out.println("PDF Loaded Successfully!");
}
```  
2. **Understand Parameters and Methods**  
   - `dataDir`: المجلد الذي يحتوي على ملف PDF المصدر.  
   - `Document doc`: المثيل الذي يمثل ملف PDF المحمّل، والذي تُجرى من خلاله جميع العمليات اللاحقة.

### تحويل PDF إلى PPTX
**التعريف:** يضبط `PptxSaveOptions` كيفية تحويل PDF إلى شرائح PowerPoint، مما يتيح التحكم في جودة الصورة، أبعاد الشريحة، واستبدال الخطوط.

#### تنفيذ خطوة بخطوة:
1. **Load and Initialize Document**  
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.PptxSaveOptions;

String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Replace with your actual directory path
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // Replace with your output directory

public static void convertPDFToPPTX(String dataDir, String outputDir) {
    Document doc = new Document(dataDir + "/input.pdf");
```  
2. **Configure Conversion Options**  
```java
    PptxSaveOptions pptx_save = new PptxSaveOptions();
```  
3. **Save the Converted Document**  
```java
    doc.save(outputDir + "/output.pptx", pptx_save);
    System.out.println("Conversion to PPTX Complete!");
}
```  

#### اعتبارات رئيسية
- يتيح `PptxSaveOptions` تعيين `setImageQuality(int)` (0‑100) و`setSlideSize(SlideSize)` لتتناسب مع قالب العرض الخاص بك.  
- بالنسبة لملفات PDF التي تحتوي على خطوط مدمجة، فعّل `setEmbedFonts(true)` لتجنب تحذيرات الخطوط المفقودة.

## المشكلات الشائعة والحلول
| المشكلة | الحل |
|-------|----------|
| **OutOfMemoryError on large PDFs** | عالج الملفات على دفعات أصغر، استدعِ `System.gc()` بعد كل تحويل، وفعل `doc.optimizeResources()` قبل الحفظ. |
| **Missing fonts in PPTX** | دمج الخطوط في PDF المصدر أو عيّن `saveOptions.setFontSubstitution(true)` لاستبدال الخطوط المفقودة بالخطوط الافتراضية. |
| **Poor image quality** | زد `saveOptions.setImageQuality(100)` أو اختر `ImageCompression.LOSSLESS`. |

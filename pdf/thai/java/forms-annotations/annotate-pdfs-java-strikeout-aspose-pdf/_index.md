# วิธีการใส่คำอธิบายประกอบ PDF ใน Java โดยใช้ Strike-Out Annotations โดยใช้ Aspose.PDF

ปลดล็อกศักยภาพทั้งหมดของการจัดการ PDF โดยใช้ Aspose.PDF สำหรับ Java โดยการเรียนรู้วิธีการใส่คำอธิบายประกอบและบันทึกไฟล์ PDF ด้วยคำอธิบายประกอบแบบขีดฆ่า คำแนะนำที่ครอบคลุมนี้จะแนะนำคุณตลอดขั้นตอนการเริ่มต้นเอกสาร การค้นหาส่วนข้อความ การสร้างคำอธิบายประกอบ และการบันทึกเอกสารที่มีคำอธิบายประกอบของคุณ

## การแนะนำ

การนำทางผ่านความซับซ้อนของ PDF อาจเป็นเรื่องท้าทาย ไม่ว่าจะเป็นการแยกข้อมูลหรือการแก้ไขเนื้อหา หากคุณต้องการเพิ่มคำอธิบายประกอบภาพ เช่น การขีดฆ่าโดยตรงภายในแอปพลิเคชัน Java ของคุณโดยใช้ Aspose.PDF สำหรับ Java คู่มือนี้เหมาะสำหรับคุณ คุณจะได้เรียนรู้วิธีผสานรวมฟังก์ชัน PDF เข้ากับโปรเจ็กต์ของคุณอย่างราบรื่น

### สิ่งที่คุณจะได้เรียนรู้
- วิธีการเริ่มต้นและโหลดเอกสาร PDF ด้วย Aspose.PDF
- เทคนิคในการค้นหาข้อความบางส่วนที่เจาะจงใน PDF
- ขั้นตอนในการสร้างและกำหนดค่าคำอธิบายการขีดฆ่า
- วิธีการบันทึกเอกสาร PDF ที่ถูกแก้ไข

มาเจาะลึกข้อกำหนดเบื้องต้นที่คุณจะต้องมีก่อนที่เราจะเริ่มกัน

## คู่มือการใช้งาน
เราจะแบ่งการใช้งานของเราออกเป็นส่วนๆ ตามฟังก์ชันการทำงาน: การเริ่มต้นเอกสาร การค้นหาส่วนของข้อความ การสร้างคำอธิบายประกอบ และการบันทึกเอกสาร

### การเริ่มต้นและการโหลดเอกสาร
#### ภาพรวม
ขั้นแรก เราต้องโหลดไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ Java ขั้นตอนนี้เกี่ยวข้องกับการเริ่มต้น `Document` คลาสที่มีเส้นทางไปยัง PDF ของคุณ

**ขั้นตอนที่ 1**นำเข้าคลาสที่จำเป็น
```java
import com.aspose.pdf.Document;
```

**ขั้นตอนที่ 2**: กำหนดไดเรกทอรีเอกสารของคุณและเริ่มต้นวัตถุเอกสาร
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document document = new Document(dataDir + "/input.pdf");
```
โค้ดนี้โหลด PDF จาก `dataDir` เข้าไปใน `Document` เช่น อนุญาตให้ดำเนินการเพิ่มเติมได้

### การดูดซับชิ้นส่วนข้อความ
#### ภาพรวม
การค้นหาข้อความใน PDF ของคุณเป็นสิ่งสำคัญก่อนที่จะสร้างคำอธิบายประกอบ เราจะใช้ `TextFragmentAbsorber`-

**ขั้นตอนที่ 1**: นำเข้าคลาส
```java
import com.aspose.pdf.TextFragmentAbsorber;
```

**ขั้นตอนที่ 2**: สร้างอินสแตนซ์ของ TextFragmentAbsorber ด้วยคำค้นหา
```java
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("Sample");
```

**ขั้นตอนที่ 3**:ยอมรับตัวดูดซับบนแต่ละหน้าเพื่อค้นหาชิ้นส่วนข้อความที่ตรงกันทั้งหมด
```java
for (int i = 1; i <= document.getPages().size(); i++) {
    document.getPages().get_Item(i).accept(textFragmentAbsorber);
}
```
ลูปนี้จะค้นหาในแต่ละหน้าเพื่อรวบรวมรายการที่ตรงกับ "ตัวอย่าง"

### ดึงข้อมูลคอลเลกชันเศษส่วนข้อความ
#### ภาพรวม
เมื่อดูดซับชิ้นส่วนข้อความแล้ว เราจะดึงชิ้นส่วนเหล่านั้นมาไว้ในคอลเลกชันเพื่อประมวลผลคำอธิบายประกอบ

**ขั้นตอนที่ 1**: นำเข้าคลาสที่จำเป็น
```java
import com.aspose.pdf.TextFragmentCollection;
```

**ขั้นตอนที่ 2**: นำคอลเลกชันจากตัวดูดซับ
```java
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.getTextFragments();
```
วิธีการนี้ทำให้เราสามารถเข้าถึงอินสแตนซ์ที่พบทั้งหมดของ "ตัวอย่าง"

### สร้างคำอธิบายขีดฆ่า
#### ภาพรวม
เราจะเพิ่มคำอธิบายขีดฆ่าเหนือส่วนข้อความที่ระบุแต่ละส่วน โดยเน้นหรือทำเครื่องหมายให้ชัดเจน

**ขั้นตอนที่ 1**นำเข้าคลาสที่จำเป็น
```java
import com.aspose.pdf.StrikeOutAnnotation;
import com.aspose.pdf.Rectangle;
import com.aspose.pdf.Border;
import com.aspose.pdf.Color;
import com.aspose.pdf.Page;
```

**ขั้นตอนที่ 2**:วนซ้ำผ่านชิ้นส่วนข้อความและสร้างคำอธิบายประกอบ
```java
for (int j = 1; j <= textFragmentCollection.size(); j++) {
    com.aspose.pdf.TextFragment textFragment = textFragmentCollection.get_Item(j);

    // กำหนดพื้นที่สี่เหลี่ยมสำหรับคำอธิบายตามตำแหน่งของชิ้นส่วนข้อความ
    Rectangle rect = new Rectangle((float) textFragment.getPosition().getXIndent(),
                                   (float) textFragment.getPosition().getYIndent(),
                                   (float) textFragment.getPosition().getXIndent() + (float) textFragment.getRectangle().getWidth(),
                                   (float) textFragment.getPosition().getYIndent() + (float) textFragment.getRectangle().getHeight());

    // สร้างและกำหนดค่าคำอธิบายการขีดฆ่า
    StrikeOutAnnotation strikeOut = new StrikeOutAnnotation(textFragment.getPage(), rect);
    strikeOut.setOpacity(0.80f);  // ตั้งค่าความทึบเป็น 80%
    strikeOut.setBorder(new Border(strikeOut));  // กำหนดขอบเขตให้กับคำอธิบายประกอบ
    strikeOut.setColor(Color.getRed());  // ตั้งค่าสีคำอธิบายเป็นสีแดง

    // เพิ่มคำอธิบายลงในหน้า
    textFragment.getPage().getAnnotations().add(strikeOut);
}
```
คำอธิบายประกอบแต่ละรายการจะได้รับการกำหนดค่าความทึบ ขอบ และการตั้งค่าสีก่อนที่จะเพิ่ม

### บันทึกเอกสารพร้อมคำอธิบายประกอบ
#### ภาพรวม
หลังจากสร้างคำอธิบายแล้ว เราจะบันทึกเอกสารที่มีคำอธิบาย

**ขั้นตอนที่ 1**: นำเข้าคลาสที่จำเป็นหากยังไม่ได้ดำเนินการ
```java
import com.aspose.pdf.Document;
```

**ขั้นตอนที่ 2**: กำหนดไดเรกทอรีเอาท์พุตและบันทึกเอกสาร
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
document.save(outputDir + "/StrikeOut.pdf");
```
ขั้นตอนนี้จะทำให้ PDF ของคุณพร้อมคำอธิบายสมบูรณ์โดยที่การปรับเปลี่ยนทั้งหมดยังคงเหมือนเดิม

### สิ่งที่คุณจะได้เรียนรู้:

## คู่มือการใช้งาน

เรามาแบ่งกระบวนการออกเป็นสองคุณสมบัติหลัก: การตั้งค่าและการรับปัจจัยการซูมในเอกสาร PDF โดยใช้ Java

### การตั้งค่าปัจจัยการซูม

#### ภาพรวม
คุณลักษณะนี้ช่วยให้คุณระบุได้ว่าเนื้อหาจะปรากฏขนาดใหญ่หรือเล็กแค่ไหนเมื่อเปิด PDF เพื่อเพิ่มประสบการณ์การใช้งานของผู้ใช้ตั้งแต่เริ่มต้น

#### การดำเนินการทีละขั้นตอน:
**1. การนำเข้าพัสดุที่จำเป็น**
เริ่มต้นด้วยการนำเข้าคลาสที่จำเป็น:
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.XYZExplicitDestination;
import com.aspose.pdf.GoToAction;
```

**2. กำหนดเส้นทางไฟล์และระดับการซูม**
ตั้งค่าไดเร็กทอรีเอกสารและปัจจัยการซูมที่ต้องการ
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";
double zoom = 0.5; // ตัวอย่าง: ระดับการซูม 50%
```

**3. โหลดเอกสาร PDF**
สร้าง `Document` วัตถุที่จะโหลดไฟล์ PDF ที่มีอยู่ที่คุณต้องการปรับเปลี่ยน
```java
Document doc = new Document(dataDir + "/HelloWorld.pdf");
```

**4. สร้าง XYZExplicitDestination**
ตั้งค่าปลายทางด้วยระดับการซูม ความกว้าง และความสูงที่คุณต้องการตามขนาดกล่องสื่อของหน้าแรก
```java
XYZExplicitDestination destination = new XYZExplicitDestination(
    doc.getPages().get_Item(1),
    doc.getPages().get_Item(1).getMediaBox().getWidth(),
    doc.getPages().get_Item(1).getMediaBox().getHeight(), 
    zoom);
```

**5. กำหนดการซูมเป็นการดำเนินการแบบเปิด**
สร้าง `GoToAction` โดยใช้จุดหมายปลายทางของคุณและตั้งค่าเป็นการดำเนินการเปิดเอกสาร
```java
GoToAction actionzoom = new GoToAction(destination);
doc.setOpenAction(actionzoom);
```

**6. บันทึกการเปลี่ยนแปลงของคุณ**
สุดท้ายให้บันทึก PDF ที่แก้ไขแล้วเพื่อใช้การตั้งค่าการซูม
```java
String outputPath = outputDir + "/Zoomed_actionzoom.pdf";
doc.save(outputPath);
```

### การดึงข้อมูลปัจจัยการซูม

#### ภาพรวม
การเข้าถึงระดับการซูมปัจจุบันที่กำหนดไว้ใน PDF ถือเป็นสิ่งสำคัญในการทำความเข้าใจว่าเอกสารของคุณนำเสนออย่างไรเมื่อเปิดขึ้น

#### การดำเนินการทีละขั้นตอน:
**1. โหลดเอกสารที่แก้ไขแล้ว**
เริ่มต้นด้วยการโหลดเอกสารที่บันทึกไว้ก่อนหน้านี้
```java
Document doc1 = new Document(outputDir + "/Zoomed_actionzoom.pdf");
```

**2. เข้าถึงการดำเนินการแบบเปิดและดึงระดับการซูม**
สกัด `GoToAction` จากการดำเนินการเปิดแล้วรับระดับการซูมจาก `XYZExplicitDestination`-
```java
GoToAction action = (GoToAction) doc1.getOpenAction();
XYZExplicitDestination destination = (XYZExplicitDestination) action.getDestination();
double retrievedZoom = destination.getZoom();
```
ตัวแปร `retrievedZoom` ตอนนี้ถือปัจจัยการซูมปัจจุบันของเอกสารของคุณแล้ว

3. **เครื่องมือการทำงานร่วมกัน**:ให้แน่ใจว่าสมาชิกในทีมเห็นเอกสารในขนาดที่เหมาะสมในระหว่างการตรวจสอบ

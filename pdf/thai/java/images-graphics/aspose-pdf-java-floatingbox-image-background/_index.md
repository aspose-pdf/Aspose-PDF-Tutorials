## คู่มือการใช้งาน
### การเพิ่มรูปภาพเป็นพื้นหลังให้กับกล่องลอย
สำรวจวิธีการเพิ่มรูปภาพพื้นหลังให้กับกล่องลอยในเอกสาร PDF โดยใช้ Aspose.PDF สำหรับ Java

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์ของวัตถุเอกสาร
สร้างใหม่ `Document` วัตถุ:
```java
doc = new Document();
```

#### ขั้นตอนที่ 2: เพิ่มหน้าลงใน PDF ของคุณ
เพิ่มหน้าสำหรับกล่องลอยของคุณ:
```java
Page page = doc.getPages().add();
```

#### ขั้นตอนที่ 3: สร้างและกำหนดค่า FloatingBox
ตั้งค่า `FloatingBox` ดังต่อไปนี้:
```java
FloatingBox aBox = new FloatingBox(200, 100);
aBox.setLeft(40); // ตั้งค่าตำแหน่งด้านซ้าย
aBox.setTop(80);  // ตั้งตำแหน่งบนสุด
aBox.setHorizontalAlignment(HorizontalAlignment.Center); // จัดวางแนวนอน

// เพิ่มข้อความภายในกล่องลอย
aBox.getParagraphs().add(new TextFragment("main text"));

// กำหนดคุณสมบัติของเส้นขอบ
aBox.setBorder(new BorderInfo(BorderSide.All, Color.getRed()));
```

#### ขั้นตอนที่ 4: ตั้งค่าภาพพื้นหลัง
วิธีเพิ่มรูปภาพเป็นพื้นหลัง:
```java
Image backgroundImage = new Image();
aBox.setBackgroundImage(backgroundImage);
backgroundImage.setFile("YOUR_DOCUMENT_DIRECTORY/aspose-logo.jpg");
```

**คำอธิบาย**: เดอะ `setBackgroundImage` วิธีการแนบรูปภาพที่ระบุลงในกล่องลอย ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกต้อง

#### ขั้นตอนที่ 5: เพิ่ม FloatingBox ลงในเพจ
ผนวกกล่องลอยที่กำหนดค่าแล้วลงในหน้าของคุณ:
```java
page.getParagraphs().add(aBox);
```

#### ขั้นตอนที่ 6: บันทึกเอกสารของคุณ
บันทึกเอกสารที่มีพื้นหลังเป็นรูปภาพ:
```java
doc.save("YOUR_OUTPUT_DIRECTORY/AddImageStampAsBackgroundInFloatingBox_out.pdf");
```

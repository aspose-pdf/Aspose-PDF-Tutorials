Java\nหากต้องการเริ่มใช้ Aspose.PDF สำหรับ Java ให้เพิ่มเป็นส่วนที่ต้องพึ่งพาในโครงการของคุณ นี่คือวิธีที่คุณสามารถทำได้ด้วย Maven และ Gradle:\n\n**เมเวน:**\น```xml\n<dependency>\n    <groupId>com.aspose</groupId>\n    <artifactId>aspose-pdf</artifactId>\n    <version>25.3</version>\n</dependency>\n```\n\n**เกรเดิ้ล:**\น```gradle\nimplementation

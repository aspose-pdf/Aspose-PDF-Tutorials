## คู่มือการใช้งาน

### การแยกข้อความจากหน้าใดหน้าหนึ่ง

**ภาพรวม:** ฟีเจอร์นี้ช่วยให้คุณแยกข้อความจากหน้าใดหน้าหนึ่งในเอกสาร PDF ได้ เหมาะมากในกรณีที่คุณต้องการข้อมูลจากเพียงส่วนหนึ่งของเอกสาร

#### ขั้นตอน:

##### ตั้งค่าเอกสารและอุปกรณ์
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.Page;
import com.aspose.pdf.TextExtractionOptions;
import com.aspose.pdf.devices.TextDevice;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

Document pdfDocument = new Document(dataDir + "/input.pdf");
TextDevice textDevice = new TextDevice();
```

##### กำหนดค่าตัวเลือกการแยกข้อความ
```java
TextExtractionOptions textExtOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Raw);
textDevice.setExtractionOptions(textExtOptions);
```
ที่นี่เราตั้งค่าโหมดการสกัดเป็น `Raw`ซึ่งจะแยกข้อความออกมาโดยไม่ต้องจัดรูปแบบใดๆ

##### ประมวลผลหน้าเฉพาะ
```java
// ดึงข้อความจากหน้าแรกของเอกสาร PDF
textDevice.process(pdfDocument.getPages().get_Item(1), outputDir + "/ExtractedText.txt");
```
การ `process` วิธีการนี้จะกำหนดเป้าหมายไปที่หน้าเฉพาะและส่งออกข้อความไปยังไฟล์

#### เคล็ดลับการแก้ไขปัญหา:
- **ไม่พบไฟล์:** ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีของคุณถูกต้องและสามารถเข้าถึงได้

### ดึงข้อความจากทุกหน้าของ PDF

**ภาพรวม:** ฟีเจอร์นี้จะดึงข้อความจากทุกหน้าในเอกสาร ซึ่งมีประโยชน์สำหรับการวิเคราะห์แบบครอบคลุมหรือการทำงานดึงข้อมูล

#### ขั้นตอน:

##### เริ่มต้นสตรีมเอาท์พุต
```java
import java.io.FileOutputStream;
import java.io.OutputStream;

OutputStream text_stream = new FileOutputStream(outputDir + "/ExtractedText.txt", false);
```
สตรีมเอาต์พุตจะเขียนข้อความที่แยกออกมาทั้งหมดลงในไฟล์เดียว

##### วนซ้ำผ่านทุกหน้า
```java
for (Page page : (Iterable<Page>) pdfDocument.getPages()) {
    TextDevice textDevice = new TextDevice();
    TextExtractionOptions textExtOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Raw);
    textDevice.setExtractionOptions(textExtOptions);

    // ดึงข้อความจากแต่ละหน้า
    textDevice.process(page, text_stream);
}
```
ทำซ้ำในทุกหน้าและแยกข้อความโดยใช้การตั้งค่าแบบเดียวกับก่อนหน้านี้

##### ปิดสตรีมเอาท์พุต
```java
text_stream.close();
```
ปิดสตรีมเพื่อปล่อยทรัพยากรเสมอเมื่อการประมวลผลเสร็จสิ้น

#### เคล็ดลับการแก้ไขปัญหา:
- **ปัญหาหน่วยความจำ:** สำหรับ PDF ขนาดใหญ่ ควรพิจารณาแยกส่วนที่เล็กกว่าหรือเพิ่มประสิทธิภาพการใช้หน่วยความจำในสภาพแวดล้อมของคุณ

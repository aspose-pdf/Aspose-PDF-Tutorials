## คู่มือการใช้งาน

### การฝัง LaTeX ใน PDF ด้วย Aspose.PDF สำหรับ Java

#### ภาพรวม
การฝังสคริปต์ LaTeX ลงในไฟล์ PDF จะช่วยเพิ่มประสิทธิภาพการใช้งาน โดยเฉพาะสมการทางคณิตศาสตร์และเอกสารทางวิทยาศาสตร์

#### การดำเนินการแบบทีละขั้นตอน

**1. สร้างเอกสารใหม่**

```java
import com.aspose.pdf.Document;
import com.aspose.pdf.LatexFragment;
import com.aspose.pdf.Page;

// เริ่มต้นอินสแตนซ์เอกสารใหม่
Document doc = new Document();
```
*คำอธิบาย*: เดอะ `Document` คลาสแสดงถึงเอกสาร PDF ที่คุณจะใช้งาน

**2. เพิ่มหน้า**

```java
Page page = doc.getPages().add();
```
*คำอธิบาย*การเพิ่มหน้าลงในเอกสารของคุณเป็นสิ่งที่จำเป็นก่อนที่จะแทรกเนื้อหาใด ๆ รวมถึงชิ้นส่วน LaTeX

**3. กำหนดสคริปต์ LaTeX**

กำหนดสคริปต์เป็น `String`นี่คือตัวอย่างที่รวมถึงสัญลักษณ์ทางคณิตศาสตร์:

```java
String latexScript =
    "\\usepackage{amsmath,amsthm}" +
    "\\begin{document}" +
    "\\begin{proof} The proof is as follows: " +
    "\\begin{align}" +
    "(x+y)^3&=(x+y)(x+y)^2" +
    "(x+y)(x^2+2xy+y^2)\" +
    "&=x^3+3x^2y+3xy^3+x^3.\\qedhere" +
    "\\end{align}" +
    "\\end{proof}" +
    "\\end{document}";
```
*วัตถุประสงค์*สคริปต์นี้สาธิตวิธีการรวมสมการทางคณิตศาสตร์โดยใช้ไวยากรณ์ LaTeX

**4. สร้าง LatexFragment**

```java
LatexFragment latex = new LatexFragment(latexScript);
```
*คำอธิบาย*: เดอะ `LatexFragment` คลาสจะแปลงสตริง LaTeX ของคุณให้เป็นอ็อบเจ็กต์ที่สามารถฝังลงในเอกสาร PDF ได้

**5. เพิ่มชิ้นส่วนลงในหน้า**

```java
page.getParagraphs().add(latex);
```
*คำอธิบาย*ขั้นตอนนี้จะเพิ่มส่วน LaTeX ลงในคอลเล็กชันย่อหน้าของหน้า เพื่อให้แน่ใจว่าส่วนดังกล่าวจะแสดงผลเมื่อดู PDF

**6. บันทึกเอกสาร**

```java
doc.save("output/Script_out.pdf");
```
*วัตถุประสงค์*การบันทึกเอกสารจะทำให้การเพิ่มทั้งหมดเสร็จสมบูรณ์และสร้างไฟล์ PDF ในไดเร็กทอรีที่คุณระบุ

### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าไวยากรณ์ LaTeX ของคุณถูกต้องเพื่อหลีกเลี่ยงปัญหาการเรนเดอร์
- ตรวจสอบแพ็คเกจที่ขาดหายไปในสคริปต์ LaTeX ของคุณซึ่งอาจทำให้เกิดข้อผิดพลาดระหว่างการคอมไพล์

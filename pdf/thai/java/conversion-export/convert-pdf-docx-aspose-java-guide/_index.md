## คู่มือการใช้งาน

สำรวจคุณลักษณะหลักสามประการของการแปลงไฟล์ PDF เป็นรูปแบบ DOC/DOCX โดยใช้ Aspose.PDF สำหรับ Java

### บันทึก PDF เป็นรูปแบบ DOC
#### ภาพรวม
คุณลักษณะนี้สาธิตการแปลงพื้นฐานจากไฟล์ PDF เป็นรูปแบบ Microsoft Word DOC เหมาะสำหรับการแปลงแบบตรงไปตรงมาโดยไม่ต้องมีข้อกำหนดเค้าโครงที่ซับซ้อน

#### ขั้นตอน:
##### ขั้นตอนที่ 1: โหลดเอกสาร PDF ต้นฉบับ
```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/input2.pdf");
```
##### ขั้นตอนที่ 2: บันทึกเป็นไฟล์ DOC
```java
pdfDocument.save("YOUR_OUTPUT_DIRECTORY/TableHeightIssue.doc", SaveFormat.Doc);
```

### บันทึก PDF เป็นรูปแบบ DOCX
#### ภาพรวม
ฟีเจอร์นี้แสดงวิธีการแปลงไฟล์ PDF เป็นรูปแบบ Microsoft Word DOCX โดยใช้ `DocSaveOptions` เพื่อการควบคุมการตั้งค่าเอาท์พุตที่มากยิ่งขึ้น

#### ขั้นตอน:
##### ขั้นตอนที่ 1: โหลดเอกสาร PDF ต้นฉบับ
```java
Document doc = new Document(dataDir + "/input.pdf");
```
##### ขั้นตอนที่ 2: กำหนดค่า DocSaveOptions
```java
import com.aspose.pdf.DocSaveOptions;

DocSaveOptions saveOptions = new DocSaveOptions();
saveOptions.setFormat(DocSaveOptions.DocFormat.DocX);
```
##### ขั้นตอนที่ 3: บันทึกเป็นไฟล์ DOCX
```java
doc.save("YOUR_OUTPUT_DIRECTORY/savingToDOCX.docx", saveOptions);
```

### การใช้ DocSaveOptions สำหรับการตั้งค่าการแปลงขั้นสูง
#### ภาพรวม
คุณสมบัตินี้จะสำรวจการตั้งค่าขั้นสูงโดยใช้ `DocSaveOptions` เพื่อเพิ่มความแม่นยำในการแปลง โดยเฉพาะอย่างยิ่งมีประโยชน์เมื่อทำการรักษาเค้าโครงที่ซับซ้อน

#### ขั้นตอน:
##### ขั้นตอนที่ 1: โหลดเอกสาร PDF ต้นฉบับ
```java
Document document = new Document(dataDir + "/source.pdf");
```
##### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการบันทึกขั้นสูง
```java
DocSaveOptions saveOption = new DocSaveOptions();
saveOption.setMode(DocSaveOptions.RecognitionMode.Flow);
saveOption.setRelativeHorizontalProximity(2.5f);
saveOption.setRecognizeBullets(true);
```
##### ขั้นตอนที่ 3: บันทึกด้วยการตั้งค่าขั้นสูง
```java
document.save("YOUR_OUTPUT_DIRECTORY/usingTheDocSaveOptionsClass.doc", saveOption);
```

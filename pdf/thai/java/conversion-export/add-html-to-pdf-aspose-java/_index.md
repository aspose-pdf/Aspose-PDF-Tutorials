## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นสองคุณลักษณะหลัก: การเพิ่มเนื้อหา HTML และการตั้งค่าสถานะข้อความ

### การเพิ่มเนื้อหา HTML โดยใช้ DOM

**ภาพรวม:** คุณลักษณะนี้ช่วยให้คุณแทรกส่วน HTML ลงในเอกสาร PDF ทำให้สามารถจัดรูปแบบที่สมบูรณ์ภายในหน้า PDF ของคุณได้

**ขั้นตอน:**

1. **สร้างวัตถุเอกสาร**
   
   ```java
   import com.aspose.pdf.Document;
   
   Document doc = new Document();
   ```

2. **เพิ่มหน้าลงในเอกสาร**
   
   ```java
   import com.aspose.pdf.Page;

   Page page = doc.getPages().add();
   ```

3. **สร้างส่วน HTML**
   
   ```java
   import com.aspose.pdf.HtmlFragment;
   
   String htmlContent = "<p style='font-family: Verdana'><b><i>Table contains text</i></b></p>";
   HtmlFragment title = new HtmlFragment(htmlContent);
   ```

4. **ตั้งค่าสถานะข้อความ**
   
   กำหนดค่าแบบอักษรและขนาดสำหรับเนื้อหา HTML ของคุณ:
   
   ```java
   import com.aspose.pdf.TextState;

   title.setTextState(new TextState("Arial", 20));
   ```

5. **กำหนดค่าระยะขอบ**
   
   ปรับระยะขอบเพื่อวางตำแหน่งส่วน HTML ของคุณอย่างถูกต้อง:
   
   ```java
   title.getMargin().setBottom(10);
   title.getMargin().setTop(400);
   ```

6. **เพิ่มชิ้นส่วนลงในหน้า**
   
   ```java
   page.getParagraphs().add(title);
   ```

7. **บันทึกเอกสาร**
   
   ```java
   String outputDir = "YOUR_OUTPUT_DIRECTORY";
   doc.save(outputDir + "/AddHTMLUsingDOMAndOverwrite_out.pdf");
   ```

**เคล็ดลับการแก้ไขปัญหา:**
- ตรวจสอบให้แน่ใจว่าเนื้อหา HTML ของคุณมีรูปแบบถูกต้องเพื่อหลีกเลี่ยงปัญหาการแสดงผล
- ตรวจสอบเส้นทางไฟล์อีกครั้งเพื่อบันทึกเอกสาร

### การตั้งค่าสถานะข้อความและระยะขอบ

**ภาพรวม:** คุณลักษณะนี้มุ่งเน้นที่การปรับแต่งรูปแบบข้อความในส่วน HTML ของคุณ เพื่อเพิ่มความสามารถในการอ่านและการนำเสนอ

**ขั้นตอน:**

ทำซ้ำขั้นตอนที่ 3 ถึง 7 จากส่วนก่อนหน้าด้วยการกำหนดค่าที่แตกต่างกันหรือมีข้อกำหนดการจัดรูปแบบข้อความเพิ่มเติมตามต้องการ 

## การประยุกต์ใช้งานจริง

ต่อไปนี้คือสถานการณ์จริงบางสถานการณ์ที่การเพิ่ม HTML ลงใน PDF สามารถเป็นประโยชน์อย่างยิ่ง:

1. **รายงานแบบไดนามิก:** สร้างรายงานทางการเงินโดยอัตโนมัติซึ่งรวมตารางและสรุปที่มีรูปแบบต่างๆ
2. **ใบแจ้งหนี้:** สร้างใบแจ้งหนี้ที่ดูเป็นมืออาชีพด้วยแบบอักษรและเค้าโครงที่กำหนดเองสำหรับลูกค้าที่แตกต่างกัน
3. **โบรชัวร์การตลาด:** พัฒนาโบรชัวร์ดิจิทัลที่รักษาความสอดคล้องของแบรนด์ในทุกแพลตฟอร์ม
4. **สื่อการเรียนรู้:** สร้างสื่อการสอนพร้อมเนื้อหาแบบโต้ตอบเช่นแบบทดสอบหรือกราฟิก

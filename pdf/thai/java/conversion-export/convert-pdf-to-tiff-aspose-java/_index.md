## คู่มือการใช้งาน
### แปลงหน้า PDF ทั้งหมดเป็นรูปภาพ TIFF
#### ภาพรวม
แปลงหน้าทั้งหมดของเอกสาร PDF ให้เป็นภาพ TIFF เดียวเพื่อการเก็บถาวรหรือเมื่อจำเป็นต้องใช้รูปแบบรวมกัน

#### คำแนะนำทีละขั้นตอน
**1. เปิดเอกสาร PDF**
```java
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
บรรทัดนี้จะเริ่มต้นเอกสาร PDF ของคุณจากเส้นทางที่ระบุ ช่วยให้คุณสามารถทำงานกับเนื้อหาผ่านโปรแกรมได้

**2. สร้างสตรีมเอาท์พุตสำหรับภาพ TIFF**
```java
java.io.OutputStream imageStream = new java.io.FileOutputStream("YOUR_OUTPUT_DIRECTORY/Converted_Image.tiff");
```
ที่นี่ เราตั้งค่าสตรีมเอาท์พุตโดยที่ไฟล์ TIFF ที่ได้จะถูกบันทึก

**3. ตั้งค่าความละเอียดและการตั้งค่า Tiff**
```java
Resolution resolution = new Resolution(300);
TiffSettings tiffSettings = new TiffSettings();
tiffSettings.setCompression(CompressionType.CCITT4);
tiffSettings.setDepth(ColorDepth.Format8bpp);
tiffSettings.setSkipBlankPages(true);
```
- **ปณิธาน:** กำหนด DPI ของภาพเอาต์พุต 300 DPI ถือเป็นมาตรฐานสำหรับการพิมพ์คุณภาพสูง
- **การบีบอัด:** การบีบอัด CCITT4 ช่วยลดขนาดไฟล์อย่างมีประสิทธิภาพโดยไม่สูญเสียคุณภาพ เหมาะอย่างยิ่งสำหรับภาพขาวดำ
- **ความลึกของสี:** 8bpp (บิตต่อพิกเซล) สร้างความสมดุลระหว่างรายละเอียดสีและขนาดไฟล์

**4. เริ่มต้น TiffDevice**
```java
TiffDevice tiffDevice = new TiffDevice(resolution, tiffSettings);
```
การ `TiffDevice` วัตถุใช้ความละเอียดและการตั้งค่าที่กำหนดไว้ข้างต้นเพื่อประมวลผลการแปลง

**5. แปลงและบันทึกภาพ TIFF**
```java
tiffDevice.process(pdfDocument, imageStream);
imageStream.close();
```
การ `process()` วิธีการนี้จะแปลงหน้าทั้งหมดเป็นไฟล์ TIFF ไฟล์เดียว อย่าลืมปิดสตรีมเสมอเพื่อให้แน่ใจว่าข้อมูลถูกเขียนอย่างถูกต้องและทรัพยากรได้รับการปลดปล่อย

### แปลงไฟล์ PDF หนึ่งหน้าเป็นภาพ TIFF
#### ภาพรวม
แปลงหน้าเฉพาะจาก PDF ของคุณเป็นรูปแบบ TIFF ตามความต้องการ

#### คำแนะนำทีละขั้นตอน
**1. เปิดเอกสาร PDF**
```java
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

**2. สร้างสตรีมเอาท์พุตสำหรับภาพ TIFF ของหน้าเฉพาะ**
```java
java.io.OutputStream imageStream = new java.io.FileOutputStream("YOUR_OUTPUT_DIRECTORY/Converted_Image_Page_1.tiff");
```

**3. ตั้งค่าความละเอียดและการตั้งค่า Tiff (เหมือนเดิม)**
ขั้นตอนการกำหนดค่าจะเหมือนกันกับขั้นตอนที่ใช้ในการแปลงหน้าทั้งหมด

**4. เริ่มต้น TiffDevice**
```java
TiffDevice tiffDevice = new TiffDevice(resolution, tiffSettings);
```

**5. แปลงหน้าเฉพาะ**
```java
tiffDevice.process(pdfDocument, 1, 1, imageStream); // แปลงเฉพาะหน้าแรกเท่านั้น
imageStream.close();
```
การ `process()` วิธีการนี้ใช้กับพารามิเตอร์เพิ่มเติมเพื่อระบุว่าจะแปลงหน้าใด (เริ่มต้นและสิ้นสุด)

## การประยุกต์ใช้งานจริง
- **การจัดเก็บถาวร:** แปลงเอกสารทางกฎหมายหรือประวัติศาสตร์เป็น TIFF เพื่อการจัดเก็บในระยะยาว
- **การประมวลผลภาพ:** ดึงภาพจาก PDF ที่สแกนเพื่อวิเคราะห์เพิ่มเติมโดยใช้เครื่องมือประมวลผลภาพ
- **การแบ่งปันเอกสาร:** แบ่งปันเอกสารหน้าเดียวในรูปแบบภาพโดยการแปลงเป็นรูปแบบรูปภาพ เพื่อให้มั่นใจว่าจะดูได้สอดคล้องกันในทุกอุปกรณ์

## การพิจารณาประสิทธิภาพ
เพื่อให้มั่นใจถึงประสิทธิภาพที่เหมาะสมที่สุด:
- ตรวจสอบการใช้หน่วยความจำเมื่อทำงานกับเอกสารขนาดใหญ่ พิจารณาแยกการแปลงออกเป็นงานย่อยๆ หากจำเป็น
- ปรับการตั้งค่าความละเอียดตามคุณภาพเอาต์พุตที่ต้องการและข้อจำกัดขนาดไฟล์
- ใช้ประเภทการบีบอัดที่เหมาะสมเพื่อสร้างสมดุลระหว่างคุณภาพของภาพและความต้องการในการจัดเก็บ

## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการใช้งานการแปลงข้อความเป็น PDF โดยใช้ Aspose.PDF ใน Java

### ขั้นตอนที่ 1: อ่านไฟล์ข้อความลงในอาร์เรย์ไบต์
เราเริ่มต้นด้วยการอ่านไฟล์ข้อความต้นฉบับลงในหน่วยความจำ ซึ่งจะทำให้เราสามารถประมวลผลเนื้อหาได้อย่างมีประสิทธิภาพ

```java
try {
    // เปิดไฟล์ข้อความต้นฉบับเป็นสตรีมอินพุต
    FileInputStream fis = new FileInputStream(dataDir + "test.txt");
    ByteArrayOutputStream bos = new ByteArrayOutputStream();
    byte[] buf = new byte[1024];
    int readNum;

    // อ่านข้อมูลจากไฟล์เป็นชิ้นๆ และเขียนลงใน ByteArrayOutputStream
    for (; (readNum = fis.read(buf)) != -1; ) {
        bos.write(buf, 0, readNum);
    }
    
    byte[] bytes = bos.toByteArray();
    ByteArrayInputStream srcStream = new ByteArrayInputStream(bytes);
    BufferedReader reader = new BufferedReader(new InputStreamReader(srcStream));
```

- **วัตถุประสงค์:** ขั้นตอนนี้จะช่วยให้แน่ใจว่าเนื้อหาทั้งหมดของไฟล์ข้อความของคุณถูกโหลดลงในหน่วยความจำเพื่อประมวลผล
  
### ขั้นตอนที่ 2: แปลงข้อมูลข้อความเป็นสตริงเดี่ยว
ถัดไป เราจะแปลงข้อมูลอาร์เรย์ไบต์ให้เป็นสตริงเดียว ซึ่งจะทำให้การเพิ่มข้อมูลลงในเอกสาร PDF ของเราง่ายดายขึ้น

```java
    // แปลงข้อมูลข้อความเป็นสตริงเดี่ยว
    StringBuilder builder = new StringBuilder(5024);
    String line;
    while ((line = reader.readLine()) != null) {
        builder.append(line).append("\r\n");
    }
```

- **วัตถุประสงค์:** การรวมบรรทัดเป็นสตริงเดียวทำให้การเพิ่มบรรทัดเป็นเนื้อหาข้อความใน PDF ของเราเป็นเรื่องง่าย
  
### ขั้นตอนที่ 3: สร้างเอกสาร PDF และเพิ่มเนื้อหาข้อความ
ในที่สุดเราก็สร้างเอกสาร PDF และแทรกข้อมูลข้อความของเรา

```java
    // สร้างเอกสาร PDF และเพิ่มเนื้อหาข้อความ
    Document doc = new Document();
    Page page = doc.getPages().add();
    TextFragment text = new TextFragment(builder.toString());
    page.getParagraphs().add(text);
    
    // บันทึก PDF ที่สร้างขึ้นไปยังไดเร็กทอรีเอาท์พุตที่ระบุ
    doc.save(outputDir + "TextFile_TexttoPDF.pdf");
} catch (IOException e) {
    e.printStackTrace();
}
```

- **วัตถุประสงค์:** ขั้นตอนนี้เกี่ยวข้องกับการสร้างเอกสารใหม่และผนวกเนื้อหาข้อความ จากนั้นบันทึกเป็น PDF
  
### เคล็ดลับการแก้ไขปัญหา
ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ (`dataDir` และ `outputDir`) ถูกกำหนดไว้อย่างถูกต้อง จัดการข้อยกเว้นเพื่อแก้ไขปัญหาที่เกี่ยวข้องกับการเข้าถึงไฟล์หรือการดำเนินการ I/O

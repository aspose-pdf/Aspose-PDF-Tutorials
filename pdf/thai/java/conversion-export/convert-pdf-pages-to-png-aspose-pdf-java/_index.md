## คู่มือการใช้งาน

มาแบ่งการใช้งานออกเป็นส่วนๆ ตามตรรกะ เพื่อทำความเข้าใจว่าแต่ละส่วนมีส่วนช่วยในการแปลงหน้า PDF เป็นภาพ PNG อย่างไร

### ขั้นตอนที่ 1: โหลดเอกสาร PDF

เริ่มต้นด้วยการโหลดไฟล์ PDF ที่ต้องการ ขั้นตอนนี้จะเตรียมเอกสารของคุณให้พร้อมสำหรับการประมวลผล

```java
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

**คำอธิบาย**: เดอะ `Document` คลาสโหลด PDF จากเส้นทางที่ระบุ ช่วยให้เราสามารถจัดการหน้าต่างๆ ได้

### ขั้นตอนที่ 2: วนซ้ำหน้าต่างๆ

ทำซ้ำในแต่ละหน้าใน PDF ลูปนี้คือจุดที่เราจะแปลงแต่ละหน้าเป็นรูปภาพ

```java
for (int pageCount = 1; pageCount <= pdfDocument.getPages().size(); pageCount++) {
    // ตรรกะการแปลงจะถูกวางไว้ที่นี่
}
```

**คำอธิบาย**โดยการวนซ้ำจาก `pageCount = 1` ถึง `pdfDocument.getPages().size()`เรามั่นใจว่ามีการประมวลผลทุกหน้า

### ขั้นตอนที่ 3: ตั้งค่าสตรีมเอาท์พุตและความละเอียด

สำหรับแต่ละหน้า ตั้งค่าสตรีมเอาท์พุตเพื่อบันทึกภาพ PNG และกำหนดความละเอียดสำหรับภาพคุณภาพสูง

```java
FileOutputStream imageStream = new FileOutputStream("YOUR_OUTPUT_DIRECTORY/Converted_Image" + pageCount + ".png");
Resolution resolution = new Resolution(300);
```

**คำอธิบาย**: เดอะ `FileOutputStream` เขียนข้อมูลภาพที่แปลงแล้วในขณะที่ `Resolution(300)` รับประกันผลลัพธ์ที่คมชัด

### ขั้นตอนที่ 4: แปลงและบันทึกแต่ละหน้า

สร้างวัตถุ PNGDevice ที่มีความละเอียดที่คุณระบุเพื่อจัดการกระบวนการแปลง

```java
PngDevice pngDevice = new PngDevice(resolution);
pngDevice.process(pdfDocument.getPages().get_Item(pageCount), imageStream);
imageStream.close();
```

**คำอธิบาย**: เดอะ `process` วิธีการแปลงและบันทึกแต่ละหน้าเป็น PNG และการปิดสตรีมจะทำให้ทรัพยากรว่างขึ้น

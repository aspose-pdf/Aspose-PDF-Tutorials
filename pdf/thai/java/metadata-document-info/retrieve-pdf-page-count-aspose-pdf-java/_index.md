## คู่มือการใช้งาน

มาแบ่งการใช้งานของเราออกเป็นสองคุณสมบัติหลัก: ดึงจำนวนหน้าจากเอกสารที่มีอยู่และการสร้างเอกสารเพื่อกำหนดจำนวนหน้าที่เป็นไปได้

### คุณสมบัติ 1: รับจำนวนหน้าของเอกสาร PDF

**ภาพรวม:** คุณสมบัตินี้ช่วยให้คุณเปิดไฟล์ PDF ที่มีอยู่และดึงจำนวนหน้าทั้งหมดโดยใช้ Aspose.PDF สำหรับ Java

#### ขั้นตอนที่ 1: เปิดเอกสาร PDF
เริ่มต้นโดยระบุไดเรกทอรีที่เอกสาร PDF อินพุตของคุณตั้งอยู่ แทนที่ `"YOUR_DOCUMENT_DIRECTORY"` ด้วยเส้นทางจริง:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // ตั้งค่าไดเร็กทอรี PDF ของคุณที่นี่
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

#### ขั้นตอนที่ 2: ดึงข้อมูลและแสดงจำนวนหน้า
เข้าถึง `Pages` ขนาดของคอลเลกชันเพื่อกำหนดจำนวนหน้าในเอกสารของคุณ:
```java
int pageCount = pdfDocument.getPages().size();
System.out.printf("Page Count :- " + pageCount);
```
### คุณสมบัติที่ 2: รับจำนวนหน้าโดยไม่ต้องบันทึกเอกสาร PDF

**ภาพรวม:** ฟีเจอร์นี้สาธิตการสร้าง PDF ใหม่ การเพิ่มเนื้อหาแบบไดนามิก และการกำหนดจำนวนหน้าผลลัพธ์โดยไม่ต้องบันทึก

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์ของวัตถุเอกสารใหม่
```java
document doc = new Document();
```

#### ขั้นตอนที่ 2: เพิ่มหน้าใหม่ลงในคอลเล็กชันหน้าของเอกสาร
```java
Page page = doc.getPages().add();
```

#### ขั้นตอนที่ 3: จำลองเนื้อหาหลายหน้าโดยการเพิ่มส่วนข้อความ
เพิ่มข้อความหลายส่วนเพื่อดูว่าเนื้อหาส่งผลต่อจำนวนหน้าอย่างไร:
```java
for (int i = 0; i < 300; i++) {
    page.getParagraphs().add(new com.aspose.pdf.TextFragment("Pages count test"));
}
```
ดึงข้อมูลจำนวนหน้าหลังจากเพิ่มเนื้อหา:
```java
int simulatedPageCount = doc.getPages().size();
System.out.printf("Simulated Page Count :- " + simulatedPageCount);
```

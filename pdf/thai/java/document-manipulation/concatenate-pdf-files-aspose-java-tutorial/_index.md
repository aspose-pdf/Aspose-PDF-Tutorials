PdfFileEditor\n\n `PdfFileEditor` คลาสนี้ให้วิธีการสำหรับการแก้ไขและจัดการไฟล์ PDF ซึ่งเป็นสิ่งสำคัญสำหรับงานของเรา:\n\n```java\n// Instantiate PdfFileEditor\nPdfFileEditor pdfEditor = new

# วิธีการเพิ่มวันที่หมดอายุลงใน PDF โดยใช้ Aspose.PDF Java

## การแนะนำ
คุณกำลังต้องการเพิ่มวันที่หมดอายุในเอกสาร PDF อยู่หรือไม่ ฟีเจอร์นี้มีประโยชน์อย่างยิ่งเมื่อต้องแชร์ข้อมูลที่ละเอียดอ่อนหรือเกี่ยวข้องกับเวลา โดยรับรองว่าเอกสารจะมีผลใช้ได้เฉพาะช่วงเวลาที่กำหนดเท่านั้น ในบทช่วยสอนนี้ เราจะมาเรียนรู้วิธีตั้งค่าการดำเนินการ JavaScript ใน PDF โดยใช้ไลบรารี Aspose.PDF อันทรงพลังสำหรับ Java

### สิ่งที่คุณจะได้เรียนรู้:
- วิธีการเพิ่มฟังก์ชันการหมดอายุให้กับไฟล์ PDF ด้วย Aspose.PDF
- การตั้งค่าและกำหนดค่าสภาพแวดล้อมของคุณให้ทำงานกับ Aspose.PDF
- การนำ JavaScript มาใช้งานใน PDF เพื่อตรวจสอบวันที่และแจ้งเตือน

ก่อนจะเริ่มดำเนินการ ตรวจสอบให้แน่ใจก่อนว่าคุณมีทุกสิ่งที่จำเป็นเพื่อให้ปฏิบัติตามได้อย่างราบรื่น

## คู่มือการใช้งาน
ในส่วนนี้ เราจะแบ่งการใช้งานออกเป็นส่วนๆ ที่จัดการได้ โดยเน้นที่การเพิ่มฟังก์ชันการหมดอายุด้วยการดำเนินการ JavaScript โดยใช้ Aspose.PDF Java
### การเพิ่มฟังก์ชันการหมดอายุ
คุณสมบัติหลักของบทช่วยสอนของเราคือการฝังการดำเนินการ JavaScript ลงใน PDF ของคุณ ซึ่งจะตรวจสอบว่าเอกสารหมดอายุหรือไม่เมื่อเปิดขึ้น นี่คือวิธีดำเนินการดังกล่าว:
#### ขั้นตอนที่ 1: โหลดเอกสาร PDF ของคุณ
เริ่มต้นด้วยการโหลดไฟล์ PDF ที่มีอยู่จากไดเร็กทอรีของคุณโดยใช้ Aspose.PDF
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // ตั้งค่าเส้นทางไดเร็กทอรี PDF อินพุต
Document doc = new Document(dataDir + "/input.pdf");
```
รหัสนี้จะเริ่มต้น `Document` วัตถุที่แสดงถึง PDF ของคุณ คุณจะต้องแทนที่ `"YOUR_DOCUMENT_DIRECTORY"` ด้วยเส้นทางจริงที่คุณเก็บไฟล์ PDF ไว้
#### ขั้นตอนที่ 2: กำหนด JavaScript สำหรับการตรวจสอบวันหมดอายุ
ถัดมา ให้กำหนด JavaScript ที่จะตรวจสอบว่าวันที่ปัจจุบันเกินวันที่หมดอายุหรือไม่ และจะแสดงการแจ้งเตือนหากเป็นเช่นนั้น
```java
JavascriptAction javaScript = new JavascriptAction(
    "var year=2014; var month=4; today = new Date(); today = new Date(today.getFullYear(), today.getMonth());" +
    "expiry = new Date(year, month);" +
    "if (today.getTime() > expiry.getTime()) app.alert('The file is expired. You need a new one.');"
);
```
สคริปต์นี้จะตั้งวันที่หมดอายุ (`2014-04`) และเปรียบเทียบกับวันที่ปัจจุบันทุกครั้งที่เปิด PDF หากวันที่เปิดเอกสารเกินกว่านี้ ระบบจะส่งการแจ้งเตือนไปยังผู้ใช้
#### ขั้นตอนที่ 3: ตั้งค่า JavaScript เป็น Open Action
รวมการดำเนินการ JavaScript ลงใน PDF ของคุณเพื่อให้ทำงานเมื่อเปิด:
```java
doc.setOpenAction(javaScript);
```
บรรทัดโค้ดนี้จะช่วยให้มั่นใจว่าทุกครั้งที่คุณเปิด PDF JavaScript จะทำงานโดยอัตโนมัติเพื่อตรวจสอบวันหมดอายุ
#### ขั้นตอนที่ 4: บันทึก PDF ที่แก้ไขแล้ว
สุดท้าย ให้บันทึกเอกสารที่มีการดำเนินการ JavaScript ที่เพิ่มใหม่ไปยังไดเร็กทอรีเอาต์พุตที่ระบุ
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // ตั้งค่าเส้นทางไดเร็กทอรี PDF เอาท์พุต
doc.save(outputDir + "/JavaScript-Added.pdf");
```
แทนที่ `"YOUR_OUTPUT_DIRECTORY"` ด้วยเส้นทางโฟลเดอร์เอาต์พุตที่คุณต้องการ การดำเนินการนี้จะบันทึก PDF ที่คุณแก้ไขแล้ว พร้อมสำหรับการแจกจ่ายพร้อมฟีเจอร์การหมดอายุ
### เคล็ดลับการแก้ไขปัญหา
ปัญหาทั่วไปอาจรวมถึงเส้นทางไฟล์ไม่ถูกต้องหรือข้อผิดพลาดของ JavaScript ตรวจสอบเส้นทางไดเรกทอรีของคุณอีกครั้งและให้แน่ใจว่าไวยากรณ์ JavaScript ของคุณไม่มีข้อผิดพลาด หากปัญหายังคงมีอยู่ โปรดดูเอกสารหรือฟอรัมของ Aspose.PDF เพื่อขอรับการสนับสนุน

# ปิดใช้งานการบีบอัดไฟล์ใน PDF โดยใช้ Aspose.PDF สำหรับ Java: คำแนะนำทีละขั้นตอน

## การแนะนำ

## การตั้งค่า Aspose.PDF สำหรับ Java

ในการทำงานกับ Aspose.PDF สำหรับ Java ให้ตั้งค่าไลบรารีโดยใช้ Maven หรือ Gradle:

### การใช้ Maven
เพิ่มการอ้างอิงนี้ให้กับคุณ `pom.xml`-
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```
### การใช้ Gradle
รวมสิ่งนี้ไว้ในของคุณ `build.gradle` ไฟล์:
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## คู่มือการใช้งาน

ตอนนี้สภาพแวดล้อมของคุณได้รับการตั้งค่าเรียบร้อยแล้ว ให้ปิดการใช้งานการบีบอัดไฟล์สำหรับทรัพยากรที่ฝังไว้

### ขั้นตอนที่ 1: โหลดเอกสาร PDF ของคุณ
โหลดไฟล์ PDF ต้นฉบับลงใน `Document` วัตถุ:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอกสารของคุณ
Path pdfPath = Paths.get(dataDir + "input.pdf");
byte[] data = Files.readAllBytes(pdfPath);
InputStream inputStream = new ByteArrayInputStream(data);

Document pdfDocument = new Document(inputStream);
```
*เหตุใดจึงก้าวขั้นนี้?* การโหลด PDF เป็นสิ่งจำเป็นสำหรับการจัดการเนื้อหา

### ขั้นตอนที่ 2: เพิ่มไฟล์เป็นทรัพยากรที่ฝังไว้
สร้าง `FileSpecification` และตั้งค่าการเข้ารหัสเป็น `None` เพื่อป้องกันการบีบอัด:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอาท์พุตของคุณ

FileSpecification fileSpecification = new FileSpecification(dataDir + "test.txt", "Sample text file");
fileSpecification.setEncoding(FileEncoding.None);

pdfDocument.getEmbeddedFiles().add(fileSpecification);
```
*เหตุใดจึงต้องตั้งค่าการเข้ารหัสเป็น `None`-* วิธีนี้จะช่วยให้แน่ใจว่าทรัพยากรที่ฝังไว้จะถูกเพิ่มโดยไม่บีบอัด

### ขั้นตอนที่ 3: บันทึก PDF ของคุณ
บันทึกเอกสารที่แก้ไข:
```java
pdfDocument.save(outputDir + "outputNoCompression.pdf");
```

## คู่มือการใช้งาน
ตอนนี้เรามาดูขั้นตอนที่จำเป็นในการเพิ่มรายการเรียงลำดับ HTML ลงในเอกสาร PDF ของคุณโดยใช้ Aspose.PDF สำหรับ Java กัน
### การสร้างเอกสาร PDF ใหม่
ประการแรก ให้สร้างอินสแตนซ์ใหม่ `Document` วัตถุเป็นฐานของไฟล์ PDF ของเรา:
```java
// สร้างอินสแตนซ์ของวัตถุเอกสารเพื่อสร้างเอกสาร PDF ใหม่
doc = new Document();
```
### การเพิ่มส่วน HTML
ใช้ `HtmlFragment` คลาสสำหรับตีความและแสดงผล HTML ภายใน PDF
#### ขั้นตอนที่ 1: สร้าง HtmlFragment
สร้าง `HtmlFragment` ด้วยเนื้อหา HTML ที่คุณต้องการ:
```java
// สร้าง HtmlFragment ด้วยเนื้อหา HTML สำหรับรายการสั่งซื้อ
t = new HtmlFragment("<body style='line-height: 100px;'><ul><li>First</li><li>Second</li><li>Third</li><li>Fourth</li><li>Fifth</li></ul>Text after the list.<br/>Next line<br/>Last line</body>");
```
#### ขั้นตอนที่ 2: เพิ่มหน้าใหม่ลงในเอกสาร
เพิ่มหน้าใหม่ลงในเอกสารของคุณ:
```java
// เพิ่มหน้าใหม่ลงในเอกสาร PDF
page = doc.getPages().add();
```
#### ขั้นตอนที่ 3: แทรก HtmlFragment ลงในหน้า
เพิ่มของคุณ `HtmlFragment` ไปยังหน้าที่เพิ่งสร้างใหม่:
```java
// แทรก HtmlFragment ลงในหน้าที่เพิ่มใหม่
page.getParagraphs().add(t);
```
### การบันทึกเอกสาร PDF
บันทึกเอกสารไปยังไดเร็กทอรีที่ระบุ:
```java
// บันทึกเอกสาร PDF ที่สร้างขึ้น
doc.save(outputDir + "/AddHTMLOrderedListIntoDocuments_out.pdf");
```

## คู่มือการใช้งาน
ตอนนี้สภาพแวดล้อมของคุณได้รับการตั้งค่าเรียบร้อยแล้ว มาใช้งานคุณลักษณะในการเพิ่มตราประทับหมายเลขหน้ากัน

### การเปิดเอกสาร
ขั้นแรก โหลดเอกสาร PDF ที่คุณต้องการแก้ไข:
```csharp
using Aspose.Pdf;

// โหลดเอกสาร PDF ที่มีอยู่
document = new Document("YOUR_DOCUMENT_DIRECTORY/PageNumberStamp.pdf");
```

### การสร้างและกำหนดค่าตราประทับหมายเลขหน้า
เป้าหมายคือการสร้างแสตมป์หมายเลขหน้าที่ปรากฎบนแต่ละหน้าของ PDF ของคุณ เพื่อช่วยในการนำทาง

#### การดำเนินการแบบทีละขั้นตอน
##### สร้างตราประทับหมายเลขหน้า
```csharp
using Aspose.Pdf.Text;

// สร้างอินสแตนซ์ใหม่ของคลาส PageNumberStamp
pageNumberStamp = new PageNumberStamp();
```

##### ตั้งค่าคุณสมบัติของแสตมป์
กำหนดค่าคุณสมบัติต่างๆ เพื่อปรับแต่งแสตมป์หมายเลขหน้าของคุณ:
```csharp
// ตรวจสอบให้แน่ใจว่าแสตมป์อยู่เบื้องหน้า
pageNumberStamp.Background = false;

// จัดรูปแบบข้อความของแสตมป์
pageNumberStamp.Format = "Page # of " + document.Pages.Count;

// กำหนดระยะขอบจากด้านล่างของหน้า
pageNumberStamp.BottomMargin = 10;

// วางแสตมป์ให้อยู่กึ่งกลางแนวนอน
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;

// เริ่มนับเลขจากหน้า 1
pageNumberStamp.StartingNumber = 1;
```

##### ปรับแต่งลักษณะข้อความ
ตั้งค่าแบบอักษร ขนาด รูปแบบ และสีเพื่อให้แสตมป์ของคุณโดดเด่น:
```csharp
// ตั้งค่าแบบอักษร ขนาด และรูปแบบ
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;

// เลือกสีข้อความ
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;
```

##### เพิ่มแสตมป์ลงในหน้า
ประทับตราบนทุกหน้าในเอกสารของคุณ:
```csharp
foreach (Page page in document.Pages)
{
    // เพิ่มตราประทับที่กำหนดค่าไว้ในแต่ละหน้า
    page.AddStamp(pageNumberStamp);
}
```

### บันทึกเอกสาร
หลังจากเพิ่มแสตมป์แล้ว ให้บันทึก PDF ของคุณด้วยการเปลี่ยนแปลง:
```csharp
// บันทึกเอกสารที่อัพเดตไปยังเส้นทางที่ระบุ
document.Save("YOUR_OUTPUT_DIRECTORY/PageNumberStamp_out.pdf");
```

1. **เอกสารทางกฎหมาย**:ช่วยให้สามารถอ้างอิงและตรวจยืนยันได้อย่างง่ายดายระหว่างการตรวจสอบ
2. **หนังสือและสิ่งพิมพ์**:ช่วยอำนวยความสะดวกในการนำทาง โดยเฉพาะในเวอร์ชันพิมพ์
3. **รายงานและการนำเสนอ**:เสริมสร้างความเป็นมืออาชีพด้วยการรักษาโครงสร้าง

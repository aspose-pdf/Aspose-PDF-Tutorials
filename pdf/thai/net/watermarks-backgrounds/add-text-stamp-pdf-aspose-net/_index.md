## คู่มือการใช้งาน

### การเพิ่มแสตมป์ข้อความ

#### ภาพรวม
การเพิ่มแสตมป์ข้อความเกี่ยวข้องกับการสร้างและกำหนดค่า `Stamp` วัตถุที่สามารถวางบนหน้าเฉพาะของเอกสาร PDF คุณสมบัตินี้มีประโยชน์สำหรับการติดตราสินค้าหรือทำเครื่องหมายเอกสารว่าเป็นความลับ

#### การดำเนินการแบบทีละขั้นตอน

##### เริ่มต้นวัตถุ PdfFileStamp

ขั้นตอนแรกในกระบวนการของเราคือการเริ่มต้น `PdfFileStamp` วัตถุซึ่งทำหน้าที่เป็นอินเทอร์เฟซสำหรับการทำงานกับไฟล์ PDF

```csharp
// ขั้นตอนที่ 1: เริ่มต้นวัตถุ PdfFileStamp
PdfFileStamp fileStamp = new PdfFileStamp();
```

**คำอธิบาย**: เดอะ `PdfFileStamp` คลาสนี้มีวิธีการเพิ่มแสตมป์และบันทึกการเปลี่ยนแปลง โดยจะจัดการทรัพยากรอย่างมีประสิทธิภาพโดยการปิดการเชื่อมต่อเมื่อเสร็จสิ้น

##### เปิดเอกสาร PDF

ถัดไป ระบุเส้นทางไปยังเอกสาร PDF อินพุตของคุณโดยใช้ `BindPdf` วิธี.

```csharp
// ขั้นตอนที่ 2: เปิดเอกสาร PDF
string dataDir = @"YOUR_DOCUMENT_DIRECTORY"; // ตัวแทนสำหรับไดเรกทอรีอินพุต
fileStamp.BindPdf(dataDir + "AddTextStamp-Page.pdf");
```

**คำอธิบาย**: เดอะ `dataDir` ตัวแปรควรชี้ไปที่ไฟล์ PDF ของคุณ ตรวจสอบให้แน่ใจว่าคุณได้แทนที่ `"YOUR_DOCUMENT_DIRECTORY"` ด้วยเส้นทางที่แท้จริง

##### สร้างและกำหนดค่าแสตมป์ข้อความ

ที่นี่เราจะสร้างใหม่ `Stamp` วัตถุและกำหนดคุณสมบัติสำหรับเนื้อหาข้อความ สี สไตล์แบบอักษร และตำแหน่ง

```csharp
// ขั้นตอนที่ 3: สร้างและกำหนดค่าแสตมป์ข้อความ
Aspose.Pdf.Facades.Stamp stamp = new Aspose.Pdf.Facades.Stamp();
stamp.BindLogo(new FormattedText(
    "Hello World!", 
    Color.Blue, 
    Color.Gray, 
    FontStyle.Helvetica, 
    EncodingType.Winansi, 
    true, 14));
stamp.SetOrigin(200, 200); // ตำแหน่งของแสตมป์บนหน้ากระดาษ
stamp.Rotation = 90.0F; // มุมการหมุนเป็นองศา
stamp.IsBackground = true; // วางแสตมป์ไว้ด้านหลังเนื้อหา
```

**คำอธิบาย**: เดอะ `FormattedText` constructor กำหนดคุณสมบัติของข้อความ เช่น สีและแบบอักษร ปรับพิกัดต้นทางเพื่อวางตำแหน่งแสตมป์ให้ตรงตามที่คุณต้องการ

##### ระบุหน้าสำหรับประทับตรา

คุณสามารถจำกัดหน้าที่จะรับแสตมป์ได้โดยการตั้งค่า `Pages` ทรัพย์สินของ `Stamp` วัตถุ.

```csharp
// ขั้นตอนที่ 4: ระบุหน้าที่จะเพิ่มแสตมป์
stamp.Pages = new int[] { 1 };
```

**คำอธิบาย**ตัวอย่างนี้จะเพิ่มตราประทับเฉพาะหน้าแรกเท่านั้น แก้ไขอาร์เรย์เพื่อรวมหน้าเพิ่มเติมหรือหน้าอื่นตามต้องการ

##### เพิ่มแสตมป์และบันทึก

สุดท้ายให้ใช้ตราประทับที่กำหนดค่าไว้กับเอกสารของคุณและบันทึกไว้

```csharp
// ขั้นตอนที่ 5: เพิ่มตราประทับลงในเอกสาร PDF
fileStamp.AddStamp(stamp);

// ขั้นตอนที่ 6: บันทึกเอกสาร PDF ที่อัปเดต
string outputDir = @"YOUR_OUTPUT_DIRECTORY"; // ตัวแทนสำหรับไดเรกทอรีเอาท์พุต
fileStamp.Save(outputDir + "AddTextStamp-Page_out.pdf");

// ขั้นตอนที่ 7: ปิดวัตถุ PdfFileStamp เพื่อปล่อยทรัพยากร
fileStamp.Close();
```

**คำอธิบาย**:อย่าลืมปิดฝาทุกครั้ง `PdfFileStamp` เพื่อปลดปล่อยทรัพยากรระบบ เอกสารสุดท้ายจะถูกบันทึกไว้ในไดเร็กทอรีเอาต์พุตที่ระบุ

## คู่มือการใช้งาน

### การสร้าง PDF พร้อมวันที่หมดอายุ

#### ภาพรวม

เราจะสร้าง PDF ง่ายๆ และกำหนดวันที่หมดอายุโดยใช้ JavaScript ภายในไฟล์ PDF ซึ่งจะแจ้งเตือนผู้ใช้เมื่อเอกสารหมดอายุ

#### การดำเนินการแบบทีละขั้นตอน

**1. การตั้งค่าเอกสาร**

เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Document` คลาสซึ่งแสดงถึง PDF ของคุณ:

```csharp
// สร้างอินสแตนซ์ของวัตถุเอกสาร
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

**2. การเพิ่มเนื้อหาลงใน PDF**

เพิ่มหน้าและข้อความลงในเอกสารของคุณเพื่อวัตถุประสงค์ในการสาธิต:

```csharp
// เพิ่มหน้าเข้าไปยังคอลเลคชันไฟล์ PDF
doc.Pages.Add();

// เพิ่มข้อความส่วนย่อยลงในคอลเล็กชั่นย่อหน้าของวัตถุหน้า
doc.Pages[1].Paragraphs.Add(new TextFragment("Hello World..."));
```

**3. การนำ Expiry Logic มาใช้กับ JavaScript**

ใช้ JavaScript ใน PDF เพื่อบังคับใช้วันที่หมดอายุ:

```csharp
// สร้างวัตถุ JavaScript เพื่อกำหนดวันที่หมดอายุของ PDF
JavascriptAction javaScript = new JavascriptAction(
    "var year=2023;"  // อัพเดตถึงปีปัจจุบัน
    + "var month=10;"  // กำหนดเดือนหมดอายุที่ต้องการ
    + "today = new Date(); today = new Date(today.getFullYear(), today.getMonth());"
    + "expiry = new Date(year, month);"
    + "if (today.getTime() > expiry.getTime())"
    + "app.alert('The file is expired. You need a new one.');");

// ตั้งค่า JavaScript เป็นการดำเนินการเปิด PDF
doc.OpenAction = javaScript;
```

**4. การบันทึกเอกสาร**

สุดท้าย ให้บันทึกเอกสารของคุณด้วยตรรกะการหมดอายุที่กำหนดไว้:

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
dataDir += "SetExpiryDate_out.pdf";
// บันทึกเอกสาร PDF
doc.Save(dataDir);
Console.WriteLine(\nPDF expiry date setup successfully.\nFile saved at " + dataDir);
```

## คู่มือการใช้งาน

หัวข้อนี้จะแบ่งขั้นตอนการสร้าง PDF ที่มีข้อความและรูปภาพโดยใช้ Aspose.PDF สำหรับ .NET ออกเป็นขั้นตอนที่สามารถจัดการได้

### การสร้างเอกสารด้วยข้อความ

**ภาพรวม:**
การสร้างเอกสาร PDF ง่ายๆ โดยการเพิ่มข้อความนั้นเป็นเรื่องง่าย คุณลักษณะนี้ช่วยให้คุณสร้างเอกสารด้วยโปรแกรม ซึ่งสามารถปรับแต่งเพื่อรวมสไตล์และการจัดรูปแบบเฉพาะได้

#### การดำเนินการทีละขั้นตอน:
1. **การเริ่มต้นเอกสาร**
   เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Document` ระดับ.
   ```csharp
   Document doc = new Document();
   ```
2. **เพิ่มหน้า**
   หน้าจะถูกเพิ่มลงในเอกสารที่คุณจะวางเนื้อหาของคุณ
   ```csharp
   Page page = doc.getPages().add();
   ```
3. **สร้างและกำหนดค่าส่วนข้อความ**
   ที่นี่เราสร้าง `TextFragment` ซึ่งมีข้อความที่เราต้องการพร้อมทั้งการตั้งค่ารูปแบบเฉพาะเช่นสีและขนาด
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;

   TextFragment t1 = new TextFragment("paragraph 3 segment");
   t1.getTextState().setForegroundColor(Color.RED);
   t1.setTextIsInLineParagraph(true);
   t1.getTextState().setFontSize(12);
   ```
4. **เพิ่มข้อความลงในหน้า**
   สุดท้าย เพิ่มชิ้นส่วนข้อความลงในคอลเล็กชั่นย่อหน้าของหน้า
   ```csharp
   page.getParagraphs().add(t1);
   doc.save("YOUR_DOCUMENT_DIRECTORY/CreateTextPDF_out.pdf");
   ```

### การสร้าง FloatingBox ด้วยข้อความ

**ภาพรวม:**
กล่องลอยเป็นวิธีขั้นสูงในการจัดระเบียบเนื้อหาภายใน PDF ช่วยให้มีรูปแบบแบบไดนามิกได้

#### การดำเนินการทีละขั้นตอน:
1. **สร้างกล่องลอยน้ำ**
   กำหนดนิยาม `FloatingBox` ที่มีขนาดที่เฉพาะเจาะจง
   ```csharp
   FloatingBox textFloatingBox = new FloatingBox(117, 21);
   ```
2. **กำหนดค่าตำแหน่งและดัชนี Z**
   ปรับแต่งตำแหน่งบนหน้า
   ```csharp
   textFloatingBox.setZIndex(1);
   textFloatingBox.setLeft(-4);
   textFloatingBox.setTop(-4);
   ```
3. **เพิ่มกล่องลอยลงในเพจ**
   วางไว้ภายในโครงสร้างเอกสาร
   ```csharp
   page.getParagraphs().add(textFloatingBox);
   textFloatingBox.getParagraphs().add(t1); // เพิ่มข้อความส่วนที่สร้างไว้ก่อนหน้านี้
   doc.save("YOUR_DOCUMENT_DIRECTORY/TextFloatingBox_out.pdf");
   ```

### การเพิ่มรูปภาพลงใน FloatingBox

**ภาพรวม:**
การรวมรูปภาพไว้ในกล่องลอยช่วยให้สามารถออกแบบและจัดเค้าโครงได้อย่างยืดหยุ่น

#### การดำเนินการทีละขั้นตอน:
1. **การเริ่มต้นภาพ**
   เตรียมวัตถุรูปภาพของคุณพร้อมด้วยเส้นทางไฟล์
   ```csharp
   using Aspose.Pdf;

   Image image = new Image();
   image.setFile("YOUR_DOCUMENT_DIRECTORY/test_image.png");
   ```
2. **สร้างและกำหนดค่ากล่องลอยสำหรับภาพ**
   คล้ายกับข้อความ กำหนดกล่องลอยแต่ได้รับการปรับแต่งให้เหมาะกับรูปภาพ
   ```csharp
   FloatingBox imageFloatingBox = new FloatingBox(117, 21);
   page.getParagraphs().add(imageFloatingBox);

   imageFloatingBox.setLeft(-4);
   imageFloatingBox.setTop(-4);
   imageFloatingBox.setZIndex(2);
   ```
3. **เพิ่มรูปภาพลงในกล่องลอย**
   วางรูปภาพของคุณไว้ในกล่องลอยและบันทึกไว้
   ```csharp
   imageFloatingBox.getParagraphs().add(image);
   doc.save("YOUR_DOCUMENT_DIRECTORY/ImageInFloatingBox_out.pdf");
   ```

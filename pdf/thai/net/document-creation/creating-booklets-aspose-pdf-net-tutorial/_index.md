## คู่มือการใช้งาน

มาแยกขั้นตอนการสร้างหนังสือเล่มเล็กกัน

### การสร้างวัตถุ PdfFileEditor

ประการแรก คุณต้องสร้างอินสแตนซ์ของ `PdfFileEditor`คลาสนี้นำเสนอวิธีการแก้ไขไฟล์ PDF รวมถึงการทำหนังสือเล่มเล็ก
```csharp
using Aspose.Pdf.Facades;

// สร้างวัตถุ PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

### การตั้งค่าสตรีมไฟล์

ขั้นตอนต่อไป ตั้งค่าสตรีมไฟล์เพื่ออ่าน PDF อินพุตและเขียนสมุดเอาต์พุต ขั้นตอนนี้เกี่ยวข้องกับการสร้าง `FileStream` วัตถุสำหรับทั้งสองการดำเนินการ
```csharp
using System.IO;

// กำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุต
string dataDir = "your_data_directory_path";
string inputFile = Path.Combine(dataDir, "input.pdf");
string outputFile = Path.Combine(dataDir, "output_booklet.pdf");

// สร้างสตรีม
using FileStream inputStream = new FileStream(inputFile, FileMode.Open);
using FileStream outputStream = new FileStream(outputFile, FileMode.Create);
```

### การทำสมุดเล่มเล็ก

ใช้ประโยชน์ `PdfFileEditor.MakeBooklet` เพื่อแปลงไฟล์ PDF ของคุณเป็นรูปแบบหนังสือเล่มเล็ก วิธีนี้ต้องระบุขนาดหน้าของหนังสือเล่มเล็ก
```csharp
// ทำสมุดขนาดหน้า A5
pdfEditor.MakeBooklet(inputStream, outputStream, PageSize.A5);
```

**พารามิเตอร์ที่สำคัญ:**
- `inputStream`: สตรีมไฟล์อินพุต
- `outputStream`:สตรีมไฟล์เอาท์พุตที่จะบันทึกหนังสือเล่มเล็ก
- `PageSize.A5`:กำหนดขนาดมาตรฐานของแผ่นงานแต่ละแผ่นในเล่ม

### เคล็ดลับการแก้ไขปัญหา

หากคุณพบข้อผิดพลาด โปรดตรวจสอบ:
- ไฟล์ PDF ของคุณสามารถเข้าถึงได้และระบุไว้อย่างถูกต้องในเส้นทาง
- คุณมีสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาท์พุต

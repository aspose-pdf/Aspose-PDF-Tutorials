## คู่มือการใช้งาน
### ขั้นตอนที่ 1: การโหลดเอกสาร PDF
เริ่มต้นด้วยการโหลดเอกสาร PDF ที่ต้องการ นี่เป็นจุดเริ่มต้นสำหรับการใช้การปรับแต่งใดๆ

**โหลดเอกสารของคุณ**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Optimization;

// กำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
string dataDir = "path/to/your/documents/";

// เปิดเอกสาร PDF ที่มีอยู่
Document doc = new Document(dataDir + "AddImage.pdf");
```

### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการเพิ่มประสิทธิภาพ
ขั้นตอนต่อไปคือการตั้งค่าตัวเลือกการเพิ่มประสิทธิภาพที่ปรับแต่งให้เหมาะกับการบีบอัด FlateDecode ขั้นตอนนี้มีความสำคัญในการระบุวิธีการบีบอัดรูปภาพใน PDF

**ตั้งค่าตัวเลือกการเพิ่มประสิทธิภาพ**
```csharp
// ตัวเลือกการเพิ่มประสิทธิภาพการเริ่มต้น
var optimizationOptions = new Aspose.Pdf.Optimization.OptimizationOptions();

// กำหนดค่าการบีบอัดภาพให้ใช้การเข้ารหัส FlateDecode
optimizationOptions.ImageCompressionOptions.Encoding = ImageEncoding.Flate;
```

### ขั้นตอนที่ 3: ใช้การเพิ่มประสิทธิภาพ
เมื่อกำหนดค่าการตั้งค่าแล้ว ให้นำการเพิ่มประสิทธิภาพเหล่านี้ไปใช้กับเอกสารของคุณ

**เพิ่มประสิทธิภาพทรัพยากร**
```csharp
// ใช้ตัวเลือกการเพิ่มประสิทธิภาพให้กับเอกสาร
doc.OptimizeResources(optimizationOptions);

// บันทึกเอกสาร PDF ที่ได้รับการเพิ่มประสิทธิภาพ
doc.Save(dataDir + "FlateDecodeCompression.pdf");
```

3. **ประสิทธิภาพการจัดเก็บเอกสาร**ประหยัดพื้นที่เก็บข้อมูลด้วยการบีบอัดเอกสารจำนวนมากในสภาพแวดล้อมขององค์กร

5. **การแบ่งปันและการแจกจ่ายที่ได้รับการปรับปรุง**:การแชร์เนื้อหาภาพคุณภาพสูงในรูปแบบบีบอัดได้ง่ายขึ้น เหมาะสำหรับสื่อการตลาด

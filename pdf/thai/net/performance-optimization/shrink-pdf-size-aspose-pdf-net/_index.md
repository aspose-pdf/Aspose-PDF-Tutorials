## คู่มือการใช้งาน: การลดขนาด PDF ด้วย Aspose.PDF สำหรับ .NET

### การโหลดเอกสาร PDF
เริ่มต้นด้วยการโหลดไฟล์ PDF ที่มีอยู่ของคุณลงใน `Document` วัตถุ ตรวจสอบให้แน่ใจว่าคุณระบุเส้นทางที่ถูกต้องไปยังไฟล์ของคุณ:
```csharp
// ระบุไดเรกทอรีของเอกสารอินพุตของคุณ
string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
Document pdfDocument = new Document(dataDir + "ShrinkDocument.pdf");
```

### การเพิ่มประสิทธิภาพการใช้ทรัพยากร
คุณสมบัติหลักที่เรามุ่งเน้นคือการปรับปรุงทรัพยากร PDF ขั้นตอนนี้จะช่วยลดขนาดไฟล์โดยการจัดระเบียบและบีบอัดส่วนประกอบภายในของเอกสารของคุณ:
```csharp
// เพิ่มประสิทธิภาพทรัพยากรเพื่อลดขนาดไฟล์ PDF
pdfDocument.OptimizeResources();
```
วิธีนี้จะจัดการการเพิ่มประสิทธิภาพต่างๆ โดยอัตโนมัติ รวมถึงการบีบอัดภาพและการลบวัตถุที่ไม่ได้ใช้

### การบันทึก PDF ที่ได้รับการเพิ่มประสิทธิภาพ
หลังจากเพิ่มประสิทธิภาพแล้ว ให้บันทึกเอกสารไปยังตำแหน่งใหม่หรือเขียนทับด้วยเวอร์ชันที่เพิ่มประสิทธิภาพแล้ว:
```csharp
// ระบุไดเรกทอรีสำหรับเอกสารผลลัพธ์ของคุณ
string outputDir = "YOUR_OUTPUT_DIRECTORY/";
pdfDocument.Save(outputDir + "ShrinkDocument_out.pdf");
```

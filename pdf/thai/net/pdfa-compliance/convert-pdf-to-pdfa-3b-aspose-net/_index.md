## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแปลงไฟล์ PDF เป็นรูปแบบ PDF/A-3B

### ขั้นตอนที่ 1: เปิดเอกสาร

เริ่มต้นด้วยการโหลดเอกสาร PDF ของคุณโดยใช้ `Document` ระดับ:

```csharp
// โหลดเอกสาร PDF ที่มีอยู่
string dataDir = RunExamples.GetDataDir_AsposePdf_DocumentConversion();
Document pdfDocument = new Document(dataDir + "input.pdf");
```

### ขั้นตอนที่ 2: แปลงเป็น PDF/A-3B

ใช้ `Convert` วิธีการแปลงเอกสารของคุณ:

```csharp
// ดำเนินการแปลงเป็นรูปแบบ PDF/A-3B
pdfDocument.Convert(new MemoryStream(), PdfFormat.PDF_A_3B, ConvertErrorAction.Delete);
```
**คำอธิบาย**- 
- **เมมโมรี่สตรีม**: แสดงถึงสตรีมสำหรับการอ่านและการเขียนไบต์
- **รูปแบบไฟล์ PDF.PDF_A_3B**: ระบุระดับการปฏิบัติตามเป้าหมาย
- **แปลงErrorAction.ลบ**:จัดการข้อผิดพลาดโดยการลบองค์ประกอบที่ไม่สามารถแปลงได้

### ขั้นตอนที่ 3: บันทึกเอกสารที่แปลงแล้ว

สุดท้ายให้บันทึกเอกสารของคุณ:

```csharp
// กำหนดเส้นทางออกและบันทึกเอกสาร
dataDir = dataDir + "PDFToPDFA3b_out.pdf";
pdfDocument.Save(dataDir);
```

**เคล็ดลับการแก้ไขปัญหา**- 
- ให้แน่ใจว่าไฟล์อินพุตสามารถเข้าถึงได้
- จัดการข้อยกเว้นที่อาจเกิดขึ้นในระหว่างการแปลง

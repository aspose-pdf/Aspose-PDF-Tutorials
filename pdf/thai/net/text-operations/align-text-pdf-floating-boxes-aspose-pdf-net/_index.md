## คู่มือการใช้งาน

เราจะแบ่งกระบวนการจัดตำแหน่งข้อความภายในกล่องลอยออกเป็นขั้นตอนที่สามารถจัดการได้

### การสร้างและการจัดตำแหน่งกล่องลอย

#### ภาพรวม

กล่องลอยช่วยให้คุณควบคุมการวางข้อความได้อย่างอิสระจากการไหลของหน้า เหมาะสำหรับแถบด้านข้างหรือคำบรรยายภาพ เราจะสร้างกล่องลอย 3 กล่องที่จัดวางในตำแหน่งแนวตั้งที่แตกต่างกัน (ด้านล่าง ตรงกลาง บน) บนหน้า PDF

#### การดำเนินการแบบทีละขั้นตอน

**1. สร้างเอกสารและเพิ่มหน้า**

```csharp
// เริ่มต้นอินสแตนซ์เอกสารใหม่
doc = new Aspose.Pdf.Document();
doc.Pages.Add(); // เพิ่มหน้าใหม่ให้กับเอกสาร
```

**2. กำหนดกล่องลอยพร้อมการจัดตำแหน่งด้านล่าง**

```csharp
// สร้างกล่องลอยสำหรับการจัดตำแหน่งด้านล่าง
Aspose.Pdf.FloatingBox floatBox = new Aspose.Pdf.FloatingBox(100, 100);
floatBox.VerticalAlignment = VerticalAlignment.Bottom;
floatBox.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right;

// เพิ่มข้อความลงในกล่องลอย
doc.Pages[1].Paragraphs.Add(floatBox.Paragraphs.Add(new TextFragment("FloatingBox_bottom")));

// กำหนดขอบเขตสำหรับการมองเห็น
floatBox.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**3. กำหนดกล่องลอยที่มีการจัดตำแหน่งกึ่งกลาง**

```csharp
// สร้างกล่องลอยสำหรับการจัดตำแหน่งกึ่งกลาง
doc.Pages[1].Paragraphs.Add(floatBox = new Aspose.Pdf.FloatingBox(100, 100) {
    VerticalAlignment = VerticalAlignment.Center,
    HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right
});

floatBox.Paragraphs.Add(new TextFragment("FloatingBox_center"));
floatBox.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**4. กำหนดกล่องลอยที่มีการจัดตำแหน่งด้านบน**

```csharp
// สร้างกล่องลอยสำหรับการจัดตำแหน่งด้านบน
doc.Pages[1].Paragraphs.Add(floatBox2 = new Aspose.Pdf.FloatingBox(100, 100) {
    VerticalAlignment = VerticalAlignment.Top,
    HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right
});

floatBox2.Paragraphs.Add(new TextFragment("FloatingBox_top"));
floatBox2.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**5. บันทึกเอกสาร**

```csharp
// กำหนดไดเรกทอรีเอาท์พุตและบันทึกเอกสาร
string dataDir = RunExamples.GetDataDir_AsposePdf_Text();
doc.Save(dataDir + "FloatingBox_alignment_review_out.pdf");
```

### คำอธิบายพารามิเตอร์หลัก

- **ขนาด FloatingBox (100, 100):** กำหนดความกว้างและความสูง
- **การจัดวางแนวตั้ง:** ควบคุมการวางตำแหน่งแนวตั้งภายในหน้า
- **การจัดวางแนวนอน:** ปรับการจัดตำแหน่งแนวนอนเมื่อเทียบกับองค์ประกอบอื่น
- **ข้อมูลชายแดน:** ปรับแต่งลักษณะขอบเพื่อให้มองเห็นได้ชัดเจนยิ่งขึ้นระหว่างการพัฒนา

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเนมสเปซทั้งหมดได้รับการนำเข้าอย่างถูกต้อง
- ตรวจสอบว่าไดเรกทอรีเอาท์พุตมีอยู่หรือไม่ก่อนบันทึกเอกสาร

# เรียนรู้การเว้นระยะบรรทัดใน PDF ด้วยฟอนต์ TrueType โดยใช้ Aspose.PDF สำหรับ .NET

## การแนะนำ

กำลังดิ้นรนเพื่อรักษาระยะห่างบรรทัดที่สม่ำเสมอในเอกสาร PDF ของคุณในขณะที่ใช้แบบอักษรที่กำหนดเองหรือไม่ ด้วยพลังของ **Aspose.PDF สำหรับ .NET**คุณสามารถควบคุมและปรับแต่งลักษณะข้อความใน PDF ได้อย่างง่ายดาย ในบทช่วยสอนนี้ เราจะมาเรียนรู้วิธีจัดการระยะห่างระหว่างบรรทัดอย่างมีประสิทธิภาพโดยใช้แบบอักษร TrueType กับ Aspose.PDF สำหรับ .NET เพื่อให้แน่ใจว่าเอกสารของคุณดูสวยงามและเป็นมืออาชีพ

- การเพิ่มชิ้นส่วนข้อความลงในหน้าใหม่ภายในเอกสาร

เริ่มต้นด้วยการตรวจสอบให้แน่ใจว่าคุณมีเครื่องมือและความรู้ที่จำเป็นทั้งหมดอยู่ในมือ

## คู่มือการใช้งาน

### การกำหนดค่าระยะห่างระหว่างบรรทัดด้วยตัวเลือกการจัดรูปแบบข้อความ

ขั้นแรกมาทำความเข้าใจกันก่อนว่าจะใช้การกำหนดค่าระยะห่างระหว่างบรรทัดอย่างไร `TextFormattingOptions`คุณลักษณะนี้ช่วยให้คุณควบคุมระยะห่างแนวตั้งระหว่างบรรทัดข้อความเพื่อให้ดูสวยงามยิ่งขึ้น

#### ขั้นตอนที่ 1: กำหนดเอกสารของคุณและตั้งค่าตัวเลือกการจัดรูปแบบ

วิธีการสร้างเอกสารใหม่และตั้งค่าตัวเลือกการจัดรูปแบบด้วยระยะห่างระหว่างบรรทัดขนาดเต็มมีดังนี้
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

public class SpecifyLineSpacing
{
    public static void Run()
    {
        string dataDir = "Path_to_your_directory";
        string fontFile = dataDir + "HPSimplified.TTF";

        Document doc = new Document();
        TextFormattingOptions formattingOptions = new TextFormattingOptions
        {
            LineSpacing = TextFormattingOptions.LineSpacingMode.FullSize
        };
```

#### ขั้นตอนที่ 2: โหลดฟอนต์ TrueType

ขั้นตอนต่อไปคือโหลดแบบอักษรที่คุณกำหนดเองจากไฟล์ ซึ่งเป็นสิ่งสำคัญสำหรับการรักษารูปลักษณ์ของข้อความตามต้องการ
```csharp
if (fontFile != "")
{
    using (FileStream fontStream = File.OpenRead(fontFile))
    {
        TextFragment textFragment = new TextFragment("Hello world");
        textFragment.TextState.Font = FontRepository.OpenFont(fontStream, FontTypes.TTF);
```

#### ขั้นตอนที่ 3: วางข้อความของคุณและเพิ่มลงในเอกสาร

กำหนดตำแหน่งของข้อความในเอกสาร ขั้นตอนนี้จะช่วยให้ข้อความปรากฏตรงตำแหน่งที่คุณต้องการ
```csharp
textFragment.Position = new Position(100, 600);
        textFragment.TextState.FormattingOptions = formattingOptions;

        var page = doc.Pages.Add();
        page.Paragraphs.Add(textFragment);

        dataDir += "SpecifyLineSpacing_out.pdf";
        doc.Save(dataDir);
    }
}
```

### เคล็ดลับการแก้ไขปัญหา

- **เส้นทางไฟล์แบบอักษร:** ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์แบบอักษรถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาดในการโหลด
- **การจัดการสตรีมไฟล์:** ใช้เสมอ `using` คำชี้แจงสำหรับสตรีมไฟล์เพื่อปล่อยทรัพยากรอย่างถูกต้อง

1. **เอกสารทางกฎหมาย:** เพื่อให้มั่นใจถึงความสามารถในการอ่านและรูปลักษณ์ที่เป็นมืออาชีพ
2. **การจัดพิมพ์หนังสืออิเล็กทรอนิกส์:** ให้การจัดรูปแบบที่สอดคล้องกันในทุกอุปกรณ์

## คู่มือการใช้งาน
เมื่อทุกอย่างพร้อมแล้ว มาดูขั้นตอนการใช้งานกัน

### เริ่มต้น PdfExtractor และเชื่อมโยง PDF
#### ภาพรวม
เริ่มต้นด้วยการเริ่มทำ `PdfExtractor` และผูกเข้ากับอินสแตนซ์ PDF เป้าหมายของคุณ

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Text();

// เปิดอินพุต PDF
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "ExtractText-Page.pdf");
```
#### คำอธิบาย
- **`BindPdf(string)`**เตรียมเครื่องมือแยกไฟล์โดยการเข้าเล่มไฟล์ PDF และตั้งค่าให้พร้อมสำหรับการแยกข้อความ
- การ `dataDir` จุดตัวแปรไปยังไดเร็กทอรีของคุณที่มี PDF

### ดึงข้อความจากแต่ละหน้า
#### ภาพรวม
วนซ้ำแต่ละหน้าของ PDF และแยกเนื้อหาออกเป็นไฟล์ข้อความแยกกัน

```csharp
// ใช้เมธอด ExtractText แบบไม่มีพารามิเตอร์
currentPage = 1;
pdfExtractor.ExtractText();
while (pdfExtractor.HasNextPageText())
{
    using (MemoryStream tempMemoryStream = new MemoryStream())
    {
        pdfExtractor.GetNextPageText(tempMemoryStream);

        string text = Encoding.Unicode.GetString(tempMemoryStream.ToArray());

        File.WriteAllText(dataDir + "output" + currentPage + "_out.txt", text);
        currentPage++;
    }
}
```
#### คำอธิบาย
- **`ExtractText()`**: เริ่มกระบวนการแยกข้อมูลสำหรับทุกหน้า
- **`HasNextPageText()`**: ตรวจสอบว่ามีหน้าอื่นให้ประมวลผลอีกหรือไม่
- **`GetNextPageText(MemoryStream)`**: ดึงข้อความจากหน้าถัดไปลงใน `MemoryStream`-
- **การเข้ารหัสยูนิโค้ด**:ช่วยให้มั่นใจว่าการอ่านข้อความที่แยกออกมาถูกต้อง โดยเฉพาะอย่างยิ่งสำหรับอักขระพิเศษ

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าไฟล์ PDF ของคุณมีเลเยอร์ข้อความที่สามารถเข้าถึงได้และไม่ได้ใช้รูปภาพ
- ตรวจสอบเส้นทางไฟล์และการอนุญาตเพื่อหลีกเลี่ยงข้อผิดพลาด I/O
- ตรวจสอบการตั้งค่าการเข้ารหัสหากคุณพบปัญหาความเสียหายของอักขระ

## การประยุกต์ใช้งานจริง
ต่อไปนี้คือสถานการณ์จริงบางสถานการณ์ที่การแยกข้อความทีละหน้าสามารถมีประโยชน์อย่างยิ่ง:
1. **การเก็บเอกสารถาวร**:แปลงส่วนต่างๆ ของ PDF ขนาดใหญ่เป็นไฟล์ข้อความที่ค้นหาได้เพื่อการเก็บถาวรและดึงข้อมูลได้ง่ายขึ้น
2. **การวิเคราะห์เนื้อหา**:ดำเนินการวิเคราะห์ความรู้สึกหรือการดึงคำสำคัญบนหน้าเฉพาะในเอกสาร
3. **การโยกย้ายข้อมูล**:ถ่ายโอนข้อมูลจากระบบ PDF เดิมไปยังฐานข้อมูลที่ทันสมัย โดยรักษาข้อมูลเมตาในระดับหน้า

## การพิจารณาประสิทธิภาพ
การแยกข้อความอย่างมีประสิทธิภาพถือเป็นสิ่งสำคัญต่อประสิทธิภาพการทำงาน นี่คือเคล็ดลับบางประการ:
- ใช้สตรีมหน่วยความจำอย่างรอบคอบเพื่อจัดการการใช้ทรัพยากรอย่างมีประสิทธิภาพ
- ประมวลผล PDF ขนาดใหญ่เป็นส่วนๆ หากเป็นไปได้ เพื่อลดการใช้หน่วยความจำ
- ใช้ประโยชน์จากการประมวลผลแบบอะซิงโครนัสเมื่อเหมาะสมเพื่อปรับปรุงการตอบสนอง

**แนวทางปฏิบัติที่ดีที่สุด:**
- กำจัดทิ้ง `MemoryStream` และวัตถุใช้แล้วทิ้งอื่นๆ ทันที
- สร้างโปรไฟล์ประสิทธิภาพแอปพลิเคชันของคุณโดยใช้เครื่องมือ เช่น .NET Profiler เพื่อระบุจุดคอขวด

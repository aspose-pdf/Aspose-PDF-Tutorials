## คู่มือการใช้งาน

ในส่วนนี้ เราจะนำการนับหน้ามาใช้โดยใช้คุณลักษณะ FloatingBox

### การเพิ่ม FloatingBox สำหรับการกำหนดหมายเลขหน้า

เอ `FloatingBox` ช่วยให้คุณสามารถวางเนื้อหาในตำแหน่งเฉพาะในหน้า PDF ของคุณได้ ต่อไปนี้คือวิธีใช้เพื่อเพิ่มหมายเลขหน้า:

#### ขั้นตอนที่ 1: สร้างเอกสารและเพิ่มหน้า
ขั้นแรก ให้สร้างเอกสารใหม่และเพิ่มหน้าที่จะวางกล่องลอย

```csharp
// สร้างเอกสาร PDF ใหม่
Document document = new Document();

// เพิ่มหน้าลงในเอกสาร PDF
Page page = document.Pages.Add();
```

#### ขั้นตอนที่ 2: เริ่มต้น FloatingBox

สร้างอินสแตนซ์ของ `FloatingBox` ตามขนาดที่ต้องการและวางไว้ให้เหมาะสมบนหน้าของคุณ

```csharp
// สร้าง FloatingBox ที่มีความกว้างและความสูง
FloatingBox box1 = new FloatingBox(140, 80);

// กำหนดตำแหน่งซ้ายและบนเพื่อการวางที่แม่นยำ
box1.Left = 2;
box1.Top = 10;

// เพิ่มข้อความการนับหน้าลงในย่อหน้า FloatingBox
box1.Paragraphs.Add(new Text.TextFragment("Page: ($p/ $P )"));

// เพิ่มกล่องลอยลงในหน้าปัจจุบัน
page.Paragraphs.Add(box1);
```

**คำอธิบาย:**  
- `FloatingBox(140, 80)`: กำหนดขนาดของกล่อง
- `$p` และ `$P`:ตัวแทนสำหรับหน้าปัจจุบันและหน้าทั้งหมด

#### ขั้นตอนที่ 3: บันทึกเอกสาร

สุดท้ายให้บันทึกเอกสารของคุณในตำแหน่งที่ระบุ

```csharp
// กำหนดเส้นทางเอาท์พุต
string outputPath = "YOUR_OUTPUT_DIRECTORY/PageNumberinHeaderFooterUsingFloatingBox_out.pdf";

// บันทึกเอกสาร
document.Save(outputPath);
```

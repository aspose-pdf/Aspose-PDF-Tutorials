## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การเปิดเอกสาร PDF และค้นหากลุ่มข้อความ

### เปิดเอกสาร PDF

#### ภาพรวม
ฟีเจอร์นี้จะแสดงวิธีเปิดไฟล์ PDF ที่มีอยู่โดยใช้ Aspose.PDF ช่วยให้สามารถดำเนินการอื่นๆ เช่น การอ่านหรือการแก้ไขได้

**ขั้นตอนที่ 1:** ระบุเส้นทางไดเร็กทอรีเอกสารของคุณ
```csharp
string YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
```

**ขั้นตอนที่ 2:** สร้าง `Document` วัตถุสำหรับ PDF ที่คุณต้องการใช้งาน
```csharp
using Aspose.Pdf;

public static void OpenPdfDocument()
{
    // โหลดไฟล์ PDF ที่มีอยู่ลงในวัตถุเอกสาร
    Document pdfDocument = new Document(YOUR_DOCUMENT_DIRECTORY + "SearchTextSegmentsPage.pdf");
}
```

### ค้นหาส่วนข้อความบนหน้าเฉพาะ

#### ภาพรวม
คุณลักษณะนี้ช่วยให้คุณค้นหาข้อความที่ต้องการภายในหน้าใดหน้าหนึ่งของเอกสาร PDF ของคุณได้

**ขั้นตอนที่ 1:** เปิดเอกสาร PDF
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

public static void SearchTextOnSpecificPage()
{
    string YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
    Document pdfDocument = new Document(YOUR_DOCUMENT_DIRECTORY + "SearchTextSegmentsPage.pdf");
}
```

**ขั้นตอนที่ 2:** ใช้ `TextFragmentAbsorber` เพื่อค้นหาข้อความ
```csharp
// สร้างวัตถุ TextFragmentAbsorber เพื่อค้นหาอินสแตนซ์ทั้งหมดของวลี "ข้อความ"
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("text");

// ยอมรับตัวดูดซับในหน้า 2 ของเอกสาร
pdfDocument.Pages[2].Accept(textFragmentAbsorber);
```

**ขั้นตอนที่ 3:** ดึงข้อมูลและจัดการผลลัพธ์การค้นหา
```csharp
// รับการรวบรวมชิ้นส่วนข้อความที่แยกออกมา
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.TextFragments;

// ทำซ้ำผ่านแต่ละ TextFragment ในคอลเล็กชัน (ทางเลือก)
foreach (TextFragment fragment in textFragmentCollection)
{
    Console.WriteLine($"Found text: {fragment.Text}");
}
```

**เคล็ดลับการแก้ไขปัญหา:**
- ตรวจสอบให้แน่ใจว่าเส้นทางเอกสารของคุณถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าหมายเลขหน้าที่คุณระบุมีอยู่ใน PDF ของคุณ

## การประยุกต์ใช้งานจริง

ต่อไปนี้เป็นสถานการณ์จริงบางกรณีที่คุณลักษณะเหล่านี้อาจเป็นประโยชน์ได้:

1. **ระบบจัดการเอกสาร:** ระบบอัตโนมัติในการดึงข้อมูลเฉพาะจากเอกสารปริมาณมาก
2. **การตรวจสอบเอกสารทางกฎหมาย:** ค้นหาส่วนข้อความที่เกี่ยวข้องในสัญญาหรือเอกสารทางกฎหมายได้อย่างรวดเร็ว
3. **เครื่องมือทางการศึกษา:** สร้างแอปพลิเคชันที่ช่วยให้ผู้เรียนค้นหาคำสำคัญภายในหนังสือเรียนได้

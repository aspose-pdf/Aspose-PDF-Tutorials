# สร้างและหมุนข้อความใน PDF โดยใช้ Aspose.PDF .NET: คู่มือครอบคลุมสำหรับนักพัฒนา

## การแนะนำ

คุณกำลังมองหาวิธีสร้างเอกสาร PDF แบบไดนามิกที่มีเค้าโครงข้อความที่กำหนดเองและการหมุนโดยใช้ C# หรือไม่ ด้วยความสามารถอันทรงพลังของ Aspose.PDF สำหรับ .NET คุณสามารถเริ่มต้นเอกสาร PDF เพิ่มหน้า กำหนดค่าแอตทริบิวต์ข้อความ และแม้แต่หมุนข้อความเพื่อให้เหมาะกับความต้องการในการออกแบบของคุณได้อย่างง่ายดาย คู่มือที่ครอบคลุมนี้จะแนะนำคุณตลอดกระบวนการสร้างและจัดการข้อความใน PDF โดยใช้ Aspose.PDF เพื่อให้แน่ใจว่าคุณมีเครื่องมือทั้งหมดที่จำเป็นในการสร้างเอกสารระดับมืออาชีพด้วยโปรแกรม

**สิ่งที่คุณจะได้เรียนรู้:**
- การเริ่มต้นเอกสาร PDF ด้วย Aspose.PDF สำหรับ .NET
- การเพิ่มและกำหนดค่าส่วนข้อความใน PDF ของคุณ
- การหมุนข้อความโดยใช้ TextParagraph สำหรับเค้าโครงที่สร้างสรรค์
- การประยุกต์ใช้เทคนิคเหล่านี้ในโลกแห่งความเป็นจริง

## คู่มือการใช้งาน

ตอนนี้เรามาแบ่งการใช้งานออกเป็นคุณสมบัติหลักกัน

### เริ่มต้นและเพิ่มหน้าลงในเอกสาร PDF

**ภาพรวม**:ส่วนนี้สาธิตวิธีเริ่มต้นด้วยเอกสาร PDF ใหม่และเพิ่มหน้าเริ่มต้น

1. **สร้างเอกสารใหม่**
   เริ่มต้นด้วยการเริ่มทำ `Document` วัตถุ:
   
   ```csharp
   Document pdfDocument = new Document();
   ```

2. **เพิ่มหน้าใหม่**
   ผนวกหน้าใหม่โดยใช้ `Pages.Add()` วิธี:
   
   ```csharp
   Page pdfPage = (Page)pdfDocument.Pages.Add();
   ```

3. **บันทึกเอกสาร**
   สุดท้ายให้บันทึกเอกสารของคุณไปยังไดเร็กทอรีที่ระบุ:
   
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   pdfDocument.Save(outputDir + "/EmptyPagePdf.pdf");
   ```

### สร้างและกำหนดค่าส่วนข้อความ

**ภาพรวม**:เรียนรู้วิธีการสร้างชิ้นส่วนข้อความด้วยคุณสมบัติเฉพาะเช่นขนาดและสีของแบบอักษร

1. **เริ่มต้นส่วนข้อความ**
   เริ่มต้นด้วยการสร้าง `TextFragment` วัตถุ:
   
   ```csharp
   TextFragment textFragment = new TextFragment("Sample Text");
   ```

2. **ตั้งค่าคุณสมบัติ**
   ปรับแต่งลักษณะที่ปรากฏของข้อความของคุณ:
   - ตั้งค่าขนาดตัวอักษร:
     
     ```csharp
     textFragment.TextState.FontSize = 12;
     ```
   - เลือกแบบอักษรที่ต้องการ:
     
     ```csharp
     textFragment.TextState.Font = FontRepository.FindFont("TimesNewRoman");
     ```
   - ใช้สีพื้นหลังและสีพื้นหน้า:
     
     ```csharp
     textFragment.TextState.BackgroundColor = Aspose.Pdf.Color.LightGray;
     textFragment.TextState.ForegroundColor = Aspose.Pdf.Color.Blue;
     ```

3. **ตัวอย่างคุณสมบัติเพิ่มเติม**
   วิธีการขีดเส้นใต้ข้อความมีดังนี้:
   
   ```csharp
   TextFragment textUnderlinedFragment = new TextFragment("Underlined Text");
   textUnderlinedFragment.TextState.Underline = true;
   ```

### หมุนข้อความโดยใช้ TextParagraph และ Builder

**ภาพรวม**:ส่วนนี้จะครอบคลุมข้อความหมุนโดยใช้ `TextParagraph` ชั้นเรียนสำหรับการออกแบบหน้าเพจอย่างสร้างสรรค์

1. **เริ่มต้นเอกสารและหน้า**
   เริ่มต้นด้วยการสร้างเอกสารใหม่และเพิ่มหน้า:
   
   ```csharp
   Document pdfDocument = new Document();
   Page pdfPage = (Page)pdfDocument.Pages.Add();
   ```

2. **สร้างและกำหนดค่าย่อหน้าข้อความ**
   ใช้ `TextParagraph` สำหรับการหมุนข้อความ:
   - ตั้งค่าตำแหน่ง:
     
     ```csharp
     TextParagraph paragraph = new TextParagraph();
     paragraph.Position = new Position(200, 600);
     ```
   - หมุนย่อหน้า:
     
     ```csharp
     paragraph.Rotation = i * 90 + 45; // ตัวอย่าง: 45, 135, 225, 315 องศา
     ```

3. **เพิ่มข้อความบางส่วนลงในย่อหน้า**
   ผนวกข้อความหลายส่วน:
   
   ```csharp
   TextFragment textFragment1 = new TextFragment("Paragraph Text");
   paragraph.AppendLine(textFragment1);
   ```

4. **ใช้ TextBuilder เพื่อผนวกย่อหน้า**
   สุดท้ายใช้ `TextBuilder` เพื่อเพิ่มย่อหน้าที่คุณกำหนดค่าลงในหน้า:
   
   ```csharp
   TextBuilder textBuilder = new TextBuilder(pdfPage);
   textBuilder.AppendParagraph(paragraph);
   ```

5. **บันทึกเอกสารของคุณ**
   บันทึกด้วยการกำหนดค่าข้อความที่หมุน:
   
   ```csharp
   pdfDocument.Save(outputDir + "/RotatedTextPdf.pdf");
   ```

2. **แบบฟอร์มใบแจ้งหนี้**:นำข้อความที่หมุนมาใช้กับเค้าโครงใบแจ้งหนี้ที่มีเอกลักษณ์เฉพาะ

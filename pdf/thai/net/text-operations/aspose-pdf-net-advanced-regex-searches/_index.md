## คู่มือการใช้งาน

### ค้นหาคำที่ตรงกันเป๊ะ

**ภาพรวม:** คุณลักษณะนี้ช่วยให้คุณค้นหาคำที่ตรงกันในเอกสารได้โดยใช้นิพจน์ทั่วไป

#### ขั้นตอนการดำเนินการ:

**1. สร้าง TextFragmentAbsorber:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"\bWord\b", new TextSearchOptions(true));
```
- **คำอธิบาย:** การ `\b` หมายถึงขอบเขตของคำ โดยให้แน่ใจว่ามีการจับคู่เฉพาะ "คำ" ที่ตรงกันเท่านั้น

**2. ยอมรับตัวดูดซับ:**

```csharp
pdfDocument.Pages.Accept(textFragmentAbsorber);
TextFragmentCollection textFragments = textFragmentAbsorber.TextFragments;
```
- **วัตถุประสงค์:** วิธีนี้จะประมวลผลแต่ละหน้าของเอกสารและแยกส่วนที่ตรงกัน

### การค้นหาสตริงโดยไม่คำนึงถึงตัวพิมพ์ใหญ่-เล็ก

**ภาพรวม:** ดำเนินการค้นหาโดยไม่ต้องกังวลเกี่ยวกับความแตกต่างของตัวพิมพ์เล็ก/ใหญ่ มีประโยชน์สำหรับเนื้อหาที่ผู้ใช้สร้างขึ้นหรือการจัดรูปแบบที่ไม่สอดคล้องกัน

#### ขั้นตอนการดำเนินการ:

**1. กำหนด TextFragmentAbsorber:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("(?i)Line", new TextSearchOptions(true));
```
- **คำอธิบาย:** การ `(?i)` แฟล็กจะทำให้การค้นหาไม่คำนึงถึงตัวพิมพ์เล็ก/ใหญ่

### การแยกวิเคราะห์สตริงทั้งหมดภายในเอกสาร PDF

**ภาพรวม:** แยกเนื้อหาข้อความทั้งหมดเพื่อให้แน่ใจว่าคุณไม่พลาดข้อมูลใดๆ ในเอกสารของคุณ

#### ขั้นตอนการดำเนินการ:

**1. เริ่มต้น Absorber สำหรับทุกคำ:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"[\S]+");
```
- **คำอธิบาย:** `[\S]+` ตรงกับลำดับของอักขระที่ไม่ใช่ช่องว่างใดๆ จึงสามารถจับสตริงทั้งหมดได้อย่างมีประสิทธิภาพ

### ค้นหาการจับคู่ของสตริงการค้นหาและแยกเนื้อหาต่อไปนี้

**ภาพรวม:** มีประโยชน์สำหรับการแยกเนื้อหาที่เป็นไปตามรูปแบบหรือคำสำคัญที่เฉพาะเจาะจงในเอกสารของคุณ

#### ขั้นตอนการดำเนินการ:

**1. สร้าง Regex Absorber:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"(?i)the ((.)*)");
```
- **คำอธิบาย:** วิธีนี้จะจับ "the" ตามด้วยอักขระใดๆ ก็ตามจนถึงหนึ่งบรรทัด

### การค้นหาข้อความที่ตามการจับคู่ Regex

**ภาพรวม:** แยกเนื้อหาตามการจับคู่ regex เหมาะสำหรับการระบุและประมวลผลข้อมูลในภายหลัง

#### ขั้นตอนการดำเนินการ:

**1. นิยามของตัวดูดซับ:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"(?<=word).*");
```
- **คำอธิบาย:** การ `(?<=word)` เป็นการยืนยันการมองย้อนหลังในแง่บวก โดยครอบคลุมทุกสิ่งหลังจาก "คำ"

### การค้นหาไฮเปอร์ลิงก์/URL ในเอกสาร PDF

**ภาพรวม:** ระบุและแยก URL ที่เป็นประโยชน์สำหรับการจัดทำแคตตาล็อกหรือการตรวจสอบความถูกต้องของลิงก์ภายในเอกสาร

#### ขั้นตอนการดำเนินการ:

**1. กำหนดค่า TextFragmentAbsorber:**

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"(http|ftp|https):\/\/(?:[\w\-_]+(?:(?:\.[\w\-_]+)+))(?:[\w\-.,@?^=%&amp;:/~\\#+]*[\w\-@?^=%&amp;/~\\#+])?");
```
- **คำอธิบาย:** รูปแบบนี้ตรงกับ URL ที่เริ่มต้นด้วย http, https หรือ ftp

## การประยุกต์ใช้งานจริง

1. **การดึงข้อมูลเพื่อการวิเคราะห์:** ดึงและวิเคราะห์ข้อมูลจากรายงาน PDF โดยอัตโนมัติ
2. **การตรวจสอบเนื้อหา:** ตรวจสอบไฮเปอร์ลิงก์ในเอกสารทางเทคนิคก่อนการเผยแพร่
3. **การจัดทำดัชนีเอกสาร:** สร้างดัชนีที่ค้นหาได้ของคอลเลกชันเอกสารขนาดใหญ่โดยการแยกคำสำคัญและวลี
4. **การตรวจสอบการปฏิบัติตาม:** ตรวจสอบให้แน่ใจว่าข้อมูลที่จำเป็นทั้งหมดรวมอยู่ในเอกสารทางกฎหมายหรือทางการเงิน

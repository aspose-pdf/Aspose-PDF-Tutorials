## คําแนะนําการติดตั้ง  
เพื่อใช้ Aspose.PDF สําหรับ .NET ในโครงการของคุณ ติดตามขั้นตอนนี้  

1. โครงการผ่าน NuGet  
กรอกคําสั่งต่อไปนี้ใน NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. เพิ่ม namespace ใน C#  
รวมพื้นที่ชื่อที่ต้องการในตอนต้นของไฟล์ C # ของคุณ:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

[Aspose.PDF สําหรับเอกสาร .NET](https://docs.aspose.com/pdf/net/)

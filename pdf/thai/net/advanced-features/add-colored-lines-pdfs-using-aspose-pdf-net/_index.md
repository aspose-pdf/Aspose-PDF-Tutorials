## คู่มือการใช้งาน
ในส่วนนี้เราจะแนะนำการเพิ่มเลเยอร์เส้นสีต่างๆ ให้กับเอกสาร PDF ของคุณโดยใช้ Aspose.PDF สำหรับ .NET

### การเพิ่มเลเยอร์เส้นสีแดง
เลเยอร์เส้นสีแดงอาจมีประโยชน์อย่างยิ่งในการเน้นส่วนที่สำคัญหรือสร้างแนวทางภาพภายในเอกสารของคุณ

#### ภาพรวม
เราจะสร้างและเพิ่มบรรทัดสีแดงบนหน้าแรกของเอกสาร PDF ของเรา

#### ขั้นตอน:

**1. สร้างอินสแตนซ์เอกสาร**
```csharp
Document doc = new Document();
```
- **ทำไม**: เอ `Document` วัตถุแสดงถึงไฟล์ PDF ทั้งหมดที่คุณกำลังใช้งานอยู่

**2. เพิ่มหน้า**
```csharp
Page page = doc.Pages.Add();
```
- **ทำไม**:หน้าเป็นส่วนประกอบพื้นฐานของเอกสาร PDF ใดๆ

**3. กำหนดและกำหนดค่าเลเยอร์สีแดง**
```csharp
Layer redLayer = new Layer("oc1", "Red Line");
redLayer.Contents.Add(new SetRGBColorStroke(1, 0, 0));
redLayer.Contents.Add(new MoveTo(500, 700));
redLayer.Contents.Add(new LineTo(400, 700));
redLayer.Contents.Add(new Stroke());
```
- **ทำไม**: เดอะ `SetRGBColorStroke` กำหนดสีของเส้น `MoveTo` และ `LineTo` กำหนดจุดเริ่มต้นและจุดสิ้นสุดของเส้น

**4. เพิ่มเลเยอร์ให้กับหน้า**
```csharp
page.Layers = new List<Layer>();
page.Layers.Add(redLayer);
```
- **ทำไม**:เลเยอร์ช่วยให้คุณจัดการองค์ประกอบต่างๆ ใน PDF ของคุณได้อย่างอิสระ

**5. บันทึกเอกสาร**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "AddRedLine_out.pdf";
doc.Save(dataDir);
```
- **ทำไม**การบันทึกจะสร้างไฟล์ทางกายภาพที่สะท้อนถึงการเปลี่ยนแปลงทั้งหมดของคุณ

### การเพิ่มเลเยอร์เส้นสีเขียว
เส้นสีเขียวสามารถใช้เพื่อแยกแยะส่วนต่าง ๆ หรือเน้นเส้นทางความคืบหน้าในแผนโครงการ

#### ภาพรวม
เราจะเพิ่มเลเยอร์เส้นสีเขียวลงในเอกสาร PDF เดียวกัน เพื่อแสดงให้เห็นว่าเลเยอร์ต่างๆ สามารถอยู่ร่วมกันได้อย่างไร

#### ขั้นตอน:

**1. สร้างและเพิ่มหน้า**
ทำซ้ำขั้นตอนจากส่วนเลเยอร์สีแดงสำหรับการเริ่มต้น `Document` และเพิ่มหน้า

**2. กำหนดและกำหนดค่าเลเยอร์สีเขียว**
```csharp
Layer greenLayer = new Layer("oc2", "Green Line");
greenLayer.Contents.Add(new SetRGBColorStroke(0, 1, 0));
greenLayer.Contents.Add(new MoveTo(500, 750));
greenLayer.Contents.Add(new LineTo(400, 750));
greenLayer.Contents.Add(new Stroke());
```
- **ทำไม**: คล้ายกับเส้นสีแดง แต่มีค่าสี RGB ที่แตกต่างกัน

**3. เพิ่มเลเยอร์ให้กับหน้า**
ทำตามขั้นตอนที่คล้ายกับการเพิ่มเลเยอร์สีแดง โดยให้แน่ใจว่าแต่ละเลเยอร์ได้รับการเริ่มต้นและเพิ่มเข้าไปอย่างถูกต้อง `page-Layers`.

**4. บันทึกเอกสาร**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "AddGreenLine_out.pdf";
doc.Save(dataDir);
```

### การเพิ่มเลเยอร์เส้นสีน้ำเงิน
เส้นสีน้ำเงินสามารถใช้แสดงขอบเขตหรือเชื่อมต่อส่วนต่างๆ ของเอกสารของคุณได้

#### ภาพรวม
เราจะเพิ่มเลเยอร์เส้นสีน้ำเงินเพื่อเติมเต็มเลเยอร์สีแดงและสีเขียวที่มีอยู่

#### ขั้นตอน:

**1. สร้างและเพิ่มหน้า**
ทำซ้ำขั้นตอนจากส่วนก่อนหน้าเพื่อตั้งค่า `Document` และเพิ่มหน้า

**2. กำหนดและกำหนดค่าเลเยอร์สีน้ำเงิน**
```csharp
Layer blueLayer = new Layer("oc3", "Blue Line");
blueLayer.Contents.Add(new SetRGBColorStroke(0, 0, 1));
blueLayer.Contents.Add(new MoveTo(500, 800));
blueLayer.Contents.Add(new LineTo(400, 800));
blueLayer.Contents.Add(new Stroke());
```
- **ทำไม**:ค่า RGB กำหนดสีน้ำเงิน และ `MoveTo`-`LineTo` กำหนดเส้นทางของมัน

**3. เพิ่มเลเยอร์ให้กับหน้า**
ตรวจสอบให้แน่ใจว่าแต่ละชั้นได้รับการเพิ่มอย่างถูกต้องตามที่สาธิตไว้ก่อนหน้านี้

**4. บันทึกเอกสาร**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "AddBlueLine_out.pdf";
doc.Save(dataDir);
```

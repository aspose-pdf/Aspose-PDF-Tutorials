## คู่มือการใช้งาน (H2)
เราจะแบ่งกระบวนการออกเป็นขั้นตอนที่สามารถจัดการได้

### การสร้างเอกสาร PDF (H3)
**ภาพรวม:** เริ่มต้นด้วยการสร้างเอกสารใหม่และเพิ่มหน้าลงไป
```csharp
Document pdf = new Aspose.Pdf.Document();
Page sec1 = pdf.Pages.Add(); // เพิ่มหน้าใหม่
```
- `Aspose.Pdf.Document`:แสดงถึง PDF ทั้งหมดของคุณ
- `Pages.Add()`: เพิ่มหน้าใหม่ที่คุณสามารถวางเนื้อหาได้

### การเพิ่มส่วนข้อความ (H3)
**ภาพรวม:** แทรกชิ้นส่วนข้อความด้วยตัวเลือกการจัดรูปแบบเช่นสีและขนาดแบบอักษร
```csharp
TextFragment t1 = new Aspose.Pdf.Text.TextFragment("paragraph 3 segment");
sec1.Paragraphs.Add(t1);

t1.TextState.ForegroundColor = Color.Red; // ตั้งค่าสีข้อความเป็นสีแดง
t1.TextState.FontSize = 12;                // ตั้งค่าขนาดตัวอักษรเป็น 12
```
- `TextFragment`: หมายถึงกลุ่มข้อความ
- `TextState.ForegroundColor`: กำหนดสีข้อความ
- `TextState.FontSize`: ควบคุมขนาดตัวอักษร

### การแทรกรูปภาพ (H3)
**ภาพรวม:** เพิ่มรูปภาพลงในเอกสาร PDF ของคุณ เพื่อให้เนื้อหามีความสมบูรณ์ยิ่งขึ้น
```csharp
Image image1 = new Aspose.Pdf.Image();
image1.File = "path\to\your\test_image.png"; // ระบุเส้นทางของภาพ
sec1.Paragraphs.Add(image1);
```
- `Aspose.Pdf.Image`: แสดงถึงรูปภาพในเอกสารของคุณ
- `image1.File`: กำหนดเส้นทางไฟล์สำหรับรูปภาพ

### การเพิ่มกล่องลอย (H3)
**ภาพรวม:** ใช้กล่องลอยเพื่อจัดระเบียบและจัดตำแหน่งเนื้อหาอย่างมีประสิทธิภาพภายในหน้า
```csharp
FloatingBox box1 = new Aspose.Pdf.FloatingBox(117, 21);
box1.Left = -4; // การวางตำแหน่งจากขอบด้านซ้าย
box1.Top = -4;  // การวางตำแหน่งจากขอบด้านบน
sec1.Paragraphs.Add(box1);

box1.Paragraphs.Add(image1); // เพิ่มรูปภาพลงในกล่องลอย
```
- `FloatingBox`:ภาชนะสำหรับจัดระเบียบเนื้อหา
- คุณสมบัติตำแหน่ง (`Left`- `Top`): ตั้งค่าตำแหน่งของกล่องบนหน้า

### การบันทึกเอกสาร PDF (H3)
**ภาพรวม:** สุดท้ายให้บันทึกเอกสารของคุณลงในไฟล์
```csharp
pdf.Save("path\to\your\CreateMultiLayerPdf_out.pdf");
```
- `Document.Save()`: เขียนผลลัพธ์สุดท้ายลงในดิสก์

## คู่มือการใช้งาน
ปฏิบัติตามขั้นตอนเหล่านี้เพื่อเพิ่มข้อความที่แรเงาแบบไล่ระดับ:

### การเพิ่มข้อความด้วยสีไล่เฉด
#### ภาพรวม
ปรับแต่งข้อความใน PDF ของคุณโดยใช้สีไล่เฉดเพื่อให้ดูสวยงามยิ่งขึ้น ใช้ Aspose.PDF `GradientAxialShading` เพื่อการเปลี่ยนสีที่เรียบเนียน

#### การดำเนินการแบบทีละขั้นตอน
**1. โหลดเอกสาร PDF**
โหลดไฟล์ PDF ที่มีอยู่ลงใน `Document` วัตถุ.
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_Text();
using (Document pdfDocument = new Document(dataDir + "text_sample4.pdf"))
{
    // โค้ดสำหรับจัดการเอกสารจะอยู่ที่นี่
}
```
**2. ค้นหาข้อความเฉพาะ**
ใช้ `TextFragmentAbsorber` เพื่อค้นหาชิ้นส่วนข้อความที่คุณต้องการแก้ไข
```csharp
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Lorem ipsum");
pdfDocument.Pages.Accept(absorber);
TextFragment textFragment = absorber.TextFragments[1];
```
**3. ใช้การแรเงาแบบไล่ระดับ**
ตั้งค่า `ForegroundColor` โดยใช้รูปแบบช่องสี
```csharp
textFragment.TextState.ForegroundColor = new Aspose.Pdf.Color()
{
    PatternColorSpace = new Aspose.Pdf.Drawing.GradientAxialShading(Color.Red, Color.Blue)
};
```
**4. เพิ่มขีดเส้นใต้ (ไม่บังคับ)**
ปรับปรุงลักษณะข้อความโดยการขีดเส้นใต้
```csharp
textFragment.TextState.Underline = true;
```
**5. บันทึกเอกสาร**
บันทึกการเปลี่ยนแปลงของคุณลงในไฟล์ PDF ใหม่
```csharp
pdfDocument.Save(dataDir + "text_out.pdf");
```

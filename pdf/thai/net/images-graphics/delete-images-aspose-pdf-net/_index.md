## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการลบรูปภาพเฉพาะหรือทั้งหมดจากเอกสาร PDF

### การลบรูปภาพเฉพาะ

หากต้องการลบภาพออกจาก PDF ของคุณ:

#### ขั้นตอนที่ 1: โหลดเอกสาร PDF

สร้างอินสแตนซ์ของ `Document` ชั้นเรียนและโหลดไฟล์ PDF ของคุณ ระบุไฟล์ PDF ที่คุณกำลังทำงานด้วย

```csharp
// ExStart: โหลดเอกสาร PDF
using System.IO;
using Aspose.Pdf;

string dataDir = "your_directory_path/";
Document pdfDocument = new Document(dataDir + "DeleteImages.pdf");
```

#### ขั้นตอนที่ 2: เข้าถึงและลบรูปภาพ

เข้าถึงหน้าที่มีภาพเป้าหมายของคุณ ใช้ `Delete` วิธีการที่จะลบมันออกไป

```csharp
// ExStart:ลบภาพเฉพาะ
pdfDocument.Pages[1].Resources.Images.Delete(1);
dataDir = dataDir + "DeleteImages_out.pdf";
```

ในตัวอย่างนี้ เราจะลบรูปภาพแรกจากหน้าแรก ปรับพารามิเตอร์เพื่อกำหนดเป้าหมายรูปภาพหรือหน้าอื่นตามต้องการ

#### ขั้นตอนที่ 3: บันทึกเอกสารที่อัปเดต

หลังจากทำการเปลี่ยนแปลงแล้วให้บันทึกเอกสารโดยใช้ `Save` วิธี.

```csharp
// ExStart:บันทึกอัปเดต PDF
pdfDocument.Save(dataDir);
```

### การลบรูปภาพทั้งหมดจากหน้า

หากต้องการลบรูปภาพทั้งหมดจากหน้าที่ระบุ:

```csharp
// วนซ้ำรูปภาพแต่ละภาพในทรัพยากรของหน้าและลบออก
foreach (var img in pdfDocument.Pages[1].Resources.Images)
{
    pdfDocument.Pages[1].Resources.Images.Delete(img.Index);
}
```

## คู่มือการใช้งาน

### ภาพรวม

ในส่วนนี้ เราจะอธิบายการแยกภาพจากลายเซ็นดิจิทัลภายในเอกสาร PDF ซึ่งมีประโยชน์อย่างยิ่งเมื่อคุณต้องตรวจสอบหรือจัดเก็บรายละเอียดลายเซ็นแยกกัน

#### โหลดเอกสาร PDF

ขั้นแรก โหลดไฟล์ PDF ของคุณโดยใช้ Aspose.PDF `Document` ระดับ:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFilePath = Path.Combine(dataDir, "DigitallySign.pdf");

// สร้างอินสแตนซ์เอกสารใหม่
Document doc = new Document(inputFilePath);
```

#### ดึงภาพจากลายเซ็น

วิธีการแยกภาพที่ฝังอยู่ในลายเซ็น PDF มีดังนี้

```csharp
using (PdfFileSignature signature = new PdfFileSignature(doc))
{
    // ตรวจสอบว่าเอกสารมีลายเซ็นดิจิทัลหรือไม่
    if (signature.ContainsSignature())
    {
        foreach (string sigName in signature.GetSignNames())
        {
            string outputFilePath = Path.Combine(dataDir, "ExtractImages_out.jpg");

            using (Stream imageStream = signature.ExtractImage(sigName))
            {
                if (imageStream != null)
                {
                    // แปลงสตรีมเป็นวัตถุรูปภาพ
                    using (System.Drawing.Image image = Bitmap.FromStream(imageStream))
                    {
                        // บันทึกภาพที่แยกออกมาเป็นไฟล์ JPEG
                        image.Save(outputFilePath, System.Drawing.Imaging.ImageFormat.Jpeg);
                    }
                }
            }
        }
    }
}
```

**คำอธิบาย-**
- **`PdfFileSignature`:** คลาสนี้ช่วยอำนวยความสะดวกในการใช้งานลายเซ็น PDF
- **`ContainsSignature()`-** ตรวจสอบว่ามีลายเซ็นดิจิทัลอยู่ในเอกสารหรือไม่
- **`ExtractImage(sigName)`-** ดึงข้อมูลภาพจากลายเซ็นที่ระบุ

### เคล็ดลับการแก้ไขปัญหา

## คู่มือการใช้งาน
ในหัวข้อนี้ เราจะแนะนำคุณผ่านแต่ละขั้นตอนของการเรนเดอร์ LaTeX ใน PDF อย่างชัดเจน

### การสร้างเอกสารและหน้าใหม่ (H2)
#### ภาพรวม:
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Document` คลาสและเพิ่มหน้าใหม่เข้าไป

**ขั้นตอนที่ 1: เริ่มต้นเอกสาร**
```csharp
var doc = new Document();
```
*คำอธิบาย:* บรรทัดนี้จะสร้างเอกสาร PDF ใหม่ จากนั้นคุณจะเพิ่มเนื้อหา รวมถึงสคริปต์ LaTeX ลงในเอกสารนี้

#### ขั้นตอนที่ 2: เพิ่มหน้า
```csharp
var page = doc.Pages.Add();
```
*วัตถุประสงค์:* เพิ่มหน้าใหม่ให้กับเอกสารซึ่งคุณสามารถเรนเดอร์สคริปต์ LaTeX ได้ แต่ละหน้าทำหน้าที่เป็นตัวบรรจุเนื้อหาในไฟล์ PDF

### การเพิ่มสคริปต์ LaTeX (H3)
ตอนนี้เรามาเพิ่มสคริปต์ LaTeX ของเราโดยใช้ Aspose.PDF กัน `TeXFragment`-

#### ขั้นตอนที่ 1: กำหนดสคริปต์ LaTeX
```csharp
string latexScript = """
    \usepackage{amsmath,amsthm}
    \begin{document}
    \begin{proof} The proof is as follows:
    \begin{align}
    (x+y)^3&=(x+y)(x+y)^2
    (x+y)(x^2+2xy+y^2)\\
    &=x^3+3x^2y+3xy^2+x^3.\qedhere
    \end{align}
    \end{proof}
    \end{document}"
```
*คำอธิบาย:* สคริปต์นี้ประกอบด้วยโค้ด LaTeX สำหรับการเรนเดอร์พิสูจน์ทางคณิตศาสตร์ รวมถึงแพ็คเกจและการจัดตำแหน่งเพื่อการจัดรูปแบบที่ถูกต้อง

#### ขั้นตอนที่ 2: สร้าง TeXFragment
```csharp
var latex = new TeXFragment(latexScript);
```
*วัตถุประสงค์:* แปลงสตริง LaTeX ของคุณเป็น `TeXFragment`ซึ่ง Aspose.PDF สามารถตีความเพื่อแสดงผลภายในเอกสาร PDF ได้

#### ขั้นตอนที่ 3: เพิ่มส่วนลงในหน้า
```csharp
page.Paragraphs.Add(latex);
```
*คำอธิบาย:* เพิ่ม `TeXFragment` ไปสู่คอลเลกชันย่อหน้าของหน้า โดยฝังสคริปต์ LaTeX ของคุณลงในเนื้อหา PDF

### การบันทึกเอกสาร (H2)
สุดท้ายให้บันทึกเอกสารของคุณไปยังตำแหน่งที่ระบุ:

#### ขั้นตอนที่ 1: กำหนดเส้นทางการบันทึก
```csharp
var dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
*วัตถุประสงค์:* กำหนดไดเรกทอรีที่จะบันทึกไฟล์ PDF เอาต์พุต ตรวจสอบว่ามีเส้นทางนี้อยู่หรือแอปพลิเคชันของคุณสามารถเขียนได้

#### ขั้นตอนที่ 2: บันทึกเอกสาร
```csharp

```
*คำอธิบาย:* คำสั่งนี้จะเขียนเอกสารที่เรนเดอร์แล้วลงในดิสก์ โดยสร้างไฟล์ PDF ที่มีเนื้อหา LaTeX ในตำแหน่งที่ระบุ

## คู่มือการใช้งาน (H2)

### ตั้งค่ารูปภาพเป็นพื้นหลัง (H2)

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการตั้งค่าพื้นหลังรูปภาพใน PDF ของคุณ

#### ขั้นตอนที่ 1: สร้างเอกสารใหม่

เริ่มต้นด้วยการสร้าง `Document` วัตถุ:

```csharp
// การเริ่มต้นเอกสาร
document doc = new Document();
```

สไนปเป็ตนี้จะเริ่มต้นอินสแตนซ์ใหม่ของ `Document` คลาสที่แสดงไฟล์ PDF ของคุณ

#### ขั้นตอนที่ 2: เพิ่มหน้าลงในเอกสาร

เพิ่มหน้าที่คุณจะตั้งค่ารูปภาพพื้นหลัง:

```csharp
Page page = doc.Pages.Add();
```

ที่นี่เราเพิ่มหน้าว่างโดยใช้ `Pages.Add()` วิธี.

#### ขั้นตอนที่ 3: สร้างและกำหนดค่า BackgroundArtifact

การ `BackgroundArtifact` คลาสนี้ใช้สำหรับตั้งค่าพื้นหลังใน Aspose.PDF วิธีกำหนดค่ามีดังนี้:

```csharp
BackgroundArtifact background = new BackgroundArtifact();
background.BackgroundImage = File.OpenRead(@"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg");
page.Artifacts.Add(background);
```

- **ทำไม** การ `BackgroundArtifact` คลาสนี้ได้รับการออกแบบมาเพื่อจัดการพื้นหลังในหน้า PDF การเพิ่มอาร์ทิแฟกต์นี้จะช่วยให้มั่นใจว่ารูปภาพจะปรากฏเป็นพื้นหลัง
- แทนที่ `"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg"` ตามเส้นทางภาพพื้นหลังที่คุณต้องการ

#### ขั้นตอนที่ 4: บันทึกเอกสารของคุณ

สุดท้ายให้บันทึกเอกสารของคุณ:

```csharp
string dataDir = @"YOUR_OUTPUT_DIRECTORY\ImageAsBackground_out.pdf";
doc.Save(dataDir);
```

รหัสนี้จะบันทึกไฟล์ PDF ของคุณด้วยไดเรกทอรีเอาต์พุตและชื่อไฟล์ที่ระบุ แทนที่ `"YOUR_OUTPUT_DIRECTORY"` ตามตำแหน่งที่คุณต้องการบันทึก

- **ปัญหาด้านประสิทธิภาพ**:ปรับขนาดภาพขนาดใหญ่ก่อนตั้งเป็นพื้นหลังเพื่อปรับปรุงประสิทธิภาพ

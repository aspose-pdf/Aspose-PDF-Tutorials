# วิธีการเพิ่มคำอธิบายบรรทัดที่วัดได้ลงใน PDF ด้วย Aspose.PDF .NET

## การแนะนำ

คุณเคยจำเป็นต้องใส่คำอธิบายประกอบในเอกสาร PDF ด้วยการวัดที่แม่นยำหรือไม่ ไม่ว่าจะเป็นภาพวาดทางเทคนิค แผนผังสถาปัตยกรรม หรือเอกสารใดๆ ที่ความแม่นยำเป็นสิ่งสำคัญ การเพิ่มคำอธิบายประกอบเส้นที่วัดได้จะช่วยเพิ่มความชัดเจนและการสื่อสารได้อย่างมาก บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ไลบรารี Aspose.PDF .NET ที่มีประสิทธิภาพเพื่อเพิ่มคำอธิบายประกอบเส้นที่มีความสามารถในการวัดลงในไฟล์ PDF ของคุณได้อย่างง่ายดาย

**สิ่งที่คุณจะได้เรียนรู้:**

- วิธีตั้งค่าสภาพแวดล้อมของคุณสำหรับการทำงานกับ Aspose.PDF .NET
- ขั้นตอนการสร้างคำอธิบายบรรทัดพร้อมการวัดในเอกสาร PDF
- การกำหนดค่าการตั้งค่าต่างๆ เช่น เส้นผู้นำ หน่วยการวัด และการแสดงคำบรรยาย

มาเจาะลึกข้อกำหนดเบื้องต้นที่จำเป็นก่อนที่เราจะเริ่มใช้งานฟีเจอร์นี้กัน

## คู่มือการใช้งาน

ในส่วนนี้ เราจะอธิบายกระบวนการการเพิ่มคำอธิบายแบบวัดเส้นลงในเอกสาร PDF โดยใช้ Aspose.PDF .NET

### การเพิ่มคำอธิบายบรรทัดด้วยการวัด

#### ภาพรวม

การเพิ่มคำอธิบายบรรทัดช่วยให้คุณสามารถทำเครื่องหมายเฉพาะส่วนต่างๆ ใน PDF และวัดระยะทางได้ คุณลักษณะนี้มีประโยชน์อย่างยิ่งสำหรับเอกสารทางเทคนิคที่ความแม่นยำมีความสำคัญ

#### ขั้นตอนการดำเนินการ

1. **โหลดเอกสาร**

   เริ่มต้นด้วยการโหลดเอกสาร PDF ที่มีอยู่ที่คุณต้องการเพิ่มคำอธิบายประกอบ

   ```csharp
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY/input.pdf";
   Document doc = new Document(dataDir);
   ```

2. **กำหนดพื้นที่คำอธิบาย**

   ระบุพื้นที่สี่เหลี่ยมบนหน้าของคุณที่คำอธิบายประกอบจะปรากฏขึ้น

   ```csharp
   Rectangle rect = new Rectangle(260, 630, 451, 662); // กำหนดพิกัดสำหรับคำอธิบายประกอบ
   ```

3. **สร้างคำอธิบายบรรทัด**

   สร้าง `LineAnnotation` วัตถุที่มีจุดเริ่มต้นและจุดสิ้นสุดภายในพื้นที่สี่เหลี่ยมที่กำหนด

   ```csharp
   LineAnnotation line = new LineAnnotation(doc.Pages[1], rect, new Point(266, 657), new Point(446, 656));
   ```

4. **กำหนดค่าลักษณะที่ปรากฏ**

   ตั้งค่าสี เส้นนำ และสไตล์สำหรับคำอธิบายประกอบ

   ```csharp
   line.Color = Color.Red; // กำหนดสีของคำอธิบายเป็นสีแดง
   line.LeaderLine = -15; // เส้นผู้นำเคลื่อนออกจากจุดเริ่มต้น
   line.LeaderLineExtension = 5; // ความยาวของสายต่อขยายของสายนำ
   line.StartingStyle = LineEnding.OpenArrow;
   line.EndingStyle = LineEnding.OpenArrow;
   ```

5. **ตั้งค่ารายละเอียดการวัด**

   ใช้ `Measure` วัตถุที่จะกำหนดรายละเอียดการวัด

   ```csharp
   line.Measure = new Measure(line);
   line.Measure.DistanceFormat.Add(new Measure.NumberFormat(line.Measure));
   line.Measure.DistanceFormat[1].UnitLabel = "mm"; // ตั้งค่าหน่วยสำหรับการวัด
   ```

6. **เพิ่มคำบรรยายและบันทึก**

   เพิ่มคำอธิบายเพื่อแสดงการวัดและบันทึกเอกสารของคุณ

   ```csharp
   line.Contents = "155 mm";
   line.ShowCaption = true;
   line.CaptionPosition = CaptionPosition.Top;

   doc.Pages[1].Annotations.Add(line); // เพิ่มคำอธิบายลงในหน้าที่ 1

   string outputDir = @"YOUR_OUTPUT_DIRECTORY/UseMeasureWithLineAnnotation_out.pdf";
   doc.Save(outputDir);
   ```

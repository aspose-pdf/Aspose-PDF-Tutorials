## คู่มือการใช้งาน

### สร้างคำอธิบายลิงก์เอกสาร

คุณลักษณะนี้จะแสดงวิธีการเพิ่มคำอธิบายประกอบลิงก์ลงใน PDF ที่มีอยู่

#### ขั้นตอนที่ 1: โหลดเอกสาร PDF

ขั้นแรก โหลดเอกสาร PDF ของคุณโดยใช้ `Document` ระดับ:

```csharp
using Aspose.Pdf;

// กำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุต
string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\CreateDocumentLink.pdf";
string outputFilePath = @"YOUR_OUTPUT_DIRECTORY\CreateDocumentLink_out.pdf";

// โหลดเอกสาร PDF ที่มีอยู่
Document document = new Document(inputFilePath);
```

#### ขั้นตอนที่ 2: เข้าถึงหน้าที่ต้องการ

ขั้นตอนต่อไป ให้เข้าถึงหน้าเฉพาะที่คุณต้องการเพิ่มคำอธิบาย:

```csharp
// เข้าถึงหน้าแรกของเอกสาร
Page page = document.Pages[1];
```

#### ขั้นตอนที่ 3: สร้างและกำหนดค่าคำอธิบายลิงก์

สร้าง `LinkAnnotation` วัตถุ โดยระบุตำแหน่งและขนาดบนหน้า วิธีกำหนดค่ามีดังนี้

```csharp
using Aspose.Pdf.Annotations;

// กำหนดรูปสี่เหลี่ยมผืนผ้าสำหรับพื้นที่ลิงก์
Rectangle rect = new Rectangle(100, 100, 300, 300);

// สร้างคำอธิบายลิงก์
LinkAnnotation link = new LinkAnnotation(page, rect);

// ตั้งค่าสีของลิงค์เป็นสีเขียว
link.Color = Color.FromRgb(System.Drawing.Color.Green);

// กำหนดการดำเนินการระยะไกลสำหรับลิงก์ (เช่น นำทางไปยัง PDF อื่น)
link.Action = new GoToRemoteAction(@"YOUR_DOCUMENT_DIRECTORY\RemoveOpenAction.pdf", 1);
```

#### ขั้นตอนที่ 4: เพิ่มคำอธิบายและบันทึก

เพิ่มคำอธิบายประกอบที่คุณกำหนดค่าลงในคอลเล็กชันคำอธิบายประกอบของหน้า จากนั้นบันทึกเอกสาร:

```csharp
// เพิ่มคำอธิบายลิงก์ลงในหน้า
page.Annotations.Add(link);

// บันทึก PDF ที่ได้รับการอัพเดต
document.Save(outputFilePath);
```

### กำหนดค่าคุณสมบัติคำอธิบายลิงก์

ส่วนนี้สาธิตวิธีการตั้งค่าคุณสมบัติ เช่น สีและการกระทำสำหรับ `LinkAnnotation`-

#### ขั้นตอนที่ 1: กำหนดหน้าและคำอธิบายประกอบ

สมมติว่าคุณมี `Page` วัตถุ:

```csharp
Page page = new Page(); // แทนที่ด้วยอินสแตนซ์หน้าจริง
LinkAnnotation link = new LinkAnnotation(page, new Rectangle(100, 100, 300, 300));
```

#### ขั้นตอนที่ 2: ตั้งค่าคุณสมบัติ

กำหนดค่าคุณสมบัติของคำอธิบายประกอบ เช่น สีและการดำเนินการ:

```csharp
// ตั้งค่าสีคำอธิบายเป็นสีเขียว
link.Color = Color.FromRgb(System.Drawing.Color.Green);

// กำหนดการดำเนินการ (เช่น นำทางไปยังหน้าเฉพาะในเอกสารอื่น)
link.Action = new GoToRemoteAction(@"YOUR_DOCUMENT_DIRECTORY\TargetDocument.pdf", 2);
```

#### ขั้นตอนที่ 3: เพิ่มคำอธิบายประกอบ

เพิ่มคำอธิบายลงในหน้าที่คุณต้องการ:

```csharp
// สมมติว่า 'หน้า' คือชุดของหน้า
page.Annotations.Add(link);
```

## การพิจารณาประสิทธิภาพ

การเพิ่มประสิทธิภาพเป็นสิ่งสำคัญเมื่อทำงานกับไฟล์ PDF ขนาดใหญ่:

- **การจัดการหน่วยความจำ**: ใช้ `using` คำชี้แจงเพื่อให้แน่ใจว่ามีการกำจัดทรัพยากรอย่างเหมาะสม
- **คำอธิบายประกอบที่มีประสิทธิภาพ**จำกัดจำนวนคำอธิบายประกอบหากต้องจัดการกับข้อจำกัดด้านหน่วยความจำ
- **การประมวลผลแบบแบตช์**:เมื่อใช้คำอธิบายประกอบกับเอกสารหลายฉบับ ควรพิจารณาการประมวลผลแบบชุดเพื่อจัดการการใช้ทรัพยากรอย่างมีประสิทธิภาพ

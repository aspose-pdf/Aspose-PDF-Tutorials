## คู่มือการใช้งาน
### การตรวจสอบลายเซ็นดิจิทัล
การตรวจสอบลายเซ็นดิจิทัลช่วยยืนยันว่าเอกสารไม่มีการเปลี่ยนแปลงตั้งแต่ลงนาม ช่วยให้มั่นใจถึงความถูกต้องและสมบูรณ์

#### ขั้นตอนที่ 1: ตั้งค่าสภาพแวดล้อมของคุณ
ให้แน่ใจว่าสภาพแวดล้อมของคุณพร้อมแล้วด้วยการติดตั้งไลบรารีที่จำเป็นทั้งหมด:
```csharp
using Aspose.Pdf.Facades;
```

#### ขั้นตอนที่ 2: สร้างวิธีการตรวจสอบลายเซ็น
ใช้ชิ้นส่วนโค้ดนี้เพื่อสร้างวิธีการที่ตรวจสอบลายเซ็นดิจิทัลใน PDF:
```csharp
class VerifyValidSignature
{
    public static void Run()
    {
        try
        {
            // ตั้งค่าเส้นทางไดเรกทอรีสำหรับเอกสารของคุณ
            string dataDir = "path_to_your_documents_directory";
            
            // สร้างอินสแตนซ์ของ PdfFileSignature
            PdfFileSignature pdfSign = new PdfFileSignature();
            
            // ผูกเอกสาร PDF
            pdfSign.BindPdf(dataDir + "DigitallySign.pdf");
            
            // ตรวจสอบลายเซ็น
            if (pdfSign.VerifySignature("Signature1"))
            {
                Console.WriteLine("Signature Verified");
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine(ex.Message);
        }
    }       
}
```

#### คำอธิบายพารามิเตอร์โค้ด
- `dataDir`:ไดเร็กทอรีที่คุณจัดเก็บไฟล์ PDF ของคุณ
- `PdfFileSignature pdfSign = new PdfFileSignature()`: เริ่มต้นอินสแตนซ์ใหม่สำหรับการจัดการลายเซ็น PDF
- `pdfSign.BindPdf(dataDir + "DigitallySign.pdf")`:ผูกไฟล์ PDF ที่จะประมวลผล
- `pdfSign.VerifySignature("Signature1")`: ตรวจสอบว่าลายเซ็นที่ระบุถูกต้องหรือไม่

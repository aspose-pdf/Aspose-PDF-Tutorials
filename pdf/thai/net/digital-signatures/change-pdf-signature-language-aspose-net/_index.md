# วิธีการเปลี่ยนภาษาลายเซ็น PDF ด้วย Aspose.PDF สำหรับ .NET

## การแนะนำ

คุณกำลังมองหาวิธีปรับแต่งภาษาของลายเซ็นดิจิทัลในเอกสาร PDF ของคุณโดยใช้ .NET หรือไม่ ไม่ว่าคุณจะเตรียมสัญญา ข้อตกลง หรือเอกสารอื่นใดที่ต้องมีลายเซ็น การแปลข้อความเป็นภาษาต่างๆ ถือเป็นสิ่งสำคัญ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการเปลี่ยนการตั้งค่าภาษาของลายเซ็นดิจิทัลใน PDF ด้วย Aspose.PDF สำหรับ .NET เพื่อให้แน่ใจว่าเอกสารของคุณเป็นไปตามมาตรฐานสากลและรองรับผู้ชมทั่วโลก

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีตั้งค่า Aspose.PDF สำหรับ .NET
- การเปลี่ยนภาษาข้อความลายเซ็นโดยใช้ Aspose.PDF `SignatureCustomAppearance` ระดับ.
- การกำหนดค่าพารามิเตอร์ลายเซ็น เช่น รูปแบบวันที่ ตำแหน่ง เหตุผล ฯลฯ
- บันทึกเอกสาร PDF ที่ลงนามแล้วพร้อมการตั้งค่าภาษาที่กำหนดเอง

ขณะที่เราเจาะลึกคู่มือนี้ ให้แน่ใจว่าคุณพร้อมแล้วโดยปฏิบัติตามข้อกำหนดเบื้องต้นที่ระบุไว้ด้านล่าง เพื่อให้เริ่มต้นได้อย่างราบรื่น

## คู่มือการใช้งาน

มาแบ่งขั้นตอนการเปลี่ยนแปลงการตั้งค่าภาษาสำหรับลายเซ็นดิจิทัลออกเป็นขั้นตอนที่สามารถจัดการได้

### การตั้งค่าลักษณะลายเซ็น

#### ภาพรวม
เราจะกำหนดค่าลักษณะที่ลายเซ็นจะปรากฏบน PDF ของคุณ โดยเน้นที่การปรับแต่งองค์ประกอบข้อความ เช่น วันที่ เหตุผล และตำแหน่งให้เหมาะกับภาษาต่างๆ

**โหลดเอกสารของคุณ**
ขั้นแรก ให้สร้างอินสแตนซ์ของ `PdfFileSignature` และผูกเข้ากับอินพุต PDF ของคุณ:

```csharp
using (Aspose.Pdf.Facades.PdfFileSignature pdfSign = new Aspose.Pdf.Facades.PdfFileSignature())
{
    pdfSign.BindPdf("input.pdf");
}
```

**กำหนดรูปสี่เหลี่ยมลายเซ็น**
กำหนดพื้นที่บนหน้าที่ลายเซ็นจะปรากฏ:

```csharp
System.Drawing.Rectangle rect = new System.Drawing.Rectangle(310, 45, 200, 50);
```

### การกำหนดค่ารายละเอียดลายเซ็น

#### ภาพรวม
ตั้งค่าพารามิเตอร์ต่างๆ เช่น เหตุผลและตำแหน่งสำหรับลายเซ็นดิจิทัลของคุณ ปรับแต่งพารามิเตอร์เหล่านี้ให้สะท้อนถึงภาษาใดก็ได้

**สร้างวัตถุลายเซ็น PKCS#7**
สร้างตัวอย่าง `PKCS7` วัตถุพร้อมรายละเอียดใบรับรองที่จำเป็น:

```csharp
PKCS7 pkcs = new PKCS7("certificate.pfx", "12345");
pkcs.Reason = "Pruebas Firma";
pkcs.ContactInfo = "Contacto Pruebas";
pkcs.Location = "Población (Provincia)";
pkcs.Date = DateTime.Now;
```

**ปรับแต่งลักษณะลายเซ็น**
ใช้ประโยชน์ `SignatureCustomAppearance` เพื่อปรับแต่งคุณสมบัติข้อความและการตั้งค่าภาษา:

```csharp
SignatureCustomAppearance signatureCustomAppearance = new SignatureCustomAppearance();
signatureCustomAppearance.DateSignedAtLabel = "Fecha";
signatureCustomAppearance.DigitalSignedLabel = "Digitalmente firmado por";
signatureCustomAppearance.ReasonLabel = "Razón";
signatureCustomAppearance.LocationLabel = "Localización";
signatureCustomAppearance.FontFamilyName = "Arial";
signatureCustomAppearance.FontSize = 10d;
signatureCustomAppearance.Culture = CultureInfo.InvariantCulture;
signatureCustomAppearance.DateTimeFormat = "yyyy.MM.dd HH:mm:ss";

pkcs.CustomAppearance = signatureCustomAppearance;
```

### การลงนามใน PDF
**สมัครลายเซ็น**
ใช้ `Sign` วิธีการใช้ลายเซ็นที่คุณกำหนดค่าไว้:

```csharp
pdfSign.Sign(1, true, rect, pkcs);
```

### การบันทึกไฟล์เอาท์พุต
**บันทึกเอกสารที่ลงนาม**
สุดท้าย ให้บันทึก PDF ที่คุณลงนามโดยใช้การตั้งค่าภาษาใหม่:

```csharp
pdfSign.Save("output.pdf");
```

## การประยุกต์ใช้งานจริง
ต่อไปนี้เป็นสถานการณ์จริงบางสถานการณ์ที่สามารถใช้คุณลักษณะนี้ได้:
1. **สัญญาธุรกิจระหว่างประเทศ**:การแปลข้อความลายเซ็นให้กับพันธมิตรในประเทศต่างๆ
2. **เอกสารทางกฎหมาย**:การรับประกันการปฏิบัติตามข้อกำหนดทางกฎหมายของภูมิภาค โดยการแสดงลายเซ็นเป็นภาษาของท้องถิ่น
3. **สื่อการเรียนรู้**:การมอบใบรับรองหรือเอกสารให้กับนักศึกษาต่างชาติในภาษาแม่ของพวกเขา
4. **แบบฟอร์มราชการ**:การปรับแต่งแบบฟอร์มที่ต้องมีลายเซ็นอย่างเป็นทางการ เช่น ใบอนุญาตและการลงทะเบียน
5. **บริษัทข้ามชาติ**:ปรับปรุงเวิร์กโฟลว์เอกสารสำหรับพนักงานในแต่ละภูมิภาค

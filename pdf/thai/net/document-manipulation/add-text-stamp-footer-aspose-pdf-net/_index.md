## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่า Aspose.PDF เรียบร้อยแล้ว มาลองดูวิธีการเพิ่มตราประทับข้อความเป็นส่วนท้ายกัน

### การเพิ่มข้อความประทับตราท้ายกระดาษ

#### ภาพรวม

ฟีเจอร์นี้ช่วยให้คุณวางข้อความที่กำหนดเองที่ด้านล่างของแต่ละหน้าในเอกสาร PDF โดยอัตโนมัติ เหมาะอย่างยิ่งสำหรับการเพิ่มประกาศลิขสิทธิ์หรือองค์ประกอบการสร้างแบรนด์ให้สอดคล้องกันในไฟล์ของคุณ

#### การดำเนินการแบบทีละขั้นตอน

1. **เปิดเอกสาร PDF ของคุณ**
   
   โหลดไฟล์ PDF จากไดเร็กทอรีที่คุณระบุ
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY/TextinFooter.pdf";
   Document pdfDocument = new Document(dataDir);
   ```

2. **สร้างวัตถุ TextStamp**
   
   เริ่มต้นการใช้งาน `TextStamp` พร้อมข้อความท้ายกระดาษที่คุณต้องการ
   
   ```csharp
   TextStamp textStamp = new TextStamp("Footer Text");
   ```

3. **กำหนดค่าคุณสมบัติแสตมป์**
   
   ตั้งค่าระยะขอบและการจัดตำแหน่งเพื่อวางเป็นส่วนท้ายของแต่ละหน้า
   
   ```csharp
   textStamp.BottomMargin = 10;
   textStamp.HorizontalAlignment = HorizontalAlignment.Center;
   textStamp.VerticalAlignment = VerticalAlignment.Bottom;
   ```

4. **เพิ่มตราประทับข้อความให้กับทุกหน้า**
   
   ดำเนินการซ้ำผ่านหน้าเอกสารและประทับตรา
   
   ```csharp
   foreach (Page page in pdfDocument.Pages)
   {
       page.AddStamp(textStamp);
   }
   ```

5. **บันทึกเอกสารที่อัปเดต**
   
   ระบุเส้นทางเอาต์พุตและบันทึก PDF ที่แก้ไขแล้ว
   
   ```csharp
   string outputPath = "YOUR_OUTPUT_DIRECTORY/TextinFooter_out.pdf";
   pdfDocument.Save(outputPath);
   ```

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ได้รับการระบุอย่างถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
- ตรวจสอบคุณสมบัติการจัดตำแหน่งหากส่วนท้ายปรากฏว่าวางผิดที่

## การประยุกต์ใช้งานจริง

การเพิ่มส่วนท้ายกระดาษสามารถทำได้หลากหลายวัตถุประสงค์:

1. **การสร้างแบรนด์**:แสดงโลโก้หรือชื่อบริษัทของคุณอย่างสม่ำเสมอในเอกสารทั้งหมด
2. **ประกาศทางกฎหมาย**: รวมข้อมูลลิขสิทธิ์หรือข้อความปฏิเสธความรับผิดชอบโดยอัตโนมัติ
3. **การติดตามเอกสาร**:เพิ่มหมายเลขหน้าหรือรายละเอียดเวอร์ชันเป็นเชิงอรรถเพื่อให้ง่ายต่อการอ้างอิง

การรวมฟีเจอร์นี้เข้ากับระบบอื่นๆ จะช่วยปรับปรุงเวิร์กโฟลว์การทำงานอัตโนมัติของเอกสาร โดยเฉพาะในภาคกฎหมายและการเงิน

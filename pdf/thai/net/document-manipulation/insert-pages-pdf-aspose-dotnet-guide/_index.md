## คู่มือการใช้งาน

### ภาพรวมของการแทรกหน้า
การแทรกหน้าระหว่างส่วนต่างๆ ใน PDF จะช่วยปรับปรุงการจัดระเบียบเอกสารหรือเพิ่มเนื้อหาที่จำเป็นได้โดยไม่ต้องแก้ไขด้วยตนเอง หัวข้อนี้จะแนะนำคุณตลอดการใช้งาน `PdfFileEditor` เพื่อให้บรรลุภารกิจนี้ได้อย่างมีประสิทธิภาพ

#### การดำเนินการแบบทีละขั้นตอน
**1. สร้างอินสแตนซ์ PdfFileEditor**
เริ่มต้นด้วยการเริ่มต้น `PdfFileEditor` วัตถุซึ่งให้วิธีการสำหรับการจัดการไฟล์ PDF:

```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**2. เตรียมสตรีมอินพุตและเอาต์พุต**
ตั้งค่าสตรีมเพื่ออ่านไฟล์ PDF ต้นฉบับและเขียนเอาต์พุต:

```csharp
FileStream inputStream = new FileStream("source.pdf", FileMode.Open);
FileStream portStream = new FileStream("pagesToInsert.pdf", FileMode.Open);
FileStream outputStream = new FileStream("output.pdf", FileMode.Create);
```

**3. แทรกหน้าโดยใช้สตรีม**
ตอนนี้แทรกหน้าจาก PDF หนึ่งไปยังอีก PDF หนึ่งในตำแหน่งที่ระบุ:

```csharp
// แทรกหน้าที่ 1 ถึง 4 ของสตรีมที่สองหลังจากหน้าที่ 1 ของสตรีมแรก
pdfEditor.Insert(inputStream, 1, portStream, 1, 4, outputStream);
```

**คำอธิบาย:**
- `inputStream`:แหล่งที่มา PDF ส่วนที่แทรกหน้า
- `1`:หมายเลขหน้าในไฟล์ PDF ต้นฉบับที่เนื้อหาใหม่เริ่มต้น
- `portStream`:สตรีมพร้อมหน้าเพิ่มเติมให้แทรก
- `1, 4`: ช่วงหน้าตั้งแต่ `portStream` ที่จะรวมอยู่ด้วย

#### เคล็ดลับการแก้ไขปัญหา
- **ข้อผิดพลาดไม่พบไฟล์**: ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้องและสามารถเข้าถึงได้
- **ปัญหาด้านความจำ**:ใช้สตรีมอย่างรอบคอบเพื่อหลีกเลี่ยงการใช้หน่วยความจำมากเกินไป

## คู่มือการใช้งาน

### คุณสมบัติ: หน้า PDF แบบ N-Up

คุณสมบัตินี้จะรวมหน้าต่างๆ จาก PDF ต่างๆ เข้าเป็นหน้าจำนวนน้อยลงโดยใช้รูปแบบ N-Up ซึ่งเหมาะสำหรับการย่อเอกสารโดยไม่สูญเสียข้อมูล

#### ขั้นตอนที่ 1: สร้างวัตถุ PdfFileEditor
```csharp
using Aspose.Pdf.Facades;

// เริ่มต้นวัตถุ PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

#### ขั้นตอนที่ 2: กำหนดเส้นทางไฟล์ต้นทางและปลายทางเอาท์พุต
ระบุเส้นทางสำหรับไฟล์ PDF อินพุตและไฟล์เอาท์พุต แทนที่ตัวแทนด้วยไดเร็กทอรีจริง
```csharp
string inputFile1 = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
string inputFile2 = "YOUR_DOCUMENT_DIRECTORY/input2.pdf";
string outputFile = "YOUR_OUTPUT_DIRECTORY/MakeNUpUsingPaths_out.pdf";
```

#### ขั้นตอนที่ 3: สร้าง N-Up
ใช้ `MakeNUp` วิธีการรวมเอกสารให้เป็นไฟล์เอาต์พุตเดียว
```csharp
// รวม 'input.pdf' และ 'input2.pdf' โดยใช้การกำหนดค่า N-Up
pdfEditor.MakeNUp(inputFile1, inputFile2, outputFile);
```

**คำอธิบายพารามิเตอร์:**
- `inputFile1`- `inputFile2`:เส้นทางไปยังไฟล์ PDF ที่คุณต้องการรวมเข้าด้วยกัน
- `outputFile`:เส้นทางปลายทางสำหรับ PDF รวม

### คุณสมบัติ: การตั้งค่าไดเรกทอรีสำหรับตัวอย่าง Aspose.PDF

คู่มือการตั้งค่าเส้นทางไดเรกทอรี และการจัดเก็บไฟล์อินพุตและเอาต์พุตอย่างเป็นระเบียบ

#### กำหนดไดเรกทอรีตัวแทน
```csharp
using System;

string dataDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

Console.WriteLine("Using document directory: " + dataDirectory);
Console.WriteLine("Output will be saved to: " + outputDirectory);
```

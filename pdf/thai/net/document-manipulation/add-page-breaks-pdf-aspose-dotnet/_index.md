# วิธีการเพิ่มตัวแบ่งหน้าในเอกสาร PDF โดยใช้ Aspose.PDF สำหรับ .NET

**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน

### คุณสมบัติ 1: เพิ่มตัวแบ่งหน้าในเอกสาร PDF

**ภาพรวม:** คุณลักษณะนี้จะแสดงวิธีการเพิ่มตัวแบ่งหน้าในตำแหน่งที่ระบุภายในเอกสาร PDF โดยใช้เส้นทางไฟล์

**ขั้นตอน:**

#### ขั้นตอนที่ 1: กำหนดเส้นทาง
เริ่มต้นด้วยการกำหนดไดเร็กทอรีสำหรับไฟล์ PDF อินพุตและเอาท์พุตของคุณ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### ขั้นตอนที่ 2: โหลดเอกสารต้นฉบับ
โหลดเอกสาร PDF ต้นฉบับของคุณลงใน `Aspose.Pdf.Document` วัตถุ:
```csharp
Document doc = new Document(dataDir + "/input.pdf");
```

#### ขั้นตอนที่ 3: สร้างเอกสารปลายทาง
สร้างเอกสาร PDF ปลายทางว่างเปล่าเพื่อจัดเก็บผลลัพธ์การแบ่งหน้า:
```csharp
Document dest = new Document();
```

#### ขั้นตอนที่ 4: เริ่มต้นใช้งาน PdfFileEditor
ตั้งค่า `PdfFileEditor` วัตถุที่จะใช้ในการแก้ไขไฟล์ PDF:
```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
```

#### ขั้นตอนที่ 5: เพิ่มตัวแบ่งหน้า
เพิ่มตัวแบ่งหน้าที่ตำแหน่ง 450 บนหน้าแรกของเอกสารของคุณและเพิ่มลงในเอกสารปลายทาง:
```csharp
fileEditor.AddPageBreak(doc, dest, new PdfFileEditor.PageBreak[] {
    new PdfFileEditor.PageBreak(1, 450)
});
```

#### ขั้นตอนที่ 6: บันทึก PDF ที่ได้
สุดท้าย ให้บันทึก PDF ที่ได้พร้อมเพิ่มตัวแบ่งหน้าด้วย:
```csharp
dest.Save(outputDir + "/PageBreak_out.pdf");
```

### คุณสมบัติ 2: เพิ่มตัวแบ่งหน้าและบันทึกลงในเส้นทางปลายทาง

**ภาพรวม:** คุณลักษณะนี้จะเพิ่มตัวแบ่งหน้าลงใน PDF โดยตรงตามเส้นทางที่ระบุ

#### ขั้นตอนที่ 1: กำหนดเส้นทาง
ระบุไดเรกทอรีเอกสารของคุณ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

#### ขั้นตอนที่ 2: เริ่มต้นใช้งาน PdfFileEditor
สร้างอินสแตนซ์ของ `PdfFileEditor` ในการจัดการไฟล์ PDF:
```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
```

#### ขั้นตอนที่ 3: เพิ่มตัวแบ่งหน้าและบันทึกโดยตรง
เพิ่มตัวแบ่งหน้าโดยตรงที่ตำแหน่ง 450 บนหน้าแรกของเอกสารของคุณ และบันทึกลงในเส้นทางเอาต์พุต:
```csharp
fileEditor.AddPageBreak(dataDir + "/input.pdf", dataDir + "/PageBreakWithDestPath_out.pdf", new PdfFileEditor.PageBreak[] {
    new PdfFileEditor.PageBreak(1, 450)
});
```

### คุณสมบัติที่ 3: เพิ่มการแบ่งหน้าโดยใช้สตรีม

**ภาพรวม:** คุณลักษณะนี้สาธิตการเพิ่มตัวแบ่งหน้าโดยใช้สตรีมอินพุตและเอาต์พุต

#### ขั้นตอนที่ 1: เปิดสตรีมอินพุต
เปิดสตรีมเพื่ออ่านเอกสาร PDF ต้นฉบับ:
```csharp
using (Stream src = new FileStream(dataDir + "/input.pdf", FileMode.Open, FileAccess.Read))
```

#### ขั้นตอนที่ 2: เปิดสตรีมเอาท์พุต
สร้างสตรีมเอาท์พุตเพื่อเขียนการเปลี่ยนแปลง:
```csharp
using (Stream dest = new FileStream(dataDir + "/PageBreakWithStream_out.pdf", FileMode.Create, FileAccess.Write))
```

#### ขั้นตอนที่ 3: เริ่มต้นใช้งาน PdfFileEditor
ตั้งค่า `PdfFileEditor` วัตถุสำหรับการแก้ไข:
```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
```

#### ขั้นตอนที่ 4: เพิ่มการแบ่งหน้าโดยใช้สตรีม
เพิ่มตัวแบ่งหน้าที่ตำแหน่ง 450 บนหน้าหนึ่ง โดยใช้สตรีมในการจัดการทั้งอินพุตและเอาต์พุต:
```csharp
fileEditor.AddPageBreak(src, dest, new PdfFileEditor.PageBreak[] {
    new PdfFileEditor.PageBreak(1, 450)
});
```

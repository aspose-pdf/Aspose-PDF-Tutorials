# เพิ่มรูปภาพและหมายเลขหน้าลงใน PDF โดยใช้ Aspose.PDF สำหรับ .NET: คู่มือฉบับสมบูรณ์

ในโลกดิจิทัลทุกวันนี้ การสร้างเอกสาร PDF ที่ดูเป็นมืออาชีพถือเป็นสิ่งสำคัญ ไม่ว่าคุณจะกำลังร่างรายงาน จดหมายข่าว หรือเอกสารทางธุรกิจใดๆ การเพิ่มรายละเอียดส่วนตัว เช่น รูปภาพและหมายเลขหน้า จะช่วยปรับปรุงการนำเสนอเอกสารของคุณได้อย่างมาก คู่มือนี้จะแนะนำคุณตลอดกระบวนการผสานรวมคุณลักษณะเหล่านี้เข้ากับ PDF ของคุณอย่างราบรื่นโดยใช้ Aspose.PDF สำหรับ .NET

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีการเพิ่มรูปภาพลงในส่วนหัว PDF

- การตั้งค่าและติดตั้ง Aspose.PDF สำหรับ .NET
- การประยุกต์ใช้งานจริงของฟีเจอร์เหล่านี้

มาดูว่าคุณสามารถแปลงเอกสาร PDF ของคุณได้อย่างง่ายดายอย่างไร

## คู่มือการใช้งาน

### การเพิ่มรูปภาพลงในส่วนหัว PDF

#### ภาพรวม
การเพิ่มรูปภาพลงในส่วนหัว PDF จะทำให้เอกสารดูน่าสนใจและเป็นมืออาชีพมากขึ้น ในส่วนนี้จะแนะนำวิธีการดำเนินการดังกล่าวด้วย Aspose.PDF สำหรับ .NET

**ขั้นตอนที่ 1: เริ่มต้นวัตถุเอกสาร**
สร้างใหม่ `Document` วัตถุซึ่งแสดงถึงไฟล์ PDF ทั้งหมด:
```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

**ขั้นตอนที่ 2: สร้างหน้าและส่วนหัว**
เพิ่มหน้าลงในเอกสารของคุณและสร้างส่วนหัวสำหรับหน้านั้น:
```csharp
// เพิ่มหน้า
Aspose.Pdf.Page page = doc.Pages.Add();

// เริ่มต้นส่วนหัว
Aspose.Pdf.HeaderFooter header = new Aspose.Pdf.HeaderFooter();
page.Header = header;
```

**ขั้นตอนที่ 3: แทรกภาพลงในส่วนหัว**
สร้างวัตถุรูปภาพ ตั้งค่าเส้นทางไฟล์ และเพิ่มลงในส่วนหัว:
```csharp
Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
image1.File = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
header.Paragraphs.Add(image1);
```

**ขั้นตอนที่ 4: บันทึกเอกสาร**
สุดท้าย ให้บันทึกเอกสารของคุณโดยรวมภาพส่วนหัวไว้ด้วย:
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "HeaderWithImage_out.pdf");
doc.Save(outputFilePath);
```

### การเพิ่มหมายเลขหน้าลงในส่วนท้ายของ PDF

#### ภาพรวม
การใส่หมายเลขหน้าไว้ในส่วนท้ายของ PDF ถือเป็นสิ่งสำคัญสำหรับเอกสารที่มีหลายหน้า คู่มือนี้จะอธิบายวิธีดำเนินการนี้ด้วย Aspose.PDF สำหรับ .NET

**ขั้นตอนที่ 1: เริ่มต้นเอกสารและเพิ่มหน้า**
เริ่มต้นด้วยการสร้างใหม่ `Document` วัตถุ:
```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
// เพิ่มหน้า
Aspose.Pdf.Page page = doc.Pages.Add();
```

**ขั้นตอนที่ 2: สร้างส่วนท้าย**
สร้างส่วนท้ายของเอกสารของคุณ:
```csharp
Aspose.Pdf.HeaderFooter footer = new Aspose.Pdf.HeaderFooter();
page.Footer = footer;
```

**ขั้นตอนที่ 3: เพิ่มหมายเลขหน้าลงในส่วนท้าย**
แทรกส่วนข้อความที่แสดงหมายเลขหน้าแบบไดนามิก:
```csharp
Aspose.Pdf.Text.TextFragment txt = new Aspose.Pdf.Text.TextFragment("Page: ($p of $P)");
footer.Paragraphs.Add(txt);
```

**ขั้นตอนที่ 4: บันทึกเอกสารด้วยส่วนท้าย**
บันทึกเอกสารของคุณเพื่อรวมส่วนท้ายพร้อมหมายเลขหน้า:
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "FooterWithPageNumber_out.pdf");
doc.Save(outputFilePath);
```

## คู่มือการใช้งาน

ในส่วนนี้เราจะครอบคลุมการโหลดและการวนซ้ำผ่านหน้า PDF รวมถึงการปรับขนาดหน้า

### การโหลดและการวนซ้ำผ่านหน้า PDF
คุณสมบัตินี้ช่วยให้คุณโหลดเอกสาร PDF และวนซ้ำผ่านหน้าต่างๆ เพื่อเข้าถึงหรือแก้ไขได้

#### ขั้นตอนที่ 1: โหลดเอกสาร
```csharp
using System.IO;
using Aspose.Pdf;

// โหลดไฟล์ PDF ของคุณ
document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

#### ขั้นตอนที่ 2: ทำซ้ำในแต่ละหน้า
คุณสามารถทำซ้ำในแต่ละหน้าได้โดยใช้ `foreach` ลูป:
```csharp
foreach (Page page in doc.Pages)
{
    // เข้าถึงสี่เหลี่ยมผืนผ้า MediaBox ของหน้าปัจจุบัน
    Rectangle r = page.MediaBox;
}
```
การ `MediaBox` คุณสมบัตินี้จะให้ขนาดและตำแหน่งแก่คุณ ซึ่งเป็นสิ่งสำคัญสำหรับการปรับทิศทาง

### การปรับขนาดหน้า
การเปลี่ยนทิศทางเกี่ยวข้องกับการแก้ไขความกว้างและความสูงของหน้า ดังต่อไปนี้:

#### ขั้นตอนที่ 1: คำนวณมิติใหม่
หากต้องการสลับหน้าจากแนวตั้งเป็นแนวนอนหรือในทางกลับกัน ให้คำนวณขนาดใหม่ดังนี้:
```csharp
foreach (Page page in doc.Pages)
{
    Rectangle r = page.MediaBox;

    // สลับความสูงและความกว้างเพื่อเปลี่ยนทิศทาง
    double newWidth = r.Height;
    double newHeight = r.Width;

    // ใช้มิติใหม่กับ MediaBox
    page.MediaBox = new Rectangle(0, 0, newWidth, newHeight);
}
```
**คำอธิบาย:**
- `r.Height` กลายเป็นความกว้างใหม่ และ `r.Width` กลายเป็นความสูงใหม่สำหรับการวางแนวนอน

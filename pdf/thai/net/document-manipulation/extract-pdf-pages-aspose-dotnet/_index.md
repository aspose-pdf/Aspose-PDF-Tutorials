## คู่มือการใช้งาน

ให้เราแบ่งการดำเนินการออกเป็นขั้นตอนที่สามารถจัดการได้

### ขั้นตอนที่ 1: สร้างอินสแตนซ์ของคลาส PdfFileEditor

การ `PdfFileEditor` คลาสนี้มีความสำคัญต่อการแก้ไขไฟล์ PDF โดยจะมีวิธีการต่างๆ มากมาย รวมถึงการแยกหน้าด้วย

```csharp
// สร้างอินสแตนซ์ของวัตถุ PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

วัตถุนี้จะใช้เพื่อดำเนินการสกัด

### ขั้นตอนที่ 2: แยกหน้าจากไฟล์ PDF

ใช้ `Extract` วิธีการของ `PdfFileEditor` คลาสเพื่อระบุว่าจะแยกหน้าใด

```csharp
// กำหนดเส้นทางต้นทางและปลายทางออก
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFile = dataDir + "/MultiplePages.pdf";
string outputFile = dataDir + "/ExtractPagesBetweenNumbers_out.pdf";

// แยกหน้าตั้งแต่ 1 ถึง 3
pdfEditor.Extract(inputFile, 1, 3, outputFile);
```

**คำอธิบายพารามิเตอร์:**
- `inputFile`: เส้นทางของแหล่งที่มา PDF.
- `1`: หมายเลขหน้าเริ่มต้นสำหรับการแยก
- `3`เลขหน้าสุดท้ายสำหรับการแยก
- `outputFile`:เส้นทางปลายทางสำหรับหน้าที่แยกออกมา

### เคล็ดลับการแก้ไขปัญหา

- **ปัญหาทั่วไป:** "FileNotFoundException" – ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกต้อง
- **ข้อกังวลเกี่ยวกับการใช้งานหน่วยความจำ:** สำหรับไฟล์ PDF ขนาดใหญ่ ให้ตรวจสอบการใช้หน่วยความจำและพิจารณาการประมวลผลเป็นกลุ่มหากจำเป็น

## คู่มือการใช้งาน
### คุณสมบัติ: การจัดการสตรีมไฟล์
#### ภาพรวม
การจัดการสตรีมไฟล์ถือเป็นพื้นฐานเมื่อทำงานกับไฟล์ PDF ใน .NET ในส่วนนี้ เราจะเปิดและจัดการสตรีมไฟล์ PDF โดยใช้ `FileStream`-

**ขั้นตอนที่ 1: เปิดไฟล์ PDF อินพุต**
```csharp
using System.IO;

// เปิดสตรีมไฟล์ PDF ที่มีอยู่เพื่ออ่าน
FileStream inputStream = new FileStream("YOUR_DOCUMENT_DIRECTORY/input.pdf", FileMode.Open);
// เปิดสตรีมไฟล์ PDF ที่มีอยู่อื่นเพื่ออ่าน
FileStream portStream = new FileStream("YOUR_DOCUMENT_DIRECTORY/input2.pdf", FileMode.Open);
```
ที่นี่, `FileMode.Open` ช่วยให้เราสามารถเข้าถึงไฟล์ในโหมดอ่านอย่างเดียวได้

**ขั้นตอนที่ 2: สร้างสตรีมไฟล์ PDF เอาท์พุต**
```csharp
// สร้างสตรีมไฟล์ PDF ใหม่สำหรับการเขียนเอาต์พุต
FileStream outputStream = new FileStream("YOUR_OUTPUT_DIRECTORY/AppendFilesUsingStreams_out.pdf", FileMode.Create);
```
การ `FileMode.Create` รับประกันว่าหากไฟล์เอาต์พุตมีอยู่แล้ว จะถูกเขียนทับ

### คุณสมบัติ: การแก้ไขไฟล์ PDF โดยใช้ Aspose.Pdf.Facades
#### ภาพรวม
ตอนนี้เรามาผนวกไฟล์ PDF สองไฟล์เข้าเป็นไฟล์เดียวโดยใช้ Aspose.PDF ฟีเจอร์นี้แสดงให้เห็นถึงวิธีการใช้ไลบรารีเพื่อผสานเอกสารอย่างมีประสิทธิภาพ

**ขั้นตอนที่ 1: เริ่มต้นใช้งาน PdfFileEditor**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Facades;

// สร้างอินสแตนซ์ของ PdfFileEditor ที่ให้วิธีการแก้ไขเอกสาร PDF
PdfFileEditor pdfEditor = new PdfFileEditor();
```
`PdfFileEditor` เป็นคลาสเอนกประสงค์ที่ช่วยให้สามารถดำเนินการต่างๆ กับไฟล์ PDF ได้ รวมถึงการผสานไฟล์ด้วย

**ขั้นตอนที่ 2: ผนวกไฟล์ PDF**
```csharp
// ผนวกเนื้อหาของ portStream (เริ่มจากหน้าที่ 1) ลงใน inputStream
// และบันทึกผลลัพธ์ลงใน outputStream
pdfEditor.Append(inputStream, portStream, 1, 1, outputStream);
```
ที่นี่เราระบุว่าเอกสารทั้งหมดจาก `portStream` ควรจะผนวกเข้าไปด้วย `inputStream`เริ่มตั้งแต่หน้า 1

## คู่มือการใช้งาน

ทำตามขั้นตอนเหล่านี้เพื่อปิดการใช้งานการบีบอัดไฟล์ภายในไฟล์แนบ PDF

### ภาพรวม

การปิดใช้งานการบีบอัดไฟล์จะช่วยรักษาคุณภาพเดิมของไฟล์ที่ฝังไว้ ซึ่งเป็นสิ่งสำคัญสำหรับข้อมูลที่ละเอียดอ่อนหรือภาพที่มีความเที่ยงตรงสูง คุณสามารถทำได้ดังนี้:

#### ขั้นตอนที่ 1: ตั้งค่าสภาพแวดล้อมโครงการของคุณ

สร้างแอปพลิเคชันคอนโซล C# ใหม่และเพิ่มโค้ดต่อไปนี้ลงในวิธีการหลักของคุณ:

```csharp
using System.IO;
using Aspose.Pdf;

namespace AsposePdfExamples
{
    public class DisableFilesCompressionExample
    {
        public static void Run()
        {
            string dataDir = "/path/to/your/documents/directory/";
```

#### ขั้นตอนที่ 2: โหลดเอกสาร PDF

โหลดเอกสาร PDF ที่มีอยู่ของคุณ:
```csharp
Document pdfDocument = new Document(dataDir + "GetAlltheAttachments.pdf");
```
นี่จะโหลด PDF เข้าไปในหน่วยความจำเพื่อการจัดการ

#### ขั้นตอนที่ 3: สร้างข้อกำหนดไฟล์สำหรับสิ่งที่แนบมา

สร้าง `FileSpecification` วัตถุที่จะแสดงไฟล์ที่คุณต้องการแนบ:
```csharp
FileSpecification fileSpecification = new FileSpecification("test_out.txt", "Sample text file");
fileSpecification.Encoding = FileEncoding.None; // ปิดการใช้งานการบีบอัดโดยตั้งค่าการเข้ารหัสเป็นไม่มี
```

#### ขั้นตอนที่ 4: เพิ่มสิ่งที่แนบมา

เพิ่มของคุณ `FileSpecification` คัดค้านการรวบรวมไฟล์ที่ฝังไว้ในเอกสาร:
```csharp
pdfDocument.EmbeddedFiles.Add(fileSpecification);
```
การเชื่อมโยงไฟล์นี้เป็นไฟล์แนบภายใน PDF ของคุณ

#### ขั้นตอนที่ 5: บันทึกเอกสาร

บันทึกเอกสารที่แก้ไขแล้วพร้อมแนบไฟล์โดยไม่บีบอัด:
```csharp
string outputFilePath = dataDir + "DisableFilesCompression_out.pdf";
pdfDocument.Save(outputFilePath);

Console.WriteLine("\nFile compression disabled successfully.\nFile saved at " + outputFilePath);
```

### ตัวเลือกการกำหนดค่าคีย์

- **ข้อมูลจำเพาะไฟล์.การเข้ารหัส**: การตั้งค่านี้เป็น `FileEncoding.None` ป้องกันไม่ให้ไฟล์ถูกบีบอัด
  
### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไดเร็กทอรีเอกสารของคุณถูกต้องและสามารถเข้าถึงได้
- หากสิ่งที่แนบมาไม่ปรากฏขึ้น โปรดตรวจสอบว่าเส้นทางและชื่อไฟล์ถูกต้อง

## การประยุกต์ใช้งานจริง

การปิดการใช้งานการบีบอัดใน PDF มีการใช้งานจริงหลายประการ:
1. **เอกสารทางกฎหมาย**:รักษารูปแบบดั้งเดิมและความสมบูรณ์ของสัญญา
2. **การถ่ายภาพทางการแพทย์**:ป้องกันการสูญเสียรายละเอียดหรือคุณภาพในภาพทางการแพทย์ความละเอียดสูงที่แนบมากับรายงาน
3. **วัตถุประสงค์ด้านการเก็บถาวร**:รักษาความเที่ยงตรงของเอกสารประวัติศาสตร์เมื่อแปลงเอกสารเป็นดิจิทัล

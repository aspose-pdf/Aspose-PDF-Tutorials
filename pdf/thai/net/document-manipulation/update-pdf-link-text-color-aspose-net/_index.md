## คู่มือการใช้งาน
เรามาดูการใช้งานฟังก์ชันการอัปเดตสีข้อความลิงก์ในเอกสาร PDF กัน

### การโหลดและการแยกวิเคราะห์เอกสาร PDF
ขั้นแรก โหลดไฟล์ PDF ของคุณ ขั้นตอนนี้จะเริ่มต้นการทำงาน `Document` วัตถุซึ่งทำหน้าที่เป็นจุดเข้าสำหรับการจัดการเพิ่มเติม:

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_LinksActions();
Document doc = new Document(dataDir + "UpdateLinks.pdf");
```

### การระบุตำแหน่งลิงก์คำอธิบาย
ขั้นตอนต่อไป ให้ระบุคำอธิบายประกอบลิงก์ภายในเอกสารของคุณ ซึ่งเกี่ยวข้องกับการวนซ้ำผ่านคำอธิบายประกอบในหน้าเฉพาะและตรวจสอบว่าคำอธิบายประกอบเหล่านั้นเป็นประเภทใด `LinkAnnotation`-

```csharp
foreach (Annotation annotation in doc.Pages[1].Annotations)
{
    if (annotation is LinkAnnotation)
    {
        // ดำเนินการต่อไปที่นี่...
    }
}
```

### การแก้ไขสีข้อความ
เมื่อคุณได้ระบุคำอธิบายลิงก์แล้ว ให้ใช้ `TextFragmentAbsorber` เพื่อค้นหาชิ้นส่วนข้อความภายใต้ลิงก์เหล่านี้และอัปเดตสี:

```csharp
TextFragmentAbsorber ta = new TextFragmentAbsorber();
Rectangle rect = annotation.Rect;
rect.LLX -= 10; // ขยายพื้นที่การค้นหา
rect.LLY -= 10;
rect.URX += 10;
rect.URY += 10;
ta.TextSearchOptions = new TextSearchOptions(rect);
ta.Visit(doc.Pages[1]);

// เปลี่ยนสีข้อความ
foreach (TextFragment tf in ta.TextFragments)
{
    tf.TextState.ForegroundColor = Color.Red; // ตั้งเป็นสีแดง
}
```

### บันทึก PDF ที่อัปเดต
สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณ:

```csharp
dataDir = dataDir + "UpdateLinkTextColor_out.pdf";
doc.Save(dataDir);
Console.WriteLine("\nLinkAnnotation text color updated successfully.\nFile saved at " + dataDir);
```

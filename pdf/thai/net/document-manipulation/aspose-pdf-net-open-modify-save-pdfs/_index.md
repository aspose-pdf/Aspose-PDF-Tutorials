## คู่มือการใช้งาน

### คุณสมบัติ 1: เปิดและผูกเอกสาร PDF

#### ภาพรวม

การเปิดและเข้าเล่มเอกสาร PDF เป็นขั้นตอนแรกในกระบวนการแก้ไขใดๆ ซึ่งจะทำให้คุณสามารถโหลด PDF ลงในหน่วยความจำเพื่อดำเนินการต่อไป

**ขั้นตอน:**

##### เริ่มต้น FormEditor

```csharp
// เริ่มต้นวัตถุ FormEditor
FormEditor formEditor = new FormEditor();
```

นี่คือการเริ่มต้นอินสแตนซ์ของ `FormEditor` คลาสที่จะจัดการการดำเนินงานทั้งหมดในไฟล์ PDF ของคุณ

##### กำหนดไดเรกทอรีเอกสาร

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
แทนที่ `"YOUR_DOCUMENT_DIRECTORY"` ด้วยเส้นทางที่จัดเก็บไฟล์ PDF ของคุณ ตัวแทนไดเรกทอรีนี้ช่วยให้คุณสลับไดเรกทอรีได้อย่างง่ายดายเมื่อใช้งานแอปพลิเคชันของคุณ

##### เปิดและผูกไฟล์ PDF

```csharp
// เปิดและผูกเอกสาร PDF
formEditor.BindPdf(dataDir + "DeleteFormField.pdf");
```

การ `BindPdf` วิธีการนี้จะโหลดไฟล์ PDF ที่ระบุลงในหน่วยความจำเพื่อเตรียมการแก้ไข ตรวจสอบให้แน่ใจว่าเส้นทางถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาดขณะรันไทม์

### คุณสมบัติ 2: ลบฟิลด์ออกจากเอกสาร PDF

#### ภาพรวม

การลบฟิลด์ (เช่น กล่องข้อความหรือช่องกาเครื่องหมาย) สามารถทำให้เอกสารของคุณมีประสิทธิภาพมากขึ้นโดยลบองค์ประกอบที่ไม่จำเป็นออกไป

**ขั้นตอน:**

##### เปิดและผูกไฟล์ PDF

ขั้นแรก ให้แน่ใจว่าคุณได้เปิดและเข้าเล่ม PDF ตามที่อธิบายไว้ในหัวข้อก่อนหน้าแล้ว

```csharp
formEditor.BindPdf(dataDir + "DeleteFormField.pdf");
```

##### ลบฟิลด์ออก

```csharp
// ลบฟิลด์ที่ระบุออกจากเอกสาร PDF
formEditor.RemoveField("textfield");
```
การ `RemoveField` วิธีนี้จะใช้ชื่อของฟิลด์ที่คุณต้องการลบ แทนที่ `"textfield"` ด้วยชื่อจริงของฟิลด์เป้าหมายของคุณ

### คุณสมบัติที่ 3: บันทึกเอกสาร PDF ที่แก้ไขแล้ว

#### ภาพรวม

หลังจากทำการเปลี่ยนแปลงแล้ว สิ่งสำคัญคือต้องบันทึกงานของคุณ ขั้นตอนนี้จะช่วยให้มั่นใจว่าการแก้ไขทั้งหมดจะถูกเก็บไว้ในไฟล์ใหม่

**ขั้นตอน:**

##### กำหนดไดเรกทอรีเอาท์พุต

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```
คล้ายกับไดเรกทอรีอินพุต แทนที่ `"YOUR_OUTPUT_DIRECTORY"` กับที่คุณต้องการเก็บไฟล์ที่แก้ไข

##### บันทึกไฟล์ที่อัปเดต

```csharp
// บันทึกไฟล์ที่อัพเดตไปยังไดเร็กทอรีที่ระบุ
formEditor.Save(outputDir + "DeleteFormField_out.pdf");
```

การ `Save` วิธีการเขียนการเปลี่ยนแปลงของคุณไปยังไฟล์ PDF ใหม่ในตำแหน่งที่ระบุ โดยรักษาเอกสารต้นฉบับไว้

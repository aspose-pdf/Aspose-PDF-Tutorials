## คู่มือการใช้งาน
มาสำรวจวิธีการนำแต่ละคุณลักษณะไปใช้โดยใช้ Aspose.PDF สำหรับ .NET กัน

### การสร้าง TextFragment
#### ภาพรวม
การสร้างและเพิ่มส่วนข้อความลงใน PDF นั้นทำได้ง่าย ในส่วนนี้จะแนะนำคุณเกี่ยวกับการสร้าง "Hello World" เป็นเนื้อหาตัวอย่าง
**ขั้นตอนที่ 1: สร้างอินสแตนซ์เอกสาร**
```csharp
Document doc = new Document();
```
*ทำไม*:เริ่มต้นเอกสาร PDF ใหม่เพื่อการจัดการ

**ขั้นตอนที่ 2: เพิ่มหน้า**
```csharp
Page page = doc.Pages.Add();
```
*ทำไม*:เพิ่มหน้าว่างลงในเอกสารของคุณซึ่งจะวางส่วนข้อความไว้

**ขั้นตอนที่ 3: สร้างและเพิ่ม TextFragment**
```csharp
TextFragment text = new TextFragment("Hello World.. ");
page.Paragraphs.Add(text);
```
*ทำไม*:สไนปเป็ตนี้จะสร้างวัตถุข้อความและเพิ่มลงในคอลเล็กชันย่อหน้าของหน้า

**ขั้นตอนที่ 4: บันทึกเอกสาร**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(Path.Combine(outputDir, "TextFragment_out.pdf"));
```
*ทำไม*:บันทึกการเปลี่ยนแปลงของคุณเป็นไฟล์ PDF ในไดเร็กทอรีที่ระบุ

### การเพิ่มรูปภาพแบบอินไลน์
#### ภาพรวม
การเพิ่มรูปภาพลงในข้อความจะช่วยให้ PDF ของคุณดูน่าสนใจยิ่งขึ้น ต่อไปนี้เป็นวิธีฝังรูปภาพทันทีหลังจากส่วนข้อความที่เพิ่มไว้ก่อนหน้านี้
**ขั้นตอนที่ 1: สร้างภาพตัวอย่าง**
```csharp
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
image.IsInLineParagraph = true;
```
*ทำไม*:กำหนดค่ารูปภาพให้ได้รับการปฏิบัติแบบอินไลน์ เพื่อให้สามารถไหลไปในเนื้อหาข้อความได้อย่างราบรื่น

**ขั้นตอนที่ 2: ตั้งค่าคุณสมบัติของรูปภาพ**
```csharp
string imagePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
image.File = imagePath;
image.FixHeight = 30;
image.FixWidth = 100;
```
*ทำไม*:ระบุเส้นทางและขนาดของภาพเพื่อให้แน่ใจว่าพอดีกับเค้าโครงเอกสารของคุณ

**ขั้นตอนที่ 3: เพิ่มรูปภาพลงในหน้า**
```csharp
page.Paragraphs.Add(image);
```
*ทำไม*: เพิ่มรูปภาพที่กำหนดค่าเป็นองค์ประกอบย่อหน้าในคอลเล็กชั่นของหน้า

**ขั้นตอนที่ 4: บันทึกเอกสาร**
```csharp
doc.Save(Path.Combine(outputDir, "InlineImage_out.pdf"));
```
*ทำไม*:บันทึกเอกสารที่อัปเดตของคุณโดยมีรูปภาพอินไลน์รวมอยู่ด้วย

### การเพิ่ม Inline TextFragment หลังภาพ
#### ภาพรวม
ดำเนินการต่อจากจุดที่เราหยุดไว้ มาเพิ่มส่วนข้อความอีกส่วนหลังรูปภาพเพื่อสร้างเนื้อหาที่ต่อเนื่องไร้รอยต่อ
**ขั้นตอนที่ 1: เริ่มต้น TextFragment อีกครั้ง**
```csharp
TextFragment text = new TextFragment(" Hello Again..");
text.IsInLineParagraph = true;
```
*ทำไม*:เตรียมบล็อกข้อความใหม่สำหรับการเพิ่มแบบอินไลน์ โดยต่อจากรูปภาพโดยตรง

**ขั้นตอนที่ 2: เพิ่ม TextFragment ใหม่ลงในหน้า**
```csharp
page.Paragraphs.Add(text);
```
*ทำไม*:วางชิ้นส่วนข้อความให้ตรงกับองค์ประกอบเนื้อหาอื่น ๆ บนหน้า

**ขั้นตอนที่ 3: บันทึกเอกสาร**
```csharp
doc.Save(Path.Combine(outputDir, "InlineTextAfterImage_out.pdf"));
```
*ทำไม*:สรุปและบันทึกเอกสารของคุณซึ่งมีทั้งรูปภาพและส่วนข้อความเพิ่มเติม

## คู่มือการใช้งาน

### เปลี่ยนขนาดหน้า PDF

หัวข้อนี้สาธิตวิธีการเปลี่ยนขนาดหน้าของหน้าที่เลือกโดยใช้ Aspose.PDF สำหรับ .NET

#### ภาพรวม

เราจะใช้ `PdfPageEditor` จาก Aspose.Pdf.Facades เพื่อปรับเปลี่ยนเอกสาร PDF โดยการเปลี่ยนขนาดหน้า

**1. นำเข้าไลบรารี**

ขั้นแรก ให้แน่ใจว่าคุณมีเนมสเปซที่จำเป็นรวมอยู่ด้วย:

```csharp
using System;
using Aspose.Pdf.Facades;
```

**2. เริ่มต้นใช้งาน PdfPageEditor**

สร้างอินสแตนซ์ของ `PdfPageEditor` ในการทำงานกับไฟล์ PDF ของคุณ:

```csharp
PdfPageEditor pEdit = new PdfPageEditor();
```

**3. ผูกไฟล์ PDF**

ใช้ `BindPdf()` วิธีการโหลดไฟล์ PDF ลงในโปรแกรมแก้ไข:

```csharp
pEdit.BindPdf("YOUR_DOCUMENT_DIRECTORY/FilledForm.pdf");
```
แทนที่ "YOUR_DOCUMENT_DIRECTORY" ด้วยเส้นทางจริงของคุณ

**4. ระบุหน้าและเปลี่ยนขนาด**

กำหนดหน้าที่จะแก้ไขและกำหนดขนาดโดยใช้ `PageSize` การแจงนับ:

```csharp
// ดำเนินการเฉพาะหน้าแรกเท่านั้น (ดัชนี 1)
pEdit.ProcessPages = new int[] { 1 };
pEdit.PageSize = PageSize.PageLetter;
```
ที่นี่เราเลือกหน้าแรกและเปลี่ยนขนาดเป็น "LETTER"

**5. บันทึก PDF ที่แก้ไขแล้ว**

หลังจากทำการเปลี่ยนแปลงแล้ว ให้บันทึก PDF ของคุณด้วยชื่ออื่น:

```csharp
pEdit.Save("YOUR_OUTPUT_DIRECTORY/ChangePageSizes_out.pdf");
```
แทนที่ "YOUR_OUTPUT_DIRECTORY" ด้วยตำแหน่งที่คุณต้องการบันทึกไฟล์ที่แก้ไข

#### ตรวจสอบการเปลี่ยนแปลง

เชื่อมโยงใหม่และตรวจสอบว่าขนาดหน้าได้รับการอัปเดตอย่างถูกต้องหรือไม่:

```csharp
pEdit.BindPdf("YOUR_DOCUMENT_DIRECTORY/FilledForm.pdf");
PageSize size = pEdit.GetPageSize(1);
Console.WriteLine($"New Page Size: {size}");
```

## คู่มือการใช้งาน
มาแบ่งกระบวนการใช้งานออกเป็นขั้นตอนที่สามารถจัดการได้

### การแยกหน้าจากเอกสาร PDF
#### ภาพรวม
เราจะใช้ `PdfFileEditor` จาก `Aspose.Pdf.Facades` เพื่อแยกหน้าเฉพาะออกมา ซึ่งมีประโยชน์ในการแยกบทหรือส่วนต่างๆ โดยไม่ต้องใช้เอกสารทั้งหมด

#### ขั้นตอนการดำเนินการ
**ขั้นตอนที่ 1: เริ่มต้นใช้งาน PdfFileEditor**
สร้างอินสแตนซ์ของ `PdfFileEditor` ระดับ:

```csharp
// สร้างวัตถุ PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**ขั้นตอนที่ 2: กำหนดหน้าที่จะแยกออกมา**
ระบุหน้าที่คุณต้องการโดยใช้ตัวแปรจำนวนเต็ม ตัวอย่างเช่น การแยกหน้า 1 และ 2:

```csharp
int[] pagesToExtract = new int[] { 1, 2 };
```

**ขั้นตอนที่ 3: ดำเนินการแยกหน้า**
ใช้ `Extract` วิธีการบันทึกหน้าที่ระบุลงในไฟล์ PDF ใหม่ ระบุเส้นทางที่ถูกต้องสำหรับไฟล์อินพุตและเอาต์พุตของคุณ:

```csharp
// แยกหน้าออกมา
pdfEditor.Extract("path/to/your/input.pdf", pagesToExtract, "path/to/output/extracted_pages.pdf");
```

#### คำอธิบายส่วนประกอบหลัก
- **โปรแกรมแก้ไขไฟล์ PDF**:จัดการการดำเนินการต่างๆ เช่น การแตกไฟล์หรือรวมไฟล์ PDF
- **วิธีการสกัด**:ใช้เส้นทาง PDF ต้นทาง อาร์เรย์ของหมายเลขหน้าที่จะแยก และเส้นทางไฟล์ปลายทาง

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่ามี PDF อินพุตที่เส้นทางที่ระบุ
- ตรวจสอบว่าคุณมีสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาต์พุตหรือไม่

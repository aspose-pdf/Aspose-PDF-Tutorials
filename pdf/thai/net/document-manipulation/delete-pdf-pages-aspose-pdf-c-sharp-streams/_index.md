## คู่มือการใช้งาน
ปฏิบัติตามคำแนะนำนี้เพื่อลบหน้าเฉพาะจาก PDF โดยใช้สตรีม:

### การเริ่มต้นวัตถุ PdfFileEditor
การ `PdfFileEditor` คลาสเป็นศูนย์กลางในการแก้ไขไฟล์ PDF เริ่มต้นด้วยการสร้างอินสแตนซ์:
```csharp
// สร้างวัตถุ PdfFileEditor
PfPdFileEditor pdfEditor = new PfPdFileEditor();
```
วัตถุนี้จัดการงานการจัดการหน้าทั้งหมด รวมถึงการลบ

### การสร้างสตรีม
การเริ่มต้น `FileStream` วัตถุสำหรับการอ่านและเขียนข้อมูล PDF:
- **กระแสอินพุต:** ชี้ไปยังไฟล์ PDF แหล่งที่มา
- **กระแสข้อมูลขาออก:** กำหนดตำแหน่งที่จะบันทึก PDF ที่แก้ไข
```csharp
// สร้างสตรีมอินพุตและเอาต์พุต
using (FileStream inputStream = new FileStream("input.pdf", FileMode.Open))
using (FileStream outputStream = new FileStream("output.pdf", FileMode.Create))
{
    // การดำเนินการไปที่นี่
}
```

### การลบหน้า
ระบุหน้าที่จะลบโดยใช้จำนวนเต็มอาร์เรย์ ด้านล่างนี้เป็นวิธีการลบหน้า 1 และ 3:
```csharp
// อาร์เรย์ของหน้าที่จะลบ
int[] pagesToDelete = new int[] { 1, 3 };

// ลบหน้าที่ระบุ
cpdfEditor.Delete(inputStream, pagesToDelete, outputStream);
```
- **พารามิเตอร์:**
  - `inputStream`:แหล่งที่มาของสตรีม PDF
  - `pagesToDelete`:อาร์เรย์ที่มีหมายเลขหน้าที่ต้องการลบ
  - `outputStream`:ปลายทางสำหรับ PDF ที่ถูกแก้ไข

### การปิดลำธาร
ปิดสตรีมทุกครั้งหลังปฏิบัติการเพื่อปลดปล่อยทรัพยากร:
```csharp
// สตรีมจะถูกปิดโดยอัตโนมัติโดยใช้คำสั่ง 'using' ด้านบน
```

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้องและสามารถเข้าถึงได้
- ยืนยันว่าหมายเลขหน้าใน `pagesToDelete` ถูกต้อง (คือ อยู่ในช่วงของ PDF)

2. **การปรับแต่งของผู้ใช้:** อนุญาตให้ผู้ใช้ปรับแต่งรายงานโดยลบหน้าที่ไม่จำเป็นออกก่อนการแชร์
3. **การปรับเปลี่ยนการปฏิบัติตาม:** แก้ไขไฟล์ PDF ด้านกฎหมายหรือการเงินอย่างรวดเร็วเพื่อให้เป็นไปตามข้อกำหนดด้านกฎระเบียบ

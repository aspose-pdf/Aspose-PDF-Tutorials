## คู่มือการใช้งาน
สำรวจฟังก์ชันหลักบางอย่างที่คุณสามารถนำไปใช้กับ Aspose.PDF

### การสร้างและการจัดการ PDF

#### การสร้างเอกสารใหม่
การสร้าง PDF จากศูนย์เป็นเรื่องง่าย:

**การเริ่มต้นเอกสาร:**
```csharp
// สร้างวัตถุเอกสารใหม่
document = new Document();
```
**คำอธิบาย:** นี่เป็นการเริ่มต้นเอกสาร PDF ที่ว่างเปล่า

#### การเพิ่มข้อความและรูปภาพ
การเพิ่มเนื้อหาลงใน PDF ของคุณเป็นเรื่องง่ายด้วย Aspose.PDF

**เพิ่มข้อความลงในหน้า:**
```csharp
Page page = document.Pages.Add();
TextFragment textFragment = new TextFragment("Hello, World!");
page.Paragraphs.Add(textFragment);
```
**คำอธิบาย:** สไนปเป็ตนี้จะเพิ่มส่วนข้อความ "Hello, World!" ลงในหน้าที่เพิ่งสร้างขึ้น

#### การเพิ่มรูปภาพ

**ฝังรูปภาพ:**
```csharp
Image image = new Image();
image.File = "path/to/your/image.jpg";
page.Resources.Images.Add(image);
```
**คำอธิบาย:** โหลดและเพิ่มรูปภาพจากเส้นทางที่ระบุลงในเอกสาร PDF ของคุณ

### การผสานและแยกเอกสาร

#### การรวมไฟล์ PDF หลายไฟล์

**รวมไฟล์ PDF:**
```csharp
Document pdf1 = new Document("file1.pdf");
Document pdf2 = new Document("file2.pdf");
// เพิ่มหน้าของเอกสารที่สองลงในเอกสารแรก
pdf1.Pages.Add(pdf2.Pages);
pdf1.Save("merged.pdf");
```
**คำอธิบาย:** นี่รวมเข้าด้วยกัน `file2.pdf` เข้าไปข้างใน `file1.pdf`, บันทึกเอาไว้เป็น `merged-pdf`.

### การดึงข้อมูล

#### การแยกข้อความจาก PDF

**ดึงข้อความ:**
```csharp
TextAbsorber textAbsorber = new TextAbsorber();
document.Pages.Accept(textAbsorber);
string extractedText = textAbsorber.Text;
```
**คำอธิบาย:** ใช้ `TextAbsorber` เพื่อดึงเนื้อหาข้อความทั้งหมดจากเอกสาร

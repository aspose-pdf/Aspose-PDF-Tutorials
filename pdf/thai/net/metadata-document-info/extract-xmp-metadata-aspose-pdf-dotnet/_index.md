### การแยกข้อมูลเมตา XMP

**ภาพรวม:**
การแยกข้อมูลเมตา XMP เกี่ยวข้องกับการอ่านคุณสมบัติเฉพาะที่ฝังอยู่ในไฟล์ PDF ข้อมูลนี้อาจรวมถึงวันที่สร้าง รายละเอียดผู้เขียน และฟิลด์ที่กำหนดเอง

#### การดำเนินการทีละขั้นตอน:

1. **โหลดเอกสาร PDF**
   เริ่มต้นด้วยการโหลดเอกสาร PDF เป้าหมายของคุณลงใน `Aspose.Pdf.Document` วัตถุ.

   ```csharp
   // เส้นทางไปยังไดเรกทอรีที่บรรจุเอกสาร
   string dataDir = "path/to/your/documents/";

   // โหลดไฟล์ PDF
   Document pdfDocument = new Document(dataDir + "GetXMPMetadata.pdf");
   ```

2. **การเข้าถึงคุณสมบัติข้อมูลเมตา**
   ใช้ `Metadata` คุณสมบัติของวัตถุเอกสารของคุณเพื่อเข้าถึงข้อมูลเมตา XMP

   ```csharp
   // ดึงข้อมูลและแสดงคุณสมบัติเมตาข้อมูลเฉพาะ
   Console.WriteLine(pdfDocument.Metadata["xmp:CreateDate"]);
   Console.WriteLine(pdfDocument.Metadata["xmp:Nickname"]);
   Console.WriteLine(pdfDocument.Metadata["xmp:CustomProperty"]);
   ```

**คำอธิบายพารามิเตอร์:**
- `dataDir`:เส้นทางไดเร็กทอรีที่จัดเก็บไฟล์ PDF ของคุณ
- `Metadata`:วัตถุประเภทพจนานุกรมที่เก็บรายการเมตาข้อมูล XMP ทั้งหมด

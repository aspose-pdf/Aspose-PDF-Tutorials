## คู่มือการใช้งาน

ตอนนี้คุณได้ติดตั้ง Aspose.PDF และตั้งค่าสภาพแวดล้อมของคุณเสร็จเรียบร้อยแล้ว ลองมาดูวิธีนำคุณลักษณะแต่ละอย่างไปใช้กัน

### คุณสมบัติ 1: การแปลง PDF เป็น PPTX ขั้นพื้นฐาน

#### ภาพรวม
หัวข้อนี้ครอบคลุมการแปลงเอกสาร PDF เป็นงานนำเสนอ PowerPoint ในรูปแบบ PPTX โดยใช้ Aspose.PDF สำหรับ .NET

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: โหลดเอกสาร PDF**
```csharp
using Aspose.Pdf;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Aspose.Pdf.Document doc = new Aspose.Pdf.Document(dataDir + "/input.pdf");
```
*คำอธิบาย*:ที่นี่เราโหลด PDF จากไดเร็กทอรีที่ระบุโดยใช้ `Aspose-Pdf.Document`.

**ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการบันทึก PPTX**
```csharp
PptxSaveOptions pptx_save = new PptxSaveOptions();
```
*คำอธิบาย*: เราสร้างอินสแตนซ์ของ `PptxSaveOptions` เพื่อจัดการวิธีการบันทึก PDF เป็นไฟล์ PPTX

**ขั้นตอนที่ 3: บันทึกเอกสารในรูปแบบ PPTX**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputDir + "/PDFToPPT_out.pptx", pptx_save);
```
*คำอธิบาย*:ในที่สุดเราจะบันทึก PDF ที่โหลดเป็นไฟล์ PPTX ในตำแหน่งที่ต้องการ

### คุณสมบัติ 2: แปลงสไลด์ PDF เป็นรูปภาพใน PPTX

#### ภาพรวม
คุณสมบัตินี้ช่วยให้คุณแปลงสไลด์แต่ละภาพใน PDF ให้เป็นรูปภาพภายในงานนำเสนอ PPTX ที่ได้

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: ตั้งค่า SlidesAsImages**
```csharp
pptx_save.SlidesAsImages = true;
```
*คำอธิบาย*: โดยการตั้งค่า `SlidesAsImages` หากเป็นจริง แต่ละหน้าของ PDF จะถูกแปลงเป็นภาพสไลด์ในไฟล์ PowerPoint

### คุณสมบัติที่ 3: ติดตามความคืบหน้าระหว่างการแปลง PDF เป็น PPTX

#### ภาพรวม
ติดตามและแสดงความคืบหน้าการแปลงโดยใช้ตัวจัดการเหตุการณ์แบบกำหนดเองเพื่อให้ได้รับคำติชมจากผู้ใช้ที่ดีขึ้นในระหว่างกระบวนการ

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: กำหนดค่าตัวจัดการความคืบหน้าแบบกำหนดเอง**
```csharp
pptx_save.CustomProgressHandler = ShowProgressOnConsole;
```
*คำอธิบาย*:เราได้กำหนดวิธีการจัดการเหตุการณ์ความคืบหน้าซึ่งจะให้การอัปเดตแบบเรียลไทม์บนคอนโซล

### ฟังก์ชั่น: ตัวจัดการการติดตามความคืบหน้า

ฟังก์ชันนี้จัดการเหตุการณ์ความคืบหน้าต่างๆ ในระหว่างการแปลง:

```csharp
void ShowProgressOnConsole(PptxSaveOptions.ProgressEventHandlerInfo eventInfo)
{
    switch (eventInfo.EventType)
    {
        case ProgressEventType.TotalProgress:
            Console.WriteLine($"{DateTime.Now.TimeOfDay} - Conversion progress: {eventInfo.Value}%.");
            break;
        case ProgressEventType.ResultPageCreated:
            Console.WriteLine($"{DateTime.Now.TimeOfDay} - Result page's {eventInfo.Value} of {eventInfo.MaxValue} layout created.");
            break;
        // จัดการประเภทกิจกรรมอื่น ๆ ตามความจำเป็น
    }
}
```
*คำอธิบาย*:ฟังก์ชันนี้ให้ข้อเสนอแนะตามขั้นตอนต่างๆ ของกระบวนการแปลง เช่น ความคืบหน้าทั้งหมด และการสร้างหน้าผลลัพธ์

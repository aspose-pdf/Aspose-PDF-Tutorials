## คู่มือการใช้งาน
หัวข้อนี้จะสรุปการแปลงเอกสาร PDF เป็นไฟล์ XPS โดยใช้ Aspose.PDF สำหรับ .NET

### แปลง PDF เป็น XPS
แปลงไฟล์ PDF ที่มีอยู่ของคุณเป็นรูปแบบ XPS ได้อย่างง่ายดาย:

#### ขั้นตอนที่ 1: โหลดเอกสาร PDF
โหลด PDF ที่ต้องการการแปลง:
```csharp
using System;
using Aspose.Pdf;

// ระบุไดเรกทอรีอินพุตสำหรับไฟล์ PDF ของคุณ
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";

document = new Document(inputDirectory + "/input.pdf");
```

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการบันทึก XPS
ตั้งค่าตัวเลือกที่จำเป็นสำหรับการบันทึกในรูปแบบ XPS:
```csharp
// สร้างอินสแตนซ์ตัวเลือกการบันทึก XPS
var saveOptions = new Aspose.Pdf.XpsSaveOptions();
```

#### ขั้นตอนที่ 3: บันทึกเป็นเอกสาร XPS
แปลงและบันทึก PDF ของคุณเป็นไฟล์ XPS:
```csharp
// กำหนดไดเรกทอรีเอาท์พุตสำหรับไฟล์ XPS ที่แปลงแล้ว
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

document.Save(outputDirectory + "/PDFToXPS_out.xps", saveOptions);
```
**คำอธิบาย**: เดอะ `XpsSaveOptions` คลาสอนุญาตให้ปรับแต่งเฉพาะสำหรับการแปลง XPS

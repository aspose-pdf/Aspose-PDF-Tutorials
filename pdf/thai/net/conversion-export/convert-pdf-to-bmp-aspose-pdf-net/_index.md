## คู่มือการใช้งาน

ตอนนี้เรามาแปลงหน้า PDF เป็นภาพ BMP กัน

### แปลงหน้า PDF เป็นภาพ BMP

ฟีเจอร์นี้จะแปลงหน้า PDF แต่ละหน้าเป็นไฟล์ภาพ BMP แยกกัน ซึ่งมีประโยชน์สำหรับรูปแบบที่พร้อมพิมพ์หรือเนื้อหาดิจิทัลที่มีความละเอียดสูง

#### ขั้นตอนที่ 1: ตั้งค่าไดเร็กทอรีของคุณ

กำหนดไดเรกทอรีสำหรับไฟล์ PDF ต้นฉบับและรูปภาพเอาท์พุต:

```csharp
private const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
private const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
```
แทนที่ตัวแทนด้วยเส้นทางจริง

#### ขั้นตอนที่ 2: โหลดเอกสาร PDF ของคุณ

โหลดเอกสารโดยใช้ Aspose.PDF `Document` ระดับ:

```csharp
Document pdfDocument = new Document(DocumentDirectory + "/AddImage.pdf");
```

โหลด "AddImage.pdf" จากไดเร็กทอรีที่คุณระบุ

#### ขั้นตอนที่ 3: กำหนดค่าการตั้งค่าเอาท์พุต

ตั้งค่าความละเอียดสำหรับภาพเอาท์พุตและสร้าง `BmpDevice`-

```csharp
Resolution resolution = new Resolution(300); // ตั้งค่า DPI ที่ต้องการ เช่น 300 DPI สำหรับภาพคุณภาพสูง
BmpDevice bmpDevice = new BmpDevice(resolution);
```

#### ขั้นตอนที่ 4: แปลงแต่ละหน้าเป็นภาพ BMP

ทำซ้ำผ่านแต่ละหน้า PDF และแปลงเป็นภาพ BMP:

```csharp
for (int pageCount = 1; pageCount <= pdfDocument.Pages.Count; pageCount++)
{
    string outputPath = Path.Combine(OutputDirectory, "image" + pageCount + "_out.bmp");

    using (FileStream imageStream = new FileStream(outputPath, FileMode.Create))
    {
        bmpDevice.Process(pdfDocument.Pages[pageCount], imageStream);
    }
}
```

ลูปนี้จะประมวลผลแต่ละหน้าและบันทึกเป็นไฟล์ BMP ในไดเร็กทอรีเอาต์พุตที่ระบุ

### เคล็ดลับการแก้ไขปัญหา

- **ไฟล์ที่หายไป**: ตรวจสอบให้แน่ใจว่าเส้นทางเอกสารของคุณถูกต้อง
- **ปัญหาการอนุญาต**: ตรวจสอบว่าคุณมีสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาต์พุต
- **การตั้งค่าความละเอียด**:ปรับความละเอียดตามความต้องการด้านคุณภาพ DPI ที่สูงขึ้นส่งผลให้ไฟล์มีขนาดใหญ่ขึ้นแต่คุณภาพของภาพดีขึ้น

การแปลงหน้า PDF เป็นภาพ BMP มีประโยชน์สำหรับ:
1. **การเก็บถาวรและการสำรองข้อมูล**: จัดเก็บเวอร์ชันเอกสารคุณภาพสูงเป็นรูปภาพ
2. **การแบ่งปันเนื้อหา**:แบ่งปันหน้าเอกสารเฉพาะโดยไม่ต้องใช้โปรแกรมอ่าน PDF
3. **การประมวลผลภาพ**:ใช้รูปภาพที่แปลงแล้วในแอปพลิเคชันที่ต้องมีการปรับแต่ง

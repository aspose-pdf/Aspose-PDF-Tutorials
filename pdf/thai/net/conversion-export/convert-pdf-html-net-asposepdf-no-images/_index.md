## คู่มือการใช้งาน

ตอนนี้มาแบ่งกระบวนการใช้งานออกเป็นขั้นตอนที่สามารถจัดการได้

### การกำหนดค่าตัวเลือกการบันทึก HTML

**ภาพรวม**
เราจะกำหนดค่าตัวเลือกในการแปลง PDF เป็น HTML โดยยังคงฝังรูปภาพไว้ในพื้นหลังของหน้า วิธีนี้จะช่วยให้มั่นใจว่าเอกสารที่แปลงแล้วของคุณยังคงเค้าโครงเดิมไว้โดยไม่ต้องส่งออกรูปภาพแยกต่างหาก

#### ขั้นตอนที่ 1: เริ่มต้นเอกสารและบันทึกตัวเลือก
ตั้งค่าอินสแตนซ์เอกสารและระบุตัวเลือกบันทึกสำหรับการแปลง:

```csharp
// เส้นทางไปยังไฟล์ PDF อินพุตของคุณ
string dataDir = "YourDirectoryPath";

Document pdfDocument = new Document(dataDir + "/input.pdf");
HtmlSaveOptions saveOptions = new HtmlSaveOptions();
```

#### ขั้นตอนที่ 2: ตั้งค่าเค้าโครงและโหมดบันทึกรูปภาพ

กำหนดค่า `HtmlSaveOptions` เพื่อรักษาเค้าโครงคงที่และฝังรูปภาพ:

```csharp
saveOptions.FixedLayout = true;
// รูปภาพจะถูกบันทึกเป็น PNG ที่ฝังไว้ในพื้นหลังของหน้า
saveOptions.RasterImagesSavingMode = HtmlSaveOptions.RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground;
```

#### ขั้นตอนที่ 3: บันทึกเอกสาร
สุดท้ายให้บันทึกเอกสาร PDF ของคุณในรูปแบบ HTML โดยใช้การตั้งค่าเหล่านี้:

```csharp
pdfDocument.Save(dataDir + "/AvoidSavingImages_out.html", saveOptions);
```

**เคล็ดลับการแก้ไขปัญหา**: ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์อินพุตถูกต้องและสามารถเข้าถึงได้เพื่อหลีกเลี่ยงข้อยกเว้นในระหว่างการโหลด

## คู่มือการใช้งาน

มาดูขั้นตอนสำคัญในการแปลงไฟล์ MHT เป็น PDF กัน

### แปลง MHT เป็น PDF

**ภาพรวม:**
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ MHT ซึ่งเป็นไฟล์เก็บถาวร HTML ที่มีรีซอร์สฝังอยู่ ให้เป็นเอกสาร PDF โดยใช้ Aspose.PDF สำหรับ .NET

**ขั้นตอนการดำเนินการ:**
1. **ตั้งค่าสภาพแวดล้อมของคุณ**
   กำหนดเส้นทางไดเร็กทอรีสำหรับไฟล์อินพุตและเอาต์พุตของคุณ:
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **สร้างตัวเลือกการโหลด**
   การเริ่มต้น `MhtLoadOptions` ในการกำหนดค่าการตั้งค่าการโหลดที่เฉพาะเจาะจงที่คุณอาจต้องการ:
   
   ```csharp
   MhtLoadOptions options = new MhtLoadOptions();
   // ตัวอย่างการกำหนดค่า (ลบความคิดเห็นหากจำเป็น):
   // ตัวเลือก.PageInfo.IsLandscape = true;
   ```

3. **โหลดเอกสาร MHT**
   ใช้ `MhtLoadOptions` เพื่อโหลดไฟล์ MHT ของคุณลงใน `Document` วัตถุ:
   
   ```csharp
   Document document = new Document(dataDir + "/test.mht", options);
   ```

4. **บันทึกเป็น PDF**
   สุดท้ายให้บันทึกเอกสารที่โหลดเป็น PDF ในไดเร็กทอรีเอาต์พุตที่ต้องการ:
   
   ```csharp
   document.Save(outputDir + "/MHTToPDF_out.pdf");
   ```

### กำหนดค่าตัวเลือกการโหลด MHT

**ภาพรวม:**
การปรับแต่งตัวเลือกการโหลดจะช่วยเพิ่มประสิทธิภาพวิธีการประมวลผลไฟล์ MHT ของคุณก่อนการแปลง

**ขั้นตอนการดำเนินการ:**
1. **สร้างและปรับแต่ง `MhtLoadOptions`**
   
   ```csharp
   MhtLoadOptions options = new MhtLoadOptions();
   // ตั้งค่าคุณสมบัติตามต้องการ เช่น การวางแนวหน้า:
   // ตัวเลือก.PageInfo.IsLandscape = true;
   ```

### โหลดเอกสารจากไฟล์

**ภาพรวม:**
การโหลดเอกสารที่มีอยู่อย่างถูกต้องเป็นสิ่งสำคัญสำหรับการประมวลผลและการแปลง

**ขั้นตอนการดำเนินการ:**
1. **โหลดไฟล์ MHT**
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   MhtLoadOptions loadOptions = new MhtLoadOptions();
   Document loadedDocument = new Document(dataDir + "/test.mht", loadOptions);
   ```

### บันทึกเอกสารลงในไฟล์

**ภาพรวม:**
การบันทึกเอกสารที่ได้รับการประมวลผลเป็นขั้นตอนสุดท้ายในเวิร์กโฟลว์การแปลงของคุณ

**ขั้นตอนการดำเนินการ:**
1. **บันทึกเป็น PDF**
   
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   loadedDocument.Save(outputDir + "/output.pdf");
   ```

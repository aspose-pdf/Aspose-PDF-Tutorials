## คู่มือการใช้งาน

### การแปลงไฟล์ PDF เป็น TIFF ด้วยความลึกของสีที่เฉพาะเจาะจง

คุณสมบัตินี้ช่วยให้คุณแปลงไฟล์ PDF เป็นภาพ TIFF พร้อมทั้งระบุความลึกของสี เช่น 1 บิตต่อพิกเซล (bpp)

#### ภาพรวม
การแปลง PDF เป็น TIFF ขาวดำสามารถช่วยเพิ่มประสิทธิภาพการจัดเก็บและการประมวลผลสำหรับแอพพลิเคชั่นบางอย่างได้

#### การดำเนินการแบบทีละขั้นตอน

##### กำหนดไดเรกทอรีอินพุตและเอาต์พุต

ตั้งค่าไดเร็กทอรีของคุณด้วยช่องว่างสำหรับไฟล์ PDF อินพุตและภาพ TIFF เอาท์พุต

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

##### เริ่มต้นวัตถุ PdfConverter

สร้าง `PdfConverter` วัตถุ โดยระบุความละเอียดที่ต้องการ (เช่น 300 DPI)

```csharp
PdfConverter pdfConverter = new PdfConverter();
pdfConverter.Resolution = new Resolution(300);
```

##### ผูกไฟล์ PDF อินพุต

ผูกไฟล์ PDF อินพุตของคุณเข้ากับตัวแปลง

```csharp
pdfConverter.BindPdf(dataDir + "/inFile.pdf");
```

##### ดำเนินการแปลงด้วยความลึกของสีที่เฉพาะเจาะจง

การกำหนดค่า `TiffSettings` สำหรับความลึกสีที่เฉพาะเจาะจงเช่น 1bpp และดำเนินการแปลง

```csharp
if (pdfConverter.DoConvert())
{
    TiffSettings tiffSettings = new TiffSettings();
tiffSettings.Depth = ColorDepth.Format1bpp;
pdfConverter.SaveAsTIFF(outputDir + "/PDFToTIFFConversion_out.tif", 300, 300, tiffSettings);
}
```

##### ทำความสะอาด

ปิด `PdfConverter` คัดค้านการปล่อยทรัพยากร

```csharp
pdfConverter.Close();
```

### การแปลงไฟล์ PDF เป็น TIFF ด้วยโปรแกรมแปลงรูปภาพแบบกำหนดเอง

สำหรับสถานการณ์การแปลงขั้นสูง ให้ใช้ตัวแปลงรูปภาพแบบกำหนดเอง

#### ภาพรวม
แนวทางนี้ช่วยให้สามารถปรับแต่งการประมวลผลได้ในระหว่างการแปลง เช่น การเปลี่ยนแปลงความลึกบิตแบบไดนามิก

#### การดำเนินการแบบทีละขั้นตอน

##### ใช้ตัวแปลงรูปภาพที่กำหนดเอง

หลังจากผูก PDF และดำเนินการตรวจสอบเบื้องต้น:

```csharp
if (pdfConverter.DoConvert())
{
    TiffSettings tiffSettings = new TiffSettings();
pdfConverter.SaveAsTIFF(outputDir + "/PDFToTIFFConversion_out.tif", 300, 300, tiffSettings, new WinAPIIndexBitmapConverter());
}
```

### โปรแกรมแปลงรูปภาพแบบกำหนดเองสำหรับการแปลง TIFF

สร้างตัวแปลงแบบกำหนดเองเพื่อจัดการกับความลึกบิตที่แตกต่างกัน

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณแปลงรูปภาพเป็นความลึกบิตต่างๆ ได้โดยใช้การเรียก Windows API โดยตรงจาก C#

#### รายละเอียดการดำเนินการ

กำหนดความหมาย `WinAPIIndexBitmapConverter` คลาสซึ่งใช้วิธีการเช่น `Get1BppImage`- `Get4BppImage`, และ `Get8BppImage`-

```csharp
class WinAPIIndexBitmapConverter : IIndexBitmapConverter
{
    // การนำวิธีการไปใช้งานเพื่อแปลงภาพให้มีความลึกบิตที่แตกต่างกัน
}
```

ตัวแปลงที่กำหนดเองนี้ใช้การเรียก P/Invoke เพื่อโต้ตอบกับฟังก์ชัน Windows GDI สำหรับการจัดการรูปภาพ

### การประยุกต์ใช้งานจริง
- **การจัดเก็บข้อมูลถาวร**:แปลงเอกสาร PDF คุณภาพสูงเป็นรูปแบบ TIFF เพื่อการจัดเก็บในระยะยาว
- **การสแกนเอกสาร**:รวมฟีเจอร์การแปลงนี้ไว้ในแอปพลิเคชันการสแกนเอกสาร
- **ท่อประมวลผลภาพ**:ใช้ตัวแปลงแบบกำหนดเองในไปป์ไลน์ที่ต้องการความลึกบิตที่เฉพาะเจาะจง
- **การถ่ายภาพทางการแพทย์**:ปรับความลึกของสีเพื่อให้เหมาะกับความต้องการด้านการถ่ายภาพทางการแพทย์

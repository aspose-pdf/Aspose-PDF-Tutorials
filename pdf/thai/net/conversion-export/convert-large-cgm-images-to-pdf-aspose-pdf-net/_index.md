### แปลงภาพ CGM ขนาดใหญ่เป็น PDF (H3)
**ภาพรวม**
ในส่วนนี้ เราจะแปลงไฟล์ CGM ขนาดใหญ่เป็น PDF โดยใช้ Aspose.PDF .NET ฟีเจอร์นี้มีประโยชน์อย่างยิ่งเมื่อต้องจัดการกับกราฟิกความละเอียดสูงที่ต้องแชร์หรือพิมพ์ในรูปแบบที่เข้าถึงได้ง่ายกว่า

#### การดำเนินการแบบทีละขั้นตอน
1. **ตั้งค่าเส้นทางและตัวเลือก**
   เริ่มต้นด้วยการระบุเส้นทางไฟล์อินพุตและเอาต์พุต นอกจากนี้ เราจะกำหนดค่าตัวเลือกขนาดหน้าสำหรับกระบวนการแปลงด้วย
   ```csharp
   using System;
   using Aspose.Pdf.Facades;

   // ตั้งค่าไดเร็กทอรีเอกสารและไดเร็กทอรีเอาท์พุตของคุณ
   string inputFile = "YOUR_DOCUMENT_DIRECTORY/corvette.cgm";
   string outputFile = "YOUR_OUTPUT_DIRECTORY/LargeCGMImageToPDF_out.pdf";

   // สร้างอินสแตนซ์ของ CgmImportOptions เพื่อตั้งค่าตัวเลือก
   CgmImportOptions options = new CgmImportOptions();
   options.PageSize = new SizeF(1000, 1000); // ระบุขนาดหน้าตามต้องการ
   ```
2. **ดำเนินการแปลง**
   ใช้ `PdfProducer.Produce` เพื่อแปลงไฟล์ CGM เป็นรูปแบบ PDF
   ```csharp
   PdfProducer.Produce(inputFile, ImportFormat.Cgm, outputFile);
   ```

#### คำอธิบาย
- **ตัวเลือกการนำเข้า Cgm**:คลาสนี้ช่วยให้คุณสามารถระบุตัวเลือกการนำเข้าต่างๆ เช่น ขนาดหน้า ในตัวอย่างนี้ เราตั้งค่าเป็นหน่วย 1000x1000
- **PdfProducer.ผลิต**:จัดการกระบวนการแปลงโดยการเลือกเส้นทางไฟล์อินพุต รูปแบบ และเส้นทางไฟล์เอาต์พุต

## คู่มือการใช้งาน (H2)
ให้เราแบ่งกระบวนการแปลงออกเป็นสองคุณสมบัติหลัก: การแปลงไฟล์โดยตรงและการแปลงแบบสตรีม

### คุณสมบัติ 1: การแปลง PCL เป็น PDF โดยตรง
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ PCL เป็นเอกสาร PDF โดยตรงโดยใช้ Aspose.PDF สำหรับ .NET

#### การดำเนินการทีละขั้นตอน:
**เอช3**:ตัวเลือกการเริ่มต้นโหลด PCL
```csharp
// ขั้นตอนที่ 1: เริ่มต้นตัวเลือกการโหลด PCL
Aspose.Pdf.LoadOptions loadopt = new Aspose.Pdf.PclLoadOptions();
```
*คำอธิบาย*: เดอะ `PclLoadOptions` มีความจำเป็นในการระบุว่าจะตีความไฟล์ PCL อย่างไร

**เอช3**: สร้างวัตถุเอกสาร
```csharp
// ขั้นตอนที่ 2: สร้างวัตถุเอกสารโดยใช้ไฟล์ PCL อินพุตและโหลดตัวเลือก
Aspose.Pdf.Document doc = new Aspose.Pdf.Document(dataDir + "/hidetext.pcl", loadopt);
```
*คำอธิบาย*: เดอะ `Document` คลาสนี้ใช้ในการจัดการเอกสาร PDF และที่นี่จะเริ่มต้นด้วยไฟล์ PCL

**เอช3**: บันทึกผลลัพธ์ PDF
```csharp
// ขั้นตอนที่ 3: บันทึกเอกสาร PDF ที่ได้ผลลัพธ์
doc.Save(outputDir + "/PCLToPDF_out.pdf");
```
*คำอธิบาย*: เดอะ `Save` วิธีการเขียนเอกสารที่แปลงแล้วไปยังเส้นทางที่คุณระบุ

### คุณสมบัติ 2: การแปลง PCL เป็น PDF แบบสตรีม
คุณลักษณะนี้สาธิตการแปลงไฟล์ PCL เป็น PDF โดยใช้สตรีมในหน่วยความจำ ซึ่งมีประโยชน์ในการจัดการไฟล์ขนาดใหญ่หรือการบูรณาการกับแอปพลิเคชันเว็บ

#### การดำเนินการทีละขั้นตอน:
**เอช3**: เปิดและอ่าน FileStream
```csharp
// ขั้นตอนที่ 1: เปิด FileStream สำหรับไฟล์ PCL
using (FileStream fileStream = new FileStream(dataDir + "/sample.pcl", FileMode.Open))
```
*คำอธิบาย*- `FileStream` ใช้เพื่ออ่านข้อมูลจากไฟล์ PCL ของคุณลงในหน่วยความจำ

**เอช3**: ตั้งค่าตัวเลือกการโหลด
```csharp
// ขั้นตอนที่ 2: ตั้งค่าตัวเลือกโหลด PCL ด้วยเครื่องมือแปลงที่ต้องการ
PclLoadOptions pclLoadOptions = new PclLoadOptions
{
    ConversionEngine = PclLoadOptions.ConversionEngines.LegacyEngine
};
```
*คำอธิบาย*: การกำหนดค่า `ConversionEngine` ช่วยควบคุมวิธีการตีความเอกสารของคุณในระหว่างการแปลง

**เอช3**: จัดการ MemoryStream
```csharp
// ขั้นตอนที่ 4: รีเซ็ตตำแหน่งของสตรีมหน่วยความจำไปที่จุดเริ่มต้น
memoryStream.Seek(0, SeekOrigin.Begin);
```
*คำอธิบาย*:สิ่งสำคัญคือต้องรีเซ็ตตำแหน่งสตรีมหน่วยความจำเพื่อให้สามารถอ่านข้อมูลได้อย่างถูกต้องในการดำเนินการครั้งต่อไป

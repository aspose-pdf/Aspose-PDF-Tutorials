## คู่มือการใช้งาน

### คุณสมบัติ 1: การแปลง PDF เป็น HTML พร้อมการฝังทรัพยากร

คุณลักษณะนี้มุ่งเน้นที่การแปลงเอกสาร PDF เป็นรูปแบบ HTML ในขณะที่ฝังทรัพยากรทั้งหมดไว้ในไฟล์ HTML เอาท์พุต

#### ภาพรวม:
การแปลงจะทำให้แน่ใจว่าแบบอักษร รูปภาพ และสไตล์ทั้งหมดจะถูกรวมไว้ใน HTML ซึ่งจะทำให้ได้ประสบการณ์ที่เป็นอิสระและสอดคล้องกันทางภาพในเบราว์เซอร์ต่างๆ

##### ขั้นตอนที่ 1: โหลด PDF ต้นฉบับ

```csharp
using System;
using Aspose.Pdf;

string documentPath = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document doc = new Document(documentPath);
```
ที่นี่ เราจะโหลดไฟล์ PDF ต้นฉบับ ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาดในการโหลด

##### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการบันทึก HTML

```csharp
using Aspose.Pdf.Saving;

HtmlSaveOptions options = new HtmlSaveOptions();
options.PartsEmbeddingMode = PartsEmbeddingModes.EmbedAllIntoHtml;
```
เราสร้างตัวอย่าง `HtmlSaveOptions` และกำหนดค่าให้ฝังทรัพยากรทั้งหมดลงในไฟล์ HTML

##### ขั้นตอนที่ 3: เพิ่มประสิทธิภาพสำหรับ Internet Explorer

```csharp
options.LettersPositioningMethod = LettersPositioningMethods.UseEmUnitsAndCompensationOfRoundingErrorsInCss;
```
การตั้งค่านี้จะช่วยให้แน่ใจว่าข้อความปรากฏอย่างถูกต้องใน IE ด้วยการใช้หน่วย em และชดเชยข้อผิดพลาดในการปัดเศษ CSS

##### ขั้นตอนที่ 4: บันทึกภาพแรสเตอร์เป็น PNG ที่ฝังไว้

```csharp
options.RasterImagesSavingMode = RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground;
```
แสดงรูปภาพเป็นส่วนที่ฝังอยู่ในพื้นหลังของหน้า ช่วยให้มั่นใจถึงความสมจริงของภาพในทุกแพลตฟอร์ม

##### ขั้นตอนที่ 5: กำหนดค่าโหมดการบันทึกแบบอักษร

```csharp
options.FontSavingMode = FontSavingModes.SaveInAllFormats;
```
ตัวเลือกนี้จะบันทึกแบบอักษรในทุกรูปแบบเพื่อความเข้ากันได้สูงสุดและรักษาคุณภาพ

##### ขั้นตอนที่ 6: บันทึกไฟล์ HTML ที่แปลงแล้ว

```csharp
string outputHtmlPath = "YOUR_OUTPUT_DIRECTORY/SingleHTML_out.html";
doc.Save(outputHtmlPath, options);
```
สุดท้าย ให้บันทึกไฟล์ HTML ที่แปลงแล้วไปยังไดเร็กทอรีที่คุณระบุ ตรวจสอบให้แน่ใจว่าเส้นทางเอาต์พุตของคุณถูกต้องเพื่อป้องกันข้อผิดพลาด

### คุณสมบัติ 2: เพิ่มประสิทธิภาพการแปลง PDF เป็น HTML

คุณลักษณะนี้ขยายการแปลงพื้นฐานโดยรวมการกำหนดค่าเพิ่มเติมเพื่อเพิ่มประสิทธิภาพและประสิทธิภาพการทำงาน

ขั้นตอนการใช้งานจะคล้ายกับที่ระบุไว้ข้างต้น แต่มุ่งเน้นไปที่การปรับปรุงความเข้ากันได้ระหว่างเบราว์เซอร์ต่างๆ ที่มีการตั้งค่าขั้นสูง

## การประยุกต์ใช้งานจริง

1. **เว็บพอร์ทัล**:แปลงคู่มือผู้ใช้เป็นคู่มือแบบโต้ตอบบนเว็บ
2. **ระบบจัดการเอกสาร**:รวมรายงานบนพื้นฐาน PDF ลงในแอปพลิเคชันเว็บโดยตรงในรูปแบบหน้า HTML
3. **แพลตฟอร์มการเรียนรู้ทางอิเล็กทรอนิกส์**:เปลี่ยนแปลงสื่อการเรียนรู้จากรูปแบบดั้งเดิมเป็นแบบไดนามิกเพื่อการมีส่วนร่วมที่ดีขึ้น
4. **เครือข่ายการส่งมอบเนื้อหา**:ฝังเนื้อหาสื่อที่หลากหลายภายในบทความและบทช่วยสอนโดยไม่ต้องใช้ทรัพยากรภายนอก

## การพิจารณาประสิทธิภาพ

เมื่อแปลงไฟล์ PDF ขนาดใหญ่หรือซับซ้อน ควรพิจารณาเคล็ดลับต่อไปนี้:

- เพิ่มประสิทธิภาพการใช้ทรัพยากรโดยลดขนาดไฟล์ที่ฝังไว้ให้น้อยที่สุดเมื่อทำได้
- ใช้แนวทางการจัดการหน่วยความจำที่มีประสิทธิภาพใน .NET เพื่อจัดการเอกสารขนาดใหญ่ได้อย่างราบรื่น
- สร้างโปรไฟล์ประสิทธิภาพการทำงานของแอปพลิเคชันของคุณเพื่อระบุจุดคอขวดระหว่างการแปลง

**คำถามที่ 2: ประโยชน์จากการฝังทรัพยากรทั้งหมดไว้ในไฟล์ HTML มีอะไรบ้าง**

**คำถามที่ 3: ฉันสามารถใช้ Aspose.PDF สำหรับ .NET ในแอปพลิเคชันเชิงพาณิชย์ได้หรือไม่**

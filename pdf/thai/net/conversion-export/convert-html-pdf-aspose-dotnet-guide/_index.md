## คู่มือการใช้งาน

เราจะแนะนำฟีเจอร์ต่างๆ ของ Aspose.PDF .NET เพื่อแปลงเนื้อหา HTML เป็น PDF โดยเน้นที่การโหลดทรัพยากรแบบกำหนดเอง การเรนเดอร์หน้าเดียว และการจัดการข้อมูล SVG ภายในเอกสาร

### คุณสมบัติ 1: แปลง HTML เป็น PDF ด้วยทรัพยากรที่กำหนดเอง

ฟีเจอร์นี้ช่วยให้คุณปรับแต่งวิธีการโหลดทรัพยากรภายนอกระหว่างการแปลงได้ โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับรูปภาพหรือทรัพยากรเฉพาะที่ต้องการการจัดการเป็นพิเศษ

#### ภาพรวม
จุดมุ่งหมายคือการโหลดรูปภาพเป็นอาร์เรย์ไบต์และใช้ใน PDF ผลลัพธ์ โดยแทนที่กลไกการโหลดทรัพยากรเริ่มต้นของ Aspose.PDF

**ขั้นตอนที่ 1: เริ่มต้น HtmlLoadOptions**

เราเริ่มต้นด้วยการสร้าง `HtmlLoadOptions` และการกำหนดตัวโหลดแบบกำหนดเองสำหรับทรัพยากรภายนอก

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// กลยุทธ์การโหลดทรัพยากรที่กำหนดเอง
HtmlLoadOptions options = new HtmlLoadOptions();
options.CustomLoaderOfExternalResources = new LoadOptions.ResourceLoadingStrategy(SamePictureLoader);
```

**ขั้นตอนที่ 2: สร้างเอกสาร PDF จาก HTML**

เมื่อตั้งค่าตัวเลือกแล้ว เราจะโหลดไฟล์ HTML และแปลงเป็นเอกสาร PDF

```csharp
Document pdfDocument = new Document(dataDir + "/HTMLToPDF.html", options);
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/HTMLToPDF_out.pdf");
```

**ขั้นตอนที่ 3: กำหนดตัวโหลดทรัพยากรแบบกำหนดเอง**

การ `SamePictureLoader` ฟังก์ชันนี้ระบุวิธีการโหลดทรัพยากรภายนอก ที่นี่ เราจะโหลดไฟล์รูปภาพเฉพาะเป็นไบต์

```csharp
private static LoadOptions.ResourceLoadingResult SamePictureLoader(string resourceURI)
{
    byte[] resultBytes = File.ReadAllBytes(dataDir + "/aspose-logo.jpg");
    return new LoadOptions.ResourceLoadingResult(resultBytes);
}
```

### คุณสมบัติ 2: เรนเดอร์เนื้อหา HTML ลงใน PDF หน้าเดียว

การแปลงเนื้อหา HTML หลายหน้าให้เป็นหน้าเดียวอาจมีความจำเป็นสำหรับการสร้างรายงานหรือสรุปแบบกระชับ

#### ภาพรวม
เราจะเปิดใช้งาน `IsRenderToSinglePage` คุณสมบัติในตัวเลือกของเราเพื่อให้แน่ใจว่าเอกสารทั้งหมดพอดีกับหนึ่งหน้า

**ขั้นตอนที่ 1: ตั้งค่า HtmlLoadOptions**

เปิดใช้งานการเรนเดอร์หน้าเดียวโดยการกำหนดค่า `HtmlLoadOptions`-

```csharp
HtmlLoadOptions options = new HtmlLoadOptions();
options.IsRenderToSinglePage = true;
```

**ขั้นตอนที่ 2: โหลดและบันทึก HTML เป็น PDF หน้าเดียว**

ตอนนี้ เราโหลดเอกสารที่มีตัวเลือกเหล่านี้และบันทึก

```csharp
Document doc = new Document(dataDir + "/HTMLToPDF.html", options);
doc.Save("YOUR_OUTPUT_DIRECTORY/RenderContentToSamePage.pdf");
```

### คุณสมบัติที่ 3: เรนเดอร์ HTML ด้วยข้อมูล SVG ใน PDF

การจัดการข้อมูล SVG ภายใน HTML อาจเป็นเรื่องยุ่งยาก แต่ Aspose.PDF ช่วยให้ทำได้อย่างราบรื่น

#### ภาพรวม
คุณลักษณะนี้มุ่งเน้นที่การแสดงไฟล์ HTML ที่มีรูปภาพ SVG ที่ฝังอยู่ในเอกสาร PDF

**ขั้นตอนที่ 1: ระบุเส้นทางอินพุตและเอาต์พุต**

กำหนดเส้นทางสำหรับไฟล์อินพุต HTML และเอาต์พุต PDF

```csharp
string inFile = dataDir + "/HTMLSVG.html";
string outFile = "YOUR_OUTPUT_DIRECTORY/RenderHTMLwithSVGData.pdf";
```

**ขั้นตอนที่ 2: เริ่มต้นตัวเลือกด้วยเส้นทางไดเร็กทอรี**

ตั้งค่า `HtmlLoadOptions` พร้อมด้วยเส้นทางไดเร็กทอรีของไฟล์ HTML เพื่อให้แน่ใจว่า SVG ได้รับการแก้ไขอย่างถูกต้อง

```csharp
HtmlLoadOptions options = new HtmlLoadOptions(Path.GetDirectoryName(inFile));
Document pdfDocument = new Document(inFile, options);
pdfDocument.Save(outFile);
```

## การประยุกต์ใช้งานจริง

ความสามารถของ Aspose.PDF .NET ขยายออกไปนอกเหนือจากการแปลงพื้นฐาน นี่คือแอปพลิเคชันในโลกแห่งความเป็นจริงบางส่วน:

1. **แคตตาล็อกสินค้าอีคอมเมิร์ซ:** แปลงคำอธิบายผลิตภัณฑ์แบบไดนามิกเป็นแคตตาล็อก PDF ที่สามารถพิมพ์ได้

3. **การสร้างรายงานอัตโนมัติ:** สร้างบทสรุปผู้บริหารแบบหน้าเดียวจากรายงาน HTML หลายหน้าที่ครอบคลุม

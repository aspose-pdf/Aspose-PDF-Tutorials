## คู่มือการใช้งาน

มาแบ่งกระบวนการแปลงออกเป็นขั้นตอนกัน

### โหลดเอกสาร PDF

**ภาพรวม**ขั้นตอนแรกคือการโหลดเอกสาร PDF ที่คุณต้องการแปลง ซึ่งเกี่ยวข้องกับการสร้างอินสแตนซ์ของ `Document` ระดับ.

```csharp
// โหลดเอกสาร PDF จากไดเร็กทอรีที่ระบุ
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

โค้ดตัวอย่างนี้จะเริ่มต้น `Document` วัตถุที่แสดงไฟล์ PDF ของคุณสำหรับการจัดการและการแปลง

### กำหนดค่าตัวเลือกการบันทึก SVG

**ภาพรวม**ต่อไป ให้กำหนดค่าวิธีการบันทึก PDF เป็น SVG ซึ่งรวมถึงการตั้งค่าต่างๆ เช่น การตั้งค่าการบีบอัด

```csharp
// สร้างอินสแตนซ์ของวัตถุของ SvgSaveOptions ด้วยการกำหนดค่าเฉพาะ
SvgSaveOptions saveOptions = new SvgSaveOptions();

// ตั้งค่าเป็นเท็จเพื่อให้แน่ใจว่าแต่ละหน้าได้รับการบันทึกเป็นไฟล์ .svg แยกกัน
saveOptions.CompressOutputToZipArchive = false;
```

**คำอธิบาย**- 
- `CompressOutputToZipArchive` ถูกตั้งเป็น `false` เพื่อให้แต่ละหน้า PDF บันทึกเป็นไฟล์แยกกัน `.svg` ไฟล์.

### บันทึกเอกสารเป็น SVG

**ภาพรวม**สุดท้ายให้บันทึกเอกสารของคุณในรูปแบบ SVG โดยใช้ตัวเลือกที่ระบุ

```csharp
// บันทึกเอกสารเป็นไฟล์ SVG ในไดเร็กทอรีที่ระบุ
doc.Save("YOUR_OUTPUT_DIRECTORY/PDFToSVG_out.svg", saveOptions);
```

ขั้นตอนนี้จะเขียนไฟล์ SVG ที่แปลงแล้วไปยังไดเร็กทอรีเอาต์พุตที่คุณต้องการ โดยแต่ละหน้า PDF จะบันทึกเป็นไฟล์ SVG แยกกันหากกำหนดค่าไว้อย่างเหมาะสม

### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาเส้นทางไฟล์**: ตรวจสอบให้แน่ใจว่าระบุเส้นทางอินพุตและเอาต์พุตอย่างถูกต้อง
- **การอนุญาต**: ตรวจสอบว่าแอปพลิเคชันของคุณมีสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาต์พุต

2. **การออกแบบกราฟิก**:แปลงไดอะแกรม PDF โดยละเอียดเป็นไฟล์ SVG ที่แก้ไขได้เพื่อการจัดการเพิ่มเติม

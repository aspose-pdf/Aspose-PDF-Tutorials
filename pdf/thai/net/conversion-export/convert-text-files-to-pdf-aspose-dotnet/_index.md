## คู่มือการใช้งาน

### คุณสมบัติการแปลงข้อความเป็น PDF
ฟีเจอร์นี้ช่วยให้คุณแปลงไฟล์ข้อความธรรมดาเป็นเอกสาร PDF ได้อย่างง่ายดาย โดยวิธีการทำงานแบบทีละขั้นตอนมีดังนี้:

#### ขั้นตอนที่ 1: อ่านไฟล์ข้อความต้นฉบับ
ขั้นแรก ให้อ่านเนื้อหาไฟล์ข้อความต้นฉบับของคุณโดยใช้ `StreamReader`-
```csharp
using System.IO;
// ตั้งค่าเส้นทางสำหรับไดเรกทอรีอินพุต
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

using (TextReader tr = new StreamReader(dataDir + "/log.txt"))
{
    // โค้ดยังคงดำเนินต่อไป...
}
```
**ทำไม**:การอ่านเนื้อหาไฟล์ข้อความช่วยให้เราจัดการและแปลงเป็นรูปแบบ PDF ได้

#### ขั้นตอนที่ 2: สร้างอินสแตนซ์ของวัตถุเอกสาร
สร้างว่างเปล่า `Document` วัตถุที่จะทำหน้าที่เป็นคอนเทนเนอร์เอกสาร PDF ของคุณ
```csharp
// สร้างวัตถุเอกสารใหม่
Document doc = new Document();
```
**ทำไม**ขั้นตอนนี้เป็นการเริ่มต้นโครงสร้างที่เนื้อหาข้อความจะถูกเพิ่มและบันทึกเป็นไฟล์ PDF ในที่สุด

#### ขั้นตอนที่ 3: เพิ่มหน้าใหม่
เพิ่มหน้าลงใน `Document`การรวบรวมหน้าของเอกสารแต่ละฉบับต้องมีอย่างน้อยหนึ่งหน้า
```csharp
// เพิ่มหน้าใหม่
Page page = doc.Pages.Add();
```
**ทำไม**:จำเป็นต้องมีหน้าหนึ่งหน้าสำหรับเอกสาร PDF ใดๆ โดยทำหน้าที่เป็นภาชนะสำหรับเนื้อหาข้อความของคุณ

#### ขั้นตอนที่ 4: สร้างและกำหนดค่า TextFragment
สร้าง `TextFragment` เพื่อเก็บข้อความที่อ่านไว้ ตั้งค่าคุณสมบัติแบบอักษรได้ตามต้องการ
```csharp
// สร้าง TextFragment ด้วยข้อความจากไฟล์
TextFragment text = new TextFragment(tr.ReadToEnd());

// ตัวเลือก: ตั้งค่าแบบอักษรสำหรับข้อความ
// ข้อความ.TextState.Font = FontRepository.FindFont("Arial Unicode MS");
```
**ทำไม**- `TextFragment` ใช้เพื่อรวมและจัดการเนื้อหาข้อความภายในเอกสาร PDF

#### ขั้นตอนที่ 5: เพิ่มข้อความลงในหน้า
เพิ่มสิ่งที่สร้างขึ้น `TextFragment` สู่คอลเลกชันย่อหน้าของหน้า
```csharp
// เพิ่มส่วนข้อความลงในหน้า
page.Paragraphs.Add(text);
```
**ทำไม**ขั้นตอนนี้จะวางข้อความของคุณลงบนหน้า PDF เพื่อพร้อมสำหรับการแปลงและการดู

#### ขั้นตอนที่ 6: บันทึกเอกสาร
สุดท้ายนี้ให้บันทึก `Document` วัตถุเป็นไฟล์ PDF ในไดเร็กทอรีเอาต์พุตที่คุณต้องการ
```csharp
// ตั้งค่าเส้นทางสำหรับไดเรกทอรีเอาท์พุต
string outputDir = "YOUR_OUTPUT_DIRECTORY";

doc.Save(outputDir + "/TexttoPDF_out.pdf");
```
**ทำไม**การบันทึกเอกสารจะทำให้กระบวนการแปลงเสร็จสมบูรณ์ และสร้างไฟล์ PDF จริง

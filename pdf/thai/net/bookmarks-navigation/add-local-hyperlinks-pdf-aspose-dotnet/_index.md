# เพิ่มไฮเปอร์ลิงก์ภายในไปยัง PDF โดยใช้ Aspose.PDF สำหรับ .NET: คู่มือที่ครอบคลุม

การปรับปรุงเอกสาร PDF ของคุณด้วยการเพิ่มไฮเปอร์ลิงก์ภายในนั้นมีความสำคัญสำหรับการนำทางที่ราบรื่นภายในเอกสารเดียว คู่มือนี้นำเสนอแนวทางทีละขั้นตอนโดยใช้ไลบรารี Aspose.PDF เพื่อปรับปรุงประสบการณ์ของผู้ใช้และการโต้ตอบ

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า Aspose.PDF สำหรับ .NET
- การเพิ่มไฮเปอร์ลิงก์ภายในไฟล์ PDF
- การประยุกต์ใช้งานจริงและเคล็ดลับประสิทธิภาพ

## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มไฮเปอร์ลิงก์ภายในลงในเอกสาร PDF โดยใช้ Aspose.PDF สำหรับ .NET

### การสร้างและการกำหนดค่าไฮเปอร์ลิงก์ภายใน
ไฮเปอร์ลิงก์ภายในช่วยให้ผู้ใช้สามารถนำทางภายในเอกสารเดียวกันได้อย่างราบรื่น

#### ขั้นตอนที่ 1: เพิ่มไฮเปอร์ลิงก์ท้องถิ่นแรกของคุณ
สร้างข้อความที่ทำหน้าที่เป็นไฮเปอร์ลิงก์:
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // ระบุไดเร็กทอรีของคุณ
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // ไดเรกทอรีผลลัพธ์

// สร้างอินสแตนซ์เอกสารใหม่
document doc = new Document();
Page page = doc.Pages.Add();

// ขั้นตอนที่ 1: เพิ่มไฮเปอร์ลิงก์ท้องถิ่นแรก
textFragment text1 = new TextFragment("link to page 7");
hyperlink link1 = new LocalHyperlink();
link1.TargetPageNumber = 7; // กำหนดหมายเลขหน้าเป้าหมายสำหรับการนำทาง
text1.Hyperlink = link1;
doc.Pages[page.Number].Paragraphs.Add(text1);
```

#### ขั้นตอนที่ 2: เพิ่มไฮเปอร์ลิงก์ท้องถิ่นอื่น
เพิ่มไฮเปอร์ลิงก์อื่นบนหน้าใหม่ภายในเอกสาร:
```csharp
// ขั้นตอนที่ 2: เพิ่มไฮเปอร์ลิงก์ท้องถิ่นที่สอง
textFragment text2 = new TextFragment("link to page 1");
text2.IsInNewPage = true; // ตรวจสอบให้แน่ใจว่าลิงก์นี้ปรากฏบนหน้าใหม่
hyperlink link2 = new LocalHyperlink();
link2.TargetPageNumber = 1;
text2.Hyperlink = link2;
doc.Pages.Add().Paragraphs.Add(text2);
```

### การบันทึกเอกสารของคุณ
บันทึกเอกสารของคุณด้วยไฮเปอร์ลิงก์ที่รวมอยู่ด้วย:
```csharp
string outputFilePath = Path.Combine(outputDir, "CreateLocalHyperlink_out.pdf");
doc.Save(outputFilePath);
```

## คู่มือการใช้งาน

### การเพิ่มสารบัญ
การเพิ่ม TOC เกี่ยวข้องกับการโหลด PDF ที่มีอยู่ สร้างหน้า TOC ใหม่ และเชื่อมโยงกับหัวข้อจากหน้าอื่น วิธีการมีดังนี้:

#### ขั้นตอนที่ 1: โหลดเอกสาร PDF ของคุณ
```csharp
// เริ่มต้นวัตถุเอกสาร
document doc = new Document("YOUR_DOCUMENT_DIRECTORY\AddTOC.pdf");
```
โหลดไฟล์ PDF ที่มีอยู่ของคุณลงใน Aspose.PDF `Document` วัตถุสำหรับการจัดการ

#### ขั้นตอนที่ 2: สร้างหน้า TOC
```csharp
// แทรกหน้าใหม่ในตอนต้นเพื่อใช้เป็นสารบัญ
Page tocPage = doc.Pages.Insert(1);
```
แทรกหน้าใหม่ที่จุดเริ่มต้นของ PDF เพื่อจัดเก็บ TOC ช่วยให้เข้าถึงได้ง่ายจากทุกที่ในเอกสาร

#### ขั้นตอนที่ 3: ตั้งค่าข้อมูล TOC
```csharp
// กำหนดค่าชื่อและลักษณะที่ปรากฏของ TOC
tocInfo tocInfo = new TocInfo();
TextFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20;
title.TextState.FontStyle = FontStyles.Bold;
tocInfo.Title = title;
tocPage.TocInfo = tocInfo;
```
กำหนดลักษณะและเนื้อหาของ TOC ด้วยชื่อเรื่องที่ชัดเจนและโดดเด่น

#### ขั้นตอนที่ 4: กำหนดและเพิ่มหัวข้อ
```csharp
// อาร์เรย์ของชื่อเรื่องที่จะเพิ่มเป็นองค์ประกอบ TOC
string[] titles = { "First page", "Second page", "Third page", "Fourth page" };

for (int i = 0; i < 2; i++)
{
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
    TextSegment segment2 = new TextSegment();
    heading2.TocPage = tocPage;
    heading2.Segments.Add(segment2);

    // เชื่อมโยงรายการ TOC แต่ละรายการกับหน้าที่เกี่ยวข้อง
    heading2.DestinationPage = doc.Pages[i + 2];
    heading2.Top = doc.Pages[i + 2].Rect.Height;
    segment2.Text = titles[i];

    tocPage.Paragraphs.Add(heading2);
}
```
วนซ้ำผ่านหัวข้อที่ต้องการ สร้าง `Heading` วัตถุสำหรับแต่ละรายการ ตั้งค่าหน้าปลายทางและตำแหน่งเพื่อให้แน่ใจว่านำทางได้โดยตรง

#### ขั้นตอนที่ 5: บันทึกเอกสาร
```csharp
// ส่งออกเอกสารที่แก้ไขแล้วพร้อม TOC
string outputFilePath = "YOUR_OUTPUT_DIRECTORY\TOC_out.pdf";
doc.Save(outputFilePath);
```
บันทึก PDF ที่อัปเดตของคุณพร้อม TOC ที่ผสานรวมไว้ที่จุดเริ่มต้น

1. **รายงานทางธุรกิจ**:ปรับปรุงการนำทางผ่านส่วนต่างๆ ทำให้รายงานเป็นมิตรต่อผู้ใช้มากขึ้น
2. **บทความวิชาการ**:ปรับปรุงการอ่านให้ง่ายขึ้นด้วยการเข้าถึงบทและหัวข้อย่อยได้อย่างง่ายดาย
3. **หนังสือดิจิตอล**:เพิ่มประสิทธิภาพการนำทางสำหรับผู้ใช้เครื่องอ่านอีบุ๊กหรืออ่าน PDF

## คู่มือการใช้งาน

ในส่วนนี้เราจะแนะนำการอัปเดตบุ๊กมาร์กย่อยในเอกสาร PDF โดยใช้ C#

### ขั้นตอนที่ 1: เปิดเอกสาร

เริ่มต้นด้วยการโหลดไฟล์ PDF ที่ต้องการ ตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่าเส้นทางที่ถูกต้องแล้ว:

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_Bookmarks();
Document pdfDocument = new Document(dataDir + "UpdateChildBookmarks.pdf");
```

**คำอธิบาย**: ที่นี่, `RunExamples.GetDataDir_AsposePdf_Bookmarks()` ดึงเส้นทางไดเรกทอรีของคุณ จากนั้นเราจะสร้าง `Document` วัตถุที่จะโหลด PDF

### ขั้นตอนที่ 2: เข้าถึงและอัปเดตบุ๊กมาร์กย่อย

ค้นหาและแก้ไขคุณสมบัติของบุ๊กมาร์กย่อย:

```csharp
OutlineItemCollection pdfOutline = pdfDocument.Outlines[1];
OutlineItemCollection childOutline = pdfOutline[1];

// ปรับปรุงหัวเรื่องให้เป็นตัวเอียงและตัวหนา
childOutline.Title = "Updated Outline";
childOutline.Italic = true;
childOutline.Bold = true;

dataDir = dataDir + "UpdateChildBookmarks_out.pdf"; // กำหนดเส้นทางเอาท์พุต

pdfDocument.Save(dataDir);
```

**คำอธิบาย**: เราเข้าถึงบุ๊กมาร์กผู้ปกครอง (`pdfOutline`) และลูกของมัน (`childOutline`). การอัปเดตคุณสมบัติ เช่น `Title`- `Italic`, และ `Bold` ช่วยให้สามารถปรับแต่งรูปแบบได้ สุดท้าย บันทึกการเปลี่ยนแปลงด้วย `pdfDocument-Save()`.

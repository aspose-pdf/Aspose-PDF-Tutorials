## คู่มือการใช้งาน
ตอนนี้ เราลองนำฟังก์ชันการซูมบุ๊กมาร์กไปใช้งานในแอปพลิเคชัน .NET ทีละขั้นตอนกัน

### ขั้นตอนที่ 1: โหลดเอกสาร PDF
ขั้นแรก โหลดเอกสาร PDF ที่คุณต้องการตั้งค่าบุ๊กมาร์ก:
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = RunExamples.GetDataDir_AsposePdf_Bookmarks();

// เปิดเอกสารที่มีอยู่
Document doc = new Document(dataDir + "input.pdf");
```

### ขั้นตอนที่ 2: สร้างและกำหนดค่าบุ๊กมาร์ก
ขั้นต่อไป ให้สร้างบุ๊กมาร์กด้วยระดับการซูมแบบกำหนดเองโดยใช้ `XYZExplicitDestination`-
```csharp
// เริ่มต้นรายการที่คั่นหน้าด้วยการซูมแบบกำหนดเอง
OutlineItemCollection item = new OutlineItemCollection(doc.Outlines);

// ตั้งค่าการซูมเป็น 0 (ดูหน้าเต็ม) ที่พิกัดเฉพาะ
XYZExplicitDestination dest = new XYZExplicitDestination(2, 100, 100, 0);
item.Action = new GoToAction(dest);

// เพิ่มบุ๊กมาร์กลงในคอลเล็กชันโครงร่างของเอกสาร
doc.Outlines.Add(item);
```

### ขั้นตอนที่ 3: บันทึกเอกสารที่อัปเดต
สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณกลับเป็นไฟล์ PDF:
```csharp
dataDir += "InheritZoom_out.pdf";

// บันทึก PDF ที่แก้ไขแล้วพร้อมบุ๊กมาร์กที่อัปเดต
doc.Save(dataDir);

Console.WriteLine("\nBookmarks updated successfully.\nFile saved at " + dataDir);
```

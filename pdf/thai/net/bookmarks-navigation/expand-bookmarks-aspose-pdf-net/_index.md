# วิธีการขยายบุ๊กมาร์กทั้งหมดในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET

### คู่มือการใช้งาน:
ให้เราแบ่งการใช้งานออกเป็นสองฟีเจอร์หลัก: การขยายบุ๊กมาร์กและการตั้งค่าโหมดมุมมองหน้า

#### คุณสมบัติ 1: ขยายบุ๊กมาร์กทั้งหมดใน PDF
คุณสมบัตินี้ช่วยให้คุณเปิดบุ๊กมาร์กทั้งหมดภายในเอกสาร PDF ช่วยให้การนำทางมีประสิทธิภาพมากขึ้น

**ขั้นตอนที่ 1:** โหลดเอกสาร PDF ของคุณ
เริ่มต้นด้วยการโหลดไฟล์ PDF ของคุณโดยใช้ Aspose.PDF `Document` คลาส ตรวจสอบให้แน่ใจว่าคุณระบุเส้นทางที่ถูกต้องไปยังไฟล์ PDF อินพุตของคุณ
```csharp
using System;
using Aspose.Pdf;

// โหลดเอกสาร PDF จากไดเร็กทอรีที่ระบุ
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input-bookmark.pdf");
```

**ขั้นตอนที่ 2:** ตั้งค่าโหมดการดูหน้า
ให้แน่ใจว่าบุ๊กมาร์กสามารถมองเห็นได้โดยตั้งค่าโหมดการดูหน้าเป็น `UseOutlines`-
```csharp
// เปิดใช้งานการแสดงโครงร่าง (บุ๊กมาร์ก) ในโปรแกรมดู PDF
doc.PageMode = PageMode.UseOutlines;
```

**ขั้นตอนที่ 3:** เปิดบุ๊กมาร์กทั้งหมด
ทำซ้ำผ่านบุ๊กมาร์กแต่ละรายการและเปิดมัน แก้ไข `OutlineItemCollection` วัตถุดังต่อไปนี้:
```csharp
// วนซ้ำผ่านบุ๊กมาร์กทั้งหมดและตั้งค่าให้เปิด
foreach (OutlineItemCollection item in doc.Outlines)
{
    // ทำเครื่องหมายบุ๊กมาร์กปัจจุบันเป็นเปิด
    item.Open = true;
}
```

**ขั้นตอนที่ 4:** บันทึกเอกสารที่แก้ไข
บันทึกเอกสารของคุณด้วยการขยายบุ๊กมาร์กไปยังไฟล์ใหม่
```csharp
// บันทึกไฟล์ PDF พร้อมบุ๊กมาร์กขยายไปยังไดเร็กทอรีเอาท์พุต
doc.Save("YOUR_OUTPUT_DIRECTORY/ExpandBookmarks_out.pdf");
```

#### คุณสมบัติ 2: ตั้งค่าโหมดการดูหน้า
ปรับโหมดการดูหน้าของ PDF เพื่อแสดงบุ๊กมาร์กตามค่าเริ่มต้น

**ขั้นตอนที่ 1:** โหลดเอกสาร PDF ของคุณ
เริ่มต้นด้วยการโหลดเอกสารเหมือนเช่นเคย ขั้นตอนนี้จะคล้ายกับที่เราทำในฟีเจอร์ก่อนหน้านี้
```csharp
using Aspose.Pdf;

// เปิดเอกสาร PDF จากไดเร็กทอรีที่คุณระบุ
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input-bookmark.pdf");
```

**ขั้นตอนที่ 2:** กำหนดค่าโหมดการดูหน้า
ตั้งค่าโหมดหน้าเป็น `UseOutlines` อีกครั้งหนึ่ง ตรวจสอบให้แน่ใจว่ามีการแสดงบุ๊กมาร์กเมื่อเปิดไฟล์
```csharp
// ตั้งค่าโหมดการแสดงหน้าให้แสดงโครงร่าง (บุ๊กมาร์ก)
doc.PageMode = PageMode.UseOutlines;
```

**ขั้นตอนที่ 3:** บันทึกการเปลี่ยนแปลงของคุณ
บันทึก PDF ของคุณด้วยการตั้งค่าหน้าที่อัปเดต
```csharp
// บันทึกเอกสารที่แก้ไข
doc.Save("YOUR_OUTPUT_DIRECTORY/BookmarksPageView_out.pdf");
```

1. **การทำงานร่วมกัน**:แบ่งปันเอกสารด้วยส่วนที่สามารถเข้าถึงได้ผ่านบุ๊กมาร์กแบบขยาย ช่วยเพิ่มประสิทธิภาพของทีม
2. **เอกสารประกอบการอบรม**:สร้าง PDF ที่ง่ายต่อการนำทางสำหรับเนื้อหาการศึกษา ช่วยให้ผู้เรียนสามารถข้ามหัวข้อต่างๆ ได้อย่างง่ายดาย
3. **รายงานและการนำเสนอ**:ปรับปรุงรายงานขององค์กรด้วยองค์ประกอบแบบโต้ตอบที่ช่วยแนะนำผู้อ่านให้ไปตามส่วนต่างๆ

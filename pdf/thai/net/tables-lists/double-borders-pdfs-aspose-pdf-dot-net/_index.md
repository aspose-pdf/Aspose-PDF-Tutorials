## คู่มือการใช้งาน

ให้เราแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การสร้างตารางพื้นฐานและการกำหนดเส้นขอบคู่บนเซลล์

### คุณลักษณะที่ 1: การสร้างตารางพื้นฐานใน Aspose.PDF

#### ภาพรวม
การสร้างตาราง PDF เกี่ยวข้องกับการเพิ่มแถว คอลัมน์ และเนื้อหา หัวข้อนี้จะแสดงวิธีการตั้งค่าตารางง่ายๆ ด้วยเซลล์ข้อความหนึ่งเซลล์

#### การดำเนินการแบบทีละขั้นตอน
**สร้างเอกสารและเพิ่มหน้า**
```csharp
// สร้างอินสแตนซ์ของวัตถุเอกสาร
Document document = new Document();
// เพิ่มหน้าลงในเอกสาร PDF
Page page = document.Pages.Add();
```

**ตั้งค่าโครงสร้างตาราง**
```csharp
// สร้างอินสแตนซ์วัตถุตารางด้วยความกว้างคอลัมน์ที่ระบุ
Aspose.Pdf.Table table = new Aspose.Pdf.Table();
table.ColumnWidths = "100";  // กำหนดความกว้างของคอลัมน์

// เพิ่มแถวและเซลล์ลงในตาราง
Row row = table.Rows.Add();
Cell cell = row.Cells.Add("some text");
```

**เพิ่มตารางลงในเอกสาร**
```csharp
// ผนวกตารางเข้าในคอลเล็กชั่นย่อหน้าของหน้า
page.Paragraphs.Add(table);
```

### คุณสมบัติ 2: ตั้งค่าเส้นขอบคู่สำหรับเซลล์

#### ภาพรวม
คุณลักษณะนี้มุ่งเน้นไปที่การปรับแต่งเส้นขอบเซลล์ในตาราง PDF ของคุณโดยเฉพาะการตั้งค่าเส้นคู่ที่ด้านทั้งหมดของเซลล์

#### การดำเนินการแบบทีละขั้นตอน
**สร้างและปรับแต่งขอบ**
```csharp
// สร้างวัตถุ BorderInfo ใหม่โดยมีขอบสองด้านทุกด้าน
Aspose.Pdf.BorderInfo border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All);
border.Top.IsDoubled = true;
border.Bottom.IsDoubled = true;

// กำหนดเส้นขอบแบบกำหนดเองให้กับเซลล์
cell.Border = border;
```

**บันทึกเอกสาร**
```csharp
// กำหนดไดเรกทอรีเอาท์พุตและเส้นทางไฟล์
string dataDir = "YOUR_OUTPUT_DIRECTORY/TableBorderTest_out.pdf";

// บันทึกเอกสาร PDF
document.Save(dataDir);
```

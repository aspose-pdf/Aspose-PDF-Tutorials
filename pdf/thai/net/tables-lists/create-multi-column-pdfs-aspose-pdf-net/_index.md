## คู่มือการใช้งาน
ตอนนี้เราได้ตั้งค่าทุกอย่างเรียบร้อยแล้ว เรามาเริ่มสร้างเอกสาร PDF หลายคอลัมน์กันเลย เราจะแบ่งกระบวนการนี้ออกเป็นขั้นตอนที่จัดการได้

### 1. เริ่มต้นเอกสาร
เริ่มต้นด้วยการสร้างใหม่ `Document` วัตถุ:
```csharp
Document doc = new Document();
```
วัตถุนี้แสดงไฟล์ PDF ทั้งหมดของคุณ

### 2. กำหนดค่าระยะขอบหน้า
ตั้งค่าระยะขอบหน้าเพื่อให้แน่ใจว่าเนื้อหาอยู่ในตำแหน่งที่เหมาะสมในแต่ละหน้า:
```csharp
doc.PageInfo.Margin.Left = 40;
doc.PageInfo.Margin.Right = 40;
Page page = doc.Pages.Add();
```
ค่าเหล่านี้จะกำหนดช่องว่างระหว่างเนื้อหาเอกสารของคุณและขอบ

### 3. เพิ่มเส้นแนวนอน
หากต้องการแยกส่วนต่างๆ อย่างชัดเจน ให้เพิ่มเส้นแนวนอนขวางความกว้างของหน้า:
```csharp
Graph graph1 = new Graph(500.0, 2.0);
page.Paragraphs.Add(graph1);

float[] posArr = { 1, 2, 500, 2 };
Line l1 = new Line(posArr);
graph1.Shapes.Add(l1);
```
นี่จะสร้างเส้นโดยใช้ `Graph` และ `Line` ชั้นเรียน

### 4. แทรกหัวข้อด้วยการจัดรูปแบบ HTML
สำหรับข้อความที่มีลักษณะเฉพาะ เช่น หัวเรื่อง ให้ใช้ส่วน HTML:
```csharp
string headingHtml = "<font face=\"Times New Roman\" size=4><strong> How to Steer Clear of Money Scams</strong></font>";
HtmlFragment headingText = new HtmlFragment(headingHtml);
page.Paragraphs.Add(headingText);
```
สิ่งนี้ช่วยให้คุณใช้แท็ก HTML เพื่อจัดรูปแบบข้อความภายใน PDF ของคุณได้

### 5. สร้าง FloatingBox สำหรับเค้าโครงหลายคอลัมน์
การ `FloatingBox` คลาสนี้ใช้เพื่อจัดเรียงเนื้อหาลงในคอลัมน์:
```csharp
FloatingBox box = new FloatingBox();
box.ColumnInfo.ColumnCount = 2; // สองคอลัมน์
box.ColumnInfo.ColumnSpacing = "5";
box.ColumnInfo.ColumnWidths = "105 105";

TextFragment authorText = new TextFragment("By A Googler (The Official Google Blog)");
authorText.TextState.FontSize = 8;
authorText.TextState.FontStyle = FontStyles.Italic;
box.Paragraphs.Add(authorText);
```
สไนปเป็ตนี้จะสร้างเค้าโครงสองคอลัมน์โดยมีระยะห่างและความกว้างที่ระบุไว้

### 6. เพิ่มเนื้อหาเพิ่มเติม
ดำเนินการเพิ่มเนื้อหา เช่น เส้นแนวนอน หรือบล็อกข้อความต่อไป:
```csharp
Graph graph2 = new Graph(50.0, 10.0);
float[] posArr2 = { 1, 10, 100, 10 };
Line l2 = new Line(posArr2);
graph2.Shapes.Add(l2);
box.Paragraphs.Add(graph2);

TextFragment sampleText = new TextFragment("Your content here...");
box.Paragraphs.Add(sampleText);
```
ภาพนี้แสดงให้เห็นการเพิ่มองค์ประกอบประเภทต่างๆ ภายในคอลัมน์ของคุณ

### 7. เสร็จสิ้นและบันทึกเอกสาร
สุดท้ายเพิ่ม `FloatingBox` ไปที่หน้าและบันทึกเอกสารของคุณ:
```csharp
page.Paragraphs.Add(box);

string outputDir = "YOUR_OUTPUT_DIRECTORY/CreateMultiColumnPdf_out.pdf";
doc.Save(outputDir);
```
การดำเนินการนี้จะบันทึกไฟล์ PDF หลายคอลัมน์ของคุณไปยังไดเร็กทอรีที่ระบุ

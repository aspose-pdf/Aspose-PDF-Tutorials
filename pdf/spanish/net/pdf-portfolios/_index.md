# Tutoriales de paquetes y portafolios PDF para Aspose.PDF .NET

Domine la creación de portafolios PDF con nuestros detallados tutoriales de Aspose.PDF para desarrolladores .NET. Estas completas guías muestran cómo crear portafolios PDF con múltiples archivos, personalizar el diseño y la navegación de los portafolios, extraer y manipular documentos incrustados e implementar funciones avanzadas de portafolios mediante programación. Cada tutorial proporciona ejemplos prácticos de código C# para crear paquetes de documentos sofisticados que combinan múltiples archivos en un único contenedor PDF navegable, ideal para distribuir documentos relacionados como una unidad cohesiva.

## Tutoriales disponibles

### [Cómo crear un portafolio PDF con Aspose.PDF para .NET: una guía completa](./create-pdf-portfolio-aspose-net/)
Aprenda a crear un portafolio PDF con Aspose.PDF para .NET. Esta guía abarca la configuración, la adición de archivos como Excel y Word, y las prácticas recomendadas para una gestión eficiente de documentos.

### [Dominando la gestión de portafolios PDF en .NET con Aspose.PDF](./mastering-pdf-portfolio-management-net-asposepdf/)
Aprenda a gestionar y optimizar carteras PDF de forma eficiente con Aspose.PDF para .NET. Agilice la gestión de documentos sin esfuerzo.

# Crear y modificar un PDF con imágenes usando Aspose.PDF

Aspose.PDF para .NET es una biblioteca poderosa que permite a los desarrolladores crear, editar y manipular documentos PDF de forma programática. En este ejemplo, aprenderá a generar un documento PDF, agregar una imagen, ajustar su tamaño y modificar las propiedades de la página usando C#. El código demuestra cómo iniciar un documento PDF, insertar una imagen, establecer dimensiones de página y guardar el archivo final. Ya sea que esté trabajando con informes, facturas o contenido dinámico, este tutorial proporciona una manera simple pero efectiva de administrar imágenes dentro de PDFs usando Aspose.PDF para .NET.

## Instrucciones de instalación:  

Para usar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar a través del NuGet Package Manager:  
Abra Visual Studio y ejecute el siguiente comando en la consola del administrador de paquetes:  
```powershell
Install-Package Aspose.PDF
```

2. Añadir espacio de nombres en su código C #:  
```csharp
using Aspose.Pdf;
```

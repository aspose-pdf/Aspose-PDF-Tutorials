# Crea y modifica un PDF con imágenes usando Aspose.PDF

Aspose.PDF para .NET es una biblioteca poderosa que permite a los desarrolladores crear, editar y manipular documentos PDF de forma programática. En este ejemplo, aprenderá a generar un documento PDF, agregar una imagen, ajustar su tamaño y modificar las propiedades de la página utilizando C#. El código demuestra la inicialización de un documento PDF, la inserción de una imagen, el ajuste de dimensiones de la página y el almacenamiento del archivo final. Ya sea que esté trabajando con informes, facturas o contenido dinámico, este tutorial proporciona una forma simple pero efectiva de administrar imágenes dentro de los PDFs utilizando Aspose.PDF para .NET.

## Instrucciones de instalación:  

Para usar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar a través del administrador de paquetes NuGet:  
Abra Visual Studio y ejecuta el siguiente comando en la consola de gestión de paquetes:  
```powershell
Install-Package Aspose.PDF
```

2. Añadir espacio de nombres en su código C#:  
```csharp
using Aspose.Pdf;
```

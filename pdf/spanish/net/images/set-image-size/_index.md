# Crear y modificar un PDF con imágenes usando Aspose.PDF

Aspose.PDF para .NET es una potente biblioteca que permite a los desarrolladores crear, editar y manipular documentos PDF mediante programación. En este ejemplo, aprenderá a generar un documento PDF, añadir una imagen, ajustar su tamaño y modificar las propiedades de página con C#. El código muestra cómo inicializar un documento PDF, insertar una imagen, configurar las dimensiones de la página y guardar el archivo final. Tanto si trabaja con informes, facturas o contenido dinámico, este tutorial le ofrece una forma sencilla y eficaz de gestionar imágenes en archivos PDF con Aspose.PDF para .NET.

## Instrucciones de instalación:  

Para utilizar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar a través del Administrador de paquetes NuGet:  
Abra Visual Studio y ejecute el siguiente comando en la Consola del Administrador de paquetes:  
```powershell
Install-Package Aspose.PDF
```

2. Agregue un espacio de nombres en su código C#:  
```csharp
using Aspose.Pdf;
```

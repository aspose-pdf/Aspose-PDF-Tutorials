# Optimizar imágenes PDF con Aspose.PDF

Este ejemplo demuestra cómo optimizar un documento PDF comprimiendo y redimensionando las imágenes que contiene con Aspose.PDF para .NET. Siguiendo el código proporcionado, puede reducir eficazmente el tamaño de sus documentos PDF manteniendo una calidad de imagen aceptable. Esto es especialmente útil para mejorar el rendimiento y reducir los costos de almacenamiento.

## Instrucciones de instalación
1. Descargar Aspose.PDF para .NET: Visita el sitio [Página de descarga de Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/) para obtener la última versión.
2. Instalar mediante NuGet: abra su proyecto en Visual Studio y use el siguiente comando en la Consola del Administrador de paquetes:
   ```
   Install-Package Aspose.PDF
   ```

## Para más detalles
documentación, visite la [Referencia de la API de Aspose.PDF para .NET](https://reference.aspose.com/pdf/net/).

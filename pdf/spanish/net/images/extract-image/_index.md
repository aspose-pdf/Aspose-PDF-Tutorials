# Extraer imágenes de PDF en C# con Aspose.PDF
 
Extraer imágenes de un documento PDF es un requisito común para muchas aplicaciones, ya sea para archivar, procesar o compartir.[Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/), puede extraer imágenes de un PDF y guardarlas fácilmente en varios formatos como JPEG. Este tutorial proporciona una guía simple y paso a paso para extraer imágenes de un PDF usando C#. Aprenderá cómo abrir un PDF, recuperar imágenes, guardarlas y actualizar el documento PDF original. Siga el código de ejemplo para integrar sin problemas esta funcionalidad en sus aplicaciones .NET.

## Instrucciones de instalación:  

Para usar Aspose.PDF para .NET en su proyecto C#, siga estos pasos:  

1. Instalar a través del NuGet Package Manager:  
* Abre el estudio visual.  
* Ir a Herramientas > NuGet Package Manager > Administrar paquetes NuGet para la solución.  
* Busca Aspose.PDF y instálalo.  
* O ejecute el siguiente comando en la consola del administrador de paquetes:  

  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalar a través de DLL Descargar:  
* Descargue la última versión de [Aspose.PDF para descargar .NET](https://releases.aspose.com/pdf/net/)\- ¿ Qué ?  
* Extraer el paquete descargado.  
* Añadir una referencia a Aspose.PDF.dll en su proyecto .NET.  

3. Aplicar una licencia (opcional):  
* Puedes conseguir un ..[Licencia temporal gratuita](https://purchase.aspose.com/temporary-license/)para una funcionalidad completa.

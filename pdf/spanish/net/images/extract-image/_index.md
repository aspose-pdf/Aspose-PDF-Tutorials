# Extraer imágenes de PDF en C# usando Aspose.PDF
 
Extraer imágenes de un documento PDF es un requisito común para muchas aplicaciones, ya sea para archivar, procesar o compartir.[Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/), puede extraer fácilmente imágenes de un PDF y guardarlas en varios formatos como JPEG. Este tutorial proporciona una guía sencilla, paso a paso para extraer imágenes de un PDF usando C#. Aprenderá cómo abrir un PDF, recuperar imágenes, guardarlas y actualizar el documento PDF original. Siga junto con el código de ejemplo para integrar sin problemas esta funcionalidad en sus aplicaciones .NET.

## Instrucciones de instalación:  

Para usar Aspose.PDF para .NET en su proyecto C#, siga estos pasos:  

1. Instalar a través del administrador de paquetes NuGet:  
* Abre el estudio visual.  
* Ve a Herramientas > Gerente de paquetes NuGet > Gerenciamiento de paquetes NuGet para Solución.  
* Busque Aspose.PDF y instale.  
* O ejecuta el siguiente comando en la consola de gestión de paquetes:  

  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalar a través de descarga de DLL:  
* Descarga la última versión de [Aspose.PDF para .NET Descargar](https://releases.aspose.com/pdf/net/)¿ Qué ?  
* Extraer el paquete descargado.  
* Añadir una referencia a Aspose.PDF.dll en su proyecto .NET.  

3. Aplicar una licencia (opcional):  
* Puedes conseguir una .[licencia temporal gratuita](https://purchase.aspose.com/temporary-license/)para una funcionalidad completa.

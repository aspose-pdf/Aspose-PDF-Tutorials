# Añadir texto subrayado en PDF con Aspose.PDF para .NET

Aspose.PDF para .NET es una biblioteca de procesamiento de PDF que permite a los desarrolladores crear, modificar y manipular documentos PDF sin necesidad de Adobe Acrobat. Este ejemplo demuestra cómo crear un nuevo documento PDF y agregar texto con formato subrayado utilizando las clases TextBuilder y TextFragment. El código crea un PDF simple con un mensaje de prueba formateado en fuente Arial con estilo subrayado, posicionándolo en coordenadas específicas dentro de la página.

## Instrucciones de instalación

1: Instale Aspose.PDF para el .NET

Puede instalar Aspose.PDF para .NET usando uno de los siguientes métodos:

Utilizando el administrador de paquetes NuGet:

```csharp
Install-Package Aspose.PDF
```

Usando la interfaz de usuario .NET:

```bash
dotnet add package Aspose.PDF
```

Usando PackageReference en el archivo del proyecto:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Añadir declaraciones de uso requeridas

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

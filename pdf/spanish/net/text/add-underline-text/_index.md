# Cómo añadir texto subrayado en PDF con Aspose.PDF para .NET

Aspose.PDF para .NET es una potente biblioteca de procesamiento de PDF que permite a los desarrolladores crear, modificar y manipular documentos PDF sin necesidad de Adobe Acrobat. Este ejemplo muestra cómo crear un nuevo documento PDF y añadir texto subrayado mediante las clases TextBuilder y TextFragment. El código crea un PDF simple con un mensaje de prueba en fuente Arial y subrayado, posicionándolo en coordenadas específicas dentro de la página.

## Instrucciones de instalación

1: Instalar Aspose.PDF para .NET

Puede instalar Aspose.PDF para .NET utilizando uno de los siguientes métodos:

Uso del administrador de paquetes NuGet:

```csharp
Install-Package Aspose.PDF
```

Usando la CLI .NET:

```bash
dotnet add package Aspose.PDF
```

Usando PackageReference en su archivo de proyecto:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Agregar instrucciones Using requeridas

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

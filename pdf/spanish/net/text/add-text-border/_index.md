# Cómo crear texto con estilo y bordes en documentos PDF

Aspose.PDF para .NET es una potente API de procesamiento de PDF que permite a los desarrolladores crear, editar, manipular y convertir documentos PDF sin necesidad de Adobe Acrobat. Este ejemplo muestra cómo crear un documento PDF con un fragmento de texto con un estilo específico y un borde, lo cual resulta especialmente útil para resaltar contenido importante en los documentos.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.PDF
   ```

2. Instalar mediante la CLI de .NET:
   ```
   dotnet add package Aspose.PDF
   ```

3. Instalación manual:
   - Descargue la última versión desde [Aspose.PDF para versiones .NET](https://releases.aspose.com/pdf/net/)
   - Agregue una referencia a Aspose.PDF.dll en su proyecto

## Recursos

- [Documentación de Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/)

# Crear texto con bordes en documentos PDF

Aspose.PDF para .NET es una potente API de procesamiento de PDF que permite a los desarrolladores crear, editar, manipular y convertir documentos PDF sin necesidad de Adobe Acrobat. Este ejemplo demuestra cómo crear un documento PDF con un fragmento de texto que tiene un estilo específico y un borde, que es particularmente útil para resaltar el contenido importante en sus documentos.

## Instrucciones de instalación

1. Instalar a través del NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Instalar a través de .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Instalación manual:
   * Descargue la última versión de [Aspose.PDF para las versiones .NET](https://releases.aspose.com/pdf/net/)
   * Añadir referencia a Aspose.PDF.dll en su proyecto

## Instrucciones de instalación
Para comenzar a utilizar Aspose.PDF para .NET, siga estas instrucciones de instalación:

1. Descargar Aspose.PDF: 
   - Visita el [Sitio web de Aspose](https://www.aspose.com/downloads/pdf/net) para descargar la última versión de Aspose.PDF para .NET.

2. Instalar a través del Administrador de paquetes NuGet:
   - Abra su proyecto en Visual Studio.
   - Vaya a Herramientas > Administrador de paquetes NuGet > Administrar paquetes NuGet para la solución.
   - Busque "Aspose.PDF" e instale el paquete.

3. Añadir referencias:
   - Asegúrese de que su proyecto haga referencia a la DLL Aspose.PDF en el `packages` carpeta.
   
4. Incluya el espacio de nombres Aspose.PDF en su proyecto:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

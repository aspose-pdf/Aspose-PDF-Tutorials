## Instalación  

Para utilizar Aspose.PDF para .NET, siga estos pasos:  

1. Descargar e instalar Aspose.PDF para .NET  
   - Descargar desde [Descargas de Aspose.PDF para .NET](https://releases.aspose.com/pdf/net/)  
   - Instalar mediante NuGet:  
     ```sh
     Install-Package Aspose.PDF
     ```  

2. Solicitar licencia (opcional)  
   - Compra: [Comprar Aspose.PDF](https://purchase.aspose.com/buy)  
   - Prueba gratuita: [Prueba gratuita de Aspose.PDF](https://releases.aspose.com/)  
   - Licencia temporal: [Licencia temporal de Aspose.PDF](https://purchase.aspose.com/temporary-license/)  

3. Consulte la documentación  
   - [Documentación de Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/)

## Instrucciones de instalación  
Para usar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar a través de NuGet  
Ejecute el siguiente comando en la consola de gestión de paquetes NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Añadir espacio de nombres en C#  
Incluya los espacios de nombres requeridos al principio de su archivo C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Para obtener documentación detallada, visite:  
[Aspose.PDF para Documentación .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF para la referencia de API .NET](https://reference.aspose.com/pdf/net/)

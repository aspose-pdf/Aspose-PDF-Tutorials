## Instrucciones de instalación  
Para usar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar con NuGet  
Ejecute el siguiente comando en la consola de NuGet Package Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Añadir espacio de nombres en C#  
Incluye los espacios de nombres requeridos al principio de tu archivo C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Para obtener documentación detallada, visite:  
[Aspose.PDF para documentación .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF para la referencia de la API .NET](https://reference.aspose.com/pdf/net/)

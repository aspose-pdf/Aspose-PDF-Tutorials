# Cree un PDF con rectángulos y colores alfa en Aspose.PDF

Aspose.PDF para .NET es una potente biblioteca para crear, manipular y convertir documentos PDF en C#. En este ejemplo, aprenderá a crear un archivo PDF y a dibujar rectángulos con colores alfa. Usamos... `Aspose.Pdf.Drawing.Graph` Clase para agregar formas y aplicar efectos de transparencia con colores ARGB. Este tutorial proporciona una guía paso a paso con código de ejemplo, lo que le permitirá implementarla fácilmente en sus aplicaciones .NET.

## Instrucciones de instalación  
Para utilizar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar mediante NuGet  
Ejecute el siguiente comando en la consola del Administrador de paquetes NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Agregar espacio de nombres en C#  
Incluya los espacios de nombres requeridos al comienzo de su archivo C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Para obtener documentación detallada, visite:  
[Documentación de Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/)  
[Referencia de la API de Aspose.PDF para .NET](https://reference.aspose.com/pdf/net/)

# Crea y personaliza PDF con rectángulos

Esta guía muestra cómo crear un documento PDF con Aspose.PDF para .NET, incorporando rectángulos de colores con diferentes índices Z para controlar su superposición. Siga las instrucciones paso a paso para añadir formas a su PDF y guardarlo, optimizando así la capacidad de generación de PDF de su aplicación.

## Instrucciones de instalación
1. Asegúrese de tener configurado el entorno de desarrollo .NET (como Visual Studio).
2. Descargue la biblioteca Aspose.PDF desde [Sitio web de Aspose](https://www.aspose.com/purchase).
3. Instale el paquete NuGet Aspose.PDF ejecutando `Install-Package Aspose.PDF` en la consola del administrador de paquetes.
4. Incluya los espacios de nombres necesarios en su proyecto:
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

## Para obtener documentación detallada, visite:  
- [Documentación de Aspose.PDF para .NET](https://docs.aspose.com/pdf/net/)

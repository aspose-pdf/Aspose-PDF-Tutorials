# Dibujar líneas diagonales en PDF con Aspose.PDF para .NET

Aspose.PDF para .NET permite a los desarrolladores crear, manipular y editar documentos PDF fácilmente. En este ejemplo, mostramos cómo dibujar programáticamente dos líneas diagonales en una página PDF con C#. Aprenderá a inicializar un documento PDF, configurar las propiedades de la página, añadir elementos gráficos y guardar el resultado final. Esta guía ofrece un enfoque paso a paso que facilita la integración de esta funcionalidad en sus aplicaciones .NET.

## Instrucciones de instalación:  

Para utilizar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar a través del Administrador de paquetes NuGet  
- Abra Visual Studio y cree un nuevo proyecto .NET.  
- Abra la Consola del administrador de paquetes (Herramientas → Administrador de paquetes NuGet → Consola del administrador de paquetes) y ejecute:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalar a través de la CLI de .NET  
Alternativamente, utilice la CLI de .NET para instalar el paquete:  
```bash
dotnet add package Aspose.PDF
```

3. Agregar espacio de nombres en su código C#  
Para utilizar la funcionalidad de Aspose.PDF, incluya los siguientes espacios de nombres:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

# Dibujar líneas diagonales en PDF usando Aspose.PDF para .NET

Aspose.PDF para .NET permite a los desarrolladores crear, manipular y editar documentos PDF con facilidad. En este ejemplo, demostramos cómo dibujar programáticamente dos líneas diagonales a través de una página PDF usando C#. Aprenderá a inicializar un documento PDF, establecer propiedades de la página, agregar elementos gráficos y guardar la salida final. Esta guía proporciona un enfoque paso a paso, lo que facilita la integración de esta funcionalidad en sus aplicaciones .NET.

## Instrucciones de instalación:  

Para usar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar a través del administrador de paquetes NuGet  
* Abre Visual Studio y crea un nuevo proyecto .NET.  
* Abre la consola de gestión de paquetes (herramientas → NuGet Package Manager → Console de gestión de paquetes) y ejecuta:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalar a través de .NET CLI  
Alternativamente, utilice el CLI .NET para instalar el paquete:  
```bash
dotnet add package Aspose.PDF
```

3. Añadir espacio de nombres en su código C#  
Para utilizar la función Aspose.PDF, incluya los siguientes espacios de nombres:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

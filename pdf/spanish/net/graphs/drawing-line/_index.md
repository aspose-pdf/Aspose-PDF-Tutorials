## Instrucciones de instalación:  

Para usar Aspose.PDF para .NET en su proyecto, siga estos pasos:  

1. Instalar a través del NuGet Package Manager  
* Abra Visual Studio y cree un nuevo proyecto .NET.  
* Abrir la consola del administrador de paquetes (Herramientas → NuGet Package Manager → Package Manager Console) y ejecutar:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Instalar a través de la interfaz de usuario .NET  
Alternativamente, utilice la interfaz de usuario .NET para instalar el paquete:  
```bash
dotnet add package Aspose.PDF
```

3. Añadir espacios de nombres en su código C#  
Para utilizar la funcionalidad Aspose.PDF, incluye los siguientes espacios de nombres:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

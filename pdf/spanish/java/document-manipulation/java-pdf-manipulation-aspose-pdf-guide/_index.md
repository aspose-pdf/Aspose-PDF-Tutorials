Reemplazar `YOUR_DOCUMENT_DIRECTORY` con la ruta de su carpeta actual.\n\n#### Paso 2: Cargue el PDF\n\nCree un `Document` objeto y cargue su archivo PDF:\n\n```java\ndocument = new Document(dataDir + \"/Field_Added_output.pdf\");\n```\n\n### Cómo acceder a un campo específico en un formulario PDF\n\n**Descripción general:** Acceda a campos específicos dentro de un formulario para extraer o manipular datos.\n\n#### Paso 1: recuperar el campo por nombre\n\nUse el `getForm()` método para acceder al formulario y recuperar el campo:\n\n```java\nimport com.aspose.pdf.TextBoxField;\n\nTextBoxField textBoxField = (TextBoxField) pdfDocument.getForm().get(\"textbox1\");\n```\n\nAquí, `"textbox1"` es el nombre del campo al que desea acceder.\n\n### Recuperar información de campo de un documento PDF\n\n**Descripción general:** Extraiga información detallada, como nombres parciales y valores de los campos.\n\n#### Paso 2: Obtener detalles del campo\n\nRecupere el nombre parcial y el valor del campo especificado:\n\n```java\nString fieldName = textBoxField.getPartialName();\nString fieldValue = textBoxField.getValue();\n\nSystem.out.println(\"Field Name: \" + fieldName);\nSystem.out.println(\"Field Value: \" + fieldValue);\n```\n\n## Aplicaciones prácticas\n\nCon Aspose.PDF, puede integrar

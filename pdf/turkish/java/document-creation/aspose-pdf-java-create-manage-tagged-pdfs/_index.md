PDF'yi kaydedin**\N```java\ndocument.save(dataDir + \"/TaggedPdfDocument.pdf\");\n```\n\n### Özellik 3: Etiketli PDF'ye Metin Yapısı Öğeleri Ekleme\n\n**Genel Bakış:** Daha iyi okunabilirlik ve

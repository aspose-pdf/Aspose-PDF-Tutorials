\"YOUR_OUTPUT_DIRECTORY\";\npdfDocument.save(outputDir + \"/output.pdf\");\n```\n*Açıklama:* The `save` yöntem, işlem sırasında yapılan tüm güncellemeleri koruyarak değiştirilen PDF'yi depolamaya geri yazar.\n\n### Sorun Giderme İpuçları\n- **Dizin Sınır Dışı:** Silmeye çalıştığınız dizinin açıklamalar listesinde mevcut olduğundan emin olun.\n- **Dosya Bulunamadı:** Dosya yolunuzu ve dizin

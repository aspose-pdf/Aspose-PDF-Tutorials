## Kurulum talimatları  

Projenizdeki .NET için Aspose.PDF kullanmak için, şu adımları izleyin:  

1. NuGet ile yükle (Önerilen)  
NuGet Paket Yöneticisi Konsolunda aşağıdaki komutu çalıştır:  
```powershell
Install-Package Aspose.PDF
```

2. .NET CLI ile yükle  
.NET Core veya .NET 5+ için kullanın:  
```bash
dotnet add package Aspose.PDF
```

3. Kurulumdan sonra, projenizdeki gerekli isim alanlarını içeri aktarın:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

# Aspose.PDF for .NET ile PDF'ye Altı Çizili Metin Ekleme

Aspose.PDF for .NET, geliştiricilerin Adobe Acrobat'a ihtiyaç duymadan PDF belgeleri oluşturmasına, değiştirmesine ve düzenlemesine olanak tanıyan güçlü bir PDF işleme kütüphanesidir. Bu örnek, TextBuilder ve TextFragment sınıflarını kullanarak yeni bir PDF belgesi oluşturmayı ve alt çizgi biçimlendirmeli metin eklemeyi gösterir. Kod, Arial fontuyla biçimlendirilmiş ve alt çizgi stiliyle sayfadaki belirli koordinatlara yerleştirilmiş bir test mesajı içeren basit bir PDF oluşturur.

## Kurulum Talimatları

1: .NET için Aspose.PDF'yi yükleyin

Aspose.PDF for .NET'i aşağıdaki yöntemlerden birini kullanarak yükleyebilirsiniz:

NuGet Paket Yöneticisini Kullanma:

```csharp
Install-Package Aspose.PDF
```

.NET CLI kullanımı:

```bash
dotnet add package Aspose.PDF
```

Proje dosyanızda PackageReference'ı kullanma:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Gerekli Using Deyimlerini Ekleyin

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

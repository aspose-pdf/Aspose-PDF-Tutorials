# Aspose.PDF ile PDF'de Altyazılı Metin Ekle .NET için

Aspose.PDF .NET için geliştiricilerin Adobe Acrobat gerekmeden PDF belgelerini oluşturma, değiştirme ve manipüle etme imkanı veren güçlü bir PDF işleme kütüphanesidir. Bu örnek, yeni bir PDF belgesi oluşturmanın ve TextBuilder ve TextFragment sınıflarını kullanarak alt çizgi biçimlendirme ile metin eklemenin nasıl yapıldığını göstermektedir. Kod, sayfa içindeki belirli koordinatlarda konumlandırılarak, alt çizgi biçimlendirme ile Arial yazı tipinde biçimlendirilmiş bir test mesajı ile basit bir PDF oluşturur.

## Kurulum talimatları

1: .NET için Aspose.PDF yükleyin

Aşağıdaki yöntemlerden birini kullanarak .NET için Aspose.PDF yükleyebilirsiniz:

NuGet Paket Yöneticisi kullan:

```csharp
Install-Package Aspose.PDF
```

.NET CLI kullan:

```bash
dotnet add package Aspose.PDF
```

Proje dosyanızda PackageReference kullanmak:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Gerekli Kullanım İfade Ekle

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

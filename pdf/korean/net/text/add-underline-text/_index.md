# Aspose.PDF로 PDF에 하향된 텍스트를 추가합니다.

Aspose.PDF는 개발자가 Adobe Acrobat을 필요로 하지 않고 PDF 문서를 생성, 수정 및 조작할 수 있는 강력한 PDF 처리 라이브러리이다. 이 예제는 새로운 PDF 문서를 생성하고 텍스트 빌더와 텍스트 프래그먼트 클래스를 사용하여 하부 포맷 된 텍스트를 추가하는 방법을 보여줍니다. 코드는 하부 스타일링으로 아리얼 글꼴로 포맷 된 테스트 메시지로 간단한 PDF를 생성하고 페이지 내 특정 좌표에 배치합니다.

## 설치 지침

1: .NET에 대한 Aspose.PDF를 설치

다음 방법 중 하나를 사용하여 .NET에 Aspose.PDF를 설치할 수 있습니다:

NuGet 패키지 관리자를 사용함:

```csharp
Install-Package Aspose.PDF
```

.NET CLI를 사용함:

```bash
dotnet add package Aspose.PDF
```

프로젝트 파일에서 PackageReference를 사용함:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: 필요한 사용 문장을 추가

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

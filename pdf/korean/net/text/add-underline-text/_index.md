# Aspose.PDF for .NET을 사용하여 PDF에 밑줄이 그어진 텍스트 추가

Aspose.PDF for .NET은 개발자가 Adobe Acrobat 없이도 PDF 문서를 생성, 수정 및 조작할 수 있도록 지원하는 강력한 PDF 처리 라이브러리입니다. 이 예제에서는 TextBuilder 및 TextFragment 클래스를 사용하여 새 PDF 문서를 만들고 밑줄 서식이 적용된 텍스트를 추가하는 방법을 보여줍니다. 이 코드는 Arial 글꼴로 서식이 지정되고 밑줄 스타일이 적용된 테스트 메시지가 포함된 간단한 PDF를 생성하며, 페이지 내 특정 좌표에 배치됩니다.

## 설치 지침

1: .NET용 Aspose.PDF 설치

다음 방법 중 하나를 사용하여 .NET용 Aspose.PDF를 설치할 수 있습니다.

NuGet 패키지 관리자 사용:

```csharp
Install-Package Aspose.PDF
```

.NET CLI 사용:

```bash
dotnet add package Aspose.PDF
```

프로젝트 파일에서 PackageReference 사용하기:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: 필수 Using 문 추가

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

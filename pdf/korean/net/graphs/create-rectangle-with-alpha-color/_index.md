## 설치 지침  
프로젝트에서 .NET에 대한 Aspose.PDF를 사용하려면 다음 단계를 수행하십시오.  

1. NuGet를 통해 설치  
NuGet 패키지 관리자 콘솔에서 다음 명령을 실행하세요:  
```powershell
Install-Package Aspose.PDF
```

2. C#에서 네임스페이스를 추가  
C# 파일의 시작에 필요한 이름 공간을 포함하십시오:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

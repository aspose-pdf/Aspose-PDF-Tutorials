# Aspose.PDF로 PDF 이미지를 최적화하십시오

## 설치 지침
1. .NET에 대한 Aspose.PDF를 다운로드:[.NET 다운로드 페이지에 대한 Aspose.PDF](https://docs.aspose.com/pdf/net/)최신 버전을 얻기 위해서요.
2. NuGet를 통해 설치: Visual Studio에서 프로젝트를 열고 패키지 관리자 콘솔에서 다음 명령을 사용하십시오.
   ```
   Install-Package Aspose.PDF
   ```

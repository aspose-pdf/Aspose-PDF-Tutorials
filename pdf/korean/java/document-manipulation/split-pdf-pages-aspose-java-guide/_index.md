   newDocument.getPages().add(pdfDocument1.getPages().get_Item(pdfPage));\n    \n    // Save the new document as a separate PDF file named according to its page number\n    newDocument.save(\"YOUR_OUTPUT_DIRECTORY/page_\" + pdfPage + \".pdf\");\n}\n```\n\n**설명:**\N- **`dataDir`:** 입력된 PDF가 저장되는 디렉토리 경로입니다. **`newDocument`:** 새로운 인스턴스 `Document` 각 페이지가 분할될 때마다.\n- **`get_Item(pdfPage)`:** 원본 문서에서 특정 페이지를 검색합니다.\n\n### 문제 해결 팁\n- 디렉토리 경로가 있고 적절한 권한이 있는지 확인합니다.\n- 런타임 문제를 방지하기 위해 Aspose.PDF의 버전 호환성을 확인합니다.\n\n## 실용적 응용 프로그램\n\nPDF를 개별 페이지로 분할하는 것은 다양한 시나리오에서 매우

Csere `YOUR_DOCUMENT_DIRECTORY` a tényleges mappaútvonallal.\n\n#### 2. lépés: Töltse be a PDF-et\n\nHozzon létre egy `Document` objektum és töltse be a PDF fájlt:\n\n```java\ndocument = new Document(dataDir + \"/Field_Added_output.pdf\");\n```\n\n### PDF űrlap adott mezőjének elérése\n\n**Áttekintés:** Űrlapon belüli adott mezők elérése adatok kinyeréséhez vagy kezeléséhez.\n\n#### 1.

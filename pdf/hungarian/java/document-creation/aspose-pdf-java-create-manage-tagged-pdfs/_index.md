inicializálása**\n```java\nimport com.aspose.pdf.Document;\n\n// Define output directory for saving documents\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n\n// Create a new, empty PDF document\nDocument document = new Document();\n```\n\n**PDF dokumentum mentése**\n\nÁllítsa be a kívánt fájlútvonalat, és mentse el a dokumentumot.\n```java\n// Save the newly created PDF to the title and language for accessibility compliance\ntaggedContent.setTitle(\"Tagged Pdf Document\");\ntaggedContent.setLanguage(\"en-US\");\n```\n\n**Mentse el a címkézett PDF-et**\n```java\ndocument.save(dataDir + \"/TaggedPdfDocument.pdf\");\n```\n\n### 3. funkció: Szövegszerkezeti elemek hozzáadása címkézett PDF-hez\n\n**Áttekintés:** A dokumentum tartalmát a jobb Document();\nITaggedContent taggedContent = document.getTaggedContent();\ntaggedContent.getRootElement().appendChild(taggedContent.createParagraphElement()\n    .setText(\"This is a sample paragraph

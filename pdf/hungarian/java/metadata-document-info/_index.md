Ezek a gyakorlati útmutatók végigvezetik a szabványos dokumentum-metaadatok elérésén és módosításán, az egyéni metaadatmezők létrehozásán, az XMP-metaadatok megvalósításán és a dokumentuminformációs szótárak használatán.

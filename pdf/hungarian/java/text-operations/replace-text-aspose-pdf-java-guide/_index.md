Ez a szakasz a PDF-fájl ezen osztály használatával történő betöltését tárgyalja.\n```java\nimport com.aspose.pdf.facades.PdfContentEditor;\n\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\n//

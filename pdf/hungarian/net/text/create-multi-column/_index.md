## A szerelési utasítások  

Ha a .NET projekthez Aspose.PDF használatát szeretnéd használni, kövesd az alábbi lépéseket:  

1. A NuGet segítségével telepíteni (ajánlott)  
A következő parancsot futtassa a NuGet csomagkezelő konzolban:  
```powershell
Install-Package Aspose.PDF
```

2. A .NET CLI segítségével telepíteni  
A .NET Core vagy a .NET 5+ esetében:  
```bash
dotnet add package Aspose.PDF
```

3. Az telepítés után importálja a projektben a szükséges névtéreket:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

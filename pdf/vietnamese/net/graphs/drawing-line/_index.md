## Hướng dẫn cài đặt:  

Để sử dụng Aspose.PDF cho .NET trong dự án của bạn, hãy làm theo các bước sau:  

1. Cài đặt qua NuGet Package Manager  
* Mở Visual Studio và tạo một dự án .NET mới.  
* Mở Bộ điều khiển quản lý gói (Tools → NuGet Package Manager → Package Manager Console) và chạy:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Cài đặt qua .NET CLI  
Ngoài ra, sử dụng .NET CLI để cài đặt gói:  
```bash
dotnet add package Aspose.PDF
```

3. Thêm không gian tên trong mã C # của bạn  
Để sử dụng chức năng Aspose.PDF, bao gồm các không gian tên sau:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

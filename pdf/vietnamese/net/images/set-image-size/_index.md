## Hướng dẫn cài đặt:  

Để sử dụng Aspose.PDF cho .NET trong dự án của bạn, hãy làm theo các bước sau:  

1. Cài đặt qua NuGet Package Manager:  
Mở Visual Studio và chạy lệnh sau trong Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Thêm không gian tên trong mã C# của bạn:  
```csharp
using Aspose.Pdf;
```

# Thêm văn bản gạch dưới trong PDF với Aspose.PDF cho .NET

Aspose.PDF cho .NET là một thư viện xử lý PDF mạnh mẽ cho phép các nhà phát triển tạo, sửa đổi và thao tác các tài liệu PDF mà không cần Adobe Acrobat. Ví dụ này cho thấy cách tạo một tài liệu PDF mới và thêm văn bản với định dạng nhấn mạnh bằng cách sử dụng các lớp TextBuilder và TextFragment. Mã tạo ra một PDF đơn giản với một tin nhắn thử nghiệm định dạng bằng phông chữ Arial với kiểu chữ nhấn mạnh, định vị nó ở các tọa độ cụ thể trong trang.

## Hướng dẫn cài đặt

1: Cài đặt Aspose.PDF cho .NET

Bạn có thể cài đặt Aspose.PDF cho .NET bằng một trong các phương pháp sau:

Sử dụng NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Sử dụng .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Sử dụng PackageReference trong tệp dự án của bạn:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Thêm yêu cầu sử dụng các câu lệnh

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

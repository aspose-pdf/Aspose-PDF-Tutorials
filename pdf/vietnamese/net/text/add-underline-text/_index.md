# Thêm văn bản gạch chân trong PDF với Aspose.PDF cho .NET

Aspose.PDF for .NET là một thư viện xử lý PDF mạnh mẽ cho phép các nhà phát triển tạo, chỉnh sửa và thao tác các tài liệu PDF mà không cần Adobe Acrobat. Ví dụ này trình bày cách tạo một tài liệu PDF mới và thêm văn bản có định dạng gạch chân bằng cách sử dụng các lớp TextBuilder và TextFragment. Mã này tạo một PDF đơn giản với một thông báo thử nghiệm được định dạng bằng phông chữ Arial có kiểu gạch chân, định vị nó ở các tọa độ cụ thể trong trang.

## Hướng dẫn cài đặt

1: Cài đặt Aspose.PDF cho .NET

Bạn có thể cài đặt Aspose.PDF cho .NET bằng một trong các phương pháp sau:

Sử dụng Trình quản lý gói NuGet:

```csharp
Install-Package Aspose.PDF
```

Sử dụng .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Sử dụng PackageReference trong tệp dự án của bạn:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Thêm các câu lệnh sử dụng bắt buộc

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

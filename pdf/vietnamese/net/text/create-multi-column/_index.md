## Hướng dẫn cài đặt  

Để sử dụng Aspose.PDF cho .NET trong dự án của bạn, hãy làm theo các bước sau:  

1. Cài đặt qua NuGet (Khuyến nghị)  
Chạy lệnh sau trong Bảng điều khiển Trình quản lý gói NuGet:  
```powershell
Install-Package Aspose.PDF
```

2. Cài đặt qua .NET CLI  
Đối với .NET Core hoặc .NET 5+, hãy sử dụng:  
```bash
dotnet add package Aspose.PDF
```

3. Sau khi cài đặt, hãy nhập các không gian tên cần thiết vào dự án của bạn:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

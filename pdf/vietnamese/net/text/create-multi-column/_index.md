# Tạo một PDF đa cột sử dụng Aspose.PDF cho .NET

## Hướng dẫn cài đặt  

Để sử dụng Aspose.PDF cho .NET trong dự án của bạn, hãy làm theo các bước sau:  

1. Cài đặt qua NuGet (Được khuyến cáo)  
Chạy lệnh sau trong NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Cài đặt qua .NET CLI  
Đối với .NET Core hoặc .NET 5+, sử dụng:  
```bash
dotnet add package Aspose.PDF
```

3. Sau khi cài đặt, nhập các không gian tên cần thiết trong dự án của bạn:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

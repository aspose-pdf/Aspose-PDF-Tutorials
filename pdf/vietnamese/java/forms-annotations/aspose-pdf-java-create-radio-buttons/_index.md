Phần này trình bày cách tạo PDF và thêm trang.\n\n##### Bước 1: Khởi tạo tài liệu mới\nTạo phiên bản của `Document` lớp, đại diện cho PDF của bạn:\n```java\nimport com.aspose.pdf.Document;\nimport com.aspose.pdf.Page;\n\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nDocument pdfDocument = new Document();\n```\n\n##### Bước 2: Thêm Trang\nSử dụng `getPages().add()` phương pháp để thêm một trang:\n```java\nPage page = pdfDocument.getPages().add();\n```\n\n##### Bước 3: Lưu tài liệu\nLưu tài liệu của bạn, đảm bảo nó phản ánh tất cả các thay đổi:\n```java\npdfDocument.save(dataDir + \"/SinglePageSample.pdf\");\n```\n\n### Tạo và Cấu hình Trường Nút Radio\n\n#### Tổng quan\nViệc thêm các thành phần biểu mẫu tương tác như nút radio giúp tăng cường tương tác của người dùng trong PDF.

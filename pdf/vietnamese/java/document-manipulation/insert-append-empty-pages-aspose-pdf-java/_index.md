Thực hiện theo các bước sau:\n\n**1. Mở Tài liệu PDF hiện có**\n\nĐầu tiên, hãy tải tệp PDF của bạn bằng cách sử dụng `Document` lớp:\n```java\Nimport com.aspose.pdf.Document;\n\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nDocument pdfDocument1 = new Document(dataDir + \"/input.pdf\");\n```\n*Tại sao?* Việc tải tài liệu là cần thiết để truy cập và thao tác các trang trong đó.\n\n**2.

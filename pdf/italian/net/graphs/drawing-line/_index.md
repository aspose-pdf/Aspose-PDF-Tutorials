## Instruzioni di installazione:  

Per utilizzare Aspose.PDF per .NET nel tuo progetto, segui questi passaggi:  

1. Installare tramite NuGet Package Manager  
* Aprire Visual Studio e creare un nuovo progetto .NET.  
* Aprire Console di gestione dei pacchetti (Tulsi → NuGet Manager di pacchetti → Console di gestione dei pacchetti) e eseguire:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Installazione tramite .NET CLI  
In alternativa, utilizzare il CLI .NET per installare il pacchetto:  
```bash
dotnet add package Aspose.PDF
```

3. Aggiungi spazio di nome nel tuo codice C#  
Per utilizzare la funzionalità Aspose.PDF, includere i seguenti namespaces:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

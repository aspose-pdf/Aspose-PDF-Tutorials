## Istruzioni per l'installazione  
Per utilizzare Aspose.PDF per .NET nel tuo progetto, segui questi passaggi:  

1. Installa tramite NuGet  
Eseguire il seguente comando nella console di NuGet Package Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Aggiungere lo spazio dei nomi in C#  
Includi gli spazi dei nomi richiesti all'inizio del tuo file C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Per la documentazione dettagliata, visitare:  
[Documentazione di Aspose.PDF per .NET](https://docs.aspose.com/pdf/net/)  
[Riferimento API Aspose.PDF per .NET](https://reference.aspose.com/pdf/net/)

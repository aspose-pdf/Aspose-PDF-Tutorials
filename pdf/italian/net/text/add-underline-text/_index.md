# Aggiungere testo sottolineato in PDF con Aspose.PDF per .NET

Aspose.PDF per .NET è una potente libreria di elaborazione PDF che consente agli sviluppatori di creare, modificare e manipolare documenti PDF senza dover utilizzare Adobe Acrobat. Questo esempio illustra come creare un nuovo documento PDF e aggiungere testo con sottolineatura utilizzando le classi TextBuilder e TextFragment. Il codice crea un semplice PDF con un messaggio di prova formattato con il font Arial e sottolineato, posizionandolo in corrispondenza di coordinate specifiche all'interno della pagina.

## Istruzioni per l'installazione

1: Installa Aspose.PDF per .NET

È possibile installare Aspose.PDF per .NET utilizzando uno dei seguenti metodi:

Utilizzo di NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Utilizzo della CLI .NET:

```bash
dotnet add package Aspose.PDF
```

Utilizzo di PackageReference nel file di progetto:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Aggiungere istruzioni di utilizzo obbligatorie

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

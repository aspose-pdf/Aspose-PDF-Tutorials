# Aggiungere testo sottolineato in PDF con Aspose.PDF per .NET

Aspose.PDF per .NET è una potente libreria di elaborazione PDF che consente agli sviluppatori di creare, modificare e manipolare documenti PDF senza richiedere Adobe Acrobat. Questo esempio dimostra come creare un nuovo documento PDF e aggiungere testo con formattazione sottolineata utilizzando le classi TextBuilder e TextFragment. Il codice crea un semplice PDF con un messaggio di prova formattato in font Arial con stile sottolineato, posizionandolo a specifiche coordinate all'interno della pagina.

## Istruzioni di installazione

1: installare Aspose.PDF per .NET

È possibile installare Aspose.PDF per .NET utilizzando uno dei seguenti metodi:

Utilizzare NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Utilizzare .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Utilizzare PackageReference nel file del progetto:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Aggiungere le istruzioni utilizzate richieste

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

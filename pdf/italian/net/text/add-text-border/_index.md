## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Installa tramite .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Installazione manuale:
   - Scarica l'ultima versione da [Aspose.PDF per le versioni .NET](https://releases.aspose.com/pdf/net/)
   - Aggiungi un riferimento a Aspose.PDF.dll nel tuo progetto

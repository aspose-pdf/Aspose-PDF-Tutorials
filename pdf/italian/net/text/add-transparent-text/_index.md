## Istruzioni di installazione

1. Installare tramite NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Installare tramite .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Aggiungere il riferimento manualmente:
   * Scarica la libreria da[Aspose rilasci](https://releases.aspose.com/pdf/net/)
   * Aggiungi il riferimento a Aspose.PDF.dll nel tuo progetto

4. Aggiungere Usando Statement:
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

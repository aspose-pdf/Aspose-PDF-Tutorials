## Istruzioni per l'installazione
Per iniziare a utilizzare Aspose.PDF per .NET, seguire queste istruzioni di installazione:

1. Scarica Aspose.PDF: 
   - Visita il [Sito web di Aspose](https://www.aspose.com/downloads/pdf/net) per scaricare l'ultima versione di Aspose.PDF per .NET.

2. Installa tramite NuGet Package Manager:
   - Apri il progetto in Visual Studio.
   - Vai a Strumenti > Gestore pacchetti NuGet > Gestisci pacchetti NuGet per la soluzione.
   - Cerca "Aspose.PDF" e installa il pacchetto.

3. Aggiungi riferimenti:
   - Assicurati che il tuo progetto faccia riferimento alla DLL Aspose.PDF in `packages` cartella.
   
4. Includi lo spazio dei nomi Aspose.PDF nel tuo progetto:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

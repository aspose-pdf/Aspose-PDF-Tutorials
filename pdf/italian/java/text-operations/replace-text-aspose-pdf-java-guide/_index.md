Questa sezione illustra come caricare il file PDF utilizzando questa classe.```java\nimport com.aspose.pdf.facades.PdfContentEditor;\n\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\n// Initialize PdfContentEditor object\nPdfContentEditor pdfContentEditor = new PdfContentEditor();\n\n// Bind the PDF file\npdfContentEditor.bindPdf(dataDir + \"/Input.pdf\");\n```\n\n### Sostituisci testo nel documento\n\n#### Panoramica\nQui sostituiremo tutte le istanze di una stringa di testo specificata con un'altra in tutto il documento.\n```java\n// Specify text to find and replace\nString

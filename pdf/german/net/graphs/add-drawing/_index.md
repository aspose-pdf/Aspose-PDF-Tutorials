## Installationsanweisungen:  

Um Aspose.PDF für .NET zu verwenden, folgen Sie diesen Schritten:  

1. Installieren Sie über NuGet Package Manager  
* Öffnen Sie Visual Studio und erstellen Sie ein neues .NET-Projekt.  
* Öffnen Sie die Package Manager Console (Tools → NuGet Package Manager → Package Manager Console) und laufen Sie:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Installieren Sie über .NET CLI  
Alternativ installieren Sie das Paket mit der .NET CLI:  
```bash
dotnet add package Aspose.PDF
```

3. Fügen Sie Namespace in Ihren C#-Code hinzu  
Um mit Aspose.PDF zu beginnen, fügen Sie den folgenden Namensraum in Ihre C#-Datei hinzu:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

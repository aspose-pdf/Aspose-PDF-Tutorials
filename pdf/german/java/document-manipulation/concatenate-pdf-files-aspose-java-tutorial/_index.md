\"YOUR_DOCUMENT_DIRECTORY/input2.pdf\";\n```\n\n**Warum**Hiermit werden die Quell-PDFs definiert, die zu einer einzigen Ausgabedatei zusammengeführt werden.\n\n### Schritt 3: Verketten Sie die PDF-Dateien\n\nVerwenden Sie die `concatenate` Methode zum Zusammenführen Ihrer PDF-Dateien:\n\n```java\n// Merge specified PDF files into one\npdfEditor.concatenate(filesArray,

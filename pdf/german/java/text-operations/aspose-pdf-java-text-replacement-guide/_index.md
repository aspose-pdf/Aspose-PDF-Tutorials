Dabei werden Schrifteigenschaften wie Größe und Farbe festgelegt.\n\n#### Schritt 1: PDF-Dokument laden\n\nLaden Sie zunächst Ihre PDF-Quelldatei. Dies geschieht mithilfe des `Document` Klasse:\n\n```java\nimport com.aspose.pdf.Document;\n\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nDocument doc = new Document(dataDir + \"/input.pdf\");\n```\n\n#### Schritt 2: Erstellen Sie einen TextFragment Absorber\n\nVerwenden Sie die `TextFragmentAbsorber` um Text zu finden, der Ihrem regulären Ausdrucksmuster entspricht. Dies ermöglicht eine flexible Suche innerhalb des Dokuments:\n\n```java\nimport com.aspose.pdf.TextFragmentAbsorber;\n\nTextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(\"[Cname,companyname,Textbox,50]\");\ndoc.getPages().accept(textFragmentAbsorber);\n```\n\n#### Schritt 3: Text iterieren und ersetzen\n\nDurchlaufen Sie die gefundenen Textfragmente, um sie durch die gewünschte Zeichenfolge zu ersetzen. Dabei passen wir auch die Schrifteigenschaften an:\n\n```java\nimport com.aspose.pdf.FontRepository;\nimport com.aspose.pdf.Color;\n\nfor (TextFragment textFragment : (Iterable<TextFragment>) textFragmentAbsorber.getTextFragments()) {\n    textFragment.getTextState().setFont(FontRepository.findFont(\"Arial\"));\n    textFragment.getTextState().setFontSize(12);\n    textFragment.getTextState().setForegroundColor(Color.getNavy());\n    textFragment.setText(\"This is a Larger String to Testing of this issue\");\n}\n```\n\n#### Schritt 4: Speichern Sie die geänderte PDF-Datei\n\nSpeichern Sie Ihr Dokument nach den Änderungen:\n\n```java\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\ndoc.save(outputDir + \"/29860_out_large_NoHyphenation_1020.pdf\");\n```\n\n### Funktion 2: Anpassung der Textersetzung (optional)\n\nObwohl in unserem ersten Beispiel auskommentiert, zeigt diese Funktion, wie die Textersetzung mit Silbentrennungsoptionen angepasst wird:\n\n```java\nimport com.aspose.pdf.TextReplaceOptions;\n\n// Load source PDF file\nDocument doc = new Document(dataDir + \"/input.pdf\");\n\nTextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(\"[Cname,companyname,Textbox,50]\");\ndoc.getPages().accept(textFragmentAbsorber);\n\ntextFragmentAbsorber.getTextReplaceOptions().setReplaceAdjustmentAction(TextReplaceOptions.ReplaceAdjustment.WholeWordsHyphenation);\n```\n\nDies kann nützlich sein, um sicherzustellen, dass der Text genau in die Layoutbeschränkungen des Dokuments passt.\n\n## Praktische Anwendungen\n\n1. **Automatisierte Dokumentenverarbeitung**: Verwenden Sie

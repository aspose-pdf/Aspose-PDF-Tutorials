Ausgabedateien einrichten.\n```java\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\n```\n\n#### Schritt 3: Erstellen Sie eine Methode zum Glätten von Feldern\nDefinieren Sie die `documentForm` Methode zum Öffnen, Reduzieren und Speichern Ihres PDF-Dokuments.\n\n```java\nvoid documentForm() {\n    // Initialize Form object to interact with the PDF\n    Form pdfForm = new Form();\n    \n    // Bind the source PDF file from 'dataDir/input.pdf'\n    pdfForm.bindPdf(dataDir + \"/input.pdf\");\n    \n    // Flatten all fields in the PDF document\n    pdfForm.flattenAllFields();\n    \n    // Save the modified document to 'outputDir/output.pdf'\n    pdfForm.save(outputDir + \"/output.pdf\");\n}\n```\n\n#### Erklärung:\n- **`Form` Klasse**: Wird zum

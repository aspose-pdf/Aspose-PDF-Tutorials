## 設置說明:  
請使用以下步:  

1. 請下載並安裝Aspose.PDF  
   * 訪問[網路下載的Aspose.PDF](https://releases.aspose.com/pdf/net/)還下載最新版本.  
   * 另行選擇是使用NuGet包管理器安裝:  
     ```sh
     Install-Package Aspose.PDF
     ```  

2. 請將Aspose.PDF命名空間加入您的項目:  
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;
   ```

## 設置說明  

請使用以下步:  

1. 透過NuGet安裝 (建議)  
在 NuGet包管理器控制台執行下列命令:  
```powershell
Install-Package Aspose.PDF
```

2. 透過.NET CLI 進行安裝  
必須使用以下方式:  
```bash
dotnet add package Aspose.PDF
```

3. 在安裝後, 輸入您的項目中必要的命名空間:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

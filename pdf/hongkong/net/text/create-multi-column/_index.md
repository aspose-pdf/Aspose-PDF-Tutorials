## 安裝說明  

若要在您的專案中使用 Aspose.PDF for .NET，請依照下列步驟操作：  

1. 透過 NuGet 安裝（建議）  
在 NuGet 套件管理器控制台中執行以下命令：  
```powershell
Install-Package Aspose.PDF
```

2. 透過 .NET CLI 安裝  
對於 .NET Core 或 .NET 5+，請使用：  
```bash
dotnet add package Aspose.PDF
```

3. 安裝後，在專案中導入必要的命名空間：  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

## 設置說明

1. 透過NuGet包管理器安裝:
   ```
   Install-Package Aspose.PDF
   ```

2. 透過.NET CLI安裝:
   ```
   dotnet add package Aspose.PDF
   ```

3. 隨手添加參考:
   * 請下載圖書館[子釋放](https://releases.aspose.com/pdf/net/)
   * 在您的項目中添加"Aspose.PDF.dll"參考

4. 加入使用聲明:
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

## 設置說明

1. 透過NuGet包管理器安裝:
   ```
   Install-Package Aspose.PDF
   ```

2. 透過.NET CLI安裝:
   ```
   dotnet add package Aspose.PDF
   ```

3. 按手動安裝方式:
   * 請下載最新版本[網頁版本的Aspose.PDF](https://releases.aspose.com/pdf/net/)
   * 在您的項目中添加"Aspose.PDF.dll"參考

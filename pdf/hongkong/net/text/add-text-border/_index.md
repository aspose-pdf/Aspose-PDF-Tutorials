# 在 PDF 文件中建立帶有邊框的樣式文本

Aspose.PDF for .NET 是一個強大的 PDF 處理 API，允許開發人員無需 Adobe Acrobat 即可建立、編輯、操作和轉換 PDF 文件。此範例示範如何建立具有特定樣式和邊框的文字片段的 PDF 文檔，這對於突出顯示文檔中的重要內容特別有用。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.PDF
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.PDF
   ```

3. 手動安裝：
   - 從下載最新版本 [Aspose.PDF for .NET 發布](https://releases.aspose.com/pdf/net/)
   - 在您的專案中新增對 Aspose.PDF.dll 的引用

## 設置說明

請使用Aspose.PDF

您可以使用以下方法之一安裝Aspose.PDF

使用NuGet包管理器:

```csharp
Install-Package Aspose.PDF
```

使用 .NET CLI:

```bash
dotnet add package Aspose.PDF
```

在您的項目檔案中使用 PackageReference:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

增加必須使用的說明

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

# 使用 Aspose.PDF for .NET 在 PDF 中新增底線文字

Aspose.PDF for .NET 是一個強大的 PDF 處理庫，允許開發人員無需 Adobe Acrobat 即可建立、修改和操作 PDF 文件。此範例示範如何使用 TextBuilder 和 TextFragment 類別建立新的 PDF 文件並新增帶有下劃線格式的文字。程式碼建立了一個簡單的 PDF，其中包含以 Arial 字體格式化的測試訊息，並帶有下劃線樣式，並將其定位在頁面內的特定座標處。

## 安裝說明

1：安裝 Aspose.PDF for .NET

您可以使用下列方法之一安裝 Aspose.PDF for .NET：

使用 NuGet 套件管理器：

```csharp
Install-Package Aspose.PDF
```

使用 .NET CLI：

```bash
dotnet add package Aspose.PDF
```

在專案檔案中使用 PackageReference：

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2：新增必需的 Using 語句

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## 安裝說明：  

若要在您的專案中使用 Aspose.PDF for .NET，請依照下列步驟操作：  

1. 透過 NuGet 套件管理器安裝  
- 開啟 Visual Studio 並建立一個新的 .NET 專案。  
- 開啟套件管理器控制台（工具 → NuGet 套件管理器 → 套件管理器控制台）並執行：  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. 透過 .NET CLI 安裝  
或者，使用 .NET CLI 安裝套件：  
```bash
dotnet add package Aspose.PDF
```

3. 在 C# 程式碼中新增命名空間  
若要使用 Aspose.PDF 功能，請包含以下命名空間：  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

**Q:** *.NET 教學能在 .NET Core 上執行嗎？*  
**A:** 當然可以——所有範例皆相容於 .NET Core 3.1 及以上版本，同時支援 .NET 5/6/7。  

**Q:** *如何從既有的 PDF 中提取圖像？*  
**A:** 使用「Extract Images PDF」章節中示範的 `PdfExtractor` 類別即可抽取點陣圖資產。

**Q:** *我可以在 PDF 頁面上繪製自訂線條或形狀嗎？*

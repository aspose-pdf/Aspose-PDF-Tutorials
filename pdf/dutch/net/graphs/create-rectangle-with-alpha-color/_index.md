## Installatie-instructies  
Om Aspose.PDF voor .NET in uw project te gebruiken, volg deze stappen:  

1. Installeer via NuGet  
Voer het volgende commando uit in de NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Namespace toevoegen in C#  
Voeg de benodigde naamruimtes aan het begin van uw C#-bestand in:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Voor gedetailleerde documentatie:  
[Aspose.PDF voor .NET-documentatie](https://docs.aspose.com/pdf/net/)

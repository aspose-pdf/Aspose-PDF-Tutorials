## Installatie-instructies  
Om Aspose.PDF voor .NET in uw project te gebruiken, volgt u deze stappen:  

1. Installeer via NuGet  
Voer het volgende commando uit in de NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Voeg namenruimte toe in C#  
Voeg de vereiste namenruimtes aan het begin van je C#-bestand toe:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Voor gedetailleerde documentatie:  
[Aspose.PDF voor .NET Documentatie](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF voor .NET API Referentie](https://reference.aspose.com/pdf/net/)

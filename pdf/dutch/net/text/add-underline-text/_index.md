## Installatie-instructies

1: Installeer Aspose.PDF voor .NET

U kunt Aspose.PDF voor .NET installeren met behulp van een van de volgende methoden:

Gebruik NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Gebruik .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Gebruik PackageReference in uw projectbestand:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Voeg vereiste gebruikte uitspraken toe

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

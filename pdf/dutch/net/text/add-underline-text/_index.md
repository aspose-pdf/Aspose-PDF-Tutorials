## Installatie-instructies

1: Aspose.PDF voor .NET installeren

U kunt Aspose.PDF voor .NET installeren met behulp van een van de volgende methoden:

NuGet Package Manager gebruiken:

```csharp
Install-Package Aspose.PDF
```

Met behulp van .NET CLI:

```bash
dotnet add package Aspose.PDF
```

PackageReference gebruiken in uw projectbestand:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Voeg vereiste gebruiksinstructies toe

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

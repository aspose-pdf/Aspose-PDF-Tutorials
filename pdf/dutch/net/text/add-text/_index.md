## Installatie-instructies

1: Voorwaarden
* Visual Studio 2017 of hoger
* .NET Framework 4.0 of hoger of .NET Core 2.0+ / .NET 5.0+
* Basiskennis van C#-programmering

2: Installeer Aspose.PDF voor .NET
U kunt Aspose.PDF voor .NET installeren met behulp van een van de volgende methoden:

Gebruik van NuGet Package Manager Console:
```csharp
PM> Install-Package Aspose.PDF
```

3: Gebruik Visual Studio NuGet Package Manager:
	1. Open je project in Visual Studio
	2. Klik met de rechtermuisknop op het project in Solution Explorer en selecteer "Manage NuGet Packages"
	3. Zoek naar "Aspose.PDF" en installeer het pakket

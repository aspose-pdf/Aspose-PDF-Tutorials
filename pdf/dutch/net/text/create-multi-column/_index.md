## Installatie-instructies  

Om Aspose.PDF voor .NET in uw project te gebruiken, volgt u deze stappen:  

1. Installeer via NuGet (aanbevolen)  
Voer het volgende commando uit in de NuGet Package Manager Console:  
```powershell
Install-Package Aspose.PDF
```

2. Installeer via .NET CLI  
Voor .NET Core of .NET 5+ gebruik:  
```bash
dotnet add package Aspose.PDF
```

3. Na installatie, importeer de benodigde namenruimten in uw project:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

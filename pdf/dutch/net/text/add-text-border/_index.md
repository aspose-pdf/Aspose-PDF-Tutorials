## Installatie-instructies

1. Installeer via NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Installeer via .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Handmatige installatie:
   * Download de laatste versie van [Aspose.PDF voor .NET-releases](https://releases.aspose.com/pdf/net/)
   * Voeg verwijzing toe naar Aspose.PDF.dll in je project

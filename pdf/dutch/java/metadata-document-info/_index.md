# PDF-metagegevens en documentinformatie-tutorials voor Aspose.PDF Java

Onze tutorials over metadata en documentinformatie laten zien hoe u PDF-documenteigenschappen beheert met Aspose.PDF in Java. Deze praktische handleidingen begeleiden u bij het openen en wijzigen van standaard documentmetadata, het maken van aangepaste metadatavelden, het implementeren van XMP-metadata en het werken met woordenboeken met documentinformatie. Elke tutorial bevat gedetailleerde Java-codevoorbeelden voor het bewerken van metadata in overeenstemming met PDF-standaarden, zodat u applicaties kunt bouwen die de juiste documentattributie en -classificatie behouden.

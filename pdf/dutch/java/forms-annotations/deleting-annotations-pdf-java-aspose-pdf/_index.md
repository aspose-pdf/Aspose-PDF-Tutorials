geven.\n\n```java\NString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nDocument pdfDocument = new Document(dataDir + \"/input.pdf\");\n```\n*Uitleg:* Deze code initialiseert een `Document` object, waardoor onnodige annotaties selectief kunnen worden verwijderd.\n\n#### Stap 4: Sla uw document op\nNadat u wijzigingen hebt aangebracht, slaat u uw document op een nieuwe locatie op.\n\n```java\NString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\npdfDocument.save(outputDir + \"/output.pdf\");\n```\n*Uitleg:* De `save` methode schrijft de gewijzigde PDF terug naar de opslag, waarbij alle tijdens de verwerking gemaakte updates behouden blijven.\n\n### Tips voor probleemoplossing\n- **Index Buiten de grenzen:** Zorg ervoor dat de index die u probeert te verwijderen, in de

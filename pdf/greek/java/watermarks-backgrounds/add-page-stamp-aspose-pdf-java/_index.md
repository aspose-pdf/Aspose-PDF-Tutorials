# Πώς να προσθέσετε μια σφραγίδα σελίδας σε PDF χρησιμοποιώντας το Aspose.PDF για Java

## Οδηγός Εφαρμογής

### Προσθήκη σφραγίδας σελίδας σε έγγραφο PDF

Αυτή η λειτουργία σάς επιτρέπει να επικαλύψετε μια σφραγίδα από ένα PDF σε ένα άλλο. Ας το αναλύσουμε:

#### Επισκόπηση
Θα πάρετε μια σελίδα από `stamp.pdf` και τοποθετήστε το ως φόντο στην πρώτη σελίδα του `input.pdf`.

#### Βήμα προς βήμα εφαρμογή

**1. Τοποθετήστε τα έγγραφά σας**
Ξεκινήστε φορτώνοντας τόσο το κύριο έγγραφο όσο και την πηγή σφραγίδας.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/input.pdf");
Document pdfStampSource = new Document(dataDir + "/stamp.pdf");
```

**2. Δημιουργήστε ένα αντικείμενο σφραγίδας**
Χρησιμοποιήστε την πρώτη σελίδα του `stamp.pdf` για να δημιουργήσετε το αντικείμενο σφραγίδας σας.
```java
import com.aspose.pdf.PdfPageStamp;

PdfPageStamp pageStamp = new PdfPageStamp(pdfStampSource.getPages().get_Item(1));
pageStamp.setBackground(true); // Σφραγίδα στο φόντο
// Ρυθμίστε τη θέση και την περιστροφή για ακριβή τοποθέτηση
pageStamp.setXIndent(100);
pageStamp.setYIndent(100);
pageStamp.setRotate(com.aspose.pdf.Rotation.on180);
```

**3. Προσθέστε τη σφραγίδα στο έγγραφό σας**
Τώρα, εφαρμόστε αυτήν τη σφραγίδα στο κύριο έγγραφό σας.
```java
pdfDocument.getPages().get_Item(1).addStamp(pageStamp);
// Αποθήκευση του ενημερωμένου εγγράφου
String outputDir = "YOUR_OUTPUT_DIRECTORY";
pdfDocument.save(outputDir + "/output_pdfpagestamp.pdf");
```

### Σφραγίδα περιστρεφόμενης σελίδας σε έγγραφο PDF
Για να περιστρέψετε μια σφραγίδα, προσαρμόστε τον προσανατολισμό της πριν την προσθέσετε στο PDF προορισμού σας.

#### Επισκόπηση
Αυτή η διαδικασία περιλαμβάνει τη ρύθμιση της περιστροφής στο `PdfPageStamp` αντικείμενο και στη συνέχεια εφαρμογή του στο έγγραφό σας.

**1. Φόρτωση εγγράφων**
Όπως και πριν, φορτώστε και τα δύο έγγραφα.
```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
Document pdfStampSource = new Document(dataDir + "/stamp.pdf");
```

**2. Δημιουργία και διαμόρφωση της σφραγίδας**
Δημιουργήστε μια σφραγίδα και ορίστε την περιστροφή της.
```java
PdfPageStamp pageStamp = new PdfPageStamp(pdfStampSource.getPages().get_Item(1));
pageStamp.setRotate(com.aspose.pdf.Rotation.on180);
```

**3. Εφαρμογή και Αποθήκευση**
Προσθέστε την περιστρεφόμενη σφραγίδα στο PDF σας και, στη συνέχεια, αποθηκεύστε την.
```java
pdfDocument.getPages().get_Item(1).addStamp(pageStamp);
pdfDocument.save(outputDir + "/output_rotated_stamp.pdf");
```

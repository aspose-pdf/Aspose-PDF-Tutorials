## Πώς να Φορτώσετε Έγγραφο PDF Java
Η φόρτωση ενός εγγράφου PDF είναι το πρώτο βήμα για την πρόσβαση στα ενσωματωμένα αρχεία του. Η παρακάτω ενότητα δείχνει τον ακριβή κώδικα που χρειάζεστε.

### Οδηγός Υλοποίησης

### Λειτουργία 1: Φόρτωση Εγγράφου PDF
Η φόρτωση ενός εγγράφου PDF είναι το πρώτο βήμα για την πρόσβαση στα ενσωματωμένα αρχεία του.

**Επισκόπηση:**  
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον PDF χρησιμοποιώντας το Aspose.PDF for Java, θέτοντας τη βάση για περαιτέρω λειτουργίες όπως η εξαγωγή ιδιοτήτων ενσωματωμένων αρχείων.

#### Βήμα‑Βήμα Υλοποίηση
1. **Specify Your Document Directory:**  
   Define where your input PDF is stored.  
   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   ```
2. **Load the PDF Document:**  
   Use the `Document` class to load a PDF file from the specified path.  
   ```java
   import com.aspose.pdf.Document;

   Document pdfDocument = new Document(dataDir + "/input.pdf");
   System.out.println("PDF Loaded Successfully.");
   ```

### Λειτουργία 2: Πρόσβαση σε Ενσωματωμένα Αρχεία σε PDF
Η πρόσβαση σε ενσωματωμένα αρχεία μέσα σε ένα φορτωμένο έγγραφο είναι κρίσιμη για την εξαγωγή των ιδιοτήτων τους.

**Επισκόπηση:**  
Αυτή η λειτουργία δείχνει πώς να έχετε πρόσβαση στη λίστα των ενσωματωμένων αρχείων και να ανακτήσετε συγκεκριμένα χρησιμοποιώντας την κλάση `FileSpecification` του Aspose.PDF.

#### Βήμα‑Βήμα Υλοποίηση
1. **Retrieve the List of Embedded Files:**  
   Access the collection of embedded files in your PDF.  
   ```java
   import com.aspose.pdf.FileSpecification;

   FileSpecification fileSpecification = pdfDocument.getEmbeddedFiles().get_Item(1);
   System.out.println("Accessed Embedded File.");
   ```

### Λειτουργία 3: Ανάκτηση Ιδιοτήτων Αρχείου από Ενσωματωμένα Αρχεία
Η εξαγωγή ιδιοτήτων των ενσωματωμένων αρχείων παρέχει πολύτιμα μεταδεδομένα.

**Επισκόπηση:**  
Αυτή η λειτουργία δείχνει πώς να ανακτήσετε διάφορες ιδιότητες όπως όνομα, περιγραφή, τύπο MIME και άλλα από ένα ενσωματωμένο αρχείο μέσα σε PDF.

#### Βήμα‑Βήμα Υλοποίηση
1. **Print Basic Properties:**  
   Use the `FileSpecification` object to access basic properties.  
   ```java
   System.out.println("Name:-" + fileSpecification.getName());
   System.out.println("Description:- " + fileSpecification.getDescription());
   System.out.println("Mime Type:-" + fileSpecification.getMIMEType());
   ```
2. **Retrieve Additional Parameters:**  
   Check for additional metadata if available.  
   ```java
   if (fileSpecification.getParams() != null) {
       System.out.println("CheckSum:- " + fileSpecification.getParams().getCheckSum());
       System.out.println("Creation Date:- " + fileSpecification.getParams().getCreationDate());
       System.out.println("Modification Date:- " + fileSpecification.getParams().getModDate());
       System.out.println("Size:- " + fileSpecification.getParams().getSize());
   }
   ```

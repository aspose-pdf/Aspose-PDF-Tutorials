## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση εγγράφου PDF
Η φόρτωση ενός εγγράφου PDF είναι το πρώτο βήμα για την πρόσβαση στα ενσωματωμένα αρχεία του.

**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον PDF χρησιμοποιώντας το Aspose.PDF για Java, προετοιμάζοντας το έδαφος για περαιτέρω λειτουργίες όπως η εξαγωγή ενσωματωμένων ιδιοτήτων αρχείου.

#### Βήμα προς βήμα εφαρμογή
1. **Καθορίστε τον κατάλογο εγγράφων σας:**
   Ορίστε πού αποθηκεύεται το PDF που εισαγάγατε.
   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   ```
2. **Φόρτωση του εγγράφου PDF:**
   Χρησιμοποιήστε το `Document` κλάση για να φορτώσετε ένα αρχείο PDF από την καθορισμένη διαδρομή.
   ```java
   import com.aspose.pdf.Document;

   Document pdfDocument = new Document(dataDir + "/input.pdf");
   System.out.println("PDF Loaded Successfully.");
   ```

### Δυνατότητα 2: Πρόσβαση σε ενσωματωμένα αρχεία σε PDF
Η πρόσβαση σε ενσωματωμένα αρχεία μέσα σε ένα φορτωμένο έγγραφο είναι κρίσιμη για την εξαγωγή των ιδιοτήτων τους.

**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να αποκτήσετε πρόσβαση στη λίστα των ενσωματωμένων αρχείων και να ανακτήσετε συγκεκριμένα χρησιμοποιώντας το Aspose.PDF. `FileSpecification` τάξη.

#### Βήμα προς βήμα εφαρμογή
1. **Ανάκτηση της λίστας ενσωματωμένων αρχείων:**
   Αποκτήστε πρόσβαση στη συλλογή ενσωματωμένων αρχείων στο PDF σας.
   ```java
   import com.aspose.pdf.FileSpecification;

   FileSpecification fileSpecification = pdfDocument.getEmbeddedFiles().get_Item(1);
   System.out.println("Accessed Embedded File.");
   ```

### Δυνατότητα 3: Ανάκτηση ιδιοτήτων αρχείου από ενσωματωμένα αρχεία
Η εξαγωγή ιδιοτήτων από ενσωματωμένα αρχεία παρέχει πολύτιμα μεταδεδομένα.

**Επισκόπηση:**
Αυτή η λειτουργία παρουσιάζει τον τρόπο ανάκτησης διαφόρων ιδιοτήτων, όπως όνομα, περιγραφή, τύπο MIME και άλλα, από ένα ενσωματωμένο αρχείο μέσα σε ένα PDF.

#### Βήμα προς βήμα εφαρμογή
1. **Βασικές Ιδιότητες Εκτύπωσης:**
   Χρησιμοποιήστε το `FileSpecification` αντικείμενο για πρόσβαση σε βασικές ιδιότητες.
   ```java
   System.out.println("Name:-" + fileSpecification.getName());
   System.out.println("Description:- " + fileSpecification.getDescription());
   System.out.println("Mime Type:-" + fileSpecification.getMIMEType());
   ```
2. **Ανάκτηση πρόσθετων παραμέτρων:**
   Ελέγξτε για πρόσθετα μεταδεδομένα, εάν υπάρχουν.
   ```java
   if (fileSpecification.getParams() != null) {
       System.out.println("CheckSum:- " + fileSpecification.getParams().getCheckSum());
       System.out.println("Creation Date:- " + fileSpecification.getParams().getCreationDate());
       System.out.println("Modification Date:- " + fileSpecification.getParams().getModDate());
       System.out.println("Size:- " + fileSpecification.getParams().getSize());
   }
   ```

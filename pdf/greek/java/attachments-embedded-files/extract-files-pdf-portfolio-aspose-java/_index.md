### Εξαγωγή αρχείων από χαρτοφυλάκιο PDF

#### Επισκόπηση

Αυτή η ενότητα σας καθοδηγεί στον τρόπο εξαγωγής μεμονωμένων αρχείων που είναι ενσωματωμένα σε ένα χαρτοφυλάκιο PDF. Αυτό είναι ιδιαίτερα χρήσιμο όταν ασχολείστε με έγγραφα που περιέχουν πολλαπλούς τύπους μέσων ή αρχεία δεδομένων.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση χαρτοφυλακίου PDF πηγής**

Ξεκινήστε φορτώνοντας το έγγραφο PDF σε ένα Aspose.PDF `Document` αντικείμενο:

```java
import com.aspose.pdf.Document;

// Φόρτωση χαρτοφυλακίου PDF πηγής
document = new Document("Portfolio_output.pdf");
```

**2. Ανάκτηση Συλλογής Ενσωματωμένων Αρχείων**

Αποκτήστε πρόσβαση στη συλλογή ενσωματωμένων αρχείων χρησιμοποιώντας το `getEmbeddedFiles()` μέθοδος:

```java
import com.aspose.pdf.EmbeddedFileCollection;

// Λήψη συλλογής ενσωματωμένων αρχείων
embeddedFiles = document.getEmbeddedFiles();
```

**3. Επαναλάβετε και εξαγάγετε κάθε αρχείο**

Εκτελέστε επανάληψη σε κάθε αρχείο στο χαρτοφυλάκιο για να τα εξαγάγετε ξεχωριστά:

```java
import java.io.File;
import java.io.FileOutputStream;
import java.io.IOException;
import java.io.InputStream;

// Επαναλάβετε την επεξεργασία μεμονωμένων αρχείων του χαρτοφυλακίου
for (int counter = 1; counter <= embeddedFiles.size(); counter++) {
    com.aspose.pdf.FileSpecification fileSpecification = embeddedFiles.get_Item(counter);
    try (InputStream input = fileSpecification.getContents()) {
        File file = new File(fileSpecification.getName());
        
        // Δημιουργία διαδρομής για αρχείο από PDF
        if (!file.getParentFile().exists()) {
            file.getParentFile().mkdirs();
        }
        
        // Εξαγωγή και αποθήκευση του αρχείου
        try (FileOutputStream output = new FileOutputStream(file)) {
            byte[] buffer = new byte[4096];
            int n;
            while ((n = input.read(buffer)) != -1) {
                output.write(buffer, 0, n);
            }
        }
    } catch (IOException e) {
        System.err.println("An error occurred: " + e.getMessage());
    }
}
```

Σε αυτό το απόσπασμα:
- `input` ανακτά τη ροή περιεχομένου κάθε ενσωματωμένου αρχείου.
- Δημιουργούμε καταλόγους όπως απαιτείται χρησιμοποιώντας `mkdirs()`.
- Τα αρχεία εξάγονται και αποθηκεύονται με ένα `FileOutputStream`.

#### Συμβουλές αντιμετώπισης προβλημάτων

- **Το αρχείο δεν βρέθηκε:** Βεβαιωθείτε ότι η διαδρομή του εγγράφου PDF είναι σωστή.
- **Προβλήματα δικαιωμάτων:** Ελέγξτε τα δικαιώματα καταλόγου κατά τη δημιουργία φακέλων ή την εγγραφή αρχείων.

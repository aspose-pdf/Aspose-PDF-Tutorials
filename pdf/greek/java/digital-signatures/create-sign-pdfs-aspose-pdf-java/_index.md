## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Δημιουργία και Χειρισμός Εγγράφων

Η δημιουργία ενός εγγράφου PDF από την αρχή είναι απλή με το Aspose.PDF για Java.

#### Οδηγός βήμα προς βήμα:

**1. Δημιουργήστε ένα αντίγραφο του αντικειμένου εγγράφου**

Ξεκινήστε δημιουργώντας μια παρουσία του `Document` τάξη:
```java
import com.aspose.pdf.Document;
import java.io.ByteArrayOutputStream;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document();
```

Αυτό δημιουργεί ένα κενό έγγραφο PDF στη μνήμη.

**2. Προσθήκη σελίδας στο έγγραφο**

Η προσθήκη σελίδων είναι ζωτικής σημασίας για οποιοδήποτε μη κενό PDF:
```java
// Προσθήκη κενής σελίδας
doc.getPages().add();
```

**3. Αποθηκεύστε το έγγραφο σε ένα OutputStream**

Τέλος, αποθηκεύστε το PDF που μόλις δημιουργήσατε σε μια ροή εξόδου:
```java
import java.io.OutputStream;

OutputStream out = new ByteArrayOutputStream();
doc.save(out);
```

Αυτό το βήμα μετατρέπει το περιεχόμενο του εγγράφου σας σε μορφή πίνακα byte για περαιτέρω επεξεργασία ή αποθήκευση.

### Χαρακτηριστικό 2: Ψηφιακή υπογραφή σε αρχεία PDF

Οι ψηφιακές υπογραφές είναι απαραίτητες για την επαλήθευση της αυθεντικότητας και της ακεραιότητας των εγγράφων.

#### Οδηγός βήμα προς βήμα:

**1. Προετοιμασία του εγγράφου για υπογραφή**

Αρχικά, συνδέστε το PDF που θα υπογραφεί:
```java
import com.aspose.pdf.facades.PdfFileSignature;
import java.io.ByteArrayInputStream;

PdfFileSignature pdfSignSingle = new PdfFileSignature();
pdfSignSingle.bindPdf(new ByteArrayInputStream(((ByteArrayOutputStream) out).toByteArray()));
```

Αυτός ο κώδικας μετατρέπει τη ροή εξόδου από το βήμα δημιουργίας εγγράφου σε έναν πίνακα byte για σύνδεση.

**2. Εφαρμογή ψηφιακής υπογραφής**

Χρησιμοποιήστε το PKCS1 για να υπογράψετε το PDF σας:
```java
import com.aspose.pdf.PKCS1;
import java.awt.Rectangle;

pdfSignSingle.sign(1, true, new Rectangle(100, 100, 150, 50), new PKCS1(dataDir + "certificate.pfx", "password"));
```

Αυτό το τμήμα κώδικα τοποθετεί την ψηφιακή υπογραφή σε καθορισμένες συντεταγμένες στην πρώτη σελίδα του εγγράφου.

**3. Ορισμός εμφάνισης υπογραφής**

Η προσθήκη οπτικής αναπαράστασης βελτιώνει την αναγνωσιμότητα:
```java
pdfSignSingle.setSignatureAppearance(dataDir + "signature-image.jpg");
```

**4. Αποθηκεύστε το υπογεγραμμένο PDF**

Τέλος, αποθηκεύστε το υπογεγραμμένο PDF σας σε έναν καθορισμένο κατάλογο εξόδου:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
pdfSignSingle.save(outputDir + "signed_document.pdf");
```

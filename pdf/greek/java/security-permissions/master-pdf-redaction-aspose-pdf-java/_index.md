## Οδηγός Εφαρμογής
Θα εξερευνήσουμε δύο μεθόδους για την επεξεργασία περιοχών μέσα σε ένα PDF: τη χρήση των σχολίων επεξεργασίας και της πρόσοψης του PdfAnnotationEditor. Κάθε μέθοδος έχει τα πλεονεκτήματά της, ανάλογα με τις συγκεκριμένες ανάγκες σας.

### Επεξεργασία με σχολιασμό επεξεργασίας
Αυτή η προσέγγιση επιτρέπει την λεπτομερή προσαρμογή της κωδικοποιημένης περιοχής.

#### Επισκόπηση
Μπορείτε να καθορίσετε ακριβώς πού θα εφαρμόσετε τις διορθώσεις και να προσαρμόσετε την εμφάνισή τους.

#### Βήματα
##### 1. Τοποθετήστε το έγγραφο
```java
document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
Εδώ, φορτώνουμε το έγγραφο PDF στη μνήμη.

##### 2. Ορίστε την περιοχή επεξεργασίας
```java
Rectangle rect = new Rectangle(200, 500, 300, 600);
```
Αυτό το ορθογώνιο ορίζει τις συντεταγμένες της περιοχής που θέλετε να επεξεργαστείτε.

##### 3. Δημιουργία και διαμόρφωση της σχολίασης
```java
RedactionAnnotation annot = new RedactionAnnotation(doc.getPages().get_Item(1), rect);
annot.setFillColor(Color.getBlack());
annot.setBorderColor(Color.getYellow());
annot.setColor(Color.getBlue());
annot.setFontSize(20);
annot.setOverlayText("REDACTED");
annot.setTextAlignment(HorizontalAlignment.Center);
annot.setRepeat(true);
```
Αυτό το απόσπασμα δημιουργεί μια σχολίαση επιμέλειας, ορίζοντας το χρώμα και το κείμενο επικάλυψης.

##### 4. Προσθέστε την σχολίαση στη σελίδα
```java
doc.getPages().get_Item(1).getAnnotations().add(annot);
```
Η επιμέλεια προστίθεται στην πρώτη σελίδα του εγγράφου.

##### 5. Αποθήκευση του εγγράφου
```java
doc.save("YOUR_OUTPUT_DIRECTORY/Redaction_out.pdf");
```
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο PDF.

### Επεξεργασία με την προσέγγιση PdfAnnotationEditor Facades
Αυτή η μέθοδος παρέχει έναν γρήγορο τρόπο εφαρμογής διορθώσεων χωρίς προσαρμογή σχολίων.

#### Επισκόπηση
Χρήση `PdfAnnotationEditor` για απλές εργασίες επιμέλειας.

#### Βήματα
##### 1. Αρχικοποίηση του Επεξεργαστή
```java
PdfAnnotationEditor editor = new PdfAnnotationEditor();
editor.bindPdf("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
Φορτώστε το έγγραφό σας με το πρόγραμμα επεξεργασίας.

##### 2. Εφαρμογή Επιμέλειας
```java
color redactColor = Color.WHITE;
editor.redactArea(1, new Rectangle(100, 100, 20, 70), redactColor);
```
Ορίστε και εφαρμόστε μια επιμέλεια στη σελίδα 1 χρησιμοποιώντας καθορισμένες συντεταγμένες.

##### 3. Αποθήκευση του εγγράφου
```java
color outputDirectory = "YOUR_OUTPUT_DIRECTORY/Redaction_out.pdf";
editor.save(outputDirectory);
```
Αποθηκεύστε το τροποποιημένο έγγραφό σας με τις εφαρμοσμένες διορθώσεις.

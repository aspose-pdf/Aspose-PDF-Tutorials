## Οδηγός Εφαρμογής
Θα αναλύσουμε την υλοποίησή μας σε ενότητες με βάση τη λειτουργικότητα: αρχικοποίηση εγγράφου, αναζήτηση τμημάτων κειμένου, δημιουργία σχολίων και αποθήκευση του εγγράφου.

### Αρχικοποίηση και Φόρτωση Εγγράφου
#### Επισκόπηση
Αρχικά, πρέπει να φορτώσουμε ένα αρχείο PDF χρησιμοποιώντας το Aspose.PDF για Java. Αυτό το βήμα περιλαμβάνει την αρχικοποίηση του `Document` τάξη με τη διαδρομή προς το PDF σας.

**Βήμα 1**Εισαγωγή απαραίτητων κλάσεων.
```java
import com.aspose.pdf.Document;
```

**Βήμα 2**Ορίστε τον κατάλογο του εγγράφου σας και αρχικοποιήστε το αντικείμενο Document.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document document = new Document(dataDir + "/input.pdf");
```
Αυτός ο κώδικας φορτώνει ένα PDF από `dataDir` σε ένα `Document` για παράδειγμα, επιτρέποντας περαιτέρω λειτουργίες.

### Απορρόφηση Θραυσμάτων Κειμένου
#### Επισκόπηση
Η αναζήτηση κειμένου μέσα στο PDF σας είναι απαραίτητη πριν από τη δημιουργία σχολίων. Θα χρησιμοποιήσουμε το `TextFragmentAbsorber`.

**Βήμα 1**Εισαγωγή της κλάσης.
```java
import com.aspose.pdf.TextFragmentAbsorber;
```

**Βήμα 2**Δημιουργήστε μια παρουσία του TextFragmentAbsorber με το ερώτημα αναζήτησης.
```java
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("Sample");
```

**Βήμα 3**Αποδεχτείτε τον απορροφητή σε κάθε σελίδα για να βρείτε όλα τα αντίστοιχα τμήματα κειμένου.
```java
for (int i = 1; i <= document.getPages().size(); i++) {
    document.getPages().get_Item(i).accept(textFragmentAbsorber);
}
```
Αυτός ο βρόχος αναζητά σε κάθε σελίδα, συλλέγοντας αντιστοιχίσεις για το "Δείγμα".

### Ανάκτηση συλλογής τμημάτων κειμένου
#### Επισκόπηση
Μόλις απορροφηθούν τα αποσπάσματα κειμένου, τα ανακτούμε σε μια συλλογή για να επεξεργαστούμε τις σχολιασμούς.

**Βήμα 1**Εισαγάγετε την απαιτούμενη κλάση.
```java
import com.aspose.pdf.TextFragmentCollection;
```

**Βήμα 2**: Λήψη της συλλογής από τον απορροφητή.
```java
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.getTextFragments();
```
Αυτό μας δίνει πρόσβαση σε όλες τις περιπτώσεις του "Δείγματος" που βρέθηκαν.

### Δημιουργία σχολίου διαγραφής
#### Επισκόπηση
Θα προσθέσουμε μια διαγραμμένη σημείωση πάνω από κάθε αναγνωρισμένο τμήμα κειμένου, επισημαίνοντάς το ή επισημαίνοντάς το οπτικά.

**Βήμα 1**Εισαγωγή απαραίτητων κλάσεων.
```java
import com.aspose.pdf.StrikeOutAnnotation;
import com.aspose.pdf.Rectangle;
import com.aspose.pdf.Border;
import com.aspose.pdf.Color;
import com.aspose.pdf.Page;
```

**Βήμα 2**: Περιηγηθείτε στα αποσπάσματα κειμένου και δημιουργήστε σχολιασμούς.
```java
for (int j = 1; j <= textFragmentCollection.size(); j++) {
    com.aspose.pdf.TextFragment textFragment = textFragmentCollection.get_Item(j);

    // Ορίστε την περιοχή ορθογωνίου για σχολιασμό με βάση τη θέση του τμήματος κειμένου.
    Rectangle rect = new Rectangle((float) textFragment.getPosition().getXIndent(),
                                   (float) textFragment.getPosition().getYIndent(),
                                   (float) textFragment.getPosition().getXIndent() + (float) textFragment.getRectangle().getWidth(),
                                   (float) textFragment.getPosition().getYIndent() + (float) textFragment.getRectangle().getHeight());

    // Δημιουργήστε και διαμορφώστε τη σχολίαση διαγραφής.
    StrikeOutAnnotation strikeOut = new StrikeOutAnnotation(textFragment.getPage(), rect);
    strikeOut.setOpacity(0.80f);  // Ορισμός αδιαφάνειας στο 80%
    strikeOut.setBorder(new Border(strikeOut));  // Αρχικοποίηση περιγράμματος για την σχολίαση
    strikeOut.setColor(Color.getRed());  // Ορισμός χρώματος σχολιασμού σε κόκκινο

    // Προσθέστε τη σχολίαση στη σελίδα.
    textFragment.getPage().getAnnotations().add(strikeOut);
}
```
Κάθε σχολιασμός διαμορφώνεται με ρυθμίσεις αδιαφάνειας, περιγράμματος και χρώματος πριν προστεθεί.

### Αποθήκευση εγγράφου με σχολιασμούς
#### Επισκόπηση
Αφού δημιουργήσουμε σχολιασμούς, αποθηκεύουμε το σχολιασμένο έγγραφο.

**Βήμα 1**Εισαγάγετε τις απαραίτητες κλάσεις, εάν δεν τις έχετε ήδη εισαγάγει.
```java
import com.aspose.pdf.Document;
```

**Βήμα 2**: Ορίστε τον κατάλογο εξόδου και αποθηκεύστε το έγγραφο.
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
document.save(outputDir + "/StrikeOut.pdf");
```
Αυτό το βήμα ολοκληρώνει το σχολιασμένο PDF σας με όλες τις τροποποιήσεις άθικτες.

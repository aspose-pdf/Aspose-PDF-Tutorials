## Οδηγός Υλοποίησης

Αυτός ο οδηγός χωρίζεται σε δύο κύρια χαρακτηριστικά: μετατροπή μιας εικόνας σε PDF χρησιμοποιώντας άμεσες ροές αρχείου και προσθήκη μιας εικόνας από `BufferedImage` σε PDF.

### Μετατροπή Εικόνας σε PDF Χρησιμοποιώντας Άμεσο Ροή Αρχείου

#### Βήμα 1: Δημιουργία Αντικειμένου Document
```java
doc = new Document();
```
Δημιουργήστε ένα αντικείμενο `Document`, το οποίο θα λειτουργήσει ως κοντέινερ για το αρχείο PDF σας.

#### Βήμα 2: Προσθήκη Σελίδας στο Document
```java
page = doc.getPages().add();
```
Προσθέστε μια νέα σελίδα στο έγγραφο. Εδώ θα τοποθετηθεί η εικόνα σας.

#### Βήμα 3: Φόρτωση Αρχείου Εικόνας
```java
FileInputStream fs = new FileInputStream("YOUR_DOCUMENT_DIRECTORY/source.tif");
```
Φορτώστε το αρχικό αρχείο εικόνας σε ένα `FileInputStream`. Αντικαταστήστε το `"YOUR_DOCUMENT_DIRECTORY/source.tif"` με την πραγματική διαδρομή της εικόνας σας.

#### Βήμα 4: Ορισμός Περιθωρίων Σελίδας και Crop Box
```java
page.getPageInfo().getMargin().setBottom(0);
page.getPageInfo().getMargin().setTop(0);
page.getPageInfo().getMargin().setLeft(0);
page.getPageInfo().getMargin().setRight(0);
page.setCropBox(new Rectangle(0, 0, 400, 400));
```
Ρυθμίστε τα περιθώρια και το crop box ώστε η εικόνα να ταιριάζει καλά μέσα στη σελίδα.

#### Βήμα 5: Δημιουργία και Προσθήκη Αντικειμένου Image
```java
Image image1 = new Image();
page.getParagraphs().add(image1);
image1.setImageStream(fs);
```
Δημιουργήστε ένα αντικείμενο `Image`, προσθέστε το στη συλλογή παραγράφων της σελίδας και ορίστε τη ροή του με το `FileInputStream` σας.

#### Βήμα 6: Αποθήκευση του PDF Εγγράφου
```java
doc.save("YOUR_OUTPUT_DIRECTORY/Image2PDF_DOM.pdf");
```
Αποθηκεύστε το έγγραφο στην επιθυμητή τοποθεσία. Αντικαταστήστε το `"YOUR_OUTPUT_DIRECTORY/Image2PDF_DOM.pdf"` με τη διαδρομή εξόδου σας.

### Προσθήκη Εικόνας από BufferedImage σε PDF

#### Βήμα 1: Δημιουργία Document και Προσθήκη Σελίδας
```java
doc = new Document();
page = doc.getPages().add();
```
Ξεκινήστε δημιουργώντας το έγγραφο και προσθέτοντας μια σελίδα, όπως στο προηγούμενο χαρακτηριστικό.

#### Βήμα 2: Δημιουργία BufferedImage από Αρχείο Εικόνας
```java
Image image1 = new Image();
java.awt.image.BufferedImage bufferedImage = ImageIO.read(new File("YOUR_DOCUMENT_DIRECTORY/source.gif"));
ByteArrayOutputStream baos = new ByteArrayOutputStream();

// Write the BufferedImage as GIF
ImageIO.write(bufferedImage, "gif", baos);
baos.flush();
ByteArrayInputStream bais = new ByteArrayInputStream(baos.toByteArray());
```
Μετατρέψτε το αρχείο εικόνας σας σε `BufferedImage` και στη συνέχεια γράψτε το σε ένα `ByteArrayOutputStream`. Αυτή η ροή θα χρησιμοποιηθεί για την προσθήκη της εικόνας στο PDF.

#### Βήμα 3: Προσθήκη Εικόνας στη Σελίδα και Ορισμός Ροής
```java
page.getParagraphs().add(image1);
image1.setImageStream(bais);
```
Προσθέστε το αντικείμενο εικόνας στη συλλογή παραγράφων της σελίδας και ορίστε τη ροή του χρησιμοποιώντας το `ByteArrayInputStream`.

#### Βήμα 4: Αποθήκευση του PDF Εγγράφου
```java
doc.save("YOUR_OUTPUT_DIRECTORY/BufferedImage.pdf");
```
Αποθηκεύστε το έγγραφό σας. Φροντίστε να καθορίσετε το σωστό φάκελο εξόδου.

## Οδηγός Εφαρμογής

Τώρα που η εγκατάσταση έχει ολοκληρωθεί, ας εφαρμόσουμε τη μετατροπή PCL σε PDF.

### Επισκόπηση λειτουργιών: Μετατροπή από PCL σε PDF
Αυτή η λειτουργία επιτρέπει την απρόσκοπτη μετατροπή ενός αρχείου PCL σε μορφή PDF χρησιμοποιώντας τη βιβλιοθήκη Aspose.PDF. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Ορισμός καταλόγων
Καθορίστε καταλόγους για τα αρχεία PCL εισόδου και τα PDF εξόδου.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος που περιέχει αρχεία PCL
String outputDir = "YOUR_OUTPUT_DIRECTORY";  // Κατάλογος για την αποθήκευση των PDF που έχουν μετατραπεί
```

#### Βήμα 2: Φόρτωση αρχείου PCL
Χρήση `PclLoadOptions` για να φορτώσετε το αρχείο PCL.
```java
PclLoadOptions loadoptions = new PclLoadOptions();
```

#### Βήμα 3: Δημιουργία αντικειμένου εγγράφου
Δημιουργήστε ένα υπόδειγμα `Document` αντικείμενο με τη διαδρομή αρχείου PCL και τις επιλογές φόρτωσης:
```java
// Αντικαταστήστε το 'test.pcl' με το πραγματικό όνομα αρχείου PCL σας
Document doc = new Document(dataDir + "test.pcl", loadoptions);
```

#### Βήμα 4: Αποθήκευση ως PDF
Αποθηκεύστε το έγγραφο σε ένα αρχείο PDF στον καθορισμένο κατάλογο εξόδου.
```java
doc.save(outputDir + "output.pdf");
```

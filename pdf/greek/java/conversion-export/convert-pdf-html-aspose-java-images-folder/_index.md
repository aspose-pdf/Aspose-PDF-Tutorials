### Καθορισμός Φακέλου Εικόνων για τη Μετατροπή
**Overview:** Θα ορίσουμε έναν κατάλογο όπου όλες οι εξαγόμενες εικόνες από το PDF αποθηκεύονται κατά τη μετατροπή σε HTML. Αυτό διατηρεί το έργο σας τακτοποιημένο, ειδικά όταν εργάζεστε με μεγάλα PDFs που περιέχουν πολλές εικόνες.

#### Βήμα 1: Ορισμός Διαδρομών
Η κλάση `Document` αντιπροσωπεύει ένα αρχείο PDF στη μνήμη. Ορίστε πρώτα τις τοποθεσίες προέλευσης και εξόδου:

```java
// Define paths for your input PDF and output directory
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";
```
*Explanation:* Replace `YOUR_DOCUMENT_DIRECTORY` and `YOUR_OUTPUT_DIRECTORY` with the actual locations on your machine.

#### Βήμα 2: Φόρτωση του PDF Εγγράφου
Η κλάση `Document` φορτώνει το πηγαίο PDF για επεξεργασία.

```java
// Load the document from the specified directory
Document pdfDocument = new Document(dataDir + "SampleDataTable.pdf");
```
*Explanation:* This creates a `Document` object for the source PDF. Ensure the file name matches an existing PDF.

#### Βήμα 3: Διαμόρφωση Επιλογών Αποθήκευσης HTML
`HtmlSaveOptions` ελέγχει πώς συμπεριφέρεται η μετατροπή. Ορίζοντας `setSpecialFolderForAllImages` λέει στο Aspose.PDF να τοποθετήσει κάθε εξαγόμενη εικόνα στον υποκατάλογο `imageFolder` που παρέχετε.

```java
// Set up HtmlSaveOptions for conversion settings
HtmlSaveOptions saveOptions = new HtmlSaveOptions();

// Specify the folder where images will be stored
saveOptions.setSpecialFolderForAllImages(outputDir + "imageFolder/");
```
*Explanation:* `setSpecialFolderForAllImages` tells Aspose.PDF to place every extracted image into the `imageFolder` sub‑directory you provide.

#### Βήμα 4: Αποθήκευση του HTML Εγγράφου
Η μετατροπή ολοκληρώνεται εδώ, παράγοντας ένα αρχείο HTML που αναφέρεται στις εικόνες που αποθηκεύονται στον φάκελο που ορίσατε.

```java
// Convert and save the PDF as an HTML document with specified image storage location
pdfDocument.save(outputDir + "resultant.html", saveOptions);
```
*Explanation:* The conversion finishes here, producing an HTML file that references images stored in the folder you defined.

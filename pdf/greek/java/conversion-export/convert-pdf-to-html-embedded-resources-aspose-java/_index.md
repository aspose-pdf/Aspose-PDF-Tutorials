## Ρύθμιση του Aspose.PDF για Java

### Ρύθμιση Maven
Προσθέστε την ακόλουθη εξάρτηση στο αρχείο `pom.xml` σας:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

### Ρύθμιση Gradle
Συμπεριλάβετε αυτή τη γραμμή στο αρχείο `build.gradle` σας:
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

**Απόκτηση Άδειας**: Αποκτήστε μια δοκιμαστική ή πλήρη άδεια από το Aspose. Η δοκιμή σας επιτρέπει να εξερευνήσετε όλες τις δυνατότητες, ενώ μια αγορασμένη άδεια αφαιρεί τους περιορισμούς αξιολόγησης.

**Βασική Αρχικοποίηση**: Εισάγετε την κύρια κλάση στον κώδικα Java σας:
```java
import com.aspose.pdf.Document;
```

## Υλοποίηση Βήμα‑Βήμα

### Βήμα 1: Φόρτωση του Πηγαίου PDF
Πρώτα, υποδείξτε στο Aspose.PDF το αρχείο εισόδου σας:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document doc = new Document(dataDir);
```
Αν το PDF είναι προστατευμένο με κωδικό, περάστε τον κωδικό στον κατασκευαστή `Document` (δείτε τη δευτερεύουσα λέξη-κλειδί *convert password protected pdf*).

### Βήμα 2: Δημιουργία Επιλογών Αποθήκευσης HTML
Δημιουργήστε ένα αντικείμενο `HtmlSaveOptions` για να ελέγξετε τη μετατροπή:
```java
HtmlSaveOptions options = new HtmlSaveOptions();
```

### Βήμα 3: Ενσωμάτωση Όλων των Πόρων στο HTML
Εντοπίστε το Aspose να ενσωματώνει κάθε εικόνα, γραμματοσειρά και τμήμα CSS απευθείας στο αρχείο HTML:
```java
options.setPartsEmbeddingMode(HtmlSaveOptions.PartsEmbeddingModes.EmbedAllIntoHtml);
```

### Βήμα 4: Βελτιστοποίηση Τοποθέτησης Γραμμάτων
Για απόδοση pixel‑perfect—ιδιαίτερα σε παλαιότερα προγράμματα περιήγησης όπως το Internet Explorer—χρησιμοποιήστε μονάδες EM με αντιστάθμιση σφάλματος στρογγυλοποίησης:
```java
options.setLettersPositioningMethod(LettersPositioningMethods.UseEmUnitsAndCompensationOfRoundingErrorsInCss);
```

### Βήμα 5: Αποθήκευση Ραστερ Εικόνων ως Ενσωματωμένα PNG Υπόβαθρα
Η ενσωμάτωση ραστερ εικόνων ως μέρος του φόντου της σελίδας διατηρεί το HTML τακτοποιημένο:
```java
options.setRasterImagesSavingMode(HtmlSaveOptions.RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground);
```

### Βήμα 6: Διατήρηση Γραμματοσειρών σε Όλες τις Μορφές
Εξασφαλίστε μέγιστη συμβατότητα αποθηκεύοντας τις γραμματοσειρές σε κάθε διαθέσιμη μορφή:
```java
options.setFontSavingMode(HtmlSaveOptions.FontSavingModes.SaveInAllFormats);
```

### Βήμα 7: Δημιουργία του Μονού HTML Αποτελέσματος
Τέλος, δημιουργήστε το αυτό-συμπεριλαμβανόμενο αρχείο HTML:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/Single_output.html";
doc.save(outputDir, options);
```
Το αποτέλεσμα είναι ένα αυτόνομο έγγραφο HTML έτοιμο για άμεση δημοσίευση στο web.

## Συχνές Ερωτήσεις

**Q:** Μπορώ να μετατρέψω αρχεία PDF με κωδικό πρόσβασης χρησιμοποιώντας αυτή τη μέθοδο;  
**A:** Ναι. Παρέχετε τον κωδικό πρόσβασης όταν δημιουργείτε το αντικείμενο `Document`. Αυτό ευθυγραμμίζεται με τη δευτερεύουσα λέξη-κλειδί *convert password protected pdf*.

**Q:** Πώς μπορώ να διαχειριστώ πολύ μεγάλα αρχεία PDF αποδοτικά;  
**A:** Κατανείμετε περισσότερη μνήμη heap, επεξεργαστείτε τα αρχεία σε παρτίδες και σκεφτείτε να καλέσετε `doc.optimizeResources()` πριν από την αποθήκευση.

**Q:** Είναι δυνατόν να προσαρμόσετε τον τρόπο ενσωμάτωσης των γραμματοσειρών στο αποτέλεσμα HTML;  
**A:** Απόλυτα. Χρησιμοποιήστε διαφορετικές `FontSavingModes` (π.χ., `SaveInWoff`, `SaveInTtf`) για να ταιριάξετε με τους προορισμένους browsers.

**Q:** Ποια προγράμματα περιήγησης υποστηρίζουν το ενσωματωμένο HTML που παράγει το Aspose.PDF;  
**A:** Όλα τα σύγχρονα προγράμματα περιήγησης (Chrome, Edge, Firefox, Safari) αποδίδουν σωστά το αυτό-συμπεριλαμβανόμενο HTML. Παλαιότερες εκδόσεις του IE μπορεί να χρειάζονται τη ρύθμιση τοποθέτησης μονάδων EM που εφαρμόσαμε.

**Q:** Μπορώ να διατηρήσω τα διαδραστικά πεδία φόρμας κατά τη μετατροπή σε HTML;  
**A:** Η μετατροπή φόρμας είναι περιορισμένη· ίσως χρειαστεί να εξάγετε τα δεδομένα της φόρμας ξεχωριστά και να δημιουργήσετε τα πεδία χειροκίνητα σε HTML.

## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε την υλοποίηση σε συγκεκριμένα χαρακτηριστικά.

### Χαρακτηριστικό 1: Μετατροπή αρχείου HTML σε PDF
#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει τη μετατροπή ενός απλού αρχείου HTML σε έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για Java.
##### Βήμα προς βήμα εφαρμογή:
###### Ρύθμιση επιλογών φόρτωσης
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
HtmlLoadOptions htmloptions = new HtmlLoadOptions(dataDir);
```
*Εξήγηση:* Ο `HtmlLoadOptions` Η κλάση επιτρέπει τον καθορισμό επιλογών, όπως η βασική διαδρομή, για τη φόρτωση αρχείων HTML.
###### Φόρτωση και μετατροπή εγγράφου
```java
document doc = new Document(dataDir + "/EmailDemo_updated.html", htmloptions);
doc.save("YOUR_OUTPUT_DIRECTORY/Web+URL_output.pdf");
```
*Εξήγηση:* Ο `Document` Ο κατασκευαστής φορτώνει το αρχείο HTML σας. Καλώντας `save()`, το μετατρέπετε σε PDF που είναι αποθηκευμένο στον καθορισμένο κατάλογο.

### Λειτουργία 2: Απόδοση περιεχομένου HTML σε PDF μίας σελίδας
#### Επισκόπηση
Αποδώστε εκτεταμένο περιεχόμενο HTML σε ένα PDF μίας σελίδας, ιδανικό για τη δημιουργία συνοπτικών αναφορών ή περιλήψεων.
##### Βήμα προς βήμα εφαρμογή:
###### Ρύθμιση παραμέτρων επιλογών για απόδοση μίας σελίδας
```java
HtmlLoadOptions options = new HtmlLoadOptions();
options.setRenderToSinglePage(true);
```
*Εξήγηση:* Σύνθεση `setRenderToSinglePage(true)` διασφαλίζει ότι όλο το περιεχόμενο χωράει σε μία σελίδα, ανεξάρτητα από το αρχικό του μήκος.
###### Φόρτωση και αποθήκευση του εγγράφου
```java
document doc = new Document(dataDir + "/HTMLToPDF.html", options);
doc.save("YOUR_OUTPUT_DIRECTORY/RenderContentToSamePage_out.pdf");
```
*Εξήγηση:* Παρόμοια με τη Λειτουργία 1, αλλά με απόδοση μίας σελίδας που εφαρμόζεται μέσω `HtmlLoadOptions`.

### Χαρακτηριστικό 3: Απόδοση HTML με ενσωματωμένα δεδομένα SVG
#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στη μετατροπή ενός αρχείου HTML που περιέχει ενσωματωμένα δεδομένα SVG σε ένα έγγραφο PDF.
##### Βήμα προς βήμα εφαρμογή:
###### Αρχικοποίηση επιλογών φόρτωσης με διαδρομή SVG
```java
HtmlLoadOptions options = new HtmlLoadOptions(dataDir + "/HTMLSVG.html");
```
*Εξήγηση:* Η βασική διαδρομή έχει οριστεί για να χειρίζεται σωστά το περιεχόμενο SVG κατά τη μετατροπή.
###### Δημιουργία και αποθήκευση εγγράφου
```java
document pdfDocument = new Document(dataDir + "/HTMLSVG.html", options);
pdfDocument.save("YOUR_OUTPUT_DIRECTORY/RenderHTMLwithSVGData_out.pdf");
```
*Εξήγηση:* Φορτώνει HTML με ενσωματωμένα SVG, διατηρώντας την πιστότητα των γραφικών στην έξοδο PDF.

### Δυνατότητα: Προειδοποίηση αντικατάστασης γραμματοσειράς κατά τη μετατροπή PDF σε HTML

Αυτή η λειτουργία σάς επιτρέπει να παρακολουθείτε και να καταγράφετε τυχόν αντικαταστάσεις γραμματοσειρών που συμβαίνουν κατά τη διαδικασία μετατροπής από μορφή PDF σε μορφή HTML.

#### Βήμα 1: Φόρτωση του εγγράφου PDF
Φορτώστε το υπάρχον αρχείο PDF χρησιμοποιώντας το Aspose.PDF `Document` κλάση. Αυτό το βήμα είναι απαραίτητο για την πρόσβαση στο περιεχόμενό της και την εφαρμογή περαιτέρω λειτουργιών.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDoc = new Document(dataDir + "input1.pdf");
```

#### Βήμα 2: Ρύθμιση ενός χειριστή αντικατάστασης γραμματοσειράς
Υλοποιήστε έναν χειριστή αντικατάστασης γραμματοσειρών για να καταγράψετε τυχόν αλλαγές στις γραμματοσειρές κατά τη μετατροπή. Αυτός ο χειριστής καταγράφει τις αρχικές και τις αντικατεστημένες γραμματοσειρές.

```java
final Map<String, String> names = new HashMap<>();
pdfDoc.FontSubstitution.add(new Document.FontSubstitutionHandler() {
    public void invoke(Font font, Font newFont) {
        // Το αρχείο καταγραφής αντικατέστησε το FontNames σε έναν χάρτη.
        names.put(font.getFontName(), newFont.getFontName());
    }
});
```

**Γιατί αυτό το βήμα;**
Η καταγραφή των αντικαταστάσεων γραμματοσειρών σάς επιτρέπει να λάβετε διορθωτικά μέτρα εάν η οπτική ακεραιότητα του εγγράφου σας έχει παραβιαστεί.

```java
HtmlSaveOptions htmlSaveOps = new HtmlSaveOptions();
```

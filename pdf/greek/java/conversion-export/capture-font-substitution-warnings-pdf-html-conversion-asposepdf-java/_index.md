# Μετατροπή PDF σε HTML: Καταγραφή Προειδοποιήσεων Αντικατάστασης Γραμματοσειράς με το Aspose.PDF για Java

## Εισαγωγή

Όταν εκτελείτε μια **pdf to html conversion**, η αντικατάσταση γραμματοσειράς μπορεί σιωπηρά να αλλάξει την εμφάνιση των σελίδων σας, προκαλώντας μεταβολές διάταξης ή ελλιπή χαρακτήρες. Η καταγραφή αυτών των προειδοποιήσεων σας επιτρέπει να επαληθεύσετε ότι η μετατροπή διατηρεί το αρχικό σχέδιο και βοηθά στον εντοπισμό ελλιπών γραμματοσειρών pdf πριν γίνουν πρόβλημα. Σε αυτό το σεμινάριο, θα μάθετε πώς να συνδέεστε στη γραμμή μετατροπής του Aspose.PDF for Java, να καταγράφετε τυχόν αλλαγές γραμματοσειράς και να αποθηκεύετε το παραγόμενο αρχείο HTML με σιγουριά.

**Τι θα επιτύχετε:**
- Κατανοήστε γιατί η παρακολούθηση της αντικατάστασης γραμματοσειράς είναι σημαντική για τη pdf to html conversion.
- Ρυθμίστε έναν διαχειριστή αντικατάστασης γραμματοσειράς που καταγράφει κάθε αλλαγή γραμματοσειράς.
- Διαμορφώστε το `HtmlSaveOptions` για να ρυθμίσετε λεπτομερώς το αποτέλεσμα της μετατροπής.

Ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε πριν προχωρήσουμε.

## Γρήγορες Απαντήσεις
- **Τι κάνει ο διαχειριστής αντικατάστασης γραμματοσειράς;** Καταγράφει το αρχικό όνομα γραμματοσειράς και τη γραμματοσειρά που το Aspose.PDF αντικαθιστά κατά τη μετατροπή.  
- **Μπορώ να το χρησιμοποιήσω με έργα pdf to html java;** Ναι, ο κώδικας λειτουργεί με οποιαδήποτε εφαρμογή Java που αναφέρεται στο Aspose.PDF.  
- **Χρειάζομαι άδεια για παραγωγική χρήση;** Απαιτείται έγκυρη άδεια Aspose.PDF για εμπορικές εγκαταστάσεις.  
- **Θα εντοπιστούν αυτόματα οι ελλιπείς γραμματοσειρές;** Ο διαχειριστής καταγράφει κάθε αντικατάσταση, επιτρέποντάς σας να εντοπίσετε τις ελλιπείς γραμματοσειρές pdf.  
- **Απαιτείται επιπλέον διαμόρφωση;** Μόνο η τυπική ρύθμιση του Aspose.PDF και η καταγραφή του διαχειριστή όπως φαίνεται παρακάτω.

## Τι είναι η pdf to html conversion;
Η pdf to html conversion μετατρέπει ένα έγγραφο PDF σε ένα φιλικό προς το web αρχείο HTML, προσπαθώντας να διατηρήσει την αρχική διάταξη, τις γραμματοσειρές και τις εικόνες. Αυτή η διαδικασία είναι χρήσιμη για την εμφάνιση PDF σε προγράμματα περιήγησης χωρίς την ανάγκη πρόσθετου προβολέα PDF.

## Γιατί να καταγράψετε προειδοποιήσεις αντικατάστασης γραμματοσειράς;
Κατά τη μετατροπή, εάν η αρχική γραμματοσειρά δεν είναι ενσωματωμένη ή δεν είναι διαθέσιμη στο σύστημα, το Aspose.PDF την αντικαθιστά με εναλλακτική. Χωρίς ορατότητα, το HTML μπορεί να φαίνεται αισθητά διαφορετικό. Καταγράφοντας τις προειδοποιήσεις μπορείτε:
- Να εντοπίσετε τις ελλιπείς γραμματοσειρές νωρίς.
- Να επιλέξετε την ενσωμάτωση των απαιτούμενων γραμματοσειρών.
- Να παρέχετε μια εναλλακτική στρατηγική για τους τελικούς χρήστες.

## Οδηγός Υλοποίησης

### Λειτουργία: Προειδοποίηση Αντικατάστασης Γραμματοσειράς στη pdf to html conversion

This feature lets you monitor and capture any font substitutions that occur while converting a PDF to HTML.

#### Βήμα 1: Φόρτωση του PDF εγγράφου σας
(Already shown above) Loading the document gives you access to its content and font information.

#### Βήμα 2: Ρύθμιση Διαχειριστή Αντικατάστασης Γραμματοσειράς
Register a handler that logs each substitution into a map for later inspection.

```java
final Map<String, String> names = new HashMap<>();
pdfDoc.FontSubstitution.add(new Document.FontSubstitutionHandler() {
    public void invoke(Font font, Font newFont) {
        // Log substituted FontNames into a map.
        names.put(font.getFontName(), newFont.getFontName());
    }
});
```

**Γιατί είναι σημαντικό:**  
Εάν η μετατροπή αντικαταστήσει μια ιδιόκτητη γραμματοσειρά με μια γενική, το HTML μπορεί να εμφανιστεί με απρόσμενα κενά ή ελλιπή σύμβολα. Ο χάρτης `names` σας παρέχει σαφή καταγραφή.

#### Βήμα 3: Διαμόρφωση Επιλογών Αποθήκευσης HTML
Create an `HtmlSaveOptions` instance to control how the PDF is saved as HTML.

```java
HtmlSaveOptions htmlSaveOps = new HtmlSaveOptions();
```

Μπορείτε να προσαρμόσετε περαιτέρω ιδιότητες όπως `SplitIntoPages`, `EmbedFonts` ή `ImageCompression` ανάλογα με τις ανάγκες του έργου σας.

#### Βήμα 4: Αποθήκευση του Μετατρεπόμενου Εγγράφου
Finally, write the HTML output to disk.

```java
pdfDoc.save("YOUR_OUTPUT_DIRECTORY/getWarningForFontSubstitution.html\
```

Μετά την εκτέλεση, εξετάστε τον χάρτη `names` για να δείτε ποιες γραμματοσειρές αντικαταστάθηκαν. Εάν παρατηρήσετε απροσδόκητες καταχωρήσεις, σκεφτείτε να ενσωματώσετε τις ελλιπείς γραμματοσειρές ή να προσαρμόσετε τις ρυθμίσεις μετατροπής.

## Κοινά Προβλήματα & Επίλυση

| Σύμπτωμα | Πιθανή Αιτία | Διόρθωση |
|----------|--------------|----------|
| Δεν υπάρχουν καταχωρήσεις στον χάρτη `names` | Η αντικατάσταση γραμματοσειράς είναι απενεργοποιημένη ή όλες οι γραμματοσειρές είναι ενσωματωμένες | Βεβαιωθείτε ότι το `EmbedFonts` είναι ορισμένο σε `false` στο `HtmlSaveOptions` εάν θέλετε να δείτε τις αντικαταστάσεις. |
| Η διάταξη HTML διασπάται | Η αντικατεστημένη γραμματοσειρά δεν περιέχει τα απαιτούμενα σύμβολα | Ενσωματώστε τη λείπουσα γραμματοσειρά ή παρέχετε εναλλακτικό CSS που ταιριάζει με το αρχικό σχέδιο. |
| `pdfDoc.save` προκαλεί εξαίρεση | Λανθασμένη διαδρομή εξόδου ή έλλειψη δικαιωμάτων εγγραφής | Επαληθεύστε ότι το `YOUR_OUTPUT_DIRECTORY` υπάρχει και είναι εγγράψιμο. |

## Συχνές Ερωτήσεις

**Ε: Μπορώ να χρησιμοποιήσω αυτή την προσέγγιση με άλλες μορφές εξόδου (π.χ., DOCX);**  
Α: Ναι. Το Aspose.PDF παρέχει παρόμοια γεγονότα αντικατάστασης γραμματοσειράς για τους περισσότερους στόχους μετατροπής.

**Ε: Πώς να εντοπίσω ελλιπείς γραμματοσειρές pdf πριν από τη μετατροπή;**  
Α: Εξετάστε τη συλλογή `pdfDoc.FontInfo` ή βασιστείτε στον διαχειριστή αντικατάστασης κατά τη μετατροπή.

**Ε: Υπάρχει τρόπος να ενσωματωθούν αυτόματα οι ελλιπείς γραμματοσειρές;**  
Α: Ορίστε `htmlSaveOps.setEmbedFonts(true)`· το Aspose.PDF θα ενσωματώσει τις διαθέσιμες γραμματοσειρές, αλλά οι πραγματικά ελλιπείς γραμματοσειρές πρέπει να παρασχεθούν χειροκίνητα.

**Ε: Λειτουργεί αυτό με κρυπτογραφημένα PDF;**  
Α: Ναι, εφόσον παρέχετε τον κωδικό πρόσβασης κατά τη φόρτωση του εγγράφου: `new Document(path, new LoadOptions(password))`.

**Ε: Θα αυξήσει αυτό τον χρόνο μετατροπής;**  
Α: Το κόστος καταγραφής των αντικαταστάσεων είναι ελάχιστο, συνήθως προσθέτει μόνο μερικά χιλιοστά του δευτερολέπτου.

## Χαρακτηριστικό 1: Μετατροπή Όλων των Σελίδων PDF σε Εικόνες JPEG

### Τι σημαίνει το “convert all pdf pages”;

Αυτή η λειτουργία διασχίζει κάθε σελίδα ενός εγγράφου PDF και αποθηκεύει κάθε σελίδα ως ξεχωριστό αρχείο JPEG. Είναι ιδανική για σενάρια **batch convert pdf jpeg** όπως η δημιουργία αρχείων εικόνας ή η προετοιμασία μικρογραφιών για μια γκαλερί στο web.

### Υλοποίηση Βήμα‑βήμα

**Βήμα 1: Φόρτωση του PDF Εγγράφου**  
Ξεκινάμε δημιουργώντας ένα αντικείμενο `Document` που αντιπροσωπεύει το πηγαίο PDF.

```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

**Βήμα 2: Επανάληψη σε Κάθε Σελίδα**  
Ένας απλός βρόχος `for` περνάει από κάθε δείκτη σελίδας.

```java
for (int pageCount = 1; pageCount <= pdfDocument.getPages().size(); pageCount++) {
    // Processing code here
}
```

**Βήμα 3: Ρύθμιση Εξόδου και Παραμέτρων Μετατροπής**  
Ρυθμίστε το `JpegDevice` με ανάλυση 300 DPI και μέγιστη ποιότητα (100).

```java
Resolution resolution = new Resolution(300);
JpegDevice jpegDevice = new JpegDevice(resolution, 100);
```

**Βήμα 4: Μετατροπή Κάθε Σελίδας σε JPEG**  
Μέσα στον βρόχο, γράφουμε κάθε σελίδα σε αρχείο JPEG χρησιμοποιώντας ένα μπλοκ try‑with‑resources για να διασφαλίσουμε ότι η ροή κλείνει σωστά.

```java
try (OutputStream imageStream = new FileOutputStream(outputDir + "/Converted_Image" + pageCount + ".jpg")) {
    jpegDevice.process(pdfDocument.getPages().get_Item(pageCount), imageStream);
}
```

#### Συχνά Πιθανά Σφάλματα & Συμβουλές
- **Path correctness:** Ελέγξτε ξανά τις τιμές `dataDir` και `outputDir`; οι σχετικές διαδρομές είναι εύκολο να τοποθετηθούν λανθασμένα.  
- **File permissions:** Βεβαιωθείτε ότι η διαδικασία Java έχει δικαίωμα εγγραφής στον φάκελο εξόδου.  
- **Memory usage:** Για πολύ μεγάλα PDF, σκεφτείτε την επεξεργασία σε μικρότερες παρτίδες ή την αύξηση του μεγέθους heap του JVM.

## Χαρακτηριστικό 2: Μετατροπή Μίας Σελίδας PDF σε Εικόνα JPEG

### Πότε θα “convert single pdf page”;
Μερικές φορές χρειάζεστε μόνο ένα στιγμιότυπο μιας συγκεκριμένης σελίδας—ίσως την εξώφυλλο, ένα γράφημα ή μια σελίδα υπογραφής. Αυτή η λειτουργία δείχνει πώς να στοχεύσετε μια μόνο σελίδα.

### Υλοποίηση Βήμα‑βήμα

**Βήμα 1: Φόρτωση του PDF Εγγράφου** (όπως πριν)

```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

**Βήμα 2: Ρύθμιση Ροής Εξόδου και Παραμέτρων Μετατροπής**  
Επαναχρησιμοποιήστε την ίδια διαμόρφωση `Resolution` και `JpegDevice`.

```java
Resolution resolution = new Resolution(300);
JpegDevice jpegDevice = new JpegDevice(resolution, 100);
```

**Βήμα 3: Μετατροπή της Καθορισμένης Σελίδας σε JPEG**  
Εδώ μετατρέπουμε μόνο την πρώτη σελίδα (αλλάξτε το δείκτη για να στοχεύσετε διαφορετική σελίδα).

```java
try (OutputStream imageStream = new FileOutputStream(outputDir + "/Converted_Image.jpg")) {
    jpegDevice.process(pdfDocument.getPages().get_Item(1), imageStream);
}
```

#### Γρήγορη Λίστα Ελέγχου
- Επαληθεύστε ότι ο αριθμός σελίδας υπάρχει (`pdfDocument.getPages().size()`).  
- Διαχειριστείτε το `IndexOutOfBoundsException` εάν ο δείκτης σελίδας είναι εκτός εύρους.  
- Χρησιμοποιήστε το ίδιο μοτίβο try‑with‑resources για να αποφύγετε διαρροές πόρων.

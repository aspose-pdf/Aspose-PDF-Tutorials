#### Μετατροπή PDF σε HTML με ενσωματωμένες εικόνες PNG

**1. Άνοιγμα του εγγράφου PDF πηγής**

Φορτώστε το έγγραφο PDF στη μνήμη χρησιμοποιώντας το Aspose.PDF:
```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document pdfDocument = new Document(dataDir);
```
Βεβαιωθείτε ότι η διαδρομή σας δείχνει σωστά το σημείο όπου βρίσκεται το PDF σας.

**2. Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML**

Ρυθμίστε τον τρόπο με τον οποίο η μετατροπή θα πρέπει να χειρίζεται εικόνες και διάταξη:
```java
import com.aspose.pdf.HtmlSaveOptions;

HtmlSaveOptions saveOptions = new HtmlSaveOptions();
// Διατηρήστε την οπτική ομοιομορφία με μια σταθερή διάταξη
saveOptions.setFixedLayout(true);
// Ενσωματώστε εικόνες ως PNG στο φόντο της σελίδας αντί για SVG
saveOptions.setRasterImagesSavingMode(HtmlSaveOptions.RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground);
```
Ο `setFixedLayout(true)` διασφαλίζει ότι η HTML αντικατοπτρίζει την οπτική δομή του PDF σας. Η βασική μέθοδος εδώ, `setRasterImagesSavingMode`, κατευθύνει το Aspose.PDF να ενσωματώσει εικόνες ως PNG, αποφεύγοντας τη μορφή SVG η οποία ενδέχεται να μην αποδίδεται με συνέπεια σε όλα τα προγράμματα περιήγησης.

**3. Αποθήκευση του μετατρεπόμενου αρχείου HTML**

Αποθηκεύστε το έγγραφό σας σε μορφή HTML με τις καθορισμένες επιλογές σας:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/converted_file.html";
pdfDocument.save(outputDir, saveOptions);
```
Αυτό αποθηκεύει το αρχείο που έχει μετατραπεί στην επιθυμητή θέση, διατηρώντας τις ενσωματωμένες εικόνες PNG.

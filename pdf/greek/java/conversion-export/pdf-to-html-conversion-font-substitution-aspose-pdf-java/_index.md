## Ρύθμιση του Aspose.PDF για Java

### Εγκατάσταση μέσω Maven

Για να προσθέσετε το Aspose.PDF στο έργο σας χρησιμοποιώντας το Maven, συμπεριλάβετε την ακόλουθη εξάρτηση στο `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

### Εγκατάσταση μέσω Gradle

Για έργα που χρησιμοποιούν Gradle, προσθέστε την ακόλουθη γραμμή στο `build.gradle`:

```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Μετατροπή PDF σε HTML με αντικατάσταση γραμματοσειράς

Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε έγγραφα PDF σε μορφή HTML, αντικαθιστώντας τις μη διαθέσιμες γραμματοσειρές με συγκεκριμένες εναλλακτικές, διασφαλίζοντας οπτική συνέπεια.

#### Βήμα 1: Ρύθμιση καταλόγων και φόρτωση του εγγράφου

Ξεκινήστε καθορίζοντας διαδρομές για το αρχείο PDF εισόδου και το αρχείο HTML εξόδου:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/input1.pdf"; // Εισαγωγή διαδρομής PDF
String outputDir = "YOUR_OUTPUT_DIRECTORY/pdfToHTMLWithFontSubstitution.html"; // Διαδρομή HTML εξόδου

// Φόρτωση του εγγράφου από τον καθορισμένο κατάλογο
Document pdf = new Document(dataDir);
```

#### Βήμα 2: Ρύθμιση παραμέτρων λογικής αντικατάστασης γραμματοσειρών

Δημιουργήστε μια προσαρμοσμένη υποκλάση του `CustomFontSubstitutionBase` για τη διαχείριση αντικαταστάσεων γραμματοσειρών:

```java
CustomSubst1 subst1 = new CustomSubst1();
FontRepository.getSubstitutions().add(subst1);
```

#### Βήμα 3: Προσθήκη ειδοποίησης για συμβάντα αντικατάστασης γραμματοσειράς

Καταχωρίστε έναν χειριστή συμβάντων για να καταγράφει ή να εκτυπώνει λεπτομέρειες σχετικά με την αντικατάσταση γραμματοσειράς:

```java
pdf.FontSubstitution.add(new Document.FontSubstitutionHandler() {
    public void invoke(Font font, Font newFont) {
        System.out.println("Original Font: " + font.getFontName() + ", New Font: " + newFont.getFontName());
    }
});
```

#### Βήμα 4: Μετατροπή PDF σε HTML

Ρυθμίστε το `HtmlSaveOptions` και αποθηκεύστε το έγγραφο ως αρχείο HTML:

```java
HtmlSaveOptions options = new HtmlSaveOptions();
pdf.save(outputDir, options);
```

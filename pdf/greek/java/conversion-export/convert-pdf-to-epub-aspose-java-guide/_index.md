## Οδηγός Εφαρμογής

### Μετατροπή PDF σε μορφή EPUB
Σε αυτήν την ενότητα, θα δούμε πώς να μετατρέπετε ένα έγγραφο PDF σε μορφή EPUB χρησιμοποιώντας το Aspose.PDF για Java.

#### Φόρτωση του εγγράφου PDF
Ξεκινήστε φορτώνοντας το υπάρχον αρχείο PDF σας στο `Document` αντικείμενο. Αυτό είναι κρίσιμο καθώς αποτελεί το αρχικό σας υλικό για μετατροπή:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf"; // Διαδρομή προς το έγγραφο PDF προέλευσης

// Φόρτωση του υπάρχοντος εγγράφου PDF
Document pdfDocument = new Document(dataDir);
```

#### Ρύθμιση παραμέτρων EpubSaveOptions
Στη συνέχεια, δημιουργήστε ένα `EpubSaveOptions` αντικείμενο. Αυτό σας επιτρέπει να ορίσετε διάφορες επιλογές για μετατροπή:

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/convertPDFToEPUBFormat.epub"; // Διαδρομή για την αποθήκευση του αρχείου EPUB που έχει μετατραπεί

// Δημιουργήστε ένα αντίγραφο του αντικειμένου EpubSaveOptions για να καθορίσετε επιλογές μετατροπής
EpubSaveOptions options = new EpubSaveOptions();

// Ορίστε τη λειτουργία αναγνώρισης περιεχομένου σε «Ροή», η οποία προσαρμόζει τη ροή κειμένου για καλύτερη αναγνωσιμότητα σε ηλεκτρονικά αναγνώστες
options.setContentRecognitionMode(EpubSaveOptions.RecognitionMode.Flow);
```

**Γιατί να χρησιμοποιήσετε το `Flow` Τρόπος;**
Ο `Flow` Η λειτουργία είναι επωφελής καθώς αναδιαμορφώνει το κείμενο σε μια μορφή που είναι πιο κατάλληλη για διαφορετικά μεγέθη οθόνης, βελτιώνοντας την αναγνωσιμότητα σε διάφορες συσκευές ηλεκτρονικής ανάγνωσης.

#### Αποθήκευση ως EPUB
Τέλος, αποθηκεύστε το έγγραφό σας σε μορφή EPUB χρησιμοποιώντας τις καθορισμένες επιλογές:

```java
// Αποθηκεύστε το έγγραφο σε μορφή EPUB χρησιμοποιώντας τις καθορισμένες επιλογές αποθήκευσης
doc.save(outputDir, options);
```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι η διαδρομή εισόδου PDF είναι σωστή για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε ότι έχετε δικαιώματα εγγραφής για τον κατάλογο εξόδου.

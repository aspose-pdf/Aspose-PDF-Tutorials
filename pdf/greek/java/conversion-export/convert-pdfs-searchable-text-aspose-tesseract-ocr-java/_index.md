# Πώς να μετατρέψετε PDF σε κείμενο με δυνατότητα αναζήτησης χρησιμοποιώντας Aspose.PDF και Tesseract OCR σε Java

## Εισαγωγή

Αντιμετωπίζετε προκλήσεις κατά την εξαγωγή κειμένου από σαρωμένα έγγραφα PDF ή έγγραφα PDF με εικόνες; Η μετατροπή αυτών των μη αναζητήσιμων αρχείων σε αναζητήσιμες μορφές μπορεί να φέρει επανάσταση στην εξαγωγή δεδομένων, την αρχειοθέτηση και την προσβασιμότητα. Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση του Aspose.PDF για Java σε συνδυασμό με το Tesseract OCR για να επιτύχετε αποτελεσματικά αυτόν τον μετασχηματισμό.

**Τι θα μάθετε:**
- Μετατροπή εικόνων PDF σε κείμενο χρησιμοποιώντας Tesseract OCR σε Java
- Εφαρμόστε τις ισχυρές λειτουργίες του Aspose.PDF για την επεξεργασία εγγράφων

Με αυτές τις δεξιότητες, μπορείτε να αξιοποιήσετε πλήρως τις δυνατότητες των εγγράφων σας. Πριν ξεκινήσετε την υλοποίηση, βεβαιωθείτε ότι έχετε όλα έτοιμα.

## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Μετατροπή OCR χρησιμοποιώντας το Tesseract

Αυτή η λειτουργία δείχνει πώς να μετατρέψετε εικόνες μέσα σε ένα PDF σε κείμενο χρησιμοποιώντας το Tesseract OCR.

#### Βήμα 1: Δημιουργήστε την Επανάκληση για Επεξεργασία OCR

Η συνάρτηση επανάκλησης `CallBackGetHocr` επεξεργάζεται κάθε εικόνα από το PDF:

```java
import com.aspose.pdf.Document;
import com.aspose.pdf.Document.CallBackGetHocr;
import javax.imageio.ImageIO;
import java.awt.image.BufferedImage;
import java.io.File;
import java.io.FileNotFoundException;
import java.io.IOException;
import java.util.Scanner;

CallBackGetHocr cbgh = new CallBackGetHocr() {
    @Override
    public String invoke(BufferedImage img) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        String outputDir = "YOUR_OUTPUT_DIRECTORY";

        File outputfile = new File(dataDir + "/test.jpg");
        try {
            // Εγγραφή της εικόνας που έχει αποθηκευτεί στο buffer ως αρχείο JPEG
            ImageIO.write(img, "jpg", outputfile);
        } catch (IOException e1) {
            e1.printStackTrace();
        }
        
        try {
            // Εκτελέστε το Tesseract OCR για να μετατρέψετε την εικόνα σε κείμενο και να την αποθηκεύσετε σε μορφή HOCR
            java.lang.Process process = Runtime.getRuntime().exec(
                "tesseract " + dataDir + "/test.jpg " + outputDir + "/out hocr"
            );
            process.waitFor();
        } catch (IOException | InterruptedException e) {
            e.printStackTrace();
        }

        // Ανάγνωση του δημιουργημένου αρχείου HTML HOCR σε μια συμβολοσειρά
        File file = new File(outputDir + "/out.html");
        StringBuilder fileContents = new StringBuilder((int) file.length());
        Scanner scanner = null;
        try {
            scanner = new Scanner(file);
            String lineSeparator = System.getProperty("line.separator");
            while (scanner.hasNextLine()) {
                fileContents.append(scanner.nextLine() + lineSeparator);
            }
        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } finally {
            if (scanner != null)
                scanner.close();
        }

        // Διαγραφή προσωρινών αρχείων
        File fileOut = new File(outputDir + "/out.html");
        if (fileOut.exists()) {
            fileOut.delete();
        }
        File fileTest = new File(dataDir + "/test.jpg");
        if (fileTest.exists()) {
            fileTest.delete();
        }
        return fileContents.toString();
    }
};
```

#### Βήμα 2: Μετατροπή PDF σε κείμενο με δυνατότητα αναζήτησης

Φορτώστε το υπάρχον PDF χωρίς δυνατότητα αναζήτησης και μετατρέψτε το χρησιμοποιώντας την επανάκληση:

```java
// Ορισμός διαδρομών καταλόγου κράτησης θέσης
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του υπάρχοντος εγγράφου PDF
Document doc = new Document(dataDir + "/outFile.pdf");

// Μετατρέψτε το έγγραφο χρησιμοποιώντας την επανάκληση για επεξεργασία OCR
doc.convert(cbgh);

// Αποθήκευση του μετατρεπόμενου εγγράφου ως PDF με δυνατότητα αναζήτησης
doc.save(outputDir + "/output971.pdf");
```

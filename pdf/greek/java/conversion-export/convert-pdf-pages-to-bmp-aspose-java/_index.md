## Οδηγός Εφαρμογής

Θα εξερευνήσουμε δύο κύριες εργασίες: τη μετατροπή μιας μεμονωμένης σελίδας PDF και τη μαζική μετατροπή όλων των σελίδων ενός εγγράφου PDF σε εικόνες BMP.

### Μετατροπή μιας μεμονωμένης σελίδας PDF σε εικόνα BMP

#### Επισκόπηση
Εξαγάγετε την πρώτη σελίδα ενός εγγράφου PDF και αποθηκεύστε την ως αρχείο εικόνας BMP υψηλής ανάλυσης. Αυτό είναι χρήσιμο για τη δημιουργία προεπισκοπήσεων ή μικρογραφιών.

#### Βήμα προς βήμα εφαρμογή

**1. Ανοίξτε το έγγραφο PDF**

```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
```
*Γιατί;*: Η φόρτωση του εγγράφου σάς επιτρέπει να έχετε πρόσβαση στις σελίδες του για επεξεργασία.

**2. Δημιουργήστε ένα OutputStream για αποθήκευση εικόνας**

```java
OutputStream imageStream = new FileOutputStream(outputDir + "/Converted_Image.bmp");
```
*Γιατί;*Απαιτείται μια ροή εξόδου για την εγγραφή και αποθήκευση δεδομένων BMP στο δίσκο.

**3. Ορισμός ανάλυσης εξόδου**

```java
Resolution resolution = new Resolution(300);
```
*Γιατί;*Η υψηλότερη ανάλυση εξασφαλίζει καλύτερη ποιότητα εικόνας, κάτι που είναι ζωτικής σημασίας για την ευκρίνεια.

**4. Αρχικοποίηση BmpDevice με ανάλυση**

```java
BmpDevice bmpDevice = new BmpDevice(resolution);
```
*Γιατί;*: Το `BmpDevice` διαχειρίζεται τη μετατροπή χρησιμοποιώντας καθορισμένες ρυθμίσεις ανάλυσης.

**5. Μετατροπή και αποθήκευση της πρώτης σελίδας ως BMP**

```java
bmpDevice.process(pdfDocument.getPages().get_Item(1), imageStream);
imageStream.close();
```
*Γιατί;*Αυτό επεξεργάζεται την πρώτη σελίδα σε μορφή BMP, αποθηκεύοντάς την στο δίσκο. Το κλείσιμο της ροής απελευθερώνει σωστά τους πόρους.

#### Συμβουλές αντιμετώπισης προβλημάτων
- **IOException**Βεβαιωθείτε ότι υπάρχουν διαδρομές αρχείων και ότι έχετε δικαιώματα εγγραφής.
- **Προβλήματα επίλυσης**: Εάν οι εικόνες εμφανίζονται με pixel, δοκιμάστε να αυξήσετε την τιμή ανάλυσης.

### Μετατροπή όλων των σελίδων PDF σε εικόνες BMP

#### Επισκόπηση
Μετατρέψτε κάθε σελίδα ενός εγγράφου PDF σε μεμονωμένα αρχεία BMP. Ιδανικό για μαζική επεξεργασία όπου κάθε σελίδα χρειάζεται ξεχωριστό χειρισμό ή αρχειοθέτηση.

#### Βήμα προς βήμα εφαρμογή

**1. Ανοίξτε το έγγραφο PDF**

```java
Document pdfDocument = new Document(dataDir + "/input.pdf");
```
*Γιατί;*: Αυτό φορτώνει το έγγραφό σας στη μνήμη για επεξεργασία, όπως και πριν.

**2. Επαναλάβετε την επεξεργασία σε κάθε σελίδα**

```java
for (int pageCount = 1; pageCount <= pdfDocument.getPages().size(); pageCount++) {
    OutputStream imageStream = new FileOutputStream(outputDir + "/Converted_Image" + pageCount + ".bmp");
```
*Γιατί;*Η επανάληψη κάθε σελίδας επιτρέπει την επεξεργασία και την αποθήκευσή τους μεμονωμένα.

**3. Ρύθμιση παραμέτρων ανάλυσης και συσκευής Bmp**

```java
Resolution resolution = new Resolution(300);
BmpDevice bmpDevice = new BmpDevice(resolution);
```
*Γιατί;*Η ρύθμιση μιας ομοιόμορφης ανάλυσης διασφαλίζει σταθερή ποιότητα σε όλες τις σελίδες.

**4. Επεξεργαστείτε κάθε σελίδα σε μορφή BMP**

```java
bmpDevice.process(pdfDocument.getPages().get_Item(pageCount), imageStream);
imageStream.close();
}
```
*Γιατί;*Αυτός ο βρόχος μετατρέπει κάθε σελίδα σε ξεχωριστό αρχείο BMP, κλείνοντας τις ροές μετά την εγγραφή για την αποφυγή διαρροών πόρων.

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Αντικατάσταση αρχείου**: Βεβαιωθείτε ότι υπάρχουν μοναδικά ονόματα αρχείων για κάθε έξοδο.
- **Προβλήματα απόδοσης**Εξετάστε το ενδεχόμενο επεξεργασίας σε παρτίδες ή αύξησης των πόρων του συστήματος εάν χειρίζεστε μεγάλα έγγραφα.

- Μέθοδοι συμπίεσης εικόνων μέσα σε αρχεία PDF, βελτιώνοντας τους χρόνους φόρτωσης.

## Οδηγός Εφαρμογής

### Αφαίρεση περιττών αντικειμένων

**Επισκόπηση:** Αυτή η λειτουργία αποσυμφορεί το PDF σας αφαιρώντας αχρησιμοποίητα αντικείμενα και υποδιαιρώντας γραμματοσειρές, μειώνοντας έτσι το μέγεθος του αρχείου.

#### Βήμα 1: Φόρτωση του εγγράφου PDF
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.optimization.OptimizationOptions;

String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή εγγράφου
Document pdfDocument = new Document(dataDir + "/YourPDFFile.pdf");
```

#### Βήμα 2: Διαμόρφωση επιλογών βελτιστοποίησης
```java
OptimizationOptions optimizeOptions = new OptimizationOptions();
optimizeOptions.setRemoveUnusedObjects(true);
optimizeOptions.setSubsetFonts(true);
```
- **`setRemoveUnusedObjects(boolean)`:** Αφαιρεί τυχόν αντικείμενα που δεν χρησιμοποιούνται στο έγγραφο.
- **`setSubsetFonts(boolean)`:** Μειώνει τα δεδομένα γραμματοσειράς συμπεριλαμβάνοντας μόνο τους απαραίτητους χαρακτήρες.

#### Βήμα 3: Βελτιστοποίηση και Αποθήκευση
```java
pdfDocument.optimizeResources(optimizeOptions);
pdfDocument.save("YOUR_OUTPUT_DIRECTORY/removeUnnecessaryObjects_out.pdf");
```

### Συμπίεση PDF με εικόνες

**Επισκόπηση:** Αυτή η λειτουργία μειώνει το μέγεθος των εικόνων σε ένα PDF, επιταχύνοντας τη φόρτωσή του χωρίς να επηρεάζεται σημαντικά η ποιότητα.

#### Βήμα 1: Φόρτωση και προετοιμασία
```java
Document doc = new Document(dataDir + "/YourPDFFile.pdf");

OptimizationOptions opt = new OptimizationOptions();
opt.setRemoveUnusedObjects(false); // Διατήρηση αντικειμένων για αυτήν τη λειτουργία
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών συμπίεσης εικόνας
```java
opt.getImageCompressionOptions().setCompressImages(true);
opt.getImageCompressionOptions().setImageQuality(10); // Χαμηλότερη τιμή σημαίνει υψηλότερη συμπίεση
```
- **`setCompressImages(boolean)`:** Ενεργοποιεί τη συμπίεση εικόνας.
- **`setImageQuality(int)`:** Ορίζει το επίπεδο ποιότητας των εικόνων (1-100).

#### Βήμα 3: Εφαρμογή Βελτιστοποίησης και Αποθήκευση
```java
doc.optimizeResources(opt);
doc.save("YOUR_OUTPUT_DIRECTORY/compressingPDFWithImages_out.pdf");
```

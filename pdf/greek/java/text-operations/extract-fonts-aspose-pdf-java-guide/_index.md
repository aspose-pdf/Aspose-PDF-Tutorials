## Οδηγός Εφαρμογής

### Αρχικοποίηση και φόρτωση του εγγράφου PDF
#### Επισκόπηση
Το πρώτο βήμα είναι να αρχικοποιήσετε ένα αντικείμενο εγγράφου χρησιμοποιώντας το Aspose.PDF φορτώνοντας ένα υπάρχον αρχείο PDF στην εφαρμογή Java που χρησιμοποιείτε.

#### Βήματα:
**1. Δημιουργήστε τον κατάλογό σας**
Ξεκινήστε ορίζοντας τον κατάλογο όπου αποθηκεύονται τα αρχεία PDF:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String inputFilePath = dataDir + "/input.pdf";
```

**2. Τοποθετήστε το έγγραφο**
Δημιουργήστε ένα `Document` αντικείμενο από το αρχείο PDF σας:
```java
import com.aspose.pdf.Document;
// Αρχικοποίηση νέου αντικειμένου εγγράφου από ένα υπάρχον αρχείο
Document doc = new Document(inputFilePath);
```
Αυτό το βήμα προετοιμάζει το έγγραφό σας για περαιτέρω λειτουργίες, όπως η ανάκτηση γραμματοσειρών.

### Ανάκτηση όλων των γραμματοσειρών από το έγγραφο PDF
#### Επισκόπηση
Μόλις φορτωθεί το έγγραφο, μπορείτε να εξαγάγετε όλες τις πληροφορίες γραμματοσειράς χρησιμοποιώντας τα βοηθητικά προγράμματα γραμματοσειρών του Aspose.PDF για να λάβετε μια λεπτομερή λίστα με τις γραμματοσειρές που χρησιμοποιούνται στο έγγραφο.

#### Βήματα:
**1. Πρόσβαση σε βοηθητικά προγράμματα γραμματοσειρών**
Χρησιμοποιήστε το `getFontUtilities()` μέθοδος στο αντικείμενο του εγγράφου σας:
```java

```

**2. Επανάληψη και εξαγωγή λεπτομερειών γραμματοσειράς**
Πραγματοποιήστε επανάληψη σε κάθε γραμματοσειρά για να συγκεντρώσετε λεπτομέρειες, όπως ονόματα γραμματοσειρών:
```java
for (com.aspose.pdf.Font f : fonts) {
    String fontName = f.getFontName(); // Πρόσβαση στο όνομα κάθε γραμματοσειράς
    // Εκτελέστε λειτουργίες με αυτές τις πληροφορίες, όπως καταγραφή ή ανάλυση
}
```

## Οδηγός Εφαρμογής

### Εξαγωγή κειμένου από μια συγκεκριμένη σελίδα

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε κείμενο από μια συγκεκριμένη σελίδα στο έγγραφο PDF σας. Είναι ιδανική όταν χρειάζεστε πληροφορίες μόνο από μία ενότητα του εγγράφου.

#### Βήματα:

##### Ρύθμιση εγγράφου και συσκευής
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.Page;
import com.aspose.pdf.TextExtractionOptions;
import com.aspose.pdf.devices.TextDevice;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

Document pdfDocument = new Document(dataDir + "/input.pdf");
TextDevice textDevice = new TextDevice();
```

##### Ρύθμιση παραμέτρων επιλογών εξαγωγής κειμένου
```java
TextExtractionOptions textExtOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Raw);
textDevice.setExtractionOptions(textExtOptions);
```
Εδώ, ορίζουμε τη λειτουργία εξαγωγής σε `Raw`, το οποίο εξάγει κείμενο χωρίς μορφοποίηση.

##### Επεξεργασία της συγκεκριμένης σελίδας
```java
// Εξαγωγή κειμένου από την πρώτη σελίδα του εγγράφου PDF
textDevice.process(pdfDocument.getPages().get_Item(1), outputDir + "/ExtractedText.txt");
```
Ο `process` Η μέθοδος στοχεύει σε μια συγκεκριμένη σελίδα και εξάγει το κείμενό της σε ένα αρχείο.

#### Συμβουλή αντιμετώπισης προβλημάτων:
- **Το αρχείο δεν βρέθηκε:** Βεβαιωθείτε ότι οι διαδρομές καταλόγου σας είναι σωστές και προσβάσιμες.

### Εξαγωγή κειμένου από όλες τις σελίδες ενός PDF

**Επισκόπηση:** Αυτή η λειτουργία εξάγει κείμενο από κάθε σελίδα του εγγράφου, κάτι που είναι χρήσιμο για ολοκληρωμένες εργασίες ανάλυσης ή εξαγωγής δεδομένων.

#### Βήματα:

##### Αρχικοποίηση ροής εξόδου
```java
import java.io.FileOutputStream;
import java.io.OutputStream;

OutputStream text_stream = new FileOutputStream(outputDir + "/ExtractedText.txt", false);
```
Η ροή εξόδου γράφει όλα τα εξαγόμενα κείμενα σε ένα μόνο αρχείο.

##### Επανάληψη σε όλες τις σελίδες
```java
for (Page page : (Iterable<Page>) pdfDocument.getPages()) {
    TextDevice textDevice = new TextDevice();
    TextExtractionOptions textExtOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Raw);
    textDevice.setExtractionOptions(textExtOptions);

    // Εξαγωγή κειμένου από κάθε σελίδα
    textDevice.process(page, text_stream);
}
```
Επαναλάβετε όλες τις σελίδες και εξαγάγετε κείμενο χρησιμοποιώντας τις ίδιες ρυθμίσεις όπως πριν.

##### Κλείσιμο της ροής εξόδου
```java
text_stream.close();
```
Να κλείνετε πάντα τις ροές σε ελεύθερους πόρους μόλις ολοκληρωθεί η επεξεργασία.

#### Συμβουλή αντιμετώπισης προβλημάτων:
- **Προβλήματα μνήμης:** Για μεγάλα PDF, εξετάστε το ενδεχόμενο εξαγωγής μικρότερων ενοτήτων ή βελτιστοποίησης της χρήσης μνήμης στο περιβάλλον σας.

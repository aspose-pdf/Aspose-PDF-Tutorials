Ξεκινήστε φορτώνοντας το έγγραφο PDF χρησιμοποιώντας το Aspose.PDF:
```java
import com.aspose.pdf.*;

Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

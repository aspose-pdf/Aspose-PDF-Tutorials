## Οδηγός Εφαρμογής

Τώρα, ας δούμε τη διαδικασία υλοποίησης βήμα προς βήμα.

### Δημιουργία σφραγίδας κειμένου στο κέντρο σε PDF

**1. Ανοίξτε ένα υπάρχον έγγραφο PDF**

Αρχικά, φορτώνουμε το υπάρχον έγγραφο PDF χρησιμοποιώντας το `Document` τάξη.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/input.pdf");
```

**2. Ρύθμιση παραμέτρων αντικειμένου FormattedText**

Δημιουργούμε ένα `FormattedText` αντικείμενο για να ορίσουμε το περιεχόμενο του κειμένου μας και τη δομή του.
```java
FormattedText text = new FormattedText("This");
text.addNewLineText("is sample");
text.addNewLineText("Center Aligned");
text.addNewLineText("TextStamp");
text.addNewLineText("Object");
```
- **Γιατί αυτό το βήμα;** Χρησιμοποιούμε `FormattedText` για να δημιουργήσετε ένα μπλοκ κειμένου με πλούσια μορφοποίηση που μπορεί να προσαρμοστεί με διάφορα χαρακτηριστικά.

**3. Δημιουργία και ρύθμιση παραμέτρων αντικειμένου TextStamp**

Στη συνέχεια, δημιουργούμε ένα παράδειγμα του `TextStamp` αντικείμενο χρησιμοποιώντας το διαμορφωμένο μας `FormattedText`.
```java
TextStamp stamp = new TextStamp(text);
```

**4. Ορισμός ιδιοτήτων ευθυγράμμισης**

Εδώ είναι που συμβαίνει η μαγεία: ο καθορισμός ευθυγραμμίσεων.
```java
stamp.setHorizontalAlignment(HorizontalAlignment.Center);
stamp.setVerticalAlignment(VerticalAlignment.Center);
stamp.setTextAlignment(HorizontalAlignment.Center);
```
- **Παράμετροι:** Αυτές οι μέθοδοι προσαρμόζουν τον τρόπο τοποθέτησης του κειμένου μέσα στη σφραγίδα και στη σελίδα.

**5. Ορισμός περιθωρίων**

Καθορίζουμε ένα επάνω περιθώριο για να βελτιώσουμε τη θέση της σφραγίδας κειμένου μας.
```java
stamp.setTopMargin(20);
```

**6. Προσθήκη σφραγίδας στη σελίδα PDF**

Τέλος, προσθέτουμε τη διαμορφωμένη σφραγίδα στην επιθυμητή σελίδα του εγγράφου.
```java
pdfDocument.getPages().get_Item(1).addStamp(stamp);
```
- **Σημείωμα:** Μπορείτε να στοχεύσετε οποιαδήποτε σελίδα αλλάζοντας `get_Item(1)`.

**7. Αποθήκευση του τροποποιημένου εγγράφου**

Αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο.
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
pdfDocument.save(outputDir + "/TextStamp_output.pdf");
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Συνηθισμένο πρόβλημα:** Το κείμενο δεν είναι σωστά κεντραρισμένο; Ελέγξτε ξανά τις ρυθμίσεις ευθυγράμμισης.
- **Συμβουλή απόδοσης:** Για μεγάλα έγγραφα, εξετάστε το ενδεχόμενο επεξεργασίας σε παρτίδες για αποτελεσματική διαχείριση της χρήσης μνήμης.

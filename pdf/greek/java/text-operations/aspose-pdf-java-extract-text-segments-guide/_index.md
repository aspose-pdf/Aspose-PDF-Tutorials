## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην υλοποίηση της εξαγωγής τμημάτων κειμένου χρησιμοποιώντας το Aspose.PDF για Java.

### Αναζήτηση και εξαγωγή τμημάτων κειμένου (H2)

Ας αναλύσουμε την υλοποίηση της λειτουργίας σε λογικά βήματα:

#### Βήμα 1: Φόρτωση του εγγράφου PDF

Ξεκινήστε φορτώνοντας το έγγραφο PDF. Αυτό γίνεται δημιουργώντας μια παρουσία του `Document` κλάση, παρέχοντάς της τη διαδρομή προς το αρχείο σας.

```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String inputFilePath = dataDir + "/input.pdf";

// Φόρτωση του εγγράφου PDF
Document pdfDocument = new Document(inputFilePath);
```

#### Βήμα 2: Δημιουργήστε ένα TextFragmentAbsorber

Ο `TextFragmentAbsorber` Η κλάση χρησιμοποιείται για την αναζήτηση συγκεκριμένου κειμένου μέσα στο έγγραφό σας. Θα χρειαστεί να δημιουργήσετε μια παρουσία αυτής της κλάσης, καθορίζοντας το κείμενο που αναζητάτε.

```java
import com.aspose.pdf.TextFragmentAbsorber;

// Δημιουργήστε ένα αντικείμενο TextFragmentAbsorber με τη λέξη-κλειδί αναζήτησης 'sample'
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("sample");
```

#### Βήμα 3: Εφαρμόστε το Absorber σε όλες τις σελίδες

Χρησιμοποιήστε το `accept` σε όλες τις σελίδες του εγγράφου σας. Αυτό το βήμα εντοπίζει εμφανίσεις του καθορισμένου κειμένου σε ολόκληρο το έγγραφο.

```java
// Εφαρμόστε το απορροφητικό σε όλες τις σελίδες
pdfDocument.getPages().accept(textFragmentAbsorber);
```

#### Βήμα 4: Ανάκτηση και επανάληψη τμημάτων κειμένου

Αφού εφαρμόσετε τον απορροφητή, ανακτήστε τη συλλογή των θραυσμάτων κειμένου που βρέθηκαν. Επαναλάβετε κάθε θραύσμα και τα τμήματά του για να εξαγάγετε λεπτομερείς πληροφορίες.

```java
import com.aspose.pdf.TextFragmentCollection;
import com.aspose.pdf.TextFragment;
import com.aspose.pdf.TextSegment;

// Αποκτήστε τη συλλογή των θραυσμάτων κειμένου που βρέθηκαν
textFragmentCollection = textFragmentAbsorber.getTextFragments();

// Επαναλάβετε κάθε TextFragment και τα τμήματά του για να εξαγάγετε λεπτομέρειες
for (TextFragment textFragment : (Iterable<TextFragment>) textFragmentCollection) {
    for (TextSegment textSegment : (Iterable<TextSegment>) textFragment.getSegments()) {
        // Εξαγωγή διαφόρων ιδιοτήτων από κάθε τμήμα κειμένου
        String text = textSegment.getText();
        Object position = textSegment.getPosition();
        
        double xIndent = position != null ? ((com.aspose.pdf.Point)position).getXIndent() : 0;
        double yIndent = position != null ? ((com.aspose.pdf.Point)position).getYIndent() : 0;
        
        String fontName = textSegment.getTextState().getFont().getFontName();
        boolean isAccessible = textSegment.getTextState().getFont().isAccessible();
        boolean isEmbedded = textSegment.getTextState().getFont().isEmbedded();
        boolean isSubset = textSegment.getTextState().getFont().isSubset();
        
        double fontSize = textSegment.getTextState().getFontSize();
        String foregroundColor = textSegment.getTextState().getForegroundColor() != null ? textSegment.getTextState().getForegroundColor().toString() : "null";
        
        // Παράδειγμα: Λεπτομέρειες εξαγωγής που εξάγονται
        System.out.println("Text: " + text);
        System.out.println("XIndent: " + xIndent);
    }
}
```

## Οδηγός Εφαρμογής

Αφού ολοκληρωθεί η εγκατάσταση, ας προσθέσουμε συνδέσμους JavaScript σε ένα PDF.

### Δημιουργία και σύνδεση εγγράφου PDF

Αρχικά, δημιουργήστε μια παρουσία του `PdfContentEditor` και συνδέστε το με το αρχείο PDF σας:
```java
import com.aspose.pdf.facades.PdfContentEditor;

// Δημιουργήστε μια παρουσία του PdfContentEditor
PdfContentEditor editor = new PdfContentEditor();

// Σύνδεση του προγράμματος επεξεργασίας σε ένα υπάρχον έγγραφο PDF
editor.bindPdf("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
**Εξήγηση:** Ο `bindPdf` η μέθοδος συσχετίζει το αρχείο PDF σας με το `PdfContentEditor`, επιτρέποντας τροποποιήσεις.

### Δημιουργία συνδέσμου JavaScript σε έγγραφο PDF

Στη συνέχεια, ορίστε μια περιοχή με δυνατότητα κλικ που εκτελεί μια ενέργεια JavaScript. Αυτό περιλαμβάνει τη δημιουργία ενός ορθογωνίου και τη σύνδεσή του με ένα απόσπασμα κώδικα JavaScript:
```java
import java.awt.Rectangle;
import java.awt.Color;

// Ορίστε ένα ορθογώνιο με δυνατότητα κλικ μέσα στο έγγραφο
Rectangle rect = new Rectangle(50, 50, 200, 200);

// Ορισμός χρώματος υπερσυνδέσμου (πράσινο)
Color linkColor = new Color(0, 255, 0); 

// Ενέργεια JavaScript που θα εκτελεστεί όταν κάνετε κλικ
String code = "app.alert('Welcome to Aspose!');";

// Προσθήκη συνδέσμου JavaScript στην πρώτη σελίδα του εγγράφου PDF
editor.createJavaScriptLink(code, rect, 1, linkColor);
```
**Βασικά σημεία:**
- `Rectangle` καθορίζει την περιοχή με δυνατότητα κλικ.
- Ο `Color` Το αντικείμενο ορίζει την οπτική εμφάνιση του υπερσυνδέσμου.
- `createJavaScriptLink` συνδέει τον κώδικα JavaScript σας με αυτό το ορθογώνιο σε μια συγκεκριμένη σελίδα.

### Αποθήκευση αλλαγών σε ένα έγγραφο PDF

Αποθηκεύστε το τροποποιημένο έγγραφο με τις νέες διαδραστικές λειτουργίες του:
```java

editor.save("YOUR_OUTPUT_DIRECTORY/JavaScriptAdded_output.pdf");
```
**Εξήγηση:** Αυτό το βήμα εγγράφει όλες τις τροποποιήσεις πίσω στον δίσκο, διασφαλίζοντας ότι τα διαδραστικά σας στοιχεία διατηρούνται.

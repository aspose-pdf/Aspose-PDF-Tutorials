## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Άνοιγμα εγγράφου PDF
Για να χειριστείτε ένα αρχείο PDF, ανοίξτε το χρησιμοποιώντας το Aspose.PDF για Java:
```java
import com.aspose.pdf.Document;
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
document = new Document(dataDir + "/input.pdf");
```
- **Παράμετροι**: Το `dataDir` Η μεταβλητή θα πρέπει να δείχνει την τοποθεσία του εγγράφου σας.
- **Σκοπός**: Φόρτωση του PDF σε ένα `Document` Το αντικείμενο επιτρέπει περαιτέρω χειρισμό.

### Λειτουργία 2: Δημιουργία και ρύθμιση παραμέτρων γονικού σελιδοδείκτη
Η δημιουργία σελιδοδεικτών βοηθά στην αποτελεσματική οργάνωση του περιεχομένου. Δείτε πώς:
```java
import com.aspose.pdf.OutlineItemCollection;
import com.aspose.pdf.GoToAction;
OutlineItemCollection pdfOutline = new OutlineItemCollection(document.getOutlines());
pdfOutline.setTitle("Parent Outline");
pdfOutline.setItalic(true);
pdfOutline.setBold(true);
pdfOutline.setDestination(new GoToAction(document.getPages().get_Item(2)));
```
- **Παράμετροι**: 
  - `getTitle()`: Ορίζει τον τίτλο του σελιδοδείκτη.
  - `setItalic()` & `setBold()`: Εφαρμόζει στυλ κειμένου.
  - `setDestination()`: Κατευθύνει τον σελιδοδείκτη σε μια συγκεκριμένη σελίδα.

### Λειτουργία 3: Δημιουργία και ρύθμιση παραμέτρων θυγατρικού σελιδοδείκτη
Βελτιώστε την πλοήγηση προσθέτοντας θυγατρικούς σελιδοδείκτες:
```java
import com.aspose.pdf.OutlineItemCollection;
import com.aspose.pdf.GoToAction;
OutlineItemCollection pdfChildOutline = new OutlineItemCollection(document.getOutlines());
pdfChildOutline.setTitle("Child Outline");
pdfChildOutline.setItalic(true);
pdfChildOutline.setBold(true);
pdfChildOutline.setDestination(new GoToAction(document.getPages().get_Item(2)));
```

### Χαρακτηριστικό 4: Προσθήκη παιδικού σελιδοδείκτη στον γονικό σελιδοδείκτη
Οργανώστε τους σελιδοδείκτες τοποθετώντας τους θυγατρικούς σελιδοδείκτες κάτω από τους γονικούς:
```java
document.getOutlines().add(pdfOutline); // Βεβαιωθείτε ότι το pdfOutline είναι το γονικό αρχείο
pdfOutline.add(pdfChildOutline);
```
- **Σκοπός**Αυτή η μέθοδος οργανώνει τους σελιδοδείκτες ιεραρχικά για καλύτερη πλοήγηση.

### Λειτουργία 5: Προσθήκη γονικού σελιδοδείκτη στο έγγραφο και αποθήκευση
Αποθηκεύστε το PDF με τις ενημερωμένες διαμορφώσεις σελιδοδεικτών:
```java
import com.aspose.pdf.Document;
String outputDir = "YOUR_OUTPUT_DIRECTORY";
document.getOutlines().add(pdfOutline);
document.save(outputDir + "/PDF_with_ChildBookmarks.pdf");
```
- **Σκοπός**: Διασφαλίζει ότι οι αλλαγές διατηρούνται αποθηκεύοντας το έγγραφο με νέο όνομα αρχείου.

## Οδηγός Υλοποίησης
### Εξαγωγή Σελιδοδεικτών από Ένα Έγγραφο PDF
Αυτή η ενότητα σας καθοδηγεί στην ανάκτηση σελιδοδεικτών χρησιμοποιώντας το Aspose.PDF για Java.

#### Άνοιγμα του Εγγράφου PDF
Φορτώστε το έγγραφο PDF σας:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf"; // Path to the input PDF file
Document pdfDocument = new Document(dataDir);
```

#### Επανάληψη Μέσω των Σελιδοδεικτών
Επανάληψη μέσω όλων των σελιδοδεικτών στο έγγραφο:
```java
for (OutlineItemCollection outlineItem : pdfDocument.getOutlines()) {
    // Access bookmark details
}
```

#### Εμφάνιση Λεπτομερειών Σελιδοδείκτη
Για κάθε σελιδοδείκτη, εμφανίστε τον τίτλο και τα χαρακτηριστικά του όπως πλάγιο, έντονο και χρώμα:
```java
String title = outlineItem.getTitle();
boolean isItalic = outlineItem.isItalic();  // Corrected method call for italic check
boolean isBold = outlineItem.getBold();     // Check if the text is bold
int color = outlineItem.getColor().getKnownColor();

// Example output
System.out.println("Title: " + title);
System.out.println("Is Italic: " + isItalic);
System.out.println("Is Bold: " + isBold);
System.out.println("Color: " + color);
```
**Επεξήγηση**: Η μέθοδος `getTitle()` ανακτά το όνομα του σελιδοδείκτη, ενώ οι `isItalic()` και `getBold()` ελέγχουν το στυλ κειμένου. Η `getColor()` παρέχει πληροφορίες χρώματος.

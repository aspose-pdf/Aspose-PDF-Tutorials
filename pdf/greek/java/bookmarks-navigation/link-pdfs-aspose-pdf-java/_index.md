## Οδηγός Εφαρμογής
### Δημιουργία συνδέσμου προς άλλο έγγραφο PDF
Αυτή η ενότητα περιγράφει τη δημιουργία ενός συνδέσμου σε ένα έγγραφο PDF που οδηγεί σε ένα άλλο αρχείο PDF.

#### Επισκόπηση
Θα δημιουργήσουμε έναν διαδραστικό υπερσύνδεσμο μέσα σε ένα PDF, κατευθύνοντας τους χρήστες σε μια συγκεκριμένη σελίδα ενός άλλου εγγράφου PDF. Αυτό είναι ιδιαίτερα χρήσιμο για την αναφορά σε σχετικά έγγραφα ή ενότητες σε μεγάλες αναφορές.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποίηση του εγγράφου**

```java
Document pdfDocument = new Document();
pdfDocument.getPages().add();
```

Εδώ, ξεκινάμε δημιουργώντας ένα κενό `Document` αντικείμενο και προσθέτοντας μια σελίδα σε αυτό. Εδώ θα βρίσκεται ο υπερσύνδεσμός μας.

**2. Ορισμός τοποθεσίας συνδέσμου**

```java
Rectangle rect = new Rectangle(100, 100, 110, 110);
LinkAnnotation link = new LinkAnnotation(pdfDocument.getPages().get_Item(1), rect);
```

Ορίζουμε ένα `Rectangle` που καθορίζει την περιοχή στη σελίδα όπου θα τοποθετηθεί ο υπερσύνδεσμός μας. `LinkAnnotation` δημιουργείται στη συνέχεια με αυτό το ορθογώνιο ως όριο.

**3. Ορισμός ιδιοτήτων συνδέσμου**

```java
link.setColor(Color.fromRgb(java.awt.Color.green));
link.setAction(new GoToRemoteAction("YOUR_DOCUMENT_DIRECTORY/SampleDataTable.pdf", 1));
```

Το χρώμα του συνδέσμου έχει οριστεί σε πράσινο για ορατότητα και το `GoToRemoteAction` καθορίζει το έγγραφο PDF προορισμού (`SampleDataTable.pdf`) και τον αριθμό σελίδας (1) στον οποίο θα πρέπει να μεταβεί.

**4. Προσθήκη σχολίου συνδέσμου**

```java
pdfDocument.getPages().get_Item(1).getAnnotations().add(link);
```

Τέλος, προσθέστε την σχολίαση συνδέσμου στην πρώτη σελίδα του εγγράφου σας.

**5. Αποθήκευση του εγγράφου**

```java
pdfDocument.save("YOUR_OUTPUT_DIRECTORY/Hyperlink_to_PDF.pdf");
```

Αποθηκεύστε το τροποποιημένο PDF με τη νέα υπερσύνδεση στη θέση της.

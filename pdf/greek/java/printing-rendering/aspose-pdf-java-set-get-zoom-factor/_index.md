## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία σε δύο κύρια χαρακτηριστικά: τον ορισμό και τη λήψη του συντελεστή ζουμ σε ένα έγγραφο PDF χρησιμοποιώντας Java.

### Ρύθμιση συντελεστή ζουμ

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε πόσο μεγάλο ή μικρό θα εμφανίζεται το περιεχόμενο όταν ανοίγει το PDF σας, βελτιώνοντας την εμπειρία χρήστη από την αρχή.

#### Βήμα προς βήμα εφαρμογή:
**1. Εισαγωγή απαραίτητων πακέτων**
Ξεκινήστε εισάγοντας τις απαραίτητες κλάσεις:
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.XYZExplicitDestination;
import com.aspose.pdf.GoToAction;
```

**2. Ορισμός διαδρομών αρχείων και επιπέδου ζουμ**
Ρυθμίστε τους καταλόγους εγγράφων σας και τον επιθυμητό συντελεστή ζουμ.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";
double zoom = 0.5; // Παράδειγμα: Επίπεδο ζουμ 50%
```

**3. Φορτώστε το έγγραφο PDF**
Δημιουργήστε ένα `Document` αντικείμενο για να φορτώσετε το υπάρχον αρχείο PDF που θέλετε να τροποποιήσετε.
```java
Document doc = new Document(dataDir + "/HelloWorld.pdf");
```

**4. Δημιουργήστε XYZExplicitDestination**
Ορίστε τον προορισμό με το επιθυμητό επίπεδο ζουμ, πλάτος και ύψος με βάση τις διαστάσεις του πλαισίου πολυμέσων της πρώτης σελίδας.
```java
XYZExplicitDestination destination = new XYZExplicitDestination(
    doc.getPages().get_Item(1),
    doc.getPages().get_Item(1).getMediaBox().getWidth(),
    doc.getPages().get_Item(1).getMediaBox().getHeight(), 
    zoom);
```

**5. Αντιστοιχίστε το ζουμ ως ανοιχτή ενέργεια**
Δημιουργήστε ένα `GoToAction` χρησιμοποιώντας τον προορισμό σας και ορίστε τον ως ενέργεια ανοίγματος του εγγράφου.
```java
GoToAction actionzoom = new GoToAction(destination);
doc.setOpenAction(actionzoom);
```

**6. Αποθηκεύστε τις αλλαγές σας**
Τέλος, αποθηκεύστε το τροποποιημένο PDF για να εφαρμόσετε τη ρύθμιση ζουμ.
```java
String outputPath = outputDir + "/Zoomed_actionzoom.pdf";
doc.save(outputPath);
```

### Ανάκτηση συντελεστή ζουμ

#### Επισκόπηση
Η πρόσβαση στο τρέχον επίπεδο ζουμ που έχει οριστεί σε ένα PDF είναι ζωτικής σημασίας για την κατανόηση του τρόπου με τον οποίο παρουσιάζεται το έγγραφό σας κατά το άνοιγμα.

#### Βήμα προς βήμα εφαρμογή:
**1. Φόρτωση του τροποποιημένου εγγράφου**
Ξεκινήστε φορτώνοντας το προηγουμένως αποθηκευμένο έγγραφο.
```java
Document doc1 = new Document(outputDir + "/Zoomed_actionzoom.pdf");
```

**2. Πρόσβαση στην Άνοιγμα Ενέργειας και Ανάκτηση Επιπέδου Ζουμ**
Εξαγωγή του `GoToAction` από την ανοιχτή ενέργεια και, στη συνέχεια, λάβετε το επίπεδο ζουμ από `XYZExplicitDestination`.
```java
GoToAction action = (GoToAction) doc1.getOpenAction();
XYZExplicitDestination destination = (XYZExplicitDestination) action.getDestination();
double retrievedZoom = destination.getZoom();
```
Η μεταβλητή `retrievedZoom` διατηρεί πλέον τον τρέχοντα συντελεστή ζουμ του εγγράφου σας.

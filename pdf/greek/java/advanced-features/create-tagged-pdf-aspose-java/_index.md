## Οδηγός Υλοποίησης

### Πώς να δημιουργήσετε ετικετοποιημένο PDF Java – Βήμα 1: Δημιουργία και διαμόρφωση του εγγράφου

Φορτώστε μια νέα παρουσία `Document`, ενεργοποιήστε την ετικετοποίηση και ορίστε βασικές ιδιότητες εγγράφου όπως τίτλο και γλώσσα. Αυτό το βήμα δημιουργεί το δοχείο λογικής δομής στο οποίο θα ανήκουν όλες οι επόμενες ετικέτες.

#### Επισκόπηση
`Document` είναι η κύρια κλάση του Aspose.PDF που αντιπροσωπεύει ένα έγγραφο PDF στη μνήμη.  
Πρώτα, δημιουργούμε ένα αντικείμενο `Document` και ανακτούμε τη διεπαφή **tagged content**. Αυτή η διεπαφή μας επιτρέπει να χειριζόμαστε τη λογική δομή.  
Η κλάση `Document` είναι το ανώτερο αντικείμενο του Aspose.PDF που αντιπροσωπεύει ένα μοναδικό αρχείο PDF στη μνήμη. Μετά τη δημιουργία, όλες οι λειτουργίες ανάγνωσης και εγγραφής περνούν από αυτό το αντικείμενο.

```java
import com.aspose.pdf.Document;

String dataDir = "YOUR_DOCUMENT_DIRECTORY"; 
Document document = new Document();
```

```java
import com.aspose.pdf.tagged.ITaggedContent;

ITaggedContent taggedContent = document.getTaggedContent();
```

```java
String title = "Tagged Pdf Document";
String language = "en-US";

taggedContent.setTitle(title);
taggedContent.setLanguage(language);
```

### Πώς να δημιουργήσετε ετικετοποιημένο PDF Java – Βήμα 2: Προσθήκη στοιχείων παραγράφου με προσαρμοσμένες ετικέτες

Δημιουργήστε ένα δοχείο `SectElement` και γεμίστε το με αντικείμενα `ParagraphElement`, το καθένα λαμβάνοντας μια προσαρμοσμένη ετικέτα για ευκολότερη ταυτοποίηση και μετέπειτα εξαγωγή.

#### Επισκόπηση
`SectElement` αντιπροσωπεύει μια λογική ενότητα στο δέντρο ετικετών του PDF.  
`ParagraphElement` ορίζει ένα μπλοκ κειμένου μέσα σε μια ενότητα.  
Δημιουργούμε μια **ενότητα** (`SectElement`) που λειτουργεί ως δοχείο, έπειτα προσθέτουμε αρκετές **παράγραφους** (`ParagraphElement`) μέσα σε αυτήν. Κάθε παράγραφος λαμβάνει μια προσαρμοσμένη ετικέτα για ευκολότερη ταυτοποίηση.

```java
import com.aspose.pdf.tagged.logicalstructure.elements.bls.ParagraphElement;
import com.aspose.pdf.tagged.logicalstructure.elements.grouping.SectElement;

SectElement sect = taggedContent.createSectElement();
taggedContent.getRootElement().appendChild(sect);

ParagraphElement p1 = taggedContent.createParagraphElement();
p1.setText("P1. ");
p1.setTag("P1");
// Repeat for other paragraphs...
```

```java
sect.appendChild(p1);
// Append other paragraphs similarly...
```

### Πώς να δημιουργήσετε ετικετοποιημένο PDF Java – Βήμα 3: Προσθήκη στοιχείων span με προσαρμοσμένες ετικέτες

Εισάγετε αντικείμενα `SpanElement` μέσα σε παραγράφους για να ετικετοποιήσετε τμήματα κειμένου εντός γραμμής, παρέχοντάς σας λεπτομερή έλεγχο του στυλ και των μεταδεδομένων προσβασιμότητας.

#### Επισκόπηση
`SpanElement` ετικετοποιεί ένα τμήμα κειμένου εντός μιας παραγράφου.  
**Spans** χωρίζουν τις παραγράφους σε πιο λεπτές μονάδες, επιτρέποντάς μας να ετικετοποιήσουμε τμήματα κειμένου εντός γραμμής. Εδώ το **tag spans pdf** ξεχωρίζει.

```java
import com.aspose.pdf.tagged.logicalstructure.elements.ils.SpanElement;

SpanElement span1 = taggedContent.createSpanElement();
span1.setText("Span 1.");
span1.setTag("SPAN");
// Repeat for other spans...
```

```java
p1.appendChild(span1);
// Append other spans similarly...
```

### Πώς να δημιουργήσετε ετικετοποιημένο PDF Java – Βήμα 4: Αποθήκευση του ετικετοποιημένου εγγράφου PDF

Διατηρήστε το πλήρως ετικετοποιημένο έγγραφο στο δίσκο με μία κλήση `save`, διασφαλίζοντας ότι όλες οι πληροφορίες λογικής δομής γράφονται στο αρχείο εξόδου.

#### Επισκόπηση
Αφού δημιουργήσουμε τη λογική δομή, απλώς αποθηκεύουμε το έγγραφο στην επιθυμητή τοποθεσία.  
Η μέθοδος `save` γράφει τόσο το οπτικό περιεχόμενο όσο και το κρυφό δέντρο ετικετών, παράγοντας ένα PDF που συμμορφώνεται με το PDF/UA και είναι έτοιμο για ανίχνευση SEO.

```java
String outputPath = dataDir + "CustomTag.pdf";
document.save(outputPath);
```

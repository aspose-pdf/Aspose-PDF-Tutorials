## Οδηγός Εφαρμογής
Ας αναλύσουμε τώρα τα βήματα που απαιτούνται για να προσθέσετε μια ταξινομημένη λίστα HTML στο έγγραφο PDF σας χρησιμοποιώντας το Aspose.PDF για Java.
### Δημιουργία νέου εγγράφου PDF
Αρχικά, δημιουργήστε ένα νέο `Document` αντικείμενο ως βάση του αρχείου PDF μας:
```java
// Δημιουργία αντικειμένου εγγράφου για τη δημιουργία ενός νέου εγγράφου PDF
doc = new Document();
```
### Προσθήκη ενός τμήματος HTML
Χρησιμοποιήστε το `HtmlFragment` κλάση για την ερμηνεία και την απόδοση HTML μέσα στο PDF.
#### Βήμα 1: Δημιουργήστε ένα HtmlFragment
Δημιουργήστε ένα `HtmlFragment` με το επιθυμητό περιεχόμενο HTML:
```java
// Δημιουργήστε ένα HtmlFragment με το περιεχόμενο HTML για την ταξινομημένη λίστα
t = new HtmlFragment("<body style='line-height: 100px;'><ul><li>First</li><li>Second</li><li>Third</li><li>Fourth</li><li>Fifth</li></ul>Text after the list.<br/>Next line<br/>Last line</body>");
```
#### Βήμα 2: Προσθήκη νέας σελίδας στο έγγραφο
Προσθήκη νέας σελίδας στο έγγραφό σας:
```java
// Προσθήκη νέας σελίδας στο έγγραφο PDF
page = doc.getPages().add();
```
#### Βήμα 3: Εισαγωγή του HtmlFragment στη σελίδα
Προσθέστε το δικό σας `HtmlFragment` στη σελίδα που μόλις δημιουργήθηκε:
```java
// Εισαγάγετε το HtmlFragment στη σελίδα που μόλις προσθέσατε
page.getParagraphs().add(t);
```
### Αποθήκευση του εγγράφου PDF
Αποθηκεύστε το έγγραφο σε έναν καθορισμένο κατάλογο:
```java
// Αποθηκεύστε το δημιουργημένο έγγραφο PDF
doc.save(outputDir + "/AddHTMLOrderedListIntoDocuments_out.pdf");
```

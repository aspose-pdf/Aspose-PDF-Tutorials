## Ρύθμιση του Aspose.PDF για Java
Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.PDF για Java στα έργα σας, πρέπει να προσθέσετε τη βιβλιοθήκη ως εξάρτηση. Δείτε πώς:

**Maven**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>25.3</version>
</dependency>
```

**Γκράντλ**
```gradle
implementation 'com.aspose:aspose-pdf:25.3'
```

## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία και ρύθμιση παραμέτρων ενός εγγράφου PDF

#### Επισκόπηση
Η δημιουργία ενός νέου εγγράφου PDF περιλαμβάνει την αρχικοποίηση μιας παρουσίας του `Document` κλάση, προσθήκη σελίδων και ορισμός ιδιοτήτων όπως μέγεθος και περιθώρια.

#### Βήμα προς βήμα εφαρμογή
**Αρχικοποίηση και Ρύθμιση Εγγράφου**
Ξεκινήστε δημιουργώντας ένα νέο έγγραφο PDF:
```java
import com.aspose.pdf.Document;
import com.aspose.pdf.Page;

String outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε μια νέα παρουσία της κλάσης Document
Document doc = new Document();
```

**Προσθήκη σελίδας στο έγγραφο**
Μπορείτε να προσθέσετε σελίδες χρησιμοποιώντας το `getPages().add()` μέθοδος:
```java
Page page = doc.getPages().add(); // Προσθέτει μια νέα σελίδα στο έγγραφο
```

**Ρύθμιση μεγέθους σελίδας και περιθωρίων**
Προσαρμόστε το μέγεθος σελίδας και τα περιθώρια όπως απαιτείται:
```java
// Ορίστε το μέγεθος της σελίδας
page.setPageSize(375, 300);

// Ορίστε τα περιθώρια για τη σελίδα
page.getPageInfo().getMargin().setLeft(0);
page.getPageInfo().getMargin().setTop(0);
```

**Αποθήκευση του εγγράφου**
Τέλος, αποθηκεύστε το έγγραφό σας σε έναν καθορισμένο κατάλογο:
```java
doc.save(outputDir + "/Created_PDF.pdf");
```

### Λειτουργία 2: Προσθήκη ορθογωνίου σε σελίδα PDF

#### Επισκόπηση
Η προσθήκη γραφικών στοιχείων όπως ορθογώνια περιλαμβάνει τη χρήση του `Graph` και `Rectangle` μαθήματα από το Aspose.PDF.

#### Βήμα προς βήμα εφαρμογή
**Αρχικοποίηση γραφήματος και ορισμός θέσης**
Δημιουργήστε ένα αντικείμενο γραφήματος για το ορθογώνιό σας:
```java
import com.aspose.pdf.drawing.Graph;
import com.aspose.pdf.drawing.Rectangle;

Page page = new Page(); // Υποθέστε ότι η σελίδα είναι μέρος του εγγράφου
Graph graph = new Graph(60, 40); // Δημιουργήστε ένα γράφημα με καθορισμένες διαστάσεις

// Διορθώστε τη θέση σε σχέση με τη σελίδα
graph.setChangePosition(false);
```

**Ορισμός θέσης και διαστάσεων**
Τοποθετήστε το ορθογώνιο στη σελίδα PDF σας:
```java
graph.setLeft(50); // Ορισμός αριστερής θέσης
graph.setTop(40);  // Ορισμός κορυφαίας θέσης

Rectangle rect = new Rectangle(0, 0, 60, 40); // Ορισμός διαστάσεων
rect.getGraphInfo().setFillColor(Color.getRed()); // Χρώμα γεμίσματος
rect.getGraphInfo().setColor(Color.getRed()); // Χρώμα περιγράμματος

graph.getShapes().add(rect); // Προσθήκη ορθογωνίου στη συλλογή σχημάτων του γραφήματος
```

**Έλεγχος σειράς Z και προσθήκη στη σελίδα**
Βεβαιωθείτε ότι η σειρά απόδοσης είναι σωστή ορίζοντας τον δείκτη Z και, στη συνέχεια, προσθέστε το γράφημά σας στη σελίδα:
```java
graph.setZIndex(2); // Ορισμός σειράς σχεδίασης

page.getParagraphs().add(graph);
```

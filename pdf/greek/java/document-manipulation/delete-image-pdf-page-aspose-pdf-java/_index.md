## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει το περιβάλλον σας, ας εφαρμόσουμε τη λειτουργία για την κατάργηση μιας συγκεκριμένης εικόνας από μια σελίδα PDF.

### Πρόσβαση και τροποποίηση πόρων PDF

Για να καταργήσετε μια εικόνα, πρέπει πρώτα να φορτώσετε το έγγραφο και να αποκτήσετε πρόσβαση στους πόρους του. Δείτε πώς:

#### Βήμα 1: Φόρτωση του εγγράφου
```java
document = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```
Αυτό φορτώνει το αρχείο PDF στη μνήμη, ώστε να είναι δυνατή η επεξεργασία του.

#### Βήμα 2: Πρόσβαση σε πόρους εικόνας
Πρόσβαση στις εικόνες από μια συγκεκριμένη σελίδα:
```java
document.getPages().get_Item(1).getResources().getImages();
```
Αυτό ανακτά όλες τις εικόνες στην πρώτη σελίδα του εγγράφου σας.

#### Βήμα 3: Αφαίρεση συγκεκριμένης εικόνας
Καταργήστε μια εικόνα καθορίζοντας τον δείκτη της:
```java
document.getPages().get_Item(1).getResources().getImages().delete(1);
```
Σε αυτό το παράδειγμα, αφαιρούμε τη δεύτερη εικόνα (το ευρετήριο ξεκινά από το 1) από την πρώτη σελίδα.

### Αποθήκευση αλλαγών

Αφού τροποποιήσετε το έγγραφό σας:

```java
document.save("YOUR_OUTPUT_DIRECTORY/output.pdf");
```

Αυτό το βήμα διασφαλίζει ότι όλες οι αλλαγές εγγράφονται ξανά σε ένα νέο ή υπάρχον αρχείο PDF.

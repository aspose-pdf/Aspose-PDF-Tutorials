## Οδηγός Εφαρμογής

Αυτή η ενότητα αναλύει δύο ισχυρές λειτουργίες: την επεξεργασία PDF με `PdfPageEditor` και μετατροπή σελίδων σε εικόνες χρησιμοποιώντας `PdfConverter`.

### Επεξεργασία PDF χρησιμοποιώντας το PdfPageEditor

#### Επισκόπηση
Ο `PdfPageEditor` Η κλάση σάς επιτρέπει να τροποποιείτε περιοχές μέσα σε ένα αρχείο PDF, προσφέροντας δυνατότητες όπως η μετακίνηση της προέλευσης μιας σελίδας.

#### Οδηγός βήμα προς βήμα

**1. Δημιουργήστε ένα αντίγραφο του PdfPageEditor**
```java
import com.aspose.pdf.facades.PdfPageEditor;
// Δημιουργήστε μια παρουσία του PdfPageEditor
PdfPageEditor editor = new PdfPageEditor();
```

**2. Δέστε το αρχείο PDF προέλευσης**
Καθορίστε τον κατάλογο και συνδέστε το αρχείο PDF προορισμού σας:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
editor.bindPdf(dataDir + "/Exported.pdf");
```

**3. Μετακίνηση προέλευσης σελίδας**
Προσαρμόστε το σημείο προέλευσης σε μια συγκεκριμένη θέση, όπως (100, 200):
```java
// Μετατόπιση της προέλευσης της σελίδας
editor.movePosition(100, 200);
```

**4. Αποθήκευση αλλαγών σε νέο αρχείο**
Χρήση `FileOutputStream` για να αποθηκεύσετε τις αλλαγές σας:
```java
import java.io.FileOutputStream;
String outputDir = "YOUR_OUTPUT_DIRECTORY";
try (FileOutputStream fout = new FileOutputStream(outputDir + "/TempFile.pdf")) {
    editor.save(fout); // Αποθήκευση του τροποποιημένου PDF
}
```

### Μετατροπή σελίδας PDF σε εικόνα χρησιμοποιώντας το PdfConverter

#### Επισκόπηση
`PdfConverter` Διευκολύνει τη μετατροπή συγκεκριμένων σελίδων ενός PDF σε μορφές εικόνας, επιτρέποντας την εύκολη κοινή χρήση ή αρχειοθέτηση.

#### Οδηγός βήμα προς βήμα

**1. Δημιουργία και σύνδεση του μετατροπέα PdfConverter**
```java
import com.aspose.pdf.facades.PdfConverter;
// Αρχικοποίηση του PdfConverter με το τροποποιημένο αρχείο PDF
PdfConverter objConverter = new PdfConverter();
objConverter.bindPdf(new FileInputStream(outputDir + "/TempFile.pdf"));
```

**2. Ορισμός παραμέτρων μετατροπής**
Ορίστε ποιες σελίδες θα μετατραπούν και ξεκινήστε τη διαδικασία:
```java
objConverter.setStartPage(1); // Ξεκινήστε από τη σελίδα 1
objConverter.setEndPage(1);   // Τέλος στη σελίδα 1
objConverter.doConvert();
```

**3. Αποθήκευση μετατρεπόμενων εικόνων**
Περιηγηθείτε στις διαθέσιμες εικόνες και αποθηκεύστε τες:
```java
int page = 1;
while (objConverter.hasNextImage()) {
    objConverter.getNextImage(outputDir + "/Specific_Region-Image" + page++ + ".jpeg");
}
```

## Εισαγωγή
Θέλετε να βελτιώσετε τα έγγραφά σας PDF προσαρμόζοντας υποσημειώσεις ή σημειώσεις τέλους; Είτε πρόκειται για ακαδημαϊκές εργασίες, επιχειρηματικές αναφορές είτε για τεχνική τεκμηρίωση, η προσαρμογή αυτών των στοιχείων μπορεί να βελτιώσει σημαντικά την αναγνωσιμότητα και την παρουσίαση. ## Οδηγός Εφαρμογής
Θα καλύψουμε τρία κύρια χαρακτηριστικά: την προσαρμογή στυλ γραμμών υποσημειώσεων, την προσαρμογή ετικετών υποσημειώσεων και τη δημιουργία σημειώσεων τέλους. Κάθε ενότητα παρέχει μια βήμα προς βήμα προσέγγιση για την εφαρμογή αυτών των χαρακτηριστικών χρησιμοποιώντας το Aspose.PDF για Java.

### Χαρακτηριστικό 1: Προσαρμοσμένο στυλ γραμμής για υποσημείωση
#### Επισκόπηση
Προσαρμόστε την εμφάνιση των υποσημειώσεων αλλάζοντας το στυλ γραμμής τους στα έγγραφα PDF σας.

#### Βήματα:
**1. Δημιουργία και ρύθμιση παραμέτρων GraphInfo**

```java
GraphInfo graph = new GraphInfo();
graph.setLineWidth(2); // Ορίστε το πλάτος γραμμής σε 2 σημεία.
graph.setColor(Color.getRed()); // Αλλάξτε χρώμα σε κόκκινο.
grapth.setDashArray(new int[] { 3 }); // Ορίστε μοτίβο παύλας: 3 μονάδες ενεργοποιημένες, 1 μονάδα απενεργοποιημένη.
grapth.setDashPhase(1); // Ξεκινήστε με μια μετατόπιση 1 μονάδας.
```

**Εξήγηση**: Αυτές οι ρυθμίσεις προσαρμόζουν το στυλ γραμμής των υποσημειώσεων. `GraphInfo` Η κλάση ελέγχει τον τρόπο απόδοσης των γραμμών.

**2. Εφαρμογή προσαρμοσμένου στυλ γραμμής στην υποσημείωση**

```java
page.setNoteLineStyle(graph);
```

Αυτή η μέθοδος εφαρμόζει την προσαρμοσμένη διαμόρφωση γραμμής σε όλες τις υποσημειώσεις στη σελίδα.

**3. Προσθήκη τμημάτων κειμένου με υποσημειώσεις**

```java
TextFragment text = new TextFragment("Hello World");
text.setFootNote(new Note("foot note for test text 1"));
pag.getParagraphs().add(text);

text = new TextFragment("Aspose.Pdf for Java");
text.setFootNote(new Note("foot note for test text 2"));
pag.getParagraphs().add(text);
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Εξασφαλίζω `GraphInfo` Οι ρυθμίσεις εφαρμόζονται πριν από την προσθήκη υποσημειώσεων.
- Ελέγξτε ξανά τις τιμές χρώματος και μοτίβου παύλας εάν το στυλ δεν εμφανίζεται όπως αναμένεται.

### Χαρακτηριστικό 2: Προσαρμογή ετικέτας υποσημείωσης
#### Επισκόπηση
Βελτιώστε τη σαφήνεια προσαρμόζοντας το κείμενο των ετικετών υποσημειώσεων στα έγγραφά σας PDF.

**1. Προσθήκη τμήματος κειμένου με προσαρμοσμένη ετικέτα υποσημείωσης**

```java
TextFragment text = new TextFragment("Hello World");
text.setFootNote(new Note("foot note for test text 1"));
text.getFootNote().setText("Aspose(2015)"); // Προσαρμόστε την ετικέτα.
pag.getParagraphs().add(text);
```

**Εξήγηση**: Το `setText` Η μέθοδος σάς επιτρέπει να επαναπροσδιορίσετε την ετικέτα της υποσημείωσης, παρέχοντας σαφήνεια ή επωνυμία όπως απαιτείται.

### Λειτουργία 3: Δημιουργία σημειώσεων τέλους σε PDF
#### Επισκόπηση
Προσθέστε σημειώσεις στο τέλος των σελίδων του εγγράφου σας για ολοκληρωμένες αναφορές.

**1. Προσθήκη τμήματος κειμένου με σημείωση τέλους**

```java
TextFragment text = new TextFragment("Hello World");
text.setEndNote(new Note("sample End note"));
text.getEndNote().setText(" Aspose(2015)"); // Προσαρμόστε την ετικέτα σημείωσης τέλους.
pag.getParagraphs().add(text);
```

**Εξήγηση**: Χρήση `setEndNote` για την προσθήκη σημειώσεων στο τέλος ενός εγγράφου, ιδανικό για μεγαλύτερες αναφορές.

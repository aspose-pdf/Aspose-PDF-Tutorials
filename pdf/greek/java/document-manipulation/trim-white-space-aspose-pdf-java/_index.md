## Οδηγός Εφαρμογής
Δείτε πώς μπορείτε να περικόψετε τον κενό χώρο από σελίδες PDF με το Aspose.PDF για Java:
### Χαρακτηριστικό: Περικοπή λευκού χώρου γύρω από τις σελίδες
Αυτή η λειτουργία δείχνει πώς να αφαιρέσετε αποτελεσματικά τα ανεπιθύμητα περιθώρια από το περιεχόμενο PDF σας.
#### Βήμα 1: Φόρτωση του εγγράφου PDF προέλευσης
Τοποθετήστε το έγγραφο PDF προέλευσης στο σημείο όπου θέλετε να περικόψετε τα κενά.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String inputFilePath = dataDir + "/input.pdf";

// Φόρτωση του εγγράφου PDF προέλευσης
document = new Document(inputFilePath);
```
#### Βήμα 2: Επιλέξτε μια σελίδα και υπολογίστε τα όρια περιεχομένου
Προσδιορίστε ποια σελίδα θέλετε να περικόψετε. Εδώ, εστιάζουμε στην πρώτη σελίδα.
```java
Page pdfPage = document.getPages().get_Item(1);

// Υπολογισμός ορίων περιεχομένου
Rectangle contentBBox = pdfPage.calculateContentBBox();
```
Ο `calculateContentBBox()` Η μέθοδος βοηθά στον προσδιορισμό της ακριβούς περιοχής που περιέχει το πραγματικό περιεχόμενο υπολογίζοντας το πλαίσιο ορίων της.
#### Βήμα 3: Προσαρμογή CropBox και MediaBox
Ορίστε και τα δύο `CropBox` και `MediaBox` για να ταιριάζει με τα υπολογισμένα όρια, ουσιαστικά περικόπτοντας τυχόν επιπλέον λευκό χώρο γύρω από το περιεχόμενο PDF σας.
```java
// Περικοπή λευκού χώρου ορίζοντας το CropBox και το MediaBox
pdfPage.setCropBox(contentBBox);
pdfPage.setMediaBox(contentBBox);
```
#### Βήμα 4: Αποθηκεύστε το PDF που προκύπτει
Αποθηκεύστε το προσαρμοσμένο PDF σας σε ένα νέο αρχείο.
```java
String outputFilePath = dataDir + "/output_trim.pdf";
document.save(outputFilePath);
```

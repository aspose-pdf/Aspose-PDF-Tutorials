## Οδηγός Εφαρμογής
### Φόρτωση και τροποποίηση εγγράφου PDF
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα υπάρχον έγγραφο PDF, να προσθέσετε περιεχόμενο και να αποθηκεύσετε τις αλλαγές.

#### Επισκόπηση
Θα μάθετε πώς να δημιουργείτε ένα νέο έγγραφο PDF, να προσθέτετε κείμενο σε αυτό και, στη συνέχεια, να αποθηκεύετε το τροποποιημένο αρχείο σας.

#### Βήμα προς βήμα εφαρμογή
##### 1. Δημιουργήστε ένα νέο έγγραφο PDF

```java
import com.aspose.pdf.Document;
import com.aspose.pdf.Page;
import com.aspose.pdf.TextFragment;

public class ModifyPdf {
    public static void main(String[] args) {
        // Ορισμός καταλόγων
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        String outputDir = "YOUR_OUTPUT_DIRECTORY";

        // Δημιουργήστε μια νέα παρουσία εγγράφου
        Document doc = new Document();
```

##### 2. Προσθήκη κειμένου στο PDF

```java
        // Προσθήκη σελίδας στο έγγραφο
        Page page = doc.getPages().add();

        // Επανάληψη και προσθήκη τμημάτων κειμένου
        for (int i = 0; i < 4; i++) {
            TextFragment text = new TextFragment("Lorem ipsum dolor sit amet...");
            text.getTextState().setFontSize(20);
            page.getParagraphs().add(text);
        }
```

##### 3. Αποθήκευση του εγγράφου

```java
        // Αποθηκεύστε το έγγραφο στον κατάλογο εξόδου
        doc.save(outputDir + "DetermineLineBreak_out.pdf");
    }
}
```

### Ανάκτηση ειδοποιήσεων από μια σελίδα PDF
Αποκτήστε πληροφορίες σχετικά με τα μεταδεδομένα ή τις ειδοποιήσεις που σχετίζονται με σελίδες στο PDF σας.

#### Επισκόπηση
Μάθετε πώς να αποκτάτε πρόσβαση και να ανακτάτε ειδοποιήσεις ή μεταδεδομένα σε επίπεδο σελίδας μέσα σε ένα αρχείο PDF.

#### Βήματα Υλοποίησης

```java
import com.aspose.pdf.Document;

public class RetrieveNotifications {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";

        // Φόρτωση ενός υπάρχοντος εγγράφου PDF
        Document doc = new Document(dataDir + "input.pdf");

        // Ανάκτηση ειδοποιήσεων από την πρώτη σελίδα
        String notifications = doc.getPages().get_Item(1).getNotifications();

        System.out.println("Page Notifications: " + notifications);
    }
}
```

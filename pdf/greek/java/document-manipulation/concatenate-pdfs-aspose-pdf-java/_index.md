## Οδηγός Εφαρμογής
### Λειτουργία 1: Συνένωση αρχείων PDF χρησιμοποιώντας διαδρομές αρχείων
#### Επισκόπηση
Η συνένωση αρχείων PDF είναι ζωτικής σημασίας όταν χρειάζεστε ένα μόνο έγγραφο από πολλαπλές πηγές. Αυτή η λειτουργία απλοποιεί τη συγχώνευση εγγράφων σε ένα χρησιμοποιώντας διαδρομές που παρέχονται στον κώδικα.

#### Βήματα:
##### Βήμα 1: Δημιουργία αντικειμένου PdfFileEditor
```java
import com.aspose.pdf.facades.PdfFileEditor;

PdfFileEditor pdfEditor = new PdfFileEditor();
```
Αυτό το αντικείμενο είναι η πύλη σας για την επεξεργασία αρχείων PDF με το Aspose.PDF.

##### Βήμα 2: Συνένωση αρχείων
Δείτε πώς καθορίζετε τις διαδρομές εισόδου και εξόδου:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

pdfEditor.concatenate(
    dataDir + "/input1.pdf",
    dataDir + "/input2.pdf",
    outputDir + "/output.pdf"
);
```
Ο `concatenate` Η μέθοδος λαμβάνει τις διαδρομές αρχείων των PDF εισόδου και μια διαδρομή εξόδου για το συγχωνευμένο έγγραφο. Βεβαιωθείτε ότι οι κατάλογοί σας έχουν ρυθμιστεί σωστά για να αποφύγετε τυχόν σφάλματα "δεν βρέθηκε αρχείο".

### Λειτουργία 2: Ρύθμιση επιλογής περιγραμμάτων αντιγραφής
#### Επισκόπηση
Κατά τη συνένωση αρχείων PDF, η διαχείριση περιγραμμάτων (σελιδοδεικτών) είναι συχνά απαραίτητη. Αυτή η λειτουργία σάς επιτρέπει να αποφασίσετε εάν αυτά τα περιγράμματα θα πρέπει να συμπεριληφθούν στο τελικό έγγραφο.

#### Βήματα:
##### Βήμα 1: Ρύθμιση παραμέτρων του PdfFileEditor για έλεγχο Outline
```java
PdfFileEditor pfe = new PdfFileEditor();
pfe.setCopyOutlines(false); // Οι σελιδοδείκτες δεν θα αντιγραφούν.
```
Σύνθεση `CopyOutlines` Η τιμή "false" διασφαλίζει ότι οι σελιδοδείκτες από μεμονωμένα PDF δεν περιλαμβάνονται στο συνενωμένο αρχείο.

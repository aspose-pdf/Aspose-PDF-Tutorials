## Οδηγός Εφαρμογής
Θα αναλύσουμε την υλοποίησή μας σε δύο κύρια χαρακτηριστικά: αλλαγή προσανατολισμού σελίδας σε οριζόντιο και προσαρμογή περιεχομένου ανάλογα.

### Λειτουργία 1: Αλλαγή προσανατολισμού σελίδας σε οριζόντιο
**Επισκόπηση**Αυτή η λειτουργία μετατρέπει κάθε σελίδα PDF από κατακόρυφη σε οριζόντια, προσαρμόζοντας τις διαστάσεις για μια απρόσκοπτη μετάβαση.

#### Βήμα προς βήμα εφαρμογή
##### Φόρτωση του εγγράφου προέλευσης
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/Input.pdf");
```
*Γιατί*Η φόρτωση του εγγράφου σας είναι απαραίτητη, καθώς μας επιτρέπει να χειριζόμαστε τις σελίδες του.

##### Αλλαγή προσανατολισμού σελίδας
Επαναλάβετε κάθε σελίδα και προσαρμόστε τις διαστάσεις:
```java
int pageCount = doc.getPages().size();
for (int i = 1; i <= pageCount; i++) {
    Rectangle r = doc.getPages().get_Item(i).getMediaBox();

    double newHeight = r.getWidth();
    double newWidth = r.getHeight();
    double newLLX = r.getLLX();
    double newLLY = r.getLLY() + (r.getHeight() - newHeight);

    doc.getPages().get_Item(i).setMediaBox(new Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight));
    doc.getPages().get_Item(i).setCropBox(new Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight));

    // Περιστρέψτε τη σελίδα κατά 90 μοίρες για οριζόντιο προσανατολισμό
    doc.getPages().get_Item(i).setRotate(Rotation.on90);
}
```
*Γιατί*Αυτό το μπλοκ κώδικα αλλάζει τις διαστάσεις κάθε σελίδας και την περιστρέφει κατά 90 μοίρες για να επιτύχει οριζόντια προβολή.

##### Αποθήκευση τροποποιημένου εγγράφου
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.save(outputDir + "/Output.pdf");
```
*Γιατί*Η αποθήκευση του εγγράφου διασφαλίζει ότι οι αλλαγές σας αποθηκεύονται μόνιμα, επιτρέποντάς σας να ελέγξετε ή να μοιραστείτε το τροποποιημένο PDF.

### Λειτουργία 2: Προσαρμογή περιεχομένου σελίδας στον νέο προσανατολισμό
**Επισκόπηση**: Προσαρμόστε τις διαστάσεις του περιεχομένου μετά την αλλαγή του προσανατολισμού της σελίδας για καλύτερη προσαρμογή.

#### Ρύθμιση διαστάσεων
```java
Rectangle r = doc.getPages().get_Item(0).getMediaBox();
double newHeight = r.getHeight();
double newWidth = r.getWidth() * (r.getHeight() / r.getWidth());

// Περαιτέρω προσαρμογές μπορούν να εφαρμοστούν εδώ, εάν είναι απαραίτητο.
```
*Γιατί*Η διασφάλιση ότι το περιεχόμενο ταιριάζει καλά στις σελίδες με νέο προσανατολισμό βελτιώνει την αναγνωσιμότητα και την ποιότητα της παρουσίασης.

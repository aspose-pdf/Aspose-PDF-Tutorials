## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη μετατροπή του RGB σε CMYK και την επαλήθευση αυτών των αλλαγών.

### Δυνατότητα 1: Μετατροπή χρώματος RGB σε CMYK σε έγγραφο PDF

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε όλες τις ρυθμίσεις χρώματος RGB στα έγγραφα PDF σας σε CMYK, καθιστώντας τα κατάλληλα για εκτύπωση υψηλής ποιότητας. 

##### Βήμα 1: Φόρτωση του εγγράφου PDF
Αρχικά, φορτώστε το έγγραφο PDF πηγής σας.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input_color.pdf");
```

##### Βήμα 2: Πρόσβαση στα περιεχόμενα της σελίδας
Αποκτήστε πρόσβαση στα περιεχόμενα της πρώτης σελίδας για να τροποποιήσετε τις ρυθμίσεις χρωμάτων.

```java
OperatorCollection contents = doc.getPages().get_Item(1).getContents();
```

##### Βήμα 3: Επανάληψη και μετατροπή χρωμάτων
Επαναλάβετε κάθε τελεστή στη συλλογή περιεχομένου. Για κάθε ρύθμιση χρώματος RGB, μετατρέψτε την σε CMYK:

```java
for (int j = 1; j <= contents.size(); j++) {
    Operator oper = contents.get_Item(j);
    
    if (oper instanceof SetRGBColor || oper instanceof SetRGBColorStroke) {
        try {
            double[] rgbFloatArray = new double[]{
                Double.valueOf(oper.getParameters().get(0).toString()),
                Double.valueOf(oper.getParameters().get(1).toString()),
                Double.valueOf(oper.getParameters().get(2).toString())
            };
            
            double[] cmyk = new double[4];
            if (oper instanceof SetRGBColor) {
                ((SetRGBColor) oper).getCMYKColor(rgbFloatArray, cmyk);
                contents.set_Item(j, new SetCMYKColor(cmyk[0], cmyk[1], cmyk[2], cmyk[3]));
            } else if (oper instanceof SetRGBColorStroke) {
                ((SetRGBColorStroke) oper).getCMYKColor(rgbFloatArray, cmyk);
                contents.set_Item(j, new SetCMYKColorStroke(cmyk[0], cmyk[1], cmyk[2], cmyk[3]));
            } else {
                throw new java.lang.Throwable("Unsupported command");
            }
        } catch (Throwable e) {
            e.printStackTrace();
        }
    }
}
```

##### Βήμα 4: Αποθήκευση του τροποποιημένου εγγράφου
Τέλος, αποθηκεύστε το έγγραφό σας με τις ρυθμίσεις χρώματος που έχουν μετατραπεί.

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.save(outputDir + "/input_colorout.pdf");
```

Α: Βεβαιωθείτε ότι όλες οι ρυθμίσεις χρώματος έχουν μετατραπεί όπως προβλέπεται, εστιάζοντας στις μετατροπές RGB σε CMYK.

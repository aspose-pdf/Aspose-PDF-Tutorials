### Εξαγωγή εικόνων από πεδία υπογραφής

**Επισκόπηση:**
Αυτή η ενότητα δείχνει πώς να επαναλάβετε τα πεδία μιας φόρμας PDF, να προσδιορίσετε πεδία υπογραφής και να εξαγάγετε εικόνες που είναι ενσωματωμένες σε αυτά.

**Βήματα:**
1. **Επανάληψη μέσω πεδίων φόρμας:**
   - Αποκτήστε πρόσβαση στα πεδία της φόρμας χρησιμοποιώντας `pdfDocument.getForm().getFields()`.
2. **Προσδιορισμός πεδίων υπογραφής:**
   ```java
   for (Field field : pdfDocument.getForm().getFields()) {
       if (field instanceof SignatureField) {
           SignatureField signatureField = (SignatureField) field;
           // Συνέχεια με την εξαγωγή εικόνας
       }
   }
   ```
3. **Εξαγωγή και αποθήκευση εικόνων:**
   - Χρησιμοποιήστε το `extractImage()` μέθοδος από το `SignatureField` αντικείμενο για να λάβετε ένα InputStream της ενσωματωμένης εικόνας.
   ```java
   try {
       FileOutputStream output = new FileOutputStream(outputDir + "/im" + i + ".jpeg");
       InputStream tempStream = signatureField.extractImage();
       byte[] b = new byte[tempStream.available()];
       tempStream.read(b);
       output.write(b);
       i++;
   } catch (IOException e) {
       e.printStackTrace(); // Χειριστείτε τις εξαιρέσεις κατάλληλα
   }
   ```
4. **Διαχείριση Πόρων:**
   - Βεβαιωθείτε για την ορθή απόρριψη των `Document` αντίρρηση για την απελευθέρωση πόρων.
   ```java
   if (pdfDocument != null)
       pdfDocument.dispose();
   ```

## Οδηγός Εφαρμογής

Τώρα που είστε έτοιμοι, ας δημιουργήσουμε ένα PDF με ετικέτες και εικόνες.

### Δυνατότητα 1: Δημιουργία και ρύθμιση παραμέτρων περιεχομένου με ετικέτες σε PDF

#### Επισκόπηση
Η δημιουργία ενός PDF με ετικέτες διασφαλίζει την προσβασιμότητα. Αυτό το βήμα καλύπτει τον ορισμό του τίτλου και της γλώσσας του εγγράφου για συμβατότητα με προγράμματα ανάγνωσης οθόνης.

**Βήμα 1: Δημιουργήστε ένα νέο έγγραφο PDF**
```java
Document document = new Document();
```

**Βήμα 2: Πρόσβαση στο Επίπεδο Περιεχομένου με Ετικέτες**
Η πρόσβαση στο επίπεδο περιεχομένου με ετικέτες διαχειρίζεται τις λειτουργίες προσβασιμότητας:
```java
import com.aspose.pdf.tagged.ITaggedContent;

ITaggedContent taggedContent = document.getTaggedContent();
```

**Βήμα 3: Ορισμός τίτλου και γλώσσας**
Ορίστε αυτά τα χαρακτηριστικά για καλύτερη υποστήριξη αναγνωστών οθόνης:
```java
taggedContent.setTitle("CreatePDFwithTaggedImage");
taggedContent.setLanguage("en-US");
```

### Λειτουργία 2: Προσθήκη εικόνας σε περιεχόμενο PDF με ετικέτες

#### Επισκόπηση
Η προσθήκη εικόνων με κατάλληλες ετικέτες διασφαλίζει ότι περιγράφονται με ακρίβεια από τους αναγνώστες οθόνης.

**Βήμα 4: Δημιουργία και διαμόρφωση ενός στοιχείου εικόνας**
Δημιουργήστε ένα στοιχείο σχήματος για προσβασιμότητα:
```java
import com.aspose.pdf.tagged.logicalstructure.elements.IllustrationElement;

IllustrationElement figure1 = taggedContent.createFigureElement();
taggedContent.getRootElement().appendChild(figure1);
figure1.setAlternativeText("Aspose Logo");
figure1.setTitle("Image 1");
figure1.setTag("Fig");

// Υποθέστε διαδρομή εικόνας υψηλής ανάλυσης
String imagePath = "YOUR_DOCUMENT_DIRECTORY/aspose-logo.jpg";
figure1.setImage(imagePath);
```

### Λειτουργία 3: Αποθήκευση εγγράφου PDF με ετικέτα

#### Επισκόπηση
Η αποθήκευση του εγγράφου διατηρεί το περιεχόμενο με ετικέτες, διασφαλίζοντας την προσβασιμότητα.

**Βήμα 6: Ορισμός διαδρομής και ονόματος αρχείου**
Βεβαιωθείτε ότι έχετε έναν καθορισμένο κατάλογο εξόδου:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
document.save(outputDir + "PDFwithTaggedImage.pdf");
```

## Οδηγός Εφαρμογής

Τώρα που όλα έχουν ρυθμιστεί, ας μετατρέψουμε το PDF RGB σε κλίμακα του γκρι.

### Φόρτωση του εγγράφου PDF προέλευσης

Φορτώστε το έγγραφο PDF προέλευσης χρησιμοποιώντας το `Document` τάξη:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος για αρχεία εισόδου
Document document = new Document(dataDir + "/input.pdf");
```
Αυτό το βήμα προετοιμάζει το έγγραφό σας για μετατροπή.

### Αρχικοποίηση στρατηγικής μετατροπής

Αρχικοποιήστε τη στρατηγική για τη διαχείριση της μετατροπής RGB σε κλίμακα του γκρι:
```java
RgbToDeviceGrayConversionStrategy strategy = new RgbToDeviceGrayConversionStrategy();
```
Ο `RgbToDeviceGrayConversionStrategy` Η κλάση παρέχει έναν απλό τρόπο μετατροπής χρωματικών χώρων.

### Επανάληψη σε κάθε σελίδα

Επαναλάβετε κάθε σελίδα και εφαρμόστε τη μετατροπή:
```java
for (int idxPage = 1; idxPage <= document.getPages().size(); idxPage++) {
    Page page = document.getPages().get_Item(idxPage);
    strategy.convert(page);
}
```
Αυτός ο βρόχος διασφαλίζει ότι κάθε σελίδα υποβάλλεται σε επεξεργασία σε κλίμακα του γκρι.

### Αποθήκευση του μετατρεπόμενου εγγράφου

Αποθηκεύστε το PDF που μετατρέψατε στον επιθυμητό κατάλογο εξόδου:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // Κατάλογος για αρχεία εξόδου
document.save(outputDir + "/output.pdf");
```
Αυτό το βήμα εγγράφει τις αλλαγές σε ένα νέο αρχείο.

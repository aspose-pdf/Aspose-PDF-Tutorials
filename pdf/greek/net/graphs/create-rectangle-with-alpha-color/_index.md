## Οδηγίες εγκατάστασης  
Για να χρησιμοποιήσετε το Aspose.PDF για το .NET στο έργο σας, ακολουθήστε τα παρακάτω βήματα:  

1. Εγκατάσταση μέσω NuGet  
Εκτελέστε την ακόλουθη εντολή στην κονσόλα NuGet Package Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Προσθήκη χώρου ονομάτων στο C#  
Συμπεριλάβετε τους απαιτούμενους χώρους ονομάτων στην αρχή του αρχείου C #:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Για λεπτομερή τεκμηρίωση, επισκεφθείτε:  
[Aspose.PDF για την τεκμηρίωση .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF για την αναφορά .NET API](https://reference.aspose.com/pdf/net/)

## Οδηγός Εφαρμογής
### Βήμα 1: Φόρτωση του εγγράφου PDF
Ξεκινήστε φορτώνοντας το έγγραφο PDF προορισμού σας. Αυτό είναι το σημείο εκκίνησης για την εφαρμογή τυχόν βελτιστοποιήσεων.

**Φόρτωση του εγγράφου σας**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Optimization;

// Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "path/to/your/documents/";

// Άνοιγμα ενός υπάρχοντος εγγράφου PDF
Document doc = new Document(dataDir + "AddImage.pdf");
```

### Βήμα 2: Διαμόρφωση επιλογών βελτιστοποίησης
Στη συνέχεια, ρυθμίστε επιλογές βελτιστοποίησης προσαρμοσμένες για συμπίεση FlateDecode. Αυτό το βήμα είναι κρίσιμο για τον καθορισμό του τρόπου συμπίεσης των εικόνων στο PDF.

**Ορισμός επιλογών βελτιστοποίησης**
```csharp
// Αρχικοποίηση επιλογών βελτιστοποίησης
var optimizationOptions = new Aspose.Pdf.Optimization.OptimizationOptions();

// Ρύθμιση παραμέτρων συμπίεσης εικόνας για χρήση κωδικοποίησης FlateDecode
optimizationOptions.ImageCompressionOptions.Encoding = ImageEncoding.Flate;
```

### Βήμα 3: Εφαρμογή βελτιστοποιήσεων
Αφού διαμορφώσετε τις ρυθμίσεις, εφαρμόστε αυτές τις βελτιστοποιήσεις στο έγγραφό σας.

**Βελτιστοποίηση πόρων**
```csharp
// Εφαρμογή επιλογών βελτιστοποίησης στο έγγραφο
doc.OptimizeResources(optimizationOptions);

// Αποθηκεύστε το βελτιστοποιημένο έγγραφο PDF
doc.Save(dataDir + "FlateDecodeCompression.pdf");
```

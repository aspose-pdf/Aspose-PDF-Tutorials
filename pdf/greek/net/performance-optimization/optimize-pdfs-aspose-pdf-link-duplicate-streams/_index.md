### Σύνδεση διπλότυπων ροών σε PDF
Η σύνδεση διπλότυπων ροών βοηθά στη μείωση του μεγέθους του αρχείου συγχωνεύοντας πανομοιότυπα δεδομένα σε διαφορετικά μέρη ενός εγγράφου. Δείτε πώς μπορείτε να το πετύχετε αυτό με το Aspose.PDF:

#### Βήμα 1: Φόρτωση του εγγράφου PDF
Ξεκινήστε φορτώνοντας το υπάρχον έγγραφο PDF σε μια παρουσία του `Document`:
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
Document pdfDocument = new Document(dataDir + "OptimizeDocument.pdf");
```

#### Βήμα 2: Διαμόρφωση επιλογών βελτιστοποίησης
Ορίστε το `LinkDuplicateStreams` ιδιότητα σε true εντός ενός `Pdf.Optimization.OptimizationOptions` αντικείμενο:
```csharp
var optimizeOptions = new Pdf.Optimization.OptimizationOptions
{
    LinkDuplcateStreams = true
};
```
Αυτή η διαμόρφωση υποδεικνύει στο Aspose.PDF να αναγνωρίζει και να συγχωνεύει διπλότυπες ροές σε όλο το έγγραφο.

#### Βήμα 3: Βελτιστοποίηση πόρων
Εφαρμόστε αυτές τις ρυθμίσεις βελτιστοποίησης στο PDF σας:
```csharp
pdfDocument.OptimizeResources(optimizeOptions);
```

#### Βήμα 4: Αποθήκευση του Βελτιστοποιημένου Εγγράφου
Τέλος, αποθηκεύστε το βελτιστοποιημένο έγγραφό σας σε ένα νέο αρχείο ή αντικαταστήστε το υπάρχον:
```csharp
dataDir = dataDir + "OptimizeDocument_out.pdf";
pdfDocument.Save(dataDir);

Console.WriteLine("\nLinked duplicate streams successfully.\nFile saved at " + dataDir);
```

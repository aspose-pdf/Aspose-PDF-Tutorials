### Εξαγωγή μεταδεδομένων XMP

**Επισκόπηση:**
Η εξαγωγή μεταδεδομένων XMP περιλαμβάνει την ανάγνωση συγκεκριμένων ιδιοτήτων που είναι ενσωματωμένες σε ένα αρχείο PDF. Αυτές οι πληροφορίες μπορούν να περιλαμβάνουν ημερομηνίες δημιουργίας, στοιχεία συντάκτη και προσαρμοσμένα πεδία.

#### Βήμα προς βήμα εφαρμογή:

1. **Φόρτωση του εγγράφου PDF**
   Ξεκινήστε φορτώνοντας το έγγραφο PDF προορισμού σας σε ένα `Aspose.Pdf.Document` αντικείμενο.

   ```csharp
   // Διαδρομή προς τον κατάλογο που περιέχει τα έγγραφα
   string dataDir = "path/to/your/documents/";

   // Φόρτωση του αρχείου PDF
   Document pdfDocument = new Document(dataDir + "GetXMPMetadata.pdf");
   ```

2. **Ιδιότητες μεταδεδομένων Access**
   Χρησιμοποιήστε το `Metadata` ιδιότητα του αντικειμένου εγγράφου σας για πρόσβαση σε μεταδεδομένα XMP.

   ```csharp
   // Ανάκτηση και εμφάνιση συγκεκριμένων ιδιοτήτων μεταδεδομένων
   Console.WriteLine(pdfDocument.Metadata["xmp:CreateDate"]);
   Console.WriteLine(pdfDocument.Metadata["xmp:Nickname"]);
   Console.WriteLine(pdfDocument.Metadata["xmp:CustomProperty"]);
   ```

**Επεξήγηση παραμέτρων:**
- `dataDir`: Η διαδρομή καταλόγου όπου είναι αποθηκευμένα τα αρχεία PDF σας.
- `Metadata`: Ένα αντικείμενο που μοιάζει με λεξικό και περιέχει όλες τις καταχωρήσεις μεταδεδομένων XMP.

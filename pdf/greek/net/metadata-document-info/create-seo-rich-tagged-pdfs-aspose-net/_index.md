## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία δημιουργίας ενός εγγράφου PDF με ετικέτες βήμα προς βήμα.

### Δημιουργία νέου εγγράφου

Ξεκινήστε ρυθμίζοντας το έργο σας με τους απαραίτητους χώρους ονομάτων:


#### Βήμα 1: Αρχικοποίηση ενός νέου εγγράφου PDF

Δημιουργήστε μια παρουσία του `Document` κλάση. Αυτό αντιπροσωπεύει το έγγραφο PDF σας.

```csharp
// Δημιουργήστε ένα νέο έγγραφο PDF
Document document = new Document();
```

#### Βήμα 2: Πρόσβαση σε περιεχόμενο με ετικέτες

Ανακτήστε τη διεπαφή περιεχομένου με ετικέτες, η οποία σας επιτρέπει να εργαστείτε με λογικές δομές εντός του PDF:

```csharp
ITaggedContent taggedContent = document.TaggedContent;
```

#### Βήμα 3: Ορισμός μεταδεδομένων εγγράφου

Ορίστε απαραίτητα μεταδεδομένα όπως τίτλο και γλώσσα για λόγους προσβασιμότητας.

```csharp
taggedContent.SetTitle("Tagged Pdf Document");
taggedContent.SetLanguage("en-US");
```

### Προσθήκη στοιχείων κειμένου με ετικέτες

Για να βελτιώσετε την προσβασιμότητα, προσθέστε στοιχεία κειμένου με σημασιολογική σημασία.

#### Στοιχείο κεφαλίδας

Δημιουργήστε ένα στοιχείο κεφαλίδας για να ορίσετε τη δομή επικεφαλίδας του εγγράφου:

```csharp
HeaderElement headerElement = taggedContent.CreateHeaderElement();
headerElement.ActualText = "Heading 1";
taggedContent.RootElement.AppendChild(headerElement);
```

#### Στοιχεία Παραγράφων

Προσθέστε πολλές παραγράφους για να δημιουργήσετε το κύριο μέρος του περιεχομένου του PDF σας.

```csharp
ParagraphElement paragraphElement1 = taggedContent.CreateParagraphElement();
paragraphElement1.ActualText = "test1";
taggedContent.RootElement.AppendChild(paragraphElement1);

// Επαναλάβετε για τις υπόλοιπες παραγράφους...
```

### Αποθήκευση του PDF με ετικέτα

Αφού προσθέσετε όλα τα απαραίτητα στοιχεία, αποθηκεύστε το έγγραφό σας:

```csharp
document.Save("PDFwithTaggedText.pdf");
```

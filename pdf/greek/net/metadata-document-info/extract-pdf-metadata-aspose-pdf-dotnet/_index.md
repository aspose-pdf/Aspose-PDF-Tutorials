## Οδηγός Εφαρμογής

### Άνοιγμα και ανάγνωση αρχείου PDF
Για να εξαγάγετε μεταδεδομένα, χρησιμοποιήστε το `PdfFileInfo` τάξη:
```csharp
using Aspose.Pdf.Facades;
using System;

namespace PdfInfoExtractor
{
    public class GetFileInfo
    {
        public static void Run()
        {
            string dataDir = "path_to_your_directory/"; // Ενημέρωση με τη διαδρομή καταλόγου σας

            // Άνοιγμα εγγράφου χρησιμοποιώντας το PdfFileInfo
            PdfFileInfo fileInfo = new PdfFileInfo(dataDir + "GetFileInfo.pdf");

            // Εξαγωγή και εκτύπωση μεταδεδομένων PDF
            Console.WriteLine("Subject: {0}", fileInfo.Subject);
            Console.WriteLine("Title: {0}", fileInfo.Title);
            Console.WriteLine("Keywords: {0}", fileInfo.Keywords);
            Console.WriteLine("Creator: {0}", fileInfo.Creator);
            Console.WriteLine("Creation Date: {0}", fileInfo.CreationDate);
            Console.WriteLine("Modification Date: {0}", fileInfo.ModDate);

            // Ελέγξτε αν το αρχείο είναι έγκυρο PDF και αν είναι κρυπτογραφημένο
            Console.WriteLine("Is Valid PDF: {0}", fileInfo.IsPdfFile);
            Console.WriteLine("Is Encrypted: {0}", fileInfo.IsEncrypted);
        }
    }
}
```

#### Βασικά σημεία:
- **Παράμετροι**: Το `PdfFileInfo` Ο κατασκευαστής δέχεται τη διαδρομή του αρχείου ως παράμετρο.
- **Μέθοδοι και Ιδιότητες**Μέθοδοι όπως `Subject`, `Title`, κ.λπ., παρέχουν πρόσβαση σε μεταδεδομένα PDF. 
- **Επικύρωση αρχείου**: Ελέγξτε αν το έγγραφο είναι έγκυρο χρησιμοποιώντας `IsPdfFile` και αν είναι κρυπτογραφημένο με `IsEncrypted`.

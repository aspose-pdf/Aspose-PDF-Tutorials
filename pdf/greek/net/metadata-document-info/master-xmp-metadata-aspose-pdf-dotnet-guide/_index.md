## Οδηγός Εφαρμογής
Αυτός ο οδηγός αναλύει κάθε λειτουργία σε διαχειρίσιμα βήματα. Θα δούμε πώς να προσθέτουμε, να τροποποιούμε και να καταργούμε ιδιότητες μεταδεδομένων χρησιμοποιώντας το Aspose.PDF για .NET.

### Προσθήκη ιδιοτήτων μεταδεδομένων
#### Επισκόπηση
Η προσθήκη μεταδεδομένων περιλαμβάνει τη δημιουργία ενός νέου `PdfXmpMetadata` αντικείμενο και σύνδεσή του στο αρχείο PDF σας.

**Βήμα 1: Αρχικοποίηση και σύνδεση**
```csharp
// Δημιουργία αντικειμένου PdfXmpMetadata
PdfXmpMetadata xmpMetaData = new PdfXmpMetadata();

// Σύνδεση αρχείου pdf με το αντικείμενο
xmpMetaData.BindPdf("SetXMPMetadata.pdf");
```

**Βήμα 2: Προσθήκη πεδίων μεταδεδομένων**
- **Ημερομηνία δημιουργίας**: Ορίστε την ημερομηνία δημιουργίας του εγγράφου.
```csharp
xmpMetaData.Add(DefaultMetadataProperties.CreateDate, DateTime.Now.ToString());
```
- **Εργαλείο δημιουργού**: Καθορίστε το λογισμικό που χρησιμοποιήθηκε για τη δημιουργία του PDF.
```csharp
xmpMetaData.Add(DefaultMetadataProperties.CreatorTool, "Your Application Name");
```

### Τροποποίηση ιδιοτήτων μεταδεδομένων
#### Επισκόπηση
Για να ενημερώσετε τις υπάρχουσες ιδιότητες μεταδεδομένων, αποκτήστε άμεση πρόσβαση σε αυτές χρησιμοποιώντας τα κλειδιά τους.

**Αλλαγή Υπάρχουσας Ιδιότητας**
```csharp
// Τροποποίηση μεταδεδομένων ημερομηνίας δημιουργίας
xmpMetaData[DefaultMetadataProperties.MetadataDate] = DateTime.Now.ToString();
```

### Αφαίρεση ιδιοτήτων μεταδεδομένων
#### Επισκόπηση
Η αφαίρεση περιττών μεταδεδομένων είναι απλή με το `Remove` μέθοδος.

**Βήμα 1: Κατάργηση ανεπιθύμητων ιδιοτήτων**
```csharp
// Αφαίρεση ημερομηνίας τροποποίησης εάν δεν χρειάζεται
xmpMetaData.Remove(DefaultMetadataProperties.ModifyDate);
```

### Προσθήκη προσαρμοσμένων χώρων ονομάτων και ιδιοτήτων
#### Επισκόπηση
Για προσαρμοσμένες ιδιότητες, πρέπει πρώτα να καταχωρίσετε ένα πρόθεμα χώρου ονομάτων.

**Βήμα 1: Καταχώριση προθέματος χώρου ονομάτων**
```csharp
// Καταχώρηση ενός προσαρμοσμένου URI χώρου ονομάτων
xmpMetaData.RegisterNamespaceURI("customNamespace", "http://www.customNameSpaces.com/ns/");
```

**Βήμα 2: Προσθήκη προσαρμοσμένης ιδιότητας**
```csharp
// Προσθήκη ιδιότητας που ορίζεται από τον χρήστη με το πρόθεμα
xmpMetaData.Add("customNamespace:UserPropertyName", "Custom Value");
```

### Αποθήκευση και Κλείσιμο
Αφού κάνετε τις αλλαγές, αποθηκεύστε τις ξανά στο αρχείο PDF:

```csharp
// Αποθήκευση μεταδεδομένων xmp στο αρχείο pdf
xmpMetaData.Save("SetXMPMetadata_out.pdf");

// Κλείστε το αντικείμενο
xmpMetaData.Close();
```

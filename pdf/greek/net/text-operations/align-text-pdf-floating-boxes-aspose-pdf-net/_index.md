## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία ευθυγράμμισης κειμένου μέσα σε αιωρούμενα πλαίσια σε διαχειρίσιμα βήματα.

### Δημιουργία και ευθυγράμμιση αιωρούμενων πλαισίων

#### Επισκόπηση

Τα αιωρούμενα πλαίσια σάς επιτρέπουν να ελέγχετε την τοποθέτηση κειμένου ανεξάρτητα από τη ροή της σελίδας, ιδανικά για πλαϊνές γραμμές ή λεζάντες. Θα δημιουργήσουμε τρία αιωρούμενα πλαίσια ευθυγραμμισμένα σε διαφορετικές κάθετες θέσεις (κάτω, κέντρο, επάνω) σε μια σελίδα PDF.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργήστε το έγγραφο και προσθέστε μια σελίδα**

```csharp
// Αρχικοποίηση μιας νέας παρουσίας εγγράφου
doc = new Aspose.Pdf.Document();
doc.Pages.Add(); // Προσθέτει μια νέα σελίδα στο έγγραφο
```

**2. Ορισμός Πλωτού Πλαισίου με Στοίχιση από Κάτω**

```csharp
// Δημιουργήστε ένα πλωτό πλαίσιο για στοίχιση από κάτω
Aspose.Pdf.FloatingBox floatBox = new Aspose.Pdf.FloatingBox(100, 100);
floatBox.VerticalAlignment = VerticalAlignment.Bottom;
floatBox.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right;

// Προσθήκη κειμένου στο πλωτό πλαίσιο
doc.Pages[1].Paragraphs.Add(floatBox.Paragraphs.Add(new TextFragment("FloatingBox_bottom")));

// Ορισμός περιγράμματος για ορατότητα
floatBox.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**3. Ορισμός Πλωτού Πλαισίου με Στοίχιση Κέντρου**

```csharp
// Δημιουργήστε ένα πλωτό πλαίσιο για στοίχιση στο κέντρο
doc.Pages[1].Paragraphs.Add(floatBox = new Aspose.Pdf.FloatingBox(100, 100) {
    VerticalAlignment = VerticalAlignment.Center,
    HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right
});

floatBox.Paragraphs.Add(new TextFragment("FloatingBox_center"));
floatBox.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**4. Ορισμός Πλωτού Πλαισίου με Στοίχιση από Πάνω**

```csharp
// Δημιουργήστε ένα πλωτό πλαίσιο για στοίχιση από πάνω
doc.Pages[1].Paragraphs.Add(floatBox2 = new Aspose.Pdf.FloatingBox(100, 100) {
    VerticalAlignment = VerticalAlignment.Top,
    HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right
});

floatBox2.Paragraphs.Add(new TextFragment("FloatingBox_top"));
floatBox2.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, Aspose.Pdf.Color.Blue);
```

**5. Αποθήκευση του εγγράφου**

```csharp
// Ορίστε τον κατάλογο εξόδου και αποθηκεύστε το έγγραφο
string dataDir = RunExamples.GetDataDir_AsposePdf_Text();
doc.Save(dataDir + "FloatingBox_alignment_review_out.pdf");
```

### Επεξήγηση Βασικών Παραμέτρων

- **Διαστάσεις FloatingBox (100, 100):** Ορίζει το πλάτος και το ύψος.
- **Κατακόρυφη Στοίχιση:** Ελέγχει την κατακόρυφη τοποθέτηση μέσα στη σελίδα.
- **Οριζόντια Στοίχιση:** Προσαρμόζει την οριζόντια ευθυγράμμιση σε σχέση με άλλα στοιχεία.
- **Πληροφορίες για τα σύνορα:** Προσαρμόζει την εμφάνιση του περιγράμματος για καλύτερη ορατότητα κατά την ανάπτυξη.

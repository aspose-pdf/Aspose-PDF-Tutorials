# Πώς να επισημάνετε κείμενο σε PDF χρησιμοποιώντας το Aspose.PDF .NET

Μία από τις συνηθισμένες προκλήσεις που αντιμετωπίζουν οι προγραμματιστές είναι η επισήμανση κειμένου μέσα σε αυτά τα έγγραφα για καλύτερη αναγνωσιμότητα ή έμφαση. Αυτό μπορεί να είναι ιδιαίτερα χρήσιμο όταν ασχολείστε με μεγάλους όγκους δεδομένων όπου συγκεκριμένες πληροφορίες πρέπει να ξεχωρίζουν. Χρησιμοποιώντας το Aspose.PDF .NET, αυτό το σεμινάριο θα σας δείξει πώς να αναζητάτε και να επισημαίνετε αποτελεσματικά κείμενο σε ένα έγγραφο PDF χρησιμοποιώντας κανονικές εκφράσεις, σχεδιάζοντας ορθογώνια γύρω από το κείμενο που βρέθηκε.

**Τι θα μάθετε:**
- Πώς να χρησιμοποιήσετε το Aspose.PDF .NET για να βρείτε κείμενο μέσα σε ένα PDF.
- Τεχνικές για την επισήμανση συγκεκριμένων φράσεων ή λέξεων σχεδιάζοντας ορθογώνια γύρω τους.
- Ρύθμιση παραμέτρων επιλογών αναζήτησης, όπως η μη διάκριση πεζών-κεφαλαίων για πιο ευέλικτη αναζήτηση κειμένου.

## Οδηγός Εφαρμογής
Ας δούμε πώς υλοποιείται η λειτουργία επισήμανσης κειμένου χρησιμοποιώντας το Aspose.PDF.

### Χαρακτηριστικό: Αναζήτηση κειμένου και σχεδίαση ορθογωνίου
Αυτό το μέρος του κώδικά μας δείχνει πώς να αναζητήσετε κείμενο μέσα σε ένα έγγραφο PDF χρησιμοποιώντας κανονικές εκφράσεις και να σχεδιάσετε ορθογώνια γύρω του. Δείτε πώς το πετυχαίνουμε αυτό:

#### Άνοιγμα εγγράφου
Αρχικά, φορτώστε το αρχείο PDF στο `Document` αντικείμενο.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document document = new Document(dataDir + "SearchAndGetTextFromAll.pdf");
```

#### Ρύθμιση αναζήτησης κειμένου με κανονικές εκφράσεις
Δημιουργήστε ένα `TextFragmentAbsorber` για να βρείτε όλο το κείμενο που ταιριάζει με την καθορισμένη κανονική έκφραση. Εδώ, χρησιμοποιούμε `[\S]+`, η οποία βρίσκει ακολουθίες χαρακτήρων που δεν είναι κενά διαστήματα.
```csharp
TextFragmentAbsorber textAbsorber = new TextFragmentAbsorber(@"[\S]+");
TextSearchOptions textSearchOptions = new TextSearchOptions(true);
textAbsorber.TextSearchOptions = textSearchOptions;
document.Pages.Accept(textAbsorber); 
```
Ο `TextSearchOptions` με μια παράμετρο true καθιστά την αναζήτηση μη ευαίσθητη σε πεζά-κεφαλαία.

#### Σχεδίαση ορθογωνίων γύρω από το κείμενο που βρέθηκε
Στη συνέχεια, κάντε επανάληψη σε κάθε εντοπισμένο `TextFragment`σχεδιάζοντας ορθογώνια γύρω τους.
```csharp
var editor = new PdfContentEditor(document); 

foreach (TextFragment textFragment in textAbsorber.TextFragments)
{
    foreach (TextSegment textSegment in textFragment.Segments)
    {
        DrawBox(editor, textFragment.Page.Number, textSegment, Color.Red);
    }
}
```

#### Αποθήκευση του εγγράφου
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο PDF.
```csharp
dataDir = dataDir + "SearchTextAndDrawRectangle_out.pdf";
document.Save(dataDir);
```

### Χαρακτηριστικό: Κουτί σχεδίασης
Αυτή η βοηθητική συνάρτηση `DrawBox` δείχνει πώς να σχεδιάσετε ένα πολύγωνο (ορθογώνιο) γύρω από συγκεκριμένα τμήματα κειμένου.

#### Ορισμός συντεταγμένων και δημιουργία πολυγώνου
Για κάθε `TextSegment`, ορίστε τις συντεταγμένες του ορθογωνίου και δημιουργήστε ένα πολύγωνο στην καθορισμένη σελίδα PDF.
```csharp
private static void DrawBox(PdfContentEditor editor, int page, TextSegment segment, Color color)
{
    var lineInfo = new LineInfo();
    lineInfo.VerticeCoordinate = new[] {
        (float)segment.Rectangle.LLX, (float)segment.Rectangle.LLY,
        (float)segment.Rectangle.LLX, (float)segment.Rectangle.URY,
        (float)segment.Rectangle.URX, (float)segment.Rectangle.URY,
        (float)segment.Rectangle.URX, (float)segment.Rectangle.LLY
    };
    lineInfo.Visibility = true;
    lineInfo.LineColor = color;

    editor.CreatePolygon(lineInfo, page, new Rectangle(0, 0, 0, 0), null);
}
```

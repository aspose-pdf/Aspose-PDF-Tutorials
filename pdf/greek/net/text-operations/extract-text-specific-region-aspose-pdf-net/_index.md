## Οδηγός Εφαρμογής

### Εξαγωγή κειμένου από μια συγκεκριμένη περιοχή
Ακολουθήστε τα παρακάτω βήματα για να εξαγάγετε κείμενο από καθορισμένες περιοχές μιας σελίδας PDF:

#### Βήμα 1: Φόρτωση του εγγράφου σας
Φορτώστε το έγγραφο PDF στην εφαρμογή σας.
```csharp
Document pdfDocument = new Document("path/to/your/ExtractTextAll.pdf");
```

#### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων του TextAbsorber
Χρήση `TextAbsorber` για να καθορίσετε ποιο κείμενο θέλετε να εξαγάγετε, περιορίζοντας την εξαγωγή σε μια συγκεκριμένη περιοχή σελίδας ορίζοντας συντεταγμένες ορθογωνίου.
```csharp
// Αρχικοποίηση αντικειμένου TextAbsorber
TextAbsorber absorber = new TextAbsorber();

// Περιορισμός αναζήτησης κειμένου εντός των ορίων σελίδας
absorber.TextSearchOptions.LimitToPageBounds = true;

// Ορίστε την ορθογώνια περιοχή από την οποία θα εξαχθεί κείμενο (x, y, πλάτος, ύψος)
absorber.TextSearchOptions.Rectangle = new Aspose.Pdf.Rectangle(100, 200, 250, 350);
```

#### Βήμα 3: Αποδοχή Απορροφητή για Σελίδα Στόχου
Αποδοχή του `TextAbsorber` αντικείμενο στην επιθυμητή σελίδα για να ξεκινήσει η εξαγωγή κειμένου.
```csharp
// Εξαγωγή κειμένου από την πρώτη σελίδα
document.Pages[1].Accept(absorber);
```

#### Βήμα 4: Ανάκτηση και αποθήκευση εξαγόμενου κειμένου
Ανακτήστε το εξαγόμενο κείμενο και αποθηκεύστε το σε ένα αρχείο.
```csharp
// Λήψη του εξαγόμενου κειμένου
string extractedText = absorber.Text;

// Εγγραφή εξαγόμενου κειμένου σε νέο αρχείο
using (TextWriter tw = new StreamWriter("path/to/extracted-text.txt"))
{
    tw.WriteLine(extractedText);
}
```

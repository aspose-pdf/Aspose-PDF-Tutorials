## Οδηγός Εφαρμογής

### Ρύθμιση απόστασης μεταξύ γραμμών με επιλογές μορφοποίησης κειμένου

Αρχικά, ας κατανοήσουμε πώς να ρυθμίσουμε την απόσταση μεταξύ γραμμών χρησιμοποιώντας `TextFormattingOptions`Αυτή η λειτουργία σάς επιτρέπει να ελέγχετε την κάθετη απόσταση μεταξύ των γραμμών κειμένου για μια πιο εκλεπτυσμένη εμφάνιση.

#### Βήμα 1: Ορίστε το έγγραφό σας και ορίστε τις επιλογές μορφοποίησης

Δείτε πώς μπορείτε να δημιουργήσετε ένα νέο έγγραφο και να ρυθμίσετε επιλογές μορφοποίησης με διάστιχο πλήρους μεγέθους:
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

public class SpecifyLineSpacing
{
    public static void Run()
    {
        string dataDir = "Path_to_your_directory";
        string fontFile = dataDir + "HPSimplified.TTF";

        Document doc = new Document();
        TextFormattingOptions formattingOptions = new TextFormattingOptions
        {
            LineSpacing = TextFormattingOptions.LineSpacingMode.FullSize
        };
```

#### Βήμα 2: Φόρτωση της γραμματοσειράς TrueType

Στη συνέχεια, φορτώστε την προσαρμοσμένη γραμματοσειρά σας από ένα αρχείο. Αυτό είναι κρίσιμο για τη διατήρηση της επιθυμητής εμφάνισης του κειμένου σας.
```csharp
if (fontFile != "")
{
    using (FileStream fontStream = File.OpenRead(fontFile))
    {
        TextFragment textFragment = new TextFragment("Hello world");
        textFragment.TextState.Font = FontRepository.OpenFont(fontStream, FontTypes.TTF);
```

#### Βήμα 3: Τοποθετήστε το κείμενό σας και προσθέστε το στο έγγραφο

Ορίστε τη θέση του τμήματος κειμένου σας μέσα στο έγγραφο. Αυτό το βήμα διασφαλίζει ότι το κείμενο εμφανίζεται ακριβώς εκεί που το θέλετε.
```csharp
textFragment.Position = new Position(100, 600);
        textFragment.TextState.FormattingOptions = formattingOptions;

    }
}
```

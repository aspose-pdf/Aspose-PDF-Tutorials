### Εξαγωγή κειμένου από ένα PDF

#### Επισκόπηση

Η εξαγωγή κειμένου είναι ζωτικής σημασίας για την ανάλυση δεδομένων, την αναφορά ή την ενσωμάτωση περιεχομένου σε άλλες εφαρμογές. Δείτε πώς μπορείτε να εξαγάγετε κείμενο αποτελεσματικά:

1. **Πλήρης εξαγωγή εγγράφων**
2. **Εξαγωγή μεμονωμένων σελίδων**

#### Πλήρης εξαγωγή εγγράφων

Για να εξαγάγετε κείμενο από ένα ολόκληρο έγγραφο, χρησιμοποιήστε το `PdfExtractor` τάξη.

```csharp
using Aspose.Pdf.Facades;

namespace PdfTextExtraction
{
    class Program
    {
        static void Main()
        {
            string dataDir = "YOUR_DOCUMENT_DIRECTORY";
            PdfExtractor extractor = new PdfExtractor();
            extractor.Password = ""; // Ορίστε εάν το έγγραφο προστατεύεται με κωδικό πρόσβασης
            extractor.BindPdf(dataDir + "/inFile.pdf");
            extractor.ExtractText();

            // Αποθήκευση εξαγόμενου κειμένου σε αρχείο
            extractor.GetText(dataDir + "/extracted_text.txt");
        }
    }
}
```

**Εξήγηση:**
- `BindPdf`: Επισυνάπτει το PDF για εξαγωγή.
- `ExtractText`: Ξεκινά τη διαδικασία εξαγωγής κειμένου.
- `GetText`: Αποθηκεύει το εξαγόμενο περιεχόμενο σε ένα καθορισμένο αρχείο.

#### Εξαγωγή μεμονωμένων σελίδων

Για την εξαγωγή κειμένου από συγκεκριμένες σελίδες, επαναλάβετε την ίδια διαδικασία σε κάθε σελίδα:

```csharp
using Aspose.Pdf.Facades;

namespace PdfPageTextExtraction
{
    class Program
    {
        static void Main()
        {
            string dataDir = "YOUR_DOCUMENT_DIRECTORY";
            PdfExtractor extractor = new PdfExtractor();
            extractor.Password = "";
            extractor.BindPdf(dataDir + "/inFile.pdf");

            while (extractor.HasNextPageText())
            {
                string fileName = $"{dataDir}/{DateTime.Now.Ticks}_out.txt";
                extractor.GetNextPageText(fileName);
            }
        }
    }
}
```

**Εξήγηση:**
- `HasNextPageText`: Ελέγχει αν υπάρχουν περισσότερες σελίδες για εξαγωγή.
- `GetNextPageText`: Εξάγει κείμενο από την επόμενη σελίδα και το αποθηκεύει.

### Εξαγωγή εικόνων από ένα PDF

#### Επισκόπηση

Η εξαγωγή εικόνων μπορεί να είναι απαραίτητη για την αναπροσαρμογή ή την ανάλυση περιεχομένου. Δείτε πώς μπορείτε να το πετύχετε αυτό χρησιμοποιώντας το Aspose.PDF:

```csharp
using System;
using Aspose.Pdf.Facades;
using System.Drawing.Imaging;

namespace PdfImageExtraction
{
    class Program
    {
        static void Main()
        {
            string dataDir = "YOUR_DOCUMENT_DIRECTORY";
            PdfExtractor extractor = new PdfExtractor();
            extractor.Password = "";
            extractor.BindPdf(dataDir + "/inFile.pdf");
            extractor.ExtractImage();

            while (extractor.HasNextImage())
            {
                string fileName = $"{dataDir}/{DateTime.Now.Ticks}_out.jpg";
                extractor.GetNextImage(fileName, ImageFormat.Jpeg);
            }
        }
    }
}
```

**Εξήγηση:**
- `ExtractImage`: Εξάγει όλες τις εικόνες από το PDF.
- `HasNextImage` και `GetNextImage`: Επαναλαμβάνει την επεξεργασία κάθε εικόνας για εξαγωγή.

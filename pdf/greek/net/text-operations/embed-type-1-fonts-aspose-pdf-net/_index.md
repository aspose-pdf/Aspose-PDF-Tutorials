## Οδηγός Εφαρμογής

### Ενσωμάτωση γραμματοσειρών τυπικού τύπου 1

Η ενσωμάτωση γραμματοσειρών είναι ζωτικής σημασίας για τη διατήρηση της ακεραιότητας και της εμφάνισης του εγγράφου σε διαφορετικές πλατφόρμες. Αυτή η λειτουργία διασφαλίζει ότι το κείμενό σας εμφανίζεται με συνέπεια, ανεξάρτητα από το πού προβάλλεται το PDF.

#### Βήμα προς βήμα διαδικασία

**Φόρτωση του υπάρχοντος εγγράφου σας**

Ξεκινήστε φορτώνοντας το PDF που θέλετε να τροποποιήσετε:
```csharp
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

**Ορισμός ιδιότητας ενσωμάτωσης τυπικών γραμματοσειρών**

Βεβαιωθείτε ότι οι τυπικές γραμματοσειρές τύπου 1 είναι ενσωματωμένες στο έγγραφό σας:
```csharp
pdfDocument.EmbedStandardFonts = true;
```

**Επανάληψη σε σελίδες και γραμματοσειρές**

Στη συνέχεια, επαναλάβετε κάθε σελίδα και τους πόρους γραμματοσειρών της για να ενσωματώσετε τις απαραίτητες γραμματοσειρές:
```csharp
foreach (Page page in pdfDocument.Pages)
{
    if (page.Resources.Fonts != null)
    {
        foreach (Text.Font pageFont in page.Resources.Fonts)
        {
            // Ενσωματώστε τη γραμματοσειρά εάν δεν είναι ήδη ενσωματωμένη
            if (!pageFont.IsEmbedded)
            {
                pageFont.IsEmbedded = true;
            }
        }
    }
}
```

Αυτό διασφαλίζει ότι όλες οι γραμματοσειρές που χρησιμοποιούνται στο έγγραφό σας είναι ενσωματωμένες, διατηρώντας την εμφάνισή τους.

**Αποθήκευση του ενημερωμένου εγγράφου**

Τέλος, αποθηκεύστε το ενημερωμένο PDF σας:
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/EmbeddedFonts-updated_out.pdf");
```

### Εξαγωγή γραμματοσειρών από PDF
#### Επισκόπηση
Ο πρωταρχικός στόχος εδώ είναι η εξαγωγή όλων των γραμματοσειρών που χρησιμοποιούνται σε ένα δεδομένο αρχείο PDF, κάτι που είναι κρίσιμο για τον έλεγχο της συνέπειας των εγγράφων ή την προετοιμασία αρχείων για συγκεκριμένες ροές εργασίας.

#### Βήμα προς βήμα εφαρμογή
**1. Τοποθετήστε το έγγραφο**
Ξεκινήστε φορτώνοντας το έγγραφο PDF σε μια παρουσία του `Document` τάξη.

```csharp
using Aspose.Pdf;

// Αρχικοποίηση αντικειμένου εγγράφου
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
Document doc = new Document(dataDir + "input.pdf");
```

**2. Εξαγωγή γραμματοσειρών**
Χρησιμοποιήστε το `FontUtilities.GetAllFonts()` μέθοδος για την ανάκτηση μιας σειράς γραμματοσειρών που χρησιμοποιούνται στο έγγραφο.

```csharp
Aspose.Pdf.Text.Font[] fonts = doc.FontUtilities.GetAllFonts();

foreach (Aspose.Pdf.Text.Font font in fonts)
{
    Console.WriteLine(font.FontName);
}
```

**Εξήγηση:**
- `FontUtilities.GetAllFonts()`Αυτή η μέθοδος σαρώνει το PDF και επιστρέφει έναν πίνακα με όλες τις μοναδικές γραμματοσειρές που χρησιμοποιούνται.
- Ο βρόχος επαναλαμβάνεται πάνω από κάθε γραμματοσειρά, εξάγοντας το όνομά της στην κονσόλα.

#### Βασικές επιλογές διαμόρφωσης
Ενώ αυτό το παράδειγμα εστιάζει στη βασική εξαγωγή, το Aspose.PDF προσφέρει διάφορες επιλογές διαμόρφωσης για πιο προηγμένες περιπτώσεις χρήσης, όπως ο καθορισμός υποσυνόλων γραμματοσειρών ή η βελτιστοποίηση για ενσωματωμένες γραμματοσειρές.

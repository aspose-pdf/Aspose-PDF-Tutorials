## Οδηγός Εφαρμογής

Ας εμβαθύνουμε σε κάθε λειτουργία για να ανοίξετε και να επεξεργαστείτε αποτελεσματικά PDF.

### Χαρακτηριστικό 1: Άνοιγμα εγγράφου PDF
#### Επισκόπηση
Ξεκινήστε ανοίγοντας το αρχείο PDF για τροποποίηση χρησιμοποιώντας το `PdfFileMend` τάξη.

#### Βήματα Υλοποίησης
**Βήμα 3.1: Δέστε το αρχείο PDF**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
PdfFileMend mender = new PdfFileMend();
mender.BindPdf(dataDir + "/AddText.pdf"); // Ανοίγει και συνδέει το αρχείο PDF για επεξεργασία.
```
*Εξήγηση*: Το `BindPdf` Η μέθοδος ανοίγει το καθορισμένο PDF, προετοιμάζοντάς το για τροποποιήσεις. Βεβαιωθείτε ότι έχετε δώσει τη σωστή διαδρομή προς το έγγραφό σας.

### Λειτουργία 2: Δημιουργία και μορφοποίηση κειμένου
#### Επισκόπηση
Δημιουργήστε ένα μορφοποιημένο αντικείμενο κειμένου που θα εισαχθεί στο PDF.

#### Βήματα Υλοποίησης
**Βήμα 3.2: Ορισμός ιδιοτήτων FormattedText**
```csharp
using System.Drawing;
using System.Text;

string textToInsert = "Aspose - Your File Format Experts!";
Color textColor = Color.AliceBlue;
FontStyle fontStyle = FontStyle.Courier;
float fontSize = 14.0f;

FormattedText formattedText = new FormattedText(
    textToInsert,
    textColor,
    Color.Gray, // Χρώμα φόντου
    fontStyle,
    EncodingType.Winansi,
    true, // Υποστήριξη Unicode
    fontSize
);
```
*Εξήγηση*: Αυτό το απόσπασμα δημιουργεί ένα `FormattedText` αντικείμενο με συγκεκριμένες ιδιότητες όπως περιεχόμενο, χρώματα, στυλ γραμματοσειράς και μέγεθος. Προσαρμόστε αυτές τις παραμέτρους ώστε να ταιριάζουν στις ανάγκες σας.

### Λειτουργία 3: Ρύθμιση παραμέτρων αναδίπλωσης κειμένου και προσθήκη κειμένου
#### Επισκόπηση
Ορίστε τον τρόπο αναδίπλωσης του κειμένου μέσα στο PDF και καθορίστε τη θέση του στη σελίδα.

#### Βήματα Υλοποίησης
**Βήμα 3.3: Ορισμός αναδίπλωσης λέξεων και προσθήκη κειμένου**
```csharp
mender.IsWordWrap = true; // Ενεργοποίηση αναδίπλωσης λέξεων.
int pageNumber = 1;
float lowerLeftX = 100, lowerLeftY = 200, upperRightX = 400, upperRightY = 200;

// Προσθέτει το κείμενο στις καθορισμένες συντεταγμένες στη σελίδα PDF.
mender.AddText(formattedText, pageNumber, lowerLeftX, lowerLeftY, upperRightX, upperRightY);
```
*Εξήγηση*: Το `IsWordWrap` Η ιδιότητα διασφαλίζει ότι το κείμενό σας χωράει εντός καθορισμένων ορίων. Προσαρμόστε τις συντεταγμένες και τις ρυθμίσεις αναδίπλωσης όπως απαιτείται.

### Λειτουργία 4: Αποθήκευση και κλείσιμο εγγράφου PDF
#### Επισκόπηση
Μετά τις τροποποιήσεις, αποθηκεύστε τις αλλαγές σε ένα νέο αρχείο και κλείστε σωστά το αντικείμενο PdfFileMend για να απελευθερώσετε πόρους.

#### Βήματα Υλοποίησης
**Βήμα 3.4: Αποθήκευση και απελευθέρωση πόρων**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
mender.Save(outputDir + "/AddText_out.pdf"); // Αποθηκεύει το τροποποιημένο PDF.
mender.Close(); // Κλείνει το αντικείμενο PdfFileMend, απελευθερώνοντας πόρους.
```
*Εξήγηση*: Το `Save` Η μέθοδος γράφει τις αλλαγές σας σε ένα νέο αρχείο. Να καλείται πάντα `Close()` για την αποφυγή διαρροών μνήμης.

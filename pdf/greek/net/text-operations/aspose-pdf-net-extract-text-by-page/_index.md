## Οδηγός Εφαρμογής
Αφού όλα είναι έτοιμα, ας αναλύσουμε τα βήματα υλοποίησης.

### Αρχικοποίηση του PdfExtractor και σύνδεση PDF
#### Επισκόπηση
Ξεκινήστε αρχικοποιώντας ένα `PdfExtractor` και σύνδεση του PDF προορισμού σας σε αυτό.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Text();

// Άνοιγμα PDF εισαγωγής
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "ExtractText-Page.pdf");
```
#### Εξήγηση
- **`BindPdf(string)`**: Προετοιμάζει τον εξαγωγέα συνδέοντας ένα αρχείο PDF, ρυθμίζοντάς το για εξαγωγή κειμένου.
- Ο `dataDir` μεταβλητά σημεία στον κατάλογό σας που περιέχει το PDF.

### Εξαγωγή κειμένου από κάθε σελίδα
#### Επισκόπηση
Περιηγηθείτε σε κάθε σελίδα του PDF και εξαγάγετε το περιεχόμενό της σε ξεχωριστά αρχεία κειμένου.

```csharp
// Χρήση μεθόδου ExtractText χωρίς παραμέτρους
currentPage = 1;
pdfExtractor.ExtractText();
while (pdfExtractor.HasNextPageText())
{
    using (MemoryStream tempMemoryStream = new MemoryStream())
    {
        pdfExtractor.GetNextPageText(tempMemoryStream);

        string text = Encoding.Unicode.GetString(tempMemoryStream.ToArray());

        File.WriteAllText(dataDir + "output" + currentPage + "_out.txt", text);
        currentPage++;
    }
}
```
#### Εξήγηση
- **`ExtractText()`**: Ξεκινά τη διαδικασία εξαγωγής για όλες τις σελίδες.
- **`HasNextPageText()`**Ελέγχει εάν υπάρχουν περισσότερες σελίδες διαθέσιμες για επεξεργασία.
- **`GetNextPageText(MemoryStream)`**: Εξάγει κείμενο από την επόμενη σελίδα σε ένα `MemoryStream`.
- **Κωδικοποίηση Unicode**Εξασφαλίζει την ακριβή ανάγνωση του εξαγόμενου κειμένου, ιδιαίτερα σημαντικό για τους ειδικούς χαρακτήρες.

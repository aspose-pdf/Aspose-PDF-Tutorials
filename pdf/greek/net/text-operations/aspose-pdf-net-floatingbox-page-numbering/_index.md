### Προσθήκη ενός FloatingBox για αρίθμηση σελίδων

ΕΝΑ `FloatingBox` σας επιτρέπει να τοποθετείτε περιεχόμενο σε συγκεκριμένες θέσεις στις σελίδες PDF σας. Δείτε πώς μπορείτε να το χρησιμοποιήσετε για να προσθέσετε αριθμούς σελίδων:

#### Βήμα 1: Δημιουργία εγγράφου και προσθήκη σελίδων
Αρχικά, δημιουργήστε ένα νέο έγγραφο και προσθέστε μια σελίδα όπου θα τοποθετηθεί το πλωτό πλαίσιο.

```csharp
// Δημιουργήστε ένα νέο έγγραφο PDF
Document document = new Document();

// Προσθήκη σελίδας στο έγγραφο pdf
Page page = document.Pages.Add();
```

#### Βήμα 2: Αρχικοποίηση του FloatingBox

Δημιουργήστε μια παρουσία του `FloatingBox` με τις επιθυμητές διαστάσεις και τοποθετήστε το κατάλληλα στη σελίδα σας.

```csharp
// Αρχικοποίηση ενός FloatingBox με πλάτος και ύψος
FloatingBox box1 = new FloatingBox(140, 80);

// Ρυθμίστε την αριστερή και την επάνω θέση για ακριβή τοποθέτηση
box1.Left = 2;
box1.Top = 10;

// Προσθήκη κειμένου αρίθμησης σελίδων στις παραγράφους του FloatingBox
box1.Paragraphs.Add(new Text.TextFragment("Page: ($p/ $P )"));

// Προσθήκη του αιωρούμενου πλαισίου στην τρέχουσα σελίδα
page.Paragraphs.Add(box1);
```

**Εξήγηση:**  
- `FloatingBox(140, 80)`: Ορίζει το μέγεθος του κουτιού.
- `$p` και `$P`: Πλήκτρα κράτησης θέσης για τις τρέχουσες και τις συνολικές σελίδες.

#### Βήμα 3: Αποθήκευση του εγγράφου

Τέλος, αποθηκεύστε το έγγραφό σας σε μια καθορισμένη τοποθεσία.

```csharp
// Ορισμός διαδρομής εξόδου
string outputPath = "YOUR_OUTPUT_DIRECTORY/PageNumberinHeaderFooterUsingFloatingBox_out.pdf";

// Αποθήκευση του εγγράφου
document.Save(outputPath);
```

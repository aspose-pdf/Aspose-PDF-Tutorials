## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα δείξουμε πώς να εφαρμόσετε στυλιζαρισμένες δομές κειμένου σε PDF με ετικέτες με το Aspose.PDF. Θα επικεντρωθούμε στη ρύθμιση του εγγράφου σας, στην προσθήκη ετικετών και στην εφαρμογή στυλ σε στοιχεία κειμένου.

### Δημιουργία εγγράφου PDF με ετικέτες
#### Αρχικοποίηση του εγγράφου
Ξεκινήστε δημιουργώντας μια παρουσία του `Document` τάξη:
```csharp
// Δημιουργήστε ένα νέο αντικείμενο εγγράφου
Document document = new Document();
```
Αυτό αρχικοποιεί ένα κενό έγγραφο PDF το οποίο μπορείτε να χειριστείτε περαιτέρω.

#### Πρόσβαση σε περιεχόμενο με ετικέτες
Για να εργαστείτε με PDF με ετικέτες, αποκτήστε πρόσβαση στο `TaggedContent` ιδιοκτησία:
```csharp
ITaggedContent taggedContent = document.TaggedContent;
```
Αυτό παρέχει μια διεπαφή για την προσθήκη και τη διαμόρφωση στοιχείων χρησιμοποιώντας ετικέτες.

### Στυλ κειμένου σε στοιχείο παραγράφου
#### Ορισμός τίτλου και γλώσσας
Πριν προσθέσετε περιεχόμενο, ορίστε τον τίτλο και τη γλώσσα του εγγράφου σας για καλύτερη προσβασιμότητα:
```csharp
taggedContent.SetTitle("Tagged Pdf Document");
taggedContent.SetLanguage("en-US");
```
Αυτά τα χαρακτηριστικά βοηθούν τους αναγνώστες οθόνης να κατανοήσουν τη δομή και το γλωσσικό πλαίσιο.

#### Δημιουργία στοιχείου παραγράφου
Δημιουργήστε ένα `ParagraphElement` για να διατηρήσετε το στυλιζαρισμένο κείμενό σας:
```csharp
ParagraphElement p = taggedContent.CreateParagraphElement();
taggedContent.RootElement.AppendChild(p);
```
Αυτό το στοιχείο χρησιμεύει ως δοχείο για το στυλιζαρισμένο κείμενό σας.

#### Εφαρμογή στυλ κειμένου
Εφαρμόστε διάφορα στυλ στο κείμενο της παραγράφου, όπως μέγεθος γραμματοσειράς, χρώμα και στυλ:
```csharp
p.StructureTextState.FontSize = 18F; // Ορισμός μεγέθους γραμματοσειράς
p.StructureTextState.ForegroundColor = Color.Red; // Ορισμός χρώματος κειμένου
p.StructureTextState.FontStyle = FontStyles.Italic; // Ορισμός στυλ κειμένου

p.SetText("Red italic text.");
```
### Αποθήκευση του εγγράφου PDF με ετικέτα
Τέλος, αποθηκεύστε το έγγραφό σας σε ένα αρχείο:
```csharp
document.Save(dataDir + "StyledTaggedPdf.pdf");
```
Αυτό το βήμα διασφαλίζει ότι όλες οι αλλαγές εγγράφονται στον δίσκο, δημιουργώντας ένα PDF με ετικέτες στυλ.

## Οδηγός Εφαρμογής

### Βήμα 1: Ανοίξτε το έγγραφο PDF σας

Αρχικά, φορτώστε το έγγραφο PDF στην εφαρμογή σας:

```csharp
// Φόρτωση του εγγράφου PDF
Document pdfDocument = new Document("input.pdf");
```

Αυτό αρχικοποιεί ένα `Document` αντικείμενο που αντιπροσωπεύει το αρχείο PDF σας.

### Βήμα 2: Ρύθμιση παραμέτρων επιλογών εξαγωγής κειμένου

Ρύθμιση επιλογών εξαγωγής κειμένου χρησιμοποιώντας το `TextExtractionOptions` κλάση. Αυτό σας επιτρέπει να καθορίσετε πώς θα πρέπει να εξαχθεί το κείμενο:

```csharp
// Ορισμός λειτουργίας εξαγωγής κειμένου (Καθαρή ή Ακατέργαστη)
TextExtractionOptions textExtOptions = new TextExtractionOptions(TextExtractionOptions.TextFormattingMode.Pure);
```

### Βήμα 3: Εξαγωγή κειμένου από κάθε σελίδα

Επαναλάβετε κάθε σελίδα του εγγράφου, εξαγάγετε κείμενο και προσαρτήστε το σε ένα εργαλείο δημιουργίας συμβολοσειρών:

```csharp
StringBuilder builder = new StringBuilder();
string extractedText;

foreach (Page pdfPage in pdfDocument.Pages)
{
    using (MemoryStream textStream = new MemoryStream())
    {
        // Δημιουργία TextDevice για εξαγωγή
        TextDevice textDevice = new TextDevice();
        textDevice.ExtractionOptions = textExtOptions;
        
        // Εξαγωγή κειμένου από τη σελίδα και αποθήκευση στη ροή μνήμης
        textDevice.Process(pdfPage, textStream);
        
        extractedText = Encoding.Unicode.GetString(textStream.ToArray());
        builder.Append(extractedText);
    }
}
```

### Βήμα 4: Αποθήκευση εξαγόμενου κειμένου

Τέλος, γράψτε το εξαγόμενο κείμενο σε ένα αρχείο εξόδου:

```csharp
// Ορίστε τη διαδρομή για το αρχείο κειμένου εξόδου
string outputPath = "input_Text_Extracted_out.txt";

// Αποθήκευση του εξαγόμενου κειμένου σε ένα αρχείο
File.WriteAllText(outputPath, builder.ToString());
Console.WriteLine("\nText extracted successfully using text device from page of PDF Document.\nFile saved at " + outputPath);
```

## Οδηγός Εφαρμογής

### Μέτρηση πλάτους συμβολοσειράς με γραμματοσειρά

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει τη μέτρηση του πλάτους μεμονωμένων χαρακτήρων χρησιμοποιώντας μια συγκεκριμένη γραμματοσειρά στο Aspose.PDF για .NET.

#### Οδηγός βήμα προς βήμα
**Αρχικοποίηση και ρύθμιση της γραμματοσειράς:**
Αρχικά, αρχικοποιήστε τη γραμματοσειρά Arial από το αποθετήριο:
```csharp
Aspose.Pdf.Text.Font font = Aspose.Pdf.Text.FontRepository.FindFont("Arial");
```
Ο `FontRepository` είναι μια συλλογή που περιέχει διάφορες γραμματοσειρές διαθέσιμες στο Aspose.PDF.

**Μέτρηση πλάτους χαρακτήρων:**
Για να μετρήσετε το πλάτος ενός χαρακτήρα, χρησιμοποιήστε το `MeasureString` μέθοδος:
```csharp
double measureA = font.MeasureString("A", 14);
```
Εδώ, μετράμε το πλάτος του χαρακτήρα 'A' στο μέγεθος 14. Το `MeasureString` Η συνάρτηση επιστρέφει μια διπλή τιμή που αντιπροσωπεύει το πλάτος σε σημεία.

**Επικύρωση της μέτρησης:**
Βεβαιωθείτε ότι η μέτρηση είναι όπως αναμένεται:
```csharp
if (Math.Abs(measureA - 9.337) > 0.001)
    throw new Exception("Unexpected font string measure!");
```
Αυτή η επικύρωση ελέγχει εάν το μετρούμενο πλάτος βρίσκεται εντός ενός αποδεκτού εύρους της αναμενόμενης τιμής.

### Μέτρηση πλάτους συμβολοσειράς με το TextState

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τη μέτρηση του πλάτους των χαρακτήρων χρησιμοποιώντας ένα `TextState` αντικείμενο, το οποίο προσφέρει επιπλέον ευελιξία.

#### Οδηγός βήμα προς βήμα
**Ορισμός Κατάστασης Κειμένου:**
Αρχικά, ρυθμίστε το `TextState`:
```csharp
Aspose.Pdf.Text.TextState ts = new Aspose.Pdf.Text.TextState();
ts.Font = font;
ts.FontSize = 14;
```
Εδώ, συσχετίζουμε τη γραμματοσειρά Arial και ορίζουμε μέγεθος 14.

**Μέτρηση πλάτους χαρακτήρων με το TextState:**
Χρήση `TextState` να μετρήσω:
```csharp
double measureZ = ts.MeasureString("z");
```
Αυτή η μέθοδος παρέχει έναν εναλλακτικό τρόπο για τον υπολογισμό του πλάτους συμβολοσειράς, αξιοποιώντας τη διαμόρφωση εντός `TextState`.

**Επικύρωση της μέτρησης:**
Βεβαιωθείτε ότι η μέτρηση είναι ακριβής:
```csharp
if (Math.Abs(measureZ - 7.0) > 0.001)
    throw new Exception("Unexpected font string measure!");
```

### Σύγκριση μετρήσεων συμβολοσειράς γραμματοσειράς και TextState

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να συγκρίνετε μετρήσεις που λαμβάνονται και από τα δύο `Font` και `TextState`.

#### Οδηγός βήμα προς βήμα
**Επανάληψη χαρακτήρων:**
Επανάληψη σε μια σειρά χαρακτήρων:
```csharp
for (char c = 'A'; c <= 'z'; c++)
{
    double fnMeasure = font.MeasureString(c.ToString(), 14);
    double tsMeasure = ts.MeasureString(c.ToString());

    if (Math.Abs(fnMeasure - tsMeasure) > 0.001)
        throw new Exception("Font and state string measuring doesn't match!");
}
```
Αυτός ο βρόχος συγκρίνει τις μετρήσεις και από τις δύο μεθόδους, διασφαλίζοντας τη συνέπεια.

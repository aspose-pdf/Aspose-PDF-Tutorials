## Οδηγός Εφαρμογής

### Αναζήτηση κειμένου σε έγγραφο PDF

**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να αναζητήσετε συγκεκριμένο κείμενο μέσα σε ένα έγγραφο PDF και να εξαγάγετε αυτά τα τμήματα για περαιτέρω επεξεργασία.

#### Βήμα 1: Ορίστε τη διαδρομή προς το έγγραφο PDF σας
Ξεκινήστε ορίζοντας τη διαδρομή του αρχείου σας. Αντικατάσταση `YOUR_DOCUMENT_DIRECTORY` με τον πραγματικό κατάλογο που περιέχει το PDF σας.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/SearchAndGetTextFromAll.pdf");
```

#### Βήμα 2: Δημιουργήστε μια παρουσία TextFragmentAbsorber

Ο `TextFragmentAbsorber` Η κλάση χρησιμοποιείται για τον εντοπισμό κειμένου μέσα στο έγγραφο. Μπορείτε να καθορίσετε τη φράση αναζήτησης κατά τη δημιουργία μιας παρουσίας.

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("text");
```

#### Βήμα 3: Αποδοχή του απορροφητή για επεξεργασία

Για να επεξεργαστείτε όλες τις σελίδες του εγγράφου, αποδεχτείτε το `TextFragmentAbsorber`:

```csharp
pdfDocument.Pages.Accept(textFragmentAbsorber);
```

#### Βήμα 4: Ανάκτηση εξαγόμενων τμημάτων κειμένου

Συλλέξτε τα αποσπάσματα κειμένου που ταιριάζουν με τη φράση αναζήτησής σας.

```csharp
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.TextFragments;
```

#### Βήμα 5: Επαναλάβετε κάθε τμήμα κειμένου

Πρόσβαση και αξιοποίηση ιδιοτήτων κάθε τμήματος:

```csharp
foreach (TextFragment textFragment in textFragmentCollection)
{
    string text = textFragment.Text;
    double xIndent = textFragment.Position.XIndent;
    double yIndent = textFragment.Position.YIndent;
    string fontName = textFragment.TextState.Font.FontName;
    bool isAccessible = textFragment.TextState.Font.IsAccessible;
    bool isEmbedded = textFragment.TextState.Font.IsEmbedded;
    bool isSubset = textFragment.TextState.Font.IsSubset;
    double fontSize = textFragment.TextState.FontSize;
}
```

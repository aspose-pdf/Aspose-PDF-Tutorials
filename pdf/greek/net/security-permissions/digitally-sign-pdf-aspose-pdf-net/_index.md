## Οδηγός Εφαρμογής

### Ψηφιακή υπογραφή εγγράφου PDF
**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να προσθέτετε ψηφιακές υπογραφές σε PDF, διασφαλίζοντας ότι έχουν επαληθευτεί η αυθεντικότητά τους και δεν έχουν παραποιηθεί.

#### Βήμα 1: Προετοιμάστε το περιβάλλον σας
Πριν από την υπογραφή, βεβαιωθείτε ότι το περιβάλλον σας έχει ρυθμιστεί σωστά. Θα χρειαστείτε:
- Ένα αρχείο .pfx για το ψηφιακό πιστοποιητικό
- Το έγγραφο PDF που θέλετε να υπογράψετε
```csharp
string pbxFile = "YOUR_PFX_FILE_PATH"; // Διαδρομή προς το αρχείο .pfx
string inFile = @"YOUR_DOCUMENT_DIRECTORY\DigitallySign.pdf";
string outFile = @"YOUR_OUTPUT_DIRECTORY\DigitallySign_out.pdf";
```

#### Βήμα 2: Φόρτωση του εγγράφου PDF
Φορτώστε το έγγραφο που σκοπεύετε να υπογράψετε χρησιμοποιώντας το Aspose.PDF `Document` τάξη.
```csharp
using (Document document = new Document(inFile))
{
    // Συνεχίστε με τα βήματα υπογραφής...
}
```

#### Βήμα 3: Αρχικοποίηση των αντικειμένων PdfFileSignature και PKCS7
Χρήση `PdfFileSignature` για να χειριστείτε τη διαδικασία υπογραφής. Δημιουργήστε ένα `PKCS7` αντικείμενο, το οποίο αντιπροσωπεύει το ψηφιακό σας πιστοποιητικό.
```csharp
using (PdfFileSignature signature = new PdfFileSignature(document))
{
    PKCS7 pkcs = new PKCS7(pbxFile, "WebSales"); // Χρησιμοποιήστε αρχείο .pfx και κωδικό πρόσβασης
```

#### Βήμα 4: Ορισμός εμφάνισης και δικαιωμάτων υπογραφής
Καθορίστε τη θέση της υπογραφής στη σελίδα και ορίστε την εμφάνισή της.
```csharp
Rectangle rect = new Rectangle(100, 100, 200, 100);
signature.SignatureAppearance = @"YOUR_DOCUMENT_DIRECTORY\aspose-logo.jpg";

// Ορίστε δικαιώματα εγγράφων χρησιμοποιώντας το DocMDPSignature
DocMDPSignature docMdpSignature = new DocMDPSignature(pkcs, DocMDPAccessPermissions.FillingInForms);
```

#### Βήμα 5: Υπογράψτε το έγγραφο
Τέλος, υπογράψτε ψηφιακά το PDF και αποθηκεύστε το.
```csharp
signature.Certify(1, "Signature Reason", "Contact", "Location", true, rect, docMdpSignature);
signature.Save(outFile); // Αποθήκευση του υπογεγραμμένου εγγράφου
}
```

### Επαλήθευση ψηφιακά υπογεγραμμένου εγγράφου PDF
**Επισκόπηση:**
Μετά την υπογραφή, ίσως θελήσετε να επαληθεύσετε τις υπογραφές για να βεβαιωθείτε για την εγκυρότητά τους.

#### Βήμα 1: Φόρτωση του υπογεγραμμένου PDF
Φορτώστε το υπογεγραμμένο PDF χρησιμοποιώντας το Aspose.PDF `Document` τάξη.
```csharp
using (Document document = new Document(outFile))
{
    // Συνεχίστε με τα βήματα επαλήθευσης...
}
```

#### Βήμα 2: Αρχικοποίηση αντικειμένου PdfFileSignature
Αρχικοποίηση ενός `PdfFileSignature` αντίρρηση για τον χειρισμό της επαλήθευσης υπογραφής.
```csharp
using (PdfFileSignature signature = new PdfFileSignature(document))
{
    IList<string> sigNames = signature.GetSignNames(); // Λάβετε τα ονόματα όλων των υπογραφών

    if (sigNames.Count > 0) // Έλεγχος για υπάρχουσες υπογραφές
    {
        string firstSigName = sigNames[0]; // Εστίαση στην πρώτη υπογραφή

        // Επαληθεύστε την υπογραφή
        if (signature.VerifySigned(firstSigName))
        {
            // Ελέγξτε εάν το έγγραφο είναι πιστοποιημένο και ανακτήστε τα δικαιώματα
            if (signature.IsCertified)
            {
                DocMDPAccessPermissions accessPermissions = signature.GetAccessPermissions();

                if (accessPermissions == DocMDPAccessPermissions.FillingInForms) 
                {
                    // Η λογική για τον χειρισμό επαληθευμένων εγγράφων μπορεί να προστεθεί εδώ
                }
            }
        }
    }
}
```

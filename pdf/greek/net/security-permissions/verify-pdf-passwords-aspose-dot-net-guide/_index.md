## Οδηγός Εφαρμογής

Τώρα, ας εξερευνήσουμε τις βασικές λειτουργίες της επαλήθευσης κωδικών πρόσβασης PDF χρησιμοποιώντας το Aspose.PDF σε C#.

### Κατανόηση της επαλήθευσης κωδικού πρόσβασης

Ο στόχος είναι να προσδιοριστεί εάν ένα δεδομένο σύνολο κωδικών πρόσβασης μπορεί να ξεκλειδώσει ένα κρυπτογραφημένο αρχείο PDF. Δείτε πώς μπορείτε να το εφαρμόσετε:

#### Βήμα 1: Φόρτωση του εγγράφου PDF

Φορτώστε το αρχείο PDF πηγής και ελέγξτε αν προστατεύεται με κωδικό πρόσβασης χρησιμοποιώντας `PdfFileInfo`.
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Facades;

// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "path/to/your/files/";

// Φόρτωση αρχείου PDF πηγής
PdfFileInfo info = new PdfFileInfo();
info.BindPdf(dataDir + "IsPasswordProtected.pdf");

// Προσδιορίστε εάν το PDF προέλευσης είναι κρυπτογραφημένο
Console.WriteLine("File is password protected: " + info.IsEncrypted);
```

#### Βήμα 2: Επαναλάβετε τους κωδικούς πρόσβασης

Δοκιμάστε κάθε κωδικό πρόσβασης από μια προκαθορισμένη λίστα και εντοπίστε τυχόν εξαιρέσεις για εσφαλμένους κωδικούς πρόσβασης.
```csharp
String[] passwords = new String[5] { "test", "test1", "test2", "test3", "sample" };

for (int passwordcount = 0; passwordcount < passwords.Length; passwordcount++)
{
    try
    {
        Document doc = new Document(dataDir + "IsPasswordProtected.pdf", passwords[passwordcount]);
        if (doc.Pages.Count > 0)
            Console.WriteLine("Password is correct. Number of pages: " + doc.Pages.Count);
    }
    catch (InvalidPasswordException)
    {
        Console.WriteLine("Password = " + passwords[passwordcount] + " is not correct");
    }
}
```

## Οδηγός Εφαρμογής

### Δημιουργία PDF με ημερομηνία λήξης

#### Επισκόπηση

Θα δημιουργήσουμε ένα απλό PDF και θα ορίσουμε μια ημερομηνία λήξης χρησιμοποιώντας JavaScript μέσα στο αρχείο PDF. Αυτό θα ειδοποιήσει τους χρήστες όταν το έγγραφο έχει λήξει.

#### Βήμα προς βήμα εφαρμογή

**1. Ρύθμιση του εγγράφου**

Ξεκινήστε δημιουργώντας μια παρουσία του `Document` κλάση, η οποία αντιπροσωπεύει το PDF σας:

```csharp
// Δημιουργία αντικειμένου εγγράφου
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

**2. Προσθήκη περιεχομένου στο PDF**

Προσθέστε μια σελίδα και κείμενο στο έγγραφό σας για σκοπούς επίδειξης:

```csharp
// Προσθήκη σελίδας σε συλλογή σελίδων αρχείου PDF
doc.Pages.Add();

// Προσθήκη τμήματος κειμένου σε συλλογή παραγράφων του αντικειμένου σελίδας
doc.Pages[1].Paragraphs.Add(new TextFragment("Hello World..."));
```

**3. Υλοποίηση Λογικής Λήξης με JavaScript**

Χρησιμοποιήστε JavaScript μέσα στο PDF για να επιβάλετε την ημερομηνία λήξης:

```csharp
// Δημιουργήστε αντικείμενο JavaScript για να ορίσετε την ημερομηνία λήξης του PDF
JavascriptAction javaScript = new JavascriptAction(
    "var year=2023;"  // Ενημέρωση για το τρέχον έτος
    + "var month=10;"  // Ορίστε τον επιθυμητό μήνα λήξης
    + "today = new Date(); today = new Date(today.getFullYear(), today.getMonth());"
    + "expiry = new Date(year, month);"
    + "if (today.getTime() > expiry.getTime())"
    + "app.alert('The file is expired. You need a new one.');");

// Ορισμός JavaScript ως ενέργεια ανοίγματος PDF
doc.OpenAction = javaScript;
```

**4. Αποθήκευση του εγγράφου**

Τέλος, αποθηκεύστε το έγγραφό σας με την καθορισμένη λογική λήξης:

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_WorkingDocuments();
dataDir += "SetExpiryDate_out.pdf";
// Αποθήκευση εγγράφου PDF
doc.Save(dataDir);
Console.WriteLine(\nPDF expiry date setup successfully.\nFile saved at " + dataDir);
```

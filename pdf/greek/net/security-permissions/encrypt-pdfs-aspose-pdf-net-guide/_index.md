## Οδηγός Εφαρμογής

### Κρυπτογράφηση αρχείων PDF με κωδικούς πρόσβασης χρήστη και κατόχου

Αυτή η λειτουργία σάς επιτρέπει να προστατεύετε τα αρχεία PDF σας ορίζοντας κωδικούς πρόσβασης χρήστη και κατόχου. Δείτε πώς λειτουργεί:

#### Βήμα 1: Δέστε το έγγραφο PDF
Αρχικά, φορτώστε το έγγραφο που θέλετε να κρυπτογραφήσετε:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        fileSecurity.BindPdf("YOUR_DOCUMENT_DIRECTORY/Encrypt.pdf");
    }
}
```
**Γιατί;** Αυτό το βήμα είναι κρίσιμο καθώς φορτώνει το PDF προορισμού σας στη μνήμη για επεξεργασία.

#### Βήμα 2: Εφαρμογή κρυπτογράφησης
Τώρα, εφαρμόστε κρυπτογράφηση με καθορισμένα δικαιώματα και κρυπτογράφηση AES 256-bit:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        // Κρυπτογράφηση αρχείου με κωδικούς πρόσβασης χρήστη και κατόχου, επιτρέποντας μόνο την εκτύπωση
        fileSecurity.EncryptFile("user", "owner", DocumentPrivilege.Print, KeySize.x256, Algorithm.AES);
    }
}
```
**Επεξήγηση παραμέτρων:**
- `"user"`: Ο κωδικός πρόσβασης που απαιτείται για το άνοιγμα του PDF.
- `"owner"`: Ο κωδικός πρόσβασης για την αλλαγή δικαιωμάτων ή την κατάργηση της κρυπτογράφησης.
- `DocumentPrivilege.Print`Περιορίζει την πρόσβαση μόνο στην εκτύπωση.
- `KeySize.x256`: Καθορίζει κρυπτογράφηση AES 256-bit.
- `Algorithm.AES`Χρησιμοποιεί το Προηγμένο Πρότυπο Κρυπτογράφησης.

#### Βήμα 3: Αποθήκευση του κρυπτογραφημένου αρχείου
Τέλος, αποθηκεύστε το κρυπτογραφημένο έγγραφό σας:

```csharp
class Program {
    static void Main() {
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        fileSecurity.Save("YOUR_OUTPUT_DIRECTORY/Encrypt_out.pdf");
    }
}
```
**Γιατί;** Αυτό το βήμα εγγράφει όλες τις αλλαγές σε ένα νέο αρχείο, διατηρώντας το αρχικό PDF.

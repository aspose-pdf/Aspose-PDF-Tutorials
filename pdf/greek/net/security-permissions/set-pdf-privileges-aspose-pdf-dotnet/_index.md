## Οδηγός Εφαρμογής

Ας αναλύσουμε τον ορισμό δικαιωμάτων PDF σε διαχειρίσιμα βήματα.

### Επισκόπηση: Ορισμός δικαιωμάτων εγγράφου

Αυτή η λειτουργία σάς επιτρέπει να ελέγχετε τι μπορούν να κάνουν οι χρήστες με το PDF σας, όπως εκτύπωση ή αντιγραφή περιεχομένου. Χρησιμοποιώντας το `DocumentPrivilege` κλάση, μπορείτε να ορίσετε και να εφαρμόσετε αυτά τα δικαιώματα αποτελεσματικά.

#### Βήμα 1: Δημιουργήστε ένα `DocumentPrivilege` Αντικείμενο

Ξεκινήστε δημιουργώντας μια παρουσία του `DocumentPrivilege`Αυτό το αντικείμενο θα διατηρήσει όλες τις ρυθμίσεις δικαιωμάτων για το έγγραφο.

```csharp
// Δημιουργήστε το αντικείμενο DocumentPrivileges για να καθορίσετε τις επιτρεπόμενες ενέργειες
DocumentPrivilege privilege = DocumentPrivilege.ForbidAll;
privilege.ChangeAllowLevel = 1; // Επιτρέπονται αλλαγές σε ένα καθορισμένο επίπεδο ασφαλείας
privilege.AllowPrint = true;     // Ενεργοποίηση εκτύπωσης
privilege.AllowCopy = true;      // Ενεργοποίηση αντιγραφής περιεχομένου
```

#### Βήμα 2: Δέστε το αρχείο PDF

Χρήση `PdfFileSecurity` για να συνδέσετε το έγγραφό σας και να το προετοιμάσετε για ρυθμίσεις δικαιωμάτων.

```csharp
// Αρχικοποίηση του PdfFileSecurity με τη διαδρομή αρχείου
class Program
{
    static void Main(string[] args)
    {
        // Αρχικοποίηση του PdfFileSecurity με τη διαδρομή αρχείου
        PdfFileSecurity fileSecurity = new PdfFileSecurity();
        fileSecurity.BindPdf("input.pdf");
```

#### Βήμα 3: Ορισμός δικαιωμάτων εγγράφου

Εφαρμόστε τα δικαιώματα που ορίστηκαν νωρίτερα στο αρχείο PDF χρησιμοποιώντας `SetPrivilege`.

```csharp
// Εφαρμογή ρυθμίσεων δικαιωμάτων στο έγγραφο
fileSecurity.SetPrivilege(privilege);
fileSecurity.Save("output.pdf"); // Αποθήκευση του ενημερωμένου εγγράφου
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Συνηθισμένο πρόβλημα**: Σφάλματα "Δεν βρέθηκε αρχείο".
  - **Διάλυμα**Ελέγξτε ξανά τις διαδρομές αρχείων και βεβαιωθείτε ότι έχουν καθοριστεί σωστά.

- **Δεν ισχύουν δικαιώματα**:
  - **Ελεγχος**: Βεβαιωθείτε `SetPrivilege` καλείται μετά τη σύνδεση του PDF με `PdfFileSecurity`.

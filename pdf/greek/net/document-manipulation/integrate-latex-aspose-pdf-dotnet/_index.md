## Οδηγός Εφαρμογής
### Λειτουργία 1: Δημιουργία και ρύθμιση παραμέτρων εγγράφου με περιεχόμενο LaTeX σε κελιά
#### Επισκόπηση
Αυτή η λειτουργία παρουσιάζει τη δημιουργία ενός νέου εγγράφου PDF, την προσθήκη μιας σελίδας, τον ορισμό περιθωρίων κελιών και την εισαγωγή κειμένου LaTeX ως τμήματος HTML σε ένα κελί πίνακα.
**Βήματα Υλοποίησης:**
##### Βήμα 1: Δημιουργία νέου αντικειμένου εγγράφου
```csharp
// Αρχικοποίηση του αντικειμένου εγγράφου
Document document = new Document();
```
*Γιατί αυτό το βήμα;* Δημιουργώντας ένα `Document` Η παρουσία είναι απαραίτητη για να ξεκινήσετε τη δημιουργία της δομής του αρχείου PDF σας.
##### Βήμα 2: Προσθήκη σελίδας στο έγγραφό σας
```csharp
// Προσθήκη σελίδας στη συλλογή σελίδων του εγγράφου
Page page = document.Pages.Add();
```
*Σκοπός:* Η προσθήκη μιας σελίδας σάς επιτρέπει να τοποθετήσετε περιεχόμενο μέσα σε αυτήν, όπως πίνακες και τμήματα κειμένου.
##### Βήμα 3: Ρύθμιση παραμέτρων περιθωρίων κελιών
```csharp
// Ορισμός κειμένου LaTeX για εισαγωγή στο κελί
string latexText = "@\documentclass{article}\n@\begin{document}\nLaTeX and the document class will normally take care of page layout issues for you.\n@\end{document}";

// Δημιουργήστε έναν νέο πίνακα, μια γραμμή και ένα κελί με καθορισμένα περιθώρια
Aspose.Pdf.Table pdfTable = new Aspose.Pdf.Table();
pdfTable.Rows.Add().Cells.Add(latexText);
MarginInfo marginInfo = new MarginInfo { Left = 20, Right = 20, Top = 20, Bottom = 20 };
pdfTable.DefaultCellPadding = marginInfo;
```
*Διαμόρφωση κλειδιού:* Ο ορισμός περιθωρίων διασφαλίζει ότι το περιεχόμενο μέσα στο κελί σας είναι ευθυγραμμισμένο και οπτικά ισορροπημένο.
##### Βήμα 4: Προσθήκη κειμένου LaTeX ως HTMLFragment σε κελί
```csharp
// Δημιουργήστε ένα HtmlFragment που περιέχει το κείμενο LaTeX
HtmlFragment htmlFragment = new HtmlFragment(latexText);
pdfTable.Rows[0].Cells[0].Paragraphs.Add(htmlFragment);

// Προσθέστε τον πίνακα στη συλλογή παραγράφων της σελίδας σας
page.Paragraphs.Add(pdfTable);

// Αποθήκευση του εγγράφου
document.Save("YOUR_DOCUMENT_DIRECTORY/LatextScriptInPdf_out.pdf");
```
*Γιατί;* Μετατροπή κειμένου LaTeX σε `HtmlFragment` σας επιτρέπει να διατηρήσετε τη μορφοποίησή του κατά την ενσωμάτωσή του σε ένα κελί PDF.
### Λειτουργία 2: Δημιουργία και συμπλήρωση πίνακα με περιεχόμενο LaTeX
#### Επισκόπηση
Αυτή η λειτουργία επεκτείνει την προηγούμενη, επιδεικνύοντας τον τρόπο δημιουργίας ενός πίνακα, προσθήκης γραμμών και κελιών και συμπλήρωσής τους με περιεχόμενο LaTeX για πιο σύνθετες δομές εγγράφων.
**Βήματα Υλοποίησης:**
##### Βήμα 1: Δημιουργία και ρύθμιση του πίνακά σας
```csharp
// Αρχικοποίηση ενός νέου αντικειμένου πίνακα
Table table = new Table();
```
*Σκοπός:* Οι πίνακες είναι βασικοί για την οργάνωση δεδομένων σε δομημένες μορφές μέσα στα έγγραφα PDF σας.
##### Βήμα 2: Προσθήκη γραμμής στον πίνακα
```csharp
// Προσθήκη γραμμής σε αυτόν τον πίνακα
Row row = table.Rows.Add();
```
*Εξήγηση:* Η προσθήκη γραμμών σάς επιτρέπει να τοποθετήσετε πολλά κελιά, καθένα από τα οποία ενδεχομένως να περιέχει διαφορετικούς τύπους περιεχομένου ή στυλ.
##### Βήμα 3: Εισαγωγή κειμένου με μορφοποίηση LaTeX σε κελιά
```csharp
// Ορισμός κειμένου LaTeX για εισαγωγή
string latexText2 = "@\documentclass{article}\n@\begin{document}\nLaTeX and the document class will normally take care of page layout issues for you.\n@\end{document}";

// Προσθήκη κελιού με καθορισμένα περιθώρια σε αυτήν τη γραμμή
Cell cell = row.Cells.Add();
MarginInfo marginInfo2 = new MarginInfo { Left = 20, Right = 20, Top = 20, Bottom = 20 };
cell.Margin = marginInfo2;

// Εισαγωγή LaTeX ως HTMLFragment στο κελί
HtmlFragment htmlFragment2 = new HtmlFragment(latexText2);
cell.Paragraphs.Add(htmlFragment2);

// Προσθέστε τον πίνακα στη συλλογή παραγράφων της σελίδας σας
page.Paragraphs.Add(table);

// Αποθήκευση του ενημερωμένου εγγράφου
document.Save("YOUR_DOCUMENT_DIRECTORY/LatextScriptInPdf_withTable_out.pdf");
```
*Γιατί;* Χρησιμοποιώντας `HtmlFragment` για το LaTeX διασφαλίζει ότι διατηρείται η σύνθετη μορφοποίηση στο αποτέλεσμα PDF.

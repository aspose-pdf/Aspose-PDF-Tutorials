# Πώς να προσθέσετε ένα υποσέλιδο σε κάθε σελίδα ενός εγγράφου PDF χρησιμοποιώντας το Aspose.PDF για .NET

Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στην προσθήκη υποσέλιδων σε κάθε σελίδα των εγγράφων PDF σας χρησιμοποιώντας το Aspose.PDF για .NET. Θα μάθετε πώς να χρησιμοποιείτε την κλάση PdfFileStamp για να προσαρμόσετε εύκολα το κείμενο του υποσέλιδου σας.

## Οδηγός Εφαρμογής

### Προσθήκη υποσέλιδου σε κάθε σελίδα

Σε αυτήν την ενότητα, θα δείξουμε πώς να προσθέσετε ένα κείμενο υποσέλιδου σε κάθε σελίδα ενός εγγράφου PDF χρησιμοποιώντας το `PdfFileStamp` κλάση από το Aspose.PDF για .NET.

#### Βήμα 1: Δημιουργήστε ένα αντικείμενο PdfFileStamp
Ξεκινήστε δημιουργώντας μια παρουσία του `PdfFileStamp`Αυτό το αντικείμενο σάς επιτρέπει να σφραγίσετε πρόσθετες πληροφορίες στις σελίδες PDF σας.

```csharp
PdfFileStamp fileStamp = new PdfFileStamp();
```

#### Βήμα 2: Δέστε το έγγραφο PDF

Συνδέστε το έγγραφο προέλευσης με το `fileStamp` αντικείμενο. Αντικατάσταση `"YOUR_DOCUMENT_DIRECTORY"` με τη διαδρομή όπου είναι αποθηκευμένο το PDF σας.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
fileStamp.BindPdf(dataDir + "/AddFooter.pdf");
```

#### Βήμα 3: Δημιουργία μορφοποιημένου κειμένου για το υποσέλιδο

Στη συνέχεια, δημιουργήστε ένα `FormattedText` αντικείμενο που καθορίζει την εμφάνιση του κειμένου του υποσέλιδου. Αυτό το βήμα επιτρέπει την προσαρμογή του στυλ, του μεγέθους και του χρώματος της γραμματοσειράς.

```csharp
FormattedText formattedText = new FormattedText(
    "Aspose - Your File Format Experts!", 
    System.Drawing.Color.Blue, 
    System.Drawing.Color.Gray, 
    Aspose.Pdf.Facades.FontStyle.Courier, 
    EncodingType.Winansi, 
    false, 14
);
```

- **Επεξήγηση παραμέτρων:**
  - `text`Το κείμενο του υποσέλιδου.
  - `foregroundColor`: Χρώμα κειμένου.
  - `backgroundColor`: Χρώμα φόντου (προαιρετικό).
  - `fontStyle`: Στυλ γραμματοσειράς που χρησιμοποιείται για το κείμενο.
  - `encodingType`: Τύπος κωδικοποίησης του κειμένου σας.
  - `isHtmlTagSupported`: Λογική τιμή για τον καθορισμό υποστήριξης HTML σε κείμενο.
  - `fontSize`: Το μέγεθος της γραμματοσειράς του υποσέλιδου.

#### Βήμα 4: Προσθήκη υποσέλιδου σε κάθε σελίδα

Χρησιμοποιήστε το `AddFooter` μέθοδος για την εφαρμογή του μορφοποιημένου κειμένου υποσέλιδου με ένα καθορισμένο περιθώριο (σε αυτήν την περίπτωση, 10 μονάδες).

```csharp
fileStamp.AddFooter(formattedText, 10);
```

#### Βήμα 5: Αποθήκευση και κλείσιμο του εγγράφου PDF

Τέλος, αποθηκεύστε το ενημερωμένο έγγραφό σας και αποδεσμεύστε τους πόρους κλείνοντας το `PdfFileStamp` αντικείμενο.

```csharp
fileStamp.Save(dataDir + "/AddFooter_out.pdf");
fileStamp.Close();
```

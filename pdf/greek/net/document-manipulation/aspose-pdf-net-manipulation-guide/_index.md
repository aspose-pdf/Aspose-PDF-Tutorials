## Οδηγός Εφαρμογής
Εξερευνήστε ορισμένες βασικές λειτουργίες που μπορείτε να εφαρμόσετε με το Aspose.PDF.

### Δημιουργία και Επεξεργασία PDF

#### Δημιουργία νέου εγγράφου
Η δημιουργία PDF από την αρχή είναι απλή:

**Αρχικοποίηση εγγράφου:**
```csharp
// Δημιουργήστε ένα νέο αντικείμενο εγγράφου
document = new Document();
```
**Εξήγηση:** Αυτό αρχικοποιεί ένα κενό έγγραφο PDF.

#### Προσθήκη κειμένου και εικόνων
Η προσθήκη περιεχομένου στο PDF σας είναι απλή με το Aspose.PDF.

**Προσθήκη κειμένου στη σελίδα:**
```csharp
Page page = document.Pages.Add();
TextFragment textFragment = new TextFragment("Hello, World!");
page.Paragraphs.Add(textFragment);
```
**Εξήγηση:** Αυτό το απόσπασμα προσθέτει ένα τμήμα κειμένου με την ένδειξη "Γεια σου, Κόσμε!" στη σελίδα που μόλις δημιουργήθηκε.

#### Προσθήκη εικόνων

**Ενσωμάτωση εικόνας:**
```csharp
Image image = new Image();
image.File = "path/to/your/image.jpg";
page.Resources.Images.Add(image);
```
**Εξήγηση:** Φορτώστε και προσθέστε μια εικόνα από μια καθορισμένη διαδρομή στο έγγραφο PDF σας.

### Συγχώνευση και διαχωρισμός εγγράφων

#### Συνδυασμός πολλαπλών PDF

**Συγχώνευση αρχείων PDF:**
```csharp
Document pdf1 = new Document("file1.pdf");
Document pdf2 = new Document("file2.pdf");
// Προσθήκη σελίδων του δεύτερου εγγράφου στο πρώτο
pdf1.Pages.Add(pdf2.Pages);
pdf1.Save("merged.pdf");
```
**Εξήγηση:** Αυτό συγχωνεύεται `file2.pdf` σε `file1.pdf`, αποθηκεύοντάς το ως `merged.pdf`.

### Εξαγωγή Δεδομένων

#### Εξαγωγή κειμένου από PDF

**Ανάκτηση κειμένου:**
```csharp
TextAbsorber textAbsorber = new TextAbsorber();
document.Pages.Accept(textAbsorber);
string extractedText = textAbsorber.Text;
```
**Εξήγηση:** Χρήση `TextAbsorber` για να εξαγάγετε όλο το κείμενο από το έγγραφο.

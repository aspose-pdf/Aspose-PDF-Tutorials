## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα επικεντρωθούμε στο βασικό χαρακτηριστικό: τον διαχωρισμό σελίδων PDF χρησιμοποιώντας ροές.

### Διαχωρισμός σελίδων PDF

Αυτή η λειτουργία σάς επιτρέπει να διαιρέσετε ένα έγγραφο PDF σε ξεχωριστά αρχεία, καθένα από τα οποία περιέχει μία σελίδα. 

#### Βήμα 1: Αρχικοποίηση του PdfFileEditor

Αρχικά, δημιουργήστε μια παρουσία του `PdfFileEditor`, το οποίο παρέχει μεθόδους για τον χειρισμό PDF:

```csharp
using Aspose.Pdf.Facades;

PdfFileEditor pdfEditor = new PdfFileEditor();
```

**Γιατί**: Αυτή η κλάση είναι κρίσιμη για την πρόσβαση στο `SplitToPages` μέθοδος που χρησιμοποιήθηκε αργότερα.

#### Βήμα 2: Ορισμός διαδρομών καταλόγου

Καθορίστε τις διαδρομές καταλόγων εισόδου και εξόδου για να διαχειριστείτε αποτελεσματικά τις τοποθεσίες αρχείων:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**Γιατί**Η οργάνωση των αρχείων βοηθά στη διατήρηση της σαφήνειας, ειδικά σε μεγαλύτερα έργα.

#### Βήμα 3: Ανοίξτε το αρχείο PDF ως ροή

Ανοίξτε το PDF εισαγωγής σας χρησιμοποιώντας το αρχείο PDF που έχετε εισαγάγει. `FileStream` για να το χειριστείτε ως ροή:

```csharp
using System.IO;

FileStream inputStream = new FileStream(dataDir + "/input.pdf", FileMode.Open);
```

**Γιατί**Οι ροές είναι αποδοτικές ως προς τη μνήμη για τον χειρισμό μεγάλων αρχείων.

#### Βήμα 4: Διαχωρίστε το PDF σε σελίδες

Χρησιμοποιώ `SplitToPages`, το οποίο επιστρέφει έναν πίνακα από `MemoryStream` αντικείμενα, καθένα από τα οποία αντιπροσωπεύει μια σελίδα:

```csharp
MemoryStream[] outBuffer = pdfEditor.SplitToPages(inputStream);
```

**Γιατί**Αυτή η μέθοδος διαχωρίζει αποτελεσματικά τις σελίδες χωρίς να δημιουργεί ενδιάμεσα αρχεία.

#### Βήμα 5: Αποθηκεύστε κάθε σελίδα ξεχωριστά

Επαναλάβετε τις ροές που προκύπτουν και αποθηκεύστε τις ως μεμονωμένα PDF χρησιμοποιώντας `FileStream`:

```csharp
int fileNumber = 1;
foreach (MemoryStream aStream in outBuffer)
{
    using (FileStream outStream = new FileStream(outputDir + "/File_" + fileNumber.ToString() + "_out.pdf", FileMode.Create))
    {
        aStream.WriteTo(outStream);
    }
    fileNumber++;
}
```

**Γιατί**Αυτό το βήμα διασφαλίζει ότι κάθε σελίδα αποθηκεύεται ως ξεχωριστό έγγραφο, διευκολύνοντας τη διαχείρισή της.

#### Βήμα 6: Καθαρισμός

Να κλείνετε πάντα τη ροή εισόδου σας μετά την επεξεργασία:

```csharp
inputStream.Close();
```

**Γιατί**Το σωστό κλείσιμο των ροών αποτρέπει τις διαρροές μνήμης και το κλείδωμα πόρων.

## Οδηγός Εφαρμογής

### Άνοιγμα και σύνδεση εγγράφου PDF

Για να χειριστείτε ένα PDF, πρέπει πρώτα να το ανοίξετε. Αυτό το βήμα περιλαμβάνει τη σύνδεση του εγγράφου χρησιμοποιώντας το `PdfPageEditor` τάξη.

#### Βήμα 1: Ρύθμιση διαδρομής αρχείου
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY/input.pdf";
```

#### Βήμα 2: Δέστε το PDF
```csharp
pageEditor.BindPdf(dataDir);
```
Αυτός ο κώδικας συνδέει το καθορισμένο έγγραφο PDF, καθιστώντας το έτοιμο για χειρισμό. `BindPdf` Η μέθοδος φορτώνει το αρχείο στη μνήμη.

### Μετακίνηση προέλευσης σελίδας

Η προσαρμογή του σημείου προέλευσης μιας σελίδας μπορεί να είναι κρίσιμη για τον σχεδιασμό της διάταξης και για σκοπούς παρουσίασης.

#### Βήμα 1: Δημιουργία αρχικού PdfPageEditor
Υποθέτοντας ότι έχετε ήδη βιβλιοδέσει ένα έγγραφο όπως φαίνεται προηγουμένως.

#### Βήμα 2: Ορισμός νέας θέσης προέλευσης
```csharp
pageEditor.MovePosition(100, 100);
```
Αυτό μετατοπίζει την προέλευση από (0, 0) σε (100, 100), επηρεάζοντας τον τρόπο τοποθέτησης του περιεχομένου σε κάθε σελίδα.

### Ορισμός περιστροφών σελίδας

Η περιστροφή σελίδων μπορεί να βοηθήσει στην ευθυγράμμιση του προσανατολισμού του εγγράφου ή για συγκεκριμένους σκοπούς εμφάνισης.

#### Βήμα 1: Προετοιμασία πίνακα περιστροφής
```csharp
Hashtable pageRotations = new Hashtable();
pageRotations.Add(1, 90); // Περιστροφή πρώτης σελίδας κατά 90 μοίρες
pageRotations.Add(2, 180); // Περιστροφή δεύτερης σελίδας κατά 180 μοίρες
pageRotations.Add(3, 270); // Περιστροφή τρίτης σελίδας κατά 270 μοίρες
```

#### Βήμα 2: Αντιστοίχιση περιστροφών στο PageEditor
```csharp
pageEditor.PageRotations = pageRotations;
```
Αυτό το απόσπασμα κώδικα αντιστοιχίζει γωνίες περιστροφής σε συγκεκριμένες σελίδες χρησιμοποιώντας ένα `Hashtable`.

### Ορισμός επιπέδου ζουμ

Η προσαρμογή του επιπέδου ζουμ είναι χρήσιμη για την προετοιμασία εγγράφων που χρειάζονται αλλαγή μεγέθους.

#### Βήμα 1: Ορισμός συντελεστή ζουμ
```csharp
pageEditor.Zoom = 2.0f; // Ορίζει ζουμ 200%
```

Αυτή η μέθοδος τροποποιεί άμεσα τον συντελεστή ζουμ ολόκληρου του εγγράφου, βελτιώνοντας την αναγνωσιμότητα ή την ποιότητα της παρουσίασης.

### Αποθήκευση ενημερωμένου αρχείου PDF

Μόλις ολοκληρωθούν όλες οι τροποποιήσεις, η αποθήκευση του ενημερωμένου PDF είναι απλή.

#### Βήμα 1: Ορισμός διαδρομής εξόδου
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY/SetPageProperties_out.pdf";
```

#### Βήμα 2: Αποθήκευση του εγγράφου
```csharp
pageEditor.Save(outputDir);
```
Αυτή η εντολή εγγράφει όλες τις αλλαγές σε ένα νέο αρχείο, διατηρώντας το αρχικό σας έγγραφο.

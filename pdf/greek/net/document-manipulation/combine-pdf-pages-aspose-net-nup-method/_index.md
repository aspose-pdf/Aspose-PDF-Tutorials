## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Σελίδες PDF N-Up

Αυτή η λειτουργία συνδυάζει πολλές σελίδες από διαφορετικά PDF σε λιγότερες σελίδες χρησιμοποιώντας τη μορφή N-Up, ιδανική για συμπύκνωση εγγράφων χωρίς απώλεια πληροφοριών.

#### Βήμα 1: Δημιουργήστε ένα αντικείμενο PdfFileEditor
```csharp
using Aspose.Pdf.Facades;

// Αρχικοποίηση του αντικειμένου PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

#### Βήμα 2: Ορισμός διαδρομών αρχείων προέλευσης και εξόδου
Καθορίστε διαδρομές για αρχεία PDF εισόδου και αρχεία εξόδου. Αντικαταστήστε τα placeholders με πραγματικούς καταλόγους.
```csharp
string inputFile1 = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
string inputFile2 = "YOUR_DOCUMENT_DIRECTORY/input2.pdf";
string outputFile = "YOUR_OUTPUT_DIRECTORY/MakeNUpUsingPaths_out.pdf";
```

#### Βήμα 3: Δημιουργήστε N-Up
Χρησιμοποιήστε το `MakeNUp` μέθοδος για τον συνδυασμό εγγράφων σε ένα ενιαίο αρχείο εξόδου.
```csharp
// Συνδυάστε τα αρχεία 'input.pdf' και 'input2.pdf' χρησιμοποιώντας τη διαμόρφωση N-Up
pdfEditor.MakeNUp(inputFile1, inputFile2, outputFile);
```

**Επεξήγηση παραμέτρων:**
- `inputFile1`, `inputFile2`: Διαδρομές προς τα αρχεία PDF που θέλετε να συνδυάσετε.
- `outputFile`: Η διαδρομή προορισμού για το συνδυασμένο PDF.

### Χαρακτηριστικό: Ρύθμιση καταλόγου για παραδείγματα Aspose.PDF

Οδηγίες για τη δημιουργία διαδρομών καταλόγων, διασφαλίζοντας την οργανωμένη αποθήκευση αρχείων εισόδου και εξόδου.

## Οδηγός Εφαρμογής
Ας δούμε πώς υλοποιείται η λειτουργικότητα της ενημέρωσης του χρώματος κειμένου συνδέσμου σε ένα έγγραφο PDF.

### Φόρτωση και ανάλυση ενός εγγράφου PDF
Αρχικά, φορτώστε το αρχείο PDF. Αυτό το βήμα αρχικοποιεί το `Document` αντικείμενο που λειτουργεί ως σημείο εισόδου για περαιτέρω χειρισμούς:

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_LinksActions();
Document doc = new Document(dataDir + "UpdateLinks.pdf");
```

### Εντοπισμός σχολίων συνδέσμων
Στη συνέχεια, εντοπίστε τις σχολιασμοί συνδέσμων μέσα στο έγγραφό σας. Αυτό περιλαμβάνει την επανάληψη των σχολιασμών σε μια συγκεκριμένη σελίδα και τον έλεγχο του τύπου τους `LinkAnnotation`.

```csharp
foreach (Annotation annotation in doc.Pages[1].Annotations)
{
    if (annotation is LinkAnnotation)
    {
        // Περαιτέρω επεξεργασία εδώ...
    }
}
```

### Τροποποίηση χρώματος κειμένου
Μόλις εντοπίσετε τις σχολιασμοί συνδέσμων, χρησιμοποιήστε ένα `TextFragmentAbsorber` για να εντοπίσετε αποσπάσματα κειμένου κάτω από αυτούς τους συνδέσμους και να ενημερώσετε το χρώμα τους:

```csharp
TextFragmentAbsorber ta = new TextFragmentAbsorber();
Rectangle rect = annotation.Rect;
rect.LLX -= 10; // Ανάπτυξη περιοχής αναζήτησης
rect.LLY -= 10;
rect.URX += 10;
rect.URY += 10;
ta.TextSearchOptions = new TextSearchOptions(rect);
ta.Visit(doc.Pages[1]);

// Αλλαγή χρώματος κειμένου.
foreach (TextFragment tf in ta.TextFragments)
{
    tf.TextState.ForegroundColor = Color.Red; // Ορισμός σε κόκκινο
}
```

### Αποθήκευση του ενημερωμένου PDF
Τέλος, αποθηκεύστε τις αλλαγές σας:

```csharp
dataDir = dataDir + "UpdateLinkTextColor_out.pdf";
doc.Save(dataDir);
Console.WriteLine("\nLinkAnnotation text color updated successfully.\nFile saved at " + dataDir);
```

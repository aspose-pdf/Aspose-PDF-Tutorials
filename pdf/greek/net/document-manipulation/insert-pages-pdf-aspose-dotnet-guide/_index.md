## Οδηγός Εφαρμογής

### Επισκόπηση της Εισαγωγής Σελίδας
Η εισαγωγή σελίδων μεταξύ συγκεκριμένων ενοτήτων σε ένα PDF μπορεί να βελτιώσει την οργάνωση του εγγράφου ή να προσθέσει το απαραίτητο περιεχόμενο χωρίς χειροκίνητη επεξεργασία. Αυτή η ενότητα σας καθοδηγεί στη χρήση `PdfFileEditor` για να εκτελέσει αποτελεσματικά αυτό το έργο.

#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργία στιγμιότυπου PdfFileEditor**
Ξεκινήστε αρχικοποιώντας το `PdfFileEditor` αντικείμενο, το οποίο παρέχει μεθόδους για τον χειρισμό αρχείων PDF:

```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
```

**2. Προετοιμασία ροών εισόδου και εξόδου**
Ρυθμίστε τις ροές για να διαβάζουν τα αρχεία PDF πηγής σας και να γράφουν την έξοδο:

```csharp
FileStream inputStream = new FileStream("source.pdf", FileMode.Open);
FileStream portStream = new FileStream("pagesToInsert.pdf", FileMode.Open);
FileStream outputStream = new FileStream("output.pdf", FileMode.Create);
```

**3. Εισαγωγή σελίδων χρησιμοποιώντας ροές**
Τώρα, εισαγάγετε σελίδες από ένα PDF σε ένα άλλο σε μια καθορισμένη θέση:

```csharp
// Εισαγωγή σελίδων 1 έως 4 της δεύτερης ροής μετά τη σελίδα 1 της πρώτης ροής
pdfEditor.Insert(inputStream, 1, portStream, 1, 4, outputStream);
```

**Εξήγηση:**
- `inputStream`: PDF πηγής όπου εισάγονται οι σελίδες.
- `1`: Αριθμός σελίδας στο PDF πηγής όπου ξεκινά το νέο περιεχόμενο.
- `portStream`: Ροή με επιπλέον σελίδες για εισαγωγή.
- `1, 4`: Εύρος σελίδων από `portStream` να συμπεριληφθεί.

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Σφάλματα "Δεν βρέθηκε αρχείο"**Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και προσβάσιμες.
- **Προβλήματα μνήμης**Χρησιμοποιήστε τις ροές με σύνεση για να αποφύγετε την υπερβολική χρήση μνήμης.

## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει τα πάντα, ας προχωρήσουμε στην εφαρμογή της λειτουργίας διαίρεσης σελίδων PDF.

### Διαχωρισμός σελίδων χρησιμοποιώντας διαδρομές

#### Επισκόπηση

Θα διαχωρίσουμε συγκεκριμένες σελίδες από ένα αρχείο PDF προέλευσης και θα τις αποθηκεύσουμε ως ξεχωριστά έγγραφα. Αυτό είναι χρήσιμο όταν χρειάζεστε μόνο συγκεκριμένα τμήματα ενός εγγράφου ή θέλετε να διανείμετε τα μέρη ξεχωριστά.

#### Βήμα προς βήμα εφαρμογή

**Δημιουργία αντικειμένου PdfFileEditor**

```csharp
// Αρχικοποίηση του αντικειμένου PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

Αυτό το αντικείμενο θα χειριστεί όλες τις λειτουργίες που σχετίζονται με τον χειρισμό σελίδων PDF.

**Ορισμός διαδρομών πηγής και εξόδου**

Βεβαιωθείτε ότι έχετε έναν κατάλογο με το PDF πηγής σας:

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdfFacades_Pages();

// Ορίστε διαδρομές για αρχεία εισόδου και εξόδου
string inputFile = $"{dataDir}MultiplePages.pdf";
string outputFile = $"{dataDir}SplitPagesUsingPaths_out.pdf";
```

**Εκτελέστε τη λειτουργία διαίρεσης**

Χρήση `SplitFromFirst` για να διαχωρίσετε σελίδες ξεκινώντας από μια συγκεκριμένη σελίδα:

```csharp
// Διαχωρίστε τις τρεις πρώτες σελίδες σε ένα νέο αρχείο PDF
pdfEditor.SplitFromFirst(inputFile, 3, outputFile);
```

- **Επεξήγηση παραμέτρων:**
  - `inputFile`: Διαδρομή προς το PDF πηγής σας.
  - `3`: Αριθμός σελίδων προς διαίρεση από την αρχή.
  - `outputFile`: Διαδρομή προορισμού για το PDF που προκύπτει.

**Συμβουλές αντιμετώπισης προβλημάτων**

- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και προσβάσιμες.
- Ελέγξτε εάν υπάρχουν επαρκή δικαιώματα για την ανάγνωση/εγγραφή αρχείων.
- Επιβεβαιώστε ότι το PDF εισόδου έχει αρκετές σελίδες για να διαιρεθεί όπως καθορίζεται.

## Οδηγός Εφαρμογής

### Λειτουργία 1: Άνοιγμα και σύνδεση εγγράφου PDF

#### Επισκόπηση

Το άνοιγμα και η βιβλιοδεσία ενός εγγράφου PDF είναι το πρώτο σας βήμα σε οποιαδήποτε διαδικασία τροποποίησης. Αυτό σας επιτρέπει να φορτώσετε το PDF στη μνήμη για περαιτέρω λειτουργίες.

**Βήματα:**

##### Αρχικοποίηση Επεξεργαστή Φόρμας

```csharp
// Αρχικοποίηση του αντικειμένου FormEditor
FormEditor formEditor = new FormEditor();
```

Αυτό αρχικοποιεί μια παρουσία του `FormEditor` κλάση, η οποία θα χειριστεί όλες τις λειτουργίες στο αρχείο PDF σας.

##### Ορισμός καταλόγου εγγράφων

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με τη διαδρομή όπου είναι αποθηκευμένα τα αρχεία PDF σας. Αυτό το σύμβολο κράτησης θέσης καταλόγου διασφαλίζει ότι μπορείτε εύκολα να αλλάζετε καταλόγους κατά την ανάπτυξη της εφαρμογής σας.

##### Άνοιγμα και σύνδεση του PDF

```csharp
// Άνοιγμα και σύνδεση του εγγράφου PDF
formEditor.BindPdf(dataDir + "DeleteFormField.pdf");
```

Ο `BindPdf` Η μέθοδος φορτώνει το καθορισμένο αρχείο PDF στη μνήμη, προετοιμάζοντάς το για τροποποίηση. Βεβαιωθείτε ότι η διαδρομή είναι σωστή για να αποφύγετε σφάλματα χρόνου εκτέλεσης.

### Λειτουργία 2: Διαγραφή πεδίου από έγγραφο PDF

#### Επισκόπηση

Η διαγραφή πεδίων (όπως πλαίσια κειμένου ή πλαίσια ελέγχου) μπορεί να βελτιστοποιήσει τα έγγραφά σας αφαιρώντας περιττά στοιχεία.

**Βήματα:**

##### Άνοιγμα και σύνδεση του PDF

Αρχικά, βεβαιωθείτε ότι έχετε ανοίξει και δέσει το PDF όπως περιγράφεται στην προηγούμενη ενότητα.

```csharp
formEditor.BindPdf(dataDir + "DeleteFormField.pdf");
```

##### Αφαίρεση πεδίου

```csharp
// Διαγραφή του καθορισμένου πεδίου από το έγγραφο PDF
formEditor.RemoveField("textfield");
```
Ο `RemoveField` Η μέθοδος παίρνει το όνομα του πεδίου που θέλετε να διαγράψετε. Αντικαταστήστε `"textfield"` με το πραγματικό όνομα του πεδίου προορισμού σας.

### Λειτουργία 3: Αποθήκευση τροποποιημένου εγγράφου PDF

#### Επισκόπηση

Αφού κάνετε αλλαγές, είναι σημαντικό να αποθηκεύσετε την εργασία σας. Αυτό το βήμα διασφαλίζει ότι όλες οι τροποποιήσεις διατηρούνται σε ένα νέο αρχείο.

**Βήματα:**

##### Ορισμός καταλόγου εξόδου

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```
Όπως και στον κατάλογο εισόδου, αντικαταστήστε `"YOUR_OUTPUT_DIRECTORY"` με το πού θέλετε να αποθηκεύσετε τα τροποποιημένα αρχεία.

##### Αποθήκευση του ενημερωμένου αρχείου

```csharp
// Αποθηκεύστε το ενημερωμένο αρχείο σε έναν καθορισμένο κατάλογο
formEditor.Save(outputDir + "DeleteFormField_out.pdf");
```

Ο `Save` Η μέθοδος γράφει τις αλλαγές σας σε ένα νέο αρχείο PDF στην καθορισμένη θέση, διατηρώντας το αρχικό έγγραφο.

## Οδηγός Εφαρμογής
### Διαχωρισμός PDF σε ξεχωριστές σελίδες
Αυτή η λειτουργία σάς επιτρέπει να διαχωρίσετε κάθε σελίδα ενός εγγράφου σε ξεχωριστό αρχείο, κάτι που μπορεί να είναι πολύ χρήσιμο για την οργάνωση και την κοινή χρήση μεμονωμένων ενοτήτων.

#### Βήμα 1: Ανοίξτε το πρωτότυπο έγγραφο
Ξεκινήστε φορτώνοντας το PDF σας χρησιμοποιώντας το `Document` τάξη:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/SplitToPages.pdf");
```
Αυτός ο κώδικας ανοίγει ένα αρχείο PDF από έναν καθορισμένο κατάλογο, προετοιμάζοντάς το για επεξεργασία.

#### Βήμα 2: Επανάληψη σε σελίδες
Περιηγηθείτε σε κάθε σελίδα για να δημιουργήσετε ξεχωριστά έγγραφα:
```csharp
int pageCount = 1;
foreach (Page pdfPage in pdfDocument.Pages)
{
    // Δημιουργήστε ένα νέο έγγραφο για αυτήν τη σελίδα.
    Document newDocument = new Document();
    newDocument.Pages.Add(pdfPage);

    // Αποθηκεύστε το έγγραφο μίας σελίδας ως αρχείο PDF.
    newDocument.Save($"YOUR_OUTPUT_DIRECTORY/page_{pageCount}_out.pdf");
    pageCount++;
}
```
**Εξήγηση**: Αυτό το απόσπασμα επαναλαμβάνεται σε κάθε σελίδα του PDF σας, δημιουργώντας ένα νέο `Document` αντικείμενο για κάθε ένα και αποθηκεύοντάς το ως μεμονωμένο αρχείο.

Σε αυτό το σεμινάριο, θα σας παρουσιάσουμε τη διαδικασία βήμα προς βήμα, διασφαλίζοντας ότι θα αποκτήσετε σαφήνεια και σιγουριά στην εφαρμογή αυτών των αλλαγών.

## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει το περιβάλλον μας, ας ξεκινήσουμε την αλλαγή μεγέθους των περιεχομένων PDF.

### Κατανόηση της αλλαγής μεγέθους περιεχομένου

Η αλλαγή μεγέθους περιεχομένου PDF περιλαμβάνει την προσαρμογή των περιθωρίων ώστε να χωρούν περισσότερες ή λιγότερες πληροφορίες σε μια σελίδα. Αυτό μπορεί να είναι κρίσιμο για τη δημιουργία καθαρότερων και πιο ευανάγνωστων εγγράφων.

#### Βήμα 1: Ανοίξτε το υπάρχον έγγραφο

Ξεκινήστε φορτώνοντας το υπάρχον έγγραφο PDF:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "/input.pdf");
```

#### Βήμα 2: Ορισμός παραμέτρων αλλαγής μεγέθους

Θα ορίσουμε συγκεκριμένα περιθώρια ως ποσοστά των διαστάσεων της σελίδας. Δείτε πώς ορίζετε αυτές τις παραμέτρους:

```csharp
PdfFileEditor.ContentsResizeParameters parameters = new PdfFileEditor.ContentsResizeParameters(
    // Αριστερό περιθώριο: 10% του πλάτους της σελίδας
    PdfFileEditor.ContentsResizeValue.Percents(10),
    
    // Δεξί περιθώριο: 10% του πλάτους της σελίδας
    PdfFileEditor.ContentsResizeValue.Percents(10),
    
    // Άνω περιθώριο: 10% του ύψους σελίδας
    PdfFileEditor.ContentsResizeValue.Percents(10),
    
    // Κάτω περιθώριο: 10% του ύψους σελίδας
    PdfFileEditor.ContentsResizeValue.Percents(10)
);
```

#### Βήμα 3: Αλλαγή μεγέθους περιεχομένου σε συγκεκριμένες σελίδες

Τώρα, εφαρμόστε αυτές τις παραμέτρους για να αλλάξετε το μέγεθος του περιεχομένου σε συγκεκριμένες σελίδες. Εδώ στοχεύουμε την πρώτη και τη δεύτερη σελίδα:

```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
fileEditor.ResizeContents(doc, new int[] { 1, 2 }, parameters);
```

#### Βήμα 4: Αποθήκευση του τροποποιημένου εγγράφου

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο PDF στον επιθυμητό κατάλογο εξόδου:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputDir + "/ResizePageContents_out.pdf");
```

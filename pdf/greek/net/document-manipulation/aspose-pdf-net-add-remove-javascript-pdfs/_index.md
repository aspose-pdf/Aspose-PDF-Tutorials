## Οδηγός Εφαρμογής
Εξερευνήστε δύο βασικές λειτουργίες: την προσθήκη JavaScript σε PDF και την αφαίρεσή της.

### Προσθήκη συναρτήσεων JavaScript σε ένα έγγραφο PDF
Η προσθήκη JavaScript μπορεί να μετατρέψει τα στατικά σας PDF σε δυναμικά εργαλεία. Δείτε πώς μπορείτε να εφαρμόσετε αυτήν τη λειτουργία:

#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει την ενσωμάτωση συναρτήσεων JavaScript στο έγγραφο PDF σας χρησιμοποιώντας το Aspose.PDF για .NET.

#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργία και ρύθμιση του εγγράφου PDF**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Αρχικοποιήστε ένα νέο έγγραφο.
Document doc = new Document();
doc.Pages.Add();  // Προσθήκη σελίδας στο έγγραφο.
```

**2. Προσθήκη συναρτήσεων JavaScript**
Εδώ, θα προσθέσουμε δύο απλές συναρτήσεις με το όνομα `func1` και `func2`.
```csharp
// Ενσωματώστε συναρτήσεις JavaScript στη συλλογή σεναρίων του PDF.
doc.JavaScript["func1"] = "function func1() { hello(); }";
doc.JavaScript["func2"] = "function func2() { hello(); }";

// Αποθηκεύστε το έγγραφο με ενσωματωμένα σενάρια.
doc.Save(dataDir + "/AddJavascript.pdf");
```
*Εξήγηση*Χρησιμοποιούμε `doc.JavaScript` για να προσθέσετε λειτουργίες. Κάθε λειτουργία σχετίζεται με ένα μοναδικό κλειδί, επιτρέποντας εύκολη πρόσβαση και χειρισμό.

**Συμβουλή αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι ο κώδικας JavaScript είναι συντακτικά σωστός και δεν έρχεται σε διένεξη με υπάρχοντα σενάρια στο PDF.

### Αφαίρεση συνάρτησης JavaScript από ένα έγγραφο PDF
Μερικές φορές, ίσως χρειαστεί να καταργήσετε συγκεκριμένες συναρτήσεις JavaScript. Δείτε πώς:

#### Επισκόπηση
Αυτή η ενότητα σάς καθοδηγεί στην αφαίρεση συναρτήσεων JavaScript από ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET.

#### Βήμα προς βήμα εφαρμογή
**1. Φορτώστε το υπάρχον PDF**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Ανοίξτε το έγγραφο που περιέχει τα σενάρια.
Document doc1 = new Document(dataDir + "/AddJavascript.pdf");
```

**2. Εμφάνιση συναρτήσεων JavaScript**
Πριν από την αφαίρεση, είναι χρήσιμο να απαριθμήσετε τις υπάρχουσες λειτουργίες.
```csharp
IList keys = (System.Collections.IList)doc1.JavaScript.Keys;

foreach (string key in keys)
{
    // Εξάγετε το όνομα κάθε συνάρτησης και τον κωδικό της.
    Console.WriteLine($"{key}: {doc1.JavaScript[key]}");
}
```

**3. Αφαιρέστε τη συγκεκριμένη λειτουργία**
Εδώ, αφαιρούμε `func1` από το έγγραφο.
```csharp
// Διαγράψτε τη λέξη 'func1' με το κλειδί της.
doc1.JavaScript.Remove("func1");

// Προαιρετικά, αποθηκεύστε το τροποποιημένο PDF, εάν χρειάζεται.
doc1.Save(dataDir + "/ModifiedJavascript.pdf");
```
*Εξήγηση*Χρησιμοποιήστε το `Remove` μέθοδο με το κλειδί της συνάρτησης για να την εξαλείψετε από τη συλλογή σεναρίων.

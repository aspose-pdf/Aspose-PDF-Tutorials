## Οδηγός Εφαρμογής
Ας αναλύσουμε την εφαρμογή της προσθήκης μιας σφραγίδας κειμένου σε σαφή βήματα.

### Προσθήκη σφραγίδων κειμένου σε PDF (H2)
Αυτή η λειτουργία σάς επιτρέπει να εισάγετε προσαρμοσμένο κείμενο σε οποιαδήποτε σελίδα του εγγράφου PDF σας, κάτι που είναι χρήσιμο για την προσθήκη επωνυμίας ή τη σήμανση εγγράφων με συγκεκριμένα αναγνωριστικά, όπως αριθμούς έκδοσης ή ημερομηνίες.

#### Βήμα 1: Ανοίξτε ένα υπάρχον έγγραφο PDF
Ξεκινήστε φορτώνοντας το έγγραφο PDF όπου θέλετε να προσθέσετε μια σφραγίδα:

```csharp
// Φόρτωση του εγγράφου PDF
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY\\AddTextStamp.pdf");
```

#### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων σφραγίδας κειμένου
Δημιουργήστε ένα `TextStamp` αντικείμενο και διαμορφώστε το με το επιθυμητό περιεχόμενο κειμένου, τοποθέτηση, στυλ και άλλες ιδιότητες.

```csharp
// Δημιουργήστε την αρχική σφραγίδα κειμένου
TextStamp textStamp = new TextStamp("Sample Stamp");

// Ορίστε το φόντο σε true εάν θέλετε τη σφραγίδα πίσω από το υπάρχον κείμενο
textStamp.Background = true;

// Τοποθέτηση της σφραγίδας
textStamp.XIndent = 100;
textStamp.YIndent = 100;

// Περιστρέψτε τη σφραγίδα κατά 90 μοίρες για δυναμική εμφάνιση
textStamp.Rotate = Rotation.on90;

// Ρύθμιση παραμέτρων γραμματοσειράς
textStamp.TextState.Font = FontRepository.FindFont("Arial");
textStamp.TextState.FontSize = 14.0F;
textStamp.TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;
textStamp.TextState.ForegroundColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Aqua);
```

#### Βήμα 3: Προσθήκη της σφραγίδας σε μια σελίδα
Στη συνέχεια, προσθέστε τη σφραγίδα κειμένου στην πρώτη σελίδα του εγγράφου PDF.

```csharp
// Προσθήκη σφραγίδας στην πρώτη σελίδα
pdfDocument.Pages[1].AddStamp(textStamp);
```

#### Βήμα 4: Αποθήκευση του τροποποιημένου εγγράφου
Τέλος, αποθηκεύστε το ενημερωμένο έγγραφο με την προσθήκη της νέας σφραγίδας:

```csharp
// Ορίστε τη διαδρομή εξόδου και αποθηκεύστε το έγγραφο
string outputDir = "YOUR_OUTPUT_DIRECTORY\\AddTextStamp_out.pdf";
pdfDocument.Save(outputDir);
```

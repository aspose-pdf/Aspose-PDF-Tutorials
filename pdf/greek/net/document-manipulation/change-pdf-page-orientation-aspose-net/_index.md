## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα καλύψουμε τη φόρτωση και την επανάληψη σελίδων PDF, καθώς και την προσαρμογή των διαστάσεων της σελίδας.

### Φόρτωση και επανάληψη σελίδων PDF
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα έγγραφο PDF και να κάνετε επανάληψη στις σελίδες του για πρόσβαση ή τροποποίηση.

#### Βήμα 1: Φόρτωση του εγγράφου
```csharp
using System.IO;
using Aspose.Pdf;

// Φορτώστε το αρχείο PDF σας
document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

#### Βήμα 2: Επανάληψη σελίδων
Μπορείτε να επαναλάβετε την περιήγηση σε κάθε σελίδα χρησιμοποιώντας ένα `foreach` βρόχος:
```csharp
foreach (Page page in doc.Pages)
{
    // Πρόσβαση στο ορθογώνιο MediaBox της τρέχουσας σελίδας
    Rectangle r = page.MediaBox;
}
```
Ο `MediaBox` Η ιδιότητα σάς δίνει διαστάσεις και θέση, κάτι που είναι κρίσιμο για την προσαρμογή του προσανατολισμού.

### Προσαρμογή διαστάσεων σελίδας
Η αλλαγή προσανατολισμού περιλαμβάνει την τροποποίηση του πλάτους και του ύψους της σελίδας. Δείτε πώς:

#### Βήμα 1: Υπολογισμός Νέων Διαστάσεων
Για να αλλάξετε μια σελίδα από κατακόρυφο σε οριζόντιο προσανατολισμό ή αντίστροφα, υπολογίστε τις νέες διαστάσεις ως εξής:
```csharp
foreach (Page page in doc.Pages)
{
    Rectangle r = page.MediaBox;

    // Εναλλαγή ύψους και πλάτους για αλλαγή προσανατολισμού
    double newWidth = r.Height;
    double newHeight = r.Width;

    // Εφαρμογή των νέων διαστάσεων στο MediaBox
    page.MediaBox = new Rectangle(0, 0, newWidth, newHeight);
}
```
**Εξήγηση:**
- `r.Height` γίνεται το νέο πλάτος, και `r.Width` γίνεται το νέο ύψος για τον οριζόντιο προσανατολισμό.

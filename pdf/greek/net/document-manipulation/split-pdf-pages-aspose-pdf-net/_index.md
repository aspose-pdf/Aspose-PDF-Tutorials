## Οδηγός Εφαρμογής

### Διαχωρισμός σελίδων PDF σε μαζικά αρχεία

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να διαχωρίσετε συγκεκριμένες σελίδες ενός εγγράφου PDF σε ξεχωριστά αρχεία, διευκολύνοντας την ανεξάρτητη διαχείριση και διανομή ενοτήτων.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1: Αρχικοποίηση του PdfFileEditor**

```csharp
using Aspose.Pdf.Facades;

PdfFileEditor pdfEditor = new PdfFileEditor();
```

**Βήμα 2: Ορισμός σελίδων για διαίρεση**
Προσδιορίστε τα εύρη σελίδων που θέλετε να διαχωρίσετε. Για παράδειγμα:

```csharp
int[][] numberOfPages = new int[][] { new int[] { 1, 2 }, new int[] { 3, 4 } };
```

**Βήμα 3: Διαχωρισμός PDF σε μαζικά αρχεία**
Εκτελέστε τη λειτουργία διαίρεσης και αποθηκεύστε κάθε τμήμα σε ένα `MemoryStream` παράταξη.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

MemoryStream[] outBuffer = pdfEditor.SplitToBulks(dataDir + "MultiplePages.pdf", numberOfPages);
```

**Βήμα 4: Αποθήκευση κάθε ροής που προκύπτει**
Επαναλάβετε τις ροές και αποθηκεύστε την καθεμία ως νέο αρχείο.

```csharp
int fileNumber = 1;
foreach (MemoryStream aStream in outBuffer)
{
    using (FileStream outStream = new FileStream(outputDir + "File_" + fileNumber.ToString() + "_out.pdf", FileMode.Create))
    {
        aStream.WriteTo(outStream);
    }
    fileNumber++;
}
```

### Χειρισμός ροών για επεξεργασία PDF

#### Επισκόπηση
Η διαχείριση ροών είναι απαραίτητη όταν χρειάζεται να επεξεργαστείτε PDF χωρίς άμεση πρόσβαση στο σύστημα αρχείων. Αυτή η τεχνική προσφέρει ευελιξία και αποτελεσματικότητα.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1: Ανάγνωση PDF σε MemoryStream**
Ξεκινήστε διαβάζοντας το PDF σας σε μια ροή μνήμης χρησιμοποιώντας `FileStream`.

```csharp
using (FileStream inputFileStream = new FileStream(dataDir + "MultiplePages.pdf", FileMode.Open))
{
    using (MemoryStream inputStream = new MemoryStream())
    {
        inputFileStream.CopyTo(inputStream);
        inputStream.Position = 0; // Επαναφορά θέσης για περαιτέρω λειτουργίες

        int[][] numberOfPages = new int[][] { new int[] { 1, 2 }, new int[] { 3, 4 } };
        MemoryStream[] outBuffer = pdfEditor.SplitToBulks(inputStream, numberOfPages);

        for (int i = 0; i < outBuffer.Length; i++)
        {
            using (FileStream outStream = new FileStream(outputDir + "File_" + (i + 1).ToString() + "_out.pdf", FileMode.Create))
            {
                outBuffer[i].WriteTo(outStream);
            }
        }
    }
}
```

## Οδηγός Εφαρμογής

### Προσθήκη εικόνας στην κεφαλίδα PDF

#### Επισκόπηση
Η προσθήκη μιας εικόνας στην κεφαλίδα του PDF σας μπορεί να κάνει τα έγγραφα να φαίνονται πιο ελκυστικά και επαγγελματικά. Αυτή η ενότητα θα σας καθοδηγήσει στον τρόπο με τον οποίο μπορείτε να το πετύχετε αυτό με το Aspose.PDF για .NET.

**Βήμα 1: Αρχικοποίηση αντικειμένου εγγράφου**
Δημιουργήστε ένα νέο `Document` αντικείμενο που αντιπροσωπεύει ολόκληρο το αρχείο PDF:
```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

**Βήμα 2: Δημιουργία σελίδας και ενότητας κεφαλίδας**
Προσθέστε μια σελίδα στο έγγραφό σας και δημιουργήστε μια ενότητα κεφαλίδας για αυτήν:
```csharp
// Προσθήκη σελίδας
Aspose.Pdf.Page page = doc.Pages.Add();

// Αρχικοποίηση της ενότητας κεφαλίδας
Aspose.Pdf.HeaderFooter header = new Aspose.Pdf.HeaderFooter();
page.Header = header;
```

**Βήμα 3: Εισαγωγή εικόνας στην κεφαλίδα**
Δημιουργήστε ένα αντικείμενο εικόνας, ορίστε τη διαδρομή αρχείου του και προσθέστε το στην κεφαλίδα:
```csharp
Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
image1.File = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
header.Paragraphs.Add(image1);
```

**Βήμα 4: Αποθήκευση εγγράφου**
Τέλος, αποθηκεύστε το έγγραφό σας με την εικόνα κεφαλίδας που περιλαμβάνεται:
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "HeaderWithImage_out.pdf");
doc.Save(outputFilePath);
```

### Προσθήκη αριθμών σελίδων στο υποσέλιδο PDF

#### Επισκόπηση
Η συμπερίληψη αριθμών σελίδων στο υποσέλιδο του PDF είναι ζωτικής σημασίας για έγγραφα που εκτείνονται σε πολλές σελίδες. Αυτός ο οδηγός εξηγεί πώς να το κάνετε αυτό με το Aspose.PDF για .NET.

**Βήμα 1: Αρχικοποίηση εγγράφου και προσθήκη σελίδας**
Ξεκινήστε δημιουργώντας ένα νέο `Document` αντικείμενο:
```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
// Προσθήκη σελίδας
Aspose.Pdf.Page page = doc.Pages.Add();
```

**Βήμα 2: Δημιουργία ενότητας υποσέλιδου**
Δημιουργήστε μια ενότητα υποσέλιδου για το έγγραφό σας:
```csharp
Aspose.Pdf.HeaderFooter footer = new Aspose.Pdf.HeaderFooter();
page.Footer = footer;
```

**Βήμα 3: Προσθήκη αριθμού σελίδας στο υποσέλιδο**
Εισαγάγετε ένα τμήμα κειμένου που εμφανίζει δυναμικά τον αριθμό σελίδας:
```csharp
Aspose.Pdf.Text.TextFragment txt = new Aspose.Pdf.Text.TextFragment("Page: ($p of $P)");
footer.Paragraphs.Add(txt);
```

**Βήμα 4: Αποθήκευση εγγράφου με υποσέλιδο**
Αποθηκεύστε το έγγραφό σας για να συμπεριλάβετε το υποσέλιδο με τους αριθμούς σελίδων:
```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "FooterWithPageNumber_out.pdf");
doc.Save(outputFilePath);
```

1. **Εταιρικές Αναφορές:** Η προσθήκη εταιρικών λογοτύπων στις κεφαλίδες μπορεί να ενισχύσει την ταυτότητα της επωνυμίας.
2. **Ακαδημαϊκές Εργασίες:** Οι αριθμοί σελίδων στα υποσέλιδα βοηθούν στη διατήρηση της δομής του εγγράφου, διευκολύνοντας την πλοήγηση για τους αναγνώστες.
3. **Συμβάσεις και Συμφωνίες:** Η συμπερίληψη ημερομηνιών αναθεώρησης ή αναγνωριστικών σε κεφαλίδες/υποσέλιδα βοηθά στην αποτελεσματική παρακολούθηση των αλλαγών.

## Οδηγός Εφαρμογής

### Διαγραφή σελίδων από ένα έγγραφο PDF
Αυτή η λειτουργία σάς επιτρέπει να αφαιρείτε αποτελεσματικά συγκεκριμένες σελίδες από ένα έγγραφο PDF. Ακολουθήστε τα παρακάτω βήματα για να εφαρμόσετε αυτήν τη λειτουργία:

#### 1. Δημιουργήστε το αντικείμενο PdfFileEditor
```csharp
using Aspose.Pdf.Facades;

// Δημιουργήστε ένα αντίγραφο του αντικειμένου PdfFileEditor
class PdfPageDeleter {
    static void Main() {
        using (PdfFileEditor pdfEditor = new PdfFileEditor()) {
            // Πίνακας αριθμών σελίδων προς διαγραφή (ευρετήριο με βάση το 1)
            int[] pagesToDelete = new int[] { 1, 2 };

            // Διαδρομές για αρχεία εισόδου και εξόδου
            string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
            string outputFile = "YOUR_OUTPUT_DIRECTORY/DeletePagesUsingFilePath_out.pdf";

            // Εκτέλεση διαγραφής σελίδας
            pdfEditor.Delete(inputFile, pagesToDelete, outputFile);
        }
    }
}
```
Αυτός ο κώδικας αρχικοποιεί μια παρουσία του `PdfFileEditor`, το οποίο παρέχει μεθόδους για την επεξεργασία αρχείων PDF.

#### 2. Καθορίστε σελίδες για διαγραφή
Ορίστε τις σελίδες που θέλετε να καταργήσετε χρησιμοποιώντας έναν πίνακα ακεραίων:
```csharp
// Πίνακας αριθμών σελίδων προς διαγραφή (ευρετήριο με βάση το 1)
int[] pagesToDelete = new int[] { 1, 2 };
```
Εδώ, καθορίζουμε ότι θέλουμε να διαγράψουμε την πρώτη και τη δεύτερη σελίδα.

#### 3. Διαγραφή σελίδων από PDF
Χρησιμοποιήστε το `Delete` μέθοδος για την κατάργηση των καθορισμένων σελίδων:
```csharp
// Διαδρομές για αρχεία εισόδου και εξόδου
class PdfPageDeleter {
    static void Main() {
        using (PdfFileEditor pdfEditor = new PdfFileEditor()) {
            string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
            string outputFile = "YOUR_OUTPUT_DIRECTORY/DeletePagesUsingFilePath_out.pdf";

            // Εκτέλεση διαγραφής σελίδας
            pdfEditor.Delete(inputFile, pagesToDelete, outputFile);
        }
    }
}
```
Αυτός ο κώδικας διαγράφει τις καθορισμένες σελίδες από `input.pdf` και αποθηκεύει το αποτέλεσμα σε `output.pdf`.

## Οδηγός Εφαρμογής
Ακολουθήστε αυτόν τον οδηγό για να διαγράψετε συγκεκριμένες σελίδες από ένα PDF χρησιμοποιώντας ροές:

### Αρχικοποίηση αντικειμένου PdfFileEditor
Ο `PdfFileEditor` Η κλάση είναι κεντρική για την τροποποίηση PDF. Ξεκινήστε δημιουργώντας μια παρουσία:
```csharp
// Δημιουργία αντικειμένου PdfFileEditor
PfPdFileEditor pdfEditor = new PfPdFileEditor();
```
Αυτό το αντικείμενο χειρίζεται όλες τις εργασίες χειρισμού σελίδας, συμπεριλαμβανομένης της διαγραφής.

### Δημιουργία ροών
Αρχικοποίηση `FileStream` αντικείμενα για ανάγνωση και εγγραφή δεδομένων PDF:
- **Ροή εισόδου:** Παραπέμπει στο αρχείο PDF προέλευσης.
- **Ροή εξόδου:** Ορίζει πού θα αποθηκευτεί το τροποποιημένο PDF.
```csharp
// Δημιουργήστε ροές εισόδου και εξόδου
using (FileStream inputStream = new FileStream("input.pdf", FileMode.Open))
using (FileStream outputStream = new FileStream("output.pdf", FileMode.Create))
{
    // Οι λειτουργίες πηγαίνουν εδώ
}
```

### Διαγραφή σελίδων
Καθορίστε ποιες σελίδες θα διαγραφούν χρησιμοποιώντας έναν πίνακα ακεραίων. Παρακάτω περιγράφεται ο τρόπος διαγραφής των σελίδων 1 και 3:
```csharp
// Πίνακας σελίδων προς διαγραφή
int[] pagesToDelete = new int[] { 1, 3 };

// Διαγραφή καθορισμένων σελίδων
cpdfEditor.Delete(inputStream, pagesToDelete, outputStream);
```
- **Παράμετροι:**
  - `inputStream`: Η ροή PDF πηγής.
  - `pagesToDelete`: Ένας πίνακας που περιέχει αριθμούς σελίδων προς αφαίρεση.
  - `outputStream`Ο προορισμός για το τροποποιημένο PDF.

### Κλείσιμο ροών

- Επιβεβαιώστε ότι οι αριθμοί σελίδων είναι `pagesToDelete` είναι έγκυρα (δηλαδή, εντός του εύρους του PDF).

## Οδηγός Εφαρμογής
Ας εξερευνήσουμε πώς να υλοποιήσουμε κάθε λειτουργία χρησιμοποιώντας το Aspose.PDF για .NET.

### Δημιουργία TextFragment
#### Επισκόπηση
Η δημιουργία και η προσθήκη ενός τμήματος κειμένου σε ένα PDF είναι απλή. Αυτή η ενότητα σας καθοδηγεί στη δημιουργία του "Hello World" ως δείγματος περιεχομένου.
**Βήμα 1: Δημιουργία μιας παρουσίας εγγράφου**
```csharp
Document doc = new Document();
```
*Γιατί;*: Αρχικοποιεί ένα νέο έγγραφο PDF για χειρισμό.

**Βήμα 2: Προσθήκη σελίδας**
```csharp
Page page = doc.Pages.Add();
```
*Γιατί;*: Προσθέτει μια κενή σελίδα στο έγγραφό σας όπου θα τοποθετηθεί το τμήμα κειμένου.

**Βήμα 3: Δημιουργία και προσθήκη TextFragment**
```csharp
TextFragment text = new TextFragment("Hello World.. ");
page.Paragraphs.Add(text);
```
*Γιατί;*Αυτό το τμήμα κειμένου δημιουργεί ένα αντικείμενο κειμένου και το προσθέτει στη συλλογή παραγράφων της σελίδας.

**Βήμα 4: Αποθήκευση του εγγράφου**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(Path.Combine(outputDir, "TextFragment_out.pdf"));
```
*Γιατί;*Αποθηκεύει τις αλλαγές σας ως αρχείο PDF στον καθορισμένο κατάλογο.

### Προσθήκη ενσωματωμένης εικόνας
#### Επισκόπηση
Η προσθήκη εικόνων εντός κειμένου βελτιώνει την οπτική ελκυστικότητα του PDF σας. Δείτε πώς μπορείτε να ενσωματώσετε μια εικόνα αμέσως μετά το τμήμα κειμένου που προσθέσατε προηγουμένως.
**Βήμα 1: Δημιουργία μιας παρουσίας εικόνας**
```csharp
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
image.IsInLineParagraph = true;
```
*Γιατί;*: Ρυθμίζει τις παραμέτρους της εικόνας ώστε να αντιμετωπίζεται ως ενσωματωμένη, επιτρέποντάς της να ρέει απρόσκοπτα μέσα στο περιεχόμενο κειμένου.

**Βήμα 2: Ορισμός ιδιοτήτων εικόνας**
```csharp
string imagePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "aspose-logo.jpg");
image.File = imagePath;
image.FixHeight = 30;
image.FixWidth = 100;
```
*Γιατί;*: Καθορίζει τη διαδρομή και τις διαστάσεις της εικόνας, διασφαλίζοντας ότι ταιριάζει σωστά στη διάταξη του εγγράφου σας.

**Βήμα 3: Προσθήκη εικόνας στη σελίδα**
```csharp
page.Paragraphs.Add(image);
```
*Γιατί;*: Προσθέτει την διαμορφωμένη εικόνα ως στοιχείο παραγράφου στη συλλογή της σελίδας.

**Βήμα 4: Αποθήκευση του εγγράφου**
```csharp
doc.Save(Path.Combine(outputDir, "InlineImage_out.pdf"));
```
*Γιατί;*Αποθηκεύει το ενημερωμένο έγγραφό σας μαζί με την ενσωματωμένη εικόνα.

### Προσθήκη Ενσωματωμένου Αποσπάσματος Κειμένου μετά την Εικόνα
#### Επισκόπηση
Συνεχίζοντας από εκεί που μείναμε, ας προσθέσουμε ένα ακόμη τμήμα κειμένου μετά την εικόνα μας για να δημιουργήσουμε μια απρόσκοπτη ροή περιεχομένου.
**Βήμα 1: Επαναρχικοποίηση του TextFragment**
```csharp
TextFragment text = new TextFragment(" Hello Again..");
text.IsInLineParagraph = true;
```
*Γιατί;*Προετοιμάζει το νέο μπλοκ κειμένου για την προσθήκη εντός γραμμής, αμέσως μετά την εικόνα.

**Βήμα 2: Προσθήκη νέου TextFragment στη σελίδα**
```csharp
page.Paragraphs.Add(text);
```
*Γιατί;*: Τοποθετεί το τμήμα κειμένου σε ευθυγράμμιση με άλλα στοιχεία περιεχομένου στη σελίδα.

**Βήμα 3: Αποθήκευση του εγγράφου**
```csharp
doc.Save(Path.Combine(outputDir, "InlineTextAfterImage_out.pdf"));
```
*Γιατί;*: Οριστικοποιεί και αποθηκεύει το έγγραφό σας που περιέχει εικόνα και πρόσθετα τμήματα κειμένου.

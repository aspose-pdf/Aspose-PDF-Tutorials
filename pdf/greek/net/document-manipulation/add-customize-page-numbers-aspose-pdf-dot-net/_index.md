# Πώς να προσθέσετε και να προσαρμόσετε αριθμούς σελίδων σε έγγραφα PDF χρησιμοποιώντας το Aspose.PDF για .NET

Αυτό το σεμινάριο θα σας καθοδηγήσει στην προσθήκη και προσαρμογή αριθμών σελίδων χρησιμοποιώντας αυτήν την ισχυρή βιβλιοθήκη.

- Προσαρμογή στυλ, συμπεριλαμβανομένων των λατινικών αριθμών
- Βελτιστοποίηση απόδοσης με μεγάλα έγγραφα

Ας καλύψουμε τις προϋποθέσεις πριν ξεκινήσουμε.

## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την προσθήκη και την προσαρμογή αριθμών σελίδων χρησιμοποιώντας το Aspose.PDF για .NET.

### Προσθήκη αριθμού σελίδας σε έγγραφο PDF (H2)

Προσθέστε αριθμούς σελίδων στο κάτω μέρος κάθε σελίδας με τη μορφή "Σελίδα X από Y".

#### Επισκόπηση
Θα χρησιμοποιήσουμε `PdfFileStamp` για να σφραγίσετε τους αριθμούς σελίδων στο έγγραφό σας. 

**Βήμα 1: Αρχικοποίηση του PdfFileStamp**
```csharp
using Aspose.Pdf.Facades;

// Δημιουργήστε ένα νέο αντικείμενο PdfFileStamp
fileStamp = new PdfFileStamp();
fileStamp.BindPdf("YOUR_DOCUMENT_DIRECTORY\AddPageNumber.pdf");
```

**Βήμα 2: Λήψη συνολικού αριθμού σελίδων**
Ανακτήστε τον συνολικό αριθμό σελίδων για να μορφοποιήσετε σωστά τους αριθμούς σελίδων.
```csharp
int totalPages = new PdfFileInfo("YOUR_DOCUMENT_DIRECTORY\AddPageNumber.pdf").NumberOfPages;
```

**Βήμα 3: Μορφοποίηση και προσθήκη αριθμών σελίδων**
Προσαρμόστε την εμφάνιση των αριθμών σελίδων σας ορίζοντας το χρώμα, το στυλ γραμματοσειράς και τη θέση τους.
```csharp
using System.Drawing;
using System.Text;

FormattedText formattedText = new FormattedText(
    "Page # Of " + totalPages,
    Color.Blue,
    Color.Gray,
    FontStyle.Courier,
    EncodingType.Winansi,
    false, 14
);

fileStamp.StartingNumber = 1;
fileStamp.AddPageNumber(formattedText, 0); // Θέση στο κάτω κέντρο

// Αποθήκευση και κλείσιμο του αντικειμένου PdfFileStamp
testDocument.Save("YOUR_OUTPUT_DIRECTORY\AddPageNumber_out.pdf");
testDocument.Close();
```

### Προσαρμογή στυλ αρίθμησης σελίδας σε ένα έγγραφο PDF (H2)

Αλλάξτε το στυλ αρίθμησης σελίδας, όπως χρησιμοποιώντας λατινικούς αριθμούς.

#### Επισκόπηση
Μπορείτε εύκολα να προσαρμόσετε το στυλ αρίθμησης με `PdfFileStamp`.

**Βήμα 1: Αρχικοποίηση του PdfFileStamp**
Επαναρχικοποιήστε εάν είναι απαραίτητο ή συνεχίστε από την προηγούμενη χρήση.
```csharp
// Υποθέτοντας ότι το fileStamp έχει ήδη αρχικοποιηθεί και είναι συνδεδεμένο με ένα έγγραφο PDF
```

**Βήμα 2: Ορισμός στυλ αρίθμησης**
Επιλέξτε κεφαλαία γράμματα για λατινικούς αριθμούς σε αυτό το παράδειγμα.
```csharp
fileStamp.NumberingStyle = NumberingStyle.NumeralsRomanUppercase;
```

**Βήμα 3: Προσθήκη αριθμών σελίδων με προσαρμοσμένη μορφή**
Εφαρμόστε το προσαρμοσμένο στυλ αρίθμησης στο έγγραφό σας.
```csharp
// Προσθήκη αριθμών σελίδων με την καθορισμένη μορφή στο κάτω μέρος του κέντρου
testDocument.AddPageNumber("#");

// Αποθήκευση και κλείσιμο του αντικειμένου PdfFileStamp
testDocument.Save("YOUR_OUTPUT_DIRECTORY\CustomNumberStyle_out.pdf");
testDocument.Close();
```

## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην περικοπή του κενού χώρου γύρω από μια σελίδα χρησιμοποιώντας το Aspose.PDF για .NET.

### Φόρτωση του υπάρχοντος PDF

Ξεκινήστε φορτώνοντας το αρχείο PDF προορισμού σας σε ένα `Aspose.Pdf.Document` αντικείμενο. Αυτό σας επιτρέπει να χειριστείτε τις σελίδες και τις ιδιότητές του.

```csharp
string dataDir = "path_to_your_directory/";
document doc = new Document(dataDir + "input.pdf");
```

### Απόδοση της σελίδας ως εικόνας

Για να περικόψετε τον λευκό χώρο, αποδώστε μια σελίδα PDF ως εικόνα χρησιμοποιώντας `PngDevice`, το οποίο παρέχει έλεγχο της ανάλυσης και της ποιότητας εξόδου.

```csharp
using Aspose.Pdf.Devices;
using System.Drawing;

// Ρύθμιση συσκευής με το επιθυμητό DPI
PngDevice device = new PngDevice(new Resolution(72));

using (MemoryStream imageStr = new MemoryStream()) {
    // Επεξεργασία της πρώτης σελίδας
    device.Process(doc.Pages[1], imageStr);
    Bitmap bmp = new Bitmap(imageStr);
}
```

### Προσδιορίστε τις μη λευκές περιοχές

Κλειδώστε τα bit bitmap για να αναλύσετε κάθε pixel και να προσδιορίσετε τα όρια των μη λευκών περιοχών. Αυτό βοηθά στον επανυπολογισμό του πλαισίου περικοπής.

```csharp
using System.Drawing.Imaging;
using Aspose.Pdf;

// Κλείδωμα bit για πρόσβαση μόνο για ανάγνωση
BitmapData imageBitmapData = bmp.LockBits(
    new Rectangle(0, 0, bmp.Width, bmp.Height),
    ImageLockMode.ReadOnly,
    PixelFormat.Format32bppRgb);

int toHeight = bmp.Height;
int toWidth = bmp.Width;
int? leftNonWhite = null, rightNonWhite = null;

// Επεξεργασία κάθε σειράς pixel
for (int y = 0; y < toHeight; y++) {
    byte[] imageRowBytes = new byte[imageBitmapData.Stride];
    
    if (IntPtr.Size == 4)
        Marshal.Copy(new IntPtr(imageBitmapData.Scan0.ToInt32() + y * imageBitmapData.Stride), 
                     imageRowBytes, 0, imageBitmapData.Stride);
    else
        Marshal.Copy(new IntPtr(imageBitmapData.Scan0.ToInt64() + y * imageBitmapData.Stride), 
                     imageRowBytes, 0, imageBitmapData.Stride);
    
    // Προσδιορίστε τις μη λευκές περιοχές στη σειρά
}
```

### Προσαρμογή πλαισίου περικοπής

Αφού προσδιορίσετε τα όρια της περιοχής περιεχομένου, προσαρμόστε το πλαίσιο περικοπής της σελίδας για να αφαιρέσετε τον πλεονάζοντα λευκό χώρο.

```csharp
// Προηγούμενη αναφορά πλαισίου καλλιέργειας
document.Rectangle prevCropBox = doc.Pages[1].CropBox;

// Υπολογισμός νέων διαστάσεων καλλιέργειας
doc.Pages[1].CropBox =
    new Aspose.Pdf.Rectangle(
        leftNonWhite.Value + prevCropBox.LLX,
        (toHeight + prevCropBox.LLY) - bottomNonWhite.Value,
        rightNonWhite.Value + doc.Pages[1].CropBox.LLX,
        (toHeight + prevCropBox.LLY) - topNonWhite.Value
    );
```

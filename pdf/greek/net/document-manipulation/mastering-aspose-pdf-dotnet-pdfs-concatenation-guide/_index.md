## Οδηγός Εφαρμογής
Θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη συνένωση αρχείων PDF και την ανάκτηση ονομάτων αρχείων από έναν κατάλογο.

### Λειτουργία 1: Συνένωση Αρχείων PDF (H2)
**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να συγχωνεύσετε όλα τα PDF σε έναν καθορισμένο κατάλογο σε ένα αρχείο εξόδου, ονομάζοντάς το αυτόματα με την τρέχουσα ημερομηνία και ώρα για εύκολη αναγνώριση.

#### Βήμα προς βήμα εφαρμογή

**H3: Ανάκτηση αρχείων PDF**
Αρχικά, πρέπει να λάβουμε τα ονόματα όλων των αρχείων PDF στον κατάλογο προορισμού μας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string[] fileEntries = Directory.GetFiles(dataDir, "*.pdf");
```

Εδώ, `fileEntries` θα περιέχει διαδρομές προς κάθε αρχείο PDF που βρέθηκε.

**H3: Δημιουργία μοναδικού ονόματος αρχείου εξόδου**
Δημιουργούμε ένα μοναδικό όνομα για την ενωμένη έξοδο χρησιμοποιώντας την τρέχουσα ημερομηνία και ώρα:

```csharp
string date = DateTime.Now.ToString("MM-dd-yyyy");
string hoursSeconds = DateTime.Now.ToString("hh-mm");
string masterFileName = $"{date}_{hoursSeconds}_out.pdf";
```

**H3: Συνένωση PDF**
Δημιουργία στιγμιαίας εικόνας `PdfFileEditor` και χρησιμοποιήστε το για να συγχωνεύσετε τα αρχεία:

```csharp
PdfFileEditor pdfEditor = new PdfFileEditor();
pdfEditor.Concatenate(fileEntries, Path.Combine(dataDir, masterFileName));
```

Αυτός ο κώδικας συνενώνει όλα τα αρχεία PDF που αναφέρονται στο `fileEntries`, αποθηκεύοντάς τα ως `masterFileName`.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι όλες οι διαδρομές είναι σωστά καθορισμένες και προσβάσιμες.
- Ελέγξτε για τυχόν δικαιώματα ανάγνωσης/εγγραφής στον κατάλογο.

### Χαρακτηριστικό 2: Ανάκτηση αρχείων καταλόγου (H2)
**Επισκόπηση**Αυτή η ενότητα καλύπτει την ανάκτηση των ονομάτων αρχείων PDF από έναν δεδομένο κατάλογο, κάτι που είναι κρίσιμο για εργασίες αυτοματισμού που αφορούν τη διαχείριση αρχείων.


## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Προσθήκη αλλαγής σελίδας σε έγγραφο PDF

**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να προσθέσετε μια αλλαγή σελίδας σε μια καθορισμένη θέση μέσα σε ένα έγγραφο PDF χρησιμοποιώντας διαδρομές αρχείων.

**Βήματα:**

#### Βήμα 1: Ορισμός διαδρομών
Ξεκινήστε ορίζοντας τους καταλόγους για τα αρχεία PDF εισόδου και εξόδου:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση εγγράφου προέλευσης
Φορτώστε το έγγραφο PDF πηγής σας σε ένα `Aspose.Pdf.Document` αντικείμενο:
```csharp
Document doc = new Document(dataDir + "/input.pdf");
```

#### Βήμα 3: Δημιουργία εγγράφου προορισμού
Δημιουργήστε ένα κενό έγγραφο PDF προορισμού για να αποθηκεύσετε τα αποτελέσματα αλλαγής σελίδας:
```csharp
Document dest = new Document();
```

#### Βήμα 4: Αρχικοποίηση του PdfFileEditor
Ρυθμίστε το `PdfFileEditor` αντικείμενο, το οποίο θα χρησιμοποιηθεί για την επεξεργασία του αρχείου PDF:
```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
```

#### Βήμα 5: Προσθήκη αλλαγής σελίδας
Προσθέστε μια αλλαγή σελίδας στη θέση 450 στην πρώτη σελίδα του εγγράφου σας και προσθέστε την στο έγγραφο προορισμού:
```csharp
fileEditor.AddPageBreak(doc, dest, new PdfFileEditor.PageBreak[] {
    new PdfFileEditor.PageBreak(1, 450)
});
```

#### Βήμα 6: Αποθηκεύστε το PDF που προκύπτει
Τέλος, αποθηκεύστε το PDF που προκύπτει με την προσθήκη αλλαγής σελίδας:
```csharp
dest.Save(outputDir + "/PageBreak_out.pdf");
```

### Λειτουργία 2: Προσθήκη αλλαγής σελίδας και αποθήκευση στη διαδρομή προορισμού

**Επισκόπηση:** Αυτή η λειτουργία προσθέτει μια αλλαγή σελίδας απευθείας στο PDF σε μια καθορισμένη διαδρομή.

#### Βήμα 1: Ορισμός διαδρομών
Προσδιορίστε τον κατάλογο εγγράφων σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

#### Βήμα 2: Αρχικοποίηση του PdfFileEditor
Δημιουργήστε μια παρουσία του `PdfFileEditor` για να χειριστείτε το αρχείο PDF:
```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
```

#### Βήμα 3: Προσθήκη αλλαγής σελίδας και άμεση αποθήκευση
Προσθέστε απευθείας μια αλλαγή σελίδας στη θέση 450 στην πρώτη σελίδα του εγγράφου σας, αποθηκεύοντάς την σε μια διαδρομή εξόδου:
```csharp
fileEditor.AddPageBreak(dataDir + "/input.pdf", dataDir + "/PageBreakWithDestPath_out.pdf", new PdfFileEditor.PageBreak[] {
    new PdfFileEditor.PageBreak(1, 450)
});
```

### Λειτουργία 3: Προσθήκη αλλαγής σελίδας χρησιμοποιώντας ροές

**Επισκόπηση:** Αυτή η λειτουργία δείχνει την προσθήκη αλλαγής σελίδας χρησιμοποιώντας ροές εισόδου και εξόδου.

#### Βήμα 1: Άνοιγμα ροής εισόδου
Άνοιγμα ροής για την ανάγνωση του πηγαίου εγγράφου PDF:
```csharp
using (Stream src = new FileStream(dataDir + "/input.pdf", FileMode.Open, FileAccess.Read))
```

#### Βήμα 2: Άνοιγμα ροής εξόδου
Δημιουργήστε μια ροή εξόδου για την εγγραφή αλλαγών:
```csharp
using (Stream dest = new FileStream(dataDir + "/PageBreakWithStream_out.pdf", FileMode.Create, FileAccess.Write))
```

#### Βήμα 3: Αρχικοποίηση του PdfFileEditor
Ρυθμίστε το `PdfFileEditor` αντικείμενο για επεξεργασία:
```csharp
PdfFileEditor fileEditor = new PdfFileEditor();
```

#### Βήμα 4: Προσθήκη αλλαγής σελίδας χρησιμοποιώντας ροές
Προσθέστε μια αλλαγή σελίδας στη θέση 450 στην πρώτη σελίδα, χρησιμοποιώντας ροές για τη διαχείριση τόσο της εισόδου όσο και της εξόδου:
```csharp
fileEditor.AddPageBreak(src, dest, new PdfFileEditor.PageBreak[] {
    new PdfFileEditor.PageBreak(1, 450)
});
```

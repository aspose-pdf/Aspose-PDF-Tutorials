## Οδηγός Εφαρμογής
Αυτή η ενότητα περιγράφει λεπτομερώς τη διαδικασία προσθήκης ενός θυγατρικού σελιδοδείκτη σε ένα υπάρχον έγγραφο PDF.

### Επισκόπηση
Η προσθήκη ενός θυγατρικού σελιδοδείκτη περιλαμβάνει τη δημιουργία μιας ιεραρχικής δομής μέσα στο PDF σας, βελτιώνοντας την πλοήγηση ομαδοποιώντας σχετικές ενότητες κάτω από τους γονικούς σελιδοδείκτες.

### Οδηγός βήμα προς βήμα
#### **1. Ρύθμιση διαδρομών εισόδου και εξόδου**
Αρχικά, ορίστε διαδρομές για το αρχείο PDF εισόδου και τη θέση εξόδου:
```csharp
string inputDir = "YOUR_DOCUMENT_DIRECTORY" + "/AddChildBookmark.pdf";
string outputDir = "YOUR_OUTPUT_DIRECTORY" + "/AddChildBookmark_out.pdf";
```
#### **2. Φορτώστε το έγγραφο PDF σας**
Ανοίξτε ένα υπάρχον έγγραφο PDF χρησιμοποιώντας το Aspose.PDF:
```csharp
Document pdfDocument = new Document(inputDir);
```
#### **3. Δημιουργία γονικού σελιδοδείκτη**
Δημιουργήστε και διαμορφώστε ένα γονικό αντικείμενο σελιδοδείκτη ώστε να ξεχωρίζει στη λίστα σελιδοδεικτών:
```csharp
OutlineItemCollection parentOutline = new OutlineItemCollection(pdfDocument.Outlines);
parentOutline.Title = "Parent Outline";
parentOutline.Italic = true;
parentOutline.Bold = true;
```
*Γιατί αυτό το βήμα;*Η διαμόρφωση βοηθά στη διαφοροποίηση των κύριων ενοτήτων, καθιστώντας την πλοήγηση διαισθητική.
#### **4. Δημιουργία σελιδοδείκτη θυγατρικών**
Δημιουργήστε έναν θυγατρικό σελιδοδείκτη με το δικό του στυλ:
```csharp
OutlineItemCollection childOutline = new OutlineItemCollection(pdfDocument.Outlines);
childOutline.Title = "Child Outline";
childOutline.Italic = true;
childOutline.Bold = true;
```
*Γιατί αυτό το βήμα;*Οι θυγατρικοί σελιδοδείκτες παρέχουν άμεση πρόσβαση σε ενσωματωμένο περιεχόμενο, βελτιώνοντας την εμπειρία χρήστη.
#### **5. Προσθέστε το θυγατρικό σελιδοδείκτη**
Συνδέστε τον θυγατρικό σελιδοδείκτη στον γονικό του:
```csharp
parentOutline.Add(childOutline);
```
*Γιατί αυτό το βήμα;*Η δημιουργία αυτής της σχέσης οργανώνει λογικά τη δομή του εγγράφου σας.
#### **6. Αποθηκεύστε το έγγραφό σας**
Αποθηκεύστε το ενημερωμένο PDF με τους νέους σελιδοδείκτες που έχουν προστεθεί:
```csharp
pdfDocument.Save(outputDir);
```

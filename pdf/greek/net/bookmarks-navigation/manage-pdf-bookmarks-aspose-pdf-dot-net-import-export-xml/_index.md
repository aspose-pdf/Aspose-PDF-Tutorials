## Οδηγός Εφαρμογής

### Εισαγωγή σελιδοδεικτών από XML σε έγγραφο PDF

Αυτή η λειτουργία σάς επιτρέπει να συμπληρώσετε σελιδοδείκτες σε ένα έγγραφο PDF χρησιμοποιώντας ένα υπάρχον αρχείο XML. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Ρύθμιση διαδρομών καταλόγου και αρχείων

ορίστε τις διαδρομές για το PDF εισόδου, την πηγή σελιδοδεικτών XML και το PDF εξόδου:
```csharp
double YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
string pdfFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ImportFromXML.pdf");
string xmlFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "bookmarks.xml");
string outputPdfPath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ImportFromXML_out.pdf");
```

#### Βήμα 2: Αρχικοποίηση του PdfBookmarkEditor

Δημιουργήστε μια παρουσία του `PdfBookmarkEditor` για να χειριστείτε τους σελιδοδείκτες PDF:
```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```

#### Βήμα 3: Σύνδεση και εισαγωγή σελιδοδεικτών

Ανοίξτε το αρχείο PDF προορισμού σας και, στη συνέχεια, εισαγάγετε τους σελιδοδείκτες από XML:
```csharp
bookmarkEditor.BindPdf(pdfFilePath);
bookmarkEditor.ImportBookmarksWithXML(xmlFilePath);
```
- **BindPdf:** Ανοίγει ένα PDF για επεξεργασία.
- **Εισαγωγή σελιδοδεικτών με XML:** Εισάγει σελιδοδείκτες που ορίζονται σε ένα αρχείο XML.

#### Βήμα 4: Αποθηκεύστε το ενημερωμένο PDF

Τέλος, αποθηκεύστε τις αλλαγές σας για να δημιουργήσετε το PDF εξόδου:
```csharp
bookmarkEditor.Save(outputPdfPath);
```

### Εξαγωγή σελιδοδεικτών από έγγραφο PDF σε αρχείο XML

Αντίστροφα, μπορείτε να εξαγάγετε υπάρχοντες σελιδοδείκτες σε μορφή XML:

#### Βήμα 1: Ρύθμιση διαδρομών καταλόγου και αρχείων

Καθορίστε διαδρομές για το αρχείο PDF προέλευσης και το αρχείο XML προορισμού:
```csharp
double YOUR_DOCUMENT_DIRECTORY = @"YOUR_DOCUMENT_DIRECTORY";
string pdfFilePath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "ExportToXML.pdf");
string xmlOutputPath = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "bookmarks_exported.xml");
```

#### Βήμα 2: Αρχικοποίηση του PdfBookmarkEditor

Ξανά, αρχικοποιήστε το `PdfBookmarkEditor`:
```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```

#### Βήμα 3: Σύνδεση και εξαγωγή σελιδοδεικτών

Ανοίξτε το αρχείο PDF και εξαγάγετε τους σελιδοδείκτες του σε ένα αρχείο XML:
```csharp
bookmarkEditor.BindPdf(pdfFilePath);
bookmarkEditor.ExportBookmarksToXML(xmlOutputPath);
```
- **Εξαγωγή σελιδοδεικτών σε XML:** Μετατρέπει και αποθηκεύει τους σελιδοδείκτες σε μορφή XML.

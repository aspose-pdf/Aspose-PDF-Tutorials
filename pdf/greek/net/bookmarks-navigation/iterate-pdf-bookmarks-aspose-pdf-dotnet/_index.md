## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Φόρτωση και ανάλυση εγγράφου PDF

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τον τρόπο φόρτωσης ενός εγγράφου PDF χρησιμοποιώντας το Aspose.PDF για .NET.

```csharp
Document pdfDocument = new Document(dataDir + "/GetChildBookmarks.pdf");
// Φορτώστε το καθορισμένο έγγραφο PDF.
```

### Λειτουργία 2: Επανάληψη μέσω σελιδοδεικτών σε ένα έγγραφο PDF

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στην επανάληψη μέσω σελιδοδεικτών και στην εκτύπωση των ιδιοτήτων τους.

#### Επανάληψη σελιδοδεικτών

Ο ακόλουθος κώδικας δείχνει πώς να αποκτήσετε πρόσβαση σε σελιδοδείκτες και να εκτυπώσετε τα χαρακτηριστικά τους:

```csharp
using System;
using Aspose.Pdf;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου σας εδώ
Document pdfDocument = new Document(dataDir + "/GetChildBookmarks.pdf");

foreach (OutlineItemCollection outlineItem in pdfDocument.Outlines)
{
    // Εκτύπωση ιδιοτήτων κάθε σελιδοδείκτη
    Console.WriteLine($"Title: {outlineItem.Title}");
    Console.WriteLine($"Italic: {outlineItem.Italic}");
    Console.WriteLine($"Bold: {outlineItem.Bold}");
    Console.WriteLine($"Color: {outlineItem.Color}");

    if (outlineItem.Count > 0)
    {
        // Ελέγξτε για θυγατρικούς σελιδοδείκτες και επαναλάβετε τη διαδικασία μέσω αυτών
        foreach (OutlineItemCollection childOutline in outlineItem)
        {
            // Εκτύπωση ιδιοτήτων κάθε θυγατρικού σελιδοδείκτη
            Console.WriteLine($"Child Title: {childOutline.Title}");
            Console.WriteLine($"Child Italic: {childOutline.Italic}");
            Console.WriteLine($"Child Bold: {childOutline.Bold}");
            Console.WriteLine($"Child Color: {childOutline.Color}");
        }
    }
}
```

#### Εξήγηση
- **Κλάση εγγράφου**: Αντιπροσωπεύει το έγγραφο PDF.
- **Συλλογή Αντικειμένων Outline**: Αντιπροσωπεύει έναν σελιδοδείκτη στο PDF. Μπορεί να περιέχει θυγατρικούς σελιδοδείκτες.

## Οδηγός Εφαρμογής

### Προσθήκη πίνακα περιεχομένων
Η προσθήκη ενός Περιεχομένου Περιεχομένου περιλαμβάνει τη φόρτωση ενός υπάρχοντος PDF, τη δημιουργία μιας νέας σελίδας Περιεχομένου Περιεχομένου και τη σύνδεσή του με επικεφαλίδες από άλλες σελίδες. Δείτε πώς:

#### Βήμα 1: Φόρτωση του εγγράφου PDF
```csharp
// Αρχικοποίηση του αντικειμένου εγγράφου
document doc = new Document("YOUR_DOCUMENT_DIRECTORY\AddTOC.pdf");
```
Φορτώστε το υπάρχον αρχείο PDF σε ένα Aspose.PDF `Document` αντικείμενο για χειραγώγηση.

#### Βήμα 2: Δημιουργήστε μια σελίδα περιεχομένων
```csharp
// Εισαγάγετε μια νέα σελίδα στην αρχή που θα χρησιμεύσει ως Πίνακας Περιεχομένων
Page tocPage = doc.Pages.Insert(1);
```
Εισαγάγετε μια νέα σελίδα στην αρχή του PDF σας για να φιλοξενήσετε τον Πίνακα Περιεχομένων, επιτρέποντας εύκολη πρόσβαση από οπουδήποτε στο έγγραφο.

#### Βήμα 3: Ρύθμιση πληροφοριών πίνακα περιεχομένων
```csharp
// Διαμορφώστε τον τίτλο και την εμφάνιση του TOC
tocInfo tocInfo = new TocInfo();
TextFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20;
title.TextState.FontStyle = FontStyles.Bold;
tocInfo.Title = title;
tocPage.TocInfo = tocInfo;
```
Ορίστε την εμφάνιση και το περιεχόμενο του TOC με έναν σαφή, έντονο τίτλο.

#### Βήμα 4: Ορισμός και προσθήκη επικεφαλίδων
```csharp
// Πίνακας τίτλων για προσθήκη ως στοιχεία TOC
string[] titles = { "First page", "Second page", "Third page", "Fourth page" };

for (int i = 0; i < 2; i++)
{
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
    TextSegment segment2 = new TextSegment();
    heading2.TocPage = tocPage;
    heading2.Segments.Add(segment2);

    // Συνδέστε κάθε καταχώρηση Περιεχομένου (TOC) με την αντίστοιχη σελίδα της
    heading2.DestinationPage = doc.Pages[i + 2];
    heading2.Top = doc.Pages[i + 2].Rect.Height;
    segment2.Text = titles[i];

    tocPage.Paragraphs.Add(heading2);
}
```
Περιηγηθείτε στις επιθυμητές επικεφαλίδες, δημιουργώντας `Heading` αντικείμενα για κάθε ένα. Ορίστε τη σελίδα προορισμού και τη θέση για να εξασφαλίσετε άμεση πλοήγηση.

#### Βήμα 5: Αποθήκευση του εγγράφου
```csharp
// Εξάγετε το τροποποιημένο έγγραφο με τον Πίνακα Περιεχομένων
string outputFilePath = "YOUR_OUTPUT_DIRECTORY\TOC_out.pdf";
doc.Save(outputFilePath);
```
Αποθηκεύστε το ενημερωμένο PDF σας με τον ενσωματωμένο πίνακα περιεχομένων στην αρχή.

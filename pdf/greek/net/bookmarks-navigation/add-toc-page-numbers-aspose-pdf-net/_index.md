# Πώς να προσθέσετε έναν πίνακα περιεχομένων με αριθμούς σελίδων σε PDF χρησιμοποιώντας το Aspose.PDF .NET

## Εισαγωγή

Στη σημερινή ψηφιακή εποχή, η αποτελεσματική πλοήγηση σε μεγάλα έγγραφα είναι απαραίτητη. Είτε χειρίζεστε αναφορές, εγχειρίδια είτε ακαδημαϊκές εργασίες, η ύπαρξη ενός καλά οργανωμένου Πίνακα Περιεχομένων (TOC) μπορεί να βελτιώσει σημαντικά τη χρηστικότητα. Αυτός ο οδηγός δείχνει πώς να προσθέσετε και να προσαρμόσετε έναν Πίνακα Περιεχομένων με αριθμούς σελίδων σε αρχεία PDF χρησιμοποιώντας το Aspose.PDF για .NET, μια ισχυρή βιβλιοθήκη σχεδιασμένη για χειρισμό PDF σε εφαρμογές .NET.

**Τι θα μάθετε:**
- Ρύθμιση του Aspose.PDF για .NET
- Φόρτωση ενός υπάρχοντος εγγράφου PDF στην εφαρμογή σας
- Δημιουργία και προσαρμογή ενός Πίνακα Περιεχομένων με αριθμούς σελίδων
- Αποθήκευση του τροποποιημένου αρχείου PDF

Είστε έτοιμοι να βελτιστοποιήσετε τη διαχείριση των εγγράφων σας; Ας ξεκινήσουμε!

### Προαπαιτούμενα

Πριν προχωρήσετε, βεβαιωθείτε ότι έχετε:

### Οδηγός Εφαρμογής

Αφού όλα είναι έτοιμα, ας προσθέσουμε έναν προσαρμοσμένο πίνακα περιεχομένων με αριθμούς σελίδων στο έγγραφο PDF σας.

#### Φόρτωση και τροποποίηση εγγράφου PDF
Αρχικά, φορτώστε το υπάρχον έγγραφο PDF στην εφαρμογή σας. Στη συνέχεια, θα εισαγάγουμε μια νέα σελίδα στην αρχή αυτού του εγγράφου για τον Πίνακα Περιεχομένων μας.

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string inFile = "YOUR_DOCUMENT_DIRECTORY/42824.pdf";
string outFile = "YOUR_OUTPUT_DIRECTORY/42824_out.pdf";

// Φόρτωση του εγγράφου PDF
document doc = new Document(inFile);

// Εισαγωγή νέας σελίδας στην αρχή για τον Πίνακα Περιεχομένων
Aspose.Pdf.Page tocPage = doc.Pages.Insert(1);
```

#### Δημιουργία και ρύθμιση παραμέτρων πληροφοριών πίνακα περιεχομένων
Στη συνέχεια, διαμορφώστε τον πίνακα περιεχομένων σας ορίζοντας τον τίτλο, το μέγεθος γραμματοσειράς, το στυλ και τους προθέματα αρίθμησης σελίδων.

```csharp
// Ρύθμιση πληροφοριών TOC
tocInfo tocInfo = new TocInfo();
textFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20; // Μέγεθος γραμματοσειράς για τον τίτλο του Πίνακα Περιεχομένων
title.TextState.FontStyle = FontStyles.Bold; // Έντονο στυλ για έμφαση
tocInfo.Title = title;
tocInfo.PageNumbersPrefix = "P"; // Πρόθεμα αριθμών σελίδων με 'P'
tocPage.TocInfo = tocInfo; // Αντιστοίχιση πληροφοριών TOC στη νέα σελίδα
```

#### Προσθήκη καταχωρίσεων στον πίνακα περιεχομένων
Τώρα, συμπληρώστε τον Πίνακα Περιεχομένων με καταχωρήσεις για κάθε σελίδα του εγγράφου σας.

```csharp
for (int i = 1; i < doc.Pages.Count; i++) {
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
textSegment segment2 = new textSegment();

    // Σύνδεση της επικεφαλίδας με τη σελίδα TOC
    heading2.TocPage = tocPage;

    // Προσθήκη τμήματος κειμένου στην επικεφαλίδα
    heading2.Segments.Add(segment2); 

    // Ορισμός σελίδας προορισμού και θέσης για κάθε καταχώρηση
    heading2.DestinationPage = doc.Pages[i + 1];
    heading2.Top = doc.Pages[i + 1].Rect.Height; 
    segment2.Text = "P" + i.ToString(); // Προσαρμοσμένο κείμενο με πρόθεμα 'P'

    // Προσθήκη της επικεφαλίδας στη σελίδα Περιεχομένων
    tocPage.Paragraphs.Add(heading2);
}
```

#### Αποθήκευση τροποποιημένου εγγράφου PDF
Τέλος, αποθηκεύστε το τροποποιημένο έγγραφό σας με τον πρόσφατα προστιθέμενο Πίνακα Περιεχομένων.

```csharp
doc.Save(outFile);
```

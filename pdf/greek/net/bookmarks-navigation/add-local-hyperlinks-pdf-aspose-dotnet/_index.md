## Οδηγός Εφαρμογής
Αυτή η ενότητα σάς καθοδηγεί στην προσθήκη τοπικών υπερσυνδέσμων σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET.

### Δημιουργία και ρύθμιση παραμέτρων τοπικών υπερσυνδέσμων
Οι τοπικοί υπερσύνδεσμοι επιτρέπουν στους χρήστες να πλοηγούνται απρόσκοπτα μέσα στο ίδιο έγγραφο.

#### Βήμα 1: Προσθέστε τον πρώτο σας τοπικό υπερσύνδεσμο
Δημιουργήστε κείμενο που λειτουργεί ως υπερσύνδεσμος:
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Καθορίστε τον κατάλογό σας
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Κατάλογος εξόδου

// Δημιουργήστε μια νέα παρουσία εγγράφου
document doc = new Document();
Page page = doc.Pages.Add();

// Βήμα 1: Προσθέστε τον πρώτο τοπικό υπερσύνδεσμο
textFragment text1 = new TextFragment("link to page 7");
hyperlink link1 = new LocalHyperlink();
link1.TargetPageNumber = 7; // Ορισμός αριθμού σελίδας-στόχου για πλοήγηση
text1.Hyperlink = link1;
doc.Pages[page.Number].Paragraphs.Add(text1);
```

#### Βήμα 2: Προσθήκη άλλου τοπικού υπερσυνδέσμου
Προσθήκη ενός ακόμη υπερσυνδέσμου σε μια νέα σελίδα μέσα στο έγγραφο:
```csharp
// Βήμα 2: Προσθέστε τον δεύτερο τοπικό υπερσύνδεσμο
textFragment text2 = new TextFragment("link to page 1");
text2.IsInNewPage = true; // Βεβαιωθείτε ότι αυτός ο σύνδεσμος εμφανίζεται σε μια νέα σελίδα
hyperlink link2 = new LocalHyperlink();
link2.TargetPageNumber = 1;
text2.Hyperlink = link2;
doc.Pages.Add().Paragraphs.Add(text2);
```

### Αποθήκευση του εγγράφου σας
Αποθηκεύστε το έγγραφό σας με τους υπερσυνδέσμους που περιλαμβάνονται:
```csharp
string outputFilePath = Path.Combine(outputDir, "CreateLocalHyperlink_out.pdf");
doc.Save(outputFilePath);
```

## Οδηγός Εφαρμογής
Τώρα, ας εφαρμόσουμε τη λειτουργία ζουμ σελιδοδεικτών στην εφαρμογή .NET σας βήμα προς βήμα.

### Βήμα 1: Φόρτωση του εγγράφου PDF
Αρχικά, φορτώστε το έγγραφο PDF εκεί που θέλετε να ορίσετε τους σελιδοδείκτες:
```csharp
// Διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir_AsposePdf_Bookmarks();

// Άνοιγμα του υπάρχοντος εγγράφου
Document doc = new Document(dataDir + "input.pdf");
```

### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων σελιδοδείκτη
Στη συνέχεια, δημιουργήστε έναν σελιδοδείκτη με προσαρμοσμένο επίπεδο ζουμ χρησιμοποιώντας `XYZExplicitDestination`:
```csharp
// Αρχικοποίηση του στοιχείου σελιδοδείκτη με προσαρμοσμένο ζουμ
OutlineItemCollection item = new OutlineItemCollection(doc.Outlines);

// Ορίστε το ζουμ σε 0 (προβολή πλήρους σελίδας) σε συγκεκριμένες συντεταγμένες
XYZExplicitDestination dest = new XYZExplicitDestination(2, 100, 100, 0);
item.Action = new GoToAction(dest);

// Προσθήκη του σελιδοδείκτη στη συλλογή περιγραμμάτων του εγγράφου
doc.Outlines.Add(item);
```

### Βήμα 3: Αποθήκευση του ενημερωμένου εγγράφου
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα αρχείο PDF:
```csharp
dataDir += "InheritZoom_out.pdf";

// Αποθηκεύστε το τροποποιημένο PDF με ενημερωμένους σελιδοδείκτες
doc.Save(dataDir);

Console.WriteLine("\nBookmarks updated successfully.\nFile saved at " + dataDir);
```

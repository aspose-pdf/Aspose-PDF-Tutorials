## Οδηγός Εφαρμογής

### Προσθήκη σφραγίδας κειμένου

#### Επισκόπηση
Η προσθήκη μιας σφραγίδας κειμένου περιλαμβάνει τη δημιουργία και τη διαμόρφωση μιας `Stamp` αντικείμενο που μπορεί να τοποθετηθεί σε συγκεκριμένες σελίδες ενός εγγράφου PDF. Αυτή η λειτουργία είναι χρήσιμη για την επωνυμία ή την επισήμανση εγγράφων ως εμπιστευτικών.

#### Βήμα προς βήμα εφαρμογή

##### Αρχικοποίηση αντικειμένου PdfFileStamp

Το πρώτο βήμα στη διαδικασία μας είναι η αρχικοποίηση του `PdfFileStamp` αντικείμενο, το οποίο λειτουργεί ως διεπαφή για την εργασία με αρχεία PDF.

```csharp
// Βήμα 1: Αρχικοποίηση αντικειμένου PdfFileStamp
PdfFileStamp fileStamp = new PdfFileStamp();
```

**Εξήγηση**: Το `PdfFileStamp` Η κλάση παρέχει μεθόδους για την προσθήκη σφραγίδων και την αποθήκευση αλλαγών. Διαχειρίζεται αποτελεσματικά τους πόρους κλείνοντας τις συνδέσεις όταν ολοκληρωθούν.

##### Άνοιγμα του εγγράφου PDF

Στη συνέχεια, καθορίστε τη διαδρομή προς το έγγραφο PDF εισόδου χρησιμοποιώντας το `BindPdf` μέθοδος.

```csharp
// Βήμα 2: Ανοίξτε το έγγραφο PDF
string dataDir = @"YOUR_DOCUMENT_DIRECTORY"; // Θέση κράτησης θέσης για τον κατάλογο εισόδου
fileStamp.BindPdf(dataDir + "AddTextStamp-Page.pdf");
```

**Εξήγηση**: Το `dataDir` η μεταβλητή θα πρέπει να δείχνει στο αρχείο PDF σας. Βεβαιωθείτε ότι έχετε αντικαταστήσει `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική διαδρομή.

##### Δημιουργία και ρύθμιση παραμέτρων σφραγίδας κειμένου

Εδώ, θα δημιουργήσουμε ένα νέο `Stamp` αντικείμενο και ορίστε τις ιδιότητές του για περιεχόμενο κειμένου, χρώμα, στυλ γραμματοσειράς και τοποθέτηση.

```csharp
// Βήμα 3: Δημιουργία και διαμόρφωση σφραγίδας κειμένου
Aspose.Pdf.Facades.Stamp stamp = new Aspose.Pdf.Facades.Stamp();
stamp.BindLogo(new FormattedText(
    "Hello World!", 
    Color.Blue, 
    Color.Gray, 
    FontStyle.Helvetica, 
    EncodingType.Winansi, 
    true, 14));
stamp.SetOrigin(200, 200); // Θέση της σφραγίδας στη σελίδα
stamp.Rotation = 90.0F; // Γωνία περιστροφής σε μοίρες
stamp.IsBackground = true; // Τοποθετήστε τη σφραγίδα πίσω από το περιεχόμενο
```

**Εξήγηση**: Το `FormattedText` Ο κατασκευαστής ορίζει τις ιδιότητες κειμένου, όπως το χρώμα και τη γραμματοσειρά. Προσαρμόστε τις συντεταγμένες προέλευσης για να τοποθετήσετε τη σφραγίδα ακριβώς εκεί που τη χρειάζεστε.

##### Καθορισμός σελίδων για τη σφραγίδα

Μπορείτε να περιορίσετε ποιες σελίδες λαμβάνουν τη σφραγίδα ορίζοντας το `Pages` ιδιοκτησία του `Stamp` αντικείμενο.

```csharp
// Βήμα 4: Καθορίστε σε ποιες σελίδες θα προσθέσετε τη σφραγίδα
stamp.Pages = new int[] { 1 };
```

**Εξήγηση**Αυτό το παράδειγμα προσθέτει μια σφραγίδα μόνο στην πρώτη σελίδα. Τροποποιήστε τον πίνακα για να συμπεριλάβετε επιπλέον ή διαφορετικές σελίδες, όπως απαιτείται.

##### Προσθήκη της σφραγίδας και αποθήκευση

Τέλος, εφαρμόστε τη διαμορφωμένη σφραγίδα στο έγγραφό σας και αποθηκεύστε την.

```csharp
// Βήμα 5: Προσθέστε τη σφραγίδα στο έγγραφο PDF
fileStamp.AddStamp(stamp);

// Βήμα 6: Αποθηκεύστε το ενημερωμένο έγγραφο PDF
string outputDir = @"YOUR_OUTPUT_DIRECTORY"; // Θέση κράτησης θέσης για τον κατάλογο εξόδου
fileStamp.Save(outputDir + "AddTextStamp-Page_out.pdf");

// Βήμα 7: Κλείστε το αντικείμενο PdfFileStamp για να απελευθερώσετε πόρους
fileStamp.Close();
```

**Εξήγηση**: Να θυμάστε πάντα να κλείνετε το `PdfFileStamp` για να ελευθερώσετε πόρους συστήματος. Το τελικό έγγραφο θα αποθηκευτεί στον καθορισμένο κατάλογο εξόδου.

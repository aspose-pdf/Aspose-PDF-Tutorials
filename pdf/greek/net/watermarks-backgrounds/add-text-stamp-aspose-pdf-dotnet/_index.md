## Οδηγός Εφαρμογής

Τώρα ας αναλύσουμε πώς να προσθέσετε μια σφραγίδα κειμένου στο έγγραφο PDF σας χρησιμοποιώντας το Aspose.PDF για .NET.

### Φόρτωση του εγγράφου PDF

**Επισκόπηση**Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο PDF στην εφαρμογή σας όπου θα εφαρμόσετε τη σφραγίδα κειμένου.

**Βήματα Υλοποίησης:**
1. **Φόρτωση υπάρχοντος PDF:**
   ```csharp
έγγραφο = νέο έγγραφο("ΚΑΤΑΛΟΓΟΣ_ΕΓΓΡΑΦΩΝ_ΟΥ/Ορισμός_Στοίχισης.pdf");
```
Replace `"YOUR_DOCUMENT_DIRECTORY/DefineAlignment.pdf"` with your file path.

### Creating the Formatted Text

**Overview**: Define the text content for your stamp using `FormattedText`, supporting multi-line formatting.

**Implementation Steps:**
2. **Create FormattedText Object:**
   ```csharp
FormattedText formattedText = new FormattedText("This is sample Center Aligned TextStamp Object");
```
Ο `FormattedText` Η κλάση επιτρέπει τη δημιουργία εμπλουτισμένου κειμένου με πολλαπλές γραμμές και στυλ.

### Δημιουργία της σφραγίδας κειμένου

**Επισκόπηση**Χρησιμοποιήστε το μορφοποιημένο κείμενο για να δημιουργήσετε ένα `TextStamp`, διαμορφώνοντάς το για ευθυγράμμιση.

**Βήματα Υλοποίησης:**
3. **Δημιουργία αντικειμένου TextStamp:**
   ```csharp
TextStamp stamp = new TextStamp(formattedText);
```
4. **Ορισμός ιδιοτήτων ευθυγράμμισης:**
   ```csharp
// Ορίστε την οριζόντια και κάθετη στοίχιση στο κέντρο
stamp.HorizontalAlignment = HorizontalAlignment.Center;
stamp.VerticalAlignment = VerticalAlignment.Center;

// Ευθυγράμμιση κειμένου μέσα στην ίδια τη σφραγίδα με το κέντρο
stamp.TextAlignment = HorizontalAlignment.Center;
```
These properties ensure your stamp is perfectly centered on the page and within its bounds.

### Adding the Stamp

**Overview**: Position and add the text stamp to a specific page in the PDF document.

**Implementation Steps:**
5. **Set Top Margin for Placement:**
   ```csharp
stamp.TopMargin = 20;
```
6. **Προσθήκη TextStamp στην πρώτη σελίδα:**
   ```csharp
έγγραφο.Σελίδες[1].ΠροσθήκηΣφραγίδας(σφραγίδα);
```
Here, we're adding the stamp to the first page of the document. Adjust the index as needed for your use case.

### Saving the Updated Document

**Overview**: Save the modified PDF with the newly added text stamp.

**Implementation Steps:**
7. **Save the Document:**
   ```csharp
document.Save("YOUR_OUTPUT_DIRECTORY/StampedPDF_out.pdf");
```
Αυτό το βήμα ολοκληρώνει και αποθηκεύει τις αλλαγές σας σε έναν καθορισμένο κατάλογο εξόδου.

## Οδηγός Εφαρμογής
Τώρα που το περιβάλλον σας έχει ρυθμιστεί, ας εφαρμόσουμε τη λειτουργία για να προσθέσουμε μια σφραγίδα αριθμού σελίδας.

### Άνοιγμα του εγγράφου
Αρχικά, φορτώστε το έγγραφο PDF που θέλετε να τροποποιήσετε:
```csharp
using Aspose.Pdf;

// Φόρτωση ενός υπάρχοντος εγγράφου PDF
document = new Document("YOUR_DOCUMENT_DIRECTORY/PageNumberStamp.pdf");
```

### Δημιουργία και διαμόρφωση της σφραγίδας αριθμού σελίδας
Ο στόχος είναι να δημιουργήσετε μια σφραγίδα αριθμού σελίδας που εμφανίζεται σε κάθε σελίδα του PDF σας, βοηθώντας στην πλοήγηση.

#### Βήμα προς βήμα εφαρμογή
##### Δημιουργήστε τη σφραγίδα αριθμού σελίδας
```csharp
using Aspose.Pdf.Text;

// Αρχικοποίηση μιας νέας παρουσίας της κλάσης PageNumberStamp
pageNumberStamp = new PageNumberStamp();
```

##### Ορισμός ιδιοτήτων σφραγίδας
Ρυθμίστε διάφορες ιδιότητες για να προσαρμόσετε τη σφραγίδα αριθμού σελίδας:
```csharp
// Βεβαιωθείτε ότι η σφραγίδα βρίσκεται στο προσκήνιο
pageNumberStamp.Background = false;

// Μορφοποίηση του κειμένου της σφραγίδας
pageNumberStamp.Format = "Page # of " + document.Pages.Count;

// Ορίστε το περιθώριο από το κάτω μέρος της σελίδας
pageNumberStamp.BottomMargin = 10;

// Κεντράρετε τη σφραγίδα οριζόντια
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;

// Έναρξη αρίθμησης από τη σελίδα 1
pageNumberStamp.StartingNumber = 1;
```

##### Προσαρμόστε την εμφάνιση κειμένου
Ορίστε τη γραμματοσειρά, το μέγεθος, το στυλ και το χρώμα για να κάνετε τη σφραγίδα σας να ξεχωρίζει:
```csharp
// Ορισμός γραμματοσειράς, μεγέθους και στυλ
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;

// Επιλέξτε ένα χρώμα κειμένου
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;
```

##### Προσθήκη σφραγίδας στις σελίδες
Εφαρμόστε τη σφραγίδα σε κάθε σελίδα του εγγράφου σας:
```csharp
foreach (Page page in document.Pages)
{
    // Προσθέστε τη διαμορφωμένη σφραγίδα σε κάθε σελίδα
    page.AddStamp(pageNumberStamp);
}
```

### Αποθήκευση του εγγράφου
Αφού προσθέσετε τη σφραγίδα, αποθηκεύστε το PDF σας με τις αλλαγές:
```csharp
// Αποθήκευση του ενημερωμένου εγγράφου σε μια καθορισμένη διαδρομή
document.Save("YOUR_OUTPUT_DIRECTORY/PageNumberStamp_out.pdf");
```

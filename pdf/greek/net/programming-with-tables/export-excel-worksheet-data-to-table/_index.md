# Εξαγωγή δεδομένων φύλλου εργασίας Excel σε πίνακα

## Εισαγωγή

Έχετε ποτέ χρειαστεί να εξαγάγετε δεδομένα από ένα φύλλο εργασίας του Excel σε ένα αρχείο PDF, τακτοποιημένα σε μορφή πίνακα; Φανταστείτε να έχετε πολλά δεδομένα στο Excel, αλλά να χρειάζεται να τα μοιραστείτε ως ένα PDF επαγγελματικής εμφάνισης. Μπορεί να ακούγεται περίπλοκο, σωστά; Αλλά με το Aspose.PDF για .NET, μπορείτε να κάνετε αυτήν την εργασία παιχνιδάκι. Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη διαδικασία εξαγωγής δεδομένων φύλλου εργασίας Excel σε έναν πίνακα μέσα σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Θα σας καθοδηγήσουμε βήμα προς βήμα, αναλύοντας τα πάντα, έτσι ώστε ακόμα κι αν είστε αρχάριοι σε αυτό, να νιώσετε σαν επαγγελματίας στο τέλος.

## Προαπαιτούμενα

Πριν προχωρήσουμε στην κωδικοποίηση, ας ρυθμίσουμε μερικά πράγματα:

1. Aspose.PDF για τη βιβλιοθήκη .NET – Βεβαιωθείτε ότι έχετε εγκαταστήσει την πιο πρόσφατη έκδοση. Μπορείτε [κατεβάστε το εδώ](https://releases.aspose.com/pdf/net/).
2. Aspose.Cells για τη βιβλιοθήκη .NET – Θα το χρειαστείτε για να χειρίζεστε λειτουργίες του Excel. Κατεβάστε το από [εδώ](https://releases.aspose.com/cells/net/).
3. Περιβάλλον ανάπτυξης .NET – Ένα εργαλείο όπως το Visual Studio θα λειτουργήσει τέλεια για τον προγραμματισμό.
4. Αρχείο Excel – Να έχετε έτοιμο ένα αρχείο Excel με τα δεδομένα που θέλετε να εξαγάγετε.

Εάν δεν έχετε τις βιβλιοθήκες Aspose.PDF και Aspose.Cells, μπορείτε να ξεκινήσετε με ένα [δωρεάν δοκιμή](https://releases.aspose.com/).

## Εισαγωγή πακέτων

Αρχικά, βεβαιωθείτε ότι έχετε εγκαταστήσει τόσο τις βιβλιοθήκες Aspose.PDF όσο και τις βιβλιοθήκες Aspose.Cells στο έργο σας. Μπορείτε να τις εγκαταστήσετε χρησιμοποιώντας το NuGet Package Manager στο Visual Studio.

Δείτε πώς μπορείτε να εισαγάγετε τα απαραίτητα πακέτα στον κώδικα C#:

```csharp
using System.Data;
using System.IO;
using System.Linq;
```

Τώρα που έχουν οριστεί οι προϋποθέσεις, ας δούμε τη διαδικασία εξαγωγής δεδομένων από ένα φύλλο Excel σε έναν πίνακα σε ένα έγγραφο PDF.

## Βήμα 1: Φόρτωση του βιβλίου εργασίας του Excel

Για να ξεκινήσετε, πρέπει να φορτώσετε το βιβλίο εργασίας του Excel στο πρόγραμμα. Σε αυτό το βήμα, θα χρησιμοποιήσουμε το Aspose.Cells για να ανοίξουμε το αρχείο Excel.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Φόρτωση του βιβλίου εργασίας του Excel
Aspose.Cells.Workbook workbook = new Aspose.Cells.Workbook(new FileStream(dataDir + "newBook1.xlsx", FileMode.Open));
```

Εξήγηση: Εδώ, καθορίζουμε τη διαδρομή καταλόγου όπου βρίσκεται το αρχείο Excel και φορτώνουμε το βιβλίο εργασίας χρησιμοποιώντας `Aspose.Cells.Workbook`Βεβαιωθείτε ότι έχετε προσαρμόσει `"YOUR DOCUMENT DIRECTORY"` για να υποδείξει την τοποθεσία του αρχείου σας.

## Βήμα 2: Πρόσβαση στο πρώτο φύλλο εργασίας

Αφού φορτώσουμε το βιβλίο εργασίας, πρέπει να έχουμε πρόσβαση στο πρώτο φύλλο εργασίας όπου είναι αποθηκευμένα τα δεδομένα μας.

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel
Aspose.Cells.Worksheet worksheet = workbook.Worksheets[0];
```

Εξήγηση: Αυτό το βήμα είναι απλό—παίρνουμε το πρώτο φύλλο εργασίας από το βιβλίο εργασίας, το οποίο περιέχει τα δεδομένα που θα εξαχθούν.

## Βήμα 3: Εξαγωγή δεδομένων σε DataTable

Τώρα, ας εξαγάγουμε τα δεδομένα από το φύλλο Excel σε ένα αντικείμενο DataTable, το οποίο θα λειτουργήσει ως ενδιάμεσος για τη μεταφορά των δεδομένων στο PDF.

```csharp
// Εξαγωγή των περιεχομένων 7 γραμμών και 2 στηλών ξεκινώντας από το 1ο κελί στον Πίνακα Δεδομένων
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, worksheet.Cells.MaxRow + 1, worksheet.Cells.MaxColumn + 1, true);
```

Εξήγηση: Το `ExportDataTable` Η μέθοδος εξάγει τα δεδομένα ξεκινώντας από το πρώτο κελί του φύλλου εργασίας και εκτείνεται σε όλες τις γραμμές και τις στήλες. Αυτά τα δεδομένα αποθηκεύονται στη συνέχεια σε ένα `DataTable` για περαιτέρω χρήση.

## Βήμα 4: Δημιουργία νέου εγγράφου PDF

Στη συνέχεια, πρέπει να δημιουργήσουμε ένα νέο έγγραφο PDF χρησιμοποιώντας το Aspose.PDF.

```csharp
// Δημιουργία στιγμιαίας εμφάνισης εγγράφου
Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document();

// Δημιουργία σελίδας στην παρουσία εγγράφου
Aspose.Pdf.Page page = pdfDocument.Pages.Add();
```

Εξήγηση: Εδώ, αρχικοποιούμε ένα νέο `Aspose.Pdf.Document` και προσθέστε μια σελίδα σε αυτήν. Αυτή η σελίδα θα περιέχει αργότερα τον πίνακα που δημιουργούμε από τα δεδομένα του Excel.

## Βήμα 5: Δημιουργήστε ένα αντικείμενο πίνακα σε PDF

Ας προχωρήσουμε στη δημιουργία ενός πίνακα μέσα στο έγγραφο PDF.

```csharp
// Δημιουργήστε ένα αντικείμενο Πίνακα
Aspose.Pdf.Table table = new Aspose.Pdf.Table();

// Προσθήκη του αντικειμένου Πίνακας στη συλλογή παραγράφων της σελίδας
page.Paragraphs.Add(table);
```

Εξήγηση: Δημιουργούμε ένα `Aspose.Pdf.Table` αντικείμενο και να το προσθέσετε στη συλλογή παραγράφων της σελίδας, κάτι που διασφαλίζει ότι ο πίνακας θα εμφανίζεται στη σελίδα.

## Βήμα 6: Ορισμός πλάτους και περιγραμμάτων στηλών

Οι πίνακες σε PDF χρειάζονται καθορισμένα πλάτη στηλών. Θα προσθέσουμε επίσης περιγράμματα για να κάνουμε τον πίνακα πιο ευανάγνωστο.

```csharp
// Ορισμός πλάτους στηλών του πίνακα
table.ColumnWidths = "40 100 100";

// Ορισμός προεπιλεγμένου περιγράμματος κελιού
table.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);
```

Εξήγηση: Ορίζουμε τα πλάτη των τριών στηλών και δίνουμε σε όλα τα κελιά ένα προεπιλεγμένο περίγραμμα με πάχος `0.1F`.

## Βήμα 7: Εισαγωγή δεδομένων από το DataTable σε πίνακα PDF

Τώρα, ήρθε η ώρα να εισαγάγουμε τα δεδομένα από το DataTable στον πίνακα PDF μας.

```csharp
// Εισαγωγή δεδομένων στο αντικείμενο Πίνακας από τον Πίνακα Δεδομένων
table.ImportDataTable(dataTable, true, 0, 0, dataTable.Rows.Count + 1, dataTable.Columns.Count);
```

Εξήγηση: Το `ImportDataTable` Η μέθοδος μεταφέρει όλα τα δεδομένα από το `DataTable` στον πίνακα PDF. Αυτό συμπληρώνει τον πίνακα με τα δεδομένα από το φύλλο Excel σας.

## Βήμα 8: Διαμόρφωση της γραμμής κεφαλίδας

Ας διαμορφώσουμε τη γραμμή κεφαλίδας του πίνακα αλλάζοντας το χρώμα φόντου, τη γραμματοσειρά και τη στοίχιση.

```csharp
// Λήψη της πρώτης γραμμής από τον πίνακα
Aspose.Pdf.Row headerRow = table.Rows[0];

// Ορισμός στυλ για τη γραμμή κεφαλίδας
foreach (Aspose.Pdf.Cell cell in headerRow.Cells)
{
    cell.BackgroundColor = Color.Blue;
    cell.DefaultCellTextState.Font = Aspose.Pdf.Text.FontRepository.FindFont("Helvetica-Oblique");
    cell.DefaultCellTextState.ForegroundColor = Color.Yellow;
    cell.DefaultCellTextState.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Center;
}
```

Εξήγηση: Κάνουμε επανάληψη σε όλα τα κελιά της πρώτης γραμμής (κεφαλίδα) και ορίζουμε το χρώμα φόντου τους σε μπλε, το χρώμα κειμένου σε κίτρινο και ευθυγραμμίζουμε το κείμενο με το κέντρο.

## Βήμα 9: Στυλίστε τις υπόλοιπες σειρές

Για να διαφοροποιήσουμε την κεφαλίδα από τις υπόλοιπες γραμμές, ας προσθέσουμε ένα διαφορετικό στυλ για τις υπόλοιπες γραμμές.

```csharp
for (int i = 1; i <= dataTable.Rows.Count; i++)
{
    foreach (Aspose.Pdf.Cell cell in table.Rows[i].Cells)
    {
        cell.BackgroundColor = Color.Gray;
        cell.DefaultCellTextState.ForegroundColor = Color.White;
    }
}
```

Εξήγηση: Για όλες τις γραμμές εκτός από την κεφαλίδα, ορίζουμε γκρι φόντο και λευκό κείμενο.

## Βήμα 10: Αποθήκευση του εγγράφου PDF

Τέλος, αποθηκεύστε το έγγραφο PDF μαζί με τον πίνακα.

```csharp
// Αποθήκευση του PDF
pdfDocument.Save(dataDir + "Exceldata_toPdf_table.pdf");
```

Εξήγηση: Αποθηκεύουμε το PDF στον καθορισμένο κατάλογο. Ιδού! Τα δεδομένα του Excel σας βρίσκονται τώρα μέσα σε έναν όμορφα μορφοποιημένο πίνακα PDF.

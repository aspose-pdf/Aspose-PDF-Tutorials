# Προσθήκη επαναλαμβανόμενης στήλης σε έγγραφο PDF

## Εισαγωγή

Αν εργάζεστε με έγγραφα PDF και χρειάζεται να προσθέσετε επαναλαμβανόμενες στήλες, βρίσκεστε στο σωστό μέρος! Χρησιμοποιώντας το Aspose.PDF για .NET, μπορείτε εύκολα να διαχειριστείτε πίνακες και περιεχόμενο μέσα σε ένα PDF. Είτε δημιουργείτε δυναμικές αναφορές, τιμολόγια είτε οποιοδήποτε άλλο δομημένο έγγραφο, οι επαναλαμβανόμενες στήλες μπορούν να αλλάξουν τα δεδομένα στην οργάνωση των δεδομένων. Ας δούμε έναν αναλυτικό οδηγό για το πώς να προσθέσετε επαναλαμβανόμενες στήλες σε ένα έγγραφο PDF.

## Εισαγωγή πακέτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων από το Aspose.PDF για .NET. Δείτε πώς μπορείτε να το κάνετε:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

Αυτά τα πακέτα παρέχουν τις απαραίτητες κλάσεις και μεθόδους που απαιτούνται για την εργασία με έγγραφα PDF και τον χειρισμό πινάκων.

Τώρα, ας χωρίσουμε τη διαδικασία σε πολλά βήματα για να προσθέσουμε επαναλαμβανόμενες στήλες σε ένα έγγραφο PDF. Ακολουθήστε μας!

## Βήμα 1: Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας

Πριν δημιουργήσουμε ή επεξεργαστούμε οποιαδήποτε αρχεία, πρέπει να ορίσουμε τη διαδρομή όπου θα αποθηκευτεί το δημιουργημένο PDF. Στο έργο C#, ορίστε τη διαδρομή του καταλόγου όπου θα βρίσκονται τα αρχεία σας:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
string outFile = dataDir + "AddRepeatingColumn_out.pdf";
```

Αυτή η διαδρομή δείχνει στον κατάλογο όπου θα αποθηκευτεί το PDF εξόδου. Αντικαταστήστε `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στο μηχάνημά σας.

## Βήμα 2: Δημιουργήστε ένα νέο έγγραφο PDF

Για να ξεκινήσετε, δημιουργήστε ένα νέο `Document` αντικείμενο. Αυτό θα χρησιμεύσει ως το δοχείο για όλες τις σελίδες και το περιεχόμενο μέσα στο PDF.

```csharp
Document doc = new Document();
Aspose.Pdf.Page page = doc.Pages.Add();
```

Εδώ, δημιουργήσαμε ένα νέο έγγραφο PDF και προσθέσαμε μια κενή σελίδα σε αυτό. Το `doc.Pages.Add()` Η μέθοδος εισάγει μια νέα σελίδα στο έγγραφο.

## Βήμα 3: Δημιουργία στιγμιαίας δημιουργίας του εξωτερικού πίνακα

Στη συνέχεια, δημιουργούμε έναν εξωτερικό πίνακα. Αυτός ο πίνακας θα καλύπτει ολόκληρο το πλάτος της σελίδας και θα χρησιμεύει ως δοχείο για άλλους πίνακες, συμπεριλαμβανομένου αυτού που θα περιέχει τις επαναλαμβανόμενες στήλες.

```csharp
Aspose.Pdf.Table outerTable = new Aspose.Pdf.Table();
outerTable.ColumnWidths = "100%";
outerTable.HorizontalAlignment = HorizontalAlignment.Left;
```

Έχουμε ορίσει το `ColumnWidths` αλλαγή της ιδιότητας σε "100%", που σημαίνει ότι ο πίνακας θα εκτείνεται σε όλο το πλάτος της σελίδας.

## Βήμα 4: Δημιουργήστε τον εσωτερικό πίνακα

Τώρα, ας δημιουργήσουμε τον εσωτερικό πίνακα, ο οποίος θα έχει επαναλαμβανόμενες στήλες. Οι βασικές ιδιότητες εδώ είναι `Broken`, το οποίο επιτρέπει στον πίνακα να συνεχίζεται στην ίδια σελίδα, και `ColumnAdjustment`, το οποίο προσαρμόζει αυτόματα τα πλάτη των στηλών ώστε να ταιριάζουν στο περιεχόμενο.

```csharp
Aspose.Pdf.Table mytable = new Aspose.Pdf.Table();
mytable.Broken = TableBroken.VerticalInSamePage;
mytable.ColumnAdjustment = ColumnAdjustment.AutoFitToContent;
```

Αυτός ο εσωτερικός πίνακας θα ενσωματωθεί μέσα στον εξωτερικό πίνακα.

## Βήμα 5: Προσθήκη πινάκων στη σελίδα

Τώρα που έχουμε έτοιμους και τους εξωτερικούς και τους εσωτερικούς πίνακες, ας τους προσθέσουμε στη σελίδα. Αυτό το βήμα διασφαλίζει ότι οι πίνακες περιλαμβάνονται στη δομή του εγγράφου.

```csharp
page.Paragraphs.Add(outerTable);
var bodyRow = outerTable.Rows.Add();
var bodyCell = bodyRow.Cells.Add();
bodyCell.Paragraphs.Add(mytable);
mytable.RepeatingColumnsCount = 5;
```

Εδώ, προσθέσαμε το `outerTable` στη σελίδα και, στη συνέχεια, μέσα στον εξωτερικό πίνακα, τοποθετήσαμε το `mytable`Επιπλέον, ορίσαμε `RepeatingColumnsCount` έως 5, καθορίζοντας πόσες στήλες θα πρέπει να επαναλαμβάνονται όταν προστίθενται δεδομένα.

## Βήμα 6: Προσθήκη γραμμής κεφαλίδας

Τώρα είναι η ώρα να προσθέσετε τις κεφαλίδες στον πίνακα. Η γραμμή κεφαλίδας δίνει το πλαίσιο στα δεδομένα και βοηθά στη δομή των στηλών. 

```csharp
Aspose.Pdf.Row row = mytable.Rows.Add();
row.Cells.Add("header 1");
row.Cells.Add("header 2");
row.Cells.Add("header 3");
row.Cells.Add("header 4");
row.Cells.Add("header 5");
row.Cells.Add("header 6");
row.Cells.Add("header 7");
row.Cells.Add("header 11");
row.Cells.Add("header 12");
row.Cells.Add("header 13");
row.Cells.Add("header 14");
row.Cells.Add("header 15");
row.Cells.Add("header 16");
row.Cells.Add("header 17");
```

Αυτό το απόσπασμα κώδικα προσθέτει την πρώτη γραμμή (την οποία θα χρησιμοποιήσουμε ως κεφαλίδες) και τη συμπληρώνει με κελιά που περιέχουν κείμενο όπως "κεφαλίδα 1", "κεφαλίδα 2" και ούτω καθεξής.

## Βήμα 7: Προσθήκη γραμμών δεδομένων

Τέλος, μπορούμε να προσθέσουμε κάποια δεδομένα στον πίνακα. Αυτός ο βρόχος δημιουργεί δυναμικά γραμμές και γεμίζει τα κελιά με περιεχόμενο:

```csharp
for (int RowCounter = 0; RowCounter <= 5; RowCounter++)
{
    Aspose.Pdf.Row row1 = mytable.Rows.Add();
    row1.Cells.Add("col " + RowCounter.ToString() + ", 1");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 2");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 3");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 4");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 5");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 6");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 7");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 11");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 12");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 13");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 14");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 15");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 16");
    row1.Cells.Add("col " + RowCounter.ToString() + ", 17");
}
```

Ο βρόχος επαναλαμβάνεται έξι φορές, προσθέτοντας γραμμές και γεμίζοντας κάθε κελί με τα αντίστοιχα δεδομένα στήλης (π.χ., "στήλη 1, 1", "στήλη 2, 2", κ.λπ.).

## Βήμα 8: Αποθήκευση του εγγράφου

Μόλις προστεθούν όλες οι γραμμές και οι στήλες, το τελευταίο βήμα είναι να αποθηκεύσετε το έγγραφο στην καθορισμένη διαδρομή αρχείου.

```csharp
doc.Save(outFile);
```

Το έγγραφό σας αποθηκεύεται πλέον με επαναλαμβανόμενες στήλες!

# Αυτόματη προσαρμογή στο παράθυρο

## Εισαγωγή

Όταν εργάζεστε με PDF, είναι σύνηθες να ασχολείστε με πίνακες και υπάρχουν φορές που χρειάζεστε αυτούς τους πίνακες να ταιριάζουν τέλεια στο πλάτος μιας σελίδας. Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να προσαρμόσετε αυτόματα έναν πίνακα σε ένα παράθυρο χρησιμοποιώντας το Aspose.PDF για .NET. Αυτό μπορεί να κάνει τους πίνακές σας να φαίνονται κομψοί και οργανωμένοι, αποτρέποντας προβλήματα όπως υπερχείλιση ή ανομοιόμορφες στήλες. Είστε έτοιμοι να μάθετε; Ας ξεκινήσουμε!

## Εισαγωγή πακέτων

Πριν εμβαθύνετε στον κώδικα, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων:

```csharp
using System.IO;
using System;
using Aspose.Pdf;
```

Τώρα που είμαστε έτοιμοι, ας το αναλύσουμε σε απλά, εύπεπτα βήματα για να κατανοήσουμε πώς μπορείτε να προσαρμόσετε αυτόματα έναν πίνακα σε ένα παράθυρο χρησιμοποιώντας το Aspose.PDF για .NET.

## Βήμα 1: Αρχικοποίηση του αντικειμένου εγγράφου

Αρχικά, πρέπει να δημιουργήσετε ένα έγγραφο PDF. Σκεφτείτε αυτό το έγγραφο ως ένα κενό φύλλο όπου θα προσθέτετε σελίδες και πίνακες.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Δημιουργήστε ένα αντίγραφο του αντικειμένου Pdf καλώντας τον κενό κατασκευαστή του
Document doc = new Document();
```
  
Εδώ, δημιουργούμε ένα νέο έγγραφο χρησιμοποιώντας το `Document` τάξη από το Aspose.PDF. Το `dataDir` είναι η τοποθεσία όπου θα αποθηκευτεί το PDF σας αφού τελειώσετε.

## Βήμα 2: Προσθήκη σελίδας στο έγγραφο

Ένα έγγραφο PDF χρειάζεται σελίδες, σωστά; Ας προσθέσουμε μία.

```csharp
// Δημιουργήστε μια ενότητα (σελίδα) στο αντικείμενο Pdf
Page sec1 = doc.Pages.Add();
```
  
Προσθέσαμε μια νέα σελίδα στο έγγραφο χρησιμοποιώντας το `Pages.Add()` μέθοδος. Μπορείτε να το σκεφτείτε αυτό ως την προσθήκη ενός νέου φύλλου στο έγγραφό σας όπου θα τοποθετήσετε τον πίνακα.

## Βήμα 3: Δημιουργία και διαμόρφωση πίνακα

Τώρα είναι η ώρα να δημιουργήσετε έναν πίνακα και να τον προσαρμόσετε ώστε να ταιριάζει στο παράθυρο.

```csharp
// Δημιουργία στιγμιαίου αντικειμένου πίνακα
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
// Προσθήκη της συλλογής πίνακα σε παραγράφους της επιθυμητής ενότητας
sec1.Paragraphs.Add(tab1);
```
  
Αρχικοποιήσαμε ένα νέο `Table` αντικείμενο και το πρόσθεσα στη συλλογή παραγράφων της σελίδας. Κάθε σελίδα PDF μπορεί να έχει διαφορετικές παραγράφους και εδώ αντιμετωπίζουμε τον πίνακα ως παράγραφο.

## Βήμα 4: Ορισμός πλάτους στηλών και αυτόματη προσαρμογή στο παράθυρο

Στη συνέχεια, ορίζουμε τα πλάτη των στηλών και διασφαλίζουμε ότι ο πίνακας προσαρμόζεται ώστε να ταιριάζει στο παράθυρο.

```csharp
// Ορισμός πλάτους στηλών για τον πίνακα
tab1.ColumnWidths = "50 50 50";
tab1.ColumnAdjustment = ColumnAdjustment.AutoFitToWindow;
```
  
Ορίσαμε σταθερά πλάτη στηλών για τον πίνακα, αλλά προσθέσαμε επίσης `ColumnAdjustment.AutoFitToWindow`, το οποίο διασφαλίζει ότι ο πίνακας προσαρμόζει το μέγεθός του ώστε να ταιριάζει στο διαθέσιμο παράθυρο.

## Βήμα 5: Ορισμός περιγραμμάτων και περιθωρίων για τον πίνακα και τα κελιά

Οι πίνακες χωρίς περιθώρια είναι συχνά δυσανάγνωστοι. Ας ορίσουμε περιθώρια και περιθώρια για να φαίνονται τακτοποιημένα.

```csharp
// Ορισμός προεπιλεγμένου περιγράμματος κελιού χρησιμοποιώντας το αντικείμενο BorderInfo
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);

// Ορισμός περιγράμματος πίνακα χρησιμοποιώντας ένα άλλο προσαρμοσμένο αντικείμενο BorderInfo
tab1.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 1F);

// Δημιουργήστε ένα αντικείμενο MarginInfo και ορίστε τα αριστερά, κάτω, δεξιά και πάνω περιθώριά του
Aspose.Pdf.MarginInfo margin = new Aspose.Pdf.MarginInfo();
margin.Top = 5f;
margin.Left = 5f;
margin.Right = 5f;
margin.Bottom = 5f;

// Ορίστε την προεπιλεγμένη συμπλήρωση κελιών στο αντικείμενο MarginInfo
tab1.DefaultCellPadding = margin;
```
  
Τα περιγράμματα προστίθενται τόσο στον πίνακα όσο και στα κελιά χρησιμοποιώντας το `BorderInfo` κλάση, όπου ορίζετε το πάχος. Τα περιθώρια ορίζονται για να δίνουν στα κελιά κάποιο χώρο συμπλήρωσης.

## Βήμα 6: Προσθήκη γραμμών και κελιών στον πίνακα

Ένας πίνακας χωρίς περιεχόμενο; Αυτό δεν είναι καλό! Ας προσθέσουμε μερικές γραμμές και κελιά.

```csharp
// Δημιουργήστε γραμμές στον πίνακα και στη συνέχεια κελιά στις γραμμές
Aspose.Pdf.Row row1 = tab1.Rows.Add();
row1.Cells.Add("col1");
row1.Cells.Add("col2");
row1.Cells.Add("col3");

Aspose.Pdf.Row row2 = tab1.Rows.Add();
row2.Cells.Add("item1");
row2.Cells.Add("item2");
row2.Cells.Add("item3");
```
  
Δημιουργούμε δύο γραμμές και προσθέτουμε τρία κελιά σε κάθε γραμμή. Εδώ θα εισαγάγετε τα πραγματικά δεδομένα σας (τα οποία θα μπορούσαν να είναι οτιδήποτε, από συμβολοσειρές έως πιο σύνθετα στοιχεία).

## Βήμα 7: Αποθήκευση του εγγράφου

Μόλις όλα ρυθμιστούν, θα πρέπει να αποθηκεύσετε το νεοδημιουργημένο έγγραφο PDF.

```csharp
dataDir = dataDir + "AutoFitToWindow_out.pdf";
// Αποθήκευση ενημερωμένου εγγράφου που περιέχει το αντικείμενο πίνακα
doc.Save(dataDir);
```
  
Ο `doc.Save()` Η μέθοδος αποθηκεύει το PDF στον καθορισμένο κατάλογο. Σε αυτήν την περίπτωση, το έγγραφο θα αποθηκευτεί ως `AutoFitToWindow_out.pdf` στον καθορισμένο κατάλογό σας.

## Σύναψη

Και να το! Μόλις δημιουργήσατε έναν πίνακα που προσαρμόζεται αυτόματα στο παράθυρο χρησιμοποιώντας το Aspose.PDF για .NET. Αυτό όχι μόνο διασφαλίζει ότι ο πίνακάς σας φαίνεται επαγγελματικός και σωστά τοποθετημένος, αλλά σας δίνει και ευελιξία όταν εργάζεστε με διάφορα μεγέθη δεδομένων. ## Συχνές ερωτήσεις

### Μπορώ να προσθέσω περισσότερες γραμμές δυναμικά;  
Ναι, μπορείτε να συνεχίσετε να προσθέτετε γραμμές χρησιμοποιώντας το `tab1.Rows.Add()` μέθοδος, δυναμικά βασισμένη στο περιεχόμενο.

### Πώς μπορώ να προσαρμόσω τον πίνακα αν δεν θέλω να προσαρμόζεται αυτόματα;  
Μπορείτε να ρυθμίσετε χειροκίνητα το `ColumnWidths` χωρίς χρήση `ColumnAdjustment.AutoFitToWindow` για να διατηρήσετε ένα σταθερό πλάτος πίνακα.

### Μπορώ να προσθέσω εικόνες ή άλλο περιεχόμενο μέσα στα κελιά;  
Ναι, το Aspose.PDF σάς επιτρέπει να προσθέτετε εικόνες, κείμενο, ακόμη και άλλους πίνακες μέσα σε κελιά!

### Τι γίνεται αν χρειαστώ πιο σύνθετα στυλ τραπεζιού;  
Μπορείτε να προσαρμόσετε περαιτέρω τα στυλ πίνακα και κελιών χρησιμοποιώντας ιδιότητες όπως το χρώμα φόντου, τη στοίχιση κειμένου και τις ρυθμίσεις γραμματοσειράς.

### Είναι δυνατή η εξαγωγή αυτού του πίνακα σε μορφή διαφορετική από PDF;  
Απολύτως! Το Aspose.PDF υποστηρίζει εξαγωγή σε διάφορες μορφές όπως HTML, DOCX και άλλες.

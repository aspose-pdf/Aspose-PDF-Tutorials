### Προσθήκη λίστας με ταξινόμηση HTML σε έγγραφο PDF

**Επισκόπηση:**  
Αυτή η λειτουργία σάς επιτρέπει να εισάγετε δομημένο περιεχόμενο HTML, όπως λίστες, απευθείας σε ένα νέο αρχείο PDF. Είναι ιδανική όταν χρειάζεται να δημιουργήσετε έγγραφα με δυναμικό περιεχόμενο ιστού.

#### Βήμα 1: Δημιουργία αντικειμένου εγγράφου

Ξεκινήστε δημιουργώντας ένα νέο `Document` αντικείμενο:

```csharp
// Δημιουργήστε μια παρουσία της κλάσης Document
Document doc = new Document();
```

Αυτό το αντικείμενο θα χρησιμεύσει ως δοχείο για το αρχείο PDF σας.

#### Βήμα 2: Ορισμός περιεχομένου HTML

Δημιουργήστε μια συμβολοσειρά που περιέχει το περιεχόμενό σας HTML. Αυτό το παράδειγμα περιλαμβάνει μια ταξινομημένη λίστα:

```csharp
// Ορισμός περιεχομένου HTML με μια ταξινομημένη λίστα
string htmlContent = "<body style='line-height: 100px;'>\n" +
                     "<ul>\n" + 
                     "<li>First</li>\n" + 
                     "<li>Second</li>\n" + 
                     "<li>Third</li>\n" + 
                     "<li>Fourth</li>\n" + 
                     "<li>Fifth</li>\n" + 
                     "</ul>\n" +
                     "Text after the list.<br/>Next line<br/>Last line\n" + 
                     "</body>";
```

Αυτή η συμβολοσειρά θα μετατραπεί σε μορφή PDF.

#### Βήμα 3: Δημιουργία αντικειμένου HtmlFragment

Μετατρέψτε το περιεχόμενο HTML σας σε `HtmlFragment` αντικείμενο:

```csharp
// Δημιουργήστε ένα HtmlFragment με το περιεχόμενο HTML
HtmlFragment htmlFragment = new HtmlFragment(htmlContent);
```

Ο `HtmlFragment` Η κλάση αναλύει και μορφοποιεί τη συμβολοσειρά HTML για συμπερίληψη σε ένα PDF.

#### Βήμα 4: Προσθήκη σελίδας στο έγγραφο

Προσθέστε μια νέα σελίδα στο έγγραφό σας όπου θα εμφανίζεται η λίστα:

```csharp
// Προσθήκη κενής σελίδας στο έγγραφο
Page page = doc.Pages.Add();
```

Αυτό διασφαλίζει ότι το περιεχόμενό μας έχει χώρο μέσα στη δομή PDF.

#### Βήμα 5: Προσθήκη HtmlFragment στη σελίδα

Συμπεριλάβετε το `HtmlFragment` στη σελίδα που μόλις δημιουργήθηκε:

```csharp
// Προσθήκη του HtmlFragment στη συλλογή παραγράφων της σελίδας
page.Paragraphs.Add(htmlFragment);
```

Η συλλογή παραγράφων περιέχει όλα τα στοιχεία μιας σελίδας, συμπεριλαμβανομένου του κειμένου και των εικόνων.

#### Βήμα 6: Αποθήκευση του εγγράφου

Τέλος, αποθηκεύστε το έγγραφό σας σε μια καθορισμένη διαδρομή με το περιεχόμενο HTML που περιλαμβάνεται:

```csharp
// Ορισμός διαδρομής αρχείου εξόδου
string outFile = "YOUR_OUTPUT_DIRECTORY" + "/AddHTMLOrderedListIntoDocuments_out.pdf";

// Αποθήκευση του εγγράφου PDF
doc.Save(outFile);
```

Αυτό το βήμα δημιουργεί το τελικό PDF με τη λίστα παραγγελίας σας.

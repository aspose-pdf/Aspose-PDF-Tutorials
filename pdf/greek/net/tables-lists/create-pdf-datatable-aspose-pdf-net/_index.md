### Δημιουργία και συμπλήρωση ενός πίνακα δεδομένων
#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει τον τρόπο δημιουργίας ενός `DataTable`, ορίστε το σχήμα του και συμπληρώστε το με δεδομένα μέσω προγραμματισμού σε C#.

**Βήμα 1: Δημιουργήστε τον Πίνακα Δεδομένων**

```csharp
using System;
using System.Data;

DataTable CreateAndPopulateDataTable()
{
    // Δημιουργήστε έναν νέο Πίνακα Δεδομένων με το όνομα «Υπάλληλος»
    DataTable dt = new DataTable("Employee");

    // Ορίστε το σχήμα του πίνακα προσθέτοντας στήλες
    dt.Columns.Add("Employee_ID", typeof(Int32));
    dt.Columns.Add("Employee_Name", typeof(string));
    dt.Columns.Add("Gender", typeof(string));

    // Προσθήκη γραμμών στον Πίνακα Δεδομένων μέσω προγραμματισμού
    DataRow dr = dt.NewRow();
    dr[0] = 1; // Αναγνωριστικό εργαζομένου
    dr[1] = "John Smith"; // Όνομα υπαλλήλου
    dr[2] = "Male"; // Γένος
    dt.Rows.Add(dr);

    dr = dt.NewRow();
    dr[0] = 2;
    dr[1] = "Mary Miller";
    dr[2] = "Female";
    dt.Rows.Add(dr);

    return dt; // Επιστρέψτε τον συμπληρωμένο Πίνακα Δεδομένων
}
```
**Εξήγηση:**
- **Δημιουργία Πίνακα Δεδομένων**: Ένα νέο `DataTable` με το όνομα "Υπάλληλος" δημιουργείται ως παράδειγμα.
- **Ορισμός σχήματος**: Προστίθενται στήλες για να οριστεί η δομή, καθορίζοντας τύπους δεδομένων για κάθε στήλη.
- **Συμπλήρωση δεδομένων**: Δημιουργούνται γραμμές και συμπληρώνονται με δείγματα δεδομένων υπαλλήλων.

### Δημιουργία εγγράφου PDF με πίνακα από το DataTable
#### Επισκόπηση
Αυτό το μέρος εξηγεί πώς να δημιουργήσετε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF και να το συμπληρώσετε με έναν πίνακα που προέρχεται από το αρχείο σας. `DataTable`.

**Βήμα 2: Αρχικοποίηση του εγγράφου PDF**

```csharp
using System.IO;
using Aspose.Pdf;

void CreatePdfWithTable(DataTable dataTable)
{
    // Αρχικοποίηση νέου εγγράφου PDF
    Document doc = new Document();
    doc.Pages.Add();

    // Δημιουργήστε ένα αντικείμενο Πίνακα και ορίστε τις ιδιότητές του
    Aspose.Pdf.Table table = new Aspose.Pdf.Table();
    table.ColumnWidths = "40 100 100 100"; // Ορισμός πλάτους στηλών
    table.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, .5f, Aspose.Pdf.Color.FromRgb(System.Drawing.Color.LightGray));
    table.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, .5f, Aspose.Pdf.Color.FromRgb(System.Drawing.Color.LightGray));

    // Εισαγωγή δεδομένων από τον Πίνακα Δεδομένων στον Πίνακα PDF
    table.ImportDataTable(dataTable, true, 0, 1, 3, 3);

    // Προσθήκη του πίνακα στην πρώτη σελίδα του εγγράφου
    doc.Pages[1].Paragraphs.Add(table);

    string outputDir = "YOUR_OUTPUT_DIRECTORY";
    string outputFile = Path.Combine(outputDir, "DataIntegrated_out.pdf");

    // Αποθηκεύστε το έγγραφο PDF με τον ενσωματωμένο πίνακα δεδομένων
    doc.Save(outputFile);
}
```
**Εξήγηση:**
- **Αρχικοποίηση εγγράφου**: Ένα νέο `Document` Το αντικείμενο δημιουργείται για να αναπαραστήσει το PDF.
- **Διαμόρφωση πίνακα**Η εμφάνιση και η διάταξη του πίνακα διαμορφώνονται χρησιμοποιώντας ιδιότητες όπως το πλάτος των στηλών και τα περιγράμματα.
- **Εισαγωγή Δεδομένων**: Δεδομένα από το `DataTable` εισάγεται στο Aspose.PDF `Table`.
- **Οικονομία**: Το έγγραφο αποθηκεύεται σε έναν καθορισμένο κατάλογο.

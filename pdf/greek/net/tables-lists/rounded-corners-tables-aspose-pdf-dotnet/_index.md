## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει το περιβάλλον σας, ας προχωρήσουμε στη δημιουργία ενός πίνακα με στρογγυλεμένες γωνίες χρησιμοποιώντας το Aspose.PDF για .NET.

### Βήμα 1: Δημιουργήστε μια παρουσία της κλάσης πίνακα

Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Table` κλάση. Αυτό θα χρησιμεύσει ως το κοντέινερ μας για δεδομένα και επιλογές προσαρμογής.

```csharp
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
```

### Βήμα 2: Αρχικοποίηση του GraphInfo για τον ορισμό ιδιοτήτων περιγράμματος

Για να προσαρμόσετε το περίγραμμα του πίνακα, αρχικοποιήστε ένα `GraphInfo` αντικείμενο. Εδώ, ορίζουμε το χρώμα του περιγράμματος:

```csharp
GraphInfo graph = new GraphInfo();
graph.Color = Aspose.Pdf.Color.Red; // Ορισμός χρώματος περιγράμματος σε κόκκινο
```

### Βήμα 3: Δημιουργία αντικειμένου BorderInfo

Δημιουργήστε ένα `BorderInfo` αντικείμενο χρησιμοποιώντας το προηγουμένως ορισμένο `GraphInfo`Αυτό το βήμα περιλαμβάνει τον καθορισμό ότι όλες οι πλευρές του πίνακα θα χρησιμοποιούν αυτήν τη διαμόρφωση:

```csharp
BorderInfo bInfo = new BorderInfo(BorderSide.All, graph);
```

### Βήμα 4: Ορισμός ακτίνας στρογγυλεμένου περιγράμματος

Για να επιτύχετε στρογγυλεμένες γωνίες, καθορίστε μια ακτίνα περιγράμματος στο `BorderInfo` αντικείμενο. Εδώ το ορίζουμε σε 15 μονάδες για μια ανεπαίσθητη καμπύλη:

```csharp
bInfo.RoundedBorderRadius = 15;
```

### Βήμα 5: Αντιστοίχιση στυλ στρογγυλής γωνίας

Βεβαιωθείτε ότι οι γωνίες του τραπεζιού έχουν σχεδιαστεί σύμφωνα με τις προδιαγραφές μας, ρυθμίζοντας το `CornerStyle` ιδιοκτησία σε `Round`.

```csharp
tab1.CornerStyle = Aspose.Pdf.BorderCornerStyle.Round;
```

### Βήμα 6: Εφαρμογή πληροφοριών περιγράμματος στον πίνακα

Τέλος, αντιστοιχίστε τις διαμορφωμένες πληροφορίες περιγράμματος στον πίνακα. Αυτό ολοκληρώνει τη ρύθμιση για έναν οπτικά ελκυστικό πίνακα με στρογγυλεμένες γωνίες:

```csharp
tab1.Border = bInfo;
```

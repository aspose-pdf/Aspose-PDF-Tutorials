## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία και διαμόρφωση εγγράφου PDF με περιεχόμενο με ετικέτες

#### Επισκόπηση
Η δημιουργία ενός νέου εγγράφου PDF περιλαμβάνει τη ρύθμιση βασικών μεταδεδομένων όπως ο τίτλος και η γλώσσα, τα οποία είναι ζωτικής σημασίας για την προσβασιμότητα και το SEO.

**Οδηγίες βήμα προς βήμα**

##### Βήμα 3.1: Δημιουργία νέας παρουσίας εγγράφου
```csharp
Document document = new Document();
```
Εδώ, δημιουργείτε μια παρουσία του `Document` κλάση, η οποία λειτουργεί ως το αρχείο PDF σας.

##### Βήμα 3.2: Πρόσβαση και διαμόρφωση περιεχομένου με ετικέτες
```csharp
ITaggedContent taggedContent = document.TaggedContent;
taggedContent.SetTitle("Tagged Pdf Document");
taggedContent.SetLanguage("en-US");
```
Πρόσβαση στο `TaggedContent` ιδιότητα του εγγράφου σας για να ορίσετε ιδιότητες μεταδεδομένων όπως τίτλο και γλώσσα, βελτιώνοντας τόσο την προσβασιμότητα όσο και το SEO.

### Λειτουργία 2: Προσθήκη στοιχείων δομής σε ένα PDF με ετικέτες

#### Επισκόπηση
Η προσθήκη στοιχείων δομής, όπως ενότητες και κεφαλίδες, είναι απαραίτητη για τη δημιουργία λογικής ροής περιεχομένου στα PDF σας. 

**Οδηγίες βήμα προς βήμα**

##### Βήμα 3.3: Πρόσβαση στο βασικό στοιχείο
```csharp
StructureElement rootElement = taggedContent.RootElement;
```
Ξεκινήστε αποκτώντας πρόσβαση στο `RootElement` όπου θα προσθέσετε όλα τα δομημένα στοιχεία.

##### Βήμα 3.4: Δημιουργία και προσθήκη ενότητας
```csharp
SectElement sect = taggedContent.CreateSectElement();
rootElement.AppendChild(sect);
```
Δημιουργήστε ενότητες για να διαιρέσετε λογικά το περιεχόμενό σας, διευκολύνοντας την πλοήγηση στο έγγραφο από τους αναγνώστες οθόνης και τις μηχανές αναζήτησης.

##### Βήμα 3.5: Προσθήκη στοιχείου κεφαλίδας
```csharp
HeaderElement h1 = taggedContent.CreateHeaderElement(1);
h1.SetText("The Header");
h1.Title = "Title";
h1.Language = "en-US";
// Πρόσθετες ιδιότητες
h1.AlternativeText = "Alternative Text";
h1.ExpansionText = "Expansion Text";
h1.ActualText = "Actual Text";
sect.AppendChild(h1);
```
Οι κεφαλίδες παρέχουν σημασιολογικό νόημα στη δομή του εγγράφου. Εδώ, ορίζετε διάφορες ιδιότητες για να βελτιώσετε την προσβασιμότητα και το SEO.

### Λειτουργία 3: Αποθήκευση εγγράφου PDF με ετικέτα

#### Επισκόπηση
Μόλις το PDF σας δομηθεί με περιεχόμενο με ετικέτες, αποθηκεύστε το για να βεβαιωθείτε ότι όλες οι τροποποιήσεις θα διατηρηθούν.

**Οδηγίες βήμα προς βήμα**

##### Βήμα 3.6: Αποθήκευση του Δομημένου Εγγράφου
```csharp
string outputFile = "YOUR_OUTPUT_DIRECTORY/StructureElementsProperties.pdf";
document.Save(outputFile);
```
Χρησιμοποιήστε το `Save` μέθοδος για να διατηρήσετε τις αλλαγές σας σε έναν καθορισμένο κατάλογο, διασφαλίζοντας ότι διατηρούνται όλα τα δομημένα στοιχεία.

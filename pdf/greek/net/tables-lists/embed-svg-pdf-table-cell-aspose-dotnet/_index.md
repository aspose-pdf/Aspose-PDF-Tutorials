## Οδηγός Εφαρμογής

### Επισκόπηση της ενσωμάτωσης SVG σε κελιά πίνακα PDF
Αυτή η ενότητα θα σας καθοδηγήσει στην ενσωμάτωση μιας εικόνας SVG σε ένα κελί πίνακα μέσα σε ένα έγγραφο PDF. Αυτή η λειτουργία είναι χρήσιμη για την προσθήκη λογότυπων, εικονιδίων ή οποιωνδήποτε άλλων διανυσματικών γραφικών.

#### Βήμα 1: Δημιουργήστε την παρουσία εγγράφου και εικόνας
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργία αντικειμένου εγγράφου
Document doc = new Document();

// Δημιουργήστε μια παρουσία εικόνας για SVG
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.FileType = Aspose.Pdf.ImageFileType.Svg; // Ορίστε τον τύπο εικόνας ως SVG
img.File = dataDir + "SVGToPDF.svg"; // Καθορίστε τη διαδρομή προς το αρχείο SVG σας
img.FixWidth = 50; // Ορισμός πλάτους για την παρουσία εικόνας
img.FixHeight = 50; // Ορισμός ύψους για την παρουσία εικόνας
```

#### Βήμα 2: Ρύθμιση παραμέτρων και προσθήκη πίνακα
```csharp
// Δημιουργήστε έναν πίνακα και ρυθμίστε τα πλάτη των στηλών του
Aspose.Pdf.Table table = new Aspose.Pdf.Table();
table.ColumnWidths = "100 100";

// Προσθήκη γραμμής στον πίνακα
Aspose.Pdf.Row row = table.Rows.Add();

// Προσθήκη πρώτου κελιού με κείμενο
Aspose.Pdf.Cell cell1 = row.Cells.Add();
cell1.Paragraphs.Add(new TextFragment("First cell")); // Προσθήκη τμήματος κειμένου στη συλλογή παραγράφων του κελιού

// Προσθήκη δεύτερου κελιού και συμπερίληψη εικόνας SVG σε αυτό
Aspose.Pdf.Cell cell2 = row.Cells.Add();
cell2.Paragraphs.Add(img); // Προσθήκη εικόνας SVG στη συλλογή παραγράφων του κελιού
```

#### Βήμα 3: Εισαγωγή πίνακα στο έγγραφο
```csharp
// Δημιουργήστε μια νέα σελίδα και προσθέστε έναν πίνακα στη συλλογή παραγράφων της
Page page = doc.Pages.Add();
page.Paragraphs.Add(table);

// Ορισμός καταλόγου εξόδου για την αποθήκευση του PDF
string outputPath = outputDir + "AddSVGObject_out.pdf";
doc.Save(outputPath); // Αποθήκευση του εγγράφου με το αντικείμενο SVG που έχει προστεθεί
```

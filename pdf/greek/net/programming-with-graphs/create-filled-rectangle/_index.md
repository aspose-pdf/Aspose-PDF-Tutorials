# Δημιουργία γεμισμένου ορθογωνίου

## Εισαγωγή

Θέλατε ποτέ να δημιουργήσετε οπτικά ελκυστικά PDF μέσω προγραμματισμού; Αν ναι, τότε βρίσκεστε στο σωστό μέρος! Σε αυτό το σεμινάριο, θα βυθιστούμε στον κόσμο του Aspose.PDF για .NET, μιας ισχυρής βιβλιοθήκης που σας επιτρέπει να χειρίζεστε έγγραφα PDF με ευκολία. Σήμερα, θα επικεντρωθούμε στη δημιουργία ενός γεμισμένου ορθογωνίου μέσα σε ένα αρχείο PDF. Είτε είστε έμπειρος προγραμματιστής είτε μόλις ξεκινάτε, αυτός ο οδηγός θα σας καθοδηγήσει σε κάθε βήμα με φιλικό και ελκυστικό τρόπο. Πάρτε λοιπόν το καπέλο σας στον προγραμματισμό και ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν προχωρήσουμε στον κώδικα, υπάρχουν μερικά πράγματα που πρέπει να έχετε στη διάθεσή σας:

1. Visual Studio: Βεβαιωθείτε ότι έχετε εγκαταστήσει το Visual Studio στον υπολογιστή σας. Είναι ένα φανταστικό IDE για ανάπτυξη .NET.
2. Aspose.PDF για .NET: Θα χρειαστεί να κατεβάσετε και να εγκαταστήσετε τη βιβλιοθήκη Aspose.PDF. Μπορείτε να τη βρείτε [εδώ](https://releases.aspose.com/pdf/net/).
3. Βασικές γνώσεις C#: Λίγη εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τα αποσπάσματα κώδικα.

## Εισαγωγή πακέτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τα απαραίτητα πακέτα στο έργο σας σε C#. Δείτε πώς μπορείτε να το κάνετε:

### Δημιουργία νέου έργου

Ανοίξτε το Visual Studio και δημιουργήστε ένα νέο έργο C#. Μπορείτε να επιλέξετε μια εφαρμογή κονσόλας για απλότητα.

### Προσθήκη αναφοράς Aspose.PDF

1. Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων.
2. Επιλέξτε "Διαχείριση πακέτων NuGet".
3. Αναζητήστε το "Aspose.PDF" και εγκαταστήστε την πιο πρόσφατη έκδοση.

```csharp
using System.IO;
using System;
using Aspose.Pdf;
```

Τώρα που έχουμε όλα ρυθμισμένα, ας περάσουμε στον κώδικα!

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Πρώτα απ 'όλα, πρέπει να καθορίσετε τη διαδρομή όπου θα αποθηκευτεί το PDF σας. Αυτό είναι κρίσιμο επειδή υποδεικνύει στο πρόγραμμα πού να δημιουργήσει το αρχείο.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στον υπολογιστή σας όπου θέλετε να αποθηκεύσετε το PDF.

## Βήμα 2: Δημιουργία μιας παρουσίας εγγράφου

Στη συνέχεια, θα δημιουργήσουμε μια παρουσία του `Document` κλάση. Αυτή η κλάση αντιπροσωπεύει το έγγραφο PDF με το οποίο θα εργαστείτε.

```csharp
// Δημιουργία στιγμιότυπου εγγράφου
Document doc = new Document();
```

Αυτή η γραμμή αρχικοποιεί ένα νέο έγγραφο PDF το οποίο μπορούμε να χειριστούμε.

## Βήμα 3: Προσθήκη σελίδας στο έγγραφο

Τώρα, ας προσθέσουμε μια σελίδα στο έγγραφό μας. Κάθε PDF χρειάζεται τουλάχιστον μία σελίδα, σωστά;

```csharp
// Προσθήκη σελίδας σε συλλογή σελίδων αρχείου PDF
Page page = doc.Pages.Add();
```

Αυτός ο κώδικας προσθέτει μια νέα σελίδα στο έγγραφο, επιτρέποντάς μας να σχεδιάσουμε σχήματα πάνω σε αυτήν.

## Βήμα 4: Δημιουργία μιας παρουσίας γραφήματος

Για να σχεδιάσουμε σχήματα, πρέπει να δημιουργήσουμε ένα `Graph` παράδειγμα. Σκεφτείτε ένα γράφημα ως έναν καμβά όπου μπορείτε να σχεδιάσετε διάφορα σχήματα.

```csharp
// Δημιουργία στιγμιότυπου γραφήματος
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
```

Εδώ, δημιουργούμε ένα γράφημα με πλάτος 100 και ύψος 400.

## Βήμα 5: Προσθήκη του γραφήματος στη σελίδα

Τώρα που έχουμε το γράφημά μας, ας το προσθέσουμε στη σελίδα που δημιουργήσαμε νωρίτερα.

```csharp
// Προσθήκη αντικειμένου γραφήματος στη συλλογή παραγράφων της παρουσίας σελίδας
page.Paragraphs.Add(graph);
```

Αυτή η γραμμή συνδέει το γράφημα με τη σελίδα, καθιστώντας το έτοιμο για σχεδίαση.

## Βήμα 6: Δημιουργήστε μια παρουσία ορθογωνίου

Στη συνέχεια, θα δημιουργήσουμε ένα ορθογώνιο που θέλουμε να γεμίσουμε με χρώμα.

```csharp
// Δημιουργία αντιγράφου ορθογωνίου
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 120);
```

Σε αυτόν τον κώδικα, ορίζουμε τη θέση και το μέγεθος του ορθογωνίου. Οι παράμετροι αντιπροσωπεύουν τις συντεταγμένες x και y, το πλάτος και το ύψος.

## Βήμα 7: Καθορίστε το χρώμα γεμίσματος

Τώρα, ας επιλέξουμε ένα χρώμα για το ορθογώνιό μας. Θα το γεμίσουμε με κόκκινο για αυτό το παράδειγμα.

```csharp
// Καθορισμός χρώματος γεμίσματος για το αντικείμενο γραφήματος
rect.GraphInfo.FillColor = Aspose.Pdf.Color.Red;
```

Αυτή η γραμμή ορίζει το χρώμα γεμίσματος του ορθογωνίου σε κόκκινο. Μπορείτε να επιλέξετε οποιοδήποτε χρώμα θέλετε!

## Βήμα 8: Προσθέστε το ορθογώνιο στο γράφημα

Αφού ετοιμάσουμε το ορθογώνιο μας, ήρθε η ώρα να το προσθέσουμε στο γράφημα.

```csharp
// Προσθήκη αντικειμένου ορθογωνίου σε συλλογή σχημάτων του αντικειμένου Γράφημα
graph.Shapes.Add(rect);
```

Αυτός ο κώδικας προσθέτει το ορθογώνιο στο γράφημα, καθιστώντας το μέρος του σχεδίου μας.

## Βήμα 9: Αποθήκευση του εγγράφου PDF

Τέλος, πρέπει να αποθηκεύσουμε το έγγραφό μας στον καθορισμένο κατάλογο.

```csharp
dataDir = dataDir + "CreateFilledRectangle_out.pdf";
// Αποθήκευση αρχείου PDF
doc.Save(dataDir);
```

Αυτός ο κώδικας αποθηκεύει το αρχείο PDF με το όνομα `CreateFilledRectangle_out.pdf` στον κατάλογο που καθορίσατε νωρίτερα.

## Βήμα 10: Μήνυμα επιβεβαίωσης

Για να μας ενημερώσετε ότι όλα πήγαν ομαλά, μπορούμε να εκτυπώσουμε ένα μήνυμα επιβεβαίωσης.

```csharp
Console.WriteLine("\nFilled rectangle object created successfully.\nFile saved at " + dataDir);
```

Αυτή η γραμμή θα εμφανίσει ένα μήνυμα στην κονσόλα, επιβεβαιώνοντας ότι το γεμισμένο ορθογώνιο έχει δημιουργηθεί με επιτυχία.

# Αφαίρεση ενσωματωμένων γραμματοσειρών και βελτιστοποίηση αρχείων PDF

## Εισαγωγή

Στην ψηφιακή εποχή, τα PDF είναι πανταχού παρόντα. Είτε μοιράζεστε αναφορές, παρουσιάσεις είτε ηλεκτρονικά βιβλία, η Φορητή Μορφή Εγγράφου (PDF) είναι η ιδανική επιλογή για τη διατήρηση της ακεραιότητας των εγγράφων σας. Ωστόσο, καθώς δημιουργούμε και κοινοποιούμε περισσότερα PDF, τα μεγέθη των αρχείων μπορεί να διογκωθούν, καθιστώντας τα δυσκίνητα στην αποστολή ή την αποθήκευσή τους. Εδώ μπαίνει στο παιχνίδι το Aspose.PDF για .NET, προσφέροντας ισχυρά εργαλεία για τη βελτιστοποίηση των αρχείων PDF σας. Σε αυτό το σεμινάριο, θα εμβαθύνουμε στον τρόπο κατάργησης της ενσωμάτωσης γραμματοσειρών και βελτιστοποίησης αρχείων PDF χρησιμοποιώντας το Aspose.PDF για .NET.

## Προαπαιτούμενα

Πριν μπούμε στα πιο σημαντικά, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε για να ξεκινήσετε:

1. Visual Studio: Βεβαιωθείτε ότι έχετε εγκατεστημένο το Visual Studio στον υπολογιστή σας. Είναι το IDE που θα χρησιμοποιήσουμε για να γράψουμε και να εκτελέσουμε τον κώδικα .NET.
2. Aspose.PDF για .NET: Θα χρειαστεί να κατεβάσετε και να εγκαταστήσετε τη βιβλιοθήκη Aspose.PDF. Μπορείτε να την κατεβάσετε από το [σύνδεσμος λήψης](https://releases.aspose.com/pdf/net/).
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε τα τμήματα κώδικα που θα χρησιμοποιήσουμε.
4. Ένα αρχείο PDF: Να έχετε έτοιμο ένα αρχείο PDF που θέλετε να βελτιστοποιήσετε. Μπορείτε να χρησιμοποιήσετε οποιοδήποτε PDF, αλλά για επίδειξη, θα το αναφέρουμε ως `OptimizeDocument.pdf`.

## Εισαγωγή πακέτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τα απαραίτητα πακέτα στο έργο σας σε C#. Δείτε πώς μπορείτε να το κάνετε:

1. Ανοίξτε το έργο σας στο Visual Studio.
2. Προσθήκη αναφοράς στο Aspose.PDF: Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων, επιλέξτε "Διαχείριση πακέτων NuGet" και αναζητήστε `Aspose.PDF`Εγκαταστήστε το πακέτο.

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

Τώρα που έχουμε ρυθμίσει τα πάντα, ας αναλύσουμε τη διαδικασία βελτιστοποίησης σε διαχειρίσιμα βήματα.

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Πρώτα απ 'όλα, πρέπει να ορίσετε τη διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ θα αποθηκευτούν τα αρχεία PDF σας. Δείτε πώς μπορείτε να το κάνετε:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου βρίσκεται το αρχείο PDF σας. Αυτό είναι κρίσιμο επειδή το πρόγραμμα πρέπει να γνωρίζει πού να βρει το PDF που θέλετε να βελτιστοποιήσετε.

## Βήμα 2: Ανοίξτε το έγγραφο PDF

Τώρα που έχουμε ρυθμίσει τον κατάλογό μας, ήρθε η ώρα να ανοίξουμε το έγγραφο PDF που θέλουμε να βελτιστοποιήσουμε. Ορίστε ο κώδικας για να το κάνουμε αυτό:

```csharp
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "OptimizeDocument.pdf");
```

Αυτή η γραμμή κώδικα δημιουργεί ένα νέο `Document` αντικείμενο, το οποίο αντιπροσωπεύει το αρχείο PDF σας. Βεβαιωθείτε ότι το όνομα του αρχείου ταιριάζει με αυτό που έχετε στον κατάλογό σας.

## Βήμα 3: Ορισμός επιλογών βελτιστοποίησης

Στη συνέχεια, πρέπει να καθορίσουμε τις επιλογές βελτιστοποίησης. Σε αυτήν την περίπτωση, θέλουμε να καταργήσουμε την ενσωμάτωση γραμματοσειρών. Δείτε πώς μπορείτε να το ρυθμίσετε:

```csharp
// Ορισμός επιλογής UnembedFonts 
var optimizeOptions = new Pdf.Optimization.OptimizationOptions
{
    UnembedFonts = true
};
```

Ρυθμίζοντας `UnembedFonts` να `true`, δίνουμε οδηγίες στο Aspose.PDF να βελτιστοποιήσει το PDF καταργώντας την ενσωμάτωση των γραμματοσειρών. Αυτό μπορεί να μειώσει σημαντικά το μέγεθος του αρχείου, ειδικά εάν το PDF περιέχει πολλές ενσωματωμένες γραμματοσειρές.

## Βήμα 4: Βελτιστοποίηση του εγγράφου PDF

Αφού ορίσαμε τις επιλογές μας, ήρθε η ώρα να βελτιστοποιήσουμε το έγγραφο PDF. Ορίστε ο κώδικας για να το κάνουμε αυτό:

```csharp
Console.WriteLine("Start");
// Βελτιστοποιήστε ένα έγγραφο PDF χρησιμοποιώντας το OptimizationOptions
pdfDocument.OptimizeResources(optimizeOptions);
```

Αυτό το απόσπασμα κώδικα καλεί το `OptimizeResources` μέθοδος στο `pdfDocument` αντικείμενο, εφαρμόζοντας τις επιλογές βελτιστοποίησης που ορίσαμε νωρίτερα. Θα δείτε ένα μήνυμα στην κονσόλα που υποδεικνύει ότι η διαδικασία βελτιστοποίησης έχει ξεκινήσει.

## Βήμα 5: Αποθήκευση του ενημερωμένου εγγράφου

Αφού βελτιστοποιήσουμε το PDF, πρέπει να αποθηκεύσουμε το ενημερωμένο έγγραφο. Δείτε πώς μπορείτε να το κάνετε:

```csharp
// Αποθήκευση ενημερωμένου εγγράφου
pdfDocument.Save(dataDir + "OptimizeDocument_out.pdf");
Console.WriteLine("Finished");
```

Αυτός ο κώδικας αποθηκεύει το βελτιστοποιημένο PDF ως `OptimizeDocument_out.pdf` στον ίδιο κατάλογο. Μπορείτε να επιλέξετε διαφορετικό όνομα, αν προτιμάτε, αλλά η διατήρηση της ομοιότητας βοηθά στην αναγνώριση της αρχικής και της βελτιστοποιημένης έκδοσης.

## Βήμα 6: Σύγκριση μεγεθών αρχείων

Τέλος, είναι πάντα καλό να ελέγχετε πόσο χώρο έχετε εξοικονομήσει. Δείτε πώς μπορείτε να συγκρίνετε το αρχικό και το βελτιστοποιημένο μέγεθος αρχείων:

```csharp
var fi1 = new System.IO.FileInfo(dataDir + "OptimizeDocument.pdf");
var fi2 = new System.IO.FileInfo(dataDir + "OptimizeDocument_out.pdf");
Console.WriteLine("Original file size: {0}. Reduced file size: {1}", fi1.Length, fi2.Length);
```

Αυτός ο κώδικας ανακτά τα μεγέθη αρχείων τόσο των αρχικών όσο και των βελτιστοποιημένων PDF και τα εκτυπώνει στην κονσόλα. Είναι μια ικανοποιητική στιγμή να βλέπεις πόσο έχεις μειώσει το μέγεθος του αρχείου!

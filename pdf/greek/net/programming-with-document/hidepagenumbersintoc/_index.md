# Απόκρυψη αριθμών σελίδων στον πίνακα περιεχομένων

## Εισαγωγή

Όταν εργάζεστε με PDF, μερικές φορές μπορεί να θέλετε να δημιουργήσετε έναν Πίνακα Περιεχομένων (TOC), αλλά να διατηρήσετε την κομψότητα αποκρύπτοντας τους αριθμούς σελίδων. Ίσως το έγγραφο να ρέει καλύτερα χωρίς αυτούς ή ίσως είναι μια αισθητική επιλογή. Όποιος και αν είναι ο λόγος σας, εάν εργάζεστε με το Aspose.PDF για .NET, αυτό το σεμινάριο θα σας δείξει ακριβώς πώς να αποκρύψετε τους αριθμούς σελίδων στον Πίνακα Περιεχομένων σας.

## Εισαγωγή πακέτων

Πριν ξεκινήσετε τον κώδικα, βεβαιωθείτε ότι έχετε εισαγάγει τους ακόλουθους χώρους ονομάτων στο έργο σας σε C#. Αυτοί θα παρέχουν τις απαραίτητες κλάσεις και μεθόδους για την εργασία με έγγραφα PDF και τη δημιουργία του Πίνακα Περιεχομένων (TOC).

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

Τώρα που το περιβάλλον σας είναι έτοιμο και τα πακέτα έχουν εισαχθεί, ας αναλύσουμε κάθε βήμα της διαδικασίας. Θα καλύψουμε κάθε μέρος του κώδικα για να διασφαλίσουμε τη σαφήνεια, ώστε να μπορείτε να παρακολουθείτε εύκολα.

## Βήμα 1: Αρχικοποίηση του εγγράφου PDF

Το πρώτο πράγμα που πρέπει να κάνουμε είναι να δημιουργήσουμε ένα νέο έγγραφο PDF και να προσθέσουμε μια σελίδα για τον Πίνακα Περιεχομένων (TOC).


```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
string outFile = dataDir + "HiddenPageNumbers_out.pdf";
Document doc = new Document();
Page tocPage = doc.Pages.Add();
```

- dataDir: Αυτός είναι ο κατάλογος όπου θα αποθηκευτεί το αρχείο εξόδου σας.
- Document(): Αρχικοποιεί ένα νέο έγγραφο PDF.
- Pages.Add(): Προσθέτει μια νέα κενή σελίδα στο έγγραφο, η οποία αργότερα θα περιέχει τον Πίνακα Περιεχομένων σας.

## Βήμα 2: Ρύθμιση πληροφοριών περιεχομένων και τίτλου

Στη συνέχεια, θα ορίσουμε τις πληροφορίες του Πίνακα Περιεχομένων, συμπεριλαμβανομένου του τίτλου που θα εμφανίζεται στην κορυφή του Πίνακα Περιεχομένων.

```csharp
TocInfo tocInfo = new TocInfo();
TextFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20;
title.TextState.FontStyle = FontStyles.Bold;
tocInfo.Title = title;
tocPage.TocInfo = tocInfo;
```

- TocInfo: Αυτό το αντικείμενο περιέχει όλες τις πληροφορίες σχετικά με το TOC.
- TextFragment: Αντιπροσωπεύει το κείμενο του τίτλου του Πίνακα Περιεχομένων, εδώ το ορίζουμε ως "Πίνακας Περιεχομένων".
- Στυλ γραμματοσειράς: Διαμορφώνουμε τον τίτλο του πίνακα περιεχομένων ορίζοντας το μέγεθός του σε 20 και κάνοντας τον έντονο.
- tocPage.TocInfo: Αντιστοιχίζουμε τις πληροφορίες Περιεχομένου (TOC) στη σελίδα που θα εμφανίζει τον Πίνακα Περιεχομένων.

## Βήμα 3: Απόκρυψη αριθμών σελίδων στον Πίνακα Περιεχομένων

Τώρα για το διασκεδαστικό μέρος! Εδώ ρυθμίζουμε τον Πίνακα Περιεχομένων για να αποκρύπτει τους αριθμούς σελίδων.

```csharp
tocInfo.IsShowPageNumbers = false;
tocInfo.FormatArrayLength = 4;
```

- IsShowPageNumbers: Αυτός είναι ο μαγικός διακόπτης που αποκρύπτει τους αριθμούς σελίδων. Ορίστε τον σε `false`και οι αριθμοί σελίδων δεν θα εμφανίζονται στον Πίνακα Περιεχομένων.
- FormatArrayLength: Ορίζουμε αυτήν την τιμή σε 4, υποδεικνύοντας ότι θέλουμε να ορίσουμε μορφοποίηση για τέσσερα επίπεδα επικεφαλίδων TOC.

## Βήμα 4: Προσαρμογή μορφοποίησης πίνακα περιεχομένων

Για να προσθέσετε περισσότερο στυλ στον Πίνακα Περιεχομένων σας, θα ορίσουμε τώρα τη μορφοποίηση για διαφορετικά επίπεδα επικεφαλίδων.

```csharp
tocInfo.FormatArray[0].Margin.Right = 0;
tocInfo.FormatArray[0].TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;
tocInfo.FormatArray[1].Margin.Left = 30;
tocInfo.FormatArray[1].TextState.Underline = true;
tocInfo.FormatArray[1].TextState.FontSize = 10;
tocInfo.FormatArray[2].TextState.FontStyle = FontStyles.Bold;
tocInfo.FormatArray[3].TextState.FontStyle = FontStyles.Bold;
```

- FormatArray: Αυτός ο πίνακας ελέγχει τη μορφοποίηση των καταχωρίσεων TOC. Κάθε ευρετήριο αντιπροσωπεύει ένα διαφορετικό επίπεδο επικεφαλίδας.
- Περιθώριο και Στυλ Κειμένου: Ορίζουμε περιθώρια και εφαρμόζουμε στυλ γραμματοσειράς όπως έντονη γραφή, πλάγια γραφή και υπογράμμιση για κάθε επίπεδο επικεφαλίδας.

## Βήμα 5: Προσθήκη επικεφαλίδων στο έγγραφο

Τέλος, ας προσθέσουμε τις πραγματικές επικεφαλίδες που θα αποτελούν μέρος του Πίνακα Περιεχομένων.

```csharp
Page page = doc.Pages.Add();
for (int Level = 1; Level != 5; Level++)
{ 
    Heading heading2 = new Heading(Level); 
    TextSegment segment2 = new TextSegment(); 
    heading2.TocPage = tocPage; 
    heading2.Segments.Add(segment2); 
    heading2.IsAutoSequence = true; 
    segment2.Text = "this is heading of level " + Level; 
    heading2.IsInList = true; 
    page.Paragraphs.Add(heading2); 
}
```

- Επικεφαλίδα και Τμήμα Κειμένου: Αυτές αντιπροσωπεύουν τις επικεφαλίδες που θα εμφανίζονται στον Πίνακα Περιεχομένων σας. Κάθε επίπεδο έχει τη δική του επικεφαλίδα.
- IsAutoSequence: Αριθμεί αυτόματα τις επικεφαλίδες.
- IsInList: Εξασφαλίζει ότι κάθε επικεφαλίδα εμφανίζεται στον Πίνακα Περιεχομένων.

## Βήμα 6: Αποθήκευση του εγγράφου

Μόλις όλα ρυθμιστούν, αποθηκεύστε το έγγραφο PDF στο καθορισμένο αρχείο εξόδου.

```csharp
doc.Save(outFile);
```

Και αυτό είναι όλο! Δημιουργήσατε με επιτυχία ένα PDF με Πίνακα Περιεχομένων και οι αριθμοί σελίδων είναι κρυφοί!

Ακολουθώντας αυτόν τον οδηγό βήμα προς βήμα, μάθατε πώς να προσαρμόζετε τη μορφή Πίνακα Περιεχομένων, να αποκρύπτετε τους αριθμούς σελίδων και να εφαρμόζετε διαφορετικά στυλ στις επικεφαλίδες σας. ## Συχνές ερωτήσεις

### Μπορώ να εμφανίσω αριθμούς σελίδων για συγκεκριμένες επικεφαλίδες στον Πίνακα Περιεχομένων;
Όχι, το Aspose.PDF αποκρύπτει ή εμφανίζει τους αριθμούς σελίδων για ολόκληρο τον Πίνακα Περιεχομένων. Δεν μπορείτε να τους αποκρύψετε επιλεκτικά για συγκεκριμένες καταχωρήσεις.

### Είναι δυνατόν να προσθέσω περισσότερα επίπεδα στον Πίνακα Περιεχομένων;
Ναι, μπορείτε να αυξήσετε την `FormatArrayLength` για να ορίσετε περισσότερα επίπεδα επικεφαλίδων TOC.

### Πώς μπορώ να αλλάξω τη γραμματοσειρά για όλες τις καταχωρήσεις TOC;
Μπορείτε να αλλάξετε τη γραμματοσειρά τροποποιώντας το `TextState.Font` ιδιότητα για κάθε επίπεδο στο `FormatArray`.

### Μπορώ να εισάγω υπερσυνδέσμους στον Πίνακα Περιεχομένων;
Ναι, μπορείτε να συνδέσετε κάθε καταχώρηση Πίνακα Περιεχομένων με μια συγκεκριμένη ενότητα στο έγγραφο χρησιμοποιώντας το `Heading.TocPage` ιδιοκτησία.

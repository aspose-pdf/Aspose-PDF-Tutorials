## Οδηγός Εφαρμογής

### Μετατροπή PDF σε HTML με προσαρμοσμένες διευθύνσεις URL εικόνων

Η μετατροπή ενός εγγράφου PDF σε HTML, ενώ παράλληλα ελέγχονται οι διευθύνσεις URL εικόνων, απαιτεί την κατανόηση των δυνατοτήτων του Aspose.PDF και τη διαμόρφωση των κατάλληλων επιλογών. Θα αναλύσουμε τα πάντα βήμα προς βήμα.

#### Βήμα 1: Φόρτωση του εγγράφου
Αρχικά, φορτώστε το έγγραφο PDF χρησιμοποιώντας το `Document` τάξη:

```csharp
// Φόρτωση ενός υπάρχοντος εγγράφου PDF
document pdfDocument = new Document("input.pdf");
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης
Στήνω `HtmlSaveOptions` για να καθορίσετε τον τρόπο χειρισμού των εικόνων κατά τη μετατροπή. Το κλειδί εδώ είναι η ρύθμιση του `RasterImagesSavingMode` και ορισμός μιας προσαρμοσμένης στρατηγικής εξοικονόμησης πόρων:

```csharp
// Ρύθμιση επιλογών αποθήκευσης HTML
HtmlSaveOptions options = new HtmlSaveOptions();

// Ορισμός λειτουργίας αποθήκευσης εικόνας
options.RasterImagesSavingMode = HtmlSaveOptions.RasterImagesSavingModes.AsExternalPngFilesReferencedViaSvg;

// Ορίστε προσαρμοσμένη στρατηγική εξοικονόμησης πόρων
customResourceSavingStrategy: new HtmlSaveOptions.ResourceSavingStrategy(Custom_processor_of_embedded_images);
```

#### Βήμα 3: Εφαρμογή προσαρμοσμένης στρατηγικής εξοικονόμησης πόρων
Εδώ μπορείτε να προσαρμόσετε τον τρόπο αποθήκευσης και αναφοράς των εικόνων:

```csharp
private static string Custom_processor_of_embedded_images(SaveOptions.ResourceSavingInfo resourceSavingInfo)
{
    if (!(resourceSavingInfo is HtmlSaveOptions.HtmlImageSavingInfo))
    {
        resourceSavingInfo.CustomProcessingCancelled = true;
        return "";
    }

    // Ορισμός καταλόγου εξόδου για εικόνες
    string dataDir = "your/data/directory/path";
    string outDir = Path.Combine(dataDir, @"output\files");
    string outPath = Path.Combine(outDir, Path.GetFileName(resourceSavingInfo.SupposedFileName));

    if (!Directory.Exists(outDir))
    {
        Directory.CreateDirectory(outDir);
    }

    // Αποθήκευση της εικόνας
    using (BinaryReader reader = new BinaryReader(resourceSavingInfo.ContentStream))
    {
        System.IO.File.WriteAllBytes(outPath, reader.ReadBytes((int)resourceSavingInfo.ContentStream.Length));
    }

    // Επιστρέψτε μια προσαρμοσμένη διεύθυνση URL για αναφορά στην εικόνα σε HTML/SVG
    HtmlSaveOptions.HtmlImageSavingInfo imgInfo = (HtmlSaveOptions.HtmlImageSavingInfo)resourceSavingInfo;
    
    if (imgInfo.ParentType == HtmlSaveOptions.ImageParentTypes.SvgImage)
    {
        return "http://localhost:255/" + resourceSavingInfo.SupposedFileName;
    }
    else
    {
        return "http://localhost:GetImage/imageID=" + resourceSavingInfo.SupposedFileName;
    }
}
```

Αυτή η συνάρτηση καθορίζει τον τρόπο αποθήκευσης και αναφοράς των εικόνων, επιτρέποντάς σας να καθορίσετε προσαρμοσμένες διευθύνσεις URL.

#### Βήμα 4: Εκτέλεση μετατροπής
Τέλος, αποθηκεύστε το έγγραφο σε μορφή HTML χρησιμοποιώντας τις διαμορφωμένες επιλογές σας:

```csharp

```

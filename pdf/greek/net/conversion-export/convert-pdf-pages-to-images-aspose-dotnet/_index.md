### Μετατροπή σελίδας PDF σε εικόνα

#### Επισκόπηση

Αυτή η λειτουργία μετατρέπει μια συγκεκριμένη σελίδα από ένα έγγραφο PDF σε εικόνα, επιτρέποντας την προσαρμογή της απόδοσης κειμένου μέσω των ρυθμίσεων γραμματοσειράς.

#### Βήμα προς βήμα εφαρμογή

**1. Φορτώστε το έγγραφο PDF**

```csharp
// Φορτώστε το αρχείο PDF χρησιμοποιώντας το Aspose.PDF
using (Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf"))
{
    // Συνεχίστε με τα βήματα μετατροπής εντός αυτού του μπλοκ
}
```

*Εξήγηση:* Αυτός ο κώδικας αρχικοποιεί ένα `Document` αντικείμενο, το οποίο είναι απαραίτητο για την πρόσβαση στις σελίδες του PDF.

**2. Διαμόρφωση ρυθμίσεων εξόδου**

```csharp
// Ρύθμιση ροής εξόδου και ανάλυσης
using (FileStream imageStream = new FileStream("YOUR_OUTPUT_DIRECTORY/SetDefaultFontName.png", FileMode.Create))
{
    Resolution resolution = new Resolution(300);
    PngDevice pngDevice = new PngDevice(resolution);

    // Επιλογές απόδοσης για ρυθμίσεις γραμματοσειράς
    RenderingOptions ro = new RenderingOptions();
    ro.DefaultFontName = "Arial";  // Ορίστε την επιθυμητή προεπιλεγμένη γραμματοσειρά

    // Εφαρμογή επιλογών απόδοσης στη συσκευή
    pngDevice.RenderingOptions = ro;
}
```

*Εξήγηση:* Εδώ, διαμορφώνουμε ένα `FileStream` και ρυθμίστε την ανάλυση. Το `RenderingOptions` Η κλάση μας επιτρέπει να καθορίσουμε μια προεπιλεγμένη γραμματοσειρά για τα στοιχεία κειμένου κατά τη μετατροπή.

**3. Εκτελέστε μετατροπή**

```csharp
// Μετατρέψτε την πρώτη σελίδα ενός PDF σε εικόνα
pngDevice.Process(pdfDocument.Pages[1], imageStream);
```

*Εξήγηση:* Τέλος, μετατρέπουμε και αποθηκεύουμε την καθορισμένη σελίδα ως εικόνα χρησιμοποιώντας `PngDevice`.

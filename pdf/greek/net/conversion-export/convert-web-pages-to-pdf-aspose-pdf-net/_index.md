## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την βήμα προς βήμα υλοποίηση, χωρισμένη σε δύο κύρια χαρακτηριστικά: ανάκτηση περιεχομένου ιστοσελίδας και μετατροπή του σε PDF.

### Ανάκτηση περιεχομένου ιστοσελίδας
#### Επισκόπηση
Θα ξεκινήσουμε χρησιμοποιώντας αιτήματα HTTP σε C# για να ανακτήσουμε περιεχόμενο από μια ιστοσελίδα. Αυτό μας επιτρέπει να ανακτούμε δυναμικά δεδομένα HTML για μετατροπή.

#### Βήματα Υλοποίησης
**Βήμα 1:** Ορίστε τη διεύθυνση URL και δημιουργήστε ένα αίτημα HTTP.
```csharp
using System.IO;
using System.Net;

string url = "https://el.wikipedia.org/wiki/Main_Page";
WebRequest request = WebRequest.Create(url);
request.Credentials = CredentialCache.DefaultCredentials; // Χρήση προεπιλεγμένων διαπιστευτηρίων για έλεγχο ταυτότητας
```
*Γιατί να χρησιμοποιήσετε `CredentialCache.DefaultCredentials`Επιτρέπει στην εφαρμογή να πραγματοποιήσει έλεγχο ταυτότητας χρησιμοποιώντας τα διαπιστευτήρια του τρέχοντος χρήστη.*

**Βήμα 2:** Στείλτε το αίτημα και λάβετε απάντηση.
```csharp
HttpWebResponse response = (HttpWebResponse)request.GetResponse();
Stream dataStream = response.GetResponseStream(); // Λήψη της ροής δεδομένων από την απάντηση
```

**Βήμα 3:** Διαβάστε και αποθηκεύστε το περιεχόμενο.
```csharp
StreamReader reader = new StreamReader(dataStream);
string responseFromServer = reader.ReadToEnd(); // Μετατροπή της ροής σε συμβολοσειρά

// Καθαρίστε τους πόρους
reader.Close();
dataStream.Close();
response.Close();
```
*Το κλείσιμο των ροών είναι ζωτικής σημασίας για την απελευθέρωση πόρων συστήματος.*

### Μετατροπή περιεχομένου ιστοσελίδας σε PDF
#### Επισκόπηση
Στη συνέχεια, θα μετατρέψουμε το περιεχόμενο HTML που ανακτήθηκε σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET.

#### Βήματα Υλοποίησης
**Βήμα 1:** Προετοιμάστε τα δεδομένα HTML για μετατροπή.
```csharp
using System.Text;

MemoryStream stream = new MemoryStream(Encoding.UTF8.GetBytes(responseFromServer));
```
*Η κωδικοποίηση σε UTF-8 διασφαλίζει τον σωστό χειρισμό των χαρακτήρων κειμένου.*

**Βήμα 2:** Ρυθμίστε τις παραμέτρους των επιλογών PDF και δημιουργήστε ένα έγγραφο.
```csharp
HtmlLoadOptions options = new HtmlLoadOptions(url);
Document pdfDocument = new Document(stream, options);

options.PageInfo.IsLandscape = true; // Ορίστε τον προσανατολισμό της σελίδας σε οριζόντιο, εάν χρειάζεται
```

**Βήμα 3:** Αποθηκεύστε το έγγραφο ως αρχείο PDF.
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/WebPageToPDF_out.pdf");
```
*Αντικαθιστώ `YOUR_OUTPUT_DIRECTORY` με την επιθυμητή διαδρομή εξόδου.*

## Ενότητα Συχνών Ερωτήσεων
**Ε1: Πώς μπορώ να χειριστώ τον έλεγχο ταυτότητας κατά την ανάκτηση ιστοσελίδων;**
A1: Χρήση `CredentialCache.DefaultCredentials` ή καθορίστε προσαρμοσμένα διαπιστευτήρια στο αίτημά σας HTTP για πρόσβαση με έλεγχο ταυτότητας.

**Ε2: Μπορώ να μετατρέψω πολλές ιστοσελίδες σε ένα μόνο έγγραφο PDF;**
A2: Ναι, μπορείτε να προσθέσετε πολλά περιεχόμενα HTML σε μία ροή πριν δημιουργήσετε το PDF.

**Ε3: Υπάρχουν περιορισμοί στη μετατροπή μεγάλων ιστοσελίδων;**
A3: Ενώ το Aspose.PDF χειρίζεται καλά μεγάλα έγγραφα, βεβαιωθείτε ότι έχετε επαρκή κατανομή μνήμης και σκεφτείτε να διαχωρίσετε υπερβολικά μεγάλο περιεχόμενο.

**Ε4: Πώς μπορώ να αλλάξω τη διάταξη σελίδας στο PDF που δημιουργείται;**
A4: Τροποποίηση `HtmlLoadOptions.PageInfo` ιδιότητες για να προσαρμόσετε τα περιθώρια, τον προσανατολισμό ή το μέγεθος.

**Ε5: Τι γίνεται αν η ιστοσελίδα μου έχει διαδραστικά στοιχεία όπως φόρμες;**
A5: Το Aspose.PDF εστιάζει σε στατικό περιεχόμενο. Τα διαδραστικά στοιχεία συνήθως δεν διατηρούνται κατά τη μετατροπή.

## Οδηγός Εφαρμογής
Θα εξερευνήσουμε κάθε λειτουργία με οδηγίες βήμα προς βήμα, αποσπάσματα κώδικα και λεπτομερείς εξηγήσεις.

### Μετατροπή PDF σε Excel (XLS)
#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει τον τρόπο μετατροπής ενός εγγράφου PDF σε αρχείο XLS χρησιμοποιώντας το Aspose.PDF για .NET.

**Βήμα 1: Φόρτωση του εγγράφου PDF**
```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
document pdfDocument = new Document(documentPath);
```

**Βήμα 2: Δημιουργία στιγμιότυπου ExcelSaveOptions**
```csharp
ExcelSaveOptions excelsave = new ExcelSaveOptions();
```

**Βήμα 3: Αποθηκεύστε το PDF ως XLS**
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/PDFToXLS_out.xls", excelsave);
```

### Μετατροπή PDF σε Excel (XLSX)
#### Επισκόπηση
Μάθετε πώς να μετατρέψετε ένα έγγραφο PDF σε αρχείο XLSX, προσφέροντας καλύτερη συμβατότητα με νεότερες εκδόσεις του Excel.

**Βήμα 1: Αρχικοποίηση του ExcelSaveOptions για μορφή XLSX**
```csharp
ExcelSaveOptions excelsave = new ExcelSaveOptions();
excelsave.Format = ExcelSaveOptions.ExcelFormat.XLSX;
```

**Βήμα 2: Αποθηκεύστε το PDF ως XLSX**
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/PDFToXLSX_out.xlsx", excelsave);
```

### Εισαγωγή στήλης ελέγχου σε μετατροπή
#### Επισκόπηση
Ελέγξτε εάν θα εισάγεται μια κενή στήλη στην αρχή του αρχείου Excel κατά τη μετατροπή.

**Βήμα 1: Ρύθμιση παραμέτρων του ExcelSaveOptions**
```csharp
excelsave.InsertBlankColumnAtFirst = false;
```

**Βήμα 2: Αποθήκευση με ελεγχόμενες στήλες**
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/PDFToXLSControlColumn_out.xls", excelsave);
```

### Εξαγωγή όλων των σελίδων PDF σε ένα μόνο φύλλο Excel
#### Επισκόπηση
Εξαγάγετε όλες τις σελίδες από ένα PDF πολλαπλών σελίδων σε ένα μόνο φύλλο στο Excel, απλοποιώντας την ανάλυση δεδομένων.

**Βήμα 1: Ορισμός επιλογής MinimizeWorksheets**
```csharp
excelsave.MinimizeTheNumberOfWorksheets = true;
```

**Βήμα 2: Αποθήκευση με όλες τις σελίδες σε ένα φύλλο**
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/PDFToXLSExportAllPagesToSingle_out.xls", excelsave);
```

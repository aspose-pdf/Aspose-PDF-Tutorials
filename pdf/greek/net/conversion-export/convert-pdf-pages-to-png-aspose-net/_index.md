## Οδηγός Εφαρμογής

### Μετατροπή σελίδων PDF σε PNG

#### Επισκόπηση
Η μετατροπή κάθε σελίδας ενός εγγράφου PDF σε μεμονωμένα αρχεία PNG υψηλής ανάλυσης μπορεί να είναι χρήσιμη για εφαρμογές όπως η δημοσίευση στο διαδίκτυο και οι λύσεις ψηφιακής αποθήκευσης.

##### Βήμα 1: Ορισμός καταλόγων
Αρχικά, ορίστε τις διαδρομές για το PDF προέλευσης και τον κατάλογο εξόδου:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

Βεβαιωθείτε ότι αυτοί οι κατάλογοι υπάρχουν ή ενημερώστε τους σύμφωνα με τη δομή του έργου σας.

##### Βήμα 2: Φόρτωση του εγγράφου
Φορτώστε το έγγραφο PDF χρησιμοποιώντας το Aspose.PDF `Document` τάξη:

```csharp
document pdfDocument = new Document(dataDir + "/ConvertAllPagesToPNG.pdf");
```

Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής αποκτώντας πρόσβαση στο επιθυμητό αρχείο PDF.

##### Βήμα 3: Μετατροπή κάθε σελίδας
Επαναλάβετε κάθε σελίδα, μετατρέποντάς την σε αρχεία PNG:

```csharp
for (int pageCount = 1; pageCount <= pdfDocument.Pages.Count; pageCount++)
{
    string imagePath = Path.Combine(outputDir, "image" + pageCount + "_out.png");
    
    using (FileStream imageStream = new FileStream(imagePath, FileMode.Create))
    {
        Resolution resolution = new Resolution(300); // Ορίστε σε υψηλό DPI για ποιότητα
        PngDevice pngDevice = new PngDevice(resolution);
        
        pngDevice.Process(pdfDocument.Pages[pageCount], imageStream);
    }
}
```
- **Ψήφισμα**Η ρύθμιση 300 DPI εξασφαλίζει ευκρινείς και καθαρές εικόνες.
- **Συσκευή Png**: Χειρίζεται τη μετατροπή από PDF σε PNG με καθορισμένες ρυθμίσεις ανάλυσης.

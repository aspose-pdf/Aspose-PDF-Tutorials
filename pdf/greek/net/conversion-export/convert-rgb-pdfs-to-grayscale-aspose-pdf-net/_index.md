### Μετατροπή κάθε σελίδας από RGB σε κλίμακα του γκρι

#### Επισκόπηση
Η μετατροπή έγχρωμων σελίδων σε κλίμακα του γκρι μπορεί να βοηθήσει στην τυποποίηση της εμφάνισης του εγγράφου και στη μείωση του μεγέθους του αρχείου. Δείτε πώς μπορείτε να εφαρμόσετε αυτήν τη λειτουργία:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
using Aspose.Pdf.Text;

// Φόρτωση του αρχείου PDF προέλευσης
using (Document document = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf"))
{
    // Δημιουργήστε μια παρουσία του RgbToDeviceGrayConversionStrategy
    Aspose.Pdf.RgbToDeviceGrayConversionStrategy strategy = new Aspose.Pdf.RgbToDeviceGrayConversionStrategy();
    
    // Επαναλάβετε κάθε σελίδα στο έγγραφο PDF
    for (int idxPage = 1; idxPage <= document.Pages.Count; idxPage++)
    {
        // Πρόσβαση στη συγκεκριμένη σελίδα
        Page page = document.Pages[idxPage];
        
        // Μετατροπή χρωματικού χώρου RGB σε κλίμακα του γκρι
        strategy.Convert(page);
    }
    
    // Αποθηκεύστε το τροποποιημένο αρχείο στην επιθυμητή θέση
    document.Save("YOUR_OUTPUT_DIRECTORY/Test-gray_out.pdf");
}
```

**Εξήγηση:**
- **`RgbToDeviceGrayConversionStrategy`:** Αυτή η κλάση διευκολύνει τη διαδικασία μετατροπής, χειριζόμενη τον μετασχηματισμό χρωμάτων κάθε σελίδας.
- **Επανάληψη σελίδων:** Κάθε σελίδα υποβάλλεται σε ξεχωριστή επεξεργασία για να διασφαλιστεί ο ακριβής έλεγχος της μετατροπής.

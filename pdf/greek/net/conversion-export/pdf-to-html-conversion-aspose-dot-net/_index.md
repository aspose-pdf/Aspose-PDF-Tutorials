## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Μετατροπή PDF σε HTML με ενσωμάτωση πόρων

Αυτή η λειτουργία εστιάζει στη μετατροπή ενός εγγράφου PDF σε μορφή HTML, ενσωματώνοντας παράλληλα όλους τους πόρους στο αρχείο HTML εξόδου.

#### Επισκόπηση:
Η μετατροπή διασφαλίζει ότι όλες οι γραμματοσειρές, οι εικόνες και τα στυλ ενσωματώνονται μέσα στην HTML, παρέχοντας μια αυτοτελή και οπτικά ομοιόμορφη εμπειρία σε διάφορα προγράμματα περιήγησης.

##### Βήμα 1: Φόρτωση του PDF πηγής

```csharp
using System;
using Aspose.Pdf;

string documentPath = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document doc = new Document(documentPath);
```
Εδώ, φορτώνουμε το αρχείο PDF προέλευσης. Βεβαιωθείτε ότι η διαδρομή του αρχείου σας έχει οριστεί σωστά για να αποφύγετε σφάλματα φόρτωσης.

##### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML

```csharp
using Aspose.Pdf.Saving;

HtmlSaveOptions options = new HtmlSaveOptions();
options.PartsEmbeddingMode = PartsEmbeddingModes.EmbedAllIntoHtml;
```
Δημιουργούμε ένα υπόδειγμα `HtmlSaveOptions` και ρυθμίστε το ώστε να ενσωματώνει όλους τους πόρους στο αρχείο HTML.

##### Βήμα 3: Βελτιστοποίηση για τον Internet Explorer

```csharp
options.LettersPositioningMethod = LettersPositioningMethods.UseEmUnitsAndCompensationOfRoundingErrorsInCss;
```
Αυτή η ρύθμιση διασφαλίζει ότι το κείμενο εμφανίζεται με ακρίβεια στον IE χρησιμοποιώντας μονάδες em και αντισταθμίζοντας τα σφάλματα στρογγυλοποίησης CSS.

##### Βήμα 4: Αποθήκευση εικόνων ράστερ ως ενσωματωμένα PNG

```csharp
options.RasterImagesSavingMode = RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground;
```
Αποδίδει τις εικόνες ως ενσωματωμένα μέρη του φόντου της σελίδας, διασφαλίζοντας οπτική πιστότητα σε όλες τις πλατφόρμες.

##### Βήμα 5: Ρύθμιση παραμέτρων λειτουργίας αποθήκευσης γραμματοσειράς

```csharp
options.FontSavingMode = FontSavingModes.SaveInAllFormats;
```
Αυτή η επιλογή αποθηκεύει γραμματοσειρές σε όλες τις μορφές για μέγιστη συμβατότητα και διατήρηση της ποιότητας.

##### Βήμα 6: Αποθήκευση του αρχείου HTML που έχει μετατραπεί

```csharp
string outputHtmlPath = "YOUR_OUTPUT_DIRECTORY/SingleHTML_out.html";
doc.Save(outputHtmlPath, options);
```
Τέλος, αποθηκεύστε το αρχείο HTML που έχει μετατραπεί στον καθορισμένο κατάλογο. Βεβαιωθείτε ότι η διαδρομή εξόδου είναι ακριβής για να αποτρέψετε σφάλματα.

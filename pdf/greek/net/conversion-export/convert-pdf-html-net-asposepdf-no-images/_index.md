## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε τη διαδικασία υλοποίησης σε διαχειρίσιμα βήματα.

### Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML

**Επισκόπηση**
Θα διαμορφώσουμε επιλογές για τη μετατροπή PDF σε HTML, διατηρώντας παράλληλα τις εικόνες ενσωματωμένες στο φόντο της σελίδας. Αυτή η προσέγγιση διασφαλίζει ότι τα έγγραφα που έχετε μετατρέψει διατηρούν την αρχική τους διάταξη χωρίς να εξάγονται εικόνες ξεχωριστά.

#### Βήμα 1: Αρχικοποίηση εγγράφου και επιλογών αποθήκευσης
Ρυθμίστε την παρουσία του εγγράφου και καθορίστε επιλογές αποθήκευσης για μετατροπή:

```csharp
// Διαδρομή προς το αρχείο PDF εισαγωγής σας.
string dataDir = "YourDirectoryPath";

Document pdfDocument = new Document(dataDir + "/input.pdf");
HtmlSaveOptions saveOptions = new HtmlSaveOptions();
```

#### Βήμα 2: Ορισμός διάταξης και λειτουργίας αποθήκευσης εικόνας

Διαμορφώστε το `HtmlSaveOptions` για να διατηρήσετε μια σταθερή διάταξη και να ενσωματώσετε εικόνες:

```csharp
saveOptions.FixedLayout = true;
// Οι εικόνες αποθηκεύονται ως ενσωματωμένα PNG μέσα στο φόντο της σελίδας.
saveOptions.RasterImagesSavingMode = HtmlSaveOptions.RasterImagesSavingModes.AsEmbeddedPartsOfPngPageBackground;
```

#### Βήμα 3: Αποθήκευση εγγράφου
Τέλος, αποθηκεύστε το έγγραφο PDF σε μορφή HTML χρησιμοποιώντας αυτές τις ρυθμίσεις:

```csharp
pdfDocument.Save(dataDir + "/AvoidSavingImages_out.html", saveOptions);
```

**Συμβουλή αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι η διαδρομή του αρχείου εισόδου είναι σωστή και προσβάσιμη για να αποφύγετε εξαιρέσεις κατά τη φόρτωση.

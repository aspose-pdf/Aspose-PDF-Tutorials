## Οδηγός Εφαρμογής

### Μετατροπή PDF σε EPUB

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στη μετατροπή ενός αρχείου PDF σε μορφή EPUB χρησιμοποιώντας συγκεκριμένες διαμορφώσεις που παρέχονται από το Aspose.PDF.

##### Βήμα 1: Φόρτωση του εγγράφου PDF εισόδου
Ξεκινήστε φορτώνοντας το έγγραφο PDF:
```csharp
// Δημιουργήστε μια παρουσία της κλάσης Document
document = new Document("YOUR_DOCUMENT_DIRECTORY/PDFToEPUB.pdf");
```
**Εξήγηση**: Αυτό αρχικοποιεί ένα `Document` αντικείμενο, που αντιπροσωπεύει το PDF που θέλετε να μετατρέψετε.

##### Βήμα 2: Διαμόρφωση επιλογών μετατροπής
Ρυθμίστε τις προτιμήσεις μετατροπής χρησιμοποιώντας `EpubSaveOptions`:
```csharp
// Δημιουργήστε μια παρουσία του EpubSaveOptions
options = new EpubSaveOptions();

// Ορισμός λειτουργίας αναγνώρισης περιεχομένου σε «Ροή»
options.ContentRecognitionMode = EpubSaveOptions.RecognitionMode.Flow;
```
**Εξήγηση**: Το `ContentRecognitionMode.Flow` διασφαλίζει ότι το περιεχόμενο προσαρμόζεται καλά σε διάφορα μεγέθη οθόνης.

##### Βήμα 3: Αποθήκευση του εγγράφου EPUB εξόδου
Τέλος, αποθηκεύστε το έγγραφο που μετατρέψατε:
```csharp
document.Save("YOUR_OUTPUT_DIRECTORY/PDFToEPUB_out.epub", options);
```
**Εξήγηση**Αυτό το βήμα εγγράφει το αρχείο EPUB σε έναν καθορισμένο κατάλογο με όλες τις καθορισμένες ρυθμίσεις εφαρμοσμένες.

## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε βασικά χαρακτηριστικά:

### Χαρακτηριστικό 1: Ορισμός διαστάσεων αρχείου εξόδου

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσαρμόσετε τις διαστάσεις των σελίδων PDF πριν τις μετατρέψετε σε HTML. Διασφαλίζει ότι το περιεχόμενο ταιριάζει απόλυτα στα νέα μεγέθη σελίδων υπολογίζοντας ένα κατάλληλο επίπεδο ζουμ.

#### Βήμα προς βήμα εφαρμογή

**Αρχικοποίηση και σύνδεση εγγράφου PDF**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Facades;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εγγράφων σας
PdfPageEditor pdfEditor = new PdfPageEditor();
pdfEditor.BindPdf(dataDir + "/input.pdf"); // Δέσμευση του PDF εισόδου
```

**Ορισμός νέων διαστάσεων σελίδας**
```csharp
// Επιλέξτε το επιθυμητό μέγεθος σελίδας
float newPageWidth = 400f;
float newPageHeight = 400f;

// Ορισμός νέων διαστάσεων και στοίχισης
pdfEditor.PageSize = new PageSize(newPageWidth, newPageHeight);
pdfEditor.VerticalAlignmentType = VerticalAlignment.Center;
pdfEditor.HorizontalAlignment = HorizontalAlignment.Center;
```

**Υπολογισμός συντελεστή ζουμ**
```csharp
// Υπολογισμός ζουμ για αναλογική προσαρμογή περιεχομένου στο νέο μέγεθος σελίδας
float zoom = Math.Min((float)newPageWidth / (float)pdfEditor.Document.Pages[1].Rect.Width,
                      (float)newPageHeight / (float)pdfEditor.Document.Pages[1].Rect.Height);
pdfEditor.Zoom = zoom; // Εφαρμογή υπολογισμένου ζουμ
```

**Αποθήκευση για ροή και μετατροπή**
```csharp
// Αποθηκεύστε το ενημερωμένο PDF με τις νέες διαστάσεις σε μια ροή μνήμης
MemoryStream output = new MemoryStream();
pdfEditor.Save(output);

// Επαναφόρτωση του κλιμακωμένου εγγράφου για μετατροπή HTML
Document exportDoc = new Document(output);
HtmlSaveOptions htmlOptions = new HtmlSaveOptions();

// Ρύθμιση παραμέτρων περιγραμμάτων σελίδας στο αρχείο HTML που προκύπτει
SaveOptions.BorderPartStyle borderStyle = new SaveOptions.BorderPartStyle()
{
    LineType = SaveOptions.HtmlBorderLineType.Dotted,
    Color = System.Drawing.Color.Gray
};
htmlOptions.PageBorderIfAny = new SaveOptions.BorderInfo(borderStyle);

// Μετατροπή και αποθήκευση του PDF σε μορφή HTML
exportDoc.Save("YOUR_OUTPUT_DIRECTORY/SetOutputFileDimensions_out.html", htmlOptions);
output.Close(); // Κλείστε τη ροή μνήμης
```

### Χαρακτηριστικό 2: Διαμόρφωση μετατροπής

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στη διαμόρφωση της διαδικασίας μετατροπής από PDF σε HTML, συμπεριλαμβανομένης της ρύθμισης περιγραμμάτων σελίδας για καλύτερη ορατότητα.

**Ρύθμιση παραμέτρων και μετατροπή**
```csharp
// Αρχικοποίηση HtmlSaveOptions για διαμόρφωση
HtmlSaveOptions htmlOptions = new HtmlSaveOptions();

// Ρύθμιση ορατών περιγραμμάτων σελίδας στο αρχείο HTML που προκύπτει
SaveOptions.BorderPartStyle borderStyle = new SaveOptions.BorderPartStyle()
{
    LineType = SaveOptions.HtmlBorderLineType.Dotted,
    Color = System.Drawing.Color.Gray
};
htmlOptions.PageBorderIfAny = new SaveOptions.BorderInfo(borderStyle);

// Ας υποθέσουμε ότι δημιουργείται ένα αντικείμενο εγγράφου (π.χ., από PDF)
Document exportDoc = new Document(dataDir + "/input.pdf");

// Μετατροπή και αποθήκευση του εγγράφου ως HTML με διαμορφωμένες επιλογές
exportDoc.Save("YOUR_OUTPUT_DIRECTORY/ConvertedWithBorders_out.html", htmlOptions);
```

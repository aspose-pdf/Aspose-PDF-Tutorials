## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία μετατροπής σε βήματα.

### Φόρτωση του εγγράφου PDF

**Επισκόπηση**Το πρώτο βήμα είναι να φορτώσετε το έγγραφο PDF που θέλετε να μετατρέψετε. Αυτό περιλαμβάνει τη δημιουργία μιας παρουσίας του `Document` τάξη.

```csharp
// Φόρτωση του εγγράφου PDF από έναν καθορισμένο κατάλογο
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/input.pdf");
```

Αυτό το απόσπασμα κώδικα αρχικοποιεί ένα `Document` αντικείμενο, που αντιπροσωπεύει το αρχείο PDF σας για χειρισμό και μετατροπή.

### Ρύθμιση παραμέτρων επιλογών αποθήκευσης SVG

**Επισκόπηση**Στη συνέχεια, διαμορφώστε τον τρόπο αποθήκευσης του PDF ως SVG. Αυτό περιλαμβάνει τον ορισμό διαφόρων επιλογών, όπως ρυθμίσεις συμπίεσης.

```csharp
// Δημιουργήστε ένα αντικείμενο του SvgSaveOptions με συγκεκριμένη διαμόρφωση
SvgSaveOptions saveOptions = new SvgSaveOptions();

// Ορίστε την τιμή σε false για να διασφαλίσετε ότι κάθε σελίδα αποθηκεύεται ως ξεχωριστό αρχείο .svg
saveOptions.CompressOutputToZipArchive = false;
```

**Εξήγηση**: 
- `CompressOutputToZipArchive` έχει οριστεί σε `false` έτσι ώστε κάθε σελίδα PDF να αποθηκεύεται ως ξεχωριστή `.svg` αρχείο.

### Αποθήκευση του εγγράφου ως SVG

**Επισκόπηση**Τέλος, αποθηκεύστε το έγγραφό σας σε μορφή SVG χρησιμοποιώντας τις καθορισμένες επιλογές.

```csharp
// Αποθηκεύστε το έγγραφο ως αρχείο SVG στον καθορισμένο κατάλογο
doc.Save("YOUR_OUTPUT_DIRECTORY/PDFToSVG_out.svg", saveOptions);
```

Αυτό το βήμα εγγράφει τα αρχεία SVG που έχουν μετατραπεί στον επιθυμητό κατάλογο εξόδου. Κάθε σελίδα PDF αποθηκεύεται ως ξεχωριστό αρχείο SVG, εάν έχει ρυθμιστεί ανάλογα.

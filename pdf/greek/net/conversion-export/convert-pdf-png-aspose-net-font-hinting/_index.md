## Οδηγός Εφαρμογής

### Επισκόπηση

Θα μετατρέψουμε σελίδες PDF σε εικόνες PNG χρησιμοποιώντας το Aspose.PDF .NET, αξιοποιώντας τις υποδείξεις γραμματοσειράς για βελτιωμένη σαφήνεια κειμένου. Αυτή η διαδικασία περιλαμβάνει τη φόρτωση ενός εγγράφου PDF, τον ορισμό επιλογών απόδοσης και την αποθήκευση κάθε σελίδας ως αρχείο PNG.

#### Βήμα 1: Φόρτωση του εγγράφου PDF

Ανοίξτε το PDF προορισμού σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/input.pdf");
```
Αυτός ο κώδικας φορτώνει ένα PDF από τον καθορισμένο κατάλογο. Βεβαιωθείτε ότι έχετε αντικαταστήσει `"YOUR_DOCUMENT_DIRECTORY"` με τη διαδρομή προς το έγγραφό σας.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών απόδοσης

Ρυθμίστε τις επιλογές απόδοσης για να ενεργοποιήσετε την υπόδειξη γραμματοσειράς, βελτιώνοντας την ευκρίνεια του κειμένου στις εικόνες που προκύπτουν:
```csharp
RenderingOptions opts = new RenderingOptions();
opts.UseFontHinting = true;
```
Η υποδείξεις γραμματοσειράς διασφαλίζει ότι οι γραμματοσειρές αποδίδονται με ευκρίνεια, προσαρμόζοντας τα σχήματα των χαρακτήρων ώστε να ταιριάζουν καλύτερα στα pixel.

#### Βήμα 3: Μετατροπή κάθε σελίδας

Επαναλάβετε κάθε σελίδα του PDF και μετατρέψτε την σε εικόνα PNG:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";

for (int pageCount = 1; pageCount <= pdfDocument.Pages.Count; pageCount++)
{
    using (FileStream imageStream = new FileStream(outputDir + "/image" + pageCount + "_out.png", FileMode.Create))
    {
        Resolution resolution = new Resolution(300);
        PngDevice pngDevice = new PngDevice(resolution);
        pngDevice.RenderingOptions = opts;
        
        pngDevice.Process(pdfDocument.Pages[pageCount], imageStream);
    }
}
```
Αυτό το απόσπασμα ορίζει μια έξοδο υψηλής ανάλυσης (300 DPI) και επεξεργάζεται κάθε σελίδα ξεχωριστά, αποθηκεύοντάς την ως αρχείο PNG. Προσαρμόστε την ανάλυση με βάση τις απαιτήσεις ποιότητας που έχετε.

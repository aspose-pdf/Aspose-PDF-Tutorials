## Οδηγός Εφαρμογής

Θα εξετάσουμε διάφορες λειτουργίες του Aspose.PDF .NET για τη μετατροπή περιεχομένου HTML σε PDF, εστιάζοντας στην προσαρμοσμένη φόρτωση πόρων, την απόδοση μίας σελίδας και τον χειρισμό δεδομένων SVG μέσα στο έγγραφο.

### Χαρακτηριστικό 1: Μετατροπή HTML σε PDF με προσαρμοσμένους πόρους

Αυτή η λειτουργία σάς επιτρέπει να προσαρμόσετε τον τρόπο φόρτωσης των εξωτερικών πόρων κατά τη μετατροπή. Είναι ιδιαίτερα χρήσιμη όταν πρόκειται για συγκεκριμένες εικόνες ή στοιχεία που χρειάζονται ειδικό χειρισμό.

#### Επισκόπηση
Ο στόχος εδώ είναι να φορτώσετε μια εικόνα ως πίνακα byte και να τη χρησιμοποιήσετε στο PDF που προκύπτει, παρακάμπτοντας τον προεπιλεγμένο μηχανισμό φόρτωσης πόρων του Aspose.PDF.

**Βήμα 1: Αρχικοποίηση HtmlLoadOptions**

Ξεκινάμε δημιουργώντας `HtmlLoadOptions` και ορισμός ενός προσαρμοσμένου φορτωτή για εξωτερικούς πόρους.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// Προσαρμοσμένη στρατηγική φόρτωσης πόρων
HtmlLoadOptions options = new HtmlLoadOptions();
options.CustomLoaderOfExternalResources = new LoadOptions.ResourceLoadingStrategy(SamePictureLoader);
```

**Βήμα 2: Δημιουργία εγγράφου PDF από HTML**

Με τις επιλογές που έχουμε ορίσει, φορτώνουμε το αρχείο HTML και το μετατρέπουμε σε έγγραφο PDF.

```csharp
Document pdfDocument = new Document(dataDir + "/HTMLToPDF.html", options);
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/HTMLToPDF_out.pdf");
```

**Βήμα 3: Ορισμός Προσαρμοσμένου Φορτωτή Πόρων**

Ο `SamePictureLoader` Η συνάρτηση καθορίζει τον τρόπο φόρτωσης των εξωτερικών πόρων. Εδώ, φορτώνουμε ένα συγκεκριμένο αρχείο εικόνας ως bytes.

```csharp
private static LoadOptions.ResourceLoadingResult SamePictureLoader(string resourceURI)
{
    byte[] resultBytes = File.ReadAllBytes(dataDir + "/aspose-logo.jpg");
    return new LoadOptions.ResourceLoadingResult(resultBytes);
}
```

### Δυνατότητα 2: Απόδοση περιεχομένου HTML σε PDF μίας σελίδας

Η μετατροπή περιεχομένου HTML πολλαπλών σελίδων σε μία μόνο σελίδα μπορεί να είναι απαραίτητη για τη δημιουργία συνοπτικών αναφορών ή περιλήψεων.

#### Επισκόπηση
Θα ενεργοποιήσουμε το `IsRenderToSinglePage` ιδιότητα στις επιλογές μας για να διασφαλίσουμε ότι ολόκληρο το έγγραφο χωράει σε μία σελίδα.

**Βήμα 1: Ρύθμιση HtmlLoadOptions**

Ενεργοποιήστε την απόδοση μίας σελίδας διαμορφώνοντας `HtmlLoadOptions`.

```csharp
HtmlLoadOptions options = new HtmlLoadOptions();
options.IsRenderToSinglePage = true;
```

**Βήμα 2: Φόρτωση και αποθήκευση HTML ως PDF μίας σελίδας**

Τώρα, φορτώνουμε το έγγραφο με αυτές τις επιλογές και το αποθηκεύουμε.

```csharp
Document doc = new Document(dataDir + "/HTMLToPDF.html", options);
doc.Save("YOUR_OUTPUT_DIRECTORY/RenderContentToSamePage.pdf");
```

### Δυνατότητα 3: Απόδοση HTML με δεδομένα SVG σε PDF

Η διαχείριση δεδομένων SVG μέσα σε HTML μπορεί να είναι δύσκολη, αλλά το Aspose.PDF την κάνει απρόσκοπτη.

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στην απόδοση αρχείων HTML που περιέχουν ενσωματωμένες εικόνες SVG σε ένα έγγραφο PDF.

**Βήμα 1: Καθορισμός διαδρομών εισόδου και εξόδου**

Ορίστε τις διαδρομές για το αρχείο HTML εισόδου και το PDF εξόδου.

```csharp
string inFile = dataDir + "/HTMLSVG.html";
string outFile = "YOUR_OUTPUT_DIRECTORY/RenderHTMLwithSVGData.pdf";
```

**Βήμα 2: Αρχικοποίηση επιλογών με διαδρομή καταλόγου**

Στήνω `HtmlLoadOptions` με τη διαδρομή καταλόγου του αρχείου HTML για να διασφαλίσετε ότι τα SVG έχουν επιλυθεί σωστά.

```csharp
HtmlLoadOptions options = new HtmlLoadOptions(Path.GetDirectoryName(inFile));
Document pdfDocument = new Document(inFile, options);
pdfDocument.Save(outFile);
```

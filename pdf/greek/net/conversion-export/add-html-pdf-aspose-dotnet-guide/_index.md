## Οδηγός Εφαρμογής

Έχοντας έτοιμο το περιβάλλον μας, ας εξερευνήσουμε πώς να εφαρμόσουμε την προσθήκη HTML χρησιμοποιώντας DOM και ακριβή τοποθέτηση μέσα σε σελίδες PDF.

### Χαρακτηριστικό 1: Προσθήκη HTML χρησιμοποιώντας DOM
Αυτή η λειτουργία σάς επιτρέπει να ενσωματώνετε πλούσιο περιεχόμενο HTML σε έγγραφα PDF.

#### Επισκόπηση
Προσθέστε δυναμική HTML για να δημιουργήσετε οπτικά ελκυστικά PDF με μορφοποιημένο κείμενο, εικόνες ή στυλιζαρισμένους πίνακες.

#### Βήματα Υλοποίησης

**Βήμα 1: Αρχικοποίηση του εγγράφου**
```csharp
// Δημιουργήστε ένα νέο αντικείμενο εγγράφου
Document doc = new Document();
```

**Βήμα 2: Προσθήκη σελίδας**
```csharp
// Προσθήκη σελίδας στη συλλογή σελίδων του εγγράφου
Page page = doc.Pages.Add();
```

**Βήμα 3: Δημιουργήστε ένα τμήμα HTML**
```csharp
// Αρχικοποιήστε το HtmlFragment με το επιθυμητό περιεχόμενο HTML
HtmlFragment title = new HtmlFragment("<fontsize=10><b><i>Table</i></b></fontsize>");
```

**Βήμα 4: Ρύθμιση παραμέτρων περιθωρίων**
```csharp
// Ορισμός περιθωρίων για έλεγχο της τοποθέτησης
title.Margin.Bottom = 10;
title.Margin.Top = 200;
```

**Βήμα 5: Προσθήκη τμήματος HTML στη σελίδα**
```csharp
// Εισαγωγή του τμήματος HTML στη συλλογή παραγράφων της σελίδας
page.Paragraphs.Add(title);
```

**Βήμα 6: Αποθήκευση του εγγράφου**
```csharp
// Ορίστε τη διαδρομή εξόδου και αποθηκεύστε το έγγραφο
string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/AddHTMLUsingDOM_out.pdf";
doc.Save(dataDir);
```

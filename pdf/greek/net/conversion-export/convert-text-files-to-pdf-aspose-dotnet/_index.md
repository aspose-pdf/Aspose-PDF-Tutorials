## Οδηγός Εφαρμογής

### Δυνατότητα μετατροπής κειμένου σε PDF
Αυτή η λειτουργία σάς επιτρέπει να μετατρέπετε αρχεία απλού κειμένου σε έγγραφα PDF με ευκολία. Δείτε πώς λειτουργεί βήμα προς βήμα:

#### Βήμα 1: Διαβάστε το αρχείο κειμένου πηγής
Αρχικά, διαβάστε το περιεχόμενο του αρχείου κειμένου πηγής χρησιμοποιώντας ένα `StreamReader`.
```csharp
using System.IO;
// Ορίστε τη διαδρομή για τον κατάλογο εισόδου.
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

using (TextReader tr = new StreamReader(dataDir + "/log.txt"))
{
    // Ο κώδικας συνεχίζεται...
}
```
**Γιατί;**Η ανάγνωση του περιεχομένου του αρχείου κειμένου μάς επιτρέπει να το χειριστούμε και να το μετατρέψουμε σε μορφή PDF.

#### Βήμα 2: Δημιουργία αντικειμένου εγγράφου
Δημιουργήστε ένα κενό `Document` αντικείμενο που θα χρησιμεύσει ως δοχείο του εγγράφου PDF σας.
```csharp
// Δημιουργήστε ένα νέο αντικείμενο εγγράφου.
Document doc = new Document();
```
**Γιατί;**Αυτό το βήμα αρχικοποιεί τη δομή όπου θα προστεθεί το περιεχόμενο κειμένου και τελικά θα αποθηκευτεί ως αρχείο PDF.

#### Βήμα 3: Προσθήκη νέας σελίδας
Προσθήκη σελίδας στο `Document`συλλογή σελίδων του. Κάθε έγγραφο πρέπει να έχει τουλάχιστον μία σελίδα.
```csharp
// Προσθήκη νέας σελίδας.
Page page = doc.Pages.Add();
```
**Γιατί;**Μια σελίδα είναι απαραίτητη για οποιοδήποτε έγγραφο PDF, η οποία χρησιμεύει ως δοχείο για το περιεχόμενο κειμένου σας.

#### Βήμα 4: Δημιουργία και ρύθμιση παραμέτρων του TextFragment
Δημιουργήστε ένα `TextFragment` για να διατηρήσετε το αναγνωσμένο κείμενο. Προαιρετικά, ορίστε ιδιότητες γραμματοσειράς, εάν χρειάζεται.
```csharp
// Δημιουργήστε ένα TextFragment με το κείμενο από το αρχείο.
TextFragment text = new TextFragment(tr.ReadToEnd());

// Προαιρετικά: Ορισμός γραμματοσειράς για κείμενο.
// text.TextState.Font = FontRepository.FindFont("Arial Unicode MS");
```
**Γιατί;**: `TextFragment` Χρησιμοποιείται για την ενσωμάτωση και τη διαχείριση περιεχομένου κειμένου μέσα σε ένα έγγραφο PDF.

#### Βήμα 5: Προσθήκη κειμένου στη σελίδα
Προσθέστε το δημιουργημένο `TextFragment` στη συλλογή παραγράφων της σελίδας.
```csharp
// Προσθέστε το τμήμα κειμένου στη σελίδα.
page.Paragraphs.Add(text);
```
**Γιατί;**: Αυτό το βήμα τοποθετεί το κείμενό σας στη σελίδα PDF, έτοιμο για μετατροπή και προβολή.

#### Βήμα 6: Αποθήκευση του εγγράφου
Τέλος, αποθηκεύστε το `Document` αντικείμενο ως αρχείο PDF στον επιθυμητό κατάλογο εξόδου.
```csharp
// Ορίστε τη διαδρομή για τον κατάλογο εξόδου.
string outputDir = "YOUR_OUTPUT_DIRECTORY";

doc.Save(outputDir + "/TexttoPDF_out.pdf");
```
**Γιατί;**Η αποθήκευση του εγγράφου ολοκληρώνει τη διαδικασία μετατροπής και δημιουργεί το πραγματικό αρχείο PDF.

## Οδηγός Εφαρμογής

### Αντικατάσταση γραμματοσειράς σε PDF

Η αντικατάσταση γραμματοσειράς είναι ζωτικής σημασίας για τη διατήρηση της εμφάνισης του εγγράφου όταν η αρχική γραμματοσειρά δεν είναι διαθέσιμη. Ας εφαρμόσουμε αυτήν τη λειτουργία χρησιμοποιώντας το Aspose.PDF .NET.

#### Επισκόπηση
Αυτή η ενότητα εστιάζει στην αντικατάσταση γραμματοσειρών που λείπουν με μια συγκεκριμένη εναλλακτική λύση για τη διατήρηση της μορφοποίησης κειμένου και της οπτικής συνέπειας.

#### Βήμα προς βήμα εφαρμογή

**1. Ρύθμιση αντικατάστασης γραμματοσειράς**

Αρχικά, ορίστε τον κανόνα αντικατάστασης για το έγγραφό σας:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;

// Διαδρομή καταλόγου για αποθήκευση εγγράφων
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; 

// Δημιουργήστε ένα αντικείμενο SimpleFontSubstitution για να αντικαταστήσετε το "AgencyFB" με το "Arial"
var substituteFont = new SimpleFontSubstitution("AgencyFB", "Arial");

try
{
    // Προσπάθεια εντοπισμού της αρχικής γραμματοσειράς στο αποθετήριο
    FontRepository.FindFont("AgencyFB");
} catch (Exception)
{
    // Εάν δεν βρεθεί, προσθέστε κανόνα αντικατάστασης
    FontRepository.Substitutions.Add(substituteFont);
}
```

**Εξήγηση:**
- `SimpleFontSubstitution` αντιστοιχίζει μια μη διαθέσιμη γραμματοσειρά ("AgencyFB") σε μια εναλλακτική επιλογή ("Arial").
- Ο κώδικας ελέγχει εάν υπάρχει το "AgencyFB". Εάν δεν υπάρχει, το μπλοκ εξαίρεσης προσθέτει τον κανόνα αντικατάστασης.

### Μετατροπή και αποθήκευση PDF

Η μετατροπή PDF σε τυποποιημένες μορφές όπως PDF/A διασφαλίζει τη συμβατότητα και τη συμμόρφωση με τα πρότυπα αρχειοθέτησης.

#### Επισκόπηση
Αυτή η ενότητα σας καθοδηγεί στη μετατροπή και αποθήκευση ενός εγγράφου PDF σε διαφορετική μορφή χρησιμοποιώντας το Aspose.PDF .NET.

**2. Μετατροπή και αποθήκευση εγγράφων PDF**

Δείτε πώς μπορείτε να μετατρέψετε ένα τυπικό PDF σε μορφή PDF/A-1b, η οποία χρησιμοποιείται ευρέως για μακροπρόθεσμη αρχειοθέτηση:

```csharp
using System.IO;
using Aspose.Pdf;

// Ορίστε καταλόγους και διαδρομές αρχείων
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; 
string outputFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "newfile_out.pdf");

// Φόρτωση του εγγράφου PDF εισόδου
var pdfDocument = new Document(Path.Combine(dataDir, "input.pdf"));

// Μετατρέψτε σε μορφή PDF/A-1b και αποθηκεύστε το αποτέλεσμα
pdfDocument.Convert(Path.Combine(dataDir, "log.xml"), PdfFormat.PDF_A_1B, ConvertErrorAction.Delete);
pdfDocument.Save(outputFile);
```

**Εξήγηση:**
- `Convert` Η μέθοδος χρησιμοποιείται για τη μετατροπή του εγγράφου σε μια καθορισμένη μορφή (PDF/A-1b).
- Οι παράμετροι περιλαμβάνουν τη διαδρομή του αρχείου καταγραφής και τις ενέργειες χειρισμού σφαλμάτων (`ConvertErrorAction.Delete`).

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι τα καθορισμένα αρχεία γραμματοσειρών υπάρχουν στον κατάλογο του συστήματός σας ή της εφαρμογής σας.
- Επαληθεύστε τις διαδρομές αρχείων για τους καταλόγους εισόδου/εξόδου για να αποτρέψετε σφάλματα χρόνου εκτέλεσης.

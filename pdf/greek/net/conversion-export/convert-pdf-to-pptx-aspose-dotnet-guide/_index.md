### Χαρακτηριστικό 1: Βασική μετατροπή PDF σε PPTX

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τη μετατροπή ενός εγγράφου PDF σε παρουσίαση PowerPoint σε μορφή PPTX χρησιμοποιώντας το Aspose.PDF για .NET.

#### Βήματα Υλοποίησης

**Βήμα 1: Φόρτωση του εγγράφου PDF**
```csharp
using Aspose.Pdf;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Aspose.Pdf.Document doc = new Aspose.Pdf.Document(dataDir + "/input.pdf");
```
*Εξήγηση*Εδώ, φορτώνουμε ένα PDF από έναν καθορισμένο κατάλογο χρησιμοποιώντας `Aspose.Pdf.Document`.

**Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης PPTX**
```csharp
PptxSaveOptions pptx_save = new PptxSaveOptions();
```
*Εξήγηση*Δημιουργούμε μια παρουσία του `PptxSaveOptions` για να διαχειριστείτε τον τρόπο αποθήκευσης του PDF ως αρχείο PPTX.

**Βήμα 3: Αποθηκεύστε το έγγραφο σε μορφή PPTX**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
doc.Save(outputDir + "/PDFToPPT_out.pptx", pptx_save);
```
*Εξήγηση*Τέλος, αποθηκεύουμε το φορτωμένο PDF ως αρχείο PPTX στην επιθυμητή θέση.

### Χαρακτηριστικό 2: Μετατροπή διαφανειών PDF σε εικόνες σε PPTX

#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει τη μετατροπή κάθε διαφάνειας στο PDF σας σε εικόνα μέσα στην προκύπτουσα παρουσίαση PPTX.

#### Βήματα Υλοποίησης

**Βήμα 1: Ορισμός διαφανειών ως εικόνων**
```csharp
pptx_save.SlidesAsImages = true;
```
*Εξήγηση*: Με ρύθμιση `SlidesAsImages` Εάν η τιμή είναι true, κάθε σελίδα του PDF θα μετατραπεί σε εικόνα διαφάνειας στο αρχείο PowerPoint.

### Χαρακτηριστικό 3: Παρακολούθηση προόδου κατά τη μετατροπή PDF σε PPTX

#### Επισκόπηση
Παρακολουθήστε και εμφανίστε την πρόοδο της μετατροπής χρησιμοποιώντας προσαρμοσμένους χειριστές συμβάντων για καλύτερη ανατροφοδότηση από τους χρήστες κατά τη διάρκεια της διαδικασίας.

#### Βήματα Υλοποίησης

**Βήμα 1: Ρύθμιση παραμέτρων προσαρμοσμένου χειριστή προόδου**
```csharp
pptx_save.CustomProgressHandler = ShowProgressOnConsole;
```
*Εξήγηση*: Ορίζουμε μια μέθοδο για τον χειρισμό συμβάντων προόδου, η οποία θα παρέχει ενημερώσεις σε πραγματικό χρόνο στην κονσόλα.

### Λειτουργία: Χειριστής παρακολούθησης προόδου

Αυτή η συνάρτηση χειρίζεται διάφορα συμβάντα προόδου κατά τη μετατροπή:

```csharp
void ShowProgressOnConsole(PptxSaveOptions.ProgressEventHandlerInfo eventInfo)
{
    switch (eventInfo.EventType)
    {
        case ProgressEventType.TotalProgress:
            Console.WriteLine($"{DateTime.Now.TimeOfDay} - Conversion progress: {eventInfo.Value}%.");
            break;
        case ProgressEventType.ResultPageCreated:
            Console.WriteLine($"{DateTime.Now.TimeOfDay} - Result page's {eventInfo.Value} of {eventInfo.MaxValue} layout created.");
            break;
        // Χειρισμός άλλων τύπων συμβάντων, όπως απαιτείται
    }
}
```
*Εξήγηση*Αυτή η συνάρτηση παρέχει ανατροφοδότηση με βάση διαφορετικά στάδια της διαδικασίας μετατροπής, όπως η συνολική πρόοδος και η δημιουργία σελίδας αποτελεσμάτων.

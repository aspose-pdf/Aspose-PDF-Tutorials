### Μετατροπή PDF σε XPS
Μετατρέψτε τα υπάρχοντα αρχεία PDF σας σε μορφή XPS χωρίς κόπο:

#### Βήμα 1: Φόρτωση του εγγράφου PDF
Φορτώστε το PDF που απαιτεί μετατροπή:
```csharp
using System;
using Aspose.Pdf;

// Καθορίστε τον κατάλογο εισαγωγής για το αρχείο PDF σας
string inputDirectory = "YOUR_DOCUMENT_DIRECTORY";

document = new Document(inputDirectory + "/input.pdf");
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης XPS
Ορίστε τις επιλογές που απαιτούνται για την αποθήκευση σε μορφή XPS:
```csharp
// Δημιουργία επιλογών αποθήκευσης XPS
var saveOptions = new Aspose.Pdf.XpsSaveOptions();
```

#### Βήμα 3: Αποθήκευση ως έγγραφο XPS
Μετατρέψτε και αποθηκεύστε το PDF σας σε αρχείο XPS:
```csharp
// Ορίστε τον κατάλογο εξόδου για το αρχείο XPS που έχει μετατραπεί
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

document.Save(outputDirectory + "/PDFToXPS_out.xps", saveOptions);
```
**Εξήγηση**: Το `XpsSaveOptions` Η κλάση επιτρέπει την προσαρμογή ειδικά για μετατροπή XPS.

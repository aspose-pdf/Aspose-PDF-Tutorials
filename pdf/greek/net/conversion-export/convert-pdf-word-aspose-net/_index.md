## Οδηγός Εφαρμογής

Εξερευνήστε πώς να μετατρέψετε έγγραφα PDF σε μορφή Word χρησιμοποιώντας διάφορες λειτουργίες του Aspose.PDF.

### Μετατροπή PDF σε μορφή DOC
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα απλό έγγραφο PDF σε μορφή Microsoft Word (DOC) χωρίς πρόσθετες ρυθμίσεις.

#### Επισκόπηση
Φορτώστε ένα αρχείο PDF και αποθηκεύστε το απευθείας ως αρχείο DOC, διατηρώντας τη βασική μορφοποίηση.

#### Βήματα Υλοποίησης
**Βήμα 1: Φόρτωση του εγγράφου PDF**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/PDFToDOC.pdf");
```

**Βήμα 2: Αποθήκευση ως μορφή Word (DOC)**
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/PDFToDOC_out.doc", SaveFormat.Doc);
```
Αυτή η μέθοδος αποθηκεύει το PDF σας σε μορφή DOC, διατηρώντας την αρχική διάταξη και δομή.

### Αποθήκευση PDF σε DOC χρησιμοποιώντας τις επιλογές αποθήκευσης
Βελτιώστε τη διαδικασία μετατροπής σας με προσαρμοσμένες επιλογές αποθήκευσης, όπως λειτουργία αναγνώρισης ή χειρισμό κουκκίδων.

#### Επισκόπηση
Καθορίστε πώς θα πρέπει να ερμηνεύεται το περιεχόμενο PDF κατά τη μετατροπή.

#### Βήματα Υλοποίησης
**Βήμα 1: Φόρτωση του εγγράφου προέλευσης**
```csharp
Document pdfDocument = new Document(dataDir + "/PDFToDOC.pdf");
```

**Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης**
Δημιουργήστε ένα `DocSaveOptions` αντικείμενο για λεπτότερο έλεγχο:
```csharp
DocSaveOptions saveOptions = new DocSaveOptions();
saveOptions.Mode = DocSaveOptions.RecognitionMode.Flow;
saveOptions.RelativeHorizontalProximity = 2.5f;
saveOptions.RecognizeBullets = true;
```
- **Λειτουργία αναγνώρισης**: `Flow` Η λειτουργία ανάγνωσης διασφαλίζει μια φυσική σειρά ανάγνωσης.
- **Σχετική οριζόντια εγγύτητα**: Προσαρμόζει την απόσταση για τα στοιχεία κειμένου.
- **Αναγνώριση Σφαίρες**: Ενεργοποιεί την ανίχνευση κουκκίδων.

**Βήμα 3: Αποθήκευση με προσαρμοσμένες επιλογές**
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/saveOptionsOutput_out.doc", saveOptions);
```

### Μετατροπή PDF σε μορφή DOCX
Μετατρέψτε τα έγγραφά σας PDF στη σύγχρονη μορφή DOCX χρησιμοποιώντας συγκεκριμένες διαμορφώσεις.

#### Επισκόπηση
Αυτή η μετατροπή σάς επιτρέπει να αξιοποιήσετε τις δυνατότητες που είναι διαθέσιμες σε νεότερες εκδόσεις του Word.

#### Βήματα Υλοποίησης
**Βήμα 1: Φόρτωση του εγγράφου σας**
```csharp
Document pdfDocument = new Document(dataDir + "/PDFToDOC.pdf");
```

**Βήμα 2: Ρύθμιση του DocSaveOptions για το DOCX**
Ρυθμίστε τις επιλογές αποθήκευσης για έξοδο ως DOCX:
```csharp
DocSaveOptions saveOptions = new DocSaveOptions();
saveOptions.Format = DocSaveOptions.DocFormat.DocX;
```

**Βήμα 3: Αποθήκευση σε μορφή DOCX**
```csharp
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/ConvertToDOCX_out.docx", saveOptions);
```

## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει λεπτομερώς τη μετατροπή ενός εγγράφου PDF σε αρχείο HTML, με εικόνες που αποθηκεύονται ως εξωτερικά αρχεία PNG στα οποία γίνεται αναφορά μέσω SVG, χρησιμοποιώντας το Aspose.PDF για .NET.

### Βήμα 1: Φόρτωση του εγγράφου PDF
Ξεκινήστε φορτώνοντας το αρχείο PDF σε ένα `Document` αντικείμενο:
```csharp
// Ορίστε τις διαδρομές του καταλόγου σας εδώ
string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";

try
{
    // Δημιουργήστε ένα αντικείμενο εγγράφου για να φορτώσετε το αρχείο PDF
    Document doc = new Document(YOUR_DOCUMENT_DIRECTORY + "/input.pdf");
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message);
}
```

### Βήμα 2: Αρχικοποίηση HtmlSaveOptions
Διαμορφώστε τις επιλογές μετατροπής χρησιμοποιώντας `HtmlSaveOptions`:
```csharp
// Αρχικοποίηση του HtmlSaveOptions με συγκεκριμένες διαμορφώσεις
HtmlSaveOptions saveOptions = new HtmlSaveOptions();
saveOptions.FixedLayout = true;  // Διατηρήστε την αρχική διάταξη του PDF
saveOptions.SplitIntoPages = false;  // Μην χωρίζετε σε ξεχωριστές σελίδες HTML για κάθε σελίδα PDF
```

### Βήμα 3: Ρύθμιση παραμέτρων λειτουργίας αποθήκευσης εικόνας
Ορίστε τον τρόπο αποθήκευσης και αναφοράς των εικόνων:
```csharp
// Αποθήκευση εικόνων ως εξωτερικά αρχεία PNG που αναφέρονται μέσω SVG
saveOptions.RasterImagesSavingMode = HtmlSaveOptions.RasterImagesSavingModes.AsExternalPngFilesReferencedViaSvg;
```

### Βήμα 4: Αποθήκευση του μετατρεπόμενου εγγράφου
Τέλος, αποθηκεύστε το έγγραφό σας σε ένα αρχείο HTML με συγκεκριμένες επιλογές:
```csharp
// Αποθήκευση του μετατρεπόμενου εγγράφου ως αρχείο HTML με καθορισμένες επιλογές
doc.Save(YOUR_OUTPUT_DIRECTORY + "/SaveImages_out.html", saveOptions);
```

**Βασικές επιλογές διαμόρφωσης:**
- `FixedLayout`Διατηρεί την αρχική διάταξη του PDF στην έξοδο HTML.
- `RasterImagesSavingMode`Αποθηκεύει εικόνες ως εξωτερικά αρχεία PNG που αναφέρονται μέσω SVG για καλύτερη απόδοση ιστού.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές καταλόγων έχουν οριστεί σωστά για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Επαληθεύστε ότι το Aspose.PDF έχει εγκατασταθεί και έχει λάβει άδεια χρήσης σωστά για να αποτρέψετε εξαιρέσεις κατά τον χρόνο εκτέλεσης.

## Οδηγός Εφαρμογής

### Μετατροπή σελίδων PDF σε εικόνες BMP

Αυτή η λειτουργία μετατρέπει κάθε σελίδα PDF σε ξεχωριστά αρχεία εικόνας BMP, κάτι χρήσιμο για μορφές έτοιμες για εκτύπωση ή ψηφιακό περιεχόμενο υψηλής ανάλυσης.

#### Βήμα 1: Ρύθμιση των καταλόγων σας

Ορίστε καταλόγους για το PDF πηγής και τις εικόνες εξόδου:

```csharp
private const string DocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
private const string OutputDirectory = "YOUR_OUTPUT_DIRECTORY";
```
Αντικαταστήστε τα placeholders με πραγματικές διαδρομές.

#### Βήμα 2: Φόρτωση του εγγράφου PDF

Φορτώστε το έγγραφο χρησιμοποιώντας το Aspose.PDF `Document` τάξη:

```csharp
Document pdfDocument = new Document(DocumentDirectory + "/AddImage.pdf");
```

Αυτό φορτώνει το "AddImage.pdf" από τον καθορισμένο κατάλογο.

#### Βήμα 3: Διαμόρφωση ρυθμίσεων εξόδου

Ορίστε την ανάλυση για τις εικόνες εξόδου και δημιουργήστε μια `BmpDevice`:

```csharp
Resolution resolution = new Resolution(300); // Ορίστε την επιθυμητή ανάλυση DPI, π.χ. 300 DPI για εικόνες υψηλής ποιότητας
BmpDevice bmpDevice = new BmpDevice(resolution);
```

#### Βήμα 4: Μετατροπή κάθε σελίδας σε εικόνα BMP

Επαναλάβετε κάθε σελίδα PDF και μετατρέψτε την σε εικόνα BMP:

```csharp
for (int pageCount = 1; pageCount <= pdfDocument.Pages.Count; pageCount++)
{
    string outputPath = Path.Combine(OutputDirectory, "image" + pageCount + "_out.bmp");

    using (FileStream imageStream = new FileStream(outputPath, FileMode.Create))
    {
        bmpDevice.Process(pdfDocument.Pages[pageCount], imageStream);
    }
}
```

Αυτός ο βρόχος επεξεργάζεται κάθε σελίδα και την αποθηκεύει ως αρχείο BMP στον καθορισμένο κατάλογο εξόδου.

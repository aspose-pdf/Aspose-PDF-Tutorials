## Οδηγός Εφαρμογής

Ας δούμε τα βασικά βήματα για τη μετατροπή αρχείων MHT σε PDF.

### Μετατροπή MHT σε PDF

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο MHT, το οποίο ουσιαστικά είναι ένα αρχείο HTML με ενσωματωμένους πόρους, σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET.

**Βήματα Υλοποίησης:**
1. **Ρυθμίστε το περιβάλλον σας**
   Ορίστε τις διαδρομές καταλόγων για τα αρχεία εισόδου και εξόδου:
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **Δημιουργία επιλογών φόρτωσης**
   Αρχικοποίηση `MhtLoadOptions` για να διαμορφώσετε τυχόν συγκεκριμένες ρυθμίσεις φόρτωσης που μπορεί να χρειαστείτε:
   
   ```csharp
   MhtLoadOptions options = new MhtLoadOptions();
   // Παράδειγμα διαμόρφωσης (αποσχολιάστε εάν χρειάζεται):
   // options.PageInfo.IsLandscape = true;
   ```

3. **Φόρτωση του εγγράφου MHT**
   Χρήση `MhtLoadOptions` για να φορτώσετε το αρχείο MHT σας σε ένα `Document` αντικείμενο:
   
   ```csharp
   Document document = new Document(dataDir + "/test.mht", options);
   ```

4. **Αποθήκευση ως PDF**
   Τέλος, αποθηκεύστε το φορτωμένο έγγραφο ως PDF στον επιθυμητό κατάλογο εξόδου:
   
   ```csharp
   document.Save(outputDir + "/MHTToPDF_out.pdf");
   ```

### Ρύθμιση παραμέτρων επιλογών φόρτωσης MHT

**Επισκόπηση:**
Η προσαρμογή των επιλογών φόρτωσης μπορεί να βοηθήσει στη βελτιστοποίηση του τρόπου επεξεργασίας του αρχείου MHT πριν από τη μετατροπή.

**Βήματα Υλοποίησης:**
1. **Δημιουργία και Προσαρμογή `MhtLoadOptions`**
   
   ```csharp
   MhtLoadOptions options = new MhtLoadOptions();
   // Ορίστε τις ιδιότητες όπως απαιτείται, π.χ., προσανατολισμό σελίδας:
   // options.PageInfo.IsLandscape = true;
   ```

### Φόρτωση εγγράφου από αρχείο

**Επισκόπηση:**
Η σωστή φόρτωση ενός υπάρχοντος εγγράφου είναι ζωτικής σημασίας για την επεξεργασία και τη μετατροπή.

**Βήματα Υλοποίησης:**
1. **Φόρτωση του αρχείου MHT**
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   MhtLoadOptions loadOptions = new MhtLoadOptions();
   Document loadedDocument = new Document(dataDir + "/test.mht", loadOptions);
   ```

### Αποθήκευση εγγράφου σε αρχείο

**Επισκόπηση:**
Η αποθήκευση του επεξεργασμένου εγγράφου είναι το τελευταίο βήμα στη ροή εργασίας μετατροπής.

**Βήματα Υλοποίησης:**
1. **Αποθήκευση ως PDF**
   
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   loadedDocument.Save(outputDir + "/output.pdf");
   ```

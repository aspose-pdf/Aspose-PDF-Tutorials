## Οδηγός Εφαρμογής

### Ιδιότητες σχολιασμού γραμμής ανάγνωσης

#### Επισκόπηση
Σε αυτήν την ενότητα, θα εξαγάγουμε ιδιότητες από σχολιασμούς γραμμών σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποίηση του εγγράφου**
Ξεκινήστε φορτώνοντας το αρχείο PDF σε ένα `Aspose.Pdf.Document` αντικείμενο. Βεβαιωθείτε ότι έχετε έτοιμη τη διαδρομή προς το PDF σας.

```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_Miscellaneous();
Document doc = new Document(dataDir + "ReadMeasure.pdf");
```

**2. Σχολιασμοί γραμμής πρόσβασης**
Αποκτήστε πρόσβαση σε σχολιασμούς από μια συγκεκριμένη σελίδα και μεταδώστε τους σε `LineAnnotation`.

```csharp
// Ανάκτηση της σχολίασης πρώτης γραμμής από την πρώτη σελίδα
var lineAnnotation = (LineAnnotation)doc.Pages[1].Annotations[1];
```

**3. Ιδιότητες εξαγωγής**
Χρησιμοποιήστε τα ακόλουθα τμήματα κώδικα για να διαβάσετε διάφορες ιδιότητες της σχολίασης γραμμής.

- **Αναλογία κλίμακας**
  ```csharp
  Console.WriteLine(lineAnnotation.Measure.ScaleRatio);
  ```

- **Ετικέτα μονάδας**
  ```csharp
  Console.WriteLine(lineAnnotation.Measure.AreaFormat[1].UnitLabel);
  ```

- **Συντελεστής μετατροπής**
  ```csharp
  Console.WriteLine(lineAnnotation.Measure.AreaFormat[1].ConvresionFactor);
  ```

- **Διαχωριστής κλασμάτων**
  ```csharp
  Console.WriteLine(lineAnnotation.Measure.AreaFormat[1].FractionSeparator);
  ```

#### Βασικές επιλογές διαμόρφωσης
- Βεβαιωθείτε ότι οι σχολιασμοί έχουν καταχωρηθεί σωστά στη σελίδα.
- Επικυρώστε αυτό `Measure` Υπάρχουν ιδιότητες για την αποφυγή εξαιρέσεων μηδενικής αναφοράς.

### Συμβουλές αντιμετώπισης προβλημάτων
- Εάν η πρόσβαση σε μια σχολίαση εμφανίσει σφάλμα, βεβαιωθείτε ότι το ευρετήριο βρίσκεται εντός ορίων.
- Επαληθεύστε ότι το PDF περιέχει σχολιασμούς γραμμής πριν επιχειρήσετε πρόσβαση.

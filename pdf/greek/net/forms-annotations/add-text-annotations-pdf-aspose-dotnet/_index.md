## Οδηγός Εφαρμογής

### Προσθήκη σχολίου κειμένου σε ένα έγγραφο PDF

Ας αναλύσουμε τη διαδικασία προσθήκης σχολίων κειμένου σε διαχειρίσιμα βήματα.

#### Φόρτωση του εγγράφου PDF σας

Ξεκινήστε φορτώνοντας ένα υπάρχον έγγραφο PDF. Καθορίστε τον κατάλογο όπου θα αποθηκευτούν τα έγγραφα εισόδου και εξόδου.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/AddAnnotation.pdf");
```

#### Δημιουργία αντικειμένου σχολιασμού κειμένου

Στη συνέχεια, δημιουργήστε ένα `TextAnnotation` αντικείμενο σε μια συγκεκριμένη σελίδα με επιθυμητές ιδιότητες όπως τίτλο, θέμα και τοποθεσία.

```csharp
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

#### Ρύθμιση παραμέτρων ιδιοτήτων περιγράμματος

Ορίστε ιδιότητες περιγράμματος για να βελτιώσετε την οπτική ελκυστικότητα του σχολιασμού σας.

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1); // Ορίστε στυλ παύλας για το περίγραμμα.
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

#### Προσθήκη σχολίου στο έγγραφο

Τέλος, προσθέστε τη διαμορφωμένη σχολίασή σας στη συλλογή σχολίων μιας συγκεκριμένης σελίδας και αποθηκεύστε το τροποποιημένο έγγραφο.

```csharp
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
pdfDocument.Save("YOUR_OUTPUT_DIRECTORY/AddAnnotation_out.pdf");
```

## Οδηγός Εφαρμογής
### Προσθήκη πεδίου πλαισίου κειμένου
Αυτή η ενότητα εξηγεί πώς να προσθέσετε ένα πεδίο πλαισίου κειμένου στο έγγραφο PDF σας.

#### Επισκόπηση
Τα πεδία TextBox επιτρέπουν στους χρήστες να εισάγουν δεδομένα απευθείας, διευκολύνοντας την υποβολή φορμών και τη συλλογή δεδομένων.

##### Βήμα 1: Ανοίξτε το έγγραφο PDF σας
Φορτώστε ένα υπάρχον αρχείο PDF χρησιμοποιώντας το Aspose.PDF:
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();

// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "TextField.pdf");
```

##### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων ενός TextBoxField
Δημιουργήστε ένα πεδίο πλαισίου κειμένου που καθορίζει τη θέση του:
```csharp
// Δημιουργήστε ένα πεδίο
TextBoxField textBoxField = new TextBoxField(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(100, 200, 300, 300));
textBoxField.PartialName = "textbox1";
textBoxField.Value = "Text Box";
```
**Εξήγηση:**
- `PartialName`: Μοναδικό αναγνωριστικό για το πεδίο.
- `Value`: Το προεπιλεγμένο κείμενο εμφανίζεται στο πλαίσιο.

##### Βήμα 3: Προσαρμογή εμφάνισης
Προσαρμόστε το πλαίσιο κειμένου σας με περιγράμματα και χρώματα:
```csharp
// Ρύθμιση παραμέτρων περιγράμματος
Border border = new Border(textBoxField)
{
    Width = 5,
    Dash = new Dash(1, 1) // Διακεκομμένη γραμμή
};
textBoxField.Border = border;
textBoxField.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
```
**Βασικές επιλογές διαμόρφωσης:**
- `Width`: Πάχος περιγράμματος.
- `Dash`: Μοτίβο για διακεκομμένες γραμμές.
- `Color`: Χρώμα γεμίσματος του πλαισίου κειμένου.

##### Βήμα 4: Προσθήκη πεδίου στο έγγραφο
Προσθέστε το διαμορφωμένο πεδίο πλαισίου κειμένου σας:
```csharp
// Προσθήκη πεδίου στο έγγραφο
pdfDocument.Form.Add(textBoxField, 1);
```

##### Βήμα 5: Αποθηκεύστε το τροποποιημένο PDF σας
Αποθήκευση αλλαγών σε νέο αρχείο:
```csharp
dataDir = dataDir + "TextBox_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nTextbox field added successfully.\nFile saved at " + dataDir);
```

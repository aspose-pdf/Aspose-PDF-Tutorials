## Οδηγός Εφαρμογής

### Προσθήκη αόρατων σχολίων

Τώρα ας επικεντρωθούμε στην κύρια εργασία—την προσθήκη μιας αόρατης σχολίασης στο έγγραφο PDF σας.

#### Βήμα 1: Φόρτωση του εγγράφου PDF

Ξεκινήστε φορτώνοντας το αρχείο PDF με το οποίο θέλετε να εργαστείτε. Αυτό περιλαμβάνει τον καθορισμό της διαδρομής και τη δημιουργία ενός νέου `Document` αντικείμενο.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir_AsposePdf_Annotations();

// Άνοιγμα του εγγράφου
Document doc = new Document(dataDir + "input.pdf");
```

#### Βήμα 2: Δημιουργήστε την σχολίαση

Δημιουργήστε μια παρουσία του `FreeTextAnnotation`Αυτός ο τύπος σάς επιτρέπει να προσθέσετε ελεύθερο κείμενο ως μορφή σχολιασμού.

```csharp
FreeTextAnnotation annotation = new FreeTextAnnotation(doc.Pages[1], 
    new Aspose.Pdf.Rectangle(50, 600, 250, 650), 
    new DefaultAppearance("Helvetica", 16, System.Drawing.Color.Red));
```

- **Επεξήγηση παραμέτρων:**
  - `Rectangle`: Ορίζει τη θέση και το μέγεθος της σχολίασης.
  - `DefaultAppearance`: Ορίζει τη γραμματοσειρά, το μέγεθος και το χρώμα για το κείμενο.

#### Βήμα 3: Ρύθμιση παραμέτρων ιδιοτήτων σχολιασμού

Ορίστε ιδιότητες όπως περιεχόμενο, χρώμα περιγράμματος και σημαίες για να κάνετε την σχολίαση αόρατη.

```csharp
// Ορίστε το περιεχόμενο και τα χαρακτηριστικά
annotation.Contents = "ABCDEFG";
annotation.Characteristics.Border = System.Drawing.Color.Red;
annotation.Flags = AnnotationFlags.Print | AnnotationFlags.NoView;
```

- **Σημαίες σχολίων:**
  - `Print`: Επιτρέπει την εκτύπωση της σχολίασης.
  - `NoView`: Κάνει την σχολίαση αόρατη στην οθόνη.

#### Βήμα 4: Προσθήκη και αποθήκευση της σχολίασης

Προσθέστε τη διαμορφωμένη σχολίασή σας στο έγγραφο και αποθηκεύστε την σε ένα νέο αρχείο.

```csharp
// Προσθήκη της σχολίασης στην πρώτη σελίδα
doc.Pages[1].Annotations.Add(annotation);

dataDir = dataDir + "InvisibleAnnotation_out.pdf";

// Αποθήκευση του αρχείου εξόδου
doc.Save(dataDir);
```

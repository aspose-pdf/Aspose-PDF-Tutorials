## Οδηγός Εφαρμογής
### Επισκόπηση των σχολίων μελανιού
Οι σχολιασμοί με μελάνι επιτρέπουν στους χρήστες να σχεδιάζουν σε σελίδες PDF χρησιμοποιώντας γραφίδα ή ποντίκι. Αυτοί οι σχολιασμοί μπορούν να προσαρμοστούν ως προς το χρώμα, την αδιαφάνεια και το στυλ.

#### Δημιουργία σχολίου μελανιού
Ας αναλύσουμε την υλοποίηση σε διαχειρίσιμα βήματα:

##### Βήμα 1: Αρχικοποίηση του εγγράφου σας
Ξεκινήστε δημιουργώντας ένα νέο έγγραφο PDF και προσθέτοντας μια σελίδα:
```csharp
document doc = new Document();
Page pdfPage = doc.Pages.Add();
```

##### Βήμα 2: Ορίστε την περιοχή σχολίων
Ρυθμίστε την περιοχή ορθογωνίου για τη σχολίαση με μελάνι. Αυτό το παράδειγμα καλύπτει ολόκληρη τη σελίδα:
```csharp
System.Drawing.Rectangle drect = new System.Drawing.Rectangle()
{
    Height = (int)pdfPage.Rect.Height,
    Width = (int)pdfPage.Rect.Width,
    X = 0,
    Y = 0
};
Aspose.Pdf.Rectangle arect = Aspose.Pdf.Rectangle.FromRect(drect);
```

##### Βήμα 3: Δημιουργήστε την σχολίαση μελανιού
Ορίστε σημεία για τις διαδρομές μελάνης σας και προσθέστε τα σε μια λίστα:
```csharp
IList<Point[]> inkList = new List<Point[]>();
Aspose.Pdf.Point[] arrpt = { 
    new Aspose.Pdf.Point(100, 800), 
    new Aspose.Pdf.Point(200, 800), 
    new Aspose.Pdf.Point(200, 700) 
};
inkList.Add(arrpt);

InkAnnotation ia = new InkAnnotation(pdfPage, arect, inkList)
{
    Title = "XXX",
    Color = Aspose.Pdf.Color.LightBlue,
    CapStyle = CapStyle.Rounded
};
```

##### Βήμα 4: Προσαρμόστε την σχολίαση
Προσαρμόστε ιδιότητες όπως το πλάτος και την αδιαφάνεια των περιγραμμάτων:
```csharp
Border border = new Border(ia) { Width = 25 };
ia.Opacity = 0.5;

pdfPage.Annotations.Add(ia);
```

#### Αποθήκευση του εγγράφου σας
Τέλος, αποθηκεύστε το έγγραφό σας σε έναν καθορισμένο κατάλογο:
```csharp
string dataDir = RunExamples.GetDataDir_AsposePdf_Annotations();
dataDir += "AddInkAnnotation_out.pdf";
doc.Save(dataDir);

Console.WriteLine("\nInk annotation added successfully.\nFile saved at " + dataDir);
```

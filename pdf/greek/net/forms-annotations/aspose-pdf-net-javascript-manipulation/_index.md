# Βελτίωση PDF με JavaScript χρησιμοποιώντας το Aspose.PDF .NET

Αναζητάτε τρόπους για να προσθέσετε δυναμικές λειτουργίες όπως η JavaScript στα έγγραφά σας PDF; Αυτός ο ολοκληρωμένος οδηγός θα σας διδάξει πώς να φορτώνετε, να αποθηκεύετε και να εισάγετε διαδραστικά στοιχεία σε αρχεία PDF χρησιμοποιώντας το Aspose.PDF για .NET. ## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση και αποθήκευση εγγράφου PDF

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον έγγραφο PDF, να κάνετε τροποποιήσεις και να το αποθηκεύσετε ξανά.

##### Βήμα προς βήμα εφαρμογή

**3.1 Φόρτωση του εγγράφου PDF**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
Document doc = new Document(dataDir);
```

**3.2 Αποθήκευση του τροποποιημένου εγγράφου**
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/ModifiedOutput.pdf";
doc.Save(outputPath);
```
*Παράμετροι*: Ενημέρωση `dataDir` και `outputPath` με τις συγκεκριμένες διαδρομές καταλόγου σας.

### Λειτουργία 2: Προσθήκη JavaScript σε επίπεδο εγγράφου

#### Επισκόπηση
Μάθετε πώς να προσθέτετε JavaScript που ζητά από τον χρήστη να κάνει ενέργειες, όπως το άνοιγμα ενός παραθύρου διαλόγου εκτύπωσης όταν ανοίγει το έγγραφο.

##### Βήμα προς βήμα εφαρμογή

**3.1 Ορισμός Ενέργειας JavaScript**
```csharp
JavascriptAction javaScript = new JavascriptAction("this.print({bUI:true,bSilent:false,bShrinkToFit:true});");
```

**3.2 Ανάθεση σε ενέργεια ανοίγματος εγγράφου**
```csharp
doc.OpenAction = javaScript;
```

**3.3 Αποθήκευση του ενημερωμένου PDF**
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/JavaScriptDocumentLevel_out.pdf";
doc.Save(outputPath);
```

### Λειτουργία 3: Προσθήκη JavaScript σε επίπεδο σελίδας

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέτετε διαδραστικές ενέργειες JavaScript ειδικά για μεμονωμένες σελίδες μέσα σε ένα PDF.

##### Βήμα προς βήμα εφαρμογή

**3.1 Ορισμός JavaScript για Συγκεκριμένες Σελίδες**
```csharp
doc.Pages[2].Actions.OnOpen = new JavascriptAction("app.alert('Page opened')");
doc.Pages[2].Actions.OnClose = new JavascriptAction("app.alert('Page closed')");
```

**3.2 Αποθήκευση του εγγράφου**
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/JavaScriptPageLevel_out.pdf";
doc.Save(outputPath);
```
*Σημείωμα*: Προσαρμόστε τους δείκτες σελίδων και τις ενέργειες JavaScript όπως απαιτείται.

## Οδηγός Εφαρμογής
Αυτή η ενότητα χωρίζεται σε δύο κύρια χαρακτηριστικά: χρήση `Document` και `Form` τάξεις.

### Διατήρηση δικαιωμάτων χρησιμοποιώντας την κλάση εγγράφων

#### Επισκόπηση
Ο `Aspose.Pdf.Document` Η κλάση σάς επιτρέπει να ανοίγετε και να τροποποιείτε υπάρχοντα έγγραφα PDF, συμπεριλαμβανομένων των πεδίων φόρμας τους. Αυτή η μέθοδος διατηρεί τα δικαιώματα του εγγράφου μετά τις τροποποιήσεις.

#### Βήμα προς βήμα εφαρμογή

**1. Ανοίξτε το αρχείο PDF**
Ξεκινήστε δημιουργώντας ένα FileStream με δικαιώματα ανάγνωσης-εγγραφής:

```csharp
using System.IO;
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
FileStream fs = new FileStream(dataDir + "/input.pdf", FileMode.Open, FileAccess.ReadWrite);
```

**2. Δημιουργία στιγμιαίας παρουσίας εγγράφου**
Δημιουργήστε μια παρουσία του `Aspose.Pdf.Document` για να αλληλεπιδράσετε με το αρχείο PDF:

```csharp
Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document(fs);
```

**3. Τροποποίηση πεδίων φόρμας**
Επαναλάβετε τη διαδικασία στα πεδία της φόρμας, τροποποιώντας τις τιμές όπως απαιτείται:

```csharp
foreach (Field formField in pdfDocument.Form)
{
    if (formField.FullName.Contains("A1"))
    {
        TextBoxField textBoxField = formField as TextBoxField;
        textBoxField.Value = "New Value";  // Αλλάξτε την «Νέα Τιμή» στην επιθυμητή τιμή.
    }
}
```

**4. Αποθήκευση και κλείσιμο**
Αποθηκεύστε τις αλλαγές και κλείστε το FileStream:

```csharp
pdfDocument.Save();
fs.Close();
```

### Διατήρηση δικαιωμάτων χρησιμοποιώντας την κλάση φόρμας

#### Επισκόπηση
Ο `Aspose.Pdf.Facades.Form` Η κλάση παρέχει μια απλούστερη διεπαφή για την άμεση συμπλήρωση πεδίων φόρμας.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργία στιγμιαίας παρουσίας φόρμας**
Δημιουργήστε μια παρουσία του `Form` τάξη για να φορτώσετε το PDF σας:

```csharp
using Aspose.Pdf.Facades;
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Form form = new Form(dataDir + "/input.pdf");
```

**2. Συμπληρώστε ένα συγκεκριμένο πεδίο**
Χρησιμοποιήστε το `FillField` μέθοδος για την ενημέρωση τιμών πεδίων:

```csharp
form.FillField("topmostSubform[0].Page1[0].f1_29_0_[0]", "New Value");  // Ενημερώστε με το πεδίο-στόχο και την τιμή σας.
```

**3. Αποθήκευση αλλαγών**
Αποθηκεύστε το τροποποιημένο έγγραφο σε μια καθορισμένη διαδρομή:

```csharp
string outputPath = dataDir + "/PreserveRightsUsingFormClass_out.pdf";
form.Save(outputPath);
```

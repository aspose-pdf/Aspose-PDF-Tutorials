## Οδηγός Εφαρμογής
### Δημιουργία σχολίου συνημμένου αρχείου
Αυτή η ενότητα θα σας καθοδηγήσει στη δημιουργία σχολίων συνημμένου αρχείου χρησιμοποιώντας το Aspose.PDF για .NET.

#### Βήμα 1: Δημιουργία αντικειμένου PdfContentEditor
Ξεκινήστε δημιουργώντας ένα στιγμιότυπο του `PdfContentEditor` κλάση, η οποία επιτρέπει την επεξεργασία αρχείων PDF:

```csharp
// Δημιουργήστε μια παρουσία του PdfContentEditor
PdfContentEditor editor = new PdfContentEditor();
```
Ο `PdfContentEditor` Το αντικείμενο είναι κρίσιμο για την τροποποίηση υπαρχόντων PDF.

#### Βήμα 2: Σύνδεση Εισαγωγή αρχείου PDF
Στη συνέχεια, συνδέστε το αρχείο PDF εισόδου στο `PdfContentEditor` αντικείμενο:

```csharp
// Καθορίστε τη διαδρομή προς το έγγραφο εισαγωγής σας
editor.BindPdf("YOUR_DOCUMENT_DIRECTORY\\CreateFileAnnotation.pdf");
```
Ensure you replace "YOUR_DOCUMENT_DIRECTORY\\CreateFileAnnotation.pdf" with the actual path to your PDF.

#### Βήμα 3: Δημιουργήστε μια σχολίαση συνημμένου αρχείου
Προσθέστε μια σχολίαση συνημμένου αρχείου ορίζοντας την τοποθεσία, τη διαφάνειά του και άλλες ιδιότητες:

```csharp
// Ορίστε παραμέτρους για το συνημμένο αρχείο
editor.CreateFileAttachment(
    new System.Drawing.Rectangle(50, 50, 10, 10), // Τοποθεσία στη σελίδα
    "here",                                     // Τίτλος της σχολίασης
    "YOUR_DOCUMENT_DIRECTORY\\AddFreeTextAnnotation.pdf", // Διαδρομή προς το συνημμένο αρχείο
    1,                                          // Αριθμός σελίδας για συνημμένο (προαιρετικό)
    "Paperclip",                                // Τύπος εικονιδίου
    0.005                                       // Επίπεδο διαφάνειας
);
```
Σε αυτό το παράδειγμα:
- **Ορθογώνιο παραλληλόγραμμο**: Καθορίζει πού στη σελίδα θα εμφανίζεται το εικονίδιο σχολιασμού.
- **Διαδρομή Συνημμένου**: Διαδρομή προς το αρχείο που επισυνάπτετε.
- **Εικόνισμα**Επιλέγει ένα αναγνωρίσιμο εικονίδιο, όπως "Συνδετήρας".
- **Διαφάνεια**: Ελέγχει τον τρόπο με τον οποίο εμφανίζεται το εικονίδιο ως διαφανές.

#### Βήμα 4: Αποθηκεύστε το ενημερωμένο PDF
Αφού δημιουργήσετε τη σχολίασή σας, αποθηκεύστε το ενημερωμένο έγγραφο:

```csharp
// Αποθηκεύστε το PDF εξόδου με σχολιασμούς
editor.Save("YOUR_OUTPUT_DIRECTORY\\PdfWith_Transparent_Annotation_out.pdf");
```
Replace "YOUR_OUTPUT_DIRECTORY\\PdfWith_Transparent_Annotation_out.pdf" with your desired file path.

## Οδηγός Υλοποίησης: Προσθήκη Ελεύθερου Κειμένου σε Σχόλια

### Επισκόπηση της λειτουργίας

Η προσθήκη σχολίων με ελεύθερο κείμενο σάς επιτρέπει να εισάγετε προσαρμοσμένο κείμενο σε συγκεκριμένες θέσεις στο PDF σας. Αυτή η λειτουργία είναι απαραίτητη για την επισήμανση ενοτήτων, την προσθήκη σχολίων ή την επισήμανση σημαντικών πληροφοριών.

### Βήμα προς βήμα εφαρμογή

#### Βήμα 1: Ορισμός διαδρομών και αρχικοποίηση του PdfContentEditor

Ξεκινήστε ορίζοντας τις διαδρομές για τα έγγραφα εισόδου και εξόδου και αρχικοποιήστε το `PdfContentEditor` αντικείμενο:

```csharp
using System.IO;
using Aspose.Pdf.Facades;

public static void AddFreeTextAnnotation()
{
    string dataDir = @"YOUR_DOCUMENT_DIRECTORY";  
    string outputPath = @"YOUR_OUTPUT_DIRECTORY";

    using (PdfContentEditor contentEditor = new PdfContentEditor())
    {
        // Άνοιγμα ενός υπάρχοντος εγγράφου PDF
        contentEditor.BindPdf(dataDir + "AddFreeTextAnnotation.pdf");
```

#### Βήμα 2: Ορισμός ορθογωνίου σχολιασμού

Προσδιορίστε την περιοχή όπου θέλετε να τοποθετήσετε την σχολίασή σας. Εδώ, χρησιμοποιούμε ένα ορθογώνιο:

```csharp
        System.Drawing.Rectangle rect = new System.Drawing.Rectangle(50, 50, 100, 100);
```

#### Βήμα 3: Δημιουργία και προσθήκη σχολίων ελεύθερου κειμένου

Τώρα, δημιουργήστε την σχολίαση ελεύθερου κειμένου μέσα στο καθορισμένο ορθογώνιο και προσθέστε την στο PDF σας:

```csharp
        // Δημιουργήστε μια δωρεάν σχολίαση κειμένου
        contentEditor.CreateFreeText(rect, "Sample content", 1);

        // Αποθήκευση του ενημερωμένου εγγράφου
        contentEditor.Save(outputPath + "AddFreeTextAnnotation_out.pdf");
    }
}
```

### Επεξήγηση Παραμέτρων και Μεθόδων

- `rect`: Ορίζει πού στη σελίδα θα εμφανίζεται η σχολίασή σας.
- `CreateFreeText()`: Προσθέτει μια σχολίαση ελεύθερου κειμένου με καθορισμένες διαστάσεις και κείμενο.
- `Save()`Αποθηκεύει τις αλλαγές σε ένα νέο αρχείο PDF.

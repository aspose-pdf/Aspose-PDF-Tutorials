## Οδηγός Εφαρμογής
Θα εξερευνήσουμε δύο κύρια χαρακτηριστικά: την εξαγωγή πεδίων φόρμας και την προσθήκη νέων.

### Εξαγωγή ονομάτων και θέσεων πεδίων φόρμας
#### Επισκόπηση
Ανάκτηση ονομάτων και συντεταγμένων πλαισίου οριοθέτησης όλων των πεδίων φόρμας σε ένα PDF, κάτι ζωτικής σημασίας για τη δυναμική διαχείριση και τον αυτοματισμό φορμών.

#### Βήμα προς βήμα εφαρμογή
**1. Εισαγωγή απαραίτητων βιβλιοθηκών**
```csharp
using Aspose.Pdf.Facades;
```

**2. Αρχικοποίηση του αντικειμένου φόρμας**
Δημιουργήστε μια παρουσία του `Aspose.Pdf.Facades.Form` για να αλληλεπιδράσετε με το PDF σας.
```csharp
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form(dataDir + "/FilledForm.pdf");
```

**3. Εξαγωγή ονομάτων πεδίων**
Ανάκτηση όλων των ονομάτων πεδίων χρησιμοποιώντας `FieldNames`.
```csharp
String[] allfields = form.FieldNames;
Rectangle[] box = new Rectangle[allfields.Length];
for (int i = 0; i < allfields.Length; i++) {
    FormFieldFacade facade = form.GetFieldFacade(allfields[i]);
    box[i] = facade.Box;
}
```
*Εξήγηση:* Αυτός ο βρόχος επαναλαμβάνεται σε κάθε πεδίο, εξάγοντας τις συντεταγμένες του πλαισίου οριοθέτησης χρησιμοποιώντας `GetFieldFacade()`.

**4. Αποθήκευση αλλαγών**
Αποθηκεύστε τις τροποποιήσεις σας σε ένα νέο αρχείο PDF.
```csharp
form.Save(dataDir + "/ExtractedFields_out.pdf");
```

### Προσθήκη νέων πεδίων κειμένου σε υπάρχουσα φόρμα PDF
#### Επισκόπηση
Προσθέστε νέα πεδία κειμένου κάτω από τα υπάρχοντα, βελτιώνοντας την προσαρμογή της φόρμας και την αλληλεπίδραση με τον χρήστη.

#### Βήμα προς βήμα εφαρμογή
**1. Τοποθετήστε το έγγραφο**
```csharp
Document document = new Document(dataDir + "/FilledForm - 2.pdf");
```

**2. Αρχικοποίηση του FormEditor**
Χρήση `FormEditor` για την προσθήκη νέων πεδίων.
```csharp
FormEditor editor = new FormEditor(document);
```

**3. Προσθέστε πεδία κάτω από τα υπάρχοντα**
Περιηγηθείτε στα υπάρχοντα πεδία και προσθέστε νέα ακριβώς από κάτω.
```csharp
for (int i = 0; i < allfields.Length; i++) {
    editor.AddField(Aspose.Pdf.Forms.FieldType.Text, "TextField" + i, allfields[i], 1,
        box[i].X, box[i].Y - 10, box[i].X + 50, box[i].Y);
}
```
*Εξήγηση:* Ο `AddField()` Η μέθοδος τοποθετεί τα νέα πεδία σε σχέση με τα υπάρχοντα.

**4. Αποθηκεύστε το τροποποιημένο PDF**
```csharp
editor.Save(dataDir + "/AddedFields_out.pdf");
```

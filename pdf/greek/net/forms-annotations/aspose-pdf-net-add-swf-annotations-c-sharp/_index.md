## Οδηγός Εφαρμογής

### Άνοιγμα και φόρτωση εγγράφου PDF

#### Επισκόπηση

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον έγγραφο PDF χρησιμοποιώντας **Aspose.PDF για .NET**.

##### Βήμα 1: Ορισμός του καταλόγου

```csharp
using System;
using Aspose.Pdf;

public class FeatureOpenPDF
{
    public static void Run()
    {
        // Καθορίστε τον κατάλογο εγγράφων σας
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        // Άνοιγμα ενός υπάρχοντος αρχείου PDF
        Document doc = new Document(dataDir + "AddSwfFileAsAnnotation.pdf");
        
        Console.WriteLine("The document is now loaded and ready for manipulation.");
    }
}
```

- **Παράμετροι:** Η διαδρομή προς τον κατάλογο εγγράφων σας.
- **Σκοπός:** Φορτώνει το έγγραφο PDF στη μνήμη, προετοιμάζοντάς το για περαιτέρω λειτουργίες.

### Προσθήκη αρχείου SWF ως σχολιασμού

#### Επισκόπηση

Προσθέστε περιεχόμενο πολυμέσων εισάγοντας μια σχολίαση αρχείου SWF σε μια συγκεκριμένη σελίδα του PDF σας.

##### Βήμα 2: Αναφορά και σχολιασμός στην πρώτη σελίδα

```csharp
using System;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;

public class FeatureAddSwfAnnotation
{
    public static void Run()
    {
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        // Φόρτωση του εγγράφου
        Document doc = new Document(dataDir + "AddSwfFileAsAnnotation.pdf");
        
        // Λάβετε αναφορά στην πρώτη σελίδα
        Page page = doc.Pages[1];
        
        // Δημιουργία και προσθήκη σχολίων SWF
        ScreenAnnotation annotation = new ScreenAnnotation(page, 
            new Aspose.Pdf.Rectangle(0, 400, 600, 700), dataDir + "input.swf");
        
        page.Annotations.Add(annotation);
        
        Console.WriteLine("An SWF file annotation is added to the first page.");
    }
}
```

- **Παράμετροι:** Διαδρομή αρχείου για το SWF και θέση εγγράφου.
- **Σκοπός:** Τοποθετεί ένα διαδραστικό αρχείο πολυμέσων SWF μέσα στο PDF σας.

### Αποθήκευση ενημερωμένου εγγράφου PDF

#### Επισκόπηση

Αφού κάνετε τις αλλαγές, βεβαιωθείτε ότι έχετε αποθηκεύσει ξανά το ενημερωμένο PDF στον δίσκο.

##### Βήμα 3: Αποθήκευση τροποποιήσεων

```csharp
using System;
using Aspose.Pdf;

public class FeatureSavePDF
{
    public static void Run()
    {
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        // Φόρτωση και τροποποίηση του εγγράφου όπως απαιτείται
        Document doc = new Document(dataDir + "AddSwfFileAsAnnotation.pdf");
        
        // Καθορίστε τη διαδρομή εξόδου για αποθήκευση
        string outputPath = dataDir + "AddSwfFileAsAnnotation_out.pdf";
        
        // Αποθήκευση αλλαγών σε νέο αρχείο
        doc.Save(outputPath);
        
        Console.WriteLine("The modified PDF document is saved with annotations.");
    }
}
```

- **Σκοπός:** Διατηρεί τις προστιθέμενες σχολιασμούς και τυχόν άλλες τροποποιήσεις.

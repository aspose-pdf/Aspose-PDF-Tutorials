# Πώς να προσθέσετε κουμπιά επιλογής σε PDF χρησιμοποιώντας το Aspose.PDF για .NET (Οδηγός C#)

## Οδηγός Εφαρμογής

### Δημιουργία και προσθήκη πεδίων κουμπιών ραδιοφώνου

#### Επισκόπηση
Αυτή η ενότητα σας καθοδηγεί στη δημιουργία πεδίων κουμπιών επιλογής στο έγγραφο PDF σας χρησιμοποιώντας C#. Θα μάθετε πώς να δημιουργήσετε ένα αντίγραφο ενός `RadioButtonField`, προσθέστε επιλογές και επισυνάψτε το στη φόρμα.

**Βήμα 1: Δημιουργία αντικειμένου εγγράφου**
Ξεκινήστε δημιουργώντας ένα νέο `Document` αντικείμενο. Αυτό χρησιμεύει ως δοχείο για το περιεχόμενο PDF σας.
```csharp
// Δημιουργήστε ένα νέο έγγραφο PDF
Document pdfDocument = new Document();
```

**Βήμα 2: Προσθήκη σελίδας στο έγγραφό σας**
Πρέπει να προστεθεί μια σελίδα πριν από την τοποθέτηση πεδίων σε αυτήν.
```csharp
// Προσθήκη κενής σελίδας
pdfDocument.Pages.Add();
```

**Βήμα 3: Δημιουργία αντικειμένου RadioButtonField**
Ο `RadioButtonField` Το αντικείμενο αντιπροσωπεύει την ομάδα κουμπιών επιλογής. Καθορίστε τον αριθμό σελίδας κατά τη δημιουργία του.
```csharp
// Δημιουργήστε ένα νέο πεδίο κουμπιού επιλογής στη σελίδα 1
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

**Βήμα 4: Προσθήκη επιλογών στα κουμπιά ραδιοφώνου σας**
Ορίστε πολλαπλές επιλογές μέσα στο πεδίο του κουμπιού επιλογής σας, καθορίζοντας τις θέσεις τους χρησιμοποιώντας `Rectangle` αντικείμενα.
```csharp
// Προσθήκη πρώτης επιλογής με συγκεκριμένη θέση
radio.AddOption("Test", new Rectangle(0, 0, 20, 20));

// Προσθήκη δεύτερης επιλογής σε διαφορετική τοποθεσία
radio.AddOption("Test1", new Rectangle(20, 20, 40, 40));
```

**Βήμα 5: Επισυνάψτε το πεδίο του κουμπιού επιλογής στη φόρμα**
Προσθέστε το πεδίο του κουμπιού επιλογής σας στη συλλογή φορμών του εγγράφου.
```csharp
// Προσθήκη του πεδίου του κουμπιού επιλογής στη φόρμα PDF
pdfDocument.Form.Add(radio);
```

**Βήμα 6: Αποθήκευση του εγγράφου σας**
Μόλις ολοκληρωθεί η διαμόρφωση, αποθηκεύστε το έγγραφό σας με όλα τα στοιχεία του ανέπαφα.
```csharp
// Ορίστε τη διαδρομή αρχείου και αποθηκεύστε το έγγραφο
string dataDir = RunExamples.GetDataDir_AsposePdf_Forms();
dataDir += "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```

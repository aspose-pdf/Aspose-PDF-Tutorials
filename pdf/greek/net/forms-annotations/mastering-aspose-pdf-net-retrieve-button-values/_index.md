# Ανάκτηση τιμών κουμπιών σε φόρμες PDF χρησιμοποιώντας το Aspose.PDF .NET

Ας εμβαθύνουμε στην εξαγωγή κρίσιμων δεδομένων από φόρμες PDF χρησιμοποιώντας το Aspose.PDF .NET.

## Οδηγός Εφαρμογής
Θα ασχοληθούμε με δύο κύρια χαρακτηριστικά: την ανάκτηση τιμών επιλογών κουμπιών και τη λήψη της τρέχουσας επιλεγμένης τιμής ενός πεδίου κουμπιού.

### Χαρακτηριστικό 1: Λήψη τιμών επιλογών κουμπιού
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε όλες τις διαθέσιμες επιλογές για ένα συγκεκριμένο πεδίο κουμπιού από μια φόρμα PDF, παρέχοντας πολύτιμες πληροφορίες σχετικά με τις επιλογές των χρηστών ή τις διαμορφώσεις φόρμας.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1:** Δημιουργήστε και αρχικοποιήστε το αντικείμενο Form.
```csharp
Aspose.Pdf.Facades.Form pdfForm = new Aspose.Pdf.Facades.Form();
```

**Βήμα 2:** Συνδέστε το έγγραφο PDF σας με το αντικείμενο Form.
```csharp
pdfForm.BindPdf("YOUR_DOCUMENT_DIRECTORY/FormField.pdf");
```
*Εξήγηση:* Η σύνδεση ενός PDF διασφαλίζει ότι όλα τα στοιχεία του είναι προσβάσιμα για χειρισμό.

**Βήμα 3:** Ανάκτηση και εμφάνιση τιμών επιλογών κουμπιού.
```csharp
var optionValues = pdfForm.GetButtonOptionValues("Gender");

IDictionaryEnumerator optionValueEnumerator = optionValues.GetEnumerator();
while (optionValueEnumerator.MoveNext()) {
    Console.WriteLine("Key : {0} , Value : {1}", optionValueEnumerator.Key, optionValueEnumerator.Value);
}
```
*Εξήγηση:* Ο `GetButtonOptionValues` Η μέθοδος ανακτά μια απαρίθμηση όλων των επιλογών κουμπιών για το καθορισμένο πεδίο.

**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι το όνομα πεδίου ("Φύλο") ταιριάζει ακριβώς με τα ονόματα πεδίων της φόρμας PDF σας για να αποφύγετε σφάλματα.

### Λειτουργία 2: Λήψη τρέχουσας τιμής επιλογής κουμπιού
#### Επισκόπηση
Η κατανόηση της τρέχουσας επιλογής σε μια μορφή PDF μπορεί να είναι κρίσιμη, ειδικά κατά την επεξεργασία δεδομένων ή ρυθμίσεων χρήστη.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1:** Όπως και πριν, αρχικοποιήστε το αντικείμενο Form και συνδέστε το έγγραφό σας.
```csharp
Aspose.Pdf.Facades.Form pdfForm = new Aspose.Pdf.Facades.Form();
pdfForm.BindPdf("YOUR_DOCUMENT_DIRECTORY/FormField.pdf");
```

**Βήμα 2:** Λήψη και εξαγωγή της τρέχουσας επιλεγμένης τιμής.
```csharp
string optionValue = pdfForm.GetButtonOptionCurrentValue("Gender");
Console.WriteLine("Current Value : {0}", optionValue);
```
*Εξήγηση:* `GetButtonOptionCurrentValue` Ανακτά την τρέχουσα ενεργή επιλογή, παρέχοντας άμεση εικόνα για τις καταστάσεις της φόρμας.

**Συμβουλή αντιμετώπισης προβλημάτων:** Εάν δεν επιστραφεί καμία τιμή, επαληθεύστε ότι το πεδίο PDF περιέχει έγκυρα δεδομένα και ταιριάζει με το όνομα πεδίου που καθορίσατε.

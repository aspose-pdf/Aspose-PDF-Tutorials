## Οδηγός Εφαρμογής

### Δημιουργία σχολίου συνδέσμου εγγράφου

Αυτή η λειτουργία δείχνει πώς να προσθέσετε μια σχολίαση συνδέσμου σε ένα υπάρχον PDF.

#### Βήμα 1: Φόρτωση του εγγράφου PDF

Αρχικά, φορτώστε το έγγραφο PDF χρησιμοποιώντας το `Document` τάξη:

```csharp
using Aspose.Pdf;

// Ορίστε διαδρομές για αρχεία εισόδου και εξόδου
string inputFilePath = @"YOUR_DOCUMENT_DIRECTORY\CreateDocumentLink.pdf";
string outputFilePath = @"YOUR_OUTPUT_DIRECTORY\CreateDocumentLink_out.pdf";

// Φόρτωση ενός υπάρχοντος εγγράφου PDF
Document document = new Document(inputFilePath);
```

#### Βήμα 2: Πρόσβαση στην επιθυμητή σελίδα

Στη συνέχεια, αποκτήστε πρόσβαση στη συγκεκριμένη σελίδα όπου θέλετε να προσθέσετε τη σχολίαση:

```csharp
// Πρόσβαση στην πρώτη σελίδα του εγγράφου
Page page = document.Pages[1];
```

#### Βήμα 3: Δημιουργία και ρύθμιση παραμέτρων σχολιασμού συνδέσμου

Δημιουργήστε ένα `LinkAnnotation` αντικείμενο, καθορίζοντας τη θέση και το μέγεθός του στη σελίδα. Δείτε πώς μπορείτε να το διαμορφώσετε:

```csharp
using Aspose.Pdf.Annotations;

// Ορίστε ένα ορθογώνιο για την περιοχή του συνδέσμου
Rectangle rect = new Rectangle(100, 100, 300, 300);

// Δημιουργήστε την σχολίαση συνδέσμου
LinkAnnotation link = new LinkAnnotation(page, rect);

// Ορίστε το χρώμα του συνδέσμου σε πράσινο
link.Color = Color.FromRgb(System.Drawing.Color.Green);

// Ορίστε μια απομακρυσμένη ενέργεια για τον σύνδεσμο (π.χ., πλοήγηση σε άλλο PDF)
link.Action = new GoToRemoteAction(@"YOUR_DOCUMENT_DIRECTORY\RemoveOpenAction.pdf", 1);
```

#### Βήμα 4: Προσθήκη σχολίου και αποθήκευση

Προσθέστε τη διαμορφωμένη σχολίασή σας στη συλλογή σχολιασμών της σελίδας και, στη συνέχεια, αποθηκεύστε το έγγραφο:

```csharp
// Προσθήκη σχολιασμού συνδέσμου στη σελίδα
page.Annotations.Add(link);

// Αποθήκευση του ενημερωμένου PDF
document.Save(outputFilePath);
```

### Ρύθμιση παραμέτρων ιδιοτήτων σχολιασμού συνδέσμου

Αυτή η ενότητα δείχνει πώς να ορίσετε ιδιότητες όπως χρώμα και ενέργειες για ένα `LinkAnnotation`.

#### Βήμα 1: Ορισμός σελίδας και σχολιασμού

Υποθέτοντας ότι έχετε ένα `Page` αντικείμενο:

```csharp
Page page = new Page(); // Αντικατάσταση με την πραγματική παρουσία σελίδας
LinkAnnotation link = new LinkAnnotation(page, new Rectangle(100, 100, 300, 300));
```

#### Βήμα 2: Ορισμός ιδιοτήτων

Ρυθμίστε τις παραμέτρους των ιδιοτήτων της σχολίασης, όπως το χρώμα και την ενέργεια:

```csharp
// Ορισμός χρώματος σχολιασμού σε πράσινο
link.Color = Color.FromRgb(System.Drawing.Color.Green);

// Ορισμός μιας ενέργειας (π.χ., πλοήγηση σε μια συγκεκριμένη σελίδα σε ένα άλλο έγγραφο)
link.Action = new GoToRemoteAction(@"YOUR_DOCUMENT_DIRECTORY\TargetDocument.pdf", 2);
```

#### Βήμα 3: Προσθήκη σχολίου

Προσθέστε τη σχολίαση στη σελίδα που επιθυμείτε:

```csharp
// Υποθέτοντας ότι το 'σελίδες' είναι μια συλλογή σελίδων
page.Annotations.Add(link);
```

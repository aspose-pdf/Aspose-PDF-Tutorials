## Οδηγός Εφαρμογής

### Ρύθμιση μορφοποίησης σχολίων ελεύθερου κειμένου

Αυτή η λειτουργία παρουσιάζει τον τρόπο δημιουργίας και προσαρμογής σχολιασμών FreeText χρησιμοποιώντας το Aspose.PDF.

#### Επισκόπηση
Η προσθήκη σχολίων FreeText επιτρέπει στους χρήστες να επισημαίνουν πληροφορίες ή να προσθέτουν σημειώσεις απευθείας σε μια σελίδα PDF, βελτιώνοντας την διαδραστικότητα και την αναγνωσιμότητα.

**1. Ορίστε τον Κατάλογο Εγγράφων**
Καθορίστε πού βρίσκεται το PDF που εισαγάγατε:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

**2. Ανοίξτε το υπάρχον έγγραφο PDF**
Τοποθετήστε το έγγραφο που θέλετε να σχολιάσετε:

```csharp
Document pdfDocument = new Document(dataDir + "/SetFreeTextAnnotationFormatting.pdf");
```

**3. Δημιουργήστε προεπιλεγμένη εμφάνιση για το στυλ και το χρώμα κειμένου**
Ορίστε πώς θα εμφανίζεται το κείμενό σας χρησιμοποιώντας `DefaultAppearance`:

```csharp
DefaultAppearance default_appearance = new DefaultAppearance("Arial", 28, System.Drawing.Color.Red);
```
- **Παράμετροι:**
  - Οικογένεια γραμματοσειράς (π.χ., "Arial")
  - Μέγεθος γραμματοσειράς (π.χ., 28)
  - Χρώμα κειμένου (`System.Drawing.Color.Red`)

**4. Ορισμός Ορίων Ορθογώνιου για Σχολιασμό**
Ορίστε τη θέση και το μέγεθος της σχολίασής σας:

```csharp
Aspose.Pdf.Rectangle rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```
- **Παράμετροι:**
  - Συντεταγμένη Χ κάτω αριστερά (200)
  - Συντεταγμένη Y κάτω αριστερά (400)
  - Συντεταγμένη Χ άνω δεξιά (400)
  - Συντεταγμένη Y άνω δεξιά (600)

**5. Δημιουργία και προσθήκη σχολίων FreeTextAnnotation**
Προσθέστε τη σχολίαση στο έγγραφό σας:

```csharp
FreeTextAnnotation freetext = new FreeTextAnnotation(pdfDocument.Pages[1], rect, default_appearance);
freetext.Contents = "Free Text";
pdfDocument.Pages[1].Annotations.Add(freetext);
```

**6. Αποθηκεύστε το ενημερωμένο έγγραφο**
Καθορίστε πού θα αποθηκεύσετε το σχολιασμένο PDF σας:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/SetFreeTextAnnotationFormatting_out.pdf";
pdfDocument.Save(outputDir);
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι όλες οι διαδρομές έχουν καθοριστεί σωστά.
- Επαληθεύστε ότι το ευρετήριο σελίδας (π.χ. `Pages[1]`) υπάρχει στο έγγραφό σας.

### Φόρτωση και χειρισμός ιδιοτήτων εγγράφου
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα PDF και να τροποποιήσετε τις ιδιότητες μεταδεδομένων του.

**1. Ανοίξτε το υπάρχον έγγραφο PDF**
Τοποθετήστε το έγγραφο που θέλετε να χειριστείτε:

```csharp
Document pdfDocument = new Document(dataDir + "/SamplePDF.pdf");
```

**2. Πρόσβαση και τροποποίηση ιδιοτήτων εγγράφου**
Ανάκτηση και ενημέρωση πληροφοριών εγγράφου:

```csharp
Console.WriteLine("Original Title: " + pdfDocument.Info.Title);
pdfDocument.Info.Author = "Author Name";
pdfDocument.Info.Title = "New Title";
```
- **Παράμετροι:**
  - `Info.Title` και `Info.Author` είναι ιδιότητες που μπορείτε να τροποποιήσετε.

**3. Αποθήκευση αλλαγών σε νέο αρχείο**
Αποθηκεύστε τις αλλαγές σας:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/ModifiedSamplePDF.pdf";
pdfDocument.Save(outputDir);
```

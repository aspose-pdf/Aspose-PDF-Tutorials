# Δημιουργία διαδραστικών PDF με το Aspose.PDF .NET: Υλοποίηση ενεργειών κατάδειξης για βελτιωμένη αλληλεπίδραση

## Εισαγωγή

Στο σημερινό ψηφιακό τοπίο, η βελτίωση της αλληλεπίδρασης των χρηστών στα έγγραφα μπορεί να διαφοροποιήσει το περιεχόμενό σας από τα υπόλοιπα. Είτε δημιουργείτε αναφορές, φόρμες είτε διαδραστικά εγχειρίδια, η προσθήκη διαδραστικότητας σε PDF μπορεί να βελτιώσει σημαντικά την εμπλοκή και την κατανόηση των χρηστών. Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση του Aspose.PDF για .NET για να δημιουργήσετε ένα έγγραφο με κείμενο που ανταποκρίνεται δυναμικά στις ενέργειες του δείκτη του ποντικιού, καθιστώντας το ιδανικό για προγραμματιστές που επιθυμούν να δημιουργήσουν πιο ελκυστικά PDF.

**Τι θα μάθετε:**
- Πώς να δημιουργήσετε ένα έγγραφο PDF με ένα αρχικό μπλοκ κειμένου
- Τεχνικές για τη φόρτωση και τροποποίηση υπαρχόντων εγγράφων με την προσθήκη κρυφών πεδίων κειμένου
- Μέθοδοι για τη βελτίωση της διαδραστικότητας με αόρατα κουμπιά που ενεργοποιούν αλλαγές ορατότητας κατά την τοποθέτηση του ποντικιού

Καθώς προχωράμε σε αυτόν τον οδηγό, θα μάθετε πώς αυτές οι λειτουργίες μπορούν να ενσωματωθούν απρόσκοπτα στις εφαρμογές .NET σας. Ας εμβαθύνουμε στις προϋποθέσεις πριν ξεκινήσουμε.

## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Δημιουργία εγγράφου με κείμενο

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα νέο έγγραφο PDF που περιέχει ένα αρχικό μπλοκ κειμένου που θέτει τις βάσεις για περαιτέρω βελτιώσεις στην διαδραστικότητα.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργία και αποθήκευση νέου εγγράφου**
```csharp
using Aspose.Pdf;

// Δημιουργία δείγματος εγγράφου με κείμενο
Document doc = new Document();
doc.Pages.Add().Paragraphs.Add(new TextFragment("Move the mouse cursor here to display floating text"));
doc.Save(@"YOUR_DOCUMENT_DIRECTORY\TextBlock_HideShow_MouseOverOut_out.pdf");
```

- **Εξήγηση:** Ξεκινάμε δημιουργώντας ένα `Document` αντικείμενο και προσθήκη σελίδας. A `TextFragment` Στη συνέχεια, προστίθεται σε αυτήν τη σελίδα, το οποίο περιέχει οδηγίες για την αλληλεπίδραση του χρήστη.

### Λειτουργία 2: Φόρτωση εγγράφου και δημιουργία κρυφού πεδίου κειμένου

#### Επισκόπηση
Αυτή η λειτουργία περιλαμβάνει τη φόρτωση του εγγράφου που δημιουργήθηκε προηγουμένως, τον εντοπισμό συγκεκριμένου κειμένου και την ενσωμάτωση ενός κρυφού πεδίου κειμένου που γίνεται ορατό όταν τοποθετήσετε τον δείκτη του ποντικιού.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση υπάρχοντος εγγράφου**
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
using Aspose.Pdf.Forms;
using System.Drawing;

// Άνοιγμα του προηγουμένως αποθηκευμένου εγγράφου με κείμενο
Document document = new Document(@"YOUR_DOCUMENT_DIRECTORY\TextBlock_HideShow_MouseOverOut_out.pdf");
```

**2. Εύρεση κειμένου και προσθήκη κρυφού πεδίου**
```csharp
// Δημιουργήστε ένα αντικείμενο TextAbsorber για να βρείτε όλες τις φράσεις που ταιριάζουν με το καθορισμένο κείμενο
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Move the mouse cursor here to display floating text");
document.Pages.Accept(absorber);
TextFragmentCollection textFragments = absorber.TextFragments;
TextFragment fragment = textFragments[1];

// Δημιουργία κρυφού πεδίου κειμένου για αιωρούμενο κείμενο στο καθορισμένο ορθογώνιο της σελίδας
TextBoxField floatingField = new TextBoxField(fragment.Page, new Rectangle(100, 700, 220, 740));
floatingField.Value = "This is the \"floating text field\".";
floatingField.ReadOnly = true;
floatingField.Flags |= AnnotationFlags.Hidden;
floatingField.PartialName = "FloatingField_1";

// Προσαρμόστε την εμφάνιση του αιωρούμενου πεδίου
floatingField.DefaultAppearance = new DefaultAppearance("Helv", 10, Color.Blue);
floatingField.Characteristics.Background = Color.LightBlue;
floatingField.Characteristics.Border = Color.DarkBlue;
floatingField.Border = new Border(floatingField);
floatingField.Border.Width = 1;
floatingField.Multiline = true;

// Προσθήκη πεδίου κειμένου στη φόρμα του εγγράφου
document.Form.Add(floatingField);
```

- **Εξήγηση:** Εντοπίζουμε συγκεκριμένο κείμενο χρησιμοποιώντας `TextFragmentAbsorber` και δημιουργήστε ένα κρυφό `TextBoxField`Αυτό το πεδίο έχει διαμορφωθεί με προσαρμοσμένα στυλ, διασφαλίζοντας ότι παραμένει αόρατο μέχρι να ενεργοποιηθεί από την αλληλεπίδραση του χρήστη.

### Χαρακτηριστικό 3: Προσθήκη αόρατου κουμπιού με ενέργειες

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει την προσθήκη ενός αόρατου κουμπιού που εναλλάσσει την ορατότητα του πεδίου κειμένου στις ενέργειες κατά την τοποθέτηση του δείκτη του ποντικιού.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργία και ρύθμιση παραμέτρων αόρατου κουμπιού**
```csharp
using Aspose.Pdf.Forms;
using System.Drawing;

// Δημιουργήστε ένα αόρατο κουμπί στη θέση του τμήματος κειμένου
ButtonField buttonField = new ButtonField(fragment.Page, fragment.Rectangle);

// Προσθήκη ενεργειών για εμφάνιση/απόκρυψη του αιωρούμενου πεδίου όταν το ποντίκι εισέρχεται/εξέρχεται από την περιοχή κουμπιών
buttonField.Actions.OnEnter = new HideAction(floatingField, false); // Εμφάνιση πεδίου με την εισαγωγή του ποντικιού
buttonField.Actions.OnExit = new HideAction(floatingField); // Απόκρυψη πεδίου κατά την έξοδο από το ποντίκι

// Προσθήκη του πεδίου κουμπιού στη φόρμα του εγγράφου
document.Form.Add(buttonField);
```

- **Εξήγηση:** Ο `ButtonField` τοποθετείται στη θέση του τμήματος κειμένου. Ορίζουμε ενέργειες χρησιμοποιώντας `HideAction` για τον έλεγχο της ορατότητας του αιωρούμενου κειμένου, ενισχύοντας την διαδραστικότητα.

### Αποθήκευση αλλαγών

## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε διαχειρίσιμα βήματα.

### Χαρακτηριστικό 1: Διαμόρφωση πληροφοριών γραμμής

#### Επισκόπηση
Σε αυτήν την ενότητα, ρυθμίζουμε τις ιδιότητες μιας γραμμής, όπως τις συντεταγμένες των κορυφών, την ορατότητα, το χρώμα και το πλάτος χρησιμοποιώντας το `LineInfo` τάξη.

**Βήμα 1**: Ορισμός ιδιοτήτων γραμμής
```csharp
// Ρύθμιση παραμέτρων συντεταγμένων κορυφής για τη γραμμή
LineInfo lineInfo = new LineInfo();
lineInfo.VerticeCoordinate = new float[] { 55, 55, 70, 70, 70, 90, 150, 60 };
lineInfo.Visibility = true;
lineInfo.LineColor = System.Drawing.Color.Red;
lineInfo.LineWidth = 2;

// Υπολογίστε τον αριθμό των σημείων από τις συντεταγμένες των κορυφών
int length = lineInfo.VerticeCoordinate.Length / 2;
Aspose.Pdf.Point[] gesture = new Aspose.Pdf.Point[length];
for (int i = 0; i < length; i++) {
    // Δημιουργήστε ένα σημείο για κάθε ζεύγος συντεταγμένων
    gesture[i] = new Aspose.Pdf.Point(lineInfo.VerticeCoordinate[2 * i], lineInfo.VerticeCoordinate[2 * i + 1]);
}
```

**Εξήγηση**: 
- Ο `VerticeCoordinate` Ο πίνακας ορίζει τη διαδρομή της γραμμής χρησιμοποιώντας συντεταγμένες x και y.
- Ορίσαμε τη γραμμή να είναι ορατή, χρωματισμένη κόκκινη και με πλάτος 2 μονάδες.

### Χαρακτηριστικό 2: Δημιουργία και διαμόρφωση σχολίων μελανιού

#### Επισκόπηση
Στη συνέχεια, δημιουργούμε μια σχολίαση μελανιού που χρησιμοποιεί τη διαμορφωμένη γραμμή μας ως γραφικό στοιχείο.

**Βήμα 2**: Δημιουργία και ρύθμιση παραμέτρων σχολίων μελανιού
```csharp
using Aspose.Pdf.Annotations;

IList<Point[]> inkList = new List<Point[]>();
inkList.Add(gesture);

Document doc = new Document();
doc.Pages.Add();

InkAnnotation a1 = new InkAnnotation(doc.Pages[1], new Aspose.Pdf.Rectangle(100, 100, 300, 300), inkList);
a1.Subject = "Test";
a1.Title = "Title";
a1.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);

doc.Pages[1].Annotations.Add(a1);
```

**Εξήγηση**: 
- Προσθέτουμε τη διαμόρφωση γραμμής σε μια νέα σχολίαση μελανιού.
- Ορίστε ιδιότητες όπως θέμα, τίτλο και χρώμα.

### Χαρακτηριστικό 3: Διαμόρφωση περιγράμματος σχολίων

#### Επισκόπηση
Προσαρμόστε τα σχόλιά σας με συγκεκριμένα στυλ περιγράμματος για βελτιωμένη οπτική ελκυστικότητα.

**Βήμα 3**: Ρύθμιση παραμέτρων περιγραμμάτων σχολίων
```csharp
using Aspose.Pdf.Facades;

Border border = new Border(a1);
border.Width = 3;
border.Effect = BorderEffect.Cloudy;
border.Dash = new Dash(1, 1); // 1 μονάδα ενεργοποιημένη, 1 μονάδα απενεργοποιημένη, μοτίβο ταμπλό
border.Style = BorderStyle.Solid;
```

**Εξήγηση**: 
- Εφαρμόζουμε ένα συμπαγές περίγραμμα με εφέ "θολού" και συγκεκριμένα μοτίβα παύλας.

Τέλος, αποθηκεύστε το έγγραφο:

```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/lnkAnnotationLineWidth_out.pdf";
doc.Save(dataDir);
```

## Οδηγός Εφαρμογής

### Πρόσβαση και επιλογή κουμπιών ραδιοφώνου

#### Επισκόπηση
Μάθετε πώς να αποκτάτε πρόσβαση σε ομάδες κουμπιών επιλογής μέσα στα έγγραφα PDF σας και να επιλέγετε επιλογές μέσω προγραμματισμού.

#### Οδηγίες βήμα προς βήμα

**Αποκτήστε πρόσβαση στο RadioButtonField:**
```csharp
// Ανάκτηση του πεδίου του κουμπιού επιλογής με το όνομά του
RadioButtonField radioButtonField = document.Form["radio"] as RadioButtonField;
```
*Εξήγηση*: Χρήση `document.Form` για να αποκτήσετε πρόσβαση στα πεδία της φόρμας. Θα χρειαστείτε το όνομα του πεδίου, το οποίο μπορείτε να βρείτε χρησιμοποιώντας προγράμματα επεξεργασίας PDF όπως το Adobe Acrobat.

**Επιλέξτε μια συγκεκριμένη επιλογή:**
```csharp
// Επιλέξτε την τρίτη επιλογή (ο δείκτης ξεκινά από το 0)
radioButtonField.Selected = 2;
```
*Εξήγηση*Οι επιλογές των κουμπιών επιλογής έχουν μηδενικό δείκτη. Εδώ, η επιλογή δείκτη `2` επιλέγει την τρίτη επιλογή στην ομάδα.

#### Αποθήκευση αλλαγών
Μετά τις τροποποιήσεις, αποθηκεύστε το έγγραφό σας:
```csharp
// Ορίστε τη διαδρομή εξόδου και αποθηκεύστε το τροποποιημένο PDF
document.Save("YOUR_OUTPUT_DIRECTORY/SelectRadioButton_out.pdf");
```

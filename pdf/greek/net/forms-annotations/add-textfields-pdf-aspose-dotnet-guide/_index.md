## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην προσθήκη πεδίων κειμένου σε υπάρχουσες φόρμες PDF χρησιμοποιώντας το Aspose.PDF για .NET. Κάθε λειτουργία αναλύεται σε σαφή βήματα.

### Προσδιορισμός πεδίων φόρμας

#### Επισκόπηση:
Αρχικά, πρέπει να εντοπίσουμε και να εξαγάγουμε τα πεδία της φόρμας από ένα υπάρχον έγγραφο PDF.

**Βήμα 1: Φόρτωση του εγγράφου PDF**
```csharp
string dataDir = "path/to/your/pdf/";
Aspose.Pdf.Facades.Form form = new Aspose.Pdf.Facades.Form(dataDir + "FilledForm.pdf");
```

**Βήμα 2: Ανάκτηση όλων των ονομάτων πεδίων**
```csharp
String[] allfields = form.FieldNames;
```

#### Εξήγηση:
Αυτός ο κώδικας φορτώνει ένα PDF και ανακτά τα ονόματα όλων των πεδίων, παρέχοντας μια βάση για περαιτέρω τροποποίηση.

### Προσθήκη νέων πεδίων κειμένου

#### Επισκόπηση:
Τώρα που έχουμε εντοπίσει τα υπάρχοντα πεδία φόρμας, ας προσθέσουμε νέα πεδία κειμένου σε αυτές τις θέσεις.

**Βήμα 1: Προετοιμασία συντεταγμένων**
```csharp
System.Drawing.Rectangle[] box = new System.Drawing.Rectangle[allfields.Length];
for (int i = 0; i < allfields.Length; i++)
{
    FormFieldFacade facade = form.GetFieldFacade(allfields[i]);
    box[i] = facade.Box;
}
```

**Βήμα 2: Δημιουργία και προσθήκη νέων πεδίων κειμένου**
```csharp
Document doc = new Document(dataDir + "FilledForm - 2.pdf");
FormEditor editor = new FormEditor(doc);

for (int i = 0; i < allfields.Length; i++)
{
    editor.AddField(FieldType.Text, "TextField" + i, allfields[i], 1,
                    box[i].Left, box[i].Top, box[i].Left + 50, box[i].Top + 10);
}
editor.Save(dataDir + "IdentifyFormFields_out.pdf");
```

#### Εξήγηση:
Αυτό το απόσπασμα προσθέτει νέα πεδία κειμένου ακριβώς πάνω από κάθε υπάρχον πεδίο φόρμας χρησιμοποιώντας τις συντεταγμένες που ελήφθησαν νωρίτερα.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Αναντιστοιχία συντεταγμένων**Βεβαιωθείτε ότι οι τιμές συντεταγμένων του ορθογωνίου έχουν υπολογιστεί σωστά.

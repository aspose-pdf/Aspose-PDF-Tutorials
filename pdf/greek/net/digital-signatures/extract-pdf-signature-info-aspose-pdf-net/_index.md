## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Εξαγωγή πληροφοριών υπογραφής

Αυτή η λειτουργία εστιάζει στην εξαγωγή πληροφοριών πιστοποιητικού από πεδία υπογραφής PDF για την επαλήθευση της αυθεντικότητας και της ακεραιότητας του εγγράφου.

#### Βήμα 1: Ορισμός διαδρομής εγγράφου και φόρτωση PDF

Καθορίστε τη διαδρομή καταλόγου για το έγγραφο PDF και φορτώστε το:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string input = dataDir + "ExtractSignatureInfo.pdf";

using (Document pdfDocument = new Document(input))
{
    // Συνέχεια επεξεργασίας...
}
```
**Εξήγηση**: Το `dataDir` Η μεταβλητή περιέχει τη διαδρομή προς τα έγγραφά σας. Η `input` Η συμβολοσειρά κατασκευάζει την πλήρη διαδρομή αρχείου για φόρτωση.

#### Βήμα 2: Επαναλάβετε τα πεδία φόρμας

Περιηγηθείτε σε κάθε πεδίο στη φόρμα PDF για να βρείτε πεδία υπογραφής:

```csharp
foreach (Field field in pdfDocument.Form)
{
    SignatureField sf = field as SignatureField;
    if (sf != null)
    {
        // Πεδίο υπογραφής διεργασίας
    }
}
```
**Εξήγηση**Χρησιμοποιούμε ένα `foreach` επανάληψη σε κάθε βρόχο `Field` αντικείμενο. Το `as` η λέξη-κλειδί ελέγχει αν το πεδίο είναι `SignatureField`.

#### Βήμα 3: Εξαγωγή ροής πιστοποιητικών

Εξαγωγή και ανάγνωση της ροής πιστοποιητικών από το πεδίο υπογραφής:

```csharp
Stream cerStream = sf.ExtractCertificate();
if (cerStream != null)
{
    byte[] bytes = new byte[cerStream.Length];
    using (cerStream)
    {
        cerStream.Read(bytes, 0, bytes.Length);
    }

    // Αποθήκευση πιστοποιητικού σε αρχείο
    string outputCerPath = dataDir + "input.cer";
    using (FileStream fs = new FileStream(outputCerPath, FileMode.CreateNew))
    {
        fs.Write(bytes, 0, bytes.Length);
    }
}
```
**Εξήγηση**: Το `ExtractCertificate()` Η μέθοδος ανακτά τη ροή πιστοποιητικών. Διαβάζουμε αυτήν τη ροή σε έναν πίνακα byte και την αποθηκεύουμε ως `.cer` αρχείο.

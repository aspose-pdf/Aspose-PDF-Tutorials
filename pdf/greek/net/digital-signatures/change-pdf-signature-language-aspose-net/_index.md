# Πώς να αλλάξετε τη γλώσσα υπογραφής PDF με το Aspose.PDF για .NET

Αυτό το σεμινάριο θα σας καθοδηγήσει στην αλλαγή των ρυθμίσεων γλώσσας των ψηφιακών υπογραφών σε PDF με το Aspose.PDF για .NET, διασφαλίζοντας ότι τα έγγραφά σας πληρούν τα διεθνή πρότυπα και απευθύνονται σε παγκόσμιο κοινό.

## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία αλλαγής των ρυθμίσεων γλώσσας για τις ψηφιακές υπογραφές σε διαχειρίσιμα βήματα.

### Ρύθμιση εμφάνισης υπογραφής

#### Επισκόπηση
Θα διαμορφώσουμε τον τρόπο εμφάνισης της υπογραφής στο PDF σας, εστιάζοντας στην προσαρμογή στοιχείων κειμένου όπως η ημερομηνία, η αιτία και η τοποθεσία ώστε να ταιριάζουν σε διαφορετικές γλώσσες.

**Φόρτωση του εγγράφου σας**
Αρχικά, δημιουργήστε μια παρουσία του `PdfFileSignature` και συνδέστε το με το PDF εισόδου σας:

```csharp
using (Aspose.Pdf.Facades.PdfFileSignature pdfSign = new Aspose.Pdf.Facades.PdfFileSignature())
{
    pdfSign.BindPdf("input.pdf");
}
```

**Ορισμός ορθογωνίου υπογραφής**
Προσδιορίστε την περιοχή στη σελίδα όπου θα εμφανίζεται η υπογραφή:

```csharp
System.Drawing.Rectangle rect = new System.Drawing.Rectangle(310, 45, 200, 50);
```

### Ρύθμιση παραμέτρων λεπτομερειών υπογραφής

#### Επισκόπηση
Ορίστε διάφορες παραμέτρους όπως τον λόγο και την τοποθεσία για τις ψηφιακές υπογραφές σας. Προσαρμόστε τις ώστε να αντικατοπτρίζονται σε οποιαδήποτε γλώσσα.

**Δημιουργία αντικειμένου υπογραφής PKCS#7**
Δημιουργήστε ένα υπόδειγμα `PKCS7` αντικείμενο με τα απαραίτητα στοιχεία πιστοποιητικού:

```csharp
PKCS7 pkcs = new PKCS7("certificate.pfx", "12345");
pkcs.Reason = "Pruebas Firma";
pkcs.ContactInfo = "Contacto Pruebas";
pkcs.Location = "Población (Provincia)";
pkcs.Date = DateTime.Now;
```

**Προσαρμόστε την εμφάνιση της υπογραφής**
Χρησιμοποιώ `SignatureCustomAppearance` για να προσαρμόσετε τις ιδιότητες κειμένου και τις ρυθμίσεις γλώσσας:

```csharp
SignatureCustomAppearance signatureCustomAppearance = new SignatureCustomAppearance();
signatureCustomAppearance.DateSignedAtLabel = "Fecha";
signatureCustomAppearance.DigitalSignedLabel = "Digitalmente firmado por";
signatureCustomAppearance.ReasonLabel = "Razón";
signatureCustomAppearance.LocationLabel = "Localización";
signatureCustomAppearance.FontFamilyName = "Arial";
signatureCustomAppearance.FontSize = 10d;
signatureCustomAppearance.Culture = CultureInfo.InvariantCulture;
signatureCustomAppearance.DateTimeFormat = "yyyy.MM.dd HH:mm:ss";

pkcs.CustomAppearance = signatureCustomAppearance;
```

### Υπογραφή του PDF
**Εφαρμογή υπογραφής**
Χρησιμοποιήστε το `Sign` μέθοδος για την εφαρμογή της διαμορφωμένης υπογραφής σας:

```csharp
pdfSign.Sign(1, true, rect, pkcs);
```

### Αποθήκευση του αρχείου εξόδου
**Αποθήκευση υπογεγραμμένου εγγράφου**
Τέλος, αποθηκεύστε το υπογεγραμμένο PDF σας με τις νέες ρυθμίσεις γλώσσας που έχουν εφαρμοστεί:

```csharp
pdfSign.Save("output.pdf");
```

## Οδηγός Εφαρμογής

Τώρα που είστε έτοιμοι, ας εφαρμόσουμε την ψηφιακή υπογραφή.

### Δυνατότητα: Ψηφιακή υπογραφή PDF με προσαρμοσμένη εμφάνιση

Αυτή η λειτουργία επιτρέπει την προσαρμογή του τρόπου εμφάνισης της υπογραφής σας σε PDF. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Αρχικοποίηση αντικειμένου PdfFileSignature

Δημιουργήστε μια παρουσία του `PdfFileSignature` να δέσει και να υπογράψει το έγγραφο.

```csharp
using (PdfFileSignature pdfSign = new PdfFileSignature()) {
    // Δέστε το αρχείο PDF που θέλετε να υπογράψετε
    pdfSign.BindPdf("input.pdf");
}
```

#### Βήμα 2: Ορισμός τοποθεσίας υπογραφής

Δημιουργήστε ένα ορθογώνιο που καθορίζει πού θα εμφανίζεται η υπογραφή σας.

```csharp
Rectangle rect = new Rectangle(310, 45, 200, 50);
```

#### Βήμα 3: Αρχικοποίηση αντικειμένου PKCS7

Χρησιμοποιήστε το αρχείο PFX και τον κωδικό πρόσβασής σας για να αρχικοποιήσετε ένα `PKCS7` αντικείμενο. Αυτό αντιπροσωπεύει το ψηφιακό πιστοποιητικό για υπογραφή.

```csharp
PKCS7 pkcs = new PKCS7("SampleCertificate.pfx", "idsrv3test");
```

#### Βήμα 4: Προσαρμόστε την εμφάνιση της υπογραφής

Προσαρμόστε την εμφάνιση της υπογραφής σας χρησιμοποιώντας `SignatureCustomAppearance`.

```csharp
SignatureCustomAppearance signatureCustomAppearance = new SignatureCustomAppearance();
signatureCustomAppearance.FontSize = 6;
signatureCustomAppearance.FontFamilyName = "Times New Roman";
signatureCustomAppearance.DigitalSignedLabel = "Signed by me";
pkcs.CustomAppearance = signatureCustomAppearance;
```

#### Βήμα 5: Υπογράψτε το PDF

Εφαρμόστε την ψηφιακή σας υπογραφή στο έγγραφο χρησιμοποιώντας το `Sign` μέθοδος.

```csharp
pdfSign.Sign(1, true, rect, pkcs);
pdfSign.Save("output.pdf");
```

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το αρχείο PFX και ο κωδικός πρόσβασής σας είναι σωστά.
- Βεβαιωθείτε ότι έχετε δικαιώματα ανάγνωσης/εγγραφής των αρχείων PDF που εμπλέκονται.

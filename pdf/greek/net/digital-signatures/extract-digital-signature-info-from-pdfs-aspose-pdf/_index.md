## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε τη διαδικασία εξαγωγής πληροφοριών ψηφιακής υπογραφής από ένα έγγραφο PDF.

### Επισκόπηση

Η εργασία είναι η εξαγωγή και αποθήκευση πληροφοριών πιστοποιητικού που σχετίζονται με ψηφιακές υπογραφές σε ένα PDF χρησιμοποιώντας το Aspose.PDF. `PdfFileSignature` τάξη.

### Βήμα προς βήμα εφαρμογή

#### **1. Εισαγωγή απαιτούμενων χώρων ονομάτων**

Ξεκινήστε εισάγοντας τους απαραίτητους χώρους ονομάτων:

```csharp
using System.IO;
using Aspose.Pdf.Facades;
```

Αυτοί οι χώροι ονομάτων παρέχουν πρόσβαση στη διαχείριση αρχείων και στις λειτουργίες χειρισμού PDF που χρειαζόμαστε.

#### **2. Ορίστε την κύρια κλάση και τη μέθοδο σας**

Δημιουργήστε μια τάξη `ExtractSignatureInfo` με μια στατική μέθοδο `Run()` όπου θα εφαρμόσετε τη λογική σας:

```csharp
public class ExtractSignatureInfo
{
    public static void Run()
    {
        // Η εφαρμογή θα γίνει εδώ
    }
}
```

#### **3. Φορτώστε το έγγραφο PDF**

Χρήση `PdfFileSignature` για να φορτώσετε το αρχείο PDF που θέλετε να επεξεργαστείτε:

```csharp
string dataDir = "PathToYourDataDirectory";
string inputFilePath = Path.Combine(dataDir, "DigitallySign.pdf");

using (PdfFileSignature pdfFileSignature = new PdfFileSignature())
{
    pdfFileSignature.BindPdf(inputFilePath);
```

`BindPdf` συνδέει το έγγραφο PDF με το `PdfFileSignature` αντικείμενο.

#### **4. Ανάκτηση Ονομάτων Υπογραφής**

Ελέγξτε για υπογραφές μέσα στο PDF:

```csharp
IList<string> sigNames = pdfFileSignature.GetSignNames();
if (sigNames.Count > 0)
{
    // Έχουμε τουλάχιστον μία υπογραφή προς επεξεργασία
}
```

Αυτό το βήμα ανακτά όλα τα ονόματα ψηφιακών υπογραφών που υπάρχουν στο έγγραφο.

#### **5. Εξαγωγή πληροφοριών πιστοποιητικού**

Εξαγωγή και αποθήκευση δεδομένων πιστοποιητικού για κάθε υπογραφή:

```csharp
string sigName = sigNames[0] as string;
if (!string.IsNullOrEmpty(sigName))
{
    using (Stream cerStream = pdfFileSignature.ExtractCertificate(sigName))
    {
        if (cerStream != null)
        {
            byte[] bytes = new byte[cerStream.Length];
            cerStream.Read(bytes, 0, (int)cerStream.Length);
            
            string outputFilePath = Path.Combine(dataDir, "ExtractedCertificate.pfx");
            using (FileStream fs = new FileStream(outputFilePath, FileMode.CreateNew))
            {
                fs.Write(bytes, 0, bytes.Length);
            }
        }
    }
}
```

Ο `ExtractCertificate` Η μέθοδος ανακτά το πιστοποιητικό που σχετίζεται με μια υπογραφή και το αποθηκεύει στο δίσκο.

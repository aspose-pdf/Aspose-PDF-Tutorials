## Οδηγός Εφαρμογής

Ας αναλύσουμε κάθε χαρακτηριστικό βήμα προς βήμα.

### Χαρακτηριστικό 1: Αντιγραφή αρχείου και δημιουργία υπογραφής

#### Επισκόπηση

Αυτή η λειτουργία επιτρέπει την αντιγραφή ενός αρχείου PDF για τη δημιουργία ενός νέου εγγράφου για υπογραφή και, στη συνέχεια, προσθέτει ένα πεδίο υπογραφής.

#### Βήματα

##### **Βήμα 1: Αντιγράψτε το PDF**

Ξεκινήστε αντιγράφοντας ένα υπάρχον PDF που θα χρησιμεύσει ως βασικό σας έγγραφο:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
File.Copy(dataDir + "/blank.pdf", dataDir + "/externalSignature1.pdf", true);
```

**Γιατί:** Αυτό διασφαλίζει ότι έχετε ένα νέο σημείο εκκίνησης χωρίς να τροποποιήσετε το αρχικό αρχείο.

##### **Βήμα 2: Δημιουργία και προσθήκη πεδίου υπογραφής**

Φορτώστε το αντιγραμμένο PDF και προσθέστε ένα πεδίο υπογραφής:

```csharp
using (FileStream fs = new FileStream(dataDir + "/externalSignature1.pdf", FileMode.Open, FileAccess.ReadWrite))
{
    Document doc = new Document(fs);
    Rectangle rect = new Rectangle(100, 400, 200, 450); // Ορίστε τη θέση και το μέγεθος για το πεδίο υπογραφής.
    SignatureField field1 = new SignatureField(doc.Pages[1], rect);
    doc.Form.Add(field1, 1);
}
```

**Γιατί:** Το πεδίο υπογραφής καθορίζει πού θα τοποθετούν οι χρήστες τις ψηφιακές τους υπογραφές.

### Λειτουργία 2: Υπογραφή εγγράφου PDF

#### Επισκόπηση

Αυτή η ενότητα καλύπτει τον τρόπο υπογραφής του PDF που δημιουργήθηκε προηγουμένως χρησιμοποιώντας ένα πιστοποιητικό από το χώρο αποθήκευσης πιστοποιητικών των Windows.

#### Βήματα

##### **Βήμα 3: Πιστοποιητικά πρόσβασης**

Ανοίξτε το χώρο αποθήκευσης πιστοποιητικών και επιλέξτε ένα πιστοποιητικό:

```csharp
X509Store store = new X509Store(StoreLocation.CurrentUser);
store.Open(OpenFlags.ReadOnly);
X509Certificate2Collection sel = X509Certificate2UI.SelectFromCollection(store.Certificates, null, "Select a Certificate:", SelectionFlag.SingleSelection);
```

**Γιατί:** Τα πιστοποιητικά παρέχουν τα απαραίτητα διαπιστευτήρια για την υπογραφή εγγράφων.

##### **Βήμα 4: Υπογράψτε το PDF**

Δημιουργήστε μια εξωτερική υπογραφή και εφαρμόστε την στο έγγραφό σας:

```csharp
using (FileStream fs = new FileStream(dataDir + "/externalSignature1.pdf", FileMode.Open, FileAccess.ReadWrite))
{
    Document doc = new Document(fs);
    SignatureField field1 = (SignatureField)doc.Form["sig1"];
    
    Aspose.Pdf.Forms.ExternalSignature externalSignature = new Aspose.Pdf.Forms.ExternalSignature(sel[0])
    {
        Authority = "Me",
        Reason = "Reason",
        ContactInfo = "Contact"
    };

    field1.Sign(externalSignature);
    doc.Save();
}
```

**Γιατί:** Αυτό το βήμα εφαρμόζει την ψηφιακή σας υπογραφή, επαληθεύοντας την αυθεντικοποίηση του εγγράφου.

### Λειτουργία 3: Επαλήθευση υπογεγραμμένου εγγράφου PDF

#### Επισκόπηση

Διασφαλίστε την ακεραιότητα των υπογεγραμμένων εγγράφων επαληθεύοντας τις υπογραφές τους.

#### Βήματα

##### **Βήμα 5: Επαλήθευση υπογραφών**

Ελέγξτε όλες τις υπογραφές στο PDF για να επιβεβαιώσετε την εγκυρότητά τους:

```csharp
using (PdfFileSignature pdfSign = new PdfFileSignature(new Document(dataDir + "/externalSignature1.pdf")))
{
    IList<string> sigNames = pdfSign.GetSignNames();
    foreach (string sigName in sigNames)
    {
        bool isVerified = pdfSign.VerifySigned(sigName) && pdfSign.VerifySignature(sigName);
        if (!isVerified)
        {
            throw new ApplicationException("Not verified");
        }
    }
}
```

**Γιατί:** Η επαλήθευση διασφαλίζει ότι το έγγραφο δεν έχει τροποποιηθεί από την υπογραφή του.

## Οδηγός Εφαρμογής
### Επαλήθευση ψηφιακής υπογραφής
Η επαλήθευση μιας ψηφιακής υπογραφής επιβεβαιώνει ότι το έγγραφο δεν έχει τροποποιηθεί από την υπογραφή του, διασφαλίζοντας την αυθεντικότητα και την ακεραιότητα.

#### Βήμα 1: Ρύθμιση του περιβάλλοντος σας
Βεβαιωθείτε ότι το περιβάλλον σας είναι έτοιμο με όλες τις απαραίτητες βιβλιοθήκες εγκατεστημένες:
```csharp
using Aspose.Pdf.Facades;
```

#### Βήμα 2: Δημιουργήστε μια μέθοδο για την επαλήθευση της υπογραφής
Χρησιμοποιήστε αυτό το απόσπασμα κώδικα για να δημιουργήσετε μια μέθοδο που επαληθεύει τις ψηφιακές υπογραφές σε PDF:
```csharp
class VerifyValidSignature
{
    public static void Run()
    {
        try
        {
            // Ορίστε τη διαδρομή καταλόγου για τα έγγραφά σας
            string dataDir = "path_to_your_documents_directory";
            
            // Δημιουργήστε μια παρουσία του PdfFileSignature
            PdfFileSignature pdfSign = new PdfFileSignature();
            
            // Δέστε το έγγραφο PDF
            pdfSign.BindPdf(dataDir + "DigitallySign.pdf");
            
            // Επαληθεύστε την υπογραφή
            if (pdfSign.VerifySignature("Signature1"))
            {
                Console.WriteLine("Signature Verified");
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine(ex.Message);
        }
    }       
}
```

#### Επεξήγηση παραμέτρων κώδικα
- `dataDir`: Κατάλογος όπου αποθηκεύονται τα αρχεία PDF σας.
- `PdfFileSignature pdfSign = new PdfFileSignature()`: Αρχικοποιεί μια νέα παρουσία για τον χειρισμό υπογραφών PDF.
- `pdfSign.BindPdf(dataDir + "DigitallySign.pdf")`: Συνδέει το αρχείο PDF που θα υποβληθεί σε επεξεργασία.
- `pdfSign.VerifySignature("Signature1")`Ελέγχει αν η καθορισμένη υπογραφή είναι έγκυρη.

### Φόρτωση Πηγαίου PDF Χαρτοφυλάκιο (H2)
Αρχικά, φορτώστε το έγγραφο PDF που περιέχει τα ενσωματωμένα αρχεία σας:
```csharp

// Φόρτωση χαρτοφυλακίου PDF πηγής

```

### Πρόσβαση σε ενσωματωμένα αρχεία (H2)
Στη συνέχεια, αποκτήστε πρόσβαση και επαναλάβετε τα ενσωματωμένα αρχεία:
```csharp
// Λήψη συλλογής ενσωματωμένων αρχείων
EmbeddedFileCollection embeddedFiles = pdfDocument.EmbeddedFiles;

// Επαναλάβετε μέσω μεμονωμένου αρχείου του χαρτοφυλακίου
foreach (FileSpecification fileSpecification in embeddedFiles)
{
    // Εξαγωγή περιεχομένου και αποθήκευση σε αρχείο ή ροή
    byte[] fileContent = new byte[fileSpecification.Contents.Length];
    fileSpecification.Contents.Read(fileContent, 0, fileContent.Length);
    
    string filename = Path.GetFileName(fileSpecification.Name);

    // Αποθηκεύστε το εξαγόμενο αρχείο σε κάποια τοποθεσία
    using (FileStream fileStream = new FileStream(dataDir + "_out" + filename, FileMode.Create))
    {
        fileStream.Write(fileContent, 0, fileContent.Length);
    }
}
```
#### Εξήγηση
- **Συλλογή Ενσωματωμένων Αρχείων:** Παρέχει πρόσβαση σε όλα τα ενσωματωμένα αρχεία στο PDF.
- **Προδιαγραφές Αρχείου:** Αντιπροσωπεύει κάθε αρχείο μέσα στη συλλογή. Το `Contents` Η ιδιότητα σάς επιτρέπει να διαβάζετε και να εξάγετε δεδομένα.

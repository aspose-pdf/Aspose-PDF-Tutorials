## Οδηγός Εφαρμογής
### Άνοιγμα εγγράφου PDF
**Επισκόπηση:** Ανοίξτε ένα υπάρχον έγγραφο PDF για να αποκτήσετε πρόσβαση στις σελίδες και τις ιδιότητές του.

#### Βήμα 1: Ορισμός διαδρομής αρχείου
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/GetAlltheAttachments.pdf";
```

#### Βήμα 2: Ανοίξτε το έγγραφο
```csharp
document pdfDocument = new Document(dataDir);
// Το έγγραφο είναι πλέον προσβάσιμο για περαιτέρω λειτουργίες.
```
### Πρόσβαση στη συλλογή ενσωματωμένων αρχείων
**Επισκόπηση:** Ανακτήστε όλα τα ενσωματωμένα αρχεία σε ένα PDF για να τα προετοιμάσετε για εξαγωγή.

#### Βήμα 3: Λήψη ενσωματωμένων αρχείων
```csharp
EmbeddedFileCollection embeddedFiles = pdfDocument.EmbeddedFiles;
// Τώρα έχετε πρόσβαση στη συλλογή συνημμένων.
```
### Επανάληψη και εξαγωγή συνημμένων
**Επισκόπηση:** Επαναλάβετε κάθε συνημμένο, εξαγάγετε τις λεπτομέρειες του και αποθηκεύστε το στο δίσκο.

#### Βήμα 4: Συνημμένα με βρόχο
```csharp
int count = 1;
foreach (FileSpecification fileSpecification in embeddedFiles)
{
    string fileName = fileSpecification.Name;
    string description = fileSpecification.Description;
    string mimeType = fileSpecification.MIMEType;

    if (fileSpecification.Params != null)
    {
        // Εξαγωγή πρόσθετων μεταδεδομένων, εάν είναι διαθέσιμα
        string checksum = fileSpecification.Params.CheckSum;
        DateTime creationDate = fileSpecification.Params.CreationDate;
        DateTime modificationDate = fileSpecification.Params.ModDate;
        int size = fileSpecification.Params.Size;
    }

    // Διαβάστε και αποθηκεύστε το περιεχόμενο του συνημμένου
    byte[] fileContent = new byte[fileSpecification.Contents.Length];
    fileSpecification.Contents.Read(fileContent, 0, fileContent.Length);
    string outputFilePath = dataDir + "/YOUR_OUTPUT_DIRECTORY/" + count + "_out.txt";
    File.WriteAllBytes(outputFilePath, fileContent);
    count++;
}
```

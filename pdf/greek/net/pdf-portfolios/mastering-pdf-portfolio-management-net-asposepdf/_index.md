## Οδηγός Εφαρμογής

Θα επικεντρωθούμε σε δύο κύριες λειτουργίες: την αφαίρεση αρχείων από ένα χαρτοφυλάκιο PDF και τη φόρτωση/αποθήκευση εγγράφων PDF.

### Αφαίρεση αρχείων από ένα χαρτοφυλάκιο PDF

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να βελτιστοποιήσετε τα χαρτοφυλάκια PDF σας εξαλείφοντας τα περιττά αρχεία, καθιστώντας τη διαχείριση εγγράφων πιο αποτελεσματική.

#### Βήμα προς βήμα εφαρμογή
**1. Φορτώστε το χαρτοφυλάκιο PDF πηγής**
Ξεκινήστε δημιουργώντας ένα νέο `Document` αντικείμενο με το χαρτοφυλάκιο PDF προορισμού σας:

```csharp
using Aspose.Pdf;

// Φόρτωση πηγαίου κώδικα PDF Χαρτοφυλάκιο
document = new Document("YOUR_DOCUMENT_DIRECTORY/PDFPortfolio.pdf");
```

**2. Αφαίρεση όλων των αρχείων στο χαρτοφυλάκιο**
Για να διαγράψετε όλα τα αρχεία από το χαρτοφυλάκιο, χρησιμοποιήστε το `Delete()` μέθοδος:

```csharp
// Διαγραφή όλων των αρχείων από το χαρτοφυλάκιο
document.Collection.Delete();
```

**3. Αποθηκεύστε το τροποποιημένο έγγραφο**
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο χωρίς περιεχόμενο χαρτοφυλακίου:

```csharp
// Αποθήκευση του τροποποιημένου εγγράφου
document.Save("YOUR_OUTPUT_DIRECTORY/No_PortFolio_out.pdf");
```

### Φόρτωση και αποθήκευση εγγράφων PDF

#### Επισκόπηση
Αυτή η λειτουργία καλύπτει βασικές λειτουργίες για το άνοιγμα υπαρχόντων PDF και την αποθήκευσή τους μετά από τροποποιήσεις.

#### Βήμα προς βήμα εφαρμογή
**1. Καθορισμός διαδρομής και φόρτωση εγγράφου**
Ορίστε τη διαδρομή του αρχείου προέλευσης και, στη συνέχεια, φορτώστε το έγγραφο:

```csharp
string documentPath = "YOUR_DOCUMENT_DIRECTORY/YourPDFDocument.pdf";

// Φόρτωση του εγγράφου PDF
document = new Document(documentPath);
```

**2. Εκτέλεση λειτουργιών (Προαιρετικό)**
Μπορείτε να χειριστείτε το φορτωμένο PDF όπως απαιτείται—αν και για λόγους επίδειξης, θα προχωρήσουμε απευθείας στην αποθήκευση.

**3. Αποθηκεύστε το τροποποιημένο PDF**
Επιλέξτε μια διαδρομή εξόδου και αποθηκεύστε τις αλλαγές σας:

```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/ModifiedPDF.pdf";

// Αποθήκευση του εγγράφου PDF
document.Save(outputPath);
```

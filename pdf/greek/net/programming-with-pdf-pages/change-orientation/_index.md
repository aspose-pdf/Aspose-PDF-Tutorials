Ευτυχώς για εμάς, το Aspose.PDF για .NET παρέχει έναν εύκολο και ισχυρό τρόπο χειρισμού αρχείων PDF με σχεδόν κάθε δυνατό τρόπο—συμπεριλαμβανομένης της αλλαγής του προσανατολισμού των σελίδων σας. Είτε θέλετε να αλλάξετε από κατακόρυφο σε οριζόντιο είτε αντίστροφα, αυτός ο οδηγός θα σας καθοδηγήσει στη διαδικασία βήμα προς βήμα.

## Εισαγωγή χώρων ονομάτων

Πριν μπορέσετε να χειριστείτε τον προσανατολισμό των σελίδων στο PDF σας, θα πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο C# σας. Βεβαιωθείτε ότι έχετε τα εξής:

```csharp
using System.IO;
using Aspose.Pdf;
```

Αφού εισαγάγαμε αυτό, ας προχωρήσουμε στο κύριο μέρος του σεμιναρίου.

## Βήμα 1: Φόρτωση του εγγράφου PDF

Το πρώτο πράγμα που πρέπει να κάνουμε είναι να φορτώσουμε το αρχείο PDF που θέλουμε να τροποποιήσουμε. Μπορείτε να χρησιμοποιήσετε το `Document` κλάση από τον χώρο ονομάτων Aspose.PDF για να ανοίξετε το PDF σας.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "input.pdf");
```

Αυτή η γραμμή φορτώνει το PDF από τον καθορισμένο κατάλογο. Βεβαιωθείτε ότι έχετε αντικαταστήσει `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς το αρχείο σας. Το `"input.pdf"` είναι το PDF του οποίου τον προσανατολισμό θέλετε να αλλάξετε.

## Βήμα 2: Επανάληψη σε κάθε σελίδα

Τώρα που έχουμε φορτώσει το έγγραφο, ας κάνουμε επανάληψη σε κάθε σελίδα του PDF. Θα χρησιμοποιήσουμε ένα `foreach` βρόχο για να διασχίσουμε κάθε σελίδα, επιτρέποντάς μας να εφαρμόσουμε την αλλαγή προσανατολισμού σε όλες.

```csharp
foreach (Page page in doc.Pages)
{
    // Χειρισμός κάθε σελίδας
}
```

Αυτός ο βρόχος θα επαναληφθεί σε όλες τις σελίδες του εγγράφου.

## Βήμα 3: Αποκτήστε το MediaBox της σελίδας

Κάθε σελίδα σε ένα PDF έχει ένα `MediaBox` που ορίζει τα όρια της σελίδας. Πρέπει να έχουμε πρόσβαση σε αυτό για να προσδιορίσουμε τον τρέχοντα προσανατολισμό και να τον τροποποιήσουμε.

```csharp
Aspose.Pdf.Rectangle r = page.MediaBox;
```

Ο `MediaBox` μας δίνει τις διαστάσεις της σελίδας, όπως το πλάτος, το ύψος και τη θέση της.

## Βήμα 4: Αλλάξτε το πλάτος και το ύψος

Για να αλλάξουμε τον προσανατολισμό της σελίδας από κατακόρυφο σε οριζόντιο ή από οριζόντιο σε κατακόρυφο, απλώς αλλάζουμε τις τιμές πλάτους και ύψους. Αυτό το βήμα θα προσαρμόσει τις διαστάσεις της σελίδας.

```csharp
double newHeight = r.Width;
double newWidth = r.Height;
double newLLX = r.LLX;
double newLLY = r.LLY + (r.Height - newHeight);
```

Αυτός ο κώδικας εναλλάσσει το ύψος και το πλάτος και επανατοποθετεί την κάτω αριστερή γωνία (`LLY`) έτσι ώστε το περιεχόμενο να ταιριάζει άψογα μετά την περιστροφή.

## Βήμα 5: Ενημέρωση του MediaBox και του CropBox

Τώρα που έχουμε το νέο ύψος και πλάτος, ας εφαρμόσουμε τις αλλαγές στη σελίδα `MediaBox` και `CropBox`Το `CropBox` είναι απαραίτητο εάν το πρωτότυπο έγγραφο είχε ένα σετ, διασφαλίζοντας ότι ολόκληρη η σελίδα εμφανίζεται σωστά.

```csharp
page.MediaBox = new Aspose.Pdf.Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight);
page.CropBox = new Aspose.Pdf.Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight);
```

Αυτό το βήμα αλλάζει το μέγεθος της σελίδας με βάση τις νέες διαστάσεις που μόλις υπολογίσαμε.

## Βήμα 6: Περιστροφή σελίδας

Τέλος, ορίζουμε τη γωνία περιστροφής της σελίδας. Το Aspose.PDF το κάνει αυτό εξαιρετικά απλό. Μπορούμε να περιστρέψουμε τη σελίδα κατά 90 μοίρες για να αλλάξουμε από κατακόρυφο σε οριζόντιο προσανατολισμό ή το αντίστροφο.

```csharp
page.Rotate = Rotation.on90;
```

Αυτός ο κώδικας περιστρέφει τη σελίδα κατά 90 μοίρες, αναστρέφοντάς την στον επιθυμητό προσανατολισμό.

## Βήμα 7: Αποθήκευση του PDF εξόδου

Αφού εφαρμόσουμε τις αλλαγές προσανατολισμού σε όλες τις σελίδες, αποθηκεύουμε το τροποποιημένο έγγραφο σε ένα νέο αρχείο. 

```csharp
dataDir = dataDir + "ChangeOrientation_out.pdf";
doc.Save(dataDir);
System.Console.WriteLine("\nPage orientation changed successfully.\nFile saved at " + dataDir);
```

Βεβαιωθείτε ότι έχετε δώσει ένα νέο όνομα αρχείου (σε αυτήν την περίπτωση, `ChangeOrientation_out.pdf`) για να αποθηκεύσετε το αποτέλεσμα. Με αυτόν τον τρόπο, δεν αντικαθιστάτε το αρχικό σας αρχείο.

## Οδηγός Εφαρμογής

### Δημιουργία και προσθήκη κειμένου σε μια σελίδα

#### Επισκόπηση

Ο πυρήνας αυτού του σεμιναρίου είναι η δημιουργία ενός απλού εγγράφου PDF και η προσθήκη κειμένου "Hello World!" σε αυτό χρησιμοποιώντας το Aspose.PDF για .NET.

#### Βήμα 1: Αρχικοποίηση του εγγράφου

Ξεκινήστε δημιουργώντας μια παρουσία του `Document`.

```csharp
// Αρχικοποιήστε ένα νέο αντικείμενο εγγράφου.
Document document = new Document();
```

#### Βήμα 2: Προσθήκη σελίδας

Στη συνέχεια, προσθέστε μια σελίδα στο έγγραφό σας. Εδώ θα τοποθετήσετε το κείμενό σας.

```csharp
// Προσθέστε μια κενή σελίδα στο έγγραφο.
Page page = document.Pages.Add();
```

#### Βήμα 3: Προσθήκη κειμένου στη σελίδα

Χρήση `TextFragment` για να δημιουργήσετε και να προσθέσετε το κείμενό σας στη σελίδα που μόλις προσθέσατε.

```csharp
// Δημιουργήστε ένα τμήμα κειμένου με το "Hello World!"
page.Paragraphs.Add(new Aspose.Pdf.Text.TextFragment("Hello World!"));
```

#### Βήμα 4: Αποθήκευση του εγγράφου σας

Τέλος, αποθηκεύστε το έγγραφό σας. Επιλέξτε μια κατάλληλη τοποθεσία για το αρχείο εξόδου.

```csharp
// Ορίστε τον κατάλογο δεδομένων και τη διαδρομή για την αποθήκευση του PDF.
string dataDir = RunExamples.GetDataDir_AsposePdf_QuickStart();

// Αποθηκεύστε το έγγραφο στην καθορισμένη διαδρομή.
document.Save(dataDir + "HelloWorld_out.pdf");

Console.WriteLine("\nFile saved at " + dataDir);
```

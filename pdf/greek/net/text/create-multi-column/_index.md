## Οδηγίες εγκατάστασης  

Για να χρησιμοποιήσετε το Aspose.PDF για .NET στο έργο σας, ακολουθήστε τα εξής βήματα:  

1. Εγκατάσταση μέσω NuGet (Συνιστάται)  
Εκτελέστε την ακόλουθη εντολή στην κονσόλα NuGet Package Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Εγκατάσταση μέσω .NET CLI  
Για .NET Core ή .NET 5+, χρησιμοποιήστε:  
```bash
dotnet add package Aspose.PDF
```

3. Μετά την εγκατάσταση, εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
using Aspose.Pdf.Text;
```

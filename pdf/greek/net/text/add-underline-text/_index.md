# Προσθήκη υπογραμμισμένου κειμένου σε PDF με το Aspose.PDF για .NET

Το Aspose.PDF για .NET είναι μια ισχυρή βιβλιοθήκη επεξεργασίας PDF που επιτρέπει στους προγραμματιστές να δημιουργούν, να τροποποιούν και να χειρίζονται έγγραφα PDF χωρίς να απαιτούν το Adobe Acrobat. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε ένα νέο έγγραφο PDF και να προσθέσετε κείμενο με μορφοποίηση υπογράμμισης χρησιμοποιώντας τις κλάσεις TextBuilder και TextFragment. Ο κώδικας δημιουργεί ένα απλό PDF με ένα δοκιμαστικό μήνυμα μορφοποιημένο σε γραμματοσειρά Arial με στυλ υπογράμμισης, τοποθετώντας το σε συγκεκριμένες συντεταγμένες μέσα στη σελίδα.

## Οδηγίες εγκατάστασης

1: Εγκατάσταση του Aspose.PDF για .NET

Μπορείτε να εγκαταστήσετε το Aspose.PDF για .NET χρησιμοποιώντας μία από τις ακόλουθες μεθόδους:

Χρησιμοποιώντας το NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Χρησιμοποιώντας το .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Χρησιμοποιώντας το PackageReference στο αρχείο του έργου σας:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Προσθήκη Απαιτούμενων Με Δηλώσεις

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

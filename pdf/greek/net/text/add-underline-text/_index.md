# Προσθήκη υπογραμμισμένου κειμένου σε PDF με το Aspose.PDF για .NET

Το Aspose.PDF για το .NET είναι μια ισχυρή βιβλιοθήκη επεξεργασίας PDF που επιτρέπει στους προγραμματιστές να δημιουργούν, να τροποποιούν και να χειρίζονται έγγραφα PDF χωρίς να απαιτείται Adobe Acrobat. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε ένα νέο έγγραφο PDF και να προσθέσετε κείμενο με υπογραμμισμένη μορφοποίηση χρησιμοποιώντας τις τάξεις TextBuilder και TextFragment. Ο κώδικας δημιουργεί ένα απλό PDF με ένα μήνυμα δοκιμής που μορφοποιείται σε γραμματοσειρά Arial με υπογραμμισμένο στυλ, τοποθετώντας το σε συγκεκριμένες συντεταγμένες εντός της σελίδας.

## Οδηγίες εγκατάστασης

1: Εγκατάσταση του Aspose.PDF για το .NET

Μπορείτε να εγκαταστήσετε το Aspose.PDF για το .NET χρησιμοποιώντας μία από τις ακόλουθες μεθόδους:

Χρησιμοποιώντας το NuGet Package Manager:

```csharp
Install-Package Aspose.PDF
```

Χρησιμοποιώντας το .NET CLI:

```bash
dotnet add package Aspose.PDF
```

Χρησιμοποιώντας το PackageReference στο αρχείο του έργου σας:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Προσθήκη απαιτούμενων χρησιμοποιώντας δηλώσεις

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

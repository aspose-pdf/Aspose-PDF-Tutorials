## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Εγκατάσταση μέσω .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Προσθήκη αναφοράς χειροκίνητα:
   - Κατεβάστε τη βιβλιοθήκη από [Aspose Κυκλοφορίες](https://releases.aspose.com/pdf/net/)
   - Προσθέστε αναφορά στο Aspose.PDF.dll στο έργο σας

4. Προσθήκη χρησιμοποιώντας δήλωση:
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

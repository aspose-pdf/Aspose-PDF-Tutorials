## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.PDF
   ```

2. Εγκατάσταση μέσω .NET CLI:
   ```
   dotnet add package Aspose.PDF
   ```

3. Προσθήκη αναφοράς με το χέρι:
   * Κατεβάστε τη βιβλιοθήκη από το [Απολύσεις](https://releases.aspose.com/pdf/net/)
   * Προσθέστε αναφορά στο Aspose.PDF.dll στο έργο σας

4. Προσθήκη Using Statement:
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

# Αναζήτηση και λήψη κειμένου σε όλα

## Εισαγωγή

Χρειάστηκε ποτέ να εξαγάγετε συγκεκριμένο κείμενο από ένα PDF αλλά το βρήκατε δύσκολο; Τα PDF μπορεί μερικές φορές να μοιάζουν με κλειδωμένα κοντέινερ, γεγονός που καθιστά δύσκολη την εύρεση των πληροφοριών που χρειάζεστε. Αλλά να τα καλά νέα: με το Aspose.PDF για .NET, μπορείτε εύκολα να αναζητήσετε και να ανακτήσετε κείμενο από οποιοδήποτε PDF. Αυτή η ισχυρή βιβλιοθήκη παρέχει όλα όσα χρειάζεστε για να εργαστείτε με PDF στις εφαρμογές σας .NET, κάνοντας την εξαγωγή κειμένου παιχνιδάκι. Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη διαδικασία αναζήτησης και εξαγωγής κειμένου από ένα αρχείο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Είτε δημιουργείτε ένα εργαλείο ανάλυσης κειμένου είτε απλώς χρειάζεται να αυτοματοποιήσετε την εξαγωγή δεδομένων από αναφορές PDF, βρίσκεστε στο σωστό μέρος!

## Προαπαιτούμενα

Πριν προχωρήσουμε στον κώδικα, ας βεβαιωθούμε ότι έχετε ρυθμίσει τα πάντα:

1. Aspose.PDF για .NET: Θα χρειαστεί να κατεβάσετε και να εγκαταστήσετε το Aspose.PDF για .NET. Μπορείτε να το λάβετε από τη σελίδα λήψης. [εδώ](https://releases.aspose.com/pdf/net/).
2. Περιβάλλον .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει το .NET Framework ή το .NET Core στον υπολογιστή ανάπτυξης.
3. Βασικές γνώσεις C##: Συνιστάται κάποια εξοικείωση με την C# και την εργασία με έργα .NET.
4. Έγγραφο PDF: Ένα δείγμα αρχείου PDF από το οποίο θα εξάγουμε κείμενο. Σε αυτό το παράδειγμα, θα χρησιμοποιήσουμε `SearchAndGetTextFromAll.pdf`.

## Εισαγωγή πακέτων

Πριν γράψετε οποιονδήποτε κώδικα, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας για να εργαστείτε με το Aspose.PDF.

```csharp
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System;
```

Αυτοί οι χώροι ονομάτων παρέχουν πρόσβαση στο μοντέλο αντικειμένου εγγράφου του PDF και μας επιτρέπουν να χειριζόμαστε κείμενο μέσα στο αρχείο.

Ας χωρίσουμε τη διαδικασία σε απλά βήματα, ώστε να μπορείτε να την παρακολουθείτε με ευκολία.

## Βήμα 1: Ορισμός του καταλόγου εγγράφων

Πρώτα απ 'όλα, πρέπει να καθορίσετε τη διαδρομή προς τον κατάλογο όπου βρίσκεται το PDF σας. Αυτό βοηθά την εφαρμογή να εντοπίσει το αρχείο από το οποίο θα εξαγάγετε το κείμενο.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

- Ο `dataDir` Η μεταβλητή θα πρέπει να δείχνει στον κατάλογο όπου βρίσκεται το `SearchAndGetTextFromAll.pdf` το αρχείο αποθηκεύεται.
- Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στο μηχάνημά σας.

## Βήμα 2: Ανοίξτε το έγγραφο PDF

Στη συνέχεια, θα ανοίξουμε το έγγραφο PDF χρησιμοποιώντας το Aspose.PDF. `Document` αντικείμενο.

```csharp
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "SearchAndGetTextFromAll.pdf");
```

- Δημιουργούμε μια νέα παρουσία του `Document` κλάσης περνώντας την πλήρη διαδρομή αρχείου του PDF.
- Αυτό θα φορτώσει το PDF στη μνήμη, καθιστώντας το έτοιμο για επεξεργασία.

## Βήμα 3: Δημιουργήστε ένα πρόγραμμα απορρόφησης κειμένου

Ο `TextFragmentAbsorber` Το αντικείμενο χρησιμοποιείται για την αναζήτηση συγκεκριμένου κειμένου μέσα στο PDF. Σε αυτήν την περίπτωση, θα αναζητήσουμε τη λέξη «κείμενο».

```csharp
// Δημιουργήστε ένα αντικείμενο TextAbsorber για να βρείτε όλες τις εμφανίσεις της φράσης αναζήτησης που έχετε εισαγάγει
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("text");
```

- Ο `TextFragmentAbsorber` αρχικοποιείται με τη συμβολοσειρά `"text"`Αυτό σημαίνει ότι θα αναζητήσει τυχόν εμφανίσεις της λέξης «κείμενο» μέσα στο έγγραφο PDF.

## Βήμα 4: Αποδοχή του Απορροφητή για Όλες τις Σελίδες

Τώρα, θα δώσουμε εντολή στο έγγραφο PDF να αποδεχτεί τον απορροφητή και να αναζητήσει το κείμενο σε όλες τις σελίδες του.

```csharp
// Αποδεχτείτε τον απορροφητή για όλες τις σελίδες
pdfDocument.Pages.Accept(textFragmentAbsorber);
```

- Ο `Accept` Η μέθοδος εφαρμόζεται στις σελίδες του εγγράφου. Αυτό θα αναζητήσει σε όλες τις σελίδες το καθορισμένο κείμενο.

## Βήμα 5: Εξαγωγή τμημάτων κειμένου

Μόλις ο απορροφητής σαρώσει το έγγραφο, μπορούμε να ανακτήσουμε τα εξαγόμενα τμήματα κειμένου.

```csharp
// Λήψη των εξαγόμενων τμημάτων κειμένου
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.TextFragments;
```

- Ο `TextFragments` ιδιοκτησία του `TextFragmentAbsorber` Επιστρέφει μια συλλογή από όλα τα τμήματα κειμένου που ταιριάζουν με τον όρο αναζήτησης.

## Βήμα 6: Επανάληψη των τμημάτων κειμένου

Τώρα που έχουμε τη συλλογή των τμημάτων κειμένου, θα τα εξετάσουμε επανειλημμένα και θα εξαγάγουμε τις λεπτομέρειες.

```csharp
// Περιπλανηθείτε μέσα από τα θραύσματα
foreach (TextFragment textFragment in textFragmentCollection)
{
    Console.WriteLine("Text : {0} ", textFragment.Text);
    Console.WriteLine("Position : {0} ", textFragment.Position);
    Console.WriteLine("XIndent : {0} ", textFragment.Position.XIndent);
    Console.WriteLine("YIndent : {0} ", textFragment.Position.YIndent);
    Console.WriteLine("Font - Name : {0}", textFragment.TextState.Font.FontName);
    Console.WriteLine("Font - IsAccessible : {0} ", textFragment.TextState.Font.IsAccessible);
    Console.WriteLine("Font - IsEmbedded : {0} ", textFragment.TextState.Font.IsEmbedded);
    Console.WriteLine("Font - IsSubset : {0} ", textFragment.TextState.Font.IsSubset);
    Console.WriteLine("Font Size : {0} ", textFragment.TextState.FontSize);
    Console.WriteLine("Foreground Color : {0} ", textFragment.TextState.ForegroundColor);
}
```

- Ο `foreach` ο βρόχος επαναλαμβάνεται μέσω κάθε `TextFragment` στη συλλογή.
- Εκτυπώνουμε διάφορες ιδιότητες κάθε τμήματος, όπως το πραγματικό κείμενο, τη θέση του στη σελίδα, τις λεπτομέρειες της γραμματοσειράς και το μέγεθος της γραμματοσειράς.
- Ο `XIndent` και `YIndent` Οι ιδιότητες δίνουν τις ακριβείς συντεταγμένες του τμήματος κειμένου μέσα στο PDF.

## Συχνές ερωτήσεις

### Μπορώ να αναζητήσω πολλές λέξεις ταυτόχρονα;  
Ναι, μπορείτε να τροποποιήσετε το `TextFragmentAbsorber` για να αναζητήσετε πολλαπλές φράσεις προσαρμόζοντας ανάλογα τη συμβολοσειρά αναζήτησης.

### Τι γίνεται αν το κείμενο εκτείνεται σε πολλές γραμμές;  
Το Aspose.PDF θα εξακολουθεί να αναγνωρίζει και να εξάγει κείμενο ακόμα κι αν εκτείνεται σε πολλές γραμμές. Μπορείτε να χειριστείτε αυτά τα τμήματα ξεχωριστά.

### Πώς μπορώ να αποθηκεύσω το εξαγόμενο κείμενο σε ένα αρχείο;  
Μπορείτε να γράψετε το εξαγόμενο κείμενο σε ένα αρχείο χρησιμοποιώντας τυπικές λειτουργίες εισόδου/εξόδου αρχείων C#, όπως `StreamWriter`.

### Υποστηρίζει το Aspose.PDF την εξαγωγή κειμένου από σαρωμένα PDF;  
Το Aspose.PDF δεν υποστηρίζει OCR. Για σαρωμένα PDF, θα χρειαστείτε ένα εργαλείο OCR για την αναγνώριση του κειμένου.

### Πώς μπορώ να χειριστώ κρυπτογραφημένα PDF;  
Εάν το PDF σας προστατεύεται με κωδικό πρόσβασης, μπορείτε να το ξεκλειδώσετε χρησιμοποιώντας το Aspose.PDF παρέχοντας τον κωδικό πρόσβασης κατά τη φόρτωση του εγγράφου.

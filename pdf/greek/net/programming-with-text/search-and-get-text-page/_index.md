# Αναζήτηση και λήψη σελίδας κειμένου σε αρχείο PDF

## Εισαγωγή

Έχετε βρεθεί ποτέ να χρειάζεται να αναζητήσετε συγκεκριμένο κείμενο μέσα σε ένα έγγραφο PDF και να το εξαγάγετε για περαιτέρω χρήση; Ίσως δημιουργείτε μια εφαρμογή που επεξεργάζεται έγγραφα και απαιτεί ακριβή εξαγωγή δεδομένων ή ίσως απλώς χρειάζεται να αναλύσετε αποτελεσματικά τα PDF. Όποια και αν είναι η περίπτωσή σας, βρίσκεστε στο σωστό μέρος! Σε αυτό το σεμινάριο, θα εμβαθύνουμε στον τρόπο αναζήτησης και λήψης κειμένου από μια σελίδα σε ένα αρχείο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Είτε είστε αρχάριος είτε έμπειρος προγραμματιστής, αυτός ο οδηγός θα σας καθοδηγήσει σε κάθε βήμα με έναν εύχρηστο και ενδιαφέρον τρόπο. Είστε έτοιμοι να ξεκινήσετε; Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν ξεκινήσουμε τον προγραμματισμό, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε:

1. Aspose.PDF για τη βιβλιοθήκη .NET: Μπορείτε να το κατεβάσετε από [εδώ](https://releases.aspose.com/pdf/net/) ή αποκτήστε μια δωρεάν δοκιμαστική έκδοση από τον ίδιο σύνδεσμο. Για αγορά, επισκεφθείτε τη διεύθυνση [Κατάστημα Aspose](https://purchase.aspose.com/buy).
2. .NET Framework: Θα χρειαστείτε ένα λειτουργικό περιβάλλον ανάπτυξης .NET, όπως το Visual Studio.
3. Ένα αρχείο PDF: Θα χρειαστείτε ένα δείγμα αρχείου PDF όπου μπορούμε να αναζητήσουμε και να εξαγάγουμε το κείμενο. Για αυτό το σεμινάριο, ας υποθέσουμε ότι το αρχείο ονομάζεται `SearchAndGetTextPage.pdf`.

## Εισαγωγή πακέτων

Πρώτα απ' όλα, πρέπει να εισαγάγουμε τους απαραίτητους χώρους ονομάτων για να λειτουργήσουμε με το Aspose.PDF για .NET. Βεβαιωθείτε ότι αυτοί περιλαμβάνονται στην αρχή του κώδικά σας.

```csharp
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System
```

Τώρα που καλύψαμε τις προϋποθέσεις, ας αναλύσουμε τον κώδικα βήμα προς βήμα. Κάθε βήμα έχει περιγραφεί με σαφήνεια για να είναι εύκολο να το παρακολουθήσετε.

## Βήμα 1: Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας

Πριν αλληλεπιδράσετε με το PDF σας, θα πρέπει να ορίσετε τη διαδρομή προς το σημείο όπου αποθηκεύεται το έγγραφο PDF σας. Αυτό διασφαλίζει ότι το πρόγραμμα μπορεί να έχει πρόσβαση στο αρχείο.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

- dataDir: Αυτή είναι η διαδρομή προς τον φάκελο όπου είναι αποθηκευμένα τα αρχεία PDF σας. Αντικαταστήστε `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου βρίσκεται το PDF.

## Βήμα 2: Φόρτωση του εγγράφου PDF

Το επόμενο βήμα είναι να φορτώσετε το έγγραφο PDF στη μνήμη, ώστε να μπορείτε να εργαστείτε με αυτό. Δείτε πώς:

```csharp
Document pdfDocument = new Document(dataDir + "SearchAndGetTextPage.pdf");
```

- Έγγραφο: Αυτή είναι η κλάση Aspose.PDF που φορτώνει το αρχείο PDF.
- pdfDocument: Η μεταβλητή όπου αποθηκεύεται το αρχείο PDF μετά τη φόρτωσή του.

## Βήμα 3: Δημιουργήστε ένα αντικείμενο απορρόφησης κειμένου

Ο `TextFragmentAbsorber` Η κλάση σάς επιτρέπει να αναζητήσετε συγκεκριμένο κείμενο μέσα στο PDF. Ας δημιουργήσουμε μια παρουσία αυτής της κλάσης για να βρούμε όλες τις εμφανίσεις μιας δεδομένης φράσης αναζήτησης.

```csharp
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("Figure");
```

- TextFragmentAbsorber: Αυτή η κλάση είναι υπεύθυνη για την εύρεση και εξαγωγή κειμένου από το PDF.
- "Σχήμα": Αντικαταστήστε το με οποιοδήποτε κείμενο θέλετε να αναζητήσετε μέσα στο PDF.

## Βήμα 4: Εφαρμόστε το Text Absorber σε ολόκληρο το PDF

Μόλις ρυθμιστεί το πρόγραμμα απορρόφησης κειμένου, πρέπει να ορίσετε στο πρόγραμμα να πραγματοποιήσει αναζήτηση σε όλες τις σελίδες του PDF.

```csharp
pdfDocument.Pages.Accept(textFragmentAbsorber);
```

- Accept(): Αυτή η μέθοδος εφαρμόζει την απορρόφηση κειμένου στο PDF, σαρώνοντας κάθε σελίδα για το κείμενο που καθορίσατε.

## Βήμα 5: Ανάκτηση και επανάληψη του εξαγόμενου κειμένου

Τώρα που σαρώσαμε το PDF, ήρθε η ώρα να ανακτήσουμε τα αποτελέσματα και να τα εμφανίσουμε. Θα επανέλθουμε στα αποσπάσματα κειμένου που εξαγάγαμε.

```csharp
TextFragmentCollection textFragmentCollection = textFragmentAbsorber.TextFragments;
```

- TextFragmentCollection: Αυτή η συλλογή περιέχει όλες τις εμφανίσεις των τμημάτων κειμένου που βρέθηκαν από το εργαλείο απορρόφησης κειμένου.

## Βήμα 6: Επανάληψη κάθε τμήματος και εξαγωγή δεδομένων

Θα κάνουμε τώρα μια επανάληψη μέσα από το `textFragmentCollection` και να εξαγάγετε διάφορες ιδιότητες κάθε τμήματος κειμένου, όπως τη θέση του, τις λεπτομέρειες της γραμματοσειράς και το χρώμα.

```csharp
foreach (TextFragment textFragment in textFragmentCollection)
{
	foreach (TextSegment textSegment in textFragment.Segments)
	{
		Console.WriteLine("Text : {0} ", textSegment.Text);
		Console.WriteLine("Position : {0} ", textSegment.Position);
		Console.WriteLine("XIndent : {0} ", textSegment.Position.XIndent);
		Console.WriteLine("YIndent : {0} ", textSegment.Position.YIndent);
		Console.WriteLine("Font - Name : {0}", textSegment.TextState.Font.FontName);
		Console.WriteLine("Font - IsAccessible : {0} ", textSegment.TextState.Font.IsAccessible);
		Console.WriteLine("Font - IsEmbedded : {0} ", textSegment.TextState.Font.IsEmbedded);
		Console.WriteLine("Font - IsSubset : {0} ", textSegment.TextState.Font.IsSubset);
		Console.WriteLine("Font Size : {0} ", textSegment.TextState.FontSize);
		Console.WriteLine("Foreground Color : {0} ", textSegment.TextState.ForegroundColor);
	}
}
```

- Απόσπασμα κειμένου: Κάθε απόσπασμα περιέχει τμήματα του κειμένου που βρέθηκε.
- Τμήμα κειμένου: Κάθε τμήμα μπορεί να έχει πολλά τμήματα, που αντιπροσωπεύουν διαφορετικά μέρη του κειμένου.
- Κατάσταση κειμένου: Παρέχει λεπτομερείς πληροφορίες σχετικά με τη γραμματοσειρά, το μέγεθος και το χρώμα του κειμένου.

Σε αυτόν τον βρόχο, εκτυπώνουμε λεπτομέρειες όπως το πραγματικό κείμενο, τη θέση του (συντεταγμένες X και Y), τη γραμματοσειρά, το αν η γραμματοσειρά είναι ενσωματωμένη στο PDF και το χρώμα προσκηνίου του κειμένου.

Ναι, μπορείτε να τροποποιήσετε τον κώδικα για να αναζητήσετε πολλαπλές φράσεις δημιουργώντας πολλαπλές `TextFragmentAbsorber` αντικείμενα.

### Πώς μπορώ να εξαγάγω κείμενο από μια συγκεκριμένη σελίδα;  
Μπορείτε να στοχεύσετε μια συγκεκριμένη σελίδα εφαρμόζοντας την `TextFragmentAbsorber` σε μία μόνο σελίδα αντί για ολόκληρο το έγγραφο. Για παράδειγμα: `pdfDocument.Pages[1].Accept(textFragmentAbsorber);`.

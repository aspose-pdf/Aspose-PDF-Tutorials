# Προσθήκη και αναζήτηση κρυφού κειμένου σε αρχείο PDF

## Εισαγωγή

Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε βήμα προς βήμα για το πώς να προσθέσετε και να αναζητήσετε κρυφό κείμενο σε ένα αρχείο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Είτε είστε έμπειρος προγραμματιστής είτε αρχάριος που θέλει να βελτιώσει τις δεξιότητές του στον προγραμματισμό, αυτό το άρθρο θα σας παρέχει τις πληροφορίες που χρειάζεστε για να ενσωματώσετε τη λειτουργικότητα κρυφού κειμένου στις εφαρμογές σας.

## Προαπαιτούμενα

Πριν προχωρήσουμε στο κομμάτι του προγραμματισμού, υπάρχουν μερικές προϋποθέσεις που πρέπει να λάβετε υπόψη:

### Λίστα ελέγχου απαιτήσεων
- Visual Studio: Βεβαιωθείτε ότι έχετε εγκατεστημένο το Visual Studio. Αυτό το σεμινάριο προϋποθέτει ότι χρησιμοποιείτε .NET Framework.
- Aspose.PDF για .NET: Πρέπει να έχετε τη βιβλιοθήκη Aspose.PDF για .NET. Μπορείτε να την κατεβάσετε. [εδώ](https://releases.aspose.com/pdf/net/).
- Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τα αποσπάσματα κώδικα.

## Εισαγωγή πακέτων

Πριν ξεκινήσετε με τον κώδικά σας, πρέπει να βεβαιωθείτε ότι έχετε εισαγάγει τους απαραίτητους χώρους ονομάτων Aspose.PDF. Δείτε πώς μπορείτε να το κάνετε:

### Ρύθμιση του έργου σας
1. Ανοίξτε το Visual Studio και δημιουργήστε ένα νέο έργο C# ή χρησιμοποιήστε ένα υπάρχον.
2. Εγκαταστήστε το Aspose.PDF προσθέτοντας το πακέτο NuGet. Μπορείτε να το κάνετε αυτό μεταβαίνοντας στον Διαχειριστή Πακέτων NuGet και αναζητώντας `Aspose.PDF`. 
3. Εναλλακτικά, μπορείτε να κατεβάσετε απευθείας τη βιβλιοθήκη από [εδώ](https://releases.aspose.com/pdf/net/) και προσθέστε το ως αναφορά στο έργο σας.

### Εισαγωγή απαιτούμενων χώρων ονομάτων
Στο επάνω μέρος του αρχείου C#, εισαγάγετε τους ακόλουθους χώρους ονομάτων:

```csharp
using Aspose.Pdf.Text;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

Αυτό το βήμα είναι κρίσιμο, καθώς αυτοί οι χώροι ονομάτων περιέχουν τις κλάσεις και τις μεθόδους που είναι απαραίτητες για τον χειρισμό εγγράφων PDF.

## Δημιουργία εγγράφου PDF με κρυφό κείμενο

Τώρα που είστε έτοιμοι, ας δούμε τα βήματα για να δημιουργήσετε ένα έγγραφο PDF που περιέχει ορατό και αόρατο κείμενο.

### Βήμα 1: Ορισμός καταλόγου εγγράφων
Αρχικά, θα πρέπει να ορίσετε τη διαδρομή όπου θα αποθηκευτεί το PDF. Εδώ ξεκινά η μαγεία!

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // Αλλάξτε αυτό στον κατάλογό σας
```

Αυτή η γραμμή καθορίζει πού θα αποθηκευτεί το PDF που δημιουργήσατε. Μην ξεχάσετε να αντικαταστήσετε την `YOUR DOCUMENT DIRECTORY` με την πραγματική σου πορεία.

### Βήμα 2: Δημιουργήστε ένα έγγραφο PDF
Στη συνέχεια, ας δημιουργήσουμε ένα νέο έγγραφο PDF και ας προσθέσουμε σελίδες σε αυτό.

```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
Page page = doc.Pages.Add();
```

Εδώ, αρχικοποιούμε ένα νέο έγγραφο και προσθέτουμε μια σελίδα όπου θα τοποθετήσουμε τα τμήματα κειμένου μας.

### Βήμα 3: Προσθήκη ορατού και κρυφού κειμένου
Τώρα θα προσθέσουμε τόσο ορατό όσο και αόρατο κείμενο στο PDF μας.

```csharp
TextFragment frag1 = new TextFragment("This is common text.");
TextFragment frag2 = new TextFragment("This is invisible text.");
```

Σε αυτό το απόσπασμα, `frag1` θα είναι ορατό, ενώ `frag2` θα οριστεί σε αόρατο στη συνέχεια.

### Βήμα 4: Ορίστε το κείμενο σε αόρατο
Για να φτιάξετε το κείμενο του `frag2` αόρατο, απλώς το τροποποιείτε `TextState`.

```csharp
frag2.TextState.Invisible = true;
```

Ορίζοντας αυτήν την ιδιότητα, οποιοδήποτε κείμενο σχετίζεται με `frag2` δεν θα αποδοθεί κατά την προβολή του PDF.

### Βήμα 5: Προσθήκη τμημάτων κειμένου στη σελίδα
Τέλος, προσθέτουμε αυτά τα τμήματα κειμένου στη σελίδα και αποθηκεύουμε το PDF.

```csharp
page.Paragraphs.Add(frag1);
page.Paragraphs.Add(frag2);
doc.Save(dataDir + "39400_out.pdf");
doc.Dispose();
```

Αυτό το μέρος του κώδικα προσθέτει τα αποσπάσματα κειμένου μας στη σελίδα. Στη συνέχεια, αποθηκεύουμε και απορρίπτουμε το έγγραφο σωστά.

## Αναζήτηση κρυφού κειμένου σε PDF

Τώρα που δημιουργήσαμε το PDF μας με ορατό και κρυφό κείμενο, πώς μπορούμε να αναζητήσουμε αυτό το κρυφό κείμενο; Ας το αναλύσουμε.

### Βήμα 1: Φόρτωση του εγγράφου PDF
Για να αναζητήσουμε κείμενο μέσα στο PDF, πρέπει πρώτα να φορτώσουμε το έγγραφο που μόλις δημιουργήσαμε.

```csharp
doc = new Aspose.Pdf.Document(dataDir + "39400_out.pdf");
```

### Βήμα 2: Δημιουργήστε ένα πρόγραμμα απορρόφησης θραυσμάτων κειμένου
Θα χρησιμοποιήσουμε `TextFragmentAbsorber` για να καταγράψετε όλα τα αποσπάσματα κειμένου στο PDF.

```csharp
TextFragmentAbsorber absorber = new TextFragmentAbsorber();
absorber.Visit(doc.Pages[1]);
```

Εδώ, καθορίζουμε ότι θέλουμε να απορροφήσουμε όλα τα τμήματα κειμένου από την πρώτη σελίδα.

### Βήμα 3: Επαναλάβετε τα τμήματα
Τώρα, μπορούμε να επανεξετάσουμε τα συλλεγμένα αποσπάσματα κειμένου για να διαπιστώσουμε ποια είναι ορατά και ποια είναι κρυμμένα.

```csharp
foreach (TextFragment fragment in absorber.TextFragments)
{
    Console.WriteLine("Text '{0}' on pos {1} invisibility: {2}",
        fragment.Text, fragment.Position.ToString(), fragment.TextState.Invisible);
}
```

Αυτός ο βρόχος ελέγχει κάθε τμήμα κειμένου και εκτυπώνει τα περιεχόμενά του μαζί με τη θέση και την κατάσταση ορατότητάς του. Εάν `fragment.TextState.Invisible` αν οριστεί σε true, σημαίνει ότι το κείμενο είναι κρυφό!

### Βήμα 4: Απόρριψη εγγράφου
Τέλος, θυμηθείτε να απορρίψετε ξανά το έγγραφο μόλις τελειώσετε.

```csharp
doc.Dispose();
```

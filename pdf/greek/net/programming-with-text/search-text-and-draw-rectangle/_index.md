# Αναζήτηση κειμένου και σχεδίαση ορθογωνίου

## Εισαγωγή

Θέλετε να βελτιώσετε τις δεξιότητές σας στον χειρισμό PDF; Θέλετε να μάθετε πώς να αναζητάτε συγκεκριμένο κείμενο σε αρχεία PDF και να το επισημαίνετε με ένα ορθογώνιο; Βρήκατε τον τέλειο οδηγό! Σήμερα, θα σας καθοδηγήσω στον τρόπο χρήσης του Aspose.PDF για .NET για να αναζητήσετε κείμενο σε ένα έγγραφο PDF και να σχεδιάσετε ορθογώνια γύρω του. Αυτό το άρθρο θα παρέχει ένα βήμα προς βήμα σεμινάριο σχεδιασμένο με γνώμονα τη σαφήνεια και τη χρησιμότητα, διασφαλίζοντας ότι μπορείτε να παρακολουθείτε και να εφαρμόζετε αυτές τις τεχνικές στα έργα σας. 

## Προαπαιτούμενα

Πριν ξεκινήσουμε το σεμινάριο, ας προετοιμάσουμε ό,τι χρειάζεστε για να διασφαλίσετε μια ομαλή ροή εργασίας:

1. Βασική Κατανόηση του .NET: Θα πρέπει να είστε εξοικειωμένοι με τον προγραμματισμό C# και το .NET framework για να παρακολουθήσετε αποτελεσματικά αυτό το σεμινάριο.
   
2. Εγκατεστημένο Visual Studio: Θα χρειαστείτε ένα ολοκληρωμένο περιβάλλον ανάπτυξης (IDE) για να γράψετε και να δοκιμάσετε τον κώδικά σας. Η Κοινότητα του Visual Studio είναι μια εξαιρετική επιλογή και είναι δωρεάν.
   
3. Aspose.PDF για .NET: Πρέπει να έχετε εγκατεστημένη τη βιβλιοθήκη Aspose.PDF στο έργο σας. Μπορείτε να την κατεβάσετε. [εδώ](https://releases.aspose.com/pdf/net/) ή σκεφτείτε ένα [προσωρινή άδεια](https://purchase.aspose.com/temporary-license/) για εκτεταμένες λειτουργίες.
   
4. Δείγμα εγγράφου PDF: Για αυτό το σεμινάριο, θα χρειαστείτε ένα δείγμα αρχείου PDF με το όνομα `SearchAndGetTextFromAll.pdf` αποθηκευμένο στον κατάλογο του έργου σας.

### Άνοιγμα Visual Studio

Εκκινήστε το Visual Studio και δημιουργήστε μια νέα εφαρμογή κονσόλας ή χρησιμοποιήστε μια υπάρχουσα όπου θέλετε να εφαρμόσετε τις λειτουργίες PDF.

## Εισαγωγή χώρων ονομάτων

Στο επάνω μέρος του αρχείου προγράμματος, θα πρέπει να εισαγάγετε τους σχετικούς χώρους ονομάτων από τη βιβλιοθήκη Aspose:

```csharp
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System;
using Aspose.Pdf.Facades;
```

Αυτό διευκολύνει την πρόσβαση σε κλάσεις και μεθόδους εντός της βιβλιοθήκης Aspose.PDF για τις εργασίες σας.


Τώρα που έχετε ρυθμίσει τα πάντα, ας αναλύσουμε τη διαδικασία αναζήτησης κειμένου σε ένα PDF και τη σχεδίαση ενός ορθογωνίου γύρω του σε διαχειρίσιμα βήματα.

## Βήμα 1: Ορίστε τη διαδρομή για το έγγραφό σας

Αρχικά, ορίστε τη διαδρομή προς το αρχείο PDF. Βεβαιωθείτε ότι έχετε αντικαταστήσει `YOUR DOCUMENT DIRECTORY` με την πραγματική διαδρομή όπου εσύ `SearchAndGetTextFromAll.pdf` αποθηκεύεται.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 2: Ανοίξτε το έγγραφο PDF

Στη συνέχεια, δημιουργήστε μια παρουσία του `Document` τάξη για να φορτώσετε το PDF σας:

```csharp
Document document = new Document(dataDir + "SearchAndGetTextFromAll.pdf");
```

Αυτή η γραμμή κώδικα ανοίγει το καθορισμένο αρχείο PDF, επιτρέποντάς σας να το χειριστείτε περαιτέρω.

## Βήμα 3: Δημιουργήστε ένα πρόγραμμα απορρόφησης κειμένου

Τώρα, θα χρειαστείτε έναν τρόπο αναζήτησης κειμένου μέσα σε αυτό το έγγραφο. Για αυτό, χρησιμοποιούμε το `TextFragmentAbsorber`:

```csharp
TextFragmentAbsorber textAbsorber = new TextFragmentAbsorber(@"[\S]+");
```

Η κανονική έκφραση `@"[\S]+"` έχει σχεδιαστεί για να ταιριάζει με οποιαδήποτε συμβολοσειρά που δεν περιέχει κενά διαστήματα στο PDF. 

## Βήμα 4: Ρύθμιση παραμέτρων επιλογών αναζήτησης κειμένου

Στη συνέχεια, θα πρέπει να ρυθμίσετε τις επιλογές αναζήτησης κειμένου:

```csharp
TextSearchOptions textSearchOptions = new TextSearchOptions(true);
textAbsorber.TextSearchOptions = textSearchOptions;
```

Εδώ, το `true` η παράμετρος σημαίνει ότι η αναζήτηση θα κάνει διάκριση πεζών-κεφαλαίων. Μπορείτε να την ορίσετε σε `false` αν θέλετε αναζήτηση χωρίς διάκριση πεζών-κεφαλαίων.

## Βήμα 5: Αποδοχή του Απορροφητή Κειμένου στο Έγγραφο

Με το δικό σας `TextFragmentAbsorber` και οι επιλογές αναζήτησης είναι έτοιμες, ήρθε η ώρα να απορροφήσετε κείμενο από το έγγραφο:

```csharp
document.Pages.Accept(textAbsorber);
```

Αυτή η μέθοδος εξετάζει κάθε σελίδα στο PDF σας για να βρει τμήματα κειμένου που ταιριάζουν με το καθορισμένο μοτίβο.

## Βήμα 6: Δημιουργήστε ένα πρόγραμμα επεξεργασίας PdfContent

Για να σχεδιάσετε σχήματα στο έγγραφο, θα χρειαστείτε το `PdfContentEditor`:

```csharp
var editor = new PdfContentEditor(document);
```

Αυτό το πρόγραμμα επεξεργασίας σάς επιτρέπει να χειρίζεστε και να επεξεργάζεστε εύκολα το περιεχόμενο PDF.

## Βήμα 7: Επανάληψη σε εντοπισμένα τμήματα κειμένου

Τώρα, θα θελήσετε να κάνετε επανάληψη στα αποσπάσματα κειμένου που βρέθηκαν για να σχεδιάσετε ορθογώνια γύρω τους:

```csharp
foreach (TextFragment textFragment in textAbsorber.TextFragments)
{
    foreach (TextSegment textSegment in textFragment.Segments)
    {
        DrawBox(editor, textFragment.Page.Number, textSegment, System.Drawing.Color.Red);
    }
}
```

Αυτός ο βρόχος επαναλαμβάνεται πάνω από κάθε τμήμα κειμένου και τα τμήματά του, καλώντας ένα `DrawBox` μέθοδος για το σχεδιασμό ορθογωνίων.

## Βήμα 8: Ορισμός της μεθόδου DrawBox

Πρέπει να ορίσετε το `DrawBox` μέθοδος, η οποία θα χειριστεί τη λογική σχεδίασης ορθογωνίων. Ακολουθεί μια απλή υλοποίηση:

```csharp
private static void DrawBox(PdfContentEditor editor, int pageNumber, TextSegment textSegment, System.Drawing.Color color)
{
    // Υπολογίστε τις διαστάσεις του ορθογωνίου με βάση το τμήμα κειμένου
    float x = textSegment.Rectangle.LLX;
    float y = textSegment.Rectangle.LLY;
    float width = textSegment.Rectangle.Width;
    float height = textSegment.Rectangle.Height;

    // Σχεδιάστε ένα ορθογώνιο χρησιμοποιώντας τις υπολογισμένες τιμές
    editor.DrawRectangle(pageNumber, x, y, width, height, color, 1);
}
```

Αυτή η μέθοδος καθορίζει τη θέση και το μέγεθος του ορθογωνίου με βάση το ορθογώνιο που οριοθετεί το τμήμα και χρησιμοποιεί τον επεξεργαστή για να το σχεδιάσει.

## Βήμα 9: Αποθήκευση του τροποποιημένου εγγράφου

Αφού σχεδιάσετε τα ορθογώνια γύρω από το κείμενο που βρέθηκε, μπορείτε να αποθηκεύσετε το τροποποιημένο έγγραφο:

```csharp
dataDir = dataDir + "SearchTextAndDrawRectangle_out.pdf";
document.Save(dataDir);
```

Βεβαιωθείτε ότι το νέο σας αρχείο αποθηκεύεται με ξεχωριστό όνομα για να αποφύγετε την αντικατάσταση του αρχικού σας εγγράφου.

## Βήμα 10: Μήνυμα επιβεβαίωσης

Τέλος, εκτυπώστε ένα μήνυμα επιβεβαίωσης στην κονσόλα για να σας ενημερώσουμε ότι η λειτουργία ήταν επιτυχής:

```csharp
Console.WriteLine("\nRectangle drawn successfully on searched text.\nFile saved at " + dataDir);
```

Και να το! Δημιουργήσατε με επιτυχία ένα σενάριο για την αναζήτηση κειμένου σε ένα PDF και την επισήμανσή του με ορθογώνια.

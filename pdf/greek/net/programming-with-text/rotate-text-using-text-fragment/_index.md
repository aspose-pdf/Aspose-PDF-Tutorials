## Εισαγωγή πακέτων

Πριν ξεκινήσουμε την κωδικοποίηση, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων που συνοδεύουν το Aspose.PDF. Αυτό είναι κρίσιμο για την εργασία με έγγραφα, σελίδες, τμήματα κειμένου και άλλα. Προσθέστε τον ακόλουθο κώδικα στην αρχή του αρχείου C#:

```csharp
using System;
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
using Aspose.Pdf.Facades;
```

Τώρα, ας αναλύσουμε τον κώδικα του παραδείγματος βήμα προς βήμα, ώστε να μπορείτε να περιστρέψετε κείμενο σαν επαγγελματίας!

## Βήμα 1: Αρχικοποίηση του αντικειμένου εγγράφου

Κάθε χειρισμός PDF ξεκινά με τη δημιουργία ή τη φόρτωση ενός εγγράφου PDF. Εδώ, θα αρχικοποιήσουμε ένα νέο έγγραφο PDF από την αρχή χρησιμοποιώντας το Aspose.PDF.

Δημιουργούμε ένα νέο `Document` αντικείμενο που αντιπροσωπεύει το αρχείο PDF. Αρχικά, αυτό το έγγραφο είναι κενό.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
// Αρχικοποίηση αντικειμένου εγγράφου
Document pdfDocument = new Document();
```

Εξήγηση:  
- `dataDir`: Αυτός είναι ο κατάλογος όπου θα αποθηκευτεί το τελικό σας PDF.
- `Document pdfDocument = new Document();`: Αυτό αρχικοποιεί ένα νέο, κενό έγγραφο PDF. 

## Βήμα 2: Προσθήκη σελίδας στο έγγραφο

Στη συνέχεια, πρέπει να προσθέσουμε μια σελίδα στο έγγραφο. Ένα PDF είναι ουσιαστικά μια συλλογή σελίδων και χρειάζεστε τουλάχιστον μία σελίδα για να προσθέσετε το περιεχόμενό σας.

```csharp
// Λήψη συγκεκριμένης σελίδας
Page pdfPage = (Page)pdfDocument.Pages.Add();
```

Χωρίς να προσθέσετε μια σελίδα, δεν υπάρχει καμβάς για να σχεδιάσετε ή να τοποθετήσετε το κείμενό σας!

## Βήμα 3: Δημιουργήστε το πρώτο τμήμα κειμένου

Και τώρα έρχεται το συναρπαστικό κομμάτι! Ας προσθέσουμε ένα τμήμα κειμένου στο PDF. Ένα τμήμα κειμένου είναι ένα τμήμα κειμένου με συγκεκριμένες ιδιότητες όπως γραμματοσειρά, μέγεθος και θέση.

```csharp
// Δημιουργία τμήματος κειμένου
TextFragment textFragment1 = new TextFragment("main text");
textFragment1.Position = new Position(100, 600);
textFragment1.TextState.FontSize = 12;
textFragment1.TextState.Font = FontRepository.FindFont("TimesNewRoman");
```

- TextFragment("κύριο κείμενο"): Αυτό δημιουργεί ένα νέο τμήμα κειμένου με περιεχόμενο "κύριο κείμενο".
- Θέση(100, 600): Ορίζει τη θέση του κειμένου στη σελίδα. Ο πρώτος αριθμός είναι η συντεταγμένη x και ο δεύτερος είναι η συντεταγμένη y.
- TextState.FontSize: Ορίζει το μέγεθος γραμματοσειράς του κειμένου.
- FontRepository.FindFont: Εντοπίζει την καθορισμένη γραμματοσειρά που θα εφαρμοστεί στο κείμενο.

## Βήμα 4: Δημιουργήστε τα περιστρεφόμενα τμήματα κειμένου

Ας προσθέσουμε περισσότερα τμήματα κειμένου, αλλά αυτή τη φορά θα τα περιστρέψουμε σε διαφορετικές γωνίες!

### Περιστροφή τμήματος κειμένου κατά 45 μοίρες

```csharp
// Δημιουργία περιστρεφόμενου τμήματος κειμένου
TextFragment textFragment2 = new TextFragment("rotated text");
textFragment2.Position = new Position(200, 600);
textFragment2.TextState.FontSize = 12;
textFragment2.TextState.Font = FontRepository.FindFont("TimesNewRoman");
textFragment2.TextState.Rotation = 45;
```

Εδώ, η βασική αλλαγή είναι:
- TextState.Rotation: Αυτή η ιδιότητα ορίζει τη γωνία περιστροφής για το τμήμα κειμένου και, σε αυτήν την περίπτωση, είναι 45 μοίρες.

### Περιστροφή τμήματος κειμένου κατά 90 μοίρες

```csharp
// Δημιουργία περιστρεφόμενου τμήματος κειμένου
TextFragment textFragment3 = new TextFragment("rotated text");
textFragment3.Position = new Position(300, 600);
textFragment3.TextState.FontSize = 12;
textFragment3.TextState.Font = FontRepository.FindFont("TimesNewRoman");
textFragment3.TextState.Rotation = 90;
```

Σε αυτή την περίπτωση, η περιστροφή είναι 90 μοίρες.

## Βήμα 5: Προσθήκη τμημάτων κειμένου στη σελίδα PDF

Τώρα που έχουμε έτοιμα όλα τα τμήματα κειμένου μας, ήρθε η ώρα να τα προσθέσουμε στη σελίδα PDF χρησιμοποιώντας την κλάση TextBuilder.

```csharp
// δημιουργία αντικειμένου TextBuilder
TextBuilder textBuilder = new TextBuilder(pdfPage);
// Προσθήκη του τμήματος κειμένου στη σελίδα PDF
textBuilder.AppendText(textFragment1);
textBuilder.AppendText(textFragment2);
textBuilder.AppendText(textFragment3);
```

Η κλάση TextBuilder βοηθά στην προσθήκη πολλαπλών τμημάτων κειμένου σε μία σελίδα, δίνοντάς σας την ευελιξία να τα χειρίζεστε ξεχωριστά.

## Βήμα 6: Αποθήκευση του εγγράφου PDF

Τέλος, αποθηκεύστε το έγγραφο στον καθορισμένο κατάλογο. Χωρίς αυτό το βήμα, όλη η σκληρή δουλειά σας θα εξαφανιστεί στο πουθενά!

```csharp
// Αποθήκευση εγγράφου
pdfDocument.Save(dataDir + "TextFragmentTests_Rotated1_out.pdf");
```

Ολοκληρώσατε με επιτυχία την εναλλαγή κειμένου σε ένα αρχείο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Μπορείτε τώρα να ανοίξετε το PDF για να δείτε τα περιστραμμένα τμήματα κειμένου!

## Συχνές ερωτήσεις

### Μπορώ να περιστρέψω τμήματα κειμένου σε οποιαδήποτε γωνία;
Ναι! Μπορείτε να ρυθμίσετε το `TextState.Rotation` ιδιότητα σε οποιονδήποτε βαθμό (ακόμα και σε αρνητικές γωνίες) για να περιστρέψετε το κείμενο όπως απαιτείται.

### Μπορώ να χρησιμοποιήσω διαφορετικές γραμματοσειρές για κάθε τμήμα κειμένου;
Απολύτως. Μπορείτε να προσαρμόσετε τη γραμματοσειρά κάθε τμήματος κειμένου χρησιμοποιώντας `FontRepository.FindFont` και περάστε τη γραμματοσειρά που θέλετε να εφαρμόσετε.

### Υποστηρίζει το Aspose.PDF πολυσέλιδα PDF;
Ναι, μπορείτε να προσθέσετε πολλές σελίδες στο έγγραφο PDF σας και να χειριστείτε κάθε σελίδα ξεχωριστά.

### Υπάρχει όριο στον αριθμό των τμημάτων κειμένου που μπορώ να προσθέσω;
Όχι, μπορείτε να προσθέσετε όσα τμήματα κειμένου χρειάζεστε. Απλώς βεβαιωθείτε ότι είναι τοποθετημένα σωστά στη σελίδα.

### Μπορώ να τροποποιήσω τμήματα κειμένου αφού τα προσθέσω;
Ναι, μόλις προστεθεί ένα τμήμα κειμένου, μπορείτε να ενημερώσετε τις ιδιότητές του ή να το αφαιρέσετε από τη σελίδα.

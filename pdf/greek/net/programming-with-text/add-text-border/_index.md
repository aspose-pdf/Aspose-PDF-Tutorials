# Προσθήκη περιγράμματος κειμένου σε αρχείο PDF

## Εισαγωγή

Η δημιουργία και ο χειρισμός εγγράφων PDF έχει γίνει μια απαραίτητη δεξιότητα στον σημερινό ψηφιακό κόσμο. Είτε δημιουργείτε αναφορές, τιμολόγια είτε οποιοδήποτε άλλο είδος τεκμηρίωσης, ο έλεγχος του τρόπου εμφάνισης του κειμένου σας μπορεί να κάνει σημαντική διαφορά. Μια τέτοια βελτίωση που ίσως θελήσετε να εφαρμόσετε είναι η προσθήκη ενός περιγράμματος γύρω από το κείμενό σας σε ένα αρχείο PDF. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε στα βήματα για να προσθέσετε ένα περίγραμμα κειμένου σε ένα αρχείο PDF χρησιμοποιώντας τη βιβλιοθήκη Aspose.PDF για .NET. Ας ξεκινήσουμε, λοιπόν!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, υπάρχουν μερικά πράγματα που πρέπει να έχετε στη διάθεσή σας. Μην ανησυχείτε, είναι αρκετά απλό!

2. Aspose.PDF για .NET: Θα χρειαστεί να κατεβάσετε και να εγκαταστήσετε τη βιβλιοθήκη Aspose.PDF. Μπορείτε να την αποκτήσετε από το [Σελίδα λήψης Aspose PDF για .NET](https://releases.aspose.com/pdf/net/)Αν θέλετε να το δοκιμάσετε πρώτα, μπορείτε επίσης να αποκτήσετε ένα [δωρεάν δοκιμή εδώ](https://releases.aspose.com/).
3. Βασικές γνώσεις C#: Η βασική κατανόηση της γλώσσας προγραμματισμού C# θα σας βοηθήσει να παρακολουθήσετε εύκολα τα παραδείγματα.

## Εισαγωγή πακέτων

Τώρα που έχουμε ρυθμίσει τα πάντα, ας εισαγάγουμε τα απαραίτητα πακέτα για να χρησιμοποιήσουμε το Aspose.PDF στο έργο μας. Μπορείτε να το κάνετε αυτό προσθέτοντας τα ακόλουθα χρησιμοποιώντας οδηγίες στο επάνω μέρος του αρχείου C#:

```csharp
using Aspose.Pdf.Text;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;

```

Αυτοί οι χώροι ονομάτων θα σας επιτρέψουν να εργαστείτε αποτελεσματικά με έγγραφα PDF και αποσπάσματα κειμένου. 

Τώρα, ας αναλύσουμε τη διαδικασία προσθήκης ενός περιγράμματος κειμένου σε λεπτομερή βήματα. Θα εξετάσουμε κάθε βήμα, ώστε να κατανοήσετε ακριβώς τι συμβαίνει στο παρασκήνιο.

## Βήμα 1: Ρύθμιση του εγγράφου

Πρώτα απ 'όλα, πρέπει να δημιουργήσουμε ένα νέο έγγραφο PDF. Εδώ θα συμβεί όλη μας η μαγεία.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// Δημιουργία νέου αντικειμένου εγγράφου 
Document pdfDocument = new Document();
```

Σε αυτό το βήμα, καθορίζουμε τον κατάλογο όπου θέλουμε να αποθηκεύσουμε το αρχείο PDF. Στη συνέχεια, δημιουργούμε μια νέα παρουσία του `Document` κλάση, η οποία αντιπροσωπεύει το έγγραφο PDF μας.

## Βήμα 2: Προσθήκη νέας σελίδας

Στη συνέχεια, πρέπει να προσθέσουμε μια σελίδα στο έγγραφό μας. Σκεφτείτε το σαν να προσθέτουμε έναν κενό καμβά όπου θα τοποθετήσουμε το κείμενό μας.

```csharp
// Λήψη συγκεκριμένης σελίδας
Page pdfPage = (Page)pdfDocument.Pages.Add();
```

Εδώ, ονομάζουμε το `Add()` μέθοδος στο `Pages` συλλογή μας `pdfDocument` αντικείμενο. Αυτό προσθέτει μια νέα σελίδα στο έγγραφο και αποθηκεύουμε μια αναφορά σε αυτήν στο `pdfPage` μεταβλητός.

## Βήμα 3: Δημιουργήστε ένα τμήμα κειμένου

Τώρα, ας δημιουργήσουμε το κείμενο που θέλουμε να εμφανίζεται στο PDF μας. Εδώ ορίζουμε το περιεχόμενο του τμήματος κειμένου μας.

```csharp
// Δημιουργία τμήματος κειμένου
TextFragment textFragment = new TextFragment("main text");
textFragment.Position = new Position(100, 600);
```

Σε αυτόν τον κώδικα, δημιουργούμε ένα νέο `TextFragment` αντικείμενο με το κείμενο "κύριο κείμενο". Ορίζουμε επίσης τη θέση του στη σελίδα χρησιμοποιώντας το `Position` κλάση. Οι συντεταγμένες (100, 600) καθορίζουν πού θα τοποθετηθεί το κείμενο στη σελίδα.

## Βήμα 4: Ορισμός ιδιοτήτων κειμένου

Στη συνέχεια, θα προσαρμόσουμε το τμήμα κειμένου μας ώστε να είναι οπτικά ελκυστικό. Αυτό περιλαμβάνει τη ρύθμιση του μεγέθους γραμματοσειράς, του τύπου γραμματοσειράς, του χρώματος φόντου και του χρώματος προσκηνίου.

```csharp
// Ορισμός ιδιοτήτων κειμένου
textFragment.TextState.FontSize = 12;
textFragment.TextState.Font = FontRepository.FindFont("TimesNewRoman");
textFragment.TextState.BackgroundColor = Aspose.Pdf.Color.LightGray;
textFragment.TextState.ForegroundColor = Aspose.Pdf.Color.Red;
```

Εδώ, ορίζουμε το μέγεθος της γραμματοσειράς σε 12, χρησιμοποιούμε την γραμματοσειρά "Times New Roman" και εφαρμόζουμε ένα ανοιχτό γκρι χρώμα φόντου με κόκκινο κείμενο. Αυτές οι ιδιότητες βοηθούν στην ενίσχυση της ορατότητας του κειμένου.

## Βήμα 5: Ορίστε το χρώμα περιγράμματος για το περίγραμμα

Τώρα, φτάνουμε στο συναρπαστικό κομμάτι—την προσθήκη ενός περιγράμματος γύρω από το κείμενό μας!

```csharp
// Ορισμός της ιδιότητας StrokingColor για σχεδίαση περιγράμματος (χτενίσματος) γύρω από το ορθογώνιο κειμένου
textFragment.TextState.StrokingColor = Aspose.Pdf.Color.DarkRed;
```

Σε αυτό το βήμα, καθορίζουμε το χρώμα του περιγράμματος που θέλουμε να σχεδιάσουμε γύρω από το κείμενό μας. Εδώ, έχουμε επιλέξει ένα σκούρο κόκκινο χρώμα.

## Βήμα 6: Ενεργοποίηση του περιγράμματος ορθογωνίου κειμένου

Για να σχεδιάσουμε το περίγραμμα γύρω από το κείμενό μας, πρέπει να ενεργοποιήσουμε το `DrawTextRectangleBorder` ιδιοκτησία.

```csharp
// Ορισμός της τιμής της ιδιότητας DrawTextRectangleBorder σε true
textFragment.TextState.DrawTextRectangleBorder = true;
```

Ορίζοντας αυτήν την ιδιότητα σε `true`, λέμε στο Aspose.PDF να σχεδιάσει το περίγραμμα γύρω από το ορθογώνιο κειμένου με βάση το καθορισμένο χρώμα περιγράμματος.

## Βήμα 7: Προσθήκη του τμήματος κειμένου στη σελίδα

Τώρα που έχουμε έτοιμο το τμήμα κειμένου μας με όλες τις ιδιότητες που έχουν οριστεί, ήρθε η ώρα να το προσθέσουμε στη σελίδα.

```csharp
TextBuilder tb = new TextBuilder(pdfPage);
tb.AppendText(textFragment);
```

Εδώ, δημιουργούμε ένα `TextBuilder` αντικείμενο που σχετίζεται με εμάς `pdfPage`Στη συνέχεια χρησιμοποιούμε το `AppendText` μέθοδος για να προσθέσουμε το δικό μας `textFragment` στη σελίδα. 

## Βήμα 8: Αποθήκευση του εγγράφου

Τέλος, πρέπει να αποθηκεύσουμε το έγγραφο PDF στον καθορισμένο κατάλογο. Αυτή είναι η στιγμή της αλήθειας!

```csharp
// Αποθήκευση του εγγράφου
pdfDocument.Save(dataDir + "PDFWithTextBorder_out.pdf");
```

Σε αυτό το βήμα, ονομάζουμε το `Save` μέθοδος στο δικό μας `pdfDocument` αντικείμενο, παρέχοντας τη διαδρομή όπου θέλουμε να αποθηκεύσουμε το αρχείο. Μόλις εκτελέσετε τον κώδικα, θα πρέπει να βρείτε το νεοδημιουργημένο PDF με το περίγραμμα κειμένου στον καθορισμένο κατάλογο!

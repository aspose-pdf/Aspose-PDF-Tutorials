# Προσδιορίστε τον σωστό κωδικό πρόσβασης σε αρχείο PDF

## Εισαγωγή

Όσον αφορά την εργασία με αρχεία PDF, όλοι έχουμε συναντήσει εκείνη την εξοργιστική στιγμή όταν προσπαθείτε να ανοίξετε ένα έγγραφο, μόνο και μόνο για να συναντήσετε ένα εμπόδιο κωδικού πρόσβασης. Είναι ένα συνηθισμένο πρόβλημα που μπορεί να οδηγήσει είτε σε παραγωγική διαχείριση εγγράφων είτε σε μια απογοητευτική αναμέτρηση. Ευτυχώς, με την ισχυρή βιβλιοθήκη Aspose.PDF για .NET, μπορείτε να ανακτήσετε τον έλεγχο και να προσδιορίσετε εάν ένα αρχείο PDF προστατεύεται με κωδικό πρόσβασης και, εάν ναι, ποιος κωδικός πρόσβασης θα το ξεκλειδώσει. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε στη διαδικασία αναγνώρισης του σωστού κωδικού πρόσβασης για ένα προστατευμένο PDF χρησιμοποιώντας το Aspose.PDF, με εύκολα βήματα.

## Βήμα 1: Ορισμός της διαδρομής αρχείου

Πριν από οτιδήποτε άλλο, θα πρέπει να καθορίσετε τη διαδρομή προς το αρχείο PDF με το οποίο εργάζεστε. Βεβαιωθείτε ότι έχετε αντικαταστήσει το `"YOUR DOCUMENTS DIRECTORY"` με την πραγματική διαδρομή όπου βρίσκεται το αρχείο PDF σας.

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## Βήμα 2: Φόρτωση του αρχείου PDF προέλευσης

Στη συνέχεια, χρησιμοποιήστε `PdfFileInfo` για να φορτώσετε το αρχείο PDF πηγής:

```csharp
PdfFileInfo info = new PdfFileInfo();
info.BindPdf(dataDir + "IsPasswordProtected.pdf");
```

Αυτό το βήμα συνδέει το αρχείο PDF με το `info` αντικείμενο, επιτρέποντάς μας να έχουμε πρόσβαση στις ιδιότητές του.

## Βήμα 3: Ελέγξτε αν το PDF είναι κρυπτογραφημένο

Τώρα, ήρθε η ώρα να διαπιστώσετε εάν το έγγραφο PDF προστατεύεται πράγματι με κωδικό πρόσβασης:

```csharp
Console.WriteLine("File is password protected " + info.IsEncrypted);
```

Ελέγχοντας το `IsEncrypted` ιδιότητα, μπορείτε να διαπιστώσετε την κατάσταση κλειδώματος του εγγράφου. Εάν είναι `true`, τότε θα πρέπει να σπάσεις τον κώδικα!

## Βήμα 4: Προετοιμάστε μια λίστα με πιθανούς κωδικούς πρόσβασης

Για να ξεκινήσετε μια αναζήτηση κωδικών πρόσβασης, προετοιμάστε έναν πίνακα συμβολοσειρών που περιέχει πιθανούς κωδικούς πρόσβασης που θέλετε να δοκιμάσετε:

```csharp
String[] passwords = new String[5] { "test", "test1", "test2", "test3", "sample" };
```

Μπορείτε να τροποποιήσετε αυτόν τον πίνακα με βάση τις ανάγκες σας ή τους πιο πιθανούς κωδικούς πρόσβασης.

## Βήμα 5: Προσπαθήστε να ανοίξετε το PDF με κάθε κωδικό πρόσβασης

Τώρα, θα επαναλάβουμε κάθε κωδικό πρόσβασης, προσπαθώντας να ανοίξουμε το αρχείο PDF. 

```csharp
for (int passwordcount = 0; passwordcount < passwords.Length; passwordcount++)
{
    try
    {
        Document doc = new Document(dataDir + "IsPasswordProtected.pdf", passwords[passwordcount]);
        if (doc.Pages.Count > 0)
            Console.WriteLine("Number of Page in document are = " + doc.Pages.Count);
    }
    catch (InvalidPasswordException)
    {
        Console.WriteLine("Password = " + passwords[passwordcount] + "  is not correct");
    }
}
```

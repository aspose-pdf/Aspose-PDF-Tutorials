## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα δούμε πώς να προσθέτουμε διαφορετικά χρωματιστά επίπεδα γραμμών στα έγγραφα PDF σας χρησιμοποιώντας το Aspose.PDF για .NET.

### Προσθήκη επιπέδου κόκκινης γραμμής
Η στρώση κόκκινης γραμμής μπορεί να είναι ιδιαίτερα χρήσιμη για την επισήμανση σημαντικών ενοτήτων ή τη δημιουργία οπτικών οδηγών μέσα στο έγγραφό σας.

#### Επισκόπηση
Θα δημιουργήσουμε και θα προσθέσουμε μια κόκκινη γραμμή στην πρώτη σελίδα του εγγράφου PDF μας.

#### Βήματα:

**1. Δημιουργήστε μια παρουσία εγγράφου**
```csharp
Document doc = new Document();
```
- **Γιατί**: Α `Document` Το αντικείμενο αντιπροσωπεύει ολόκληρο το αρχείο PDF με το οποίο εργάζεστε.

**2. Προσθήκη σελίδας**
```csharp
Page page = doc.Pages.Add();
```
- **Γιατί**Οι σελίδες είναι θεμελιώδη στοιχεία οποιουδήποτε εγγράφου PDF.

**3. Ορισμός και διαμόρφωση κόκκινης στρώσης**
```csharp
Layer redLayer = new Layer("oc1", "Red Line");
redLayer.Contents.Add(new SetRGBColorStroke(1, 0, 0));
redLayer.Contents.Add(new MoveTo(500, 700));
redLayer.Contents.Add(new LineTo(400, 700));
redLayer.Contents.Add(new Stroke());
```
- **Γιατί**: Το `SetRGBColorStroke` ορίζει το χρώμα της γραμμής. `MoveTo` και `LineTo` ορίστε τα σημεία έναρξης και λήξης της γραμμής.

**4. Προσθήκη επιπέδου στη σελίδα**
```csharp
page.Layers = new List<Layer>();
page.Layers.Add(redLayer);
```
- **Γιατί**Τα επίπεδα σάς επιτρέπουν να διαχειρίζεστε διαφορετικά στοιχεία στο PDF σας ανεξάρτητα.

**5. Αποθήκευση του εγγράφου**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "AddRedLine_out.pdf";
doc.Save(dataDir);
```
- **Γιατί**: Η αποθήκευση δημιουργεί ένα φυσικό αρχείο που αντικατοπτρίζει όλες τις αλλαγές σας.

### Προσθήκη επιπέδου πράσινης γραμμής
Οι πράσινες γραμμές μπορούν να χρησιμοποιηθούν για να διακρίνουν διαφορετικά τμήματα ή να επισημαίνουν διαδρομές προόδου στα σχέδια του έργου.

#### Επισκόπηση
Θα προσθέσουμε μια πράσινη γραμμή στο ίδιο έγγραφο PDF, δείχνοντας πώς μπορούν να συνυπάρχουν πολλαπλά επίπεδα.

#### Βήματα:

**1. Δημιουργία και προσθήκη σελίδας**
Επαναλάβετε τα βήματα από την ενότητα κόκκινης στρώσης για την αρχικοποίηση `Document` και προσθήκη σελίδας.

**2. Ορισμός και διαμόρφωση ενός πράσινου επιπέδου**
```csharp
Layer greenLayer = new Layer("oc2", "Green Line");
greenLayer.Contents.Add(new SetRGBColorStroke(0, 1, 0));
greenLayer.Contents.Add(new MoveTo(500, 750));
greenLayer.Contents.Add(new LineTo(400, 750));
greenLayer.Contents.Add(new Stroke());
```
- **Γιατί**Παρόμοιο με την κόκκινη γραμμή αλλά με διαφορετική τιμή χρώματος RGB.

**3. Προσθήκη επιπέδου στη σελίδα**
Ακολουθήστε παρόμοια βήματα όπως και με την προσθήκη της κόκκινης στρώσης, διασφαλίζοντας ότι κάθε στρώση έχει αρχικοποιηθεί και προστεθεί σωστά. `page.Layers`.

**4. Αποθήκευση του εγγράφου**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "AddGreenLine_out.pdf";
doc.Save(dataDir);
```

### Προσθήκη επιπέδου μπλε γραμμής
Οι μπλε γραμμές μπορούν να είναι ιδανικές για την ένδειξη ορίων ή τη σύνδεση διαφορετικών τμημάτων του εγγράφου σας.

#### Επισκόπηση
Θα προσθέσουμε μια στρώση μπλε γραμμής για να συμπληρώσουμε τις υπάρχουσες κόκκινες και πράσινες στρώσεις.

#### Βήματα:

**1. Δημιουργία και προσθήκη σελίδας**
Επαναλάβετε τα βήματα από τις προηγούμενες ενότητες για τη ρύθμιση `Document` και προσθήκη σελίδας.

**2. Ορισμός και διαμόρφωση μπλε επιπέδου**
```csharp
Layer blueLayer = new Layer("oc3", "Blue Line");
blueLayer.Contents.Add(new SetRGBColorStroke(0, 0, 1));
blueLayer.Contents.Add(new MoveTo(500, 800));
blueLayer.Contents.Add(new LineTo(400, 800));
blueLayer.Contents.Add(new Stroke());
```
- **Γιατί**Οι τιμές RGB ορίζουν το μπλε χρώμα και `MoveTo`/`LineTo` χαράξει την πορεία του.

**3. Προσθήκη επιπέδου στη σελίδα**
Βεβαιωθείτε ότι κάθε στρώση έχει προστεθεί σωστά όπως έχει αποδειχθεί προηγουμένως.

**4. Αποθήκευση του εγγράφου**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "AddBlueLine_out.pdf";
doc.Save(dataDir);
```

## Οδηγός Εφαρμογής

### Ορισμός τίτλου και εναλλακτικού κειμένου για PDF με ετικέτες
**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να ορίσετε έναν τίτλο για ολόκληρο το έγγραφό σας και να παρέχετε εναλλακτικές περιγραφές κειμένου για εικόνες, καθιστώντας τες προσβάσιμες σε προγράμματα ανάγνωσης οθόνης.

#### Βήμα 1: Εισαγωγή απαιτούμενων χώρων ονομάτων
```csharp
using Aspose.Pdf.LogicalStructure;
using Aspose.Pdf.Tagged;
```

#### Βήμα 2: Ανοίξτε το έγγραφο PDF
```csharp
Document document = new Document("YOUR_DOCUMENT_DIRECTORY/TH.pdf");
ITaggedContent taggedContent = document.TaggedContent;
StructureElement rootElement = taggedContent.RootElement;
```

#### Βήμα 3: Ορίστε τον τίτλο για το PDF με ετικέτες
```csharp
taggedContent.SetTitle("Document with images");
```
**Εξήγηση:**  
Ο `SetTitle` Η μέθοδος αντιστοιχίζει έναν τίτλο σε ολόκληρο το έγγραφό σας, βελτιώνοντας την προσβασιμότητά του.

#### Βήμα 4: Αντιστοίχιση εναλλακτικού κειμένου σε εικόνες
```csharp
foreach (FigureElement figureElement in rootElement.FindElements<FigureElement>(true)) {
    figureElement.AlternativeText = "Figure alternative text (technique 2)";
}
```
**Εξήγηση:**  
Ο `AlternativeText` Η ιδιότητα παρέχει μια περιγραφή για κάθε εικόνα, απαραίτητη για χρήστες με προβλήματα όρασης.

#### Βήμα 5: Αποθήκευση του εγγράφου
```csharp
document.Save("YOUR_OUTPUT_DIRECTORY/TH_out.pdf");
```
### Ορισμός χαρακτηριστικού BBox για στοιχεία σχήματος σε PDF με ετικέτες
**Επισκόπηση:**
Αυτή η λειτουργία ορίζει χαρακτηριστικά διάταξης όπως πλαίσια οριοθέτησης (BBox) για στοιχεία σχήματος, ορίζοντας τις χωρικές τους ιδιότητες μέσα σε μια σελίδα.

#### Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων
```csharp
using Aspose.Pdf.LogicalStructure;
using Aspose.Pdf.Tagged;
```

#### Βήμα 2: Ανοίξτε το έγγραφο PDF
```csharp
Document document = new Document("YOUR_DOCUMENT_DIRECTORY/TH.pdf");
ITaggedContent taggedContent = document.TaggedContent;
StructureElement rootElement = taggedContent.RootElement;
```

#### Βήμα 3: Δημιουργία και ορισμός χαρακτηριστικού BBox
```csharp
foreach (FigureElement figureElement in rootElement.FindElements<FigureElement>(true)) {
    StructureAttribute bboxAttribute = new StructureAttribute(AttributeKey.BBox);
    bboxAttribute.SetRectangleValue(new Rectangle(0.0, 0.0, 100.0, 100.0));

    StructureAttributes figureLayoutAttributes = figureElement.Attributes.GetAttributes(AttributeOwnerStandard.Layout);
    figureLayoutAttributes.SetAttribute(bboxAttribute);
}
```
**Εξήγηση:**  
Ο `SetRectangleValue` Η μέθοδος ορίζει τη θέση και το μέγεθος των σχημάτων σας μέσα στη διάταξη σελίδας.

#### Βήμα 4: Αποθήκευση του ενημερωμένου εγγράφου
```csharp
document.Save("YOUR_OUTPUT_DIRECTORY/TH_out_boundingbox.pdf");
```
### Μετακίνηση στοιχείου Span σε παράγραφο σε PDF με ετικέτες
**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να μετακινήσετε ένα στοιχείο span σε μια παράγραφο, βελτιώνοντας τη δομή και την προσβασιμότητα του εγγράφου.

#### Βήμα 1: Εισαγωγή απαιτούμενων χώρων ονομάτων
```csharp
using Aspose.Pdf.LogicalStructure;
using Aspose.Pdf.Tagged;
```

#### Βήμα 2: Ανοίξτε το έγγραφο PDF
```csharp
Document document = new Document("YOUR_DOCUMENT_DIRECTORY/TH.pdf");
ITaggedContent taggedContent = document.TaggedContent;
StructureElement rootElement = taggedContent.RootElement;
```

#### Βήμα 3: Εύρεση και μετακίνηση στοιχείων
```csharp
TableElement tableElement = rootElement.FindElements<TableElement>(true)[0];
SpanElement spanElement = tableElement.FindElements<SpanElement>(true)[0];
TableTDElement firstTdElement = tableElement.FindElements<TableTDElement>(true)[0];
ParagraphElement paragraph = firstTdElement.FindElements<ParagraphElement>(true)[0];

spanElement.ChangeParentElement(paragraph);
```
**Εξήγηση:**  
Η αλλαγή του γονικού στοιχείου ενός span σε παράγραφο διασφαλίζει καλύτερη σημασιολογική δομή.

#### Βήμα 4: Αποθήκευση αλλαγών
```csharp
document.Save("YOUR_OUTPUT_DIRECTORY/TH_out_movedspan.pdf");
```

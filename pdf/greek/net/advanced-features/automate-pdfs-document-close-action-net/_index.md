## Οδηγός Εφαρμογής
Τώρα που η ρύθμιση ολοκληρώθηκε, ας προσθέσουμε μια ενέργεια κλεισίματος εγγράφου στο αρχείο PDF σας.

### Προσθήκη εγγράφου Ενέργεια κλεισίματος
Αυτή η λειτουργία εκτελεί JavaScript όταν ο χρήστης κλείνει το έγγραφο PDF. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Προετοιμάστε το περιβάλλον σας
Βεβαιωθείτε ότι έχετε ένα υπάρχον αρχείο PDF με το όνομα `CreateDocumentLink.pdf` στον καθορισμένο κατάλογο.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
```

#### Βήμα 2: Ανοίξτε το έγγραφο PDF
Χρησιμοποιώ `PdfContentEditor` για να ανοίξετε και να επεξεργαστείτε το PDF:
```csharp
PdfContentEditor contentEditor = new PdfContentEditor();
contentEditor.BindPdf(dataDir + "CreateDocumentLink.pdf");
```
Αυτό το βήμα συνδέει το υπάρχον έγγραφό σας για επεξεργασία.

#### Βήμα 3: Ορίστε την ενέργεια κλεισίματος
Καθορίστε την ενέργεια χρησιμοποιώντας `AddDocumentAdditionalAction`Μια ειδοποίηση JavaScript ενεργοποιείται κατά το κλείσιμο:
```csharp
contentEditor.AddDocumentAdditionalAction(PdfContentEditor.DocumentClose,
    "app.alert('Thank you for using Aspose products!');");
```
Ο `PdfContentEditor.DocumentClose` Η παράμετρος προσδιορίζει το συμβάν και η συμβολοσειρά JavaScript ορίζει την ενέργεια.

#### Βήμα 4: Αποθηκεύστε το ενημερωμένο PDF
Αφού ρυθμίσετε το έγγραφό σας με πρόσθετες ενέργειες, αποθηκεύστε το για να εφαρμόσετε τις αλλαγές:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/";
contentEditor.Save(outputDir + "CreateDocAdditionalAction_out.pdf");
```
Αυτό το βήμα αποθηκεύει το τροποποιημένο PDF σας στην επιθυμητή θέση.

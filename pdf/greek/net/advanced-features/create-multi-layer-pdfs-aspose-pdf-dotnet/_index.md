## Οδηγός Εφαρμογής (H2)
Θα χωρίσουμε τη διαδικασία σε διαχειρίσιμα βήματα.

### Δημιουργία του εγγράφου PDF (H3)
**Επισκόπηση:** Ξεκινήστε δημιουργώντας ένα νέο έγγραφο και προσθέτοντας μια σελίδα σε αυτό.
```csharp
Document pdf = new Aspose.Pdf.Document();
Page sec1 = pdf.Pages.Add(); // Προσθήκη νέας σελίδας
```
- `Aspose.Pdf.Document`: Αντιπροσωπεύει ολόκληρο το PDF σας.
- `Pages.Add()`: Προσθέτει μια νέα σελίδα όπου μπορείτε να τοποθετήσετε περιεχόμενο.

### Προσθήκη τμημάτων κειμένου (H3)
**Επισκόπηση:** Εισαγάγετε τμήματα κειμένου με επιλογές μορφοποίησης όπως χρώμα και μέγεθος γραμματοσειράς.
```csharp
TextFragment t1 = new Aspose.Pdf.Text.TextFragment("paragraph 3 segment");
sec1.Paragraphs.Add(t1);

t1.TextState.ForegroundColor = Color.Red; // Ορισμός χρώματος κειμένου σε κόκκινο
t1.TextState.FontSize = 12;                // Ορισμός μεγέθους γραμματοσειράς σε 12
```
- `TextFragment`: Αντιπροσωπεύει ένα μπλοκ κειμένου.
- `TextState.ForegroundColor`: Ορίζει το χρώμα του κειμένου.
- `TextState.FontSize`: Ελέγχει το μέγεθος της γραμματοσειράς.

### Εισαγωγή εικόνων (H3)
**Επισκόπηση:** Προσθέστε εικόνες στο έγγραφο PDF σας, εμπλουτίζοντας οπτικά το περιεχόμενό του.
```csharp
Image image1 = new Aspose.Pdf.Image();
image1.File = "path\to\your\test_image.png"; // Καθορίστε τη διαδρομή της εικόνας
sec1.Paragraphs.Add(image1);
```
- `Aspose.Pdf.Image`: Αντιπροσωπεύει μια εικόνα στο έγγραφό σας.
- `image1.File`: Ορίζει τη διαδρομή αρχείου για την εικόνα.

### Προσθήκη αιωρούμενων πλαισίων (H3)
**Επισκόπηση:** Χρησιμοποιήστε αιωρούμενα πλαίσια για να οργανώσετε και να τοποθετήσετε αποτελεσματικά το περιεχόμενο μέσα σε μια σελίδα.
```csharp
FloatingBox box1 = new Aspose.Pdf.FloatingBox(117, 21);
box1.Left = -4; // Τοποθέτηση από την αριστερή άκρη
box1.Top = -4;  // Τοποθέτηση από την πάνω άκρη
sec1.Paragraphs.Add(box1);

box1.Paragraphs.Add(image1); // Προσθήκη εικόνας στο αιωρούμενο πλαίσιο
```
- `FloatingBox`: Ένα κοντέινερ για την οργάνωση περιεχομένου.
- Χαρακτηριστικά θέσης (`Left`, `Top`): Ορίστε τη θέση του πλαισίου στη σελίδα.

### Αποθήκευση του εγγράφου PDF (H3)
**Επισκόπηση:** Τέλος, αποθηκεύστε το έγγραφό σας σε ένα αρχείο.
```csharp
pdf.Save("path\to\your\CreateMultiLayerPdf_out.pdf");
```
- `Document.Save()`: Εγγράφει την τελική έξοδο στο δίσκο.

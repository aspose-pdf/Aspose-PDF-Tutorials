## Οδηγός Εφαρμογής
Εξερευνήστε την πρόσβαση και την τροποποίηση θυγατρικών στοιχείων σε PDF με ετικέτες με λεπτομερείς οδηγίες.

### Πρόσβαση σε θυγατρικά στοιχεία
Η πρόσβαση σε θυγατρικά στοιχεία είναι ζωτικής σημασίας για τον χειρισμό της λογικής δομής ενός PDF. Αυτή η ενότητα σας καθοδηγεί στην ανάκτηση αυτών των στοιχείων χρησιμοποιώντας το API του Aspose.PDF.

#### Βήμα προς βήμα εφαρμογή
**Λήψη περιεχομένου με ετικέτες**
Ανακτήστε το περιεχόμενο με ετικέτες από το έγγραφό σας:
```csharp
ITaggedContent taggedContent = document.TaggedContent;
```

**Πρόσβαση στα παιδιά του στοιχείου Root**
Για να αποκτήσετε πρόσβαση σε θυγατρικά στοιχεία του ριζικού στοιχείου:
```csharp
ElementList elementList = taggedContent.StructTreeRootElement.ChildElements;

foreach (Element element in elementList)
{
    if (element is StructureElement structureElement)
    {
        // Ανάκτηση ιδιοτήτων
        string title = structureElement.Title;
        string language = structureElement.Language;
        string actualText = structureElement.ActualText;
        string expansionText = structureElement.ExpansionText;
        string alternativeText = structureElement.AlternativeText;

        // Παράδειγμα χρήσης:
        Console.WriteLine($"Title: {title}, Language: {language}");
    }
}
```
#### Εξήγηση
- **Λίστα Στοιχείων**: Αντιπροσωπεύει μια συλλογή από θυγατρικά στοιχεία.
- **Στοιχείο Δομής**: Ένας συγκεκριμένος τύπος στοιχείου με πρόσθετες ιδιότητες όπως τίτλος και γλώσσα.

### Τροποποίηση θυγατρικών στοιχείων
Η τροποποίηση των θυγατρικών στοιχείων σάς επιτρέπει να προσαρμόσετε τη δομή και τις δυνατότητες προσβασιμότητας των PDF σας. Αυτή η ενότητα καλύπτει τον τρόπο αποτελεσματικής ρύθμισης αυτών των ιδιοτήτων.

**Ορισμός ιδιοτήτων για ένα συγκεκριμένο στοιχείο**
Για να τροποποιήσετε τις ιδιότητες ενός στοιχείου:
```csharp
elementList = taggedContent.RootElement.ChildElements[1].ChildElements;

foreach (Element element in elementList)
{
    if (element is StructureElement structureElement)
    {
        // Ορισμός ιδιοτήτων
        structureElement.Title = "Modified Title";
        structureElement.Language = "en-US";
        structureElement.ActualText = "Updated Actual Text";
        structureElement.ExpansionText = "exp";
        structureElement.AlternativeText = "alt";

        // Παράδειγμα χρήσης:
        Console.WriteLine($"New Title: {structureElement.Title}");
    }
}
```
#### Εξήγηση
- **Τίτλος, Γλώσσα**: Ορίζεται για να οριστούν τα χαρακτηριστικά του στοιχείου.
- **Πραγματικό Κείμενο, Κείμενο Επέκτασης, Εναλλακτικό Κείμενο**: Παροχή περιεχομένου κειμένου για εργαλεία προσβασιμότητας.

### Αποθήκευση αλλαγών
Αποθηκεύστε τις τροποποιήσεις σας:
```csharp
document.Save("ModifiedDocument.pdf");
```

## Οδηγός Εφαρμογής

### Ορισμός προκαθορισμένων ιδιοτήτων για το παράθυρο διαλόγου εκτύπωσης (H2)
Αυτή η λειτουργία επιδεικνύει τον ορισμό ιδιοτήτων αμφίδρομης εκτύπωσης σε ένα έγγραφο PDF. Ας εξερευνήσουμε πώς να το ρυθμίσετε χρησιμοποιώντας το Aspose.PDF.

#### Επισκόπηση
Ο ακόλουθος κώδικας ορίζει την ιδιότητα διπλής όψης έτσι ώστε οι σελίδες να εκτυπώνονται κατά μήκος της μεγάλης πλευράς, ιδανική για επαγγελματικές παρουσιάσεις ή αναφορές.

#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργία και διαμόρφωση εγγράφου**
```csharp
using Aspose.Pdf;

// Αρχικοποίηση νέου εγγράφου PDF
Document doc = new Document();

// Προσθήκη σελίδας στο έγγραφο
doc.Pages.Add();

// Ορισμός ιδιότητας διπλής όψης
doc.Duplex = PrintDuplex.DuplexFlipLongEdge;
```
- **Εξήγηση:** Δημιουργούμε ένα νέο `Document` παράδειγμα και προσθήκη μιας σελίδας. Ρύθμιση `doc.Duplex` να `PrintDuplex.DuplexFlipLongEdge` διασφαλίζει ότι οι σελίδες εκτυπώνονται κατά μήκος της μακρύτερης πλευράς τους.

**2. Αποθήκευση του εγγράφου**
```csharp
// Ορισμός διαδρομής αρχείου εξόδου
string outputFilePath = "YOUR_OUTPUT_DIRECTORY/SetPresetPropertiesForPrintDialog_out.pdf";

// Αποθήκευση εγγράφου με καθορισμένες ρυθμίσεις
doc.Save(outputFilePath, SaveFormat.Pdf);
```
- **Εξήγηση:** Ο `Save` Η μέθοδος γράφει το διαμορφωμένο PDF σας στο δίσκο. Θυμηθείτε να αντικαταστήσετε `"YOUR_OUTPUT_DIRECTORY"` με την πραγματική διαδρομή όπου θέλετε να αποθηκεύσετε το αρχείο.

### Ορισμός ιδιοτήτων παραθύρου διαλόγου εκτύπωσης χρησιμοποιώντας το πρόγραμμα επεξεργασίας περιεχομένου PDF (H2)
Για τα υπάρχοντα έγγραφα, η τροποποίηση των προτιμήσεων του προγράμματος προβολής μπορεί να είναι κρίσιμη για τη συνεπή συμπεριφορά εκτύπωσης σε διαφορετικές πλατφόρμες.

#### Επισκόπηση
Αυτή η ενότητα τροποποιεί τις ρυθμίσεις διπλής όψης ενός υπάρχοντος PDF χρησιμοποιώντας το PdfContentEditor του Aspose.PDF.

#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση και Δέσμευση Εγγράφου**
```csharp
using Aspose.Pdf.Facades;

string inputFile = "YOUR_DOCUMENT_DIRECTORY/input.pdf";
string documentPath = "YOUR_OUTPUT_DIRECTORY/SetPrintDlgPropertiesUsingPdfContentEditor_out.pdf";

// Δημιουργήστε μια παρουσία του PdfContentEditor
using (PdfContentEditor editor = new PdfContentEditor())
{
    // Δέστε το PDF για επεξεργασία
    editor.BindPdf(inputFile);
```
- **Εξήγηση:** `PdfContentEditor` επιτρέπει τροποποιήσεις σε υπάρχοντα PDF. Το έγγραφο είναι δεσμευμένο για επεξεργασία καθορίζοντας τη διαδρομή του.

**2. Τροποποίηση προτιμήσεων θεατή**
```csharp
    // Ανάκτηση και έλεγχος των τρεχουσών προτιμήσεων
    ViewerPreferences prefs = editor.GetViewerPreference();
    
    if ((prefs & ViewerPreference.DuplexFlipShortEdge) > 0)
    {
        // Η τρέχουσα προτίμηση περιλαμβάνει αναστροφή από τη μικρή πλευρά
    }

    // Ορισμός νέας προτίμησης θεατή για αναστροφή από τη μικρή πλευρά
    editor.ChangeViewerPreference(ViewerPreference.DuplexFlipShortEdge);
```
- **Εξήγηση:** Αυτό ελέγχει τις τρέχουσες ρυθμίσεις και τις ενημερώνει ώστε να αναστρέφονται κατά μήκος της μικρότερης πλευράς του χαρτιού, βελτιώνοντας την ευελιξία στις διατάξεις εκτύπωσης.

**3. Αποθήκευση αλλαγών**
```csharp
    // Αποθήκευση του τροποποιημένου εγγράφου
    editor.Save(documentPath);
}
```
- **Εξήγηση:** Ο `Save` Η μέθοδος επιμένει και αλλάζει πίσω στην τοποθεσία αποθήκευσής σας.

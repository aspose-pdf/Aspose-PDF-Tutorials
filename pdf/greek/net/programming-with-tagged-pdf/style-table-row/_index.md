# Στυλ Πίνακα Σειρά

## Εισαγωγή

Όσον αφορά τη δημιουργία καλά δομημένων και όμορφα μορφοποιημένων εγγράφων PDF, το Aspose.PDF για .NET είναι μια λύση που μπορείτε να χρησιμοποιήσετε. Είτε αυτοματοποιείτε αναφορές, τιμολόγια είτε δημιουργείτε δυναμικούς πίνακες, η μορφοποίηση πινάκων με διάφορα στυλ είναι το κλειδί για ένα κομψό έγγραφο. Σε αυτό το σεμινάριο, θα εμβαθύνουμε στη διαμόρφωση μιας γραμμής πίνακα χρησιμοποιώντας το Aspose.PDF για .NET. Και μην ανησυχείτε, θα σας καθοδηγήσω βήμα προς βήμα, όπως ακριβώς μια καλή συζήτηση με καφέ!

## Εισαγωγή πακέτων

Πριν ξεκινήσουμε να δουλεύουμε με το Aspose.PDF, πρέπει να εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Στο έργο σας σε C#, βεβαιωθείτε ότι έχετε συμπεριλάβει τα εξής:

```csharp
using Aspose.Pdf.LogicalStructure;
using Aspose.Pdf.Tagged;
using Aspose.Pdf.Text;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

Αυτά είναι απαραίτητα για τη δημιουργία και τη διαμόρφωση του πίνακα και, φυσικά, για την εργασία με περιεχόμενο με ετικέτες για λόγους συμμόρφωσης.

Τώρα ας αναλύσουμε την εργασία βήμα προς βήμα, ώστε να μπορείτε να διαμορφώσετε τις σειρές του πίνακά σας σαν επαγγελματίας!

## Βήμα 1: Δημιουργήστε ένα νέο έγγραφο PDF

Πρώτα απ 'όλα: ας δημιουργήσουμε ένα ολοκαίνουργιο έγγραφο PDF. Αυτό το έγγραφο θα περιέχει όλες τις γραμμές του πίνακα με στυλ.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Δημιουργία εγγράφου
Document document = new Document();
```

Εδώ, απλώς αρχικοποιούμε ένα νέο `Document` αντικείμενο που θα αντιπροσωπεύει το αρχείο PDF μας. Βεβαιωθείτε ότι έχετε ορίσει τη διαδρομή καταλόγου όπου θα αποθηκεύσετε τα αρχεία εξόδου σας.

## Βήμα 2: Εργασία με περιεχόμενο με ετικέτες

Για να δομήσουμε το PDF σας με γνώμονα την προσβασιμότητα, θα χρησιμοποιήσουμε περιεχόμενο με ετικέτες. Αυτό βοηθά στη δημιουργία δομημένων στοιχείων όπως πίνακες, διασφαλίζοντας ότι συμμορφώνονται με πρότυπα προσβασιμότητας όπως το PDF/UA.

```csharp
ITaggedContent taggedContent = document.TaggedContent;
taggedContent.SetTitle("Example table row style");
taggedContent.SetLanguage("en-US");
```

Εδώ, ορίζουμε τον τίτλο και τη γλώσσα για το περιεχόμενο με ετικέτες του PDF. Είναι σαν να δίνετε στο PDF σας ένα όνομα και να του λέτε σε ποια γλώσσα πρέπει να μιλάει!

## Βήμα 3: Ορισμός της δομής του πίνακα

Στη συνέχεια, ας ορίσουμε τη δομή του πίνακα που πρόκειται να δημιουργήσουμε. Κάθε πίνακας χρειάζεται κεφαλίδα, σώμα και υποσέλιδο – όπως ακριβώς μια καλά οργανωμένη ανάρτηση ιστολογίου!

```csharp
// Λήψη στοιχείου δομής ρίζας
StructureElement rootElement = taggedContent.RootElement;

// Δημιουργία στοιχείου δομής πίνακα
TableElement tableElement = taggedContent.CreateTableElement();
rootElement.AppendChild(tableElement);
TableTHeadElement tableTHeadElement = tableElement.CreateTHead();
TableTBodyElement tableTBodyElement = tableElement.CreateTBody();
TableTFootElement tableTFootElement = tableElement.CreateTFoot();
```

Αυτό που κάνουμε εδώ είναι να δημιουργήσουμε έναν πίνακα με κεφαλίδα (`THead`), σώμα (`TBody`), και υποσέλιδο (`TFoot`). Αυτά τα στοιχεία θα συγκρατήσουν τις σειρές μας.

## Βήμα 4: Προσθήκη της γραμμής κεφαλίδας πίνακα

Οι πίνακες χωρίς κεφαλίδες είναι σαν βιβλία χωρίς τίτλους. Ας δημιουργήσουμε πρώτα τη γραμμή κεφαλίδας για να παρέχουμε το περιβάλλον για τα δεδομένα.

```csharp
TableTRElement headTrElement = tableTHeadElement.CreateTR();
headTrElement.AlternativeText = "Head Row";
for (int colIndex = 0; colIndex < 3; colIndex++)
{
    TableTHElement thElement = headTrElement.CreateTH();
    thElement.SetText(String.Format("Head {0}", colIndex));
}
```

Εδώ, κάνουμε επανάληψη και προσθέτουμε τρία κελιά κεφαλίδας (`TableTHElement`), δίνοντας σε κάθε ένα ένα περιγραφικό κείμενο. Απλό, σωστά;

## Βήμα 5: Προσθήκη στυλιζαρισμένων γραμμών σώματος

Τώρα έρχεται το διασκεδαστικό κομμάτι – η διαμόρφωση των γραμμών! Ας δημιουργήσουμε επτά γραμμές με προσαρμοσμένα στυλ. Θα ορίσουμε χρώματα φόντου, περιγράμματα, συμπλήρωση και στοίχιση κειμένου.

```csharp
for (int rowIndex = 0; rowIndex < 7; rowIndex++)
{
    TableTRElement trElement = tableTBodyElement.CreateTR();
    trElement.AlternativeText = String.Format("Row {0}", rowIndex);
    trElement.BackgroundColor = Color.LightGoldenrodYellow;
    trElement.Border = new BorderInfo(BorderSide.All, 0.75F, Color.DarkGray);
    trElement.DefaultCellBorder = new BorderInfo(BorderSide.All, 0.50F, Color.Blue);
    trElement.MinRowHeight = 100.0;
    trElement.FixedRowHeight = 120.0;
    trElement.IsInNewPage = (rowIndex % 3 == 1);
    trElement.IsRowBroken = true;

    for (int colIndex = 0; colIndex < 3; colIndex++)
    {
        TableTDElement tdElement = trElement.CreateTD();
        tdElement.SetText(String.Format("Cell [{0}, {1}]", rowIndex, colIndex));
    }
}
```

- Χρώμα φόντου: Χρησιμοποιήσαμε ένα ανοιχτό κίτρινο χρυσής ράβδου για μια επαγγελματική αλλά ζεστή πινελιά.
- Περιγράμματα: Κάθε σειρά έχει ένα σκούρο γκρι εξωτερικό περίγραμμα και μπλε περιγράμματα κελιών για πιο ευκρινή εμφάνιση.
- Ύψος και συμπλήρωση: Τα ύψη των γραμμών ορίζονται και προστίθεται συμπλήρωση για καθαρή εμφάνιση.
- Αλλαγές σελίδας: Για να γίνει ο πίνακας πιο ευανάγνωστος, κάθε δεύτερη γραμμή ξεκινά σε μια νέα σελίδα.

## Βήμα 6: Προσθήκη της γραμμής υποσέλιδου

Όπως ακριβώς και η κεφαλίδα, το υποσέλιδο αποτελεί αγκύρωση του πίνακα. Ας δημιουργήσουμε ένα.

```csharp
TableTRElement footTrElement = tableTFootElement.CreateTR();
footTrElement.AlternativeText = "Foot Row";
for (int colIndex = 0; colIndex < 3; colIndex++)
{
    TableTDElement tdElement = footTrElement.CreateTD();
    tdElement.SetText(String.Format("Foot {0}", colIndex));
}
```

Απλώς κάνουμε επανάληψη σε τρία κελιά υποσέλιδου και προσθέτουμε λίγο κείμενο. Το εναλλακτικό κείμενο για το υποσέλιδο είναι "Foot Row" για να είναι προσβάσιμο.

## Βήμα 7: Αποθήκευση του εγγράφου PDF

Τώρα που το τραπέζι είναι έτοιμο, ήρθε η ώρα να αποθηκεύσετε το αριστούργημά σας!

```csharp
document.Save(dataDir + "StyleTableRow.pdf");
```

Έτσι απλά, το PDF σας αποθηκεύεται με όλες τις όμορφες σειρές πίνακα που μόλις διαμορφώσαμε.

## Βήμα 8: Επικύρωση συμμόρφωσης με PDF/UA

Για να διασφαλίσουμε ότι το PDF μας συμμορφώνεται με τα πρότυπα προσβασιμότητας, θα το επικυρώσουμε για συμμόρφωση με τα πρότυπα PDF/UA.

```csharp
document = new Document(dataDir + "StyleTableRow.pdf");
bool isPdfUaCompliance = document.Validate(dataDir + "StyleTableRow.xml", PdfFormat.PDF_UA_1);
Console.WriteLine(String.Format("PDF/UA compliance: {0}", isPdfUaCompliance));
```

Αυτό διασφαλίζει ότι το PDF σας πληροί το πρότυπο PDF/UA, καθιστώντας το προσβάσιμο σε όλους. Η προσβασιμότητα είναι το παν!

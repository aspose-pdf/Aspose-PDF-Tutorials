# PDF σε XPS

## Εισαγωγή

Στον σημερινό ψηφιακό κόσμο, η ανάγκη μετατροπής εγγράφων από τη μία μορφή στην άλλη είναι πιο συχνή από ποτέ. Είτε είστε προγραμματιστής που θέλει να ενσωματώσει την επεξεργασία εγγράφων στην εφαρμογή του είτε επαγγελματίας που χρειάζεται να μοιραστεί αρχεία σε μια παγκοσμίως αποδεκτή μορφή, η κατανόηση του τρόπου μετατροπής αρχείων PDF σε XPS (XML Paper Specification) μπορεί να είναι εξαιρετικά χρήσιμη. Σε αυτό το σεμινάριο, θα εμβαθύνουμε στη διαδικασία μετατροπής PDF σε XPS χρησιμοποιώντας την ισχυρή βιβλιοθήκη Aspose.PDF για .NET.

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Πριν μπορέσετε να μετατρέψετε ένα PDF σε XPS, πρέπει να καθορίσετε τον κατάλογο όπου βρίσκεται το αρχείο PDF. Αυτό είναι κρίσιμο επειδή το πρόγραμμα πρέπει να γνωρίζει πού να βρει το αρχείο εισόδου.

Σε αυτό το βήμα, θα ορίσετε μια μεταβλητή συμβολοσειράς που περιέχει τη διαδρομή προς τον κατάλογο εγγράφων σας. Αυτή η διαδρομή θα πρέπει να δείχνει την τοποθεσία του αρχείου PDF σας.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στον υπολογιστή σας όπου είναι αποθηκευμένο το αρχείο PDF.

## Βήμα 2: Φόρτωση του εγγράφου PDF

Τώρα που έχετε ρυθμίσει τον κατάλογο εγγράφων σας, το επόμενο βήμα είναι να φορτώσετε το έγγραφο PDF που θέλετε να μετατρέψετε.

Θα δημιουργήσετε μια παρουσία του `Document` κλάση από τη βιβλιοθήκη Aspose.PDF και μεταβιβάστε τη διαδρομή του αρχείου PDF στον κατασκευαστή του. Αυτό θα φορτώσει το έγγραφο PDF στη μνήμη.

```csharp
// Φόρτωση εγγράφου PDF
Document pdfDocument = new Document(dataDir + "input.pdf");
```

Φροντίστε να αντικαταστήσετε `"input.pdf"` με το όνομα του πραγματικού αρχείου PDF σας.

## Βήμα 3: Δημιουργία επιλογών αποθήκευσης XPS

Πριν αποθηκεύσετε το έγγραφο σε μορφή XPS, πρέπει να δημιουργήσετε μια παρουσία του `XpsSaveOptions` κλάση. Αυτή η κλάση σάς επιτρέπει να καθορίσετε διάφορες επιλογές για την αποθήκευση του εγγράφου.

Με την δημιουργία στιγμιότυπων `XpsSaveOptions`, μπορείτε να προσαρμόσετε τον τρόπο μετατροπής του PDF σε XPS. Για αυτήν τη βασική μετατροπή, μπορείτε να χρησιμοποιήσετε τις προεπιλεγμένες ρυθμίσεις.

```csharp
// Δημιουργία επιλογών αποθήκευσης XPS
Aspose.Pdf.XpsSaveOptions saveOptions = new Aspose.Pdf.XpsSaveOptions();
```

## Βήμα 4: Αποθήκευση του εγγράφου ως XPS

Επιτέλους, ήρθε η ώρα να αποθηκεύσετε το φορτωμένο έγγραφο PDF ως αρχείο XPS. Εδώ συμβαίνει η μαγεία!

Θα καλέσετε το `Save` μέθοδος στο `pdfDocument` αντικείμενο, περνώντας το επιθυμητό όνομα αρχείου εξόδου και το `saveOptions` δημιουργήσατε νωρίτερα.

```csharp
// Αποθήκευση του εγγράφου XPS
pdfDocument.Save("PDFToXPS_out.xps", saveOptions);
```

Αυτή η γραμμή κώδικα θα δημιουργήσει ένα αρχείο XPS με το όνομα `PDFToXPS_out.xps` στον κατάλογο του έργου σας.

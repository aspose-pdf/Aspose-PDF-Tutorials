# Αφαίρεση υπερσυνδέσμων μετά τη μετατροπή από Html

## Εισαγωγή

Στην ψηφιακή εποχή, η μετατροπή εγγράφων HTML σε PDF είναι μια συνηθισμένη εργασία. Ωστόσο, μερικές φορές μπορεί να θέλετε να αφαιρέσετε υπερσυνδέσμους από το PDF που έχει μετατραπεί για διάφορους λόγους, όπως η βελτίωση της αναγνωσιμότητας ή η αποτροπή ανεπιθύμητης πλοήγησης. Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να το πετύχουμε αυτό χρησιμοποιώντας το Aspose.PDF για .NET. 

## Προαπαιτούμενα

Πριν ξεκινήσετε να μελετάτε τον κώδικα, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:

1. Visual Studio: Βεβαιωθείτε ότι έχετε εγκατεστημένο το Visual Studio στον υπολογιστή σας. Αυτό θα είναι το περιβάλλον ανάπτυξής σας.
2. Aspose.PDF για .NET: Χρειάζεται να έχετε τη βιβλιοθήκη Aspose.PDF. Μπορείτε να την κατεβάσετε από [εδώ](https://releases.aspose.com/pdf/net/).
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τον κώδικα.

## Εισαγωγή πακέτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τα απαραίτητα πακέτα στο έργο σας σε C#. Δείτε πώς μπορείτε να το κάνετε:

1. Ανοίξτε το έργο σας στο Visual Studio.
2. Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων και επιλέξτε "Διαχείριση πακέτων NuGet".
3. Αναζήτηση για `Aspose.PDF` και εγκαταστήστε το.

```csharp
using Aspose.Pdf.Annotations;
using Aspose.Pdf.Text;
using System.IO;
```

Τώρα που έχετε ρυθμίσει τα πάντα, ας αναλύσουμε τη διαδικασία αφαίρεσης υπερσυνδέσμων από ένα αρχείο HTML μετά τη μετατροπή του σε PDF.

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων

Πρώτα απ 'όλα, πρέπει να καθορίσετε τη διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ βρίσκεται το αρχείο HTML και εκεί θα αποθηκευτεί το PDF εξόδου.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου είναι αποθηκευμένο το αρχείο HTML σας.

## Βήμα 2: Φόρτωση του εγγράφου HTML

Στη συνέχεια, θα φορτώσετε το έγγραφο HTML χρησιμοποιώντας το `Document` κλάση από το Aspose.PDF. Αυτή η κλάση σάς επιτρέπει να εργάζεστε εύκολα με έγγραφα PDF.

```csharp
Document doc = new Document(dataDir + "SampleHtmlFile.html", new HtmlLoadOptions());
```

Εδώ, φορτώνουμε το αρχείο HTML με το όνομα `SampleHtmlFile.html`Βεβαιωθείτε ότι αυτό το αρχείο υπάρχει στον καθορισμένο κατάλογο.

## Βήμα 3: Αποθήκευση του εγγράφου στη ροή μνήμης

Πριν ξεκινήσουμε την επεξεργασία των σχολίων, πρέπει να αποθηκεύσουμε το έγγραφο σε μια ροή μνήμης. Αυτό το βήμα είναι κρίσιμο, καθώς προετοιμάζει το έγγραφο για περαιτέρω χειρισμό.

```csharp
doc.Save(new MemoryStream());
```

Αυτή η γραμμή αποθηκεύει το έγγραφο στη μνήμη, επιτρέποντάς μας να εργαστούμε με αυτό χωρίς να το γράψουμε ακόμα στο δίσκο.

## Βήμα 4: Επανάληψη μέσω σχολίων

Τώρα, θα εξετάσουμε επανειλημμένα τις σχολιασμοί στο έγγραφο. Οι σχολιασμοί είναι στοιχεία όπως σύνδεσμοι, σχόλια και επισημάνσεις. Μας ενδιαφέρουν συγκεκριμένα οι σχολιασμοί συνδέσμων.

```csharp
foreach (Annotation a in doc.Pages[1].Annotations)
{
    if (a.AnnotationType == AnnotationType.Link)
    {
        // Επεξεργασία της σχολίασης συνδέσμου
    }
}
```

Σε αυτόν τον βρόχο, ελέγχουμε αν ο τύπος σχολιασμού είναι σύνδεσμος. Εάν ναι, προχωράμε στα επόμενα βήματα.

## Βήμα 5: Κατάργηση της ενέργειας υπερσύνδεσης

Για κάθε σχολιασμό συνδέσμου, πρέπει να ελέγξουμε αν διαθέτει ενέργεια υπερσύνδεσης. Εάν ναι, θα καταργήσουμε τον υπερσύνδεσμο ορίζοντας το URI του σε μια κενή συμβολοσειρά.

```csharp
LinkAnnotation la = (LinkAnnotation)a;
if (la.Action is GoToURIAction)
{
    GoToURIAction gta = (GoToURIAction)la.Action;
    gta.URI = "";
```

Αυτό το απόσπασμα κώδικα διασφαλίζει ότι η ενέργεια υπερσύνδεσης καταργείται αποτελεσματικά.

## Βήμα 6: Απορροφήστε τμήματα κειμένου

Στη συνέχεια, θα απορροφήσουμε τα τμήματα κειμένου που σχετίζονται με την σχολίαση συνδέσμου. Αυτό μας επιτρέπει να χειριστούμε την εμφάνιση του κειμένου.

```csharp
TextFragmentAbsorber tfa = new TextFragmentAbsorber();
tfa.TextSearchOptions = new TextSearchOptions(a.Rect);
doc.Pages[a.PageIndex].Accept(tfa);
```

Εδώ, δημιουργούμε ένα `TextFragmentAbsorber` και ορίστε τις επιλογές αναζήτησης στο ορθογώνιο της σχολίασης. Αυτό μας βοηθά να βρούμε το κείμενο που συνδέθηκε.

## Βήμα 7: Τροποποίηση εμφάνισης κειμένου

Μόλις έχουμε τα αποσπάσματα κειμένου, μπορούμε να τροποποιήσουμε την εμφάνισή τους. Σε αυτήν την περίπτωση, θα αφαιρέσουμε την υπογράμμιση και θα αλλάξουμε το χρώμα του κειμένου σε μαύρο.

```csharp
foreach (TextFragment tf in tfa.TextFragments)
{
    tf.TextState.Underline = false;
    tf.TextState.ForegroundColor = Color.Black;
}
```

Αυτό το βήμα βελτιώνει την αναγνωσιμότητα του κειμένου αφαιρώντας το στυλ υπερσυνδέσμων.

## Βήμα 8: Διαγραφή της σχολίασης

Αφού τροποποιήσουμε το κείμενο, μπορούμε να διαγράψουμε με ασφάλεια την σχολίαση συνδέσμου από το έγγραφο.

```csharp
doc.Pages[a.PageIndex].Annotations.Delete(a);
}
```

Αυτή η γραμμή αφαιρεί τον υπερσύνδεσμο από το PDF, διασφαλίζοντας ότι δεν υπάρχει πλέον στο τελικό αποτέλεσμα.

## Βήμα 9: Αποθήκευση του τροποποιημένου εγγράφου

Τέλος, πρέπει να αποθηκεύσουμε το τροποποιημένο έγγραφο σε ένα νέο αρχείο PDF. Αυτό είναι το τελευταίο βήμα της διαδικασίας μας.

```csharp
doc.Save(dataDir + "RemoveHyperlinksFromText_out.pdf");
```

Αυτή η γραμμή αποθηκεύει το έγγραφο χωρίς τους υπερσυνδέσμους, δημιουργώντας ένα νέο αρχείο PDF με το όνομα `RemoveHyperlinksFromText_out.pdf`.

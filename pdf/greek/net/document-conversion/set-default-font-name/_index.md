# Ορισμός προεπιλεγμένου ονόματος γραμματοσειράς

## Εισαγωγή

Έχετε προσπαθήσει ποτέ να αποδώσετε ένα έγγραφο PDF σε εικόνα, αλλά διαπιστώσατε ότι οι γραμματοσειρές δεν φαίνονται σωστά; Ίσως το κείμενο να φαίνεται παραμορφωμένο ή ίσως η αρχική γραμματοσειρά να μην υποστηρίζεται. Εδώ είναι που ο ορισμός μιας προεπιλεγμένης γραμματοσειράς μπορεί να σώσει την κατάσταση! Χρησιμοποιώντας το Aspose.PDF για .NET, μπορείτε εύκολα να ορίσετε μια προεπιλεγμένη γραμματοσειρά για την απόδοση PDF, διασφαλίζοντας ότι το έγγραφό σας θα φαίνεται καθαρό και επαγγελματικό. Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στον τρόπο ορισμού του προεπιλεγμένου ονόματος γραμματοσειράς κατά την απόδοση ενός PDF σε εικόνα. Μέχρι το τέλος αυτού του οδηγού, θα έχετε τις δεξιότητες για να αντιμετωπίσετε τυχόν προκλήσεις απόδοσης PDF που θα προκύψουν. Είστε έτοιμοι; Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν προχωρήσουμε στον κώδικα, υπάρχουν μερικά πράγματα που θα πρέπει να έχετε στη διάθεσή σας:

- Aspose.PDF για .NET: Αυτή η ισχυρή βιβλιοθήκη είναι αυτή που θα χρησιμοποιήσουμε για να χειριστούμε το έγγραφο PDF μας. Μπορείτε να την κατεβάσετε από το [Ιστότοπος Aspose](https://releases.aspose.com/pdf/net/).
- Visual Studio: Βεβαιωθείτε ότι έχετε εγκατεστημένο το Visual Studio στον υπολογιστή σας. Αυτό θα είναι το περιβάλλον ανάπτυξής μας.
- .NET Framework: Βεβαιωθείτε ότι έχετε εγκαταστήσει το .NET Framework. Το Aspose.PDF για .NET υποστηρίζει διάφορες εκδόσεις, επομένως ελέγξτε την τεκμηρίωση ώστε να ταιριάζει στις ανάγκες σας.
- Ένα έγγραφο PDF: Θα χρειαστείτε ένα δείγμα εγγράφου PDF για να εργαστείτε. Εάν δεν έχετε, δημιουργήστε ένα απλό PDF ή κατεβάστε ένα δείγμα από το διαδίκτυο.

Μόλις τα ρυθμίσετε όλα, είμαστε έτοιμοι να ξεκινήσουμε τον προγραμματισμό!

## Εισαγωγή πακέτων

Πριν εμβαθύνουμε στον κώδικα, είναι απαραίτητο να εισαγάγουμε τα απαραίτητα πακέτα. Αυτό διασφαλίζει ότι έχουμε πρόσβαση σε όλες τις κλάσεις και τις μεθόδους που χρειαζόμαστε για το έργο μας.

```csharp
using Aspose.Pdf.Devices;
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Text;
```

Αυτές οι εισαγωγές είναι ζωτικής σημασίας, καθώς φέρνουν τους απαιτούμενους χώρους ονομάτων για τη διαχείριση χειρισμού PDF, απόδοσης εικόνων και λειτουργιών ροής αρχείων.

## Βήμα 1: Ρύθμιση της διαδρομής του έργου και του εγγράφου σας

Καταρχάς, ας ορίσουμε τη διαδρομή καταλόγου όπου βρίσκεται το έγγραφο PDF σας. Αυτό θα είναι το σημείο εκκίνησης για τον χειρισμό του αρχείου PDF.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```
Εδώ, `dataDir` είναι ο κατάλογος όπου βρίσκεται το έγγραφο PDF σας. Βεβαιωθείτε ότι έχετε αντικαταστήσει το `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς το έγγραφό σας. Αυτό είναι απαραίτητο, καθώς ο κώδικας πρέπει να γνωρίζει από πού να ανακτήσει το αρχείο PDF.

## Βήμα 2: Φόρτωση του εγγράφου PDF

Τώρα που έχουμε τη διαδρομή του εγγράφου, το επόμενο βήμα είναι να φορτώσουμε το έγγραφο PDF στη μνήμη, ώστε να μπορέσουμε να ξεκινήσουμε να εργαζόμαστε σε αυτό.

```csharp
using (Document pdfDocument = new Document(dataDir + "input.pdf"))
```
Χρησιμοποιούμε το `Document` κλάση από τη βιβλιοθήκη Aspose.PDF για να φορτώσουμε το αρχείο PDF μας. Αυτή η κλάση παρέχει διάφορες μεθόδους και ιδιότητες για να εργαστούμε με το έγγραφο PDF. Το `"input.pdf"` θα πρέπει να αντικατασταθεί με το πραγματικό όνομα αρχείου του PDF σας. Αυτό το αρχείο θα χρησιμοποιηθεί ως είσοδος για την απόδοση.

## Βήμα 3: Δημιουργήστε μια ροή εικόνας για την έξοδο

Μόλις φορτωθεί το έγγραφο, πρέπει να ρυθμίσουμε μια ροή για να αποθηκεύσουμε την εικόνα που αποδόθηκε. Εδώ θα αποθηκευτεί η εικόνα εξόδου.

```csharp
using (FileStream imageStream = new FileStream(dataDir + "SetDefaultFontName.png", FileMode.Create))
```
Ο `FileStream` Η κλάση χρησιμοποιείται για τη δημιουργία ενός νέου αρχείου όπου θα αποθηκευτεί η εικόνα που αποδίδεται. Σε αυτό το παράδειγμα, αποθηκεύουμε την εικόνα ως `"SetDefaultFontName.png"`Το `FileMode.Create` διασφαλίζει ότι δημιουργείται ένα νέο αρχείο ή ότι αντικαθίσταται ένα υπάρχον αρχείο.

## Βήμα 4: Ορίστε την ανάλυση για την εικόνα

Πριν από την απόδοση του PDF σε εικόνα, είναι σημαντικό να ορίσετε την ανάλυση. Αυτή καθορίζει την ποιότητα και την ευκρίνεια της εικόνας εξόδου.

```csharp
Resolution resolution = new Resolution(300);
```
Ο `Resolution` Η κλάση ορίζει την ανάλυση της εικόνας εξόδου. Εδώ, έχουμε επιλέξει ανάλυση 300 DPI (κουκκίδες ανά ίντσα), η οποία είναι τυπική για εικόνες υψηλής ποιότητας. Αυτό διασφαλίζει ότι το κείμενο και τα γραφικά στο PDF σας αποδίδονται καθαρά χωρίς να χάνονται λεπτομέρειες.

## Βήμα 5: Ρύθμιση παραμέτρων της συσκευής PNG

Στη συνέχεια, πρέπει να ρυθμίσουμε τη συσκευή που θα χειριστεί την απόδοση του PDF σε εικόνα PNG.

```csharp
PngDevice pngDevice = new PngDevice(resolution);
```
Ο `PngDevice` η κλάση είναι υπεύθυνη για την απόδοση του εγγράφου PDF σε εικόνα PNG. Περνώντας την `resolution` Εάν έχετε αντίρρηση, διασφαλίζουμε ότι η εικόνα δημιουργείται με το καθορισμένο DPI.

## Βήμα 6: Ορισμός του προεπιλεγμένου ονόματος γραμματοσειράς

Εδώ είναι το κρίσιμο μέρος - ο ορισμός του προεπιλεγμένου ονόματος γραμματοσειράς. Αυτή θα είναι η εφεδρική γραμματοσειρά σε περίπτωση που η αρχική γραμματοσειρά στο PDF δεν είναι διαθέσιμη.

```csharp
RenderingOptions ro = new RenderingOptions();
ro.DefaultFontName = "Arial";
pngDevice.RenderingOptions = ro;
```
Δημιουργούμε μια παρουσία του `RenderingOptions` και ορίστε το `DefaultFontName` ιδιοκτησία σε `"Arial"`Αυτό σημαίνει ότι εάν δεν είναι δυνατή η εύρεση της αρχικής γραμματοσειράς στο PDF, θα χρησιμοποιηθεί η γραμματοσειρά Arial. Αυτό το βήμα είναι κρίσιμο για τη διατήρηση της αναγνωσιμότητας και της εμφάνισης του κειμένου στην εικόνα που αποδίδεται.

## Βήμα 7: Απόδοση της σελίδας PDF σε εικόνα

Τέλος, αφού έχουμε ρυθμίσει όλα τα στοιχεία, μπορούμε τώρα να αποδώσουμε την πρώτη σελίδα του εγγράφου PDF σε εικόνα και να την αποθηκεύσουμε χρησιμοποιώντας τη ροή αρχείων που δημιουργήσαμε νωρίτερα.

```csharp
pngDevice.Process(pdfDocument.Pages[1], imageStream);
```
Ο `Process` μέθοδος του `PngDevice` Η κλάση χρησιμοποιείται για την απόδοση της καθορισμένης σελίδας PDF (σε αυτήν την περίπτωση, της πρώτης σελίδας) σε εικόνα. Το αποτέλεσμα αποθηκεύεται στη συνέχεια στο `imageStream`Αυτό το βήμα μετατρέπει τη σελίδα PDF σε εικόνα PNG με την καθορισμένη ανάλυση και την προεπιλεγμένη γραμματοσειρά.

## Βήμα 8: Κλείστε τη Ροή Αρχείων και το Έγγραφο PDF

Μετά την απόδοση της εικόνας, είναι απαραίτητο να κλείσετε τη ροή αρχείων και το έγγραφο PDF για να ελευθερώσετε πόρους.

```csharp
imageStream.Close();
pdfDocument.Dispose();
```
Κλείσιμο του `imageStream` διασφαλίζει ότι το αρχείο αποθηκεύεται σωστά και δεν χάνονται δεδομένα. Η απόρριψη του `pdfDocument` απελευθερώνει μνήμη και πόρους, αποτρέποντας τυχόν διαρροές μνήμης.

## Σύναψη

Και να το! Με λίγες μόνο γραμμές κώδικα, μάθατε πώς να ορίσετε ένα προεπιλεγμένο όνομα γραμματοσειράς κατά την απόδοση ενός PDF σε μια εικόνα χρησιμοποιώντας το Aspose.PDF για .NET. Αυτή η δεξιότητα είναι εξαιρετικά χρήσιμη, ειδικά όταν ασχολείστε με PDF που ενδέχεται να περιέχουν μη υποστηριζόμενες γραμματοσειρές. Ορίζοντας μια προεπιλεγμένη γραμματοσειρά, διασφαλίζετε ότι οι εικόνες που αποδίδονται διατηρούν την αναγνωσιμότητα και την επαγγελματική τους εμφάνιση.

## Συχνές ερωτήσεις

### Τι συμβαίνει εάν η καθορισμένη προεπιλεγμένη γραμματοσειρά δεν είναι εγκατεστημένη στο σύστημα;
Εάν η προεπιλεγμένη γραμματοσειρά που καθορίζεται στο `RenderingOptions` δεν είναι εγκατεστημένο στο σύστημα, το Aspose.PDF θα χρησιμοποιήσει μια γραμματοσειρά εφεδρικής μορφής που ορίζεται από το σύστημα.

### Μπορώ να χρησιμοποιήσω άλλες γραμματοσειρές εκτός από την Arial ως προεπιλεγμένη γραμματοσειρά;
Απολύτως! Μπορείτε να ορίσετε οποιαδήποτε γραμματοσειρά είναι εγκατεστημένη στο σύστημά σας ως προεπιλεγμένη γραμματοσειρά.

### Είναι δυνατή η απόδοση πολλαπλών σελίδων ενός PDF σε εικόνες ταυτόχρονα;
Ναι, μπορείτε να κάνετε επανάληψη στις σελίδες του PDF σας και να αποδώσετε κάθε σελίδα ξεχωριστά χρησιμοποιώντας την ίδια διαδικασία.

### Επηρεάζει η ρύθμιση υψηλής ανάλυσης την απόδοση της απόδοσης PDF;
Ναι, οι υψηλότερες αναλύσεις θα έχουν ως αποτέλεσμα μεγαλύτερα αρχεία εικόνας και ενδέχεται να αυξήσουν τον χρόνο απόδοσης, αλλά θα παράγουν επίσης πιο καθαρές εικόνες.

### Μπορώ να αποδώσω το PDF σε άλλες μορφές εικόνας εκτός από PNG;
Ναι, το Aspose.PDF υποστηρίζει την απόδοση σε διάφορες μορφές εικόνας όπως JPEG, BMP και TIFF.

## Οδηγός Εφαρμογής
Τώρα, ας δούμε κάθε βήμα για να δημιουργήσουμε το PDF μας με σενάρια LaTeX.

### Βήμα 1: Δημιουργία νέου αντικειμένου εγγράφου
Ξεκινήστε αρχικοποιώντας ένα `Document` αντικείμενο. Αυτό χρησιμεύει ως το κοντέινερ για τις σελίδες και το περιεχόμενό σας.

```csharp
// Αρχικοποίηση νέου εγγράφου
Document document = new Document();
```

### Βήμα 2: Προσθήκη σελίδας στη συλλογή σελίδων του εγγράφου
Προσθέστε μια σελίδα στο έγγραφό σας όπου θα τοποθετήσετε το περιεχόμενό σας.

```csharp
// Προσθήκη νέας σελίδας
Page page = document.Pages.Add();
```

### Βήμα 3: Δημιουργήστε έναν πίνακα και προσθέστε τον στη σελίδα
Μια δομή πίνακα θα βοηθήσει στην οργάνωση του περιεχομένου. Εδώ, θα το προσθέσουμε απευθείας στη σελίδα μας.

```csharp
// Αρχικοποίηση ενός νέου αντικειμένου Πίνακα
Table table = new Table();
page.Paragraphs.Add(table);
```

### Βήμα 4: Προσθήκη γραμμής στον πίνακα
Στη συνέχεια, εισαγάγετε μια γραμμή στον πίνακά σας όπου μπορείτε να τοποθετήσετε κελιά που περιέχουν περιεχόμενο.

```csharp
// Προσθήκη γραμμής στον πίνακα
Row row = table.Rows.Add();
```

### Βήμα 5: Ορισμός Σεναρίου LaTeX για Μαθηματική Έκφραση
Προετοιμάστε το σενάριο LaTeX. Εδώ ορίζετε τη μαθηματική παράσταση που θέλετε να εμφανίσετε.

```csharp
// Έκφραση LaTeX για πρόσθεση, τετραγωνική ρίζα και ολοκλήρωμα
string latexText1 = "$123456789+\\sqrt{1}+\\int_a^b f(x)dx$";
```

### Βήμα 6: Δημιουργήστε ένα κελί και προσθέστε το τμήμα LaTeX
Δημιουργήστε ένα κελί στη γραμμή σας για να περιέχει το σενάριο LaTeX. Ορίστε τα περιθώριά του για καλύτερη αναγνωσιμότητα.

```csharp
// Προσθήκη νέου κελιού με καθορισμένα περιθώρια
Cell cell = row.Cells.Add();
cell.Margin = new MarginInfo { Left = 20, Right = 20, Top = 20, Bottom = 20 };

// Δημιουργία και προσθήκη τμήματος LaTeX
TeXFragment ltext1 = new TeXFragment(latexText1, true);
cell.Paragraphs.Add(ltext1);
```

### Βήμα 7: Αποθήκευση του εγγράφου
Τέλος, αποθηκεύστε το έγγραφό σας σε έναν καθορισμένο κατάλογο. Βεβαιωθείτε ότι έχετε δικαιώματα εγγραφής για αυτήν την τοποθεσία.

```csharp
// Καθορίστε τον κατάλογο εξόδου και αποθηκεύστε το αρχείο
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εξόδου σας
document.Save(Path.Combine(outputDir, "LatextScriptInPdf_out.pdf"));
```

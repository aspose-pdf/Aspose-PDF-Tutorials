#### Αρχικοποίηση εγγράφου και προσθήκη σελίδας

Αυτή η λειτουργία σάς δείχνει πώς να δημιουργήσετε ένα νέο έγγραφο PDF και να προσθέσετε σελίδες σε αυτό, ρυθμίζοντας τον χώρο εργασίας σας για περαιτέρω προσθήκες.

**1. Δημιουργία αντικειμένου εγγράφου**

```csharp
using Aspose.Pdf;

public class InitializeDocumentAndAddPage
{
    public void Run()
    {
        // Δημιουργήστε μια νέα παρουσία της κλάσης Document
        Document doc = new Document();

        // Προσθήκη σελίδας στη συλλογή σελίδων του εγγράφου
        Page page = doc.Pages.Add();
    }
}
```
Εδώ, `Document` χρησιμοποιείται για την αναπαράσταση του αρχείου PDF σας. Το `Pages.Add()` Η μέθοδος προσθέτει μια νέα σελίδα όπου μπορείτε να εισαγάγετε περιεχόμενο.

#### Προσθήκη τμήματος HTML σε PDF

Στη συνέχεια, θα δούμε πώς να προσθέσετε στυλιζαρισμένο περιεχόμενο HTML στο έγγραφο PDF σας.

**2. Δημιουργία και διαμόρφωση ενός τμήματος HTML**

```csharp
using Aspose.Pdf.Text;

public class AddHTMLFragmentToPDF
{
    public void Run()
    {
        // Δημιουργία στιγμιαίου HtmlFragment με περιεχόμενα HTML
        HtmlFragment title = new HtmlFragment("<p style='font-family: Verdana'><b><i>Table contains text</i></b></p>");

        // Αλλαγή οικογένειας και μεγέθους γραμματοσειράς
        title.TextState = new TextState("Arial")
        {
            FontSize = 20
        };

        // Ορίστε περιθώρια για καλύτερο έλεγχο της διάταξης
        title.Margin.Bottom = 10;
        title.Margin.Top = 400;
    }
}
```
Ο `HtmlFragment` Η κλάση επιτρέπει την ενσωμάτωση περιεχομένου HTML απευθείας στο PDF σας. Τροποποιώντας το `TextState`, μπορείτε να προσαρμόσετε τη γραμματοσειρά και το μέγεθος ώστε να ταιριάζουν στις ανάγκες σχεδιασμού σας.

#### Αποθήκευση του εγγράφου PDF

Τέλος, αποθηκεύστε την εργασία σας με αυτήν τη λειτουργία:

**3. Αποθηκεύστε το PDF σας**

```csharp
using Aspose.Pdf;

public class SavePDFDocument
{
    public void Run()
    {
        // Υποθέτοντας ότι το `doc` έχει ήδη δημιουργηθεί και τροποποιηθεί
        Document doc = new Document();

        // Ορίστε τη διαδρομή εξόδου για την αποθήκευση του PDF
        string dataDir = "YOUR_OUTPUT_DIRECTORY";
        string outputPath = dataDir + "/AddHTMLUsingDOMAndOverwrite_out.pdf";

        // Αποθηκεύστε το έγγραφο στην καθορισμένη διαδρομή
        doc.Save(outputPath);
    }
}
```
Ο `Save` Η μέθοδος γράφει το τελικό PDF σε ένα αρχείο, καθιστώντας το έτοιμο για διανομή ή περαιτέρω επεξεργασία.

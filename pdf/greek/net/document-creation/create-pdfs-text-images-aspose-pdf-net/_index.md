## Οδηγός Εφαρμογής

Αυτή η ενότητα αναλύει τη διαδικασία δημιουργίας PDF με κείμενο και εικόνες χρησιμοποιώντας το Aspose.PDF για .NET σε διαχειρίσιμα βήματα.

### Δημιουργία εγγράφου με κείμενο

**Επισκόπηση:**
Η δημιουργία ενός απλού εγγράφου PDF με την προσθήκη κειμένου είναι απλή. Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε έγγραφα μέσω προγραμματισμού, τα οποία μπορούν να προσαρμοστούν ώστε να περιλαμβάνουν συγκεκριμένο στυλ και μορφοποίηση.

#### Βήμα προς βήμα εφαρμογή:
1. **Αρχικοποίηση του εγγράφου**
   Ξεκινήστε δημιουργώντας μια παρουσία του `Document` τάξη.
   ```csharp
   Document doc = new Document();
   ```
2. **Προσθήκη σελίδας**
   Οι σελίδες προστίθενται στο έγγραφο όπου θα τοποθετήσετε το περιεχόμενό σας.
   ```csharp
   Page page = doc.getPages().add();
   ```
3. **Δημιουργία και ρύθμιση παραμέτρων τμήματος κειμένου**
   Εδώ, δημιουργούμε ένα `TextFragment` που περιέχει το κείμενο που θέλουμε, μαζί με συγκεκριμένες ρυθμίσεις στυλ όπως χρώμα και μέγεθος.
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Text;

   TextFragment t1 = new TextFragment("paragraph 3 segment");
   t1.getTextState().setForegroundColor(Color.RED);
   t1.setTextIsInLineParagraph(true);
   t1.getTextState().setFontSize(12);
   ```
4. **Προσθήκη κειμένου στη σελίδα**
   Τέλος, προσθέστε το τμήμα κειμένου στη συλλογή παραγράφων της σελίδας.
   ```csharp
   page.getParagraphs().add(t1);
   doc.save("YOUR_DOCUMENT_DIRECTORY/CreateTextPDF_out.pdf");
   ```

### Δημιουργία ενός Πλωτού Πλαισίου με Κείμενο

**Επισκόπηση:**
Τα αιωρούμενα πλαίσια παρέχουν έναν προηγμένο τρόπο οργάνωσης περιεχομένου μέσα σε ένα PDF, επιτρέποντας δυναμικές διατάξεις.

#### Βήμα προς βήμα εφαρμογή:
1. **Δημιουργήστε το Πλωτό Κουτί**
   Ορίστε ένα `FloatingBox` με συγκεκριμένες διαστάσεις.
   ```csharp
   FloatingBox textFloatingBox = new FloatingBox(117, 21);
   ```
2. **Ρύθμιση θέσης και δείκτη Z**
   Προσαρμόστε την τοποθέτησή του στη σελίδα.
   ```csharp
   textFloatingBox.setZIndex(1);
   textFloatingBox.setLeft(-4);
   textFloatingBox.setTop(-4);
   ```
3. **Προσθήκη αιωρούμενου πλαισίου στη σελίδα**
   Τοποθετήστε το στη δομή του εγγράφου.
   ```csharp
   page.getParagraphs().add(textFloatingBox);
   textFloatingBox.getParagraphs().add(t1); // Προσθήκη προηγουμένως δημιουργημένου τμήματος κειμένου
   doc.save("YOUR_DOCUMENT_DIRECTORY/TextFloatingBox_out.pdf");
   ```

### Προσθήκη εικόνας σε ένα FloatingBox

**Επισκόπηση:**
Η ενσωμάτωση εικόνων μέσα σε αιωρούμενα πλαίσια επιτρέπει ευέλικτες επιλογές σχεδιασμού και διάταξης.

#### Βήμα προς βήμα εφαρμογή:
1. **Αρχικοποίηση της εικόνας**
   Προετοιμάστε το αντικείμενο εικόνας σας με τη διαδρομή αρχείου.
   ```csharp
   using Aspose.Pdf;

   Image image = new Image();
   image.setFile("YOUR_DOCUMENT_DIRECTORY/test_image.png");
   ```
2. **Δημιουργία και ρύθμιση παραμέτρων ενός αιωρούμενου πλαισίου για την εικόνα**
   Παρόμοια με το κείμενο, ορίστε ένα αιωρούμενο πλαίσιο αλλά προσαρμοσμένο για μια εικόνα.
   ```csharp
   FloatingBox imageFloatingBox = new FloatingBox(117, 21);
   page.getParagraphs().add(imageFloatingBox);

   imageFloatingBox.setLeft(-4);
   imageFloatingBox.setTop(-4);
   imageFloatingBox.setZIndex(2);
   ```
3. **Προσθήκη της εικόνας στο αιωρούμενο πλαίσιο**
   Τοποθετήστε την εικόνα σας μέσα στο πλωτό πλαίσιο και αποθηκεύστε την.
   ```csharp
   imageFloatingBox.getParagraphs().add(image);
   doc.save("YOUR_DOCUMENT_DIRECTORY/ImageInFloatingBox_out.pdf");
   ```

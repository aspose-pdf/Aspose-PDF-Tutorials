## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία δημιουργίας ενός φυλλαδίου.

### Δημιουργία αντικειμένου PdfFileEditor

Αρχικά, πρέπει να δημιουργήσετε μια παρουσία του `PdfFileEditor`Αυτή η τάξη παρέχει μεθόδους για την επεξεργασία αρχείων PDF, συμπεριλαμβανομένης της δημιουργίας φυλλαδίων.
```csharp
using Aspose.Pdf.Facades;

// Δημιουργία αντικειμένου PdfFileEditor
PdfFileEditor pdfEditor = new PdfFileEditor();
```

### Ρύθμιση ροών αρχείων

Στη συνέχεια, ρυθμίστε τις ροές αρχείων για να διαβάσουν το PDF εισόδου και να γράψουν το φυλλάδιο εξόδου. Αυτό το βήμα περιλαμβάνει τη δημιουργία `FileStream` αντικείμενα και για τις δύο λειτουργίες.
```csharp
using System.IO;

// Ορίστε διαδρομές για αρχεία εισόδου και εξόδου
string dataDir = "your_data_directory_path";
string inputFile = Path.Combine(dataDir, "input.pdf");
string outputFile = Path.Combine(dataDir, "output_booklet.pdf");

// Δημιουργία ροών
using FileStream inputStream = new FileStream(inputFile, FileMode.Open);
using FileStream outputStream = new FileStream(outputFile, FileMode.Create);
```

### Δημιουργία του φυλλαδίου

Χρησιμοποιώ `PdfFileEditor.MakeBooklet` για να μετατρέψετε το PDF σας σε μορφή φυλλαδίου. Αυτή η μέθοδος απαιτεί τον καθορισμό του μεγέθους σελίδας για το φυλλάδιο.
```csharp
// Φτιάξτε φυλλάδιο με μέγεθος σελίδας A5
pdfEditor.MakeBooklet(inputStream, outputStream, PageSize.A5);
```

**Βασικές παράμετροι:**
- `inputStream`: Η ροή αρχείων εισόδου.
- `outputStream`: Η ροή αρχείων εξόδου όπου θα αποθηκευτεί το φυλλάδιο.
- `PageSize.A5`: Ορίζει το τυπικό μέγεθος κάθε φύλλου στο φυλλάδιο.

## Οδηγός Εφαρμογής
### Προσθήκη σφραγίδας εικόνας σε συγκεκριμένες σελίδες
Αυτή η λειτουργία επιτρέπει την επιλεκτική δημιουργία επωνυμίας ή υδατογραφήματος σε όλο το έγγραφό σας. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Ανοίξτε το έγγραφο PDF σας
Δημιουργήστε ένα `PdfFileStamp` και συνδέστε το με το επιθυμητό αρχείο PDF.
```csharp
// Ορίστε τη διαδρομή καταλόγου για έγγραφα
string dataDir = "YOUR_DOCUMENT_DIRECTORY/";

// Δημιουργήστε μια παρουσία του PdfFileStamp
PdfFileStamp fileStamp = new PdfFileStamp();

// Ανοίξτε το έγγραφο που θέλετε να σφραγίσετε
fileStamp.BindPdf(dataDir + "AddImageStamp-Page.pdf");
```

#### Βήμα 2: Αρχικοποίηση και διαμόρφωση της σφραγίδας σας
Δημιουργήστε ένα `Stamp` αντικείμενο, ορίστε τις ιδιότητές του όπως θέση, περιστροφή και φόντο.
```csharp
// Αρχικοποίηση ενός νέου αντικειμένου Stamp
Stamp stamp = new Stamp();

// Σύνδεση αρχείου εικόνας στη σφραγίδα
stamp.BindImage(dataDir + "aspose-logo.jpg");

// Ορίστε τις συντεταγμένες προέλευσης (x, y) για την τοποθέτηση της σφραγίδας
stamp.SetOrigin(200, 200);

// Περιστρέψτε τη σφραγίδα κατά 90 μοίρες
stamp.Rotation = 90.0F;

// Ρύθμιση της σφραγίδας ώστε να εμφανίζεται στο φόντο της σελίδας
stamp.Background = true;
```

#### Βήμα 3: Καθορισμός σελίδων και εφαρμογή της σφραγίδας
Επιλέξτε ποιες σελίδες θα λάβουν τη σφραγίδα.
```csharp
// Ορίστε συγκεκριμένες σελίδες (π.χ., σελίδα 1) για τη σφραγίδα
stamp.Pages = new int[] { 1 };

// Προσθέστε τη διαμορφωμένη σφραγίδα στο αρχείο PDF
fileStamp.AddStamp(stamp);
```

#### Βήμα 4: Αποθήκευση και κλείσιμο
Αποθηκεύστε το έγγραφό σας και κλείστε το `PdfFileStamp` αντικείμενο.
```csharp
// Αποθήκευση του ενημερωμένου εγγράφου
fileStamp.Save(dataDir + "AddImageStamp-Page_out.pdf");

// Απελευθερώστε πόρους κλείνοντας το αντικείμενο PdfFileStamp
fileStamp.Close();
```

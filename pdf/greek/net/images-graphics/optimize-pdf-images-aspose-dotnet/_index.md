## Οδηγός Εφαρμογής
Αυτή η ενότητα σάς καθοδηγεί στη βελτιστοποίηση εικόνων στα PDF σας χρησιμοποιώντας το Aspose.PDF για .NET.

### Βελτιστοποίηση εικόνας PDF
Η βελτιστοποίηση εικόνας περιλαμβάνει την αλλαγή μεγέθους, τη συμπίεση και την προσαρμογή της ποιότητας των εικόνων για τη μείωση του μεγέθους του αρχείου, διατηρώντας παράλληλα την οπτική πιστότητα. Αυτή η λειτουργία είναι ιδιαίτερα χρήσιμη για μεγάλα έγγραφα με εικόνες υψηλής ανάλυσης.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποίηση παρακολούθησης χρόνου**
Ξεκινήστε καταγράφοντας την τρέχουσα ώρα για τη μέτρηση της απόδοσης:
```csharp
var startTime = DateTime.Now.Ticks;
```

**2. Ορισμός διαδρομών εισόδου και εξόδου**
Καθορίστε το αρχείο PDF εισόδου και τον κατάλογο εξόδου:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/ResizeImage.pdf";
string outputDataDir = "YOUR_OUTPUT_DIRECTORY/ResizeImages_out.pdf";
```

**3. Ανοίξτε το έγγραφο PDF**
Φορτώστε το PDF σε ένα `Aspose.Pdf.Document` αντικείμενο:
```csharp
Document pdfDocument = new Document(dataDir);
```

**4. Διαμόρφωση επιλογών βελτιστοποίησης**
Δημιουργία και διαμόρφωση `OptimizationOptions` για συμπίεση εικόνας:
```csharp
var optimizeOptions = new Pdf.Optimization.OptimizationOptions();
optimizeOptions.ImageCompressionOptions.CompressImages = true;
optimizeOptions.ImageCompressionOptions.ImageQuality = 75; // Προσαρμογή ποιότητας (0-100)
optimizeOptions.ImageCompressionOptions.ResizeImages = true;
optimizeOptions.ImageCompressionOptions.MaxResolution = 300; // Μέγιστη ανάλυση σε DPI
```

**5. Βελτιστοποίηση Πόρων**
Εφαρμόστε τις ρυθμίσεις βελτιστοποίησης στο PDF σας:
```csharp
pdfDocument.OptimizeResources(optimizeOptions);
```

**6. Αποθήκευση βελτιστοποιημένου εγγράφου**
Τέλος, αποθηκεύστε το βελτιστοποιημένο έγγραφο σε ένα νέο αρχείο:
```csharp
pdfDocument.Save(outputDataDir);
```

### Μέτρηση χρόνου για την ολοκλήρωση της διαδικασίας
Η μέτρηση του χρόνου εκτέλεσης βοηθά στην αξιολόγηση της απόδοσης και στον εντοπισμό σημείων συμφόρησης.

#### Βήματα Υλοποίησης
**1. Ώρα έναρξης λήψης**
```csharp
var startTime = DateTime.Now.Ticks;
```

**2. Εκτελέστε Βελτιστοποίηση Εικόνας**
Υποθέστε μια συνάρτηση `ResizeImages()` χρησιμοποιείται για την εκτέλεση της βελτιστοποίησης:
```csharp
ResizeImages();
```

**3. Υπολογίστε τον Χρόνο που Παρήλθε**
Μετρήστε και εμφανίστε τον χρόνο που απαιτείται για τη βελτιστοποίηση:
```csharp
var endTime = DateTime.Now.Ticks;
long elapsedTime = (endTime - startTime) / TimeSpan.TicksPerMillisecond;
Console.WriteLine($"Image optimization completed in {elapsedTime} ms.");
```

## Οδηγός Εφαρμογής (H2)

### Ορισμός εικόνας ως φόντου (H2)

Αυτή η ενότητα σας καθοδηγεί στη ρύθμιση ενός φόντου εικόνας στο PDF σας.

#### Βήμα 1: Δημιουργία νέου εγγράφου

Ξεκινήστε δημιουργώντας ένα `Document` αντικείμενο:

```csharp
// Αρχικοποίηση εγγράφου
document doc = new Document();
```

Αυτό το απόσπασμα αρχικοποιεί μια νέα παρουσία του `Document` κλάση, που αντιπροσωπεύει το αρχείο PDF σας.

#### Βήμα 2: Προσθήκη σελίδας στο έγγραφο

Προσθέστε μια σελίδα όπου θα ορίσετε την εικόνα φόντου:

```csharp
Page page = doc.Pages.Add();
```

Εδώ, προσθέτουμε μια κενή σελίδα χρησιμοποιώντας το `Pages.Add()` μέθοδος.

#### Βήμα 3: Δημιουργία και ρύθμιση παραμέτρων του BackgroundArtifact

Ο `BackgroundArtifact` Η κλάση χρησιμοποιείται για τον ορισμό φόντων στο Aspose.PDF. Δείτε πώς μπορείτε να τη διαμορφώσετε:

```csharp
BackgroundArtifact background = new BackgroundArtifact();
background.BackgroundImage = File.OpenRead(@"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg");
page.Artifacts.Add(background);
```

- **Γιατί;** Ο `BackgroundArtifact` Η κλάση έχει σχεδιαστεί για τη διαχείριση φόντων σε σελίδες PDF. Η προσθήκη αυτού του τεχνουργήματος διασφαλίζει ότι η εικόνα εμφανίζεται ως φόντο.
- Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY\aspose-total-for-net.jpg"` με την επιθυμητή διαδρομή εικόνας φόντου.

#### Βήμα 4: Αποθήκευση του εγγράφου σας

Τέλος, αποθηκεύστε το έγγραφό σας:

```csharp
string dataDir = @"YOUR_OUTPUT_DIRECTORY\ImageAsBackground_out.pdf";
doc.Save(dataDir);
```

Αυτός ο κώδικας αποθηκεύει το αρχείο PDF σας με τον καθορισμένο κατάλογο εξόδου και όνομα αρχείου. Αντικαταστήστε `"YOUR_OUTPUT_DIRECTORY"` με την επιθυμητή τοποθεσία αποθήκευσης.

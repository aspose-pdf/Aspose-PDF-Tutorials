Σε αυτό το σεμινάριο, θα μάθετε πώς να χρησιμοποιείτε το Aspose.PDF για .NET για να προσθέσετε μια σφραγίδα εικόνας σε κάθε σελίδα ενός εγγράφου PDF.

**Τι θα μάθετε:**

- Αποτελεσματική διαχείριση διαδρομών αρχείων εισόδου και εξόδου

Ας εξετάσουμε τις προϋποθέσεις πριν ξεκινήσουμε.

## Οδηγός Εφαρμογής

### Προσθήκη σφραγίδας εικόνας σε κεφαλίδες PDF

Προσθέστε μια προσαρμοσμένη σφραγίδα εικόνας στην κεφαλίδα κάθε σελίδας στο έγγραφο PDF σας για εμπορική προβολή ή πρόσθετες πληροφορίες.

#### Βήμα 1: Άνοιγμα υπάρχοντος εγγράφου PDF
Φορτώστε το υπάρχον PDF σας στο αντικείμενο Aspose.Pdf.Document:
```csharp
using System.IO;
using Aspose.Pdf;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή καταλόγου

// Φόρτωση ενός υπάρχοντος εγγράφου PDF
document pdfDocument = new Document(dataDir + "/ImageinHeader.pdf");
```

#### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων αντικειμένου ImageStamp
Ρυθμίστε το `ImageStamp` αντικείμενο καθορίζοντας το αρχείο εικόνας, τα περιθώρια και τη στοίχιση:
```csharp
// Δημιουργήστε ένα ImageStamp με την εικόνα που επιθυμείτε
ImageStamp imageStamp = new ImageStamp(dataDir + "/aspose-logo.jpg");
imageStamp.TopMargin = 10;
imageStamp.HorizontalAlignment = HorizontalAlignment.Center;
imageStamp.VerticalAlignment = VerticalAlignment.Top;
```

#### Βήμα 3: Επαναλάβετε και προσθέστε σφραγίδα σε κάθε σελίδα
Εφαρμόστε τη σφραγίδα εικόνας στην κεφαλίδα κάθε σελίδας:
```csharp
// Εφαρμόστε τη σφραγίδα εικόνας στην κεφαλίδα κάθε σελίδας
document.ForEach(page => page.AddStamp(imageStamp));
```

#### Βήμα 4: Αποθηκεύστε το ενημερωμένο έγγραφο PDF
Ορίστε τη διαδρομή του αρχείου εξόδου, διασφαλίζοντας ότι ο κατάλογος υπάρχει πριν από την αποθήκευση:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου εξόδου σας

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
if (!Directory.Exists(outputDir))
{
    Directory.CreateDirectory(outputDir);
}

currentOutputPath = Path.Combine(outputDir, "ImageinHeader_out.pdf");
document.Save(currentOutputPath); // Αποθήκευση του τροποποιημένου PDF
```

### Διαχείριση διαδρομών εξόδου αρχείων
Η σωστή διαχείριση των διαδρομών αρχείων διασφαλίζει ότι τα αρχεία αποθηκεύονται σωστά χωρίς σφάλματα.

#### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου
```csharp
string dataDirectory = "YOUR_DOCUMENT_DIRECTORY"; // Διαδρομή καταλόγου κράτησης θέσης
string outputDirectory = "YOUR_OUTPUT_DIRECTORY"; // Διαδρομή καταλόγου εξόδου κράτησης θέσης

currentInputPath = Path.Combine(dataDirectory, "ImageinHeader.pdf");
currentOutputPath = Path.Combine(outputDirectory, "ImageinHeader_out.pdf");
```

#### Βήμα 2: Βεβαιωθείτε ότι υπάρχει κατάλογος
Ελέγχετε πάντα αν ο κατάλογος προορισμού υπάρχει για να αποφύγετε σφάλματα χρόνου εκτέλεσης.
```csharp
if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

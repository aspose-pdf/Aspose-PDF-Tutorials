## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία και συμπλήρωση ορθογωνίου σε PDF

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε ένα ορθογώνιο γεμάτο με χρώμα σε ένα έγγραφο PDF. Είναι ιδιαίτερα χρήσιμη για την προσθήκη οπτικών στοιχείων ή την επισήμανση τμημάτων κειμένου μέσα στα έγγραφά σας.

#### Βήμα προς βήμα εφαρμογή

##### 1. Αρχικοποίηση του εγγράφου
Αρχικά, δημιουργήστε μια παρουσία του `Document` τάξη και προσθέστε μια σελίδα:
```csharp
using Aspose.Pdf;

// Δημιουργήστε ένα νέο έγγραφο PDF
doc = new Document();

// Προσθήκη σελίδας στο έγγραφο
Page page = doc.Pages.Add();
```
Εδώ, αρχικοποιούμε ένα νέο έγγραφο PDF και προσθέτουμε μια κενή σελίδα όπου θα σχεδιαστεί το ορθογώνιό μας.

##### 2. Ρύθμιση αντικειμένου γραφήματος
Στη συνέχεια, ρυθμίστε ένα `Graph` αντικείμενο με καθορισμένες διαστάσεις:
```csharp
using Aspose.Pdf.Drawing;

// Δημιουργήστε ένα αντικείμενο Graph με καθορισμένο πλάτος και ύψος
graph = new Graph(100.0, 400.0);

// Προσθήκη του αντικειμένου γραφήματος στη συλλογή παραγράφων της σελίδας
page.Paragraphs.Add(graph);
```
Ο `Graph` Το αντικείμενο λειτουργεί ως δοχείο για σχεδιαστικά στοιχεία όπως ορθογώνια.

##### 3. Δημιουργία και διαμόρφωση ορθογωνίου
Τώρα, δημιουργήστε ένα ορθογώνιο με καθορισμένη θέση και μέγεθος και, στη συνέχεια, ορίστε το χρώμα γεμίσματός του:
```csharp
// Δημιουργήστε ένα ορθογώνιο αντικείμενο με κάτω αριστερή (llx, lly) και πάνω δεξιά (urx, ury) γωνία
Rectangle rect = new Rectangle(100, 100, 200, 120);

// Ορίστε το χρώμα γεμίσματος σε κόκκινο
rect.GraphInfo.FillColor = Aspose.Pdf.Color.Red;

// Προσθήκη του ορθογωνίου στη συλλογή σχημάτων του γραφήματος
graph.Shapes.Add(rect);
```
Ο `Rectangle` Η κλάση σάς επιτρέπει να ορίσετε διαστάσεις και θέση. Η `GraphInfo.FillColor` Η ιδιότητα ορίζει το χρώμα γεμίσματος.

##### 4. Αποθήκευση του εγγράφου
Τέλος, αποθηκεύστε το έγγραφο PDF σε έναν καθορισμένο κατάλογο:
```csharp
// Ορίστε τη διαδρομή εξόδου για το έγγραφο
dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/CreateFilledRectangle_out.pdf";

// Αποθήκευση του εγγράφου
doc.Save(dataDir);
```
Αυτό το βήμα γράφει το τροποποιημένο έγγραφο με το γεμισμένο ορθογώνιο στον δίσκο.

### Λειτουργία 2: Αρχικοποίηση και ρύθμιση παραμέτρων του εγγράφου Aspose.PDF

#### Επισκόπηση
Η βασική αρχικοποίηση ενός PDF χρησιμοποιώντας το Aspose.PDF για .NET είναι απλή. Αυτή η ενότητα καλύπτει τον τρόπο δημιουργίας ενός κενού εγγράφου και αποθήκευσής του, το οποίο χρησιμεύει ως βάση για πιο σύνθετες λειτουργίες όπως η προσθήκη ορθογωνίων.

#### Βήμα προς βήμα εφαρμογή

##### 1. Δημιουργήστε την παρουσία εγγράφου
```csharp
// Δημιουργία ενός νέου αντικειμένου εγγράφου
doc = new Document();
```
Αυτό το βήμα αρχικοποιεί ένα κενό έγγραφο PDF.

##### 2. Προσθήκη σελίδας και αποθήκευση
Προσθέστε μια σελίδα στο έγγραφο και αποθηκεύστε την:
```csharp
// Προσθήκη νέας σελίδας στο έγγραφο
Page page = doc.Pages.Add();

// Ορίστε τη διαδρομή εξόδου για το αρχικοποιημένο έγγραφο
outputDir = "YOUR_OUTPUT_DIRECTORY" + "/InitializedPdf_out.pdf";

// Αποθήκευση του αρχικοποιημένου εγγράφου PDF
doc.Save(outputDir);
```
Ο `Pages.Add()` η μέθοδος προσθέτει μια κενή σελίδα και η `Save` Η μέθοδος το γράφει στην καθορισμένη θέση.

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε σε κάθε βήμα της απόδοσης LaTeX σε PDF με σαφήνεια.

### Δημιουργία νέου εγγράφου και σελίδας (H2)
#### Επισκόπηση:
Ξεκινήστε δημιουργώντας μια παρουσία του `Document` τάξη και προσθέστε μια νέα σελίδα σε αυτήν.

**Βήμα 1: Αρχικοποίηση εγγράφου**
```csharp
var doc = new Document();
```
*Εξήγηση:* Αυτή η γραμμή δημιουργεί ένα νέο έγγραφο PDF. Στη συνέχεια, θα προσθέσετε περιεχόμενο, συμπεριλαμβανομένου του σεναρίου LaTeX, σε αυτό το έγγραφο.

#### Βήμα 2: Προσθήκη σελίδας
```csharp
var page = doc.Pages.Add();
```
*Σκοπός:* Προσθέτει μια νέα σελίδα στο έγγραφο όπου μπορείτε να αποδώσετε το σενάριο LaTeX. Κάθε σελίδα λειτουργεί ως δοχείο για το περιεχόμενό της σε αρχεία PDF.

### Προσθήκη σεναρίου LaTeX (H3)
Τώρα, ας προσθέσουμε το σενάριο LaTeX χρησιμοποιώντας το Aspose.PDF `TeXFragment`.

#### Βήμα 1: Ορισμός σεναρίου LaTeX
```csharp
string latexScript = """
    \usepackage{amsmath,amsthm}
    \begin{document}
    \begin{proof} The proof is as follows:
    \begin{align}
    (x+y)^3&=(x+y)(x+y)^2
    (x+y)(x^2+2xy+y^2)\\
    &=x^3+3x^2y+3xy^2+x^3.\qedhere
    \end{align}
    \end{proof}
    \end{document}"
```
*Εξήγηση:* Αυτό το σενάριο περιέχει κώδικα LaTeX για την απόδοση μιας μαθηματικής απόδειξης, συμπεριλαμβανομένων πακέτων και στοίχισης για σωστή μορφοποίηση.

#### Βήμα 2: Δημιουργία TeXFragment
```csharp
var latex = new TeXFragment(latexScript);
```
*Σκοπός:* Μετατρέπει τη συμβολοσειρά LaTeX σε `TeXFragment`, το οποίο το Aspose.PDF μπορεί να ερμηνεύσει για να το αποδώσει μέσα στο έγγραφο PDF.

#### Βήμα 3: Προσθήκη τμήματος στη σελίδα
```csharp
page.Paragraphs.Add(latex);
```
*Εξήγηση:* Προσθέτει το `TeXFragment` στη συλλογή παραγράφων της σελίδας, ενσωματώνοντας το σκριπτ LaTeX στο περιεχόμενο του PDF.

### Αποθήκευση του εγγράφου (H2)
Τέλος, αποθηκεύστε το έγγραφό σας σε μια καθορισμένη τοποθεσία:

#### Βήμα 1: Ορισμός διαδρομής αποθήκευσης
```csharp
var dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
*Σκοπός:* Ορίζει τον κατάλογο όπου θα αποθηκευτεί το PDF εξόδου. Βεβαιωθείτε ότι αυτή η διαδρομή υπάρχει ή είναι εγγράψιμη από την εφαρμογή σας.

#### Βήμα 2: Αποθήκευση εγγράφου
```csharp
doc.Save(dataDir + "Script_out.pdf");
```
*Εξήγηση:* Αυτή η εντολή γράφει το αποτυπωμένο έγγραφο στον δίσκο, δημιουργώντας ένα αρχείο PDF με το περιεχόμενο LaTeX σας στην καθορισμένη θέση.

## Οδηγός Εφαρμογής

### Επισκόπηση

Σε αυτήν την ενότητα, θα δούμε πώς να εξάγετε εικόνες από ψηφιακές υπογραφές μέσα σε ένα έγγραφο PDF. Αυτό είναι ιδιαίτερα χρήσιμο όταν χρειάζεται να επαληθεύσετε ή να αποθηκεύσετε ξεχωριστά τα στοιχεία της υπογραφής.

#### Φόρτωση του εγγράφου PDF

Αρχικά, φορτώστε το αρχείο PDF χρησιμοποιώντας το Aspose.PDF `Document` τάξη:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string inputFilePath = Path.Combine(dataDir, "DigitallySign.pdf");

// Δημιουργήστε μια νέα παρουσία εγγράφου
Document doc = new Document(inputFilePath);
```

#### Εξαγωγή εικόνων από υπογραφές

Δείτε πώς μπορείτε να εξαγάγετε εικόνες ενσωματωμένες στις υπογραφές PDF:

```csharp
using (PdfFileSignature signature = new PdfFileSignature(doc))
{
    // Ελέγξτε εάν το έγγραφο περιέχει ψηφιακές υπογραφές
    if (signature.ContainsSignature())
    {
        foreach (string sigName in signature.GetSignNames())
        {
            string outputFilePath = Path.Combine(dataDir, "ExtractImages_out.jpg");

            using (Stream imageStream = signature.ExtractImage(sigName))
            {
                if (imageStream != null)
                {
                    // Μετατροπή της ροής σε αντικείμενο εικόνας
                    using (System.Drawing.Image image = Bitmap.FromStream(imageStream))
                    {
                        // Αποθηκεύστε την εξαγόμενη εικόνα ως αρχείο JPEG
                        image.Save(outputFilePath, System.Drawing.Imaging.ImageFormat.Jpeg);
                    }
                }
            }
        }
    }
}
```

**Εξήγηση:**
- **`PdfFileSignature`:** Αυτή η κλάση διευκολύνει τις λειτουργίες σε υπογραφές PDF.
- **`ContainsSignature()`:** Ελέγχει εάν υπάρχουν ψηφιακές υπογραφές στο έγγραφο.
- **`ExtractImage(sigName)`:** Εξάγει δεδομένα εικόνας από μια καθορισμένη υπογραφή.

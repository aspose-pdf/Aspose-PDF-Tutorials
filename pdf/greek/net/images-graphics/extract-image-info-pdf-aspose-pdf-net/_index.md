## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε διαχειρίσιμα βήματα:

### 1. Εξαγωγή πληροφοριών εικόνας
**Επισκόπηση**Αυτή η λειτουργία εξάγει και εμφανίζει πληροφορίες σχετικά με εικόνες σε ένα αρχείο PDF, όπως οι διαστάσεις και η ανάλυσή τους.

#### Φόρτωση του αρχείου PDF προέλευσης
Ξεκινήστε καθορίζοντας τον κατάλογο όπου βρίσκεται το έγγραφό σας και φορτώστε το χρησιμοποιώντας το Aspose.PDF. `Document` τάξη.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
Document doc = new Document(dataDir + "ImageInformation.pdf");
```

#### Αρχικοποίηση στοίβας κατάστασης γραφικών
Πρέπει να διαχειριστείτε τους μετασχηματισμούς που εφαρμόζονται σε εικόνες, επομένως αρχικοποιήστε μια στοίβα για να διατηρείτε τις καταστάσεις γραφικών και μια λίστα πινάκων για τα ονόματα εικόνων:
```csharp
System.Collections.Stack graphicsState = new System.Collections.Stack();
System.Collections.ArrayList imageNames = new System.Collections.ArrayList(doc.Pages[1].Resources.Images.Names);
graphicsState.Push(new System.Drawing.Drawing2D.Matrix(1, 0, 0, 1, 0, 0));
```

#### Επανάληψη σε τελεστές PDF
Επαναλάβετε κάθε τελεστή στην πρώτη σελίδα για να διαχειριστείτε τις αλλαγές κατάστασης γραφικών και το σχέδιο εικόνας:
```csharp
foreach (Operator op in doc.Pages[1].Contents)
{
    // Χειρισμός GSave/GRestore για καταστάσεις γραφικών
    if (op is Aspose.Pdf.Operators.GSave opSaveState) 
        graphicsState.Push(((System.Drawing.Drawing2D.Matrix)graphicsState.Peek()).Clone());
    else if (op is Aspose.Pdf.Operators.GRestore opRestoreState)
        graphicsState.Pop();
    
    // Χειρισμός του ConcatenateMatrix για μετασχηματισμούς
    else if (op is Aspose.Pdf.Operators.ConcatenateMatrix opCtm)
    {
        System.Drawing.Drawing2D.Matrix cm = new System.Drawing.Drawing2D.Matrix(
            (float)opCtm.Matrix.A, (float)opCtm.Matrix.B,
            (float)opCtm.Matrix.C, (float)opCtm.Matrix.D,
            (float)opCtm.Matrix.E, (float)opCtm.Matrix.F);
        ((System.Drawing.Drawing2D.Matrix)graphicsState.Peek()).Multiply(cm);
    }

    // Εξαγωγή πληροφοριών εικόνας χρησιμοποιώντας τον τελεστή Do
    if (op is Aspose.Pdf.Operators.Do opDo && imageNames.Contains(opDo.Name))
    {
        System.Drawing.Drawing2D.Matrix lastCTM = (System.Drawing.Drawing2D.Matrix)graphicsState.Peek();
        XImage image = doc.Pages[1].Resources.Images[opDo.Name];

        double scaledWidth = Math.Sqrt(Math.Pow(lastCTM.Elements[0], 2) + Math.Pow(lastCTM.Elements[1], 2));
        double scaledHeight = Math.Sqrt(Math.Pow(lastCTM.Elements[2], 2) + Math.Pow(lastCTM.Elements[3], 2));
        double originalWidth = image.Width;
        double originalHeight = image.Height;

        int defaultResolution = 72; // Προεπιλεγμένη ανάλυση σε DPI
        double resHorizontal = originalWidth * defaultResolution / scaledWidth;
        double resVertical = originalHeight * defaultResolution / scaledHeight;

        Console.WriteLine(string.Format("Image {0} ({1:.##}:{2:.##}): Resolution {3:.##} x {4:.##}",
            opDo.Name, scaledWidth, scaledHeight, resHorizontal, resVertical).Replace(dataDir, ""));
    }
}
```

## Ενότητα Συχνών Ερωτήσεων
### Πώς μπορώ να εγκαταστήσω το Aspose.PDF για .NET;
Μπορείτε να εγκαταστήσετε το Aspose.PDF μέσω του .NET CLI με `dotnet add package Aspose.PDF`, μέσω της Κονσόλας Διαχείρισης Πακέτων χρησιμοποιώντας `Install-Package Aspose.PDF`ή κάνοντας αναζήτηση και εγκατάσταση από το περιβάλλον χρήστη του NuGet Package Manager.

### Τι είναι ένας τελεστής GSave στην επεξεργασία PDF;
Ένας τελεστής GSave αποθηκεύει την τρέχουσα κατάσταση των γραφικών, επιτρέποντάς σας να την επαναφέρετε αργότερα με ένα GRestore. Αυτό είναι απαραίτητο για τη διαχείριση μετασχηματισμών που εφαρμόζονται σε εικόνες μέσα σε ένα έγγραφο PDF.

### Μπορώ να εξαγάγω πληροφορίες από πολλές σελίδες;
Ναι, επέκταση της υλοποίησης επαναλαμβάνοντας όλες τις σελίδες αντί μόνο `doc.Pages[1]`.

### Πώς μπορώ να χειριστώ διαφορετικές μορφές εικόνας σε ένα PDF;
Το Aspose.PDF υποστηρίζει την εξαγωγή μεταδεδομένων και διαστάσεων ανεξάρτητα από την ενσωματωμένη μορφή εικόνας (JPEG, PNG, κ.λπ.).

### Τι γίνεται αν μια εικόνα δεν έχει όνομα στους πόρους του εγγράφου;
Εάν μια εικόνα δεν έχει όνομα, δεν θα συμπεριληφθεί `doc.Pages[1].Resources.Images.Names`Βεβαιωθείτε ότι όλες οι εικόνες έχουν τα κατάλληλα ονόματα ή χειριστείτε τέτοιες περιπτώσεις μέσω προγραμματισμού.

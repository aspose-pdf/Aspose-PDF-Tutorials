## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη διαγραφή συγκεκριμένων ή όλων των εικόνων από ένα έγγραφο PDF.

### Διαγραφή συγκεκριμένης εικόνας

Για να αφαιρέσετε μια εικόνα από το PDF σας:

#### Βήμα 1: Φόρτωση του εγγράφου PDF

Δημιουργήστε μια παρουσία του `Document` τάξη και φορτώστε το αρχείο PDF. Καθορίστε με ποιο PDF εργάζεστε.

```csharp
// ExStart:LoadPDFDocument
using System.IO;
using Aspose.Pdf;

string dataDir = "your_directory_path/";
Document pdfDocument = new Document(dataDir + "DeleteImages.pdf");
```

#### Βήμα 2: Πρόσβαση και διαγραφή της εικόνας

Αποκτήστε πρόσβαση στη σελίδα που περιέχει την εικόνα-στόχο σας. Χρησιμοποιήστε το `Delete` μέθοδος για να το αφαιρέσετε.

```csharp
// ExStart:ΔιαγραφήΕιδικήςΕικόνας
pdfDocument.Pages[1].Resources.Images.Delete(1);
dataDir = dataDir + "DeleteImages_out.pdf";
```

Σε αυτό το παράδειγμα, διαγράφουμε την πρώτη εικόνα από την πρώτη σελίδα. Προσαρμόστε τις παραμέτρους για να στοχεύσετε διαφορετικές εικόνες ή σελίδες, όπως απαιτείται.

#### Βήμα 3: Αποθήκευση του ενημερωμένου εγγράφου

Αφού κάνετε τις αλλαγές, αποθηκεύστε το έγγραφο χρησιμοποιώντας το `Save` μέθοδος.

```csharp
// ExStart:ΑποθήκευσηΕνημερωμένοPDF
pdfDocument.Save(dataDir);
```

### Διαγραφή όλων των εικόνων από μια σελίδα

Για να καταργήσετε όλες τις εικόνες από μια συγκεκριμένη σελίδα:

```csharp
// Περιηγηθείτε σε κάθε εικόνα στους πόρους της σελίδας και διαγράψτε την.
foreach (var img in pdfDocument.Pages[1].Resources.Images)
{
    pdfDocument.Pages[1].Resources.Images.Delete(img.Index);
}
```

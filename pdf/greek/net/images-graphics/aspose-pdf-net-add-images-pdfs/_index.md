## Οδηγός Εφαρμογής
Τώρα, ας αναλύσουμε τα βήματα για να προσθέσετε μια εικόνα σε μια σελίδα PDF.

### Προσθήκη εικόνας σε συγκεκριμένη θέση σε μια σελίδα PDF
**Επισκόπηση**
Αυτή η λειτουργία σάς επιτρέπει να επικαλύπτετε εικόνες στα έγγραφα PDF σας σε οποιαδήποτε καθορισμένη συντεταγμένη, βελτιώνοντας την οπτική τους ελκυστικότητα και τη λειτουργικότητά τους.

#### Βήμα 1: Ανοίξτε το υπάρχον έγγραφο PDF σας
```csharp
Document pdfDocument = new Document("YOUR_DOCUMENT_DIRECTORY/PDFOperators.pdf");
```
- **Εξήγηση**: Αυτή η γραμμή φορτώνει ένα υπάρχον αρχείο PDF στο `pdfDocument` αντικείμενο.

#### Βήμα 2: Καθορισμός συντεταγμένων εικόνας
```csharp
int lowerLeftX = 100;
int lowerLeftY = 100;
int upperRightX = 200;
int upperRightY = 200;
```
- **Εξήγηση**Αυτές οι συντεταγμένες καθορίζουν πού θα τοποθετηθεί η εικόνα στη σελίδα, με το (lowerLeftX, lowerLeftY) ως σημείο εκκίνησης και το (upperRightX, upperRightY) ως σημείο τερματισμού.

#### Βήμα 3: Φόρτωση εικόνας σε ροή αρχείων
```csharp
FileStream imageStream = new FileStream("YOUR_DOCUMENT_DIRECTORY/PDFOperators.jpg", FileMode.Open);
```
- **Εξήγηση**Ανοίγει ένα αρχείο εικόνας που θα προστεθεί στη σελίδα PDF. Βεβαιωθείτε ότι η διαδρομή και το όνομα αρχείου είναι σωστά.

#### Βήμα 4: Προσθήκη εικόνας στους πόρους σελίδας
```csharp
Page page = pdfDocument.Pages[1];
page.Resources.Images.Add(imageStream);
```
- **Εξήγηση**: Ανακτά την πρώτη σελίδα του εγγράφου και προσθέτει τη ροή εικόνων στους πόρους του.

#### Βήμα 5: Ορισμός κατάστασης γραφικών με τον τελεστή GSave
```csharp
page.Contents.Add(new Aspose.Pdf.Operators.GSave());
```
- **Εξήγηση**Αποθηκεύει την τρέχουσα κατάσταση των γραφικών, διασφαλίζοντας ότι τυχόν τροποποιήσεις δεν επηρεάζουν άλλα στοιχεία της σελίδας.

#### Βήμα 6: Δημιουργία και ορισμός πίνακα τοποθέτησης εικόνας
```csharp
Aspose.Pdf.Rectangle rectangle = new Aspose.Pdf.Rectangle(lowerLeftX, lowerLeftY, upperRightX, upperRightY);
Matrix matrix = new Matrix(new double[] { rectangle.URX - rectangle.LLX, 0, 0, rectangle.URY - rectangle.LLY, rectangle.LLX, rectangle.LLY });
page.Contents.Add(new Aspose.Pdf.Operators.ConcatenateMatrix(matrix));
```
- **Εξήγηση**: Ορίζει έναν πίνακα μετασχηματισμού που τοποθετεί την εικόνα σύμφωνα με καθορισμένες συντεταγμένες.

#### Βήμα 7: Σχεδιάστε μια εικόνα χρησιμοποιώντας τον τελεστή Do
```csharp
XImage ximage = page.Resources.Images[page.Resources.Images.Count];
page.Contents.Add(new Aspose.Pdf.Operators.Do(ximage.Name));
```
- **Εξήγηση**: Ανακτά την πρόσφατα προστιθέμενη εικόνα και την τοποθετεί στη σελίδα χρησιμοποιώντας το `Do` χειριστής.

#### Βήμα 8: Επαναφορά κατάστασης γραφικών με τον τελεστή GRestore
```csharp
page.Contents.Add(new Aspose.Pdf.Operators.GRestore());
```
- **Εξήγηση**: Επαναφέρει την κατάσταση των γραφικών στην αρχική τους μορφή μετά τη σχεδίαση της εικόνας.

#### Βήμα 9: Αποθήκευση του τροποποιημένου εγγράφου PDF
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/PDFOperators_out.pdf";
pdfDocument.Save(outputDir);
```
- **Εξήγηση**Αποθηκεύει όλες τις αλλαγές που έγιναν σε ένα νέο αρχείο στον καθορισμένο κατάλογο.

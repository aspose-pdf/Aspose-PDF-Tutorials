## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία σε λογικά βήματα για την υλοποίηση της μετατροπής εικόνας σε PDF.

### Βήμα 1: Προετοιμασία του έργου σας και εισαγωγή των απαραίτητων χώρων ονομάτων
Βεβαιωθείτε ότι το έργο σας έχει αναφορές σε απαραίτητους χώρους ονομάτων:

```csharp
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Devices;
using System.Drawing; // Απαιτείται για λειτουργίες Bitmap
```

### Βήμα 2: Φόρτωση της εικόνας σε μια ροή
Φορτώστε το αρχείο εικόνας σας σε μια ροή. Αυτό το παράδειγμα χρησιμοποιεί μια εικόνα JPEG, αλλά μπορεί να προσαρμοστεί και σε άλλες μορφές:

```csharp
string dataDir = "your_directory_path";
using (FileStream fs = new FileStream(dataDir + "aspose-logo.jpg", FileMode.Open, FileAccess.Read))
{
    byte[] tmpBytes = new byte[fs.Length];
    fs.Read(tmpBytes, 0, int.Parse(fs.Length.ToString()));

    MemoryStream mystream = new MemoryStream(tmpBytes);
}
```

### Βήμα 3: Δημιουργήστε ένα έγγραφο PDF και προσθέστε μια σελίδα εικόνας
Δημιουργήστε ένα στιγμιότυπο του `Document` αντικείμενο και προσθέστε μια σελίδα. Χρησιμοποιήστε ένα `Bitmap` για να διαχειριστείτε τις ιδιότητες της εικόνας:

```csharp
Document doc = new Document();
Page page = doc.Pages.Add();

using (MemoryStream mystream = new MemoryStream(tmpBytes))
{
    Bitmap b = new Bitmap(mystream);

    // Ορισμός περιθωρίων στο μηδέν για πλήρη προσαρμογή εικόνας
    page.PageInfo.Margin.Bottom = 0;
    page.PageInfo.Margin.Top = 0;
    page.PageInfo.Margin.Left = 0;
    page.PageInfo.Margin.Right = 0;

    page.CropBox = new Aspose.Pdf.Rectangle(0, 0, b.Width, b.Height);
    
    // Δημιουργήστε ένα αντικείμενο εικόνας και ορίστε τη ροή του
    Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
    page.Paragraphs.Add(image1);

    image1.ImageStream = mystream;
}
```

### Βήμα 4: Αποθήκευση του εγγράφου PDF
Τέλος, αποθηκεύστε το έγγραφό σας σε ένα αρχείο:

```csharp
string outputDir = dataDir + "ImageToPDF_out.pdf";
doc.Save(outputDir);
Console.WriteLine("\nImage converted to PDF successfully.\nFile saved at " + outputDir);
```

# Δημιουργήστε οπτικά ελκυστικά έγγραφα PDF με επισημάνσεις περιγράμματος χρησιμοποιώντας το Aspose.PDF .NET

## Οδηγός Εφαρμογής
Ας χωρίσουμε τη διαδικασία υλοποίησης σε διαχειρίσιμα μέρη.
### Εξαγωγή παραγράφων και σχεδίαση περιγραμμάτων (H2)
Αυτή η λειτουργία σάς επιτρέπει να επισημάνετε συγκεκριμένες παραγράφους μέσα σε ένα PDF σχεδιάζοντας περιγράμματα γύρω τους, βελτιώνοντας την αναγνωσιμότητα και την εστίαση.
#### Βήμα 1: Φόρτωση του εγγράφου PDF (H3)
Αρχικά, φορτώστε το έγγραφο PDF χρησιμοποιώντας το Aspose.PDF:
```csharp
Document doc = new Document("input.pdf");
Page page = doc.Pages[2]; // Αποκτήστε πρόσβαση στη συγκεκριμένη σελίδα στην οποία θέλετε να εργαστείτε.
```
#### Βήμα 2: Αρχικοποίηση του Απορροφητή Παραγράφων (H3)
Ο `ParagraphAbsorber` Η κλάση βοηθά στην αναγνώριση και εξαγωγή παραγράφων από ένα PDF:
```csharp
ParagraphAbsorber absorber = new ParagraphAbsorber();
absorber.Visit(page);
PageMarkup markup = absorber.PageMarkups[0];
```
#### Βήμα 3: Σχεδιάστε περιγράμματα γύρω από παραγράφους (H3)
Για να τονίσετε οπτικά το εξαγόμενο κείμενο, σχεδιάστε ορθογώνια και πολύγωνα γύρω τους:
```csharp
foreach (MarkupSection section in markup.Sections)
{
    // Σχεδιάστε ένα ορθογώνιο γύρω από κάθε τμήμα.
    DrawRectangleOnPage(section.Rectangle, page);

    foreach (MarkupParagraph paragraph in section.Paragraphs)
    {
        // Επισημάνετε παραγράφους με περίγραμμα πολυγώνου.
        DrawPolygonOnPage(paragraph.Points, page);
    }
}
// Αποθηκεύστε το τροποποιημένο έγγραφο.
doc.Save("output_out.pdf");
```
#### Επεξήγηση μεθόδων σχεδίασης
- **Σχεδίασηορθογώνιουστησελίδα**Αυτή η μέθοδος σκιαγραφεί τις ενότητες χρησιμοποιώντας ορθογώνια. Ορίζει το χρώμα της γραμμής σε πράσινο και προσαρμόζει το πλάτος της γραμμής για ορατότητα.
```csharp
private static void DrawRectangleOnPage(Rectangle rectangle, Page page)
{
    page.Contents.Add(new Aspose.Pdf.Operators.GSave());
    page.Contents.Add(new Aspose.Pdf.Operators.ConcatenateMatrix(1, 0, 0, 1, 0, 0));
    page.Contents.Add(new Aspose.Pdf.Operators.SetRGBColorStroke(0, 1, 0)); // Πράσινο χρώμα.
    page.Contents.Add(new Aspose.Pdf.Operators.SetLineWidth(2));
    page.Contents.Add(
        new Aspose.Pdf.Operators.Re(rectangle.LLX,
            rectangle.LLY,
            rectangle.Width,
            rectangle.Height));
    page.Contents.Add(new Aspose.Pdf.Operators.ClosePathStroke());
    page.Contents.Add(new Aspose.Pdf.Operators.GRestore());
}
```
- **ΣχεδίασηΠολυγώνουΣτηΣελίδα**: Αυτή η συνάρτηση επισημαίνει μεμονωμένες παραγράφους συνδέοντας σημεία με γραμμές, χρησιμοποιώντας μπλε χρώμα περιγράμματος.
```csharp
private static void DrawPolygonOnPage(Point[] polygon, Page page)
{
    page.Contents.Add(new Aspose.Pdf.Operators.GSave());
    page.Contents.Add(new Aspose.Pdf.Operators.ConcatenateMatrix(1, 0, 0, 1, 0, 0));
    page.Contents.Add(new Aspose.Pdf.Operators.SetRGBColorStroke(0, 0, 1)); // Μπλε χρώμα.
    page.Contents.Add(new Aspose.Pdf.Operators.SetLineWidth(1));
    page.Contents.Add(new Aspose.Pdf.Operators.MoveTo(polygon[0].X, polygon[0].Y));

    for (int i = 1; i < polygon.Length; i++)
    {
        page.Contents.Add(new Aspose.Pdf.Operators.LineTo(polygon[i].X, polygon[i].Y));
    }
    
    // Κλείστε τη διαδρομή συνδέοντας ξανά το πρώτο σημείο.
    page.Contents.Add(new Aspose.Pdf.Operators.LineTo(polygon[0].X, polygon[0].Y));
    page.Contents.Add(new Aspose.Pdf.Operators.ClosePathStroke());
    page.Contents.Add(new Aspose.Pdf.Operators.GRestore());
}
```

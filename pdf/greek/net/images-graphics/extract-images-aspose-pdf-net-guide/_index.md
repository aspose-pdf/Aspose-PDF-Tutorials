## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Φόρτωση εγγράφων και εξαγωγή εικόνων

Αυτή η λειτουργία έχει σχεδιαστεί για τη φόρτωση ενός εγγράφου PDF και την εξαγωγή εικόνων που είναι ενσωματωμένες σε αυτό χρησιμοποιώντας τη βιβλιοθήκη Aspose.PDF.

#### Επισκόπηση
Ο κύριος στόχος εδώ είναι η πρόσβαση μέσω προγραμματισμού σε κάθε εικόνα στο αρχείο PDF σας, επιτρέποντας περαιτέρω χειρισμό ή ανάλυση αυτών των εικόνων.

##### Βήμα 1: Φόρτωση του εγγράφου PDF
Αρχικά, ανοίξτε το έγγραφο PDF από το οποίο θέλετε να εξαγάγετε εικόνες:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/SearchAndGetImages.pdf";
Document doc = new Document(dataDir);
```

Αυτό το απόσπασμα κώδικα ορίζει τη διαδρομή του αρχείου σας και φορτώνει το έγγραφο στη μνήμη.

##### Βήμα 2: Δημιουργήστε ένα αντικείμενο ImagePlacementAbsorber
Στη συνέχεια, δημιουργήστε μια παρουσία του `ImagePlacementAbsorber` για να εντοπίσετε τις τοποθετήσεις εικόνων μέσα στο PDF:
```csharp
ImagePlacementAbsorber abs = new ImagePlacementAbsorber();
```

##### Βήμα 3: Αποδοχή του Absorber για όλες τις σελίδες
Επαναλάβετε κάθε σελίδα στο έγγραφό σας και αποδεχτείτε τον απορροφητή, επιτρέποντάς του να αναζητήσει εικόνες:
```csharp
foreach (Page page in doc.Pages)
{
    page.Accept(abs);
}
```

Αυτός ο βρόχος διασφαλίζει ότι λαμβάνεται υπόψη κάθε εικόνα μέσα στο PDF.

##### Βήμα 4: Εξαγωγή εικόνων από το ImagePlacements
Τέλος, εξαγάγετε κάθε εικόνα που βρέθηκε χρησιμοποιώντας το `ImagePlacement` αντικείμενο:
```csharp
foreach (ImagePlacement imagePlacement in abs.ImagePlacements)
{
    XImage image = imagePlacement.Image;

    // Εμφάνιση ορισμένων ιδιοτήτων για επίδειξη
    Console.WriteLine("Image width: " + imagePlacement.Rectangle.Width);
    Console.WriteLine("Image height: " + imagePlacement.Rectangle.Height);
}
```

Σε αυτήν την ενότητα, εξάγουμε και εκτυπώνουμε το πλάτος και το ύψος κάθε εικόνας. Μπορείτε να το τροποποιήσετε για να αποθηκεύσετε εικόνες ή να εκτελέσετε άλλες λειτουργίες.

## Οδηγός Εφαρμογής

Τώρα, ας δούμε βήμα προς βήμα την εξαγωγή εικόνων από αρχεία PDF.

### Αρχικοποίηση του PdfExtractor

Ξεκινήστε ρυθμίζοντας ένα `PdfExtractor` αντικείμενο. Αυτή θα είναι η πύλη σας για την πρόσβαση στο περιεχόμενο ενός εγγράφου PDF.

```csharp
// Δημιουργία αντικειμένου PdfExtractor
PdfExtractor pdfExtractor = new PdfExtractor();
pdfExtractor.BindPdf(dataDir + "ExtractImages-Stream.pdf");
```

#### Δέστε το PDF

Εδώ, συνδέουμε το αρχείο PDF προορισμού με το `PdfExtractor` παράδειγμα. Αυτό το βήμα προετοιμάζει τον εξάγοντα για την εξαγωγή εικόνας.

### Εξαγωγή εικόνων από PDF

Επικαλέστε το `ExtractImage()` μέθοδος για την έναρξη της διαδικασίας εξαγωγής:

```csharp
// Έναρξη εξαγωγής εικόνων
pdfExtractor.ExtractImage();
```

#### Χειρισμός εξαγόμενων εικόνων

Επαναλάβετε τις εξαγόμενες εικόνες χρησιμοποιώντας έναν βρόχο, καταγράφοντας την καθεμία σε μια ροή μνήμης για περαιτέρω επεξεργασία ή αποθήκευση σε δίσκο.

```csharp
while (pdfExtractor.HasNextImage())
{
    MemoryStream memoryStream = new MemoryStream();
    pdfExtractor.GetNextImage(memoryStream);

    // Αποθήκευση αρχείου εικόνας στον δίσκο με ένα μοναδικό όνομα με βάση την τρέχουσα χρονική σήμανση
    using (FileStream fileStream = new FileStream(dataDir + DateTime.Now.Ticks.ToString() + "_out.jpg", FileMode.Create))
    {
        memoryStream.WriteTo(fileStream);
    }
}
```

**Σημείωμα:** Η χρήση του `MemoryStream` Επιτρέπει την επεξεργασία εντός της μνήμης, η οποία μπορεί να είναι πιο αποτελεσματική από την απευθείας εγγραφή στο δίσκο.

## Οδηγός Εφαρμογής

### Προσθήκη εικόνας σε υπάρχον PDF

Αυτή η λειτουργία σάς επιτρέπει να προσθέτετε εικόνες σε συγκεκριμένες θέσεις μέσα σε υπάρχοντα έγγραφα PDF.

#### Επισκόπηση

Μάθετε πώς να τοποθετείτε και να εισάγετε εικόνες σε ένα PDF, ιδανικό για την προσθήκη λογότυπων ή προσαρμοσμένων γραφικών στα έγγραφά σας.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Φόρτωση του εγγράφου PDF

Ξεκινήστε ανοίγοντας ένα υπάρχον αρχείο PDF χρησιμοποιώντας το Aspose.PDF `Document` τάξη.

```csharp
using System.IO;
using Aspose.Pdf;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Document pdfDocument = new Document(dataDir + "/AddImage.pdf");
```

##### Βήμα 2: Ορισμός συντεταγμένων εικόνας

Προσδιορίστε πού θέλετε να εμφανίζεται η εικόνα στη σελίδα PDF σας ορίζοντας συντεταγμένες.

```csharp
int lowerLeftX = 100;
int lowerLeftY = 100;
int upperRightX = 200;
int upperRightY = 200;

Page page = pdfDocument.Pages[1];
```

##### Βήμα 3: Φόρτωση της εικόνας σε μια ροή

Φορτώστε το αρχείο εικόνας σας σε μια ροή για να επιτρέψετε στο Aspose.PDF να έχει πρόσβαση και να το χειριστεί.

```csharp
FileStream imageStream = new FileStream(dataDir + "/aspose-logo.jpg", FileMode.Open);
page.Resources.Images.Add(imageStream);
```

##### Βήμα 4: Θέση και σχεδίαση της εικόνας

Χρησιμοποιήστε τελεστές όπως `GSave`, `ConcatenateMatrix`, και `Do` για να τοποθετήσετε και να αποδώσετε την εικόνα.

```csharp
page.Contents.Add(new GSave());

Rectangle rectangle = new Rectangle(lowerLeftX, lowerLeftY, upperRightX, upperRightY);
Matrix matrix = new Matrix(new double[] { rectangle.URX - rectangle.LLX, 0, 0, rectangle.URY - rectangle.LLY, rectangle.LLX, rectangle.LLY });

page.Contents.Add(new ConcatenateMatrix(matrix));
XImage ximage = page.Resources.Images[page.Resources.Images.Count];
page.Contents.Add(new Do(ximage.Name));

page.Contents.Add(new GRestore());
```

##### Βήμα 5: Αποθήκευση του ενημερωμένου εγγράφου

Τέλος, αποθηκεύστε το έγγραφό σας με την εικόνα που μόλις προσθέσατε.

```csharp
string outputDir = dataDir + "/AddImage_out.pdf";
pdfDocument.Save(outputDir);
```

### Προσθήκη εικόνας DICOM σε νέο PDF

Αυτή η λειτουργία επιδεικνύει τη δημιουργία ενός νέου PDF από ένα αρχείο DICOM, το οποίο χρησιμοποιείται συνήθως στην ιατρική απεικόνιση.

#### Επισκόπηση

Μάθετε πώς να μετατρέπετε και να συμπεριλαμβάνετε εικόνες DICOM σε νεοδημιουργημένα PDF, βελτιώνοντας τις δυνατότητες επεξεργασίας εγγράφων σας.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Δημιουργία νέου εγγράφου

Αρχικοποίηση νέου `Document` αντικείμενο που θα χρησιμεύσει ως κοντέινερ PDF.

```csharp
using Aspose.Pdf;
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

using (Document pdfDocument = new Document())
{
    pdfDocument.Pages.Add();
```

##### Βήμα 2: Ρύθμιση παραμέτρων της εικόνας DICOM

Ρύθμιση ενός `Aspose.Pdf.Image` αντικείμενο για τον χειρισμό του αρχείου DICOM.

```csharp
    Aspose.Pdf.Image image = new Aspose.Pdf.Image();
    image.FileType = ImageFileType.Dicom;
    image.ImageStream = new FileStream(dataDir + "/0002.dcm", FileMode.Open, FileAccess.Read);
```

##### Βήμα 3: Προσθήκη εικόνας στο PDF

Εισαγάγετε την εικόνα στη σελίδα του εγγράφου σας.

```csharp
    pdfDocument.Pages[1].Paragraphs.Add(image);

    string dicomOutputDir = dataDir + "/PdfWithDicomImage_out.pdf";
    pdfDocument.Save(dicomOutputDir);
}
```

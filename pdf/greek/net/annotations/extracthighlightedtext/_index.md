## Εισαγωγή πακέτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας σε C#. Αυτό είναι κρίσιμο για την πρόσβαση στις κλάσεις και τις μεθόδους που παρέχονται από το Aspose.PDF για .NET.

```csharp
using Aspose.Pdf.Text;
using System;

using System.Linq;
```

Τώρα, ας αναλύσουμε τη διαδικασία εξαγωγής επισημασμένου κειμένου από ένα αρχείο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Κάθε βήμα εξηγείται λεπτομερώς για να σας βοηθήσει να κατανοήσετε τις υποκείμενες έννοιες και την υλοποίηση.

## Βήμα 1: Ρύθμιση του καταλόγου έργου σας

Πρώτα απ 'όλα, πρέπει να ρυθμίσετε τον κατάλογο του έργου σας όπου βρίσκεται το αρχείο PDF. Εδώ συμβαίνει η μαγεία.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογό σας όπου βρίσκεται το αρχείο PDF. Σε αυτόν τον κατάλογο η εφαρμογή θα ανακτήσει το PDF για επεξεργασία.

## Βήμα 2: Φόρτωση του εγγράφου PDF

Στη συνέχεια, πρέπει να φορτώσετε το έγγραφο PDF από το οποίο θέλετε να εξαγάγετε το επισημασμένο κείμενο. Αυτό γίνεται χρησιμοποιώντας το `Document` κλάση που παρέχεται από το Aspose.PDF.

```csharp
Document doc = new Document(dataDir + "ExtractHighlightedText.pdf");
```

Ο `Document` Η κλάση δημιουργείται με τη διαδρομή προς το αρχείο PDF. Εδώ, `"ExtractHighlightedText.pdf"` είναι το όνομα του αρχείου PDF που περιέχει το επισημασμένο κείμενο. Βεβαιωθείτε ότι αυτό το αρχείο υπάρχει στον καθορισμένο κατάλογο.

## Βήμα 3: Πρόσβαση στη Συλλογή Σχολιασμών

Μόλις φορτωθεί το έγγραφο PDF, το επόμενο βήμα είναι να αποκτήσετε πρόσβαση στις σημειώσεις στην πρώτη σελίδα του εγγράφου. Οι σημειώσεις χρησιμοποιούνται στα PDF για την προσθήκη πρόσθετων πληροφοριών, όπως επισημάνσεις, σχόλια και άλλα.

```csharp
foreach (Annotation annotation in doc.Pages[1].Annotations)
{
```

Ο `Annotations` ιδιοκτησία του `Page` Το αντικείμενο παρέχει πρόσβαση σε όλες τις σχολιασμοί σε μια συγκεκριμένη σελίδα του PDF. Εδώ, κάνουμε επανάληψη σε κάθε σχολιασμό στην πρώτη σελίδα.

## Βήμα 4: Φιλτράρισμα για σχολιασμούς με επισημασμένο κείμενο

Τώρα που έχουμε πρόσβαση σε όλες τις σχολιάσεις, πρέπει να φιλτράρουμε μόνο τις επισημασμένες σχολιάσεις κειμένου. Αυτό επιτυγχάνεται ελέγχοντας τον τύπο κάθε σχολιασμού.

```csharp
if (annotation is TextMarkupAnnotation)
{
    TextMarkupAnnotation highlightedAnnotation = annotation as TextMarkupAnnotation;
```

Ο `TextMarkupAnnotation` Η κλάση χρησιμοποιείται για την αναπαράσταση σχολίων σήμανσης κειμένου, συμπεριλαμβανομένων των επισημάνσεων. `is` η λέξη-κλειδί ελέγχει αν η σημείωση είναι του τύπου `TextMarkupAnnotation`και αν είναι, μεταδίδει την σχολίαση σε `TextMarkupAnnotation`.

## Βήμα 5: Εξαγωγή του επισημασμένου κειμένου

Αφού προσδιοριστεί η επισημασμένη σχολίαση, το επόμενο βήμα είναι η εξαγωγή του κειμένου που σχετίζεται με την επισήμανση.

```csharp
TextFragmentCollection collection = highlightedAnnotation.GetMarkedTextFragments();
foreach (TextFragment tf in collection)
{
    Console.WriteLine(tf.Text);
}
```

Ο `GetMarkedTextFragments()` η μέθοδος επιστρέφει μια συλλογή από `TextFragment` αντικείμενα, καθένα από τα οποία αντιπροσωπεύει ένα τμήμα του επισημασμένου κειμένου. Κάνουμε επανάληψη σε αυτήν τη συλλογή και εκτυπώνουμε το κείμενο κάθε τμήματος στην κονσόλα.

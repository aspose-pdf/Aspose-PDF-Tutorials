# Διαγράψτε λέξεις

## Εισαγωγή

Έχετε ποτέ χρειαστεί να δώσετε έμφαση σε συγκεκριμένο κείμενο σε ένα PDF διαγράφοντάς το; Είτε εξετάζετε έγγραφα, είτε προσθέτετε σημειώσεις σε κείμενο, είτε απλώς χρειάζεται να επισημάνετε συγκεκριμένες ενότητες, η διαγραφή λέξεων μπορεί να αποτελέσει ένα πολύτιμο εργαλείο. Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να το κάνετε αυτό χρησιμοποιώντας το Aspose.PDF για .NET. Αυτός ο περιεκτικός οδηγός θα σας καθοδηγήσει σε κάθε βήμα, διασφαλίζοντας ότι έχετε όλες τις απαραίτητες πληροφορίες για να εφαρμόσετε αποτελεσματικά αυτήν τη λειτουργία στις εφαρμογές .NET σας. 

## Προαπαιτούμενα

Πριν προχωρήσουμε στον κώδικα, υπάρχουν μερικές προϋποθέσεις που θα πρέπει να πληροίτε για να παρακολουθήσετε αυτό το σεμινάριο:

1. Aspose.PDF για βιβλιοθήκη .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.PDF για .NET. Μπορείτε να [κατεβάστε το εδώ](https://releases.aspose.com/pdf/net/).

2. .NET Framework: Βεβαιωθείτε ότι έχετε εγκαταστήσει το .NET Framework στον υπολογιστή σας. Αυτό το σεμινάριο έχει σχεδιαστεί για εφαρμογές .NET.

3. Περιβάλλον Ανάπτυξης: Θα χρειαστείτε ένα IDE όπως το Visual Studio για να γράψετε και να εκτελέσετε τον κώδικά σας.

4. Έγγραφο PDF: Να έχετε έτοιμο ένα δείγμα αρχείου PDF με το οποίο θέλετε να εργαστείτε. Αυτό θα είναι το έγγραφο από το οποίο θα διαγράψουμε το κείμενο.

5. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# είναι απαραίτητη για την κατανόηση και την εφαρμογή των βημάτων σε αυτό το σεμινάριο.

## Εισαγωγή πακέτων

Πριν ξεκινήσουμε τον προγραμματισμό, πρέπει να εισαγάγουμε τους απαραίτητους χώρους ονομάτων στο έργο .NET μας. Αυτό θα μας δώσει πρόσβαση στις κλάσεις και τις μεθόδους που απαιτούνται για τον χειρισμό αρχείων PDF χρησιμοποιώντας το Aspose.PDF.

```csharp
using System;
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```

Αυτοί οι χώροι ονομάτων είναι απαραίτητοι για την εργασία με έγγραφα PDF, τον χειρισμό κειμένου και την προσθήκη σχολίων όπως διακριτές γραμμές.

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία διαγραφής λέξεων σε ένα έγγραφο PDF σε απλά και εύχρηστα βήματα. Κάθε βήμα θα συνοδεύεται από μια λεπτομερή εξήγηση για να διασφαλίσουμε ότι κατανοείτε τον τρόπο λειτουργίας.

## Βήμα 1: Φόρτωση του εγγράφου PDF

Το πρώτο βήμα είναι να φορτώσετε το έγγραφο PDF που θέλετε να επεξεργαστείτε. Αυτό το έγγραφο θα είναι αυτό από το οποίο θα διαγράφετε συγκεκριμένες λέξεις ή φράσεις.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Άνοιγμα του εγγράφου PDF
Document document = new Document(dataDir + "input.pdf");
```

- `dataDir`: Αυτή η μεταβλητή περιέχει τη διαδρομή προς τον κατάλογο εγγράφων σας. Αντικαταστήστε `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου βρίσκεται το αρχείο PDF σας.
- `Document`: Το `Document` Η κλάση αντιπροσωπεύει ένα έγγραφο PDF. Μεταβιβάζοντας τη διαδρομή του αρχείου στον κατασκευαστή του, ανοίγουμε το αρχείο PDF για επεξεργασία.

## Βήμα 2: Δημιουργήστε ένα πρόγραμμα απορρόφησης θραυσμάτων κειμένου για να βρείτε συγκεκριμένο κείμενο

Στη συνέχεια, θα δημιουργήσουμε μια παρουσία του `TextFragmentAbsorber` για να αναζητήσετε ένα συγκεκριμένο τμήμα κειμένου στο έγγραφο PDF. Αυτό μας επιτρέπει να εντοπίσουμε το κείμενο που θέλουμε να διαγράψουμε.

```csharp
// Δημιουργήστε μια παρουσία TextFragment Absorber για να αναζητήσετε ένα συγκεκριμένο τμήμα κειμένου
Aspose.Pdf.Text.TextFragmentAbsorber textFragmentAbsorber = new Aspose.Pdf.Text.TextFragmentAbsorber("Estoque");
```

- `TextFragmentAbsorber`Αυτή η κλάση χρησιμοποιείται για την εύρεση και την επεξεργασία συγκεκριμένων τμημάτων κειμένου μέσα σε ένα έγγραφο PDF. Σε αυτό το παράδειγμα, αναζητούμε τη λέξη "Estoque". Αντικαταστήστε τη λέξη "Estoque" με τη λέξη ή τη φράση που θέλετε να βρείτε στο έγγραφό σας.

## Βήμα 3: Επαναλάβετε τις σελίδες του εγγράφου PDF

Τώρα που έχουμε το δικό μας `TextFragmentAbsorber`, πρέπει να επαναλάβουμε την αναζήτηση σε κάθε σελίδα του εγγράφου PDF για να βρούμε το καθορισμένο κείμενο.

```csharp
// Επαναλάβετε την περιήγηση στις σελίδες του εγγράφου PDF
for (int i = 1; i <= document.Pages.Count; i++)
{
    // Λήψη της τρέχουσας σελίδας του εγγράφου PDF
    Page page = document.Pages[i];
    page.Accept(textFragmentAbsorber);
}
```

- `for (int i = 1; i <= document.Pages.Count; i++)`: Αυτός ο βρόχος επαναλαμβάνεται σε κάθε σελίδα του εγγράφου PDF.
- `document.Pages[i]`: Ανακτά την τρέχουσα σελίδα που βρίσκεται υπό επεξεργασία.
- `page.Accept(textFragmentAbsorber)`: Αυτή η μέθοδος εφαρμόζει το `TextFragmentAbsorber` στην τρέχουσα σελίδα, αναζητώντας το καθορισμένο κείμενο.

## Βήμα 4: Συλλογή και επεξεργασία των τμημάτων κειμένου

Αφού επανεξετάσουμε τις σελίδες, θα συλλέξουμε τα αποσπάσματα κειμένου που βρέθηκαν και θα τα προετοιμάσουμε για περαιτέρω επεξεργασία.

```csharp
// Δημιουργήστε μια συλλογή από απορροφημένα αποσπάσματα κειμένου
Aspose.Pdf.Text.TextFragmentCollection textFragmentCollection = textFragmentAbsorber.TextFragments;
```

- `TextFragmentCollection`Αυτή η συλλογή αποθηκεύει όλα τα αποσπάσματα κειμένου που βρέθηκαν στο έγγραφο. Θα χρησιμοποιήσουμε αυτήν τη συλλογή στο επόμενο βήμα για να διαγράψουμε το κείμενο.

## Βήμα 5: Επαναλάβετε τα αποσπάσματα κειμένου και διαγράψτε τα

Σε αυτό το βήμα, θα επανέλθουμε σε κάθε τμήμα κειμένου στη συλλογή μας και θα εφαρμόσουμε μια διακριτή σημείωση σε αυτό.

```csharp
// Επανάληψη της συλλογής τμημάτων κειμένου
for (int j = 1; j <= textFragmentCollection.Count; j++)
{
	Aspose.Pdf.Text.TextFragment textFragment = textFragmentCollection[j];

    // Λήψη των ορθογώνιων διαστάσεων του αντικειμένου TextFragment
    Aspose.Pdf.Rectangle rect = new Aspose.Pdf.Rectangle(
        (float)textFragment.Position.XIndent,
        (float)textFragment.Position.YIndent,
        (float)textFragment.Position.XIndent + (float)textFragment.Rectangle.Width,
        (float)textFragment.Position.YIndent + (float)textFragment.Rectangle.Height);

    // Δημιουργία στιγμιαίας παρουσίας σχολιασμού StrikeOut
    StrikeOutAnnotation strikeOut = new StrikeOutAnnotation(textFragment.Page, rect);

    // Ορισμός ιδιοτήτων της σχολίασης διαγραφής
    strikeOut.Opacity = .80f;
    strikeOut.Border = new Border(strikeOut);
    strikeOut.Color = Aspose.Pdf.Color.Red;

    // Προσθήκη της σχολίασης στη συλλογή σχολίων της σελίδας του τμήματος κειμένου
    textFragment.Page.Annotations.Add(strikeOut);
}
```

- `TextFragment textFragment = textFragmentCollection[j]`: Αυτή η γραμμή ανακτά το τρέχον τμήμα κειμένου.
- `Aspose.Pdf.Rectangle`Υπολογίζουμε τις ορθογώνιες διαστάσεις του τμήματος κειμένου για να προσδιορίσουμε πού θα εφαρμόσουμε τη διαγραφή.
- `StrikeOutAnnotation`Αυτή η κλάση αντιπροσωπεύει την σχολίαση διαγραφής. Την δημιουργούμε με το υπολογισμένο ορθογώνιο και την τρέχουσα σελίδα.
- `strikeOut.Opacity`Αυτή η ιδιότητα ορίζει την αδιαφάνεια της διακριτής διαγραφής, καθιστώντας την ορατή κατά 80%.
- `strikeOut.Color`Ορίσαμε το χρώμα της διαγραφής σε κόκκινο. Μπορείτε να το αλλάξετε σε οποιοδήποτε χρώμα προτιμάτε.
- `textFragment.Page.Annotations.Add(strikeOut)`: Αυτό προσθέτει τη σχολίαση διακριτής διαγραφής στη σελίδα.

## Βήμα 6: Αποθήκευση του τροποποιημένου εγγράφου PDF

Το τελευταίο βήμα είναι να αποθηκεύσετε το τροποποιημένο έγγραφο PDF με τις διαγραμμένες γραφές.

```csharp
// Αποθήκευση του ενημερωμένου εγγράφου PDF
dataDir = dataDir + "StrikeOutWords_out.pdf";
document.Save(dataDir);
```

- `dataDir + "StrikeOutWords_out.pdf"`Αυτό δημιουργεί ένα νέο όνομα αρχείου για το τροποποιημένο έγγραφο. Το αρχικό αρχείο παραμένει αμετάβλητο.
- `document.Save(dataDir)`Αποθηκεύει το έγγραφο PDF με τις διαγραμμένες γραμμές στην καθορισμένη θέση.

## Σύναψη

Συγχαρητήρια! Διαγράψατε με επιτυχία συγκεκριμένες λέξεις σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Ακολουθώντας αυτόν τον αναλυτικό οδηγό, μπορείτε πλέον να προσαρμόσετε τα έγγραφα PDF επισημαίνοντας ή διαγράφοντας κείμενο, κάνοντάς τα πιο δυναμικά και προσαρμοσμένα στις ανάγκες σας. Είτε προσθέτετε σχόλια σε νομικά έγγραφα, είτε προετοιμάζετε αναφορές, είτε απλώς επισημαίνετε κείμενο για αναθεώρηση, αυτό το σεμινάριο σας έχει εξοπλίσει με τις δεξιότητες για να το κάνετε αποτελεσματικά.

## Συχνές ερωτήσεις

### Μπορώ να αλλάξω το χρώμα της διαγραφής;

Ναι, μπορείτε να αλλάξετε το χρώμα τροποποιώντας το `strikeOut.Color` ιδιότητα. Για παράδειγμα, μπορείτε να την ορίσετε σε `Aspose.Pdf.Color.Blue` για ένα μπλε strikeout.

### Είναι δυνατόν να διαγράψω πολλές λέξεις ταυτόχρονα;

Απολύτως! Το `TextFragmentAbsorber` μπορεί να χρησιμοποιηθεί για την αναζήτηση οποιασδήποτε λέξης ή φράσης στο έγγραφο. Μπορείτε να εφαρμόσετε τη διαγραφή σε πολλαπλές περιπτώσεις επαναλαμβάνοντας την `TextFragmentCollection`.

### Τι γίνεται αν θέλω να διαγράψω κείμενο μόνο σε συγκεκριμένες σελίδες;

Μπορείτε να τροποποιήσετε τον βρόχο που επαναλαμβάνεται στις σελίδες ώστε να περιλαμβάνει μόνο τις σελίδες που θέλετε να τροποποιήσετε. Για παράδειγμα, `for (int i = 1; i <= 3; i++)` θα εφάρμοζε τη διαγραφή μόνο στις τρεις πρώτες σελίδες.

### Πώς μπορώ να ρυθμίσω το πάχος της γραμμής διαγραφής;

Μπορείτε να προσαρμόσετε το πάχος της γραμμής διαγραφής τροποποιώντας το `Border` ιδιοκτησία του `StrikeOutAnnotation`Αυτό επιτρέπει την προσαρμογή της εμφάνισης της διακριτής διαγραφής.

### Υπάρχει τρόπος να αναιρέσω τη διαγραφή μετά την αποθήκευση του εγγράφου;

Μόλις αποθηκευτεί το έγγραφο, η διαγραφή είναι μόνιμη. Εάν χρειάζεται να διατηρήσετε το αρχικό κείμενο χωρίς τη διαγραφή, σκεφτείτε να αποθηκεύσετε ένα αντίγραφο ασφαλείας του αρχικού εγγράφου πριν εφαρμόσετε οποιεσδήποτε τροποποιήσεις.

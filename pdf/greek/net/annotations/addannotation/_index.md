## Εισαγωγή πακέτων

Πριν ξεκινήσετε την κωδικοποίηση, βεβαιωθείτε ότι έχετε εισαγάγει τους απαραίτητους χώρους ονομάτων:

```csharp
using Aspose.Pdf;

```

Αυτοί οι χώροι ονομάτων παρέχουν πρόσβαση στις κλάσεις και τις μεθόδους που χρειάζεστε για χειρισμό και σχολιασμό PDF.

## Βήμα 1: Φόρτωση του εγγράφου PDF

Πρώτα απ 'όλα, πρέπει να φορτώσετε το έγγραφο PDF όπου σκοπεύετε να προσθέσετε τη σχολίαση.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DATA DIRECTORY";
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

Να τι συμβαίνει: καθορίζετε τον κατάλογο όπου είναι αποθηκευμένο το αρχείο PDF και, στη συνέχεια, το φορτώνετε χρησιμοποιώντας το `Document` κλάση που παρέχεται από το Aspose.PDF. Αυτό το βήμα είναι κρίσιμο επειδή χωρίς να φορτώσετε το έγγραφο, δεν μπορείτε να κάνετε καμία αλλαγή σε αυτό.

## Βήμα 2: Δημιουργήστε μια σχολίαση

### Ορισμός των ιδιοτήτων σχολιασμού
Τώρα, ας δημιουργήσουμε την ίδια την σχολίαση. Θα χρησιμοποιήσουμε ένα `TextAnnotation`, το οποίο είναι ιδανικό για την προσθήκη σχολίων ή σημειώσεων στο PDF σας.

```csharp
// Δημιουργία σχολιασμού
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

Σε αυτό το απόσπασμα:
- Τοποθεσία και Μέγεθος: Το `Rectangle` Η κλάση ορίζει πού στη σελίδα θα εμφανίζεται η σχολίασή σας και τις διαστάσεις της.
- Τίτλος, Θέμα και Περιεχόμενα: Αυτές οι ιδιότητες σάς επιτρέπουν να καθορίσετε περί τίνος πρόκειται η σχολίασή σας και τι θα περιέχει.
- Εικονίδιο: Το `TextIcon.Key` ορίζει ένα εικονίδιο για τη σχολίαση, καθιστώντας την οπτικά πιο ελκυστική.

## Βήμα 3: Προσαρμόστε την εμφάνιση του σχολιασμού

Στη συνέχεια, ας κάνουμε αυτήν την σχολίαση να ξεχωρίζει προσθέτοντας ένα περίγραμμα και τροποποιώντας την εμφάνισή της.

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

Ακολουθεί μια ανάλυση των όσων συμβαίνουν:
- Σύνορο: Δημιουργούμε ένα `Border` αντικείμενο και ορίστε το πλάτος του σε 5, δίνοντας στην σχολίασή μας ένα εμφανές περίγραμμα.
- Μοτίβο παύλας: Το `Dash` Η ιδιότητα σάς επιτρέπει να δημιουργήσετε ένα διακεκομμένο περίγραμμα, προσθέτοντας λίγο στυλ στη σχολίαση.

## Βήμα 4: Προσθέστε την σχολίαση στη σελίδα PDF

Αφού δημιουργήσετε και προσαρμόσετε τη σχολίαση, ήρθε η ώρα να την προσθέσετε στη σελίδα PDF σας.

```csharp
// Προσθήκη σχολιασμού στη συλλογή σχολιασμών της σελίδας
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

Αυτός ο κώδικας προσθέτει την σχολίαση στην πρώτη σελίδα του PDF σας. `Annotations` Η συλλογή περιέχει όλες τις σχολιάσεις για μια συγκεκριμένη σελίδα και αυτό το βήμα διασφαλίζει ότι η νέα σας σχολίαση αποτελεί μέρος αυτής της συλλογής.

## Βήμα 5: Αποθηκεύστε το ενημερωμένο έγγραφο PDF

Τέλος, ας αποθηκεύσουμε το έγγραφο, ώστε η σχολίασή σας να προστεθεί μόνιμα.

```csharp
// Αποθήκευση αρχείου εξόδου
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nAnnotation added successfully.\nFile saved at " + dataDir);
```

Αποθηκεύοντας το έγγραφο με νέο όνομα (`AddAnnotation_out.pdf`), διατηρείτε το αρχικό αρχείο και δημιουργείτε ένα νέο με την προσθήκη της σχολίασης. Το μήνυμα της κονσόλας επιβεβαιώνει ότι όλα ήταν επιτυχή και τώρα μπορείτε να βρείτε το σχολιασμένο PDF σας στον καθορισμένο κατάλογο.

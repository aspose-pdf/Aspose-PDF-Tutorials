# Ajout de texte souligné dans un PDF avec Aspose.PDF pour .NET

Aspose.PDF pour .NET est une puissante bibliothèque de traitement PDF qui permet aux développeurs de créer, modifier et manipuler des documents PDF sans Adobe Acrobat. Cet exemple montre comment créer un document PDF et ajouter du texte souligné à l'aide des classes TextBuilder et TextFragment. Le code crée un PDF simple avec un message de test formaté en police Arial avec un style souligné, et le positionne à des coordonnées spécifiques dans la page.

## Instructions d'installation

1 : Installer Aspose.PDF pour .NET

Vous pouvez installer Aspose.PDF pour .NET en utilisant l’une des méthodes suivantes :

Utilisation du gestionnaire de packages NuGet :

```csharp
Install-Package Aspose.PDF
```

Utilisation de .NET CLI :

```bash
dotnet add package Aspose.PDF
```

Utilisation de PackageReference dans votre fichier de projet :

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2 : Ajouter des instructions d'utilisation obligatoires

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

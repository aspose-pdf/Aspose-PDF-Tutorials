# Ajouter du texte souligné dans PDF avec Aspose.PDF pour .NET

Aspose.PDF pour .NET est une puissante bibliothèque de traitement de PDF qui permet aux développeurs de créer, modifier et manipuler des documents PDF sans avoir besoin d'Adobe Acrobat. Cet exemple démontre comment créer un nouveau document PDF et ajouter du texte avec mise en forme sous-tendue en utilisant les classes TextBuilder et TextFragment. Le code crée un PDF simple avec un message de test formaté en police Arial avec mise en forme sous-tendue, le positionnant à des coordonnées spécifiques dans la page.

## Instructions d'installation

1: Installez Aspose.PDF pour le .NET

Vous pouvez installer Aspose.PDF pour .NET en utilisant l'une des méthodes suivantes:

Utiliser le gestionnaire de paquets NuGet:

```csharp
Install-Package Aspose.PDF
```

Utiliser l' interface utilisateur .NET:

```bash
dotnet add package Aspose.PDF
```

Utiliser PackageReference dans votre fichier de projet:

```xml
<PackageReference Include="Aspose.PDF" Version="23.1.0" />
```

2: Ajouter des instructions d'utilisation requises

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

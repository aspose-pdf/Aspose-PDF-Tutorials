# Tracer des lignes diagonales dans un PDF avec Aspose.PDF pour .NET

Aspose.PDF pour .NET permet aux développeurs de créer, manipuler et modifier facilement des documents PDF. Dans cet exemple, nous montrons comment tracer deux lignes diagonales sur une page PDF en C#. Vous apprendrez à initialiser un document PDF, à définir les propriétés de la page, à ajouter des éléments graphiques et à enregistrer le résultat final. Ce guide propose une approche étape par étape pour faciliter l'intégration de cette fonctionnalité dans vos applications .NET.

## Instructions d'installation :  

Pour utiliser Aspose.PDF pour .NET dans votre projet, suivez ces étapes :  

1. Installer via le gestionnaire de packages NuGet  
- Ouvrez Visual Studio et créez un nouveau projet .NET.  
- Ouvrez la console du gestionnaire de packages (Outils → Gestionnaire de packages NuGet → Console du gestionnaire de packages) et exécutez :  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Installation via .NET CLI  
Vous pouvez également utiliser l'interface de ligne de commande .NET pour installer le package :  
```bash
dotnet add package Aspose.PDF
```

3. Ajoutez un espace de noms dans votre code C#  
Pour utiliser la fonctionnalité Aspose.PDF, incluez les espaces de noms suivants :  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

# Dessiner des lignes diagonales en PDF à l'aide de Aspose.PDF pour .NET

Aspose.PDF pour .NET permet aux développeurs de créer, manipuler et modifier facilement des documents PDF. Dans cet exemple, nous démontrons comment dessiner programmatiquement deux lignes diagonales sur une page PDF en utilisant C#. Vous apprendrez à initialiser un document PDF, à définir les propriétés de la page, à ajouter des éléments graphiques et à enregistrer la sortie finale. Ce guide fournit une approche étape par étape, ce qui facilite l'intégration de cette fonctionnalité dans vos applications .NET.

## Instructions d'installation:  

Pour utiliser Aspose.PDF pour .NET dans votre projet, suivez ces étapes:  

1. Installer par le gestionnaire de paquets NuGet  
* Ouvrez Visual Studio et créez un nouveau projet .NET.  
* Ouvrez la console du gestionnaire de paquets (outils → NuGet Package Manager → Package Manager Console) et exécutez:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Installer par l'intermédiaire de l'interface utilisateur .NET  
Utilisez l'interface utilisateur .NET pour installer le package:  
```bash
dotnet add package Aspose.PDF
```

3. Ajouter des espaces de noms dans votre code C#  
Pour utiliser la fonctionnalité Aspose.PDF, inclure les espaces de noms suivants:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

# Dessinez des lignes diagonales en PDF à l'aide de Aspose.PDF pour .NET

Aspose.PDF pour .NET permet aux développeurs de créer, manipuler et modifier facilement des documents PDF. Dans cet exemple, nous montrons comment dessiner de manière programmatique deux lignes diagonales sur une page PDF en utilisant C#. Vous apprendrez à initialiser un document PDF, définir les propriétés de la page, ajouter des éléments graphiques et enregistrer la sortie finale. Ce guide fournit une approche étape par étape, ce qui facilite l'intégration de cette fonctionnalité dans vos applications .NET.

## Instructions d'installation:  

Pour utiliser Aspose.PDF pour .NET dans votre projet, suivez ces étapes:  

1. Installez par le gestionnaire de paquets NuGet  
* Ouvrez Visual Studio et créez un nouveau projet .NET.  
* Ouvrez la console de gestion de paquets (outils → NuGet Package Manager → Console de gestion de paquets) et exécutez:  
  ```powershell
  Install-Package Aspose.PDF
  ```

2. Installer par le biais de .NET CLI  
Alternativement, utilisez le CLI .NET pour installer le package:  
```bash
dotnet add package Aspose.PDF
```

3. Ajouter l'espace de noms dans votre code C#  
Pour utiliser la fonction Aspose.PDF, inclure les espaces de noms suivants:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

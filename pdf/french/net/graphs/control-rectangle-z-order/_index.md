# Créer et personnaliser un PDF avec des rectangles

Ce guide explique comment créer un document PDF avec Aspose.PDF pour .NET, en intégrant des rectangles colorés avec différents indices Z pour contrôler leur superposition. Suivez les instructions étape par étape pour ajouter des formes à votre PDF et enregistrer le document, améliorant ainsi les capacités de génération de PDF de votre application.

## Instructions d'installation
1. Assurez-vous que vous disposez d’un environnement de développement .NET configuré (tel que Visual Studio).
2. Téléchargez la bibliothèque Aspose.PDF depuis le [Site Web d'Aspose](https://www.aspose.com/purchase).
3. Installez le package NuGet Aspose.PDF en exécutant `Install-Package Aspose.PDF` dans la console du gestionnaire de paquets.
4. Incluez les espaces de noms nécessaires dans votre projet :
   ```csharp
   using Aspose.Pdf;
   using Aspose.Pdf.Drawing;
   ```

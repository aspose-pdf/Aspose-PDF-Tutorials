## Instructions d'installation  
Pour utiliser Aspose.PDF pour .NET dans votre projet, suivez ces étapes:  

1. Installez avec NuGet  
Exécutez la commande suivante dans la console NuGet Package Manager:  
```powershell
Install-Package Aspose.PDF
```

2. Ajouter des espaces de noms en C#  
Incluez les espaces de noms requis au début de votre fichier C#:  
```csharp
using Aspose.Pdf;
using Aspose.Pdf.Drawing;
```

Pour une documentation détaillée, consultez:  
[Aspose.PDF pour la documentation .NET](https://docs.aspose.com/pdf/net/)  
[Aspose.PDF pour la référence .NET API](https://reference.aspose.com/pdf/net/)

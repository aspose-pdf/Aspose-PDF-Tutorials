# Programmation avec annotations

Une collection complète de tutoriels expliquant comment utiliser les annotations dans les documents PDF avec Aspose.PDF pour .NET. Elle propose un large éventail d'extraits de code expliquant comment créer et manipuler différents types d'annotations, telles que du texte, des lignes, des cercles, des rectangles, des polygones, des hyperliens, et bien d'autres. Chaque exemple inclut le code C# complet, ainsi qu'une explication détaillée de ses fonctionnalités et du résultat attendu. C'est une excellente ressource pour les développeurs .NET souhaitant apprendre à utiliser les annotations dans les documents PDF avec Aspose.PDF.

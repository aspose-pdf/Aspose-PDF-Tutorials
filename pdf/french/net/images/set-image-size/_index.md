# Créer et modifier un PDF avec des images à l'aide d'Aspose.PDF

Aspose.PDF pour .NET est une bibliothèque puissante qui permet aux développeurs de créer, modifier et manipuler des documents PDF par programmation. Dans cet exemple, vous apprendrez à générer un document PDF, à ajouter une image, à ajuster sa taille et à modifier les propriétés de page en C#. Le code illustre l'initialisation d'un document PDF, l'insertion d'une image, la définition des dimensions de page et l'enregistrement du fichier final. Que vous travailliez avec des rapports, des factures ou du contenu dynamique, ce tutoriel propose une méthode simple et efficace pour gérer les images dans les PDF avec Aspose.PDF pour .NET.

## Instructions d'installation :  

Pour utiliser Aspose.PDF pour .NET dans votre projet, suivez ces étapes :  

1. Installer via le gestionnaire de packages NuGet :  
Ouvrez Visual Studio et exécutez la commande suivante dans la console du gestionnaire de packages :  
```powershell
Install-Package Aspose.PDF
```

2. Ajoutez un espace de noms dans votre code C# :  
```csharp
using Aspose.Pdf;
```

# Optimiser les images PDF avec Aspose.PDF

Cet exemple montre comment optimiser un document PDF en comprimant et redimensionnant les images à l'intérieur de celui-ci en utilisant Aspose.PDF pour .NET. En suivant le code fourni, vous pouvez réduire efficacement la taille du fichier de vos documents PDF tout en maintenant une qualité d'image acceptable. Ceci est particulièrement utile pour améliorer les performances et réduire les coûts de stockage.

## Instructions d'installation
1. Télécharger Aspose.PDF pour .NET: Visitez le[Aspose.PDF pour la page de téléchargement .NET](https://docs.aspose.com/pdf/net/)pour obtenir la dernière version.
2. Installez via NuGet: ouvrez votre projet dans Visual Studio et utilisez la commande suivante dans la console du gestionnaire de paquets:
   ```
   Install-Package Aspose.PDF
   ```

# Optimiser les images PDF avec Aspose.PDF

Cet exemple montre comment optimiser un document PDF en compressant et en redimensionnant les images qu'il contient avec Aspose.PDF pour .NET. En suivant le code fourni, vous pouvez réduire efficacement la taille de vos documents PDF tout en conservant une qualité d'image acceptable. Ceci est particulièrement utile pour améliorer les performances et réduire les coûts de stockage.

## Instructions d'installation
1. Téléchargez Aspose.PDF pour .NET : Visitez le [Page de téléchargement d'Aspose.PDF pour .NET](https://docs.aspose.com/pdf/net/) pour obtenir la dernière version.
2. Installer via NuGet : ouvrez votre projet dans Visual Studio et utilisez la commande suivante dans la console du gestionnaire de packages :
   ```
   Install-Package Aspose.PDF
   ```

## Pour plus de détails
documentation, visitez le [Référence de l'API Aspose.PDF pour .NET](https://reference.aspose.com/pdf/net/).

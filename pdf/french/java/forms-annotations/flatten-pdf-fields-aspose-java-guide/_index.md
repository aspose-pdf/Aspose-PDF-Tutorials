enregistre votre document PDF.\n\n```java\nvoid documentForm() {\n    // Initialize Form object to interact with the PDF\n    Form pdfForm = new Form();\n    \n    // Bind the source PDF file from 'dataDir/input.pdf'\n    pdfForm.bindPdf(dataDir + \"/input.pdf\");\n    \n    // Flatten all fields in the PDF document\n    pdfForm.flattenAllFields();\n    \n    // Save the modified document to 'outputDir/output.pdf'\n    pdfForm.save(outputDir + \"/output.pdf\");\n}\n```\n\n#### Explication :\n- **`Form` Classe**: Utilisé pour manipuler les champs de formulaire dans un PDF.\n- **`bindPdf`

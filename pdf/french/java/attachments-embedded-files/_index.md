# Tutoriels sur les pièces jointes PDF et les fichiers intégrés pour Aspose.PDF Java

Maîtrisez les fonctionnalités des pièces jointes PDF grâce à nos tutoriels Aspose.PDF détaillés pour développeurs Java. Ces guides pas à pas expliquent comment ajouter, extraire et gérer des pièces jointes, intégrer des ressources telles que des polices et des images, utiliser du contenu lié et créer des documents composés par programmation. Chaque tutoriel fournit des exemples concrets de code Java qui montrent comment implémenter des documents PDF enrichis avec du contenu intégré provenant de diverses sources, et enrichir vos documents avec des ressources complémentaires.

## Tutoriels disponibles

### [Supprimez efficacement toutes les pièces jointes d'un PDF à l'aide d'Aspose.PDF pour Java](./remove-attachments-pdf-aspose-java/)
Découvrez comment supprimer efficacement toutes les pièces jointes de vos documents PDF avec Aspose.PDF pour Java. Ce guide couvre la configuration, l'implémentation du code et les applications pratiques.

### [Comment ajouter des pièces jointes à des fichiers PDF avec Aspose.PDF pour Java : Guide du développeur](./add-attachments-pdf-aspose-pdf-java/)
Découvrez comment ajouter des pièces jointes, telles que des images ou des fichiers texte, à vos documents PDF avec Aspose.PDF pour Java. Ce guide couvre toutes les étapes, de la configuration à la mise en œuvre.

### [Comment extraire des fichiers intégrés à partir de PDF avec Aspose.PDF pour Java : guide complet](./extract-embedded-files-pdf-aspose-java-guide/)
Maîtrisez l'extraction de fichiers intégrés à partir de documents PDF avec Aspose.PDF pour Java. Ce guide couvre la configuration, la mise en œuvre étape par étape et des conseils de performance.

### [Comment extraire des fichiers intégrés d'un portfolio PDF à l'aide d'Aspose.PDF Java](./extract-files-pdf-portfolio-aspose-java/)
Apprenez à extraire efficacement des fichiers intégrés de portfolios PDF avec Aspose.PDF pour Java. Simplifiez la gestion de vos données grâce à ce guide étape par étape.

### [Maîtriser Aspose.PDF Java : accéder et gérer les fichiers intégrés dans les PDF](./master-aspose-pdf-java-access-manage-embedded-files/)
Découvrez comment utiliser Aspose.PDF pour Java pour accéder, gérer et extraire efficacement les propriétés des fichiers intégrés dans les documents PDF.

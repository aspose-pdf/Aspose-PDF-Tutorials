L'extraction de métadonnées telles que l'auteur, la date de création et les mots-clés permet d'organiser et de gérer efficacement les documents.\n\n##### Mise en œuvre étape par étape\n\n**1. Importer les bibliothèques requises**\n```java\nimport com.aspose.pdf.Document;\nimport com.aspose.pdf.DocumentInfo;\n```\n\n**2. Chargez le document PDF**\nDéfinissez votre répertoire de documents et chargez le fichier PDF.\n```java\nString dataDir = \"YOUR_DOCUMENT_DIRECTORY\";\nDocument pdfDocument = new Document(dataDir + \"/Original.pdf\");\n```\n\n**3. Extraire les informations de métadonnées**\nAccédez et récupérez les propriétés des métadonnées du document.\n\n```java\nDocumentInfo docInfo = pdfDocument.getInfo();\nString author = docInfo.getAuthor();\nString creationDate = docInfo.getCreationDate().toString();\nString keywords = docInfo.getKeywords();\nString modifyDate = docInfo.getModDate().toString();\nString subject = docInfo.getSubject();\nString title = docInfo.getTitle();\n```\n\n**Explication**: Cet extrait ouvre un document PDF et récupère ses métadonnées à l'aide de `DocumentInfo`qui fait partie de la bibliothèque Aspose.PDF. Les méthodes comme `getAuthor()` et `getCreationDate()` Extraire des éléments spécifiques de métadonnées.\n\n### Fonctionnalité 2 : Enregistrer les métadonnées du fichier PDF\nCette fonctionnalité couvre l'enregistrement des métadonnées extraites dans un fichier texte pour une utilisation ultérieure ou la conservation d'archives.\n\n#### Présentation\nUne fois les métadonnées extraites, il est souvent utile d'enregistrer ces informations dans un format structuré, tel qu'un fichier texte brut.\n\n##### Mise en œuvre étape par étape\n\n**1. Écrire des métadonnées dans un fichier**\nConfigurez votre répertoire de sortie et préparez le FileWriter.\n\n```java\nString outputDir = \"YOUR_OUTPUT_DIRECTORY\";\ntry (FileWriter writer = new FileWriter(outputDir + \"/pdf_info.txt\")) {\n    // Writing each metadata field on a separate line for clarity\n    writer.write(\"Author:-\" + author + \"\\n\");\n    writer.write(\"Creation Date:-\" + creationDate + \"\\n\");\n    writer.write(\"Keywords:-\" + keywords + \"\\n\");\n    writer.write(\"Modify Date:-\" + modifyDate + \"\\n\");\n    writer.write(\"Subject:-\" + subject + \"\\n\");\n    writer.write(\"Title:-\" + title);\n} catch (IOException e) {\n    e.printStackTrace();\n}\n```\n\n**Explication**: Cet extrait de code écrit chaque élément de métadonnées sur une ligne distincte dans `pdf_info.txt`.

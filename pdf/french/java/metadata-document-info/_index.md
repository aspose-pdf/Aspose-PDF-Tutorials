# Tutoriels sur les métadonnées PDF et les informations sur les documents pour Aspose.PDF Java

Nos tutoriels sur les métadonnées et les informations documentaires vous expliquent comment gérer les propriétés des documents PDF avec Aspose.PDF en Java. Ces guides pratiques expliquent comment accéder aux métadonnées standard des documents et les modifier, créer des champs de métadonnées personnalisés, implémenter des métadonnées XMP et utiliser des dictionnaires d'informations documentaires. Chaque tutoriel inclut des exemples de code Java détaillés pour manipuler les métadonnées conformément aux normes PDF, vous aidant ainsi à créer des applications qui respectent l'attribution et la classification des documents.

### [Extraire les métadonnées XMP des fichiers PDF avec Aspose.PDF pour Java : guide complet](./extract-xmp-metadata-pdf-aspose-pdf-java/)
Apprenez à extraire et gérer les métadonnées XMP dans les fichiers PDF avec Aspose.PDF pour Java. Ce guide couvre la configuration, des exemples de code et des applications pratiques.

Découvrez comment accéder aux propriétés PDF et les modifier avec Aspose.PDF pour Java. Ce guide couvre les métadonnées des documents, les paramètres des fenêtres, l'ordre de lecture, etc.
